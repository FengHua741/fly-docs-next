# FLY-Tools 簡介

    FLY-Toolsは広州鎂倫電子科技有限公司がKlipper向けにカスタマイズしたWebデバッグツールで、Flyホストコンピュータ（システムにバンドルされているためインストール不要）や、市場に出回っている大半のKlipperサービスを搭載したホストコンピュータに対応しています。

    :::tip ヒント

    * FLY-ToolsはKlipperをインストールしたLinuxシステムのみ対応しています

    :::
        

## FLY-Toolsの特長

    * 多言語対応、ダークテーマ対応
    * USBシリアルポートID、USBデバイスID、CANバスUUID、ビデオデバイスIDの照会とワンクリックコピー機能をサポート
    * 特定のCANデバイスの有効化または無効化をワンクリックでサポート
    * オンラインでCANデバイスのレートおよび送信バッファサイズの変更をサポート
    * オンラインでFLYマザーボードのklipperファームウェアのコンパイルをサポート（Katapultファームウェアのコンパイルは調整中です）
    * DFU、HID、CANなどの焼込方法をサポートするオンライン焼込をサポート
    * すべてのコンパイル済みファームウェアファイルのダウンロードをサポート
    * オンラインでKlipperロードグラフの生成をサポート
    * 共振グラフの自動測定と生成をワンクリックでサポート
    * Web SSH接続をサポート
    * [ttyd](https://github.com/tsl0922/ttyd)ベースのWebページSSHをサポート
    * FLYOS 3.3.4バージョンシステムをサポートし、画面の設定、ウェブ表示などを構成できます
