# FLY-Tools 簡介

    FLY-Toolsは広州鎂倫電子科技有限公司がKlipper向けにカスタマイズしたWebデバッグツールです。Fly上位機（システムにバンドルされているため、インストール不要）に対応しており、市販のほとんどのKlipperサービスを搭載した上位機とも互換性があります。

    :::tip 注意

    * FLY-ToolsはKlipperがインストールされたLinuxシステムでのみ使用可能です

    :::
        

## FLY-Toolsの特長

    * 多言語対応、ダークテーマ対応
    * USBシリアルポートID、USBデバイスID、CANbus UUID、VideoデバイスIDを一括で検索し、一键コピー機能をサポート
    * 指定したCANデバイスの有効化や無効化を一键でサポート
    * オンラインでCANデバイスの通信速度や送信バッファサイズを変更可能
    * FLYマザーボードのKlipperファームウェアをオンラインでコンパイル可能（Katapultファームウェアのコンパイルは現在調整中）
    * DFU、HID、CANなど複数の焼込方法をサポートするオンライン焼込機能
    * すべてのコンパイル済みファームウェアファイルをダウンロード可能
    * Klipperの負荷グラフをオンラインで生成可能
    * 共振グラフを自動測定し、一键で生成可能
    * Web SSH接続をサポート
    * [ttyd](https://github.com/tsl0922/ttyd)ベースのWebページSSHをサポート
    * FLYOS 3.3.4バージョンシステムに対応し、画面設定やWeb表示などをカスタマイズ可能
