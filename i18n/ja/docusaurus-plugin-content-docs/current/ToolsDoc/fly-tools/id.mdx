# 機能デモ

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/flash.gif').default} size="100%" align="left" />

    * FLYマザーボードのファームウェアコンパイルをサポート
    * CANレートを選択可能
    * オンライン焼付をサポート、DFU、HID、CANなどの焼付方法をサポート
    * Katapult焼付をサポート（旧CanBoot）
    * すべてのコンパイル済みファームウェアファイルをダウンロード可能
