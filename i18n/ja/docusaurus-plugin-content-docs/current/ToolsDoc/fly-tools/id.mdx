# 機能デモンストレーション

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/flash.gif').default} size="100%" align="left" />

* FLYメインボードファームウェアのコンパイルをサポート
* CANレートを選択可能
* オンラインプログラミングをサポート、DFU、HID、CANなどのプログラミング方法をサポート
* Katapultプログラミングをサポート（元CanBoot）
* すべてのコンパイル済みファームウェアファイルのダウンロードをサポート
