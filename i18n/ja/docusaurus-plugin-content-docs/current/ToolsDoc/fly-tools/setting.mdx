# 機能デモンストレーション

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.webp').default} size="100%" align="left" />


    * ホスト名の変更がサポートされています
    * システムのタイムゾーンの変更がサポートされています
    * 上位機40PIN拡張の有効化がサポートされています（PWMを有効にすると、上位機とのシリアル接続が使用できなくなります。画面を有効にすると、上位機の加速度センサーへの接続が使用できなくなる可能性があります）
    * マザーボードのモデルの変更がサポートされています
    * Klipperウェブページの変更がサポートされており、FuiddとMainsailもサポートされています
    * KlipperScreenの開閉がサポートされています
    * Klipperプロセスを指定されたCPUコアで強制的に実行する機能がサポートされており、選択したコア上でKlipperを実行できます
