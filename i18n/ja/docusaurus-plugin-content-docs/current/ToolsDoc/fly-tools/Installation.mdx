# FLY-Toolsのインストール

    1. 最新バージョンのKlipperを取得する

    ```bash
    cd ~/klipper
    git pull
    ```

    * ローカルKlipperソースコードを変更した場合、取得に失敗することがあります。その場合は以下のコマンドを使用してください。（この操作により、以前の変更されたファイルは失われます）

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. 最新版のKatapult（元Canboot）を取得する

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Fly-Toolsサービスをインストールする

        * インストールプロセスはGitサービスプロバイダーに依存します。ダウンロードが停止している場合は、`CTRL+C` を押して再試行してください
        * Githubの最大リクエスト回数は毎時60回です。Giteeは不明です
        * 国内のユーザーはインストールスクリプトをGiteeから取得することをお勧めします
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Githubからインストールスクリプトを取得する

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
