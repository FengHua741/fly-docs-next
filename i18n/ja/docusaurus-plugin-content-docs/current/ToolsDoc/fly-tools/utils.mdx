# 機能デモンストレーション

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/utils.gif').default} size="100%" align="left" />

    * Klipperのデフォルト負荷図の生成をサポートしています
    * MCU周波数図の生成をサポートしています
    * システム負荷図の生成をサポートしています
    * 温度曲線図の生成をサポートしています
    * フリクエンシーレスポンス図の生成をサポートしており、既存のファイルから直接チャートを生成することも可能です


# `matplotlib`が不足している場合の警告メッセージが出た場合は、以下のコマンドを実行してください

    ```bash
    ~/klippy-env/bin/pip install -v matplotlib
    ```
