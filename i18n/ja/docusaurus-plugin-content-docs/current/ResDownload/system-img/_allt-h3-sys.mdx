# {props.name}システムダウンロード

:::warning

SDカードの注意点:

* 容量は`16GB`以上または`128G`以下のTFカードであること
* TFカードの転送速度は`class10`級またはそれ以上であること、サンディスクなどのブランドのTFカードを使用することを推奨します

:::

:::danger 必ずお読みください

* サーバーの問題により、システムやklipperを随意に更新しないでください
* FLY OSシステムをダウンロードしてください。オリジナルのArmbianのインストールは難易度が高いため推奨しません

:::

:::info 重要なご案内

* ダウンロードしたシステムは解凍する必要があります。拡張子を変更したり、直接焼成したりすることはできません！！！！
* 解凍せずに焼成すると起動できません！！！

:::
