# {props.name}システムダウンロード

:::warning

SDカードの注意事項:

* 容量は`16GB`以上または`128GB`以下のTFカードであること
* TFカードの転送速度は`class10`級またはそれ以上であること、サンディスクなどのブランドを使用することをお勧めします

:::

:::danger 注意

* サーバーの問題により、システムやklipperを随意に更新しないでください
* FLYOS-Armbianシステムをダウンロードしてください。オリジナルのArmbianの使用難易度が高いため、インストールはお勧めしません

:::

:::info 重要なポイント

* ダウンロードしたシステムは解凍する必要があります。後缀を変更したり、直接書き込むことはできません！！！！
* 解凍せずに書き込むと起動しません！！

:::
