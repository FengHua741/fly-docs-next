# {props.name}システムダウンロード

:::warning

SDカードの注意点:

* 容量は`16GB`以上`128GB`以下のTFカードであること
* TFカードの転送速度は`class10`級またはそれ以上で、SanDiskなどのブランドのTFカードが推奨されます

:::

:::danger 必読

* サーバーの問題により、システムやklipperを随意に更新しないでください
* FLY OSシステムをダウンロードしてください。オリジナルのArmbianはインストールが難しく推奨されません

:::

:::info 重要な注意点

* ダウンロードしたシステムは解凍する必要があります。後ろの拡張子を変更したり、直接焼成することはできません!!!!
* 解凍せずに焼成すると起動できません!!!!

:::
