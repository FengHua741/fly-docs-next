# {props.name}システムダウンロード

:::warning

SDカードの注意事項:

* 容量は`16GB`以上`128GB`以下のTFカードであること
* TFカードの転送速度は`class10`級またはそれ以上であることが必要で、サンディスクなどのブランドのTFカードを推奨します

:::

:::danger 注意点

* サーバーの問題があるため、システムやklipperを随意に更新しないでください
* FLYOS-Armbianシステムをダウンロードしてください。オリジナルのArmbianのインストールは難易度が高いため、推奨されません

:::

:::info 重要なポイント

* ダウンロードしたシステムは解凍する必要があります。後缀名を変更したり、直接書き込むことはできません！！！！
* 解凍せずに書き込むと起動しません！！

:::
