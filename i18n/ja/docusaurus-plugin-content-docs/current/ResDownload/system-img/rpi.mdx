# ラズベリーパイイメージ


  :::warning

  SDカードの注意点:

  * 容量は必ず`16GB`以上`128GB`以下に設定してください
  * SDカードの転送速度は`class10`級またはそれ以上のもので、サンディスクなどのブランドのSDカードを使用することをお勧めします

  :::

  :::danger 注意事項

  * サーバーの問題により、システムやklipperを随意にアップデートしないでください
  * ラズベリーパイ3B/3B+/4B/CM4/5B向けのシステムイメージです
  * このイメージは各種サービスを`bookworm`ベースでインストールしています
  * このイメージではTFTスクリーンがデフォルトでインストールされ、起動しています。TFTスクリーンを使用しない場合は、`/boot/config.txt`を手動で変更する必要があります

  :::

  :::info 重要なヒント

  * ダウンロードしたシステムは解凍する必要があります。後ろの拡張子を変更したり、そのまま焼成することはできません！！！
  * 解凍せずに焼成すると、起動しません！！

  :::
