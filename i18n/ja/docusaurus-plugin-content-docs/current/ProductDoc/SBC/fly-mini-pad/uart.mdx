# 設定方法

:::danger 注意
この方法を使用した後、mini padの加速度インターフェースを介して加速度計を外付けすることはできません。
:::

    * BOOTドライブを開き、`armbianEnv.txt`を見つけ開く
    * `overlays=usbhost2 usbhost3 uart2`の**uart2**を**uart3**に変更し保存するだけで完了です。

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart.webp').default} size="80%" align="center" />

## ケーブル接続方法

    * mini-padのTXはシリアルポートファームウェアが焼かれたメインボードのRXに接続してください
    * mini-padのRXはシリアルポートファームウェアが焼かれたメインボードのTXに接続してください
    * 地線も共有する必要があります
    * そうしないと、シリアルポートファームウェアが焼かれたメインボードに接続できません

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart1.webp').default} size="80%" align="center" />

## Klipperリファレンス設定

    ```cfg
    [mcu]
    serial: /dev/ttyS3
    baud：250000
    #   使用されるボーレート。デフォルトは250000
    restart_method:command
    ```
