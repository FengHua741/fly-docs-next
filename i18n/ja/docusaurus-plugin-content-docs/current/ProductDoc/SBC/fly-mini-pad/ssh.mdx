# 上位機とのSSH接続


* シリアルポートまたはWiFiでデバイスに接続することができます
* 事前に**MobaXterm_Personal**または他の**SSHターミナルツール**を準備してください
* SDカードまたはM2WEに**対応する上位機のシステム**が焼き込まれていることを確認してください
* **もし上位機が正常に起動しない場合はドライバ、リミッター、ファンなどの外部機器を取り外してください！！**
* **最後にネットワーク経由でSSHに接続してファームウェアをコンパイルすることが便利です！！**
* **ファームウェアをコンパイルする必要がある場合は、WiFiまたはイーサネットケーブルで上位機に接続するチュートリアルを参照してください**

        ## 上位機SSHにネットワークIPで接続します

        * マザーボードが起動しており、ネットワークに接続されている場合、IPでSSHに接続することができます
        * MobaXtermを開き、以下の設定を行います。IPは確認したものを入力します。また、ルーターのバックエンドでも確認できます
