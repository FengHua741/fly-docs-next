# WIFIに接続

## MINI PADにアンテナをインストールする

    * 赤い矢印が指している場所がWIFIアンテナインターフェースです
    * WIFIアンテナポートは`ipex1`です
    * アンテナをMINI PADにインストールしないと、信号が弱いためネットワークに接続できない可能性があります
