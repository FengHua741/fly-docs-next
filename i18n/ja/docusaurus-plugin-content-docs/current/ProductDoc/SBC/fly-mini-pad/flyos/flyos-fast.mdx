# システムイメージ焼込

<FASTFASLH name="H3" />


## システムカードのインストール


    * TFカードにシステムを焼込んだ後、システムが起動するにはTFカードをMINI PADの背面に接続する必要があります。そうでない場合、システムは起動しません。
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/tf.webp').default} size="100%" align="left" />  


## アンテナの取り付け

    1. アンテナを取り付けます
    :::warning
    * 上位機器では中国語名前のWi-Fiはサポートされていません。また、5G Wi-Fiもサポートされていません。
    :::
    * 赤い矢印で示されている部分がWi-Fiアンテナのインターフェースです。
    * Wi-Fiアンテナポートは`ipex1`です。
    * 信号強度が弱いため、ルーターに接続できない可能性がありますので、必ずアンテナを取り付けしてください。
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードのインストールが完了したら、次に進み、SSHで上位機器に接続できます。
    :::

    2. 上位機器に電源を供給します。  
    **ここでは、あなたの上位機器はすでに起動しています！！！**  
    次の手順へ進んでください：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-mini-pad/ssh">上位機器へのSSH接続</Button>
