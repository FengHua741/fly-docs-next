# システム焼込み

## 上位機初心者ガイド

<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/baodian/host/">上位機初心者ガイド</Button>

## システム焼込み前の準備

* リーダー
* 容量が16GB以上または128GB以下のTFカード1枚、転送速度はクラス10またはそれ以上のもの
* 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-minipad">システムイメージ</Button> をダウンロード
* 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込みツール</Button> をダウンロード

## システム焼込み
:::info
* MINI PAD ではシステムをTFカードにしか焼込めません。M2WEは使用できません。
* システム焼込み後はTFカードをMINI PADに取り付けなければなりません。
:::
    
<TF />

## システムカードのインストール

* システムをTFカードに焼込んだ後、MINI PADの背面にTFカードを取り付けなければ、システムが起動しません。
<ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/tf.webp').default} size="100%" align="left" />  

## アンテナの取り付け

1. アンテナを設置します。
:::warning
* 注意してください、上位機は中国語名前のWi-Fiをサポートしていません。また、5G Wi-Fiもサポートしていません。
:::
* 赤い矢印で示された部分がWi-Fiアンテナ接続口です。
* Wi-Fiアンテナポートは`ipex1`です。
* アンテナを設置しないと、信号が弱いためルーターに接続できない可能性があります。
<ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/wifi.webp').default} size="80%" align="left" /> 

:::tip
- システムカードのインストールが完了したら、次に進み、SSHで上位機に接続できます。
:::

2. 上位機に電源を供給します。  
**ここでは、あなたの上位機はすでに起動できます！！！**  
次の手順に進んでください：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-mini-pad/ssh">上位機にSSH接続</Button>
