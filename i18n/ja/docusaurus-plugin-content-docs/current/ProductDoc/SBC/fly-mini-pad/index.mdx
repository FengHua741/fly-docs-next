# 簡介

    Fly-MINI-PadはエントリーレベルのKlipperホスト機器です

## 特點

    * CPU：高性能全志H3チップ、4コア64ビットCortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：最大128GB SDCARDをサポート
    * WIFI :  ボードに内蔵された2.4G Wi-Fi
    * 外設：SPI x1、USB x2、Type-C x1
    * 接口：三つのXH2.54-2Pインターフェース、それぞれがPOWER-IN電源供給をサポートし、最高24Vの入力が可能です、CAN-BUSはCANバス接続をサポートします、LI-POは3.7Vで保護付きのリチウムイオンバッテリー供給/充電インターフェースをサポートします
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、共振補償プラグインnumpyとFLY-Toolsがプリインストールされています。複雑な操作を必要とせず、初心者でも使いやすい（TFカードを購入する必要があります）
    * スクリーンは静電容量式タッチスクリーン/抵抗式タッチスクリーンで、解像度は320×480です
