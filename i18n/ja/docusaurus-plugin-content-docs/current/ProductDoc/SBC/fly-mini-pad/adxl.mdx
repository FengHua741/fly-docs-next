# 加速度計の使用

    :::tip

    * Mini Padはボード内インターフェースを使用して加速度計に直接接続するか、ツールボードなどに組み込まれた加速度計を使用できます

    :::

## ボード内インターフェースを介した外付け加速度計

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.webp').default} size="80%" align="left" />

    * 参考設定

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # an example
    ```
