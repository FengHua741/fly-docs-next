# ボード内蔵UTOCの使用

:::danger 注意

2つのUTOICを使用することは推奨されません。書き込みと接続は非常に煩雑になります。

:::

* ボード内蔵UTOCを使用しない場合、以下のコマンドを実行する必要があります。実行には任何提示も表示されません。
    ```bash
    sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
    ```
* 復元方法、実行には任何提示も表示されません。
    ```bash
    sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
    ```
* MINI Padには既にボード内蔵のUTOCが搭載されています。
