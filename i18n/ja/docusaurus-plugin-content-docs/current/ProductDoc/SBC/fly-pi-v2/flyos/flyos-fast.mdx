## システム紹介

* FlyOS-Fast は、3Dプリンタ向けに最適化された軽量なLinuxシステムで、高速起動を実現し、30秒以内で起動できます。FLY全シリーズのアップストリームプラットフォームに対応しています。
* カーネルの最適化により、Klipperのリアルタイム応答性能を向上させます。
* ファイルシステムの最適化により、突然の電源切断によるシステム破損を防ぎます。
* Klipper、RRF、Mainsail、Fluidd、Moonraker、KlipperScreenなどのサービスがプリインストールされています。
* KlipperとRRFシステムの切り替えが可能です。
* MainsailとFluiddのオンラインでのワンクリック切り替えが可能です。
* USBメモリによる印刷が可能です。
* KPPMモジュールを使用すると、電源断継続印刷が可能です。
* FluiddなどのフロントエンドUIのデフォルト言語やデフォルトプリンタ名を事前に設定可能です。
* 更なるOEMカスタマイズ機能：起動ロゴの設定、プリセット設定ファイルの適用、Mainsailタブページの非表示設定などはお問い合わせください。

## 準備作業

1. システムイメージのダウンロード
    - イメージリンク：[Piv2システムイメージダウンロード](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "クリックするとリンクにジャンプします")（Windowsのみ対応）  
    - イメージリンク：[Piv2システムイメージダウンロード](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "クリックするとリンクにジャンプします")（Windows、MacOS、Linux対応）

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. 焼き込みツールのダウンロード。このチュートリアルでは`Rufus`を使用します。実際には他のツールも利用可能で、例えば`Etcher`、`Win32 Disk Imager`などです。
    - Rufusのダウンロードリンク：[クリックしてダウンロード（中国語）](https://rufus.ie/zh/#download) または [クリックしてダウンロード（英語）](https://rufus.ie/en/#download) （Windowsのみ対応）。
    - Etcherのダウンロードリンク：[クリックしてダウンロード](https://etcher.balena.io/#download-etcher) （Windows、MacOS、Linux対応）。
    - Win32 Disk Imagerのダウンロードリンク：[クリックしてダウンロード](https://sourceforge.net/projects/win32diskimager/) （Windowsのみ対応）。

3. このチュートリアルでは`FlyOS-Fast-H5.exe`を使用します。

4. システムイメージファイルの解凍を行い、`.img`ファイルを取得します。
    - `FlyOS-Fast-H5.exe`をダブルクリックまたは右クリックで開き、数秒待つと以下のウィンドウがポップアップします。（システムが危険と警告する場合は「より多くの情報」→「実行」を選択してください）
    - 解凍先フォルダを選択し、`Extract`ボタンをクリックして解凍を待ちます。
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - 解凍が完了すると、解凍フォルダ内に`FlyOS-Fast-H5-vx.x.x.img`ファイルが生成されます。`x.x.x`がシステムバージョン番号です。

## イメージ焼込み

    <Tabs>
        <TabItem value="TF" label="TFカードに焼込み" default>

            1. - 焼き込み用のTFカードは4GB以上必要で、推奨はSanDiskの32GB C10以上のTFカードを使用（テスト結果によると、安価なカードは焼込み後に起動しない場合がある）。
               - 焼き込み用のTFカードはSDカードリーダーを使用し、焼込みカードをリーダーに挿入し、その後パソコンにリーダーを挿入します。
            2. `Rufus`を開き、デバイスを選択します。選択したデバイスの容量は、パソコンに挿入した焼込みカードの容量と一致している必要があります。

            4. イメージファイルを選択すると、`ブートタイプ選択`はイメージファイルのファイル名になります。
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="40%" align="center" />

            6. 焼き込みが完了すると、コンピューターからピューピューという音が鳴り、進行状況バーが`準備完了`と表示されます。

## WiFi接続

    1. `vscode`または専門的なテキストエディターで`config.txt`を開きます。（**できる限りWindows標準のメモ帳は使用しないでください**）
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

    2. `WIFI_SSID=""`の`""`の中に、**WiFi名**を入力します。  
       `WIFI_PASS="""`の`""`の中に、**WiFiパスワード**を入力します。
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/osfast_wifi.webp').default} size="70%" align="center" />  
  
    3. これで、あなたのアップストリームマシンは起動できます！！！  
    次のステップへ進む：<Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh">SSHでアップストリームマシンに接続</Button>
