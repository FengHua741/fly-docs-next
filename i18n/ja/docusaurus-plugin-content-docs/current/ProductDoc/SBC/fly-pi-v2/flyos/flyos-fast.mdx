## システム概要

* FlyOS-Fast は、3Dプリントに深くカスタマイズされた軽量化されたLinuxシステムで、起動が速く、30秒で起動できます。Flyの全シリーズの上位機プラットフォームをサポートしています
* カーネルを最適化し、Klipperのリアルタイム応答パフォーマンスを向上させます
* ファイルシステムを最適化し、直ちに電源を切るとシステムが損傷するのを防ぎます
* Klipper、RRF、Mainsail、Fluidd、Moonraker、KlipperScreenなどのサービスを事前にインストールしています
* KlipperとRRFシステムの切り替えをサポート
* MainsailとFluiddのオンラインでのワンクリック切り替えをサポート
* USBメモリ印刷をサポート
* KPPMモジュールと組み合わせることで、電源障害継続印刷を実現できます
* FluiddなどのフロントエンドUIのデフォルト言語やデフォルトプリンタ名を事前に設定できます
* 他のOEMカスタマイズ機能（例：起動ロゴ、事前設定済みの設定ファイル、Mainsailタブページの非表示など）についてはお問い合わせください。

## 準備作業

1. システムイメージのダウンロード
    - イメージリンク：[Piv2 システムイメージダウンロード](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "クリックするとリンクに移動します")（Windowsのみ対応）  
    - イメージリンク：[Piv2 システムイメージダウンロード](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "クリックするとリンクに移動します")（Windows、MacOS、Linuxに対応）

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. バーンツールのダウンロード、チュートリアルでは`Rufus`を使用します。実際には他のツール（例：`Etcher`、`Win32 Disk Imager`など）も使用可能です。
    - Rufus ダウンロードアドレス：[クリックしてダウンロード (日本語)](https://rufus.ie/zh/#download) または [Click Download English](https://rufus.ie/en/#download) （Windowsのみ対応）。
    - Etcher ダウンロードアドレス：[クリックしてダウンロード](https://etcher.balena.io/#download-etcher) （Windows、MacOS、Linux対応）。
    - Win32 Disk Imager ダウンロードアドレス：[クリックしてダウンロード](https://sourceforge.net/projects/win32diskimager/) （Windowsのみ対応）。

3. チュートリアルでは`FlyOS-Fast-H5.exe`を使用します。

4. システムイメージファイルを解凍して`.img`ファイルを得ます。
    - `FlyOS-Fast-H5.exe`をダブルクリックまたは右クリックして開き、数秒待つと以下のウィンドウが表示されます（システムが危険と表示される場合、「詳細情報」→「実行」を選択してください）
    - 解凍ディレクトリを選択してから`Extract`ボタンをクリックし、解凍が完了するまで待ちます。
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - 解凍が完了すると、解凍ディレクトリに`FlyOS-Fast-H5-vx.x.x.img`ファイルが生成されます。`x.x.x`がシステムバージョン番号です。

## WiFiの設定

    1. `vscode`やその他の専門的なテキストエディターで`config.txt`を開きます（**できるだけWindowsの標準テキストエディターを使用しないでください**）
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

    2. `WIFI_SSID=""`オプション内の`""`に**WiFi名**を入力します  
       `WIFI_PASS="""`オプション内の`""`に**WiFiパスワード**を入力します  
       保存して終了し、`FlyOS-Conf`という名前のUディスクまたは可搬型ディスクがポップアップ表示されます。
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/osfast_wifi.webp').default} size="70%" align="center" />  

## 上位機の起動

    1. 起動する前に、**メモリカード**または**M2WE**がアップステーションに正しく取り付けられていることを確認してください
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/tf.webp').default} size="70%" align="center" />
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/m2we.webp').default} size="70%" align="center" />
  
    2. `M2WE`または`2.4G WiFi`モジュールのアンテナを設置します
    :::warning
    * FLYアップステーションは`M2WE`または`2.4G WiFi`モジュールを使用する必要があります。そうでないとWiFiを使用できません
    * 注意：アップステーションは中国語名前のWiFiをサポートしていません。また、2.4Gと5Gの同時統合WiFiはサポートしていません
    :::
    * 紅色矢印で示された部分がWiFiアンテナインターフェースです
    * WiFiアンテナ口は`ipex1`です
    * PI V2にアンテナを接続しないと、信号が弱いためルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" /> 

    3. アップステーションに電力を供給します。  
    **ここまでの作業で、あなたのアップステーションは起動しました！！！**  
    次のステップに進むには<Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh">SSHでアップステーションに接続</Button>
