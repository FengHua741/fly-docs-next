## アンテナの取り付け

    1. `M2WE`または`2.4G WiFi`モジュールのアンテナを取り付けます
    :::warning
    * FLY上位機は`M2WE`または`2.4G WIFI`モジュールを使用する必要があります。それ以外ではWiFiを使用できません。
    * 注意してください、上位機は中国語名前のWiFiをサポートしておらず、2.4Gと5Gの両方を統合したWiFiもサポートしていません。
    :::
    * レッドアローが指している部分がWiFiアンテナ接続部です
    * WiFiアンテナポートは`ipex1`です
    * 信号が悪い場合に接続がルーターにできない可能性があるため、必ずM2WEにアンテナを取り付けてください
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードのインストールが完了したら、次の手順に進み、SSHで上位機に接続できます
    :::

    2. 上位機に電源を供給します。  
    **ここでは、あなたの上位機はすでに起動できます！！！**  
    次の手順に進んでください:<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-pi-v2/ssh">上位機にSSH接続</Button>
