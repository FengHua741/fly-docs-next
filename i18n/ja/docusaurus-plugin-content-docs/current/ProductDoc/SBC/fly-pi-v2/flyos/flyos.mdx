## システム焼込前の準備

```html
* リーダー

* 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
* 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button> をダウンロード
```

## システム焼込

:::info
TFカード焼込とTFカードからM2WEへの焼込はどちらか一方を使用してください
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEはTFカードでシステムを焼込むことを推奨します。USB経由での焼込みはドライバーの互換性の問題などで失敗する可能性があるため、直接[M2WEへのメモリカード焼込](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)をお勧めします
* 適切に対応するシステムイメージをダウンロードしていることを確認してください。対応しない場合は起動できません
:::

## アンテナの設置

1. `M2WE`または`2.4G Wi-Fi`モジュールのアンテナを設置してください
:::warning
* FLY上位機器では`M2WE`または`2.4G Wi-Fi`モジュールを使用する必要があります。Wi-Fiを利用できない場合があります
* 注意：上位機器は中国語名前のWi-Fiをサポートしていません。2.4Gと5Gの同時対応Wi-Fiもサポートしていません
:::
* 赤い矢印が示す場所がWi-Fiアンテナインターフェースです
* Wi-Fiアンテナポートは`ipex1`です
* PI V2にアンテナを設置しないと、信号が弱いためルーターに接続できない可能性があります
