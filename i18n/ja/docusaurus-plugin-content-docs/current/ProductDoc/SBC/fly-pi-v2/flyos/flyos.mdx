## システム書き込み前の準備

    * リーダー
    * 容量が16GB以上または128GB以下のTFカード、転送速度がクラス10以上である必要があります
    * 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">書き込みツール</Button> をダウンロード

## システム書き込み

:::info
TFカードへの書き込みとTFカード経由でM2WEに書き込む方法はどちらかを選択できます
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードでシステムを書き込むことをお勧めします。USB経由でM2WEに書き込むと、ドライバの互換性などの理由で失敗する可能性があります。直接 [メモリーカードをM2WEに書き込む](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os) 方法をお勧めします
* ダウンロードしたシステムイメージが正しいかどうかに注意してください。正しくない場合は起動できません
:::
