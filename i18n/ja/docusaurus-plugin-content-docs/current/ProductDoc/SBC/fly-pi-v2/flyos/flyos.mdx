## システム焼込前の準備

- リーダー
- TFカード1枚、容量は16GB以上または128GB以下で、TFカードの転送速度はクラス10またはそれ以上が必要です
- 対応するシステムイメージをダウンロードする：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-pi-v2">システムイメージ</Button>
- 必要な焼込ツールをダウンロードする：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button>

## システム焼込

:::info
TFカード焼込とTFカードからM2WEへの焼込はどちらか一方を使用してください
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードからシステム焼込をお勧めします。USB経由での焼込はドライバ互換性の問題などで失敗する可能性があるため、直接[M2WEへのメモリーカード焼込](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を使用することをお勧めします
* 使用されるシステムイメージが正しいことを確認してください。そうでない場合は起動できません
:::

## アンテナの取り付け

1. `M2WE`または`2.4G Wi-Fi`モジュールのアンテナを取り付けます
:::warning
* FLY上位機器では`M2WE`または`2.4G Wi-Fi`モジュールを使用する必要があります。それがない場合、Wi-Fiを使用することはできません
* 注意：上位機器は中国語名前のWi-Fiをサポートしておらず、2.4Gと5GのシングルバンドWi-Fiもサポートしていません
:::
* 赤い矢印が示す部分がWi-Fiアンテナの接続口です
* Wi-Fiアンテナの接続口は`ipex1`です
* 天線をM2WEに取り付けないと、信号が弱いためルーターに接続できない可能性があります
