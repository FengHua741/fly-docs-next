## システム焼付け前の準備

    * SDカードリーダー
    * TFカード1枚、容量は16GB以上かつ128GB以下、転送速度はClass10またはそれ以上
    * 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-pi-v2">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼付けツール</Button> をダウンロード

## システム焼付け

:::info
TFカード焼付けとTFカード経由でM2WEに焼付けを行う場合はどちらか一方を使用してください
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードでシステムを焼付けることを推奨します。USB経由での焼付けはドライバの互換性の問題などで失敗する可能性があります。直接 [M2WEへのメモリーカード焼付け](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os) を参照してください
* 使用するシステムイメージが正しいものであるか注意してください。正しくない場合は起動できません
:::

## アンテナの取り付け

    1. `M2WE`または`2.4G Wi-Fi`モジュールのアンテナを取付けます
    :::warning
    * FLY上位機器には`M2WE`または`2.4G Wi-Fi`モジュールが必要です。Wi-Fiを使用するにはこれらが必要です
    * 注意: 上位機器は中国語名前のWi-Fiネットワークをサポートしていません。また、2.4Gと5Gの統合Wi-Fiもサポートしていません
    :::
    * 赤い矢印で示された部分がWi-Fiアンテナコネクタです
    * Wi-Fiアンテナコネクタは`ipex1`です
    * アンテナをM2WEに取り付けることを忘れないでください。信号が弱いため、ルーターに接続できないことがあります
