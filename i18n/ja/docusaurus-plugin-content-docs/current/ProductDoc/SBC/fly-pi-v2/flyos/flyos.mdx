## システム焼付け前の準備

    * リーダー
    * TFカード1枚、容量は16GB以上または128GB以下のもので、転送速度はクラス10またはそれ以上のものが必須です
    * 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼付けツール</Button> をダウンロード

## システム焼付け

:::info
TFカード焼付けとTFカード経由でM2WEに焼付けを行う場合、どちらか一つの焼付け方法を使用すればOKです
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードを使用してシステムを焼付けることを推奨します。USB経由でM2WEに焼付けを行う場合、ドライバーの互換性などの理由で焼付けに失敗することがあります。直接 [メモリーカード経由でM2WEに焼付け](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os) することをお勧めします
* 適切なシステムイメージをダウンロードしたことを確認してください。適切でない場合はシステムが起動しません
:::
