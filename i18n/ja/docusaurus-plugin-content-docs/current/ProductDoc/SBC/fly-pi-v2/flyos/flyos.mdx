## システム焼込前の準備

* リーダー
* TFカード1枚、容量は16GB以上または128GB以下のもので、TFカードの転送速度はクラス10またはそれ以上である必要があります
* 対応するシステムイメージをダウンロードする<BR><Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>
* 必要なツールをダウンロードする<BR><Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button>

## システム焼込

:::info
TFカード焼込とTFカードからM2WEへの焼込はどちらか一方を使用すれば十分です
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEはTFカードでシステムを焼込むことを推奨します。USB経由でM2WEに焼込むと、ドライバの互換性などの理由で焼込に失敗することがあります。直接[メモリカードからM2WEに焼込む](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)ことをお勧めします
* 使用しているシステムイメージが正しいかどうか注意してください。正しくない場合は起動できません
:::
