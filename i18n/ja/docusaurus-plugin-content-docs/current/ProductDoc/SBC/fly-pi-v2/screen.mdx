# TFTスクリーン設定

<Tabs>
    <TabItem value="tft" label="TFTスクリーン" default>
        * FLY TOOLSでKSを開く必要があります。
            <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/ks.webp').default} size="80%" align="center" />
        <Tabs>
    <TabItem value="tft" label="TFTタッチスクリーン設定" default>
## 方法一    
         * タッチスクリーンの設定には、以下のコマンドを使用して`fly-tft-v2-c`に設定する必要があります。
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * 画面の裏側のダイアルが`Cap`に設定されていることを確認してください。
    * 設定が正常に行われたかどうかを確認するには、以下のコマンドを使用します。
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * タッチスクリーンを抵抗膜スクリーンに変更するには、以下のコマンドを使用します。

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

        <TabItem value="hdmi" label="TFT抵抗膜スクリーン設定">
## 方法一        
           * 抵抗膜スクリーンの設定には、以下のコマンドを使用して`fly-tft-v2-r`に設定する必要があります。
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
```
                                * 画面の裏側のダイアルが`Resi`に設定されていることを確認してください。
        * 設定が正常に行われたかどうかを確認するには、以下のコマンドを使用します。
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
                                    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-r.webp').default} size="80%" align="center" />
 * 抵抗膜スクリーンをタッチスクリーンに変更するには、以下のコマンドを使用します。

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v2-c/}' /boot/FlyOS-Env.txt
```
</TabItem>
