## システム焼成の準備

* リーダー
* TFカード1枚、容量は16GB以上または128GB以下のもので、TFカードの転送速度はClass10以上である必要がある
* 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロードする
* 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼成ツール</Button> をダウンロードする

## システム焼成

:::info
TFカード焼成とTFカードからM2WEへの焼成はどちらか一方を使用すればよい
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEはTFカードを使用してシステムを焼成することをお勧めします。USB経由での焼成はドライバ互換性などの理由で失敗する可能性がありますので、直接[メモリカード焼成](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を使用することをお勧めします
* 配布されたシステムイメージが正しいかどうかに注意してください。正しくない場合は起動できません
:::
