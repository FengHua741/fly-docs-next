# 上位機SSHに接続

* **シリアルポート**または**WiFi**を使用してデバイスに接続できます
* 事前に**MobaXterm_Personal**または他の**SSHターミナルツール**を準備してください
* SDカードまたはM2WEに**対応するシステム**が焼込まれていることを確認してください
* **上位機が正常に起動しない場合、ドライバー、限界設定、ファンなどの周辺機器を取り外してください！！**

:::warning 注意
- 初めて上位機を起動する場合は、必ず**シリアルポートで上位機SSHに接続**してください
- ファームウェアのコンパイルを行う際は、**WiFiで上位機SSHに接続**することをお勧めします。ファームウェアのコンパイルや焼込が簡単になります！
- `FLYOS-Armbian`システムと`FLYOS-FAST`システムでは、SSHへの接続手順が異なります。焼込んだシステムに応じて適切なチュートリアルを選択してください！！
:::

    <TabItem value="net" label="FLYOS-Armbianシステム WiFi接続" default>
        ## FLYOS-Armbian WiFiで上位機SSHに接続

        * 上位機システムがすでに**正常に起動している**ことを確認してください
        * **上位機のユーザー名とパスワードが明確である**ことを確認してください。FLYシステムのデフォルトのユーザー名は`fly`、パスワードは`mellow`です
        * WiFiで上位機SSHに接続するための前提条件は、**上位機がルーターに接続されており、上位機のIPがわかっている**ことです  

        * MobaXtermを開き、次のように設定します。`Remote host`には、**シリアルポート接続時に確認した上位機のIP**を入力してください。また、ルーターのバックエンドから上位機のIPを確認することも可能です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **注意：ここでの第4ステップでは上位機のユーザー名を入力してください**
        :::tip
        - 第4ステップでは上位機のユーザー名を入力します。FLYOS-Armbianシステムを使用するfly上位機の場合は`fly`と入力してください。他の上位機は実際の状況に応じて正しいユーザー名を入力してください
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * 最初の接続時にこのウィンドウが表示されます。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * パスワードを入力するよう促されますので、上位機のパスワードを入力してEnterキーを押してください
        :::tip
        - パスワードの入力中は文字が表示されません。入力が終わったらEnterキーを押してください
        :::
        
        FLYOS-Armbianシステムのデフォルトのユーザー名は：`fly`、パスワードは：`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        下図のようなウィンドウが表示された場合、このポップアップはユーザーのパスワードを保存するかどうかを尋ねています。`No`をクリックしてください
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * 下図のような画面が表示された場合、接続が成功しました！

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
