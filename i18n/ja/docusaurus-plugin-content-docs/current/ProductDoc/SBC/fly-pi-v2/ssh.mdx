# 上位機SSHへの接続

* **シリアルポート**または**WiFi**を使用してデバイスに接続することができます
* **MobaXterm_Personal**や他の**SSHターミナルツール**を事前に準備してください
* SDカードまたはM2WEが上位機と一致するシステムに焼込まれていることを確認してください
* **上位機が正常に起動しない場合、ドライバー、限界、ファンなどの外部機器を取り外してください！！！**

:::warning 注意
- 初めて上位機を起動する場合は、必ず**シリアルポートで上位機のSSHに接続**してください
- ファームウェアのコンパイルを行う際は、便利なため**WiFiで上位機のSSHに接続**してください！
- `FLYOS-Armbian`システムと`FLYOS-FAST`システムでは、SSHへの接続手順が異なりますので、焼込られたシステムに応じて適切なチュートリアルを選択してください！！！
::>

    <TabItem value="net" label="FLYOS-ArmbianシステムのWiFi接続" default>
        ## FLYOS-Armbian WiFiで上位機SSHに接続

        * 上位機システムがすでに**正常に起動している**ことを確認してください
        * **上位機のユーザ名とパスワードが明確である**ことを確認してください。FLYシステムのデフォルトユーザ名は`fly`、デフォルトパスワードは`mellow`です
        * WiFiで上位機SSHに接続するための前提条件は、**上位機がルーターに接続されており、上位機のIPがわかっている**ことです  

        * MobaXtermを開き、下図のように設定し、`Remote host`には**シリアルポート接続時に取得した上位機のIP**を入力してください。また、ルーターのバックエンドで上位機の**IP**も確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **注意：ここではステップ4で上位機のユーザ名を入力します**
        :::tip
        - ステップ4で上位機のユーザ名を入力します。FLYOS-Armbianシステムの上位機の場合、`fly`を入力してください。他の上位機は実際の状況に応じて正しいユーザ名を入力してください
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * 最初の接続時にこのウィンドウが表示されることがあります。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * パスワードを入力するよう促されるので、上位機のパスワードを入力してエンターキーを押してください
        :::tip
        - パスワードは見えませんが、入力後すぐにエンターキーを押してください
        :::
        
        FLYOS-Armbianシステムのデフォルトユーザ名は：`fly`、デフォルトパスワードは：`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        * 下図のようなウィンドウが表示される場合があります。このウィンドウはユーザーのパスワードを保存するかどうかを尋ねています。`No`をクリックしてください
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * 下図のような画面が表示されればログイン成功です
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
