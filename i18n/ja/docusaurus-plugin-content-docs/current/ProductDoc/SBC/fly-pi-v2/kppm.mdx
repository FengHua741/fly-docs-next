# Klipper電源喪失継続印刷機能

## 概要

- Klipper電源喪失継続印刷機能は、Klipperが電源を切られた後に再度電源を入れると、Klipperの印刷状態が自動的に復元できる機能です。
- 断電継続印刷機能を使用するには、FASTシステムと組み合わせる必要があります。
- PI-V2はKPPM電源喪失モジュールと組み合わせて使用する必要があります。
- 停電時にZ軸が移動する機種には適用されません。
