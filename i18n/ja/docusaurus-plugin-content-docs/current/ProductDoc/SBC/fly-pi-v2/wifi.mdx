# WiFiに接続する
:::tip
- `FLYOS-FAST`システムと`FLYOS-Armbian`システムのWiFi接続の手順と方法には若干の違いがありますので、お持ちのシステムに対応するチュートリアルを選んでください。
:::

:::warning 警告
    * FLYホストPCは`M2WE`モジュールまたは`2.4G WiFi`モジュール、または`Linux対応の無線LANUSBアダプター`を使用する必要があります。それ以外ではWiFiを使用できません。
    * ご注意ください、ホストPCは中国語名前のWiFiには対応していません。また、2.4Gと5Gを統合したWiFiにも対応していません。
    * `mini pad`または`lite2/lite2.1`をお持ちの場合、この警告を無視してください。
:::
