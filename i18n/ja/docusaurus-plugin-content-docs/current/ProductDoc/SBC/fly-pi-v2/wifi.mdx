# WiFiに接続する
:::tip
- `FLYOS-FAST`システムと`FLYOS-Armbian`システムのWiFi接続手順には若干の違いがありますので、お使いの焼込済みシステムに対応するチュートリアルを選んでください。
:::

:::warning 警告
    * FLYホストコンピューターでは、`M2WE`モジュールまたは`2.4G WiFi`モジュール、または`Linux対応ドライバ不要USBワイヤレスネットワークアダプター`を使用してください。それ以外の場合、WiFiは使用できません。
    * 注意：ホストコンピューターは中国語名前のWiFiをサポートしていません。また、`2.4Gと5Gの両方を統合したWiFi`もサポートしていません。
    * `mini pad`または`lite2/lite2.1`をお使いの場合は、この警告を無視してください。
:::
