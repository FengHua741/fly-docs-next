# はじめに

私たちは喜んで新しい FLY-Pi V2 上位機をリリースします。このボードはラズベリーパイと同じサイズで、ラズベリーパイの代わりに上位機として動作することができます。1GB のオンボード DDR3 メモリを搭載しており、完全なLINUXデスクトップ（Armbian をカスタム最適化したものに基づく）を実行することができます。このボードは Klipper と RRF ファームウェアと連携して使用でき、SHT ツールボードを使用している場合でも Klipper スクリーンを実行することができます。
この上位機を開発する目的は、Klipper ファームウェアや RRF ファームウェアを使用するのがお客様にとってより簡単にすることです。私たちは Linux システムイメージを最適化し、3D プリンターに最適化され、操作とインストールが簡単になりました。初心者でも基礎があればすぐに学習して使用することができます。他の上位機ボードと比較して学習コストを節約し、あなたは 3D プリントによる楽しさに集中することができます（SDカード付きのパッケージを購入して Klipper ファームウェアをプリロードすると、電源を入れるだけで使用可能で、時間と手間を節約します）。

## 特徴

* ラズベリーパイ 3B および 4B と同じフォームファクタ
* CPU：高性能の全志 H5 チップ、4コア 64ビット Cortex-A53
* GPU：高性能 6コア Mali 450、ピクセルフィリングレートは 2.7 gpixel/s より大きい
* RAM：1GB DDR3（GPUと共有）(512+512)
* ROM：最大 128GB SDカードをサポート
* 外付け：Spi x2、UART x2、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN （ボード内USB変換CANモジュール、つまりボード内UTOC）
* 接続：ボード内 M.2 インターフェース、Wi-Fi や eMMC などの拡張が可能です。（専用プロトコル、FLY π 専用の M.2 デバイス以外には接続しないでください）
* 40ピンインターフェース、ラズベリーパイと互換性あり
* 12-24V 直流電源をサポート、供給がより安定しています
* 5Vファン ZH1.5インターフェースを予備で用意しています
* システムは klipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグイン numpy と FLY-Tools をプリロードしており、複雑な操作を必要とせずに初心者も使いやすい（TFカードまたは M2WE を購入する必要があります）
