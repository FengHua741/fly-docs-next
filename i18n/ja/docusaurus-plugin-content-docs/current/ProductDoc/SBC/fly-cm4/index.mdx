# 簡介

      Fly-CM4はRaspberry Pi 3Bと4Bと完全に互換性があり、特にRaspberry Pi CM4の拡張機能を改善するために特別に設計されています。この拡張ボードは、3Dプリンターに最適化された一連の機能を提供します。12-24VのDC電源は、3Dプリンターの正確な制御と連続的な動作にとって重要な電源の安定性を確保します。FLY-TFTおよびFly-HDMIスクリーンFPCコネクション機能がサポートされ、より便利な配線方法を提供します。CANポートが装備されており、上位機器からCANで直接下位機器に接続することは現在一般的な方法です。

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-cm4/img/cm4.webp').default} size="80%" align="center" />

## 特徴

    * Raspberry Pi CM4コアボードをサポートしています
    * 12V-24VのDC電源をサポートしています
    * ボードマウント型UTOCモジュールを搭載しています
    * TFTとHDMIの1本通しをサポートしています
    * DSI/spiスクリーンインターフェースをサポートしています
    * CSI/USBカメラインターフェースをサポートしています
    * 40PIN拡張インターフェース、Raspberry Piと互換性があります
