# WIFIに接続する


## PIにアンテナを取り付ける

    * 赤い矢印が指している場所がWIFIアンテナインターフェースです
    * WIFIアンテナポートは`ipex1`です
    * アンテナをPIに取り付けないと、信号が弱いためネットワークに接続できない可能性があります
