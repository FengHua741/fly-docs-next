## システム焼込み

:::info
TFカードへの焼込みとTFカードからM2WEへの焼込みはどちらか一方を使用すれば十分です。
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します。
* M2WEの場合はTFカードでのシステム焼込みをお勧めします。USB経由での焼込みはドライバ互換性などの理由で失敗する可能性があります。直接 [M2WEへのメモリカード焼込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os) を使用することをお勧めします。
* 適切なシステムイメージをダウンロードしていることを確認してください。適切でない場合、システムは起動しません。
:::

## アンテナの取り付け

    1. `M2WE`または`2.4G WiFi`モジュールのアンテナを取り付けます。
    :::warning
    * FLY上位機器には`M2WE`または`2.4G WiFi`モジュールを使用する必要があります。それがないとWiFiを使用できません。
    * 注意: 上位機器は中国語名前のWiFiには対応していません。また、2.4Gと5Gの同時対応WiFiには対応していません。
    :::
    * 赤い矢印が指している場所がWiFiアンテナの接続口です。
    * WiFiアンテナの接続口は`ipex1`です。
    * PI V2にアンテナを接続しないと、信号が弱いためルーターに接続できない可能性があります。
