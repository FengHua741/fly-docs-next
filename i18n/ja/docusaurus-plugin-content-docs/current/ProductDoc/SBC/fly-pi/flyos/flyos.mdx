## システム焼込前の準備

    * SDカードリーダー
    * 容量が16GB以上または128GB以下のTFカード1枚、転送速度はクラス10以上が必要
    * 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-pi-v2">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button> をダウンロード

## システム焼込

:::info
TFカード焼込とTFカードからM2WEに焼込む方法はどちらか一つを使用してください
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEはTFカードでシステムを焼込むことを推奨します。USB経由での焼込みはドライバ互換性などの理由で失敗する可能性があります。直接[M2WEへのメモリカード焼込](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を使用することをお勧めします
* 使用するシステムイメージが正しいかどうかに注意してください。そうでない場合は起動できません
:::

## アンテナの取り付け

    1. `M2WE`または`2.4G Wi-Fi`モジュールのアンテナを取り付けます
    :::warning
    * FLY PCは`M2WE`または`2.4G Wi-Fi`モジュールを使用する必要があります。それがないとWi-Fiが使えません
    * 注意：PCは中国語名前のWi-Fiはサポートしていません。また、2.4Gと5Gの統合Wi-Fiはサポートしていません
    :::
    * 赤い矢印が示す部分がWi-Fiアンテナインターフェースです
    * Wi-Fiアンテナポートは`ipex1`です
    * PI V2にアンテナを接続しないと、信号が弱いためルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi/img/wifi.webp').default} size="80%" align="left" />
