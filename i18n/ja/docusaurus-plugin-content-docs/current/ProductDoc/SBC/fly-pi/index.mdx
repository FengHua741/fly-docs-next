# 概要

我々は新しいFLY-πマザーボードを発表することを非常に喜んでいます。このボードはRaspberry Piの代わりとして上位機器で動作でき、1GBのオンボードDDR3メモリを搭載しており、完全なLINUXデスクトップ（Armbianカスタム最適化に基づく）を実行することができます。このボードはKlipperおよびRRFファームウェアと共に使用でき、さらにはSHTツールボードを使用している場合でもKlipperスクリーンを実行することができます。

この上位機器の設計目的は、KlipperファームウェアまたはRRFファームウェアの使用を顧客に容易にするためです。我々はLINUXシステムイメージを最適化し、3Dプリンタでの使用に最適化されており、操作とインストールがより簡単になっています。初心者でも少しの基礎があれば、すぐに学習して使用できます。他の上位機器ボードと比較して、学習コストを削減し、3Dプリントによる楽しみに集中できるようにします（SDカード付きのパッケージを購入するとKlipperファームウェアがプリインストールされているので、電源を入れるだけで使えます）。

## 特徴

* CPU：高性能の全志H5チップ、4コア64ビットCortex-A53
* GPU：高性能6コアMali 450、ピクセルフィリングレートは2.7gpixel/s以上
* RAM：1GB DDR3（GPUと共有）(512+512)
* ROM：最大128GB SDカード対応
* パーティション：Spi x2、UART x2、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN x1（CANモジュールを別途購入する必要があります）
* インターフェース：オンボードM.2インターフェース、Wi-FiやeMMCなどの拡張が可能です。（専用プロトコルを使用しているため、非FLY π専用のM.2デバイスを接続しないでください）
* 40Pinインターフェース、Raspberry Piと互換性あり
* 12-24V DC電源供給に対応、電源供給がより安定しています
* 5V制御可能なファンインターフェースが予備で設置され、上位機器の冷却に使用できます
* CAN拡張インターフェースが予備で設置されています
* システムはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyとFLY-Toolsがプリインストールされています。複雑な操作、例えばソースの変更などが不要で、初心者でも簡単に始められます（TFカードまたはM2WEを購入する必要があります）
