# WIFIに接続

## LITE 2.1にアンテナを取り付ける

    * 赤い矢印が示す場所がWIFIアンテナインターフェースです
    * WIFIアンテナポートは`ipex1`です
    * アンテナをLITE 2.1に取り付けない場合、信号が弱いためネットワークに接続できない可能性があります

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="100%" align="left" />  

* 正常にWIFIに接続されていない場合は、まずアンテナが接続されていることを確認し、次に以下の方法を使用してWIFIに接続してください
