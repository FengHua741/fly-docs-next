# システム焼込

## 上位機初心者ガイド

    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/baodian/host/">上位機初心者ガイド</Button>

## システム焼込前の準備

    * リードカード
    * TFカード1枚、容量は16GB以上または128GB以下、転送速度はClass10以上である必要があります
    * 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-lite2">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button> をダウンロード

## システム焼込
:::info
* MINI PAD ではシステムをTFカードにしか焼込めません。M2WEは使用できません
* システム焼込が完了したら、TFカードをMINI PADに取り付けなければなりません
:::
    
    <TF />

## システムカードのインストール

    * システムをTFカードに焼込んだ後、LITE2.1にTFカードを取り付ける必要があります。そうしないとシステムが起動しません
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tf.webp').default} size="100%" align="left" />  

## アンテナの設置

    1. アンテナを設置します
    :::warning
    * 注意してください。上位機は中国語名前のWi-Fiには対応していません。5G Wi-Fiにも対応していません
    :::
    * 赤い矢印が指しているのがWi-Fiアンテナインターフェースです
    * Wi-Fiアンテナポートは`ipex1`です
    * アンテナを接続しないと、信号が弱いためルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードのインストールが完了したら、次の手順に進み、SSHで上位機に接続できます
    :::

    2. 上位機に電源を供給します。  
    **ここから、あなたの上位機は起動できるようになりました！！！**  
    次の手順へ進むには<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2.1/ssh">SSHで上位機に接続</Button>をクリックしてください
