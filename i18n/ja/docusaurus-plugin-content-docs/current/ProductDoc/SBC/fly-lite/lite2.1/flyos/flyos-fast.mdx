# システムイメージ焼込

<FASTFASLH name="H3" />

            
## システムカードのインストール


    * システムをTFカードに焼込んだ後、カードをLITE2.1の後ろに接続する必要があります。そうでないとシステムが起動しません
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tf.webp').default} size="100%" align="left" />  

## アンテナの設置

    1. アンテナを設置します
    :::warning
    * 注意してください、上位機は中国語名前のWi-Fiをサポートしていません。また、2.4Gと5Gのデュアルバンド統合Wi-Fiもサポートしていません
    :::
    * 赤い矢印で示されている部分がWi-Fiアンテナインターフェースです
    * Wi-Fiアンテナポートは`ipex1`です
    * アンテナを接続しない場合、信号が弱いためルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードのインストールが完了したら、次の手順に進むことができます。SSHで上位機に接続します
    :::

    2. 上位機に電源を供給します。  
    **ここでは、あなたの上位機はすでに起動できます！！！**  
    次の手順に進んでください：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2.1/ssh">上位機にSSH接続</Button>
