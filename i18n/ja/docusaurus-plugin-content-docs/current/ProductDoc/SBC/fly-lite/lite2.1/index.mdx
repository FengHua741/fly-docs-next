# 簡介

    Fly-Pi-lite2.1は入門レベルのKlipperホスト機です
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.webp').default} size="80%" align="center" />

## 特點

    * CPU：高性能全志H3チップ、4コア64ビットCortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：最大128GB SDCARDをサポート
    * WIFI: ボード内蔵2.4G WiFi
    * 外設：Spi x1、URAT x2、USB x2、HDMI x1
    * 接口：FPCを予約、HDMI + USB FPC一体型
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、共振補償プラグインのnumpyとFLY-Toolsが事前にインストールされています。煩雑な操作を必要とせずに、初心者も簡単に始められます（TFカードの購入が必要です）

## 更新機能

    * TFT V1インターフェースをTFT V2にアップグレード
    * リセットとシャットダウンピンを追加
    * 電源投入指示灯を追加
    * BOOTボタンを削除し、リセットボタンに置き換え
    * ワイヤリングを最適化
