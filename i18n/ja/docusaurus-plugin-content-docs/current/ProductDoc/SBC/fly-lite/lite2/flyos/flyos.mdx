# システム焼込

## 上位機初心者ガイド

    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/baodian/host/">上位機初心者ガイド</Button>

## システム焼込の準備

    * SDカードリーダー
    * 容量が16GB以上または128GB以下のTFカード、転送速度はclass10級またはそれ以上
    * 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-lite2">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button> をダウンロード

## システム焼込
:::info
* MINI PAD ではシステムをTFカードにしか焼込めません。M2WEは使用できません。
* システム焼込が完了したら、TFカードをMINI PADに挿入してください。
:::
    
    <TF />

## システムカードのインストール

    * システムをTFカードに焼込んだ後、lite2に接続する必要があります。それがないとシステムが起動しません
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/tf.webp').default} size="100%" align="left" />  

## アンテナの取り付け

    1. アンテナを取り付けます
    :::warning
    * 注意: 上位機は中国語名前のWi-Fiをサポートしていません。5G Wi-Fiもサポートしていません。
    :::
    * 赤い矢印が示す部分がWi-Fiアンテナポートです
    * Wi-Fiアンテナポートは`ipex1`です
    * 電波が弱いため、接続できない場合がありますので、必ずアンテナを取付けてください
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードのインストールが完了したら、次に進み、SSHで上位機に接続してください
    :::

    2. 上位機に電源供給します。  
    **ここまできたら、あなたの上位機は起動できるようになりました！！！**  
    次の手順へ進んでください: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2/ssh">上位機にSSH接続</Button>
