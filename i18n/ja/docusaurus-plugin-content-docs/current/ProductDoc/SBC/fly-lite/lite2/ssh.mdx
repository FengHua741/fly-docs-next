# 上位机SSH接続


* シリアルポートまたはWi-Fiを使用してデバイスに接続することができます
* 事前に**MobaXterm_Personal**または他の**SSHターミナルツール**を準備してください
* SDカードまたはM2WEに**対応する上位機のシステム**が焼入れられていることを確認してください
* **上位機が正常に起動しない場合はドライバーや限界、ファンなどの周辺機器を外してください！！**
* **最後にネットワーク経由でSSHに接続してファームウェアのコンパイルが便利です！！**
* **ファームウェアをコンパイルする必要がある場合は、Wi-Fiまたはイーサネットケーブルで上位機に接続するチュートリアルをご覧ください**

        ## 上位機SSHにネットワークIPで接続する方法

        * 上位機が接続されている**ルーターのバックエンド**にログインしてください
        * システムが**正常に起動している**ことを確認してください
        * 自分自身が**上位機のユーザー名とパスワードを知っている**ことを確認してください。FLYシステムのデフォルトのユーザー名は**fly**、デフォルトのパスワードは**mellow**です
        * メインボードが起動し、ネットワークに接続されている場合、IPを使ってSSHに接続できます
        * MobaXtermを開き、次の設定を行います。IPは確認したものを入力します。また、ルーターのバックエンドでも確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **注意：ここでのステップ4に入力するユーザー名は上位機のものです**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 初回接続時にこのウィンドウがポップアップしますので、`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * パスワードの入力を求められるので、上位機のパスワードを入力し、Enterキーを押します

        **注意：ここで入力するパスワードは表示されません。入力後、Enterキーを押してください**

        FLYシステムのデフォルトでは、ユーザー名は`fly`、パスワードは`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * 接続成功

    </TabItem>
