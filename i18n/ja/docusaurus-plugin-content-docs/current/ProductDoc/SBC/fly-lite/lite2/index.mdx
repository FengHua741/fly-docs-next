# 簡介

    Fly-Pi-lite2は入門向けのKlipperホストコンピュータです

## 特點

    * CPU：高性能の全志H3チップ、4コア64ビットCortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：最大128GB SDCARDをサポート
    * WIFI: ボードに内蔵された2.4G Wi-Fi
    * 外設：Spi x1、URAT x2、USB x2、HDMI x1
    * 接口：FPCを予備、HDMI + USB FPCワンケーブル
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、共振補償プラグインnumpyとFLY-Toolsが事前にインストールされており、ソースの交換などの複雑な操作が必要なく、初心者でも簡単に始められます（TFカードの購入が必要です）
