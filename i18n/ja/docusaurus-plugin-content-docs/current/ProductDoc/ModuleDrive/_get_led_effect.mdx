# ドライバのLEDをプラグインで拡張する方法

上位機にSSHで接続した後、以下のリンクに従ってインストールと設定を行ってください

```
https://github.com/julianschill/klipper-led_effect
```

## インストール

**MobaXterm_Personal**などの**SSHツール**を使用して**WIFI**を通じて上位機に接続し、以下の点を確認してください

1. **上位機にKlipperファームウェアがインストールされていることを確認してください**
2. **Klipperがインストールされているユーザとしてログインしていることを確認してください**
3. **入力法が英語であることを確認してください**
4. **上位機がデバイスを正常に検出できることを確認してください**
5. **上記の注意事項をすべて守っていない場合、次のステップに進めません**

このモジュールは、既存のKlipperインストールにインストールスクリプトを使用してインストールできます。

```
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## 参照設定

```
[neopixel TP_led]
pin: PB7
chain_count: 25
# LEDの数
# 灯珠数量
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)
```
