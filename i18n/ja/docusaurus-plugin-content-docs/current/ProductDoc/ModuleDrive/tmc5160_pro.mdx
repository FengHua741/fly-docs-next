# TMC 5160 Pro V1.2

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/5160pro.webp').default} size="80%" align="center" />

## 基本参数

    * 入力電圧 ：24-48V
    * 最大電流 ：6A
    * 対応するファームウェア ： Marlin/Klipper/RRF
    * ドライバーモードサポート：TMC：SPI
    * ステップ/ディレクションインターフェースにマイクロポリヤのマイクロステッピングを備えています
    * 最高分解能は256分割です
    * stealthChop2による静かな動作と滑らかな動きをサポートしています
    * 中周波数の共振を抑制します
    * spreadCycleによる高ダイナミックモーターコントロールチャープをサポートしています
    * dcStepによる負荷に関連した速度制御をサポートしています
    * stallGuard2による高精度な無センサーモーターロード検出をサポートしています
    * coolStepによる電流制御をサポートし、最大で75％の省エネルギーを実現します

## メインボードジャンパ説明

    * TMC5160 ProはSPIジャンパーを使用します
    * FLYドライバーのSPIピンは左から下に数えて4番目のピンで、CSピンはSPIピンです。メインボードが対応するピン位置でない場合、SPIを使用することはできません

## TMC5160 Proリファレンス設定

    :::danger

    * ドライバーのサンプリング抵抗は`sense_resistor: 0.033`に設定してください。間違った設定は避けてください
    * spi_busとspi_softwareはどちらか一方を選択する必要があります
    
    ::: 

### SPIモードリファレンス設定

    ```cfg
    [tmc5160 stepper_x]
    cs_pin:
    spi_bus:
    # spi_software_mosi_pin:
    # spi_software_miso_pin:
    # spi_software_sclk_pin: 
    run_current: 1.0                    # モーターの運転電流値
    interpolate: False                  # 256マイクロステッピングのマイクロポリヤ（オンはTrue、オフはFalse）
    sense_resistor: 0.033
    stealthchop_threshold: 0
    driver_DISS2G: 1
    driver_DISS2VS: 1
    ```
