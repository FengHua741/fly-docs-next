# 概要

    * いくつかのドライバーに関する注意点


## ドライバーのインストール

    * ENを左上に合わせてインストールする必要があります
    * FLYマザーボードのドライバーポートのENピンは左上にあります
    * FLYのDP5、PRO X10、MMUはTPドライバーのみを使用できます
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />


## ドライバーの設定

    * TPドライバーと通常のドライバーの設定方法は同じです
    * FLY-2240ドライバーはUART接続をサポートしていますが、抵抗を手動で変更する必要があります
    * ドライバーサンプリング抵抗には注意が必要です。設定が間違っているとモーター、ドライバーが損傷する可能性があります
