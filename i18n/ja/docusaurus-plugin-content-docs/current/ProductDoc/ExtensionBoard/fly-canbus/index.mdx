# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.webp').default} size="40%" align="left" />


## 製品特徴

    * CANと電源を1分4にサポートし、一度に4つのCANツールボードを拡張できます
    * ボードにRJ11インターフェースが搭載されており、CANプロトコルを使用するシステムとの接続と通信に設計されています。RJ11はCANシステムでは標準的なインターフェースではありませんが、当社の製品は両者のスムーズな接続を可能にします
    * 人間工学に基づいた設計で、プリントパッドを直接レールに取り付けることができます

## 製品仕様

    * 通信インターフェース：RJ11*1, XH2.54*5
    * 電源インターフェース：KF128-5.028-2P*5
    * 入力電圧：5V-24V
    * 互換性：RRF/Klipper/等CAN

## 注意事項

    :::warning 注意事項

    * 電源入力量は出力量と同じであることに注意してください
    * ラインの配線が間違っていないことを確認してください
    * 拡張ボードをインストールする前に、アップストリームマシンの電源がオフになっていることを確認してください
    * 湿気やほこりの多い環境での拡張ボードの使用は避けてください
    * 設備をインストールする際には、ツールボードの電圧仕様に注意して、損傷を防いでください

    :::
