# 簡介

# LLLバッファーの使用

    LLLエクストルーディングバッファーは、3Dプリンターのリモートフィラメント供給を目的としたデバイスで、フィラメントのロードとアンロードを簡素化するように設計されています。このバッファーには、フィードインとリターンの各操作を行うための2つのボタンが装備されており、ユーザーが電動でフィードインとリターン操作を行うことができます。

### バッファ機能：

    3Dプリンターの押出機がフィードインする際に、遠端のフィラメントスタンドに引っ張られた場合、バッファーはこの動作を検出し、熱端側へフィラメントを自動的に送出します。この動作は、各引っ張り動作ごとに繰り返されます。これにより、押出機がフィラメントを引っ張ることを効果的に減らし、押出機の推力を安定させ、印刷品質の安定性を向上させることができます。

### エンドセンサー機能：

    バッファーには内蔵のエンドセンサーがあり、バッファー内にフィラメントがないことを検知すると、3Dプリンターに印刷の一時停止を指示します。この機能を使用するには、信号ピンへの接続と3Dプリンターの設定の変更が必要です。一方、バッファ機能は設定を変更せずに直接使用できます。

    これらの機能により、LLLエクストルーディングバッファーはフィラメントの操作を簡素化し、3Dプリントプロセスの安定性と印刷品質を向上させます。

* 注意：機能を使用するには、フィラメントを接続した後に使用する必要があります。それ以外の場合、ボタンでモーターを制御するのみです。

## 設定リファレンス

    ```cfg
    [filament_switch_sensor エンドセンサー]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 缺料
    insert_gcode:
            M118 上料
    ```
