## 參考配置

    ```cfg
    [fan]
    ##    Print Cooling Fan 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## 開機自動轉解決方法

    :::info
    * 這邊只簡單說明
    * 在固件編譯時候添加上PWM的信號引腳，並且反轉他狀態
    * 測試使用SUPER8PRO，引腳使用PF8，在固件編譯時候選擇`GPIO pins to set at micro-controller startup`添入`!PF8`即可
    * 請注意填入的`!`是英文的感嘆號
    * 其他主板同理
    :::
