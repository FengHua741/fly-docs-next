# RRF構成設定

## カットオフおよび詰まり検出機能の設定

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} size="70%" align="center" />

1. RRFプリンターのウェブインターフェースを開き、`Files` → `System` に進み、`config.g` ファイルを見つけ、クリックしてファイルを編集します。
2. ファイルに以下のコードを追加してカットオフ検出機能を有効にします。

```gcode
M591 D0 P2 C"PA9" S1
```

3. ファイルに以下のコードを追加して詰まり検出機能を有効にします。（`L` パラメータは使用するエンコーダーにより異なります。誤検出が発生した場合は適切にこの値を増やしてください）

```gcode
M591 D0 P7 C"PA10" L3.7 R75:125 S1
```
