# 製品概要

  - MDM切れ検知センサーは、小型で使いやすく、トリガーが非常に敏感です。
  - Klipper、Marlin、RRFなどの3Dプリンタに対応しています。
  - 切れ検出：高速赤外線スキャンにより、0.25秒以内にフィラメントの断裂をキャッチし、直ちに印刷を停止して警告を発します。これにより、無駄なエネルギー消費を防ぎます。
  - ブロック検知：エンコーダーの光センサーシグナルによって、ノズルやホッターの詰まりの前兆を識別します。これにより廃棄物率を80％以上削減できます。
  - 機能的な互換性：PLA、ABS、PETG、TPUなどすべてのタイプのフィラメント（直径2.85mm）をサポートし、硬質材料から柔軟なフィラメントまで安定して認識できます。
  - 簡単な取り付け：取り付けと使用を容易にするため、ホールピッチ図、配線図、設定ファイルなどの資料を提供しています。
