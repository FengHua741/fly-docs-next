# モジュールのトリガ値を調整する方法
シリアルツールを使用すると、以下のコマンドが利用可能です：
Klipperコンソールで「SET_PIN PIN=_probe_ready VALUE=1」と送信し、トリガインジケーターを観察しながら、インジケーターが点灯しなくなるまでしきい値を小さくして大きくしてください。以下のコマンドを入力します：
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
シリアルツールで以下のコマンドを使用してトリガ値を調整します：
```cfg
set 触发值
```
デフォルトのトリガ値：20000
* シリアルツールで使用するコマンド:

* `v` をシリアルに送信すると、adc値がリアルタイムで出力されます。a:フィルタ前の値、b:フィルタ後の値。
* `uv` をシリアルに送信すると、出力を停止します。
* `rt` をシリアルに送信すると、現在のしきい値を確認できます。
