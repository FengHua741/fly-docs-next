# ケーブル接続のKlipper構成

:::danger 注意

* STM32F072にファームウェアを焼く前に接続すると、ケーブル接続の操作はサポートされません

:::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押すと、**Save configuration** が表示されます。その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * これで設定が保存され、コマンドラインに戻ります
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が表示されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## フラッシュモードに入る

    * AirClickが電源オフ状態でSTM32のBOOTを押しながら、STM32のフラッシュポートをType-Cでホストマシンに接続します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/boot.webp').default} size="100%" align="center" />

## フラッシュモードに入ったか確認

    <LS />
