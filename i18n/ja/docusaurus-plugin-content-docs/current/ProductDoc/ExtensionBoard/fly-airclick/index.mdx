# AirClick 無線ボタン
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.webp').default} size="100%" align="Center" />



## 製品特長

    * 高度な互換性：Klipper、RRF、BambuLabをサポートし、直接マシンに接続してプリセットのコマンドを送信できます。シームレスな互換性と操作体験を提供します。
    * カスタムマクロコマンド：ユーザーは自身のニーズに応じてコマンドをカスタマイズできます。これにより、機械の操作がより柔軟で便利になります。
    * 無線接続：ESP32の設計により、Klipperにカスタムコマンドやマクロを直接送信できます。これにより、ユーザーには非常に高い柔軟性と拡張性が提供されます。
    * 有線接続：ネットワークがKlipperに接続できない場合でも、STM32F072チップを使用して簡単な操作を行うことができます。合計10個のカスタムコマンドまたはマクロをサポートします。
    * 利用しやすい：直感的なユーザーインターフェースにより、ユーザーは無線ボタンを簡単に追加および管理できます。これにより、操作の利便性が大幅に向上します。

## 製品仕様

    * メインコントローラーチップ：ESP32
    * 追加チップ：STM32F072
    * ESP32-WiFi経由でコマンドを送信：制限なし
    * STM32F072-IO経由でコマンドを送信：10個
    * 互換性：Klipper、RRF、BambuLab
    * 電源：5V

## 注意事項

    :::danger 注意事項

    * 本製品は5Vで給電され、`type c`または`PH2.0-2P`コネクタによる給電が可能です。
    * 本製品の内蔵アンテナはボタン信号を受け取るためだけに使用されます。Wi-Fiに接続する場合は、アンテナを適切に取り付ける必要があります。アンテナコネクタは`ipex3`です。
    * アンテナを適切に取り付けてください！！！信号が悪ければ発熱が激しくなる可能性があります！！！
    * 長押しボタンを15秒間押すと、システムは自動的にそれをシャットダウンしますのでご注意ください。

    :::
