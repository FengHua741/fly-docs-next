# ファームウェア更新と焼付け

    :::danger 注意

    * 工場出荷時にファームウェアがプリインストールされています

    :::

## ファームウェアのダウンロード

    * ファームウェアの説明
    * `wbtn-vX.X.X-X-XXXXXX-full.bin` はフルファームウェアで、`0x0` アドレスから焼付けが必要です。焼付け後、すべての設定は保持されません
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin` はデルタファームウェアで、`0x10000` アドレスから焼付けが必要です。ファームウェアのみが更新され、既存の設定には影響しません
    * `wbtn-vX.X.X-X-XXXXXX-www.bin` はUIファームウェアで、`0x290000` アドレスから焼付けが必要です。ウェブフロントエンドUIのみが更新され、既存のファームウェアや設定には影響しません
    * <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/airclick/packages">ファームウェアパッケージのダウンロード</Button>

## ファームウェアの更新
    * `AirClick` を電源オフにし、BOOTキーを押しながら `AirClick` のESP32プログラミングポートをコンピュータに接続する必要があります
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.webp').default} size="80%" align="Center" />

    1. USBプログラミングツールをダウンロードして展開する <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">クリックしてダウンロード</Button>
    2. ``flash_download_tool_3.9.4.exe`` を開きます。注意：プログラミングツールを開いた後に最初にCMDウィンドウが表示され、その後にプログラミングツールのインターフェースが表示されます。CMDウィンドウは必ず閉じないでください
    3. プログラミングツールの ``ChipType:`` で ``ESP32-C3`` を選択します
    4. プログラミングツールの ``WorkMode:`` で ``Develop`` を選択します
    5. プログラミングツールの ``LoadMode:`` で ``USB`` を選択します
    6. 最後に ``OK`` をクリックします。以下の図を参照してください
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.webp').default} size="50%" align="Center" />
    7. クリック後、以下の画像が表示されます
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.webp').default} size="50%" align="Center" />

    :::tip

    * 上記の画像内の緑色の選択領域は、画像中のものと一致している必要があります。自分で変更しないでください

    :::

    8. AirClickをUSB-TypeCケーブルでコンピュータに接続します
    9. 上記の画像内の青色の領域1のボタンをクリックして、焼付けたいファームウェアを選択します
    10. 上記の画像内の青色の領域2のボックスに入力して、焼付けアドレスを入力します
    11. 上記の画像内の青色の領域3のボタンをクリックして、AirClickのシリアルポートを選択します（ない場合は、AirClickの焼付けボタンを押し続けながらコンピュータに接続します）
    12. 上記の画像内の青色の領域4のボックスで、ボーレートを最大の `921600` に設定します
    13. 上記の画像内の青色の領域5のボタンをクリックして、焼付けを開始します。焼付けが完了すると、AirClickはファームウェアのアップデートが完了します（ファームウェアの焼付けが完了するまで、AirClickとの接続ケーブルを切断しないでください）
