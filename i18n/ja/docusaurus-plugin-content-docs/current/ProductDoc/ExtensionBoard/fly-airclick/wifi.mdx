# WiFiに接続する

    :::info 重要な注意点

    * 本製品の内蔵アンテナは受信用のみで、送信用のアンテナは自分で取り付ける必要があります。アンテナ端子は`ipex3`です。
    * アンテナを正しく取り付けてください！！！信号が弱いと発熱が激しくなる可能性があります！！！
    * 最初にAirClickホットスポットに手動で接続し、その後対応するWIFIに接続する必要があります。

    :::

## AirClickでWiFiに接続する

    * AirClickのホットスポットに接続します。ホットスポットの名前は`FLY-WBTN-XXXXXX`です
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi.webp').default} size="50%" align="Center" />
    * ホットスポットに接続した後、ブラウザで[http://192.168.4.1/](http://192.168.4.1/#/setting)にアクセスします
    * ここでWIFI設定画面が表示され、プリンターと同じルーターに手動で接続してパスワードを入力し、下部の青いボックスをクリックしてWIFIに接続します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-1.webp').default} size="50%" align="Center" />
    * WiFiに接続した後、AirClickの接続WiFi後のIPアドレスが表示されます。下部の青いボックスをクリックしてデバイスを再起動します
        * 1番目に囲まれているのがAirClickのIPアドレスです
        * 2番目に囲まれたものをコピーしてIPアドレスを再起動します
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/wifi-2.webp').default} size="80%" align="Center" />
    * 接続するには、現在のネットワークを対応するルーターに手動で切り替える必要があります。それ以外の場合、アクセスできません
    * 接続が成功したら、AirClickを再起動することをお勧めします
