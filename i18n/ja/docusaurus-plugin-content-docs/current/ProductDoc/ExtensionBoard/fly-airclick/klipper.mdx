# Klipperに接続する

    :::info 重要な注意点
    * AirClickは目標プリンターと同じネットワークに接続する必要があります
    :::

1. 左側のシステム設定をクリックします
    
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. `Printer Type` を `Klipper` に選択します
3. `Moonraker IP` にklipperマシンのIPアドレスを入力します
4. `Moonraker Port` にklipper上位機のポートを入力します。デフォルトは`7125`です。上位機が他のポートを使用している場合は、対応するポートを手動で入力してください
5. `Apikey` は無視し、一時的にApikeyはサポートされていません

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. 上記の設定を行った後、右下隅の`提出`をクリックして設定を保存します
7. 正しく接続すると、ウェブページの上部にメッセージが表示されます

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. 接続に失敗した場合
    - IPアドレスとポートが正しいことを確認します
    - プリンターとAirClickが同じネットワークに接続されていることを確認します
    - klipper上位機が正常に動作していることを確認します
    - クラウドブラウザがklipper上位機のウェブページを開けるかどうかを確認します
