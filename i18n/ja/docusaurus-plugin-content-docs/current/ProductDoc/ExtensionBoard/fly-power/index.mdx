# FLY VoltUp Power

    FLY-VoltUp Powerこのブーストモジュールは高電圧駆動用に設計されており、ユーザーは高電圧駆動のために追加の高電圧電源を購入する必要がなく、3Dプリンターデザイン時にスペースを節約できます。

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-power/img/power.webp').default} size="50%" align="left" />


## 产品特性

    * 出力電源48V/60Vを選択可能
    * ボードファンポートがあり、4010ファン固定が内蔵されており、電源入力が何Vであれ出力も同じVになります
    * ダブル電源インターフェース拡張：1つの電源インターフェースを提供し、ユーザーが同時に2つの電源出力を拡張でき、それぞれの電源出力を48Vまたは60Vに個別に設定できます
    * インストールが簡単：標準サイズで設計され、さまざまな環境に対応し、追加のドライバーアンダルームのインストールは必要ありません
    * コンパクトサイズ：拡張ボードのサイズは適切で、過度なスペースを占めず、さまざまなプロジェクトに統合しやすくなります。

## 产品规格：

    * 電源インターフェース：12V/24V
    * 出力インターフェース：2つの選択可能な電源 48V/60V（ジャンパーキャップを選択することで選択可能）
    * ファンインターフェース：2つのXH2.54 2P VCC FAN


## 注意事項：

    * インストール前に電源を切り、すべての機器をオフにしてください
    * 48V出力時、モーターカレントは最大2Aまで設定できます。60V出力時は、モーターカレントは最大1Aまで設定できます
    * 2つのVCC FANインターフェースの電圧は、電源入力と一致します。出力電圧は、電源電圧と同じです
    * ブーストモジュールは大きな熱を発生させるため、4010ファンの固定穴が予備されています。冷却のためにファンを装着する必要があります
    * 内蔵の逆接続保護回路と逆流防止チップにより、顧客の誤操作による部品損傷を防ぎます
    * 電圧選択ジャンパーキャップを同時に挿入しないでください。これにより部品が損傷する可能性があります
    * 正負極を正しく接続して、部品を損傷させないようにしてください。
