# FLY-UTOR

    FLY-UTORは高性能でマルチユースのインターフェース拡張ソリューションであり、ユーザーに追加のUSBポートと電源拡張機能を提供し、UARTからRS232への通信もサポートします。この拡張ボードはコンパクトで簡単にインストールでき、さまざまな追加インターフェースや電源出力が必要なアプリケーションに適しています。

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.webp').default} size="100%" align="left" />


## 製品特長

    * デュアルUSBポート拡張：拡張ボードは2つの追加のUSBポートを提供し、USB 2.0規格に対応し、あらゆるUSBデバイスとの互換性を確保し、高速データ転送を実現します。
    * UARTからRS232への変換サポート：内蔵のUARTからRS232へのコンバーターにより、ユーザーはシリアルデバイスをシステムに接続し、デバイスの通信能力を拡張できます。
    * デュアル電源ポート拡張：2つの電源ポートが提供され、同時に2つの外部電源を拡張できます。
    * ステディーな電源設計：拡張ボードには安定した電源回路が設計されており、接続されたデバイスの安定した動作を保証します。
    * 簡単なインストール：標準サイズのデザインで、さまざまな環境に適応でき、追加のドライバーインストールは不要です。
    * コンパクトサイズ：拡張ボードのサイズは合理的で、多くのスペースを占めず、さまざまなプロジェクトに簡単に統合できます。

## 製品仕様：

    * 利用可能なインターフェース：USB X2、UARTからRS232 X2、電源ポート X2
    * 電源ポート：12V/24拡張
    * 兼容性：USBポートを搭載したホストコンピューター

## 注意事項：

    * インストール前に電源を切り、すべてのデバイスを停止してください。
    * 多機能USB拡張ボードをメインボードのUSBポートまたは他のUSB拡張ポートに挿入します。
    * 必要なUSBデバイスや/または電源を拡張ボードに接続します。
    * UARTからRS232機能を使用する場合は、対応するシリアルデバイスを接続します。
    * 再び電源を接続し、デバイスを起動します。
    * システムは自動的に拡張ボードを検出し、設定を行います。必要な場合は後述の設定に関する注意事項を参照してください。
