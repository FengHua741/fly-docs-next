# FLY-UTOR

    FLY-UTORは高性能で多目的なインターフェース拡張ソリューションであり、ユーザーに追加のUSBポートと電源拡張機能を提供し、UARTからRS232への通信もサポートしています。この拡張ボードはコンパクトで、インストールが容易で、追加のインターフェースと電源出力が必要なさまざまなアプリケーションシナリオに適しています。

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.webp').default} size="100%" align="left" />


## 製品特長

    * デュアルUSBポート拡張：拡張ボードは2つの追加のUSBポートを提供し、USB 2.0規格をサポートし、さまざまなUSBデバイスとの互換性と高速データ転送を確保します。
    * UARTからRS232へのサポート：内蔵のUARTからRS232変換器により、ユーザーはシリアルデバイスをシステムに接続し、デバイスの通信能力を拡張することができます。
    * デュアル電源ポート拡張：2つの電源ポートを提供し、ユーザーが同時に2つの外部電源を拡張でき、追加の電源ニーズを満たします。
    * 安定した電源設計：拡張ボードは安定した電源回路で設計されており、接続されたデバイスの安定した動作を確保します。
    * インストールが簡単：標準サイズで設計されており、さまざまな環境に適応し、追加のドライバーインストールは必要ありません。
    * コンパクトなサイズ：拡張ボードのサイズは適切で、多くのスペースを占めず、さまざまなプロジェクトに組み込むことができます。

## 製品仕様：

    * 利用可能なインターフェース：USB X2、UARTからRS232 X2、電源インターフェース X2
    * 電源インターフェース：12V/24V拡張
    * 互換性：USBポート付きのホストマシン

## 注意事項：

    * インストール前に電源を切り、すべてのデバイスをシャットダウンします。
    * 多機能USB拡張ボードをメインボードのUSBポートまたは他のUSB拡張ポートに挿入します。
    * 必要なUSBデバイスや/または電源を拡張ボードに接続します。
    * UARTからRS232機能を使用する場合は、対応するシリアルデバイスを接続します。
    * 電源を再接続し、デバイスを起動します。
    * システムは自動的に拡張ボードを検出し、設定を行います。必要に応じて、後述の設定に関する注意事項を参照してください。
