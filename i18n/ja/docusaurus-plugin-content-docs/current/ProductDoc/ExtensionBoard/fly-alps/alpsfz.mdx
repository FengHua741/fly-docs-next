# モジュールのトリガ値の調整方法

## レベルセンサモジュールを開く
    `Klipper`コンソールに以下を入力して、`SET_PIN PIN=_probe_ready VALUE=1`を送信し、トリガインジケータライトが点灯しないようにしきい値を小さなものから徐々に大きく調整します。以下のコマンドを入力してください:
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/sewt_pin.webp').default} size="100%" align="left" />

## 触発値の調整
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">シリアルポートツールアシスタントを取得</Button>

    * USBケーブルを使用してモジュールとパソコンを接続し、シリアルポートツールアシスタントを開き、対応するポート番号を選択し、ボーレートを115200に設定して「接続」をクリックします。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    シリアルポートツールで以下コマンドを入力して触発値を調整します:
    ```cfg
    set 触発値
    ```
    デフォルトの触発値: 20000
    数値が小さいほど感度が高く、数値が大きいほど感度が低くなります。
    * 例: `set 20000`、触発値を20000に設定します。数値が小さいほど感度が高く、数値が大きいほど感度が低くなります。
    * シリアルポートツールで使用するコマンド:

    * シリアルポートに`v`を送信すると、実時間でabc値が表示されます。a:フィルタリング前；b:フィルタリング後。
    * シリアルポートに`uv`を送信すると、出力を停止します。
    * シリアルポートに`rt`を送信すると、現在のしきい値を確認できます。
