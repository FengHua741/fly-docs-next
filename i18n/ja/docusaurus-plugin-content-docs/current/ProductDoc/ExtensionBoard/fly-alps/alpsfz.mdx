# モジュールのトリガ値の調整方法
シリアルツールを使用して、以下のコマンドが利用可能です：
Klipperコンソールに「SET_PIN PIN=_probe_ready VALUE=1」と送信し、トリガインジケーターを観察しながら、インジケーターが点灯しなくなるまで閾値を小さなものから大きく調整します。次のコマンドを入力してください：
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
シリアルツールで以下のコマンドを入力してトリガ値を調整します：
```cfg
set 触发值
```
デフォルトのトリガ値：20000
* シリアルツールで使用可能なコマンド：

* `v`をシリアルに送信すると、adc値がリアルタイムで出力されます。（a:フィルタ前；b:フィルタ後）
* `uv`をシリアルに送信すると、出力を停止します。
* `rt`をシリアルに送信すると、現在の閾値を確認できます。
