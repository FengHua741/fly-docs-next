# 製品概要

  - MDM切れ検知センサーは、小型で使いやすく、トリガーが非常に敏感です。
  - Klipper、Marlin、RRFなどの3Dプリンタに対応しています。
  - 切れ検知：高速赤外線スキャンにより、0.25秒以内にフィラメントの切断を検出し、即座に印刷を停止して警告します。無駄な消費を防ぎます；
  - ブロック警告：エンコーダー光信号による識別で、ノズルやホッパーの詰まりの前兆を検出します。廃棄物率を80％以上削減します。
  - 兼容性が広い：PLA、ABS、PETG、TPUなど、すべてのタイプのフィラメント（直径1.75mm）に対応し、硬質材料や柔軟なフィラメントも安定して認識できます；
  - インストールが簡単：取り付け図、配線図、設定ファイルなどの資料が提供され、インストールが容易です
