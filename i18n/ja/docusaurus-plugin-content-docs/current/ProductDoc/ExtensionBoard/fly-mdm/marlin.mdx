# Marlin設定

## 断料および詰り機能の設定

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/marlin.webp').default} size="70%" align="center" />

1. Marlin設定ファイル `Configuration.h` を開き、`#define FILAMENT_RUNOUT_SENSOR` を見つけ、その前のコメント記号を削除します。
2. `#define FILAMENT_RUNOUT_DISTANCE_MM` を見つけ、その前のコメント記号を削除し、値を `3.5` に設定します（使用しているディスクによって異なります。誤検出が発生した場合はこの値を適宜増やしてください）。
3. `#define FILAMENT_MOTION_SENSOR` を見つけ、その前のコメント記号を削除します。
4. Marlinのフィラメントセンサー機能を使用するには、`#define NOZZLE_PARK_FEATURE` と `#define ADVANCED_PAUSE_FEATURE` も有効にする必要があります。コメントを削除して、プリンターの設定に応じて具体的なパラメーターを設定してください。
