# モデルのダウンロードと組み立て

## モデルのダウンロードと説明

### 3Dモデルファイルのダウンロード

<Button variant="contained" disableElevation href="http://cdn.mellow.klipper.cn/STEP/MDM-filament_sensor.zip">クリックしてモデル圧縮パッケージをダウンロード</Button>

----

### モデルファイルの説明

| モデルファイル名 | 説明 |
| :---: | :--- |
| pcb_cover.stp | PCBカバー |
| switch_sensor_trigger_block.stp | 断料検出トリガーブロック |
| rear_cover.stp | 後ろカバー |
| upper_cover.stp | 上カバー |
| main_body.stp | モデル本体 |
| encoder_3_5.8mm.stp | エンコーダー盤、3バンド、最小検出距離5.8mm <br/> `高流量（20 mm³/s以上）印刷にはこのモデルを使用できます` |
| encoder_5_3.5mm.stp | エンコーダー盤、5バンド、最小検出距離3.5mm <br/> `適度で、推奨されます` |
| encoder_10_1.73mm.stp | エンコーダー盤、10バンド、最小検出距離1.73mm <br/> `低速高精度印刷にはこのモデルを使用できます` <br/> 3Dプリンターのファームウェアによる素材検出のサンプリングレートの制限があります |

## 組み立て

:::warning モデル印刷に関する注意事項

- 外殻は `PETG` および `透明素材` での印刷は避けるべきです。センサーの感応に影響します。

:::
