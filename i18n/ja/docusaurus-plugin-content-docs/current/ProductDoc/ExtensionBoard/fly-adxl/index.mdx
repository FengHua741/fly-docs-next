# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />



## 製品特長

    * 36モーターなどの場所に適用できます
    * Raspberry Pi、FLY PI、Orange Pi、マザーボードなどのSPIインターフェースに接続できます

## 仕様

    * ボード搭載インターフェース：ZH1.5 6Pコネクタ
    * コミュニケーション方法：SPI
    * 電圧：5V

## 注意事項

    :::warning 注意事項

    * 電圧は5Vです。間違えないでください
    * SPI接続が可能であることを確認してください

    :::
