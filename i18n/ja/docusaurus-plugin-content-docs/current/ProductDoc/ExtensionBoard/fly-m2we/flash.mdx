# システム焼成

:::danger 注意

* M2WEは工場出荷時に既にFLYOSがプリインストールされています
* 再び焼成する場合は下記の手順を参照してください

:::

## オペレーティングシステム初心者ガイド

<Button variant="contained" disableElevation href="/docs/DebugDoc/baodian/host/">オペレーティングシステム初心者ガイド</Button>

## システム焼成前の準備

* SDカードリーダー
* 容量が16GB以上または128GB以下のTFカード、転送速度はクラス10以上である必要があります
* 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロードします
* 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼成ツール</Button> をダウンロードします

## システム焼成

:::info
M2WEは3つのモードをサポートしています。私はTFカードをM2WEに移行することと、M2SDバーナーを使用してM2WEに焼成することをお勧めします。
:::

:::info
* マシンにM2WEとメモリカードがインストールされている場合、システムはデフォルトでメモリカードから起動します
* M2WEはTFカードを使用してシステムを焼成することをお勧めします。USB経由での焼成はドライバーの互換性などの理由により失敗する可能性があります。直接[メモリカードを使用してM2WEに焼成する](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)ことをお勧めします
* ダウンロードしたシステムイメージが対応していることを確認してください。対応していない場合は起動できません
:::
