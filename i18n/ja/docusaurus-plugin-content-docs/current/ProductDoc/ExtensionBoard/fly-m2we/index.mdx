## 注意事項

:::danger 注意事項

* M2WE出廠已經預刷好FLYOS可以直接啟動
* M2WE只能使用在FLY的產品上，不支援其他設備
* M2WE的天線端子是`IPEX1`
* M2WE不支援2.4G與5G雙頻合一、中文、特殊字元、預設隱藏的WIFI

:::
