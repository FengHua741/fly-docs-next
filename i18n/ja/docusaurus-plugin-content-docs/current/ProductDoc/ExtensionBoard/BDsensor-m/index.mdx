# BDsensor-m

    BDセンサーは3Dプリンターと組み合わせて使用できる距離センサーで、リアルタイムで高速なベッドレベルングとZ軸の調整が可能です。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## 機能
    * インストール紹介
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Z軸リミッターの代わり
    * スキャン時にはZ軸の昇降を必要とせず、プリント速度を向上させる
    * XY移動スキャン、密なベッド探査が可能
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## 接続

    * 電源：5V
    * ファームウェア：Klipper/Marlin
    * 接続：マザーボード（BDsensorはマザーボードのexp1インターフェースを使用する）
    * ヒーターベッド：対応：普通のPEI、アルミニウム製ヒーターベッドなどの金属製ヒーターベッド、マグネットヒーターベッドや非金属製ヒーターベッドには対応しない

## 注意点

    * Klipperはklipperと最新のBD-SENSOR-mパッチをダウンロードし、マザーボードのファームウェアを再インストールする必要があります
    * デバッグ時は緊急停止ボタンが常に触れる場所に置いておくこと、PEIを傷つけないため
    * スキャン速度は200速を推奨！
    * Z軸リセット速度は5以下を推奨
