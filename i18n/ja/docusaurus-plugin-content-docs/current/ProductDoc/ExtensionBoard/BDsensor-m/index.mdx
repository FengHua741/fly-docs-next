# BDsensor-m

    BDsensorは、3Dプリンタと一緒に使用できる距離センサーで、リアルタイムで高速ベッドレベリングとZ軸の調整を行うことができます。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.webp').default} size="50%" align="left" />

## 機能

    * Z軸リミットスイッチの代わり
    * 走査時にZ軸を上げ下げする必要がないため、印刷速度が向上
    * XY移動による走査が可能で、密集網ベッド検出が可能

## 接続

    * 電源：5V
    * ファームウェア：Klipper/Marlin
    * 接続：メインボード（BDsensorはメインボードのexp1ポートを使用）
    * ヒートベッド：サポート：通常のPEI、アルミ板ヒートベッドなどの金属表面ヒートベッド。磁石の穴あきヒートベッドと非金属表面ヒートベッドはサポートしていません

## 注意事項

    * Klipperを使用する場合、Klipperと最新のBD-SENSOR-mパッチをダウンロードして、メインボードのファームウェアを再フラッシュする必要があります
    * デバッグ中は、PEIを傷つけないように緊急停止スイッチをすぐに作動できる場所に置いてください
    * ベッド走査速度は200を推奨！
    * Z軸のホーム速度は5を超えないことを推奨します
