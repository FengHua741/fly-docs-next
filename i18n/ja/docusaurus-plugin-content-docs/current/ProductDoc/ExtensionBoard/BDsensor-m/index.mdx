# BDsensor-m

    BDセンサーは、3Dプリンターと連携して使用できる距離センサーで、リアルタイムで高速ベッド調整とZ軸の調整が可能です。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## 機能
    * インストールガイド
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Z軸リミットスイッチの代替
    * スキャン時にZ軸を上下させる必要がなく、プリント速度が向上
    * XY移動スキャンにより、密集したネットベッド探査が可能
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## 接続

    * 電源：5V
    * ファームウェア：Klipper/Marlin
    * 接続：メインボード（BDセンサーはメインボードのexp1インターフェースを使用します）
    * ヒートベッド：サポート：通常のPEI、アルミ板ヒートベッドなどの金属表面ヒートベッド、磁石穴あきヒートベッドと非金属表面ヒートベッドはサポートしません

## 注意事項

    * Klipperを使用する場合は、klipperと最新のBD-SENSOR-mパッチをダウンロードし、メインボードのファームウェアを再フラッシュする必要があります
    * デバッグ時には、緊急停止スイッチをすぐにトリガーできる場所に置いてください。PEIを傷つけないように注意してください
    * ベッドスキャンの速度は200を推奨！
    * Z軸の原点復帰速度は5を超えないようにしてください
