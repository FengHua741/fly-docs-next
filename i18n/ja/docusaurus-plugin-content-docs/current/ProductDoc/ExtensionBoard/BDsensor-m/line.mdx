    :::tip

    * **BDsensor-m**は直接マザーボードのEXP1と*BL-TOUCH**インターフェースを使用できます

    :::

    :::danger 注意事項

    * 配線の順序が正しいか確認してください。そうでないと、デバイスが破損する可能性があります

    :::
