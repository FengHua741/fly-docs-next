# FLY-G2T

    FLY-G2T多機能拡張ボードは、Raspberry PiとFLY上位機の機能を拡張するための高性能モジュールです。UART通信をサポートし、FPCインターフェイスを統合してTFT-V2に外部接続でき、さらに5Vファン制御もサポートしています。これにより、Raspberry Piのインタラクティブ性と利用可能性を向上させ、産業自動化、3Dプリンタ、ホームオートメーション、ロボット制御、または人間と機械のインターフェイスと外部通信が必要なあらゆる場面で理想的なソリューションとなります。

## 产品特点

    * 対応性：Raspberry Piの複数バージョンをサポートし、Raspberry Pi Zero/3B/3B+/4Bなどを含みます。
    * UARTからRS232への変換：安定したシリアル通信を提供し、Raspberry PiがRS232インターフェイスを介して他のハードウェアデバイスとデータ交換できるようにします。
    * SPIタッチスクリーン：FPCインターフェイスを介して接続されたTFT-V2タッチスクリーンは、直感的なタッチ操作を提供し、klipperscreenなどのグラフィカルユーザーインターフェイスが必要なアプリケーションに適しています。
    * 5Vファン対応：5Vファンインターフェイスが統合されており、ファンを接続して放熱することができます。これにより、長時間の動作や高負荷状態でも上位機の安定性が保たれます。
    * プラグアンドプレイ設計：拡張ボードは簡単にインストールでき、上位機に直接プラグアンドプレイできます。追加のワイヤーや複雑な設定は必要ありません。
    * コンパクトサイズ：拡張ボードのサイズは適切であり、多くのスペースを占めず、さまざまなプロジェクトに簡単に統合できます。

## 产品规格

    * 通信インターフェイス：UARTからRS232への変換
    * 表示インターフェイス：SPIからFPCインターフェイスへの変換
    * ファン電源：5V
    * 対応性：Raspberry Pi Zero/3B/3B+/4B、FLY Pi V2など

## 注意事項

    :::warning 注意事項

    * RS232接続ツールボードを使用する際は、必ず上位機と共地にしてください！！！！！

    :::

    * 拡張ボードのインストール前に、必ず上位機の電源をオフにしてください。
    * 拡張ボードは湿気やほこりの多い環境での使用を避けてください。
    * スクリーンをインストールする際は、電源を入れたまま抜き差ししないよう注意してください。また、画面線の向きにも注意して、破損を防いでください。
    * ファンをインストールする際は、ファンの電流と電圧仕様に注意してください。破損を防ぐために、適切な仕様のものを使用してください。
