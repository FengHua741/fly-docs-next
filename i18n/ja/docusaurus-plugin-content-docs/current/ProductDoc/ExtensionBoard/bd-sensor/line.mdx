    :::tip

    * **BDsensor**はメインボードとツールボードの**BL-TOUCH**インターフェイスを使用できます。

    :::

    :::danger 注意点

    * ワイヤーの接続順が正しくない場合、デバイスが破損する可能性があります。

    :::
