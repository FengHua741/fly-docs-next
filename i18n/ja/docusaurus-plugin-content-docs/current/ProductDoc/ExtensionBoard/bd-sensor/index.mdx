# BDsensor

    BDセンサーは3Dプリンターと連携して使用できる距離センサーで、リアルタイムに高速なベッドレベルングとZ軸調整が可能です。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.webp').default} size="50%" align="left" />

## 機能

    * Z軸リミッタの代わりとして使用可能
    * スキャン時にZ軸を上げ下げる必要がないため、プリント速度が向上
    * XY移動スキャンにより、密なベッド探査が可能です

## 接続

    * 電源：5V
    * ファームウェア：Klipper/Marlin
    * 接続：ツールボード/メインボード（BDsensorはサーボポートとリミットポートを使用します）
    * ヒートベッド：普通のPEI、アルミニウム製ヒートベッドなどの金属表面のヒートベッド、マグネットヒートベッドや非金属表面のヒートベッドはサポートされていません

## 注意事項

    * KlipperにはKlipperと最新のBD-SENSORパッチをダウンロードし、メインボードのファームウェアを再インストールする必要があります
    * デバッグ時は常に緊急停止がすぐにトリガーできる場所に置いてください、PEIを傷つけないように
    * スキャン速度は推奨200速です！
    * Z軸リセット速度は5を超えないことをお勧めします
