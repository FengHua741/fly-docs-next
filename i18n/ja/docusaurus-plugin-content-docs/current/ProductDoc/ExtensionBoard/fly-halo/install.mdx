## インストールガイド

## Voron Stealthburnerへのインストール

    * HaloをStealthburnerに取り付けるには、提供されるアダプターを使用する必要があります
    * 提供される2つのSTLファイルをダウンロードし、自分で印刷してください
        * [Stealthburner_main_body(Adapt_to_Halo).stl(クリックしてダウンロード)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body\(Adapt_to_Halo\).stl)
        * [Halo_mounting_bracket.stl(クリックしてダウンロード)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(クリックしてダウンロード)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.webp').default} size="50%" align="left" />


    :::warning 注意事項

    * 出力色の説明 **黒が負極GND**、**赤が正極VCC DC12-24Vを受け入れ可能**、**白がGPIO47 ブロック検出に使用可能**、**黄色がGPIO48 打ち止め停止に使用可能**

    :::
