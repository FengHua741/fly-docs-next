# 使用教程

## 注意事項

    * 初回使用必須接続ネットワーク、もし、あなたのWi-Fiがインターネットに接続できない場合。haloは初期設定の取得ができない
    * 設定ページで全ての設定は「設定を保存」ボタンをクリックし、デバイスを再起動した後に効力を持つ
    * 料理詰まり検出機能はデフォルトでオフになっており、設定で有効化して再起動した後からホーム画面から上スワイプでアクセスできる
    * 打ち終わって電源を切る機能はデフォルトでオフになっており、設定で有効化できる
    * 全てのポップアップウィンドウで印刷をキャンセルするか一時停止するボタンは長押しで有効であり、単純なクリックは無効（誤操作を防ぐため）
    * システムサービスページではHostサービスは上位機器ホストである
    * 公式ファームウェアを使用している場合は、メインコントローラチップまたはフラッシュチップを交換しないでください

    :::tip

    * 注意：あなたの`moonraker.conf`設定ファイルのサイズが4MBを超える場合、正常に動作しない可能性があります。問題が発生した場合は、`moonraker.conf`設定ファイルのサイズを縮小してください

    :::

## システムファームウェアの更新

    * haloバージョンの更新はフルアップデートとOTA増分アップデートの2種類がある
        * フルアップデート：基盤となる大きなアップデートが必要な場合、USBを使用してフルファームウェアパッケージを焼く必要がある
        * 増分アップデート：通常のOTAアップデート、デバイス情報ページで更新を確認できる

## スマートフォンでQRコードを読み取りWIFIを設定

    :::tip

    * 一部のモバイルデバイスにはQRコードでWi-Fiに接続する機能がない場合、スマートフォンのWi-Fi設定で名前が**FLY-halo-xxxxxxxx**のホットスポットに接続できる

    :::

    :::tip

    * ホットスポットに接続した後、ウェブページが自動的に表示されない場合は、手動でブラウザを開いてリンクを開く必要があります

    :::

    :::tip

    * 注意：HALOは2.4GWi-Fiのみ使用可能です

    :::

    :::tip

    * Haloディスプレイは隠されたWi-Fiを検索できません

    :::
