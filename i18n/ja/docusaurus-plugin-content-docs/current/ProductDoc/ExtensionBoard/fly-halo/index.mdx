# 製品紹介

    * メインコントローラはESP32-S3R8を使用しており、性能はESP32よりも優れています。8MBのPSRAMと16MBのフラッシュメモリを搭載しています。画面はIPSディスプレイで、解像度が高く、明るさも高く、視野角が広く、色再現性も優れています。電気容量タッチをサポートします。

## 製品特長

    :::tip

    * 現在6つのページがあり、左右上下にスライドして切り替えることができます。

    :::

    * ホームページ（待機、印刷、停止、エラーなどの状態を表示）

    * エキスルージョン（エキスルージョン、引き抜き、エキスルージョン温度を制御）

    * システムサービス（ホスト、Klipper、Moonrakerなどのサービスの起動、停止、再起動を制御）

    * 設定（Haloのさまざまなパラメータ、Wi-Fi、IP、言語、バックライトの明るさなどを設定）

    * デバイス情報（Haloの情報を表示し、ファームウェアバージョンを確認、更新チェックなど）

    * コンジューム・デテクション：フィラメント切れや詰まりの検出、フィラメント長の検出、詰まりが指定された長さを超えた場合に自動的に一時停止し、GPIO47を上げることができます（これらはすべてデフォルトでオフになっており、設定で有効にすることができます）

    * プリント完了後のシャットダウン：プリントが完了した後に遅延時間を設定し、その時間後にホストをシャットダウンし、GPIO48を上げることができます（これらはすべてデフォルトでオフになっており、設定で有効にすることができます）
