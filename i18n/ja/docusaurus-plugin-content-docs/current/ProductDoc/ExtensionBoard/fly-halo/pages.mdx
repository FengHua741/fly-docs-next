# Haloページ操作の紹介

## ページ構造

```bash
  プリントファイルページ      ホームページ         エクストルページ            システムサービスページ             設定ページ              デバイス情報ページ
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Print Files |<-->|     Home    |<-->|   Extrude   |<-->| Sys Service |<-->|   Setting   |<-->|  Dev Info   |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Blocking Det|
                   |             |
                   +-------------+
                    ブロッキング検出ページ
            （設定でブロッキング検出を有効にするとこのページが表示されます）

```

## エクストルページ

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.webp').default} size="50%" align="left" />
    * 上図1の操作可能な領域は下記の番号に対応しています
    1. クリック：エクストルヘッド目標温度を1増加させます
    2. ロングタップ：エクストル温度設定に進み、リングバーをドラッグして温度を調整します。再度クリックして設定した温度を確認します
    3. クリック：エクストルヘッド目標温度を1減少させます
    4. クリック：エクストル
    5. 上下スワイプ：エクストルまたはリトラクトの長さを選択します、単位はmmです
    6. クリック：リトラクト
    7. ロングタップ：上図2のエクストル速度設定に進み、上下スワイプでエクストル速度を選択し、単位はmm/sです。選択後、周囲の空白部分をクリックして終了します
