# FLY_halo

## 1. ホームページ

    シリーズドライバーまたはヒートエンドをクリックし、画面の上から時計回りに到達したい温度にスワイプしてから、シリーズドライバーやヒートエンドをクリックして温度を設定します。

    最高温度は設定ファイルで読み取られるため、mcu温度とホスト機器温度を表示するには次の構成を追加する必要があります。

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # mcuに関連付け（デフォルト）

    [temperature_sensor host]        # ホスト機器温度
    sensor_type: temperature_host     # ホスト機器に関連付け
    ```


    :::warning 注意点

    * 最高温度は設定ファイルの設定に基づいています

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.webp').default} size="50%" align="left" />


## 2. エキストラージョン

    * イコンを長押しすると、ヒートエンドの温度、エキストラード長さ、エキストラード速度を設定できます。

    :::warning 注意点

    * haloは単一のヒートエンドの制御のみをサポートしています

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/Extrusion.webp').default} size="50%" align="left" />
