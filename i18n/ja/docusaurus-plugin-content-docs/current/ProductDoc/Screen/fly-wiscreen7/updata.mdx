## 更新
    * 1.ダウンロードしたファイルを`update.img`にリネームします
    * 2.`update.img`ファイルをSDカード/USBドライブに入れてスクリーンに挿入できます（スクリーンが起動している状態でも挿入可能です）
    * 3.スクリーンを選択し、アップグレードをクリックしてアップデートを完了させることができます
