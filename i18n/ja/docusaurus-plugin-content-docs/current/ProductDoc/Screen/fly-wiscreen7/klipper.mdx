## Klipper設定

### LED設定

    ```cfg
    [led LED]
    white_pin:PA15
    ```

### PTC設定

    ```cfg
    [heater_generic warehouse]
    heater_pin:E_CS
    sensor_type:ATC Semitec 104GT-2
    sensor_pin:PA1
    min_temp:-270
    max_temp:600
    control = pid
    pid_kp = 16.804
    pid_ki = 0.794
    pid_kd = 88.854
    ```

### FAN設定

    ```cfg
    [fan_generic fan1]
    pin: FAN1
    ```

    ```cfg
    [fan_generic fan2]
    pin: FAN2
    ```

 

### カスタムGcode

    * スクリーンを下にスワイプするとカスタムGcodeが表示されます。
    * 1つのボタンで複数のGcode形式を送信できます：
    * 例：

    ```cfg
    G91 
    G1 X10 F6000
    G90
    ```

    * スクリーン上で入力する形式:(2行目以降、各コマンドの前に\nを追加)

    ```cfg
    G91\nG1 X10 F6000\nG90
    ```



## 使用オブジェクトの除外

    * moonrakerを置き換える必要があります。

    * SSHに接続し、正常にインターネットに接続できることを確認してください。

    1. moonrakerファイルのバックアップ

    ```bash
    mv ~/moonraker ~/moonraker-bak
    ```

    2. moonrakerサービスを停止

    ```bash
    sudo systemctl restart moonraker
    ```

    3. moonrakerをpull

    ```bash
    git clone https://e.coding.net/g-ofpa1390/3D-Printers/moonraker-dev.git -b flylcd-dev moonraker
    ```

    4. ブラウザで上位機にアクセスし、`moonraker.conf`を追加してからシステムを再起動します

    ```bash
    [network]
    ```

    5. 次に、以下のアドレスをブラウザに入力します。IPは上位機のIPに置き換えてください。

    ```bash
    http://ip:7125/server/network/wifi/scan
    ```

* このステップが表示されたら操作完了です
