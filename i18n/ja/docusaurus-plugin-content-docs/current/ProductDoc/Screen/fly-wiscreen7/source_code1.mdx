# プロジェクト構造の紹介

## 介绍
#### Logicファイル夹
* 私たちは99％のuiコードプログラミングをLogicファイル夹で行います。他のファイル夹は気にする必要はありません
* gcodeLogic.ccはmain.cに似ていて、99％のuiコードがここに存在します。他のファイルはあまり気にする必要はありません

#### i18nファイル夹
* 各国の言語翻訳表を保存しています

#### resourcesファイル夹
* uiで使用される画像リソースを保存しています

#### uiファイル夹
* uiインターフェイスファイルを保存しています。ダブルクリックして開くと表示されます
* 左下の矢印が指す「小さな目」をダブルクリックすると、隠されたインターフェイスを開くことができます
