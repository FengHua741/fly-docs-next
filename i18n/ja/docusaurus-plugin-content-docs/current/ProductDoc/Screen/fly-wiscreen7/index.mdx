# 簡介

      Fly_WiScreenは広州メイレン電子科技有限公司Fly3Dメインボード開発チームが提供するハイエンドKlipper制御画面で、Wi-Fiと有線LANの2つのモードで3Dプリンタを制御できます。SDカードやUSBメモリを使って印刷ファイルを転送でき、画面の応答速度が非常に速いためユーザーに良い体験を与えます。マスターよりスレーブへの設定をサポートし、複数の機器を同期操作してユーザーが機器を管理するのを容易にします。また、5V給電ポートには充電モジュールを取り付けることができ、バッテリーへの改造を容易にし、モバイル性を向上させます。

    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/wiscreen.webp').default} size="100%" align="left" />

## 製品機能

    * ウィークリーパーインストール済みの上位機器とのWi-Fiおよび有線LAN接続をサポート
    * Wi-Fi検索をサポートし、Wi-Fiに接続すると自動的に上位機器を検索できます
    * プリンタ群を制御して複数のプリンタを一括管理
    * USBおよびTFカードを使用して印刷データをアップロード
    * カスタムマクロをサポートし、操作とデバッグをより便利にします

## 製品仕様

    * CPU: 全志T113-S3、デュアルコアCORTEX-A7、最大1.2GHz
    * メモリ: 128MB DDR3統合
    * ストレージ: 128MB NANDフラッシュ、TFカードスロット
    * 表示: 1024x800キャパシティブスクリーン
    * 外部インターフェース: USB 2.0ポート1つ、TFカードスロット1つ
    * Wi-Fiプロトコル: 802.11b/g/n
    * スクリーンタイプ: IPS
    * スクリーンサイズ: 7インチ
    * マルチタッチ: 対応
    * 定格電圧: 5V
