# 開機圖片修改
:::tip
    請嚴格按著`解像度`以及`檔案名`製作圖片，否則系統將不會識別

    將製作好的圖片，放入到SD卡/U盤根目錄，在螢幕開機狀態下，插入螢幕，點擊升級即可
:::
## 開機LOGO修改

    * 圖片要求`800*480`
    * 名字要求`boot_logo.JPG`
    * 不得大於`105KB`




## 第一介面圖片修改

    * 圖片要求`500*400`
    * 名字要求`print.png`
    * 不得大於`45KB`



## 打印介面圖片修改

    * 圖片要求`311*480`
    * 名字要求`print_smail.png`
    * 不得大於`45KB`
