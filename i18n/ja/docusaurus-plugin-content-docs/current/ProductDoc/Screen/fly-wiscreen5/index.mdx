# 簡介

      Fly_WiScreenは広州メイレン電子科技有限公司Fly3Dメインボード開発チームが提供するハイエンドKlipper制御スクリーンで、Wi-Fi経由で3Dプリンターを制御できます。SDカードやUSBメモリによるプリントファイルの転送をサポートし、画面の応答速度が非常に速く、ユーザーに優れた体験を提供します。マスターよりスレーブへの設定をサポートし、複数の機器を同期操作することでユーザーが機器を管理するのが容易になります。5V給電ポートには充電モジュールを接続でき、バッテリーへの改造を容易にし、モバイル性を向上させます。

## 製品機能

    * Wi-Fi接続を介して他のKlipperがインストールされた上位機器と接続をサポートします
    * Wi-Fi検索をサポートし、Wi-Fiに接続した後は自身で上位機器を検索できます
    * 複数のプリンターをクラスター制御できます
    * USBとTFカードを使用してアップロード印刷をサポートします
    * カスタムマクロを使用し、操作デバッグがより簡単になります

## 製品仕様

    * CPU: 全志T113-S3、デュアルコアCORTEX-A7 最大1.2GHZ
    * メモリ: 統合128M DDR3
    * ストレージ: 128MB NANDフラッシュ、TFカードスロット
    * 表示: 800x480 キャパシティブディスプレイ
    * 外部インターフェース: USB 2.0ポート1つ、TFカードスロット1つ
    * Wi-Fiプロトコル: 802.11b/g/n
    * ディスプレイタイプ: IPS
    * ディスプレイサイズ: 5インチ
    * マルチタッチ: 対応
    * 定格電圧: 5V
