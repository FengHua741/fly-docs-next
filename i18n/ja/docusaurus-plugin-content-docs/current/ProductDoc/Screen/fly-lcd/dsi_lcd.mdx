## 产品简介

    * スクリーンサイズ：120.7mm*76.0mm
    * スクリーン表示領域：108mm*66.2mm
    * 解像度：800*480
    * スクリーンタイプ：IPS
    * 通信方式：DSI/Mipi
    * ボードインターフェース：15ピンDSI/22ピンMipi
    * 使用温度範囲：0°-60°C
    * タッチタイプ：Capacitive（静電容量式）
    * ロジック電圧：3.3V
    * 重量：135.5g

        <TabItem value="ラズベリーパイ5BはMipiインターフェースを使用" default>

        :::danger 注意事項

        * ラズベリーパイ5のDSIとCSIインターフェースは今やMIPIに名前が変わりました
        * MIPIインターフェースは設定によりCSIカメラまたはDSIスクリーンを直接使用できます
        * ゴールドフィンガーの向きに注意してください

        :::

        * ラズベリーパイ5には現在2つのMIPIインターフェースがあり、正面から見ると1と0のマークが見えます

        * DSIスクリーンの配線


            * 1番インターフェースを使用する場合は、以下の画面設定を行ってください

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * 0番インターフェースを使用する場合は、以下の画面設定を行ってください

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
