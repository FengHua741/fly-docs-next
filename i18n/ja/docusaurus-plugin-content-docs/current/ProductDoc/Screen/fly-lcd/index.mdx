# 簡介

    * 分体式設計，支援多種核心板DSI與HDMI
    * 多種尺寸，4.3吋，4吋，7吋
    * 高清解析度，不錯過每一個細節
    * 支援10級亮度調節
    * 支援180°螢幕翻轉

:::warning 注意事項
七吋螢幕適用HDMI核心板無法選擇，DSI核心板正常控制
:::
