# 紹介

* このデバイスは、Fly設計のメインボードに直接接続可能なユニークなFPC（フレキシブルプリント基板）ポート設計を採用しており、HDMIやUSBケーブルは不要です。この簡略化されたFPC接続は、ケーブルの混乱を減らし、クリーンで効率的なインターフェースを保証します。しかし、この先進的な接続方法を採用しているにもかかわらず、従来の接続オプションを好むユーザーのために標準のHDMIポートも保持しています。
柔軟な接続性以外にも、このデバイスには明るさ調整、画面回転、再起動を簡単に行えるスマート機能キーが含まれています。高品質の輸入泡状接着剤が強力な画面接着力と優れた耐久性を確保します。両面低抵抗放熱技術はタッチ感度を強化し、ユーザーエクスペリエンスをよりスムーズにします。黄光技術は正確な色再現を提供し、標準輝度は500で過度な露出を避けます。画面は高強度強化ガラスで保護されており、堅牢な防壊保護を確保します。
このデバイスは、Raspberry Piなどのホストシステムと完全に互換性があり、隠しHDMIポート設計を採用し、ケーブルはデバイスの後ろにきれいに隠れ、外観がクリーンでプロフェッショナルです。その高性能回路設計により、シームレスな動作が保証され、安定して信頼性の高いパフォーマンスを発揮する3Dプリンタに最適な選択肢となっています。
