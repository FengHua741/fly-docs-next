# 紹介

* このデバイスは、HDMIやUSBケーブルを使用せずにFly設計のホストマザーボードに直接接続できる独自のFPC（フレキシブルプリント回路）ポート設計を採用しています。この簡素化されたFPC接続は、ケーブルの混乱を減らし、クリーンで効率的なインターフェースを確保します。しかし、この先進的な接続方法を採用しているにもかかわらず、従来の接続オプションを好むユーザーのために、標準のHDMIポートも保持されています。
柔軟な接続性に加え、このデバイスには、明るさの調整、画面の回転、再起動が簡単にできるスマート機能キーが含まれています。高品質の輸入フォーム接着剤は、強力な画面接着力と優れた耐久性を保証します。両面低抵抗放熱技術により、タッチ感度が強化され、ユーザーエクスペリエンスがよりスムーズになります。黄色光技術は、正確な色再現を提供し、標準輝度は500で、過度の露出を避けます。画面は、高強度強化ガラスで保護されており、堅牢な防壊保護を確保しています。
このデバイスは、Raspberry Piなどのホストシステムと完全に互換性があり、隠されたHDMIポート設計を採用しており、ケーブルはデバイスの背面にきれいに隠れています。外観は清潔でプロフェッショナルです。その高性能回路設計は、シームレスな動作を保証し、安定して信頼性の高い3Dプリンタの理想的な選択肢となっています。
