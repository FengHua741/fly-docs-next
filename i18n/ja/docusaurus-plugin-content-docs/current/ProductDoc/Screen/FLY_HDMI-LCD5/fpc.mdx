# 画面パラメータの変更ガイド
:::tip
ブラウザであなたのIP:9999を開いてFLY-Toolsにアクセスしてください
:::

:::tip
FLYの上位機板載のHDMIポートはMicro HDMIです。HDMIケーブルを購入する際は注意してください！！！
:::

:::tip
FLYの上位機がFPC一線通をサポートしているのは`FLY-PI-V2`,`FLY-C8`と`FLY-PRO-X10`または`FLY_LITE2/2.1`です

:::
    * HDMIを使用するには下記のオプションを有効にする必要があります
    * FLY Toolsに入った後
* 第一歩はfly_TOOLSに入ってパラメータを変更することです
