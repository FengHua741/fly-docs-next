# 画面パラメータ変更ガイド
:::tip
ブラウザであなたのIP:9999を開いてFLY-Toolsにアクセスしてください
:::

:::tip
FLY 上位機に搭載されているHDMIインターフェースはMicro HDMIです、HDMIケーブルを購入する際に注意してください！！！
:::

:::tip
FLYの上位機でFPC一線通をサポートしているものは`FLY-PI-V2`,`FLY-C8`と`FLY-PRO-X10`または`FLY_LITE2/2.1`です。

:::
    * HDMIを使用するには下のオプションを開いて画面を有効にする必要があります
    * FLY Toolsに入った後
* 第一段階ではfly_TOOLSに入ってパラメータを変更する必要があります
