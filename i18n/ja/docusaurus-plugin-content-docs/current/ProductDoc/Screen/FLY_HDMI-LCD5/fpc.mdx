# スクリーンパラメータ変更チュートリアル
:::tip
ブラウザでIP:9999を開いてFLY-Toolsにアクセスしてください。
:::

:::tip
FLYのオンボードHDMIインターフェースはマイクロHDMIです。HDMIケーブルを購入する際にはご注意ください！！！
:::

:::tip
FLYのオンボードは`FLY-PI-V2`,`FLY-C8`と`FLY-PRO-X10`または`FLY_LITE2/2.1`でFPCラインを使用できます。

:::
    * HDMIを使用するには、下記のオプションを開く必要があります。
    * FLY Toolsに入ったら
* まず最初にfly_TOOLSでパラメータを変更します
