# 画面パラメータの変更ガイド
:::tip
ブラウザであなたのIP:9999を開いてFLY-Toolsにアクセスしてください
:::

:::tip
FLY上位機に搭載されているHDMIポートはMicro HDMIです。HDMIケーブルを購入する際はご注意ください！！！
:::

:::tip
FLYの上位機はFPC一線通をサポートしているモデルが`FLY-PI-V2`,`FLY-C8`および`FLY-PRO-X10`または`FLY_LITE2/2.1`です。

:::
    * HDMIを使用するには、下記のオプションを有効にして画面を起動してください
    * FLY Toolsに入った後
* 第一歩としてfly_TOOLSに入ってパラメータを変更する必要があります
