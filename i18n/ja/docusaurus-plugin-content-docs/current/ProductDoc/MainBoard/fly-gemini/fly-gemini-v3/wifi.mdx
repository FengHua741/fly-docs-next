# Wi-Fiに接続する


## Geminiにアンテナを取り付ける

    * 赤い矢印が指している場所がWi-Fiアンテナインターフェースです
    * Wi-Fiアンテナポートは`ipex1`です
    * アンテナをGeminiに取り付けない場合、信号が弱いためネットワークに接続できない可能性があります
