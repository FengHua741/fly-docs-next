## システム焼き込みの準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上128GB以下で、伝送速度はclass10以上である必要があります
    * 対応する<Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">システムイメージ</Button>をダウンロード
    * 必要な<Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼き込みツール</Button>をダウンロード

## システム焼き込み

:::info
TFカードへの焼き込みとTFカードを介してM2WEへの焼き込みは、そのうちの一つだけを使用すれば良い
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードを使用してシステムを焼き込むことをお勧めします。USB経由でM2Wに焼き込むと、ドライバーの互換性などの理由で失敗する可能性があります。[メモリーカードからM2WEに焼き込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を直接使用してください
* ダウンロードしたシステムイメージが対応しているかどうかを確認してください。対応していない場合、起動できません
:::
