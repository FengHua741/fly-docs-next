## システムの焼き込み前準備

    * カードリーダー
    * TFカード1枚、容量は16GB以上128GB以下で、TFカードの転送速度はclass10以上である必要があります
    * 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼き込みツール</Button> をダウンロード

## システムの焼き込み

:::info
TFカードへの焼き込みとTFカードを介してM2WEへの焼き込みは、そのうちの一つだけを選択して使用してください
:::

:::info
* 機械にM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードを使用してシステムをフラッシュすることをお勧めします。USB経由でM2Wに焼き込みを行うと、ドライバーの互換性等の理由で失敗する可能性があるため、[メモリーカードを介してM2WEに焼き込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)を直接使用することをお勧めします
* ダウンロードしたシステムイメージが対応しているかどうかを確認してください。対応していない場合は起動できません
:::
