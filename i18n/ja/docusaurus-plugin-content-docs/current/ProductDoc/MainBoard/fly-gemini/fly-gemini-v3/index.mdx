# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.webp').default} size="80%" align="center" />

## 製品特長
    * ボードマウントのホストコンピューターを搭載し、ラズベリーパイを置き換えてホストとして動作させることができます。1GBのオンボードDDR3メモリーを搭載しています。
    * 強化された5V電源保護により、チップの損傷を防ぎます。
    * MCUチップのフラッシュメモリ設定を取りやめ、ファームウェアの書き込みが容易になります。
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyとFLY-Toolsがプリインストールされており、複雑な操作は不要で、初心者でも使いやすくなっています（TFカードまたはM2WEの購入が必要です）。

## 製品仕様

    * CPU：高性能の全志H5チップ、4コア64ビットCortex-A53
    * GPU：高性能6コアMali 450、ピクセルフィルレートは2.7gピクセル/秒以上
    * RAM：1GB DDR3（GPUと共有可能）
    * ROM：最大128GB SDカード対応
    * 外部インターフェース：Spi x2、UART x2、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN x1
    * インターフェース：オンボードM.2インターフェース、Wi-FiやeMMCなどの拡張が可能です。（専用プロトコルを使用しているため、非FLY π専用のM.2デバイスへの接続は避けてください）
    * 12-24Vの直流電源に対応しており、電源供給が安定し、降圧モジュールを省略できます。
    * 5V制御可能なファンポートが予備され、ホストコンピューターの冷却に使用できます。
    * CAN拡張インターフェースが予備されています。
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyとFLY-Toolsがプリインストールされており、複雑な操作は不要で、初心者でも使いやすくなっています（TFカードまたはM2WEの購入が必要です）
    * ドライバー: 高圧ドライバー*4
    * MCU: STM32F405
