## システム焼込前の準備

- リーダー
- TFカード1枚、容量は16GB以上または128GB以下のもので、転送速度はClass10以上である必要があります
- 対応するシステムイメージをダウンロード：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>
- 必要な焼込ツールをダウンロード：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button>

## システム焼込

:::info
TFカード焼込とTFカードからM2WEに焼込む場合、どちらか一つを使用してください
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードでシステムを焼込むことを推奨します。USB経由での焼込みはドライバ互換性などの理由で失敗することがありますので、直接[メモリーカードをM2WEに焼込む](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)ことをお勧めします
* 使用しているシステムイメージが正しいかどうかを確認してください。正しくない場合は起動できません
:::

## アンテナの設置

1. `M2WE`または`2.4G WiFi`モジュールのアンテナを設置します
:::warning
* FLY上位機器では`M2WE`または`2.4G WiFi`モジュールを使用する必要があります。それがないとWiFiを使用できません
* 注意：上位機器では中国語名前のWiFiはサポートされていません。2.4Gと5Gの同時接続はサポートされていません
:::
* 赤い矢印が示す部分がWiFiアンテナインターフェースです
* WiFiアンテナインターフェースは`ipex1`です
* 天線をM2WEに取り付けないと、信号が弱いためルーターに接続できない可能性があります
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

:::tip
- システムカード設置が完了したら、次にSSHで上位機器に接続できます
:::

2. 上位機器に電源を供給します。  
**ここまできたら、あなたの上位機器は起動可能です！！！**  
次のステップに進むには：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">SSHで上位機器に接続</Button>
