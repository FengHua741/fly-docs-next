## システム焼込前の準備

    * SDカードリーダー
    * 容量が16GB以上または128GB以下のTFカード1枚、転送速度はClass10以上である必要がある
    * 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button> をダウンロード

## システム焼込

:::info
TFカード焼込とTFカード経由でM2WEに焼込む場合、どちらか一つを使用すればよいです
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムは自動的にメモリーカードから起動します
* M2WEはTFカードを使用してシステムを焼込むことを推奨します。USB経由での焼込みはドライバ互換性の問題などで失敗する可能性があります。直接 <Button variant="contained" disableElevation component="a" rel="noopener" href="https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os">メモリーカードからM2WEに焼込む</Button> を推奨します
* 使用するシステムイメージが正しいかどうか注意してください。正しくない場合は起動できません
:::

## アンテナの取り付け

    1. `M2WE`または`2.4G Wi-Fi`モジュールのアンテナを取り付けます
    :::warning
    * FLY上位機器では`M2WE`または`2.4G Wi-Fi`モジュールを使用する必要があります。Wi-Fiを使用するにはそれらが必要です
    * 注意：上位機器は中国語名前のWi-Fiには対応していません。また、2.4Gと5GのシングルバンドWi-Fiには対応していません
    :::
    * 赤い矢印が示す場所がWi-Fiアンテナの接続口です
    * Wi-Fiアンテナポートは`ipex1`です
    * M2WEにアンテナを接続しない場合、信号が弱いためルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードの取り付けが完了したら、次の手順に進み、SSHで上位機器に接続できます
    :::

    2. 上位機器に電源供給を行います。  
    **ここまできたら、あなたの上位機器は起動できるようになりました！！！**  
    次の手順に進むには：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">上位機器にSSH接続</Button>
