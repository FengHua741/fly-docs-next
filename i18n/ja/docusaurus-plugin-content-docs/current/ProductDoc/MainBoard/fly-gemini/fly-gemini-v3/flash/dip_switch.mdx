# 拨码介绍

:::tip
FLY Geminiはダイアルスイッチを使用してホストコンピューターに接続します。ダイアルの位置が正しくない場合、正常に使用できません。
:::

:::danger 注意
ダイアルスイッチは指定された位置に設定する必要があります。それ以外の場合、ホストコンピューターに正常に接続できません。
:::

## ホストコンピューターへの接続

    * ダイアルスイッチ`1`,`2`を右側に倒すと、下位機器を内蔵USBでホストコンピューターに接続できます
    * ダイアルスイッチ`3`,`4`を左側に倒すと、それ以外の場合、下位機器とホストコンピューターの接続に影響します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * この方法でダイアルスイッチを動かした後、下部のUSBポートは下位機器によって占有されますので、他のデバイスを接続しないでください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## 外付けホストコンピューターへの接続

    * ダイアルスイッチ`3`,`4`を右側に倒すと、下位機器を内蔵USBで外付けホストコンピューターに接続できます
    * ダイアルスイッチ`1`,`2`を左側に倒すと、それ以外の場合、下位機器と外付けホストコンピューターの接続に影響します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
