# ディップスイッチの紹介

:::tip
FLY Geminiはディップスイッチを介してホストマシンに接続します。ディップスイッチの位置が正しくない場合、正常に使用できません。
:::

:::danger 注意
ディップスイッチを指定された位置に切り替えないと、ホストマシンに正常に接続できません。
:::

## オンボードホストマシンへの接続

    * ディップスイッチ`1`,`2`を右に切り替え、下位機とホストマシンを内蔵USBで接続します
    * ディップスイッチ`3`,`4`を左に切り替え、そうしないと下位機とホストマシンの接続に影響します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * この方法でディップスイッチを切り替えた後、下部のUSBポートは下位機によって占有されるため、他のデバイスを接続しないでください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## 外部ホストマシンへの接続

    * ディップスイッチ`3`,`4`を右に切り替え、下位機と外部ホストマシンを内蔵USBで接続します
    * ディップスイッチ`1`,`2`を左に切り替え、そうしないと下位機と外部ホストマシンの接続に影響します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
