# CANブリッジ

CANバスは、リアルタイムアプリケーションに使用されるシリアル通信プロトコルバスで、シールド線を使用して信号を伝送できます。これは世界中で最も広く使用されているフィールドバスの1つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価で嵩張るワイヤーハーネスの代わりに使用されます。その堅牢性により、他の自動化および産業アプリケーションにも使用されています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、およびエラーチェック機能が含まれます。
3Dプリンターでは、CANバスを使用して、プリントヘッドがマザーボードに接続するための配線数を減らします。もともと数十本の線が必要でしたが、CANを使用することで、必要な線は4本までに減少し、大幅に配線の数を減らし、配線の難易度を下げることができます。
この章では、CANのブリッジの使用について簡単に概説します。

## ツールボードはCANでマザーボードに接続

* ツールボードの電源線はCANポートにつなぐのではなく、12V-24Vの電源ラインにつなぐことに注意してください
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/canbridge.webp').default} size="80%" align="center" />
* 120Ωのジャンパーピンを接続することに注意してください
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/120.webp').default} size="80%" align="center" />

## マザーボードのファームウェアをアップデート

* Geminiマザーボードに焼込むファームウェアは`USBブリッジCANファームウェア設定`であることを確認してください
