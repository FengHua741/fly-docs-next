# CDY V3_V2

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-cdy/fly-cdy-v3/img/cdy.webp').default} size="80%" align="center" />

## 产品特点

    * 32位ARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * ドライバ電圧選択がサポートされ、デフォルトは5Vです
    * RRFファームウェアがサポートされます

## 产品参数

    * 32位ARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * CANバス接続がサポートされ、データ伝送がより安定し、遅延が小さくなり、接続がより安定します
    * ファームウェア：Reprap/Klipper
    * ドライバサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライバモードサポート：TMC：UART/SPI
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * Reprapファームウェアを使用すると、拡張ESP32 Wi-Fiモジュールがサポートされ、最大1.6m/sの速度をサポートします
    * 独自のPWMファンMOSボード設計、故障した場合に直接交換でき、顧客の修理の難易度を低減します
    * PCBは2OZ銅厚を使用しており、より大きな電流をサポートできます
    * 3つのADCインターフェースがあり、最大3つのヒートセンサーをサポートします
    * 6つのIOインターフェースがあり、レベル調整とリミットスイッチに使用できます
    * 4つのヒーターコネクタがあり、最大4つのヒーターをサポートします
