# 固件編譯と焼込

    :::tip

    * CDVはデフォルトでTFカードを使用してファームウェアを焼込ます
    * 容量が`32G`以下のTFカードを準備してください
    * ブランド外のメモリーカードは焼込に失敗する可能性があるため、推奨されるメモリーカードを使用してファームウェアの更新を行ってください
    * TFカードの転送速度は`class10`級またはそれ以上であることが望ましく、サンディスクなどのブランドのTFカードをお勧めします

    :::

## Klipperファームウェアのコンパイル

    <MakeOperate/>

    <USB />

    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻っています
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
