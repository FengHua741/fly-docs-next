# ファームウェアのコンパイルと書き込み

    :::tip

    * CDVはデフォルトでTFカードを使用してファームウェアを書き込んでいます
    * 容量が`32G`以下のTFカードをご用意ください
    * 互換性のないメモリーカードは書き込みに失敗する可能性がありますので、できるだけ推奨されるメモリーカードを使用してください
    * TFカードの転送速度は`class10`級またはそれ以上であることが望ましく、サンディスクなどのブランドのTFカードをお勧めします

    :::

## Klipperファームウェアのコンパイル

    <MakeOperate/>

    <USB />

    * `Q`キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 再び `Q` キーを押して **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * これで設定が保存され、コマンドラインインターフェースに戻ります
    * コンパイルを開始するために次のコマンドを入力します。時間がかかります
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下のような出力が出ればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

        
    * コンパイル成功の例がこちらです

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make-done.webp').default} size="50%" align="left" />
