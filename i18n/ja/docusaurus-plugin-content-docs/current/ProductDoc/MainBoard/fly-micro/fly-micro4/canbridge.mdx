# CANブリッジ

CANバスは、リアルタイムアプリケーションのためのシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送でき、世界で最も広く利用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価でかさばるワイヤーハーネスの代わりにします。その堅牢性により、他の自動化および産業応用にも使用されるようになりました。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、エラー検出機能が含まれます。
3Dプリンターでは、CANバスを使用して、プリントヘッドをメインボードに接続するための配線の数を減らします。元々は数十本の線が必要でしたが、CANを使用することで、わずか4本の線で済むようになり、配線の数を大幅に削減し、配線の難易度も下げました。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードはCANを使ってメインボードと接続します

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24V電源ラインに接続することに注意してください
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/canbridge.webp').default} size="80%" align="center" />
    * 120Ωジャンパーピンを接続することに注意してください
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/120.webp').default} size="80%" align="center" />

## メインボードのファームウェアをフラッシュします

* `USB-CANブリッジファームウェア設定`のファームウェアを使用してD5メインボードを焼くことを確認してください
