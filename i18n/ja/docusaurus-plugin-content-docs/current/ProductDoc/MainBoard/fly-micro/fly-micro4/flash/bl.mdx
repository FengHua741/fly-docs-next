# Katapultファームウェア

    :::tip

    * ご注意：出荷時すでに`katapult`がプリインストールされているため、再フラッシュする必要はありません。

    :::

## フラッシュモードに入る

    :::tip

    図で囲まれているのが`BOOTボタン`です。

    :::

    * フラッシュモードに入るには、マザーボードを完全に電源オフにしてから、BOOTボタンを押しながら、データ転送用のケーブルをマザーボードとデバイスに接続します。
