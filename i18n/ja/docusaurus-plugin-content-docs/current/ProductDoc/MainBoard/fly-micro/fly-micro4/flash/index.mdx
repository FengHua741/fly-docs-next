# 固件焼込み説明

:::danger 注意点
* メインボードのファームウェア焼込みにはKatapultを介してKlipperを焼込みます
* メインボードにKatapultが既に焼込まれていることが確定している場合、最後のKatapultファームウェアの焼込みは必要ありません。これはメインボードのKatapultファームウェアの焼込みに失敗した場合の予防策です
* リセットボタンを素早く2回押して、メインボード上のLEDが点滅するかどうか確認できます。もしこのLEDが点滅している場合は、メインボードがKatapultファームウェアに入っていることを示します
:::

## どの種類のファームウェアが必要かを見分ける方法

    :::tip

    * 注意：あなたが焼込む必要があるのは1つのファームウェアのみです

    :::
    
    * E3 V2は以下の方法でホストコンピュータに接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼性があり、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスに接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションオーバータイムなどのエラーが発生する可能性があります。
