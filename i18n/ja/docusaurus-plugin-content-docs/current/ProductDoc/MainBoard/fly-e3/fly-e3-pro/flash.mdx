# ファームウェアのコンパイルと書き込み

    :::tip

    * E3 ProはデフォルトでTFカードを使用してファームウェアを焼付けます
    * 容量が`32G`以下のTFカードを用意してください
    * メモリーカードが正常に動作しない場合がありますので、推奨されるメモリーカードを使用してファームウェアを更新することをお勧めします
    * TFカードの転送速度は`class10`級またはそれ以上であることが望ましく、SanDiskなどのブランドのTFカードを使用することをお勧めします

    :::

## Klipperファームウェアのコンパイル

    <MakeOperate/>

    <USB />

    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存されコマンドラインインターフェースに戻ったはずです
    * コンパイルを開始するには次のコマンドを入力してください。時間がかかります
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が表示されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
        
    * コンパイル成功の例を以下に示します

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make-done.webp').default} size="50%" align="left" />
