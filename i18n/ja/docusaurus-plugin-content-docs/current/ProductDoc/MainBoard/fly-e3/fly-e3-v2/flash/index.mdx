    :::tip

    * E3 V2はデフォルトでTFカードを使用してファームウェアを焼込みます
    * 容量が`32G`以下のTFカードを用意してください
    * 互換品のメモリーカードは焼込みに失敗する可能性がありますので、推奨されるメモリーカードを使用することをお勧めします
    * TFカードの転送速度は`class10`またはそれ以上であることが望ましいです。サンディスクなどのブランドのTFカードを使用することをお勧めします

    :::

# 固件焼込み説明


## どの種類のファームウェアが必要かを見分ける方法

    :::tip

    * あなたが必要とするファームウェアは一つだけです

    :::
    
    * E3 V2は以下の方法でホストコンピュータに接続できます
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼でき、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ伝送に影響を与える可能性があります
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスに接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジショニングタイムアウトなどのエラーが発生する可能性があります。
