# CANブリッジ

CANバスは、リアルタイムアプリケーションに使用されるシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送し、世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価で嵩張る配線ハーネスの代わりとして機能します。このプロトコルの堅牢性により、他の自動化および産業応用へと用途が拡大しています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、およびエラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用して、プリントヘッドをメインボードに接続するための配線数を減らします。元々は数十本の線が必要でしたが、CANを使用することで、必要な線は4本に減少し、配線の数を大幅に減らし、配線の難易度も下げることができます。
本章では、CANのブリッジを使用することに関する簡単な概要のみを説明します。

## ツールボードはCANでメインボードに接続します

* ツールボードの電源ラインはCANポートに接続されていないことに注意してください。12V-24V電源ラインに接続されています
