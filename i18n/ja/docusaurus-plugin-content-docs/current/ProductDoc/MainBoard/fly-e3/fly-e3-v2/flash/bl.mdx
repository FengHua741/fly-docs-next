# BLファームウェア

    :::tip

    * ご注意ください、出荷時に`bootloader`が事前に焼かれていますので、再び焼く必要はありません。

    :::

## フラッシュモードに入る

    :::tip

    `BOOT`ボタンは右側にあり、`RESET`ボタンは左側にあります。

    :::

    * フラッシュモードに入るには2つの方法があります
    * 方法1：マザーボードを完全に電源オフにした状態で、`BOOT`ボタンを押しながらデータ転送用のケーブルをマザーボードとあなたのデバイスに接続します
    * 方法2：データ転送用のケーブルをマザーボードとあなたのデバイスに接続した後、`BOOT`ボタンを押し続けながら`RST`ボタンを押してから`RST`ボタンを離し、最後に`BOOT`ボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-e3/fly-e3-v2/flash/img/boot.webp').default} size="100%" align="center" />
