# E3V2

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-e3/fly-e3-v2/img/e3v2.webp').default} size="80%" align="center" />

## 产品特点

    * 32位ARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * CANブリッジをサポート
    * RRFファームウェアをサポート

## 产品参数

    * 32位ARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * CANバス接続をサポート、データ転送がより安定し、遅延が小さく、接続が安定しています
    * ファームウェア：Reprap/Klipper
    * ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライバーモードサポート：TMC：UART/SPI
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 1つの高圧INポート、自動ベッドセンサーをサポート：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
    * Reprapファームウェアを使用すると、拡張ESP32 WiFiモジュールをサポートし、1.6m/sをサポートします
    * 独自のPWMファンMOSボード設計、故障した場合に直接交換でき、顧客の修理の難易度を下げ、4つの2ピンファンインターフェースがあります
    * PCBは2OZ銅厚を使用しており、より大きな電流をサポートできます
    * 2つのADCインターフェース、最大で2つの熱センサーをサポート
    * 4つのIOインターフェース、レベル調整とリミットスイッチに使用できます
    * 3つのヒーターコネクタ、最大で3つのヒーターをサポート
