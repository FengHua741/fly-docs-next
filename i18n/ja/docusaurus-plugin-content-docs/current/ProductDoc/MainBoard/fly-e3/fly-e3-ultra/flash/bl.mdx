# BLファームウェア

    :::tip

    * デバイス出荷時にブートローダーが事前に書き込まれているため、再び書き込む必要はありません。

    :::

## フラッシュモードに入る

    :::tip

    右側が「BOOTボタン」、左側が「RESETボタン」です。

    :::

    * フラッシュモードに入るには2つの方法があります。
    * 方法1：マザーボードを完全に電源オフにして、「BOOTボタン」を押しながら、データ転送用のケーブルをマザーボードとデバイスの一方に接続します。
    * 方法2：データ転送用のケーブルをマザーボードとデバイスに接続した後、「BOOTボタン」を押しながら「RSTボタン」を押し、「RSTボタン」を離し、最後に「BOOTボタン」を離します。
