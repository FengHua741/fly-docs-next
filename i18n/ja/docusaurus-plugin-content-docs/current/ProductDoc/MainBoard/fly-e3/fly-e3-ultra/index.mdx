## 商品仕様

* 32ビットARM Cortex-M4シリーズ550 MHz、STM32H723VGT6チップ
* ファームウェア：Reprap/Klipper
* オンボードで5つのTMC2209ドライバを搭載、XYZの3つのドライバ位置は無限位モードをサポート
* 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
* 1つのハイプレッシャーINポート、自動フラットベッドセンサーサポート：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
* Repapファームウェアを使用すると、ESP32 WiFiモジュールが使用でき、1.6m/sをサポート
* 独自のPWMファンMOSボード設計、故障時は直接交換可能で、顧客の修理の難易度を低減、4つの2ピンファンインターフェース
* 3つのADCインターフェースは温度センサーに使用可能
3つのリミットポートにはリミットスイッチを接続できます
* 4つのIOインターフェース、近接スイッチ、リミット、レーザーヘッド、RGBライトなどに使用可能
* 3つのヒーターアンプインターフェース
* 1つのシリアルスクリーンインターフェースはRRFをサポート
* Maxバージョンでは1つのKタイプの熱電対と1つのPT100温度センサーをサポート
* 新しくダイアルスイッチを追加し、PT100温度センサーの2/3/4線モードを選択可能
* 新しい一体型ヒートシンクを追加し、ドライバのヒートシンク面積を増大
* CANインターフェースを新規追加
