## 製品仕様

    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * CANバス接続をサポートし、データ伝送がより安定し、遅延が小さくなり、接続がより安定します
    * ファームウェア：Reprap/Klipper
    * ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライバーモードサポート：TMC：UART/SPI
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * Reprapファームウェアを使用すると、拡張ESP32 Wi-Fiモジュールを使用でき、1.6m/sをサポートします
    * 独自のPWMファンMOSボード設計、損傷した場合に直接交換でき、顧客の修理の難易度を低減します
    * PCBは2OZ銅厚を使用しており、より大きな電流をサポートできます
    * 6つのADCインターフェース、最大で6つの熱素子をサポート
    * 7つのIOインターフェース、レベル合わせとリミッターに使用できます
    * 7つのヒーターコネクタ、最大で7つのヒーターをサポート
