# ファームウェアのコンパイルと書き込み

    :::tip

    * 407ZGはデフォルトでTFカードを使用してファームウェアを書き込む
    * 容量が`32G`以下のTFカードを準備してください
    * 不明ブランドのメモリーカードは書き込みに失敗する可能性がありますので、推奨されるメモリーカードを使用してファームウェアを更新してください
    * TFカードの転送速度は`class10`級またはそれ以上が望ましく、サンディスクなどのブランドのTFカードをお勧めします

    :::

## Klipperファームウェアのコンパイル

    <MakeOperate/>

    <USB />

    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存されコマンドラインに戻ったはずです
    * コンパイルを開始するために以下のコマンドを入力します。時間はかかります
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
        
    * コンパイル成功の例が以下です

        <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make-done.webp').default} size="50%" align="left" />
