# 製品仕様

* 32 ビット 550 MHz の ARM Cortex-M7 シリーズ STM32H723ZGT6 マスターチップを採用
* ファームウェア：Marlin 2.0/Reprap/Klipper
* ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
* ドライバーモードサポート：TMC：UART/SPI
* 5つの高電圧ドライバーピン、5つの独立したモータードライバーサポート
* 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
* 1つの高電圧入力ピン、自動ベッドレベルセンサー対応：BLTouch、PL08Nなど近接センサー、Klicky、Voron Tapなど
* オプションのリミットスイッチ電源：5V/12V/24V
* Reprapファームウェアを使用すると、搭載されたESP32 WiFiモジュールを使用でき、1.6m/sの速度をサポート
* 独自のPWMファンMOSボード設計、故障時は直接交換可能で、顧客の修理の難易度を下げ、2つの2ピンファンインターフェースは4ピンPWMファンもサポート
* PCBは2OZ銅厚を使用し、より大きな電流をサポートできる
* 3つのADCインターフェース、最大3つのサーモカップルに対応
* 7つのIOインターフェース、レベル調整とリミットスイッチに使用可能
* 3つのヒータージャック、最大3つのヒーターバーに対応
