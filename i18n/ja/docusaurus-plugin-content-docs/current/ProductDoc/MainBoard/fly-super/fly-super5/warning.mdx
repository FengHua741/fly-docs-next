# メインボードの注意事項

## 電源インジケーター

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/power.webp').default} size="80%" align="center" />
* TYPE-Cで電源を供給し、他のメインボードの電源に接続していない場合、少なくとも2つのライト `3.3V` と `5V` が点灯します
* メインボード電源が接続された後、電源インジケーターは `3.3V`、`5V`、`12V` のすべてのライトを点灯します
* ドライバーパワーコネクタ0-2、ドライバーパワーコネクタ3-7、ヒートベッド電源が接続された後、メインボードの左下の電源インジケーターが正常に点灯します

### 問題のトラブルシューティング

    :::danger 注意点

* 何も外部機器が接続されていない状態（ドライバ、リミットスイッチ、12864ディスプレイなどがない場合）
* TYPE-Cで電源を供給した場合、少なくとも `3.3V` と `5V` の2つのLEDが常時点灯します

    :::

* `3.3V` と `5V` が点灯しない場合は、メインボードの5Vジャンパーを確認してください
* ジャンパーを交換しても同じであれば、TYPE-Cの正常な電源供給を確認し、カスタマーサポートに連絡してください
* `3.3V` または `5V` のいずれかのライトが点滅する場合、チップがショートしていることを示します。カスタマーサポートに連絡してください
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/5v.webp').default} size="60%" align="center" />

## ドライバ

* ドライバの電源供給は48Vを使用できますが、すべてのドライバが48Vを使用する必要があります
* メインボードは外部ドライバをサポートしています     <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定チュートリアル</Button>

## リミットスイッチポート

* リミットスイッチポートが使用できない場合、リミットスイッチポートを交換するか、ドライバのリミットスイッチフットをカットすることができます
    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/2209diag.webp').default} size="80%" align="center" />
