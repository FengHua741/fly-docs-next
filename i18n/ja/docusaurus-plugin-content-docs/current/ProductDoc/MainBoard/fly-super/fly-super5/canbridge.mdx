# CANブリッジ

CANバスは、リアルタイムアプリケーション用のシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送し、世界で最も広く使用されているフィールドバスの1つです。CANプロトコルは、車両内のさまざまな要素間の通信に使用され、高価で嵩張る配線ハーネスの代わりにします。その堅牢性により、他の自動化や産業アプリケーションにも使用されるようになりました。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、およびエラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用してプリントヘッドをマザーボードに接続するための配線の数を減らします。もともと数十本の配線が必要でしたが、CANを使用することで、必要な配線は4本だけになりました。これにより、配線の数が大幅に削減され、配線の難易度も下がります。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードはCANでマザーボードに接続します

* ツールボードの電源線はCANポートに接続するのではなく、12V-24V電源ラインに接続することに注意してください

## マザーボードのファームウェアを更新します

* SUPER5 PROマザーボードを焼成する際には、ファームウェアが`USBブリッジCANファームウェア設定`であることを確認してください
