# CANブリッジ

CANバスはリアルタイムアプリケーションのためのシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができます。それは世界中で最も広く使われているフィールドバスの一つです。CANプロトコルは自動車内で異なるコンポーネント間の通信に使用され、高価でかさばる配電ハーネスに取って代わります。このプロトコルの堅牢性により、その用途は他の自動化および産業アプリケーションにも広がっています。CANプロトコルの特性には、完全性のあるシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの転送速度、11ビットのアドレッシングおよびエラー検出能力が含まれます。
3Dプリンターでは、CANバスを使用してプリントヘッドとメインボードの間の配線数を減らします。元々必要だった十数本の線が、CANバスを使用することで4本の線だけで済むようになり、配線数を大幅に減らし、配線の難易度を下げます。
本章ではCANのブリッジ使用について簡単に概要を説明します。

## ツールボードでメインボードにCAN接続する

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24vの電源線に接続してください。
