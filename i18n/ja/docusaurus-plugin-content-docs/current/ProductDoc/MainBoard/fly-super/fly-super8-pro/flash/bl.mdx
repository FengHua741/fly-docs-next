# BLファームウェア

    :::tip

    * 注意: 出荷時に`bootloader`が事前に書き込まれているため、再度書き込む必要はありません。

    :::
## フラッシュモードへの移行


    * フラッシュモードへの移行には2つの方法があります。
    * 方法1: マザーボードを完全に電源オフにして、BOOTをショートし、データ転送用のケーブルをマザーボードとデバイスの間に接続します。
    * 方法2: データ転送用のケーブルをマザーボードとデバイスの間に接続した後、BOOTをショートし、RSTボタンを押してから離します。最後にBOOTショートを解除します。
