# 產品參數

    * 采用 32 位主頻 550 MHz 的 ARM Cortex-M7 系列 STM32H723ZGT6 主控晶片
    * 固件：Marlin 2.0/Reprap/Klipper
    * 驅動支援：A4988、LV8729、DRV8225、TMC2208/2209/5160等
    * 驅動模式支援：TMC：UART/SPI
    * 8個高壓驅動位、支援8個獨立電機驅動、5個擠出機、10個PWM風扇
    * 顯示：串行觸摸屏、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 1個高壓IN口，支援自動平床傳感器：BLTouch、PL08N等接近傳感器、Klicky、Voron Tap等
    * 可選限位開關電源：5V/12V/24V
    * 使用repap固件，可以使用板載ESP32 WiFi模組，支援1.6m/s
    * 獨特的PWM Fan MOS板設計，損壞可直接更換，降低客戶維修難度，兩個2pin風扇介面可支援4pin PWM風扇
    * PCB採用2OZ銅厚，能夠支援更大的電流
    * 6個ADC介面，最多支援六路熱敏
    * 8個IO介面，可以用於調平和限位
    * 5個加熱棒介面，最多支援五路加熱棒
    * 2個PMW介面
