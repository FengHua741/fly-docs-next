    :::tip

    * Super8 PROはデフォルトでTFカードを使用してファームウェアを焼込みます
    * 容量が`32G`以下のTFカードを用意してください
    * ブランド外のメモリーカードは焼込みに失敗する可能性がありますので、推奨されるメモリーカードを使用することをお勧めします
    * TFカードの転送速度は`class10`級またはそれ以上であることが望ましく、SanDiskなどのブランドのTFカードを使用することをお勧めします

    :::

# 固件焼込み説明

    :::tip

    * Super8 Proは出荷時にTFカードブートローダーがプリフラッシュされていますので、自分自身で再焼込みする必要はありません

    :::

## どの種類のファームウェアが必要かを判断する方法

    :::tip

    * 注意：あなたが焼込む必要があるのは一つのファームウェアだけです

    :::
    
    * Super8 PROは以下の方法でホストコンピュータに接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼性があり、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ伝送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスを接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用すると、ホーム位置決定のタイムアウトエラーなどが発生する可能性があります。
