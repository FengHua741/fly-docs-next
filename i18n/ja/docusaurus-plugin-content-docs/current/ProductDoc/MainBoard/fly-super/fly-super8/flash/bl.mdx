# BL固件

    :::tip

    * 出廠時にブートローダーが事前にフラッシュされていることに注意してください。再度フラッシュする必要はありません。

    :::

## フラッシュモードに入る


    * フラッシュモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源を切られている状態で、BOOTピンをショートし、データ転送用のケーブルをマザーボードとデバイスの間に接続します
    * 方法2：データ転送用のケーブルをマザーボードとデバイスの間に接続した後、BOOTピンをショートし、RSTボタンを押してからRSTボタンを離し、最後にBOOTピンのショートを解除します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/flash/img/boot.webp').default} size="100%" align="center" />
