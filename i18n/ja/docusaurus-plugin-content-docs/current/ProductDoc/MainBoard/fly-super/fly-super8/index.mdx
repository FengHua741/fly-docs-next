# 製品パラメータ


    * 32ビットARM Cortex-M4シリーズ168 MHz、STM32F407ZGT6チップ
    * ファームウェア：Marlin 2.0/Reprap/Klipper
    * ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ドライバーモードサポート：TMC：UART/SPI
    * 8つの高圧ドライバーポート、8つの独立したモーター駆動、5つの押出機、10つのPWMファン
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * 1つの高電圧入力ポート、自動ベッドレベルセンサーがサポートされています：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
    * オプションのリミットスイッチ電源：5V/12V/24V
    * Repapファームウェアを使用すると、オンボードESP32 Wi-Fiモジュールが使用でき、最大1.6m/sをサポートします
    * 独自のPWMファンMOSボード設計、故障時は直接交換可能で、顧客の修理の難易度を下げます。2つの2ピンファンインターフェースは4ピンPWMファンをサポートできます
    * PCBは2OZ銅厚を使用しており、より大きな電流をサポートできます
    * 6つのADCインターフェース、最大6つのサーモカップルをサポート
    * 8つのIOインターフェース、レベル調整とリミットスイッチに使用できます
    * 5つのヒータージャック、最大5つのヒータージャックをサポート
    * 2つのPMWインターフェース
