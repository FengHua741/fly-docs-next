## メインボードの配線

:::warning 注意事項
画像の解像度が高いため、ウェブページで閲覧すると鮮明に見えません。マウスカーソルを接線図の上に置いて右クリックし、「画像を別の名前で保存」を選択してデスクトップに保存してから確認してください！
:::

:::warning 警告
購入した完成品の「3線式リミットスイッチ」は使用しないでください。この種のリミットスイッチは、**メインボードがショート**し、**メインボードを破損**する可能性があります！！！接線図に示された接線方法に従って接線してください！！！
:::
