# CANブリッジ

CANバスは、リアルタイムアプリケーションのためのシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送し、世界で最も広く使用されているフィールドバスの1つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価で嵩張る配線ハーネスの代わりに使用されます。このプロトコルの堅牢性により、他の自動化および産業アプリケーションにも拡張されています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、およびエラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用して、プリントヘッドがメインボードに接続されるワイヤーの数を減らします。もともと十数本のワイヤーが必要でしたが、CANを使用することで、必要なワイヤー数が4本に減少し、ワイヤーの数を大幅に減らし、配線の難易度を低減します。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードはCANでメインボードに接続する

    * ツールボードの電源線はCANポートに接続されるものではなく、12V-24V電源ラインに接続されることに注意してください
