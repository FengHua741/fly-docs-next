# CANブリッジ

CANバスはリアルタイムアプリケーション用のシリアル通信プロトコルバスであり、ツイストペアケーブルを使用して信号を伝送することができ、世界で最も広く使用されているフィールドバスの一つです。CANプロトコルは、自動車内で異なるコンポーネント間での通信に使用され、高価でかさばる配電ハーネスを置き換えるために使われます。このプロトコルの堅牢性により、その用途は他の自動化および産業用アプリケーションにまで広がっています。CANプロトコルの特性には、シリアルデータ通信の完全性、リアルタイムサポートの提供、最大1Mb/sの伝送速度、11ビットのアドレッシングおよびエラー検出能力が含まれます。
3Dプリンターでは、プリントヘッドからマザーボードへの配線を減らすためにCANバスが使用されます。元々は十数本の線が必要でしたが、CANを使用すると4本の線だけで済み、配線の数を大幅に減らし、配線の難易度を下げます。
本章ではCANブリッジの使用について簡単に概説します。

## ツールボードがマザーボードにCAN接続を使用

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24Vの電源線に接続してください
