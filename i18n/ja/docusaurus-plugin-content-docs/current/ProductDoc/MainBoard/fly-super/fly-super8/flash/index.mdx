    :::tip

    * Super8はデフォルトでTFカードを使用してファームウェアを焼込みます
    * 容量が32G未満のTFカードを用意してください
    * ブランド外のメモリーカードは焼込みに失敗する可能性がありますので、推奨されるメモリーカードを使用することをお勧めします
    * TFカードの転送速度はclass10級またはそれ以上であることが望ましく、SanDiskなどのブランドのTFカードを使用することをお勧めします

    :::

# 固件焼込説明


## どの種類のファームウェアが必要かを見分ける方法

    :::tip

    * 注意してください、あなたは一つのファームウェアだけを焼込む必要があります

    :::
    
    * Super8は以下の方法でホストコンピュータと接続できます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼でき、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスに接続でき、USBポートを占有しません
        * 欠点：KlipperScreenと一緒に使用すると、ホーム位置決めがタイムアウトするなどのエラーが発生する可能性があります。
