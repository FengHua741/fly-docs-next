## システム焼成の準備

* SDカードリーダー
* 容量が16GB以上または128GB以下のTFカード、転送速度はClass10以上のもの
* 対応する <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
* 必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">焼成ツール</Button> をダウンロード

## システム焼成

:::info
TFカード焼成とTFカードからM2WEへの焼成はどちらか一方を使用すれば十分です
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEへの焼成はTFカードを使用することをお勧めします。USBからの焼成はドライバーの互換性などの理由で失敗することがありますので、直接 [メモリーカードからM2WEへの焼成](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os) を使用することをおすすめします
* 間違ったシステムイメージをダウンロードしないように注意してください。そうでなければ起動できません
:::
