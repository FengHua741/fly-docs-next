# 上位機SSH接続


* シリアルポートまたはWIFIを介してデバイスに接続することができます
* 事前に**MobaXterm_Personal**または他の**SSHターミナルツール**を準備してください
* SDカードまたはM2WEに**対応する上位機のシステム**が焼入されていることを確認してください
* **もし上位機が正常に起動しない場合はドライバー、限界、ファンなどの周辺機器を取り外してください！！**
* **最後にネットワークを通じてSSHに接続し、ファームウェアのビルドが容易になります！！**
* **ファームウェアのビルドが必要な場合は、WIFIまたはイーサネットケーブルで上位機に接続するチュートリアルを参照してください**

        ## 上位機SSHにネットワークIPを使用して接続する

        * 上位機が接続された**ルーターのバックエンド**にログインしてください
        * システムが**正常に起動している**ことを確認してください
        * 自分で**上位機のユーザー名とパスワードを知っている**ことを確認してください、FLYシステムのデフォルトユーザー名は**fly**、デフォルトパスワードは**mellow**です
        * メインボードが起動し、ネットワークに接続されている場合、IPを使用してSSHに接続できます
        * MobaXtermを開き、次のように設定します。IPは確認したものを入力します。また、ルーターのバックエンドでも確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **注意：ここでのステップ4に入力する上位機のユーザー名**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 初めて接続するとこのウィンドウがポップアップします、`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * パスワードの入力を求められたら、上位機のパスワードを入力し、エンターを押してください

        **注意：ここで入力するパスワードは見えません、入力後すぐにエンターを押してください**

        FLYシステムのデフォルト、ユーザー名は`fly`、パスワードは`mellow`です

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * 接続が成功しました
