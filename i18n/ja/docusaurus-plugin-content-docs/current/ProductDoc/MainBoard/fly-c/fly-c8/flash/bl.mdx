# BLファームウェア

    :::tip

    * 注意：出荷時に`bootloader`がプリフラッシュされているため、再フラッシュする必要はありません。

    :::

## フラッシュモードへの進入


    * フラッシュモードに入るには2つの方法があります。
    * 方法1：マザーボードを完全に電源オフにして、BOOTを長押ししながら、データ転送用のケーブルの一方をマザーボードに、もう一方をデバイスに接続します。Windowsに適しています。
    * 方法2：データ転送用のケーブルの一方をマザーボードに、もう一方をデバイスに接続した後、BOOTを長押ししながらコマンドを入力します。ホストコンピュータに適しています。
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.webp').default} size="100%" align="center" />

                ```bash
                sudo wget https://cdn.mellow.klipper.cn/BL/HIDBL.bin -O ~/HIDBL.bin && sudo fly-flash -d c8 -u -f ~/HIDBL.bin
                ```
