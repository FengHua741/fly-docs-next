# 主板注意事项

## 电源指示灯
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/power.webp').default} size="80%" align="center" />
    * TYPE-Cのみで電源を供給し、他のマザーボード電源に接続していない場合、少なくとも2つのランプ `3.3V` と `5V` が点灯します
    * マザーボード電源に接続した後、マザーボード電源ランプは `3.3V`、`5V`、`VCC` が点灯するはずです

### トラブルシューティング

    :::danger 注意

    * すべての外部機器（ドライバー、リミットスイッチ、12864など）が接続されていない場合
    * TYPE-Cのみで電源を供給すると、少なくとも `3.3V` が常に点灯します

    :::

    * マザーボード電源とドライバ電源を接続した後、`5V` ランプが点灯しない場合は、カスタマーサポートに連絡してください

## ドライバ

    * マザーボードは外付けドライバをサポートしています     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">接続と設定のチュートリアル</Button>

## リミットスイッチ

    * 接続順序に注意してください。3線リミットスイッチの場合、5V、GND、Sが間違っているとマザーボードが損傷する可能性があります
    * 2線の場合、SとGNDのみ接続すればよいです
    * PL08および他の近接スイッチは、信号ポートにダイオードを外付けして電圧を3.3V未満に下げなければなりません
