# 拨码介绍

:::tip
FLY Geminiはダイアルスイッチを介してアップストリームマシンに接続します。ダイアルスイッチの位置が正しくない場合は、正常に使用できません。
:::

:::danger 注意
ダイアルスイッチは指定された位置にセットする必要があります。それ以外の場合、アップストリームマシンへの正常な接続に影響します。
:::

## アップストリームマシンに接続

    * ダイアルスイッチ `1`, `2` を右に倒すと、ダウンストリームマシンとアップストリームマシンを内蔵USBで接続します
    * ダイアルスイッチ `3`, `4` を左に倒すと、ダウンストリームマシンとアップストリームマシンの接続に影響を与えます
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * この方法でダイアルスイッチを操作した後、下部のUSBポートはダウンストリームマシンによって占有されますので、他のデバイスを接続しないでください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## 外部アップストリームマシンに接続

    * ダイアルスイッチ `3`, `4` を右に倒すと、ダウンストリームマシンとアップストリームマシンを内蔵USBで接続します
    * ダイアルスイッチ `1`, `2` を左に倒すと、ダウンストリームマシンとアップストリームマシンの接続に影響を与えます
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
