# コードスイッチ紹介

:::tip
FLY c8はコードスイッチを使用してホストコンピュータに接続しますが、コードスイッチの位置が正しい場合にのみ正常に使用できます。
:::

:::danger 注意事項
コードスイッチは指定された位置に切り替える必要があります。それ以外の場合、ホストコンピュータとの接続に影響を与える可能性があります。
:::

## ホストコンピュータへの接続（ボード内蔵）

    * コードスイッチ `1`, `2` を右側に倒すと、下位機器を内部USB経由でホストコンピュータに接続します
    * コードスイッチ `3`, `4` を左側に倒さない場合、下位機器とホストコンピュータの接続に影響を与えます
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * この方法でコードスイッチを操作した後、下位機器によって下のUSBポートが占有されるため、他のデバイスを接続しないでください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## 外部ホストコンピュータへの接続

    * コードスイッチ `3`, `4` を右側に倒すと、下位機器を内部USB経由でホストコンピュータに接続します
    * コードスイッチ `1`, `2` を左側に倒さない場合、下位機器とホストコンピュータの接続に影響を与えます
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
