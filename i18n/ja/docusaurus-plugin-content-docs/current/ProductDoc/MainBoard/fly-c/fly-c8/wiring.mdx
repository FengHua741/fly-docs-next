## メインボード配線

:::warning 注意事項
画像の解像度が高いため、ウェブページで閲覧すると見えづらい場合があります。マウスカーソルを接線図に合わせて右クリックし、「画像を別名で保存」してデスクトップに保存してからご覧ください！
:::

:::warning 警告
購入した完成品の「3線式リミットスイッチ」は使用しないでください。このようなリミットスイッチは**メインボードにショートを引き起こし**、**メインボードを破損**する可能性があります！！！接線図に示された接線方法に従って必ず接続してください！！！
:::
