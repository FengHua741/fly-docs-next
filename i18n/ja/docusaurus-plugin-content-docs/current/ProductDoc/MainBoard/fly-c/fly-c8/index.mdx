## 製品特長

    * 32ビットのARM Cortex-M0+シリーズSTM32F407VGT6メインコントローラーを採用し、周波数は168MHzです。
    * TMCドライバーを搭載したSPIとUARTの動作モードがあります。
    * 簡単なジャンパーキャップの挿抜で無限回帰位置を使用できるDIAG機能ピンが搭載されています。
    * 数値制御ファンは24V、12V、5V電圧を選択でき、降圧モジュールを購入する手間が省けます。
    * 新しいフラッシュ書き込み方法により、FLY-Toolsを直接使用してメインボードファームウェアを更新できます。
    * 複数の電源管理があり、ポート、リミッター、ドライバーなどのショートが発生しても、上位機器の動作に影響を与えません。
    * RGBインターフェースが予備されており、12Vまたは5Vの電圧を選択できます。
    * SPIインターフェースが予備されており、顧客が外部の加速度センサーを接続して加速度補償を行うことができます。
    * TFTスクリーンとHDMIスクリーンはワンワイヤーでサポートされます。
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共鳴補償プラグインのnumpyとFLY-Toolsが事前にインストールされており、複雑な操作を必要とせず、初心者でも簡単に始められます（TFカードまたはM2WEを購入する必要があります）。

## 製品仕様

### 上位機器
    * CPU：高性能の全志H5チップ、4コア64ビットCortex-A53
    * GPU：高性能6コアMali 450、ピクセルフィリングレートは2.7gpixel/sを超えます
    * RAM：1GB DDR3（GPUと共有）
    * ROM：最大128GB SDカードをサポート
    * 外部機器：Spi x1、TFT x1、USB x4（USB 2.0 x3、OTG x1）、Micro HDMI x1、Eth x1（100M）、CAN x1、3.5mmオーディオインターフェース x1
    * インターフェース：M.2インターフェースを搭載しており、Wi-FiやeMMCなどの拡張が可能です。（私的なプロトコルを使用しているため、FLY π専用のM.2デバイス以外には接続しないでください）
    * 予備のZH1.5制御可能な5Vファンインターフェースがあり、上位機器の冷却に使用できます
    * TFTスクリーンとHDMIスクリーンのFPCワンワイヤーをサポート
    * CANブリッジインターフェースが予備されています
    * システムにはklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共鳴補償プラグインのnumpyとFLY-Toolsが事前にインストールされており、複雑な操作を必要とせず、初心者でも簡単に始められます（TFカードまたはM2WEを購入する必要があります）

### 下位機器

    * MCU: STM32F407VGT6
    * ファームウェア:  KLIPPER/RRF/Marlin
    * 12-24Vの直流電源をサポートし、安定した電源供給が可能であり、降圧モジュールを省けます
    * サーマルベッドポートの最大出力電流:  15A
    * ヒータースティックポートの最大出力電流:  6A
    * ファンポートの最大出力電流:  1A
    * ドライバーワークモードサポート：SPI、UART、STEP/DIR
    * 対応ドライバー：外付けTMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988などに対応
    * 拡張インターフェース：BLTouch（Servos、Probe）、RGB X2、UART X1、12864 X1
    * ファン: 二線式数値制御ファン X6 (24V、12V、5Vを選択可能)
    * 温度センサー：3つのNTC 100K
    * 対応マシン構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
