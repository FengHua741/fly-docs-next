## システム焼込み前の準備

    * リーダー
    * TFカード1枚、容量は16GB以上または128GB以下のもので、転送速度がclass10またはそれ以上のもの
    * 対応する <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button> をダウンロード
    * 必要な <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込みツール</Button> をダウンロード

## システム焼込み

:::info
TFカード焼込みとTFカードからM2WEへの焼込みはどちらか一方を使用すれば十分です
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードでのシステム焼込みを推奨します。USB経由での焼込みはドライバ互換性の問題などで失敗する可能性がありますので、直接[メモリーカードからM2WEへ焼込み](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)することをお勧めします
* 適切なシステムイメージをダウンロードしていることを確認してください。適切でない場合は起動できません
:::

## アンテナの取り付け

    1. `M2WE`または`2.4G WiFi`モジュールのアンテナを取り付けます
    :::warning
    * FLY上位機は`M2WE`または`2.4G WIFI`モジュールを使用する必要があります。そうでないとWiFiを利用できません
    * 注意: 上位機は日本語名前のWiFiはサポートしていません。また、2.4Gと5Gの両方を統合したWiFiには対応していません
    :::
    * 赤い矢印が示す部分がWiFiアンテナの接続口です
    * WiFiアンテナの接続口は`ipex1`です
    * WiFiアンテナをM2WEに取り付けてください。信号が弱いため、ルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードの取り付けが完了したら、次のステップに進み、SSHで上位機に接続できます
    :::

    2. 上位機に電源供給を行います。  
    **ここまできたら、あなたの上位機は起動しました！！！**  
    次のステップに進むには：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">SSHで上位機に接続</Button>
