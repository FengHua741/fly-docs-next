## システム焼込前の準備

- リーダー
- TFカード1枚、容量は16GB以上または128GB以下、TFカードの転送速度はclass10級またはそれ以上である必要がある
- 対応するシステムイメージをダウンロードする：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">システムイメージ</Button>
- 必要な焼込ツールをダウンロードする：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">焼込ツール</Button>

## システム焼込

:::info
TFカード焼込とTFカード経由でのM2WE焼込はどちらか一つを使用してください
:::

:::info
* マシンにM2WEとメモリーカードがインストールされている場合、システムはデフォルトでメモリーカードから起動します
* M2WEはTFカードでのシステム焼込をお勧めします。USB経由での焼込はドライバ互換性の問題などで失敗する可能性があります。直接[TFカードからM2WEへの焼込](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)をお勧めします
* 適切なシステムイメージをダウンロードしたことを確認してください。適切でない場合は起動できません
:::

## アンテナの取り付け

1. `M2WE`または`2.4G WiFi`モジュールのアンテナを取り付けます
:::warning
* FLY上位機器では`M2WE`または`2.4G WiFi`モジュールを使用する必要があります。それがないとWiFiが使えません
* 注意：上位機器は中国語名前のWiFiをサポートしていません。また、2.4Gと5Gの統合WiFiはサポートしていません
:::
* 赤い矢印が指しているのがWiFiアンテナの接続口です
* WiFiアンテナ口は`ipex1`です
* 天線をM2WEに取り付けない場合、信号が弱いためルーターに接続できない可能性があります
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

:::tip
- システムカードの取り付けが完了したら、次の手順に進み、SSHで上位機器に接続できます
:::

2. 上位機器に電源を供給します。  
**ここまでで、あなたの上位機器は起動できるようになりました！！！**  
次の手順へ進む：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">SSHで上位機器に接続</Button>
