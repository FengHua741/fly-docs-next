# WIFIに接続


## C8にアンテナを設置する

    * 赤い矢印が指している場所がWIFIアンテナインターフェースです
    * WIFIアンテナポートは`ipex1`です
    * アンテナをC8に設置しないと、信号が弱いためネットワークにつながらない可能性があります
