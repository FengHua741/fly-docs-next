# ダイアルスイッチ紹介

:::tip
FLY c8 はダイアルスイッチを使用してホストコンピューターに接続しますが、ダイアルスイッチの位置が正しくない場合、正常に使用できません。
:::

:::danger 注意事項
ダイアルスイッチを指定された位置に設定する必要があります。それ以外の場合、ホストコンピューターとの接続に影響を与える可能性があります。
:::

## ホストコンピューターへの接続（ボード内蔵）

    * ダイアルスイッチ `1`, `2` を右側に倒すと、下位機器とホストコンピューターが内部USBで接続されます
    * ダイアルスイッチ `3`, `4` を左側に倒すと、下位機器とホストコンピューターの接続に影響を与える可能性があります
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * この方法でダイアルスイッチを操作した後、下位機器によって下方のUSBポートが占有されるため、他のデバイスを接続しないでください
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## 外部ホストコンピューターへの接続

    * ダイアルスイッチ `3`, `4` を右側に倒すと、下位機器とホストコンピューターが内部USBで接続されます
    * ダイアルスイッチ `1`, `2` を左側に倒すと、下位機器とホストコンピューターの接続に影響を与える可能性があります
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
