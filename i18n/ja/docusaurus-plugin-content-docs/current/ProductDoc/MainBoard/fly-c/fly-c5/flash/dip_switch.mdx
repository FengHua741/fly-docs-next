# ダイアルスイッチ紹介

:::tip
FLY c8はダイアルスイッチを介してホストコンピュータに接続します。ダイアルスイッチの位置が正しい場合にのみ正常に使用できます。
:::

:::danger 注意事項
ダイアルスイッチが指定された位置に設定されていることを確認してください。
:::
