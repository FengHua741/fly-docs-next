## 製品特徴

- 32ビット ARM Cortex-M4、550MHz、STM32H723VGT6 チップ
- TMC ドライバー対応の SPI、UART、STEP/DIR モードサポート
- DIAG 機能ピンは限界スイッチを占有せず、設定後に無限限界帰還が可能
- 数値制御ファンが 24V、12V、5V をサポートし、追加の降圧モジュールが不要
- 断電自動シャットダウンと断電継続印刷（デフォルトで断電自動シャットダウンが有効）
- USB Klipperファームウェア出荷時インストール済み
- 多機能電源管理、ポート、限界スイッチ、ドライバーなどのショートは上位機器の動作に影響しない
- 5V RGBインターフェース予備設置
- TFTとHDMI画面の1本化サポート
- システムにklipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyが事前インストールされ、ソース変更が不要で簡単に操作可能（TFカードまたはM2WEの購入が必要）

## 製品仕様

### 上位機

- **CPU**：全志 H618、4コア 64ビット Cortex-A53
- **GPU**：Arm Mali-G31
- **RAM**：1GB DDR4 3200M
- **ROM**：最大128GB SDカードサポート
- **周辺機器**：
    - TFT x1
    - USB 2.0 x2
    - Micro HDMI x1
    - エーテルネットポート（Eth）x1（100M）
    - FPC-HDMI x1
    - UART x1
- **インターフェース**：内蔵M.2、WIFIやeMMCなどの拡張が可能（プライベートプロトコル、FLY専用以外のM.2デバイスは接続しないでください）
- **サポート**：TFTディスプレイとHDMIディスプレイのFPC一本化
- **システム事前インストール**：klipper、moonraker、mainsail、fulidd、klipper-screen、Crowsmest、共振補償プラグインnumpyが含まれており、ソース変更が不要で簡単に操作可能（TFカードまたはM2WEの購入が必要）

### 下位機

- **MCU**：STM32H723VGT6
- **ファームウェア**：KLIPPER
- **電源供給**：12-24V直流電源サポート、降圧モジュール不要
- **ヒーターベッドポート最大出力電流**：15A
- **ヒーターバーポート最大出力電流**：6A
- **ファンポート最大出力電流**：1A
- **ドライバーモード**：SPI、UART、STEP/DIR
- **サポートドライバー**：TMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988など
- **拡張インターフェース**：BLTouch（Servos、Probe）、RGB x1、UART x1、CAN x1
- **ファン**：2線式数値制御ファン x4（24V、12V、5V選択可能）
- **温度センサー**：3路NTC100K
- **サポート構造**：Cartesian、Delta、Kossel、Ultimaker、CoreXYなど
