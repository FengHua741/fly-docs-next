# Klipper電源喪失時の継続印刷機能

- Klipperの電源喪失後の継続印刷機能は、Klipperが再び電源を入れたときに、Klipperの印刷状態を自動的に復元できる機能です。
- C5はKPPMなしでも使用できます
- Z軸が移動する機種には適用されません。
