# BLファームウェア

    :::tip

    * 注意: 出荷時にブートローダーが事前に焼かれているため、再び焼く必要はありません
    * C5は自社提供のPCソフトウェアを使用して書き込むことができます
    :::

## 上位機とのSSH接続

    :::danger 警告
    - BLファームウェアを焼く前に、`WiFiで上位機にSSH接続する必要があります`
    :::

## ファームウェアの焼込み


    * ダイアルスイッチを上向きにして、下位機のUSBを上位機に接続します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * 下記のコマンドを入力してBLファームウェアを焼込み開始します


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```
