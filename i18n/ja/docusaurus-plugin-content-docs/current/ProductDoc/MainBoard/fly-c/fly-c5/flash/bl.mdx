# BLファームウェア

    :::tip

    * 出荷時にブートローダーが事前にフラッシュされているため、再度フラッシュする必要はありません。

    :::

## フラッシュモードに入る

    :::tip

    * 左側がメインボードのRST、右側がメインボードのBOOTです。

    :::

    * フラッシュモードには2つの方法があります。
    * 方法1：メインボードを完全に電源オフにして、BOOTを長押ししながら、データ転送用のケーブルをメインボードとデバイスに接続します。（Windows向け）
    * 方法2：データ転送用のケーブルをメインボードとデバイスに接続した後、BOOTを長押ししながらコマンドを入力します。（上位機器向け）
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/boot.webp').default} size="60%" align="center" />
