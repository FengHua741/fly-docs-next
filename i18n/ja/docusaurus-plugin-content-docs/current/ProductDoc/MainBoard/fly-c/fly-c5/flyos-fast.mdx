## アンテナの取り付け

    1. `M2WE`または`2.4G Wi-Fi`モジュールのアンテナを取り付けます
    :::warning
    * FLY上位機は`M2WE`または`2.4G Wi-Fi`モジュールを使用する必要があります。それがないとWi-Fiが使えません
    * 注意してください、上位機は中国語名前のWi-Fiをサポートしていません。また、2.4Gと5Gの両方を統合したWi-Fiもサポートしていません
    :::
    * 赤い矢印で示されている部分がWi-Fiアンテナ接続口です
    * Wi-Fiアンテナポートは`ipex1`です
    * 天線をM2WEに取り付けてください。そうしないと信号が弱いためルーターに接続できない可能性があります
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - システムカードのインストールが完了したら、次に進んでSSHで上位機に接続できます
    :::

    2. 上位機に電源を入れます。  
    **ここでは、あなたの上位機は起動できるようになりました！！！**  
    次の手順へ進んでください：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh">上位機にSSH接続</Button>
