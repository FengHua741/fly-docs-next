# Wi-Fiに接続する

## C5にアンテナを設置する

* 赤い矢印が指している部分がWi-Fiアンテナのインターフェースです
* Wi-Fiアンテナインターフェースは`ipex1`です
* アンテナをC5に設置しないと、信号が弱いためネットワークにつながらない可能性があります
