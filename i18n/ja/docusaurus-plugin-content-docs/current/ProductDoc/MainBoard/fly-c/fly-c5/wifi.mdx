# Wi-Fiに接続する

## C5にアンテナを設置する

* 赤い矢印が指している部分がWi-Fiアンテナのインターフェースです
* Wi-Fiアンテナのインターフェースは`ipex1`です
* C5にアンテナを設置しないと、信号が弱いためネットワークに接続できない可能性があります
