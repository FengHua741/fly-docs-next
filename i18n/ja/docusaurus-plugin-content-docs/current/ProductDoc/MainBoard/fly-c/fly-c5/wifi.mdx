# Wi-Fiに接続する

## C5にアンテナを取り付ける

* 赤い矢印が示す部分がWi-Fiアンテナのインターフェースです
* Wi-Fiアンテナのインターフェースは`ipex1`です
* アンテナをC5に取り付けないと、信号が弱いためネットワークに接続できない可能性があります
