# メインボードの注意事項

## 電源インジケーター
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/power.webp').default} size="80%" align="center" />
* TYPE-Cで電源を供給し、他のメインボード電源に接続していない場合、少なくとも「3.3V」と「5V」のライトが点灯します。
* メインボード電源に接続すると、「3.3V」、「5V」、「VCC」のライトが点灯するはずです。

### 問題のトラブルシューティング

:::danger 注意

* 何も外部機器を接続していない場合（ドライバ、リミットスイッチ、12864など）
* TYPE-Cだけで電源を供給すると、「3.3V」は常に点灯します。

:::

* メインボード電源とドライバ電源を接続しても「5V」のライトが点灯しない場合は、カスタマーサポートに連絡してください。

## ドライバ

* メインボードは外部ドライバをサポートしています。<Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定チュートリアル</Button>

## リミットスイッチポート

* 接続順序に注意してください。3線式リミットスイッチの場合、5V、GND、Sを間違えるとメインボードが損傷する可能性があります。
* 2線式の場合はSとGNDに直接接続できます。
* PL08や他の近接スイッチは、信号ポートに外部ダイオードを接続して電圧を3.3V以下に下げてください。
