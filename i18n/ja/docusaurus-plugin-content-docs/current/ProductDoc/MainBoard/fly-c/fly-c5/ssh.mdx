# 上位機SSHへの接続

* シリアルポートまたはWi-Fiを使用してデバイスに接続することができます
* 事前に**MobaXterm_Personal**または他の**SSHターミナルツール**を準備してください
* SDカードまたはM2WEに**対応する上位機のシステム**が焼込まれていることを確認してください
* **もし上位機が正常に起動しない場合はドライバ、限界、ファンなどの外部機器を取り外してください！！**
* 最後にネットワーク経由でSSHに接続し、ファームウェアをコンパイルすることをお勧めします
* **ファームウェアをコンパイルする必要がある場合は、Wi-Fiまたはイーサネットで上位機に接続するチュートリアルをご覧ください**

        ## 上位機SSHにネットワークIP経由で接続する

        * 上位機が接続された**ルーターのバックエンド**にログインしてください
        * システムが**正常に起動している**ことを確認してください
        * 自分で上位機のユーザー名とパスワードを知っている必要があります。FLYシステムのデフォルトのユーザー名は**fly**、パスワードは**mellow**です
        * マザーボードが起動しており、ネットワークに接続されている場合、IP経由でSSHに接続できます
        * MobaXtermを開き、以下の設定を行い、IPは先ほど確認したものを入力してください。また、ルーターのバックエンドでも確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **注意：ここではステップ4で上位機のユーザー名を入力します**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 初回接続時にこのウィンドウがポップアップします。`Accept`をクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * パスワードを入力するように促されるので、上位機のパスワードを入力してEnterキーを押してください

        **注意：ここで入力されるパスワードは見えません。入力後直ちにEnterキーを押してください**

        FLYシステムのデフォルトでは、ユーザー名は：`fly`、パスワードは：`mellow`です
