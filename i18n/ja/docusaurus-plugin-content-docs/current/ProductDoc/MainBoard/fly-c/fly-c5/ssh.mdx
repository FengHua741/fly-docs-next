# 上位機SSHへの接続

:::danger 注意
- あなたの上位機がブートローダーインストール後に初めて起動する場合、必ず`シリアルポートで上位機SSHに接続`してください
- 最初の起動時には、電源とType-C以外のすべてのケーブルを接続しないでください！！！
- ファームウェアをコンパイルする際は、`WiFiで上位機SSHに接続`することをお勧めします。これにより、ファームウェアのコンパイルや書き込みが簡単になります！
:::

* 接続開始前に、以下のSSHソフトウェアをダウンロードしてください: <Button variant="contained" disableElevation href="https://mobaxterm.mobatek.net/download-home-edition.html ">SSHソフトウェア</Button>   右側の`Installer edition`を選択するとダウンロードできます

        ## シリアルポートを使用して上位機SSHに接続

       :::tip 注意
        - TFカードまたはM2WEが正しいシステムに焼かれていることを確認し、FLY-C5に正しく取り付けられていることを確認してください
        - FLYOS-FASTシステムのデフォルトユーザ名は`root`、デフォルトパスワードは`mellow`です
       :::

        * Type-Cデータ線を使用してFLY-C5をコンピューターに接続します
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * ボードが正常にコンピューターに接続されたら、デバイスマネージャーを開き、CH340ポートが表示されているか確認してください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/pc-device-mgr.webp').default} size="50%" align="center" />

        * `MobaXterm`を開き、設定を行い、ポートにはデバイスマネージャーで見たポートを選択してください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-2.webp').default} size="60%" align="center" />

        * 問題なく動作している場合、起動画面が表示されます

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial1.webp').default} size="60%" align="center" />  
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial2.webp').default} size="60%" align="center" />


        * FLYOS-FASTは起動速度が速いため、おそらく乱れた文字が表示されることがあります。これは正常な現象です。心配しないで、**Enterキーを数回押してください**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial3.webp').default} size="60%" align="center" />


        * Enterキーを押すと、下図のような内容が表示されます
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial4.webp').default} size="60%" align="center" />

        * ユーザー名:`root`と入力し、Enterキーを押します。次にパスワード:`mellow`と入力し、Enterキーを押します。パスワードが表示されないのは正常です  

        :::warning 注意 
        - FLYOS-FASTシステムのユーザー名は`root`、パスワードは`mellow`です
        - FLYOS-FASTシステムには`root`アカウントのみ存在します！！！
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial5.webp').default} size="60%" align="center" />

        * IPアドレスを確認したい場合は、SSHで`ip a | grep inet`と入力してEnterキーを押します。返ってきた内容からルータと同じセグメントのIPを検索してアクセスしてください

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial6.webp').default} size="60%" align="center" />

    <TabItem value="net" label="WiFiで上位機SSHに接続" default>
        ## WiFiを使用して上位機SSHに接続
       :::tip 注意
        - 上位機システムが正常に起動していることを確認してください
        - FLYOS-FASTシステムのデフォルトユーザ名は`root`、デフォルトパスワードは`mellow`です
        - WiFiで上位機SSHに接続するための**前提条件**は、上位機がルーターに接続されており、上位機のIPが分かっていることです
       :::

        * FLY-C5をコンピューターにType-Cデータ線で接続します

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/c5.webp').default} size="40%" align="center" />

        * MobaXtermを開き、下図のように設定を行います。`Remote host`にはシリアルポート接続時に確認した上位機のIPを入力してください。また、ルーターバックグラウンドから上位機のIPも確認できます

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **注意：ここでは4番目の手順で上位機のユーザ名`root`を入力してください**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_2.webp').default} size="60%" align="center" />

        * 最初の接続時にこのウィンドウが表示される場合があります。`Accept`ボタンをクリックしてください

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * パスワードの入力を求められるので、上位機のパスワードを入力し、Enterキーを押します
        :::tip
        - パスワードが見えませんが、入力後直接Enterキーを押してください
        :::
        
        **FLYOS-FAST**システムのデフォルトユーザ名は:`root`、パスワードは:`mellow`です

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_3.webp').default} size="60%" align="center" />

        **もし下図のようなウィンドウが表示された場合、これはユーザーのパスワードを保存するかどうかを尋ねるポップアップです。`No`をクリックしてください**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * 下図のような画面が表示されたら、ログイン成功です！

        :::warning 注意 
        - FLYOS-FASTシステムのユーザ名は`root`、パスワードは`mellow`です
        - FLYOS-FASTシステムには`root`アカウントのみ存在します！！！
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_5.webp').default} size="60%" align="center" />



    </TabItem>
