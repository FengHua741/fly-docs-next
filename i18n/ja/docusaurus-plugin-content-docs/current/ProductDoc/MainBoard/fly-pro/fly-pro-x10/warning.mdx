# メインボードの注意事項

## 電源インジケーター
    * Type-C給電のみで他のメインボード電源に接続しない場合、`SYS_5V`とSYS_3.3Vが点灯するはずです。
    * メインボード電源`HVIN`、`VIN`、`BOARO`、`POWER`に接続した後は、メインボード電源ランプが常に点灯するはずです。

## ドライバー

    * TP5160とTP2209は名前が異なるだけで、ジャンパーピンを設定することなく駆動モードを選択できます。
    * 駆動ポートにはTPドライバーのみをインストールでき、他のデバイスはインストールできません。

## 限界ポート
    * ProX10は5Vと12V電源を選択できます。近接スイッチを使用する場合は、降圧のためにダイオードをインストールする必要があります。

## オンボードコンピュータ
    * ProX10は外部オンボードコンピュータを使用できます。ダイアルコードでType-Cに接続を選択すると使用できます。

## ファームウェアの書き込み

    * メインボードはTFカードを使用して書き込むことが可能です。ただし、工場出荷時には`Katapult`がデフォルトです。
