## 製品特長

    * 32ビット、550 MHzのARM Cortex-M7シリーズSTM32H723ZGT6メインコントローラーを採用
    * ボードとコアボードはBTB接続を使用し、オレンジパイCM4やラズベリーパイCM4などのコアボードを使用可能
    * ボードにはBOOTボタンが設置されており、ユーザーはDFU方式でボードブートローダーを更新できる
    * 10個のプラグイン式TPドライバーを搭載、ジャンパーピンを必要とせずにドライバーモードを切り替えられる
    * 板載DIAG機能ピンがあり、簡単なプラグインジャンパーブロックを使用して無限位置リターンが可能
    * ドライバーパワーサプライはHVINとVIN電源を選択可能
    * MicroSDカードでMCUファームウェアをアップグレードでき、Klipperのmake flashコマンドを使用してDFU経由でMCUファームウェアを更新することも可能
    * 6つの2線制御ファンと、差し込み可能なMOS、2つの4線制御ファン、1つのVCCファンをサポート
    * デジタル制御ファンは24V、12V、5Vの電圧を選択可能で、外部変圧モジュールの接続を省き、マザーボードの損傷確率を減らす
    * 交換可能なヒューズを採用し、交換が簡単

## 製品仕様

    * MCU: STM32H723ZGT6
    * ドライバーパワーサプライ: VIN(12V-24V),HVIN(12V-48V)
    * ボードパワーサプライ: VIN(12V-24V)
    * サーマルベッドパワーサプライ: BED IN(12V-24V)
    * ヒーターアダプタ: サーマルベッド(BED OUT)、ヒータースティック（HETA0，HETA1，HETA2，HETA3）
    * サーマルベッドポート最大出力電流: 15A
    * ヒータースティックポート最大出力電流: 6A
    * ファンポート最大出力電流: 1A
    * ドライバーワークモードサポート：SPI、UART、STEP/DIR
    * 拡張インターフェース：BLTouch（Servos、Probe）、UART X1、高圧リミットポート X1
