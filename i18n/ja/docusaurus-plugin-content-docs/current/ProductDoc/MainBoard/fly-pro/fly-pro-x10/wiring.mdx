## メインボードの配線についての説明

:::warning 警告
  購入した完成品の「3線式リミットスイッチ」を使用しないでください。このようなリミットスイッチは、**メインボードにショートを引き起こし**、**メインボードを破損**させる可能性があります！！！必ず配線図に示された接続方法に従って接線してください！！！
:::
