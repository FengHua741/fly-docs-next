# プラグインを使用してLEDドライバーを拡張する

以下のリンクに従って、上位機にSSHで接続した後にインストールと設定を行ってください

```cfg
https://github.com/julianschill/klipper-led_effect
```

## インストール

**MobaXterm_Personal**などの**SSHツール**を使用し、**WIFI**を通じて上位機に接続してください。また、以下の点を確認してください

1. **上位機にKlipperファームウェアが正しくインストールされていることを確認してください**
2. **Klipperをインストールしたユーザーとしてログインしていることを確認してください**
3. **入力法が英語であることを確認してください**
4. **上位機が正常にデバイスを検出できるようになっていることを確認してください**
5. **上記のすべての注意事項を実施していない場合、次のステップに進めません**

このモジュールは、既存のKlipperインストールにインストールすることができます。

```cfg
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## 参照設定

```cfg
[neopixel TP_led]
pin: PA8
chain_count: 50
# LEDの数
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
