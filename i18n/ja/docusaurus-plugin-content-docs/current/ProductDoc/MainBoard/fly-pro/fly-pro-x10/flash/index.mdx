# 固件焼込み説明

    :::warning 注意事項

    * PRO X10は予めKatapultが焼込まれています
    * BLを一度でも焼込むと、その後は再びBLを焼込む必要はありません
    * PRO X10のBLには2種類あり、KatapultファームウェアとTFカードで焼込まれたBLファームウェアがあります
    * 必要に応じて、自身の選択に基づいて再び<Button variant="contained" disableElevation href="./bl">BLファームウェア</Button>を焼込むことができます

    :::
    
    :::danger 知識が必要
    * メインボードのファームウェア焼込みはKatapultを通じてKlipperを焼込む必要があります
    * メインボードがすでにKatapultが焼込まれていることが確認された場合、最後のKatapultファームウェアの焼込みは不要です。これはメインボードのKatapultファームウェアが正常に焼込まれなかった場合の予防策です
    * リセットボタンを2回素早く押すと、メインボード上のLEDが点滅しているかどうかを確認できます。これが点滅している場合は、メインボードがKatapultファームウェアに入っていることを示します
    :::

## どの種類のファームウェアが必要かをどのように判断するか

    :::tip

    * ご注意ください、あなたは一つのファームウェアだけを焼込む必要があります

    :::
    
    * Pro X10は以下の方法でホストコンピュータに接続することができます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼性があり、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスに接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションタイムアウトなどのエラーが発生する可能性があります。
