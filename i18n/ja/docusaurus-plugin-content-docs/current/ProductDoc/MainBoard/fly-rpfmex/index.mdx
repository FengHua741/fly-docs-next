    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * 固件:  Klipper
    * 支持12-24V直流供電，供電更穩定，省去降壓模塊
    * 驅動工作模式支持：UART、STEP/DIR
    * 拓展接口：可控風扇*2，RGB*1，限位*3
