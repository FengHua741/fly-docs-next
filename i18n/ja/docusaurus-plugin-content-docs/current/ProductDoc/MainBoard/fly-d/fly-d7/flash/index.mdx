    ファームウェアのビルドと焼込みを行う前に、必ずこの欄を読んでください！！

:::tip 注意事項
* D5は2024年9月1日に出荷時にKatapultファームウェアがプリインストールされています
* BLを一度でも焼込み済みの場合、後続でBLを焼込む必要はありません（D5のBLはKatapultファームウェアです）
* BLはKatapultファームウェアをブートするためのブートローダーであり、Klipperファームウェアも焼込む必要があります
:::

## 必要なファームウェアの判断

    :::tip

    * ご注意：あなたはいずれかのファームウェアのみを焼込む必要があります

    :::
    
    * D7は次の2つの方法でホストコンピュータに接続できます
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USBファームウェア（最も一般的に使用されます）</Button>
        * 利点：通信が信頼でき、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USBブリッジファームウェア（CANツールボードを使用する際によく使用されます）</Button>
        * 利点：複数のデバイスに接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションのタイムアウトエラーなどの問題が発生する可能性があります。
