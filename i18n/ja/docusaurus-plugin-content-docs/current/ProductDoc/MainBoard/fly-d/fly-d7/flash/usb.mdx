# USBファームウェアのコンパイル


## コンパイルの開始

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押して、**Save configuration** が表示されたら、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 現在は設定が保存され、コマンドラインに戻ったはずです
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * コンパイルが成功すると、次の出力が表示されます

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
## ファームウェア焼成モードに入る

    :::tip ヒント
        2024年9月1日以降に出荷されたD7には既にKatapultファームウェアが焼成されています
    :::
    * 主基板をホストマシンにType-Cケーブルで接続し、**リセットボタンをダブルクリック**します。このとき、主基板上のLEDが点滅します。
    :::warning 警告
       * LEDが点滅しない場合は、katapultファームウェアを再焼成してください：<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#進入dfu燒錄模式">katapultファームウェアの焼成</Button>
       * LEDが点滅しない場合は、katapultファームウェアの焼成後、再度実行してください
    :::
