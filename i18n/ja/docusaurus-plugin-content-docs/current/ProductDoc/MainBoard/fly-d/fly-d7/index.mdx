# 概要

> Fly-D7は広州メイレン電子科技有限公司Fly3Dメインボード開発チームが発売した高性能3Dプリンターメインボードで、最大7つのステッピングモーターをサポートしています。


    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.webp').default} size="60%" align="center" />

## **1. 製品特長**

    * 32ビット、周波数48MHzのARM Cortex-M0+シリーズSTM32F072RBT6メインコントローラーチップを採用
    * CANバス接続をサポートし、データ伝送がより安定し、遅延が少なく、接続がより安定します
    * ドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * ボードにDIAG機能ピンがあり、単純なジャンパキャップの挿抜だけで無限位リターンサポートを使用できます

## 2. 製品仕様

    * MCU: STM32F072RBT6
    * ファームウェア: Klipper/Marlin
    * 12-24V DC電源をサポートし、電源供給がより安定し、降圧モジュールを省けます
    * ドライバーワークモードサポート：SPI、UART、STEP/DIR
    * サポートドライバー：外部TMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988などをサポート
    * 拡張インターフェース：BLTouch（Servos、Probe）、UART X1
    * ファン: 2線制御可能なファン X2、並列ファン X2、常時オンファン X2
    * 温度センサー：3路NTC 100K
    * ヒーターアンプ：3路制御可能なインターフェース、最大10A
    * 対応マシン構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
