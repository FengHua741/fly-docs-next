## メインボード接線

:::warning 注意事項
由於画像の解像度が高いため、ウェブページ上で閲覧すると見えにくいです。マウスカーソルを画像上に置き、右クリックして「画像を保存」を選択し、デスクトップに保存してから確認してください！
:::

:::warning 警告
購入した完成品の「3線式リミットスイッチ」は使用しないでください。このようなリミットスイッチは、メインボードにショートを引き起こし、メインボードを破損させる可能性があります！！！絶対に接線図に示された接線方法に従って接線してください！！！
:::
