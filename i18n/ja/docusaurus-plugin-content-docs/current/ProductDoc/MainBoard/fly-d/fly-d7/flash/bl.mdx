# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip
        * D7のkatapultファームウェアは第三の接続方法を使用しています：USB接続
        * D7は2024年9月1日に出荷時にKatapultファームウェアが事前に焼付けられています
    :::

## DFU焼付けモードに入る

    :::tip

    左が`BOOTボタン`、右が`RESETボタン`

    :::

    * 焼付けモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源を切った状態で、BOOTボタンを押したままデータ転送用のケーブルの一方をマザーボードに接続し、もう一方をデバイスに接続します
    * 方法2：データ転送用のケーブルの一方をマザーボードに接続し、もう一方をデバイスに接続した後、BOOTボタンを押したままRSTボタンを押し、その後RSTボタンを離し最後にBOOTボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/img/boot.webp').default} size="50%" align="center" />

            4. ファームウェアの焼付け

            :::tip

            一つのファームウェアだけ焼付けます

            :::

            * 以下のコマンドでファームウェアを焼付けます
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/D/FLY_D7.bin
                ```

            5. 以下のような矢印部分に示される内容``File downloaded successfully``が出たら焼付け成功です
