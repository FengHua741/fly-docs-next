# CANブリッジ

CANバスは、リアルタイムアプリケーション用のシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送でき、世界で最も広く使用されているフィールドバスの1つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価でかさばる配線ハーネスを置き換えるために使用されます。このプロトコルの堅牢性により、他の自動化や産業応用にも使用されるようになりました。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの転送レート、11ビットのアドレス付け、およびエラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用して、プリントヘッドをメインボードに接続するための配線の数を減らします。元々は十数本の線が必要でしたが、CANを使用することで、4本の線のみで済むようになり、配線の数を大幅に減らし、配線の難易度も低下します。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードはCANでメインボードと接続します

* ツールボードの電源ラインはCANポートにつながず、12V-24V電源ラインにつながることに注意してください

* 120Ωジャンプワイヤーを接続することに注意してください

## メインボードのファームウェアを焼く

* D7メインボードに焼くファームウェアは`USB CANブリッジファームウェア設定`であることを確認してください
