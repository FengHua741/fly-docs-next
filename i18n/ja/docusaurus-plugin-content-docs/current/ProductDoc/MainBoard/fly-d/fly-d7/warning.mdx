# メインボードの注意事項

## 電源インジケーター
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/power.webp').default} size="80%" align="center" />
    * TYPE-Cで電源を供給し、他のメインボード電源に接続していない場合、少なくとも「3.3V」と「5V」の2つのランプが点灯します
    * メインボード電源に接続した後は、「3.3V」、「5V」、「VCC」の3つのランプが点灯するはずです

### 問題のトラブルシューティング


    :::danger 注意

    * すべての外部機器（ドライバー、リミットスイッチ、12864など）が接続されていない状態で
    * TYPE-Cで電源を供給すると、「3.3V」と「5V」の2つのLEDが常時点灯するはずです

    :::

    * ランプが全く点灯しない場合は、ショート回路が発生している可能性があります。カスタマーサービスに連絡してください
    * 「3.3V」または「5V」のいずれかのランプが点滅する場合は、ショート回路が発生している可能性があります。カスタマーサービスに連絡してください

## ドライバー

    * メインボードは外部ドライバーをサポートしています     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定チュートリアル</Button>

## リミットスイッチポート

    * 接続順序に注意してください。3ピンリミットスイッチの場合、「5V」、「GND」、「S」が間違っていると、メインボードが損傷する可能性があります
    * 2ピンの場合、「S」と「GND」を直接接続するだけで十分です
    * PL08および他の近接スイッチは、信号ポートにダイオードを外付けして電圧を3.3V未満に下げなければなりません

## ファームウェア

    * ボタンの問題により、チュートリアルはKatapultファームウェアを使用してファームウェアを書き込むように更新されました
    * Katapultファームウェアを使用してファームウェアを書き込むことをお勧めします。その後、Katapultファームウェアを使用してKlipperを書き込みます
