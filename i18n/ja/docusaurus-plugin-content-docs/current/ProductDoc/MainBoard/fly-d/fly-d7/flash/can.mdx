# USBブリッジCANファームウェアの焼込

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` キーを押すと、**Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻ります
    * `make -j4` を入力してコンパイルを開始します。時間がかかることがあります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に次の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## ファームウェア焼込モードに入る

    :::tip ヒント
        2024年9月1日以降に製造されたD7には既にKatapultファームウェアが焼込まれています
    :::
    * USBケーブルでメインボードをホストマシンに接続し、**resetボタンをダブルクリック**すると、メインボードのLEDが点滅します。
    :::warning 警告
       * LEDが点滅しない場合、<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#進入dfu燒錄模式">Katapultファームウェアの焼込み</Button>を再度実行してください
       * LEDが点滅しない場合、Katapultファームウェアの焼込み後、次の手順を実行してください
    :::
