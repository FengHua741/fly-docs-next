# USBブリッジCANファームウェア焼付け

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` キーを押して、**Save configuration** が表示されたら、その後に `Y` キーを押す
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 現在、設定が保存され、コマンドラインに戻ったはずです
    * コンパイルを開始するための次のコマンドを入力します（時間がかかります）
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## ファームウェア焼付けモードに入る

    :::tip 注意
        2024年9月1日以降に出荷されたD7はKatapultファームウェアが既に焼付けてあります
    :::
    * Type-Cケーブルを使用してメインボードをPCに接続し、**resetボタンをダブルクリック**すると、メインボードにLEDが点滅します。
    :::warning 警告
       * LEDが点滅しない場合、Katapultファームウェアの再焼付けが必要です: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#進入dfu燒錄模式">Katapultファームウェア焼付け</Button>
       * LEDが点滅しない場合、Katapultファームウェア焼付け後、再度この手順を実行してください
    :::

    <ImageView image={require('@site//docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/img/d7_dfu.webp').default} size="50%" align="center" />
