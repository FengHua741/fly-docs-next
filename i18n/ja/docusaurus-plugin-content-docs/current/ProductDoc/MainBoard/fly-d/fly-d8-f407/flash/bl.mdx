# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * D8はUSBで接続します

    :::

## フラッシュモードに入る

    :::tip

    左が`RSTボタン`、右が`BOOTボタン`です

    :::

    * フラッシュモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源を切られている状態で、BOOTボタンを押しながらデータ転送用のケーブルをマザーボードとデバイスの間に接続します
    * 方法2：データ転送用のケーブルをマザーボードとデバイスの間に接続した後、BOOTボタンを押しながらRSTボタンを押し、その後RSTボタンを離し、最後にBOOTボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/flash/img/boot.webp').default} size="50%" align="center" />
