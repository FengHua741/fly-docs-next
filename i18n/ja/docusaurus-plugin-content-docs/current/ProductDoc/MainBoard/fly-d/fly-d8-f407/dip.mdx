# ボードに搭載された周辺機器の紹介

* 右側にスイッチを倒すと、Type-Cポート隣のUSBポートが開きます。
* 左側にスイッチを倒すと、Type-Cポート隣のRS232ポートが開きます。

## USBポートの説明

* カメラやマザーボードなどのデバイスに外部接続できます。
