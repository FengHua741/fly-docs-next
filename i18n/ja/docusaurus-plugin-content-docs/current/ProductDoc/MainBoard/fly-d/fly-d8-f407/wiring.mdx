## メインボード接線

:::warning 注意事項
由於画像の解像度が高い為、ウェブページ上で閲覧すると見えづらいです。マウスカーソルを接線図の上に置き、右クリックして「画像を別名保存」でデスクトップに保存してからご覧ください！
:::

:::warning 警告
購入した完成品の「3線式リミットスイッチ」は使用しないでください。このようなリミットスイッチは、メインボードにショートを引き起こし、メインボードを破損させる可能性があります！！！接線図に示されている接線方法に従って接線してください！！！
:::
