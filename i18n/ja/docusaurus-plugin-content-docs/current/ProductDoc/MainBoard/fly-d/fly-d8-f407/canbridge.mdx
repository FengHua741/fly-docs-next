# CANブリッジ

CANバスは、リアルタイムアプリケーション用のシリアル通信プロトコルバスで、シールドツイストペアを使用して信号を伝送し、世界で最も広く使用されているフィールドバスの1つです。CANプロトコルは、高価で嵩張る配線ハーネスを置き換えるために、自動車内のさまざまな要素間の通信に使用されます。このプロトコルの堅牢性により、他の自動化および産業アプリケーションにも使用されるようになりました。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、およびエラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用して、プリントヘッドがメインボードに接続するためのワイヤーの数を減らします。元々は数十本の線が必要でしたが、CANを使用することで、4本の線のみで済むようになり、大幅にワイヤーの数を減らし、配線の難易度も下げました。
本章では、CANブリッジの使用について簡単に概覬します。

## ツールボードはCANでメインボードに接続します

* ツールボードの電源線はCANポートに接続するのではなく、12V-24V電源ラインに接続することに注意してください。

* 120Ωジャンパを接続することを忘れないでください
