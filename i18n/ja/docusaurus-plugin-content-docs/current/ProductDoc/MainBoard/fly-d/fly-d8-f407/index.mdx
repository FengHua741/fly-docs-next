# 概要

> Mellow Fly-D8は広州メイレン電子科技有限公司のFly3Dメインボード開発チームが発売した高性能3Dプリンターメインボードで、最大8つのステッピングモーターをサポートしています。

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/img/d8.webp').default} size="80%" align="center" />

## 製品特徴

* メインボードのリミットスイッチなどのインターフェースは5V電源で隔離保護されており、POWER電源に接続しない場合、電源は出力されません。
* 32ビットARM Cortex-M4シリーズの168MHz、STM32F407チップを使用しています。
* CANバスとRS232との接続をサポートし、データ伝送がより安定し、遅延が小さく、接続がより安定します。
* メインボードにはBOOTとRESETボタンが予留されており、ユーザーはUSB経由でDFUモードに進み、ファームウェアを更新することができます。
* 電源インターフェースには逆接防止保護があり、DIY時に電源ケーブルを逆接すると基盤が破損するのを防ぎます。
* 2つのUSB2.0インターフェースを提供し、より多くの接続可能性を拡張しています。そのうち1つのUSBインターフェースはスイッチによってRS232インターフェースに切り替えることができます。

# 製品仕様

* 主制御チップ：32ビットARM Cortex-M4シリーズの168MHz、STM32F407ZGT6チップ
* ファームウェア：Marlin 2.0 / Klipper
* ドライバーモードサポート：TMC：UART / SPI
* 8つのドライバーサポート：A4988、LV8729、DRV8225、TMC2208 / 2209 / 5160など
* 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
* 自動ベッドレベルセンサーをサポート：BLTouch、PL08Nなどの近接センサー、Klicky、Voron Tapなど
* 入力電圧：12V / 24Vをサポート
* 6つのリミットスイッチインターフェースがあり、正確な範囲制限と位置制御を実現できます
* 2つのRGBインターフェースがあり、多彩な視覚効果を提供できます
* 3つのヒータータブインターフェースがあり、エクストルーダーとヒートベッドを独立して制御できます
* 4つのヒータータンパーキャリアインターフェースがあり、最大4つの温度グループを同時にモニターし、正確な温度制御を実現できます
* USBインターフェース：USB2.0インターフェースを2つサポートし、そのうち1つのUSBインターフェースはスイッチによってRS232インターフェースに切り替えることができます
* 2つのUSB2.0インターフェースを提供し、より多くの接続可能性を拡張しています。そのうち1つのUSBインターフェースはスイッチによってRS232インターフェースに切り替えることができます
