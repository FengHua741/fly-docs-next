# CANブリッジ

CANバスは、リアルタイムアプリケーションのためのシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送できます。これは世界で最も広く利用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価でかさばる配線ハーネスの代わりになります。このプロトコルの堅牢性により、他の自動化や産業アプリケーションにも使用されるようになりました。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポートの提供、最大1Mb/sの転送レート、11ビットのアドレス指定、エラーチェック機能が含まれます。
3Dプリンタでは、CANバスを使用してプリントヘッドとマザーボードの間に接続するワイヤーの数を減らします。元々は数十本の線が必要でしたが、CANを使用することで、4本の線のみを使用することが可能になり、大幅にワイヤーの数を減らし、配線の難易度を下げることができます。
この章では、CANブリッジの使用について簡単に説明します。

## ツールボードはCANを使ってマザーボードと接続します

* ツールボードの電源ラインはCANポートにつながれていないことに注意してください。12V-24V電源ラインにつながれています

* 120Ωジャンパーピンを接続することに注意してください

## マザーボードのファームウェアを焼く

* D5マザーボードに焼くファームウェアは`USBブリッジCANファームウェア設定`であることを確認してください
