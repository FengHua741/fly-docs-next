# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip
    * D5のkatapultファームウェアは第3の接続方法を使用しています：USB接続
    * D5は2024年9月1日に出荷時に既にkatapultファームウェアがプリフラッシュされています
    :::

## DFU焼付けモードに入る

    :::tip

    左側が`BOOTボタン`、右側が`RESETボタン`です

    :::

    * 焼付けモードには2つの方法があります
    * 方法1：マザーボードが完全に電源オフになっている状態で、BOOTボタンを押したままデータ転送用のケーブルをマザーボードとデバイスの間に接続します
    * 方法2：データ転送用のケーブルをマザーボードとデバイスの間に接続した後、BOOTボタンを押したままRSTボタンを押し、その後RSTボタンを離し最後にBOOTボタンも離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/img/boot.webp').default} size="50%" align="center" />
