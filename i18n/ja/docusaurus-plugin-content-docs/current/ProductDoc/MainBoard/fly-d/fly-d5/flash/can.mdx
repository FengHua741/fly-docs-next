# USBブリッジCANファームウェアの焼込

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` キーを押して、**Save configuration** が表示されたら、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 現在は設定が保存され、コマンドラインに戻っています
    * 下記のコマンドを入力してコンパイルを開始してください（時間がかかります）
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下のような出力が表示されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## ファームウェア焼込モードに入る

    :::tip 注意
        2024年9月1日以降に出荷されたD5には既にKatapultファームウェアが刷られています
    :::
    * 主基板を上位機器にTypeCケーブルで接続し、**resetボタンをダブルクリック**すると、主基板のLEDが点滅します。
    :::warning 警告
       * LEDが点滅しない場合、Katapultファームウェアの再焼込みを行ってください：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#進入dfu燒錄模式">Katapultファームウェアの焼込</Button>
       * LEDが点滅しない場合は、Katapultファームウェアの焼込後に再度この手順を実行してください
    :::
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/img/d5_dfu.webp').default} size="100%" align="center" />
