# USBブリッジCANファームウェアの焼込

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` キーを押して、**Save configuration** が表示されたら、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻っているはずです
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## フラッシュモードに入る

    :::tip ヒント
        2024年9月1日以降に出荷されたD5には既にKatapultファームウェアが焼込まれています
    :::
    * USB Type-Cケーブルを使用してマザーボードをホストコンピュータに接続し、**resetボタンをダブルクリック**します。このときマザーボード上のLEDが点滅します。
    :::warning 警告
       * LEDが点滅しない場合、Katapultファームウェアを再焼込みしてください:<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#進入dfu燒錄模式">Katapultファームウェアを焼込む</Button>
       * LEDが点滅しない場合は、Katapultファームウェアの焼込み後、再度実行してください
    :::
