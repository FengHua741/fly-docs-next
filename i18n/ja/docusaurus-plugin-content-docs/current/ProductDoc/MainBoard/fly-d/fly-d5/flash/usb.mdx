# USBファームウェアのコンパイル


## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押して、**Save configuration** が表示されたら、その後に `Y` キーを押す
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * これで設定が保存され、コマンドラインインターフェースに戻るはずです
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に次の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## ブートローダーへの移行

    :::tip ヒント
        2024年9月1日以降に出荷されたD5には既にKatapultファームウェアが焼付けてあります
    :::
    * 主基板をホストマシンにTypeCケーブルで接続し、**resetボタンをダブルクリック**します。この時、主基板上のLEDが点滅します。
    :::warning 警告
       * LEDが点滅しない場合は、Katapultファームウェアを再焼付けしてください：<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#進入dfu燒錄模式">Katapultファームウェアの焼付け</Button>
       * LEDが点滅しない場合、Katapultファームウェアの焼付け後、再度実行してください
    :::
