# はじめに

> Mellow Fly-D5は広州メイレン電子科技有限公司Fly3Dメインボード開発チームが発表した高性能3Dプリンターメインボードで、最大5つのステッピングモーターをサポートしています。

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/d5.webp').default} size="60%" align="center" />

## **1. 製品特長**

* 32ビット48MHzのARM Cortex-M0+シリーズSTM32F072RBT6メインコントローラーチップを使用
* CANバス接続をサポートし、データ伝送がより安定し、遅延が小さく、接続がより安定します
* ドライバーサポート:A4988、LV8729、DRV8225、TMC2208/2209/5160など
* 板載DIAG機能ピンを搭載しており、単純なジャンパーキャップの挿抜だけで無限位リミットスイッチサポートを使用可能にする

## 2. 製品仕様

* MCU: STM32F072RBT6
* ファームウェア: Klipper/Marlin
* 12-24V DC電源をサポートし、電源がより安定し、降圧モジュールを省きます
* ドライバーワークモードサポート: SPI、UART、STEP/DIR
* サポートドライバー: 外付けTMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988など
* 拡張インターフェース: BLTouch（Servos、Probe）、UART X1、12864 X1
* ファン: 2線制御可能なファン X2、並列ファン X2、常時オンファン X2
* 温度センサー: 2つのNTC 100K
* ヒーターアダプター: 2つの制御可能なインターフェースをサポートし、10Aまで対応
* サポートするマシン構造: Cartesian、Delta、Kossel、Ultimaker、CoreXY
