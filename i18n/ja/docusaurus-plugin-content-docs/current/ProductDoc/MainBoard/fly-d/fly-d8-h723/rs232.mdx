# RS232接続

RS232標準インターフェース（別名EIA RS-232）は、一般的なシリアル通信インターフェースの一つで、様々な機器間のデータ転送に広く使用されています。その安定性と信頼性により、多くの機器の通信手段として選ばれています。このプロトコルは、通常の5Vとは異なる信号レベルを使用してデータを伝送し、信号干渉を最小限に抑えることができます。3Dプリンターシステムでは、RS232はCANよりも高いデータ転送レート、より良い安定性、簡単な設定、および上位機との互換性が良いため、選択されます。

## ツールボードはRS232でメインボードと接続する

    * ツールボードの電源線はRS232信号線ではなく、12V-24V電源線上に接続することに注意してください

    * D8メインボードファームウェアは再コンパイルする必要はありません
