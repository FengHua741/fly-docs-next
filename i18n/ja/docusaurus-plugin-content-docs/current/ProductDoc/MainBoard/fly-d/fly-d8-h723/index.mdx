# 概要

> Mellow Fly-D8は広州メイレン電子科技有限公司のFly3Dマザーボード開発チームが発表した高性能3Dプリンターマザーボードで、最大8つのステッピングモーターをサポートしています。

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-h723/img/d8.webp').default} size="80%" align="center" />

## 製品特長

    * マザーボードのリミットスイッチなどのインターフェースは5Vで電源供給され、POWER電源に接続しない場合は電源が出力されません
    * 32ビット、550MHzのARM Cortex-M7シリーズSTM32H723コントローラチップを採用しています
    * CANバスとRS232との接続をサポートし、データ転送がより安定し、遅延が小さくなり、接続がより安定します
    * マザーボードにはBOOTとRESETボタンが予備されており、ユーザーはUSB経由でDFUモードに入りファームウェアを更新できます
    * 電源インターフェースには逆接防止保護があり、DIY時に電源線を逆接してボードが破損するのを防ぎます
    * 2つのUSB2.0インターフェースを提供し、さらに多くの接続可能性を拡張しています。そのうち1つのUSBインターフェースはスイッチを切り替えることでRS232インターフェースとして機能し、さまざまな機器との接続ニーズに対応します

# 製品仕様

    * メインコントローラチップ：32ビット、550MHzのARM Cortex-M7シリーズSTM32H723コントローラチップ
    * ファームウェア：Marlin 2.0/Klipper
    * ドライバーモードサポート：TMC：UART/SPI
    * 8つのドライバーサポート：A4988、LV8729、DRV8225、TMC2208/2209/5160など
    * 表示：シリアルタッチスクリーン、mini12864 LCD、12864 LCD、2004 LCD、FLY 4.3/7.0 V1
    * オートベッドレベルセンサー対応：BLTouch、PL08Nなど近接センサー、Klicky、Voron Tapなど
    * 入力電圧：12V/24Vをサポート
    * 6つのリミットスイッチインターフェースがあり、正確な範囲制限と位置制御を実現できます
    * 2つのRGBインターフェースがあり、多彩な視覚効果を提供できます
    * 3つのヒーターチューブインターフェースがあり、エクストルーダーとヒートベッドの独立制御を提供します
    * 4つのサーマルセンサーアンプインターフェースがあり、最大4組の温度を同時に監視し、正確な温度制御を実現します
    * USBインターフェース：USB2.0インターフェースを2つサポートし、そのうち1つのUSBインターフェースはスイッチを切り替えてRS232インターフェースとして機能します
