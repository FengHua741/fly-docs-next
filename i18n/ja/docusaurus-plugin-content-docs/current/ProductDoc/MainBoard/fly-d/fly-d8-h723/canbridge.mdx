# CANブリッジ

CANバスは、リアルタイムアプリケーション用のシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送でき、世界で最も広く使用されているフィールドバスの1つです。CANプロトコルは、車両内のさまざまな要素間の通信に使用され、高価で嵩張る配線ハーネスの代わりになります。このプロトコルの堅牢性により、他の自動化や産業応用にも拡張されています。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス設定、エラーチェック機能が含まれます。
3Dプリンターでは、CANバスを使用して、プリントヘッドがメインボードに接続するための配線数を減らします。元々は十数本の線が必要でしたが、CANを使うことで、必要な線は4本だけになりました。これにより、配線の数が大幅に削減され、配線の難易度も下がります。
本章では、CANブリッジの使用について簡単に概説します。

## ツールボードはCANでメインボードに接続します

    * ツールボードの電源線はCANポートに接続するのではなく、12V-24V電源ラインに接続することに注意してください

    * 120Ωジャンパーピンを接続することを忘れないでください
