# CANブリッジ

CANバスは、リアルタイムアプリケーションのためのシリアル通信プロトコルバスで、シールドツイストペア線を使用して信号を伝送することができ、世界で最も広く利用されているフィールドバスの一つです。CANプロトコルは、自動車内のさまざまな要素間の通信に使用され、高価でかさばるワイヤーハーネスを置き換えるために使用されます。このプロトコルの堅牢性により、他の自動化および産業アプリケーションにも使用されるようになりました。CANプロトコルの特徴には、完全なシリアルデータ通信、リアルタイムサポート、最大1Mb/sの転送レート、11ビットのアドレス指定、エラーチェック機能などがあります。
3Dプリンターでは、CANバスを使用してプリントヘッドをメインボードに接続する配線の数を減らします。元々は十数本の線が必要でしたが、CANを使用することで、4本の線だけで済み、配線の数が大幅に減少し、配線の難易度も下がります。
本章では、CANのブリッジ使用について簡単に概説します。

## ツールボードはCANを使ってメインボードと接続します

* ツールボードの電源ラインはCANポートに接続されるのではなく、12V-24V電源ラインに接続されることに注意してください
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/wiring/canbridge.webp').default} size="80%" align="center" />
* 120Ωジャンパを接続することに注意してください

## メインボードのファームウェアを焼く

* DP5メインボードに焼くファームウェアは`USBブリッジCANファームウェア設定`であることを確認してください
