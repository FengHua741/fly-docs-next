# マザーボードの注意事項

## 電源インジケーター
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/power.webp').default} size="80%" align="center" />
    * TYPE-Cで電源を供給し、他のマザーボード電源に接続していない場合、少なくとも「3.3V」と「5V」の2つのランプが点灯します
    * マザーボード電源に接続した後は、「3.3V」、「5V」、「VCC」のランプが点灯するはずです

### 問題のトラブルシューティング


    :::danger 注意点

    * すべての外部機器（ドライバー、リミットスイッチ、12864など）が接続されていない場合
    * TYPE-Cで電源を供給しているだけで、少なくとも「3.3V」と「5V」の2つのLEDが常に点灯します

    :::

    * ランプが一つも点灯しない場合、ショート回路が発生しており、カスタマーサポートに連絡する必要があります
    * 「3.3V」、「5V」のいずれかのランプが点滅している場合、ショート回路が発生しており、カスタマーサポートに連絡する必要があります

## ドライバー

    * マザーボードは外部ドライバーをサポートしています     <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/ModuleDrive/servo-drive">配線と設定チュートリアル</Button>

## リミットスイッチ口

    * 接続順序に注意してください。3線リミットスイッチの場合、「5V」、「GND」、「S」が間違っているとマザーボードが損傷する可能性があります
    * 2線の場合、「S」および「GND」に直接接続するだけです
    * PL08および他の近接スイッチは、信号口にダイオードを外部接続して電圧を3.3V未満に下げなければなりません
