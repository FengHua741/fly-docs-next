# 簡介

> Mellow Fly-DP5は広州鎂倫電子科技有限公司Fly3Dメインボード開発チームが提供する高性能3Dプリンターメインボードで、最大5つのステッピングモーターをサポートしています。


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/dp5.webp').default} size="60%" align="center" />

## **1. 製品特長**

* 32ビットのARM Cortex-M0+シリーズSTM32F072RBT6メインコントローラーチップを使用し、クロック周波数は48MHzです。
* CANバス接続をサポートし、データ伝送がより安定し、遅延が少なく、接続がより安定します。
* ドライバーサポート：DP2209とDP560
* DIAG機能ピンを搭載しており、簡単なジャンパーキャップの抜き差しで無限位リセットサポートを使用できます。

## 2. 製品仕様

* MCU: STM32F072RBT6
* ファームウェア: Klipper/Marlin
* 12-24VのDC電源をサポートし、電源供給がより安定し、降圧モジュールが不要になります。
* ドライバーワークモードサポート：SPI、UART、STEP/DIR
* サポートドライバー：外付けTMC5160Pro、TMC5160、TMC2209、TMC2225、TMC2226、TMC2208、TMC2130、A4988などをサポート
* 拡張インターフェース：BLTouch（Servos、Probe）、UART X1、12864 X1
* ファン：2線制御可能なファン X2、並列ファン X2、常時オンのファン X2
* 温度センサー：2つのNTC 100K
* ヒーターアダプター：2つの制御可能なインタフェース、最大10Aまで対応
* 対応マシン構造：Cartesian、Delta、Kossel、Ultimaker、CoreXY
