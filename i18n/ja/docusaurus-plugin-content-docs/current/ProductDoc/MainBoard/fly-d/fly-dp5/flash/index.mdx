# 固件焼込み説明

:::danger 補足
* マザーボードのファームウェア焼込みにはKatapultを使用してKlipperを焼込む必要があります
* もしマザーボードが既にKatapultに焼込まれていることが確認できた場合、最後のKatapultファームウェアの焼込みは不要です。これはマザーボードのKatapultファームウェアが正常に焼込まれなかった場合の予防措置です
* リセットボタンを素早く2回押すと、マザーボード上のLEDが点滅するかどうかを確認できます。これが点滅する場合、マザーボードはKatapultファームウェアに入っていることを示します
:::

## どの種類のファームウェアが必要かを見分ける方法

    :::tip

    * 注意してください、あなたが焼込むべきファームウェアは一つだけです

    :::
    
    * DP5は以下の方法でホストコンピュータに接続することができます
    * <Button variant="contained" disableElevation href="./usb">USBファームウェア</Button>
        * 利点：通信が信頼でき、安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
    * <Button variant="contained" disableElevation href="./can">USBブリッジファームウェア</Button>
        * 利点：複数のデバイスに接続でき、USBポートを占有しません
        * 欠点：`KlipperScreen`と共に使用した場合、ホーム位置超時などのエラーが発生する可能性があります。
