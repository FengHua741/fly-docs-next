# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * D5はUSBで接続します

    :::

## フラッシュモードに入る

    :::tip

    左側が`BOOTボタン`、右側が`RESETボタン`です

    :::

    * フラッシュモードに入るには2つの方法があります
    * 方法1：マザーボードを完全に電源オフにしてから、BOOTボタンを押しながらデータ転送用のケーブルをマザーボードとデバイスの一方につなぎます
    * 方法2：データ転送用のケーブルをマザーボードとデバイスの一方につなぎ、BOOTボタンを押しながらRSTボタンを押し、その後RSTボタンを離し最後にBOOTボタンも離します
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/flash/img/boot.webp').default} size="50%" align="center" />
