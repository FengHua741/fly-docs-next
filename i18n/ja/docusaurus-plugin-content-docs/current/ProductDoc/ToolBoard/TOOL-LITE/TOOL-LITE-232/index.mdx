# 概要

# 1. 商品概要

FLY TOOL-LITE-232は広州メイレン電子科技有限公司が3Dプリンター向けに設計したツールボードです。このツールボードはXT30/KFインターフェースを提供し、従来のメインボードとヘッド間の複雑な配線を4本の線で置き換え、レイアウトを最適化します。TOOL-LITE-232は36/42ステップモーターに対応しており、RS232トランシーバー、TypeCポート、TMC2209ドライバー、LIS2DW加速度センサー、2つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperファームウェアに対応しています。

## 1.1 商品特長

    * Bootボタンが予約されています。ユーザーはTypeCを使ってブートモードに移行し、ファームウェアを更新することができます。
    * Klipperファームウェアに対応しています。
    * S232モードに対応しています。
    * RS232モードでは、より高い転送レートと強力な安定性があります。
    * 温度センサーは、ntc100kとPT1000温度センサーをサポートしています。
    * 2つの制御可能なファンをサポートし、最大出力電流は1Aです。
    * ボード内にLIS2DW加速度センサーが搭載されています。
    * 高圧プローブインターフェースが予約されており、ボード内に2209ドライバーが搭載されており、さまざまなレベルングセンサー、例えばBLtouch、klicky、Voron Tap、PL08nなどをサポートします。
    * 12-24V電圧、15A電流入力に対応し、ヒーターソケットを増大させ、定格電流は10Aです。

## 1.2 新機能

    * RS232モードが新たに追加され、より高い転送レートと強力な安定性があり、デバッグも簡単になります。
    * 1メートルの電源/信号ケーブルが新たに追加され、ケーブルはツイストペア設計を使用し、干渉に対する耐性が向上しています。

## 1.3 商品仕様

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インターフェースにはXT30(2+2)コネクタが使用され、15Aの電流をサポートし、ピーク値は30Aです。
    * ファームウェア：Klipper
    * 入力電圧：DC12V-24V
    * 論理電圧：DC3.3V
    * ボード内センサー：LIS2DW
    * ヒーターソケット：ヒーターソケット（HEAT0）、10A
    * 拡張インターフェース：高圧プローブインターフェース、EndStop、PT100/PT1000、XH2.54-2P CAN拡張ポート
    * ファンインターフェース：2つのDC制御可能なファン
    * モータードライバー：ボード内TMC2209
    * 温度センサー：100K NTCまたはPT1000（TH0）
