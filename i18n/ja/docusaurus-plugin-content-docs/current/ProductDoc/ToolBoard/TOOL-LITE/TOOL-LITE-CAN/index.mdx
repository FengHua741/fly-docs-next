# はじめに

# 1. 製品の紹介

FLY TOOL-LITE-CANは広州メイレン電子科技有限公司が設計した3Dプリンタ用のツールボードで、このツールボードにはXT30/KFインターフェイスがあり、従来のメインボードとホットエンド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。TOOL-LITE-CANは36/42ステップモーターに対応しており、CANBusトランシーバー、TypeCポート、TMC2209ドライバー、LIS2DW加速度センサー、2つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperファームウェアに適しています。

## 1.1 製品特長

    * Bootボタンが予め設置されており、ユーザーはTypeC経由でBootモードに移行してファームウェアを更新することができます
    * Klipperファームウェアをサポートしています
    * CANBusモードをサポートしています
    * CANBusモードでは、より高い転送レートと強力な安定性を持っています
    * 温度センサーのサポート：ntc100kおよびPT1000温度センサー
    * 2つの制御可能なファンをサポートし、最大出力電流は1Aです
    * ボード内にLIS2DW加速度センサーが搭載されています
    * 高圧プローブインターフェイスが予め設置され、ボード内に2209ドライバーが搭載されており、BLtouch、klicky、Voron Tap、PL08nなど、さまざまなレベルセンサーをサポートします
    * 12-24V電圧、15A電流入力をサポートし、ヒーターソケットを増大させ、定格電流は10Aです

## 1.2 新機能

    * 1m電源/信号接続線を新たに追加しました。接続線はツイストペア設計を使用し、干渉に対する耐性が向上しています

## 1.3 製品仕様

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インターフェイスにはXT30(2+2)コネクターを使用し、15Aの電流をサポートし、ピーク値は30Aです
    * ファームウェア：Klipper
    * 入力電圧：DC12V-24V
    * ロジック電圧：DC3.3V
    * ボード内センサー：LIS2DW
    * ヒーターソケット：ヒーター（HEAT0）、10A
    * 拡張インターフェイス：高圧プローブインターフェイス、EndStop、PT100/PT1000、XH2.54-2P CAN拡張ポート
    * ファンインターフェイス：2つのDC制御可能なファン
    * モータードライバー：ボード内TMC2209
    * 温度センサー：100K NTCまたはPT1000（TH0）
