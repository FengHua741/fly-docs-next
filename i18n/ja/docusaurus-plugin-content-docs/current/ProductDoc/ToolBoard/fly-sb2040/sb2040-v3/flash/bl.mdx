# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * SB2040_V3はCANとRS232をサポートしています
    * ここではCANファームウェアを提供します。500Kと1Mの速度があります
    :::

## ブートローダーモードに入る

    :::tip

    左が`リセットボタン`、右が`ブートボタン`です

    :::

    * ブートローダーモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源オフになっている場合、ブートボタンを押しながらデータ転送用のケーブルをマザーボードとデバイスに接続します
    * 方法2：データ転送用のケーブルをマザーボードとデバイスに接続した後、ブートボタンを押しながらリセットボタンを押し、その後リセットボタンを離し、最後にブートボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.webp').default} size="100%" align="center" />

            4. ファームウェアを焼く

            :::tip

            どちらか一方のファームウェアを焼くだけです

            * 以下の500K CANレートのファームウェア書き込みコマンド
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040V3_CANBOOT_CAN_500K.uf2
                ```
                       
            * 以下の1M CANレートのファームウェア書き込みコマンド
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040V3_CANBOOT_CAN_1M.uf2
                ```


            5. 以下のメッセージが表示されたら焼込み成功です

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            焼き込みが完了したら、マザーボードを完全に電源オフにする必要があります。
