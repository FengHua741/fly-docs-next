# CANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * **GPIOピンをマイクロコントローラー起動時に設定する**を選択し、入力：**!gpio5**
    :::warning
        `!gpio5`を入力する際は英数入力状態で行ってください。
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/can.webp').default} size="100%" align="center" />
    * `Q`キーを押して「Save configuration」が表示されたら、その後に`Y`キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェイスに戻っているはずです
    * `make -j4`を入力してコンパイルを開始します。時間がかかる場合があります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## ファームウェアの焼込

    * ファームウェアを焼込む前に、XT30データ線を使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードに接続します。下記は参照接続図です
    * ダイアルスイッチが正しい位置にあることを確認してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/sb2040v3_utoc.webp').default} size="80%" align="center" />
    * CAN HとCAN Lが電源オフ状態で約`60Ω`の抵抗値を測定することを確認してください。`120Ω`の場合、ダイアルスイッチをオンにしてください。`40Ω`前後の場合はダイアルスイッチをオフにしてください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.webp').default} size="100%" align="center" />
