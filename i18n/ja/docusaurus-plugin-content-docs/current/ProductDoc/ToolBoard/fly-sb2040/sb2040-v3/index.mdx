# 1. 製品概要

    FLY SB2040 V3は広州メイロン電子科技有限公司がVORON Stealthburberプリントヘッド向けに開発したツールボードです。このツールボードを使用することで、従来のメインボードとプリントヘッド間の煩雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SB2040はstealthburnerとafterburnerに適しており、CANトランシーバー、RS232トランシーバー、USBポート、TMC2209ドライバー、LIS2DW加速度センサー、3つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperとRRFファームウェアに対応しています。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/img/sb2040.webp').default} size="100%" align="center" />


## 1.1 製品特長

    * Bootボタンを予備として搭載し、ユーザーはTypeCを介してBootモードでファームウェアを更新することができます
    * Klipperファームウェア、RRFファームウェアをサポート
    * CANモード、RS232モード、RRF CANモードをサポート
    * RS232モードでは、より高い転送速度と安定性を提供します。CANモードでは、CAN拡張をサポートし、CAN拡張インターフェースを予備としています
    * 温度センサー：板載MAX31865チップ、PT100およびPT1000温度センサーと互換性があります
    * 板載LIS2DW加速度センサーとLDC1612センサー
    * RGBインターフェースを予備として搭載し、板載2209ドライバーにより、さまざまなレベルセンサー（例：klicky、Voron Tap、PL08nなど）をサポート
    * 12-24V電圧、15A電流入力に対応し、ヒーターコネクタを増大させ、定格電流は10Aです

## 1.2 新機能

    * Klipper CANモード、RS232モード、RRF CANモードを切り替えることが可能です
    * RS232モードを新規追加し、より高い転送速度と安定性を提供し、デバッグも簡単になります
    * RRF CANモードを新規追加し、RRFツールボードとして切り替えることができます。適用範囲を広げることができます
    * CAN拡張インターフェースを新規追加し、CANデバイスの拡張が容易になります
    * 終端抵抗ダイアルスイッチを新規追加し、終端抵抗120Rを選択できるようにし、ジャンパーピンは不要です
    * 1メートルの電源/シグナル接続ケーブルを新規追加し、ケーブルはツイストペア設計を使用し、干渉に対する耐性が向上します

## 1.3 製品仕様


    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インタフェースはXT30(2+2)コネクタを使用し、15Aの電流に対応し、ピーク値は30Aです
    * ファームウェア：Reprap/Klipper
    * 入力電圧：DC12V-24V
    * ロジック電圧：DC3.3V
    * 板載センサー：LIS2DWとオプションのldc1612およびmax31865
    * ヒーターコネクタ：ヒーター（HEAT0）
    * 拡張インタフェース：RGB、EndStop、PT100/PT1000、XH2.54-2P CANポート
    * ファンインタフェース：1つのDCファン、2つのDCファンまたはPWMファン
    * モータードライバー：板載TMC2209
    * 温度センサー：100K NTCまたはPT1000（TH0）、PT100/PT1000を選択可能
