# Eddy 使用

    :::info
  * エディの設定が以前に構成されている場合は削除してください
  * Eddyをインストールする際は、コイルがノズルの上にあり、ノズルから0.6〜0.8cm離れている必要があります
  * テスト時はヒートベッドを開けないでください
  * 使用前に`probe`設定を削除してください
    :::


    :::warning 注意
    * マルチZ軸マシンは一度手動でレベルを合わせる必要があります
    :::

## 使用説明
   * SSHツールを使用してIPアドレスで端末にログインし、スクリプトを実行します
    必要なSSHツールをダウンロード <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button>


<Tabs>
  <TabItem value="eddy1" label="国際使用">
    # 国際は以下のコマンドを使用してください
    ### 以下からeddy自動化スクリプトのインストールが始まります
    * ダウンロードしたSSHツールを開き、以下のコマンドを入力してエンターキーを押すとコピーできます
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * `fluidd`ウェブページの`moonraker.conf`設定ファイルに以下の設定を追加して、今後の更新に便利です
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * 更新された場合、以下のコマンドを実行してください
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* 実行すると、以下の画像のようにパスワードを入力するよう表示されます

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* 以下の画像が表示されたら、対応するプリンターの番号を入力してください。中国語と英語の選択肢があります。入力後、エンターキーを押してください
* `3z`を選択して、`printer.cfg`に`z_tilt`という設定があることを確認してください/`4z`を選択して、`printer.cfg`に`quad_gantry_level`という設定があることを確認してください。独立したデュアルZを選択する場合は、`3z`を選択してください [設定例](https://mellow.klipper.cn/docs/category/klipper-%E5%8F%82%E8%80%83%E9%85%8D%E7%BD%AE)
