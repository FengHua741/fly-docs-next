# Eddyの使用

    :::info

    * ベッドスキャンを使用する際は、個別のZリミットを使用することをお勧めします。ベッドスキャンをZリミットとして使用しないでください。
    * Eddyを設置する際、底部はノズルより1〜2mm上にする必要があります。
    * Eddyの周波数を校正する際は、コイルPCBはヒートベッドから少なくとも5mm離しておく必要があります。
    * ベッド全体を常温でテストしてください。テスト中はヒートベッドを開かないでください。

    :::

## 参照設定と使用説明

### ldc1612のデフォルト周波数変更
    この方法はsht36-v3/SB2040-v3またはSB2040-Pro-v3でのみ使用可能です。
    Mobaxtermツールを使用してコマンドを実行できます。

    必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button>をダウンロードしてください。

    
    [詳細な使用方法はこちらをご覧ください](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    最初のコマンドを実行してEnterキーを押してください。

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    二つ目のコマンドを実行してEnterキーを押してください。
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    置換が成功すると、`LDC1612_FREQ = 40000000`を含む行が返されます。成功したことを示します。返事がない場合は置換が失敗しています。
    または、以下のコマンドを使用して成功したかどうかを確認できます。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   温かいアドバイス：変更後、以下のコマンドを使用してKlipperサービスを再起動し、Eddyの周波数を再校正してください。
   ```cfg
   systemctl restart klipper
   ```
   Eddyの校正周波数コマンド
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### ベッドスキャンの設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 現在、外部の1612モジュールがリリースされていないため、この設定は一時的に使用できません。
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補償設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh設定

    * [bed_mesh]内の`horizontal_move_z`パラメータは、ベッドスキャン中にEddyができるだけベッドに近づくように、2に設定することをお勧めします。

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
