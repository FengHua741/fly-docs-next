# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * SB2040_PRO_V3 はCANとRS232をサポートしています
    * ここではCANファームウェアのみを提供します。速度は500Kと1Mがあります
    :::

## ブートロードモードに入る

    :::tip

    左が`リセットボタン`、右が`ブートボタン`です

    :::

    * ブートロードモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源を切っている状態で、ブートボタンを押したままデータ転送用のケーブルの一方をマザーボードに、もう一方をデバイスに接続します
    * 方法2：データ転送用のケーブルの一方をマザーボードに、もう一方をデバイスに接続した後、ブートボタンを押したままリセットボタンを押し、その後リセットボタンを離し、最後にブートボタンも離します
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />

