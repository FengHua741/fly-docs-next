# ドライバアンフェード解決方法

## 方法一

    :::danger

    * 電源を交換する
    * 電源フィルターを追加する
    * モータの判定を交換する
    * モータの外装をツールボードのGNDに接続する
    * クリッピングドライバ構成に`driver_TPFD: 0`を追加する

    :::

## 方法二
    * SSHを使って上位機器に接続する
    * 下記のコマンドを入力する
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Klipperの更新後、このステップが復元される可能性があります
    :::
