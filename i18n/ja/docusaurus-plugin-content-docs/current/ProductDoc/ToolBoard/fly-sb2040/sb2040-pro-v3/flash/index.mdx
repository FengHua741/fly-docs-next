# ファームウェア焼付け説明

    * SB2040 PRO V3はRS232およびCANの2つの接続方法を使用してホストコンピュータに接続できます
    * RS232およびCANは`XT30(2+2)-F`を使用して接続する必要があります
    * 以下の`XT30(2+2)-F`データ線のスタイルです
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* マザーボードのファームウェア焼付けにはKatapultを介してKlipperを焼付けます
* マザーボードがすでにKatapultを焼付け済みであることが確認された場合は、最後のKatapultファームウェア焼付けを行う必要はありません。それはマザーボードのKatapultファームウェア焼付けが失敗した場合の予防策です
* マザーボード上のLEDが2回連続でリセットボタンを押すことで点灯するか確認してください。もしそのLEDが点灯すれば、マザーボードはKatapultファームウェアに入っています
:::

## どのファームウェアが必要かをどのように判断するか

    * SB2040 PRO V3は以下の方法でホストコンピュータに接続できます
    * <Button variant="contained" disableElevation href="./can">CANファームウェア焼付け</Button>
        * 利点：`Katapult`ファームウェアを焼付けた場合、マザーボードに触れずにファームウェアを更新でき、バス接続をサポートします
        * 欠点：`KlipperScreen`と一緒に使用すると、リターンタイムエラーなどの問題が発生する可能性があります
        * ヒント：CANを使用する場合は、ホストコンピュータがCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアが焼付けられたマザーボードが必要です
    * <Button variant="contained" disableElevation href="./rs232">RS232ファームウェア焼付け</Button>
        * 利点：特別なソフトウェアなしで使用でき、通信が信頼性があり安定しています
        * 欠点：ファームウェアを再焼付けするには、手動DFUモードに移行するか、RSTボタンをダブルクリックして`Katapult`に入らなければなりません
        * ヒント: G2TまたはUTORが必要です
