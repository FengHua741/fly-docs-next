# メインボードIDの読み取りと確認

    訳注を加えずにウェブページを開く際は設定を変更しないでください

    :::

    訳注を加えずにウェブページを開く際は設定を変更しないでください

    :::danger 注意

    訳注を加えずにウェブページを開く際は設定を変更しないでください

    :::

<Tabs>
    <TabItem value="can" label="CANBus UUIDの取得">

    * IDを検索する前に、XT30データ線を使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードに接続します。以下の図は参考接続図です
    * ダイアルスイッチが正しい位置にあることを確認してください
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lの抵抗値が電源オフ状態で約`60Ω`であることを確認してください。もし`120Ω`であればダイアルスイッチをオンにしてください。もし`40Ω`前後であればダイアルスイッチをオフにしてください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.webp').default} size="100%" align="center" />
        <GetCanbusUUID />

    </TabItem>

    * ダイアルスイッチが正しい位置にあることを確認してください
