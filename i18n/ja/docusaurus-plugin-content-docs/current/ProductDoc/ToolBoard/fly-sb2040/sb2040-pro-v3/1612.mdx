# Eddyの使用方法

    :::info

    * 注意：スキャンベッドの使用時は、Zリミットスイッチとしてスキャンベッドを使用しないでください。専用のZリミットスイッチを使用してください。
    * Eddyの設置時、ノズルからの高さは少なくとも1-2mm以上離してください。
    * Eddyの周波数を校正する際は、コイルPCBはヒートベッドから最低5mm離してください。
    * ヒートベッドのテストは常温で行い、テスト中にヒートベッドをオンにしないでください。

    :::

## 参照設定と使用説明

### ldc1612のデフォルト周波数変更
    この方法はsht36-v3/SB2040-v3またはSB2040-Pro-v3にのみ適用されます。
    Mobaxtermツールを使用してコマンドを実行できます。

    必要な <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button> をダウンロード

    [詳細な使用方法はこちら](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    最初のコマンドを実行してEnterキーを押します。

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    2つ目のコマンドを実行してEnterキーを押します。
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    置換が成功した場合、`LDC1612_FREQ = 40000000`を含む行が返されます。置換が成功しなかった場合、何も返されません。
    またはこのコマンドを使用して成功したかどうかを確認します。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   温かいヒント：変更後、Klipperサービスを再起動し、再度Eddyの周波数を校正してください。
   ```cfg
   systemctl restart klipper
   ```
   Eddyの周波数校正コマンド
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### スキャンベッド設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 注意：現在、外部1612モジュールは発売されていないため、この設定は一時的に使用できません。
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の`horizontal_move_z`パラメータは、Eddyがプラットフォームにできるだけ近づくように、2に設定することをお勧めします。

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
