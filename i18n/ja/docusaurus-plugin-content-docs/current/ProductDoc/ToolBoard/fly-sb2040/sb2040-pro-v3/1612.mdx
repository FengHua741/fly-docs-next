# Eddy 使用

    :::info
  * エディーの設定が以前にあれば削除してください
  * エディーをインストールするとき、コイルはノズルの上にあり、ノズルから0.6〜0.8cm離れていてください
  * テスト時にはヒートベッドを開けないでください
  * 使用前に「probe」設定を削除してください
    :::

    :::warning 注意
    * マルチZ軸マシンの場合、一度手動でレベルを合わせておいてください
    :::

    * ブランドはヒートベッドに取り付けてください

## 使用説明
   * SSHツールを使用してIPを介してターミナルにログインし、スクリプトを実行します
    SSHツールをダウンロードする <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button>


<Tabs>
  <TabItem value="eddy1" label="国際使用">
    # 国際は以下のコマンドを使用してください
    ### 以下からeddy自動化スクリプトのインストールを開始します
    * ダウンロードしたSSHツールを開き、以下のコマンドを入力してエンターを押すか、コピーをクリックしてください
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * `fluidd`ウェブページの`moonraker.conf`設定ファイルに以下の設定を追加し、今後のアップデートに便利です
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * アップデートした場合は、次のコマンドを実行してください
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* 実行すると、下図のように表示され、あなたのホストコンピュータのパスワードを入力してください

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* 下図のように表示されたら、「選択入力」を選択し、プリンターに対応する番号を入力し、エンターを押してください
* 「3z」を選択して「printer.cfg」に「z_tilt」の設定があることを確認する、または「4z」を選択して「printer.cfg」に「quad_gantry_level」の設定があることを確認する。「独立した二つのz」を選択する場合は、「3z」を選択してください [設定例](https://mellow.klipper.cn/docs/category/klipper-%E5%8F%82%E8%80%83%E9%85%8D%E7%BD%AE)
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy2.webp').default} size="100%" align="left" />

* 下図が表示された場合、あなたは「Y」を入力し、エンターを押す必要があります
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy4.webp').default} size="100%" align="left" />

* あなたが「sb2040-v3-PROV3」を持っている場合、yを入力し、エンターを押してください
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/img/eddy7.webp').default} size="100%" align="left" />

* 下図が表示された場合、成功しました
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy3.webp').default} size="70%" align="left" />
