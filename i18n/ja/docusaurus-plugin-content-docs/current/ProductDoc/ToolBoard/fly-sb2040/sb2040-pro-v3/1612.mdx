# Eddy の使用

    :::info
  * 配置eddyの設定があったら削除してください
  * Eddyをインストールするとき、ノズルの上の位置は少なくとも1-2mm以上である必要があります
  * テスト時はヒートベッドを開けないでください
  * 使用前に「probe」設定を削除してください
    :::

    :::warning 注意事項
    * マルチZ軸マシンは一度手動でレベルを合わせておいてください
    :::

## 使用説明
   * SSHツールを使ってIPでログインし、ターミナルでスクリプトを実行します
    必要なSSHツールをダウンロードします <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button>


<Tabs>
  <TabItem value="eddy1" label="国際使用">
    # 国際の場合、以下のコマンドを使用してください
    ### 以下からeddyの自動化スクリプトのインストールが始まります
    * ダウンロードしたSSHツールを開き、以下のコマンドを入力してEnterキーを押すとコピーできます
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * fluiddのウェブページのmoonraker.conf設定ファイルに以下の設定を追加することで、今後のアップデートが簡単になります
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * 更新したら、以下のコマンドを実行してください
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
