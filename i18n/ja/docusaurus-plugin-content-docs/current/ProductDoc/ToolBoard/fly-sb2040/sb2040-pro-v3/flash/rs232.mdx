# RS232 ファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <RS232 />

    * **GPIOピンをマイクロコントローラー起動時に設定する**を選択し、入力：**!gpio5**
    :::warning
        英語入力モードで `!gpio5` を入力してください
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻っているはずです
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に次の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## 書き込みモードに入る

    :::tip

    左側が `RESETボタン`、右側が `BOOTボタン` です

    :::

    * DFUモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源オフの状態で、BOOTボタンを押し続けながらデータ転送用ケーブルをマザーボードとホストPCに接続します
    * 方法2：データ転送用ケーブルをマザーボードとホストPCに接続した後、BOOTボタンを押し続けながらRSTボタンを押し、その後RSTボタンを離し最後にBOOTボタンも離します
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
