# RS232ファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <RS232 />

    * **GPIOピンをマイクロコントローラー起動時に設定する**を選択し、入力：**!gpio5,!gpio19,!gpio24**
    :::warning
        `!gpio5,!gpio19,!gpio24`を英数入力モードで入力してください。
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * `Q` キーを押して「Save configuration」が表示されたら、その後に `Y` キーを押す
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェースに戻ったはずです
    * 下記のコマンドを入力してコンパイルを開始します（時間がかかります）
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下のような出力が出ればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```

## フラッシュモードに入る

    :::tip

    左側が`RESETボタン`、右側が`BOOTボタン`です。

    :::

    * DFUモードに入るには2つの方法があります
    * 方法1：マザーボードが完全に電源オフの状態で、BOOTボタンを押したまま、データ転送用ケーブルをマザーボードと上位機器に接続する
    * 方法2：データ転送用ケーブルをマザーボードと上位機器に接続し、BOOTボタンを押したままRSTボタンを押して、その後RSTボタンを離し、最後にBOOTボタンも離す
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
