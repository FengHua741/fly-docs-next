# 1. 製品紹介

    FLY SB2040 PRO V3は、広州美倫電子科技有限公司がVORON Stealthburberプリントヘッド向けに開発したツールボードです。このツールボードを使用すると、従来のマザーボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SB2040はstealthburnerとafterburnerに対応しており、CANトランシーバー、RS232トランシーバー、USBポート、TMC2240ドライバー、LIS2DW加速度センサー、3つの制御可能なファンなど、他の機能も含むツールボードです。このツールボードはKlipperとRRFファームウェアに対応しています。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/img/sb2040.webp').default} size="100%" align="center" />


## 1.1 製品特長

    * Bootボタンを予約し、ユーザーはTypeC経由でBootモードに入り、ファームウェアを更新できます
    * Klipperファームウェア、RRFファームウェアに対応
    * CANモード、RS232モード、RRF CANモードに対応
    * RS232モードでは、より高い転送速度と安定性を提供します。CANモードでは、CAN拡張がサポートされ、CAN拡張インターフェースが予約されています
    * 温度センサーは、搭載されたMAX31865チップ、PT100およびPT1000温度センサーと互換性があります
    * LIS2DW加速度センサーとLDC1612センサーが搭載されています
    * RGBインターフェースを予約し、2240ドライバーが搭載されており、さまざまなレベルングセンサー、例えばklicky、Voron Tap、PL08nなどをサポートします
    * 12-24V電圧、15A電流入力に対応し、ヒータースタックポートを増大させ、定格電流は10Aです

## 1.2 新機能

    * Klipper CANモード、RS232モード、RRF CANモードを切り替えることができます
    * RS232モードが追加され、より高い転送速度と安定性を提供し、デバッグも簡単になります
    * RRF CANモードが追加され、RRFツールボードとして切り替えることができ、適用範囲が広がります
    * CAN拡張インターフェースが追加され、CANデバイスの拡張が容易になります
    * テリナ抵抗ダイアルスイッチが追加され、テリナ抵抗120Rを選択できるようになり、ジャンパーコードは不要です
    * 1メートルの電源/シグナル接続ケーブルが追加され、ケーブルは二重絞り設計で、干渉に対する耐性が向上しています

## 1.3 製品仕様


    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インターフェースはXT30(2+2)コネクターを使用し、15Aの電流、ピーク30Aに対応
    * ファームウェア：Reprap/Klipper
    * 入力電圧：DC12V-24V
    * ロジック電圧：DC3.3V
    * 搭載センサー：LIS2DWとオプションのldc1612とmax31865
    * ヒータースタックインターフェース：ヒータースタック（HEAT0）
    * 拡張インターフェース：RGB、EndStop、PT100/PT1000、XH2.54-2P CANポート
    * ファンインターフェース：1つのDCファン、2つのDCファンまたはPWMファン
    * モーター駆動：搭載TMC2240
    * 温度センサー：1つの100K NTCまたはPT1000（TH0）、1つのPT100/PT1000オプション
