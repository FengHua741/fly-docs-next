# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * SB2040はCANをサポートしています
    * このページではCANファームウェアを提供します。速度は500Kと1Mがあります
    :::

## フラッシュモードに入る

    :::tip

    囲まれているのは`BOOTボタン`です

    :::

    * マザーボードが完全に電源を切られた状態で、BOOTボタンを押しながらデータ転送用のケーブルをマザーボードとデバイスに接続します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/boot.webp').default} size="50%" align="center" />

            4. ファームウェアをフラッシュする

            :::tip

            どちらか一方のファームウェアだけフラッシュすれば大丈夫です

            :::

            * 以下のコマンドは500K CANレートのファームウェアをフラッシュするものです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * 以下のコマンドは1M CANレートのファームウェアをフラッシュするものです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```


            5. 以下のメッセージが出たらフラッシュ成功です

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            フラッシュが完了したら、マザーボードを一度完全に電源を切ってください。

            :::

        </TabItem>
    </Tabs>
