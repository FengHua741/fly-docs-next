# ファームウェア焼成説明

    * SB2040はCAN接続方法を使用してホストPCと接続できます
    * CANは`XT30(2+2)-F`を使用して接続する必要があります
    * 下記は`XT30(2+2)-F`データ線のスタイルです
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* メインボードのファームウェア焼成はKatapultを使用してKlipperを焼成する必要があります
* メインボードにKatapultファームウェアがすでに焼成されている場合は、最後のKatapultファームウェア焼成を行う必要はありません。それはメインボードのKatapultファームウェア焼成が失敗した場合の予防策です
* メインボードにリセットボタンを2回素早く押すと、メインボード上のLEDが点滅しているか確認できます。点滅していれば、メインボードはKatapultファームウェアに入っていることを示しています
:::

## どのファームウェアが必要かをどのように判断するか

    * SB2040はCAN接続方法を使用してホストPCと接続できます
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CANファームウェア焼成</Button>
        * 利点：`Katapult`ファームウェアを焼成すると、マザーボードに触れずにファームウェアを更新でき、バス接続もサポートされます
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションタイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する場合は、ホストPCがCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアが焼成されたマザーボードが必要です
