## 产品简介

    FLY SB2040は広州メイレン電子科技有限公司がVORON Stealthburberプリントヘッド向けに開発した噴頭ツールボードで、このツールボードを使用することで、従来のメインボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SB2040は36ステップモーターに対応しており、CANトランシーバー、USBポート、TMC2209ドライバ、ADXL345加速度センサー、2つの制御可能なファンなど、その他の機能も備えたツールボードです。このツールボードはKlipperファームウェアに適しています。

    * CANバス接続をサポートし、データ転送がより安定し、遅延が少なく、接続がより安定します
    * ボードキャリブレートされたTMC2209ドライバを使用し、UARTモードで使用でき、押出機に使用できます
    * ボードキャリブレートされたADXL345加速度センサーを使用
    * BLTOUCH、PL08Nなどのレベルセンサーをサポートし、XYリミットスイッチを統合
    * 12-24V電圧入力に対応
    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 4.7K抵抗（NTCヒータ）または1K抵抗（PT1000）切り替えをサポート
    * CANインターフェースはXT30（2+2）を使用し、15Aの電流をサポートし、ピーク値は30A
    * ボードキャリブレートされたNTC100k抵抗を追加し、倉温を測定
    * ADXL345のINT1とINT2をmcuに接続し、個別のユーザーがレベル調整に使用できるようにする
    * 高電圧入力リミットをサポート
    * 3mのCAN接続ケーブルを付属し、端子の正しくない圧着による信号ロスの問題を避ける
    * 2/3/4ラインファンをサポート
    * 分離設計、Stealthburnerは取り外しが簡単
    * hartkのStealthburner Toolhead Boardと互換性あり（ファン小板は交換不要）
    * インターフェース：制御可能なファン×3、RGB×1、リミットスイッチ×3、ヒータ×1、ヒータ抵抗×1

## 产品参数

    * Klipper CANモード、RS232モード、RRFCANモードの相互切り替えをサポート
    * XT30(2+2)コネクタを使用した接続インターフェース、15Aの電流をサポートし、ピーク値は30A
    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 入力電圧：DC12-24V
    * ヒーターバー最大出力電流：5A
    * ファンインターフェース最大出力電流1A
