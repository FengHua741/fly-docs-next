# CANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * micro-controller起動時に設定するGPIOピンを選択します。入力：gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存されコマンドラインに戻ったはずです
    * `make -j4` を入力してコンパイルを開始します（時間がかかります）
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    <LS />


## ファームウェアの焼込み

    * ファームウェアを焼込む前に、XT30データ線を使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードに接続してください。以下は参考接続図です
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/sb2040_utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lの抵抗値を電圧計で測定し、電源を切った状態で約 `60Ω` であることを確認します。もし `120Ω` なら `120Ω` のジャンパを接続し、約 `40Ω` ならジャンパを取り外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/120.webp').default} size="50%" align="center" />

    <CANFLASH />
