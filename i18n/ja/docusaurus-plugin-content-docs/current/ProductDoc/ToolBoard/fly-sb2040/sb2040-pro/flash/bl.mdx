# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * SB2040_PRO はCANをサポートしています
    * ここでは500Kと1Mの速度のCANファームウェアのみを提供します
    :::

## ブートロードモードに入る

    :::tip

    囲まれているのは`BOOTボタン`です

    :::

    * マザーボードが完全に電源オフになっている状態で、BOOTボタンを押したままデータ転送用のケーブルをマザーボードとデバイスにつなぎます
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/boot.webp').default} size="50%" align="center" />

                ```bash

                ```
                       
            * 以下は1M CANレートのファームウェア書き込みコマンドです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```


            5. 以下のメッセージが出たら焼く成功です
