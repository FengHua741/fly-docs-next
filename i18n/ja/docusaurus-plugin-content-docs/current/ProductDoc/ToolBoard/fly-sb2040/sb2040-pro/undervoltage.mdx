# ドライバ低電圧対策

## 方法一

    :::danger

    * 電源を交換する
    * 電源フィルターを追加する
    * モーターの判定を変更する
    * モーターケースをツールボードのGNDに接続する
    * クリーパードライバ構成に`driver_TPFD: 0`を追加する

    :::

## 方法二
    * SSHで上位機器に接続する
    * 下記のコマンドを入力する
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Klipperの更新後、この手順は元に戻る可能性があります
    :::
