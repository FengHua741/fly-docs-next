# 固件焼込説明

    * SB2040 PROはCAN接続方法を使用してホストコンピュータと接続することができます
    * CANは`XT30(2+2)-F`を使用して接続する必要があります
    * 下記は`XT30(2+2)-F`データ線のスタイルです
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* マザーボードファームウェア焼込みにはKatapultを介してKlipperを焼込む必要があります
* もしマザーボードがすでにKatapultを焼込まれていることが確認できた場合、最後のKatapultファームウェア焼込みを行う必要はありません。これはマザーボードのKatapultファームウェア焼込みが失敗した場合の予防措置です
* リセットボタンを素早く2回押すことで、マザーボード上のLEDが点滅しているか確認できます。もし点滅している場合は、マザーボードがKatapultファームウェアに入っていることを示します

:::

## どの種類のファームウェアが必要かどのように判断するか

    * SB2040 PROは以下の方法でホストコンピュータと接続することができます
    * [CANファームウェア](./can.mdx "クリックしてジャンプ")
        * 利点：`Katapult`ファームウェアを焼込んだ後、マザーボードに触れることなくファームウェアを更新でき、バス接続もサポートします
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションタイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する場合、ホストコンピュータがCANをサポートしているかどうかを確認し、UTOCまたはブリッジファームウェアが焼込まれたマザーボードを使用する必要があります

## Katapultファームウェアの説明

    * SB2040 PROはデフォルトでCAN接続速度1MのKatapultファームウェアが搭載されています
