# 製品紹介

    FLY SB2040 PRO は広州メイレン電子科技有限公司がVORON Stealthburberプリントヘッド向けに開発したツールボードです。このツールボードを使用すると、従来のメインボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SB2040はstealthburnerとafterburnerに適用され、CANトランシーバー、USBポート、TMC2240ドライバー、ADXL345加速度センサー、3つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperとRRFファームウェアに対応しています。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/img/sb2040.webp').default} size="100%" align="center" />


## 製品特長

    * Bootボタンを予備として設置し、ユーザーはTypeCを介してBootモードでファームウェアを更新できます
    * Klipperファームウェアをサポート
    * 温度センサー：搭載されたMAX31865チップ、PT100との互換性あり
    * 搭載されたADXL345加速度センサー
    * RGBインターフェースを予備として設置、搭載された2240ドライバー、klicky、Voron Tap、PL08nなど、さまざまなレベルセンサーをサポート
    * 12-24V電圧、15A電流入力に対応し、ヒーターソケットを拡大し、定格電流は10Aです


## 製品仕様


    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インターフェースはXT30(2+2)コネクタを使用し、15Aの電流をサポートし、ピーク値は30Aです
    * ファームウェア：Klipper
    * 入力電圧：DC12V-24V
    * 論理電圧：DC3.3V
    * 搭載センサー：ADXL345とオプションのMAX31865
    * ヒーターソケット：ヒーター（HEAT0）
    * 拡張インターフェース：制御可能なファン*3、RGB*1、リミットスイッチ*3、ヒーターチューブ*1、サーモカップル*1
    * ファンインターフェース：3つの制御可能なファン
    * モータードライバー：搭載されたTMC2240
