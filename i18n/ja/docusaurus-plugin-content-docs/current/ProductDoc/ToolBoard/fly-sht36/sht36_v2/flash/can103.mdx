# 103バージョンCANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * **GPIOピンをマイクロコントローラー起動時に設定する**を選択し、「!PC13」を入力してください（大文字で入力してください）
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/103.webp').default} size="100%" align="center" />
    * `Q` キーを押すと「Save configuration」と表示され、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェースに戻っているはずです
    * `make -j4` を入力してコンパイルを開始します（時間がかかります）
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    <LS />

## ファームウェアのフラッシュ

    * ファームウェアをフラッシュする前に、XT30データケーブルを使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードに接続してください。参照接線図は以下をご覧ください
    * ダイアルスイッチが正しい位置にセットされているか確認してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lが電源オフ状態で、マルチメーターで約60Ωの抵抗値を測定することを確認してください。もし120Ωであれば「120Ω」ジャンパーを接続し、約40Ωであればジャンパーを取り外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.webp').default} size="50%" align="center" />

    <CANFLASH />
