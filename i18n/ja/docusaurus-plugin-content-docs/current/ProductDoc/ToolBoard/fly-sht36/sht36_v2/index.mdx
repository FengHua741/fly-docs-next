# 概要

## 製品概要

    FLY SHT36 V2は広州メイレン電子科技有限公司が設計した3Dプリンターアクチュエータ用のツールボードです。このツールボードを使用すると、従来のメインボードとプリントヘッド間の複雑な配線を4本の線に置き換えることができ、レイアウトを最適化します。SHT36 V2は36ステッピングモーターに対応しており、CANトランシーバー、USBポート、TMC2209ドライバー、ADXL345加速度センサー、2つの制御可能なファンなどの機能を統合しています。このツールボードはKlipperファームウェアに対応しています。

    * 体積が最大限に圧縮され、6層基板設計を採用しています
    * CANバス接続に対応し、データ伝送がより安定し、遅延が少なく、接続が安定しています
    * ボード内蔵TMC2209ドライバー、UARTモードを使用し、エキスパンドアを使用できます
    * ボード内蔵ADXL345加速度センサー
    * PT100（MAX31865）対応
    * ファンMOS交換可能設計
    * BLTOUCH、PL08Nなどレベルセンサーに対応し、XYリミットスイッチが内蔵されています
    * 12-24V電圧入力に対応
    * MCU: STM32F072
    * 通信: CAN
    * BLTOUCH、PL08Nなどレベルセンサーに対応し、XYリミットスイッチが内蔵されています

## 新機能

    * 4.7K抵抗（NTCヒータ）または1.1K抵抗（PT1000）切り替えに対応するようになりました
    * CANインターフェースはMX3.0コネクタを使用しなくなりました。新しいコネクタは15Aの電流をサポートし、ピーク値は30Aです
    * 5/12/24Vファン電圧を選択できるようになりました
    * ボード内蔵NTC100k抵抗、倉温測定用
    * 電源逆接保護が追加されました
    * ADXL345のINT1がMCUに接続され、一部のユーザーがレベル調整に使用できます
    * エルボーヒーター端子が拡大され、定格電流は10Aです
    * 高電圧入力制限が追加されました
    * 主制御装置が国産MCUに変更されました
    * マグネットエンコーダー（AS5047D）サポート、Klipperによる運動解析用（現在Klipperはサポートしていません）
    * 1m CANコネクターラインが追加され、ユーザーが端子を正しく圧着できないために信号が失われる問題を解決しました
    * インストールサイズ：穴間隔43.84mm、M3インストール穴径
    * プロセッサ：GD32F103（2022年10月28日以前購入）、APM32F072（2022年10月28日以降購入）
