# ファームウェア焼成説明

    * ファームウェアのビルドと焼成にはPCソフトウェアが必要です
    * SHT36 V2はCAN接続方法でPCソフトウェアと接続できます
    * CAN接続には`XT30(2+2)-F`を使用します
    * 以下が`XT30(2+2)-F`ケーブルの様子です
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意点
* メインボードのファームウェア焼成にはKatapultからKlipperを焼成する必要があります
* メインボードがすでにKatapultファームウェアが焼成されていることが確認されたら、最後のKatapultファームウェア焼成ステップはスキップできます。これはメインボードのKatapultファームウェア焼成が失敗した場合の予防措置です
* メインボードのリセットボタンを素早く二回押すと、メインボード上のLEDが点滅しているか確認できます。これが点滅していれば、メインボードがKatapultファームウェアに入っていることを示しています
:::

## CANファームウェアビルド

    * SHT36 V2はCAN接続方法のみでPCソフトウェアと接続できます

    * APM 072とSTM 072については <Button variant="contained" disableElevation href="./can072">072CANファームウェアビルド</Button> をご覧ください
