# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * SHT36 LIS3DH はCANをサポートしています
    * このページではCANファームウェアのみを提供します。500Kと1Mの速度があります
    :::

## フラッシュモードに入る

    :::tip

    囲まれているのが`BOOTボタン`です

    :::

    * マザーボードが完全に電源を切られている状態で、BOOTボタンを押したままデータ転送用のケーブルの一方をマザーボードに、もう一方をデバイスに接続します
    * 次にBOOTボタンを押しっぱなしにしないでください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/boot.webp').default} size="50%" align="center" />

## ファームウェアをフラッシュする
    <Tabs>
        <TabItem value="win" label="Windowsでのフラッシュチュートリアル" default>

            * SHT36_LIS3DHのKatapultファームウェア <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/BL/SHT36_LIS3DH.zip">ファームウェアを自動ダウンロードするにはこちらをクリック</Button>

            <WIN />

        </TabItem>
        <TabItem value="linux" label="ホストでのフラッシュチュートリアル">
            
            <Katapult />

            4. ファームウェアをフラッシュする

            :::tip

            どちらか一つのファームウェアだけをフラッシュすれば大丈夫です

            :::

            * 以下のコマンドは500K CANレートのファームウェアをフラッシュするためのものです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_500K.uf2
                ```
                        
                * 以下のコマンドは1M CANレートのファームウェアをフラッシュするためのものです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_1M.uf2
                ```


            5. フラッシュが成功すると以下のメッセージが表示されます

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            フラッシュが完了したら、マザーボードを一度完全に電源を切ってください。

            :::

        </TabItem>
    </Tabs>
