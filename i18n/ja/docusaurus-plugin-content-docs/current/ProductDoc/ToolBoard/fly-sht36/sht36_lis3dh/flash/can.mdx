# CANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。「!gpio20」を入力してください。（「!」は大文字です）
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` キーを押して、「Save configuration」が表示されたら、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェースに戻ったはずです
    * `make -j4` を入力してコンパイルを開始します（時間がかかります）
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が表示されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    <LS />


## ファームウェアの焼込み

    * ツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードにXT30データラインで接続します。参考接続図は以下にあります
    * ダイアルスイッチが正しい位置にセットされていることを確認してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lが電源オフ状態で、マルチメーターで約60Ωの抵抗値が測定されるか確認します。もし約120Ωなら「120Ω」ジャンパーを接続し、約40Ωならジャンパーを外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/120.webp').default} size="50%" align="center" />

    <CANFLASH />
