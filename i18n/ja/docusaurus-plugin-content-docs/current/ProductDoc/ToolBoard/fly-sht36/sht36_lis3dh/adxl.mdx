# 加速度計の使用


import ADXL from '@site/i18n/ja/docusaurus-plugin-content-docs/current/DebugDoc/faq/kipper-adxl.mdx';

## クリッパーを更新する

* Klipperは2024年11月13日にLIS3DH加速度計をサポートしています。LIS3DH加速度計を使用するにはKlipperを更新する必要があります。
* 上位機器にネットワーク接続し、以下のコードを入力してください（ネットワーク問題により更新が失敗することがあります）
    ```bash
    cd ~/klipper 
    git pull
    ```

## 設定を追加する


    ```cfg
    #####################################################################
    #                       LIS3DH加速度計設定（必要に応じて有効化）
    #####################################################################
    [lis3dh]
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis3dh
    probe_points:
        125, 125, 20  #ここに共振測定時にノズルがいる座標を設定します
    ```
