# 固件焼録説明

    * SHT36 LIS3DHはCAN接続方法のみで上位機器と接続できます
    * CANは`XT30(2+2)-F`を使用して接続する必要があります
    * 以下の`XT30(2+2)-F`データ線のスタイルです
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* メインボードのファームウェア焼録にはKatapultを通じてKlipperを焼録する必要があります
* メインボードにKatapultがすでに焼録されていることが確認された場合、最後のKatapultファームウェア焼録は必要ありません。それはメインボードのKatapultファームウェア焼録が失敗した場合の予防措置です
* メインボードをリセットボタンを2回素早く押すことで、メインボード上のLEDが点滅しているかどうかを確認できます。もし点滅していたら、メインボードはKatapultファームウェアに入っています
:::

## どの種類のファームウェアが必要か判断する方法

    * SHT36 LIS3DHは以下の方法で上位機器と接続できます
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CANファームウェア焼録</Button>
        * 利点：`Katapult`ファームウェアを焼録すると、マザーボードに触れずにファームウェアを更新でき、バス接続もサポートされます
        * 欠点：`KlipperScreen`と一緒に使用すると、ホーム位置決定タイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する場合は、上位機器がCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアを焼録したマザーボードが必要です
