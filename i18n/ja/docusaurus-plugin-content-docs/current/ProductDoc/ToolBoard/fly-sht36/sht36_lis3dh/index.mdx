# 製品紹介

    FLY-SHT36-LIS3DHは広州メイロン電子科技有限公司が3Dプリンターアクチュエーターヘッド向けに設計したノズルツールボードです。このツールボードを使用すると、従来のメインボードとプリントヘッド間の煩雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。FLY-SHT36-LIS3DHは36ステップモーターに対応しており、CANトランシーバー、USBポート、TMC2209ドライバー、LIS3DH加速度センサー、2つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperファームウェアに対応しています。

    * 最大限の小型化、6層基板設計
    * CANバス接続をサポートし、データ伝送がより安定し、遅延が小さく、接続がより安定しています
    * ボード搭載TMC2209ドライバー、UARTモードを使用し、エクストルーダーに使用できます
    * ボード搭載ADXL345加速度センサー
    * PT100（MAX31865）をサポート
    * 取り替え可能なファンMOS設計
    * BLTOUCH、PL08Nなどのレベルセンサーをサポートし、統合されたXYリミットスイッチがあります
    * 12-24V電圧入力に対応
    * MCU: STM32F072
    * 通信: CANBUS, USB
