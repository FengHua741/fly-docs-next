# 固件焼込み説明

    * SHT36 PROはCAN接続方法を使用してホストコンピュータと接続することができます
    * CANは`XT30(2+2)-F`を使用して接続する必要があります
    * 以下は`XT30(2+2)-F`データ線のスタイルです
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* メインボードのファームウェア焼込みにはKatapultを介してKlipperを焼込みます
* メインボードにKatapultがすでに焼込まれていることが確認された場合、最後のKatapultファームウェア焼込みは必要ありません。これはメインボードのKatapultファームウェアが正常に焼込まれなかった場合の予防策です
* リセットボタンを素早く2回押して、メインボード上のLEDが点滅しているか確認できます。これがある場合は、メインボードがKatapultファームウェアに入っていることを示します
:::

## CANファームウェアのコンパイル

    * SHT36 PROはCAN接続方法のみを使用してホストコンピュータと接続することができます
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CANファームウェア焼込み</Button>
        * 利点：`Katapult`ファームウェアが焼込まれた場合、マザーボードに触れることなくファームウェアを更新でき、バス接続もサポートします
        * 欠点：`KlipperScreen`と同時に使用すると、リスティングタイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する必要がある場合は、ホストコンピュータがCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアが焼込まれたマザーボードが必要です
