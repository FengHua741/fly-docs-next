# 製品紹介

    FLY-SHT36-PROは広州メイレン電子科技有限公司が3Dプリンターカラビナのために設計したノズルツールボードです。このツールボードを使用すると、従来のメインボードとプリントヘッド間の煩雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。FLY-SHT36-PROは36ステッピングモーターに対応しており、CANトランシーバー、USBポート、TMC5160ドライバー、ADXL345加速度センサー、2つの制御可能なファンなどの他の機能を統合しています。

    * 極限のボリューム圧縮、6層基板設計
    * CANバス接続をサポートし、データ転送がより安定し、遅延が少なく、接続が安定しています
    * TMC5160を搭載しており、エクストルーダー用です
    * ADXL345加速度センサーを搭載しています
    * PT100（MAX31865）をサポートしています
    * 可換ファンMOS設計
    * BLTOUCH、PL08Nなどのレベルセンサーをサポートし、XYリミットスイッチを統合しています
    * 12-24V電圧入力に対応
    * MCU: STM32F072
    * コミュニケーション: CANBUS, USB

## 製品仕様

    1. Klipper CANモード、RS232モード、RRFCANモードの相互切り替えをサポートしています
    2. XT30(2+2)コネクターを使用した接続インターフェースは15Aの電流をサポートし、ピーク値は30Aです
    3. MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    4. 入力電圧：DC12-24V 5A
    5. ヒーターショック最大出力電流：5A
    6. ファンインターフェース最大出力電流1A
