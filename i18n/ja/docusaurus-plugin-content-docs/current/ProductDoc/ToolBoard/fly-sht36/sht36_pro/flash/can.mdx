# CANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。入力：`!gpio13` （`!`は大文字）
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/can.webp').default} size="100%" align="center" />
    * `Q`キーを押して、**Save configuration**が表示されたら、その後に`Y`キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインに戻っているはずです
    * 下記のコマンドを入力してコンパイルを開始します（時間がかかります）
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
    <LS />


## ファームウェアの書き込み

    * ファームウェアを書き込む前に、XT30データ線を使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードに接続してください。以下は参照接続図です
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/sht36pro_utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lが電源オフ状態で約`60Ω`の抵抗値になることを確認し、`120Ω`の場合には`120Ω`のジャンパを接続し、約`40Ω`の場合にはジャンパを外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/120.webp').default} size="80%" align="center" />

    <CANFLASH />

## ファームウェアの起動確認

    * ファームウェアが正常に起動するとLEDが常時点灯します
