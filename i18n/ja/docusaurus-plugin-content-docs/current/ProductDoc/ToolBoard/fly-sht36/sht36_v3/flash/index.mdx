# ファームウェア焼成説明
    
    * ファームウェアのコンパイルとファームウェア焼成にはアップペースを使用する必要があります
    * SHT36 V3はRS232とCANの2つの接続方法でアップペースと接続できます
    * RS232とCANには`XT30(2+2)-F`を使用する必要があります
    * 以下の`XT30(2+2)-F`データ線のスタイルです
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* マザーボードのファームウェア焼成にはKatapultを介してKlipperを焼成する必要があります
* マザーボードがすでにKatapultを焼成していることが確定した場合は、最後のKatapultファームウェアの焼成は必要ありません。これはマザーボードのKatapultファームウェアの焼成に失敗した場合の予防策です
* マザーボードをリセットボタンを2回素早く押すことで確認することができます。もしLEDが点滅すれば、マザーボードはKatapultファームウェアに入っています
:::

## どの種類のファームウェアが必要かを判断する方法

    * SHT36 V3は以下の方法でアップペースと接続することができます
    * <Button variant="contained" disableElevation href="./can">CANファームウェア</Button>
        * 利点：`Katapult`ファームウェアを焼成すると、マザーボードを触らずにファームウェアを更新することができます。さらに、バス接続もサポートしています
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションタイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する場合は、アップペースがCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアが焼成されたマザーボードが必要です
    * <Button variant="contained" disableElevation href="./rs232">RS232ファームウェア</Button>
        * 利点：特別なソフトウェアなしで使用でき、通信は信頼性があり安定しています
        * 欠点：ファームウェアを再焼成するには、手動DFUに進むか、RSTボタンをダブルクリックして`Katapult`に入らなければなりません
        * ヒント：G2TまたはUTORと組み合わせて使用する必要があります
