# RS232 ファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <RS232 />

    * **GPIOピンをマイクロコントローラー起動時に設定する**を選択し、入力：**!gpio5**
    :::warning
        英語入力モードで `!gpio5` を入力してください
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **設定の保存** が表示され、その後 `Y` キーを押します
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェイスに戻ったはずです
    * コンパイルを開始するために `make -j4` を入力します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="65%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です
       <ImageView image={require('@site/docs/General/dfu-flash/img/make_uf2_finish.webp').default} size="65%" align="center" />




## フラッシュモードに入る

    :::tip

    矢印で指されているのが `BOOT` ボタンです

    :::

    * ツールボードは**完全に電源を切った状態**で、**BOOTボタン**を押したまま、データ転送機能付きのTypeCケーブルを使用して**ツールボード**を**ホストコンピューターのUSBポート**に接続します。正しく接続されたら `BOOT` ボタンを離します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="80%" align="center" />
