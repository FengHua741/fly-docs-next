# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * SHT36_V3はCANとRS232をサポートしています
    * ここではCANファームウェアのみを提供します。500Kと1Mの速度があります
    :::

## フラッシュモードに入る

    :::tip

    矢印が示すのが`BOOTボタン`です

    :::

    * マザーボードが完全に電源オフになっている状態で、BOOTボタンを押しながらデータ転送用のケーブルをマザーボードとデバイスに接続する
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="100%" align="center" />

## ファームウェアをフラッシュする
    <Tabs>
        <TabItem value="win" label="Windowsのフラッシュチュートリアル" default>

            * SHT36 V3のファームウェアのダウンロード先   <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/firmware/BL/SHT36_V3.zip">ファームウェアを自動的にダウンロードするにはクリックしてください</Button>

            <WIN />
            
        </TabItem>
        <TabItem value="linux" label="アップストリームのフラッシュチュートリアル">
            
            <Katapult />

            4. ファームウェアをフラッシュする

            :::tip

            一つだけファームウェアをフラッシュする必要があります

            :::

            * 以下は500K CANレートのファームウェアのフラッシュコマンドです
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * 以下は1M CANレートのファームウェアのフラッシュコマンドです
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. 以下のメッセージが出たらフラッシュ成功です

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            フラッシュが完了したら、マザーボードを完全に電源オフにする必要があります。

            :::

        </TabItem>
    </Tabs>
