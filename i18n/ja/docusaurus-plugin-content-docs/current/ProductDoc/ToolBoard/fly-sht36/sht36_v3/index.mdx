# 1. 製品概要

FLY SHT36 V3は、広州メイレン電子科技有限公司が設計した3Dプリンターエキスパート用のヘッドツールボードです。このツールボードを使用すると、従来のマザーボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SHT36 V3は36ステッピングモーターに適しており、CANトランシーバー、RS232トランシーバー、TypeCポート、TMC2209ドライバー、LIS2DW加速度センサー、2つの制御可能なファンなどの他の機能を統合しています。このツールボードはKlipperおよびRRFファームウェアに対応しています。

    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/sht36.webp').default} size="100%" align="center" />


## 1.1 製品特長

    * Bootボタンを予約し、ユーザーはTypeC経由でブートモードに入り、ファームウェアを更新することができます。
    * Klipperファームウェア、RRFファームウェアをサポート
    * CANモード、RS232モード、RRF CANモードをサポート
    * RS232モードでは、より高い転送レートと強力な安定性を提供します。CANモードでは、CAN拡張をサポートし、CAN拡張インターフェースを予約します。
    * 温度センサーは、搭載されたMAX31865チップにより、PT100およびPT1000温度センサーと互換性があります。
    * 2つの制御可能なファンをサポートし、電圧を選択できます。Mos小板を交換可能で、修理・交換が容易です。最大出力電流は1Aです。
    * LIS2DW加速度センサーとLDC1612センサーを搭載しています。
    * RGBインターフェースを予約し、2209ドライバーを搭載しています。さまざまなレベルセンサー、例えばBLtouch、klicky、Voron Tap、PL08nなどに対応しています。
    * 12-24V電圧、15A電流入力に対応し、ヒーターバー端子を拡大し、定格電流は10Aです。

## 1.2 新機能

    * Klipper CANモード、RS232モード、RRF CANモードを切り替えることができます。
    * RS232モードを新規追加しました。これにより、より高い転送レートと強力な安定性を提供し、デバッグも簡単になります。
    * RRF CANモードを新規追加しました。これにより、RRFツールボードとして切り替えることができ、適用範囲が広がります。
    * CAN拡張インターフェースを新規追加しました。これにより、CANデバイスの拡張が容易になります。
    * 終端抵抗ダイアルスイッチを新規追加しました。その終端抵抗は120Rで、ダイアルで選択できます。ジャンパーピンは必要ありません。
    * 1メートルの電源/信号接続線を新規追加しました。接続線はツイストペア設計を使用し、干渉に対する耐性が向上しています。

## 1.3 製品仕様

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * 接続インタフェースはXT30(2+2)コネクターを使用し、15A電流をサポートし、ピーク値は30Aです。
    * ファームウェア：Reprap/Klipper
    * 入力電圧：DC12V-24V
    * 論理電圧：DC3.3V
    * 搭載センサー：LIS2DWとLDC1612
    * ヒーターバーインターフェース：ヒーターバー（HEAT0）、10A
    * 拡張インターフェース：RGB、EndStop、PT100/PT1000、XH2.54-2P CAN拡張口
    * ファンインターフェース：2つのDC制御可能なファン
    * モータードライバー：搭載TMC2209
    * 温度センサー：100K NTCまたはPT1000（TH0）、PT100/PT1000を選択可能
