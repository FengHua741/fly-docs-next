# Eddyの使用

    :::info
    * Eddyのインストール時、ノズルの位置はノズルの上に少なくとも1-2 mm離れてください
    * Eddyの周波数を校正する際、線圈PCBはヒートベッドから少なくとも5mm離してください
    * ヒートベッドは常温でテストし、テスト中はヒートベッドをオンにしないでください

    :::

## 参照設定および使用説明

    ### ldc16121612のデフォルト周波数の変更
    この方法はsht36-v3/SB2040-v3またはSB2040-Pro-v3でのみ使用します
    Mobaxtermツールを使用してコマンドを実行します
    
    <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button>のダウンロードが必要です

    [詳細な使用方法はこちら](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    最初のコマンドを実行し、Enterキーを押します

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    2番目のコマンドを実行し、Enterキーを押します
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    置換が成功すると`LDC1612_FREQ = 40000000`を含む行が返されます。置換が成功したことを示します。何も返されない場合は置換に失敗しています。
    または、次のコマンドで確認できます
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   注意：変更後、このコマンドでklipperサービスを再起動し、Eddyの周波数を再校正してください
   ```cfg
   systemctl restart klipper
   ```
   Eddyの周波数校正コマンド
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```


    ### ベッドスキャン設定

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 現在、外部1612モジュールがリリースされていないため、この設定は一時的に使用できません
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 温度補正設定

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh設定

    * [bed_mesh] の `horizontal_move_z` パラメーターは、2に設定することをお勧めします。プラットフォームをスキャンする際に、Eddyがプラットフォームにできるだけ近づくようにするためです

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
