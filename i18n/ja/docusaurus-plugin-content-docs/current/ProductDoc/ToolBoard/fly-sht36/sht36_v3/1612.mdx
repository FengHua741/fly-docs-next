# Eddy の使用

    :::info
  * 配置eddyの設定がある場合は削除してください
  * Eddyをインストールするとき、コイルはノズルの上にあり、ノズルから0.6-0.8cm離れている必要があります
  * テスト中は熱ベッドを開けないでください
  * 使用前に「probe」設定を削除してください
    :::

    :::warning 注意事項
    * 多Z軸マシンは一度手動でレベルを合わせる必要があります
    :::

## 使用説明
   * SSHツールを使用してIPを介してターミナルにログインし、スクリプトを実行します
    必要なSSHツールをダウンロード <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxtermツール</Button>


<Tabs>
  <TabItem value="eddy1" label="国際使用">
    # 国際の場合、以下のコマンドを使用してください
    ### 以下からeddyの自動化スクリプトのインストールを開始します
    * ダウンロードしたSSHツールを開き、次のコマンドを入力してエンターを押すとコピーできます
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * `fluidd`ウェブページの`moonraker.conf`設定ファイルに以下の設定を追加し、今後の更新を容易にします
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * 更新された場合、SSHツールを使用して次のコマンドを実行します
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* 実行すると、以下の図のように表示され、アッパーコンピュータのパスワードを入力します

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* 以下の図が表示される場合、「選択入力」に入力してください。プリンターに対応する番号を入力し、エンターを押します
* 「3z」を選択して「printer.cfg」に「z_tilt」が含まれていることを確認してください/「4z」を選択して「printer.cfg」に「quad_gantry_level」が含まれていることを確認してください。「独立双z」を選択する場合は、「3z」を選択してください
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy2.webp').default} size="100%" align="left" />

* 「sht36_v3」を使用している場合、「N」を入力し、エンターを押します
