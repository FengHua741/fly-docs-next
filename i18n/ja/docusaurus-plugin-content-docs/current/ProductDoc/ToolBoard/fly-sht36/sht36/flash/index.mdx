# 固件焼成説明

    * SHT36/42はCAN接続方法で上位機器と接続することができます
    * CANはMX3.0メタルターミナルを使用して圧着する必要があります
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />

## どの種類のファームウェアが必要かを判断する方法

    * SHT36/42はCAM方法で上位機器と接続することができます
    * <Button variant="contained" disableElevation href="./can">CANファームウェア焼成</Button>
        * 利点：`Katapult`ファームウェアを焼成すると、マザーボードに触れることなくファームウェアを更新でき、バス接続もサポートします
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションタイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する場合、上位機器がCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアがプリインストールされたマザーボードが必要です
