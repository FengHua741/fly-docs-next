# CANファームウェアのコンパイル

## コンパイルの開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` キーを押して **Save configuration** が表示されたら、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインに戻っているはずです
    * 下記のコマンドを入力してコンパイルを開始します（時間がかかります）
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    <LS />


## ファームウェアの書き込み

    * ファームウェアを書き込む前に、XT30データケーブルを使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードに接続します。参照接続図は以下です
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lの抵抗値が約`60Ω`になるように、電源を切った状態でマルチメーターで測定してください。もし約`120Ω`であれば`120Ω`のジャンパーを接続し、約`40Ω`であればジャンパーを外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/flash/img/120.webp').default} size="100%" align="center" />
