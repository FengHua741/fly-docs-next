# 製品紹介

    FLY SHT36/42は広州メイレン電子科技有限公司が3Dプリンターアクチュエータヘッド向けに設計したノズルツールボードです。このツールボードを使用すると、従来のメインボードとプリントヘッド間の複雑な配線を4本の線で置き換えることができ、レイアウトを最適化します。SHT36は36ステッパーモーターに適しており、SHT42は42ステッパーモーターに適しています。このツールボードにはCANトランシーバー、USBポート、TMC2209ドライバー、ADXL345加速度センサー、2つの制御可能なファンなどの他の機能が統合されています。このツールボードはKlipperファームウェアに対応しています。

    * 体積を極限まで圧縮し、6層基板設計を採用しています
    * CANバス接続をサポートし、データ転送がより安定し、遅延が小さくなり、接続が安定しています
    * ボード搭載のTMC2209ドライバーはUARTモードで動作し、押出機に使用できます
    * ボード搭載のADXL345加速度センサー
    * PT100（MAX31865）をサポートしています
    * 交換可能なファンMOS設計
    * BLTOUCH、PL08Nなどのレベルセンサーをサポートし、統合されたXYリミットスイッチを備えています
    * 12-24V電圧入力に対応しています
    * MCU: STM32F072
    * 通信: CANBUS, USB
    * インターフェース：制御可能なファン*2、RGB*1、リミット*3、プローブ*1、サーボ*1、ヒーターシース*1、サーマルセンサー*1、PT100*1

## 製品仕様

    1. 外形寸法：51.27mm*36.26mm（SHT36）、42mm*42mm（SHT42）
    2. 取り付け寸法：穴間隔43.84mm、M3取り付け穴径
    3. プロセッサ：STM32F072C8T6
    4. 入力電圧：DC12-24V 5A
    5. ヒーターシース最大出力電流：5A
    6. ファンインターフェース最大出力電流1A
