# USBファームウェアのコンパイル

    :::tip

    * USB加速度計は工場出荷時にUSB接続用のKlipperファームウェアがプリインストールされています
    * 既存のファームウェアを更新するだけです
    * USBケーブルでコンピュータとUSB加速度計を接続できます

    :::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />
    * マイクロコントローラー起動時に設定するGPIOピンを選択します: gpio8
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後に `Y` キーを押します
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインに戻っているはずです
    * コンパイルを開始するには `make -j4` を入力します。時間がかかります
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
