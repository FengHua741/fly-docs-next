# Katapultファームウェア

## フラッシュモードに入る

    :::tip

    囲まれているのが`BOOTボタン`です

    :::

    * フラッシュモードに入れるには、マザーボードが完全に電源オフになっている状態で、BOOTボタンを押しながらデータ転送用のケーブルをマザーボードとデバイスに接続します
