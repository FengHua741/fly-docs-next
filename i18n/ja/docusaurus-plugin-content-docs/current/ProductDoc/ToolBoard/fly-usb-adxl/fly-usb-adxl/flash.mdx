# USBファームウェアのコンパイル

    :::tip

    * USB加速度計は出荷時にUSB接続のKlipperファームウェアがプリインストールされています
    * 既存のファームウェアを更新するだけです
    * USBケーブルでパソコンとUSB加速度計を接続できます

    :::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押すと **Save configuration** が表示されるので、その後に `Y` キーを押します
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻っているはずです
    * コンパイルを開始するには `make -j4` を入力します、時間がかかります
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下のような内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
