# USB ファームウェアコンパイル

    :::tip

    * USB加速度計は出荷時にUSB接続のKatapultファームウェアがプリインストールされています
    * もしプリインストールされていない場合、自分で焼くことができます
    * [Katapultファームウェア焼录](./bl.mdx "クリックするとジャンプします")

    :::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` キーを押して、**Save configuration** が出るまで待ちます。その後、`Y` キーを押します
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 今では設定が保存され、コマンドラインインターフェースに戻ったはずです
    * `make -j4` を入力してコンパイルを開始します。時間がかかるかもしれません
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出たらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
