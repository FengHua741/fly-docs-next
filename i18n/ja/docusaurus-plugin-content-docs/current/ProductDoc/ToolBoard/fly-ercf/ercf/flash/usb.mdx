# USBファームウェアのコンパイル

    :::tip

    * ERCFは出荷時にCAN接続のKatapultファームウェアがプリインストールされています
    * USBファームウェアを焼录する場合、USBのKatapultファームウェアを焼录する必要があります
    * 一度焼录したら、再度焼录する必要はありません
    * <Button variant="contained" disableElevation href="./bl">Katapultファームウェア焼录</Button>

    :::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。入力：gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` キーを押すと、**Save configuration** が表示されます。このときに `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェースに戻っています
    * `make -j4` を入力してコンパイルを開始します。時間がかかることがあります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に次の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
