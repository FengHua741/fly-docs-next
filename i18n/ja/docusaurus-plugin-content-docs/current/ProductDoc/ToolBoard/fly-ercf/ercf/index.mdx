# 製品紹介

    FLY-ERCF は RP2040 をベースにした 3D プリンター多色制御ボードで、主に Enraged Rabbit プロジェクト用です。また、他の2軸ステッピングモーター制御にも使用できます。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.webp').default} size="100%" align="left" />

## 製品特長

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * 2つのドライバーポジション、UARTおよびSPIドライバーをサポート
    * オリジナルの ERCF EASY Board と互換性あり
    * 最大限のボディ圧縮、4レイヤーボード設計
    * 2.54mm デュアルレーン 7ピン拡張ポートを搭載、リミットスイッチなどの用途に拡張可能
    * 12-24V 電圧をサポート
    * CANバスと RS232 への接続をサポート、データ伝送がより安定し、遅延が小さく、接続が安定します
    * CANインターフェイスは XT30（2+2）を使用し、15A の電流をサポート

## 製品仕様

    * MCU：Raspberry Pi rp2040、デュアルコア ARM Cortex-M0+@133MHz
    * ファームウェア: Klipper
    * 入力電圧: DC12V-24V
    * インターフェイス: 2.54mm デュアルレーン 7ピン拡張ポート、SERVO X1、ENDSTOP X2、EXTRA X1
    * ドライバー：UART/SPI と STEP/DIR をサポート

    * 12-24V 電圧をサポート
    * CANバスをサポート、データ伝送がより安定し、遅延が小さく、接続が安定します
    * CANインターフェイスは XT30（2+2）を使用し、15A の電流をサポート
