# 固件焼込み説明

    * ERCFはCAN接続方法を使用してホストコンピュータと接続できます
    * CANは`XT30(2+2)-F`を使用して接続する必要があります
    * 下方は`XT30(2+2)-F`データ線スタイルです
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 注意
* マザーボードのファームウェア焼込みにはKatapultを介してKlipperを焼込む必要があります
* もしマザーボードがすでにKatapultを焼込んでいれば、最後のKatapultファームウェア焼込みは不要です。これはマザーボードのKatapultファームウェア焼込みに失敗した場合の予防策です
* リセットボタンを2回素早く押してみてください。マザーボード上のLEDが点滅しているかどうかを確認してください。これが点滅している場合は、マザーボードがKatapultファームウェアに入っていることを示します
:::

## どの種類のファームウェアが必要かをどのように判断するか

    * ERCFは以下の方法でホストコンピュータと接続できます
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CANファームウェア焼込み</Button>
        * 利点：`Katapult`ファームウェアを焼込んだ場合、マザーボードを触れずにファームウェアを更新することができます。また、バス接続もサポートしています
        * 欠点：`KlipperScreen`と一緒に使用すると、ホームポジションタイムアウトなどのエラーが発生する可能性があります
        * ヒント：CANを使用する場合は、ホストコンピュータがCANをサポートしていることを確認し、UTOCまたはブリッジファームウェアが焼込まれたマザーボードが必要です
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USBファームウェア焼込み</Button>
        * 利点：`Katapult`ファームウェアを焼込んだ場合、マザーボードを触れずにファームウェアを更新することができます。メモリーカードやDFU焼込みを使用する場合は、従来の方法でファームウェアを更新する必要があります。通信は信頼性が高く安定しています
        * 欠点：複数のUSBデバイスがある場合、データ転送に影響を与える可能性があります
