# Katapultファームウェア

## Katapultファームウェアの説明

    <CANBOOT />

    :::tip

    * ERCF V2はCANR、S232、USBをサポートしています
    * このページではCANファームウェアのみを提供し、500Kと1Mの速度のものがあります

    :::

## ブートロードモードに入る

    :::tip

    囲まれた部分が`BOOTボタン`です

    :::

    * ブートロードモードに入るには、マザーボードが完全に電源オフになっている状態で、BOOTボタンを押しながら、データ転送用のケーブルをマザーボードとホストコンピュータに接続します
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.webp').default} size="100%" align="center" />

            4. ファームウェアの書き込み

            :::tip

            一つのファームウェアだけ書き込めば大丈夫です

            :::

            * 以下は500K CAN速度のファームウェアの書き込みコマンドです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_500K.uf2
                ```
                       
            * 以下は1M CAN速度のファームウェアの書き込みコマンドです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_1M.uf2
                ```
            * 以下はUSBのファームウェアの書き込みコマンドです
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCFV2_CANBOOT_USB.uf2
                ```

            5. 以下のメッセージが出たら、書き込み成功です

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            書き込みが完了したら、マザーボードの電源を一度完全に切りましょう。

            :::

        </TabItem>
    </Tabs>
