# CANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。入力：gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後に `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻っています
    * `make -j4` を入力してコンパイルを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## ファームウェアの焼付

    * ファームウェアを焼付前に、XT30データ線を使用してツールボードをUTOCまたはブリッジファームウェアがインストールされたマザーボードに接続します。以下が参照配線図です
    * ダイアルスイッチが対応する位置にセットされていることを確認してください
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lが電源オフ状態で、マルチメーターで約`60Ω`の抵抗値を測定することを確認してください。もし`120Ω`なら`120Ω`ジャンパーを接続し、約`40Ω`ならジャンパーを外してください
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
