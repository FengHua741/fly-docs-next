# CANファームウェアのコンパイル

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <CAN />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。「gpio17」を入力してください。
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * 「Q」キーを押して「Save configuration」が表示されたら、「Y」キーを押します。
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインに戻ったはずです。
    * 下記のコマンドを入力してコンパイルを開始します。時間がかかります。
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下のような出力が出ればコンパイル成功です。

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```

## ファームウェアの書き込み

    * ツールボードをUTOCまたはブリッジファームウェアがインストールされたメインボードにXT30データラインで接続してください。以下の配線図を参考にしてください。
    * ダブルジャンパーが正しい位置に設定されているか確認してください。
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * CAN HとCAN Lが電源オフ状態で60Ω程度の抵抗値になることを確認してください。もし120Ωであれば120Ωのジャンパーリングを接続し、40Ω程度であればジャンパーリングを外してください。
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />

    <CANFLASH />
