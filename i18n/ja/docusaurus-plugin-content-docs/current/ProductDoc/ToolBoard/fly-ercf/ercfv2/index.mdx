# 产品简介

    FLY-ERCF は RP2040 をベースにした 3D プリンターのマルチカラー制御ボードで、主に Enraged Rabbit プロジェクト用です。また、他の2軸ステッピングモーター制御にも使用できます。
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.webp').default} size="100%" align="left" />

## 产品特点

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * CANモードとRS232モードをサポート
    * 2つのドライバーポジション、UARTおよびSPIドライバーをサポート
    * 原版ERCF EASY Boardと互換性あり
    * 最大限のボリューム圧縮、4層基板設計を使用
    * 14PIN拡張ポートをXHD2.54に変更、配線がより確実になり断線を防ぎます
    * 12-24V電圧をサポート
    * CANバスとRS232との接続をサポート、データ伝送がより安定し、遅延が小さく、接続が安定します
    * CANインターフェースにはXT30（2+2）を使用、15Aの電流をサポート

## 产品参数

    * MCU：Raspberry Pi rp2040、デュアルコアARM Cortex-M0+@133MHz
    * ファームウェア: Klipper CAN/RS232
    * 入力電圧: DC12V-24V
    * インターフェース: XHD2.54デュアル列7PIN拡張ポート、SERVO X1、ENDSTOP X2、EXTRA X1
    * ドライバー：UART/SPIとSTEP/DIRをサポート
    * 12-24V電圧をサポート
    * CANバスとRS232をサポート、データ伝送がより安定し、遅延が小さく、接続が安定します
    * CANインターフェースにはXT30（2+2）を使用、15Aの電流をサポート
