# USBファームウェアのコンパイル

    :::tip

    * ERCF工場出荷時にCAN接続用のKatapultファームウェアがプリインストールされています
    * USBファームウェアの焼付けが必要な場合は、USB用のKatapultファームウェアを焼付けます
    * 一度焼付けたら再度焼付けは必要ありません
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapultファームウェアの焼付け</Button>

    :::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。入力：gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェースに戻ったはずです
    * コンパイルを開始するために `make -j4` を入力します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されればコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
