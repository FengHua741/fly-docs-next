# 产品简介

    Fly-MMUは高性能コントロールボードで、3Dプリンターや自動化装置向けに設計されています。強力なH723マイクロコントローラーを統合しており、最大4つのTMCステッピングモータードライバーをサポートし、高電圧ドライブに対応しています。また、内蔵の11個のリミットスイッチ入力に加えて、FPCを介して16個のリミットを拡張でき、さらに4つのサーボ制御出力があります。このツールボードはKlipperファームウェアと互換性があり、CANバスとRS232接続オプションを提供することで、柔軟な通信と制御ソリューションを実現します。

## 产品特点

    * 主控は32ビットの550MHzのARM Cortex-M7シリーズSTM32H723マイクロコントローラーを使用
    * 4つのドライバーアンプをサポートし、最高48Vの高電圧TP5160ドライバに対応
    * 複数のリミットスイッチを拡張可能で、ボードに11個のリミットポートが内蔵されており、FPCを通じて16個のリミットポートを拡張できます
    * Klipperファームウェアの使用をサポートし、USB、CAN、およびRS232接続を使用
    * CAN拡張ポートが予備され、CANを使用する場合、他のデバイスとの接続にCAN拡張ポートを利用できます
    * CANバスとRS232接続をサポートし、データ転送がより安定し、遅延が小さく、接続がより安定します
    * CANインターフェースはXT30（2+2）を使用し、15Aの電流をサポートします

## 产品参数

    * MCU: STM32H723
    * ファームウェア: Klipper CAN/RS232
    * 電源入力：HVIN:12V/48VとVIN:12V/24V
    * 拡張インターフェース：11個のリミット、4つのサーボ、1つのRGB、1つのCAN拡張ポート、FPC拡張ポートは16個のリミットを拡張可能
    * ドライバ：TP2209とTP5160をサポートし、他のFLYのTPドライバも使用可能
    * 12-24Vの電圧をサポート
    * CANバスとRS232をサポートし、データ転送がより安定し、遅延が小さく、接続がより安定します
    * CANインターフェースはXT30（2+2）を使用し、15Aの電流をサポートします
