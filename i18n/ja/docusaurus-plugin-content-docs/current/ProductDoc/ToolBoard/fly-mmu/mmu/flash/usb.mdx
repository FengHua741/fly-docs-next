# USBファームウェアのビルド

    :::tip

    * MMUに事前に焼付けられたCAN接続のKatapultファームウェア
    * USBファームウェアを焼付ける場合はUSB用のKatapultファームウェアを焼付ける必要があります
    * 一度焼付けたら再度焼付ける必要はありません
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapultファームウェアの焼付け</Button>

    :::

## ビルド開始

    <SSH />

    <MakeOperate/>

    <USB />

    * microコントローラー起動時に設定するGPIOピンを選択します。入力：!PA15 请注意!は大文字です
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在は設定が保存され、コマンドラインインターフェースに戻っています
    * `make -j4` を入力してビルドを開始します。時間がかかります
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に次の内容が出力されたらビルド成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
