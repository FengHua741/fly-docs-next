# USBファームウェアのコンパイル

    :::tip

    * MMUは工場出荷時にCAN接続用のKatapultファームウェアがプリロードされています
    * USBファームウェアを使用する場合、USBのKatapultファームウェアを焼付けが必要です
    * 一度焼付けすれば再度焼付けの必要はありません
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapultファームウェアの焼付け</Button>

    :::

## コンパイル開始

    <SSH />

    <MakeOperate/>

    <USB />

    * マイクロコントローラー起動時に設定するGPIOピンを選択します。入力：!PA15 （!は大文字）
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` キーを押すと **Save configuration** が表示され、その後 `Y` キーを押します
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 現在、設定が保存され、コマンドラインインターフェースに戻っているはずです
    * 下記のコマンドを入力してコンパイルを開始します（時間がかかります）
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最後に以下の内容が出力されたらコンパイル成功です

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
