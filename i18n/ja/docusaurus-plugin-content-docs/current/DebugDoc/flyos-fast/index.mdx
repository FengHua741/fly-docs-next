# システム紹介とダウンロード

## 簡介

* FlyOS-Fast は軽量なLinuxシステムで、3Dプリンティングに最適化され、Flyシリーズのすべての上位機プラットフォームをサポートしています。
* FlyOS-Fast は軽量で、高速に起動します。
* カーネルはKlipperのリアルタイム性を確保するため最適化されています。
* ファイルシステムは最適化されており、突然の電源オフによる起動不能のリスクを軽減します。
* 30秒でKlipperscreenを起動できます。
* Klipper、RRF、Mainsail、Fluidd、Moonraker、KlipperScreenなどのサービスが事前にインストールされています。
* KlipperとRRFシステムの切り替えが可能です。
* MainsailとFluiddのオンラインでのワンクリック切り替えが可能です。
* USBメモリでの印刷がサポートされています。
* 断電継続印刷がサポートされています。
* シリアルポートスクリーンがサポートされています。
* fly-c8、fly-geminiv3を使用して、起動時にファームウェアの書き込みがサポートされています。
* FluiddなどのフロントエンドUIのデフォルト言語やデフォルトプリンター名を事前に設定できます。
* さらに多くのOEMカスタマイズ機能：起動ロゴ、事前設定された設定ファイル、Mainsailタブページの非表示設定などについてはお問い合わせください。

## システムイメージファイル名の説明

* システムファイル名の命名規則：`FlyOS-Fast-[チップシリーズ]-[バージョン番号].img`、例えば：`FlyOS-Fast-H5-v1.0.0.img`。
* ソースファイルは`.img`形式ですが、ダウンロードリンクでは`.exe`と`.7z`形式の圧縮ファイルのみが提供されます。
* Windowsシステムの場合、`.exe`ファイルをダウンロードします。これは7zの自己解凍圧縮パッケージで、ダウンロード後ダブルクリックして解凍すると、`.img`ファイルが得られます（この解凍はWindowsシステム専用です）。
* MacOSやLinuxシステムの場合、`.7z`ファイルをダウンロードします。これは7z圧縮パッケージで、ダウンロード後7z形式に対応したツールを使用して解凍すると、`.img`ファイルが得られます。
* システムファイルが大きいので、一時的に他の圧縮形式やソースイメージのダウンロードは提供していません。

:::tip システム選択

* fly-piシリーズ、fly-c8シリーズ、fly-geminiシリーズなど、全志H5チップを使用するすべてのFly製品を選択する場合は、`FlyOS-Fast-H5.exe`または`FlyOS-Fast-H5.7z`を選択してください。
* fly-pi-lite2シリーズ、fly-miniPadシリーズなど、全志H3チップを使用するすべてのFly製品を選択する場合は、`FlyOS-Fast-H3.exe`または`FlyOS-Fast-H3.7z`を選択してください。
* 両者のチップモデルは異なるため、システムファイルも異なります。実際の状況に応じて選択してください。

:::

## システムイメージダウンロード

* FlyOS-Fast の公式ダウンロードアドレス：[FlyOS-Fast製品ライブラリ](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/packages)

- デフォルトユーザ名：`root`
- デフォルトパスワード：`mellow`

:::tip 注意

このシステムは軽量なカスタムシステムで、組み込まれたklipperなどのコンポーネントとサービスは公式のオリジナルではありません。ユーザーによる独自の変更やアップグレードはサポートされません。

:::

## システムイメージダウンロードプロセス

* FlyOS-Fast イメージのダウンロードプロセス：

    1. ブラウザで上のFlyOS-Fast製品ライブラリにアクセスします。

    2. 必要な圧縮形式を選択し、ファイルページに進みます。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down1.webp').default} size="50%" align="center" />

    3. バージョンを選択します（デフォルトでは最新バージョン）。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down2.webp').default} size="50%" align="center" />

    4. ダウンロードボタンが見えない場合は、マウスをファイルリスト内に移動させ、Shiftキーを押しながらマウスホイールをスクロールすることで、ファイルリストを左右にスクロールできます。
    5. または、下部のスクロールバーにマウスを置いて左ボタンを長押しし、右にドラッグすることで、ダウンロードボタンを見ることができます。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down3.webp').default} size="50%" align="center" />

    6. ダウンロードボタンをクリックすると、対応するバージョンのシステムファイルをダウンロードできます。
