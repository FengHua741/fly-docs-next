# Klipper電源遮断継続印刷機能

- Klipper電源遮断継続印刷機能は、Klipperが電源を切られても、再び電源が入った後にKlipperの印刷状態が自動的に復元できる機能です。
- FLY上位機器では、C5メインボード以外はすべてKPPM電源遮断モジュールと組み合わせて使用する必要があります。
- Z軸が電力遮断後に移動する機種には適用されません。
