# システムリリース記録

## V1.2.40

### 変更点

- 既知の問題を修正
- Klipperの電源断継続印刷機能で新しいパラメーター `paused_recover_z` を追加: 停止中に電源が切れた後のZ軸補正高さを設定するためのもの
- Klipperを最新バージョンに同期更新
- Moonrakerを最新バージョンに同期更新
- KlipperScreenを最新バージョンに同期更新
- Katapultを最新バージョンに同期更新
- FluidDを最新バージョンに同期更新
- Mainsailを最新バージョンに同期更新

### 追加

- 新しいKlipper拡張機能 [klipper-led_effect](https://github.com/julianschill/klipper-led_effect.git) を追加

----

## V1.2.39

### 変更点

- Windows11 24H2システムでの焼込時に起動できない問題を解決

### 追加

- システム構成 `canbus_bitrate` を追加: デフォルトCAN0の速度を設定可能。デフォルトは `1000000` (1M)
- システム構成 `localtime` を追加: デフォルトタイムゾーンを設定可能。デフォルトは `Asia/Shanghai`

----
