# システムリリース履歴

## V1.2.39

### 変更点

- Windows11 24H2システムでのブート不能問題を解決しました。

### 新機能

- システム構成`canbus_bitrate`を追加し、デフォルトのcan0のレートを設定できます。デフォルト値は`1000000`（1M）です。
- システム構成`localtime`を追加し、デフォルトのタイムゾーンを設定できます。デフォルト値は`Asia/Shanghai`です。
