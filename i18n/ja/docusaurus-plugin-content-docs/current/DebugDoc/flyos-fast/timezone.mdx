# 世界時区対照表

:::tip 必読

- タイムゾーン設定は、`localtime`設定項目にタイムゾーンIDを入力してください。例：`Asia/Shanghai`。
- リブート後に有効になります。
- システムはネットワークに接続していない場合、時間は誤っていますが、接続すると自動的に同期されます。

:::
