# TMCエラーに関する関連情報

* `Unable to read tmc uart ‘stepper_x’ register IFCNT`または `Unable to read tmc spi ‘stepper_x’ register register`

    1. エラーの詳細

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart.webp').default} size="80%" align="center" />

    2. エラーの原因

        * ドライバとの通信ができない

    3. 主な解決方法

        * 対応するドライバのUARTピンの設定を確認する
        * UARTジャンパーが正しいか確認する
        * ドライバに電源供給があるか確認する
        * ドライバやドライバポートの設定を変更して診断を行う
        * 他のメーカーのドライバを使用している場合、ドライバのUARTピンがFLYと一致しているか自分で確認する必要がある。（FLYドライバのUARTピンは左から上数えて4番目のピン）
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/2209uart.webp').default} size="80%" align="center" />
        * 購入したドライバのUARTピンが5番目にある場合、以下のジャンパで解決できる可能性がある
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart1.webp').default} size="80%" align="center" />

* `TMC XXXX reports error: GSTAT: 00000001 reset=1(Reset) `

    1. エラーの詳細

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/reset.webp').default} size="80%" align="center" />

    2. エラーの原因

        * ドライバリセット

    3. 主な解決方法

        * 電源ケーブルの接触不良
        * モータのケーブルが切れている
        * モータの金属外殻をマザーボードまたはツールボードのGNDと接続する
---
<br/>
<br/>

* `TMC 'extruder' reports error: DRV_STATUS: c00d0103 otpw=1(OvertempWarning!) `

    1. エラーの詳細

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/overtemperror.webp').default} size="80%" align="center" />

    2. エラーの原因

        * ドライバが過熱しています！
        * モータが過熱しているわけではありません！

    3. 主な解決方法

        * 運転電流を減らす
        * ホールド電流を削除する
        * ドライバのサンプル抵抗が正しいか確認する
        * ドライバの冷却を強化する（例：ファンを追加する）
---
<br/>
<br/>

* `TMC ‘extruder’ reports error: DRV_STATUS: 601f1000 s2vsa=1(ShortToSupply_A!) csactual=31 ola=1(OpenLoad_A!) olb=1(OpenLoad_B!)）`

    1. エラーの詳細

        * 画像なし

    2. エラーの原因

        * ドライバの電流チェックが正しくない

    3. 主な解決方法

        * ステッピングモータのABグループのコイル抵抗が一致しているか確認する
        * ステッピングモータや部品モータ内部の配線が緩んでいるかショートしているか確認する
        * サイレントモードをオフにする
---
<br/>
<br/>

* `TMC 'extruder' reports error: DRV_STATUS: 00190060 s2vsb=1(ShortToSupply_B!) ola=1(0penLoad_A!) cs_actual-25`

    1. エラーの詳細

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/ab.webp').default} size="80%" align="center" />

    2. エラーの原因

        * 接線エラーまたはモータ自体に問題がある
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/motor_wiring.webp').default} size="60%" align="center" />
---
<br/>
<br/>

    3. 主な解決方法

        * ステッピングモータのABグループのコイル抵抗が一致しているか確認する
        * ステッピングモータと接線をチェックする
        * モータの配線順序を確認し、左側が1組、右側が1組になっているか確認する
