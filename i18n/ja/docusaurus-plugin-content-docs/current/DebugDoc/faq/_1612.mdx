:::info
* `z_tilt`機能または`quad_gantry_level（QGL）`機能を搭載したプリンターでは、複数のzレベル調整を実行してください。
:::

### 第一步校准EDDY
* イメージに示されているマクロコマンド`CALIBRATE_EDDY`をクリックして実行します
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy5.webp').default} size="50%" align="left" />
* その後、次のUIインターフェースが表示されます
* A4紙を使用してZ—offsetを調整します。ポップアップされるUIインターフェースで`z—offset`の数値を調整する前に、klipperドキュメントのチュートリアル[Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)を確認してください
* 適切な数値を見つけたら`ACCEPT`をクリックします
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* 校正が完了したら`SAVE_CONFIG`でパラメータを保存します

### 第二步：温度补偿
:::info
* 温度補償を行う前に、ヒートベッドとノズルを加熱しないでください
:::

* 下の画像に示されるマクロコマンド`TEMP_COMPENSATION`をクリックして実行します
<ImageView image={require('@site/docs/DebugDoc/faq/img/temp.webp').default} size="50%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy6.webp').default} size="50%" align="left" />

1. この操作を行った後、UIはZ軸調整ボックスを表示します。手動[Z軸オフセット校正（Paper Test）方法](https://www.klipper3d.org/Bed_Level.html#the-paper-test)を使用して、ペーパーをノズルとベッドの間に挟み、その値を確認してください。
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
2. 値を受け入れると、自動的にヒートベッド温度が80℃に、ノズル温度が250℃に設定されます。
3. 部屋にエアコンや窓がある場合、Eddyの温度上昇を確実にするために、エアコンを切るか窓を閉めてください。風は温度の上昇に影響を与える可能性があります。
4. Eddyの温度が上昇すると、システムは自動的に約3℃ごとに手動Zオフセット校正を促します。ヒートベッドの温度は非常に高いため、やけどに注意してください!!!
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
5. 手動Zオフセット校正（Paper Test）を繰り返し行い、校正が完了するまで続けます。Eddyの温度が上がらなくなった場合は、以下の関連コマンドを使用して校正を事前に終了することができます。

* ドリフト校正中に使用できる追加のgcodeコマンドには、`TEMPERATURE_PROBE_NEXT`があります
* `TEMPERATURE_PROBE_NEXT`は、EDDyが目標温度`80℃`に達する前に新しいデータを強制的に取得するために使用されます。
* `TEMPERATURE_PROBE_COMPLETE`は、EDDyが設定温度`80℃`に達する前に校正を完了するために使用されます。
* `ABORT`は、校正を終了し結果を無視するために使用されます。
* 校正が完了したら、温度オフセット設定を保存するために`SAVE_CONFIG`を使用してください！
* 上記の校正プロセスは、他の多くのプログラムと比較してより挑戦的で時間がかかる場合があります。広範囲の温度範囲で優れた初層プリント結果を得るために、練習と複数回の試行が必要になるかもしれません。

## 如何微调z—offset
* 第一步：pursa/OrcaSlicer/cura/Bambu Studioを使用して、高さ0.2の200x200mmの四角を切ってz—offsetの微調整をします
* 第二步：プリンターのIPアドレスから`fluidd` `mainsail`にアクセスし、ダッシュボードの`移動制御`でz軸オフセットを微調整します。z_OFFset
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* 校正が完了したら`SAVE_CONFIG`でパラメータを保存します
