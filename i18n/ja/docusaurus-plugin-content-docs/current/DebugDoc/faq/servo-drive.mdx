# 外置ドライブ使用チュートリアル

## 注意事項

:::danger 知るべきこと

* このチュートリアルはKlipperを使用して制御されます
* モジュールの配線は共陽極を使用します

:::


## ドライブ配線

    :::info
    * 下記の2つの方法からどちらか一つを選択してください
    :::

    * 最初の配線方法
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/servo-drive.webp').default} size="80%" align="center" />
    * 2つ目の配線方法
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/servo-drive-1.webp').default} size="80%" align="center" />

## ドライブ細分ジャンパ
    * どのジャンパも必要ありません！！！

## ドライブ内蔵ダイアル
    * ドライブ内蔵ダイアルは`3200`を使用することをお勧めします
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/3200.webp').default} size="80%" align="center" />

## Klipperリファレンス設定

    * 対応するドライブの設定をシャットアウトする必要があります。例えば`[tmc5160 stepper_x]`または`[tmc2209 stepper_x]`
    * `microsteps:`を`16`に設定します
    * `step_pulse_duration: 0.000004`を追加します
    * モーターの有効化: `enable_pin:!`の`!`を削除します
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/cfg.webp').default} size="50%" align="center" />

## ドライブパルス説明

    :::info

    * モデルのプリントがずれる
    * モーターが振動し、異音が出たり、様々な問題が発生する
    * `step_pulse_duration: 0.000004`を変更できます
    * この値を大きくする`0.000009`例えば`0.00001`または`0.00002`など

    :::
