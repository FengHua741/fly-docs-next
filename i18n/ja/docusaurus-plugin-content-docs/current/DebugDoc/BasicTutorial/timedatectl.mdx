# システムのタイムゾーンを変更する

    请注意FLY上位机如果使用FLY OS 3.3.4版本可以直接使用FLY-Tools変更タイムゾーン
    その他の上位機器は下記の手順をご覧ください


    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    `<TIMEZONE>`を希望するタイムゾーンコードに置き換えてください。タイムゾーンコードは[iana.org](https://www.iana.org/time-zones)ウェブサイトで確認できます。例えば、システムのタイムゾーンを北京時刻（Asia/Shanghai）に変更したい場合は、次のコマンドを入力できます：

    ```bash
    sudo timedatectl set-timezone Asia/Shanghai
    ```

    これで、システムのタイムゾーンは上海時間に変更されました。タイムゾーン設定を確認するには、以下のコマンドを実行してください：

```bash
date
```
