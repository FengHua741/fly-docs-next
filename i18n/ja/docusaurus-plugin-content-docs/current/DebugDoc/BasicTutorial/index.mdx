# 上位机SSHに接続する

:::danger

* このドキュメントはFLY上位機のみに適用されます。他のデバイスは参考程度にしてください。
* ファームウェアのコンパイルと焼付けについては、ネットワーク接続チュートリアルを参照してください。シリアル接続チュートリアルではありません。

:::

* シリアルまたはWi-Fiでデバイスに接続することができます。
* 事前に**MobaXterm_Personal**または他の**SSHターミナルツール**を準備してください。
* SDカードまたはM2WEに**対応する上位機のシステム**が焼付けられていることを確認してください。
* **もし上位機が正常に起動しない場合は、ドライバー、リミッター、ファンなどの周辺機器を取り外してください！！**
* **最後に、ネットワーク経由でSSHに接続してファームウェアのコンパイルを容易にしましょう！！**
* **ファームウェアのコンパイルが必要な場合は、Wi-Fiまたはイーサネットケーブルで上位機に接続するチュートリアルをご覧ください。**

        ## 上位機SSHにネットワークIPで接続する

        * システムが**正常に起動している**ことを確認してください。
        * 上位機が接続されている**ルーターバックエンド**にログインしてください。上位機のIPアドレスを知っている場合を除きます。
        * **上位機のユーザー名とパスワードを把握している**ことを確認してください。FLYシステムのデフォルトのユーザー名は**fly**で、デフォルトのパスワードは**mellow**です。
        * マザーボードが起動し、ネットワークに接続済みの場合、以下のようにSSHに接続できます。
        * MobaXtermを開き、設定を行います。IPは前に見たものに入力してください。ルーターバックエンドでも確認できます。

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **注意：ここでは4番目の手順で上位機のユーザー名を入力します。**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 初回接続時はこのウィンドウが表示されるので、`Accept`をクリックしてください。

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * パスワードを入力するように促されます。上位機のパスワードを入力し、エンターキーを押します。

        **注意：ここで入力したパスワードは見えません。入力後直接エンターキーを押してください。**

        FLYシステムのデフォルトでは、ユーザー名は`fly`で、パスワードは`mellow`です。

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * 接続が成功しました。

    </TabItem>
