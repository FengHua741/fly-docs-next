# バックアップ構成方法

:::danger 認識するべきこと

* このチュートリアルはFLY-Pi、ラズベリーパイ、オレンジパイなどのホストマシンに適用されます
* EMMCを使用している場合、焼成モードに入れることができなければこのチュートリアルは適用されません
* このチュートリアルはWindowsシステムでの操作を前提としています。フォーマットされていない不明なディスクなどを行わないでください！！！
* フォーマットしないでください！！！ フォーマットしないでください！！！ フォーマットしないでください！！！

:::

## 必要なソフトウェアのダウンロード

    * クリックしてダウンロードすると、ウェブページはあなたのコンピュータに適したバージョンを示します
    * ソフトウェアはインストール不要の緑色版です。ダウンロード後に解凍して使用します。
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="https://www.diskgenius.cn/download.php">DGハードディスクツール</Button>

## バックアップ開始

    :::info
    * フォーマットしないでください！！！
    * フォーマット後には復元できません！！！
    * TFカードの場合、直接コンピュータに接続してください
    * EMMCの場合、手動で焼成モードに入らなければなりません！！！

:::

### 接続
    * バックアップするシステムをコンピュータに接続します
    * `DiskGenius.exe`を開きます
    * 正常であれば、バックアップするディスクが表示されます
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/boot.webp').default} size="80%" align="center" />

### 見つける
    * 左側で主パーティションをクリックし、上の「ファイルを参照」をクリックします。最後に下のファイルが表示されます
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/home.webp').default} size="80%" align="center" />
    * `home`をダブルクリックして入ります
    * 入ると通常は一つのフォルダーがあり、その名前はあなたシステムユーザーの名前です
    * 二つまたはそれ以上の場合は、どのフォルダーがあなたのシステム構成を持っているか自分で確認する必要があります
    * その後ダブルクリックして入ります
    * 入ると `printer_data`を見つけてください
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/printer-data.webp').default} size="80%" align="center" />
    * `printer_data`フォルダーには、印刷ファイル、klipper設定、klipperログなどが含まれています。直接すべてをエクスポートできます

### 構成のエクスポート
    * `printer_data`を右クリックすると、下のメッセージが表示されます
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export.webp').default} size="40%" align="center" />
    * 保存したいディレクトリを選択するか、直接エクスポートします
    * エクスポートを確定します
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export1.webp').default} size="80%" align="center" />
    * エクスポートしたファイルを開き、`config`を見つけます。ここがあなたのシステム構成です
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export2.webp').default} size="80%" align="center" />
    * 構成のエクスポートが完了しました
    * 注意: `moonraker`ファイルを他のシステムに無断でアップロードしないでください。システムが異なるとklipperが接続できない可能性があります
