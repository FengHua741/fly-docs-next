# SSH経由でネットワークに接続する


* システムが起動し、SSHを使用して上位機器にシリアルポートで接続されていることを確認してください。推奨されるのは**MobaXterm_Personal**です
* 上位機器に**M2WE**または**2.4G_WiFiモジュール**がインストールされていることを確認してください。**Lite2**と**MiniPad**の場合はモジュールをインストールする必要はありません
* **2.4G_WiFiモジュール**、**Lite2**、および**MiniPad**は2.4G_WiFiのみを使用でき、5G_WiFiは使用できません
* ネットワークがデュアルバンド統合を使用している場合、正常に接続できないことに注意してください
* アンテナが完全に取り付けられていることを確認してください！！

 1. SSHソフトウェアで**nmtui**コマンドを入力してエンターキーを押すと、次の図のようなインターフェースが表示されます

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. エンターキーを押し、**Activate a connection**を選択してエンターキーを押します

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. 接続したいネットワークを選択し、エンターキーを押し、WiFiパスワードを入力し、**選択 OK**をして接続を保存します

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. WIFIに接続した後、キーボードで**Esc**キーを数回押し、現在のインターフェースを終了し、次に**ip a**と入力してエンターキーを押します

 * 図中の192.168.2.136が上位機器のIPアドレスです

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
