1. フラッシュツールのインストール

* FLYアップペースカンマニはフラッシュツールのインストールを実行する必要はありません
* インストールが必要な場合はご注意ください：アップペースカンマニがインターネットに接続できる必要があります
* すでにインストール済みの場合はインストールする必要はありません

```bash
sudo apt update
sudo apt install dfu-util -y
```

2. 下記のコマンドを実行して接続が正常であることを確認してください。

```bash
lsusb
```

* DFUデバイスがない場合、再び**フラッシュモード**に入ることを試みてください
* システムバージョンの問題により、他の名前が表示される可能性があります。`0483:df11`と表示される場合、フラッシュモードに入ったことを示します


    <ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.webp').default} size="80%" align="left" />    


3. ファームウェアのフラッシュ

* 下記のコードを実行してフラッシュを開始します

```bash
sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/klipper/out/klipper.bin
```

4. 下図矢印部分に表示されている内容``File downloaded successfully``が表示されたらフラッシュ成功です

    <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="80%" align="left" />    

:::tip

フラッシュが完了したら、マザーボードに完全に電源を切ってください。

:::
