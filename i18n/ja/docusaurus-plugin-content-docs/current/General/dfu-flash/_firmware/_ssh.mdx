* `klipper`サービスがインストールされた上位機を使用する必要があります  <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/BasicTutorial/">ネットワーク経由でSSHに接続</Button>
* そしてデバイスを上位機に接続してください
* 注意：一般的な上位機を使用してください。例えば、ラズベリーパイやFLY派などです。改造された上位機（例：WiFiスティック、レッドミーなど）は問題が多いので、技術サポートを提供できません


    
:::warning
* 下記のコマンドを入力して`ROOT`ユーザーから通常のユーザーに切り替えることができます
* `ROOT`ユーザーを使用しないでください
* flyユーザーの場合、`su fly`と直接入力できます。flyユーザー以外の場合、`<あなたの上位機のユーザーネーム>`をあなたの上位機のユーザーネームに置き換えてください！
```bash
su <あなたの上位機のユーザーネーム>
```
:::
