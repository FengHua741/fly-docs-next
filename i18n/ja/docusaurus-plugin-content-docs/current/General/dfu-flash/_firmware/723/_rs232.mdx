以下内容はファームウェアのコンパイル方法を説明しています：
* SSHに接続した後、次のコマンドを入力し、エンターを押します：

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* その中で` rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は以前のコンパイルデータとファームウェアを削除し、`make menuconfig`はファームウェアをコンパイルします。実行すると以下のインターフェースが表示されるはずです。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択し、エンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* メニュー**Micro-controller Architecture**に進み、**STMicroelectronics STM32**を選択し、エンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* メニュー**Processor model**に進み、**STM32H723**を選択し、エンターを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.webp').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**NO bootloader**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.webp').default} size="100%" align="Center" />

* **Clock Reference (8 MHz crystal)**を選択し、**25 MHz crystal**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.webp').default} size="100%" align="Center" />

* **Communication interface**を選択し、**Serial (on USART1 PA10/PA9)**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/uart.webp').default} size="100%" align="Center" />
