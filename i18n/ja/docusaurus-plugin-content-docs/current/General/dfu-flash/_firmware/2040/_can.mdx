以下はファームウェアをビルドする方法です：

* SSHに接続した後、次のコマンドを入力し、Enterを押します。

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`は以前のビルドデータとファームウェアを削除し、`make menuconfig`はファームウェアをビルドします。実行後、以下のインターフェースが表示されるはずです。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**を選択し、Enterを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* メニュー**Micro-controller Architecture**に進み、**Raspberry Pi RP2040**を選択し、Enterを押します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* **Bootloader offset**を選択し、**16KiB bootloader**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* **Communication interface**を選択し、**CAN bus**を選択します。

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/can.webp').default} size="100%" align="Center" />
