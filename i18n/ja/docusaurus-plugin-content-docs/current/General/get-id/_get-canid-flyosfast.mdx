## CAN IDを入力する

1. `DEVICES`を閉じてIDを入力します
    :::danger 注意

    * 最初の赤枠前の#は削除してください（スペースも含む）。そして`serial:`を`canbus_uuid:`に変更してください。
    * FLYシステムのデフォルト設定には[MCU]が含まれる場合がありますが、下のIDが`serial: /tmp/klipper_host_mcu`の場合、`[MCU]`を`[mcu HOST]`に変更するか、この2行を削除してください。
    * 下記の内容から`canbus_uuid:`をコピーできます。
    :::

    ```cfg
    [mcu]
    canbus_uuid:
    ```

2. IDを設定ファイルに記入します
    :::tip 注意

    * IDを入力したら右上の`SAVE & RESTART`をクリックしてください。
    :::

3. Klipperが`ADC out of range`と表示された場合、これは正常な現象です。ヒートベッドとセンサーをメインボードに接続し、ノズルやヒートベッドのセンサーピンを適切に設定した後、保存して再起動してください。

:::danger
注意: 文書に記載されているすべてのIDは例示です。実際の各メインボードのIDは異なりますので、実際に入力したIDを使用してください。
:::
