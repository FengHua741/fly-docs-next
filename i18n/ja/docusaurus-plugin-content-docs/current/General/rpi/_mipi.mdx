## ラズベリーパイ5とMIPI

:::tip
FLY_MIPIはラズベリーパイ5専用のアダプタモジュールで、これを使用することでDSIとCSIデバイスを直接使用できます。
:::

### ラズベリーパイ5とMIPIの接続

    :::danger 注意点

    * ラズベリーパイ5のDSIとCSIインターフェースは現在MIPIに名前が変更されています。
    * MIPIインターフェースは設定を変更することによりCSIカメラまたはDSIディスプレイを使用できます。
    * ゴールドフィンガーの向きに注意してください。

    :::

    * ラズベリーパイ5には現在2つのMIPIインターフェースがあります。正面から見ると上部に1と0のマーキングがあります。

## DSIディスプレイ接続

    * 1番目のインターフェースを使用する場合、以下の設定でディスプレイを構成する必要があります。

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * 0番目のインターフェースを使用する場合、以下の設定でディスプレイを構成する必要があります。

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## CSIカメラ接続

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * カメラモデルは購入した製品に合わせて設定する必要があります。このチュートリアルではimx219を使用しています。

    * 左側のMIPIインターフェースを使用する場合、以下の設定でカメラを構成する必要があります。
    ```bash
    dtoverlay=imx219,cam1
    ```

    * 右側のMIPIインターフェースを使用する場合、以下の設定でカメラを構成する必要があります。
    ```bash
    dtoverlay=imx219,cam0
    ```
