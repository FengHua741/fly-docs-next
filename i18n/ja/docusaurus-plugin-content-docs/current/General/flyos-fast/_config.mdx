## 重要提示
    :::tip 必読

    - 設定ファイルの`部分`設定は、ファームウェア書き込み後にシステムを初めて起動する際にのみ有効です。
    - 本チュートリアルでは変更可能なオプションのみを提供します。

    :::

    :::warning 注意

    - 尽量は`mainsail`または`fluidd`のウェブページを使用して設定の変更を行ってください。
    - ページ翻訳機能を使用しないでください。保存後、システムが起動しなくなる可能性があります。
    - 設定ファイルには中国語や特殊文字を入力しないでください。
    - 設定ファイルの変更後、システムを再起動する必要があります。

    :::

    :::danger

    - 設定ファイル内の`######### PLEASE DO NOT EDIT THE FOLLOWING #########`以下の内容は編集しないでください。

    :::

## 設定ファイルの変更方法

### ファームウェア書き込み後の設定ファイルの変更

- 最初に起動時にのみ有効な設定項目は、ファームウェア書き込み後に専門的なテキストエディタを使用して、可搬型ドライブ`FlyOS-Conf`内の設定ファイル`config.txt`を開いて変更してください。

### システムが起動した状態でのオンライン設定変更

- 通常の設定項目は、システムのWeb設定ページで変更し、システムを再起動すると有効になります。
    - ブラウザでデバイスのIPアドレスにアクセスします。例: `http://192.168.1.2/`
    - fluiddの場合、下図左のように「チェックを外す」 -> 「隠しファイルとフォルダをフィルタリング」を選択します。
    - mainsailの場合、下図右のように「チェックを入れる」 -> 「隠しファイルを表示」を選択します。

    <table>
      <tr>
        <td>
          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config1.webp').default} size="100%" align="center" />
        </td>

          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />

    - このようにして`.flyos-config`フォルダが見えます。このフォルダ内には`sys-config.conf`ファイルがあります。
    - `sys-config.conf`ファイルは可搬型ドライブ`FlyOS-Conf`内の設定ファイル`config.txt`のシンボリックリンクです。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - `sys-config.conf`ファイルを開き、必要な設定項目を変更して「保存」 -> 「閉じる」し、その後再起動してください。

    - 注意: `sys-config.conf`ファイルの設定項目の形式は`key=value`で、`key: value`に変更しないでください。
    - 変更を保存後、システムを再起動すると有効になります。

## 設定項目

### 無線ネットワーク設定

- 以下の設定項目はシステムを起動するたびに有効になります。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | `空` | 任意のWiFi名 | WiFi名、記入すると起動時に自動的に優先的に接続します |
| `WIFI_PASS` | `空` | パスワードがない場合は空にしておく | WiFiパスワード、記入するとパスワードでWiFiに接続します |

----

### プリンターのデフォルト設定

:::tip
- 以下の設定項目は初めて起動する際のみ有効です。
:::

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :---: | :--- | :--- |
| `language` | `zh-CN` | <details><summary>詳細を展開するにはクリック</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | 初回起動時にfluiddとmainsailのデフォルト言語を設定 |
| `printer_name` | `空` | 任意の文字列 | 初回起動時にfluiddとmainsailのデフォルトプリンター名を設定 |

----

### システム設定

- 以下の設定項目はシステムを起動するたびに有効になります。
- 下記は変更可能なオプションです。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `board` |  <code>{props.board}</code> |   <code>{props.board}</code> | ボードモデル、推奨される値以外は変更しないでください |
| `localtime` | `Asia/Shanghai` | [タイムゾーン対照表](/docs/DebugDoc/flyos-fast/timezone) | システムのデフォルトタイムゾーンを設定します。ネットワーク接続後、自動的に時刻を調整します |
| `canbus_bitrate` | `1000000` | `1000000`<br/> `500000`<br/> `250000` | FASTシステムのすべてのCANデバイスのデフォルトルートを設定します。ここでのみ変更可能です。デフォルトは1M |

----

### スクリーン表示設定

- 以下の設定項目はシステムを起動するたびに有効になります。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `screen` | `hdmi` | <code>nano</code>,<code>{props.screen}</code> <br/> <code>{props.screen1}</code><code>{props.screen2}</code> | スクリーンを設定します。noneにするとすべての表示を無効化 |
| `rotate` | `90` | `0`, `90`, `180`, `270` | FLY-TFTの回転角度を設定します。hdmiには影響しません |
| `klipper_screen` | `0` | `0`, `1` | KlipperScreenを有効にするかどうか設定します。<br/>1: 有効<br/>0: 無効<br/>`printer=klipper`の時有効 |
| `moon2uart` | `0` | `0`, `1` | Moon2UARTを有効にするかどうか設定します。<br/>1: 有効<br/>0: 無効<br/>`printer=klipper`の時有効 |


----

### Klipper関連設定

- 以下の設定項目はシステムを起動するたびに有効になります。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | KlipperのデフォルトフロントエンドUIを設定します。<br/>オンラインワンクリック切り替え機能には影響しません。<br/>`printer=klipper`の時有効 |
| `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | FASTシステムのすべてのCANデバイスのデフォルトルートを設定します。ここでのみ変更可能です。デフォルトは1M |
| `crowsnest` | `0` | `0`, `1` | Crowsnestを有効にするかどうか設定します。<br/>1: 有効<br/>0: 無効<br/>KlipperとRRFモードどちらでも使用可能 |

----

### プリンター種別設定

- 以下の設定項目はシステムを起動するたびに有効になります。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `printer` | `klipper` | `klipper`, `rrf` | システムをどのサービスで起動するか設定します。<br/>`klipper`: Klipper, Moonrakerなどのサービス<br/>`rrf`: RRF DSF, DWCなどのサービス |

----

### 停電シャットダウン

- 以下の設定項目はシステムを起動するたびに有効になります。
- 停電中に印刷を継続するには、停電シャットダウン機能を無効にする必要があります。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `shutdown_pin_state` | `0` | `0`, `1` | シャットダウンボタンの電位状態を設定します。<br/> 1: ローレベル反復<br/> 0: ハイレベルトリガー |
| `shutdown_pin` | <code>{props.shutdown_pin}</code>| <code>{props.shutdown_pin}</code> | シャットダウンボタンのトリガーピンを設定します。noneにすると機能を無効にします |

----
