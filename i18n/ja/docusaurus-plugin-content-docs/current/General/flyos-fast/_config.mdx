## 重要ポイント

    :::tip 必読

    - 設定ファイル内の`部分`設定は、ファームウェア書き込み後の最初のシステム起動時にのみ有効です。
    - 設定プロセスは煩雑であるため、マシンにネットワーク接続した後にブラウザで設定することを強く推奨します。
    - このチュートリアルでは変更可能なオプションのみを提供しています。

    :::

    :::warning 注意

    - できるだけ`mainsail`または`fluidd`のウェブページを使用して設定を変更してください。
    - ページ翻訳を使用しないでください。保存後、システムが起動しなくなる可能性があります。
    - 設定ファイルには中国語や特殊文字を入力しないでください。
    - 設定ファイルの変更後、システムを再起動する必要があります。

    :::

    :::danger

    - 設定ファイル内の`######### PLEASE DO NOT EDIT THE FOLLOWING #########`以下の内容は編集しないでください。

    :::

## 設定ファイルの変更方法

### ファームウェア書き込み後の設定ファイルの変更

- 『最初の起動』時にのみ有効な設定項目は、ファームウェア書き込み後、専用のテキストエディタを使用して可搬ディスク`FlyOS-Conf`内の設定ファイル`config.txt`を開き、変更する必要があります。

### システム起動後のオンラインでの設定変更

- 普通の設定項目は、システムのウェブ設定ページで変更し、システムを再起動すると有効になります。
    - デバイスのIPアドレスをブラウザで開きます。例：`192.168.1.2/`
    - fluiddの場合、図左のように「チェックを外す」→「隠しファイルとフォルダをフィルタリング」を行います。
    - mainsailの場合、図右のように「チェックを入れる」→「隠しファイルを表示」を行います。

      <tr>

    - これにより`.flyos-config`フォルダが見えます。このフォルダ内には`sys-config.conf`ファイルがあります。
    - `sys-config.conf`ファイルは可搬ディスク`FlyOS-Conf`内の設定ファイル`config.txt`のシンボリックリンクです。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - `sys-config.conf`ファイルを開き、必要な設定項目を変更して「保存」→「閉じる」し、その後再起動します。

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config4.webp').default} size="60%" align="center" />

    - 注意：`sys-config.conf`ファイル内の設定項目の形式は`key=value`であり、`key: value`に変更しないでください。
    - 変更後、保存し、システムを再起動すると有効になります。

## 設定項目

### 無線LAN設定

- 以下の設定項目はシステムの各起動時にも有効です。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | `空` | 任意のWIFI名 | WIFI名、記入された場合、起動時に優先的にこのWIFIに接続します |
| `WIFI_PASS` | `空` | パスワードがない場合は空白 | WIFIパスワード、記入された場合、パスワードを使用してWIFIに接続します |

----

### プリンターのデフォルト設定

:::tip
- 以下の設定項目は『最初の起動』時にのみ有効です。
:::

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :---: | :--- | :--- |
| `language` | `zh-CN` | <details><summary>詳細を展開するにはクリック</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | 最初の起動時にfluiddとmainsailのデフォルト言語を設定 |
| `printer_name` | `空` | 任意の文字列 | 最初の起動時にfluiddとmainsailのデフォルトプリンター名を設定 |

----

### システム設定

- 以下の設定項目はシステムの各起動時にも有効です。
- 以下は変更可能なオプションです。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `board` |  <code>{props.board}</code> |   <code>{props.board}</code> | マザーボードの型番、この値以外は変更しないでください |
| `localtime` | `Asia/Shanghai` | [世界タイムゾーン対照表](/docs/DebugDoc/flyos-fast/timezone) | システムのデフォルトタイムゾーンを設定、ネットワーク接続後は自動で時刻を調整します |
| `canbus_bitrate` | `1000000` | `1000000`<br/> `500000`<br/> `250000` | FASTシステムのすべてのCANデバイスのデフォルトルートレートを設定、ここでは変更可能です、デフォルトは1M |

----

### スクリーン表示設定

- 以下の設定項目はシステムの各起動時にも有効です。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `screen` | `hdmi` | <code>nano</code>,<code>{props.screen}</code> <code>{props.screen3}</code> <br/> <code>{props.screen1}</code><code>{props.screen2}</code>| スクリーンを設定、noneはすべての表示をオフ |
| `rotate` | `90` | `0`, `90`, `180`, `270` | FLY-TFTの回転角度を設定、hdmiには無効 |
| `klipper_screen` | `0` | `0`, `1` | KlipperScreenを有効にするかどうかを設定<br/>1: 有効<br/>0: 無効<br/>`printer=klipper`の場合有効 |
| `moon2uart` | `0` | `0`, `1` | Moon2Uartを有効にするかどうかを設定<br/>1: 有効<br/>0: 無効<br/>`printer=klipper`の場合有効 |

----

### Klipper関連設定

- 以下の設定項目はシステムの各起動時にも有効です。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | KlipperのデフォルトUIを設定<br/>オンラインワンクリック切り替え機能には影響しません<br/>`printer=klipper`の場合有効 |
| `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | FASTシステムのすべてのCANデバイスのデフォルトルートレートを設定、ここでは変更可能です、デフォルトは1M |
| `crowsnest` | `0` | `0`, `1` | Crowsnestを有効にするかどうかを設定<br/>1: 有効<br/>0: 無効<br/>KlipperとRRFモードどちらでも使用可能 |

----

### プリンター種別設定

- 以下の設定項目はシステムの各起動時にも有効です。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `printer` | `klipper` | `klipper`, `rrf` | システムをどのサービスで起動するか設定<br/>`klipper`: Klipper, Moonrakerなどのサービス<br/>`rrf`: RRF DSF, DWCなどのサービス |

----

### 停電シャットダウン

- 以下の設定項目はシステムの各起動時にも有効です。
- 停電継続プリントを実現するには、停電シャットダウン機能を無効にしてください。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `shutdown_pin_state` |<code>{props.state}</code>| `0`, `1` | シャットダウンボタンの電位状態を設定<br/> 1: 低電位繰り返し<br/> 0: 高電位トリガー  |
| `shutdown_pin` | <code>{props.shutdown_pin}</code>| <code>{props.shutdown_pin}</code> | シャットダウンボタンのトリガーピンを設定、noneに設定すると無効化 |

----

### RRF関連設定

- 以下の設定項目はシステムの各起動時にも有効です。

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `rrf2uart` | `0` | `0`, `1` | 暫定的に無効、後続バージョンで更新予定 |

----

- 以下の設定項目は『最初の起動』時にのみ有効です。
:::

| 設定項目 | デフォルト値 | 使用可能な値 | 説明 |
| :---: | :-------: | :--- | :--- |
| `firmware` | `空` | 有効なファームウェアファイルパス | システム起動時に自動でファームウェアを焼込む設定、現在はC8、Gemini-V3、C5のみ有効 |

----
