### Klipperファームウェアのコンパイルと設定ページ操作説明

    :::tip

    キーボード入力モードを半角モード（英語モード）に設定してください

    :::

    1. キーボードの上矢印キー`↑`、下矢印キー`↓`は、カーソルを移動してメニュー項目を選択します
    2. 確認キー`Enter`またはスペースキー`Space`は、メニューにチェックを入れたり、サブメニューに進むために使用します
    3. 戻るキー`ESC`は、一つ上のメニューに戻るために使用します
    4. `Q`キーは、Klipperファームウェア設定ページから終了するために使用します
    5. `Y`キーは、Klipperファームウェア設定ページから終了する際に、確認メッセージが表示された場合に、`Y`キーを押して設定ファイルを保存します

    :::warning

    設定ページのオプションが少ない場合、まず`[ ] Enable extra low-level configuration options`にチェックを入れて、一部の隠しオプションを表示してください

    :::
