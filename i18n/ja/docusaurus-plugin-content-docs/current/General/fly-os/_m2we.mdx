# USBを介してM2WEに準備

### 準備作業

    * ソフトウェア`Win32DiskImager`をインストールしてください
    * システムの焼込には`balenaetcher`は推奨されません
    * システムを展開し、展開後のファイル名の拡張子は`.img`になります
    * [M2WE BOOTドライバー](/docs/ResDownload/auxiliary_software/ "クリックしてジャンプ") をダウンロードしてください  

### ドライバーのインストール

    * `Zadig`ソフトウェアを開きます
    * 最初の`Zadig`はインストールせずに直接実行できますが、他の2つはインストールする必要があります。インストールプロセスでは全て`次へ`または`Next`を選択してください
    * `Zadig`を開き、上部メニューの`Options`から`List All Devices`にチェックを入れます

    <ImageView image={require('@site/docs/General/fly-os/img/zadig.webp').default} size="80%" align="left" />

:::tip
デバイスをコンピュータに接続する前にボタンを押した状態にしてください
:::

    * 問題なければ、この時点でzadigに`Unknown Device #x`または`USB Device(VID_1f3a_PID_efe8)`が追加されます。それを選択します

    <ImageView image={require('@site/docs/General/fly-os/img/zadig2.webp').default} size="80%" align="left" />

    * 上の画像の2つ目の箇所は`WinUSB(vxxxxxxx)`を選択してください。バージョン番号は関係ありません
    * `ドライバーのインストール`をクリックしてドライバーをインストールします

    * インストールされたFLY-BOOTを開き、`デバイスの確認`をクリックします
    * 前の手順が正しく行われた場合、利用可能なデバイスが見つかったと表示されます
    * `BOOT`を起動し、正常であれば`BOOT`が成功したと表示されます

### M2WEのフォーマット

    * `TFFormatter.exe`ソフトウェアを使用して`BOOT`ディスクをフォーマットします

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * または、Windowsの標準機能であるディスク管理を使ってパーティションを削除し、削除後に新しいパーティションを作成します
    * これでファイルエクスプローラーを開くことができます。14.5GBのUSBドライブ（16GB eMMCの表示サイズは14.5GB）が追加されます
    * このようにすると、Win32diskimagerを使用してシステムイメージを焼込めるようになり、ドライブレターはこの14.5GB USBドライブのものに設定します
