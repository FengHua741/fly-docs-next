::: tip
ブラウザでIP:9999を開いてFLY-Toolsにアクセスしてください。
:::

* TFTを使用するには、以下のオプションを有効にする必要があります。
* TFTディスプレイは3つのタイプがあります:
    * `FLY-TFT-V1` V1 电阻屏，适用于所有 FLY 上位机
    * `FLY-TFT-V2-R` V2 电阻屏，适用于 C8 与 PI-V2 以及带有40PIN系列的树莓派，但需要与 G2T 一起使用
    * `FLY-TFT-V2-C` V2 电容屏，适用于 C8 与 PI-V2 以及带有40PIN系列的树莓派，但需要与 G2T 一起使用

* FLY Toolsにアクセスした後
