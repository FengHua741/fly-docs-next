# FLYOS_Env の使用

    :::tip ヒント

    * 3.3.4システムからFLY-TOOLSを使用してシステム構成を行うことができます。ブートに入らずに設定を変更できます。
    * お注意ください、古いシステムはFLY-TOOLSを使用した設定を直接サポートしていません。
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ToolsDoc/fly-tools/">FLY_TOOLS使用チュートリアル</Button>

    :::

### WIFI構成説明

    :::danger 知識が必要です

    * WIFI構成方法は複雑で、シリアルポートを介してホストコンピュータに接続し、その後WIFI接続を行います。

    :::
