## FLY上位机使用教程

::: tip
ブラウザでIP:9999にアクセスしてFLY-Toolsを開いてください。
:::

::: tip
FLYの上位機板載のHDMIインターフェースはMicro HDMIです。HDMIケーブルを購入する際にはご注意ください！！！
:::

::: tip
FLYの上位機は`FLY-PI-V2`,`FLY-C8`と`FLY-PRO-X10`でFPC一线通をサポートしています
:::

::: tip
FPCからHDMIモジュールを使用することで、Raspberry Piなどの上位機と互換性があります
:::
