:::danger 注意

* FLYエアロケーショニクスソフトウェアはM2WEまたは2.4G Wi-Fiモジュールを使用する必要があります。それ以外の場合、Wi-Fiを使用してください。
* ご注意ください、中国語のSSIDを持つWi-Fiと2.4Gおよび5GデュアルバンドWi-Fiはサポートされていません。
* mini padまたはlite2の場合、この警告を無視してください。

:::

## エアロケーショニクスソフトウェアへの接続

* Wi-Fi設定に失敗した場合はBOOTディスクでWi-Fiに接続するための手順を参照してください。シリアルポートを使用してエアロケーショニクスソフトウェアに接続することに注意してください。
* SSHに接続する前に、シリアルポートでエアロケーショニクスソフトウェアに接続することを確認してください。
* 次のコマンドを実行します。パスワードの入力を求められる場合があります。パスワード`mellow`を入力し、エンターを押してください。パスワードは表示されません。

```cfg
sudo nmtui
```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

* `Activate a connection`を選択し、エンターを押します。

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

* 次の画面が表示されます。キーボードの`↑ ↓`キーを使用して接続するWi-Fiを選択し、エンターを押します。パスワードを入力し、接続が完了するまで待ちます。

* ESCキーを押して終了し、コマンドラインインターフェースに戻ります。`ip a`を入力すると、図中の`192.168.2.126`がエアロケーショニクスソフトウェアのIPアドレスです。
