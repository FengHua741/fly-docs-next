:::danger 注意事項

* FLYオペレーティングマシンを使用するには、M2WEまたは2.4G Wi-Fiモジュールが必要です。それ以外の場合、Wi-Fiは使用できません。
* 注意してください。日本語のSSIDを持つWi-Fiや、2.4Gと5Gが統合されたWi-Fiはサポートされていません。
* mini padまたはlite2をお使いの場合はこの警告を無視してください。

:::

## オペレーティングマシンに接続する

* BOOTディスクでWi-Fiの設定に失敗した場合、この設定を参考にしてWi-Fiに接続できます。ただし、シリアルポートを使用してオペレーティングマシンに接続する必要があります。
* SSHに接続するためにまずシリアルポートを使用してください。([クリックしてジャンプ](/docs/DebugDoc/BasicTutorial/index.mdx "クリックしてジャンプ"))
* 次のコマンドを実行します。パスワードの入力を求められる場合があります。パスワード`mellow`を入力し、エンターキーを押してください。（パスワードは表示されません）

```cfg
sudo nmtui
```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui1.webp').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

* `Activate a connection`を選択してエンターキーを押します

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

* 次の画面が表示されます。キーボードの`↑ ↓`キーを使用して接続したいWi-Fiを選択し、エンターキーを押してパスワードを入力後、接続が完了するまで待ちます

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.webp').default} size="80%" align="left" />


        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.webp').default} size="80%" align="left" />

* 接続が成功すると以下のような画面になります

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.webp').default} size="80%" align="left" />

* ESCキーを押して終了し、コマンドラインインターフェースに戻ります。`ip a`を入力すると、図中の`192.168.2.126`がオペレーティングマシンのIPアドレスになります。

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui7.webp').default} size="80%" align="left" />
