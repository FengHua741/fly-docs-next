# システムのTFカードへの焼付け

### 準備作業

    * `Win32DiskImager`ソフトウェアをインストールする
    * システムを展開し、展開後のファイル名は`.img`で終わる
    * システム焼付けには`balenaEtcher`を使用しないことを推奨します
    * TFカードをリーダーに挿入し、リーダーをコンピューターのUSBポートに接続する
    * TFカードにデータがある場合はバックアップしてください。システムカードは焼付け後も取り出さないでください。ただし、M2WEに焼付けしてから取り外す必要がある場合は除きます

### メモリーカードのフォーマット

    * `TFFormatter.exe`ソフトウェアを使用してTFカードをフォーマットする
    * フォーマットしたいディスクを選択する

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * または、Windowsのディスク管理を使用してパーティションを削除し、パーティションを削除した後に再びパーティションを作成する。
