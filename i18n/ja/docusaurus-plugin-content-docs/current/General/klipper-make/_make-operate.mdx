    ::: tip

    * 上位机が正常にネットワーク接続されていることを確認してください
    * ファームウェアのビルド前にSSHを介してネットワーク経由で上位機にログインする必要があります
    * シリアルコンソールツールを使用せずに上位機にログインしないでください

    :::

### Klipperファームウェアのビルドと設定ページ操作説明

    ::: tip

    キーボード入力モードが半角（英語）であることを確認してください

    :::

    1. キーボードの`↑`キーは、カーソルを上に移動させ、メニュー項目を選択します
    2. 確認キー`Enter`またはスペースキー`Space`は、メニュー項目を選択またはサブメニューに進むために使用します
    3. 戻るキー`ESC`は、前のメニューに戻るために使用します
    4. `Q`キーは、Klipperファームウェア設定ページから退出するために使用します
    5. `Y`キーは、Klipperファームウェア設定ページから退出する際に、確認メッセージがある場合に、`Y`キーをクリックして設定ファイルを保存するために使用します

    ::: warning

    設定ページのオプションが少ない場合は、最初に`[ ] Enable extra low-level configuration options`を選択して、一部の隠しオプションを表示してください

    :::
