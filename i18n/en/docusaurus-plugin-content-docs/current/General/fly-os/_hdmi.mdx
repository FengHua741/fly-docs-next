## FLY Host Computer User Guide

:::tip
Open your IP:9999 in the browser to access FLY-Tools
:::

:::tip
The HDMI interface on the FLY host computer board is Micro HDMI, please pay attention when purchasing an HDMI cable!!!
:::

:::tip
The FLY host computer supports FPC one-wire-through for `FLY-PI-V2`, `FLY-C8`, and `FLY-PRO-X10`
:::

:::tip
Using an FPC to HDMI module can be compatible with host computers like Raspberry Pi
:::
