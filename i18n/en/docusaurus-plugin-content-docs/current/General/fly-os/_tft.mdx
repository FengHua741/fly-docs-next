:::tip
Open your IP:9999 in the browser to access FLY-Tools
:::

* You need to turn on the following options to enable the screen when using TFT.
* Please note that there are three types of TFT screens:
    * `FLY-TFT-V1` V1 resistive screen, compatible with all FLY hosts.
    * `FLY-TFT-V2-R` V2 resistive screen, compatible with C8 and PI-V2 and Raspberry Pi with 40-pin series, but requires G2T to be used together.
    * `FLY-TFT-V2-C` V2 capacitive screen, compatible with C8 and PI-V2 and Raspberry Pi with 40-pin series, but requires G2T to be used together.
