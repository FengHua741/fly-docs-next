:::danger Notice

* The FLY host computer needs to use the M2WE module or the 2.4G WiFi module, otherwise WiFi cannot be used.
* Please note that Chinese name Wi-Fi and 2.4G and 5G dual-band combined Wi-Fi are not supported.
* If it is mini pad or lite2, please ignore this warning.

::

## Connect Host Computer

* If configuring Wi-Fi fails under the BOOT disk, you can refer to this configuration for Wi-Fi connection. Note that a serial port connection to the host computer is required.
* First, use the serial port to [connect to SSH](/docs/DebugDoc/BasicTutorial/index.mdx "Click to jump").
* Execute the following command. You may be prompted to enter a password. Enter the password `mellow` and press Enter. Note that the password is invisible.

```cfg
sudo nmtui
```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

* Select `Activate a connection` and press Enter.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

* Enter the following interface. Use the arrow keys (`↑ ↓`) on the keyboard to select the Wi-Fi to connect to, then press Enter, input the password, and wait for the connection to succeed.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.webp').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.webp').default} size="80%" align="left" />

* Press the ESC key to exit and return to the command line interface. Enter `ip a`. The IP address `192.168.2.126` in the figure is the IP address of the host computer.
