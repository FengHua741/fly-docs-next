:::danger Notice

* System compatibility issues, please use the computer's `USB2.0` for burning
* If it still cannot be burned, please use the host machine for burning

:::

1. Burning tool <Button variant="contained" disableElevation href="https://mellow-next.klipper.cn/docs/ResDownload/auxiliary_software/">Download link</Button>
2. Need to download `SetupSTM32CubeProgrammer` and `STM32 DFU driver`
3. Copy the firmware (klipper.bin) compiled in front or the precompiled `Katapult firmware` to any directory on the computer
4. Use a Type-C data cable to connect the motherboard to the computer, make sure it is already:<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-burning-mode">Enter DFU burning mode</Button>
5. In STM32CubeProgrammer, select USB mode, refresh, and connect
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-1.webp').default} size="100%" align="center" />
6. If no error pop-up appears, then the connection is successful.
   :::danger Warning
    If the content shown below appears, it indicates a problem with the computer or software, please use the host machine for burning:<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#burn-firmware">Host machine burning tutorial</Button>
    :::
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-7.webp').default} size="100%" align="center" />
7. Click the download icon on the left and select several options below
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-2.webp').default} size="100%" align="center" />
8. Click the `Browse` button indicated by the arrow, then select the firmware you need to burn
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-3.webp').default} size="100%" align="center" />
9. Select `Start address` and enter `0x08000000`
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-4.webp').default} size="100%" align="center" />
10. Click `Start automatic mode` below to start burning the firmware
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-5.webp').default} size="100%" align="center" />
11. The prompt below indicates that the firmware burning is complete
    <ImageView image={require('@site/docs/General/dfu-flash/img/stm32cube-6.webp').default} size="100%" align="center" />
