Here's how to compile the firmware:

* After connecting to SSH, enter the following command and press Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* The `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` command deletes previous compilation data and firmware, while `make menuconfig` compiles the firmware. After execution, the following interface should appear:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Select **Enable extra low-level configuration options** and press Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Enter the **Micro-controller Architecture** menu, then select **STMicroelectronics STM32** and press Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Enter the **Processor model** menu, select **STM32F072** and press Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.webp').default} size="100%" align="Center" />

* Select **Bootloader offset**, choose: **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* Select **Communication interface**, choose: **USB to CAN bus bridge (USB on PA11/PA12)** and press Enter
* After selecting **USB to CAN bus bridge (USB on PA11/PA12)**, the default parameter **CAN bus interface (CAN bus (on PB8/PB9))--->** does not need to be changed
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc.webp').default} size="100%" align="Center" />   
    
* Please verify the following image before proceeding with the compilation

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc2.webp').default} size="100%" align="Center" />
