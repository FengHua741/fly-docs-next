Here is how to compile the firmware:

* After connecting to SSH, enter the following command and press Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Where `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` deletes previous compilation data and firmware, `make menuconfig` compiles the firmware, and after execution, the following interface should appear:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Select **Enable extra low-level configuration options** and press Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Enter the menu **Micro-controller Architecture** then select **STMicroelectronics STM32** and press Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

Updating images
* Enter the menu **Processor model**, select **STM32F405** and press Enter:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.webp').default} size="100%" align="Center" />

* Select **Bootloader offset**, choose: **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.webp').default} size="100%" align="Center" />

* Select **Communication interface**, choose: **USB to CAN bus bridge (USB on PA11/PA12)**
