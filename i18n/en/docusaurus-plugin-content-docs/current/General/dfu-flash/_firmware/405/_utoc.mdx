Here's how to compile the firmware:

* Connect to SSH and enter the following command, then press Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Here, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` deletes previous build data and firmware, while `make menuconfig` compiles the firmware. After execution, the following interface should appear:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Select **Enable extra low-level configuration options** and press Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Enter the **Micro-controller Architecture** menu, then choose **STMicroelectronics STM32** and press Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

Updating images
    * Enter the **Processor model** menu, select **STM32F405** and press Enter
