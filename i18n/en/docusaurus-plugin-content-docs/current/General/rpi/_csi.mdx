## CSI Usage Method

    * After the mirror writing is completed, enter /boot/config.txt to find `camera_auto_detect=1` and modify it to `camera_auto_detect=0`, then add configuration according to the camera model
    * The following is a reference configuration

    ```bash
    dtoverlay=imx219,cam1
    ```
