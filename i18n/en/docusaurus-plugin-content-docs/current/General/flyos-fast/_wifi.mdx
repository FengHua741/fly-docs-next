    ## FLYOS-FAST System WiFi Connection
    :::tip
    - If the **WiFi connection fails** configured in the `FlyOS-Conf` disk, you can follow this tutorial to connect to WiFi by connecting to the host machine via serial port.
    :::

    1. Please first connect to the host machine via serial port: <Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-serial-port-connection-to-host-machine-ssh">FLYOS-FAST Serial Port Connection to Host Machine SSH</Button>

    2. In SSH, enter the `nmtui` command, press the `Enter` key, and an interface as shown in the figure below will pop up
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi2.webp').default} size="40%" align="center" />

    3. Select the `Activate a connection` option, press the `Enter` key, and enter the next level menu
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi3.webp').default} size="40%" align="center" />

    4. If your router's WiFi is working properly, you will see a list of WiFi networks as shown in the figure below. The `*` after the WiFi name indicates signal strength
    :::tip
    - **WiFi with Chinese names is not supported!!!**
    - 2.4G WiFi is supported, but whether 5G WiFi is supported depends on whether the purchased WiFi module supports 5G WiFi
    - `M2WE` supports 5G WiFi
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi4.webp').default} size="40%" align="center" />

    5. Select the `WiFi` you want to connect to, press the `Enter` key, and enter the next level menu
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi5.webp').default} size="40%" align="center" />

    6. Next, the following interface will pop up. Please correctly enter your WiFi password, press the `Enter` key, and connect to WiFi
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi6.webp').default} size="40%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi7.webp').default} size="40%" align="center" />

    7. If the password is correct, a symbol `*` will appear before the corresponding `WiFi`, indicating that the WiFi connection is successful
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi8.webp').default} size="40%" align="center" />

    8. Then press the `Esc` key twice to return to the command line interface
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi9.webp').default} size="60%" align="center" />
    
    9. In SSH, enter `ip a | grep inet`, press the `Enter` key, and find the IP address in the returned content that is in the same segment as your router to access it
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi10.webp').default} size="90%" align="center" />

    10. Open a browser, enter the `IP` of the host machine you just queried in the search box, press the `Enter` key, and you can enter the fluidd/mainsail WEB control console
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi11.webp').default} size="60%" align="center" />
    
    :::tip
    - In the `FLYOS-FAST` system, you can switch fluidd/mainsail WEB control consoles in real time by clicking the mainsail/fluidd icon in the upper right corner!
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi12.webp').default} size="90%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi13.webp').default} size="90%" align="center" />
