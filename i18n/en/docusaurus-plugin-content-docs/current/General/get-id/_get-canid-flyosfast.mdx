## Fill in CAN ID

    1. Close `DEVICES` and fill in the ID
    :::danger Notice

    * The # before the first red box needs to be deleted, please note that this includes spaces, then change `serial:` to `canbus_uuid:`
    * FLY system default configuration may contain `[MCU]`, but if the ID below is `serial: /tmp/klipper_host_mcu`, rename `[MCU]` to `[mcu HOST]`, or delete these two lines
    * You can copy the following `canbus_uuid:`
    :::

    ```cfg
    [mcu]
    canbus_uuid:
    ```

    <table>
      <tr>
        <td>
          - Find [MCU] in fluidd

    2. Fill in the ID in the configuration
    :::tip

    * After filling in the ID, click `SAVE & RESTART` on the top right
    :::

    3. If Klipper prompts `ADC out of range`, this is a normal phenomenon. Connect the heated bed and thermistor to the motherboard, configure the thermistor pins for the nozzle and heated bed, then save and restart.

:::
