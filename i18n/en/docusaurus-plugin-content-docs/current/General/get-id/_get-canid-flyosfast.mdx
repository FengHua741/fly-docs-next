## Fill in CAN ID

    1. Close `DEVICES` and fill in the ID
    :::danger Note
    * Please modify the following content in the configuration file:  
    ```
    [mcu] 
    serial: /tmp/klipper_host_mcu
    ```  
    to
    ```
    [mcu host] 
    serial: /tmp/klipper_host_mcu
    ```
    * Add:
    ```
    [mcu]
    canbus_uuid: <Replace with the ID you just queried>
    ```
    :::

    <table>
      <tr>
        <td>
          - Find [MCU] in fluidd

    * After filling in the ID, click `SAVE & RESTART` on the top right
    :::

    3. If Klipper prompts `ADC out of range`, this is a normal phenomenon. Connect the heated bed and thermistor to the motherboard and configure the thermistor pins of the nozzle and heated bed, then save and restart.
