## Fill in USB ID

    1. Close `DEVICES` and fill in the ID

    :::danger Note
      * Please modify the configuration file as follows:  
      ```
      [mcu] 
      serial: /tmp/klipper_host_mcu
      ```  
      to
      ```
      [mcu host] 
      serial: /tmp/klipper_host_mcu
      ```
      * Add:
      ```
      [mcu]
      serial: <replace with the ID you just queried>
      ```
      :::
    <table>
      <tr>
        <td>
          - Find [MCU] in fluidd
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_cfg.webp').default} size="100%" align="center" />
        </td>
        <td>
          - Find [MCU] in mainsail
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_cfg.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>


    2. Fill in the ID in the configuration
    :::tip

    * After filling in the ID, click `SAVE & RESTART` on the top right
    :::
    <table>
      <tr>
        <td>
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_save.webp').default} size="100%" align="center" />
        </td>
        <td>
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_save.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>
    3. If Klipper prompts `ADC out of range`, this is a normal phenomenon. Connect the heated bed and thermistor to the motherboard, configure the thermistor pins for the nozzle and heated bed, then save and restart.

:::danger
Note: All IDs mentioned in the document are examples. Each motherboard has a different ID, please fill in the actual obtained ID.
:::
