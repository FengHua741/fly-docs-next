# {props.name} System Download

:::warning

SD Card Precautions:

* Capacity must be no less than `16GB` or no more than `128GB` TF card
* TF card transfer speed must be `class10` or higher, it is recommended to use TF cards from brands like SanDisk

:::

:::danger Notice

* Due to server issues, please do not update the system or klipper arbitrarily
* Please download FLY OS system, installing original Armbian is too difficult and not recommended

:::

:::info Important Note

* Please note that the downloaded system needs to be unzipped, not just changing the extension or burning directly!!!!
* The system cannot start if it is not unzipped before burning!!!!

:::
