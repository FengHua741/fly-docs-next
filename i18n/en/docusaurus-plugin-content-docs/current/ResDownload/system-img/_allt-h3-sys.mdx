# {props.name} System Download

:::warning

SD Card Precautions:

* Capacity must be no less than `16GB` or no more than `128GB` TF card
* The transfer speed of the TF card must be `class10` or above, it is recommended to use TF cards from brands like SanDisk

:::

:::danger Notice

* Due to server issues, please do not update the system or klipper arbitrarily
* Please download the FLYOS-Armbian system; installing the original Armbian is not recommended due to its high difficulty

:::

:::info Important Reminder

* Please note that the downloaded system needs to be decompressed instead of directly changing the suffix or burning it!!!!
* The system cannot boot if not decompressed before burning!!!!

:::
