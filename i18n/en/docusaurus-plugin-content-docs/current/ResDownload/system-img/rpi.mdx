# Raspberry Pi Image


:::warning

SD card注意事项:

* The capacity must be no less than `16GB` and no more than `128GB` for the TF card.
* The transmission speed of the TF card must be at least `class10` or above, it is recommended to use TF cards from brands like SanDisk.

:::

:::danger Notice

* Do not update the system or klipper casually due to server issues.
* This system image is suitable for Raspberry Pi 3B/3B+/4B/CM4/5B.
* This image is based on `bookworm` with various services installed.
* This image comes with TFT screen enabled by default. If you do not need to use the TFT screen, you will need to manually modify `/boot/config.txt`.

:::

:::info Important Note

* Please note that the downloaded system needs to be decompressed before use, it cannot be used directly by changing the extension or burning it directly!!!!
* Failing to decompress before burning will result in the system failing to start!!!!

:::
