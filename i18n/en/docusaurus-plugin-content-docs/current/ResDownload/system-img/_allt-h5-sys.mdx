# {props.name} System Download

:::warning

SD Card Notes:

* Capacity must be no less than `16GB` or no more than `128GB` TF card
* The transfer speed of the TF card must be Class10 or above, it is recommended to use brands like SanDisk

:::

:::danger Notice

* Due to server issues, please do not update the system or klipper arbitrarily
* Please download the FLYOS-Armbian system, installing the original Armbian is not recommended due to its high difficulty

:::

:::info Important Prompt

* Please note that the downloaded system needs to be extracted, and it cannot be directly renamed or burned without extraction!!!!
* If the system is not extracted before burning, it will fail to boot!!!!

:::
