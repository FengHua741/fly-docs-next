# {props.name} System Download

:::warning

SD Card Precautions:

* Capacity must not be less than `16GB` or greater than `128GB` TF card
* TF card transfer speed must be `class10` or above, it is recommended to use TF cards from brands like SanDisk

:::

:::danger Notice

* Due to server issues, please do not update the system or klipper arbitrarily
* Please download FLY OS system, installing the original Armbian is too difficult and not recommended

:::

:::info Important Note

* Please note that the downloaded system needs to be decompressed, do not directly change the extension or burn it directly!!!!
* The system will not start if not decompressed before burning!!!!

:::
