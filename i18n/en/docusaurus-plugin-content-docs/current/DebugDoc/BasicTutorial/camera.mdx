# Adding a USB Camera


:::danger Note

* The `crowsnest` service needs to be installed. If not installed on systems like lite2 and mini pad.
* A plug-and-play camera without the need for drivers is required. These cameras are supported by klipper under linux, armbian, etc., without the need for additional driver installation.
* This tutorial uses the latest klipper, crowsnest, fuidd as an example. If using other methods, you will need to research on your own.

:::

## Open Configuration

:::tip

* The camera ID can vary depending on the system, we just need to determine the ID that won't change.

:::

    * Find the configuration file in Fuidd, then open `crowsnest.conf`
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf.webp').default} size="100%" align="center" />

    * Find `device: ` and delete the ID configuration after it
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf1.webp').default} size="100%" align="center" />

## Search for Camera ID

:::tip

* Ensure the camera is connected to the host machine.

:::

    * Click on `DEVICES` at the top, then click `VIDEO`, and finally click refresh
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf2.webp').default} size="100%" align="center" />
    * Find `path_by_id` and copy the line following it
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf3.webp').default} size="100%" align="center" />

## Add ID

    * Add the ID and save the configuration
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf4.webp').default} size="100%" align="center" />

## Enable Camera

    * Click settings, select the camera, and then click add camera
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf5.webp').default} size="100%" align="center" />
    * Enter a preferred name and click save
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf6.webp').default} size="100%" align="center" />

## Check if Enabled

    * Click on the dashboard; normally, the camera feed should appear
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf7.webp').default} size="100%" align="center" />
    * If it does not display, restarting the system should resolve the issue
