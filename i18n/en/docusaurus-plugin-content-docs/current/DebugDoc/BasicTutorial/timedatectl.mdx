# Modify System Time Zone

    Please note that if you are using FLY OS 3.3.4 version on the FLY host machine, you can directly use FLY-Tools to modify the time zone.
    For other host machines, please see the following operations.


    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    Replace `<TIMEZONE>` with the time zone code you wish to use. You can find the time zone codes at [iana.org](https://www.iana.org/time-zones). For example, if you want to change your system time zone to Beijing time (Asia/Shanghai), you can enter the following command:

    ```bash
    sudo timedatectl set-timezone Asia/Shanghai
    ```

    Now, your system time zone has been successfully changed to Shanghai time. You can verify the time zone setting with the following command:

```bash
date
```
