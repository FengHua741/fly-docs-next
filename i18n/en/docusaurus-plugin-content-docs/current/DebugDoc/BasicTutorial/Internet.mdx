# Connecting to the Network via SSH


* Please ensure that the system has started and is connected to the host machine via SSH using a serial port, it is recommended to use **MobaXterm_Personal**
* Please ensure that the host machine has installed **M2WE** or **2.4G_WiFi module**, if it is **Lite2** or **MiniPad**, no module installation is required
* Please note that the **2.4G_WiFi module**, **Lite2**, and **MiniPad** can only use 2.4G WiFi and cannot use 5G WiFi
* Please note that the network cannot use dual-band fusion, otherwise normal connection cannot be achieved
* Please ensure that the antenna is fully installed!!!

 1. After entering the **nmtui** command in the SSH software and pressing Enter, the interface shown in the figure below will appear

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. Press Enter then select **Activate a connection** and press Enter again

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. Select the network you need to connect to, press Enter, input the WiFi password, and then select OK to save the connection

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. After connecting to WIFI, find and press the **Esc** key several times to exit the current interface, then enter **ip a** and press Enter

 * In the figure, 192.168.2.136 is the IP address of the host machine

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
