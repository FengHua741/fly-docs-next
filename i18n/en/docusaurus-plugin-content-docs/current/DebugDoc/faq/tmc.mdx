# TMC Error Related

## `Unable to read tmc uart ‘stepper_x’ register IFCNT` or `Unable to read tmc spi ‘stepper_x’ register register`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart.webp').default} size="80%" align="center" />

    2. Error Cause

        * Unable to communicate with the driver

    3. Main Solution Method

        * Check if the UART pin configuration of the corresponding driver is correct
        * Is the UART jumper correctly set?
        * Is the driver powered?
        * Replace the driver or reconfigure the driver port for judgment
        * If it's a driver from another brand, you need to determine whether the driver's UART pin is consistent with FLY's. The FLY driver UART pin is the fourth pin counting down from the top on the left side.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/2209uart.webp').default} size="80%" align="center" />
        * If the driver's UART pin is on the fifth pin, you can solve it by following the jumper settings below.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart1.webp').default} size="80%" align="center" />

## `TMC XXXX reports error: GSTAT: 00000001 reset=1(Reset)`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/reset.webp').default} size="80%" align="center" />

    2. Error Cause

        * Driver reset

    3. Main Solution Method

        * Poor contact of power wiring
        * Motor wire break
        * You can connect the metal casing of the motor with the GND of the motherboard or the tool board together.

## `TMC 'extruder' reports error: DRV_STATUS: c00d0103 otpw=1(OvertempWarning!)`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/overtemperror.webp').default} size="80%" align="center" />

    2. Error Cause

        * Driver overheating!
        * It's not the motor overheating!

    3. Main Solution Method

        * Reduce operating current
        * Delete holding current
        * Check if the sampling resistor of the driver is correct
        * Enhance driver cooling, such as adding a fan

## `TMC ‘extruder’ reports error: DRV_STATUS: 601f1000 s2vsa=1(ShortToSupply_A!) csactual=31 ola=1(OpenLoad_A!) olb=1(OpenLoad_B!)`

    1. Error Details

        * No image available

    2. Error Cause

        * Incorrect driver current check

    3. Main Solution Method

        * Check if the resistance values of the two coil groups A and B of the stepper motor are consistent
        * Check if the wires in the stepper motor or internal components are loose or short-circuited
        * Turn off quiet mode

## `TMC 'extruder' reports error: DRV_STATUS: 00190060 s2vsb=1(ShortToSupply_B!) ola=1(0penLoad_A!) cs_actual-25`

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/ab.webp').default} size="80%" align="center" />

    2. Error Cause

        * Wiring mistake or problem with the motor itself

    3. Main Solution Method

        * Check if the resistance values of the two coil groups A and B of the stepper motor are consistent
        * Check your stepper motor and wiring
        * Check the motor wiring sequence, whether one group is on the left and the other on the right
