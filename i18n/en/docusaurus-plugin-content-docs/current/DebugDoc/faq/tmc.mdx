# TMC Error Related

* `Unable to read tmc uart ‘stepper_x’ register IFCNT` or `Unable to read tmc spi ‘stepper_x’ register register` 

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart.webp').default} size="80%" align="center" />

    2. Error Cause

        * Unable to communicate with the driver

    3. Main Solution

        * Check if the UART pin configuration of the corresponding driver is correct
        * Whether the UART jumper is set correctly
        * Whether the driver is powered on
        * Replace the driver or configure the driver port for judgment
        * If it is a driver from another manufacturer, you need to determine whether the driver's UART pin is consistent with FLY's. The UART pin of the FLY driver is the fourth pin from top to bottom on the left.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/2209uart.webp').default} size="80%" align="center" />
        * If the UART pin of the purchased driver is on the fifth pin, you can solve it by following the jumper.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart1.webp').default} size="80%" align="center" />

* `TMC XXXX reports error: GSTAT: 00000001 reset=1(Reset) ` 

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/reset.webp').default} size="80%" align="center" />

    2. Error Cause

        * Driver reset


    3. Main Solution

        * Poor contact of power connection
        * Motor disconnection
        * You can connect the metal shell of the motor with the GND of the main board or the tool panel

* `TMC 'extruder' reports error: DRV_STATUS: c00d0103 otpw=1(OvertempWarning!) `

    1. Error Details

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/overtemperror.webp').default} size="80%" align="center" />

    2. Error Cause

        * Driver overheating!
        * Not motor overheating!

    3. Main Solution

        * Reduce operating current
        * Delete holding current
        * Check if the sampling resistor of the driver is correct
        * Enhance driver heat dissipation, such as adding a fan
---
<br/>
<br/>

* `TMC ‘extruder’ reports error: DRV_STATUS: 601f1000 s2vsa=1(ShortToSupply_A!) csactual=31 ola=1(OpenLoad_A!) olb=1(OpenLoad_B!)）`

    1. Error Details

        * No image available

    2. Error Cause

        * Incorrect detection of driver current


    3. Main Solution
        
        * Check if the resistance values of the two coil groups of the stepper motor are consistent
        * Check if the wires of the stepper motor or internal components of the motor are loose or short-circuited
        * Turn off silent mode
---
