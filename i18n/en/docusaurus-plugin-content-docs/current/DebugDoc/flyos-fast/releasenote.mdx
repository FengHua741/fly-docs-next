# System Release Notes

## V1.2.40

### Modifications

- Fix known issues
- Add new parameter `paused_recover_z` for klipper power-off resume support, used to set the Z-axis compensation height when resuming after a pause and power loss
- Update klipper to the latest version
- Update moonraker to the latest version
- Update KlipperScreen to the latest version
- Update katapult to the latest version
- Update fluidd to the latest version
- Update mainsail to the latest version

### Additions

- Add new klipper extension [klipper-led_effect](https://github.com/julianschill/klipper-led_effect.git)

----

## V1.2.39

### Modifications

- Solve the problem that Windows11 24H2 system burning leads to failure to start

### Additions

- Add new system configuration `canbus_bitrate` to set the default speed of can0, default value is `1000000` i.e., `1M`
- Add new system configuration `localtime` to set the default time zone, default value is `Asia/Shanghai`
