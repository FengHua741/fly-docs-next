# System Release Notes

## V1.2.39

### Modifications

- Resolve the issue where burning with Windows11 24H2 system causes failure to boot.

### New Features

- Add system configuration `canbus_bitrate` to set the default baud rate of can0, with a default value of `1000000` (i.e., 1M).
- Add system configuration `localtime` to set the default time zone, with a default value of `Asia/Shanghai`.
