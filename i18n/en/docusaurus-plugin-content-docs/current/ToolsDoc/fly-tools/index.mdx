# Introduction to FLY-Tools

FLY-Tools is a web debugging tool customized by Guangzhou Meilun Electronic Technology Co., Ltd. for Klipper. It is not only suitable for the Fly host computer (which comes with the system and does not require installation) but also compatible with most host computers on the market that have installed Klipper services.

:::tip Tip

* FLY-Tools is only suitable for Linux systems installed with Klipper

:::

## Features of FLY-Tools

* Multi-language support, dark theme support
* Support querying and one-click copying of USB serial port ID, USB device ID, CANbus UUID, Video device ID
* Support one-click enable or disable specified CAN devices
* Support online modification of CAN device speed and send buffer size
* Support online compilation of Klipper firmware for FLY mainboard (Katapult firmware compilation is being adapted)
* Support online burning, support DFU, HID, CAN and other burning methods
* Support downloading all compiled firmware files
* Support online generation of Klipper load chart
* Support one-click automatic measurement and generation of resonance diagram
* Support web SSH connection
* Support WEB web SSH (based on [ttyd](https://github.com/tsl0922/ttyd))
* Support FLYOS 3.3.4 version system, configurable screen, web display, etc.
