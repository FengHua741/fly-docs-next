# Feature Demonstration

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.webp').default} size="100%" align="left" />


    * Supports hostname modification
    * Supports system timezone modification
    * Supports enabling the上位机40PIN扩展启用（请注意PWM启用后无法使用串口连接上位机，屏幕启用后可能无法使用上位机的加速度口）upper computer 40PIN extension (please note that after PWM is enabled, the serial port cannot be used to connect to the upper computer, and the acceleration port of the upper computer may not be used after the screen is enabled)
    * Supports motherboard model modification
    * Supports Klipper web page modification, supports Fuidd and Mainsail
    * Supports opening or closing KlipperScreen
    * Supports forcing the Klippy process to run on the specified CPU core, can specify Klipper to run under the selected core
