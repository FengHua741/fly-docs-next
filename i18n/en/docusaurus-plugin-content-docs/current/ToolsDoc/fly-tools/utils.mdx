# Feature Demonstration

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/utils.gif').default} size="100%" align="left" />

    * Support generation of Klipper default load graph
    * Support generation of MCU frequency graph
    * Support generation of system load graph
    * Support generation of temperature curve graph
    * Support generation of frequency response graph, and can also directly generate charts from existing files
