# Introduction

    Fly-Pi-lite2.1 is an entry-level Klipper host machine
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.webp').default} size="80%" align="center" />

## Features

    * CPU：High-performance Allwinner H3 chip, 4-core 64-bit Cortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：Supports up to 128GB SD card
    * WIFI: Onboard 2.4G WiFi
    * Peripherals: Spi x1, URAT x2, USB x2, HDMI x1
    * Interfaces:预留FPC，HDMI + USB FPC一线通
    * System pre-installed with Klipper, Moonraker, Mainsail, Fulld, Klipper-screen, resonance compensation plugin numpy and FLY-Tools, no need for complex operations like changing sources, making it easier for beginners (requires purchasing a TF card)


## Updated Features

    * TFT V1 interface upgraded to TFT V2
    * Added reset and shutdown pins
    * Added power-on indicator light
    * Removed BOOT button, replaced with reset button
    * Optimized wiring
