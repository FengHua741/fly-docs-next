## Preparations Before System Burning

    * SD card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-lite2">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning
:::info
* MINI PAD can only burn the system to the TF card and cannot use M2WE
* After the system is burned, the TF card needs to be installed on the MINI PAD
:::
    
    <TF />

## Installing the System Card

    * After burning the system to the TF card, the TF card needs to be connected to LITE2.1, otherwise the system cannot start
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tf.webp').default} size="100%" align="left" />  

## Installing the Antenna

    1. Install the antenna
    :::warning
    * Please note that the host does not support WiFi names in Chinese, and does not support 5G WiFi
    :::
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna, otherwise poor signal may cause failure to connect to the router
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After the system card is installed, you can proceed to the next step, connecting to the host via SSH
    :::

    2. Power on the host.  
    **Your host is now ready to start up!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2.1/ssh">SSH Connect to Host</Button>
