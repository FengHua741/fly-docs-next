# System Image Flashing

<FASTFASLH name="H3" />

            
## Install System Card


    * After flashing the system to the TF card, you need to connect the TF card to the back of LITE2.1, otherwise the system cannot start
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tf.webp').default} size="100%" align="left" />  

## Install Antenna

    1. Install the antenna
    :::warning
    * Please note that the upper computer does not support WiFi with Chinese names and does not support dual-band (2.4G and 5G)合一WiFi
    :::
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna, otherwise weak signal may cause inability to connect to the router
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After completing the installation of the system card, you can proceed to the next step, connecting to the upper computer via SSH
    :::

    2. Power on the upper computer.  
    **Your upper computer can now start up!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2.1/ssh">SSH Connect to Upper Computer</Button>
