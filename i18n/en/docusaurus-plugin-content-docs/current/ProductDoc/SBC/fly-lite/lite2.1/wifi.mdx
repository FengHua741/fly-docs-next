# Connect to WIFI

## Install the antenna on LITE 2.1

    * The red arrow points to the WIFI antenna interface
    * The WIFI antenna port is `ipex1`
    * Install the antenna on LITE 2.1, otherwise poor signal may result in failure to connect to the internet

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="100%" align="left" />  

* If you find that WIFI is not connected properly, check first if the antenna is connected, and then use the method below to connect to WiFi
