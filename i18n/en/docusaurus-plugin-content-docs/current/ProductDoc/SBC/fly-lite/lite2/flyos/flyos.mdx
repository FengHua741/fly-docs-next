# System Writing

## Preparations Before System Writing

    * SD card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, and the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-lite2">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">writing tool</Button>

## System Writing
:::info
* MINI PAD can only burn the system to the TF card and cannot use M2WE
* After the system is burned, the TF card needs to be installed on the MINI PAD
:::
    
    <TF />

## Install System Card

    * After burning the system to the TF card, the TF card needs to be connected to lite2, otherwise the system cannot start
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/tf.webp').default} size="100%" align="left" />  

## Install Antenna

    1. Install the antenna
    :::warning
    * Please note that the host does not support WIFI with Chinese names and does not support 5G WIFI
    :::
    * The red arrow points to the WIFI antenna interface
    * The WIFI antenna port is `ipex1`
    * Please install the antenna, otherwise weak signal may cause failure to connect to the router
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After the system card is installed, you can proceed to the next step, connecting to the host via SSH
    :::

    2. Power on the host.  
    **Your host can now start up!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2/ssh">SSH Connect to Host</Button>
