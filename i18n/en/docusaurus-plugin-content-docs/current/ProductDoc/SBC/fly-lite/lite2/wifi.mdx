# Connect to Wi-Fi

## Install the Antenna on LITE 2

    * The red arrow points to the Wi-Fi antenna interface.
    * The Wi-Fi antenna port is `ipex1`.
    * Install the antenna on LITE 2, otherwise, poor signal may result in failure to connect to the internet.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="100%" align="left" />  

* If you find that it does not connect to Wi-Fi normally, first check if the antenna is connected, and then use the method below to connect to Wi-Fi.
