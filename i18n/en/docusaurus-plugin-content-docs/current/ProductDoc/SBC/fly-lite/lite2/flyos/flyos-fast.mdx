## Installing the System Card

* After burning the system to the TF card, you need to connect the TF card to the back of LITE2, otherwise the system cannot start
<ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/tf.webp').default} size="100%" align="left" />  

## Installing the Antenna

1. Install the antenna
:::warning
* Please note that the upper computer does not support WiFi with Chinese names and does not support dual-frequency (2.4G and 5G)合一 WiFi
:::
* The red arrow points to the WiFi antenna interface
* The WiFi antenna port is `ipex1`
* Please install the antenna, otherwise the weak signal may cause failure to connect to the router
<ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="80%" align="left" /> 

:::tip
- After the system card is installed, you can proceed to the next step and connect to the upper computer via SSH
:::

2. Power on the upper computer.  
**Your upper computer can now be started!!!**  
Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2/ssh">SSH Connect to Upper Computer</Button>
