## Preparation Before System Burning

    * Card Reader
    * One TF card, the capacity must be no less than 16GB or no more than 128GB, the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">System Image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Burning Tool</Button>

## System Burning

:::info
TF card burning and burning to M2WE through a TF card only requires using one of the burning methods.
:::

:::info
* If the machine is installed with M2WE and memory card, the system will default to boot from the memory card.
* It is recommended to use the TF card to burn the system; burning to M2WE via USB may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image is correct. If it does not match, the system cannot start.
:::
