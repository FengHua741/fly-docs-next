## Installing the Antenna

    1. Install the antenna for `M2WE` or `2.4G WiFi` module
    :::warning
    * FLY host machine requires using `M2WE` or `2.4G WIFI` module, otherwise WiFi cannot be used
    * Please note that the host machine does not support WIFI with Chinese names and does not support dual-band WIFI combining 2.4G and 5G
    :::
    * The red arrow points to the WIFI antenna interface
    * The WIFI antenna port is `ipex1`
    * Please install the antenna on the M2WE, otherwise it may fail to connect to the router due to poor signal
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After completing the installation of the system card, you can proceed to the next step and connect to the host machine via SSH
    :::

    2. Power on the host machine.  
    **At this point, your host machine has already started up!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-pi/ssh">Connect to Host Machine via SSH</Button>
