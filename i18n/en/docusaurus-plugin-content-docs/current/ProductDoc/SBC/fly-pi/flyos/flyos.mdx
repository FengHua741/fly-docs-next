## Preparations Before System Burning

    * SD card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, and the transmission speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-pi-v2">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
TF card burning and TF card burning to M2WE only require using one of the burning methods.
:::

:::info
* If the machine is installed with M2WE and memory card, the system defaults to boot from the memory card
* It is recommended to use TF card to burn the system for M2WE; burning through USB may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image corresponds. If it does not correspond, the system cannot start.
:::

## Installing the Antenna

    1. Install the antenna for `M2WE` or `2.4G WiFi` module
    :::warning
    * FLY host computer requires the use of `M2WE` or `2.4G WIFI` module; otherwise, WiFi cannot be used
    * Please note that the host computer does not support WiFi with Chinese names and does not support dual-band (2.4G and 5G) integrated WiFi
    :::
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna on the PI V2, otherwise the connection to the router may fail due to poor signal
