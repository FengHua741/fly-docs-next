# System Burning

## System Burning Preparation

    * Card reader
    * A TF card with a capacity of no less than 16GB or no more than 128GB, the transmission speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-minipad">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning
:::info
* MINI PAD can only burn the system to the TF card and cannot use M2WE
* After the system is burned, the TF card needs to be installed on the MINI PAD
::>
    
    <TF />

## Installing the System Card

    * After burning the system to the TF card, you need to connect the TF card to the back of the MINI PAD, otherwise the system cannot start
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/tf.webp').default} size="100%" align="left" />  

## Installing the Antenna

    1. Install the antenna
    :::warning
    * Please note that the host computer does not support WiFi names in Chinese and does not support 5G WiFi
    ::>
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna, otherwise the weak signal may cause failure to connect to the router
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After completing the installation of the system card, you can proceed to the next step, connecting to the host computer via SSH
    ::>

    2. Power on the host computer.  
    **Your host computer can now start up!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-mini-pad/ssh">SSH Connection to Host Computer</Button>
