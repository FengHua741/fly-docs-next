# Onboard UTOC Usage

:::danger Notice

It is not recommended to use two UTOICs; flashing and connecting will be very troublesome.

:::

- If you do not need to use the onboard UTOC, execute the following command without any prompts:
    ```bash
    sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
    ```
- Restoration method, executed without any prompts:
    ```bash
    sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
    ```
- The MINI Pad already has an onboard UTOC.
