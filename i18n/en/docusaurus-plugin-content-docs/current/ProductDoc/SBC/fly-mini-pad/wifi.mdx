# Connect to WIFI

## Install the antenna on the MINI PAD

    * The red arrow points to the WIFI antenna interface
    * The WIFI antenna port is `ipex1`
    * Install the antenna on the MINI PAD, otherwise it may fail to connect to the internet due to poor signal
