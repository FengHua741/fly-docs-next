# Introduction

    Fly-MINI-Pad is an entry-level host for Klipper.

## Features

    * CPU：High-performance Allwinner H3 chip, quad-core 64-bit Cortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：Supports up to 128GB SD card
    * WIFI : On-board 2.4G WiFi
    * Peripherals：SPI x1，USB x2，Type-C x1
    * Interfaces：Three XH2.54-2P interfaces respectively support, POWER-IN power supply supports maximum 24V input, CAN-BUS supports CAN bus connection, LI-PO supports 3.7V and protected lithium battery power/charging interface
    * System pre-installs klipper, moonraker, mainsail, fulidd, klipper-screen, resonance compensation plugin numpy and FLY-Tools, no need for complex operations such as changing sources, making it easier for beginners (requires purchase of TF card)
    * Screen capacitive/resistive, resolution 320*480
