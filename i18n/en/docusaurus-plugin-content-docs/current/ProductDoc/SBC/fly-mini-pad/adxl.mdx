# Accelerometer Usage

    :::tip

    * Mini Pad can directly connect to the accelerometer via the onboard interface or use a tool board with a built-in accelerometer.

    :::

## External Accelerometer via Onboard Interface

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.webp').default} size="80%" align="left" />

    * Reference Configuration

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # an example
    ```
