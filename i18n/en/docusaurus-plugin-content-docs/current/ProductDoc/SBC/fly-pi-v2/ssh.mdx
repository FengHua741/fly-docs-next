# Connect to Host Computer via SSH

* You can use **serial port** or **WiFi** to connect to the device
* Please prepare in advance **MobaXterm_Personal** or other **SSH terminal tools**
* Please confirm that the SD card or M2WE has burned the system **compatible with the host computer**
* **If the host computer cannot start up properly, please remove all peripherals such as drivers, limiters, fans, etc. !!!**

:::warning Note
- If this is the first time your host computer is starting up, you must use **serial port connection to the host computer SSH**.
- When compiling the firmware, you must use **WiFi connection to the host computer SSH** to facilitate firmware compilation and burning!
- The operation of connecting to SSH for the `FLYOS-Armbian` system and the `FLYOS-FAST` system is slightly different, please choose the corresponding tutorial according to the system you burned!!!
::>

    <TabItem value="net" label="FLYOS-Armbian System WiFi Connection" default>
        ## FLYOS-Armbian WiFi Connection to Host Computer SSH

        * Please ensure the host computer system has **started up normally**
        * Please make sure you know the **username and password of the host computer**. FLY system default username is `fly`, default password is `mellow`
        * The prerequisite for connecting to the host computer SSH via WiFi is: your **host computer has already connected to the router**, and you know the host computer's IP  

        * Open MobaXterm, set it up as shown below, fill in the host computer's IP you queried during **serial port connection** in the `Remote host` field, or query the host computer's IP through the router backend

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Note: In step 4 here, input the host computer's username**
        :::tip
        - In step 4, input the host computer's username. For the FLYOS-Armbian system, fill in `fly`. For other host computers, please fill in the correct username based on the actual situation
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * The first connection will pop up this window, click `Accept` to continue

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Prompt to enter the password, enter the host computer password and press Enter
        :::tip
        - The password entered here will not be displayed, just press Enter after typing
        :::
        
        Default username for FLYOS-Armbian system: `fly`, password: `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        * If the following window pops up, this window prompts whether to save the user password, click `No` to dismiss

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * If the following screen appears, login is successful!

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
