## Preparation Before System Burning

    * Card reader
    * One TF card with a capacity of not less than 16GB or not more than 128GB, and the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
TF card burning and TF card burning to M2WE only requires using one of the burning methods
:::

:::info
* If the machine is installed with M2WE and memory card, the system will boot from the memory card by default
* It is recommended to use the TF card to burn the system for M2WE; burning through USB to M2W may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image matches. If it does not match, the system cannot start
:::
