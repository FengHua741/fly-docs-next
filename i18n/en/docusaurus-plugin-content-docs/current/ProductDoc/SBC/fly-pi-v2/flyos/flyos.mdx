## System Burning Preparation

    * Card reader
    * One TF card with a capacity of not less than 16GB or not more than 128GB, the transmission speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-pi-v2">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
TF card burning and burning to M2WE via TF card only need to use one of the burnings
:::

:::info
* If the machine is installed with M2WE and memory card, the system defaults to boot from the memory card
* It is recommended to use TF card to burn the system for M2WE; burning through USB to M2W may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please pay attention to whether the downloaded system image matches. If it does not match, it cannot be started
:::

## Install Antenna

    1. Install the antenna for `M2WE` or `2.4G WiFi` module
    :::warning
    * FLY host computer requires the use of `M2WE` or `2.4G WIFI` module, otherwise WiFi cannot be used
    * Please note that the host computer does not support Chinese-named WIFI, and does not support dual-frequency integration of 2.4G and 5G WIFI
    :::
    * The red arrow points to the WIFI antenna interface
    * The WIFI antenna port is `ipex1`
    * Please install the antenna on M2WE, otherwise it may cause connection failure to the router due to poor signal
