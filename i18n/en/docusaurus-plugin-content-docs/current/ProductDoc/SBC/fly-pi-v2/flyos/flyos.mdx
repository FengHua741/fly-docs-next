## System Imaging Preparation

    * Card reader
    * One TF card with a capacity of not less than 16GB or not more than 128GB, and the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Imaging

:::info
TF card imaging and TF card imaging to M2WE only require using one of the imaging methods
:::

:::info
* If the machine is installed with M2WE and memory card, the system will boot from the memory card by default
* It is recommended to use the TF card to burn the system for M2WE; burning through USB to M2W may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please pay attention to whether the downloaded system image matches. If it does not match, it cannot start
:::

## Install Antenna

    1. Install the antenna for `M2WE` or `2.4G WiFi` module
    :::warning
    * The FLY PC needs to use the `M2WE` or `2.4G WIFI` module, otherwise WiFi cannot be used
    * Note that the PC does not support WiFi with Chinese names and does not support dual-band 2.4G and 5G combined WiFi
    :::
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna on PI V2, otherwise the connection to the router may fail due to poor signal
