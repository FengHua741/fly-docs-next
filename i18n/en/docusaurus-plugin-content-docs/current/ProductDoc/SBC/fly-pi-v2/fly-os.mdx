## Preparation Before System Burning

    * Card Reader
    * A TF card with a capacity of no less than 16GB or no more than 128GB, the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">System Image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Burning Tool</Button>

## System Burning

:::info
Burning to a TF card or burning to M2WE via a TF card only requires using one of these methods.
:::

:::info
* If the machine has installed M2WE and memory card, the system defaults to booting from the memory card.
* It is recommended to use a TF card to burn the system to M2WE; burning through USB to M2WE may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note that the downloaded system image should match. If it does not match, the system cannot start.
:::
