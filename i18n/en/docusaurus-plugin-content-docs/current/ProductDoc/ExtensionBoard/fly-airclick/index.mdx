# AirClick Wireless Button
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.webp').default} size="100%" align="Center" />


## Product Features

    * High compatibility: Supports Klipper, RRF, BambuLab, can be directly connected to the machine and send preset instructions, providing seamless compatibility and operating experience.
    * Custom macro instructions: Users can customize instructions according to their own needs, making machine operation more flexible and convenient.
    * Support wireless connection to Klipper, RRF, BambuLab. Wired connection only supports Klipper.
    * Wireless connection: Through the design of ESP32, it can directly send custom instructions or macros to Klipper, providing users with extremely high flexibility and expandability.
    * Wired connection: Through the STM32F072 chip, simple operations can also be carried out when the network cannot connect to Klipper. Supports a total of ten custom instructions or macros.
    * Easy to use: With an intuitive user interface, users can easily add and manage wireless buttons, greatly improving operational convenience.

## Product Specifications

    * Main control chip: ESP32
    * Additional chip: STM32F072
    * Number of instructions sent through ESP32-WIFI: Unlimited
    * Number of instructions sent through STM32F072-IO: 10
    * Compatibility: Supports Klipper, RRF, BambuLab
    * Power supply: 5V

## Precautions

    :::danger Precautions

    * This product uses 5V power supply and can be powered by `type c` or `PH2.0-2P` terminals.
    * The built-in antenna of this product can only be used to receive button signals. If WIFI connection is required, the antenna needs to be installed in place, and the antenna terminal is `ipex3`.
    * Please install the antenna in place!!! Otherwise, poor signal will cause serious heating!!!
    * Please note that to prevent accidental triggering, the system will automatically turn it off after pressing the button for 15 seconds.

    :::
