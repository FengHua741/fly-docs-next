# Connect to RRF

    :::info Important Note
    * AirClick must be connected to the same network as the target printer
    :::

1. Click on System Settings on the left
    
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. Select `RRF` for `Printer Type`
3. Enter the IP address of the RRF machine in `RepRapFirmware IP`

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/rrf.webp').default} size="80%" align="Center" />

4. After configuring as shown above, click `Submit` in the bottom right corner to save the settings
5. A prompt will appear at the top of the webpage if the connection is successful

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

6. If the connection fails
    - Check if the IP address is correct
    - Check if the printer and AirClick are connected to the same network
    - Check if the RRF motherboard is functioning properly
    - Check if the browser can open the RRF webpage
