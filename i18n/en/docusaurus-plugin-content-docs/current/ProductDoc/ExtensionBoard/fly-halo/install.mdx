## Installation to Voron Stealthburner

    * Installing Halo to Stealthburner requires the use of our provided adapters
    * First, download the two STL files we provided and print them yourself
        * [Stealthburner_main_body(Adapt_to_Halo).stl(Click to download)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl(Click to download)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(Click to download)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### Install the adapter board and connecting cable
    * The outlet position can be according to your habits, it does not have to be close to the back of the fan
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.webp').default} size="50%" align="left" />

    :::warning Notes

    * Explanation of outlet colors **Black is negative pole GND**, **Red is positive pole VCC acceptable voltage is DC12-24V**, **White is GPIO47 can be used for material blocking detection**, **Yellow is GPIO48 can be used for shutdown after printing**

    :::
