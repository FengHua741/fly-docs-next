# Product Overview

    * The main control uses ESP32-S3R8, which has stronger performance than ESP32. It comes with 8MB PSRAM and 16MB Flash. The screen is an IPS screen with high resolution, high brightness, wide viewing angle, and delicate colors. It supports capacitive touch.

## Product Features

    :::tip

    * Currently, there are 6 pages, which can be switched by swiping left, right, up, and down.

    :::

    * Home page (displays standby, printing, shutdown, error, etc. states)

    * Extrusion (controls extrusion, retraction, extrusion temperature)

    * System Services (controls the start, stop, and restart of services such as the host, klipper, moonraker, etc.)

    * Settings (used to set various parameters of halo, wifi, ip, language, backlight brightness, etc.)

    * Device Information (displays halo's information, firmware version, checks for updates, etc.)

    * Material Detection: Detects material breakage and blockage, detects the length of materials, and can automatically pause and raise GPIO47 after a specified blockage length (all these are disabled by default and can be enabled in settings)

    * Shutdown after Printing: Supports shutting down the host and raising GPIO48 after a set delay following the completion of printing (all these are disabled by default and can be enabled in settings)
