# FLY_halo

## 1. Home Page

    Click on the hotbed or hot end, swipe clockwise from the top of the screen to the desired temperature, and then click on the hotbed or hot end to set the temperature.

    If you need to display the MCU temperature and the host computer temperature, add this configuration

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # Associate with MCU (default)

    [temperature_sensor host]        # Host computer temperature
    sensor_type: temperature_host     # Associate with host computer
    ```


    :::warning Notice

    * Note that the maximum temperature is set in the configuration

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.webp').default} size="50%" align="left" />


## 2. Extrusion

    * Long press the icon to set the hot end temperature, extrusion length, and extrusion speed
