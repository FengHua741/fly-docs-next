# FLY-UTOR

FLY-UTOR is a high-performance, multi-purpose interface expansion solution designed to provide users with additional USB interfaces and power expansion capabilities, while also supporting UART to RS232 communication. This expansion board is compact, easy to install, and suitable for various application scenarios requiring extra interfaces and power output.

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.webp').default} size="100%" align="left"/>

## Product Features

* Dual USB port expansion: The expansion board provides two additional USB interfaces, supporting the USB 2.0 standard to ensure compatibility with various USB devices and high-speed data transfer.
* UART to RS232 support: Built-in UART to RS232 converter, making it convenient for users to connect serial devices to the system and expand the device's communication capabilities.
* Dual power interface expansion: Provides two power interfaces, allowing users to simultaneously extend two external power sources to meet additional power needs.
* Stable power design: The expansion board is designed with a stable power circuit to ensure the stable operation of connected devices.
* Easy installation: Adopting a standard size design, suitable for various environments, no need for additional driver installation.
* Compact size: The expansion board has a reasonable size and will not occupy too much space, making it convenient to integrate into various projects.

## Product Specifications:

* Available interfaces: USB X2, UART to RS232 X2, Power interface X2
* Power interface: 12V/24V expansion
* Compatibility: Host computer with USB interface

## Precautions:

* Disconnect the power and turn off all devices before installation.
* Insert the multifunctional USB expansion board into the motherboard USB port or other USB expansion interfaces.
* Connect the required USB devices and/or power to the expansion board.
* If you need to use the UART to RS232 function, connect the corresponding serial device.
* Reconnect the power and start the device.
* The system should automatically detect the expansion board and configure it. If necessary, please refer to the configuration precautions in the operation manual.
