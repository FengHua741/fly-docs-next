# How to Adjust the Module Trigger Value
Use the serial port tool, the following command is available:
Send "SET_PIN PIN=_probe_ready VALUE=1" to the Klipper console, observe the trigger indicator light, adjust the threshold from small to large until the indicator light does not light up. Input the following command:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
Enter the following command in the serial port tool to adjust the trigger value:
```cfg
set trigger value
```
* Commands used in the serial port tool:

* Send `v` via the serial port to output adc values in real time. a: before filtering; b: after filtering.
* Send `uv` via the serial port to turn off output.
* Send `rt` via the serial port to view the current threshold.
