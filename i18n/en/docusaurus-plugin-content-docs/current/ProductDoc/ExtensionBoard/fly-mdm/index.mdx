# Product Introduction

  - MDM filament sensor, small in size and easy to use, with sensitive triggering.
  - Suitable for 3D printers like Klipper, Marlin, RRF, etc.
  - Filament out detection: High-speed infrared scanning, captures the filament breakage within 0.25 seconds, immediately pauses printing and alarms to avoid idle printing loss;
  - Blockage warning: Encoder optical signal recognition, identifies the precursors of nozzle/hotend blockage, reducing scrap rate by more than 80%.
  - Wide compatibility: Supports all types of filaments (1.75mm diameter) such as PLA, ABS, PETG, TPU, etc., stable recognition for both rigid and flexible materials;
  - Easy installation: Provides installation materials including hole distance diagram, wiring diagram, and configuration files.
