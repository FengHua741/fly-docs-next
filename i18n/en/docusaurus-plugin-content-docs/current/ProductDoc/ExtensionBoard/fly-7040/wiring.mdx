## Reference Configuration

    ```cfg
    [fan]
    ##    Print Cooling Fan 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## Startup Automatic Rotation Solution

    :::info
    * This is a brief explanation
    * Add the PWM signal pin during firmware compilation and invert its state
    * Test using SUPER8PRO, pin PF8, during firmware compilation select `GPIO pins to set at micro-controller startup` and enter `!PF8`
    * Please note that the `!` entered is an English exclamation mark
    * The same applies to other motherboards
    :::
