# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />

## Product Features

    * Suitable for places like 36 motors
    * Can be connected to Raspberry Pi, FLY PI, Orange Pi, motherboards, etc. via SPI interface

## Product Specifications

    * On-board interface: ZH1.5 6P plug
    * Communication method: SPI
    * Operating voltage: 5V

## Precautions

    :::warning Precautions

    * Operating voltage is 5V, please do not connect incorrectly
    * Suitable for various firmwares, please ensure that SPI connection is available

    :::
