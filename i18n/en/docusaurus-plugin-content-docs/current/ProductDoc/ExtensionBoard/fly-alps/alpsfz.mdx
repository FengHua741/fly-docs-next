# How to Adjust the Trigger Value of the Module

## Open the Leveling Module
    Send the command `SET_PIN PIN=_probe_ready VALUE=1` in the `Klipper` console, observe the trigger indicator light, adjust the threshold from small to large until the indicator light does not light up. Enter the following command:
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```

## Adjust the Pressure Value
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">Get Serial Tool Assistant</Button>

    * Connect the module with the computer using a `USB` data cable, open the serial tool assistant, select the corresponding port number, baud rate 115200, and click connect.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    Input the following command in the serial tool to adjust the trigger value:
    ```cfg
    set 触发值
    ```
    Default trigger value: 20000
    The smaller the value, the more sensitive; the larger the value, the less sensitive.
    * For example: `set 20000`, set the trigger value to 20000. The smaller the value, the more sensitive; the larger the value, the less sensitive.
    * Commands used in the serial tool:

    * Sending `v` via the serial port outputs abc values in real time. a: before filtering; b: after filtering.
    * Sending `uv` via the serial port, closes output.
    * Sending `rt` via the serial port, checks the current threshold.
