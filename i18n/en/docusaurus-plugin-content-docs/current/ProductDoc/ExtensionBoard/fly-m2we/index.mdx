## Precautions

:::danger Precautions

* M2WE is pre-flashed with FLYOS and can be started directly
* M2WE can only be used on FLY products and does not support other devices
* The antenna terminal of M2WE is `IPEX1`
* M2WE does not support 2.4G and 5G dual-band integration, Chinese, special characters, and the default hidden WIFI

:::
