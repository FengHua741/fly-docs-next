# Introduction

> Mellow Fly-D5 is a high-performance 3D printer motherboard launched by the Fly3D motherboard R&D team of Guangzhou Meilun Electronic Technology Co., Ltd., supporting up to 5 stepper motors.

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/d5.webp').default} size="60%" align="center" />

## **1. Product Features**

* Adopt 32-bit main frequency 48MHz ARM Cortex-M0+ series STM32F072RBT6 main control chip
* Support CAN bus connection, more stable data transmission, lower delay, more stable connection
* Drive support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
* On-board DIAG function pin, infinite position homing support can be used just by simply plugging and unplugging the jumper cap

## 2. Product Parameters

* MCU: STM32F072RBT6
* Firmware: Klipper/Marlin
* Support 12-24V DC power supply, more stable power supply, eliminate the need for a step-down module
* Drive working mode support: SPI, UART, STEP/DIR
* Supported drivers: Support external TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
* Expansion interface: BLTouch (Servos, Probe), UART X1, 12864 X1
* Fans: Two-wire controllable fan X2, parallel fan X2, normally open fan X2
* Temperature sensor: 2 channels NTC 100K
* Heating interface: 2 controllable interfaces support 10A
* Support machine structure: Cartesian, Delta, Kossel, Ultimaker, CoreXY
