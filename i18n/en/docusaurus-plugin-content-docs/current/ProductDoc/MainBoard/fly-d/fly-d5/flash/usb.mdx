# USB Firmware Compilation


## Start Compilation

    <SSH />

    <USB />

    * Press the `Q` key, when **Save configuration** appears, press the `Y` key again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface
    * Enter `make -j4` to start compiling, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, then the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Enter Burning Mode

    :::tip Tip
        After September 1, 2024, D5 units that are manufactured have already been flashed with the Katapult firmware
    :::
    * Connect the main board to the host machine using a Type-C data cable, and **double-click the reset button**, at this point there should be an LED light flashing on the main board.
    :::warning Warning
       * If the LED light does not flash, please re-flash the Katapult firmware: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-burning-mode">Flash Katapult Firmware</Button>
       * If the LED light does not flash, please perform the next step after flashing the Katapult firmware
    :::
