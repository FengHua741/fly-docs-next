# Introduction

> Fly-D7 is a high-performance 3D printer mainboard developed by the Fly3D motherboard R&D team of Guangzhou Meilun Electronic Technology Co., Ltd., supporting up to 7 stepper motor channels.


    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.webp').default} size="60%" align="center" />

## **1. Product Features**

    * Uses a 32-bit ARM Cortex-M0+ series STM32F072RBT6 main control chip with a main frequency of 48MHz
    * Supports CAN bus connection, providing more stable data transmission, lower latency, and more stable connections
    * Drive support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Board-mounted DIAG function pins, enabling infinite endstop homing support with simple plug-and-play jumper caps

## 2. Product Specifications

    * MCU: STM32F072RBT6
    * Firmware: Klipper/Marlin
    * Supports 12-24V DC power supply, providing more stable power supply without the need for a voltage reduction module
    * Drive operation mode support: SPI, UART, STEP/DIR
    * Supported drives: Supports external TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
    * Expansion interfaces: BLTouch (Servos, Probe), UART X1
    * Fans: Two-wire controllable fans X2, Parallel fans X2, Always-on fans X2
    * Temperature sensors: 3 channels NTC 100K
    * Heating interfaces: 3 channels supporting 10A control
    * Supported machine structures: Cartesian, Delta, Kossel, Ultimaker, CoreXY
