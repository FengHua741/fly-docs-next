    * Press `Q` key, when **Save configuration** appears, press `Y` again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should exit to the command line interface
    * Enter the following command to start compilation, it takes some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output is as follows, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Enter Burning Mode

    :::tip Tip
        D7 produced after September 1, 2024 has been pre-flashed with Katapult firmware
    :::
    * Use a Type-C data cable to connect the mainboard to the host machine, then **double-click the reset button**, at this time there will be an LED light flashing on the mainboard.
    :::warning Warning
       * If there is no LED light flashing, please re-flash the Katapult firmware: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#enter-dfu-burning-mode">Flash Katapult Firmware</Button>
       * If there is still no LED light flashing, please execute the next step after completing the Katapult firmware flashing
    :::

    <ImageView image={require('@site//docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/img/d7_dfu.webp').default} size="50%" align="center" />
