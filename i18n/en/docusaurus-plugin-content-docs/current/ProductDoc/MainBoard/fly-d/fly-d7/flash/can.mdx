    * Press the `Q` key, and when **Save configuration** appears, press the `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should exit to the command line interface
    * Enter `make -j4` to start compiling, it takes some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following output is displayed at the end, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Enter Burning Mode

    :::tip Tip
        Starting from September 1, 2024, D7 models produced after this date have already been flashed with Katapult firmware.
    :::
    * Use a Type-C data cable to connect the main board to the host computer, then **double-click the reset button**, at which point the main board's LED light will flash.
    :::warning Warning
       * If the LED light does not flash, please re-flash the Katapult firmware: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#enter-dfu-burning-mode">Flash Katapult Firmware</Button>
       * If the LED light does not flash, please execute the next step after flashing the Katapult firmware.
    :::
