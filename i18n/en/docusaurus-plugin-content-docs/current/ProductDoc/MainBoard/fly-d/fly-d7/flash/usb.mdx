# USB Firmware Compilation


## Start Compilation

    <SSH />

    <USB />

    * Press the `Q` key, when **Save configuration** appears, press the `Y` key again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface
    * Enter `make -j4` to start compiling, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output is as follows, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Enter Burning Mode

    :::tip Tip
        Starting from September 1, 2024, D7 units produced after this date already have the Katapult firmware installed.
    :::
    * Use a Type-C data cable to connect the mainboard to the host machine, then **double-click the reset button**, at which point there should be an LED light flashing on the mainboard.
    :::warning Warning
       * If there is no LED light flashing, please re-flash the Katapult firmware: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#enter-dfu-burning-mode">Flash Katapult Firmware</Button>
       * If there is no LED light flashing, please execute the next step after flashing the Katapult firmware.
    :::
