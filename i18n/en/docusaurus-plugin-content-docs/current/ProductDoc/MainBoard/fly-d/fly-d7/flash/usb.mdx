# USB Firmware Compilation


## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q` key, a **Save configuration** prompt will appear, then press `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should exit to the command line interface
    * Enter the following command to start compilation, it may take some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output is as follows, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Enter Burning Mode

    :::tip Tip
        D7 models produced after September 1, 2024 have already been flashed with the Katapult firmware.
    :::
    * Use a Type-C data cable to connect the main board to the host machine, then **double-click the reset button**, at this point there should be an LED light flashing on the main board.
    :::warning Warning
       * If there is no LED light flashing, please re-flash the Katapult firmware: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#enter-dfu-burning-mode">Flash Katapult Firmware</Button>
       * If there is no LED light flashing, please execute the next step after flashing the Katapult firmware.
    :::

    <ImageView image={require('@site//docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/img/d7_dfu.webp').default} size="50%" align="center" />
