## Start Compilation

    <SSH />

    <CAN />

    * Press the `Q` key, and when **Save configuration** appears, press the `Y` key again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should have exited back to the command line interface
    * Enter the following command to start compiling, it may take some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed at the end, the compilation was successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
