# Board-mounted Peripheral Introduction

* When the DIP switch is set to the right, it will enable the USB interface next to the Type-C port.
* When the DIP switch is set to the left, it will enable the RS232 interface next to the Type-C port.

## USB Port Introduction

* It can be used to connect external devices such as cameras and the main board.
