## Enter Flash Mode

    :::tip

    Left is `BOOT button`, right is `RESET button`

    :::

    * There are two ways to enter flash mode
    * First method: With the main board completely powered off, press and hold the BOOT button and then connect the data transmission cable to the main board and your device.
    * Second method: After connecting the data transmission cable to the main board and your device, press and hold the BOOT button and then press the RST button, release the RST button first and then release the BOOT button
