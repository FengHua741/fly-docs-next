# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at factory and does not need to be flashed again.

    :::

## Enter programming mode


    * There are two methods to enter programming mode.
    * The first method: With the main board completely powered off, short the BOOT pin and then connect the data transfer cable with one end connected to the main board and the other end connected to your device.
    * The second method: After connecting the data transfer cable with one end connected to the main board and the other end connected to your device, short the BOOT pin, press the RST button, release the RST button, and finally disconnect the BOOT jumper.
