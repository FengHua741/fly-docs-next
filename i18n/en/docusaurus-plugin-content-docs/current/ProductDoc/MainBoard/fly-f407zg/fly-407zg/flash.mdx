# Firmware Compilation and Burning

    :::tip

    * 407ZG uses TF card for firmware burning by default.
    * Please prepare a TF card with capacity no more than `32G`.
    * Counterfeit memory cards may cause burning failure, please use the recommended memory cards below for firmware updates.
    * The transmission speed of the TF card should be Class10 or above, it is recommended to use SanDisk brand TF cards.

    :::

## Compile Klipper Firmware

    <MakeOperate/>

    <USB />

    * Press `Q` key, appear **Save configuration**, then press `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now you should have saved the configuration and exited to the command line interface
    * Enter `make -j4` to start compiling, it takes some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
