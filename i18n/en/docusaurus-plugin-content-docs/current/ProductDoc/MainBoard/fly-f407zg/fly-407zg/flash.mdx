# Firmware Compilation and Burning

    :::tip

    * The default firmware burning method for 407ZG is using the TF card.
    * Please prepare a TF card with a capacity of no more than `32G`.
    * Counterfeit memory cards may cause burning failure; please try to use the recommended memory cards below for firmware updates.
    * The transmission speed of the TF card should be at least `class10` or higher, it is recommended to use brands like SanDisk.

    :::

## Compiling Klipper Firmware

    <MakeOperate/>

    <USB />

    * Press `Q`, then when **Save configuration** appears, press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should have been saved and you should have returned to the command line interface.
    * Enter the following command to start compiling, it will take some time.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output ends with the following content, the compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    * Below is an example of a successful compilation.
