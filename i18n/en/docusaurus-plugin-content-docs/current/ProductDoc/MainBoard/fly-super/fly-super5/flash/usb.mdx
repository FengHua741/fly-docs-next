# USB Firmware Compilation

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q` key, when **Save configuration** appears, press `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * The configuration should now be saved and you should be back at the command line interface
    * Enter `make -j4` to start compilation, it might take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * The compilation is successful if the following content is output:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Flashing the Firmware 2
    <Tabs>
        <TabItem value="linux" label="DFU Flashing">
            * Follow the image below to insert the jumper cap on `boot1` and press `rst` to restart the motherboard
            <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/flash/img/boot.webp').default} size="50%" align="center" />
            * Enter the following command to check if the motherboard has entered DFU mode
            ```bash
            lsusb
            ```
            * After entering and pressing `Enter`, it will output the content shown in the image below, indicating that the motherboard has successfully entered DFU mode
            <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/dfu.webp').default} size="50%" align="center" />
           
            <dfu />

            4. Flash the firmware

            * Below is the DFU flashing command
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. If the content at the arrow in the image below shows ``File downloaded successfully``, the flashing was successful

            :::warning

            Please ignore the **error messages** after the line indicated by the red arrow!!!

            :::

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/flash/img/boot.webp').default} size="100%" align="center" />

            :::tip

            After flashing, remove the motherboard jumper cap and press the `rst` button to restart the motherboard

            :::
