# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at the factory and does not need to be flashed again.

    :::

## Enter programming mode

    :::tip

    The one on the right is the `BOOT jumper`, and the one on the left is the `RESET button`.

    :::

    * There are two methods to enter programming mode.
    * First method: With the motherboard completely powered off, press and hold the BOOT button, then connect the data transfer cable to both the motherboard and your device.
    * Second method: After connecting the data transfer cable to both the motherboard and your device, press and hold the BOOT button, then press the RST button, release the RST button, and finally release the BOOT button.
