# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at the factory and does not need to be re-flashed.

    :::
## Enter programming mode


    * There are two methods to enter programming mode.
    * First method: With the motherboard completely powered off, short the BOOT pin, then connect one end of the data transfer cable to the motherboard and the other end to your device.
    * Second method: After connecting one end of the data transfer cable to the motherboard and the other end to your device, short the BOOT pin, then press the RST button, release the RST button, and finally disconnect the shorted BOOT jumper.
