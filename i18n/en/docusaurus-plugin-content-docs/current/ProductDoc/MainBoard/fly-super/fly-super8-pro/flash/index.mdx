    :::tip

    * The Super8 PRO uses the TF card by default for firmware burning.
    * Please prepare a TF card with a capacity of no more than `32G`.
    * Counterfeit memory cards may lead to burning failure, please try to use the recommended memory cards below for firmware updates.
    * The transmission speed of the TF card should be at least `class10`, it is recommended to use TF cards from brands like SanDisk.

    :::

# Firmware Burning Instructions

    :::tip

    * The Super8 Pro comes pre-flashed with a TF card bootloader and does not require you to flash it again.

    :::

## How to Determine Which Firmware You Need

    :::tip

    * Note that you only need to flash one type of firmware.

    :::
    
    * The Super8 PRO can connect to the host computer in the following ways:
    * <Button variant="contained" disableElevation href="./usb">USB Firmware</Button>
        * Advantage: Reliable and stable communication.
        * Disadvantage: May affect data transmission if multiple USB devices are present.
    * <Button variant="contained" disableElevation href="./can">USB Bridge Firmware</Button>
        * Advantage: Can connect to multiple devices without occupying USB ports.
        * Disadvantage: May cause positioning timeouts or other errors when used together with `KlipperScreen`.
