# Product Parameters


    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip
    * Firmware: Marlin 2.0/Reprap/Klipper
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Driver mode support: TMC: UART/SPI
    * 8 high-voltage drive positions, supports 8 independent motor drives, 5 extruders, 10 PWM fans
    * Display: serial touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 high-voltage IN port, supports automatic bed leveling sensor: BLTouch, PL08N, etc. proximity sensors, Klicky, Voron Tap, etc.
    * Optional limit switch power supply: 5V/12V/24V
    * Using repap firmware, can use onboard ESP32 WiFi module, supports 1.6m/s
    * Unique PWM Fan MOS board design, damage can be directly replaced, reduces customer repair difficulty, two 2-pin fan interfaces can support 4-pin PWM fans
    * PCB uses 2OZ copper thickness, can support larger current
    * 6 ADC interfaces, can support up to six thermistors
    * 8 IO interfaces, can be used for leveling and limit switches
    * 5 hotbed interfaces, can support up to five heaters
    * 2 PMW interfaces
