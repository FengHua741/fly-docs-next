    :::tip

    * Super8 defaults to using an SD card for firmware burning.
    * Please prepare a SD card with a capacity not exceeding `32G`.
    * Counterfeit memory cards may lead to burn failure, please try to use the recommended memory cards below for firmware updates.
    * The transfer speed of the SD card should be Class10 or above, it is recommended to use SD cards from brands like SanDisk.

    :::

# Firmware Burning Instructions


## How to Determine Which Firmware You Need

    :::tip

    * Please note that you only need to flash one type of firmware.

    :::
    
    * Super8 can connect to the host computer in the following ways
    * <Button variant="contained" disableElevation href="./usb">USB Firmware</Button>
        * Advantage: Reliable and stable communication
        * Disadvantage: Data transmission may be affected if multiple USB devices are present
    * <Button variant="contained" disableElevation href="./can">USB Bridge Firmware</Button>
        * Advantage: Can connect multiple devices without occupying USB ports
        * Disadvantage: May cause positioning timeouts and other errors when used with `KlipperScreen`.
