# USB Firmware Compilation

    :::tip

    * Super8 comes with a pre-flashed TF card bootloader, no need to flash it again.

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press the `Q` key, and when you see **Save configuration**, press `Y` again.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should be back at the command line interface.
    * Enter `make -j4` to start compiling, this might take some time.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output ends with the following content, the compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
