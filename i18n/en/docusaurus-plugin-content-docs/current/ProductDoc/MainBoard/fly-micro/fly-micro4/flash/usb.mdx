# USB Firmware Compilation

    :::tip

    * The Micro4 comes with the Katapult firmware pre-flashed on the USB connection, so you don't need to flash it again.

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Select GPIO pins to set at micro-controller startup, input: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/usb.webp').default} size="100%" align="center" />
    * Press `Q`, then when **Save configuration** appears, press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should be back to the command line interface
    * Enter the following command to start compilation, it takes a while
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output is as follows, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
