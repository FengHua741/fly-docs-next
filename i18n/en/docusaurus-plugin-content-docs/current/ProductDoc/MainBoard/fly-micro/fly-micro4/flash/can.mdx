# USB Bridge CAN Firmware Burning

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Select GPIO pins to set at micro-controller startup, input: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/can.webp').default} size="100%" align="center" />
    * Press `Q`, when **Save configuration** appears, press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should exit to the command line interface
    * Enter the following command to start compilation, it takes some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed, compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
