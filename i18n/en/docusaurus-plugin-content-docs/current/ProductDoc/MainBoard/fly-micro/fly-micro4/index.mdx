# Micro4

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/micro.webp').default} size="80%" align="center" />

## Product Features

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Support CAN bridge
    * Onboard 4x2209
    * Onboard DIAG function pin, supports unlimited homing by simply plugging in a jumper cap

## Product Parameters

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Firmware: Klipper
    * Supports 12-24V DC power supply, more stable power supply, eliminating the need for a voltage drop module
    * Driver working modes supported: UART, STEP/DIR
    * Expansion interface: BLTouch (Servos, Probe), Proximity switch X1
    * Fans: Two-wire controllable fan X2, Always-on fan X1
    * Temperature sensors: 2 channels NTC 100K
    * Heating interfaces: 2 channels controllable interface supporting 10A
