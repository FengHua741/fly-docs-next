# Katapult Firmware

    :::tip

    * Please note that the `katapult` has already been pre-flashed at the factory and does not need to be flashed again.

    :::

## Enter Programming Mode

    :::tip

    The circled button is the `BOOT button`.

    :::

    * To enter programming mode, you need to press and hold the BOOT button while the main board is completely powered off, and then connect the data transfer cable to both the main board and your device.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/boot.webp').default} size="100%" align="center" />
