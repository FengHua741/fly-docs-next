## Product Features

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz

## Product Specifications

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Firmware: Klipper
    * Supports 12-24V DC power supply, more stable power supply, eliminating the need for a voltage drop module
    * Driver operating modes supported: UART, STEP/DIR
    * Expansion interfaces: Controllable fans*2, RGB*1, Limit switches*3
