# USB Firmware Compilation

    :::tip

    * The bootloader has been pre-flashed on the TF card at the factory, so there is no need to flash it again.

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press the `Q` key, and when **Save configuration** appears, press the `Y` key again.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface.
    * Enter `make -j4` to start compiling, which may take some time.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, then the compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
