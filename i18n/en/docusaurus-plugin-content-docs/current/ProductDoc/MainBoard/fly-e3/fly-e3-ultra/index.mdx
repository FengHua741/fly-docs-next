## Product Parameters

* 32-bit ARM Cortex-M4 series 550 MHz, STM32H723VGT6 chip
* Firmware: Reprap/Klipper
* Board integrates five TMC2209 drivers, XYZ three driver positions support infinite endstop mode
* Display: Serial touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
* 1 high voltage IN port, supports automatic bed leveling sensors: BLTouch, PL08N, etc., proximity sensors, Klicky, Voron Tap, etc.
* Using RepRap firmware, ESP32 WiFi module can be used, supports 1.6m/s
* Unique PWM Fan MOS board design, easy to replace when damaged, reduces customer repair difficulty, four 2-pin fan interfaces
* 3 ADC interfaces for temperature sensors
3 endstop ports can connect limit switches
* 4 IO ports can be used for proximity switches, limit switches, laser heads, RGB lights, etc.
* 3 heating ports
* 1 serial screen interface for RRF
* Max version supports 1 K-type thermocouple and 1 PT100 temperature sensor
* New addition of a DIP switch for selecting the 2/3/4-wire mode of the PT100 temperature sensor
* New integrated heat sink, increases the heat sink area of the drivers
* New CAN interface
