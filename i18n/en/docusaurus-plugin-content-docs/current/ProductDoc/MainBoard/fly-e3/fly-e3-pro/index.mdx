## Product Features

    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip

## Product Specifications

    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip
    * Firmware: Reprap/Klipper
    * On-board five TMC2209 drivers
    * Display: Serial touch screen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 high voltage IN port, supports auto bed leveling sensor: BLTouch, PL08N, etc. proximity sensors, Klicky, Voron Tap, etc.
    * Using repap firmware, can use ESP32 WiFi module, supports 1.6m/s
    * Unique PWM Fan MOS board design, damage can be directly replaced, reduces customer repair difficulty, four 2-pin fan interfaces
    * PCB uses 2OZ copper thickness, can support larger current
    * 2 ADC interfaces, up to two thermistors supported
    * 4 IO interfaces, can be used for leveling and limit switches
    * 3 hotbed interfaces, up to three hotbeds supported
