# Firmware Compilation and Burning

    :::tip

    * E3 Pro defaults to using a TF card for firmware burning.
    * Please prepare a TF card with a capacity not exceeding `32G`.
    * Counterfeit memory cards may cause burning failures, so it is recommended to use the recommended memory cards below for firmware updates.
    * The transfer speed of the TF card should be at least `class10` or higher, it is recommended to use brands like SanDisk.

    :::

## Compiling Klipper Firmware

    <USB />

    * Press the `Q` key, a **Save configuration** prompt will appear, then press the `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * You should now have saved the configuration and exited to the command line interface.
    * Enter `make -j4` to start compiling, it takes some time.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following output is seen, compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
