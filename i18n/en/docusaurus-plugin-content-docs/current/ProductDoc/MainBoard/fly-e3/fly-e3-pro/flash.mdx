# Firmware Compilation and Burning

    :::tip

    * The E3 Pro defaults to firmware burning via the TF card.
    * Please prepare a TF card with a capacity of no more than `32G`.
    * Counterfeit memory cards may cause burning failure; please try to use the recommended memory cards for firmware updates.
    * The transmission speed of the TF card should be Class10 or above, and it is recommended to use TF cards from brands like SanDisk.

    :::

## Compiling Klipper Firmware

    <MakeOperate/>

    <USB />

    * Press `Q`, and when **Save configuration** appears, press `Y` again.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should have been saved and exited to the command line interface.
    * Enter the following command to start compiling, which will take some time.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below appears at the end, compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
