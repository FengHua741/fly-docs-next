## Product Parameters

    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip
    * Supports CAN bus connection, more stable data transmission, lower delay, and more stable connection
    * Firmware: Reprap/Klipper
    * Driver support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Driver mode support: TMC: UART/SPI
    * Display: Serial touchscreen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 high voltage IN port, supports automatic bed sensor: BLTouch, PL08N, etc. proximity sensors, Klicky, Voron Tap, etc.
    * Using repap firmware, can use extended ESP32 WiFi module, supports 1.6m/s
    * Unique PWM Fan MOS board design, easy to replace when damaged, reduces customer repair difficulty, four 2-pin fan interfaces
    * PCB uses 2OZ copper thickness, capable of supporting higher current
    * 2 ADC interfaces, supports up to two thermistors
    * 4 IO interfaces, can be used for leveling and limit switches
    * 3 heater cartridge interfaces, supports up to three heater cartridges
