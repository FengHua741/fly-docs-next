# USB Firmware Compilation

    :::tip

    * The E3 V2 is pre-flashed with a TF card bootloader at the factory, so there's no need to flash it again.

    :::

## Start Compiling

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q`, and when **Save configuration** appears, press `Y` again.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now you should have saved the configuration and returned to the command line interface.
    * Enter the following command to start compiling (it may take some time):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the compilation is successful, the output will look like this:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
