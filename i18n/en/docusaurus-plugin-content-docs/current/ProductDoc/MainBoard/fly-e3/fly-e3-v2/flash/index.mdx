    :::tip

    * E3 V2 uses the TF card for firmware burning by default.
    * Please prepare a TF card with a capacity of no more than `32G`.
    * Counterfeit memory cards may cause burning to fail, please try to use the recommended memory cards below for firmware updates.
    * The transmission speed of the TF card should be Class10 or above, it is recommended to use TF cards from brands such as SanDisk.

    :::

# Firmware Burning Instructions


## How to determine which firmware you need

    :::tip

    * Please note that you only need to burn one of the firmwares.

    :::
    
    * E3 V2 can connect to the host machine using the following methods
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB Firmware</Button>
        * Advantage: Reliable and stable communication.
        * Disadvantage: If there are multiple USB devices, it may affect data transmission.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB Bridge Firmware</Button>
        * Advantage: Can connect to multiple devices without occupying USB ports.
        * Disadvantage: May cause positioning timeouts or other errors when used together with `KlipperScreen`.
