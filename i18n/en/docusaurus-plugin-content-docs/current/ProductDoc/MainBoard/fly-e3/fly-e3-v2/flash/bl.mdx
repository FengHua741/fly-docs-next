# BL firmware

    :::tip

    * Please note that the `bootloader` has already been pre-flashed at the factory and does not need to be flashed again.

    :::

## Entering programming mode

    :::tip

    The button on the right is the `BOOT button`, and the button on the left is the `RESET button`.

    :::

    * There are two ways to enter programming mode.
    * First method: With the mainboard completely powered off, press and hold the BOOT button and then connect the data transfer cable to the mainboard and your device.
    * Second method: After connecting the data transfer cable to the mainboard and your device, press and hold the BOOT button and then press the RST button, release the RST button, and finally release the BOOT button.
