# Firmware Burning Instructions
    

## How to Determine Which Firmware You Need

    :::tip

    * Please note that you only need to flash one of the firmware options.

    :::
    
    * C5 can use the following methods to connect to the host machine
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB Firmware</Button>
        * Advantage: Reliable and stable communication
        * Disadvantage: Data transmission may be affected if multiple USB devices are present
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB Bridge Firmware</Button>
        * Advantage: Can connect multiple devices without occupying a USB port
        * Disadvantage: May cause errors such as homing timeout when used with `KlipperScreen`.
