# USB Firmware Compilation

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q` key, and when the **Save configuration** prompt appears, press `Y` again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have returned to the command line interface
    * Enter `make -j4` to start compiling, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following output is displayed at the end, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware Burning

:::tip
Execute the following command to burn the firmware
:::

    ```bash
    sudo fly-flash -d c5 -h -f ~/klipper/out/klipper.bin
    ```
