# USB Bridge CAN Firmware Burning

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Press the `Q` key, when **Save configuration** appears, press the `Y` key again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and exited to the command line interface
    * Enter `make -j4` to start compilation, it takes a while
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Firmware Burning

:::tip
Execute the following command to burn the firmware
:::

    ```bash
    sudo fly-flash -d c5 -h -f ~/klipper/out/klipper.bin
    ```
