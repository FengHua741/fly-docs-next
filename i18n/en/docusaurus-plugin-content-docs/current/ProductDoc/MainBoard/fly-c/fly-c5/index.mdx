## Product Features

- 32-bit ARM Cortex-M4, 550MHz, STM32H723VGT6 chip
- Supports TMC-driven SPI, UART, STEP/DIR working modes
- DIAG function pin does not occupy limit switches; after configuration, it can achieve homing without limit switches
- Digital control fans support 24V, 12V, 5V, no additional voltage drop module required
- Supports power-off auto shutdown and power-off resume printing (power-off auto shutdown is enabled by default)
- Pre-installed USB Klipper firmware at factory
- Multi-channel power management, short circuits on ports, limit switches, and drivers do not affect host machine operation
-预留 5V RGB interface
- TFT and HDMI screens support one-line connection
- System pre-installs klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, resonance compensation plugin numpy, no need to change source, easy to use (TF card or M2WE purchase required)

## Product Parameters

### Host Computer

- **CPU**: Allwinner H618, quad-core 64-bit Cortex-A53
- **GPU**: Arm Mali-G31
- **RAM**: 1GB DDR4 3200M
- **ROM**: Maximum support for 128GB SD card
- **Peripherals**:
    - TFT x1
    - USB 2.0 x2
    - Micro HDMI x1
    - Ethernet port (Eth) x1 (100M)
    - FPC-HDMI x1
    - UART x1
- **Interface**: On-board M.2, supports expansion of WIFI, eMMC, etc. (private protocol, do not connect non-FLY dedicated M.2 devices)
- **Support**: TFT screen and HDMI screen FPC one-line connection
- **System Pre-installation**: klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, resonance compensation plugin numpy, no need to change source, easy to use (TF card or M2WE purchase required)

### Subordinate Machine

- **MCU**: STM32H723VGT6
- **Firmware**: KLIPPER
- **Power Supply**: Supports 12-24V DC power supply, no voltage drop module required
- **Maximum Output Current of Hotbed Port**: 15A
- **Maximum Output Current of Heater Port**: 6A
- **Maximum Output Current of Fan Port**: 1A
- **Driver Working Mode**: SPI, UART, STEP/DIR
- **Supported Drivers**: TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
- **Expansion Interface**: BLTouch (Servos, Probe), RGB x1, UART x1, CAN x1
- **Fans**: Two-wire digital fans x4 (support 24V, 12V, 5V selectable)
- **Temperature Sensor**: 3 channels NTC 100K
- **Supported Structures**: Cartesian, Delta, Kossel, Ultimaker, CoreXY, etc.
