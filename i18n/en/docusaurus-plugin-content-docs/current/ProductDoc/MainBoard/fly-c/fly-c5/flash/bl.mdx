# BL Firmware

    :::tip

    * Please note that the `bootloader` has been pre-flashed at the factory and does not need to be re-flashed.

    :::

## Entering Programming Mode

    :::tip

    * The left is the motherboard's RST, and the right is the motherboard's BOOT.

    :::

    * There are two ways to enter programming mode.
    * First method: With the motherboard completely powered off, long press BOOT and connect one end of the data transfer cable to the motherboard and the other end to your device, suitable for WINDOWS.
    * Second method: After connecting one end of the data transfer cable to the motherboard and the other end to your device, long press BOOT and then input commands, suitable for host machine.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/boot.webp').default} size="60%" align="center" />
