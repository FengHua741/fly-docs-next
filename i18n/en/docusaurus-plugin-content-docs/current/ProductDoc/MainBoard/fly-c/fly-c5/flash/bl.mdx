# BL Firmware

    :::tip

    * Please note that the `bootloader` has been pre-flashed at the factory and does not need to be re-flashed.
    * C5 can only use the built-in host software for burning.
    :::

## Connect to Host Machine via SSH

    :::danger Warning
    - Before burning the BL firmware, you need to connect to the host machine via `WiFi SSH`.
    :::

 * First, please: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh#using-wifi-to-connect-to-host-ssh">Use WiFi to Connect to Host SSH</Button>
 ---
 <br/>

## Flash Firmware


    * Set the dip switch upward and connect the lower computer USB to the host machine.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * Enter the following command to start flashing BL


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```
