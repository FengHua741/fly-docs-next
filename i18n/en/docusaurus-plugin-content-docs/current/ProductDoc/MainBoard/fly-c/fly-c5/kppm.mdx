# Klipper Power Loss Resume Function

- The Klipper power loss resume function refers to the ability of Klipper to automatically restore its printing status after power is restored following a power outage.
- C5 can use this feature without requiring KPPM
- Not applicable for printers where the Z-axis will move after a power outage.
