# DIP Switch Introduction

:::tip
FLY c8 connects to the host machine via the DIP switch, and if the position of the DIP switch is incorrect, it cannot be used normally.
:::

:::danger Notice
Please note that the DIP switch needs to be set to the specified position to connect to the host machine normally.
:::
