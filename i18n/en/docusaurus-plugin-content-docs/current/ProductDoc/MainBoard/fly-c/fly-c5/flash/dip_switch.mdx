# DIP Switch Introduction

:::tip
FLY c8 connects to the host machine via the DIP switch; if the position is incorrect, it cannot be used normally.
:::

:::danger Notice
Please note that the DIP switch must be set to the specified position to connect to the host machine normally.
:::

## Connecting Onboard Host Machine

    * Turn DIP switches `1` and `2` to the right to connect the lower-level machine to the host machine via the built-in USB.
    * Turn DIP switches `3` and `4` to the left, otherwise it will affect the connection between the lower-level machine and the host machine.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * After adjusting the DIP switches as described above, the lower USB port will be occupied by the lower-level machine. Please do not connect other devices.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Connecting External Host Machine

    * Turn DIP switches `3` and `4` to the right to connect the lower-level machine to the host machine via the built-in USB.
    * Turn DIP switches `1` and `2` to the left, otherwise it will affect the connection between the lower-level machine and the host machine.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
