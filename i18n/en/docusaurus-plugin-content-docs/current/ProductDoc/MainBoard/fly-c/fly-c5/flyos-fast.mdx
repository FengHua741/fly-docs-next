## Install Antenna

1. Install the antenna for the `M2WE` or `2.4G WiFi` module.
:::warning
* The FLY host machine requires the use of the `M2WE` or `2.4G WIFI` module; otherwise, WiFi cannot be used.
* Please note that the host machine does not support WiFi networks with Chinese names and does not support dual-band (2.4G and 5G) combined WiFi.
:::
* The red arrow indicates the WiFi antenna interface.
* The WiFi antenna port is `ipex1`.
* Please install the antenna on the M2WE, otherwise poor signal may cause connection failure to the router.
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/wifi.webp').default} size="80%" align="left" />

:::tip
- After completing the installation of the system card, you can proceed to the next step and connect to the host machine via SSH.
:::

2. Power on the host machine.  
**At this point, your host machine has already started up!!!**  
Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh">SSH Connect to Host Machine</Button>
