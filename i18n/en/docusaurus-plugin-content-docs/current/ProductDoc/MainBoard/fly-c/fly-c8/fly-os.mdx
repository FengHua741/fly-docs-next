## Preparation Before System Burning

    * SD card reader
    * One TF card, the capacity must not be less than 16GB or not greater than 128GB, the transmission speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
TF card burning and burning to M2WE via TF card only requires using one of the burning methods.
:::

:::info
* If the machine has installed M2WE and memory card, the system defaults to booting from the memory card
* It is recommended to use TF card to burn the system for M2WE; burning through USB to M2WE may fail due to driver compatibility issues, it is recommended to directly use [memory card burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note that the downloaded system image should match, otherwise it cannot start
:::
