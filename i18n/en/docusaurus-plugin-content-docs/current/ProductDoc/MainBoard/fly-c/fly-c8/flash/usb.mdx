# USB Firmware Compilation

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press the `Q` key, a **Save configuration** message will appear, then press the `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should be back to the command line interface
    * Enter `make -j4` to start compiling, it takes some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below appears, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware Flashing

:::tip
Run the following command to flash the firmware
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
