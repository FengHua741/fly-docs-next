# USB Bridge to CAN Firmware Flashing

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Press `Q`, and when **Save configuration** appears, press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Press `Q`, and when **Save configuration** appears, press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface
    * Enter the following command to start compilation, it takes some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * The final output should look like this to indicate successful compilation

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```



## Firmware Flashing

:::tip
Execute the following command to flash the firmware
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
