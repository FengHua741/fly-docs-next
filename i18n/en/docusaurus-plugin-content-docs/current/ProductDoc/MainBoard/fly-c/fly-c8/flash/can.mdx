# USB Bridge to CAN Firmware Flashing

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Press the `Q` key, and when **Save configuration** appears, press the `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have exited back to the command line interface
    * Enter `make -j4` to start compiling, this will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output is as follows, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Firmware Flashing

:::tip
Execute the following command to flash the firmware
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
