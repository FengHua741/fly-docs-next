        <Tabs>
    <TabItem value="tft" label="TFT Capacitive Screen Configuration" default>
## Method One    
         * Capacitive configuration requires using the following command to configure as `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Please confirm that the DIP switch on the back of the screen is set to `Cap`
    * Use the following command to check if it was successfully modified
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * To change from capacitive screen to resistive screen, use the following command

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

        <TabItem value="hdmi" label="TFT Resistive Screen Configuration">
## Method One        
           * Resistive screen configuration requires using the following command to configure as `fly-tft-v2-r` 
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
```
                                * Please confirm that the DIP switch on the back of the screen is set to `Resi`
        * Use the following command to check if it was successfully modified
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
                                    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-r.webp').default} size="80%" align="center" />
 * To change from resistive screen to capacitive screen, use the following command

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v2-c/}' /boot/FlyOS-Env.txt
```
</TabItem>
