# CDY V3_V2

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-cdy/fly-cdy-v3/img/cdy.webp').default} size="80%" align="center" />

## Product Features

    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip
    * Supports drive voltage selection, default 5V
    * Supports RRF firmware

## Product Parameters

    * 32-bit ARM Cortex-M4 series 168 MHz, STM32F407ZGT6 chip
    * Supports CAN bus connection, more stable data transmission, smaller delay, more stable connection
    * Firmware: Reprap/Klipper
    * Drive support: A4988, LV8729, DRV8225, TMC2208/2209/5160, etc.
    * Drive mode support: TMC: UART/SPI
    * Display: Serial touch screen, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Using repap firmware, can use extended ESP32 WiFi module, supports 1.6m/s
    * Unique PWM Fan MOS board design, can be directly replaced when damaged, reduces customer repair difficulty
    * PCB uses 2OZ copper thickness, can support larger current
    * 3 ADC interfaces, supports up to three thermistors
    * 6 IO interfaces, can be used for leveling and limit switches
    * 4 heating rod interfaces, supports up to four heating rods
