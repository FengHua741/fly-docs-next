# Firmware Compilation and Burning

    :::tip

    * CDV uses TF card by default for firmware burning.
    * Please prepare a TF card with a capacity of no more than `32G`.
    * Counterfeit memory cards may cause burning failure, so it is recommended to use the recommended memory cards for firmware updates.
    * The transmission speed of the TF card should be at least `class10`, it is recommended to use brands like SanDisk.

    :::

    * Now you should have saved the configuration and exited to the command line interface.
    * Enter the following command to start compiling, it will take some time.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed, then the compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
