# Introduction to DIP Switches

:::tip
FLY Gemini connects to the host computer via DIP switches; incorrect switch positions will prevent normal operation.
:::

:::danger Note
Please ensure the DIP switches are set to the specified positions for proper connection to the host computer.
:::

## Connecting to the Onboard Host Computer

    * Set DIP switches `1`, `2` to the right to connect the lower machine to the host computer via internal USB
    * Set DIP switches `3`, `4` to the left; otherwise, it will affect the connection between the lower machine and the host computer
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * After setting the DIP switches in this manner, the USB port below will be occupied by the lower machine; do not connect other devices
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## Connecting to an External Host Computer

    * Set DIP switches `3`, `4` to the right to connect the lower machine to an external host computer via internal USB
    * Set DIP switches `1`, `2` to the left; otherwise, it will affect the connection between the lower machine and the external host computer
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
