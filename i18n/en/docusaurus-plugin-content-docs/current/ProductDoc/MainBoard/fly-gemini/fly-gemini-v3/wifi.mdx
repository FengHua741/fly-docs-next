# Connect WIFI

## Install the Antenna on Gemini

    * The place pointed to by the red arrow is the WIFI antenna interface
    * The WIFI antenna port is `ipex1`
    * Install the antenna on Gemini, otherwise it may not be able to connect to the network due to poor signal
