# USB Firmware Compilation

## Starting Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press the `Q` key, and when **Save configuration** appears, press the `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * The configuration should now be saved, and you should be back at the command line interface
    * Enter `make -j4` to start the compilation, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, the compilation was successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Firmware Flashing

:::tip
Execute the following command to flash the firmware
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```

## Flashing Firmware
    <Tabs>
        <TabItem value="linux" label="DFU Flashing">
            * Follow the image below to insert the jumper cap `boot1` and then press `rst` to restart the motherboard
            <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/boot.webp').default} size="50%" align="center" />
            * Enter the following command to check if the motherboard has entered DFU mode
            ```bash
            lsusb
            ```
            * After entering and pressing `Enter`, the content in the image below will be displayed, indicating that the motherboard has successfully entered DFU mode
            <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/dfu.webp').default} size="50%" align="center" />
           
            <dfu />

            4. Flash the firmware

            * Below are the DFU flashing commands
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. If the content shown by the red arrow, `File downloaded successfully`, appears, the flashing was successful

            :::warning

            Please ignore the **error message** after the line indicated by the red arrow!!!

            :::

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/dfu1.webp').default} size="100%" align="center" />

            :::tip

            After flashing is complete, remove the motherboard jumper cap and press the `rst` button to restart the motherboard

            :::

        </TabItem>
    </Tabs>
