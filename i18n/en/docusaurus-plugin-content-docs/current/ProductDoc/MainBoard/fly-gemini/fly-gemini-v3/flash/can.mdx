# USB Bridge CAN Firmware Flashing

## Start Compiling

    <SSH />

    <MakeOperate/>

    <CAN />

    * Press the `Q` key, when **Save configuration** appears, press the `Y` key again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * The configuration should now be saved and you should be back at the command line interface
    * Enter `make -j4` to start compiling, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Firmware Flashing

:::tip
Execute the following command to flash the firmware
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```

    ## Flashing the Firmware
    <Tabs>
        <TabItem value="linux" label="dfu flashing">
            * Follow the image below to insert the jumper cap `boot1` and then press `rst` to restart the motherboard
            <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/boot.webp').default} size="50%" align="center" />
            * Enter the following command to check if the motherboard has entered dfu mode
            ```bash
            lsusb
            ```
            * After pressing `Enter`, the content shown in the image below will be output, indicating that the motherboard has successfully entered dfu mode
            <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/dfu.webp').default} size="50%" align="center" />
           
            <dfu />

            4. Flash the firmware

            * Here is the dfu flashing command
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. If the content "File downloaded successfully" appears as shown by the arrow in the image below, the flashing is successful

            :::warning

            Please ignore any error messages following the red arrow line!!!

            :::

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/dfu1.webp').default} size="100%" align="center" />

            :::tip

            After flashing is complete, remove the motherboard jumper cap and press the `rst` button to restart the motherboard

            :::
