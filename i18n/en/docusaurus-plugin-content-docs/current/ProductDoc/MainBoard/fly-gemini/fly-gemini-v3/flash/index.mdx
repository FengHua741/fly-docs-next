# Firmware Burning Instructions

## How to Determine Which Firmware You Need

    :::tip

    * Please note that you only need to flash one of the firmwares

    :::

    * Gemini V3 can be connected to the host computer in the following ways
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB Firmware</Button>
        * Advantage: Reliable communication, stable
        * Disadvantage: If multiple USB devices are used, it may affect data transmission
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB Bridge Firmware</Button>
        * Advantage: Can connect multiple devices and does not occupy USB ports
        * Disadvantage: May cause homing timeout and other errors when used together with `KlipperScreen`.
