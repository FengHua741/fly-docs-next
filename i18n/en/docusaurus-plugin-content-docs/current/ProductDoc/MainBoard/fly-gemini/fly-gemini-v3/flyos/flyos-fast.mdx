## Install Antenna

    1. Install the antenna for `M2WE` or `2.4G WiFi` module
    :::warning
    * FLY host machine needs to use the `M2WE` or `2.4G WIFI` module, otherwise WiFi cannot be used
    * Please note that the host machine does not support Chinese-named WiFi and does not support dual-band (2.4G and 5G) unified WiFi
    :::
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna on the M2WE, otherwise the connection to the router may fail due to poor signal
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After completing the installation of the system card, you can proceed to the next step and connect to the host machine via SSH
    :::

    2. Power on the host machine.  
    **Your host machine has been started successfully!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/ssh">SSH Connect to Host Machine</Button>
