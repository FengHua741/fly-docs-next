## Preparation Before System Flashing

    * Card Reader
    * A TF card with a capacity of at least 16GB or not more than 128GB, with a transfer speed of class 10 or above
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">system image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">flashing tool</Button>

## System Flashing

:::info
Flashing to a TF card or flashing to M2WE via a TF card requires only one method to be used
:::

:::info
* If the machine has both M2WE and a memory card installed, the system defaults to booting from the memory card
* It is recommended to use a TF card to flash the system to M2WE. Flashing via USB to M2W might fail due to driver compatibility issues, so it is advised to use [Memory Card Flashing to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please ensure the downloaded system image is compatible; otherwise, the system will not boot
:::
