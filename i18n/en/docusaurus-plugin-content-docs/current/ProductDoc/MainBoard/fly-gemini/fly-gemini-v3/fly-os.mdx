## Preparation Before System Burning

    * Card reader
    * One TF card, the capacity must be no less than 16GB or no more than 128GB, the transfer speed of the TF card must be class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-Gemini">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
TF card burning and burning to M2WE through TF card only need to use one of the burning methods
:::

:::info
* If the machine is installed with M2WE and memory card, the system defaults to start from the memory card
* It is recommended to use TF card to flash the system for M2WE, burning through USB to M2W may fail due to driver compatibility and other reasons, it is recommended to directly use [Memory card burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image corresponds, if not, it cannot start
:::
