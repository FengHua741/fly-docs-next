## Preparation Before System Flashing

    * Card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, the transfer speed of the TF card must be Class 10 or above
    * Download the corresponding <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">system image</Button>
    * Download the required <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">flashing tool</Button>

## System Flashing

:::info
Flashing the TF card and flashing to M2WE through the TF card only need to use one of the methods
:::

:::info
* If the machine has M2WE and a memory card installed, the system defaults to booting from the memory card
* M2WE is recommended to flash the system using a TF card, as flashing via USB to M2WE might fail due to driver compatibility issues, it is recommended to directly use [Memory Card Flashing to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image matches, otherwise, it will not boot
:::
