## Preparation Before System Burning

    * Card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, the transmission speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
TF card burning and burning to M2WE via TF card only requires using one of the burning methods
:::

:::info
* If the machine is installed with M2WE and memory card, the system will default to boot from the memory card
* It is recommended to use TF card to burn the system for M2WE. Using USB to burn to M2W may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Please note whether the downloaded system image matches, otherwise it cannot be started
:::

## Installing the Antenna

    1. Install the antenna for the `M2WE` or `2.4G WiFi` module
    :::warning
    * The FLY host machine needs to use the `M2WE` or `2.4G WIFI` module, otherwise WiFi cannot be used
    * Please note that the host machine does not support Chinese name WiFi, and does not support 2.4G and 5G dual-frequency integration WiFi
    :::
    * The red arrow points to the WiFi antenna interface
    * The WiFi antenna port is `ipex1`
    * Please install the antenna on the M2WE, otherwise the weak signal may cause failure to connect to the router
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After the system card installation is complete, you can proceed to the next step, connecting to the host machine via SSH
    :::

    2. Power on the host machine.  
    **Your host machine has already been powered on!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">SSH Connection to the Host Machine</Button>
