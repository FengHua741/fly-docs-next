## Preparations Before System Burning

    * Card reader
    * One TF card with a capacity of no less than 16GB or no more than 128GB, and the transfer speed of the TF card must be Class10 or above
    * Download the corresponding <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">system image</Button>
    * Download the required <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">burning tool</Button>

## System Burning

:::info
Burning to the TF card or burning through the TF card to M2WE only requires using one of the burning methods.
:::

:::info
* If the machine is installed with M2WE and memory card, the system will boot from the memory card by default.
* It is recommended to use the TF card to burn the system for M2WE. Burning through USB may fail due to driver compatibility issues. It is recommended to directly use [Memory Card Burning to M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os).
* Please note whether the downloaded system image corresponds. If it does not correspond, the system cannot start.
:::

## Installing the Antenna

    1. Install the antenna for `M2WE` or `2.4G WiFi` module.
    :::warning
    * A FLY host PC requires either an `M2WE` or `2.4G WIFI` module to use WiFi.
    * Please note that the host PC does not support Chinese-named Wi-Fi networks and does not support dual-band (2.4G and 5G) combined Wi-Fi.
    :::
    * The red arrow points to the Wi-Fi antenna interface.
    * The Wi-Fi antenna port is `ipex1`.
    * Please install the antenna on M2WE, otherwise weak signal may result in failure to connect to the router.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - After completing the installation of the system card, you can proceed to the next step and connect to the host PC via SSH.
    :::

    2. Power on the host PC.  
    **At this point, your host PC is already powered on and running!!!**  
    Please proceed to the next step: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">SSH Connection to the Host PC</Button>
