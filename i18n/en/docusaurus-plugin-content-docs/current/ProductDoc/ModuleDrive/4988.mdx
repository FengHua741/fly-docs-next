# A4988

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988.webp').default} size="50%" align="center" />

## Basic Parameters

    * Suitable for: Marlin/Klipper/RRF firmware
    * Overheat shutdown protection
    * Ground short-circuit protection
    * Load short-circuit protection

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988-1.webp').default} size="80%" align="center" />

## Installing FLY Motherboard
    * Please note that EN needs to be aligned with the upper left corner for installation
    * FLY's DP5, PRO X10, MMU can only use TP drive
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />

    * Drive current adjustment instructions
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988-2.webp').default} size="100%" align="center" />
    * 4988 uses normal jumper cap mode, supports configuring 16 microsteps after connecting four jumpers
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-dip.webp').default} size="100%" align="center" />

## Klipper Reference Configuration
    * 4988 does not need to configure drivers such as `[tmc5160 stepper_x]` or `[tmc2209 stepper_x]`, etc., they are not needed and cannot be used normally after configuration
    * Set `microsteps:` to `16`
