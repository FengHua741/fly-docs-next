# Extend Drive LED through Plugin

You need to connect to the host computer via SSH and follow the link below for installation and configuration

```
https://github.com/julianschill/klipper-led_effect
```

## Installation

Please use **MobaXterm_Personal** or other **SSH tools** to connect to your host computer via **WIFI**, and make sure of the following points:

1. **Please ensure that the host computer has the Klipper firmware installed**
2. **Please ensure that the logged-in user must be the one who installed Klipper**
3. **Please ensure that your input method is English**
4. **Please ensure that your host computer can normally detect the device**
5. **Please ensure that all the above precautions are taken, otherwise the next step cannot be proceed**

This module can be installed into the existing Klipper installation using the installation script.

```
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Reference Configuration

```
[neopixel TP_led]
pin: PB7
chain_count: 25
# Number of LEDs
# Number of LED beads
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)
```
