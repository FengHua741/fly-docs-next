# Screen Parameter Modification Tutorial
:::tip
Open your IP:9999 in your browser to access FLY-Tools
:::

:::tip
The HDMI port on the FLY host computer is Micro HDMI, please pay attention when purchasing HDMI cables!!!
:::

:::tip
FLY's host computers that support FPC single-line communication include `FLY-PI-V2`, `FLY-C8`, `FLY-PRO-X10`, and `FLY_LITE2/2.1`

:::
    * To enable the screen, you need to turn on the option below
    * After entering FLY Tools
* The first step requires you to enter fly_TOOLS to modify parameters
