# Screen Parameter Modification Tutorial
:::tip
Open your IP:9999 in your browser to access FLY-Tools
:::

:::tip
The HDMI interface on the FLY host board is Micro HDMI, please pay attention when purchasing HDMI cables!!!
:::

:::tip
FLY's host supports FPC One-Line Connection with `FLY-PI-V2`, `FLY-C8`, `FLY-PRO-X10`, or `FLY_LITE2/2.1`

:::
    * You need to enable the screen by turning on the option below when using HDMI
    * After entering FLY Tools
* The first step is to enter fly_TOOLS to modify parameters
