## Product Overview

    * Screen size: 120.7mm*76.0mm
    * Screen display area: 108mm*66.2mm
    * Resolution: 800*480
    * Screen type: IPS
    * Communication method: DSI/Mipi
    * Onboard interface: 15 pin DSI / 22pin Mipi
    * Working stability: 0°-60°C
    * Touch type: Capacitive (Capacitive)
    * Logic voltage: 3.3V
    * Weight: 135.5g

        :::danger Notice

        * The DSI and CSI interfaces of Raspberry Pi 5 have now been renamed to MIPI
        * The MIPI interface can be used directly through configuration with a CSI camera or DSI screen
        * Please note the direction of the gold fingers

        :::

        * Raspberry Pi 5 now has two MIPI interfaces, if you look from the front you can see marks 1 and 0 on top

        * DSI screen wiring


            * If you use interface number 1, you need to configure the screen as follows

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * If you use interface number 0, you need to configure the screen as follows

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
