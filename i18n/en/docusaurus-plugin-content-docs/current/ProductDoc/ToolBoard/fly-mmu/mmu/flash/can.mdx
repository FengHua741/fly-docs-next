        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/can.webp').default} size="100%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface
    * Enter the following command to start compiling, it takes a while
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * The final output indicating success will look like this:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Flash Firmware

    * Before flashing the firmware, use an XT30 data cable to connect the tool board to UTOC or a mainboard flashed with bridge firmware. Below is a reference wiring diagram.
    * Please ensure that the DIP switch is set correctly
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/can_mode.webp').default} size="90%" align="center" />
    * Connection to UTOC reference
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Ensure that the resistance between CAN H and CAN L is around `60立` when measured with a multimeter in a power-off state; if it is `120立`, a `120立` jumper should be connected; if it is around `40立`, the jumper should be removed
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/120.webp').default} size="50%" align="center" />
