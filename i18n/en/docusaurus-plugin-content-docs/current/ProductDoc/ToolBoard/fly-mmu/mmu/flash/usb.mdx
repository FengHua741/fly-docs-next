# USB Firmware Compilation

    :::tip

    * The CAN-connected Katapult firmware pre-installed on the MMU at factory
    * If using USB firmware burning, need to burn the USB Katapult firmware
    * Only needs to be burned once after which it does not need to be burned again
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapult Firmware Burning</Button>

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Select GPIO pins to set at micro-controller startup, input: !PA15 Please note that ! is uppercase
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * Press `Q`, a **Save configuration** prompt will appear, then press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you have exited to the command line interface
    * Enter the following command to start compilation, it takes some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Finally, if the output below appears, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
