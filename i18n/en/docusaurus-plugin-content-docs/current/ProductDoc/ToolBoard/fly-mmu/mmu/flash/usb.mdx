# USB Firmware Compilation

    :::tip

    * MMU factory pre-flashed CAN-connected Katapult firmware
    * If using USB firmware burning, you need to burn the USB Katapult firmware
    * No need to burn again after one time
    * <Button variant="contained" disableElevation href="./bl">Katapult Firmware Burning</Button>

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Select GPIO pins to set at micro-controller startup, enter: !PA15 Please note that! is uppercase
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * Press `Q` key, when **Save configuration** appears, then press `Y` key
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you have exited to the command line interface
    * Enter `make -j4` to start compilation, it takes some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
