# Product Overview

    FLY-ERCF is a multi-color control board for 3D printers based on RP2040, mainly used for the Enraged Rabbit project. It can also be used for other two-axis stepper motor control scenarios.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.webp').default} size="100%" align="left" />

## Product Features

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Two drive positions, support UART and SPI drivers
    * Compatible with the original ERCF EASY Board
    * Extremely compact size, using 4-layer board design
    * On-board 2.54MM double-row 7PIN expansion port, can be extended for limit and other uses
    * Support 12-24V voltage
    * Support CAN bus and RS232 connection, more stable data transmission, lower latency, more stable connection
    * CAN interface uses XT30 (2+2), supports 15A current

## Product Parameters

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Firmware: Klipper
    * Input voltage: DC12V-24V
    * Interface: 2.54MM double-row 7PIN expansion port, SERVO X1, ENDSTOP X2, EXTRA X1
    * Drive: Support UART/SPI and STEP/DIR

    * Support 12-24V voltage
    * Support CAN bus, more stable data transmission, lower latency, more stable connection
    * CAN interface uses XT30 (2+2), supports 15A current
