# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * ERCF supports CAN, USB
    * Here provides CAN firmware at 500K and 1M rates
    :::

## Enter Burning Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * To enter burning mode, the motherboard must be completely powered off, press and hold the BOOT button, then connect one end of the data cable with data transmission to the motherboard and the other end to the host computer
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/boot.webp').default} size="100%" align="center" />

                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * Below is the firmware writing instruction for 1M CAN rate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * Below is the firmware writing instruction for USB
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```
