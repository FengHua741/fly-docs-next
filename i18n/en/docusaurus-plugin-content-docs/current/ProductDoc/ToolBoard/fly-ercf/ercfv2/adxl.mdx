## Switching Klipper

    * Because it hasn't been merged yet, you need to switch Klipper to use it

    ```bash
    sudo cp -r klipper klipperdev
    sudo rm -rf ~/klipper
    git clone https://github.com/FLYmaker/klipper.git
    sudo systemctl restart klipper
    ```

    * Reference Configuration
    ```cfg
    #####################################################################
    #                       LIS3DH Accelerometer Configuration (Enable when needed)
    #####################################################################
    [lis2dw]
    is_lis3dh:True
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis2dw
    probe_points:
        125, 125, 20  #Set this to the coordinates of the nozzle during resonance measurement


```
