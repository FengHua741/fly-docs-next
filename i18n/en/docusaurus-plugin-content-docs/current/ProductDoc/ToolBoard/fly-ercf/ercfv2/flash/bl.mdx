# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * ERCF V2 supports CANR, S232, USB
    * Only CAN firmware is provided here at 500K and 1M rates

    :::

## Enter Boot Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * To enter boot mode, completely power off the motherboard, press and hold the BOOT button, then connect one end of the data cable with data transfer to the motherboard and the other end to the host computer
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.webp').default} size="100%" align="center" />

            * Below is the firmware flashing command for 500K CAN rate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_500K.uf2
                ```
                       
            * Below is the firmware flashing command for 1M CAN rate
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_1M.uf2
                ```
            * Below is the USB firmware flashing command
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCFV2_CANBOOT_USB.uf2
                ```

            5. If the following prompt appears, the flashing is successful
