# USB Firmware Compilation

    :::tip

    * The USB accelerometer comes pre-flashed with the USB-connected Katapult firmware
    * If it is found that it has not been burned, you can burn it yourself
    * [Katapult Firmware Burning](./bl.mdx "Click to jump")

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q` key, and when **Save configuration** appears, press `Y` key
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you have exited to the command line interface
    * Input `make -j4` to start compilation, it takes some time
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output in the end, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
