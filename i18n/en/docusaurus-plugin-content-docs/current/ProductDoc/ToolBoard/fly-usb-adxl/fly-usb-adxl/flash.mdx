# USB Firmware Compilation

    :::tip

    * The USB accelerometer has the Katapult firmware for USB connection pre-flashed at the factory.
    * If it is not flashed, you can flash it yourself.
    * [Katapult Firmware Burning](./bl.mdx "Click to jump")

    :::

## Start Compiling

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q`, and when **Save configuration** appears, press `Y` again.
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface.
    * Enter `make -j4` to start compiling, it takes a while.
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output at the end, compilation is successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
