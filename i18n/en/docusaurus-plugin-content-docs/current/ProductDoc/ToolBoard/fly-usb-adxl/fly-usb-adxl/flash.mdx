# USB Firmware Compilation

    :::tip

    * The USB accelerometer has the Katapult firmware pre-flashed for USB connection at the factory.
    * If you find that it hasn't been flashed, you can flash it yourself.
    * [Katapult Firmware Flashing](./bl.mdx "Click to jump")

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q`, and when **Save configuration** appears, press `Y` again.
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface.
    * Enter `make -j4` to start compiling, this will take some time.
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed at the end, the compilation was successful.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
