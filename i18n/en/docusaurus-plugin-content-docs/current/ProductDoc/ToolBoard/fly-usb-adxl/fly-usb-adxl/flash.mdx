# USB Firmware Compilation

    :::tip

    * The USB accelerometer has the USB-connected Klipper firmware pre-flashed at the factory.
    * Only the firmware needs to be updated.
    * You can connect the computer and the USB accelerometer with a USB cable.

    :::

## Start Compiling

    <SSH />

    <MakeOperate/>

    <USB />

    * Press `Q`, and when **Save configuration** appears, press `Y` again.
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have returned to the command line interface.
    * Enter `make -j4` to start compiling, it takes some time.
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed, the compilation is successful.

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
