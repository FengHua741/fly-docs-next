# USB Firmware Compilation

    :::tip

    * The USB accelerometer has the Katapult firmware for USB connection pre-flashed at the factory.
    * If it is not flashed, you can flash it yourself.
    * [Katapult Firmware Burning](./bl.mdx "Click to jump")

    :::

## Start Compiling

    <SSH />

    <MakeOperate/>

    <USB />
    * Select GPIO pins to set at micro-controller startup, input: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Press `Q`, and when **Save configuration** appears, press `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should be back at the command line interface
    * Enter `make -j4` to start compiling, it takes a while
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Finally, if the output below appears, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
