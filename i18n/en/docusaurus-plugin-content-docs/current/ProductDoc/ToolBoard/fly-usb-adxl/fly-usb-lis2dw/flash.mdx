# USB Firmware Compilation

    :::tip

    * The USB accelerometer has the Katapult firmware connected via USB pre-flashed at the factory.
    * If you find that it hasn't been flashed, you can flash it yourself.
    * [Katapult Firmware Flashing](./bl.mdx "Click to jump")

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Select GPIO pins to set at micro-controller startup, input: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Press `Q` key, a **Save configuration** prompt will appear, then press `Y` key
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and exited to the command line interface
    * Input `make -j4` to start compilation, it takes some time
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Finally, if the output below appears, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
