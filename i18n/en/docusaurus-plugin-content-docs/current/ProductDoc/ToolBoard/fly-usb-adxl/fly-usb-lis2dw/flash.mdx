# USB Firmware Compilation

    :::tip

    * The USB accelerometer comes with pre-installed Klipper firmware for USB connection
    * Only the firmware needs to be updated
    * You can connect the computer and the USB accelerometer using a USB cable

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Select GPIO pins to set at micro-controller startup, input: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Press `Q`, a **Save configuration** message will appear, then press `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should have been saved and you should be back to the command line interface
    * Input `make -j4` to start compiling, it takes some time
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed, compilation is successful

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
