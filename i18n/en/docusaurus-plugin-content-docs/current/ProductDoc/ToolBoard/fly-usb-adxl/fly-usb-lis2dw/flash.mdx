# USB Firmware Compilation

    :::tip

    * The USB accelerometer comes pre-flashed with Katapult firmware for USB connection
    * If no firmware is found, you can flash it yourself
    * [Katapult Firmware Burning](./bl.mdx "Click to jump")

    :::

## Start Compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Select GPIO pins to set at micro-controller startup, input: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/usb.webp').default} size="100%" align="center" />
    * Press `Q` key, and when **Save configuration** appears, press `Y` key
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should exit to the command line interface
    * Input `make -j4` to start compilation, it takes some time
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the following content is output, the compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
