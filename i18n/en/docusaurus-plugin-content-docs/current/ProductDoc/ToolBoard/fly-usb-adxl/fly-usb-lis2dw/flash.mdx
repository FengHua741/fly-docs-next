# USB Firmware Compilation

    :::tip

    * The USB accelerometer has the USB-connected Klipper firmware pre-flashed at the factory.
    * Only the firmware needs to be updated.
    * A USB cable can be used to connect the computer and the USB accelerometer.

    :::

## Start Compiling

    <SSH />

    <MakeOperate/>

    <USB />
    * Select GPIO pins to set at micro-controller startup, input: gpio8
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Press `Q` key, a **Save configuration** prompt will appear, then press `Y` key
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you are back to the command line interface
    * Enter `make -j4` to start compiling, it takes some time
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below appears, the compilation is successful

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
