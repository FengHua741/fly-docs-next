# CAN Firmware Compilation

## Start Compilation

    <SSH />

    <CAN />

    * Press `Q` key, and when the **Save configuration** prompt appears, press `Y` again
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have exited to the command line interface
    * Enter `make -j4` to start compiling, it will take some time
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below is displayed, compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    <LS />


## Firmware Burning

    * Before burning the firmware, connect the tool board to UTOC or a mainboard with bridge firmware flashed using an XT30 data cable; refer to the wiring diagram below
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />
    * Ensure that the resistance between CAN H and CAN L measures around `60立` while the power is off. If it's `120立`, install a `120立` jumper. If it's around `40立`, remove the jumper
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/flash/img/120.webp').default} size="100%" align="center" />
