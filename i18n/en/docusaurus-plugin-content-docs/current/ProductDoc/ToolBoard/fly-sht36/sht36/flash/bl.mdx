            4. Flash Firmware

            :::tip

            Only one of the firmwares needs to be flashed

            :::

            * Below is the firmware flashing command for 500K CAN rate
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_500K.BIN
                ```
                       
            * Below is the firmware flashing command for 1M CAN rate
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_1M.BIN
                ```


            5. If the content shown by the arrow in the figure below ``File downloaded successfully`` appears, the flashing is successful

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::tip

            After flashing, the motherboard needs to be completely powered off once.
