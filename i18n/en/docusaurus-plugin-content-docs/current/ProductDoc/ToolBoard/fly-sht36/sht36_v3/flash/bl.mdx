# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SHT36_V3 supports CAN and RS232
    * Only CAN firmware is provided here for 500K and 1M rates
    :::

## Enter Boot Mode

    :::tip

    The arrow points to the `BOOT button`

    :::

    * With the motherboard completely powered off, press and hold the BOOT button, then connect one end of the data cable with data transfer to the motherboard and the other end to your device
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="100%" align="center" />

            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Below is the firmware flashing command for 1M CAN rate
