# Product Introduction

FLY-SHT36-LIS3DH is a nozzle tool board designed by Guangzhou Meilun Electronic Technology Co., Ltd. for 3D printer extrusion heads. Using this tool board, four wires can be used to replace the complicated wiring between the original motherboard and the print head, optimizing the layout. FLY-SHT36-LIS3DH is suitable for 36-step motors and integrates functions such as CAN transceiver, USB port, TMC2209 driver, LIS3DH accelerometer, two controllable fans, etc. This tool board is suitable for Klipper firmware.

    * Extremely compact size, using 6-layer board design
    * Supports CAN bus connection, more stable data transmission, lower latency, more stable connection
    * On-board TMC2209 driver, using UART mode, can be used for extruders
    * On-board ADXL345 accelerometer
    * Supports PT100 (MAX31865)
    * Replaceable fan MOS design
    * Compatible with BLTOUCH, PL08N and other leveling sensors, integrated XY limit switches
    * Supports 12-24V voltage input 
    * MCU: STM32F072
    * Communication: CANBUS, USB
