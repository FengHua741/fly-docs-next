# Katapult Firmware

## Katapult Firmware Description

    <CANBOOT />

    :::tip

    * SHT36 PRO supports CAN
    * Only CAN firmware at 500K and 1M rates is provided here
    :::

## Enter Boot Mode

    :::tip

    The circled part is the `BOOT button`

    :::

    * With the motherboard completely powered off, short-circuit the BOOT button and then connect one end of the data cable with data transmission to the motherboard and the other end to your device
    * Then remove the short circuit from the BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.webp').default} size="80%" align="center" />

            :::tip

            Only one of the firmwares needs to be flashed

            * Below is the firmware flashing command for 500K CAN rate
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Below is the firmware flashing command for 1M CAN rate
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```
