# Compilation of CAN Firmware for Version 103

## Start Compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Select **GPIO pins to set at micro-controller startup**, enter: !PC13 (Note: It must be uppercase)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/103.webp').default} size="100%" align="center" />
    * Press the `Q` key, a **Save configuration** prompt will appear, then press `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Now the configuration should be saved and you should have returned to the command line interface
    * Enter the following command to start compiling, it may take some time
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * If the output below appears, compilation is successful

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

    <LS />

## Flash Firmware

    * Before flashing the firmware, connect the tool board to UTOC or the mainboard with bridge firmware using an XT30 data cable, and refer to the wiring diagram below
    * Please ensure that the DIP switch is set correctly
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * Ensure that the resistance between CAN H and CAN L measures around `60立` in a power-off state; if it is `120立`, a `120立` jumper should be connected; if it is around `40立`, remove the jumper
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.webp').default} size="50%" align="center" />
