# Product Introduction

FLY SB2040 PRO is a tool board launched by Guangzhou Meilun Electronic Technology Co., Ltd. for the VORON Stealthburber print head. Using this tool board, four wires can be used to replace the complicated wiring between the original motherboard and the print head, optimizing the layout. SB2040 is suitable for stealthburner and afterburner. It integrates CAN transceiver, USB port, TMC2240 driver, ADXL345 accelerometer, three controllable fans and other functions. This tool board is suitable for Klipper and RRF firmware.

## Product Features

* Reserved Boot button, users can enter Boot mode to update firmware through TypeC
* Support Klipper firmware
* Temperature sensor support: On-board MAX31865 chip, compatible with PT100
* On-board ADXL345 accelerometer
* Reserved RGB interface, on-board 2240 driver, support multiple leveling sensors, such as klicky, Voron Tap, PL08n, etc.
* Support 12 - 24V voltage, 15A current input, enlarged heating rod port, rated current 10A

## Product Parameters


* MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
* Connection interface uses XT30(2 + 2) connector to support 15A current, peak 30A
* Firmware: Klipper
* Input voltage: DC12V - 24V
* Logic voltage: DC3.3V
* On-board sensors: ADXL345 and optional MAX31865
* Heating rod interface: Heating rod (HEAT0)
* Expansion interface: Controllable fan * 3, RGB * 1, limit * 3, heating tube * 1, thermistor * 1
* Fan interface: Three-way controllable fan
* Motor driver: On-board TMC2240
