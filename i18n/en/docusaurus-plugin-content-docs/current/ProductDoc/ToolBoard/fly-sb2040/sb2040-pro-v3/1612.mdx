# Eddy Usage

    :::info
  * If you have previously configured eddy, please delete it
  * When installing Eddy, the coil should be above the nozzle and 0.6-0.8cm away from the nozzle
  * Do not turn on the heated bed during testing
  * Before using, delete the `probe` configuration in the configuration
    :::

    :::warning Note
    * For multi-Z-axis machines, manual leveling needs to be done once
    :::

## Usage Instructions
   * Log in to the terminal via SSH tool using the IP address, and run the script
    Download the required SSH tool <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">MobaXterm tool</Button>


<Tabs>
  <TabItem value="eddy1" label="International Use">
    # Please use the following command for international use
    ### Below is the installation of the eddy automated script
    * Open the downloaded SSH tool, enter the following command and press Enter, then click copy
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * In the `fluidd` web page, add the following configuration to the `moonraker.conf` file for easy updates later
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * If updated, execute the following command
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
