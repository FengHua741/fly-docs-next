# Eddy Usage

    :::info
  * If you have previously configured eddy, please delete it
  * When installing Eddy, the position of the bottom relative to the nozzle must be at least 1-2 mm above the nozzle
  * Do not turn on the hot bed during testing
  * Delete the `probe` configuration in the settings before use
    :::


    :::warning Notice
    * Multi-Z-axis machines need to be manually leveled once
    :::


## Wiring Diagram

    * Install with the trademark facing the hot bed

## Instructions
   * Use ssh tool to log in to the terminal via ip and execute the script
    Download the required ssh tool <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxterm tool</Button>


<Tabs>
  <TabItem value="eddy1" label="International Use">
    # For international use, please use the following commands
    ### Start installing eddy automatic script below
    * Open the downloaded ssh tool, enter the following command and press Enter, click copy
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Add the following configuration in the `moonraker.conf` configuration file on the `fluidd` webpage for easy future updates
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * If updated, please execute this command
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
