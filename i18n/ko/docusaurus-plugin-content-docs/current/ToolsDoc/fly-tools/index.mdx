# FLY-Tools 소개

    FLY-Tools는 광주 망렌 전자 기술 유한 회사에서 Klipper를 위해 맞춤화한 웹 디버그 도구입니다. Fly 상위 기계(시스템에 내장되어 있어 설치할 필요가 없습니다)뿐만 아니라 대부분의 Klipper 서비스가 설치된 상위 기계와 호환됩니다.

    :::팁
    * FLY-Tools는 Klipper가 설치된 Linux 시스템에서만 작동합니다.
    :::

## FLY-Tools 특징

    * 다국어 지원, 어두운 테마 지원
    * USB 시리얼 ID, USB 장치 ID, CANbus UUID, Video 장치 ID를 검색하고 한 번의 클릭으로 복사할 수 있습니다.
    * 특정 CAN 장치의 활성화 또는 비활성화를 한 번의 클릭으로 지원합니다.
    * 온라인으로 CAN 장치의 속도 및 전송 버퍼 크기를 수정할 수 있습니다.
    * 온라인으로 FLY 보드의 Klipper 펌웨어를 컴파일합니다 (Katapult 펌웨어 컴파일은 적합 중입니다).
    * DFU, HID, CAN 등 다양한 방법으로 온라인으로 프로그래밍할 수 있습니다.
    * 모든 컴파일된 펌웨어 파일을 다운로드할 수 있습니다.
    * 온라인으로 Klipper 부하 그래프를 생성할 수 있습니다.
    * 공명 그래프를 자동으로 측정하고 생성하는 데 필요한 단일 클릭을 지원합니다.
    * 웹 SSH 연결을 지원합니다.
    * [ttyd](https://github.com/tsl0922/ttyd) 기반의 웹 페이지 SSH를 지원합니다.
    * FLYOS 3.3.4 버전 시스템을 지원하며, 화면 및 웹 표시 등을 구성할 수 있습니다.
