# FLY-Tools 소개

    FLY-Tools는 광주 망렌 전자 기술 유한 회사에서 Klipper를 위해 맞춤 제작한 웹 디버깅 도구로, Fly 호스트 컴퓨터(시스템에 내장되어 있어 별도 설치가 필요 없음)뿐만 아니라 대부분의 시장에 출시된 Klipper 서비스를 설치한 호스트 컴퓨터와 호환됩니다.

    :::tip 팁

    * FLY-Tools는 Klipper가 설치된 Linux 시스템에만 적용됩니다.

    :::

## FLY-Tools 특징

    * 다국어 지원, 어두운 테마 지원
    * USB 시리얼 ID, USB 장치 ID, CANbus UUID, Video 장치 ID 쿼리 및 한 번의 클릭으로 복사 지원
    * 지정된 CAN 장치 활성화 또는 비활성화를 위한 한 번의 클릭 지원
    * 온라인으로 CAN 장치의 속도 및 송신 버퍼 크기 수정 지원
    * 온라인으로 FLY 보드의 Klipper 펌웨어 컴파일 지원(Katapult 펌웨어 컴파일은 현재 적합 중)
    * DFU, HID, CAN 등의 방식을 사용한 온라인 프로그래밍 지원
    * 모든 컴파일된 펌웨어 파일 다운로드 지원
    * 온라인으로 Klipper 부하 그래프 생성 지원
    * 자동으로 공명 그래프를 측정하고 생성하는 기능 지원
    * 웹SSH 연결 지원
    * [ttyd](https://github.com/tsl0922/ttyd) 기반 웹 페이지 SSH 지원
    * FLYOS 3.3.4 시스템 지원, 화면 구성, 웹 표시 등 설정 가능
