# 기능 시연

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.webp').default} size="100%" align="left" />


    * 호스트 이름 수정 지원
    * 시스템 시간대 수정 지원
    * 상위기기 40핀 확장 활성화 지원 (PWM 활성화 후 상위기기가 시리얼 포트로 연결되지 못할 수 있음, 디스플레이 활성화 후 상위기기의 가속도 센서 포트를 사용하지 못할 수 있음)
    * 메인보드 모델 수정 지원
    * Klipper 웹 페이지 수정 지원, Fuidd와 Mainsail 지원
    * KlipperScreen 열기 또는 닫기 지원
    * Klipper 프로세스가 지정된 CPU 코어에서 강제로 실행되도록 지원, Klipper가 선택한 코어에서 실행될 수 있음
