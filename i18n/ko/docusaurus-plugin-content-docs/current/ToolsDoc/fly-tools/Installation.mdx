# FLY-Tools 설치하기

    1. 최신 버전의 Klipper 가져오기

    ```bash
    cd ~/klipper
    git pull
    ```

    * 로컬 Klipper 소스 코드를 수정한 경우 가져오기에 실패할 수 있습니다. 이 경우 다음 명령어를 사용하세요. (이 작업은 이전에 수정한 파일을 삭제합니다.)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. 최신 Katapult(원 Canboot) 가져오기

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Fly-Tools 서비스 설치

        * 설치 과정은 Git 서비스 제공자에 의존합니다. 다운로드 진행 상황이 멈추면 `CTRL+C`를 눌러 종료하고 다시 시도하세요.
        * Github는 한 시간에 최대 60번의 요청을 허용합니다. Gitee는 알 수 없습니다.
        * 국내 사용자는 설치 스크립트를 Gitee를 통해 얻는 것을 권장합니다.
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Github를 통해 설치 스크립트 가져오기

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
