# RRF 구성

## 필러 감지 및 블록 감지 기능 구성

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} 크기="70%" 정렬="가운데" />

1. RRF 프린터 웹 페이지 인터페이스를 열고 `Files` -> `System`으로 이동하여 `config.g` 파일을 찾은 후, 파일을 열고 편집합니다.
2. 파일에 다음 코드를 추가하여 필러 감지 기능을 활성화합니다.

```gcode
M591 D0 P2 C"PA9" S1
```

3. 파일에 다음 코드를 추가하여 블록 감지 기능을 활성화합니다. (`L` 매개변수는 사용하는 코딩 디스크에 따라 변경되며, 오작동이 발생하면 적절히 이 값을 증가시켜야 합니다)

```gcode
M591 D0 P7 C"PA10" L3.7 R75:125 S1
```
