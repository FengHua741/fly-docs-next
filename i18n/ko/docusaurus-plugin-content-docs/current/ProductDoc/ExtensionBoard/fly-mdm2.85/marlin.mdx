# Marlin 구성

## 필라멘트 끊어짐 및 막힘 기능 설정

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/marlin.webp').default} size="70%" align="center" />

1. Marlin 구성 파일 `Configuration.h`을 열고, `#define FILAMENT_RUNOUT_SENSOR`를 찾은 후 주석 기호를 제거합니다.
2. `#define FILAMENT_RUNOUT_DISTANCE_MM`를 찾아 주석 기호를 제거하고 값을 `3.7`로 설정합니다 (사용 중인 코드 휠에 따라 다릅니다. 잘못된 트리거가 발생하면 이 값을 적절히 증가시켜야 합니다).
3. `#define FILAMENT_MOTION_SENSOR`를 찾아 주석 기호를 제거합니다.
4. Marlin의 필라멘트 센서 기능을 사용하려면 `#define NOZZLE_PARK_FEATURE`와 `#define ADVANCED_PAUSE_FEATURE`도 활성화해야 합니다. 주석을 제거한 다음 프린터에 맞는 특정 파라미터를 설정하세요.
