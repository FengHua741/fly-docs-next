# 사용 가이드

## 주의사항

    * 첫 사용 시 인터넷 연결이 필요합니다. 만약 Wi-Fi가 인터넷에 연결되지 못하면 halo는 최초 부팅 설정을 받지 못할 수 있습니다.
    * 설정 페이지에서 모든 설정은 **설정 저장** 버튼을 클릭하고 장치를 재부팅한 후에야 적용됩니다.
    * 물건 막힘 감지 기능은 기본적으로 비활성화되어 있으며, 설정에서 활성화한 후 재부팅해야만 홈 화면에서 위로 스와이프하여 접근할 수 있습니다.
    * 종료 후 전원 끄기 기능은 기본적으로 비활성화되어 있으며, 설정에서 활성화할 수 있습니다.
    * 모든 팝업에서 프린트 취소 또는 프린트 일시 중지 버튼을 누르려면 길게 누르지 않으면 작동하지 않습니다 (실수 방지를 위해).
    * 시스템 서비스 페이지에서 Host 서비스는 상위 호스트 컴퓨터입니다.
    * 공식 펌웨어를 사용할 때는 메인 컨트롤러 칩 또는 플래시 칩을 교체하지 마십시오.

    :::팁

    * 주의: `moonraker.conf` 구성 파일 크기가 4MB를 초과하면 정상적으로 작동하지 않을 수 있습니다. 문제가 발생하면 `moonraker.conf` 구성 파일의 크기를 줄이십시오.

    :::

## 시스템 펌웨어 업데이트

    * halo 버전 업데이트는 전체 업데이트 및 OTA 증분 업데이트 두 가지 유형으로 나뉩니다.
        * 전체 업데이트: 하단 계층의 큰 업그레이드가 포함되므로 USB를 통해 전체 펌웨어 패키지를 번들링해야 합니다.
        * 증분 업데이트: 일반적인 OTA 업그레이드이며, 장치 정보 페이지에서 업데이트를 확인할 수 있습니다.

## 핸드폰으로 QR 코드 스캔 및 Wi-Fi 구성

    :::팁

    * 일부 모바일 기기는 Wi-Fi 연결을 위한 QR 코드 스캔 기능이 없을 수 있으므로, 휴대폰의 Wi-Fi 설정에서 **FLY-halo-xxxxxxxx**라는 네트워크에 연결할 수 있습니다.

    :::

    :::팁

    * 핫스팟에 연결한 후 웹 페이지가 자동으로 나타나지 않는다면, 브라우저에서 링크를 수동으로 열어야 합니다.

    :::

    :::팁

    * HALO는 2.4GHz Wi-Fi만 사용할 수 있다는 점에 주의하십시오.

    :::

    :::팁

    * Halo 화면은 숨겨진 Wi-Fi를 검색할 수 없습니다.

    :::
