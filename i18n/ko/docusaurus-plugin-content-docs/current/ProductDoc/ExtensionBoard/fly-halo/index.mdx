# 제품 소개

    * 메인 컨트롤러는 ESP32-S3R8를 사용합니다. ESP32보다 성능이 더 우수합니다. 8MB PSRAM과 16MB 플래시가 있습니다. 화면은 IPS 스크린으로, 해상도가 높고 밝기와 시야각이 크며 색상이 섬세합니다. 터치 기능을 지원합니다.

## 제품 특징

    :::tip

    * 현재 총 6개의 페이지가 있으며, 좌우 상하로 슬라이드하여 전환할 수 있습니다.

    :::

    * 홈 화면 (대기, 인쇄, 정지, 오류 등의 상태 표시)

    * 스queezer (extruder) (extrude, retract, extrude 온도 제어)

    * 시스템 서비스 (host, klipper, moonraker 등의 서비스 시작, 중지, 재시작 제어)

    * 설정 (halo의 다양한 매개변수, wifi, ip, 언어, 백라이트 밝기 등 설정)

    * 장비 정보 (halo 정보, 펌웨어 버전 표시. 업데이트 확인)

    * 소모품 감지: 필라멘트 끊김, 막힘 감지, 필라멘트 길이 감지, 막힘 지정 길이 후 자동 정지 및 GPIO47 높이 조정 (이 모든 것이 기본적으로 꺼져 있으며 설정에서 활성화 가능)

    * 인쇄 완료 후 종료: 인쇄 완료 후 지정된 시간 후에 호스트를 종료하고 GPIO48 높이 조정 (이 모든 것이 기본적으로 꺼져 있으며 설정에서 활성화 가능)
