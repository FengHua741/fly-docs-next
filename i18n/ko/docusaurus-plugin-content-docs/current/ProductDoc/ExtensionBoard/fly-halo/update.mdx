## 인크리멘탈 업그레이드 (OTA)

    * Halo 장치 정보 페이지에서 버전 번호 위치를 클릭하여 업데이트 확인이 가능합니다. 사용 가능한 업데이트가 있으면 알림이 표시됩니다.
        이때 Update를 클릭하면 업그레이드가 시작됩니다. 전원을 끄지 말고, 진행 상황이 100%가 되면 자동으로 재부팅됩니다.


## 풀 업그레이드 (USB)

    * 전체 업그레이드 펌웨어가 발표되면 Halo를 USB로 컴퓨터에 연결하여 업그레이드해야 합니다
    :::팁

    * 전체 업그레이드로 기존 설정은 보존되지 않으며 모든 설정이 기본값으로 재설정됩니다

    :::

    1. USB 플래싱 도구 [flash_download_tool_3.9.4.zip(클릭하여 다운로드)](https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip)를 다운로드하고 압축을 해제합니다.
    2. 펌웨어 [Halo 펌웨어 다운로드](docs/ProductDoc/ExtensionBoard/fly-halo/firmware.mdx)를 다운로드합니다.
    3. ``flash_download_tool_3.9.4.exe``를 엽니다. 주의: 플래싱 도구를 열면 먼저 CMD 창이 표시되고 그 다음에 플래싱 도구 화면이 나타납니다. CMD 창을 닫지 마십시오.
    4. 플래싱 도구에서 ``ChipType:``을 ``ESP32-S3``로 선택합니다.
    5. 플래싱 도구에서 ``WorkMode:``를 ``Develop``로 선택합니다.
    6. 플래싱 도구에서 ``LoadMode:``를 ``USB``로 선택합니다.
    7. 마지막으로 ``OK``를 클릭할 수 있습니다. 참고: 그림 1을 참조하십시오.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/flash-2.webp').default} size="50%" align="left" />
    
    :::팁

    * 그림 2의 ``초록색 영역``은 이미지와 동일하게 유지되어야 합니다. 임의로 수정하지 마십시오.

    :::

    8. Halo를 USB-TypeC 데이터 케이블로 컴퓨터에 연결합니다.
    9. 그림 2의 ``파란색 영역1`` 버튼을 클릭하여 다운로드한 펌웨어를 선택합니다.
    10. 그림 2의 ``파란색 영역2``를 클릭하여 Halo의 시리얼 포트를 선택합니다(포트가 없는 경우 Halo의 뒷면에 있는 ``B`` 버튼을 누른 상태로 컴퓨터에 연결할 수 있습니다).
    11. 그림 2의 ``파란색 영역3``을 클릭하여 플래싱을 시작합니다. 플래싱이 완료되면 Halo를 재부팅하여 전체 업그레이드를 완료합니다(플래싱 완료 메시지를 받기 전까지 Halo의 연결을 끊어서는 안 됩니다).
