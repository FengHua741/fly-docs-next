### 어댑터 보드 및 연결 케이블 설치
    * 배선 위치는 개인의 취향에 따라 달라질 수 있습니다. 반드시 팬 뒷면에 붙일 필요는 없습니다.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.webp').default} size="50%" align="left" />

    :::warning 주의사항

    * 배선 색상 설명 **검은색은 음극 GND**, **빨간색은 양극 VCC (DC12-24V) 가능**, **하얀색은 GPIO47 (부자재 감지용)**, **노란색은 GPIO48 (작업 종료 후 정지용)**

    :::
