## 홈

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/home-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/home-2.webp').default} size="50%" align="left" />

    2. 클릭: 핫베드 온도 설정으로 이동, 원형 막대를 드래그하여 온도를 조정하고 영역 2를 다시 클릭하면 설정한 온도를 확인합니다.

## 엑트루드 페이지

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.webp').default} size="50%" align="left" />
    * 위의 그림 1의 가동 영역은 아래의 번호와 대응됩니다.
    1. 클릭: 엑트루드 헤드 목표 온도 증가
    2. 길게 누르기: 엑트루드 온도 설정으로 이동, 원형 막대를 드래그하여 온도를 조정하고 다시 클릭하면 설정한 온도를 확인할 수 있습니다.
    3. 클릭: 엑트루드 헤드 목표 온도 감소
    4. 클릭: 엑트루드
    5. 상하 스크롤: 엑트루드 또는 리트랙 길이를 선택할 수 있습니다. 단위는 mm입니다.
    6. 클릭: 리트랙
    7. 길게 누르기: 위의 그림 2의 엑트루드 속도 설정으로 이동, 상하 스크롤하여 엑트루드 속도를 선택하고 단위는 mm/s입니다. 선택 후 주변 빈 영역을 클릭하여 종료합니다.
