# FLY_halo

## 1. 홈 화면

    단순히 열판 또는 열헤드를 클릭한 후 화면 상단에서 시계 방향으로 원하는 온도를 설정하고 다시 열판 또는 열헤드를 클릭하여 온도를 설정할 수 있습니다.

    만약 MCU 온도와 상위 컴퓨터의 온도를 표시하려면 다음 구성이 필요합니다.

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # MCU와 연결 (기본값)

    [temperature_sensor host]        # 상위 컴퓨터 온도
    sensor_type: temperature_host     # 상위 컴퓨터와 연결
    ```


    :::warning 주의사항

    * 주의 최고 온도는 설정 파일에 설정된 값입니다.

    :::
