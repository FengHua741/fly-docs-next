# FLY-UTOR

    FLY-UTOR는 고성능, 다목적의 인터페이스 확장 솔루션으로, 사용자에게 추가적인 USB 인터페이스와 전원 확장 능력을 제공하며 UART to RS232 통신을 지원합니다. 이 확장 보드는 콤팩트한 설계로 쉽게 설치할 수 있으며 다양한 추가 인터페이스 및 전원 출력이 필요한 다양한 응용 분야에 적합합니다.

## 제품 특징

    * 더블 USB 포트 확장: 확장 보드는 두 개의 추가 USB 포트를 제공하며 USB 2.0 규격을 준수하여 다양한 USB 장치와의 호환성을 보장하고 고속 데이터 전송을 가능하게 합니다.
    * UART to RS232 지원: 내장된 UART to RS232 컨버터가 있어 시리얼 장치를 시스템에 연결하여 장치의 통신 능력을 확장할 수 있습니다.
    * 더블 전원 인터페이스 확장: 두 개의 전원 인터페이스를 제공하여 동시에 두 개의 외부 전원을 확장할 수 있게 해줍니다.
    * 안정적인 전원 설계: 확장 보드는 안정적인 전원 회로를 설계하여 연결된 장치의 안정적인 작동을 보장합니다.
    * 쉽게 설치: 표준 크기 설계로 다양한 환경에 적용 가능하며 별도 드라이버 설치가 필요 없습니다.
    * 콤팩트한 사이즈: 확장 보드의 크기가 적절하여 공간을 많이 차지하지 않으며 다양한 프로젝트에 통합하기 편리합니다.

## 제품 사양:

    * 사용 가능한 인터페이스: USB X2, UART to RS232 X2, 전원 인터페이스 X2
    * 전원 인터페이스: 12V/24V 확장
    * 호환성: USB 인터페이스를 가진 상위 기기

## 주의사항:

    * 전원을 끈 후 모든 장치를 끈 상태에서 설치하십시오.
    * 멀티펑션 USB 확장 보드를 메인보드의 USB 포트나 다른 USB 확장 인터페이스에 삽입하십시오.
    * 필요한 USB 장치 및/또는 전원을 확장 보드에 연결하십시오.
    * UART to RS232 기능을 사용하려면 해당 시리얼 장치를 연결하십시오.
    * 전원을 다시 연결하고 장치를 시작하십시오.
    * 시스템은 자동으로 확장 보드를 감지하고 구성해야 합니다. 필요 시 설정 참고서를 참조하십시오.
