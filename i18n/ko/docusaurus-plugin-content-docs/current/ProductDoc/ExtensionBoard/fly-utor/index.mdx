# FLY-UTOR

    FLY-UTOR는 고성능, 다목적의 인터페이스 확장 솔루션으로, 사용자에게 추가 USB 인터페이스와 전원 확장 능력을 제공하며, UART를 RS232로 변환하는 기능도 지원합니다. 이 확장 보드는 컴팩트한 설계로 쉽게 설치할 수 있으며, 다양한 응용 시나리오에서 추가 인터페이스와 전원 출력이 필요한 곳에 적합합니다.

## 제품 특징

    * 듀얼 USB 포트 확장: 확장 보드는 두 개의 추가 USB 포트를 제공하며, USB 2.0 표준을 지원하여 다양한 USB 장치와의 호환성과 빠른 데이터 전송을 보장합니다.
    * UART to RS232 지원: 내장된 UART to RS232 컨버터로 사용자는 시리얼 장치를 시스템에 연결하여 장치의 통신 능력을 확장할 수 있습니다.
    * 듀얼 전원 인터페이스 확장: 두 개의 전원 인터페이스를 제공하여 사용자가 동시에 두 개의 외부 전원을 확장할 수 있도록 합니다.
    * 안정적인 전원 설계: 확장 보드에는 안정적인 전원 회로가 설계되어 있어 연결된 장치의 안정적인 작동을 보장합니다.
    * 쉽게 설치: 표준 크기 설계로 다양한 환경에 적용 가능하며 별도의 드라이버 설치가 필요 없습니다.
    * 컴팩트한 크기: 확장 보드의 크기는 합리적이며 많은 공간을 차지하지 않아 다양한 프로젝트에 통합하기 쉽습니다.

## 제품 사양:

    * 사용 가능한 인터페이스: USB X2, UART to RS232 X2, 전원 인터페이스 X2
    * 전원 인터페이스: 12V/24V 확장
    * 호환성: USB 인터페이스를 가진 상위 컴퓨터

## 주의사항:

    * 전원을 분리하고 모든 장치를 종료한 후 설치하십시오.
    * 다중 기능 USB 확장 보드를 메인보드의 USB 포트 또는 다른 USB 확장 인터페이스에 삽입하십시오.
    * 필요한 USB 장치와/또는 전원을 확장 보드에 연결하십시오.
    * UART to RS232 기능을 사용하려면 적절한 시리얼 장치를 연결하십시오.
    * 전원을 다시 연결하고 장치를 시작하십시오.
    * 시스템은 자동으로 확장 보드를 감지하고 구성해야 합니다. 필요한 경우 설정 후 주의사항을 참조하십시오.
