## 접선

    * 메인보드의 PWM 포트에 연결, 다른 메인보드는 해당 문서를 참고하세요
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-7040/img/wiring.webp').default} size="80%" align="Center" />

## 참고 설정

    ```cfg
    [fan]
    ##    Print Cooling Fan 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## 전원 켜면 자동으로 회전하는 해결 방법

    :::info
    * 여기서 간단히 설명합니다
    * 펌웨어 컴파일 시 PWM 신호 핀을 추가하고 상태를 반전시킵니다
    * 테스트에는 SUPER8PRO를 사용했으며, 핀은 PF8을 사용했습니다. 펌웨어 컴파일 시 `GPIO pins to set at micro-controller startup`에 `!PF8`를 추가하면 됩니다
    * 주의: 입력된 `!`는 영문 느낌표입니다
    * 다른 메인보드도 동일한 방식을 따릅니다
    :::
