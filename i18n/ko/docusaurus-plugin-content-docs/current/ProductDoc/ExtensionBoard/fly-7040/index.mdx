## 제품 특징

    * 최대 회전 속도：45000
    * 최대 유량：14.7CFM
    * 선택 부품：공기 필터, 작동 소음을 줄이고 이물질이 팬에 들어가서 날개를 손상시키는 것을 방지하여 키트의 작동을 더 조용하고 안정적으로 만듭니다
