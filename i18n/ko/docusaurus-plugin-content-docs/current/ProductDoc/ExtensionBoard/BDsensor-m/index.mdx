# BDsensor-m

    BDsensor는 3D 프린터와 함께 사용할 수 있는 거리 센서로, 실시간으로 빠르게 베드 레벨링과 Z축 조정을 할 수 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.webp').default} size="50%" align="left" />

## 기능

    * Z축 리미트 대체
    * Z축을 올렸다 내릴 필요 없이 스캔하여 인쇄 속도 향상
    * XY 이동 스캔, 밀집된 베드 탐지 가능



## 연결

    * 전원: 5V
    * 펌웨어: Klipper/Marlin
    * 연결: 메인보드(BDsensor는 메인보드의 exp1 인터페이스를 사용)
    * 히트베드: 지원: 일반 PEI, 알루미늄 베드 등 금속 표면 히트베드, 지원하지 않음: 자석 구멍 베드 및 비금속 표면 히트베드


## 주의사항

    * KLipper는 klipper와 최신 BDsensor-m 패치를 다운로드해야 하며, 메인보드 펌웨어를 다시 플래시해야 합니다.
    * 디버깅 시 긴급 정지를 언제든지 트리거할 수 있도록 준비해 두십시오. PEI가 긁히는 것을 방지하기 위해
    * 베드 스캔 속도는 200으로 설정하는 것이 좋습니다!
    * Z축 홈 속도는 5를 초과하지 않는 것이 좋습니다.
