# BDsensor-m

    BD센서는 3D 프린터와 함께 사용할 수 있는 거리 센서로, 실시간으로 빠른 베드 레벨링 및 Z축 조정이 가능합니다.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## 기능
    * 설치 소개
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Z축 한계 스위치 대체
    * 스캔 시 Z축을 올리지 않고도 프린팅 속도 향상
    * XY 이동 스캔, 밀도 높은 베드 탐색 가능
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## 연결

    * 전원: 5V
    * 펌웨어: Klipper/Marlin
    * 연결: 메인보드(BD센서는 메인보드의 exp1 인터페이스를 사용합니다)
    * 히트베드: 지원: 일반 PEI, 알루미늄 베드 등 금속 표면 히트베드, 자석 구멍 베드와 비금속 표면 히트베드는 지원하지 않습니다


## 주의사항

    * Klipper는 klipper와 최신 BD-SENSOR-m 패치를 다운로드해야 하며, 메인보드 펌웨어를 다시 플래시해야 합니다
    * 디버깅 시에는 긴급 정지 버튼을 언제든지 트리거할 수 있는 위치에 두어 PEI가 긁히지 않도록 주의하세요
    * 베드 스캔 속도는 200으로 권장합니다!
    * Z축 원위치 속도는 5를 초과하지 않는 것이 좋습니다
