# **BDsensor-m-m** 설치

## 센서 케이블을 메인보드의 EXP1 포트에 연결
    * 센서 케이블의 길이가 충분하지 않으면 포장에 있는 연장 케이블을 사용하세요
    * BDsensor-m-m의 CKL과 SDA 선은 보드의 어떤 GPIO 핀에도 연결할 수 있습니다. 또한 BD 센서 케이블을 Bltouch 포트에 직접 연결할 수도 있습니다. 예를 들어:

    ```bash
    BLtouch    |    BDsensor-m
    5V       -->     5V
    GND      -->     GND
    S        -->     CLK/SCL    (Input)
    GND      -->     GND
    Zmin     -->     SDA    (Input/Output) 
    ```

    * 메인보드 커넥터의 일부 핀은 MCU의 GPIO에 직접 연결되지 않을 수 있습니다 (예: 필터 커패시터가 있거나 MOSFET, 다이오드 또는 옵토커플러를 통해 격리되어 있지만, 저항이나 저항 풀업/풀다운을 통해 격리될 수 있습니다). 따라서 BDsensor-m과 함께 사용할 수 없습니다. 그리고 펌웨어는 연결 오류를 보고할 것입니다. 예를 들어:

    * 팬과 히터의 커넥터는 MOSFET으로 격리됩니다.
    * 일부 보드에서 온도 써미스터와 엔드스탑/프로브의 커넥터는 일반적으로 필터 커패시터를 통해 GND에 연결됩니다.

## Marlin 펌웨어에 패치 설치

    BD 센서는 Marlin2.1.x에 통합되었습니다 (2022.8.27 이후),

    릴리스 버전을 다운로드할 수 있지만, 현재는 최신 버그 수정 버전을 다운로드하는 것이 권장됩니다: https://github.com/MarlinFirmware/Marlin

    설정 파일과 핀 파일을 변경해야 합니다.

### configuration.h 편집

1. BD_SENSOR 활성화

    주석 해제

    ```bash
    #define BD_SENSOR`
    #define Z_SAFE_HOMING
    #define BD_SENSOR_PROBE_NO_STOP //adding this new line for fast bed leveling without nozzle stop, 
    ```

    단, `BD_SENSOR_PROBE_NO_STOP`만

    최신 마린피쉬 버그 수정: https://github.com/MarlinFirmware/Marlin

    설명: https://github.com/MarlinFirmware/Marlin/pull/25847

2. 프로브로 홈 설정

    `Z_MIN_PROBE_USES_Z_MIN_ENDSTOP_PIN`이 비활성화되어 있는지 확인하고 `USE_PROBE_FOR_Z_HOMING`을 다음과 같이 활성화해야 합니다.

    ```bash
    //#define Z_MIN_PROBE_USES_Z_MIN_ENDSTOP_PIN
    // Force the use of the probe for Z-axis homing
    #define USE_PROBE_FOR_Z_HOMING
    ```

3. 두 번째 Z 홈 속도 느리게 하기

    ```bash
    #define Z_PROBE_FEEDRATE_SLOW (Z_PROBE_FEEDRATE_FAST / 16)
    ```

    여기서는 BD센서-m 프로세스에서 읽은 스탑퍼가 일반 스탑퍼만큼 실시간이 아니기 때문에 홈 설정 속도와 Z 홈 설정 속도를 느리게 해야 합니다.

    ### Configuration_adv.h 편집

    `#define BABYSTEPPING`을 활성화하여 실시간 평면화 기능을 구현합니다.

    ```bash
    #define HOMING_BUMP_DIVISOR { 2, 2, 8 }       // Re-Bump Speed Divisor (Divides the Homing Feedrate)
    ```

### pins_boardname.h 편집

    BDsensor-m의 SDA와 SCL 핀을 다음과 같이 추가하여 핀 파일 `pins_boardname.h`에서 설정합니다 (예: `pins_PANDA_PI_V29.h`).

        ```bash
        #define  I2C_BD_SDA_PIN    PC6   // 실제로 SDA 선이 메인보드에 연결된 번호로 변경하세요
        #define  I2C_BD_SCL_PIN    PB2   // 실제로 SLK 선이 메인보드에 연결된 번호로 변경하세요
        #define  I2C_BD_DELAY  20      // 기본값은 20이며, [20,50] 범위 내에서 설정해야 합니다.
        ```

    프린터 시작 전에 자동으로 G29를 사용하여 평면화 프로브를 하고 싶다면, 주석 해제

        ```bash
        #define AUTO_BED_LEVELING_BILINEAR
        ```

    다음 값을 편집합니다.

        ```bash
        #define Z_CLEARANCE_DEPLOY_PROBE   0 // Z Clearance for Deploy/Stow
        #define Z_CLEARANCE_BETWEEN_PROBES  1 // Z Clearance between probe points
        #define Z_CLEARANCE_MULTI_PROBE     1 // Z Clearance between multiple probes
        ```



## LCD 화면에 BD 센서 값 표시

    * 프린터가 상태 표시를 지원하는 경우 (M117 gcode 지원), 예: LCD12864 또는 일부 uart 화면, 예: ender3V2 ...

## 보정

    1. 노즐을 청소한 후 콘솔에서 Z축을 이동하여 노즐이 막대에 닿을 때까지 (BDsensor-m은 이 위치를 0점으로 사용하므로 z_offset이 필요 없고, 값을 0으로 설정합니다).
    2. gcode 명령 `M102 S-6`을 보내면, 프린터는 4mm에 도달할 때까지 Z축을 0.1mm씩 천천히 위로 이동합니다. 센서 설치 전에 M102 S-6을 실행하지 말고, 보정 중에 프린터 전원을 끄지 마세요. 그렇지 않으면 이전 보정 데이터가 삭제됩니다. 이 경우, 다시 보정만 하면 됩니다.

    3. `M102 S-5`를 통해 BD 센서가 성공적으로 보정되었는지 확인할 수 있습니다. 이는 BD 센서에 저장된 원시 보정 데이터를 반환합니다.

    보정 도구도 이것을 수행할 수 있습니다: https://github.com/markniu/Bed_Distance_sensor/raw/new/marlin/BD_Config_Tool_Marlin.zip ![img](https://raw.githubusercontent.com/markniu/Bed_Distance_sensor/main/doc/images/Read.jpg)

    참고: 데이터 값 1015 또는 > 1010은 센서 범위를 벗어남을 의미합니다. 처음 5점 (0~0.5mm) 또는 그 이상의 값이 0에서 1000 사이에 있고 증가 값 델타가 >=10이면 보정이 성공적으로 완료된 것입니다. 위의 그래프처럼 보여야 합니다.

    M102 S-5가 반환한 첫 번째 원시 보정 데이터가 400보다 크면 센서가 너무 높게 설치된 것을 의미하며, 베드에 더 가까운 위치로 다시 설치해야 합니다. 두 번째 데이터가 첫 번째 데이터 값보다 10 이상 크게 되어야 합니다.

## 테스트 및 인쇄

    메뉴 베드 레벨링

    자동 베드 레벨링

## 자동으로 베드를 평면화하는 두 가지 방법이 있습니다:

    **1. M102를 사용하여 실시간으로 앞 몇 층을 평면화**

        gcode 명령을 보내거나 gcode 파일에 추가하여 이 자동 레벨을 쉽게 활성화 또는 비활성화할 수 있습니다.

        Cura에서 베드 레벨링을 활성화하려면 프린터 기계 설정의 "시작 G 코드" 부분에서 G102(모든 축 홈) G 코드 바로 아래에 M28 G 코드를 추가하세요. 예를 들어, G28 아래에 있으면 Z축 높이가 0.2mm 이하일 때만 베드 레벨링이 수행됩니다. `M102 S2`

        BD 센서를 사용하여 베드 레벨링을 비활성화하려면 다음과 같이 보내거나 사용하세요. `M102 S0``G28``M18`

        ```bash
        M102   S-1     // 센서 정보 읽기, 연결 확인을 위해 사용할 수 있습니다.
        M102   S-2     // 현재 거리 값 읽기
        M102   S-5     // 원시 보정 데이터 읽기
        M102   S-6     // 보정 시작, 그 전에 노즐이 막대에 닿는지 확인하고, 프린터를 재시작합니다. 이전에 Z축을 홈 설정하지 마세요.
        M102   S4      // 조정 가능한 Z 높이 값 설정, 예: M102 S4는 Z 높이가 <=0.4mm일 때 조정을 수행하고, M102 S0로 비활성화합니다.
        ```

    **2. G29를 사용하여 자동으로 베드 평면화**

        다른 자동 베드 평면화 방법은 G29처럼 BLtouch를 사용하는 것과 같습니다. G29 아래에 한 줄 G28을 추가하기만 하면 됩니다.

        [설치 비디오](https://www.pandapi3d.com/post/install-bed-distance-sensor-video)

        [크리스의 지하실 설치 비디오](https://youtu.be/VLUfvkO2NFc?si=PF_6nhw39KhHBhcj)

### Z 엔드스탑 확인`M119`

    이 단계를 확인하기 전에 Z를 홈 설정하지 마세요, 그렇지 않으면 노즐이 프린터 베드에 닿을 수 있습니다.

    M119 명령(엔드스탑 상태 보고)을 보낸 후의 반환 메시지는 다음과 같습니다.

    ```bash
    Send: M119
    Recv: x:open y:open z:open
    ```

    z min이 열려 있지 않으면 설정을 확인하세요. `#define Z_MAX_ENDSTOP_HIT_STATE HIGH`

    - Z 모터가 꺼져 있거나 잠금 해제되었는지 확인
    - 손으로 Z축을 아래로 이동하여 노즐이 베드에 닿을 때까지
    - `M102 S-2`를 보내면 반환 값이 0.00mm가 되어야 하고, 다시 M119를 보내면 z 엔드스탑이 트리거된 것을 볼 수 있습니다.

    ```bash
    Send: M119
    Recv: x:open y:open z:TRIGGERED
    ```

    ### 연결 확인

    `M102 S-1`을 통해 연결을 확인합니다. 연결 및 전선 순서가 비어 있거나 다른 문자열이 반환되지 않도록 확인하세요.

    ```bash
    Send: M102 S-1
    Recv: V1.0 pandapi3d.com
    ```
