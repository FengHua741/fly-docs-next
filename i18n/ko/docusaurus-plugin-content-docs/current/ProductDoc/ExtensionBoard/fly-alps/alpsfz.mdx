# 모듈 트리거 값 조정 방법
시리얼 툴을 사용하면 다음 명령어를 사용할 수 있습니다:
Klipper 콘솔에서 "SET_PIN PIN=_probe_ready VALUE=1" 명령을 보내고 트리거 LED를 관찰하며 LED가 꺼지기 전까지 작은 값부터 점차 증가하여 임계값을 조정합니다. 아래 명령어를 입력합니다:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
시리얼 툴에서 아래 명령어를 사용하여 트리거 값을 조정합니다:
```cfg
set 트리거값
```
기본 트리거 값: 20000
* 시리얼 툴 사용 명령어:

* `v`를 전송하여 실시간으로 ADC 값을 출력합니다. a: 필터링 전; b: 필터링 후.
* `uv`를 전송하여 출력을 중지합니다.
* `rt`를 전송하여 현재 임계값을 확인합니다.
