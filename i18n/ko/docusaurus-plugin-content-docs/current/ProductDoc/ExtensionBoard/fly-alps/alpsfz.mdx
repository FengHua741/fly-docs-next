# 모듈 트리거 값 조정 방법

## 레벨링 모듈 열기
    `Klipper` 콘솔에서 명령어 `SET_PIN PIN=_probe_ready VALUE=1`을 보내서 트리거 표시등이 꺼지도록 하여 트리거 임계값을 작게 설정한 후 점차 크게 조정합니다. 아래 명령어를 입력하세요:
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```

## 압력 값 조정
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">시리얼 도구 도우미 가져오기</Button>

    * USB 데이터선으로 모듈과 컴퓨터를 연결하고, 시리얼 도구 도우미를 열어 대응하는 포트 번호를 선택하고, 보드 속도를 115200으로 설정한 후 연결합니다.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    시리얼 도구에 아래 명령어를 입력하여 트리거 값을 조정합니다:
    ```cfg
    set 트리거값
    ```
    기본 트리거 값: 20000
    숫자가 작을수록 민감하고, 숫자가 클수록 덜 민감합니다.
    * 예: `set 20000`, 트리거 값을 20000로 설정합니다. 숫자가 작을수록 더 민감하고, 클수록 덜 민감합니다.
    * 시리얼 도구 명령어 사용:

    * 시리얼 전송 `v`, 실시간으로 abc 값을 출력합니다. a: 필터 전; b: 필터 후.
    * 시리얼 전송 `uv`, 출력을 중지합니다.
    * 시리얼 전송 `rt`, 현재 임계값을 확인합니다.
