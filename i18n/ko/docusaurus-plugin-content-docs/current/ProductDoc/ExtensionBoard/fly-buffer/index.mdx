# 소개

# LLL 버퍼 사용

    LLL 소모품 버퍼는 원격 필드 펌핑을 위한 3D 프린터 장비로, 소모품의 로드 및 언로드 작업을 간소화하기 위해 설계되었습니다. 이 버퍼에는 공급 및 후방 이동을 위한 두 개의 버튼이 장착되어 있어 사용자가 전기적으로 공급 및 후방 이동을 할 수 있습니다.

### 버퍼 기능:

    3D 프린터 엑트루더가 필드를 공급할 때 원격 소모품 랙을 당길 경우 버퍼는 이 동작을 감지하고 열단방향으로 필드를 일정 길이 만큼 내보냅니다. 각각의 당김 동작마다 버퍼는 이 동작을 반복합니다. 이렇게 하면 엑트루더가 필드를 당기는 것을 줄이고 엑트루더의 밀림을 안정화시켜 프린팅 품질의 안정성을 향상시킬 수 있습니다.

### 고장 검출 기능:

    버퍼에는 고장 검출기가 내장되어 있어 버퍼 내에 소모품이 없을 때 3D 프린터에게 인쇄를 중지하라는 신호를 보냅니다. 이 기능을 사용하려면 신호 핀을 연결하고 3D 프린터의 설정을 수정해야 합니다. 반면에 버퍼 기능은 설정을 수정하지 않고 바로 사용할 수 있습니다.

    이러한 기능 설계를 통해 LLL 소모품 버퍼는 소모품 조작을 단순화하고 3D 프린팅 과정의 안정성과 프린팅 품질을 향상시킵니다.

* 주의: 기능을 사용하려면 소모품을 접속한 상태에서 사용해야 합니다. 그렇지 않으면 버튼만으로 모터 작동을 제어할 수 있습니다.

## 설정 참고

    ```cfg
    [필라멘트 스위치 센서 고장 검출]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 고장
    insert_gcode:
            M118 상품
    ```
