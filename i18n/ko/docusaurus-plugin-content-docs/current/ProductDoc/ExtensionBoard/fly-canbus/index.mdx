# CANBUS

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-canbus/img/canbus.webp').default} size="40%" align="left" />


## 제품 특징

    * CAN과 전원을 하나로 통합하여 한번에 네 개의 CAN 도구 보드를 확장할 수 있습니다.
    * RJ11 인터페이스가 내장되어 있어 CAN 프로토콜을 사용하는 시스템과의 연결 및 통신에 적합합니다. RJ11은 CAN 시스템에서 표준 인터페이스는 아니지만, 우리의 제품은 이를 원활하게 연결할 수 있는 가능성을 제공합니다.
    * 인간 중심 설계로, 프린트 부품을 통해 직접 트랙 위에 설치할 수 있습니다.

## 제품 사양

    * 통신 인터페이스: RJ11*1, XH2.54*5
    * 전원 인터페이스: KF128-5.028-2P*5
    * 입력 전압: 5V-24V
    * 호환성: RRF/Klipper/등 CAN

## 주의사항

    :::warning 주의사항

    * 주의하십시오, 전원 입력이 그대로 출력됩니다.
    * 주의하십시오, 선 순서가 잘못되지 않도록 하십시오.
    * 확장 보드를 설치하기 전에, 상위 기기의 전원이 끊겼는지 확인하십시오.
    * 습하거나 먼지가 많은 환경에서는 확장 보드를 사용하지 마십시오.
    * 장비를 설치할 때, 도구 보드의 전압 규격을 유의하십시오, 그렇지 않으면 손상이 발생할 수 있습니다.

    :::
