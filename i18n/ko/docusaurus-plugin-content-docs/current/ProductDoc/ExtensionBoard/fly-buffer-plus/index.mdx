# LLL 플러스 버퍼 사용법

    LLL 소모품 버퍼는 원격 필라멘트 공급을 위한 장치로, 소모품의 로드 및 언로드 작업을 간소화하기 위해 설계되었습니다. 이 버퍼에는 인장과 후퇴를 위한 두 개의 버튼이 장착되어 있으며, 전기식으로 필라멘트를 넣거나 빼는 동작을 수행할 수 있습니다.

### 버퍼 기능:

    3D 프린터의 엑스트루더가 필라멘트를 공급할 때 원격 필라멘트 스탠드를 당기면 버퍼는 열단 방향으로 일정 길이의 필라멘트를 자동으로 공급합니다. 이러한 당김 동작이 발생할 때마다 버퍼는 반복적으로 이와 같은 공급 동작을 수행합니다. 이를 통해 엑스트루더가 필라멘트에 미치는 당김력을 줄이고, 엑스트루더의 힘을 안정화시켜 프린팅 품질의 일관성을 높입니다.

### 필라멘트 부족 감지 기능:

    버퍼 내부에는 필라멘트가 부족한 상황을 감지하는 센서가 내장되어 있으며, 감지 시 3D 프린터는 자동으로 프린팅을 중단합니다. 이 기능을 사용하려면 신호 핀을 연결하고 3D 프린터의 설정을 수정해야 합니다. 반면, 버퍼 기능은 설정 변경 없이 바로 사용할 수 있습니다.

    위의 기능 설계 덕분에 LLL 소모품 버퍼는 소모품 관리 작업을 단순화하고, 3D 프린팅 과정의 안정성과 프린팅 품질을 향상시키는 데 도움을 줍니다.

* 주의: 기능을 사용하기 위해서는 소모품을 연결한 후 사용해야 하며, 그렇지 않으면 단순히 버튼을 눌러 모터 작동을 제어할 수 있습니다.

## 설정 참고

    ```cfg
    [필라멘트 스위치 센서 필라멘트 부족 감지]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 필라멘트 부족
    insert_gcode:
            M118 상단
    ```
