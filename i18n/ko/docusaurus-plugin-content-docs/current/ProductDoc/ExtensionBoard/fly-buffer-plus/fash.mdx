### 펌웨어 업데이트 단계
1. USB 확장기구 연결
    * USB 확장기구를 컴퓨터에 연결합니다.
    * USB 데이터 케이블을 FlyBufferPlus에 연결합니다.
    * 아래 사진 참조하여 연결합니다.
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/usb.webp').default} size="100%" align="center" />
   
2. DFU 모드 진입
    * buffer의 BOOT 버튼을 누릅니다.
    * RESET 버튼을 누릅니다.
    * RESET 버튼을 먼저 떼고, 그 후 BOOT 버튼도 뗍니다.
    * DFU 모드 진입
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/boot.webp').default} size="100%" align="center" />

3. STM32CubeProgrammer 열기
    * STM32CubeProgrammer를 실행합니다.
    * USB 연결 방식 선택
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/stm32cube.webp').default} size="100%" align="center" />   

4. 아래 그림 표시된 곳 클릭
    * 아래 그림 표시된 버튼을 클릭하면 성공적으로 펌웨어를 읽어오면 빨간색 사각형이 나타나며 16진수 값이 표시됩니다.
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/hex.webp').default} size="100%" align="center" />

5. 이제 펌웨어를 플래싱할 수 있습니다
    * 아래 그림 표시된 버튼을 클릭합니다.
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/write.webp').default} size="100%" align="center" />

6. 펌웨어 선택
    * 다운로드한 펌웨어를 선택합니다.
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/select.webp').default} size="100%" align="center" />

7. 체크박스 선택
    * 아래 그림 표시된 곳을 체크하세요.
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/check.webp').default} size="100%" align="center" />

8. 플래싱 시작
    * 아래 그림 표시된 버튼을 클릭합니다.
    * 빨간색 사각형이 나타나면 플래싱이 성공적으로 완료된 것입니다.
    * <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-buffer-plus/img/start.webp').default} size="100%" align="center" />
