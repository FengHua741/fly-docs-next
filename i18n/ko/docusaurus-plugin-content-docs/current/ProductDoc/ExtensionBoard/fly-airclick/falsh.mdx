# 펌웨어 업데이트 및 플래시

    :::danger 주의

    * 출⼚시 이미 펌웨어가 미리 프로그래밍 되어 있습니다.

    :::

## 펌웨어 다운로드

    * 펌웨어 설명
    * `wbtn-vX.X.X-X-XXXXXX-full.bin`은 전체 펌웨어이며, `0x0` 주소에서 플래시해야 하며, 플래시 후 모든 설정이 유지되지 않습니다.
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin`은 증분 펌웨어이며, `0x10000` 주소에서 플래시해야 하며, 펌웨어만 업데이트되고 기존 설정에는 영향을 미치지 않습니다.
    * `wbtn-vX.X.X-X-XXXXXX-www.bin`은 UI 펌웨어이며, `0x290000` 주소에서 플래시해야 하며, 웹 프론트엔드 UI만 업데이트되고 기존 펌웨어 및 설정에는 영향을 미치지 않습니다.
    * <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/airclick/packages">펌웨어 패키지 다운로드</Button>

## 펌웨어 업데이트
    * `AirClick`을 전원 차단 상태에서, `BOOT` 버튼을 통해 `AirClick`의 ESP32 플래시 포트를 컴퓨터에 연결해야 합니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.webp').default} size="80%" align="Center" />

    1. USB 플래시 도구를 다운로드하고 압축 해제합니다. <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">클릭하여 다운로드</Button>
    2. ``flash_download_tool_3.9.4.exe``를 엽니다. 주의: 플래시 도구를 열면 먼저 CMD 창이 나타나고 그 다음에 플래시 도구 화면이 나타납니다. CMD 창을 꼭 닫지 마세요.
    3. 플래시 도구에서 ``ChipType:``을 ``ESP32-C3``로 선택합니다.
    4. 플래시 도구에서 ``WorkMode:``를 ``Develop``로 선택합니다.
    5. 플래시 도구에서 ``LoadMode:``를 ``USB``로 선택합니다.
    6. 마지막으로 ``OK``를 클릭할 수 있습니다. 아래 그림 참조
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.webp').default} size="50%" align="Center" />
    7. 클릭하면 아래 그림이 나타납니다.
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.webp').default} size="50%" align="Center" />

    :::tip

    * 위 그림의 초록색 선택 영역은 그림과 일치해야 하며, 임의로 수정하지 마세요.

    :::

    8. AirClick을 USB-TypeC 데이터 케이블을 통해 컴퓨터에 연결합니다.
    9. 위 그림의 파란색 영역 1 버튼을 클릭하여 플래시할 펌웨어를 선택합니다.
    10. 위 그림의 파란색 영역 2를 클릭하여 플래시 주소를 입력합니다.
    11. 위 그림의 파란색 영역 3을 클릭하여 AirClick의 시리얼 포트를 선택합니다 (포트가 없는 경우, AirClick의 플래시 버튼을 누른 상태에서 컴퓨터에 연결할 수 있습니다).
    12. 위 그림의 파란색 영역 4에서 보드의 속도를 최대 `921600`으로 선택합니다.
    13. 위 그림의 파란색 영역 5를 클릭하여 플래시를 시작합니다. 플래시가 완료되면 AirClick의 펌웨어 업데이트가 완료됩니다 (연결 데이터 케이블을 끊기 전에 플래시 완료 메시지를 반드시 확인하세요).
