# 유선 연결 Klipper 구성

:::danger 주의

* STM32F072에 펌웨어를 업데이트하지 않고 Klipper에 연결하면 유선 연결 작업을 지원하지 않습니다.

:::

## 빌드 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` 키를 누르면 **Save configuration**이 나타나고, 그 다음에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 구성이 저장되고 명령줄 인터페이스로 돌아왔습니다.
    * `make -j4`를 입력하여 빌드를 시작합니다. 시간이 꽤 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래 내용이 출력되면 빌드가 성공했습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 플래시 모드로 진입

    * AirClick이 전원이 끊긴 상태에서 STM32의 BOOT 버튼을 누른 채 Type-C 케이블로 STM32의 플래시 포트를 호스트 시스템과 연결합니다.
