# AirClick 무선 버튼
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/air.webp').default} size="100%" align="Center" />



## 제품 특징

    * 높은 호환성: Klipper, RRF, BambuLab을 지원하며, 기계에 직접 연결하여 미리 설정된 명령을 보내어 원활한 호환성과 사용 경험을 제공합니다.
    * 사용자 정의 매크로 명령: 사용자는 자신의 요구에 따라 명령을 사용자 정의할 수 있어 기계 작동이 더욱 유연하고 편리해집니다.
    * 무선 Klipper, RRF, BambuLab에 연결 지원. 유선 연결은 Klipper만 지원합니다.
    * 무선 연결: ESP32 설계를 통해 Klipper에 사용자 정의 명령 또는 매크로를 직접 전송하여 사용자에게 매우 높은 유연성과 확장성을 제공합니다.
    * 유선 연결: STM32F072 칩을 통해 Klipper와의 네트워크 연결이 불가능한 경우에도 간단한 작업을 수행할 수 있습니다. 총 10개의 사용자 정의 명령 또는 매크로를 지원합니다.
    * 사용하기 쉬움: 직관적인 사용자 인터페이스를 갖추고 있어 사용자가 쉽게 무선 버튼을 추가하고 관리할 수 있으며, 이는 사용의 편의성을 크게 높입니다.

## 제품 사양

    * 주요 제어 칩: ESP32
    * 추가 칩: STM32F072
    * ESP32-WIFI를 통해 명령을 보내기: 제한 없음
    * STM32F072-IO를 통해 명령 보내기: 10개
    * 호환성: Klipper, RRF, BambuLab을 지원
    * 전원: 5V

## 주의사항

    :::danger 주의사항

    * 이 제품은 `type c` 또는 `PH2.0-2P` 단자를 통해 5V 전력을 공급받습니다.
    * 이 제품의 내장 안테나는 버튼 신호를 수신하는 용도로만 사용되며, Wi-Fi에 연결하려면 안테나를 설치해야 합니다. 안테나 단자는 `ipex3`입니다.
    * 안테나를 설치하세요!!! 그렇지 않으면 신호가 너무 약해서 심각한 발열이 발생할 수 있습니다!!!
    * 긴 시간 동안 버튼을 누르면 시스템이 자동으로 꺼지므로, 오작동을 방지하기 위해 15초 이상 길게 누르지 마세요.

    :::
