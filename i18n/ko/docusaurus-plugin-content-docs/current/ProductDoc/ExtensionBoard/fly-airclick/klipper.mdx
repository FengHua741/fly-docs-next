# Klipper에 연결

    :::info 중요 팁
    * AirClick은 대상 프린터와 동일한 네트워크에 연결되어야 합니다.
    :::

1. 왼쪽 시스템 설정을 클릭합니다.
    
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. `Printer Type`을 `Klipper`로 선택합니다.
3. `Moonraker IP`에 Klipper 기기의 IP 주소를 입력합니다.
4. `Moonraker Port`에 Klipper 상위 기기의 포트를 입력합니다. 기본값은 `7125`이며, 다른 포트를 사용하는 경우 해당 포트를 입력해야 합니다.
5. `Apikey`는 무시하십시오. 현재 Apikey는 지원되지 않습니다.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. 위의 설정이 완료되면 오른쪽 하단의 `제출`을 클릭하여 설정을 저장합니다.
7. 정상적으로 연결된 후 웹 페이지 상단에 메시지가 표시됩니다.

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. 연결 실패 시
    - IP 주소와 포트가 올바른지 확인하십시오.
    - 프린터와 AirClick이 동일한 네트워크에 연결되어 있는지 확인하십시오.
    - Klipper 상위 기기가 정상 작동 중인지 확인하십시오.
    - 브라우저에서 Klipper 상위 기기의 웹 페이지를 열 수 있는지 확인하십시오.
