# 모듈 트리거 값을 조정하는 방법
시리얼 도구를 사용하면 다음 명령을 사용할 수 있습니다:
Klipper 콘솔에 "SET_PIN PIN=_probe_ready VALUE=1" 입력, 트리거 LED를 관찰하면서 임계값을 작게 설정한 후 점차 늘려 LED가 꺼질 때까지 조정합니다. 아래 명령을 입력합니다:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
시리얼 도구에서 다음 명령을 사용하여 트리거 값을 조정합니다:
```cfg
set 트리거값
```
* 시리얼 도구 사용 명령:

* 시리얼 전송 `v`, adc 값 실시간 출력. a: 필터링 전; b: 필터링 후.
* 시리얼 전송 `uv`, 출력 비활성화.
* 시리얼 전송 `rt`, 현재 임계값 확인.
