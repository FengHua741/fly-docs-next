# 모듈 트리거 값 조정 방법
시리얼 툴을 사용하면 다음 명령을 사용할 수 있습니다:
Klipper 콘솔에서 "SET_PIN PIN=_probe_ready VALUE=1" 명령을 보내고 트리거 LED를 관찰하면서 트리거 LED가 꺼지기 전까지 작은 값에서 큰 값으로 임계값을 조정합니다. 아래 명령을 입력하십시오:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
시리얼 툴에서 아래 명령을 사용하여 트리거 값을 조정합니다:
```cfg
set 트리거값
```
기본 트리거 값: 20000
* 시리얼 툴 사용 명령:

* 시리얼로 `v`를 보내면 실시간으로 adc 값을 출력합니다. a: 필터링 전; b: 필터링 후.
* 시리얼로 `uv`를 보내면 출력을 종료합니다.
* 시리얼로 `rt`를 보내면 현재 임계값을 확인합니다.
