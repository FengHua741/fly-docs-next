# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />



## 제품 특징

    * 36 모터 등에 적합합니다.
    * 라즈베리 파이, FLY PI, 오렌지 파이, 메인보드 등의 SPI 인터페이스에 연결할 수 있습니다.

## 제품 사양

    * 보드 장착 인터페이스: ZH1.5 6P 플러그
    * 통신 방식: SPI
    * 작동 전압: 5V

## 주의사항

    :::warning 주의사항

    * 작동 전압은 5V입니다. 틀리게 연결하지 마십시오.
    * 다양한 펌웨어에 적합합니다. SPI 연결을 사용할 수 있는지 확인하십시오.

    :::
