# Marlin 설정

## 필라멘트 끊김 및 막힘 기능 설정

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/marlin.webp').default} size="70%" align="center" />

1. Marlin 설정 파일 `Configuration.h`을 열고, `#define FILAMENT_RUNOUT_SENSOR`를 찾아 주석 기호를 제거합니다.
2. `#define FILAMENT_RUNOUT_DISTANCE_MM`를 찾아 주석 기호를 제거하고 값을 `3.5`로 설정합니다 (사용 중인 코드 디스크에 따라 변경 가능하며, 오작동이 발생할 경우 이 값을 적절히 증가시키세요).
3. `#define FILAMENT_MOTION_SENSOR`를 찾아 주석 기호를 제거합니다.
4. Marlin의 필라멘트 센서 기능은 `#define NOZZLE_PARK_FEATURE`와 `#define ADVANCED_PAUSE_FEATURE`도 활성화해야 합니다. 주석을 해제하고 프린터에 맞는 파라미터를 설정하세요.
