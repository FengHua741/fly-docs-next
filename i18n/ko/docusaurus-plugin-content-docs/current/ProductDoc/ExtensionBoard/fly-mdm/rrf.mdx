# RRF 구성

## 필러 감지 및 블록 감지 기능 구성

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} size="70%" align="center" />

1. RRF 프린터 웹 페이지 인터페이스를 열고 `Files` -> `System`으로 이동하여 `config.g` 파일을 찾은 다음 열어 편집합니다.
2. 파일에 다음 코드를 추가하여 필러 감지 기능을 활성화합니다.

```gcode
M591 D0 P2 C"PA9" S1
```

3. 파일에 다음 코드를 추가하여 블록 감지 기능을 활성화합니다. (`L` 매개변수는 사용 중인 엔코더에 따라 변경되며, 잘못된 트리거가 발생하는 경우 이 값을 적절히 증가시켜야 합니다)

```gcode
M591 D0 P7 C"PA10" L3.5 R75:125 S1
```
