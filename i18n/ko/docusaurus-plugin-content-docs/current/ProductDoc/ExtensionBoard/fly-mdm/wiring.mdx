# 배선도

:::danger 주의사항
    - 입력 전원은 DC5V만 지원하며, 배선 시 정극과 음극을 구분해야 합니다. 배선 과정은 반드시 전원이 차단된 상태에서 진행해야 하며, 배선이 정확한지 확인 후 전원을 공급해야 합니다. 배선 오류로 인한 손실에 대해서는 당사가 책임을 지지 않습니다.
:::
