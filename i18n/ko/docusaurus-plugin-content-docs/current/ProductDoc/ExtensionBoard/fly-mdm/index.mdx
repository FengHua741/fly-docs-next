# 제품 소개

  - MDM斷堵料センサーは、小型で使いやすく、トリガ感度が高いです。
  - Klipper、Marlin、RRFなどの3Dプリンタに対応しています。
  - 断料検出：高速赤外線スキャンにより、0.25秒でフィラメントの断裂をキャッチし、即座に印刷を停止してアラームを発生させ、無駄な消費を防ぎます；
  - 堵料警告：エンコーダー光信号認識により、ノズルや喉管が詰まる兆候を検知し、廃棄品率を80％以上削減します。
  - 全ての線材タイプに対応：PLA、ABS、PETG、TPUなどの全タイプのフィラメント（直径1.75mm）に対応しており、剛性材料でも柔軟な素材でも安定した識別が可能です。
  - 簡単な設置：設置図、配線図、設定ファイルなどの資料を提供し、簡単に設置できます。
