### 모델 파일 설명

| 모델 파일 이름 | 설명 |
| :---: | :--- |
| pcb_cover.stp | PCB 커버 |
| switch_sensor_trigger_block.stp | 필라멘트 감지 트리거 블록 |
| rear_cover.stp | 후방 커버 |
| upper_cover.stp | 상단 커버 |
| main_body.stp | 모델 본체 |
| encoder_3_5.8mm.stp | 엔코더 디스크, 3파장, 최소 탐지 거리 5.8mm <br/> `고유량(20 mm³/s 초과) 프린팅 시 사용 가능` |
| encoder_5_3.5mm.stp | 엔코더 디스크, 5파장, 최소 탐지 거리 3.5mm <br/> `적당함, 권장 사용` |
| encoder_10_1.73mm.stp | 엔코더 디스크, 10파장, 최소 탐지 거리 1.73mm <br/> `저속 고정밀 프린팅 시 사용 가능` <br/> `3D 프린터 펌웨어의 소모재 탐지 샘플링 속도 제한에 영향 받음` |


## 프린팅 및 조립

:::warning 모델 프린팅 주의사항

- 케이스는 `PETG` 및 `투명 재료`로 프린팅하지 않는 것을 권장합니다. 센서 감응에 영향을 미칠 수 있습니다.

:::
