# USB 펌웨어 컴파일

    :::tip

    * ERCF 출하 시 CAN 연결을 위한 Katapult 펌웨어가 미리 설치되어 있습니다.
    * USB를 사용하여 펌웨어를 프로그래밍하려면 USB용 Katapult 펌웨어를 프로그래밍해야 합니다.
    * 한 번 프로그래밍하면 다시 필요하지 않습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapult 펌웨어 프로그래밍</Button>

    :::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * 마이크로 컨트롤러 시작 시 설정할 GPIO 핀을 선택합니다. 입력: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고, 그 후에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되고 명령줄 인터페이스로 돌아왔습니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 좀 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래의 내용이 출력되면 컴파일이 성공했습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
