    <TabItem value="can" label="CANBus UUID 가져오기">

    * ID를 검색하기 전에 XT30 데이터 케이블을 도구 보드를 UTOC 또는 다리 펌웨어가 설치된 메인보드에 연결합니다. 아래는 참조 배선도입니다.
    * 다이얼 스위치가 올바른 위치에 있는지 확인하십시오.
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * CAN H와 CAN L가 전원이 꺼진 상태에서 멀티미터로 저항 값을 측정했을 때 약 `60Ω` 정도인지 확인하십시오. 만약 `120Ω`라면 `120Ω` 점프를 연결해야 합니다. 만약 약 `40Ω`라면 점프를 제거해야 합니다.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
