# 제품 소개

    FLY-ERCF는 RP2040 기반의 3D 프린터 다색 제어 보드로, Enraged Rabbit 프로젝트에 주로 사용됩니다. 또한 다른 두 축 스테퍼 모터 제어 상황에서도 사용할 수 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.webp').default} size="100%" align="left" />

## 제품 특징

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * CAN 모드, RS232 모드 지원
    * 두 개의 드라이버 포지션, UART 및 SPI 드라이버 지원
    * 원래 ERCF EASY 보드와 호환
    * 공간 최적화, 4층 보드 설계
    * 14핀 확장 포트를 XHD2.54로 교체, 접속을 더 견고하게 하여 선이 끊어지는 것을 방지
    * 12-24V 전압 지원
    * CAN 버스 및 RS232 연결 지원, 데이터 전송이 더 안정적이며 지연 시간이 적고 연결이 더 안정적
    * CAN 인터페이스는 XT30(2+2)를 사용하여 15A 전류를 지원

## 제품 사양

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 펌웨어: Klipper CAN/RS232
    * 입력 전압: DC12V-24V
    * 인터페이스: XHD2.54 듀얼 핀 7핀 확장 포트, SERVO X1, ENDSTOP X2, EXTRA X1
    * 드라이버: UART/SPI 및 STEP/DIR 지원
    * 12-24V 전압 지원
    * CAN 버스 및 RS232 지원, 데이터 전송이 더 안정적이며 지연 시간이 적고 연결이 더 안정적
    * CAN 인터페이스는 XT30(2+2)를 사용하여 15A 전류를 지원
