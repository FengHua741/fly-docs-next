    <CAN />

    * 마이크로컨트롤러 시작 시 설정할 GPIO 핀 선택, 입력: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` 키 누르면 **Save configuration** 메시지가 나타나고 그 후 `Y` 키 누름
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었고 명령줄 화면으로 돌아왔습니다
    * 아래 명령어를 입력하여 빌드를 시작합니다. 시간이 다소 걸립니다
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 다음 내용이 출력되면 빌드 성공입니다

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```

## 펌웨어 플래싱

    * 펌웨어 플래싱 전에 XT30 데이터 케이블을 도구 보드를 UTOC 또는 브리지 펌웨어가 설치된 메인보드에 연결합니다. 아래는 참조 배선도입니다
    * 스위치 위치가 올바른지 확인하십시오
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * CAN H와 CAN L가 전원이 꺼진 상태에서 multimeter로 저항값을 측정했을 때 약 `60Ω` 정도인지 확인하십시오. 만약 `120Ω`라면 `120Ω` 점퍼를 연결해야 하고, 약 `40Ω`라면 점퍼를 제거해야 합니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
