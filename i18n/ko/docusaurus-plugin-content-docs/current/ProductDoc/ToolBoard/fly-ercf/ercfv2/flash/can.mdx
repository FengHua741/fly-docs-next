## 펌웨어 플래시

    * 펌웨어 플래시 전 XT30 데이터 케이블을 도구 보드를 UTOC 또는 브리지 펌웨어가 설치된 메인보드에 연결합니다. 아래는 참조 배선도입니다
    * 스위치 패들 위치를 확인해 주세요
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * CAN H와 CAN L의 저항값이 `60Ω` 정도인지 확인하세요. 전원이 꺼진 상태에서 만다테스를 사용하여 측정합니다. 만약 `120Ω`라면 `120Ω` 점퍼를 연결해야 합니다. 만약 `40Ω` 정도라면 점퍼를 제거해야 합니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
