# 펌웨어 플래시 설명

    * ERCF는 CAN 연결 방식을 통해 상위 기기와 연결될 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블의 모양입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드가 이미 Katapult로 플래시되었다면 마지막의 Katapult 펌웨어 플래시는 불필요합니다. 이는 메인보드의 Katapult 펌웨어 플래시 실패를 예방하기 위한 것입니다.
* 재설정 버튼을 빠르게 두 번 누른 후 메인보드의 LED가 깜빡이는지 확인할 수 있습니다. 만약 깜빡인다면, 메인보드가 Katapult 펌웨어로 업데이트된 것입니다.
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하는지

    * ERCF는 다음 방법으로 상위 기기에 연결할 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CAN 펌웨어 플래시</Button>
        * 장점: `Katapult` 펌웨어를 설치한 경우, 메인보드를 만지지 않고 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 홈 위치 설정 시간 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하고, UTOC 또는 다리 펌웨어가 설치된 메인보드와 함께 사용해야 합니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB 펌웨어 플래시</Button>
        * 장점: `Katapult` 펌웨어를 설치한 경우, 메인보드를 만지지 않고 펌웨어를 업데이트할 수 있습니다. 메모리 카드나 DFU로 플래시된 경우 여전히 기존 방식으로 펌웨어를 업데이트해야 하지만, 통신이 안정적이고 신뢰할 수 있습니다.
        * 단점: 여러 USB 장치가 있는 경우 데이터 전송에 영향을 줄 수 있습니다.
