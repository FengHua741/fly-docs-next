# 제품 소개

    FLY SHT36/42은 광저우 매륜 전자 과학 기술 유한 회사가 3D 프린터 노즐에 대해 설계한 노즐 도구 플레이트입니다. 이 도구 플레이트를 사용하면 복잡한 연결선을 대체하여 주보드와 프린팅 헤드 사이의 배선을 최적화할 수 있습니다. SHT36은 36 스테퍼 모터에 적합하고 SHT42은 42 스테퍼 모터에 적합하며 CAN 수신기, USB 포트, TMC2209 드라이버, ADXL345 가속도 센서 및 다른 기능을 통합한 도구 플레이트입니다. 이 도구 플레이트는 Klipper 펌웨어에 적합합니다.

    * 크기를 극대화하여 6층 보드 설계를 채택했습니다.
    * CAN 버스 연결을 지원하여 데이터 전송이 더 안정적이며 지연 시간이 줄어들고 연결이 더 안정적입니다.
    * TMC2209 드라이버가 내장되어 있으며 UART 모드로 작동하여 엑스트루더에 사용될 수 있습니다.
    * ADXL345 가속도 센서가 내장되어 있습니다.
    * PT100 (MAX31865)를 지원합니다.
    * 교체 가능한 팬 MOS 설계를 채택했습니다.
    * BLTOUCH, PL08N 등의 평탄화 센서를 지원하며 XY 리미트 스위치가 통합되어 있습니다.
    * 12-24V 전압 입력을 지원합니다.
    * MCU: STM32F072
    * 통신: CANBUS, USB
    * 인터페이스: 제어 가능한 팬\*2, RGB\*1, 리미트 스위치\*3, 프로브\*1, 서보\*1, 히터 핀\*1, 열 감지기\*1, PT100\*1

## 제품 사양

    1. 외형 크기: 51.27mm*36.26mm (SHT36), 42mm*42mm (SHT42)
    2. 설치 크기: 구멍 간격 43.84mm, M3 설치 구멍 직경
    3. 프로세서: STM32F072C8T6
    4. 입력 전압: DC12-24V 5A
    5. 히터 핀 최대 출력 전류: 5A
    6. 팬 인터페이스 최대 출력 전류: 1A
