# 펌웨어 쓰기 안내

    * SHT36 LIS3DH는 CAN 연결 방식만 사용하여 상위 기기와 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 선 스타일입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 쓰기는 Katapult를 통해 Klipper를 쓸 수 있습니다.
* 메인보드가 이미 Katapult로 쓰였다면 마지막에 Katapult 펌웨어를 쓸 필요가 없습니다. 그건 메인보드의 Katapult 펌웨어가 성공적으로 쓰이지 않은 경우에 대한 예방 조치입니다.
* 빠르게 두 번 리셋 후, 메인보드에서 LED가 깜빡이는지 확인할 수 있습니다. 만약 그렇다면, 메인보드가 이미 Katapult 펌웨어로 업데이트되었음을 나타냅니다.
:::

## 어느 종류의 펌웨어가 필요한지 어떻게 판단하는지

    * SHT36 LIS3DH는 다음과 같은 방법으로 상위 기기에 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">CAN 펌웨어 쓰기</Button>
        * 장점: `Katapult` 펌웨어를 쓴 경우, 메인보드에 손을 대지 않고 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 때, 홈 위치 설정 시간 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하는지 확인하고, UTOC 또는 다리 펌웨어를 쓴 메인보드와 함께 사용해야 합니다.
