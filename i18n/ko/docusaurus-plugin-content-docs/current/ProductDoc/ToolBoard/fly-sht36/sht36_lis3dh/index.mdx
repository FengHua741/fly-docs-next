# 제품 소개

    FLY-SHT36-LIS3DH는 광저우 맘런 전자 기술 유한 회사가 3D 프린터 엑스트루더 헤드를 위해 설계한 노즐 도구 보드입니다. 이 도구 보드를 사용하면 기존의 메인보드와 프린팅 헤드 간의 복잡한 배선을 4개의 선으로 대체하여 레이아웃을 최적화할 수 있습니다. FLY-SHT36-LIS3DH는 36 스텝 모터에 적합하며, CAN 트랜스시버, USB 포트, TMC2209 드라이버, LIS3DH 가속도 센서, 두 개의 제어 가능한 팬 등 다양한 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 펌웨어에 적합합니다.

    * 체적을 극대화하여 6층 보드 디자인을 채택했습니다.
    * CAN 버스 연결을 지원하여 데이터 전송이 더 안정적이며 지연 시간이 줄어들고 연결이 더욱 안정적입니다.
    * 보드 내장 TMC2209 드라이버를 UART 모드로 사용하여 엑스트루더에 사용할 수 있습니다.
    * 보드 내장 ADXL345 가속도 센서
    * PT100 (MAX31865)를 지원합니다.
    * 교체 가능한 팬 MOS 설계
    * BLTOUCH, PL08N 등의 평평 센서를 지원하며 XY 리미트 스위치를 통합합니다.
    * 12-24V 전압 입력을 지원합니다.
    * MCU: STM32F072
    * 통신: CANBUS, USB
