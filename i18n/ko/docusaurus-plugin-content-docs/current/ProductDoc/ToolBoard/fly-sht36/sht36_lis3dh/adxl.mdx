# 가속계 사용


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Klipper 업데이트

    * Klipper는 2024년 11월 13일에 LIS3DH 가속계를 지원합니다. LIS3DH 가속계를 사용하려면 Klipper를 업데이트해야 합니다.
    * 네트워크를 통해 상위 컴퓨터에 연결한 후 아래 코드를 입력합니다(네트워크 문제로 인해 업데이트가 실패할 수 있습니다).
        ```bash
        cd ~/klipper 
        git pull
        ```

## 구성 추가


    ```cfg
    #####################################################################
    #                       LIS3DH 가속계 구성 (필요할 때 활성화)
    #####################################################################
    [lis3dh]
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis3dh
    probe_points:
        125, 125, 20  # 여기서 설정은 공명 측정을 수행할 때 노즐이 위치하는 좌표입니다.


```
