# 펌웨어 플래시 설명

    * SHT36 PRO는 CAN 연결 방식으로 상위 기기와 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블의 모양입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드가 이미 Katapult 펌웨어로 플래시되었다면 마지막에 Katapult 펌웨어를 플래시할 필요가 없습니다. 이것은 메인보드의 Katapult 펌웨어 플래시 실패를 예방하기 위한 것입니다.
* 빠르게 두 번 리셋한 후 메인보드에 LED가 깜빡이는지 확인할 수 있습니다. 이것이 있으면 메인보드가 Katapult 펌웨어로 업데이트되었음을 나타냅니다.
:::

## CAN 펌웨어 컴파일

    * SHT36 PRO는 오직 CAN 연결 방식으로만 상위 기기를 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">CAN 펌웨어 플래시</Button>
        * 장점: `Katapult` 펌웨어가 플래시되면 메인보드에 손을 대지 않고 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우 재homes 위치 시간 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하는지 확인하고, UTOC 또는 다리 펌웨어가 플래시된 메인보드가 필요합니다.
