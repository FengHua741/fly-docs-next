    * **GPIO 핀 설정 (micro-controller 시작 시)** 선택, 입력: !PC13 (대문자로 입력)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/103.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고, 이때 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었고 명령줄 인터페이스로 돌아왔습니다.
    * 아래 명령어를 입력하여 컴파일을 시작합니다. 시간이 다소 걸릴 수 있습니다.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 다음 내용이 출력되면 컴파일 성공입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 펌웨어 플래시

    * 펌웨어를 플래시하기 전에 XT30 데이터 케이블을 도구 보드를 UTOC 또는 브리지 펌웨어가 설치된 메인보드에 연결합니다. 아래는 참조 연결도입니다.
    * 도스 witch가 올바른 위치에 있는지 확인하십시오.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * CAN H와 CAN L의 저항값이 전원이 꺼진 상태에서 만다기로 측정되었는지 확인하십시오. 약 60Ω 정도여야 합니다. 만약 120Ω이면 120Ω 점퍼를 연결하고, 약 40Ω이면 점퍼를 제거해야 합니다.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.webp').default} size="50%" align="center" />
