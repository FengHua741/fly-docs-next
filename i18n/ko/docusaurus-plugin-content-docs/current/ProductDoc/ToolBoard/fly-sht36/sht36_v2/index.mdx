# 소개

## 제품 소개

    FLY SHT36 V2는 광저우 멍런 전자 기술 유한 회사가 3D 프린터 노즐에 대해 설계한 노즐 도구 플레이트입니다. 이 도구 플레이트를 사용하면 기존의 메인보드와 프린팅 헤드 사이의 복잡한 배선을 4개의 선으로 대체하여 배치를 최적화할 수 있습니다. SHT36 V2는 36 스텝 모터에 적합하며, CAN 트랜시버, USB 포트, TMC2209 드라이버, ADXL345 가속도 센서, 두 개의 제어 가능한 팬 등 다양한 기능을 통합한 도구 플레이트입니다. 이 도구 플레이트는 Klipper 펌웨어에 적합합니다.

    * 크기 최소화, 6층 보드 설계
    * CAN 버스 연결 지원, 데이터 전송 안정성 향상, 지연 시간 감소, 연결 안정성 향상
    * TMC2209 드라이버 내장, UART 모드 사용 가능, 엑스트루더에 사용 가능
    * ADXL345 가속도 센서 내장
    * PT100 (MAX31865) 지원
    * 교체 가능한 팬 MOS 설계
    * BLTOUCH, PL08N 등의 평탄화 센서 지원, XY 리미트 스위치 통합
    * 12-24V 전압 입력 지원
    * MCU: STM32F072
    * 통신: CAN
    * BLTOUCH, PL08N 등의 평탄화 센서 지원, XY 리미트 스위치 통합

## 신규 기능

    * 4.7K 저항 (NTC 열감지기용) 또는 1.1K 저항 (PT1000용) 전환 지원 추가
    * CAN 인터페이스는 더 이상 MX3.0 단자를 사용하지 않으며, 새로운 커넥터는 15A 전류를 지원하고 피크 전류는 30A까지 지원합니다.
    * 5/12/24V 팬 전압 선택 가능
    * 보드 내 NT100k 저항 추가, 창 온도 측정 용
    * 전원 역접속 보호 기능 추가
    * ADXL345의 INT1을 MCU에 연결, 일부 사용자를 위한 평탄화에 사용
    * 히팅 바늘 단자 확대, 정격 전류 10A
    * 고전압 입력 리미트 추가
    * 주요 제어기로 국산 MCU로 교체
    * 자석 엔코더 (as5047d) 지원, Klipper에서 운동 분석용 (현재 Klipper는 아직 지원하지 않음)
    * 1m CAN 연결선 추가, 신호 손실 문제를 해결하기 위해
    * 설치 크기: 구멍 간격 43.84mm, M3 설치 구멍 직경
    * 프로세서: GD32F103 (2022-10-28일 이전 구매), APM32F072 (2022-10-28일 이후 구매)
