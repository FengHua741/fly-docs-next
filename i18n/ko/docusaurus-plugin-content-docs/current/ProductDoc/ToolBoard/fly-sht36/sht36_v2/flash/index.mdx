    :::


# 펌웨어 플래시 설명

    * 펌웨어 빌드 및 펌웨어 플래시에는 상위 기계가 필요합니다.
    * SHT36 V2는 CAN 연결 방식으로 상위 기계와 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블의 모양입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드가 이미 Katapult 펌웨어로 프로그램되었는지 확인한 후 마지막 단계에서 Katapult 펌웨어를 플래시하지 않아도 됩니다. 이는 메인보드의 Katapult 펌웨어 플래시가 성공적으로 이루어지지 않은 경우에만 필요합니다.
* 재설정 버튼을 두 번 빠르게 누른 후 메인보드에 LED가 깜빡이는지 확인할 수 있으며, 깜빡이면 메인보드가 Katapult 펌웨어로 프로그램된 것입니다.
:::

## CAN 펌웨어 빌드

    * SHT36 V2는 CAN 연결 방식만 사용하여 상위 기계와 연결할 수 있습니다.
    * STM 103의 경우 <Button variant="contained" disableElevation href="./can103">103CAN 펌웨어 빌드</Button>를 참조하세요.
    * APM 072와 STM 072의 경우 <Button variant="contained" disableElevation href="./can072">072CAN 펌웨어 빌드</Button>를 참조하세요.
