## 컴파일 시작

    <SSH />

    <RS232 />

    * **micro-controller 시작 시 설정할 GPIO 핀 선택**: 입력: **!gpio5**
    :::warning
        `!gpio5`를 입력하기 전에 반드시 영문 입력 모드로 변경해야 합니다.
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **구성 저장**이 나타나므로 그 후에 `Y` 키를 누릅니다
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 이제 구성이 저장되었고 명령줄 화면으로 돌아왔습니다
    * 아래 명령어를 입력하여 컴파일을 시작합니다 (시간이 다소 소요됩니다)
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 아래 내용이 출력되면 컴파일 성공입니다

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## 플래시 모드 진입

    :::tip

    화살표가 가리키는 부분이 **BOOT 버튼**입니다

    :::

    * 도구 보드가 **완전히 전원이 끊긴 상태**에서 **BOOT 버튼**을 누른 상태에서 데이터 전송 기능이 있는 TypeC 데이터 케이블로 **도구 보드**를 **상위 컴퓨터의 USB 단자**에 연결한 후, 정상적으로 연결된 후에 **BOOT 버튼**을 놓습니다
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="80%" align="center" />
