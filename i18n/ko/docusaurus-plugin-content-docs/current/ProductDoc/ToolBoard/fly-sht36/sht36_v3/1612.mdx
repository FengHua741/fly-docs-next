# Eddy 사용

    :::info

    * 스캔베드를 사용할 때는 독립된 Z 리미터를 사용하는 것이 좋습니다. 스캔베드를 Z 리미터로 사용하지 않는 것이 좋습니다.
    * Eddy를 설치할 때, 노즐 위치의 바닥 부분은 노즐 위로 최소 1-2 mm 있어야 합니다.
    * Eddy 주파수 보정 시 코일 PCB는 열판에서 최소 5mm 이상 떨어져 있어야 합니다.
    * 열판을 테스트할 때는 항상 상온에서 테스트하고, 테스트 중에 열판을 켜지 마세요.

    :::

## 참조 설정 및 사용 설명서

    ### ldc16121612 기본 주파수 수정
    이 방법은 sht36-v3/SB2040-v3 또는 SB2040-Pro-v3에만 사용됩니다.
    Mobaxterm 도구를 사용하여 명령을 실행할 수 있습니다.

    필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxterm 도구</Button>를 다운로드하세요.

    [자세한 사용 방법은 여기를 참조하세요](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    첫 번째 명령을 실행하고 엔터를 누르세요.

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    두 번째 명령을 실행하고 엔터를 누르세요.
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    교체가 성공하면 `LDC1612_FREQ = 40000000`가 포함된 행이 반환됩니다. 교체가 성공하지 않았다면 아무 것도 반환되지 않습니다.
    교체가 성공했는지 확인하려면 이 명령을 사용하세요.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   팁: 수정 후 klipper 서비스를 재시작하고 Eddy 주파수를 다시 보정하세요.
   ```cfg
   systemctl restart klipper
   ```
   Eddy 주파수 보정 명령어
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```


    ### 스캔베드 설정

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 현재 외부 1612 모듈이 출시되지 않았기 때문에 이 설정은 사용할 수 없습니다.
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 온도 보정 설정

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh 설정

    * [bed_mesh]의 `horizontal_move_z` 매개변수는 2로 설정하여 스캔 플랫폼 시 Eddy가 플랫폼에 최대한 가까워지도록 합니다.

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
