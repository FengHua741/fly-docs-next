# Eddy 사용

    :::info
    * Eddy를 설치할 때, 노즐 아래쪽 위치는 노즐 위 1-2mm 이상이어야 합니다.
    * Eddy 주파수를 보정할 때, 코일 PCB는 히트베드에서 최소 5mm 이상 떨어져 있어야 합니다.
    * 테스트 시 히트베드를 실온 상태로 유지하고, 테스트 중 히트베드를 켜지 마십시오.

    :::

## 참조 구성 및 사용 설명서
   
    ### ldc16121612 기본 주파수 수정
    이 방법은 sht36-v3/SB2040-v3 또는 SB2040-Pro-v3에만 사용됩니다.
    Mobaxterm 도구를 사용하여 명령을 실행할 수 있습니다.
    
    필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxterm 도구</Button>를 다운로드하세요.
    
    [자세한 사용법은 참조하세요](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    첫 번째 명령을 실행하고 엔터를 누릅니다.

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    두 번째 명령을 실행하고 엔터를 누릅니다.
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    치환에 성공하면 `LDC1612_FREQ = 40000000` 행이 반환됩니다. 반환된 행이 없으면 치환이 실패한 것입니다.
    또는 이 명령을 사용하여 성공 여부를 확인하십시오.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   따뜻한 팁: 수정 후 klipper 서비스를 다시 시작하고 Eddy 주파수를 재보정하는 명령을 사용하십시오.
   ```cfg
   systemctl restart klipper
   ```
   Eddy 주파수 보정 명령
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```


    ### 베드 스캔 설정

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * 주의: 현재 외부 1612 모듈은 출시되지 않았으므로 이 설정을 사용할 수 없습니다.
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SHT36
        i2c_bus: i2c1e
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 온도 보정 설정

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SHT36:gpio28
        ```

### bed_mesh 설정

    * [bed_mesh]의 `horizontal_move_z` 매개변수는 스캔 플랫폼 시 Eddy가 플랫폼에 최대한 가까워지도록 2로 설정하는 것이 좋습니다.

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
