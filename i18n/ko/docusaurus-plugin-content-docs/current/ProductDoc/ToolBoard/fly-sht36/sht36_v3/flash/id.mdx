# 메인보드 ID 읽기 및 확인

    :::tip

    번역기를 통해 웹 페이지를 열 때 구성 요소를 수정하지 마십시오.

    :::

    :::warning 주의사항

    번역기를 통해 웹 페이지를 열 때 구성 요소를 수정하지 마십시오.

    :::

    :::danger 주의사항

    번역기를 통해 웹 페이지를 열 때 구성 요소를 수정하지 마십시오.

    :::

<Tabs>
    <TabItem value="can" label="CANBus UUID 가져오기">
    * 코딩 스위치가 올바른 위치에 있는지 확인하십시오.
    * CAN H와 CAN L가 전원이 끊긴 상태에서 약 `60Ω` 정도의 저항을 가지는지 확인하십시오. 만약 `120Ω`라면 `120Ω` 스위치를 켜야 하며, 만약 `40Ω` 정도라면 스위치를 끄십시오.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/120.webp').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/232_mode.webp').default} size="80%" align="left" />              
    * ID를 검색하기 전에 XT30 데이터 케이블을 사용하여 도구 보드를 UTOR 또는 G2T에 연결하십시오. 아래는 참조 연결도입니다.
