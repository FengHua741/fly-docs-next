# 1. 제품 소개

FLY SHT36 V3은 광저우 멜론 전자 과학 기술 유한 회사가 3D 프린터 노즐에 대해 설계한 노즐 도구 보드입니다. 이 도구 보드를 사용하면 기존의 메인보드와 프린팅 헤드 사이의 복잡한 배선을 4개의 선으로 대체하여 배치를 최적화할 수 있습니다. SHT36 V3은 36 스텝 모터에 적합하며 CAN 트랜시버, RS232 트랜시버, TypeC 포트, TMC2209 드라이버, LIS2DW 가속도 센서, 2개의 제어 가능한 팬 등의 다양한 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 및 RRF 펌웨어에 적합합니다.

## 1.1 제품 특징

* 부트 버튼을 미리 준비했습니다. 사용자는 TypeC를 통해 부트 모드로 업데이트할 수 있습니다.
* Klipper 펌웨어, RRF 펌웨어를 지원합니다.
* CAN 모드, RS232 모드, RRF CAN 모드를 지원합니다.
* RS232 모드에서는 높은 전송 속도와 강력한 안정성을 제공합니다. CAN 모드에서는 CAN 확장을 지원하고 CAN 확장 인터페이스를 예약했습니다.
* 온도 센서 지원: 내장된 MAX31865 칩과 PT100 및 PT1000 온도 센서와 호환됩니다.
* 2개의 제어 가능한 팬을 지원하며, 전압 선택이 가능합니다. Mos 소판을 교체할 수 있어 수리 및 교체가 용이하며 최대 출력 전류는 1A입니다.
* 내장된 LIS2DW 가속도 센서와 LDC1612 센서를 지원합니다.
* RGB 인터페이스를 예약했으며, 내장된 2209 드라이버로 BLtouch, klicky, Voron Tap, PL08n 등 다양한 평평 센서를 지원합니다.
* 12-24V 전압, 15A 전류 입력을 지원하며, 열 막대 단자를 확대하여 정격 전류는 10A입니다.

## 1.2 새로운 기능

* Klipper CAN 모드, RS232 모드, RRFCAN 모드를 서로 전환할 수 있습니다.
* RS232 모드를 새로 추가하여 높은 전송 속도와 강력한 안정성을 제공하며, 디버깅이 더 간단해졌습니다.
* RRF CAN 모드를 새로 추가하여 RRF 도구 보드로 전환할 수 있으며, 적용 범위를 넓혔습니다.
* CAN 확장 인터페이스를 새로 추가하여 CAN 장비의 확장을 쉽게 만들었습니다.
* 종단 저항 다이얼 스위치를 새로 추가하여 종단 저항 120R을 다이얼로 선택할 수 있으며, 점프선이 필요 없습니다.
* 1미터 전원/신호 연결선을 새로 추가하여 연결선은 두꺼비선 설계를 사용하여 간섭 저항이 더 강합니다.

## 1.3 제품 사양

* MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
* 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류를 지원하며 피크 전류는 30A입니다.
* 펌웨어: Reprap/Klipper
* 입력 전압: DC12V-24V
* 논리 전압: DC3.3V
* 내장 센서: LIS2DW와 LDC1612
* 열 막대 인터페이스: 열 막대 (HEAT0), 10A
* 확장 인터페이스: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN 확장 단자
* 팬 인터페이스: 두 개의 DC 제어 가능한 팬
* 모터 드라이버: 내장된 TMC2209
* 온도 센서: 한 쌍의 100K NTC 또는 PT1000 (TH0), 한 쌍의 PT100/PT1000 선택 가능
