## 플래시 모드 진입

    :::팁

    위쪽이 `리셋 버튼`, 아래쪽이 `부트 버튼`입니다.

    :::

    * 플래시 모드로 진입하는 방법은 두 가지입니다.
    * 첫 번째 방법: 메인보드가 완전히 전원을 차단된 상태에서 부트 버튼을 누른 후 데이터 전송용 데이터 케이블의 한쪽 끝을 메인보드에, 다른 쪽 끝을 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송용 데이터 케이블의 한쪽 끝을 메인보드에, 다른 쪽 끝을 장치에 연결한 후, 부트 버튼을 누르고 리셋 버튼을 누른 다음 리셋 버튼을 먼저 놓고 마지막으로 부트 버튼을 놓습니다.

            4. 펌웨어 플래시

            :::팁

            하나의 펌웨어만 플래시하면 됩니다.

            :::

            * 아래는 500K CAN 속도의 펌웨어 플래시 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```
                       
            * 아래는 1M CAN 속도의 펌웨어 플래시 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```

            * 아래는 USB 펌웨어 플래시 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_USB.bin
                ```

            5. 아래 화살표 부분에 표시된 내용 ``File downloaded successfully``이 나타나면 플래시 성공입니다.

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::팁

            플래시 작업이 완료되면 메인보드를 완전히 전원 차단해야 합니다.

            :::
