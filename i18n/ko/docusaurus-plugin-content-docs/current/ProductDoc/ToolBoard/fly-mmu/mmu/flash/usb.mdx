# USB 펌웨어 컴파일

    :::tip

    * MMU는 CAN 연결을 위해 출하 시 Katapult 펌웨어가 미리 설치되어 있습니다.
    * USB를 사용하여 펌웨어를 플래시하기 위해서는 USB용 Katapult 펌웨어를 플래시해야 합니다.
    * 플래시한 번만 하면 이후에는 필요 없습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapult 펌웨어 플래시</Button>

    :::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * 마이크로컨트롤러 시작 시 GPIO 핀 선택, 입력: !PA15 (주의: 대문자 필수)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고, 그 후에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었고 명령줄 화면으로 돌아왔습니다.
    * 아래 명령어를 입력하여 컴파일을 시작합니다 (시간이 다소 소요됩니다).
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 다음 내용이 출력되면 컴파일 성공입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
