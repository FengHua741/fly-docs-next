# USB 펌웨어 컴파일

    :::tip

    * MMU 출하 시 CAN 연결용 Katapult 펌웨어가 미리 프래시됨
    * USB 펌웨어 프래싱을 사용하려면 USB용 Katapult 펌웨어를 프래시해야 함
    * 한 번 프래시하면 다시 프래시할 필요 없음
    * <Button variant="contained" disableElevation href="./bl">Katapult 펌웨어 프래싱</Button>

    :::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * 마이크로 컨트롤러 시작 시 GPIO 핀 설정 선택, 입력: !PA15 (주의: !는 대문자임)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고, 이때 `Y` 키를 누름
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 구성이 저장되고 명령줄 인터페이스로 돌아감
    * `make -j4` 입력하여 컴파일 시작, 시간이 좀 걸림
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 다음 내용이 출력되면 컴파일 성공

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
