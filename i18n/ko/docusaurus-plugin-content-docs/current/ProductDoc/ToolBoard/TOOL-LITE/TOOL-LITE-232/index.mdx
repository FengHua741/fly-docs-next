# 소개

# 1. 제품 소개

FLY TOOL-LITE-232는 광주미倫電子科技有限公司이 3D 프린터를 위해 설계한 도구 보드입니다. 이 도구 보드는 xt30/kf 인터페이스를 제공하여 기존의 메인보드와 프린트 헤드 사이의 복잡한 배선을 4개의 선으로 대체하여 레이아웃을 최적화할 수 있습니다. TOOL-LITE-232는 36/42 스텝 모터에 적합하며, RS232 트랜시버, TypeC 포트, TMC2209 드라이버, LIS2DW 가속도 센서, 2개의 제어 가능한 팬 등의 기타 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 펌웨어에 적합합니다.

## 1.1 제품 특징

    * 부트 버튼을 예약해두어 사용자는 TypeC를 통해 부트 모드로 진입하여 펌웨어를 업데이트할 수 있습니다.
    * Klipper 펌웨어를 지원합니다.
    * S232 모드를 지원합니다.
    * RS232 모드에서 더 높은 전송 속도와 더 강력한 안정성을 제공합니다.
    * 온도 센서 지원: ntc100k 및 PT1000 온도 센서.
    * 두 개의 제어 가능한 팬을 지원하며, 최대 출력 전류는 1A까지 가능합니다.
    * LIS2DW 가속도 센서가 내장되어 있습니다.
    * 고압 프로브 인터페이스를 예약해두었으며, 내장된 2209 드라이버를 통해 다양한 평탄화 센서를 지원합니다. 예를 들어, BLtouch, klicky, Voron Tap, PL08n 등이 있습니다.
    * 12-24V 전압, 15A 전류 입력을 지원하며, 히팅 로드 단자를 증가시켜 10A의 정격 전류를 제공합니다.

## 1.2 신규 기능
    * 새로운 RS232 모드가 추가되었습니다. 더 높은 전송 속도와 더 강력한 안정성을 제공하며, 디버깅 사용이 더욱 간단해졌습니다.
    * 새로운 1m 전원/신호 연결선이 추가되었습니다. 연결선은 잡음 저항성이 더 강한 트위스트 페어 설계를 사용합니다.

## 1.3 제품 사양

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류를 지원하며, 피크 전류는 30A입니다.
    * 펌웨어: Klipper
    * 입력 전압: DC12V-24V
    * 논리 전압: DC3.3V
    * 내장 센서: LIS2DW
    * 히팅 로드 인터페이스: 히팅 로드 (HEAT0), 10A
    * 확장 인터페이스: 고압 프로브 인터페이스, EndStop, PT100/PT1000, XH2.54-2P CAN 확장 포트
    * 팬 인터페이스: 두 개의 DC 제어 가능한 팬
    * 모터 드라이버: 내장된 TMC2209
    * 온도 센서: 한 개의 100K NTC 또는 PT1000 (TH0)
