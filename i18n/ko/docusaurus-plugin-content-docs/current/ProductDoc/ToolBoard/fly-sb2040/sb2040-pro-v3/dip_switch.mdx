# 내장 다이얼 소개

:::warning 주의사항

이미지 해상도가 너무 높아 웹 페이지에서 볼 때 명확하지 않습니다. **마우스 커서를 접선도면 위에 놓고**, **우클릭**하여 **이미지 저장**을 선택하고, 이미지를 데스크톱에 저장한 후 확인해주세요!!!

:::


## 작업 모드 선택

:::warning 주의사항

SB2040 PRO V3는 세 가지 연결 방식을 지원합니다. 정상적으로 연결하려면 다이얼을 수정하고 펌웨어를 다시烧录是“烧录”的意思，这里应该使用“烧录”来保持一致性。但是在韩文中没有直接对应的词，通常会使用英文“flash”或者“burn”。考虑到上下文，这里使用“재구성”来表达类似的意思。
