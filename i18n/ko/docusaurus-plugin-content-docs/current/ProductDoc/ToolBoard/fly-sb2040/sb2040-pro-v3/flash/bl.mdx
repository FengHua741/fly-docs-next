# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::팁

    * SB2040_PRO_V3은 CAN과 RS232을 지원합니다.
    * 여기서는 CAN 속도가 500K와 1M인 펌웨어만 제공됩니다.
    :::

## 플래시 모드 진입

    :::팁

    왼쪽은 `리셋 버튼`, 오른쪽은 `부트 버튼`입니다.

    :::

    * 플래시 모드에 진입하는 방법은 두 가지입니다.
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서 부트 버튼을 누르고 데이터 전송용 케이블을 메인보드와 장치 사이에 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블을 메인보드와 장치 사이에 연결한 후, 부트 버튼을 누른 상태에서 리셋 버튼을 누른 다음 리셋 버튼을 먼저 놓고 마지막으로 부트 버튼을 놓습니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />

            4. 펌웨어 플래시

            :::팁

            하나의 펌웨어만 플래시하면 됩니다.

            :::

            * 아래는 500K CAN 속도의 펌웨어 플래시 명령어입니다.
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PROV3_CANBOOT_CAN_500K.uf2
                ```
                       
            * 아래는 1M CAN 속도의 펌웨어 플래시 명령어입니다.
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PROV3_CANBOOT_CAN_1M.uf2
                ```


            5. 아래의 메시지가 나타나면 플래시 성공입니다.

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::팁

            플래시가 완료되면 메인보드를 완전히 전원을 끊어야 합니다.
