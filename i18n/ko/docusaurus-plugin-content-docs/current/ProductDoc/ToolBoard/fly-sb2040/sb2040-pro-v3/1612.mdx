# 에디 사용

    :::info

    * 침대 청소를 할 때는 별도의 Z 제한을 사용하는 것이 좋으며, 침대 청소를 Z 제한으로 사용하지 않는 것이 좋습니다.
    * 에디를 설치할 때, 바닥과 노즐의 위치는 노즐 위에서 최소 1 - 2mm가 되어야 합니다.
    * 에디 주파수를 보정할 때, 코일 PCB는 열판에서 최소 5mm 떨어져 있어야 합니다.
    * 전체 열판을 상온에서 테스트하지 말고, 테스트할 때 열판을 켜지 마세요.

    :::

## 참고 구성 및 사용 설명

##### `printer.cfg` 구성 파일에 다음 구성을 추가하고 에디를 디버깅합니다.

    ### 침대 청소 구성

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 2.0
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### 온도 보정 구성

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

    ## 망상 침대 명령

        * `printer.cfg` 구성에 아래의 매크로를 추가합니다.
            ```bash
            [gcode_macro BED_MESH_CALIBRATE]
             rename_existing: _BED_MESH_CALIBRATE
             gcode: 
                   _BED_MESH_CALIBRATE METHOD=rapid_scan {rawparams}
            ```
        * 지역 네트워크 침대 명령은 Klipper가 객체 배제 기능을 사용할 수 있도록 해야하며, 시작 매크로에 추가하는 것이 좋습니다.
            ```bash
            BED_MESH_CALIBRATE METHOD=rapid_scan adaptive=1 
            ```

### bed_mesh 구성

    * [bed_mesh]의 `horizontal_move_z` 매개변수는 2로 설정하는 것이 좋으며, 스캔 플랫폼을 할 때 에디가 플랫폼에 가까이 있도록 합니다.

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
## 충돌 방지 매크로

    ### `[quad_gantry_level]` 침대 청소 최적화
        * `[quad_gantry_level]`를 구성한 경우 아래 매크로를 구성에 추가할 수 있으며, 없으면 추가하지 않아도 됩니다.
        * 이 매크로는 용문의 높낮이 차이에 따라 탐지 높이를 낮춥니다.
            ```cfg
            [gcode_macro QUAD_GANTRY_LEVEL]
            rename_existing: _QUAD_GANTRY_LEVEL
            gcode:
                SAVE_GCODE_STATE NAME=STATE_QGL
                BED_MESH_CLEAR
                {% if not printer.quad_gantry_level.applied %}
                _QUAD_GANTRY_LEVEL horizontal_move_z=10 retry_tolerance=1
                {% endif %}
                _QUAD_GANTRY_LEVEL horizontal_move_z=2 METHOD=rapid_scan ADAPTIVE=1
                # G28 Z
                RESTORE_GCODE_STATE NAME=STATE_QGL
            ```

    ### `[z_tilt]` 침대 청소 최적화

        * `[z_tilt]`를 구성한 경우 아래 매크로를 구성에 추가할 수 있으며, 없으면 추가하지 않아도 됩니다.
        * 이 매크로는 용문의 높낮이 차이에 따라 탐지 높이를 낮춥니다.
            ```cfg
            [gcode_macro Z_TILT_ADJUST]
            rename_existing: _Z_TILT_ADJUST
            gcode:
                SAVE_GCODE_STATE NAME=STATE_Z_TILT
                BED_MESH_CLEAR
                {% if not printer.z_tilt.applied %}
                _Z_TILT_ADJUST horizontal_move_z=10 retry_tolerance=1
                {% endif %}
                _Z_TILT_ADJUST horizontal_move_z=2 METHOD=rapid_scan ADAPTIVE=1
                # G28 Z
                RESTORE_GCODE_STATE NAME=STATE_Z_TILT
            ```

    ### 에디가 대략적으로 열판 간섭으로 침대 청소를 방해하는 경우의 최적화 매크로

        * 고출력 교류 열판(500W 이상)을 사용하는 경우
        * 아래 매크로는 침대 청소 시 열판을 끄고 청소가 완료되면 다시 가열합니다.
            ```cfg
            [gcode_macro BED_MESH_CALIBRATE]
            rename_existing: _BED_MESH_CALIBRATE
            gcode:
                {% set TARGET_TEMP = printer.heater_bed.target %}
                M140 S0
                _BED_MESH_CALIBRATE {rawparams}
                M140 S{TARGET_TEMP}
            ```

   # 디버깅 단계: 
   ## 첫 번째 단계 주파수 수정

    ### ldc16121612 기본 주파수 수정
    이 방법은 sht36-v3/SB2040-v3 또는 SB2040-Pro-v3에만 사용됩니다.
    Mobaxterm 도구를 사용하여 명령을 실행할 수 있습니다.
    
    필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxterm 도구</Button>를 다운로드합니다.
    
    [상세한 사용 방법은](/docs/ProductDoc/SBC/fly-pi-v2/ssh)를 참고하세요.


    첫 번째 명령을 실행하고 Enter를 누릅니다.

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    두 번째 명령을 실행하고 Enter를 누릅니다.
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    대체가 성공하면 `LDC1612_FREQ = 40000000`이 포함된 행이 반환되며, 대체가 성공한 것입니다. 아무런 반환이 없으면 대체가 실패한 것입니다.
    또는 이 명령을 사용하여 성공 여부를 확인할 수 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
