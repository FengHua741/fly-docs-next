## 플래싱 모드 진입

    :::tip

    왼쪽은 `RESET 버튼`, 오른쪽은 `BOOT 버튼`입니다.

    :::

    * DFU 모드 진입에는 두 가지 방법이 있습니다.
    * 첫 번째 방법: 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채로 데이터 전송용 케이블을 한쪽은 보드에, 다른一侧은 상위 기기와 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블을 보드와 상위 기기에 연결한 후 BOOT 버튼을 누른 상태에서 RST 버튼을 누른 다음 RST 버튼을 먼저 놓고 BOOT 버튼을 마지막으로 놓습니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
