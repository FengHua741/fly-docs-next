# 드라이버 저압 해결 방법

## 방법 일

    :::danger

    * 전원 교체
    * 전원 필터 추가
    * 모터 판정 변경
    * 모터 외장물을 도구 보드의 GND에 연결
    * 엑트루더 드라이버 구성에 `driver_TPFD: 0` 추가

    :::

## 방법 이
    * SSH를 통해 상위 기기와 연결
    * 아래 명령어 입력
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * klipper 업데이트 후 이 단계가 다시 복원될 수 있습니다.
    :::
