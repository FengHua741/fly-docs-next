## 제품 소개

    FLY SB2040은 광주 망렌 전자 기술 유한 회사가 VORON Stealthburber 프린트 헤드를 위해 제작한 스프레이 헤드 도구 보드입니다. 이 도구 보드를 사용하면 기존의 메인보드와 프린트 헤드 사이에 복잡한 배선을 대체하여 4개의 선으로 연결할 수 있으며, 레이아웃을 최적화합니다. SB2040은 36스텝 모터에 적합하며 CAN 트랜시버, USB 포트, TMC2209 드라이버, ADXL345 가속도 센서, 두 개의 제어 가능한 팬 등 다양한 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 펌웨어에서 작동합니다.

    * CAN 버스 연결을 지원하여 데이터 전송이 더 안정적이며 지연 시간이 줄어들고 연결이 더욱 안정적입니다.
    * 내장 TMC2209 드라이버는 UART 모드로 작동하여 엑트루더에 사용될 수 있습니다.
    * 내장 ADXL345 가속도 센서
    * BLTOUCH, PL08N 등의 평탄화 센서를 지원하며 XY 리미트 스위치를 통합합니다.
    * 12-24V 전압 입력을 지원합니다.
    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 4.7K 저항 (NTC 열 감지기용) 또는 1K 저항 (PT1000용) 전환을 지원합니다.
    * CAN 인터페이스는 XT30 (2+2)를 사용하며, 15A 전류를 지원하고 피크 전류는 30A입니다.
    * 내장 NTC100k 저항, 창 온도 측정용
    * ADXL345의 INT1과 INT2를 mcu에 연결하여 일부 사용자를 위한 평탄화에 사용할 수 있습니다.
    * 고전압 입력 리미트를 지원합니다.
    * 3m CAN 연결선을 제공하여 사용자가 단자를 올바르게 눌러 신호 손실을 방지하는 문제를 해결합니다.
    * 2/3/4선 팬을 지원합니다.
    * 분리 설계, Stealthburner를 쉽게 분리할 수 있습니다.
    * hartk의 Stealthburner Toolhead Board와 호환 (팬 소형 보드는 교체하지 않아도 됩니다.)
    * 인터페이스: 제어 가능한 팬\*3, RGB\*1, 리미트\*3, 히팅 요소\*1, 열 감지기\*1

## 제품 사양

    * Klipper CAN 모드, RS232 모드, RRFCAN 모드 간의 전환을 지원합니다.
    * XT30 (2+2) 커넥터를 사용한 연결 인터페이스는 15A 전류를 지원하며, 피크 전류는 30A입니다.
    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 입력 전압: DC12-24V
    * 히팅 엘리먼트 최대 출력 전류: 5A
    * 팬 인터페이스 최대 출력 전류: 1A
