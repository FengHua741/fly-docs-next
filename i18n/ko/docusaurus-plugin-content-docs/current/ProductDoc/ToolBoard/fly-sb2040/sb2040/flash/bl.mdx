# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::tip

    * SB2040은 CAN을 지원합니다.
    * 여기에서는 500K와 1M 속도의 CAN 펌웨어만 제공됩니다.

## 플래싱 모드 진입

    :::tip

    동그라미 친 것은 `BOOT 버튼`입니다.

    :::

    * 메인보드가 완전히 전원이 끊긴 상태에서, BOOT 버튼을 누른 채 데이터 전송용 데이터 케이블의 한쪽을 메인보드에, 다른 쪽을 기기로 연결합니다.

            4. 펌웨어 플래싱

            :::tip

            하나의 펌웨어만 플래싱하면 됩니다.

            :::

            * 아래는 500K CAN 속도의 펌웨어 플래싱 명령어입니다.
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * 아래는 1M CAN 속도의 펌웨어 플래싱 명령어입니다.
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```


            5. 아래의 팁이 나타나면 플래싱이 성공한 것입니다.

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            플래싱이 완료되면, 메인보드를 완전히 전원을 끊어야 합니다.

            :::

        </TabItem>
    </Tabs>
