# Eddy 사용

    :::info
  * 이전에 Eddy의 설정을 구성한 경우, 삭제하십시오.
  * Eddy를 설치할 때, 노즐 위쪽으로 최소 1-2mm 위치해야 합니다.
  * 테스트할 때는 열판을 열지 마십시오.
  * 사용하기 전에 구성에서 `probe` 설정을 삭제하십시오.
    :::

    :::warning 주의사항
    * 멀티 Z 축 기계는 수동으로 평평하게 조정해야 합니다.
    :::


## 접속도면

    * 상표가 열판에 설치되어야 합니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/img/1612.webp').default} size="40%" align="left" />


## 사용 설명
   * SSH 도구를 통해 IP로 터미널에 로그인하고 스크립트를 실행합니다.
    필요한 SSH 도구 다운로드 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxterm 도구</Button>

  <TabItem value="eddy1" label="국제 사용">
    # 국제 사용은 아래의 명령어를 사용하세요
    ### 설치를 시작합니다
    * 다운로드한 SSH 도구를 열고, 아래의 명령어를 입력하고 Enter를 누르세요, 복사하기를 클릭하면 됩니다.
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * `fluidd` 웹 페이지의 `moonraker.conf` 구성 파일에 아래의 설정을 추가하여 나중에 업데이트하기 쉽게 합니다
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * 업데이트를 한 경우 아래의 명령어를 실행하세요
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
