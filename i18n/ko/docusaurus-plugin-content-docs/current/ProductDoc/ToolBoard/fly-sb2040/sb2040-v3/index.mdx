# 1. 제품 소개

    FLY SB2040 V3은 광주 맥론 전자 과학 기술 유한 회사가 VORON Stealthburber 프린터 헤드를 위해 제작한 도구 보드입니다. 이 도구 보드를 사용하면 원래 메인보드와 프린터 헤드 사이의 복잡한 연결선을 4개의 선으로 대체하여 배치를 최적화할 수 있습니다. SB2040은 stealthburner와 afterburner에 적합하며, CAN 트랜시버, RS232 트랜시버, USB 포트, TMC2209 드라이버, LIS2DW 가속도 센서, 세 개의 제어 가능한 팬 등의 다른 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper와 RRF 펌웨어에 적합합니다.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/img/sb2040.webp').default} size="100%" align="center" />


## 1.1 제품 특징

    * 부트 버튼을 미리 준비해 두었으며, 사용자는 TypeC를 통해 부트 모드로 업데이트할 수 있습니다.
    * Klipper 펌웨어, RRF 펌웨어를 지원합니다.
    * CAN 모드, RS232 모드, RRF CAN 모드를 지원합니다.
    * RS232 모드에서는 더 높은 전송 속도와 더 강력한 안정성을 제공합니다. CAN 모드에서는 CAN 확장을 지원하고 CAN 확장 인터페이스를 예약합니다.
    * 온도 센서 지원: 보드 내장된 MAX31865 칩으로 PT100 및 PT1000 온도 센서와 호환됩니다.
    * 보드 내장 LIS2DW 가속도 센서와 LDC1612 센서가 있습니다.
    * RGB 인터페이스를 예약하고, 보드 내장 2209 드라이버로 다양한 플랫팅 센서를 지원합니다. 예를 들어 klicky, Voron Tap, PL08n 등이 있습니다.
    * 12-24V 전압, 15A 전류 입력을 지원하며, 열 스레드 포트를 증가시키고, 정격 전류는 10A입니다.

## 1.2 신규 기능

    * Klipper CAN 모드, RS232 모드, RRF CAN 모드를 서로 전환할 수 있습니다.
    * RS232 모드를 새로 추가했으며, 더 높은 전송 속도와 더 강력한 안정성을 제공하고, 디버깅도 더 간단해졌습니다.
    * RRF CAN 모드를 새로 추가했으며, RRF 도구 보드로 전환할 수 있어 적용 범위가 더 넓어졌습니다.
    * CAN 확장 인터페이스를 새로 추가하여 CAN 장비를 쉽게 확장할 수 있습니다.
    * 터미널 저항 다이얼 스위치를 새로 추가하여, 터미널 저항 120R을 다이얼 선택으로 설정할 수 있으며, 점선을 사용하지 않아도 됩니다.
    * 1미터 전원/신호 연결선을 새로 추가했으며, 연결선은 격자 설계를 사용하여 방해 신호에 더 강하게 대응할 수 있습니다.

## 1.3 제품 사양


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류를 지원하며, 피크값은 30A입니다.
    * 펌웨어: Reprap/Klipper
    * 입력 전압: DC12V-24V
    * 논리 전압: DC3.3V
    * 보드 내장 센서: LIS2DW와 선택 가능한 ldc1612와 max31865
    * 열 스레드 인터페이스: 열 스레드 (HEAT0)
    * 확장 인터페이스: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN 포트
    * 팬 인터페이스: 1개의 DC 팬, 2개의 DC 팬 또는 PWM 팬
    * 모터 드라이버: 보드 내장 TMC2209
    * 온도 센서: 100K NTC 또는 PT1000 (TH0), PT100/PT1000 선택 가능
