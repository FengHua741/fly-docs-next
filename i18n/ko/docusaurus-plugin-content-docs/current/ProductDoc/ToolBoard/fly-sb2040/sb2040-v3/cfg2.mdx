# 참조 설정

```cfg
[probe_eddy_current fly_eddy_probe]
sensor_type: ldc1612
i2c_address: 43
i2c_mcu: SB2040
i2c_bus: i2c1b
x_offset: 0 # x 오프셋을 설정하세요.
y_offset: 0 # y 오프셋을 설정하세요.
speed:10
lift_speed: 15.0
i2c_speed: 4000000

[temperature_probe fly_eddy_probe]
sensor_type: Generic 3950
sensor_pin: SB2040:gpio28

[force_move]
enable_force_move: true
```
