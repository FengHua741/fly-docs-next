    * **마이크로컨트롤러 시작 시 설정할 GPIO 핀 선택**을 선택하고 입력: **!gpio5,!gpio19,!gpio24**
    :::warning
        `!gpio5,!gpio19,!gpio24`를 영문 입력 상태에서 입력해야 합니다.
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration**이 나타나고 그 후 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었으며 명령줄 인터페이스로 돌아왔습니다.
    * 아래 명령어를 입력하여 컴파일을 시작합니다. 시간이 다소 걸릴 수 있습니다.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 아래 내용이 출력되면 컴파일 성공입니다.

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## 프래밍 모드 진입

    :::tip

    왼쪽은 `리셋 버튼`, 오른쪽은 `부트 버튼`입니다.

    :::

    * DFU 모드로 진입하는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 보드 전원을 완전히 끈 상태에서 부트 버튼을 누른 채로 데이터 전송 가능한 데이터선을 한쪽은 보드에, 다른 쪽은 상위 기기(主机)에 연결합니다.
    * 두 번째 방법: 데이터 전송 가능한 데이터선을 한쪽은 보드에, 다른 쪽은 상위 기기에 연결한 후 부트 버튼을 누른 상태에서 리셋 버튼을 누른 다음 리셋 버튼을 놓고 마지막으로 부트 버튼을 놓습니다.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.webp').default} size="100%" align="center" />
