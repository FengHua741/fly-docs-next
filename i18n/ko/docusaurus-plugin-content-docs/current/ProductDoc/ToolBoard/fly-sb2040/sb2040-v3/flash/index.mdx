# 펌웨어 플래싱 안내

    * SB2040 V3는 RS232 및 CAN을 사용하여 상위 기기와 연결할 수 있습니다.
    * RS232 및 CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블의 모양입니다.
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 플래싱은 Katapult를 통해 Klipper를 플래싱해야 합니다.
* 메인보드에 Katapult 펌웨어가 이미 프로그래밍되어 있다면 마지막 단계의 Katapult 펌웨어 플래싱은 생략할 수 있습니다. 이는 메인보드의 Katapult 펌웨어 프로그래밍 실패를 예방하기 위함입니다.
* 메인보드에 Katapult 펌웨어가 성공적으로 프로그래밍되었다면 리셋 버튼을 두 번 빠르게 누른 후 메인보드에서 LED가 깜빡이는지 확인할 수 있습니다. LED가 깜빡인다면 메인보드는 Katapult 펌웨어로 업데이트된 것입니다.
:::

## 어떤 펌웨어를 선택해야 하는지 어떻게 판단하는지

    * SB2040 V3는 상위 기기와 다음 방법으로 연결할 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CAN 펌웨어 플래싱</Button>
        * 장점: `Katapult` 펌웨어를 프로그래밍한 경우 상위 기기에 손을 대지 않고 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 때 위치 재설정 시간 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하고, UTOC 또는 브릿지 펌웨어가 프로그래밍된 메인보드가 필요합니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">RS232 펌웨어 플래싱</Button>
        * 장점: 특별한 소프트웨어 없이 사용할 수 있으며, 통신이 신뢰성 있고 안정적입니다.
        * 단점: 펌웨어를 다시 플래싱하려면 수동 DFU로 들어가야 하거나, RST 버튼을 두 번 클릭하여 `Katapult`로 들어갈 수 있어야 합니다.
        * 팁: G2T 또는 UTOR와 함께 사용해야 합니다.
