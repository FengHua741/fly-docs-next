# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::팁

    * SB2040_V3은 CAN 및 RS232을 지원합니다.
    * 여기에서는 CAN 펌웨어를 500K와 1M 속도로 제공합니다.
    :::

## 플래싱 모드 진입

    :::팁

    왼쪽이 `리셋 버튼`, 오른쪽이 `부트 버튼`입니다.

    :::

    * 플래싱 모드에는 두 가지 방법이 있습니다.
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서 부트 버튼을 누르고 데이터 전송용 케이블의 한쪽을 메인보드에, 다른 쪽을 기기에 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블의 한쪽을 메인보드에, 다른 쪽을 기기에 연결한 후, 부트 버튼을 누른 상태에서 리셋 버튼을 누른 다음 리셋 버튼을 놓고 마지막으로 부트 버튼을 놓습니다.

            4. 펌웨어 플래싱

            :::팁

            하나의 펌웨어만 플래싱하면 됩니다.

            :::

            * 아래는 500K CAN 속도의 펌웨어 플래싱 명령어입니다.
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040V3_CANBOOT_CAN_500K.uf2
                ```
                       
            * 아래는 1M CAN 속도의 펌웨어 플래싱 명령어입니다.
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040V3_CANBOOT_CAN_1M.uf2
                ```
