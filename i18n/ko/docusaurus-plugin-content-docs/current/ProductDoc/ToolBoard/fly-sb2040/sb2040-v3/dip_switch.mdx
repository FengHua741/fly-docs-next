# 보드 내장 다이얼 소개

:::warning 주의사항

이미지 해상도가 높아 웹 페이지에서 볼 때 명확하게 확인할 수 없습니다. **마우스 커서를 접선도면 위에 놓고**, **오른쪽 클릭**하여 **이미지 저장하기**를 선택하고, 이미지를 데스크톱에 저장한 다음 확인하세요!!!

:::
