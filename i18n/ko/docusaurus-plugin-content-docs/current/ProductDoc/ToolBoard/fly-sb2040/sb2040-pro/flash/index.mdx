# 펌웨어 쓰기 설명

    * SB2040 PRO는 CAN 연결 방식을 사용하여 상위 기기와 연결할 수 있습니다.
    * CAN은 `XT30(2+2)-F`를 사용하여 연결해야 합니다.
    * 아래는 `XT30(2+2)-F` 데이터 케이블의 모습입니다.
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 주의사항
* 메인보드 펌웨어 쓰기는 Katapult를 통해 Klipper를 써야 합니다.
* 메인보드에 이미 Katapult 펌웨어가 깔려 있다면 마지막의 Katapult 펌웨어 쓰기는 생략할 수 있습니다. 그건 메인보드의 Katapult 펌웨어 쓰기가 성공하지 않았을 때 대비한 것입니다.
* 재설정 버튼을 두 번 빠르게 누른 후 메인보드에 LED가 깜빡이는지 확인할 수 있습니다. 만약 깜빡인다면 메인보드가 Katapult 펌웨어로 업데이트된 것입니다.
:::

## 어떤 펌웨어가 필요한지 어떻게 판단하는지

    * SB2040 PRO는 다음과 같은 방법으로 상위 기기에 연결할 수 있습니다.
    * [CAN 펌웨어](./can.mdx "클릭하여 이동")
        * 장점: `Katapult` 펌웨어를 깔았다면, 메인보드를 만지지 않고도 펌웨어를 업데이트할 수 있으며, 버스 연결을 지원합니다.
        * 단점: `KlipperScreen`과 함께 사용할 경우, 원점 설정 시간 초과 등의 오류가 발생할 수 있습니다.
        * 팁: CAN을 사용하려면 상위 기기가 CAN을 지원하는지 확인하고, UTOC 또는 브릿지 펌웨어를 깔은 메인보드가 필요합니다.


## Katapult 펌웨어 설명

    * SB2040 PRO는 기본적으로 CAN 연결 속도가 1M인 Katapult 펌웨어를 탑재하고 있습니다.
