# 드라이버 저압 해결 방법

## 방법 일

    :::danger

    * 전원 교체
    * 전원 필터 추가
    * 모터 판별 교체
    * 모터 외부를 도구판의 GND에 연결
    * 추출기 드라이버 구성에 `driver_TPFD: 0` 추가

    :::

## 방법 이
    * SSH를 통해 상위 기계에 연결
    * 아래 명령어 입력
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * klipper 업데이트 후 이 단계가 원상 복귀될 수 있습니다.
    :::
