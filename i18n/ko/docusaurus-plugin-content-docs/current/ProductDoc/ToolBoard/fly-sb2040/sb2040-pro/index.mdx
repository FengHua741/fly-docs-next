# 제품 소개

    FLY SB2040 PRO는 광저우 메이런 전자 기술 유한 회사가 VORON Stealthburber 프린팅 헤드를 위해 만든 도구 보드입니다. 이 도구 보드를 사용하면 기존의 메인보드와 프린팅 헤드 사이의 복잡한 배선을 4개의 선으로 간소화하여 레이아웃을 최적화할 수 있습니다. SB2040은 stealthburner와 afterburner에 적합하며 CAN 트랜시버, USB 포트, TMC2240 드라이버, ADXL345 가속도 센서, 세 개의 제어 가능한 팬 등의 다양한 기능을 통합한 도구 보드입니다. 이 도구 보드는 Klipper 및 RRF 펌웨어에 적합합니다.

## 제품 특징

    * 부트 버튼이 예약되어 있어 사용자는 TypeC를 통해 부트 모드로 업데이트할 수 있습니다.
    * Klipper 펌웨어를 지원합니다.
    * 온도 센서 지원: 내장 MAX31865 칩, PT100 호환
    * 내장 ADXL345 가속도 센서
    * RGB 인터페이스가 예약되어 있으며, 내장 2240 드라이버를 통해 다양한 평탄화 센서를 지원합니다. 예를 들어 klicky, Voron Tap, PL08n 등
    * 12-24V 전압, 15A 전류 입력을 지원하며, 열 막대 단자는 10A의 정격 전류를 갖습니다.


## 제품 사양


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * 연결 인터페이스는 XT30(2+2) 커넥터를 사용하여 15A 전류, 피크 30A를 지원합니다.
    * 펌웨어: Klipper
    * 입력 전압: DC12V-24V
    * 논리 전압: DC3.3V
    * 내장 센서: ADXL345와 선택적 MAX31865
    * 열 막대 인터페이스: 열 막대 (HEAT0)
    * 확장 인터페이스: 제어 가능한 팬*3, RGB*1, 리미트 스위치*3, 열 막대*1, 테르미스타터*1
    * 팬 인터페이스: 세 가지 제어 가능한 팬
    * 모터 드라이버: 내장 TMC2240
