# USB 펌웨어 컴파일

    :::tip

    * USB 가속계는 출하 시 USB 연결용 Katapult 펌웨어가 미리 플래시되어 있습니다.
    * 만약 플래시되지 않았다면 직접 플래시할 수 있습니다.
    * [Katapult 펌웨어 플래싱](./bl.mdx "클릭하여 이동")

    :::

## 컴파일 시작

    <SSH />

    <MakeOperate/>

    <USB />

    * `Q` 키를 누르면 **Save configuration**이 나타나고, 그 후에 `Y` 키를 누릅니다.
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었으며 명령줄 화면으로 돌아왔습니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 다소 걸립니다.
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 아래와 같은 내용이 출력되면 컴파일 성공입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
