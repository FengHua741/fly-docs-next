# Katapult 펌웨어

## 프로그래밍 모드 진입

    :::팁

    동그라미 친 부분은 `부트 버튼`입니다.

    :::

    * 프로그래밍 모드에 진입하려면, 메인보드가 완전히 전원이 꺼진 상태에서 부트 버튼을 누른 후 데이터 통신을 위한 케이블 한쪽을 메인보드에, 다른 쪽을 장치에 연결해야 합니다.
