    <TabItem value="tft" label="TFT 터치 스크린 구성" default>
## 방법 1    
         * 터치 스크린 구성은 아래 명령어로 `fly-tft-v2-c`로 설정해야 합니다.
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * 화면의 뒷면 다이얼을 `Cap`으로 설정해야 합니다.
    * 설정이 성공했는지 확인하려면 아래 명령어를 사용합니다.
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * 터치 스크린을 저항식 스크린으로 변경하려면 아래 명령어를 사용합니다.

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

        <TabItem value="hdmi" label="TFT 저항식 스크린 구성">
## 방법 1        
           * 저항식 스크린 구성은 아래 명령어로 `fly-tft-v2-r`로 설정해야 합니다. 
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
```
                                * 화면의 뒷면 다이얼을 `Resi`로 설정해야 합니다.
        * 설정이 성공했는지 확인하려면 아래 명령어를 사용합니다.
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
                                    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-r.webp').default} size="80%" align="center" />
 * 저항식 스크린을 터치 스크린으로 변경하려면 아래 명령어를 사용합니다.

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v2-c/}' /boot/FlyOS-Env.txt
```
