# 소개

우리는 새로운 FLY-π 메인보드의 출시를 기쁘게 발표합니다. 이 보드는 상위 컴퓨터로 사용할 수 있으며, Raspberry Pi를 대체할 수 있습니다. 이 보드는 1GB의 내장 DDR3 메모리를 갖추고 있어 완전한 LINUX 데스크탑을 실행할 수 있습니다(Armbian으로 최적화). 이 보드는 Klipper 및 RRF 펌웨어와 함께 사용할 수 있으며, 심지어 SHT 도구판을 사용할 때 Klipper 스크린을 실행할 수도 있습니다.

이 상위 컴퓨터 설계의 목적은 Klipper 펌웨어 또는 RRF 펌웨어를 사용하기 쉽도록 하는 것입니다. 우리는 3D 프린터에 더 적합하도록 최적화된 리눅스 시스템 이미지를 제공하며, 설치와 작동이 더 간단해집니다. 초보자라도 조금만 기초 지식이 있으면 빠르게 학습하고 사용할 수 있습니다. 다른 상위 컴퓨터 보드보다 학습 비용을 절약하여, 당신은 3D 프린팅을 즐기는 데 더 집중할 수 있습니다(3D 프린터 SD 카드가 포함된 패키지를 구매하면 Klipper 펌웨어가 미리 설치되어 있어 전원을 켤 수 있습니다).

## 특징

* CPU: 고성능 올티芯片 H5 칩, 4코어 64비트 Cortex-A53
* GPU: 고성능 6코어 Mali 450, 픽셀 채우기 속도는 2.7 gpixel/s 이상
* RAM: 1GB DDR3 (GPU와 공유) (512+512)
* ROM: 최대 128GB SD카드 지원
* 외부 장치: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1 (CAN 모듈 별도 구매 필요)
* 인터페이스: M.2 인터페이스 장착, WIFI, eMMC 등 확장 가능 (비공개 프로토콜, FLY π 전용 M.2 장치가 아닌 경우 연결하지 마십시오)
* 40핀 인터페이스, Raspberry Pi 호환
* 12-24V DC 전원 지원, 전원 공급 안정성 향상
* 5V 제어 가능한 팬 인터페이스 예약, 상위 컴퓨터 열 관리에 사용
* CAN 확장 인터페이스 예약
* 시스템에 Klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 공명 보상 플러그인 numpy와 FLY-Tools 미리 설치 (소스 변경 등 복잡한 작업 없이 초보자가 쉽게 시작할 수 있습니다 (TF 카드 또는 M2WE 구매 필요)
