# WIFI에 연결하기


## PI에 안테나 설치

    * 빨간 화살표가 가리키는 곳이 WIFI 안테나 접속부입니다.
    * WIFI 접속부는 `ipex1`입니다.
    * 안테나를 PI에 설치하지 않으면 신호가 약해서 인터넷 연결이 불가능할 수 있습니다.
