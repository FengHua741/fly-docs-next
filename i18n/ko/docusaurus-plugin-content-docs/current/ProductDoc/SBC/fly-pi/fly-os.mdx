## 시스템 번录用 준비사항

* 카드리더기
* TF 카드 한 장, 용량은 16GB 이상 또는 128GB 이하, TF 카드의 전송 속도는 Class10급 이상
* 필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">시스템 이미지</Button> 다운로드
* 필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">번录用 도구</Button> 다운로드

## 시스템 번录用

:::info
TF 카드 번录用과 TF 카드를 이용한 M2WE 번录用은 중복하여 사용할 필요가 없습니다.
:::

:::info
* 기기에 M2WE와 메모리 카드가 설치되어 있는 경우, 시스템은 기본적으로 메모리 카드에서 부팅합니다.
* M2WE는 TF 카드를 이용한 시스템 번录用을 권장합니다. USB를 통해 M2WE에 번录用하는 것은 드라이버 호환성 등으로 인해 실패할 수 있으므로, 직접 [메모리 카드를 이용한 M2WE 번录用](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 추천드립니다.
* 다운로드한 시스템 이미지가 적합한지 확인하시기 바랍니다. 적합하지 않은 경우 시스템이 부팅되지 않습니다.
:::
