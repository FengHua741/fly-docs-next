# 시스템 이미지 번들

<FASTFASLH name="H3" />

            
## 시스템 카드 설치

* 시스템을 TF 카드에 번들한 후에는 반드시 TF 카드를 LITE2 뒤쪽에 연결해야 하며, 그렇지 않으면 시스템이 시작되지 않습니다.
<ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/tf.webp').default} size="100%" align="left" />  

## 안테나 설치

1. 안테나를 설치합니다.
:::warning
* 주의하세요: 상위기기는 한글 이름의 WiFi를 지원하지 않으며, 2.4G와 5G를 합쳐 사용하는 WiFi도 지원하지 않습니다.
:::
* 빨간 화살표가 가리키는 부분이 WiFi 안테나 인터페이스입니다.
* WiFi 안테나 포트는 `ipex1`입니다.
* 안테나를 설치하지 않으면 신호가 약해져 라우터에 연결할 수 없을 가능성이 있습니다.
<ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="80%" align="left" /> 

:::tip
- 시스템 카드 설치가 완료되면 다음 단계로 진행할 수 있으며, SSH를 통해 상위기기를 연결할 수 있습니다.
:::

2. 상위기에 전원을 공급합니다.  
**여기까지 오셨다면, 이제 상위기가 시작되었습니다!!**  
다음 단계를 진행하시려면: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2/ssh">상위기기 SSH 연결</Button>를 클릭하세요.
