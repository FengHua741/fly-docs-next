# WIFI에 연결하기

## LITE 2에 안테나를 설치하기

    * 빨간 화살표가 가리키는 곳이 WIFI 안테나 접속부입니다.
    * WIFI 안테나 포트는 `ipex1` 입니다.
    * 안테나를 LITE 2에 설치하지 않으면 신호가 약해져 인터넷에 연결할 수 없을 수 있습니다.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="100%" align="left" />  

* 만약 WIFI에 정상적으로 연결되지 않는다면, 먼저 안테나가 제대로 연결되었는지 확인한 후 아래 방법으로 WIFI에 연결할 수 있습니다.
