# 소개

    Fly-Pi-lite2는入门级Klipper上位机

## 특징

    * CPU：고성능 올틱스 H3 칩, 4코어 64비트 Cortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：최대 128GB SDCARD 지원
    * WIFI：보드에 내장된 2.4G WiFi
    * 외부 장치：Spi x1, URAT x2, USB x2, HDMI x1
    * 인터페이스：FPC 예약, HDMI + USB FPC 원케이블 솔루션
    * 시스템은 klipper, moonraker, mainsail, fulidd, klipper-screen, 공명 보상 플러그인 numpy와 FLY-Tools를 미리 설치했습니다. 복잡한 작업 없이 쉽게 시작할 수 있습니다 (TF 카드 구매 필요)
