# 시스템 번들

## 상위 컴퓨터 초보자 가이드

    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/baodian/host/">상위 컴퓨터 초보자 가이드</Button>

## 시스템 번들 전 준비

    * 읽기 전용 카드 리더기
    * 최소 16GB 이상 또는 최대 128GB 이하의 TF 카드, TF 카드의 전송 속도는 반드시 Class10급 또는 그 이상
    * 대응하는 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-lite2">시스템 이미지</Button> 다운로드
    * 필요한 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">번들 도구</Button> 다운로드

## 시스템 번들

:::info
* MINI PAD는 시스템을 TF 카드에만 번들할 수 있으며, M2WE는 사용할 수 없습니다.
* 시스템 번들이 완료되면 TF 카드를 MINI PAD에 설치해야 합니다.
:::
    
    <TF />

## 시스템 카드 설치

    * 시스템을 TF 카드에 번들한 후에는 반드시 lite2에 연결해야 하며, 그렇지 않으면 시스템이 시작되지 않습니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/tf.webp').default} size="100%" align="left" />  

## 안테나 설치

    1. 안테나 설치
    :::warning
    * 상위 컴퓨터는 한글 이름의 Wi-Fi를 지원하지 않으며, 5GHz Wi-Fi도 지원하지 않습니다.
    :::
    * 빨간 화살표가 가리키는 부분은 Wi-Fi 안테나 인터페이스입니다.
    * Wi-Fi 안테나 포트는 `ipex1`입니다.
    * 신호가 약해질 수 있으므로 안테나를 설치해야 합니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - 시스템 카드 설치가 완료된 후에는 다음 단계로 넘어갈 수 있습니다. SSH를 통해 상위 컴퓨터에 연결하세요.
    :::

    2. 상위 컴퓨터에 전원 공급.  
    **여기까지 왔다면 상위 컴퓨터가 이제 시작될 수 있습니다!!**  
    다음 단계로 진행하시려면: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2/ssh">상위 컴퓨터 SSH 연결</Button>
