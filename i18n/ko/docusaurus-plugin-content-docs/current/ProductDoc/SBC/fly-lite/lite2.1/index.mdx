# 소개

    Fly-Pi-lite2.1은 입문용 Klipper 호스트 기계입니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.webp').default} size="80%" align="center" />

## 특징

    * CPU: 고성능 올티치 H3 칩셋, 4코어 64비트 Cortex-A7
    * GPU: Mali-400 MP2
    * RAM: 512MB DDR3
    * ROM: 최대 128GB microSDCARD 지원
    * WIFI: 내장 2.4G WiFi
    * 외부 장치: Spi x1, URAT x2, USB x2, HDMI x1
    * 인터페이스: FPC 예약, HDMI + USB FPC 한 줄 통
    * 시스템에 Klipper, Moonraker, Mainsail, Fulld, Klipper-Screen, 공진 보상 플러그인 numpy와 FLY-Tools가 미리 설치되어 있습니다. 복잡한 설정 없이 쉽게 시작할 수 있습니다 (TF 카드 구매 필요)

## 업데이트 기능

    * TFT V1 인터페이스 업그레이드 TFT V2
    * 리셋 및 전원 차단 핀 추가
    * 전원 켜는 LED 추가
    * 부트 버튼 제거, 리셋 버튼으로 교체
    * 배선 최적화
