# 상위 기기 SSH 연결


* 장비에 연결할 때 직렬 포트 또는 WIFI를 사용할 수 있습니다.
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해 주세요.
* SD 카드 또는 M2WE에 **해당 상위 기기의 시스템**이 녹화되어 있는지 확인하세요.
* **상위 기기가 정상적으로 시작되지 않으면 드라이버, 리미트 스위치, 팬 등 주변 장치를 제거하십시오!!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 쉽게 컴파일할 수 있습니다!!!**
* **펌웨어를 컴파일하려면 WIFI 또는 이더넷을 통해 상위 기기에 연결하는 방법을 참조하십시오.**

        ## 네트워크 IP를 사용하여 상위 기기 SSH에 연결

        * 상위 기기가 연결된 **라우터 관리자 페이지**에 로그인하세요.
        * 시스템이 **정상적으로 시작**되었는지 확인하세요.
        * 상위 기기의 **사용자 이름과 비밀번호**를 알고 있는지 확인하세요. FLY 시스템의 기본 사용자 이름은 **fly**, 기본 비밀번호는 **mellow**입니다.
        * 메인보드가 시작되고 네트워크에 연결되면 IP를 사용하여 SSH에 연결할 수 있습니다.
        * MobaXterm을 열고 다음과 같이 설정합니다. IP는 앞서 확인한 IP를 입력하거나 라우터 관리자 페이지에서 확인할 수 있습니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계는 상위 기기의 사용자 이름을 입력하는 단계입니다.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 처음 연결하면 아래와 같은 창이 뜹니다. `Accept`를 클릭하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />
