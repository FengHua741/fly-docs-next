# 시스템 이미지 번들링

<FASTFASLH name="H3" />

            
## 시스템 카드 설치


    * TF 카드에 시스템을 번들링한 후에는 LITE2.1의 뒤쪽에 TF 카드를 연결해야 합니다. 그렇지 않으면 시스템이 시작되지 않습니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tf.webp').default} size="100%" align="left" />  

## 안테나 설치

    1. 안테나를 설치합니다.
    :::warning
    * 주의하세요: 상위 컴퓨터는 한글 이름의 Wi-Fi를 지원하지 않으며, 2.4G 및 5G 병합 Wi-Fi도 지원하지 않습니다.
    :::
    * 적색 화살표가 가리키는 곳이 Wi-Fi 안테나 연결부입니다.
    * Wi-Fi 안테나 포트는 `ipex1`입니다.
    * 신호가 약해질 수 있으므로 안테나를 설치하지 않으면 라우터에 연결할 수 없습니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - 시스템 카드 설치가 완료되면 다음 단계로 넘어갈 수 있습니다. SSH를 통해 상위 컴퓨터에 연결하세요.
    :::

    2. 상위 컴퓨터에 전원을 공급합니다.  
    **여기까지 와서, 당신의 상위 컴퓨터는 실행 가능 상태입니다!!!**  
    다음 단계를 수행하세요: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2.1/ssh">상위 컴퓨터에 SSH 연결</Button>
