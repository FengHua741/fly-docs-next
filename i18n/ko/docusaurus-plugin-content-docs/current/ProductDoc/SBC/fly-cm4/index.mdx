# 소개

      Fly-CM4는 Raspberry Pi 3B와 4B와 완전히 호환되며, 특히 Raspberry Pi CM4 확장을 개선하도록 설계되었습니다. 이 확장 보드는 3D 프린터 최적화를 위한 일련의 기능을 제공합니다. 12-24V DC 전원은 3D 프린터의 정확한 제어와 지속적인 작동에 중요한 전원 안정성을 보장합니다. FLY-TFT 및 Fly-HDMI 화면 FPC 연결 기능을 지원하여 더 편리한 배선 방식을 제공합니다. Can 포트를 통해 상위 기기에서 하위 기기를 직접 Can으로 연결할 수 있으며, 이것은 현재 비교적 인기 있는 방법입니다.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-cm4/img/cm4.webp').default} size="80%" align="center" />

## 특징

    * Raspberry Pi CM4 메인보드를 지원합니다.
    * 12V-24V DC 전원을 지원합니다.
    * 내장 UTOC 모듈을 지원합니다.
    * TFT 및 HDMI One-Wire 통신을 지원합니다.
    * DSI/spi 화면 인터페이스를 지원합니다.
    * CSI/USB 카메라 인터페이스를 지원합니다.
    * 40핀 확장 인터페이스를 지원하며, Raspberry Pi와 호환됩니다.
