# 설정 방법

:::danger 주의
이 방법을 사용하면 mini pad의 가속도계 인터페이스를 통해 외부 가속도계를 사용할 수 없습니다.
:::

    * BOOT 드라이브를 열고 `armbianEnv.txt`를 찾은 다음 열어라
    * `overlays=usbhost2 usbhost3 uart2`에서 **uart2**를 **uart3**로 변경하고 저장하십시오.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart.webp').default} size="80%" align="center" />

## 연결 방법

    * mini-pad의 TX가 시리얼 포트 펌웨어가 설치된 메인보드의 RX에 연결되어야 합니다.
    * mini-pad의 RX가 시리얼 포트 펌웨어가 설치된 메인보드의 TX에 연결되어야 합니다.
    * 공용 지점을 연결해야 합니다.
    * 그렇지 않으면 시리얼 포트 펌웨어가 설치된 메인보드에 연결할 수 없습니다.

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart1.webp').default} size="80%" align="center" />

## Klipper 참조 설정

    ```cfg
    [mcu]
    serial: /dev/ttyS3
    baud：250000
    #   사용되는 보드레이트입니다. 기본값은 250000입니다.
    restart_method:command
    ```
