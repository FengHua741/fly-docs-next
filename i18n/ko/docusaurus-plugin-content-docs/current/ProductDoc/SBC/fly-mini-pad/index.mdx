# 소개

    Fly-MINI-Pad는入门级Klipper上位机 입니다.

## 특징

    * CPU：고성능 올티ン H3 칩, 4코어 64비트 Cortex-A7
    * GPU：Mali-400 MP2
    * RAM：512MB DDR3
    * ROM：최대 128GB SDCARD 지원
    * WIFI :  내장 2.4G 와이파이
    * 외부 장치：SPI x1, USB x2, Type-C x1
    * 인터페이스：세 개의 XH2.54-2P 인터페이스 각각은 POWER-IN 전원 지원 최대 24V 입력, CAN-BUS는 CAN 버스 연결을 지원, LI-PO는 3.7V 보호 기능이 있는 리튬 배터리 전원/충전 인터페이스를 지원합니다.
    * 시스템은 klipper, moonraker, mainsail, fulidd, klipper-screen, 공명 보상 플러그인 numpy와 FLY-Tools가 미리 설치되어 있습니다. 복잡한 작업 없이 바로 사용할 수 있습니다 (TF 카드 구매 필요).
    * 화면은 터치 스크린/저항식 스크린으로, 해상도는 320*480입니다.
