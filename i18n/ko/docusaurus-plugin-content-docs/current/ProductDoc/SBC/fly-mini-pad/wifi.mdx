# 와이파이 연결

## MINI PAD에 안테나 장착

    * 빨간 화살표가 가리키는 곳이 와이파이 안테나 접속부입니다.
    * 와이파이 안테나 포트는 `ipex1`입니다.
    * 안테나를 MINI PAD에 장착하지 않으면 신호가 약해져 인터넷 연결이 불가능할 수 있습니다.
