# 보드 내장 UTOC 사용

:::danger 주의

두 개의 UTOIC를 사용하지 않는 것이 좋습니다. 프로그래밍과 연결이 매우 번거로울 수 있습니다.

:::
    * 보드 내장 UTOC가 필요하지 않다면 아래 명령을 실행해야 합니다. 실행 시 아무런 메시지도 표시되지 않습니다.
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * 복원 방법은 아래와 같습니다. 실행 시 아무런 메시지도 표시되지 않습니다.
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * MINI Pad는 이미 보드 내장 UTOC가 장착되어 있습니다.
