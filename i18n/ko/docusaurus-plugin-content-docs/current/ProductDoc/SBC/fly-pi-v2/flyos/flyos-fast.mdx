## 시스템 소개

* FlyOS-Fast는 3D 프린팅을 위해 깊이 최적화된 가벼운 Linux 시스템으로, 부팅 속도가 빠르며 30초만에 시작됩니다. Fly 전체 시리즈의 상위 기기 플랫폼을 지원합니다.
* 커널 최적화로 Klipper의 실시간 응답 성능 향상
* 파일 시스템 최적화로 직접 전원이 꺼져도 시스템 손상을 방지
* Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen 등 서비스를 미리 설치
* Klipper와 RRF 시스템 간 스위치 지원
* mainsail과 fluidd 온라인 한 번 클릭으로 전환 가능
* U디스크 인쇄 지원
* KPPM 모듈과 함께 사용하면 전원 단절 후 재인쇄 가능
* Fluidd 등의 프론트엔드 UI 기본 언어 및 기본 프린터 이름 설정 가능
* 추가 OEM 맞춤 기능:开机 로고, 기본 설정 구성 파일 설정, Mainsail 탭 숨김 설정 등은 저희에게 문의하세요.

## 준비 작업

1. 시스템 이미지 다운로드
    - 이미지 링크: [Piv2 시스템 이미지 다운로드](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "클릭하여 바로 이동") (Windows 전용)  
    - 이미지 링크: [Piv2 시스템 이미지 다운로드](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "클릭하여 바로 이동") (Windows, MacOS, Linux 지원)

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. 브러싱 도구 다운로드, 여기서 예제로 `Rufus`를 사용할 것입니다. 실제로는 다른 도구, 예를 들어 `Etcher`, `Win32 Disk Imager` 등을 사용할 수 있습니다.
    - Rufus 다운로드 주소: [다운로드 (중국어)](https://rufus.ie/zh/#download) 또는 [Click Download English](https://rufus.ie/en/#download) (Windows 전용).
    - Etcher 다운로드 주소: [다운로드](https://etcher.balena.io/#download-etcher) (Windows, MacOS, Linux 지원).
    - Win32 Disk Imager 다운로드 주소: [다운로드](https://sourceforge.net/projects/win32diskimager/) (Windows 전용).

3. 예제로 `FlyOS-Fast-H5.exe`를 사용할 것입니다.

4. 시스템 이미지 파일 압축 해제, `.img` 파일 얻기.
    - `FlyOS-Fast-H5.exe`를 더블 클릭하거나 우클릭하여 열기, 몇 초 후 아래와 같은 창이 나타날 것입니다 (시스템에서 안전하지 않다고 경고하는 경우 "더 보기" -> "실행" 선택).
    - 압축 디렉토리를 선택한 후 `추출` 버튼을 클릭하고, 해제가 완료될 때까지 기다리세요.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - 압축이 완료되면 압축 디렉토리 아래에 `FlyOS-Fast-H5-vx.x.x.img` 파일이 생성됩니다. `x.x.x`는 시스템 버전 번호입니다.

            2. `Rufus`를 열고 장치를 선택합니다. 선택된 장치 표시 용량이 컴퓨터에 삽입된 브러싱 카드 용량과 일치해야 합니다.  
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="40%" align="center" />

            4. 이미지 파일 선택 후, `부트 유형 선택`이 이미지 파일 이름으로 변경됩니다.

            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />
            
            6. 브러싱이 완료되면 컴퓨터는 경고음을 내고 진행률 표시줄이 `준비 완료`로 바뀝니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="40%" align="center" />
            
            7. 브러싱이 완료되면 즉시 브러싱 카드를 빼지 마십시오.
            8. 파일 탐색기를 열고 `FlyOS-Conf`라는 이름의 U디스크 또는 이동식 디스크가 있는지 확인하십시오.

            9. `FlyOS-Conf`를 열고 다음 그림의 두 개의 파일이 존재한다면 브러싱이 성공한 것입니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />
            
            10. 그 중 `config.txt`는 시스템 구성 파일이며, 다음 페이지의 내용을 참조하여 수정하십시오.

## WiFi 설정

    1. `vscode` 또는 다른 전문 텍스트 편집기를 사용하여 `config.txt`를 엽니다 (**Windows 기본 메모장으로 구성 파일을 편집하지 마십시오**)
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

    2. `WIFI_SSID=""` 옵션의 `""`에 **WIFI 이름**을 입력합니다  
       `WIFI_PASS="""` 옵션의 `""`에 **WIFI 비밀번호**를 입력합니다  
       저장하고 종료한 후, `FlyOS-Conf`라는 이름의 U디스크 또는 이동식 디스크를 제거합니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/osfast_wifi.webp').default} size="70%" align="center" />  

## 상위 기기 시작

    1. 시작하기 전에 **메모리 카드** 또는 **M2WE**가 상위 기기에 올바르게 설치되었는지 확인하십시오
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/tf.webp').default} size="70%" align="center" />
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/m2we.webp').default} size="70%" align="center" />
  
    2. `M2WE` 또는 `2.4G WiFi` 모듈의 안테나를 설치합니다
    :::warning
    * FLY 상위 기기는 `M2WE` 또는 `2.4G WiFi` 모듈을 사용해야 하며, 그렇지 않으면 WiFi를 사용할 수 없습니다.
    * 주의하세요, 상위 기기는 중국어 이름의 WiFi를 지원하지 않으며, 2.4G와 5G를 합친 WiFi는 지원하지 않습니다.
    :::
    * 빨간 화살표가 가리키는 곳이 WiFi 안테나 접속부입니다
    * WiFi 안테나 포트는 `ipex1`입니다
    * PI V2에 안테나를 설치하지 않으면 신호가 약해질 수 있어 라우터에 연결되지 못할 수 있습니다
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" /> 

    3. 상위 기기에 전원을 공급합니다.  
    **여기까지, 상위 기기가 시작되었습니다!!**
