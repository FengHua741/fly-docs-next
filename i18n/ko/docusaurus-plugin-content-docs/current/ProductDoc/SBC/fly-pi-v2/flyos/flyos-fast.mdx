## 시스템 소개

* FlyOS-Fast는 3D 프린팅에 최적화된 가벼운 Linux 시스템으로, 부팅 속도가 빠르며 켜는 시간만 30초 밖에 걸리지 않습니다. FLY의 모든 시리즈 상위 기계 플랫폼을 지원합니다.
* 커널 최적화로 Klipper의 실시간 응답 성능을 향상시킵니다.
* 파일 시스템 최적화로 직접 전원이 꺼져도 시스템이 손상되지 않도록 합니다.
* Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen 등 다양한 서비스를 미리 설치합니다.
* Klipper와 RRF 시스템 전환을 지원합니다.
* Mainsail과 Fluidd 온라인 한 번의 클릭으로 전환 가능.
* USB 플래그로 인쇄 가능.
* KPPM 모듈과 함께 사용하면 전원이 끊겼을 때 다시 인쇄할 수 있습니다.
* Fluidd 등 프론트엔드 UI의 기본 언어, 기본 프린터 이름 설정 가능.
* 추가 OEM 맞춤 기능: 전원 로고 설정, 사전 설정된 구성 파일 설정, Mainsail 탭 숨기기 설정 등은 저희에게 문의하세요.

## 준비 작업

1. 시스템 이미지 다운로드
    - 이미지 링크: [Piv2 시스템 이미지 다운로드](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "클릭하여 바로 이동") (Windows 전용)  
    - 이미지 링크: [Piv2 시스템 이미지 다운로드](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "클릭하여 바로 이동") (Windows, MacOS, Linux 지원)

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. 번들 도구 다운로드, 여기서는 `Rufus` 예제를 사용할 것입니다. 실제로는 다른 도구도 사용 가능합니다, 예를 들어 `Etcher`, `Win32 Disk Imager` 등.
    - Rufus 다운로드 주소: [한글 다운로드](https://rufus.ie/zh/#download) 또는 [영문 다운로드](https://rufus.ie/en/#download) (Windows 전용).
    - Etcher 다운로드 주소: [다운로드](https://etcher.balena.io/#download-etcher) (Windows, MacOS, Linux).
    - Win32 Disk Imager 다운로드 주소: [다운로드](https://sourceforge.net/projects/win32diskimager/) (Windows 전용).

3. 튜토리얼에서는 `FlyOS-Fast-H5.exe`를 예제로 사용할 것입니다.

4. 시스템 이미지 파일 압축 해제, `.img` 파일 얻기.
    - `FlyOS-Fast-H5.exe`를 더블 클릭하거나 우클릭 후 열기를 선택하면 몇 초 후 아래와 같은 창이 나타납니다 (시스템에서 불안전하다고 경고하는 경우 "더 자세히 보기" -> "실행" 선택).
    - 압축 디렉토리를 선택한 후 `추출` 버튼을 클릭하고 해제 과정이 완료될 때까지 기다립니다.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - 해제 작업이 완료되면 압축 디렉토리에 `FlyOS-Fast-H5-vx.x.x.img` 파일이 생성됩니다. `x.x.x`는 시스템 버전 번호입니다.

5. 이미지 번들.

    <Tabs>
        <TabItem value="TF" label="TF 카드에 번들하기" default>

            1. - 번들용 TF 카드는 4GB보다 큰 용량이 필요하며, 추천하는 제품은 SanDisk 32GB C10급 이상의 TF 카드입니다 (저렴한 카드들은 많은 경우 번들 후 정상적으로 실행되지 않을 수 있음).
               - 번들용 TF 카드는 SD 카드 리더기를 사용해야 하며, 번들 카드를 리더기에 삽입한 후 컴퓨터에 리더기를 연결합니다.
            2. `Rufus`를 열고 장치를 선택합니다. 선택된 장치의 표시 용량이 컴퓨터에 삽입된 번들 카드 용량과 일치해야 합니다.  
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="40%" align="center" />
    
            3. 이미지 파일 선택, `유형 선택` 오른쪽의 `선택` 버튼을 클릭하여 압축해제된 `.img` 파일을 선택합니다.
            4. 이미지 파일을 선택한 후, `유형 선택`이 이미지 파일의 이름으로 바뀝니다.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="40%" align="center" />
            
            5. `시작` 버튼을 클릭하면, 데이터를 지우라는 경고 창이 나타나면 `확인`을 클릭합니다.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />
            
            6. 번들이 완료되면 컴퓨터가 경고음을 내며 진행률 표시줄이 `준비 완료`로 표시됩니다.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="40%" align="center" />
            
            7. 번들이 완료되면 바로 번들 카드를 빼지 마세요.
            8. 파일 탐색기를 열고 `FlyOS-Conf`라는 U디스크 또는 이동식 디스크가 존재하는지 확인합니다.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />
            
            9. `FlyOS-Conf`를 열면 아래 그림과 같은 두 개의 파일이 존재한다면 번들이 성공적으로 이루어졌습니다.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />
            
            10. `config.txt`는 시스템 구성 파일이며 다음 페이지의 내용을 참조하세요. 구성 파일 수정.

        </TabItem>
