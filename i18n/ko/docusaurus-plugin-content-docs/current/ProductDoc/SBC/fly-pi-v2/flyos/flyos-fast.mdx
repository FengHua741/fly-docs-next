## 시스템 소개

* FlyOS-Fast는 3D 프린팅을 위한 경량화된 Linux 시스템으로, 부팅 속도가 빠르며 30초만에 실행됩니다. 모든 Fly 시리즈의 상위기기 플랫폼을 지원합니다.
* 커널 최적화를 통해 Klipper의 실시간 응답 성능 향상
* 파일 시스템 최적화를 통해 갑작스러운 전원 차단으로 인한 시스템 손상을 방지
* Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen 등 서비스 미리 설치
* Klipper와 RRF 시스템 전환 지원
* Mainsail과 Fluidd 온라인 한 번의 클릭으로 전환 지원
* USB 플래시 프린팅 지원
* KPPM 모듈과 함께 사용하면 전원이 끊긴 후 다시 인쇄 가능
* Fluidd 등 프론트엔드 UI의 기본 언어 및 기본 프린터 이름 사전 설정 지원
* 추가 OEM 맞춤 기능: 로고 설정, 기본 구성 파일 설정, Mainsail 탭 숨김 설정 등은 저희에게 문의하세요.

## 준비 작업

1. 시스템 이미지 다운로드
    - 이미지 링크: [Piv2 시스템 이미지 다운로드](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "클릭하여 바로 이동") (Windows만 지원)  
    - 이미지 링크: [Piv2 시스템 이미지 다운로드](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "클릭하여 바로 이동") (Windows, MacOS, Linux 지원)

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. 플래시 도구 다운로드, 본 튜토리얼에서는 `Rufus`를 사용할 예정입니다. 실제로는 다른 도구를 사용할 수 있으며, 예를 들어 `Etcher`, `Win32 Disk Imager` 등.
    - Rufus 다운로드 주소: [다운로드하기 (중국어)](https://rufus.ie/zh/#download) 또는 [Click Download English](https://rufus.ie/en/#download) (Windows만 지원).
    - Etcher 다운로드 주소: [다운로드하기](https://etcher.balena.io/#download-etcher) (Windows, MacOS, Linux 지원).
    - Win32 Disk Imager 다운로드 주소: [다운로드하기](https://sourceforge.net/projects/win32diskimager/) (Windows만 지원).

3. 본 튜토리얼에서는 `FlyOS-Fast-H5.exe`를 사용할 예정입니다.

4. 시스템 이미지 파일 압축 해제, `.img` 파일 얻기.
    - `FlyOS-Fast-H5.exe`를 더블 클릭하거나 우클릭하여 열기, 몇 초 후 아래와 같은 창이 나타날 것입니다 (시스템에서 안전하지 않다고 표시되면 "더 자세히 보기" -> "실행" 선택).
    - 압축 디렉토리를 선택한 후 `Extract` 버튼을 클릭하고 해제가 완료될 때까지 기다립니다.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - 압축이 완료되면 압축 디렉토리 아래에 `FlyOS-Fast-H5-vx.x.x.img` 파일이 생성됩니다. `x.x.x`는 시스템 버전 번호입니다.

## 이미지 플래시

    <Tabs>
        <TabItem value="TF" label="TF 카드에 플래시" default>

            1. - 플래시용 TF 카드는 4GB보다 커야 하며, SanDisk 32GB C10 이상을 권장합니다 (저렴한 카드 중 많은 카드가 플래시 후 시작되지 않을 수 있음).
               - 플래시용 TF 카드는 SD 카드 리더기를 사용해야 하며, 플래시 카드를 리더기에 삽입한 후 컴퓨터에 리더기를 연결합니다.
            2. `Rufus`를 열고 장치를 선택합니다. 선택한 장치가 표시하는 용량이 컴퓨터에 연결된 플래시 카드 용량과 일치해야 합니다.  
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="40%" align="center" />
    
            3. 이미지 파일 선택, `GUID 타입 선택` 오른쪽의 `선택` 버튼을 클릭하여 압축해제된 `.img` 파일을 선택합니다.
            4. 이미지 파일을 선택하면 `GUID 타입 선택`이 이미지 파일의 이름으로 바뀝니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="40%" align="center" />
            
            5. `시작` 버튼을 클릭하면, 데이터를 지우라는 경고 창이 나오면 `확인`을 클릭합니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />
            
            6. 플래시가 완료되면 컴퓨터에서 소리가 나며 진행률 표시줄이 `준비 완료`로 표시됩니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="40%" align="center" />
            
            7. 플래시가 완료되었으나 즉시 플래시 카드를 빼지 마세요.
            8. 파일 탐색기를 열어 `FlyOS-Conf`라는 U디스크 또는 이동식 디스크가 있는지 확인합니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />
            
            9. `FlyOS-Conf`를 열고 다음 그림에 있는 두 개의 파일이 존재한다면 플래시가 성공한 것입니다.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />
            
            10. 그 중 `config.txt`는 시스템 구성 파일이며 다음 페이지의 내용을 참조하십시오. 구성 파일 수정.

## WiFi 연결

    1. `vscode` 또는 기타 전문 텍스트 편집기를 사용하여 `config.txt`를 엽니다 (**Windows 시스템의 기본 메모장을 사용하여 구성 파일을 편집하지 마십시오**)
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

    2. `WIFI_SSID=""` 옵션의 `""` 안에 **WiFi 네임**을 입력합니다.
       `WIFI_PASS="""` 옵션의 `""` 안에 **WiFi 비밀번호**를 입력합니다.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/osfast_wifi.webp').default} size="70%" align="center" />  
  
    3. 여기까지 왔다면, 상위기기는 이제 시작할 준비가 되었습니다!!
    다음 단계를 수행하세요: <Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh">SSH를 사용하여 상위기기 연결</Button>
