## 시스템 번錄 전 준비

    * 읽기 전용 카드 리더기
    * TF 카드 한 장, 용량은 반드시 16GB 이상 또는 128GB 이하, TF 카드의 전송 속도는 반드시 Class10급 또는 그 이상이어야 함
    * 대응하는 <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">시스템 이미지</Button> 다운로드
    * 필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">번錄 도구</Button> 다운로드

## 시스템 번錄

:::info
TF 카드 번錄과 TF 카드를 통해 M2WE에 번錄하는 것은 중에서 하나만 사용하면 됩니다.
:::

:::info
* 기기에 M2WE와 메모리 카드가 설치된 경우 시스템은 기본적으로 메모리 카드에서 부팅됩니다
* M2WE는 TF 카드를 사용하여 시스템을 번錄하는 것을 권장합니다. USB를 통해 M2WE에 번錄하는 경우 드라이버 호환성 등으로 인해 번錄 실패할 수 있으므로 [메모리 카드를 사용한 M2WE 번錄](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 직접 권장합니다
* 다운받은 시스템 이미지가 일치하는지 확인하시기 바랍니다. 일치하지 않으면 부팅할 수 없습니다
:::
