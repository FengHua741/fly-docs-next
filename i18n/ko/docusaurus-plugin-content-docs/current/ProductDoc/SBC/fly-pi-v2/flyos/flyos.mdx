## 시스템 번录用 준비

    * 읽기 전용 카드 리더기
    * TF 카드 한 장, 용량은 16GB 이상 또는 128GB 이하, TF 카드의 전송 속도는 Class10급 또는 그 이상이어야 함
    * 대응되는 <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">시스템 이미지</Button> 다운로드
    * 필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">번录用 도구</Button> 다운로드

## 시스템 번录用

:::info
TF 카드 번录用 M2WE에 TF 카드를 통해 번录用 중 하나만 사용하면 됩니다.
:::

:::info
* 기계에 M2WE와 메모리 카드가 설치된 경우 시스템은 기본적으로 메모리 카드에서 부팅합니다.
* M2WE는 TF 카드를 사용해 시스템을 번录用 것을 권장하며 USB를 통해 번录用 경우 드라이버 호환성 등으로 인해 번录用 실패할 수 있으므로 직접 [메모리 카드를 사용해 M2WE에 번录用](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)하는 것을 권장합니다.
* 다운받은 시스템 이미지가 맞는지 확인해야 합니다. 맞지 않으면 부팅되지 않습니다.
:::
