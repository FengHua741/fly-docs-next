## 시스템 번录用 준비사항

- 읽기 전용 카드 리더기
- 용량이 최소 16GB 이상 또는 최대 128GB 이하의 TF 카드, TF 카드의 전송 속도는 반드시 Class10급 또는 그 이상이어야 합니다.
- 대응하는 시스템 이미지를 다운로드합니다. <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">시스템 이미지</Button>
- 필요한 번录用 도구를 다운로드합니다. <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">번录用 도구</Button>

## 시스템 번录用

:::info
TF 카드 번录用과 TF 카드에서 M2WE로 번录用은 중 하나만 사용하면 됩니다.
:::

:::info
* 기기에 M2WE와 메모리 카드가 설치된 경우 시스템은 기본적으로 메모리 카드에서 부팅합니다.
* M2WE는 TF 카드에 시스템을 번录用 것을 권장합니다. USB를 통해 M2WE에 번录用 시 드라이버 호환성 등으로 인해 번录用이 실패할 수 있으므로, [메모리 카드에서 M2WE로 번录用](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 직접 사용하는 것을 권장합니다.
* 다운받은 시스템 이미지가 일치하는지 확인하시기 바랍니다. 그렇지 않으면 시스템이 시작되지 않습니다.
:::
