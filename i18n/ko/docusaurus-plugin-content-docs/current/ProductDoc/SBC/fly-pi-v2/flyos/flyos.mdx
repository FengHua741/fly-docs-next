## 시스템 번录用

:::info
TF 카드 번录用과 TF 카드를 통해 M2WE에 번录用 중 하나만 사용하면 됩니다.
:::

:::info
* 기기가 M2WE와 메모리 카드를 장착한 경우 시스템은 기본적으로 메모리 카드에서 부팅됩니다.
* M2WE는 TF 카드를 사용해 시스템을 번录用 것을 권장합니다. USB를 통한 번录用은 드라이버 호환성 등으로 인해 실패할 수 있으므로 직접 [메모리 카드를 이용한 M2WE 번录用](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 추천합니다.
* 다운로드한 시스템 이미지가 맞는지 확인하세요. 맞지 않는 경우 부팅이 불가능합니다.
:::

## 안테나 설치

    1. `M2WE` 또는 `2.4G WiFi` 모듈의 안테나를 설치합니다.
    :::warning
    * FLY 상위기기는 `M2WE` 또는 `2.4G WiFi` 모듈을 사용해야 하며, 그렇지 않으면 WiFi를 사용할 수 없습니다.
    * 주의하세요. 상위기기는 한글 이름의 WiFi를 지원하지 않으며, 2.4G와 5G의 단일화된 WiFi를 지원하지 않습니다.
    ::
    * 빨간 화살표가 가리키는 부분이 WiFi 안테나 커넥터입니다.
    * WiFi 안테나 커넥터는 `ipex1`입니다.
    * 안테나를 M2WE에 설치하지 않으면 신호가 약해져 라우터에 연결되지 않을 가능성이 있습니다.
