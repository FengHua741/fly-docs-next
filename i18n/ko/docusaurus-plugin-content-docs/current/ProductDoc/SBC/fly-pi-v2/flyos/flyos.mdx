## 시스템 번들

:::info
TF 카드 번들 또는 TF 카드를 통해 M2WE에 번들하는 것은 둘 중 하나만 사용하면 됩니다.
:::

:::info
* 기기에 M2WE와 메모리 카드가 설치된 경우 시스템은 기본적으로 메모리 카드에서 부팅합니다
* M2WE는 TF 카드를 사용하여 시스템을 번들하는 것을 권장합니다. USB를 통한 번들은 드라이버 호환성 등으로 인해 실패할 수 있으므로 [메모리 카드를 사용한 M2WE 번들](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 직접 사용하는 것이 좋습니다
* 다운받은 시스템 이미지가 일치하는지 확인하시기 바랍니다. 일치하지 않으면 부팅되지 않습니다
:::

## 안테나 설치

    1. `M2WE` 또는 `2.4G WiFi` 모듈의 안테나 설치
    :::warning
    * FLY 상위기기는 `M2WE` 또는 `2.4G WiFi` 모듈을 사용해야 하며, 그렇지 않으면 WiFi를 사용할 수 없습니다
    * 상위기기는 한글 네트워크 이름(WIFI)을 지원하지 않으며, 2.4G와 5G의 합쳐진 네트워크도 지원하지 않습니다
    :::
    * 빨간 화살표가 가리키는 곳이 WIFI 안테나 커넥터입니다
    * WIFI 안테나 포트는 `ipex1`입니다
    * 안테나를 M2WE에 설치하지 않으면 신호가 약해져 라우터에 연결할 수 없을 가능성이 있습니다
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" />
