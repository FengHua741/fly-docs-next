# Klipper 전원이 끊겼다가 다시 연결되는 기능

## 개요

- Klipper 전원이 끊겼다가 다시 연결될 때, Klipper의 인쇄 상태를 자동으로 복구할 수 있는 기능입니다.
- 이 기능은 FAST 시스템과 함께 사용해야 합니다.
- PI-V2는 KPPM 전원 모듈과 함께 사용해야만 이 기능을 사용할 수 있습니다.
- Z축이 이동하는 전원이 끊겼을 경우는 지원되지 않습니다.
