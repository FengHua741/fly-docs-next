# 상위기기 SSH 연결하기

* **시리얼 포트** 또는 **WiFi**를 사용하여 장치에 연결할 수 있습니다.
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해 주세요.
* **SD 카드** 또는 **M2WE**가 상위기기에 맞는 시스템으로 이미 번들되었는지 확인하십시오.
* 만약 상위기기가 정상적으로 시작되지 않는다면, 드라이버, 리미터, 팬 등 외부 장치들을 제거하세요!!!

:::warning 주의
- 만약 상위기기가 처음 실행 중이라면 반드시 **시리얼 포트를 통해 상위기기 SSH에 연결**해야 합니다.
- 펌웨어를 빌드할 때는 반드시 **WiFi를 통해 상위기기 SSH에 연결**하여 펌웨어 빌드와 플래싱이 더 쉬워집니다!
- `FLYOS-Armbian` 시스템과 `FLYOS-FAST` 시스템은 SSH 연결 방법이 다릅니다. 각각의 시스템에 맞는 가이드를 선택하시오!!!
:::

        ## FLYOS-Armbian 시리얼 포트를 통한 상위기기 SSH 연결

        * FLY 상위기기를 컴퓨터에 USB Type-C 케이블로 연결합니다.
        * 다른 상위기기는 연결 방법을 개별적으로 확인하시오!!

        * FLY-Pi v2 포트
                <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/piv2-port.webp').default} size="30%" align="center" />
        * 메인보드와 컴퓨터가 정상적으로 연결된 경우, 장치 관리자에서 CH340 포트를 볼 수 있습니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/pc-device-mgr.webp').default} size="50%" align="center" />

        * `MobaXterm`을 열고, 앞서 장치 관리자에서 본 포트를 선택하도록 설정하십시오.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-2.webp').default} size="60%" align="center" />

        * 모든 것이 정상이라면 부팅 화면을 볼 수 있습니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-3.webp').default} size="60%" align="center" />

        * 잠시 기다리면 이 화면이 나타나며 부팅이 완료됩니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="60%" align="center" />

        * 몇 분이 지나도 아래 그림처럼 아무런 표시가 없으면, Enter 키를 몇 번 누르세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-5.webp').default} size="60%" align="center" />

        * Enter 후 아래 그림과 같은 내용이 나타납니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-6.webp').default} size="60%" align="center" />

        :::warning
        - 현재 로그인 계정은 `root`입니다. `su fly`를 입력하여 일반 사용자로 전환한 후에 다음 작업을 진행하시오!!!
        :::
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-13.webp').default} size="60%" align="center" />

        * IP를 확인하려면 SSH에서 `ip a | grep inet`을 입력하고 Enter를 누릅니다. 반환된 내용에서 자신의 라우터와 같은 구간의 IP를 찾으면 접속 가능합니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-8.webp').default} size="70%" align="center" />

    </TabItem>

        ## FLYOS-Armbian WiFi를 통한 상위기기 SSH 연결

        * 상위기기가 정상적으로 시작되었는지 확인하시오.
        * 상위기기의 사용자 이름과 비밀번호를 명확히 알고 있으셔야 합니다. FLY 시스템의 기본 사용자는 `fly`, 기본 비밀번호는 `mellow`입니다.
        * WiFi를 통해 상위기기 SSH에 연결하기 위한 전제 조건은 상위기가 라우터에 연결되어 있으며, 상위기기의 IP를 알고 있는 것입니다.

        * MobaXterm을 열고 아래와 같이 설정합니다. `Remote host` 항목에는 시리얼 포트 연결 시 확인한 상위기기의 IP를 입력하시거나, 라우터 관리자 페이지에서 상위기기의 IP를 확인하시오.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **주의: 여기서 4번째 단계에서 상위기기의 사용자를 입력하시오**
        :::tip
        - 4번째 단계에서는 상위기기의 사용자를 입력하십시오. FLYOS-Armbian 시스템을 사용하는 fly 상위기기라면 `fly`를 입력하시오, 다른 상위기기는 상황에 맞게 적절한 사용자를 입력하시오.
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * 첫 연결 시 아래 창이 나타날 수 있습니다. `Accept`를 클릭하시오.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * 비밀번호를 입력하라는 메시지가 나타나면 상위기기의 비밀번호를 입력하고 Enter를 누르시오.
        :::tip
        - 비밀번호 입력 시 보이지 않을 수 있습니다. 입력 후 바로 Enter를 누르시오.
        :::
        
        **FLYOS-Armbian** 시스템의 기본 사용자는: `fly`, 기본 비밀번호는: `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        * 아래와 같은 창이 나타나면 이는 사용자 비밀번호를 저장할지 묻는 팝업창입니다. `No`를 클릭하시오.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * 아래와 같은 창이 나타나면 로그인 성공입니다!

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />
