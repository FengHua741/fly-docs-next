# 상위 컴퓨터에 SSH 연결


* 시리얼 포트 또는 WIFI를 통해 장치에 연결할 수 있습니다
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해 주세요
* SD 카드 또는 M2WE에 **해당 상위 컴퓨터의 시스템을 번인**했는지 확인하세요
* **상위 컴퓨터가 정상적으로 부팅되지 않으면 드라이버, 리미터, 팬 등의 주변 장치를 제거하세요!!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 편리하게 컴파일할 수 있습니다!!**
* **펌웨어를 컴파일하려면 WIFI 또는 이더넷 연결을 통해 상위 컴퓨터에 연결하는 튜토리얼을 참조하세요**

        ## 네트워크 IP를 통해 상위 컴퓨터에 SSH 연결

        * 상위 컴퓨터가 연결된 **라우터 백엔드**에 로그인하세요
        * 시스템이 **정상적으로 부팅**되었는지 확인하세요
        * **상위 컴퓨터의 사용자 이름과 비밀번호를 알고 있는지** 확인하세요. FLY 시스템의 기본 사용자 이름은 **fly**, 기본 비밀번호는 **mellow**입니다
        * 메인보드가 부팅되고 네트워크에 연결되어 있으면 IP를 사용하여 SSH에 연결할 수 있습니다
        * MobaXterm을 열고 다음과 같이 설정하세요. IP는 이전에 확인한 IP를 입력하거나 라우터 백엔드에서 확인할 수 있습니다

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계에 상위 컴퓨터의 사용자 이름을 입력합니다**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 처음 연결하면 이 창이 뜹니다. `Accept`를 클릭하세요

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호를 입력하라는 메시지가 나오면 상위 컴퓨터의 비밀번호를 입력하고 엔터를 누르세요

        **주의: 여기서 입력한 비밀번호는 보이지 않습니다. 입력 후 직접 엔터를 누르세요**

        FLY 시스템의 기본 사용자 이름은: `fly`, 비밀번호는: `mellow`입니다
