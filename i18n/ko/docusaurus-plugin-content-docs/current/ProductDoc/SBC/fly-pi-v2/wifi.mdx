# 와이파이 연결
:::tip
- `FLYOS-FAST` 시스템과 `FLYOS-Armbian` 시스템의 와이파이 연결 방법은 약간 다릅니다. 사용 중인 시스템에 맞는 가이드를 선택하세요!
:::

:::warning 경고
    * FLY 상위기기는 `M2WE` 모듈 또는 `2.4G 와이파이` 모듈 또는 `Linux 드라이버가 필요하지 않은 USB 무선 네트워크 카드`를 사용해야 합니다. 그렇지 않으면 와이파이를 사용할 수 없습니다.
    * 주의: 상위기기는 한글 이름으로 된 `와이파이`를 지원하지 않습니다. 또한 `2.4G와 5G 통합`된 와이파이는 지원되지 않습니다.
    * 만약 `mini pad`나 `lite2/lite2.1`을 사용 중이라면 이 경고를 무시하십시오.
::>
