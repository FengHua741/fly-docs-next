# 와이파이 연결
:::tip
- `FLYOS-FAST` 시스템과 `FLYOS-Armbian` 시스템의 와이파이 연결 방법에는 약간의 차이가 있으니, 사용 중인 시스템에 맞는 가이드를 선택하세요!
:::

:::warning 경고
    * FLY 상위기기는 `M2WE` 모듈 또는 `2.4G 와이파이` 모듈 또는 `Linux 드라이버가 없는 USB 무선 네트워크 카드`를 사용해야 와이파이를 사용할 수 있습니다.
    * 주의하세요! 상위기기는 한글 이름의 `와이파이`를 지원하지 않으며, `2.4G와 5G 통합형 와이파이`도 지원하지 않습니다.
    * 만약 `mini pad`나 `lite2/lite2.1`을 사용 중이라면 이 경고를 무시해도 됩니다.
:::
