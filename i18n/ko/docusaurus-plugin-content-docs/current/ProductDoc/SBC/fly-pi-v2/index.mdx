# 소개

FLY-Pi V2 상위기기를 발표하게 되어 기쁩니다. 이 보드는 라즈베리 파이와 동일한 설치 크기를 가지고 있으며, 라즈베리 파이를 대체하여 상위기기로 작동할 수 있습니다. 1GB의 내장 DDR3 메모리를 갖추고 있어 전체 LINUX 데스크톱(아ーム비안을 기반으로 커스텀 최적화)을 실행할 수 있습니다. 이 보드는 Klipper 및 RRF 펌웨어와 함께 사용할 수 있으며, 심지어 SHT 도구 보드를 사용할 때 Klipper 화면을 실행할 수도 있습니다.
이 상위기기를 디자인한 목적은 Klipper 펌웨어 또는 RRF 펌웨어를 사용하는 것이 더 쉬워지는 것입니다. 우리는 LINUX 시스템 이미지를 최적화하여 3D 프린터에 더 적합하게 만들었으며, 설치 및 운영이 더 간단해졌습니다. 초보자라도 약간의 기초 지식만 있으면 빠르게 학습하고 사용할 수 있습니다. 다른 상위기기 보드보다 학습 비용을 절약하여 3D 프린팅을 즐기는 데 더 집중할 수 있습니다(예상치 않은 SD 카드가 있는 패키지를 구매하면 Klipper 펌웨어가 미리 설치되어 있어 전원을 켜면 바로 사용 가능합니다).

## 특징

* 라즈베리 파이 3B와 4B와 같은 보드 크기
* CPU: 고성능 올티치 H5 칩셋, 4코어 64비트 Cortex-A53
* GPU: 고성능 6코어 Mali 450, 픽셀 채우기 속도 2.7G픽셀/초 초과
* RAM: 1GB DDR3 (GPU와 공유) (512+512)
* ROM: 최대 128GB microSD 카드 지원
* 외부 장치: SPI x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), 마이크로 HDMI x1, Eth x1 (100M), CAN (보드 내장 USB to CAN 모듈, 즉 보드 내장 UTOC)
* 인터페이스: 보드 내장 M.2 인터페이스, Wi-Fi, eMMC 등 확장 가능 (프라이빗 프로토콜, FLY π 전용 M.2 장치가 아닌 것은 연결하지 마십시오)
* 40핀 인터페이스, 라즈베리 파이 호환
* 12-24V DC 전원 공급, 전력 공급 안정화
* 5V 팬 ZH1.5 인터페이스 예비
* 시스템은 Klipper, Moonraker, Mainsail, Fulld, Klipper-Screen, Crowsmest, 공명 보상 플러그인, numpy 및 FLY-Tools가 미리 설치되어 있어 복잡한 작업 없이 사용하기 쉽습니다 (TF 카드 또는 M2WE를 구매해야 함)
