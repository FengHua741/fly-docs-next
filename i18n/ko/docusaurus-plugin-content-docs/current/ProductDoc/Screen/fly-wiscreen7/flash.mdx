# 부팅 화면 이미지 수정
:::tip
    해상도와 파일 이름에 맞춰서 이미지를 제작해 주세요. 그렇지 않으면 시스템이 인식하지 못할 수 있습니다.

    제작한 이미지를 SD 카드/USB 드라이브 루트 디렉토리에 넣고, 화면이 켜진 상태에서 스크린을 삽입하고 업그레이드를 클릭하면 됩니다.
:::
## 부팅 로고 수정

    * 이미지 요구사항 `1024*600`
    * 이름 요구사항 `boot_logo.JPG`
    * 105KB 이하

## 첫 번째 화면 이미지 수정

    * 이미지 요구사항 `736*563`
    * 이름 요구사항 `print.png`
    * 45KB 이하

## 프린팅 화면 이미지 수정

    * 이미지 요구사항 `355*565`
    * 이름 요구사항 `print_smail.png`
    * 45KB 이하
