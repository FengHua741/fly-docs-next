##   업데이트
    * 1.다운로드한 파일을 `update.img`으로 이름을 변경합니다.
    * 2.`update.img` 파일을 SD 카드 또는 USB 드라이브에 넣고 화면에 삽입할 수 있습니다. 화면이 켜져 있는 상태에서도 삽입할 수 있습니다.
    * 3.화면을 체크하고 업그레이드를 클릭하여 업데이트를 완료합니다.
