## 버튼 기능 설정
    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide10.gif').default} size="100%" align="left" />

* 함수를 사용하여 단일 Gcode 명령을 전송합니다.  
    ```cfg
    gcode_send2("","G28");
    ```
* 함수를 사용하여 여러 개의 Gcode 명령을 한 번에 전송합니다:
    ```cfg
    gcode_send2("","G91\\nG1 X10 F3000\\nG90");
    ```
* 또는 다음과 같은 방법으로 여러 개의 Gcode 명령을 한 번에 전송할 수 있습니다:
    ```cfg
    gcode_send2("","G91");
    gcode_send2("","G1 X10 F3000");
    gcode_send2("","G90");
    ```
