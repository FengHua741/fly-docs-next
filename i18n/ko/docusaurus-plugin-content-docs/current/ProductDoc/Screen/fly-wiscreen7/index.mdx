# 소개

      Fly_WiScreen은 광저우 맘런 전자 기술 유한 회사의 Fly3D 보드 연구 개발 팀이 출시한 고급 Klipper 제어 화면으로, Wi-Fi 및 유선 네트워크 두 가지 모드로 3D 프린터를 제어할 수 있으며, SD 카드 및 USB 플래시 드라이브를 통한 인쇄 파일 전송을 지원합니다. 화면 반응 속도가 매우 빠르기 때문에 사용자에게 매우 좋은 경험을 제공합니다. 메인/슬레이브 모드 설정을 지원하여 다중 기계 동기 작업을 수행하여 사용자가 기계를 관리하는 것이 더 쉽습니다. 그 중 5V 전원 단자는 충전 모듈을 연결하여 배터리로 변경하여 이동성을 편리하게 할 수 있습니다.

## 제품 기능

    * Wi-Fi 및 유선 네트워크를 통해 다른 Klipper가 설치된 상위 컴퓨터와 연결을 지원합니다.
    * Wi-Fi 검색을 지원하며, Wi-Fi에 연결한 후에는 자체적으로 상위 컴퓨터를 검색할 수 있습니다.
    * 클러스터 제어를 통해 여러 대의 프린터를 제어할 수 있습니다.
    * USB 및 TF 카드를 사용하여 업로드 및 인쇄를 지원합니다.
    * 사용자 정의 매크로로 조작 및 디버깅이 더 쉬워집니다.

## 제품 사양

    * CPU: Allwinner T113-S3, 듀얼 코어 CORTEX-A7 최대 1.2GHz
    * 메모리: 통합 128M DDR3
    * 저장소: 128MB Nand Flash, TF 카드 슬롯
    * 디스플레이: 1024X800 캡acic touchscreen
    * 외부 인터페이스: USB 2.0 포트 하나, TF 카드 슬롯 하나
    * Wi-Fi 프로토콜: 802.11b/g/n
    * 화면 유형: IPS
    * 화면 크기: 7인치
    * 멀티 터치: 지원
    * 정격 전압: 5V
