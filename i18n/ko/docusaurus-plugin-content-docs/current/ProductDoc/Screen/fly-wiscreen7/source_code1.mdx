# 프로젝트 구조 소개


    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide6.webp').default} size="80%" align="left" />



## 소개
#### Logic 파일夹
* 대부분의 ui 코드는 logic 파일夹에서 작성됩니다. 다른 파일夹은 신경 쓸 필요가 없습니다.
* gcodeLogic.cc는 main.c와 비슷하며, 대부분의 ui 코드는 여기에 있습니다. 다른 파일은 별로 신경 쓸 필요가 없습니다.

#### i18n 파일夹
* 각국 언어 번역 테이블이 저장되어 있습니다.

#### resources 파일夹
* ui에서 사용하는 이미지 리소스가 저장되어 있습니다.

#### ui 파일夹
* ui 인터페이스 파일이 저장되어 있습니다. 더블 클릭하여 열면 표시됩니다.
* 왼쪽 하단 화살표가 가리키는 "작은 눈"을 더블 클릭하면 숨겨진 인터페이스를 열 수 있습니다.
