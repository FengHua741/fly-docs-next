# 시작 화면 이미지 수정
:::tip
    `해상도` 및 `파일 이름`에 맞춰서 이미지를 제작해 주세요. 그렇지 않으면 시스템이 이미지를 인식하지 못할 수 있습니다.

    제작한 이미지는 SD 카드/USB 드라이브 루트 디렉토리에 넣어야 합니다. 화면이 켜진 상태에서 화면을 삽입하고 업그레이드를 클릭하면 됩니다.
:::
## 시작 로고 수정

    * 이미지 요구사항: `800*480`
    * 파일 이름 요구사항: `boot_logo.JPG`
    * 크기 제한: `105KB` 이하




## 첫 번째 화면 이미지 수정

    * 이미지 요구사항: `500*400`
    * 파일 이름 요구사항: `print.png`
    * 크기 제한: `45KB` 이하



## 프린트 화면 이미지 수정

    * 이미지 요구사항: `311*480`
    * 파일 이름 요구사항: `print_smail.png`
    * 크기 제한: `45KB` 이하
