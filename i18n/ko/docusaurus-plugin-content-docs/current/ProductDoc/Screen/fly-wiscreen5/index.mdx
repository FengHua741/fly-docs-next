# 소개

      Fly_WiScreen는 광저우 멍런 전자 과학 기술 유한 회사의 Fly3D 메인보드 연구 개발 팀이 출시한 고급 Klipper 제어 화면으로, Wi-Fi를 통해 3D 프린터를 제어할 수 있습니다. SD 카드와 USB 플래시 드라이브를 통한 프린팅 파일 전송을 지원하며, 화면 반응 속도가 매우 빠르기 때문에 사용자에게 매우 좋은 경험을 제공합니다. 주-부 기기 모드 설정을 지원하여 다중 기기 동기화 작업을 수행하여 사용자가 기기를 관리하는 것이 더 쉽습니다. 그 중 5V 전원 단자는 배터리 수정을 위한 충전 모듈을 연결할 수 있어 이동식으로 사용하기에 편리합니다.

## 제품 기능

    * Wi-Fi 연결을 통해 다른 Klipper 설치된 상위 기기와 연결을 지원합니다.
    * Wi-Fi 검색을 지원하며, Wi-Fi에 연결하면 자체적으로 상위 기기를 검색할 수 있습니다.
    * 클러스터 제어를 통해 여러 대의 프린터를 제어할 수 있습니다.
    * USB 및 TF 카드를 사용하여 업로드 프린팅을 지원합니다.
    * 사용자 정의 매크로를 지원하여 조작 및 디버깅이 더 쉬워집니다.

## 제품 사양

    * CPU: 올치프 T113-S3, 듀얼 코어 CORTEX-A7 최대 1.2GHZ
    * 메모리: 통합 128M DDR3
    * 저장소: 128MB 내장 Nand Flash, TF 카드 슬롯
    * 디스플레이: 800X480 터치 스크린
    * 외부 인터페이스: USB 2.0 단자 1개, TF 카드 슬롯 1개
    * Wi-Fi 프로토콜: 802.11b/g/n
    * 화면 유형: IPS
    * 화면 크기: 5인치
    * 멀티 터치: 지원
    * 정격 전압: 5V
