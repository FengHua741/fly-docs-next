# 소개

* 이 장치는 고유한 FPC(플렉시블 프린트 회로) 포트 설계를 채택하여 Fly 디자인의 메인 보드에 직접 연결할 수 있습니다. HDMI 및 USB 케이블을 사용하지 않고도 가능합니다. 이 단순화된 FPC 연결은 케이블 혼돈을 줄이고 깨끗하고 효율적인 인터페이스를 제공합니다. 이러한 고급 연결 방법에도 불구하고, 이 장치는 여전히 전통적인 연결 옵션을 선호하는 사용자를 위해 표준 HDMI 포트를 제공합니다.
  
  유연한 연결성 외에도, 이 장치에는 밝기 조절, 화면 회전 및 재부팅이 간편한 스마트 기능 키가 포함되어 있습니다. 고급 수입 폼 접착제는 강력한 화면 접착력과 뛰어난 내구성을 보장합니다. 양면 저저항 열방사 기술은 터치 민감성을 향상시키고 사용자 경험을 더 매끄럽게 합니다. 노란색 광기술은 정확한 색상 재현을 제공하며 표준 밝기는 500으로 과도한 노출을 방지합니다. 화면은 내구성이 뛰어난 고강도 강화 유리를 통해 보호되며, 견고한 손상 방지 보호를 제공합니다.
  
  이 장치는 Raspberry Pi와 같은 호스트 시스템과 완벽하게 호환되며, 깔끔하고 전문적인 외관을 위해 숨겨진 HDMI 포트 설계를 채택하여 케이블이 장치 후면에 깔끔하게 숨겨집니다. 그 높은 성능 회로 설계는 원활한 작동을 보장하며, 이는 안정적이고 신뢰할 수 있는 3D 프린터에 이상적입니다.
