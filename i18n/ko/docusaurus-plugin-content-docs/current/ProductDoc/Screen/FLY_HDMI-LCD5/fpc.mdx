# 화면 매개변수 수정 튜토리얼
:::tip
브라우저에서 IP:9999를 열어 FLY-Tools에 접속하세요
:::

:::tip
FLY 상위기기의 HDMI 포트는 Micro HDMI입니다. HDMI 케이블을 구매할 때 주의하세요!!!
:::

:::tip
FLY의 상위기기는 `FLY-PI-V2`, `FLY-C8` 및 `FLY-PRO-X10` 또는 `FLY_LITE2/2.1`에서 FPC 일선통을 지원합니다.

:::
    * HDMI를 사용하려면 아래 옵션을 활성화해야 화면을 사용할 수 있습니다.
    * FLY Tools에 진입한 후
* 첫 번째 단계는 fly_TOOLS에서 매개변수를 수정하는 것입니다.
