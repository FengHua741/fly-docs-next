# 화면 매개변수 수정 가이드
:::tip
브라우저에서 IP:9999를 열어 FLY-Tools에 접속하세요
:::

:::tip
FLY 상위 기기의 HDMI 인터페이스는 Micro HDMI이므로 HDMI 케이블을 구매할 때 주의하세요!!!
:::

:::tip
FLY의 상위 기기는 FPC 일선통을 지원하는 `FLY-PI-V2`,`FLY-C8`와 `FLY-PRO-X10` 또는 `FLY_LITE2/2.1`

:::
    * HDMI를 사용하려면 아래 옵션을 켜야 화면이 활성화됩니다
    * FLY Tools에 들어가면
* 첫 번째 단계는 fly_TOOLS에서 매개변수를 수정하는 것입니다
