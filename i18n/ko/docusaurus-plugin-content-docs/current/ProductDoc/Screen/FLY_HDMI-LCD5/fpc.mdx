# 화면 매개변수 수정 가이드
:::tip
브라우저에서 IP:9999을 열어 FLY-Tools에 액세스하세요.
:::

:::tip
FLY 상위기판에 내장된 HDMI 포트는 마이크로 HDMI입니다. HDMI 케이블 구매 시 유의하시기 바랍니다!!!
:::

:::tip
FLY의 상위기판은 `FLY-PI-V2`, `FLY-C8` 및 `FLY-PRO-X10` 또는 `FLY_LITE2/2.1`과 호환됩니다.

:::
    * HDMI를 사용하려면 아래 항목을 활성화해야 화면을 사용할 수 있습니다.
    * FLY Tools에 진입한 후
* 첫 번째 단계는 fly_TOOLS에서 매개변수를 수정해야 합니다.
