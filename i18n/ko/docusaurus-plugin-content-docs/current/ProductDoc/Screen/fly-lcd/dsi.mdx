# DSI 사용

    :::팁

    이 튜토리얼은 라즈베리파이 4B를 예로 들어 설명합니다.

    :::

    :::경고 주의사항

    * 라즈베리파이 시스템은 `bookworm` 버전을 사용하는 것이 좋습니다.

    :::
