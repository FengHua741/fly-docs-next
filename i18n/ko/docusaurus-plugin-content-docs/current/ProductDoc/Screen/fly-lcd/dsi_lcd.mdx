# DSI-LCD5

    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-lcd/img/dsi_lcd5.webp').default} size="50%" align="left" />

## 제품 소개

    * 화면 크기：120.7mm*76.0mm
    * 화면 표시 영역：108mm*66.2mm
    * 해상도：800*480
    * 화면 타입：IPS
    * 통신 방식：DSI/Mipi
    * 내장 인터페이스：15 핀 DSI / 22핀 Mipi
    * 작동 안정성：0°-60°C
    * 터치 타입：Capacitive（전차식）
    * 논리 전압：3.3V
    * 무게：135.5g

        <TabItem value="라즈베리 파이 5B는 Mipi 인터페이스를 사용" default>

        :::danger 주의사항

        * 라즈베리 파이 5의 DSI와 CSI 인터페이스는 이제 MIPI로 이름이 변경되었습니다.
        * MIPI 인터페이스는 구성에 따라 CSI 카메라 또는 DSI 화면을 직접 사용할 수 있습니다.
        * 금 손가락 방향에 유의하십시오.

        :::

        * 라즈베리 파이 5는 두 개의 MIPI 인터페이스를 가지고 있습니다. 정면에서 보면 위쪽에 1과 0이 표시되어 있습니다.

        * DSI 화면 배선


            * 1번 인터페이스를 사용하는 경우 아래의 화면 구성이 필요합니다.

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * 0번 인터페이스를 사용하는 경우 아래의 화면 구성이 필요합니다.

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
            * 참조 배선
