# 화면 매개변수 수정 가이드
:::tip
브라우저를 열고 IP:9999로 FLY-Tools에 접속하세요.
:::

:::
    * tft를 사용하려면 아래 옵션을 활성화해야 화면이 작동합니다.
    * FLY Tools에 들어간 후
* 첫 번째 단계는 fly_TOOLS에서 매개변수를 수정하는 것입니다.
