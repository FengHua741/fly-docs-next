# BL 펌웨어

    :::tip

    * `부트로더`는 이미 출하 시 미리 프로그램되어 있으므로 다시 프로그래밍할 필요가 없습니다.

    :::

## 플래싱 모드 진입


    * 플래싱 모드 진입 방법은 두 가지입니다.
    * 첫 번째 방법: 메인보드에 완전히 전원을 차단한 상태에서, BOOT를 단락시키고 데이터 전송용 케이블을 메인보드와 장치 사이에 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블을 메인보드와 장치 사이에 연결한 후, BOOT를 단락시키고 RST 버튼을 누른 다음 RST 버튼을 놓아 마지막으로 BOOT 단락을 해제합니다.

            4. 펌웨어 플래싱

            :::tip

            하나의 펌웨어만 플래싱하면 됩니다.

            :::


            * 아래는 katapult 펌웨어 플래싱 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * 아래는 부트로더 펌웨어 플래싱 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            5. 아래 화살표 표시된 내용 ``File downloaded successfully``이 나타나면 플래싱이 성공했습니다.

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::tip

            플래싱이 완료되면 메인보드를 완전히 전원 차단해야 합니다.
