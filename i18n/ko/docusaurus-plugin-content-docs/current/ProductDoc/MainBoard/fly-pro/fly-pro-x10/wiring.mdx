# 메인보드 배선

:::warning 주의사항
  이미지 해상도가 높아 웹 페이지에서 볼 수 없을 수 있습니다. 이미지를 확대하려면 마우스 커서를 배선도 위에 올리고 우클릭하여 '이미지 저장하기'를 선택하여 데스크톱에 이미지를 저장한 후 확인하세요!!!
:::
