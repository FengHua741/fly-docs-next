# 메인보드 케이블 연결

:::warning 주의사항
  이미지 해상도가 높아 웹 페이지에서 볼 때 명확하게 확인할 수 없습니다. 마우스 커서를 연결図 위에 올려놓고 우클릭하여 '이미지 저장하기'를 선택하여 데스크톱에 이미지를 저장한 후 확인해주세요!!!
:::

## 메인보드 케이블 연결 방법

:::warning 경고
  구매한 완제품 **3선식 리미트 스위치**를 사용하지 마십시오. 이러한 리미트 스위치는 **메인보드 단락**을 일으킬 수 있으며, **메인보드를 손상**시킬 수 있습니다!!! 반드시 연결図에 표시된 연결 방식대로 연결하십시오!!!
:::
