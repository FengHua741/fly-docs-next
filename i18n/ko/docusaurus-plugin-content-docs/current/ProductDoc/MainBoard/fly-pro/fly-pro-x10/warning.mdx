# 메인보드 주의사항

## 전원 지시등
    * Type-C를 통해 전력을 공급하고 다른 메인보드 전원에 연결하지 않을 때는 `SYS_5V`와 `SYS_3.3V`이 켜져야 합니다.
    * 메인보드 전원 `HVIN`, `VIN`, `BOARO`, `POWER`를 연결하면 메인보드 전원 LED가 항상 켜져 있어야 합니다.

## 드라이버

    * TP5160과 TP2209는 이름만 다르며, 점선 설정 없이도 드라이버 모드를 선택할 수 있습니다.
    * 드라이버 포트에는 TP 드라이버만 설치할 수 있으며, 다른 장치는 설치할 수 없습니다.

## 한계 포트
    * ProX10은 5V와 12V 전원 선택을 지원합니다. 근접 스위치를 사용하는 경우 다이오드를 설치하여 전압을 낮춰야 합니다.

## 상위 기기
    * ProX10은 외부 상위 기기를 지원하며, 다이얼 스위치로 Type-C 연결 후 사용할 수 있습니다.

## 펌웨어 플래싱

    * 메인보드는 TF 카드를 사용하여 플래싱할 수 있지만, 공장 출하 시 기본값은 `Katapult`입니다.
