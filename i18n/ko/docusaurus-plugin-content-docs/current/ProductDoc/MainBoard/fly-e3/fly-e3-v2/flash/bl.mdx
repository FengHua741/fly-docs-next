# BL 펌웨어

    :::팁

    * 주의: 출廠 시 이미 `부트로더`가 미리 프로그래밍 되었습니다. 다시 프로그래밍 할 필요가 없습니다.

    :::

## 플래시 모드 진입

    :::팁

    오른쪽은 `부트 버튼`, 왼쪽은 `리셋 버튼`입니다.

    :::

    * 플래시 모드에 진입하는 두 가지 방법이 있습니다.
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서 부트 버튼을 누르고 데이터 전송용 데이터 케이블의 한쪽을 메인보드에 연결하고 다른 쪽을 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송용 데이터 케이블의 한쪽을 메인보드에 연결하고 다른 쪽을 장치에 연결한 후, 부트 버튼을 누르고 리셋 버튼을 누릅니다. 그 다음 리셋 버튼을 놓고 마지막으로 부트 버튼을 놓습니다.
