## 제품 사양

    * 32비 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연시간이 적으며 연결이 더 안정적입니다.
    * 펌웨어: Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 디스플레이: 시리얼 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1개의 고압 IN 단자, 자동 평탄화 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
    * Repap 펌웨어 사용 시, 확장 ESP32 WiFi 모듈 지원, 최대 1.6m/s 속도 지원
    * 독특한 PWM 팬 MOS 보드 설계, 손상 시 직접 교체 가능, 고객 수리 난이도를 줄임, 4개의 2핀 팬 단자
    * PCB는 2OZ 구리 두께를 채택하여 더 큰 전류를 지원합니다.
    * 2개의 ADC 인터페이스, 최대 2개의 열 감지기 지원
    * 4개의 IO 인터페이스, 레벨링 및 제한용으로 사용 가능
    * 3개의 히팅 엘리먼트 인터페이스, 최대 3개의 히팅 엘리먼트 지원
