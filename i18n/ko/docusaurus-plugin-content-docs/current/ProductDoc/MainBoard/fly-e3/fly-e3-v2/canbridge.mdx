# CAN브리지

CAN버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스로, 이중 와이어를 사용하여 신호를 전송하며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내의 다양한 구성 요소 간의 통신에 사용되어 비용이 많이 들고 무거운 배선 하arness를 대체합니다. 이 프로토콜의 견고성으로 인해 다른 자동화 및 산업 응용 분야에도 사용됩니다. CAN 프로토콜의 특징에는 전체적인 직렬 데이터 통신, 실시간 지원, 최대 1Mbps의 전송 속도, 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린트 헤드가 메인보드에 연결되는 케이블 수를 줄입니다. 원래 십여 개의 케이블이 필요했지만, CAN을 사용하면 이제 네 개의 케이블만 사용하면 됩니다. 이렇게 함으로써 케이블 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리지를 사용하는 방법을 간단히 설명합니다.

## 도구 보드는 CAN을 통해 메인보드에 연결됩니다

* 도구 보드 전원 선은 CAN 포트에 연결되는 것이 아니라 12V-24V 전원 선에 연결됩니다.
