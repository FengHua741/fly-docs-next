# BL 펌웨어

    :::tip

    * 부트로더가 출하 시 미리 프래그되어 있으므로 다시 플래시하지 않아도 됩니다.

    :::

## 플래시 모드 진입

    :::tip

    오른쪽은 `BOOT` 버튼, 왼쪽은 `RESET` 버튼입니다.

    :::

    * 플래시 모드 진입에는 두 가지 방법이 있습니다.
    * 첫 번째: 메인보드가 완전히 전원이 꺼진 상태에서 BOOT 버튼을 누른 채 데이터 전송이 가능한 데이터선을 메인보드 한쪽과 장치 다른 쪽에 연결합니다.
    * 두 번째: 데이터 전송이 가능한 데이터선을 메인보드와 장치에 연결한 후, BOOT 버튼을 누른 상태에서 RST 버튼을 누르고 RST 버튼을 먼저 떼고 마지막으로 BOOT 버튼을 떼는 방식입니다.
