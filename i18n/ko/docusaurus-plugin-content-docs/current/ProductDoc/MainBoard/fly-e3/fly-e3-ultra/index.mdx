## 제품 사양

* 32비트 ARM Cortex-M4 시리즈 550 MHz, STM32H723VGT6 칩셋
* 펌웨어: Reprap/Klipper
* 보드에 다섯 개의 TMC2209 드라이버 장착, XYZ 3개의 드라이버 위치는 무한한 리미트 모드를 지원
* 디스플레이: 시리얼 터치스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
* 1개의 고전압 IN 포트, 자동 평탄 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
* Repap 펌웨어 사용 시 ESP32 WiFi 모듈 사용 가능, 최대 1.6m/s 속도 지원
* 독특한 PWM 팬 MOS 보드 설계, 고장 시 교체가 용이해 고객의 수리 난이도를 줄임, 4개의 2핀 팬 인터페이스 제공
* 3개의 ADC 인터페이스는 온도 센서 사용 가능
3개의 리미트 포트는 리미트 스위치 연결 가능
* 4개의 IO 인터페이스는 근접 스위치, 리미트, 레이저 헤드, RGB LED 등 사용 가능
* 3개의 히팅 인터페이스
* 1개의 RRF용 시리얼 스크린 인터페이스
* Max 버전은 K-type 열전대와 1개의 PT100 온도 센서 지원
* 새로운 디자인으로 Pt100 온도 센서의 2/3/4선 모드 선택 가능
* 새로운 통합 히트싱 추가로 드라이버의 히트싱 면적 증가
* 새로운 CAN 인터페이스 추가
