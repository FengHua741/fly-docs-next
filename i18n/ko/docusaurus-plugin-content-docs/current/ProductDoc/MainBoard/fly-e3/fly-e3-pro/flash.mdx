# 펌웨어 컴파일 및 플래시

    :::tip

    * E3 Pro는 기본적으로 TF 카드를 사용하여 펌웨어를 플래시합니다.
    * 최대 `32G` 용량의 TF 카드를 준비하십시오.
    * 브랜드 카드는 플래시 실패의 원인이 될 수 있으므로 가능한 한 아래에 나열된 카드를 사용하십시오.
    * TF 카드의 전송 속도는 `class10`급 이상이어야 하며 SanDisk 등의 브랜드를 권장합니다.

    :::

    * `Q` 키를 누르면 **Save configuration**이 표시되고, 그 다음에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되어 명령줄 인터페이스로 돌아갔습니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 다소 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래와 같은 내용이 출력되면 컴파일이 성공했습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
