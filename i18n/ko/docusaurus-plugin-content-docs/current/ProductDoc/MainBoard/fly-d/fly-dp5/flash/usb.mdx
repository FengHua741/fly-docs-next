## 플래싱 모드 진입 확인

    * 상위 컴퓨터가 정상적으로 네트워크에 연결되어 있는지 확인하고, 하위 컴퓨터와 연결된 데이터선이 데이터 전송 기능을 가졌는지 확인하십시오.
    * 상위 컴퓨터에 SSH로 접속한 후 `lsusb`를 입력하고 Enter를 누릅니다. 아래 이미지에서 빨간색으로 표시된 정보가 나타나면 다음 단계로 진행할 수 있습니다.
        * 아무런 정보도 반응이 없으면 상위 컴퓨터의 시스템 문제입니다. 해결 방법은 없으므로 정상적인 시스템으로 교체하거나 상위 컴퓨터를 교체해야 합니다.
        * `lsusb` 명령이 없다고 하는 경우 다음 명령어로 설치할 수 있습니다.
            ```bash
            sudo apt-get install usbutils
            ```
            <ImageView image={require('@site/docs/General/get-id/img/6177.webp').default} size="80%" align="left" /> 
    * 빨간색 박스 내부에 표시된 ID는 참고용이며 고정되지 않으며, `OpenMoko, Inc. stm32f072xx` 이후의 `XX`는 가변적입니다.
        * 일부 상위 컴퓨터는 시스템 문제로 인해 정보가 제대로 표시되지 않을 수 있습니다.
