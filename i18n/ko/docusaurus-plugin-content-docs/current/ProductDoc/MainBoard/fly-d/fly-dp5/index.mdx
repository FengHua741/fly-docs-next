# 소개

> Mellow Fly-DP5는 광저우 맘렌 전자 과학 기술 유한 회사의 Fly3D 메인보드 연구 개발 팀이 출시한 고성능 3D 프린터 메인보드로 최대 5개의 스테퍼 모터를 지원합니다.


<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/dp5.webp').default} size="60%" align="center" />

## **1. 제품 특징**

* 32비트 48MHz 주파수의 ARM Cortex-M0+ 시리즈 STM32F072RBT6 메인 컨트롤 칩 사용
* CAN 버스 연결을 지원하여 데이터 전송이 더 안정적이고 지연 시간이 줄어들며 연결이 더 안정적입니다.
* 드라이버 지원: DP2209와 DP560
* 보드에 DIAG 기능 핀이 장착되어 있어 간단한 점퍼 케이블 플러그를 통해 무제한 리미트 스위치 지원 가능

## 2. 제품 사양

* MCU: STM32F072RBT6
* 펌웨어: Klipper/Marlin
* 12-24V DC 전원 공급을 지원하여 전원 공급이 더 안정적이고, 다운 레벨 모듈을 제거할 수 있습니다.
* 드라이버 작동 모드 지원: SPI, UART, STEP/DIR
* 지원 드라이버: 외부 TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등
* 확장 인터페이스: BLTouch(Servos, Probe), UART X1, 12864 X1
* 팬: 2선식 제어 가능한 팬 X2, 병렬 팬 X2, 항상 켜진 팬 X2
* 온도 센서: 2개의 NTC 100K
* 가열 인터페이스: 2개의 제어 가능한 인터페이스, 각각 10A를 지원
* 지원하는 머신 구조: Cartesian, Delta, Kossel, Ultimaker, CoreXY
