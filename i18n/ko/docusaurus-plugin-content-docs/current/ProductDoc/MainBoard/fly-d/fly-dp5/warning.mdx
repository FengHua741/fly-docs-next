# 메인보드 주의사항

## 전원 지시등
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/power.webp').default} size="80%" align="center" />
    * TYPE-C를 통해 전원을 공급하고 다른 메인보드 전원에 연결하지 않을 때 최소 두 개의 LED가 켜집니다. `3.3V`와 `5V`.
    * 메인보드 전원이 연결된 후에는 메인보드 전원 LED가 `3.3V`, `5V`, `VCC`를 모두 켭니다.

### 문제 해결


    :::danger 주의

    * 아무 외부 장치도 연결되지 않은 상태에서 (드라이버, 리미트 스위치, 12864 등)
    * TYPE-C를 통해 전원을 공급하면 최소한 `3.3V`와 `5V` 두 개의 LED가 항상 켜집니다.

    :::

    * 아무 LED도 켜지지 않으면 단락이 발생했음을 나타내며, 고객 서비스에 문의해야 합니다.
    * `3.3V`, `5V` 중 하나의 LED가 깜빡인다면 단락이 발생했음을 나타내며, 고객 서비스에 문의해야 합니다.

## 드라이버

    * 메인보드는 외부 드라이버를 지원합니다.     <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">접속 및 설정 튜토리얼</Button>

## 리미트 포트

    * 접선 순서에 주의하십시오. 3선 리미트는 5V, GND, S가 잘못 연결될 경우 메인보드가 손상될 수 있습니다.
    * 2선 리미트는 S와 GND만 연결하면 됩니다.
    * PL08 및 기타 근접 스위치는 신호 포트 외부에 다이오드를 연결하여 전압을 3.3V 이하로 낮춰야 합니다.

## 펌웨어

    * 버튼 문제로 인해 터틀을 사용하여 펌웨어를 업데이트하는 새로운 튜토리얼이 제공됩니다.
    * Katapult 펌웨어를 컴퓨터를 통해 업데이트하고 Katapult 펌웨어를 사용하여 Klipper를 플래시하는 것을 권장합니다.
