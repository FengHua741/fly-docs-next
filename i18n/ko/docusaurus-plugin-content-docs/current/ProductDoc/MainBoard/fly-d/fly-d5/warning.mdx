# 메인보드 주의사항

## 전원 지시등
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/power.webp').default} size="80%" align="center" />
    * TYPE-C를 통해 전원을 공급하고 다른 메인보드 전원에 연결하지 않을 때 최소 두 개의 LED가 켜집니다. `3.3V`와 `5V`.
    * 메인보드 전원이 연결되면 `3.3V`, `5V`, `VCC` LED가 켜집니다.

### 문제 해결

    :::danger 주의

    * 어떠한 외부 장치도 연결되지 않은 상태에서 (드라이버, 리미트 스위치, 12864 등)
    * TYPE-C를 통해만 전원을 공급할 경우 최소 두 개의 LED(`3.3V`와 `5V`)가 상시 점등됩니다.

    :::

    * 아무런 LED가 켜지지 않으면 단락이 발생했음을 나타내며, 고객 서비스에 문의해야 합니다.
    * `3.3V` 또는 `5V` 중 하나의 LED가 깜빡거릴 경우 단락이 발생했음을 나타내며, 고객 서비스에 문의해야 합니다.

## 드라이버

    * 메인보드는 외부 드라이버를 지원합니다.   <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 구성 가이드</Button>

## 리미트 포트

    * 신중하게 접선 순서를 따르세요. 3-선 리미트 스위치의 경우 5V, GND, S를 잘못 연결하면 메인보드가 손상될 수 있습니다.
    * 2-선 리미트 스위치의 경우 S와 GND만 연결하면 됩니다.
    * PL08 및 기타 근접 스위치는 신호 포트에 3.3V 미만의 전압으로 낮추기 위해 외부 다이오드를 연결해야 합니다.

## 펌웨어

    * 버튼 문제가 발생하여 사용하는 Katapult 펌웨어로 펌웨어 업데이트를 진행합니다.
    * Katapult 펌웨어를 컴퓨터를 통해 업데이트한 후 Katapult 펌웨어를 사용하여 Klipper를 플래시하는 것을 권장합니다.
