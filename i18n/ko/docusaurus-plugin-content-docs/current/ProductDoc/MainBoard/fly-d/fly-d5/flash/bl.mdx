# Katapult 펌웨어

## Katapult 펌웨어 설명

    <CANBOOT />

    :::팁
    * D5의 katapult 펌웨어는 세 번째 연결 방식을 사용합니다: USB 연결
    * D5는 2024년 9월 1일 출하 시 이미 Katapult 펌웨어가 미리 프로그래밍되어 있습니다.
    :::

## DFU 베닝다운 모드 진입

    :::팁

    왼쪽은 `BOOT 버튼`, 오른쪽은 `RESET 버튼`입니다.

    :::

    * 베닝다운 모드로 들어가는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 보드가 완전히 전원이 꺼진 상태에서, BOOT 버튼을 누르고 데이터 전송용 케이블의 한 쪽을 보드에, 다른 쪽을 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블의 한 쪽을 보드에, 다른 쪽을 장치에 연결한 후, BOOT 버튼을 누르면서 RST 버튼을 누릅니다. 그런 다음 RST 버튼을 놓고 마지막으로 BOOT 버튼을 놓습니다.
