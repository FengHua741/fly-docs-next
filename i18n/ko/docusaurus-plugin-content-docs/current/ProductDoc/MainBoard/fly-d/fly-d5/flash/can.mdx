        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />

    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 좀 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 아래 내용이 출력되면 컴파일이 성공했습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 플래시 모드 진입

    :::tip 팁
        2024년 9월 1일 이후 출하된 D5는 이미 Katapult 펌웨어가 플래시되어 있습니다.
    :::
    * Type-C 데이터 케이블을 통해 보드를 상위 컴퓨터에 연결한 후, **리셋 버튼을 두 번 클릭**합니다. 이때 보드의 LED가 깜빡입니다.
    :::warning 경고
       * 만약 LED가 깜빡이지 않는다면, Katapult 펌웨어를 다시 플래시해 주세요: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-flash-mode">Katapult 펌웨어 플래시</Button>

    :::
