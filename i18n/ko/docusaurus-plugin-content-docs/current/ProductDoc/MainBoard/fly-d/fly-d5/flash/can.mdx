# USB 통합 CAN 펌웨어 플래시

## 빌드 시작

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` 키를 누르면 **Save configuration**이 나타나고, 그 후에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 이제 설정이 저장되었으며 명령줄 인터페이스로 돌아왔습니다.
    * 아래 명령어를 입력하여 빌드를 시작합니다. 시간이 다소 걸릴 수 있습니다.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막으로 아래와 같은 출력이 표시되면 빌드가 성공적으로 완료되었습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## 플래시 모드 진입

    :::tip 팁
        2024년 9월 1일 이후 출하된 D5는 이미 Katapult 펌웨어로 업데이트되어 있습니다.
    :::
    * 메인보드를 상위 컴퓨터에 Type-C 데이터 케이블로 연결한 후, **리셋 버튼을 두 번 클릭**합니다. 이때 메인보드의 LED가 깜빡입니다.
    :::warning 경고
       * 만약 LED가 깜빡이지 않는다면 Katapult 펌웨어를 다시 플래시하세요: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-flash-mode">Katapult 펌웨어 플래시</Button>
       * LED가 깜빡하지 않는 경우, Katapult 펌웨어 플래시 후 다음 단계를 진행해야 합니다.
    :::
