# CAN교차연결

CAN 버스는 실시간 응용 프로그램을 위한 직렬 통신 프로토콜 버스로, 이중 와이어를 사용하여 신호를 전송할 수 있으며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내의 다양한 요소 간의 통신에 사용되어 비싸고 무거운 배선 하arness를 대체합니다. 이 프로토콜의 강력한 특성 덕분에 다른 자동화 및 산업 응용 분야에도 활용됩니다. CAN 프로토콜의 특징에는 완전한 직렬 데이터 통신, 실시간 지원, 최대 1Mbps의 전송 속도, 11비트 주소 지정 및 오류 감지 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린팅 헤드가 메인 보드에 연결되는 케이블 수를 줄일 수 있습니다. 원래 십여 개의 케이블이 필요했지만, CAN을 사용하면 단지 네 개의 케이블만 사용하면 되므로 케이블 수를 크게 줄이고 배선 난이도를 낮출 수 있습니다.
본 장에서는 CAN 교차 연결 사용에 대한 간단한 개요만 제공합니다.

## 도구 보드가 메인 보드와 CAN으로 연결됨

* 주의: 도구 보드 전원 라인은 CAN 포트에 연결되지 않고 12V-24V 전원 라인에 연결됩니다.

* 주의: D5 메인 보드에 번들링된 펌웨어는 `USB-CAN 교차 연결 펌웨어 구성`이어야 합니다.
