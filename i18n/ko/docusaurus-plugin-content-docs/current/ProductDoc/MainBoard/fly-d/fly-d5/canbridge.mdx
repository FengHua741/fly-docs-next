# CAN 브리지

CAN 버스는 실시간 응용 프로그램에 사용되는 시리얼 통신 프로토콜 버스로, 신호를 전송하기 위해 두 개의 꼬인 선을 사용할 수 있으며, 세계적으로 가장 널리 사용되는 현장 총선 중 하나입니다. CAN 프로토콜은 자동차 내 다양한 구성 요소 간의 통신을 위해 사용되며, 비용이 많이 들고 부피가 큰 배선 하네스를 대체합니다. 이 프로토콜의 강력한 신뢰성으로 인해 기타 자동화 및 산업 응용 분야에도 활용됩니다. CAN 프로토콜의 특징에는 완전한 시리얼 데이터 통신, 실시간 지원, 최대 1Mbps의 전송 속도, 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린팅 헤드와 메인보드를 연결하는 배선 수를 줄일 수 있습니다. 원래 십여 개의 선이 필요했으나 CAN을 사용하면 단순히 네 개의 선만 사용하면 되므로, 배선 수를 크게 줄이고 배선 난이도를 낮추게 됩니다.
이 장에서는 CAN 브리지를 사용하는 방법에 대해 간단히 설명합니다.

## 도구 보드를 메인보드와 CAN으로 연결

    * 주의: 도구 보드의 전원선은 CAN 포트에 연결되지 않습니다. 12V-24V 전원선에 연결되어야 합니다.
    * 주의: 120Ω 점프 풀을 연결해야 합니다.
