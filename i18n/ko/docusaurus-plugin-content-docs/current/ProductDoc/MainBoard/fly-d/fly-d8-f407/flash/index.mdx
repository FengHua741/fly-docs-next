# 펌웨어 플래시 설명

:::danger 주의사항
* 메인보드 펌웨어 플래시는 Katapult를 통해 Klipper를 플래시해야 합니다.
* 메인보드에 이미 Katapult가 플래시되어 있다면 마지막에 Katapult 펌웨어를 플래시할 필요가 없습니다. 그건 메인보드의 Katapult 펌웨어 플래시 실패를 방지하기 위한 예방조치입니다.
* 메인보드에 Katapult 펌웨어가 성공적으로 플래시되었는지 확인하려면, 리셋 버튼을 두 번 빠르게 누른 후 메인보드의 LED가 깜빡이는지 확인하면 됩니다. LED가 깜빡인다면 메인보드는 Katapult 펌웨어로 업데이트된 것입니다.
:::

## 어떤 펌웨어를 플래시해야 하는지 어떻게 알 수 있나요?

    :::tip

    * 단, 하나의 펌웨어만 플래시하면 됩니다.

    :::
    
    * D8은 다음 방법으로 호스트 컴퓨터에 연결할 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 안정적이고 신뢰할 수 있습니다.
        * 단점: 여러 USB 장치가 있는 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않습니다.
        * 단점: KlipperScreen과 함께 사용할 때 재설정 시간 초과 등의 오류가 발생할 수 있습니다.
