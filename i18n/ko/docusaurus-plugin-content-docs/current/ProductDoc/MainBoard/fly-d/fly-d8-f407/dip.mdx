# 보드 내장 외부 장치 소개

    * 다이얼을 오른쪽으로 돌리면 Type-C 포트 옆의 USB 인터페이스가 켜집니다.
    * 다이얼을 왼쪽으로 돌리면 Type-C 포트 옆의 RS232 인터페이스가 켜집니다.

## USB 포트 소개
    
    * 외부 카메라 및 메인보드 등 장치에 연결할 수 있습니다.
