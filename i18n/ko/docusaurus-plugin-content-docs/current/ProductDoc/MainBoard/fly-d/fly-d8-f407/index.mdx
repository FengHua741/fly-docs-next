# 소개

> Mellow Fly-D8은 광저우 맘렌 전자 과학 기술 유한 회사의 Fly3D 메인보드 연구 개발 팀이 발표한 고성능 3D 프린터 메인보드로 최대 8개의 스테퍼 모터를 지원합니다.

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d8-f407/img/d8.webp').default} size="80%" align="center" />

## 제품 특징

- 메인보드 제한 등 접속구는 5V 공급을 격리 보호하고, POWER 전원에 연결하지 않으면 전원 출력이 없습니다.
- 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407 칩을 사용합니다.
- CAN 버스와 RS232 연결을 지원하여 데이터 전송이 더 안정적이고 지연 시간이 작아지며 연결이 더 안정적입니다.
- 메인보드에는 BOOT 및 RESET 버튼이 예약되어 있어 사용자는 USB를 통해 DFU 모드로 진입하여 펌웨어를 업데이트할 수 있습니다.
- 전원 접속구에는 역접속 보호가 있어 고객이 DIY 시 전원선을 역접속해 보드를 손상시키는 것을 방지합니다.
- 두 개의 USB2.0 접속구를 제공하여 더 많은 연결 가능성을 확장하며, 하나의 USB 접속구는 스위치를 통해 RS232 접속구로 변경될 수 있어 다양한 장비의 연결 요구에 맞춥니다.

# 제품 사양

- 주요 제어 칩: 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
- 펌웨어: Marlin 2.0/Klipper
- 드라이버 모드 지원: TMC: UART/SPI
- 8개의 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
- 디스플레이: 시리얼 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
- 자동 평탄 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
- 입력 전압: 12V/24V 지원
- 6개의 제한기 접속구로 정확한 범위 제한 및 위치 제어를 구현할 수 있습니다.
- 2개의 RGB 접속구로 다채로운 시각 효과를 제공할 수 있습니다.
- 3개의 히팅 튜브 접속구로 엑트러더와 열판을 독립적으로 제어할 수 있습니다.
- 4개의 열감지 접속구로 최대 4개의 그룹 온도를 동시에 모니터링하여 정밀한 온도 제어를 실현할 수 있습니다.
- USB 접속구: USB2.0 접속구를 두 개 지원하며, 하나의 USB 접속구는 스위치를 통해 RS232 접속구로 변경될 수 있습니다.
- 두 개의 USB2.0 접속구를 제공하여 더 많은 연결 가능성을 확장하며, 하나의 USB 접속구는 스위치를 통해 RS232 접속구로 변경될 수 있어 다양한 장비의 연결 요구에 맞춥니다.
