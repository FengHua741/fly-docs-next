# CAN브리지

CAN버스는 실시간 응용 프로그램을 위한 직렬 통신 프로토콜 버스로, 두 개의 와이어를 사용하여 신호를 전송할 수 있으며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내의 다양한 구성 요소 간의 통신에 사용되며, 이는 비싸고 무거운 배선 하arness를 대체하기 위함입니다. 이 프로토콜의 견고성은 다른 자동화 및 산업 응용 분야에도 확장되었습니다. CAN 프로토콜의 특징에는 전체적인 직렬 데이터 통신, 실시간 지원, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린팅 헤드가 메인보드에 연결되는 케이블 수를 줄입니다. 원래 십여 개의 선이 필요했지만, CAN을 사용하면 이제 네 개의 선만 사용하면 됩니다. 이렇게 함으로써 케이블 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리지를 사용하는 방법에 대해 간략히 설명합니다.

## 도구 보드가 CAN을 사용해 메인보드와 연결

* 도구 보드 전원 선은 CAN 포트에 연결되지 않고 12V-24V 전원 선에 연결되어야 합니다.
    
* 120Ω 점프 패치가 연결되어야 합니다.
