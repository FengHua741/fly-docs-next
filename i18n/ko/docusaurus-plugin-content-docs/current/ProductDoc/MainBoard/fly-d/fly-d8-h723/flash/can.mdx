## 펌웨어焼录 모드 확인

    * 상위 컴퓨터가 정상적으로 연결되어 있는지 확인하고, 상위 컴퓨터와 데이터 전송 기능을 가진 하위 컴퓨터를 연결합니다.
    * SSH를 통해 상위 컴퓨터에 연결한 후 `lsusb` 명령어를 입력하고 Enter 키를 누릅니다. 아래 이미지에서 빨간색 사각형으로 표시된 정보가 나타나면 다음 단계로 진행할 수 있습니다.
        * 아무런 정보도 반환되지 않는 경우, 상위 컴퓨터 시스템에 문제가 있을 수 있으며 해결 방법은 없으므로, 정상적인 시스템으로 교체하거나 상위 컴퓨터를 바꾸어야 합니다.
        * `lsusb` 명령어가 없다는 메시지가 나온 경우 아래 명령어를 실행하여 설치할 수 있습니다.
            ```bash
            sudo apt-get install usbutils
            ```
            <ImageView image={require('@site/docs/General/get-id/img/6177.webp').default} size="80%" align="left" /> 
    * 빨간색 사각형 안에 표시된 ID는 참고용이며 고정적이지 않습니다. `OpenMoko, Inc. stm32h723xx`라는 뒤의 접미사 `XX`는 변할 수 있습니다.
        * 일부 상위 컴퓨터는 시스템 문제로 인해 정보가 제대로 표시되지 않을 수 있습니다.
