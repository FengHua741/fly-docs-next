# 펌웨어 프래싱 안내

:::danger 주의사항
* 메인보드 펌웨어 프래싱은 Katapult를 통해 Klipper를 프래싱해야 합니다.
* 메인보드에 이미 Katapult 펌웨어가 프래시되어 있다면 마지막에 Katapult 펌웨어를 프래싱할 필요가 없습니다. 그건 메인보드의 Katapult 펌웨어 프래싱 실패를 방지하기 위한 예방조치입니다.
* 메인보드에 재설정 버튼을 두 번 빠르게 눌러서 LED가 깜빡이는지 확인할 수 있습니다. LED가 깜빡인다면 메인보드가 Katapult 펌웨어로 업데이트된 것입니다.
:::

## 어느 종류의 펌웨어가 필요한지 어떻게 판단할 수 있나요?

    :::tip

    * 주의하세요, 하나의 펌웨어만 설치하면 됩니다.

    :::
    
    * D8는 다음 방법으로 호스트 컴퓨터와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰성이 있으며 안정적입니다.
        * 단점: 여러 USB 장치가 있는 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있고 USB 포트를 차지하지 않습니다.
        * 단점: `KlipperScreen`과 함께 사용할 때 위치 재설정 시간 초과 등의 오류가 발생할 수 있습니다.
