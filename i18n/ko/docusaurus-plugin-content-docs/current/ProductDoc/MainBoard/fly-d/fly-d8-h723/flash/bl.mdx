## Katapult 펌웨어 설명

## 플래싱 모드 진입

    :::팁

    왼쪽은 `RST 버튼`, 오른쪽은 `BOOT 버튼`입니다.

    :::

    * 플래싱 모드에 진입하는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서, BOOT 버튼을 누르고 데이터 전송용 케이블의 한쪽 끝을 메인보드에 연결하고 다른 쪽 끝을 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블의 한쪽 끝을 메인보드에 연결하고 다른 쪽 끝을 장치에 연결한 후, BOOT 버튼을 누르면서 RST 버튼을 누른 다음 RST 버튼을 놓고 마지막으로 BOOT 버튼을 놓습니다.
