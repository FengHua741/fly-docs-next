# RS232 연결

RS232 표준 인터페이스(이하 EIA RS-232)는 일반적으로 사용되는 시리얼 통신 인터페이스 표준 중 하나로, 다양한 장치 간의 데이터 전송에 널리 사용됩니다. 이 프로토콜은 표준 5V와 다른 신호 전압을 사용하여 전선 데이터를 전송함으로써 신호 간섭을 최소화할 수 있습니다. 3D 프린터 시스템에서, RS232는 CAN보다 높은 데이터 전송 속도, 더 나은 안정성, 더 간단한 설정, 그리고 상위 컴퓨터와의 호환성이 더 좋기 때문에 선호됩니다.

## 도구 보드가 RS232로 주보드와 연결됨

    * 주의: 도구 보드 전원 선은 RS232 신호선에 연결되지 않으며, 12V-24V 전원선에 연결되어야 합니다.

    * D8 주보드 펌웨어는 재컴파일할 필요가 없습니다.
