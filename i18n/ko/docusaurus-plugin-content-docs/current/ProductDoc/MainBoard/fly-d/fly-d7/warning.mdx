# 메인보드 주의사항

## 전원 지시등
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/power.webp').default} size="80%" align="center" />
    * TYPE-C로만 전원을 공급하고 다른 메인보드 전원에 연결하지 않을 때 최소 두 개의 빛이 켜집니다. `3.3V`와 `5V`.
    * 메인보드 전원에 접속한 후에는 메인보드 전원 지시등이 `3.3V`, `5V`, `VCC`가 켜집니다.

### 문제 해결


    :::danger 주의

    * 아무 외부 장치도 연결되지 않은 상태에서 (드라이버, 리미트 스위치, 12864 등)
    * TYPE-C로만 전원을 공급할 경우 최소 두 개의 LED (`3.3V`와 `5V`)가 항상 켜집니다.

    :::

    * 아무런 LED가 켜지지 않으면 단락이 발생했음을 의미하며, 고객 서비스에 문의해야 합니다.
    * `3.3V`와 `5V` 중 하나의 LED가 깜빡이면 단락이 발생했음을 의미하며, 고객 서비스에 문의해야 합니다.

## 드라이버

    * 메인보드는 외부 드라이버를 지원합니다. <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">접선 및 설정 가이드</Button>

## 리미트 포트

    * 신중하게 배선 순서를 확인하십시오. 3선 리미트 스위치의 경우 5V, GND, S가 잘못 연결될 경우 메인보드가 손상될 수 있습니다.
    * 2선 리미트 스위치의 경우 S와 GND만 연결하면 됩니다.
    * PL08 및 기타 근접 스위치는 신호 단자에 3.3V 이하의 전압을 낮추기 위해 외부 다이오드를 연결해야 합니다.

## 펌웨어
