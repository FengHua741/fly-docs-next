    * `Q` 키를 누르면 **Save configuration**이 나타나고, 그 후에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * 이제 설정이 저장되었으며 명령줄 인터페이스로 돌아왔습니다.
    * 아래 명령어를 입력하여 빌드를 시작합니다. 시간이 다소 걸릴 수 있습니다.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 다음 내용이 출력되면 빌드가 성공적으로 완료되었습니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## 플래싱 모드 진입

    :::tip 팁
        2024년 9월 1일 이후 출하된 D7은 이미 Katapult 펌웨어가 설치되어 있습니다.
    :::
    * 메인보드를 상위 기기와 Type-C 데이터 케이블로 연결한 후, **리셋 버튼을 두 번 클릭**합니다. 이때 메인보드의 LED가 깜빡입니다.
    :::warning 경고
       * 만약 LED가 깜빡이지 않는다면 Katapult 펌웨어를 다시 플래싱하세요: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#進入dfu燒錄模式">Katapult 펌웨어 플래싱</Button>
