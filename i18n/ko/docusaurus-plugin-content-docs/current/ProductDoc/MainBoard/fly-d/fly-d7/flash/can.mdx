        ```bash

        ```

## 플래싱 모드 진입

    :::tip 팁
        2024년 9월 1일 이후에 출하된 D7은 Katapult 펌웨어가 이미 설치되어 있습니다.
    :::
    * 주보드를 상위 컴퓨터에 TypeC 데이터 케이블로 연결한 후, **리셋 버튼을 두 번 클릭**합니다. 이때 주보드에 LED 램프가 깜빡입니다.
    :::warning 경고
       * LED 램프가 깜빡이지 않는 경우, Katapult 펌웨어를 다시 프래싱해야 합니다: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#enter-dfu-flashing-mode">Katapult 펌웨어 프래싱</Button>

    :::
