    펌웨어를 컴파일하고 플래시하기 전에 이 섹션을 반드시 읽어주세요!!!

:::tip 주의사항
* D5는 2024년 9월 1일 출하일부터 Katapult 펌웨어가 미리 플래시되어 있습니다.
* BL을 한 번 플래시한 후에는 다시 플래시할 필요가 없습니다(D5의 BL은 Katapult 펌웨어입니다).
* BL은 Katapult 펌웨어를 부트로드하는 부트로더 프로그램입니다. Klipper 펌웨어도 플래시해야 합니다.
:::

## 필요한 펌웨어 판단

    :::tip

    * 주의: 하나의 펌웨어만 플래시하면 됩니다.

    :::
    
    * D7은 다음 두 가지 방법으로 호스트 컴퓨터에 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어（가장 일반적）</Button>
        * 장점: 통신이 안정적이고 신뢰할 수 있습니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">USB 브리지 펌웨어（CAN 도구 보드를 사용할 때 자주 사용됨）</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않습니다.
        * 단점: KlipperScreen과 함께 사용할 때 재homesetting 시간 초과 등의 오류가 발생할 수 있습니다.
