# Katapult 펌웨어

    :::팁

        * D7는 2024년 9월 1일에 출하될 때 이미 Katapult 펌웨어가 미리 써지도록 준비되었습니다.
    :::

## DFU 바이오깅 모드 진입

    :::팁

    왼쪽은 `BOOT 버튼`, 오른쪽은 `RESET 버튼`입니다.

    :::

    * 바이오깅 모드로 들어가는 방법은 두 가지가 있습니다
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서, BOOT 버튼을 누른 후 데이터 전송용 데이터선 한쪽을 메인보드에, 다른 한쪽을 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송용 데이터선 한쪽을 메인보드에, 다른 한쪽을 장치에 연결한 후, BOOT 버튼을 누른 다음 RST 버튼을 누릅니다. 그 후 RST 버튼을 놓고 마지막으로 BOOT 버튼도 놓습니다.
