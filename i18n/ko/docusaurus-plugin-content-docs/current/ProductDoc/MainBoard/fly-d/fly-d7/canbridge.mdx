# CAN교차연결

CAN 버스는 실시간 응용 프로그램을 위한 직렬 통신 프로토콜 버스로, 두 개의 꼬인 전선을 사용하여 신호를 전송할 수 있으며, 세계적으로 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내에서 다양한 요소 간의 통신에 사용되며, 이는 비싸고 무거운 배선 하arness를 대체하기 위함입니다. 이 프로토콜의 강력한 특성은 다른 자동화 및 산업 응용 분야로 확장될 수 있습니다. CAN 프로토콜의 특징에는 전체적인 직렬 데이터 통신, 실시간 지원, 최대 1Mbps의 전송 속도, 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린트 헤드가 메인보드에 연결되는 배선 수를 줄입니다. 원래 십여 개의 선이 필요했지만, CAN을 사용하면 단지 4개의 선만 사용하여 배선 수를 크게 줄이고 배선 난이도를 감소시킵니다.
이 장에서는 CAN 교차 연결의 사용에 대한 간단한 개요만 제공합니다.

## 도구 보드가 메인보드와 CAN으로 연결됨

    * 주의: 도구 보드 전원 선은 CAN 포트에 연결되지 않으며, 12V-24V 전원 선에 연결됩니다.
