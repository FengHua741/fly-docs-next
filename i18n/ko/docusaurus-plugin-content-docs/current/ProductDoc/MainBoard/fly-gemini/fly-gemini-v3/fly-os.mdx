## 시스템 번인 준비

    * 카드 리더기
    * TF 카드 1장, 용량은 최소 16GB 이상, 최대 128GB 이하 TF 카드, TF 카드 전송 속도는 class10 이상이어야 함
    * 해당 <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">시스템 이미지</Button> 다운로드
    * 필요한 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">번인 도구</Button> 다운로드

## 시스템 번인

:::info
TF 카드 번인과 TF 카드를 통해 M2WE로 번인하는 것은 둘 중 하나만 사용하면 됩니다.
:::

:::info
* 기기에 M2WE와 메모리 카드가 설치되어 있으면, 시스템은 기본적으로 메모리 카드에서 부팅합니다.
* M2WE는 TF 카드를 사용하여 시스템을 번인하는 것을 권장합니다. USB를 통해 M2W로 번인하는 경우 드라이버 호환성 등의 이유로 번인에 실패할 수 있으므로, [메모리 카드 번인으로 M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)를 사용하는 것이 좋습니다.
* 다운로드한 시스템 이미지가 해당되는지 주의하십시오. 그렇지 않으면 부팅되지 않습니다.
:::
