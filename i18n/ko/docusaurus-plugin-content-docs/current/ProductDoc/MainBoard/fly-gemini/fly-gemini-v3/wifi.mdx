# 와이파이 연결


## Gemini에 안테나 설치

    * 빨간 화살표가 가리키는 곳이 와이파이 안테나 접속부입니다.
    * 와이파이 안테나 포트는 `ipex1` 입니다.
    * 안테나를 Gemini에 설치하지 않으면 신호가 약해서 네트워크에 연결할 수 없을 수 있습니다.
