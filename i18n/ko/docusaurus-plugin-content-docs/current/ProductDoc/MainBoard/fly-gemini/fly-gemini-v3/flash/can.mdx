# USB橋接CAN 펌웨어 플래시

## 시작 컴파일

    <SSH />

    <MakeOperate/>

    <CAN />

    * `Q` 키를 누르면 **Save configuration**이 나타나므로 그때 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되고 명령줄 인터페이스로 돌아왔습니다.
    * `make -j4`를 입력하여 컴파일을 시작합니다. 시간이 꽤 걸립니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 다음 내용이 출력되면 컴파일이 성공한 것입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## 펌웨어 플래시

:::tip
펌웨어를 플래시하기 위해 다음 명령을 실행하세요.
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
