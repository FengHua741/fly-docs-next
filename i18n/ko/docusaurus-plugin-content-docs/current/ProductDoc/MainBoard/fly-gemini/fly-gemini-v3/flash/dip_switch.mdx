# 다이얼 소개

:::tip
FLY 제미나이를 다이얼로 상위 기기와 연결하려면, 다이얼 위치가 정확하지 않으면 정상적으로 작동하지 않습니다.
:::

:::danger 주의
다이얼 스위치는 지정된 위치에 다이얼해야만 상위 기기와 정상적으로 연결할 수 있습니다.
:::

## 내장 상위 기기 연결

    * 다이얼 스위치 `1`, `2`를 오른쪽으로 다이얼하여 하위 기기와 상위 기기를 내장 USB로 연결합니다.
    * 다이얼 스위치 `3`, `4`를 왼쪽으로 다이얼하지 않으면, 하위 기기와 상위 기기 연결에 영향을 줍니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * 이 방법으로 다이얼 스위치를 조작한 후, 아래쪽 USB 포트는 하위 기기에서 사용되므로 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## 외부 상위 기기 연결

    * 다이얼 스위치 `3`, `4`를 오른쪽으로 다이얼하여 하위 기기와 외부 상위 기기를 내장 USB로 연결합니다.
    * 다이얼 스위치 `1`, `2`를 왼쪽으로 다이얼하지 않으면, 하위 기기와 외부 상위 기기 연결에 영향을 줍니다.
