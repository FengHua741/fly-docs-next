# 디프 스위치 소개

:::tip
FLY Gemini는 디프 스위치를 통해 호스트에 연결됩니다. 디프 스위치 위치가 잘못되면 정상적으로 사용할 수 없습니다.
:::

:::danger 주의사항
디프 스위치가 지정된 위치에 맞게 설정되지 않으면 호스트에 정상적으로 연결되지 않습니다.
:::

## 온보드 호스트 연결

    * 디프 스위치 `1`, `2`를 오른쪽으로 설정하여 하위 기기와 호스트를 내장 USB로 연결합니다.
    * 디프 스위치 `3`, `4`를 왼쪽으로 설정하지 않으면 하위 기기와 호스트 연결에 영향을 미칠 수 있습니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * 이 방법으로 디프 스위치를 설정한 후, 하단의 USB 포트는 하위 기기에서 사용되므로 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## 외부 호스트 연결

    * 디프 스위치 `3`, `4`를 오른쪽으로 설정하여 하위 기기와 외부 호스트를 내장 USB로 연결합니다.
    * 디프 스위치 `1`, `2`를 왼쪽으로 설정하지 않으면 하위 기기와 외부 호스트 연결에 영향을 미칠 수 있습니다.
