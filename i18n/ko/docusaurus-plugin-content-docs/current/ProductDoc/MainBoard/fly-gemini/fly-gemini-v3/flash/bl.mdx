# BL 펌웨어

    :::tip

    * `bootloader`가 미리 프래그된 상태이므로 다시 브러시할 필요가 없습니다.

    :::

## 플래시 모드 진입

    * 플래시 모드로 진입하는 방법은 두 가지입니다.
    * 첫 번째: 메인보드가 완전히 전원을 차단된 상태에서, BOOT와 BOOT1을 단락시키고 데이터 전송용 케이블을 메인보드와 장치에 연결합니다.
    * 두 번째: 데이터 전송용 케이블을 메인보드와 장치에 연결한 후, BOOT와 BOOT1을 단락시키고 RST 버튼을 누른 다음 RST 버튼을 놓고 마지막으로 BOOT 단락을 해제합니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/flash/img/boot.webp').default} size="100%" align="center" />

            4. 펌웨어 플래시

            :::tip

            하나의 펌웨어만 플래시하면 됩니다.

            :::

            * 아래는 bootloader 펌웨어 플래시 명령어입니다.
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/Gemini/GeminiV3.bin
                ```

            5. 아래 그림의 화살표 부분에 표시된 내용 ``File downloaded successfully``가 나타나면 플래시 성공입니다.

            :::warning

            빨간색 화살표로 지정된 이 줄의 **뒤의 오류**는 무시하십시오!!

            :::
