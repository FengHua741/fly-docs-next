# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.webp').default} size="80%" align="center" />

## 제품 특징
    * 보드에 상위 기기, 라즈베리 파이를 대체하여 상위 기기를 실행할 수 있으며 1GB의 내장 DDR3 메모리를 가지고 있습니다.
    * 강화된 5V 전원 보호로 칩 손상 가능성 감소.
    * MCU 칩의 저장소 카드 설정을 제거하여 펌웨어 쓰기가 더 쉬워졌습니다.
    * 시스템에 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, numpy와 FLY-Tools를 미리 설치했으며 복잡한 작업 없이 사용하기가 더 쉽습니다 (TF 카드 또는 M2WE 구매 필요)

## 제품 사양

    * CPU: 고성능 올티치 H5 칩, 4코어 64비트 Cortex-A53
    * GPU: 고성능 6코어 Mali 450, 픽셀 채우기율 2.7Gpixel/s 초과
    * RAM: 1GB DDR3 (GPU와 공유)
    * ROM: 최대 128GB SDCARD 지원
    * 외부 장치: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1
    * 인터페이스: 내장 M.2 인터페이스로 Wi-Fi, eMMC 등을 확장 가능 (프라이빗 프로토콜, FLY π 전용 M.2 장치 외에는 연결하지 마세요)
    * 12-24V DC 전원 지원으로 전원 공급이 더 안정적이고 드롭 다운 모듈 없이 사용 가능
    * 5V 제어 가능한 팬 인터페이스 예약, 상위 기기 열 관리에 사용 가능
    * CAN 확장 인터페이스 예약
    * 시스템에 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, numpy와 FLY-Tools를 미리 설치했으며 복잡한 작업 없이 사용하기가 더 쉽습니다 (TF 카드 또는 M2WE 구매 필요)
    * 드라이버: 고압 드라이버 x4
    * MCU: STM32F405
