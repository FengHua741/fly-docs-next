# 상위 기기 SSH 연결


* 시리얼 포트 또는 WIFI를 통해 장치에 연결할 수 있습니다.
* 먼저 **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 준비하십시오.
* SD 카드 또는 M2WE에 **상위 기기에 해당하는 시스템이 번역되었는지 확인하십시오.**
* **만약 상위 기기가 정상적으로 시작되지 않으면 드라이버, 리미터, 팬 등 외부 장치를 제거하십시오!!!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 컴파일하기 쉽게 하십시오!!**
* **펌웨어를 컴파일하려면 Wi-Fi 또는 유선 네트워크를 통해 상위 기기에 연결하는 방법을 참조하십시오.**

        ## 상위 기기 SSH에 네트워크 IP를 사용하여 연결

        * 상위 기기 연결 **라우터 관리자 페이지**에 로그인하십시오.
        * 시스템이 **정상적으로 시작되었는지** 확인하십시오.
        * **자신이 상위 기기의 사용자 이름과 비밀번호를 알고 있는지** 확인하십시오. FLY 시스템의 기본 사용자 이름은 **fly**, 기본 비밀번호는 **mellow**입니다.
        * 메인보드가 시작되고 네트워크에 연결된 경우 IP를 사용하여 SSH에 연결할 수 있습니다.
        * MobaXterm을 열고 다음과 같이 설정합니다. IP는 앞서 확인한 것을 입력합니다. 라우터 관리자 페이지에서도 확인할 수 있습니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계에서 입력되는 것은 상위 기기의 사용자 이름입니다.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 첫 연결 시 이 창이 나타납니다. `Accept`를 클릭하십시오.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호를 입력하라는 메시지가 표시되면 상위 기기의 비밀번호를 입력하고 엔터를 누릅니다.

        **주의: 여기서 입력하는 비밀번호는 보이지 않습니다. 입력 후 바로 엔터를 누르십시오.**

        FLY 시스템 기본, 사용자 이름은 `fly`이며, 비밀번호는 `mellow`입니다.
