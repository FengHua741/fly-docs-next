# BL 펌웨어

    :::tip

    * `부트로더`가 이미 프래그라운드되어 있으므로 주의하세요.

    :::

## 플래싱 모드 진입

    :::tip

    오른쪽은 `부트 점퍼`, 왼쪽은 `리셋 버튼`입니다.

    :::

    * 플래싱 모드로 들어가는 두 가지 방법이 있습니다.
    * 첫 번째 방법: 메인보드가 완전히 전원을 끈 상태에서 부트 버튼을 누른 다음 데이터 전송용 데이터 케이블을 메인보드와 장치 사이에 연결합니다.
    * 두 번째 방법: 데이터 전송용 데이터 케이블을 메인보드와 장치 사이에 연결한 후, 부트 버튼을 누르고 리셋 버튼을 누른 다음 리셋 버튼을 놓고 마지막으로 부트 버튼을 놓습니다.
