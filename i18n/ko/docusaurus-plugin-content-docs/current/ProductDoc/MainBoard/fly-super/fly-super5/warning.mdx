# 메인보드 주의사항

## 전원 지시등

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/power.webp').default} size="80%" align="center" />
* TYPE-C를 통해 전원을 공급하고 다른 메인보드 전원에 연결하지 않을 때 최소 두 개의 램프가 켜집니다. `3.3V`와 `5V`
* 메인보드 전원이 연결된 후에는 메인보드 전원 램프가 `3.3V`, `5V`, `12V`가 켜져야 합니다.
* 구동 전원 0-2, 구동 전원 3-7, 열 침대 전원이 연결된 후에야 메인보드 왼쪽 아래 전원 지시등이 정상적으로 켜집니다.

### 문제 해결


    :::danger 주의

    * 구동, 리미트, 12864 등 외부 장치가 없는 경우
    * TYPE-C를 통해만 전원을 공급할 때 최소 두 개의 LED가 `3.3V`와 `5V`로 상시 켜집니다.

    :::

* `3.3V`와 `5V`가 켜지지 않는다면 메인보드의 5V 점선을 확인하세요.
* 점선을 교체한 후에도 동일하다면 TYPE-C가 정상적으로 전력을 공급하는지 확인하고 고객 서비스에 문의하세요.
* 만약 `3.3V`, `5V` 중 하나의 LED가 깜빡인다면 칩이 단락되었음을 의미하며 고객 서비스에 문의하세요.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/img/5v.webp').default} size="60%" align="center" />

## 구동

* 구동 전원은 48V를 사용할 수 있지만 모든 구동 전원이 48V를 사용해야 합니다.
* 메인보드는 외부 구동기를 지원합니다. <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 리미트 포트

* 리미트 포트가 작동하지 않는다면 리미트 포트를 교체하거나 구동기의 리미트 핀을 자르세요.
    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/2209diag.webp').default} size="80%" align="center" />



## 펌웨어 번들

* 메인보드는 TF 카드를 사용하여 번들합니다.
* 이전에는 DFU를 사용하여 번들을 진행했으나, 펌웨어를 업데이트하려면 <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/flash/bl">BL을 다시 번들</Button>해야 합니다.
