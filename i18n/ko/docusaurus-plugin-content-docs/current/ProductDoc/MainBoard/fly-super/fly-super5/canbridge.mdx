# CAN브리지

CAN 버스는 실시간 응용 프로그램을 위한 직렬 통신 프로토콜 버스입니다. 이 버스는 신호를 전송하기 위해 격자선을 사용할 수 있으며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내의 다양한 요소 간의 통신에 사용되어 비싸고 무거운 배선 하arness를 대체합니다. 이 프로토콜의 강력한 특성은 자동화 및 산업 응용 분야로 확장되었습니다. CAN 프로토콜의 특징에는 완전한 직렬 데이터 통신, 실시간 지원, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 감지 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린트 헤드가 메인보드에 연결되는 케이블 수를 줄입니다. 원래는 십여 개의 선이 필요했지만, CAN을 사용하면 단지 네 개의 선만 사용하여 케이블 수를 크게 줄이고 배선 난이도를 낮춥니다.
본 장에서는 CAN 브리지를 사용하는 방법을 간단히 설명합니다.

## 도구 보드가 메인보드와 CAN으로 연결됨

* 도구 보드 전원선은 CAN 포트에 연결되는 것이 아니라 12V-24V 전원선에 연결됩니다.

## 메인보드 펌웨어 업데이트

* SUPER5 PRO 메인보드에 번역된 펌웨어는 `USB CAN 브리지 펌웨어 설정`이어야 합니다.
