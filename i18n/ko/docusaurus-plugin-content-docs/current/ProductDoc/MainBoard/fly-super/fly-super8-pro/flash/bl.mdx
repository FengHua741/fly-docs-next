# BL 펌웨어

    :::tip

    * 주의: 출⼚시 이미 `bootloader`가 프로그래밍 되어 있으므로 다시 프로그래밍 할 필요가 없습니다.

    :::
## 플래싱 모드 진입

    * 플래싱 모드 진입에는 두 가지 방법이 있습니다.
    * 첫 번째 방법: 메인보드를 완전히 끈 상태에서, BOOT를 단락시킨 다음 데이터 전송용 케이블을 메인보드와 장치에 각각 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블을 메인보드와 장치에 각각 연결한 후, BOOT를 단락시키고 RST 버튼을 누른 다음 RST 버튼을 놓은 후 마지막으로 BOOT 단락을 해제합니다.
