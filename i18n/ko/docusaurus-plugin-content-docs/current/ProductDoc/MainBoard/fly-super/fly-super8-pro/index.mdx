# 제품 사양

    * 32비트 550MHz의 ARM Cortex-M7 시리즈 STM32H723ZGT6 마스터 칩 사용
    * 펌웨어: Marlin 2.0/Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 8개의 고전압 구동 포트, 8개의 독립 모터 구동, 5개의 엑스트루더, 10개의 PWM 팬
    * 디스플레이: 직렬 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1개의 고전압 IN 포트, 자동 평탄화 센서 지원: BLTouch, PL08N 등 근접 센서, Klicky, Voron Tap 등
    * 선택 가능한 리미트 스위치 전원: 5V/12V/24V
    * Reprap 펌웨어를 사용하면 보드 내장 ESP32 Wi-Fi 모듈을 사용할 수 있으며 1.6m/s 지원
    * 고유한 PWM 팬 MOS 보드 설계, 손상 시 직접 교체 가능, 두 개의 2핀 팬 인터페이스는 4핀 PWM 팬을 지원
    * PCB는 2OZ 쿠퍼 두께를 사용하여 더 큰 전류를 지원
    * 6개의 ADC 인터페이스, 최대 6개의 열감지기 지원
    * 8개의 IO 인터페이스, 레벨링 및 리미트에 사용 가능
    * 5개의 히터 접속점, 최대 5개의 히터 지원
    * 2개의 PMW 인터페이스
