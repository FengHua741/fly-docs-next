# CAN 브리지

CAN 버스는 실시간 애플리케이션을 위한 직렬 통신 프로토콜 버스입니다. 이는 신호 전송을 위해 꼬임쌍선을 사용할 수 있으며, 세계에서 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차에서 다양한 구성 요소 간의 통신에 사용되며, 비용이 많이 들고 무거운 분배선을 대신합니다. 이 프로토콜의 견고성 덕분에 다른 자동화 및 산업 응용 분야로도 확장되었습니다. CAN 프로토콜의 특성은 다음과 같습니다: 직렬 데이터 통신의 무결성, 실시간 지원 제공, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 검출 능력.
3D 프린터는 CAN 버스를 사용하여 프린트 헤드와 메인보드 간의 연결선 수를 줄입니다. 원래는 10여 개의 선이 필요했지만, CAN을 사용하면 4개의 선만으로 충분하여 연결선 수를 크게 줄이고 배선의 난이도를 낮춥니다.
본 장에서는 CAN 브리지 사용에 대해 간략히 개요만 설명합니다.

## 도구 보드를 사용하여 메인보드에 CAN 연결

    * 도구 보드 전원선은 CAN 포트에 연결되지 않고, 12V-24V 전원선에 연결됩니다.
