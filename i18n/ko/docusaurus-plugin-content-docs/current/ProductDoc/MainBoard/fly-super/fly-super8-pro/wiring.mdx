# 메인보드 연결

:::warning 주의사항
이미지 해상도가 너무 높아 웹 페이지에서 볼 수 없으므로 마우스 커서를 연결도면 위에 놓고 오른쪽 클릭하여 이미지를 '다른 이름으로 저장' 후 데스크탑에서 확인하십시오!
:::

:::warning 알림
Super8 pro의 핀은 super8과 동일하므로 여기서 super8의 연결도면을 직접 인용하여 추가 설명은 하지 않습니다.
:::
