# 주요 사항

## 전원 지시등

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/img/power.webp').default} size="80%" align="center" />
* TYPE-C 전원만 연결하고 다른 메인보드 전원이 연결되지 않았을 때 최소 두 개의 LED가 켜집니다. `3.3V`와 `5V`
* 메인보드 전원이 연결된 후에는 전원 지시등이 `3.3V`, `5V`, `12V`를 모두 켭니다.
* 드라이버 전원 0-2, 드라이버 전원 3-7, 열 침대 전원이 연결된 후에야 메인보드 왼쪽 아래 전원 지시등이 정상적으로 켜집니다.

### 문제 해결


    :::danger 주의사항

* 어떠한 외부 장치도 연결되지 않은 상태 (드라이버, 리미트 스위치, 12864 등)
* TYPE-C 전원만 연결하면 최소 두 개의 LED(`3.3V`와 `5V`)가 항상 켜집니다.

    :::

* `3.3V`와 `5V`가 켜지지 않는 경우 메인보드 5V 점선을 확인하세요.
* 점선을 교체한 후에도 동일한 문제가 발생하며 TYPE-C가 정상적으로 전력을 공급하는 것이 확인되면 고객 서비스에 문의하세요.
* `3.3V`, `5V`, `12V` 중 하나의 LED가 깜빡이는 경우 칩이 단락되었음을 의미하므로 고객 서비스에 문의하세요.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/img/5v.webp').default} size="60%" align="center" />

## 드라이버

* 드라이버 전원에 유의하시기 바랍니다. 드라이버 전원은 두 가지 경로로 나뉩니다. 첫 번째 경로는 드라이버 3-7에 전력을 공급하고, 두 번째 경로는 드라이버 0-2에 전력을 공급합니다.
* 메인보드는 외부 드라이버를 지원합니다. <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">접속 및 설정 가이드</Button>

## 리미트 포트

* 마지막 드라이버 포트(Drive7)를 제외하고 나머지 드라이버 포트의 리미트 스위치는 직접 연결됩니다.
* 리미트 포트가 사용되지 않는 경우에는 리미트 포트를 변경하거나 드라이버의 리미트 핀을 잘라내세요.
    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/2209diag.webp').default} size="80%" align="center" />




## 펌웨어 업데이트

* 메인보드는 TF 카드를 사용하여 펌웨어를 업데이트합니다.
* 이전에는 DFU를 통해 펌웨어를 업데이트했으나, 펌웨어를 업데이트하려면 <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-super/fly-super8-pro/flash/bl">BL을 다시 플래시</Button>해야 합니다.
