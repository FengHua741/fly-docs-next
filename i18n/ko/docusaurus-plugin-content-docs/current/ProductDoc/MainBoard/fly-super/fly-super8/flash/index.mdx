    :::tip

    * Super8는 기본적으로 TF 카드를 사용하여 펌웨어를 플래시합니다.
    * 용량이 `32G` 이하인 TF 카드를 준비해 주십시오.
    * 브랜드 카드는 플래시 실패의 원인이 될 수 있으므로 가능한 한 아래에 나열된 추천 카드를 사용하여 펌웨어를 업데이트하십시오.
    * TF 카드의 전송 속도는 `class10` 또는 그 이상이어야 하며, SanDisk 등의 브랜드를 권장합니다.

    :::

# 펌웨어 플래시 안내


## 어떤 펌웨어가 필요한지 어떻게 판단하는지

    :::tip

    * 하나의 펌웨어만 플래시하면 됩니다.

    :::
    
    * Super8는 다음 방법으로 상위 기기와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 안정적이고 신뢰할 수 있습니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않습니다.
        * 단점: KlipperScreen과 함께 사용할 경우 재설정 시간 초과 등의 오류가 발생할 수 있습니다.
