# 주요 사항

## 전원 표시등

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/power.webp').default} size="80%" align="center" />
* TYPE-C로만 전원 공급하고 다른 메인보드 전원에 연결하지 않을 때 최소 두 개의 램프 `3.3V`와 `5V`가 켜집니다.
* 메인보드 전원이 연결된 후에는 전원 표시등이 `3.3V`, `5V`, `12V`를 켭니다.
* 구동 전원 0-2, 구동 전원 3-7, 열 침대 전원이 연결된 후에야 메인보드 왼쪽 아래 전원 표시등이 정상적으로 켜집니다.

### 문제 해결

    :::danger 주의사항

* 어떤 외부 장치도 연결되지 않은 상태에서 (구동기, 리미트, 12864 등)
* TYPE-C로만 전원 공급하면 최소 두 개의 LED `3.3V`와 `5V`가 항상 켜집니다.

    :::

* 만약 `3.3V`와 `5V`가 안 켜진다면 메인보드 5V 점선을 확인하세요.
* 점선을 교체한 후에도 여전히 같은 현상이 발생하며 TYPE-C가 정상적으로 전원을 공급한다면 고객 서비스에 문의하세요.
* 만약 `3.3V`, `5V`, `12V` 중 하나의 LED가 깜빡인다면 칩이 단락된 것이므로 고객 서비스에 문의하세요.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/5v.webp').default} size="60%" align="center" />

## 구동기

* 구동기 전원에 유의하십시오. 구동기 전원은 두 가지 경로로 나뉩니다. 첫 번째 경로는 구동기 3-7에 전력을 공급하고, 두 번째 경로는 0-2에 전력을 공급합니다.
* 메인보드는 외부 구동기를 지원합니다.     <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 리미트 포트

* 마지막 구동기 포트(Drive7)를 제외하고는 모든 구동기 포트의 리미트는 직접 리미트 포트에 연결됩니다.
* 리미트 포트가 사용할 수 없을 경우 리미트 포트를 변경하거나 구동기의 리미트 핀을 자르세요.
