# BL 펌웨어

    :::tip

    * 주의: 출하 시 이미 `부트로더`가 프로그래밍되어 있으므로 다시 프로그래밍하지 않아도 됩니다.

    :::

## 프로그래밍 모드 진입


    * 프로그래밍 모드에 진입하는 두 가지 방법이 있습니다.
    * 첫 번째 방법: 메인보드를 완전히 전원을 끈 상태에서, BOOT 핀을 단락시키고 데이터 전송용 케이블의 한쪽을 메인보드에, 다른 쪽을 장치에 연결합니다.
    * 두 번째 방법: 데이터 전송용 케이블의 한쪽을 메인보드에, 다른 쪽을 장치에 연결한 후, BOOT 핀을 단락시키고 RST 버튼을 누른 다음 RST 버튼을 놓은 후 마지막으로 BOOT 핀의 단락을 해제합니다.
