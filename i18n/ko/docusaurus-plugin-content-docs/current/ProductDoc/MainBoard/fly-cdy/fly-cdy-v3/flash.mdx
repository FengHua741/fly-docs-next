# 펌웨어 컴파일 및 플래시

    :::tip

    * CDV는 기본적으로 TF 카드를 사용하여 펌웨어를 플래시합니다.
    * 용량이 `32GB` 이하인 TF 카드를 준비하십시오.
    * 불량 메모리 카드는 플래시 실패의 원인이 될 수 있으므로, 아래에서 권장하는 브랜드의 TF 카드를 사용하십시오.
    * TF 카드의 전송 속도는 최소 `class10` 이상을 권장하며, SanDisk 등 브랜드 제품을 추천합니다.

    :::
