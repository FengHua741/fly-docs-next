# CDY V3_V2

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-cdy/fly-cdy-v3/img/cdy.webp').default} size="80%" align="center" />

## 제품 특징

    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * 구동 전압 선택 지원, 기본값은 5V
    * RRF 펌웨어 지원

## 제품 사양

    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연시간이 줄어들며 연결이 더 안정적입니다
    * 펌웨어: Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 디스플레이: 시리얼 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Reprap 펌웨어 사용 시, 확장 ESP32 Wi-Fi 모듈을 지원하며 1.6m/s를 지원합니다
    * 독특한 PWM 팬 MOS 보드 설계, 손상 시 직접 교체 가능하여 고객의 수리 난이도를 줄입니다
    * PCB는 2OZ 구리 두께를 채택하여 더 큰 전류를 지원할 수 있습니다
    * 3개의 ADC 인터페이스로 최대 3개의 열감지기를 지원합니다
    * 6개의 IO 인터페이스로 평탄화 및 제한 기능을 사용할 수 있습니다
    * 4개의 히팅 엘리먼트 인터페이스로 최대 4개의 히팅 엘리먼트를 지원합니다
