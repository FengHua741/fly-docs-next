    <CAN />

    * 마이크로 컨트롤러 시작 시 설정할 GPIO 핀 선택, 입력: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/can.webp').default} size="100%" align="center" />
    * `Q` 키를 누르면 **Save configuration** 메시지가 나타나고, 그 후 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었으며 명령줄 인터페이스로 돌아왔습니다.
    * 아래 명령어를 입력하여 빌드를 시작합니다. 시간이 다소 소요됩니다.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 최종적으로 다음 내용이 출력되면 빌드 성공입니다.

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
