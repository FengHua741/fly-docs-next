# 펌웨어 프로그래밍 안내

:::danger 주의
* 메인보드 펌웨어 프로그래밍은 Katapult를 통해 Klipper를 프로그래밍해야 합니다.
* 메인보드에 이미 Katapult가 프로그래밍되어 있다면 마지막에 Katapult 펌웨어를 프로그래밍할 필요가 없습니다. 이것은 메인보드의 Katapult 펌웨어 프로그래밍 실패를 예방하기 위한 것입니다.
* 재설정 버튼을 두 번 빠르게 누른 후 메인보드에 LED가 깜빡이는지 확인하세요. 만약 LED가 깜빡인다면 메인보드가 Katapult 펌웨어로 업데이트된 것입니다.
:::

## 어떤 펌웨어를 선택해야 하는지 판단하는 방법

    :::tip

    * 주의: 하나의 펌웨어만 설치하면 됩니다.

    :::
    
    * E3 V2는 다음 방법으로 호스트 컴퓨터와 연결할 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB 펌웨어</Button>
        * 장점: 통신이 신뢰성 있고 안정적입니다.
        * 단점: 여러 USB 장치가 있을 경우 데이터 전송에 영향을 줄 수 있습니다.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB 브리지 펌웨어</Button>
        * 장점: 여러 장치를 연결할 수 있으며 USB 포트를 차지하지 않습니다.
        * 단점: KlipperScreen과 함께 사용할 때 리부팅 시간 초과 등의 오류가 발생할 수 있습니다.
