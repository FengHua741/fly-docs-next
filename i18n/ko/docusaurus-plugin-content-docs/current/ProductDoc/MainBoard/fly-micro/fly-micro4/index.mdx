# Micro4

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/micro.webp').default} size="80%" align="center" />

## 제품 특징

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * CAN 브리지 지원
    * 내장 4개의 2209
    * 무한 위치 리셋을 위한 디버그 기능 핀이 내장되어 있어, 점선 케이블을 간단히 꽂아서 사용 가능

## 제품 사양

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * 펌웨어: Klipper
    * 12-24V DC 전원 공급을 지원하여 전원 공급이 더 안정적이며, 다운 변환 모듈을 제거할 수 있습니다.
    * 드라이버 동작 모드 지원: UART, STEP/DIR
    * 확장 인터페이스: BLTouch (Servos, Probe), 근접 스위치 X1
    * 팬: 2채널 제어 가능한 팬 X2, 상시 작동 팬 X1
    * 온도 센서: 2채널 NTC 100K
    * 열 출력 인터페이스: 2채널 제어 가능한 인터페이스, 최대 10A 지원
