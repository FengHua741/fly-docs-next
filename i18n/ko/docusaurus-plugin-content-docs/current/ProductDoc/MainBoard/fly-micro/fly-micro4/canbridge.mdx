# CAN브리지

CAN버스는 실시간 응용 프로그램을 위한 직렬 통신 프로토콜 버스로, 이중 와이어를 사용하여 신호를 전송할 수 있으며, 세계적으로 가장 널리 사용되는 현장 버스 중 하나입니다. CAN 프로토콜은 자동차 내의 다양한 요소 간의 통신에 사용되어 비싸고 무거운 배선 하arness를 대체합니다. 이 프로토콜의 강력함으로 인해 다른 자동화 및 산업 응용 분야에도 활용됩니다. CAN 프로토콜의 특징에는 전체성 있는 직렬 데이터 통신, 실시간 지원, 최대 1Mb/s의 전송 속도, 11비트 주소 지정 및 오류 검출 기능이 포함됩니다.
3D 프린터에서는 CAN 버스를 사용하여 프린트 헤드와 메인 보드 사이의 연결 배선 수를 줄입니다. 원래 십여 개의 배선이 필요했지만, CAN을 사용하면 단지 4개의 배선만 필요하게 되어 배선 수를 크게 줄이고 배선 난이도를 낮춥니다.
이 장에서는 CAN 브리지를 사용하는 방법에 대해 간략히 설명합니다.

## 도구 보드는 CAN을 통해 메인 보드에 연결됩니다

* 도구 보드 전원선은 CAN 포트에 연결되는 것이 아니라 12V-24V 전원선에 연결됨에 주의하십시오.

* 120Ω 점프선이 연결되어야 함에 주의하십시오.

## 메인 보드 펌웨어 업데이트

* D5 메인 보드를 플래시할 때 `USB CAN 브리지 펌웨어 구성`이어야 합니다.
