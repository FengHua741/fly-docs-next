# 다이얼 소개

:::tip
FLY c8은 다이얼을 사용하여 상위 기계에 연결됩니다. 만약 다이얼 위치가 잘못되면 정상적으로 사용할 수 없습니다.
:::

:::danger 주의
다이얼 스위치는 정확한 위치를 선택해야만 상위 기계에 정상적으로 연결될 수 있음을 유의하시기 바랍니다.
:::
