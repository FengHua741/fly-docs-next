# 스위치 설정 소개

:::tip
FLY c8은 스위치 설정을 통해 상위 기기와 연결됩니다. 만약 설정 위치가 잘못되면 정상적으로 사용할 수 없습니다.
:::

:::danger 주의
스위치를 지정된 위치에 맞춰 올바르게 설정해야 상위 기기와 정상적으로 연결될 수 있습니다.
:::

## 내장 상위 기기 연결

    * 스위치 `1`, `2`를 오른쪽으로 돌려 하위 기기와 상위 기기를 내장 USB로 연결합니다.
    * 스위치 `3`, `4`를 왼쪽으로 돌리지 않으면 하위 기기와 상위 기기 연결에 문제가 발생할 수 있습니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * 이 방법대로 스위치를 조작한 후 아래쪽 USB 포트는 하위 기기에서 사용 중이므로 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## 외부 상위 기기 연결

    * 스위치 `3`, `4`를 오른쪽으로 돌려 하위 기기와 상위 기기를 내장 USB로 연결합니다.
    * 스위치 `1`, `2`를 왼쪽으로 돌리지 않으면 하위 기기와 상위 기기 연결에 문제가 발생할 수 있습니다.
