# Klipper 전원 차단 후 다시 인쇄 기능

- Klipper 전원 차단 후 재전원 시 Klipper의 인쇄 상태를 자동으로 복구할 수 있는 기능입니다.
- C5는 KPPM이 필요하지 않습니다.
- Z축이 이동하는 전원이 차단된 기종에는 적용되지 않습니다.
