# Klipper 전원 단절 후 이어하기 기능

## 개요

- Klipper 전원 단절 후 이어하기 기능은 Klipper이 전원이 끊겼다가 다시 켜졌을 때, 인쇄 상태를 자동으로 복구할 수 있는 기능입니다.
- 이 기능은 안전 전원 모듈과 함께 사용해야 합니다.
- Z축이 전원이 끊겼을 때 이동하는 기종에서는 지원되지 않습니다.

## 설정

### 전원 단절 종료 비활성화

    :::danger 주의사항
    - 전원 단절 종료를 비활성화해야 하며, 그렇지 않으면 전원 단절 후 이어하기를 사용할 수 없습니다.
    - 전원 단절 후 이어하기에는 진행 상태를 저장한 후 자동으로 전원이 꺼지는 기능이 포함되어 있습니다.
    :::

- 브라우저 주소 표시줄에 장치의 IP 주소를 입력합니다. 예: `http://192.168.6.179`
- 설정 페이지로 이동합니다.
    - 장치의 IP 주소를 브라우저에서 열기, 예: `http://192.168.1.2/`
    - Fluidd는 아래 왼쪽 그림처럼 `체크 해제` -> `숨겨진 파일 및 폴더 필터링` 을 수행합니다.

### plr.cfg 설정 파일

- 프린터 설정 페이지에서 `plr.cfg` 파일을 찾습니다.
- 내용을 모두 지우고 아래의 설정을 붙여넣습니다.
- 설정 파일 내용은 다음과 같습니다:

    ```cfg
    [mcu host]   
    serial: /tmp/klipper_host_mcu
    
    [power_loss_resume]
    power_pin: !host:gpiochip0/gpio260 # 안전 전원 모듈의 전원 끄기 핀, 상위 컴퓨터의 PA21 핀에 연결
    is_shutdown: True # 전원 종료 여부, 기본값은 활성화
    paused_recover_z: -2.0 # 일시 중지된 상태에서 정지했을 때 재개 시 Z 축 이동 거리, 기본값은 이동하지 않음
    start_gcode:
        # 계속 인쇄 시작 전에 실행되는 gcode
        # 전원이 끊기기 전에 저장된 모든 매개변수는 {PLR}를 통해 얻을 수 있습니다
        # M118 {PLR}를 사용하여 모든 사용 가능한 매개변수 출력
        M118 인쇄 시작: {PLR.print_stats.filename}
        M118 중단 위치: X:[{PLR.POS_X}] Y:[{PLR.POS_Y}] Z:[{PLR.POS_Z}] E:[{PLR.POS_E}]
        M140 S{PLR.bed.target}      ; 열판 온도 설정
        M104 S{PLR.extruder.target-10}  ; 엑스트루더 온도 설정
        M109 S{PLR.extruder.target-10}  ; 엑스트루더가 설정된 온도까지 대기
        G91                             ; 상대 좌표
        G1 Z2 F100                      ; Z축 상승, X,Y 리셋 준비
        G90                             ; 절대 좌표
        G28 X Y                         ; X,Y 리셋
        M140 S{PLR.bed.target}          ; 열판 온도 설정
        M104 S{PLR.extruder.target}     ; 엑스트루더 온도 설정
        M190 S{PLR.bed.target}          ; 열판이 설정된 온도까지 대기
        M109 S{PLR.extruder.target}     ; 엑스트루더가 설정된 온도까지 대기
        M83                             ; 상대 엑스트루더
        # G1 E0.5 F400                  ; 약간의 물질 엑스트루딩
    layer_count: 2 # 지정된 층수만큼 재개한 후 layer_change_gcode 실행
    layer_change_gcode:                
        # {layer_count}층만큼 재개한 후 실행할 gcode
        M118 인쇄 속도 복원
        M106 S{PLR.fan_speed}             ; 냉각 팬 켜기
        M220 S{PLR.move_speed_percent}    ; 요청 속도 퍼센트 설정
        M221 S{PLR.extrude_speed_percent} ; 요청 엑스트루드 속도 퍼센트 설정
    shutdown_gcode:
        # 전원 종료 전에 실행할 gcode
        M118 전력 부족, 전원 종료
        # M112 ; 긴급 정지

    ```

    :::warning 주의

    - 위 설정 파일의 `start_gcode` 매크로는 실제 기계 환경에 따라 수정해야 할 수 있습니다.

    :::

- 설정 파일을 저장한 후
- `printer.cfg` 파일을 열고 가장 앞에 다음 내용을 추가합니다:

    ```cfg

    [include plr.cfg]

    ```

- 오른쪽 상단의 저장 및 재부팅 버튼을 클릭하면 됩니다.
- 이제 Klipper 전원 단절 후 이어하기 기능이 설정 완료되었습니다.

## 테스트

- 임의의 파일을 인쇄하고, 인쇄 도중 `긴급 정지` 버튼을 클릭하여 전원 단절을 시뮬레이션합니다.
- 다시 `펌웨어 재시작`을 클릭하고 Klipper이 정상적으로 연결될 때까지 기다립니다.
- 웹페이지 또는 KlipperScreen에서 팝업 메시지가 나타나면 전원 단절 후 이어하기 기능이 정상 작동하는 것입니다.
- 이후 실제 전원 단절 상황에서도 테스트할 수 있습니다.
