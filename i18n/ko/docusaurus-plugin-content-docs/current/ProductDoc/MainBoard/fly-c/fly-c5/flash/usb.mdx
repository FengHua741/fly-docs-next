    ### USB 펌웨어 프래싱
        :::tip 팁
        - FLYOS-FAST 시스템에는 미리 USB 펌웨어가 컴파일되어 있으므로 별도로 손수 컴파일하지 않아도 됩니다. 아래 명령어를 실행하면 바로 프래싱할 수 있습니다.
        - 또한 `수동으로 USB 펌웨어 컴파일`도 선택 가능합니다.
        :::

        * SSH를 통해 상위 기기에 아래 명령어를 입력하면 됩니다.
        ```bash
        fly-flash -d c5 -h -f /usr/lib/firmware/klipper-h723-128k-usb.bin
        ```
