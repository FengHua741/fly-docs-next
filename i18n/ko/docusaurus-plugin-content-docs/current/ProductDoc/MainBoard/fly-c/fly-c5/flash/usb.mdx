# USB 펌웨어 컴파일


## 듀얼스위치 확인

    :::danger 주의
    - 정상적인 사용 시, `듀얼스위치를 위쪽으로 올려주세요`

    ### USB 펌웨어 플래싱
        :::tip 팁
        - FLYOS-FAST 시스템에서는 미리 USB 펌웨어를 한 번 컴파일해 두었습니다. 따라서 수동으로 펌웨어를 컴파일할 필요 없이 아래 명령어를 실행하면 플래싱이 가능합니다.
        - 또한 `수동으로 USB 펌웨어 컴파일`을 선택하여 직접 펌웨어를 컴파일할 수도 있습니다.
        :::

        * SSH를 통해 상위 기기에 연결한 후 아래 명령어를 입력하세요.
        ```bash
        fly-flash -d c5 -h -f /usr/lib/firmware/klipper-h723-128k-usb.bin
        ```
