    ### USB 펌웨어 플래시
        :::tip 팁
        - FLYOS-FAST 시스템에는 미리 USB 펌웨어가 컴파일되어 있으므로 수동으로 펌웨어를 컴파일할 필요가 없습니다. 아래 명령어를 실행하여 플래시할 수 있습니다.
        - 또는 `수동으로 USB 펌웨어 컴파일` 선택하여 직접 펌웨어를 컴파일할 수 있습니다.
        :::

        * SSH로 상위 컴퓨터에 연결한 후 아래 명령어를 입력합니다.
        ```bash
        fly-flash -d c5 -h -f /usr/lib/firmware/klipper-h723-128k-usb.bin
        ```
