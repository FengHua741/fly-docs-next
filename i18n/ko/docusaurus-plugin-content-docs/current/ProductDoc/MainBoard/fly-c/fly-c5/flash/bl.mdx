# BL 펌웨어

    :::팁

    * 공장 출하 시 이미 `부트로더`가 미리 설치되어 있으므로 다시 설치하지 않으셔도 됩니다.

    :::

## 플래싱 모드 진입

    :::팁

    * 왼쪽은 메인보드의 RST, 오른쪽은 메인보드의 BOOT입니다.

    :::

    * 플래싱 모드에 진입하는 방법은 두 가지가 있습니다.
    * 첫 번째 방법: 메인보드를 완전히 전원을 끈 상태에서 BOOT를 길게 누른 후 데이터 전송이 가능한 데이터 케이블 한쪽을 메인보드에 연결하고 다른一侧을 장치에 연결합니다. 적합한 경우 WINDOWS
    * 두 번째 방법: 데이터 전송이 가능한 데이터 케이블 한쪽을 메인보드에 연결하고 다른一侧을 장치에 연결한 후 BOOT를 길게 누르고 명령어를 입력합니다. 적합한 상위 기기
