# 와이파이 연결하기

## C5에 안테나 설치하기

* 빨간 화살표가 가리키는 부분이 와이파이 안테나 인터페이스입니다.
* 와이파이 안테나 인터페이스는 `ipex1`입니다.
* 안테나를 C5에 설치하지 않으면 신호 강도가 약해져 네트워크에 연결할 수 없을 수 있습니다.
