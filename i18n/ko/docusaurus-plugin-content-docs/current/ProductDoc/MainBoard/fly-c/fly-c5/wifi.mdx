# WIFI 연결하기

## C5에 안테나 설치하기

* 빨간 화살표가 가리키는 부분이 WIFI 안테나 인터페이스입니다.
* WIFI 안테나 인터페이스는 `ipex1`입니다.
* 안테나를 C5에 설치하지 않으면 신호가 약해져 네트워크에 연결되지 않을 수 있습니다.
