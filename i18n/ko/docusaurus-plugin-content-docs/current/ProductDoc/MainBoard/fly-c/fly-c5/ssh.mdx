# 상위 컴퓨터 SSH 연결

* 장치에 시리얼 또는 WIFI를 사용하여 연결할 수 있습니다.
* 먼저 **MobaXterm_Personal** 또는 기타 **SSH 터미널 도구**를 준비해 주세요.
* SD 카드 또는 M2WE에 **대응하는 상위 컴퓨터 시스템**을 번역했는지 확인하십시오.
* **상위 컴퓨터가 정상적으로 시작되지 않으면 드라이버, 리미터, 팬 등 외부 장치를 제거하세요!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 컴파일할 수 있도록 하세요!**
* **펌웨어를 컴파일하려면 WIFI 또는 유선 연결을 통한 상위 컴퓨터 연결 튜토리얼을 참조하세요.**

        ## 상위 컴퓨터 SSH에 시리얼로 연결하기

        * USB Type-c 케이블을 사용하여 FLY 상위 컴퓨터와 컴퓨터를 연결합니다.
        * 다른 상위 컴퓨터의 연결 방법은 각자 알아보세요!

        <Tabs>
            <TabItem value="c5" label="FLY-C5" default>
                * FLY-C5 포트

        * 메인보드와 컴퓨터가 정상적으로 연결된 경우, 장치 관리자에서 CH340 포트를 볼 수 있습니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/pc-device-mgr.webp').default} size="40%" align="center" />
        * `MobaXterm`을 열고 앞서 장치 관리자에서 본 포트를 선택하도록 설정합니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        * 잠시 기다리면 이 화면이 나타나면 부팅이 완료된 것입니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-5.webp').default} size="40%" align="center" />

        **Enter 후에는 아래와 같은 내용이 나타납니다.**

        * IP를 확인하려면 SSH에서 다음 명령어를 입력하고 Enter를 누릅니다: `ip a | grep inet`. 라우터와 동일한 세그먼트의 IP를 찾아서 접속하면 됩니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-8.webp').default} size="40%" align="center" />

        ## 상위 컴퓨터 SSH에 네트워크 IP로 연결하기

        * 상위 컴퓨터가 연결된 **라우터 관리자 페이지**에 로그인하세요.
        * 시스템이 **정상적으로 시작되었는지** 확인하세요.
        * **상위 컴퓨터의 사용자 이름과 비밀번호를 알고 있는지** 확인하세요. FLY 시스템 기본 사용자 이름은 **fly**, 기본 비밀번호는 **mellow**입니다.
        * 메인보드가 시작되고 네트워크에 연결된 경우 IP를 통해 SSH에 연결할 수 있습니다.
        * MobaXterm을 열고 앞서 본 IP를 입력하거나 라우터 관리자 페이지에서 확인하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기에서 4단계에 입력해야 하는 것은 상위 컴퓨터의 사용자 이름입니다.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 처음 연결 시 이 창이 나타날 수 있으며, "Accept"를 클릭하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호를 입력하라는 메시지가 표시되면 상위 컴퓨터의 비밀번호를 입력하고 Enter를 누릅니다.

        **주의: 비밀번호 입력 시 표시되지 않습니다. 입력 후 바로 Enter를 누르세요.**

        FLY 시스템 기본값: 사용자 이름 **fly**, 비밀번호 **mellow**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />
