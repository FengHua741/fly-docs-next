# 메인보드 주의사항

## 전원 지시등
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/power.webp').default} size="80%" align="center" />
* TYPE-C를 통해 전원만 공급하고 다른 메인보드 전원에 연결하지 않을 경우 최소 두 개의 등이 켜집니다. `3.3V`와 `5V`
* 메인보드 전원에 연결 후에는 메인보드 전원등이 `3.3V`, `5V`, `VCC`가 켜져야 합니다.

### 문제 해결

:::danger 유의사항

* 드라이버, 리미트 스위치, 12864와 같은 외부 장치가 아무것도 연결되어 있지 않은 경우
* TYPE-C를 통해만 전원을 공급할 때 최소한 `3.3V`는 항상 켜져 있어야 합니다.

:::

* 메인보드 전원과 드라이버 전원을 연결한 후에도 `5V`등이 켜지지 않는 경우 고객 서비스로 문의하십시오.

## 드라이버

* 메인보드는 외부 드라이버를 지원합니다. <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/ModuleDrive/servo-drive">연결 및 설정 가이드</Button>

## 리미트 포트

* 신호선 연결 순서를 주의하십시오. 세선 리미트는 5V, GND, S를 잘못 연결하면 메인보드가 손상될 수 있습니다.
* 두선 리미트의 경우 S와 GND만 연결하면 됩니다.
* PL08과 다른 근접 스위치는 신호 포트 외부에 이진 다이오드를 연결하여 전압을 3.3V 아래로 낮춰야 합니다.
