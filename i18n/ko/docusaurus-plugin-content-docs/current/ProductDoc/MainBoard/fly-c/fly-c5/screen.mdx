# 화면 연결 및 구성

## 구성 방법


- 브라우저의 주소 표시줄에 장치의 IP 주소를 입력합니다. 예: `http://192.168.6.179`
- 구성 페이지로 이동
    - 브라우저에서 장치 IP 주소 열기, 예: `http://192.168.1.2/`
    - fluidd는 아래 왼쪽 그림과 같이 `체크박스 해제` -> `숨겨진 파일 및 폴더 필터링`.
    - mainsail은 아래 오른쪽 그림과 같이 `체크박스 선택` -> `숨겨진 파일 표시`.
    
    <table>
      <tr>

        </td>

          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />

    - 이제 `.flyos-config` 폴더를 볼 수 있으며, 이 폴더 내에는 `sys-config.conf` 파일이 있습니다.
    - `sys-config.conf` 파일은 가동식 디스크 `FlyOS-Conf`의 구성 파일인 `config.txt`의 소프트 링크입니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - `sys-config.conf` 파일을 열고 `screen=`을 찾아 화면 유형 수정
    - 번호❶ `klipper_screen` 다음의 `0`을 `1`로 변경하여 `klipper_screen` 서비스 활성화
    - 번호❷ `screen` 다음의 구성 항목을 사용할 화면으로 수정하고 `저장` -> `닫기` 후 재부팅하면 됩니다
    <ImageView image={require('@site/docs/General/flyos-fast/img/screen.webp').default} size="90%" align="center" />

    - 주의: `sys-config.conf` 파일의 구성 항목 형식은 `key=value`이며 `key: value`로 수정하지 마십시오.
    - 수정 및 저장 후 시스템을 재부팅하면 적용됩니다.
