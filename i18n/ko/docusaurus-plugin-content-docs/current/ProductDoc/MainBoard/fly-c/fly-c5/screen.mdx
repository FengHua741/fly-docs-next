# 화면 연결 및 구성

## 구성 방법


- 브라우저의 주소 표시줄에 장치의 IP 주소를 입력합니다. 예: `http://192.168.6.179`
- 구성 페이지로 이동합니다.
    - 브라우저에서 장치의 IP 주소를 열어주세요. 예: `http://192.168.1.2/`
    - fluidd의 경우 아래 왼쪽 그림과 같이 `체크박스 해제` -> `숨김 파일 및 폴더 필터링`을 선택합니다.
    - mainsail의 경우 아래 오른쪽 그림과 같이 `체크박스 선택` -> `숨겨진 파일 표시`를 선택합니다.

    - 이제 `.flyos-config` 폴더를 볼 수 있습니다. 이 폴더 내에는 `sys-config.conf` 파일이 포함되어 있습니다.
    - `sys-config.conf` 파일은 이동식 디스크 `FlyOS-Conf`의 구성 파일인 `config.txt`의 소프트 링크입니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - `sys-config.conf` 파일을 열고 `screen=` 항목을 수정하여 화면 유형을 설정합니다.
    - 순번 ❶의 `klipper_screen` 뒤의 `0`을 `1`로 변경하여 `klipper_screen` 서비스를 활성화합니다.
    - 순번 ❷의 `screen` 뒤의 구성 항목을 사용할 화면으로 변경하고 `저장` -> `닫기` 후 재부팅하면 됩니다.
    <ImageView image={require('@site/docs/General/flyos-fast/img/screen.webp').default} size="90%" align="center" />

    - 주의: `sys-config.conf` 파일의 구성 항목 형식은 `key=value`이며 `key: value`로 변경하지 마십시오.
    - 수정 및 저장 후 시스템을 재부팅하면 적용됩니다.

## C5 화면 연결

    :::danger 주의사항
    - 화면을 연결할 때 전원을 먼저 끈 후 연결하세요. 이를 어길 경우 화면이 손상될 수 있습니다.
    - 화면 연결 시 fpc 선 방향을 주의해야 합니다. 잘못된 방향으로 연결하면 화면이 손상될 수 있습니다.
