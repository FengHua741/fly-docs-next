## 제품 특징

    * 32비트 168MHz ARM Cortex-M0+ 시리즈 STM32F407VGT6 마스터 칩을 채택했습니다.
    * TMC 드라이버를 탑재한 SPI 및 UART 작업 모드 지원
    * 간단한 점퍼 케이블 플러그인만으로 무제한 리미트 스위치 작동이 가능한 DIAG 기능 핀 탑재
    * CNC 팬 24V, 12V, 5V 전압 선택 가능, 낮은 전압 모듈 구매의 번거로움을 줄였습니다.
    * 새로운 플래시 쓰기 방식으로, FLY-Tools를 통해 직접 메인보드 펌웨어를 업데이트할 수 있습니다.
    * 다중 전원 관리, 포트, 리미트, 드라이버 등이 단락되어도 상위 컴퓨터 작동에 영향을 주지 않습니다.
    * RGB 인터페이스 예약, 12V, 5V 전압 선택 가능
    * SPI 인터페이스 예약, 가속도 센서를 외부로 연결하여 가속도 보상을 할 수 있습니다.
    * TFT 화면과 HDMI 화면 지원, 한 줄 통과
    * 시스템에 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 공명 보상 플러그인 numpy와 FLY-Tools가 미리 설치되어 있어 복잡한 작업 없이 사용자에게 쉽게 다가갈 수 있습니다( TF 카드 또는 M2WE 구매 필요)

## 제품 사양

### 상위 컴퓨터
    * CPU: 고성능 올vív H5 칩, 4코어 64비트 Cortex-A53
    * GPU: 고성능 6코어 Mali 450, 픽셀 채우기 속도는 2.7G픽셀/s를 초과합니다.
    * RAM: 1GB DDR3 (GPU와 공유)
    * ROM: 최대 128GB SDCARD 지원
    * 외부 장치: Spi x1, TFT x1, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1, 3.5mm 오디오 인터페이스 x1
    * 인터페이스: 내장 M.2 인터페이스, WIFI, eMMC 등의 확장을 지원합니다. (개인 프로토콜, FLY π 전용 M.2 장치가 아닌 것에 연결하지 마십시오.)
    * 5V 팬 컨트롤 가능한 ZH1.5 인터페이스가 예약되어 있어 상위 컴퓨터 냉각에 사용될 수 있습니다.
    * TFT 화면과 HDMI 화면 FPC 한 줄 통과 지원
    * CAN 브리지 인터페이스 예약
    * 시스템에 klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, 공명 보상 플러그인 numpy와 FLY-Tools가 미리 설치되어 있어 복잡한 작업 없이 사용자에게 쉽게 다가갈 수 있습니다( TF 카드 또는 M2WE 구매 필요)

### 하위 컴퓨터

    * MCU: STM32F407VGT6
    * 펌웨어:  KLIPPER/RRF/Marlin
    * 12-24V DC 전원 공급 지원, 전원 공급 안정화, 전압 강하 모듈 제거
    * 열 침대 포트 최대 출력 전류:  15A
    * 히팅 로드 포트 최대 출력 전류:  6A
    * 팬 포트 최대 출력 전류:  1A
    * 드라이버 작업 모드 지원: SPI, UART, STEP/DIR
    * 지원 드라이버: 외부 TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 등 지원
    * 확장 인터페이스: BLTouch(Servos, Probe), RGB X2, UART X1, 12864 X1
    * 팬: 2선 CNC 팬 X6 (24V, 12V, 5V 선택 가능)
    * 온도 센서: 3개의 NTC 100K
    * 지원하는 기계 구조: Cartesian, Delta, Kossel, Ultimaker, CoreXY
