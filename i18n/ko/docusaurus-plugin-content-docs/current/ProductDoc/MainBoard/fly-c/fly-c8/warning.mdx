# 주요 사항

## 전원 표시등
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/power.webp').default} size="80%" align="center" />
    * TYPE-C로만 전원을 공급하고 다른 메인보드 전원에 연결하지 않을 때 최소 두 개의 빛이 켜집니다. `3.3V`와 `5V`.
    * 메인보드 전원에 접속 후에는 메인보드 전원 램프가 `3.3V`, `5V`, `VCC`를 켭니다.

### 문제 해결


    :::danger 주의사항

    * 드라이버, 리미트 스위치, 12864 등 외부 장치가 없는 경우
    * TYPE-C로만 전원을 공급할 때 최소한 `3.3V`가 항상 켜집니다.

    :::

    * 메인보드 전원과 드라이버 전원을 연결한 후에도 `5V` 라이트가 켜지지 않으면 고객 서비스에 문의해야 합니다.

## 드라이버

    * 메인보드는 외부 드라이버를 지원합니다. <Button variant="contained" disableElevation href="/docs/ProductDoc/ModuleDrive/servo-drive">접선 및 구성 튜토리얼</Button>

## 리미트 포트

    * 신호 순서를 유의하십시오. 3선 리미트는 5V, GND, S가 잘못 연결되면 메인보드가 손상될 수 있습니다.
    * 2선 리미트는 S와 GND만 연결하면 됩니다.
    * PL08 및 기타 근접 스위치는 신호 포트 외부에 다이오드를 연결하여 전압을 3.3V 미만으로 낮춰야 합니다.
