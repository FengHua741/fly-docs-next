# 다이얼 소개

:::tip
FLY c8은 다이얼 스위치를 통해 상위 기계에 연결되며, 위치가 맞지 않으면 정상적으로 사용할 수 없습니다.
:::

:::danger 주의
다이얼 스위치는 지정된 위치에 설정해야만 상위 기계와 정상적으로 연결될 수 있습니다.
:::

## 내장 상위 기계 연결

    * 다이얼 스위치 `1`, `2`를 오른쪽으로 돌려 하위 기계와 상위 기계를 내장 USB로 연결합니다.
    * 다이얼 스위치 `3`, `4`를 왼쪽으로 돌리지 않으면 하위 기계와 상위 기계 연결에 영향을 미칠 수 있습니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * 이 방법대로 다이얼 스위치를 조작한 후 아래쪽 USB 포트는 하위 기계에서 점유됩니다. 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## 외부 상위 기계 연결

    * 다이얼 스위치 `3`, `4`를 오른쪽으로 돌려 하위 기계와 상위 기계를 내장 USB로 연결합니다.
    * 다이얼 스위치 `1`, `2`를 왼쪽으로 돌리지 않으면 하위 기계와 상위 기계 연결에 영향을 미칠 수 있습니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
