# 패치 코드 소개

:::tip
FLY 제미니는 패치 코드를 통해 상위 기기를 연결합니다. 패치 코드 위치가 맞지 않으면 정상적으로 사용할 수 없습니다.
:::

:::danger 주의
패치 코드 스위치가 지정된 위치에 켜져 있어야만 상위 기기와 정상적으로 연결됩니다.
:::

## 내장 상위 기기 연결

    * 패치 코드 스위치 `1`, `2`를 오른쪽으로 돌려 하위 기기와 상위 기기를 내장 USB를 통해 연결합니다.
    * 패치 코드 스위치 `3`, `4`를 왼쪽으로 돌려야 하위 기기와 상위 기기 연결에 영향을 미치지 않습니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * 이 방법으로 패치 코드 스위치를 조작한 후, 아래쪽 USB 포트는 하위 기기에 의해 점유됩니다. 다른 장치를 연결하지 마십시오.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## 외부 상위 기기 연결

    * 패치 코드 스위치 `3`, `4`를 오른쪽으로 돌려 하위 기기와 상위 기기를 내장 USB를 통해 연결합니다.
    * 패치 코드 스위치 `1`, `2`를 왼쪽으로 돌려야 하위 기기와 상위 기기 연결에 영향을 미치지 않습니다.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
