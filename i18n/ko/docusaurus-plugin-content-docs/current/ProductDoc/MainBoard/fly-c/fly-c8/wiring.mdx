## 메인보드 케이블 연결

:::warning 주의사항
웹 페이지에서 이미지를 볼 때 해상도가 높아서 명확하게 보이지 않을 수 있습니다. 이미지 위에 마우스 커서를 올리고 우클릭하여 '이미지 저장'을 선택하여 데스크톱에 저장한 후 자세히 확인해 주세요!
:::

:::warning 경고
구매한 완제품 **3선식 리미트 스위치**를 사용하지 마십시오. 이와 같은 리미트 스위치는 **메인보드 단락**을 유발하여 **메인보드를 손상**시킬 수 있습니다!!! 반드시 연결図에 나와 있는 연결 방식에 따라 연결하십시오!!!
:::
