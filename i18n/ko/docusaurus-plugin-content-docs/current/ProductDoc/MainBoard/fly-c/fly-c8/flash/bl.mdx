# BL 펌웨어

    :::tip

    * 주의: 출⼚시 이미 `bootloader`가 프로그래밍 되어 있으므로 다시 프로그래밍 할 필요가 없습니다.

    :::

## 프로그래밍 모드 진입


    * 프로그래밍 모드에 진입하는 방법은 두 가지입니다.
    * 첫 번째 방법: 메인보드가 완전히 전원이 꺼진 상태에서, BOOT 버튼을 누른 채 데이터 통신용 데이터 케이블을 메인보드와 연결 장치에 각각 연결합니다. (적합한 윈도우즈)
    * 두 번째 방법: 데이터 통신용 데이터 케이블을 메인보드와 연결 장치에 각각 연결한 후, BOOT 버튼을 누른 상태에서 명령어를 입력합니다. (적합한 상위 기기)
