## 안테나 설치

    1. `M2WE` 또는 `2.4G Wi-Fi` 모듈의 안테나를 설치합니다.
    :::warning
    * FLY 상위 기계는 `M2WE` 또는 `2.4G Wi-Fi` 모듈을 사용해야 하며, 그렇지 않으면 Wi-Fi를 사용할 수 없습니다.
    * 주의하세요. 상위 기계는 중국어 이름의 Wi-Fi를 지원하지 않습니다. 또한 2.4G와 5G를 합친 단일 Wi-Fi를 지원하지 않습니다.
    :::
    * 빨간 화살표가 가리키는 부분이 Wi-Fi 안테나 인터페이스입니다.
    * Wi-Fi 안테나 포트는 `ipex1`입니다.
    * 안테나를 M2WE에 설치하지 않으면 신호가 약해져 라우터에 연결할 수 없을 가능성이 있습니다.
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - 시스템 카드 설치가 완료되면 다음 단계로 넘어갈 수 있습니다. SSH를 통해 상위 기계에 연결하세요.
    :::

    2. 상위 기계에 전원 공급.  
    **여기까지 왔다면 상위 기계는 이제 시작할 준비가 되었습니다!!**  
    다음 단계를 수행하세요: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">상위 기계 SSH 연결</Button>
