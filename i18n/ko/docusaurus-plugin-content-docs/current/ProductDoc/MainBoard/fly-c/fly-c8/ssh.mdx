# 상위기 SSH 연결


* 장비에 **시리얼** 또는 **WIFI**를 사용하여 연결할 수 있습니다.
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비하세요.
* SD 카드 또는 M2WE에 **해당 상위기의 시스템을 번인**했는지 확인하세요.
* **상위기가 정상적으로 시작되지 않는 경우, 드라이브, 리미트, 팬 등의 주변 장치를 제거하세요!!**
* **마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 편리하게 컴파일하세요!!**
* **펌웨어를 컴파일하려면 WIFI 또는 이더넷 케이블을 통해 상위기에 연결하는 방법을 참조하세요**

        ## 네트워크 IP를 사용하여 상위기 SSH에 연결

        * 상위기가 연결된 **라우터 백엔드**에 로그인하세요.
        * 시스템이 **정상적으로 시작**되었는지 확인하세요.
        * 상위기의 **사용자 이름과 비밀번호를 알고** 있는지 확인하세요, FLY 시스템의 기본 사용자 이름은 **fly**, 기본 비밀번호는 **mellow**입니다.
        * 메인보드가 시작되고 네트워크에 연결되어 있으면 IP를 사용하여 SSH에 연결할 수 있습니다.
        * MobaXterm을 열고 다음과 같이 설정하세요, IP는 앞에서 확인한 것을 입력하거나 라우터 백엔드에서 확인할 수 있습니다.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계에 상위기의 사용자 이름을 입력하세요**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 첫 연결 시 이 창이 팝업됩니다, `Accept`를 클릭하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호를 입력하라는 메시지가 표시되면 상위기 비밀번호를 입력 후 엔터를 누르세요.

        **주의: 여기서 입력한 비밀번호는 보이지 않습니다, 입력 후 바로 엔터를 누르세요**

        FLY 시스템의 기본 사용자 이름은: `fly`, 비밀번호는: `mellow`입니다.
