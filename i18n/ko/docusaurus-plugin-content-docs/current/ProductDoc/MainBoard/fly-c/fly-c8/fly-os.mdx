## 시스템 버닝 준비

* 카드리더기
* 최소 16GB 또는 최대 128GB의 TF 카드, TF 카드의 전송 속도는 Class10 이상이어야 합니다.
* 필요한 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">시스템 이미지</Button> 다운로드
* 필요한 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">번역 도구</Button> 다운로드

## 시스템 버닝

:::info
TF 카드 번역 또는 TF 카드를 통해 M2WE에 번역하는 경우 중 하나만 사용하면 됩니다.
:::

:::info
* 기계에 M2WE와 메모리 카드가 설치된 경우 시스템은 기본적으로 메모리 카드에서 부팅합니다.
* M2WE는 TF 카드로 시스템을 번역하는 것을 권장합니다. USB를 통해 M2WE로 번역하는 것은 드라이버 호환성 등으로 인해 실패할 수 있으므로 [메모리 카드를 통해 M2WE로 번역](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)을 직접 사용하는 것을 권장합니다.
* 다운로드한 시스템 이미지가 일치하는지 주의하십시오. 일치하지 않으면 부팅할 수 없습니다.
:::
