# WIFI에 연결하기


## C8에 안테나 장착

    * 빨간 화살표가 가리키는 곳이 WIFI 안테나 커넥터입니다.
    * WIFI 안테나 포트는 `ipex1` 입니다.
    * 안테나를 C8에 장착하지 않으면 신호가 약해서 인터넷에 연결할 수 없을 수 있습니다.
