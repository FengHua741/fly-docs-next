## 제품 사양

    * 32비트 ARM Cortex-M4 시리즈 168 MHz, STM32F407ZGT6 칩
    * CAN 버스 연결 지원, 데이터 전송이 더 안정적이고 지연 시간이 줄며 연결이 더 안정적입니다
    * 펌웨어: Reprap/Klipper
    * 드라이버 지원: A4988, LV8729, DRV8225, TMC2208/2209/5160 등
    * 드라이버 모드 지원: TMC: UART/SPI
    * 표시: 시리얼 터치 스크린, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Reprap 펌웨어를 사용하면 확장 ESP32 Wi-Fi 모듈을 사용할 수 있으며, 최대 1.6m/s 속도를 지원합니다
    * 고유한 PWM 팬 MOS 보드 설계로, 손상 시 바로 교체 가능하여 고객의 수리 난이도를 줄입니다
    * PCB는 2OZ 구리를 사용하여 더 큰 전류를 지원할 수 있습니다
    * 6개의 ADC 인터페이스, 최대 6개의 열감지기 지원
    * 7개의 IO 인터페이스, 레벨링 및 리미트 스위치에 사용 가능
    * 7개의 히팅 엘리먼트 인터페이스, 최대 7개의 히팅 엘리먼트 지원
