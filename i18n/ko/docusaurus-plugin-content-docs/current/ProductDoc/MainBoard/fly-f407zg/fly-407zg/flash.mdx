# 펌웨어 컴파일 및 플래시

    :::tip

    * 407ZG는 기본적으로 TF 카드를 사용한 펌웨어 플래시를 지원합니다.
    * 용량이 `32GB` 이하인 TF 카드를 준비해 주세요.
    * 부실한 메모리 카드는 플래시 실패의 원인이 될 수 있으므로, 아래에서 권장하는 브랜드의 TF 카드를 사용하는 것이 좋습니다.
    * TF 카드의 전송 속도는 `class10` 또는 그 이상을 권장하며, SanDisk 등과 같은 브랜드를 추천합니다.

    :::

## Klipper 펌웨어 컴파일

    <MakeOperate/>

    <USB />

    * `Q` 키를 누르면 **Save configuration**이 표시되고, 그 후에 `Y` 키를 누릅니다.
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 이제 설정이 저장되었으며 명령줄 화면으로 돌아왔습니다.
    * 아래 명령어를 입력하여 컴파일을 시작합니다. 시간이 다소 걸릴 수 있습니다.
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 마지막에 다음 내용이 출력되면 컴파일이 성공한 것입니다.

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
        
    * 아래는 컴파일 성공 예제입니다.
