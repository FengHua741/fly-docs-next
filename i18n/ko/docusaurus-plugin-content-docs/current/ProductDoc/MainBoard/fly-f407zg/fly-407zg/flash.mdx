# 펌웨어 컴파일 및 플래시

    :::tip

    * 407ZG는 기본적으로 TF 카드를 통해 펌웨어 플래시를 수행합니다.
    * 최대 `32G` 용량의 TF 카드를 준비하십시오.
    * 브랜드가 없는 메모리 카드는 플래시 실패의 원인이 될 수 있으므로, 펌웨어 업데이트 시 아래에 추천된 메모리 카드를 사용하십시오.
    * TF 카드의 전송 속도는 최소 `class10` 급 이상이어야 하며, SanDisk와 같은 브랜드의 TF 카드를 권장합니다.

    :::
