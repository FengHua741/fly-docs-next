# TMC 5160 Pro V1.2

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/5160pro.webp').default} size="80%" align="center" />

## 기본 매개변수

    * 입력 전압 ：24-48V
    * 최대 전류 ：6A
    * 사용 대상 ： Marlin/Klipper/RRF 펌웨어
    * 구동 모드 지원：TMC：SPI
    * 마이크로스텝 보간이 있는 스텝/방향 인터페이스
    * 최고 해상도는 256분할
    * stealthChop2 조용한 작동 및 부드러운 동작을 지원합니다.
    * 중주파수 공진 억제를 지원합니다.
    * spreadCycle 고동적 모터 제어 스위처를 지원합니다.
    * dcStep 부하 관련 속도 제어를 지원합니다.
    * stallGuard2 고정밀 무센서 모터 부하 감지 기능을 지원합니다.
    * coolStep 전류 제어를 지원하여 최대 75%의 에너지 절약 가능

## 메인보드 점퍼 설명

    * TMC5160 Pro는 SPI 점퍼를 사용합니다.
    * FLY 드라이버의 SPI 핀은 왼쪽에서 아래로 네 번째 핀이며, CS 핀은 SPI 핀입니다. 메인보드가 해당 핀 위치에 맞지 않으면 SPI를 사용할 수 없습니다.

## TMC5160 Pro 참고 설정

    :::danger

    * 구동 샘플 저항은 `sense_resistor: 0.033`이므로 잘못 설정하지 마십시오.
    * spi_bus와 spi_software는 둘 중 하나만 선택할 수 있습니다.
    
    ::: 

### SPI 모드 참고 설정

    ```cfg
    [tmc5160 stepper_x]
    cs_pin:
    spi_bus:
    # spi_software_mosi_pin:
    # spi_software_miso_pin:
    # spi_software_sclk_pin: 
    run_current: 1.0                    # 모터 작동 전류 값
    interpolate: False                  # 256마이크로스텝 보간 여부 (켜짐은 True, 꺼짐은 False)
    sense_resistor: 0.033
    stealthchop_threshold: 0
    driver_DISS2G: 1
    driver_DISS2VS: 1
    ```
