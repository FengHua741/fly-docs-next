# KlipperScreen 확장 기능

## KlipperScreen 마우스 커서 표시 방법

    * 프린터 설정의 `KlipperScreen.conf` 파일을 엽니다. 이 파일은 KlipperScreen 설치 후 자동으로 생성됩니다. 없으면 직접 생성하세요.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.webp').default} size="100%" align="center" />
    * 설정 맨 위에 아래 코드를 추가합니다.
    ```cfg
    [main]
    show_cursor: True
    ```
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf-1.webp').default} size="80%" align="center" />
    * 저장하고 재시작하면 됩니다.

## 다른 상위 기기와 원격 연결

    * 프린터 설정의 `KlipperScreen.conf` 파일을 엽니다. 이 파일은 KlipperScreen 설치 후 자동으로 생성됩니다. 없으면 직접 생성하세요.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.webp').default} size="100%" align="center" />
    * 설정 맨 위에 아래 코드를 추가합니다.
    * `VORON-300` 프린터 이름과 `192.168.6.217` 연결할 기기의 IP 주소를 수정할 수 있습니다.
    * 주의: 두 번째 `[printer KlipperScreen]`는 원격 확장 기기를 연결하기 위한 것입니다. 이 설정을 사용하여 디버깅하거나 종료할 수 있습니다.
    :::danger 주의

    * 한 로컬 네트워크 내에서 동작해야 합니다.

    :::

    ```cfg
    # 프린터 및 이름 정의. 이름은 첫 번째 프린터 단어 이후의 아무거나입니다.
    [printer VORON-300]
    # moonraker 호스트/포트를 127.0.0.1 및 7125와 다르게 지정합니다.
    moonraker_host: 192.168.6.217
    # 포트 443 및 7130은 https/wss를 사용합니다.
    moonraker_port: 7125
    # 이 연결이 신뢰할 수 있는 클라이언트 IP로부터 이루어지지 않는 경우 Moonraker API 키를 지정합니다.
    moonraker_api_key: False

    # 프린터 및 이름 정의. 이름은 첫 번째 프린터 단어 이후의 아무거나입니다.
    [printer KlipperScreen]
    # moonraker 호스트/포트를 127.0.0.1 및 7125와 다르게 지정합니다.
    moonraker_host: 127.0.0.1
    # 포트 443 및 7130은 https/wss를 사용합니다.
    moonraker_port: 7125
    # 이 연결이 신뢰할 수 있는 클라이언트 IP로부터 이루어지지 않는 경우 Moonraker API 키를 지정합니다.
    moonraker_api_key: False

    ```
