# 백업 구성 방법

:::danger 주의사항

* 이 튜토리얼은 FLY-Pi, 라즈베리 파이, 오렌지 파이 등 상위 기기용입니다.
* EMMC를 사용하는 경우 플래시 모드로 진입할 수 있어야 하며, 그렇지 않으면 이 튜토리얼이 적용되지 않습니다.
* 이 튜토리얼은 Windows 시스템에서 작업하도록 설계되었습니다. 포맷되지 않은 알 수 없는 디스크를 포맷하지 마십시오!!!
* 포맷하지 마십시오!!! 포맷하지 마십시오!!! 포맷하지 마십시오!!!

:::

## 필요한 소프트웨어 다운로드

    * 클릭 후 다운로드하면 웹 페이지가 컴퓨터에 맞는 버전을 알려줍니다.
    * 소프트웨어는 그린 버전으로 설치할 필요가 없으며 다운로드 후 압축을 풀고 사용할 수 있습니다.
    * <Button variant="contained" disableElevation href="https://www.diskgenius.cn/download.php">DG 하드 드라이브 도구</Button>

## 백업 시작

    :::info
    * 포맷하지 마십시오!!!
    * 포맷한 후에는 복원할 수 없습니다!!!
    * TF 카드는 컴퓨터에 직접 연결하면 됩니다.
    * EMMC는 수동으로 플래시 모드로 진입해야 합니다!!!

:::

### 연결
    * 백업할 시스템을 컴퓨터에 연결합니다.
    * `DiskGenius.exe`를 엽니다.
    * 정상적으로 작동하면 백업할 디스크가 표시됩니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/boot.webp').default} size="80%" align="center" />

### 찾기
    * 왼쪽에서 기본 파티션을 클릭하고 위쪽의 파일 탐색을 클릭한 후 아래쪽에 파일이 표시됩니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/home.webp').default} size="80%" align="center" />
    * `home`을 찾아서 더블 클릭합니다.
    * 들어가면 일반적으로 하나의 폴더만 있고 이름은 시스템 사용자의 이름입니다.
    * 두 개 이상이면 시스템 구성이 있는 폴더를 스스로 확인해야 합니다.
    * 그 다음에 더블 클릭합니다.
    * 들어가서 `printer_data`를 찾습니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/printer-data.webp').default} size="80%" align="center" />
    * `printer_data` 폴더에는 프린팅 파일, Klipper 구성, Klipper 로그 등의 내용이 포함되어 있으므로 전체를 내보낼 수 있습니다.

### 구성 내보내기
    * `printer_data`를 오른쪽 클릭하면 아래의 메시지가 표시됩니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export.webp').default} size="40%" align="center" />
    * 내보낼 디렉토리를 선택하거나 바로 내보낼 수 있습니다.
    * 내보내기 확인
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export1.webp').default} size="80%" align="center" />
    * 내보낸 파일을 열고 `config`를 찾아보세요. 여기에는 시스템 구성이 포함되어 있습니다.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export2.webp').default} size="80%" align="center" />
    * 구성 내보내기가 완료되었습니다.
    * `moonraker` 파일은 다른 시스템에 무분별하게 업로드하지 마십시오. 시스템이 다르면 Klipper가 연결되지 않을 수 있습니다.
