# 상위기기 SSH 연결

::: danger

* 이 문서는 FLY 상위기기에만 적용됩니다. 다른 장치의 경우 참고용입니다.
* 펌웨어를 컴파일하고 플래시하려면 네트워크 연결 튜토리얼을 참조하세요. 시리얼 포트 연결 튜토리얼은 참고하지 마세요.

:::

* 장치에 시리얼 또는 Wi-Fi로 연결할 수 있습니다.
* **MobaXterm_Personal** 또는 다른 **SSH 터미널 도구**를 미리 준비해 주세요.
* SD 카드 또는 M2WE에 해당 상위기기의 시스템이 플래시되었는지 확인하세요.
* 만약 상위기기가 정상적으로 시작되지 않으면 드라이버, 리미터, 팬 등 외부 장치를 제거하세요!!!
* 마지막으로 네트워크를 통해 SSH에 연결하여 펌웨어를 컴파일할 수 있도록 하세요!!
* 펌웨어를 컴파일하려면 Wi-Fi 또는 이더넷 케이블을 통해 상위기기와 연결하는 튜토리얼을 참조하세요.

        ## 상위기기 SSH에 네트워크 IP를 사용하여 연결하기

        * 시스템이 **정상적으로 시작**되었는지 확인하세요.
        * 상위기기와 연결된 **라우터 관리자**에 로그인하세요. (상위기기의 IP 주소를 알고 있다면 필요 없습니다.)
        * 상위기기의 **사용자 이름과 비밀번호**를 알고 있는지 확인하세요. FLY 시스템의 기본 사용자는 **fly**, 기본 비밀번호는 **mellow**입니다.
        * 메인보드가 시작되고 네트워크에 연결되었다면 다음 방법으로 SSH에 연결할 수 있습니다.
        * MobaXterm을 열고 아래와 같이 설정하세요. IP는 앞서 확인한 것을 입력하세요. (라우터 관리자에서도 확인 가능)

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **주의: 여기서 4단계에 상위기기의 사용자 이름을 입력하세요**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * 첫 번째 연결 시 이 창이 나타납니다. `Accept`를 클릭하세요.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * 비밀번호를 입력하라는 메시지가 표시되면 상위기기의 비밀번호를 입력하고 엔터를 누르세요
