# SSH를 통해 네트워크에 연결하기


* 시스템이 시작되었는지 확인하고 SSH를 사용하여 상위 컴퓨터와 시리얼 포트로 연결하십시오. **MobaXterm_Personal**을 사용하는 것을 권장합니다.
* 상위 컴퓨터에 **M2WE** 또는 **2.4G_WiFi 모듈**이 설치되어 있는지 확인하십시오. **Lite2**와 **MiniPad**의 경우 모듈을 설치할 필요가 없습니다.
* **2.4G_WiFi 모듈**, **Lite2**와 **MiniPad** 중 하나를 사용하는 경우에는 2.4G_WiFi만 사용 가능하며 5G_WiFi는 사용할 수 없습니다.
* 네트워크가 듀얼 밴드 통합을 사용하는 경우에는 정상적으로 연결할 수 없으므로 주의하십시오.
* 안테나가 제대로 설치되어 있는지 확인하십시오!!!

 1. SSH 소프트웨어에서 **nmtui** 명령을 입력한 후 Enter 키를 누르면 아래 그림과 같은 화면이 나타납니다.

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. Enter 키를 누른 다음 **Activate a connection**을 선택하고 다시 Enter 키를 누릅니다.

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. 연결할 네트워크를 선택한 후 Enter 키를 누르고 WiFi 비밀번호를 입력하고 **선택 OK**를 눌러 연결을 저장합니다.

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. WIFI에 연결한 후, 키보드에서 **Esc** 키를 몇 번 누른 다음 현재 화면을 종료한 다음 **ip a**를 입력하고 Enter 키를 누릅니다.

 * 그림의 192.168.2.136은 상위 컴퓨터의 IP 주소입니다.

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
