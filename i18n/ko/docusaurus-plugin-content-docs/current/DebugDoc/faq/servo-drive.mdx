# 외장 드라이버 사용 가이드

## 주의사항

:::danger 알림

* 이 튜토리얼은 Klipper를 기반으로 합니다.
* 모듈은 공아노디 접선을 사용합니다.

:::

## 드라이버 분해능 점퍼

    * 어떠한 드라이버 점퍼도 필요하지 않습니다!!!

## 드라이버 내장 스위치
    * 드라이버 내장 스위치는 `3200`을 권장합니다.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/3200.webp').default} size="80%" align="center" />

## Klipper 참조 설정

    * 해당 드라이버 설정을 비활성화해야 합니다. 예를 들어 `[tmc5160 stepper_x]` 또는 `[tmc2209 stepper_x]`.
    * `microsteps:`을 `16`로 설정합니다.
    * `step_pulse_duration: 0.000004`를 추가합니다.
    * 모터 활성화: `enable_pin:!`에서 `!`를 삭제합니다.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/cfg.webp').default} size="50%" align="center" />

## 드라이버 펄스 설명

    :::info

    * 프린팅 모델 오프셋 발생
    * 모터 진동, 이상한 소리, 다양한 문제 발생
    * `step_pulse_duration: 0.000004`를 수정할 수 있습니다.
    * 이 값을 증가시킵니다. 예를 들어 `0.000009`, `0.00001` 또는 `0.00002` 등

    :::
