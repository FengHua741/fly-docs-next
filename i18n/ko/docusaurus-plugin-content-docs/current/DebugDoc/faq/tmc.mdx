# TMC 오류 관련

## `Unable to read tmc uart ‘stepper_x’ register IFCNT` 또는 `Unable to read tmc spi ‘stepper_x’ register register`

    1. 오류 상세

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart.webp').default} size="80%" align="center" />

    2. 오류 원인

        * 드라이버와의 통신 불가능

    3. 주요 해결 방법

        * 해당 드라이버의 UART 핀 설정이 올바른지 확인
        * UART 점선이 올바른지 확인
        * 드라이버에 전원 공급이 되는지 확인
        * 드라이버를 교체하거나 드라이버 포트 설정을 확인
        * 다른 회사의 드라이버일 경우, 드라이버의 UART 핀이 FLY와 동일한지 확인 필요, FLY 드라이버의 UART 핀은 왼쪽에서 위에서 아래로 네 번째 핀입니다.
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/2209uart.webp').default} size="80%" align="center" />
        * 구매한 드라이버의 UART 핀이 다섯 번째 핀에 있다면 아래 점선으로 해결 가능
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart1.webp').default} size="80%" align="center" />

## `TMC XXXX reports error: GSTAT: 00000001 reset=1(Reset)`

    1. 오류 상세

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/reset.webp').default} size="80%" align="center" />

    2. 오류 원인

        * 드라이버 리셋

    3. 주요 해결 방법

        * 전원 접속 불량
        * 모터 선단선
        * 모터 금속 케이스를 메인보드나 도구판의 GND와 연결

## `TMC 'extruder' reports error: DRV_STATUS: c00d0103 otpw=1(OvertempWarning!)`

    1. 오류 상세

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/overtemperror.webp').default} size="80%" align="center" />

    2. 오류 원인

        * 드라이버 과열!
        * 모터 과열이 아님!

    3. 주요 해결 방법

        * 작동 전류 감소
        * 유지 전류 삭제
        * 드라이버 샘플링 저항 설정이 올바른지 확인
        * 드라이버 냉각 강화, 예를 들어 팬 추가

## `TMC ‘extruder’ reports error: DRV_STATUS: 601f1000 s2vsa=1(ShortToSupply_A!) csactual=31 ola=1(OpenLoad_A!) olb=1(OpenLoad_B!)）`

    1. 오류 상세

        * 현재 이미지 없음

    2. 오류 원인

        * 드라이버 전류 측정 불량 확인

    3. 주요 해결 방법

        * 스텝 모터 A/B 두 집합의 코일 저항값이 같은지 확인
        * 스텝 모터 또는 부품 내부의 전선이 느슨하거나 단락되었는지 확인
        * 무음 모드 해제

## `TMC 'extruder' reports error: DRV_STATUS: 00190060 s2vsb=1(ShortToSupply_B!) ola=1(0penLoad_A!) cs_actual-25`

    1. 오류 상세

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/ab.webp').default} size="80%" align="center" />

    2. 오류 원인

        * 잘못된 연결 또는 모터 자체 문제

    3. 주요 해결 방법

        * 스텝 모터 A/B 두 집합의 코일 저항값이 같은지 확인
        * 스텝 모터 및 연결선 확인
        * 모터 선순서 확인, 왼쪽이 한 집합, 오른쪽이 한 집합인지 확인
