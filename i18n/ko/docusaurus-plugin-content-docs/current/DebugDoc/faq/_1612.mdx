:::info
* `z_tilt` 기능이나 `quad_gantry_level (QGL)` 기능이 있는 프린터는 다중 z 밸런싱을 실행해야 합니다.
:::

### 첫 번째 보정: EDDY
* 이미지에 표시된 매크로 명령어 `CALIBRATE_EDDY`를 클릭하여 실행합니다.
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy5.webp').default} size="50%" align="left" />
* 그런 다음 아래와 같은 UI 인터페이스가 나타납니다.
* A4 용지를 사용하여 Z-offset을 조정하십시오. 팝업된 UI 인터페이스에서 `z-offset` 값을 조정하기 전에 Klipper 문서의 가이드 [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)를 확인하십시오.
* 적절한 값이 나왔다면 `ACCEPT`를 클릭합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* 보정이 완료되면 `SAVE_CONFIG`를 클릭하여 파라미터를 저장합니다.

### 두 번째 단계: 온도 보상
:::info
* 온도 보상을 수행하기 전에는 `열판`, `노즐`을 가열하지 마십시오.
:::

* 아래 이미지에 표시된 매크로 명령어 `TEMP_COMPENSATION`를 클릭하여 실행합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/temp.webp').default} size="50%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy6.webp').default} size="50%" align="left" />

1. 이 작업을 수행하면 UI에 Z 축 조정 상자가 표시됩니다. 위에서 언급한 수동 [Z축 오프셋 보정 (Paper Test) 방법](https://www.klipper3d.org/Bed_Level.html#the-paper-test)을 사용하여 종이를 노즐과 침대 사이에 끼우고 해당 값을 확인하십시오.
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
2. 값을 수락한 후 자동으로 열판 온도가 80℃, 노즐 온도가 250℃로 설정됩니다.
3. 방에서 에어컨이나 창문을 사용하고 있다면 Eddy의 온도 상승을 보장하기 위해 에어컨을 끄거나 창문을 닫으십시오. 바람은 온도 상승에 영향을 줄 수 있습니다.
4. Eddy의 온도가 상승함에 따라 시스템은 매 3℃마다 수동 Z 오프셋 보정을 요청할 것입니다. 열판 온도가 매우 높으므로 주의하십시오!
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
5. 수동 Z 오프셋 보정 (Paper Test)을 반복하여 보정이 완료될 때까지 계속하십시오. 만약 Eddy의 온도가 더 이상 상승하지 않는다면 아래에 있는 관련 명령어를 사용하여 보정을 미리 종료할 수 있습니다.

* 드리프트 보정 중에 사용할 수 있는 추가 G 코드 명령어는 다음과 같습니다: `TEMPERATURE_PROBE_NEXT`
* `TEMPERATURE_PROBE_NEXT`는 EDDy가 목표 온도 `80℃`에 도달하기 전에 새로운 데이터 샘플링을 강제로 수행합니다.
* `TEMPERATURE_PROBE_COMPLETE`는 EDDy가 설정된 온도 `80℃`에 도달하기 전에 보정을 완료하는 데 사용됩니다.
* `ABORT`는 보정을 종료하고 결과를 무시하는 데 사용됩니다.
* 보정이 완료되면 `SAVE_CONFIG`를 사용하여 온도 오프셋 설정을 저장하십시오!
* 요약하자면, 대부분의 다른 프로그램들보다 위의 보정 과정은 더 어려우며 시간이 많이 소요됩니다. 넓은 온도 범위에서 우수한 첫 층 인쇄 효과를 얻기 위해서는 연습과 여러 번의 시도가 필요할 수 있습니다!

## Z-offset 미세 조정 방법
* 첫 번째 단계: pursa/OrcaSlicer/cura/Bambu Studio를 사용하여 높이 0.2인 200x200mm 크기의 사각형을 출력하여 Z-offset을 미세 조정합니다.
* 두 번째 단계: 프린터 IP를 통해 `fluidd` `mainsail`에 접속하여 대시보드의 `이동 제어`에서 Z축 오프셋을 Z_OFFSET으로 조정합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* 보정이 완료되면 `SAVE_CONFIG`를 클릭하여 파라미터를 저장합니다.

## 값이 변경되지 않을 경우 조정 명령어
* 아래 명령어를 사용하여 Z-offset을 조정하십시오. 팝업된 UI 인터페이스에서 `z-offset` 값을 조정하면 출력 중에 조정되는 값입니다. [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
```bash
G28
PROBE_CALIBRATE METHOD=AUTO  
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* 보정이 완료되면 `SAVE_CONFIG`를 클릭하여 파라미터를 저장합니다.

:::info
* 위 작업이 완료되었지만 스캔 실패가 발생했다면 Z 리미트를 가상 리미트로 변경하여 원래의 endstop_pin `endstop_pin: probe:z_virtual_endstop`를 덮어쓰십시오.
:::
