:::info
* 프린터에 `z_tilt` 또는 `quad_gantry_level (QGL)` 기능이 있는 경우, 다중 z 균열을 실행하도록 기억하십시오.
:::

### 첫 번째 보정 EDDY
* `CALIBRATE_EDDY` 마크다운 명령을 클릭하여 실행합니다.
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy5.webp').default} size="50%" align="left" />
* 그런 다음 다음 UI 인터페이스가 나타납니다.
* A4 용지로 Z 오프셋을 조정합니다. 팝업된 UI 인터페이스에서 `z 오프셋` 값을 조정하기 전에 klipper 문서의 튜토리얼 [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)를 확인하십시오.
* 적합한 값을 찾은 후 `ACCEPT`를 클릭합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* 보정이 완료되면 `SAVE_CONFIG`를 클릭하여 매개변수를 저장하십시오.

### 두 번째 단계: 온도 보상
:::info
* 온도 보상을 수행하기 전에 `열판, 노즐`을 가열하지 마십시오.
:::

* 아래 이미지에 표시된 `TEMP_COMPENSATION` 마크다운 명령을 클릭하여 실행합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/temp.webp').default} size="50%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy6.webp').default} size="50%" align="left" />

1. 이 작업을 수행한 후 UI는 Z 축 조정 상자를 표시할 것입니다. 위에서 언급한 수동 [Z 축 오프셋 보정(Paper Test) 방법](https://www.klipper3d.org/Bed_Level.html#the-paper-test)을 사용하여 종이를 노즐과 침대 사이에 끼운 후 해당 값을 확인하십시오.
2. 값이 수락되면 열판 온도를 최대로 설정하고 노즐 온도를 250°C로 설정하십시오.
3. 방에 에어컨이나 창문이 있다면, Eddy의 온도 상승을 보장하기 위해 에어컨을 끄거나 창문을 닫으십시오. 바람이 온도 상승에 영향을 미칠 수 있습니다.
4. Eddy의 온도가 상승하면서 시스템은 약 3°C마다 수동 Z 오프셋 보정을 수행하도록 안내할 것입니다. 열판 온도가 매우 높으므로 주의하십시오!
5. 수동 Z 오프셋 보정(Paper Test)을 반복하여 보정이 완료될 때까지 계속하십시오. Eddy의 온도가 더 이상 상승하지 않는 경우 아래에 나열된 관련 명령을 사용하여 보정을 조기에 종료할 수 있습니다.

* 드리프트 보정 중에 사용 가능한 추가 G 코드 명령은 다음과 같습니다: TEMPERATURE_PROBE_NEXT
* `TEMPERATURE_PROBE_NEXT`는 EDDy가 설정된 목표 온도 `80°C`에 도달하기 전에 EDDy가 새로운 데이터를 측정하도록 강제로 사용됩니다.
* `TEMPERATURE_PROBE_COMPLETE`는 EDDy가 설정된 목표 온도 `80°C`에 도달하기 전에 보정을 완료하도록 사용됩니다.
* `ABORT`는 보정을 종료하고 결과를 무시하도록 사용됩니다.
* 보정이 완료되면 `SAVE_CONFIG`를 사용하여 온도 오프셋 설정을 저장하십시오!
* 결론적으로, 대부분의 다른 프로그램과 비교했을 때, 위의 보정 과정은 더 어려우며 시간이 더 많이 걸립니다. 광범위한 온도 범위에서 우수한 첫 번째 층 인쇄 효과를 제공하기 위해서는 연습과 여러 번의 시도가 필요할 수 있습니다!

## Z 오프셋/Ｚ보정 조절
* 아래 명령을 사용하여 A4 용지를 사용하여 Z 오프셋을 조정합니다. 팝업된 UI 인터페이스에서 `z 오프셋` 값을 조정하기 전에 klipper 문서의 튜토리얼 [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)를 확인하십시오.
```bash
G28
PROBE_CALIBRATE METHOD=AUTO  
```
* 적합한 값을 찾은 후 `ACCEPT`를 클릭합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* 보정이 완료되면 `SAVE_CONFIG`를 클릭하여 매개변수를 저장하십시오.

## Z 오프셋 조정 미세 조정 방법
* 첫 번째 단계: pursa/OrcaSlicer/cura/Bambu Studio를 사용하여 200x200mm, 높이 0.2mm의 사각형을 자르고 Z 오프셋을 미세 조정하기 위해 프린트합니다.
* 두 번째 단계: 프린터 IP를 통해 `fluidd`, `mainsail`에 들어가서 대시보드의 `이동 제어`에서 Z 축 오프셋을 계속 미세 조정하고 Z_OFFSET을 기억하십시오.
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* 세 번째 단계: 아래 명령을 사용하여 Klipper를 재시작합니다.
```bash
FIRMWARE_RESTART
```
* 네 번째 단계: 아래 명령을 사용하여 Z 오프셋을 조정합니다. 팝업된 UI 인터페이스에서 `z 오프셋` 값을 조정하는 것은 프린팅 중에 조정한 값입니다. [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
```bash
G28
PROBE_CALIBRATE METHOD=AUTO  
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* 보정이 완료되면 `SAVE_CONFIG`를 클릭하여 매개변수를 저장하십시오.

:::info
* 위 작업을 완료하였지만 스캔 실패가 발생한 경우 z 제한을 가상 제한으로 변경하여 원래의 endstop_pin `endstop_pin: probe:z_virtual_endstop`를 덮어씁니다.
:::
