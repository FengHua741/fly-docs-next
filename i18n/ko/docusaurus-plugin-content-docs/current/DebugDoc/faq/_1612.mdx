## 작동 설명

    :::warning 주의 사항

    * 전류 보정을 먼저 수행한 후 코일 주파수 보정을 진행해야 합니다.
    * 다중 Z축 기계는 한 번 수동으로 수평을 맞춘 후 전체 보정을 완료하고 다시 한 번 수평을 맞춘 후 보정을 다시 수행해야 합니다.
    * 하위 기계 펌웨어는 Kliiper 버전과 완전히 일치해야 합니다.

    :::

    :::info

    * 전류 보정을 먼저 수행한 후 코일 주파수 보정을 진행해야 합니다.
    * 다중 Z축 기계는 한 번 수동으로 수평을 맞춘 후 전체 보정을 완료하고 다시 한 번 수평을 맞춘 후 보정을 다시 수행해야 합니다.
    * 하위 기계 펌웨어는 Kliiper 버전과 완전히 일치해야 합니다.

    :::


    * `FUIDD` 웹페이지를 사용하여 아래 구성을 추가한 후 후속 작업을 수행하세요.
        ```cfg
        [force_move]
        enable_force_move: true
        ```

    ### 두 번째 단계: EDDY 전류 보정

        * 1: 콘솔에서 아래 명령을 입력하세요.
            ```bash
            SET_KINEMATIC_POSITION z=80
            ```
        * 2: X와 Y를 귀환시키고 이동 도구 헤드를 열판 중간으로 이동시킵니다.
            ```bash
            G28 X Y
            ```
        * 상기 단계가 완료되면 Eddy 전류를 보정해야 합니다. 먼저 Eddy를 이동시켜 열판 중간으로 옮기고 노즐이 열판과 막 닿을 때까지 합니다. 그런 다음 콘솔에서 아래 명령을 실행하여 자동으로 전류를 얻고 마지막으로 `SAVE_CONFIG`로 매개변수를 저장합니다. 피드백 값이 15 정도면 정상값이며, 0 또는 30이 반환되면 연결 상태(접촉 불량 여부)를 확인하세요.
            ```bash
            LDC_CALIBRATE_DRIVE_CURRENT CHIP=fly_eddy_probe
            ```
        * 정상적인 반환 값은 `15` 정도여야 합니다.
        * 보정이 완료되면 `SAVE_CONFIG`로 매개변수를 저장합니다.

    ### 세 번째 단계: Eddy 주파수 보정


        :::info

        * Eddy를 설치할 때, eddy ocb 보드는 노즐보다 최소 1 - 2mm 높아야 합니다.
        * Eddy 주파수를 보정할 때, 코일 PCB는 열판에서 최소 5mm 떨어져 있어야 합니다.
        * 여기에서 5mm 떨어지는 것은 다음 단계에서 종이를 뽑아 `z_offset`를 얻기 편리하게 하기 위한 것입니다.

        :::

        * 그런 다음 노즐을 플랫폼 중심으로 이동시킵니다(주의: 이 단계에서는 기계가 heightmap을 로드하지 않아야 합니다).
        * 수동 z 오프셋 보정을 시작합니다. [종이 테스트](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
            ```bash
            PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
            ```
        * 보정이 완료되면 `SAVE_CONFIG`로 매개변수를 저장합니다.
        :::info
        * `z_tilt` 또는 `quad_gantry_level(QGL)` 기능이 있는 프린터는 먼저 평행도를 조정합니다. `i3는 X축의 평행도를 조정하고, 트라이던트는 열판의 평행도를 조정하며, voron 2.4는 갠트리의 평행도를 조정한 후 Eddy 주파수를 다시 한 번 보정합니다.
        :::
    ### 네 번째 단계: 온도 보상
        :::info
        * 온도 보상을 수행할 때 `열판, 노즐`을 가열하지 마세요.
        :::

        * 네 번째 단계: 온도 보상은 성공적으로 귀환할 수 있을 때 시작합니다.

        * 모든 축을 귀환시킵니다.
       ```bash
       G28
       ```

        * 만약 `[quad_gantry_level]` 또는 `[z_tilt]`가 있다면, 먼저 `Z_TILT_ADJUST` 또는 `QUAD_GANTRY_LEVEL`을 한 번 실행한 후 다시 한 번 귀환시킵니다.
        * 콘솔에서 G0 Z5를 입력하거나 웹페이지 등을 통해 Z축을 열판 위 `5mm`로 이동시킵니다.
        * 아래 코드를 실행하여 기계의 `idle timeout`을 길게 설정하여 우리가 온도를 올리는 과정에서 timeout이 발생하지 않도록 합니다.
            ```bash
            SET_IDLE_TIMEOUT TIMEOUT=65000
            ```
        * 아래 명령을 실행하여 보정을 시작합니다.
            ```bash
            TEMPERATURE_PROBE_CALIBRATE PROBE=fly_eddy_probe TARGET=56 STEP=4
            ```
        :::warning 주의 사항

        * 안내: 위 명령에서 EDDy의 목표 온도는 `56℃`로 설정되어 있으며 대부분의 장치에 적합합니다.
        * 만약 3D 프린터 챔버의 실제 온도가 더 높다면 목표 온도를 적절히 높일 수 있습니다.
        * 주목하세요. 목표 온도가 높을수록 보정에 필요한 시간이 길어지는데, 이는 Eddy가 온도 균형을 이루는데 더 많은 시간이 필요하기 때문입니다.

        :::

        1. 이 작업을 수행하면 UI에 Z축 조정 상자가 표시됩니다. 위에서 언급한 수동 [Z축 오프셋 보정(Paper Test) 방법](https://www.klipper3d.org/Bed_Level.html#the-paper-test)을 사용하여 노즐과 침대 사이에 종이를 끼우고 그 값을 확인하세요.
        2. 값을 수락한 후 열판 온도를 최고로 설정하고 노즐 온도를 220℃로 설정합니다.
        3. 만약您이 에어컨이 켜져 있거나 창문이 열려 있는 방에서 작업한다면 Eddy의 온도 상승을 보장하기 위해 에어컨을 끄거나 창문을 닫는 것이 좋습니다. 바람은 온도 상승에 영향을 줄 수 있습니다.
        4. Eddy 온도가 상승함에 따라 시스템은 4℃마다 한 번씩 수동 Z 오프셋 보정을 수행하도록 자동으로 안내합니다. 열판 온도가 매우 높으니 화상을 입지 않도록 주의하세요!!!
        5. 보정이 완료될 때까지 수동 Z 오프셋 보정(Paper Test)을 반복합니다. Eddy의 온도가 더 이상 상승하지 않는 것을 발견하면 아래 관련 명령을 사용하여 보정을 조기에 종료할 수 있습니다.

        * 드리프트 보정 동안 사용할 수 있는 추가 gcode 명령에는 다음과 같습니다: TEMPERATURE_PROBE_NEXT
            * `TEMPERATURE_PROBE_NEXT`는 EDDy가 설정된 목표 온도 `56℃`에 도달하기 전에 EDDy가 새로운 데이터를 샘플링하도록 강제합니다.
            * `TEMPERATURE_PROBE_COMPLETE`는 EDDy가 설정 온도 `56℃`에 도달하지 못한 경우 보정을 완료합니다.
            * `ABORT`는 보정을 종료하고 결과를 무시하는 데 사용할 수 있습니다.
        * 이 단계들을 완료하면 Eddy는 넓은 온도 범위에서 뛰어난 첫 번째 층 인쇄 효과를 제공할 수 있습니다!


## z—offset를 미세 조정하는 방법
* 첫 번째 단계: pursa/OrcaSlicer/cura/Bambu Studio를 사용하여 200x200 높이 0.2의 사각형을 잘라 z—offset를 미세 조정하기 위해 인쇄합니다.
* 두 번째 단계: 프린터 IP를 통해 `fluidd` `mainsail`에 들어가서 대시보드 `이동 제어` z축 오프셋을 찾아 z_OFFset를 계속 미세 조정한 후 미세 조정한 값을 기억합니다.
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* 세 번째 단계: 아래 명령을 통해 klippper를 재시작합니다.
```bash
FIRMWARE_RESTART
```
* 네 번째 단계: 아래 명령을 통해 Z—offset를 조정합니다. 팝업되는 UI 화면에서 `z—offset`를 조정합니다. 조정하는 값은 인쇄 과정에서 조정한 값입니다.
```bash
G28
PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />

:::info
* 오류 `Eddy current sensor error`가 발생하면 eddy 구성에서의 z_offset 값을 늘리세요. 아래 그림을 참조하세요.

:::

:::
