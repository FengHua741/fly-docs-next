## 운영 설명

    :::warning 주의 사항

    * 전류 보정을 먼저 한 후 코일 주파수 보정을 해야 합니다.
    * 다중 Z축 기계는 수동으로 평평하게 한 번하고 모든 보정을 완료한 후 다시 한 번 평평하게 해야 합니다.
    * 하위 기기 펌웨어는 Kliiper 버전과 완전히 일치해야 합니다.

    :::

    * `FUIDD` 웹 페이지에 아래 구성을 추가한 후 다음 작업을 실행하세요
        ```cfg
        [force_move]
        enable_force_move: true
        ```

    ### 두 번째 단계: EDDY 전류 보정

        * 1: 콘솔에 아래 명령어 입력
            ```bash
            SET_KINEMATIC_POSITION z=80
            ```
        * 2: X와 Y를 홈으로 이동하고 도구 헤드를 히트베드 중앙으로 이동
            ```bash
            G28 X Y
            ```
        * 위 단계가 완료되면 Eddy 전류를 보정해야 합니다. 먼저 EDDY를 히트베드 중앙으로 이동한 후 노즐이 히트베드에 막 닿도록 합니다. 그런 다음 콘솔에서 아래 명령어를 실행하여 자동으로 전류를 얻고 마지막으로 `SAVE_CONFIG`로 매개변수를 저장합니다. 피드백 값이 약 15가 정상 값이며, 0 또는 30이면 접촉이 불량한지 확인하세요.
            ```bash
            LDC_CALIBRATE_DRIVE_CURRENT CHIP=fly_eddy_probe
            ```
        * 정상적인 반환 값은 약 `15`여야 합니다.
        * 보정이 완료되면 `SAVE_CONFIG`로 매개변수를 저장하세요.

    ### 두 번째 단계: Eddy 주파수 보정


        :::info

        * Eddy를 설치할 때, eddy ocb 보드는 노즐보다 최소 1-2 mm 높아야 합니다.
        * Eddy 주파수를 보정할 때, 코일 PCB는 히트베드에서 최소 5mm 떨어져 있어야 합니다.
        * 여기서 5mm 떨어지는 것은 다음 단계에서 종이를 사용하여 `z_offset`을 얻기 쉽게 하기 위함입니다.

        :::

        * 그런 다음 노즐을 플랫폼 중앙으로 이동하세요 (주의: 이 단계는 기계에 heightmap이 로드되지 않았는지 확인하세요).
        * 수동 z 오프셋 보정을 시작합니다. [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
            ```bash
            PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
            ```
        * 보정이 완료되면 `SAVE_CONFIG`로 매개변수를 저장하세요
        :::info
        * `z_tilt` 또는 `quad_gantry_level（QGL）` 기능이 있는 프린터는 평행도를 조정한 후 다시 Eddy 주파수를 보정하세요. i3은 X축의 평행도를 조정하고, 삼지창은 히트베드의 평행도를 조정하고, voron 2.4는 갠트리 프레임의 평행도를 조정하세요.
        :::
    ### 세 번째 단계: 온도 보상
        * 세 번째 단계: 온도 보상은 성공적으로 홈으로 돌아가야 시작할 수 있습니다.

        * 모든 축을 홈으로 이동
       ```bash
       G28
       ```

        * `quad_gantry_level` 또는 `z_tilt`가 있으면 먼저 `Z_TILT_ADJUST` 또는 `QUAD_GANTRY_LEVEL`을 실행한 후 다시 한 번 홈으로 이동하세요.
        * 콘솔에 G0 Z5를 입력하거나 웹 페이지를 통해 Z축을 히트베드 위 `5mm`로 이동하세요.
        * 아래 코드를 실행하여 기계의 `idle timeout`을 길게 설정하여 온도 상승 중 timeout을 방지하세요.
            ```bash
            SET_IDLE_TIMEOUT TIMEOUT=65000
            ```
        * 아래 명령을 실행하여 보정을 시작하세요.
            ```bash
            TEMPERATURE_PROBE_CALIBRATE PROBE=fly_eddy_probe TARGET=56 STEP=4
            ```
        :::warning 주의 사항

        * 힌트: 위 명령어에서 EDDY 목표 온도는 `56℃`로 설정되어 있으며, 대부분의 장치에 적합합니다.
        * 3D 프린터 캐비닛의 실제 온도가 더 높다면 목표 온도를 적절히 높일 수 있습니다.
        * 목표 온도가 높을수록 보정에 필요한 시간이 길어지므로 Eddy가 온도 균형에 도달하는 데 더 오래 걸립니다.

        :::
    
        1. 이 작업을 실행하면 UI에 Z축 조정 상자가 표시됩니다. 위에서 언급한 수동 [Z축 오프셋 보정（Paper Test）방법](https://www.klipper3d.org/Bed_Level.html#the-paper-test)을 사용하여 종이를 노즐과 베드 사이에 끼우고 이 값을 확인하세요.
        2. 값을 수락한 후, 히트베드 온도를 최대로, 노즐 온도를 220℃로 설정하세요.
        3. 에어컨이나 창문이 있는 방에서 작업 중이라면 Eddy의 온도 상승을 보장하기 위해 에어컨이나 창문을 닫는 것이 좋습니다. 바람이 온도 상승에 영향을 줄 수 있습니다.
        4. Eddy의 온도가 상승함에 따라 시스템은 매 4℃마다 수동 Z 오프셋 보정을 실행하도록 자동으로 알려줄 것입니다. 히트베드 온도가 매우 높으므로 화상에 주의하세요!
        5. 수동 Z 오프셋 보정（Paper Test）을 반복하여 보정을 완료하세요. Eddy의 온도가 더 이상 상승하지 않는다면 아래 관련 명령을 사용하여 보정을 조기에 종료할 수 있습니다.
        
        * 드리프트 보정 중 사용 가능한 추가 gcode 명령어에는 다음이 포함됩니다: TEMPERATURE_PROBE_NEXT
            * `TEMPERATURE_PROBE_NEXT`    EDDy가 설정된 목표 온도 `56℃`에 도달하기 전에 강제로 EDDy가 새 데이터를 샘플링하도록 사용됩니다.
            * `TEMPERATURE_PROBE_COMPLETE`    EDDy가 설정된 온도 `56℃`에 도달하지 못했을 때 보정을 완료하는 데 사용됩니다.
            * `ABORT`  보정을 종료하고 결과를 무시하는 데 사용됩니다.
        * 이 단계를 완료하면 Eddy는 광범위한 온도 범위에서 탁월한 첫 번째 레이어 인쇄 효과를 제공할 수 있습니다!


## z-offset을 미세 조정하는 방법
* 첫 번째 단계: pursa/OrcaSlicer/cura/Bambu Studio를 사용하여 높이 0.2의 200x200 크기의 사각형을 잘라내어 z-offset을 미세 조정하기 위해 인쇄합니다.
* 두 번째 단계: 프린터 IP를 통해 `fluidd` `mainsail`에 들어가 대시보드의 `이동 제어`에서 z축 오프셋을 계속 미세 조정하고 미세 조정한 값을 기억하세요.
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* 세 번째 단계: 아래 명령어를 통해 klippper를 재시작하세요
```bash
FIRMWARE_RESTART
```
* 네 번째 단계: 아래 명령어를 통해 z-offset을 조정하는 UI 인터페이스를 표시하고 인쇄 중 조정한 값으로 `z-offset`을 조정하세요.
```bash
G28
PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />

:::info
* `Eddy current sensor error` 오류가 발생하면 eddy 설정에서 z_offset 값을 크게 조정하세요. 예시를 참조하세요.
