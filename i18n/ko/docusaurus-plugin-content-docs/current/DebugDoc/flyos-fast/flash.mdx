# 시스템 이미지 플래싱

## 준비 작업

1. 이전 페이지의 [시스템 다운로드](/docs/DebugDoc/flyos-fast/)를 완료했습니다.
2. 플래싱 도구를 다운로드합니다. 본 설명서에서는 `Rufus`를 사용하겠습니다. 실제로는 다른 도구, 예를 들어 `Etcher`, `Win32 Disk Imager` 등을 사용할 수 있습니다.
    - Rufus 다운로드 주소: [한국어 다운로드](https://rufus.ie/zh/#download) 또는 [영어 다운로드](https://rufus.ie/en/#download) (Windows 전용).
    - Etcher 다운로드 주소: [다운로드](https://etcher.balena.io/#download-etcher) (Windows, MacOS, Linux 지원).
    - Win32 Disk Imager 다운로드 주소: [다운로드](https://sourceforge.net/projects/win32diskimager/) (Windows 전용).
3. 시스템 이미지 파일을 다운로드합니다. 본 설명서에서는 `FlyOS-Fast-H5.exe`를 사용하겠습니다.

4. 시스템 이미지 파일을 압축 해제하여 `.img` 파일을 얻습니다.
    - `FlyOS-Fast-H5.exe`를 더블 클릭하거나 우클릭해서 열면 몇 초 후 아래와 같은 창이 나타납니다(시스템에서 안전하지 않다고 경고하는 경우 "더 자세히 보기" -> "실행"을 선택하세요).
    - 압축 폴더를 선택한 다음 `추출` 버튼을 클릭하고 압축이 완료될 때까지 기다립니다.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - 압축이 완료되면 압축 폴더에 `FlyOS-Fast-H5-vx.x.x.img` 파일이 생성됩니다. `x.x.x`는 시스템 버전 번호입니다.
5. 플래싱 카드를 준비합니다.
    - 플래싱 카드는 4GB 이상이어야 합니다. SanDisk 32GB C10 이상의 TF 카드를 권장합니다(저렴한 많은 카드들이 플래싱 후 부팅되지 않을 수 있다는 테스트 결과가 있음).
    - 플래싱 카드는 SD 카드 리더기를 사용하여 컴퓨터에 연결합니다.

## 시스템 플래싱

1. Rufus를 열고 장치를 선택합니다. 선택된 장치의 용량은 컴퓨터에 연결된 플래싱 카드의 용량과 일치해야 합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="30%" align="center" />

2. 부트 유형 선택, 이는 기본적으로 `이미지 파일`로 설정되어 있습니다.
3. 이미지 파일 선택, `부트 유형 선택` 오른쪽의 `선택` 버튼을 클릭하고 압축해제된 `.img` 파일을 선택합니다.
4. 이미지 파일을 선택하면 `부트 유형 선택`이 이미지 파일의 이름으로 변경됩니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="30%" align="center" />

5. `시작` 버튼을 클릭하면 디스크 데이터를 지울 필요가 있다는 경고창이 뜨면 `확인`을 클릭합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />

6. 플래싱이 완료되면 컴퓨터는 경고음을 내며 진행률 표시줄이 `준비 완료`로 바뀝니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="30%" align="center" />

7. 플래싱이 완료된 후 바로 플래싱 카드를 빼지 마세요.
8. 파일 탐색기를 열고 `FlyOS-Conf`라는 이름의 USB 드라이브나 이동식 디스크가 있는지 확인합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />

9. `FlyOS-Conf`를 열면 아래 그림에 있는 두 개의 파일이 있으면 플래싱이 성공한 것입니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

10. `config.txt`는 시스템 구성 파일입니다. 다음 페이지의 내용을 참조하여 구성 파일을 수정하세요.
