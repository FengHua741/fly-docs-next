# 시스템 이미지 플래싱

## 시스템 소개
FlyOS-Fast는 3D 프린팅에 최적화된 가벼운 Linux 시스템으로, Fly 전체 시리즈의 상위 기기 플랫폼을 지원합니다. 그 주요 특징은 다음과 같습니다:

- 빠른 부팅 속도와 커널 최적화로 Klipper 실시간성을 보장
- 파일 시스템 최적화로 파워다운으로 인한 부팅 실패 위험 감소
- 30초 내 KlipperScreen 시작
- Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen 등 서비스 미리 설치
- Klipper와 RRF 시스템 전환 지원
- Mainsail과 Fluidd 온라인 한 번 클릭 전환 지원
- U디스크 인쇄, 전원이 꺼진 상태에서 계속 인쇄, 시리얼 모니터링 지원
- fly-c8, fly-geminiv3 자동 플래시 지원
- Fluidd 등 프론트엔드 UI 기본 언어 및 프린터 이름 사전 설정 가능

추가 OEM 맞춤 기능(예: 전원 로고, 기본 구성 파일 사전 설정, Mainsail 탭 숨김 등)이 필요하시면 저희에게 연락해주세요.

:::tip 주의사항
이 시스템은 가벼운 맞춤형 시스템으로, 내장된 klipper 등의 구성 요소 및 서비스는 공식 버전이 아닙니다. 사용자가 직접 수정하거나 업그레이드할 수 없습니다.
:::


- FLYOS-FAST 시스템 계정은 `root`, 비밀번호는 `mellow`입니다.

## 시스템 플래싱

1. Rufus를 열고 장치를 선택합니다. 선택한 장치는 컴퓨터에 연결된 플래싱 카드와 용량이 동일하게 표시되어야 합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="30%" align="center" />

2. `부트 유형 선택`, 이 항목은 기본값으로 `이미지 파일`을 선택합니다.
3. 이미지 파일 선택, `부트 유형 선택` 오른쪽의 `선택` 버튼을 클릭하고 압축 해제된 `.img` 파일을 선택합니다.
4. 이미지 파일을 선택한 후, `부트 유형 선택`은 이미지 파일 이름으로 변경됩니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="30%" align="center" />

5. `시작` 버튼을 클릭하면, 디스크 데이터를 지울 필요가 있다고 경고창이 뜰 경우 `확인`을 클릭합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />

6. 플래싱이 완료되면 컴퓨터가 알림음을 내며 진행률이 `준비 완료`로 표시됩니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="30%" align="center" />

7. 플래싱이 완료되면 즉시 플래싱 카드를 뽑지 마세요.
8. 파일 탐색기를 열고 `FlyOS-Conf`라는 이름의 USB 드라이브나 이동식 디스크가 있는지 확인합니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />

9. `FlyOS-Conf`를 열고 아래와 같은 두 개의 파일이 존재하면 플래싱이 성공한 것입니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

10. 시스템 카드를 메인보드에 설치합니다.
