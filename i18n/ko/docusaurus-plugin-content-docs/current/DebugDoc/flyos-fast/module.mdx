# 시스템 내장 구성 요소 소개

:::danger

- FlyOS-Fast 시스템 내장의 모든 구성 요소 및 서비스는 사용자가 수정하거나 업그레이드 할 수 없습니다.
- 사용자가 수정함으로써 웹 페이지에 액세스할 수 없거나 시스템이 시작되지 않는 문제가 발생할 수 있습니다.
- 시스템을 다시 플래시하십시오.

:::

## Klipper

:::warning

- 기존 Klipper에 서드파티 extras를 추가하는 것은 시도해 볼 수 있지만 기존의 모든 extras 모듈은 수정하지 마십시오.

:::

### 예외 자동 일시 중지

- 프린팅 중 Klipper가 온도 이상, 드라이버 이상 등의 오류가 발생하면 자동으로 프린팅을 일시 중지하고 즉시 종료하지 않습니다.

### 실시간 최적화

- Klipper의 실시간 최적화는 커널과 Klipper 코드를 수정하여 달성되며 Klipper의 실시간성을 최적화합니다.
- `Timer too close` 오류를 효과적으로 줄입니다.

### 전원 단절 재프린팅, 예외 재프린팅 지원

- 전원 단절 재프린팅을 지원합니다.
- 오류로 인한 정지 후 재프린팅을 지원합니다.

### TCP 소켓 지원

- Unix 소켓과 TCP 소켓을 동시에 활성화할 수 있습니다.
- `-p` 매개변수를 사용하여 TCP 포트를 지정하여 TCP 소켓을 활성화합니다.

### USB 드라이브 파일 자동 처리

- USB 드라이브 파일을 자동으로 처리하여 프린팅 중단을 방지합니다.

## Moonraker

### TCP 소켓 지원

- klippy에 TCP 소켓을 사용하여 연결할 수 있습니다.
- `klippy_tcp_address` 매개변수를 사용하여 TCP 주소를 지정합니다.
- `klippy_tcp_port` 매개변수를 사용하여 TCP 포트를 지정합니다.

### 대용량 파일 업로드 최적화

- 연속적으로 대용량 파일을 업로드할 때 발생하는 오류 문제를 해결합니다.

### 네트워크 관리 - FLY5,7 화면에 적합

- 무선 네트워크 관리를 지원하며 FLY5,7 화면에 적합합니다.

### FLY 화면 지원

- FLY 화면을 지원합니다.
- 모델 첫 번째 층 그래픽 생성을 지원합니다.

## KlipperScreen

### 전원 단절 재프린팅

- 전원 단절 재프린팅을 지원합니다.
- 재프린팅 감지 시 자동으로 팝업 알림이 표시됩니다.

## Fluidd

### 전원 단절 재프린팅

- 전원 단절 재프린팅을 지원합니다.
- 재프린팅 감지 시 자동으로 팝업 알림이 표시됩니다.

## Mainsail

### 전원 단절 재프린팅

- 재프린팅 감지 시 자omatic으로 팝업 알림이 표示されます。
