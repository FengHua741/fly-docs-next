# 세계 표준시对照표

:::tip 필수 읽기

- 타임존 설정은 `localtime` 구성 항목에 타임존 ID를 입력하면 됩니다. 예: `Asia/Shanghai`입니다.
- 시스템 재부팅 후 적용됩니다.
- 네트워크 연결이 끊어진 상태에서는 시간이 잘못될 수 있으며, 연결 후 자동으로 동기화됩니다.

:::
