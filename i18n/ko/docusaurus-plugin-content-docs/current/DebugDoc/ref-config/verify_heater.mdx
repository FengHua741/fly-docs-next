# 열 저항 허용치 설정

    * 엑트루더

    ```cfg title="printer.cfg"
    [verify_heater extruder]
    max_error: 120
    check_gain_time:120
    hysteresis: 50
    heating_gain: 2
    ```
       
    * 열베드

    ```cfg title="printer.cfg"
    [verify_heater heater_bed]
    max_error: 120
    check_gain_time:120
    hysteresis: 50
    heating_gain: 2
    ```
