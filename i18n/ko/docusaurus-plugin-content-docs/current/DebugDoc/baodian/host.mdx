# 상위기기 가이드

## 새로 구매한 상위기기를 실행하기 위한 방법

  1. 먼저 상위기기가 어떤 모델인지 확인해야 합니다. Lite 2 또는 PI V2 등이 있습니다.
  2. 다음의 준비물이 있는지 확인하세요.
        * 플래싱에 사용할 소프트웨어
        * 해당 시스템 이미지
        * 읽기 전용 카드, TF 카드, 데이터 전송 기능을 갖춘 TYPE-C 케이블

  시스템 플래싱 전 주의사항:
        * 메모리 카드 용량은 `16GB` 이상 `128GB` 이하로, 그리고 전송 속도는 `class10` 급 또는 그 이상이어야 합니다. SanDisk 등의 브랜드를 권장합니다.
        * 시스템 업데이트를 임의로 하지 마세요! 서버가 해외에 위치해 있어 문제가 발생할 수 있습니다.
        * 가능한 한 추천된 소프트웨어를 사용하세요.
        * 최소한 리눅스에 대한 기본적인 지식을 배우는 것이 좋습니다. 개인적으로는 <Button variant="contained" disableElevation href="https://item.jd.com/12443890.html">《박고의 리눅스 익살레пт 기초 학습편》</Button>을 추천합니다.

  3. FLY 시스템 이미지 <Button variant="contained" disableElevation href="/docs/category/시스템_이미지">시스템 이미지</Button>
  4. 필요한 도구 <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">플래싱 도구</Button>

## 각종 도구와 대응하는 시스템 이미지를 다운로드한 후

  * 시스템 압축 해제, 압축 해제 후 파일 확장자는 `.img`입니다.
  * SDFormatter를 사용하여 메모리 카드를 다시 포맷화해야 합니다.
  * `win32DiskImager`를 사용하여 시스템에 플래싱해야 합니다.
  * 시스템 플래싱에 `balenaetcher`를 사용하지 않는 것이 좋습니다.
  * FLY 상위기기인 경우 `FlyOS-Env.txt`를 열어서 구성해야 합니다.
  * 시스템 플래싱이 완료되면 포맷화하지 마세요!!

## 시스템 시작

  * 시스템 플래싱이 완료되면 시스템 카드를 상위기에 설치하고 전원을 켜야 합니다.
  * 이때 시스템은 플래싱 및 구성이 완료되었습니다. 데이터 전송 기능을 갖춘 TYPE-C 케이블을 사용하여 컴퓨터와 연결해야 합니다.
  * 다른 상위기는 일반적으로 로그 로딩이 표시됩니다. 아직 표시되지 않은 경우에는 두 번 Enter 키를 누르거나 시스템을 다시 플래싱해야 합니다.
  * 시스템이 시작되면 Wi-Fi에 연결할 수 있습니다.

## M2WE와 메모리 카드 사용에 대한 안내

  * 기계에 M2WE와 메모리 카드가 장착되어 있는 경우 시스템은 기본적으로 메모리 카드에서 부팅합니다.
  * M2WE는 SD 카드를 사용하여 시스템을 플래싱하는 것을 권장합니다. USB를 통해 M2W로 플래싱하는 경우 드라이버 호환성 문제 등으로 플래싱 실패가 발생할 수 있으므로 메모리 카드를 직접 사용하는 것이 좋습니다.
  * 다운로드한 시스템 이미지가 일치하는지 확인하세요. 일치하지 않으면 시스템이 시작되지 않습니다.
  * Lite2와 MINI PAD는 메모리 카드만 사용할 수 있습니다.

## 일상 사용 시 주의사항

  * 직접 전원을 끄지 마십시오. 정상적으로 종료하지 않으면 데이터 손실이 발생할 수 있습니다.
  * GitHud 등의 서버가 해외에 위치해 있으므로 자체 업데이트나 다양한 플러그인 설치를 권장하지 않습니다.
