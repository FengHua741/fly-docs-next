## 장치 검색
* 상위 기기가 정상적으로 연결되어 있는지 확인하고, 하위 기기의 데이터 전송 기능이 가능한 데이터선을 사용했는지 확인하세요.
* 상위 기기에 SSH로 연결한 후 `lsusb`를 입력하고 Enter 키를 누릅니다. 아래 이미지에서 원으로 표시된 정보가 나타나면 다음 단계로 진행할 수 있습니다.
    * 아무런 정보가 반환되지 않는 경우, 이는 상위 기기의 시스템 문제입니다. 해결할 수 없으므로, 문제가 없는 시스템으로 교체하거나 상위 기기를 교체해야 합니다.
    * `lsusb` 명령이 없다고 하는 경우, 다음 명령어를 실행하여 설치할 수 있습니다:
        ```bash
        sudo apt-get install usbutils
        ```

* 빨간색 박스 내부에 표시된 ID는 참고용이며 고정된 값이 아니며 변경될 수 있습니다. 'OpenMoko, Inc. Geschwister Schneider CAN adapter'라는 문자열은 판단 근거로 사용할 수 있습니다.
* 일부 상위 기기는 시스템 문제로 인해 정보가 제대로 표시되지 않을 수 있습니다.
