:::팁
    * 플러그인 설치 시 인터넷 연결이 필요합니다. 상위 컴퓨터가 정상적으로 인터넷에 연결되어 있는지 확인하시기 바랍니다.
    * 이미 설치된 경우 다시 설치할 필요가 없습니다.
:::
