1. 프로그래밍 도구 <Button variant="contained" disableElevation href="https://mellow-next.klipper.cn/docs/ResDownload/auxiliary_software/">다운로드 주소</Button>
2. `DFUProgrammer`를 다운로드해야 합니다.
3. 앞에서 컴파일한 펌웨어(klipper.bin) 또는 미리 컴파일된 `Katapult 펌웨어`를 컴퓨터의 임의의 디렉토리에 복사합니다.
4. USB Type-C 데이터 케이블을 메인보드를 컴퓨터에 연결하고, 이미 **DFU 프래싱 모드**로 들어갔는지 확인하십시오.
5. DFUProgrammer를 열고 아래의 지침을 따르세요.
    <ImageView image={require('@site/docs/General/dfu-flash/img/apmdfu.webp').default} size="100%" align="center" />
    <ImageView image={require('@site/docs/General/dfu-flash/img/apmdfu1.webp').default} size="100%" align="center" />
6. 펌웨어 업데이트를 시작하고 잠시 기다리면 됩니다. 프래싱이 완료되면 Type-C 케이블과 점퍼 핀을 뽑으세요.
