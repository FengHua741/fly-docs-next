아래는 펌웨어를 컴파일하는 방법을 소개합니다:
* SSH에 연결한 후 다음 명령어를 입력하고 Enter 키를 누릅니다:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* 여기서 `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`은 이전의 컴파일 데이터와 펌웨어를 삭제하며, `make menuconfig`는 펌웨어를 컴파일합니다. 실행 후 아래와 같은 화면이 나타나야 합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**를 선택하고 Enter 키를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* 메뉴 **Micro-controller Architecture**로 들어가서 **Raspberry Pi RP2040/RP235x**를 선택하고 Enter 키를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* **Bootloader offset**를 선택하고 **NO bootloader**를 선택합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/0kb.webp').default} size="100%" align="Center" />

* **Communication interface**를 선택하고 **UART0 on GPIO0/GPIO1**을 선택합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/rs232.webp').default} size="100%" align="Center" />
