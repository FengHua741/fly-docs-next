다음은 펌웨어를 컴파일하는 방법을 소개합니다:
* SSH에 연결한 후 아래 명령어를 입력하고 엔터를 누릅니다:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* 여기서 `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`는 이전에 컴파일된 데이터와 펌웨어를 삭제하는 것이고 `make menuconfig`는 펌웨어를 컴파일하는 것입니다. 실행 후 아래와 같은 화면이 나타나야 합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* **Enable extra low-level configuration options**을 선택하고 엔터를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* 메뉴 **Micro-controller Architecture**로 들어가 **STMicroelectronics STM32**를 선택하고 엔터를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

이미지는 업데이트 중입니다.
* 메뉴 **Processor model**로 들어가 **STM32F405**를 선택하고 엔터를 누릅니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.webp').default} size="100%" align="Center" />

* **Bootloader offset**을 선택하고 **32KiB bootloader**를 선택합니다.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.webp').default} size="100%" align="Center" />

* **Communication interface**를 선택하고 **USB to CAN bus bridge (USB on PA11/PA12)**를 선택합니다.
* 이미지에 따라 **CAN bus interface（CAN bus（on PB8/PB9）） --->**를 선택합니다.
