::: tip
    * 플래시 플러그인 설치 시 인터넷 연결이 필요합니다. 상위 컴퓨터가 정상적으로 인터넷에 연결되어 있는지 확인하시기 바랍니다.
    * 이미 설치된 경우 다시 설치할 필요는 없습니다.
:::


### 번역하기

    ::: warning 경고
       * **Fly이외의 상위 컴퓨터** 사용 시 아래 명령어를 반드시 실행하여 펌웨어 플래시 의존 패키지를 설치해야 합니다! 그렇지 않으면 펌웨어를 플래시할 수 없습니다!
       ```
      cd && git clone https://github.com/Arksine/katapult.git
       ```

    :::

        ```bash
        ls /dev/serial/by-id/*
        ```

        ```bash
        ~/klippy-env/bin/python ~/katapult/scripts/flashtool.py -d /dev/serial/by-id/<보드 ID>
        ```
