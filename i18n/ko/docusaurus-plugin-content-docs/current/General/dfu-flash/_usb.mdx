* 상위 컴퓨터가 정상적으로 인터넷에 연결될 수 있도록 확인하고 하위 컴퓨터가 상위 컴퓨터에 연결된 데이터선이 데이터 전송 기능을 가지고 있는지 확인하세요.
* SSH 연결로 상위 컴퓨터에 연결한 후 `lsusb`를 입력하고 엔터를 누르세요. 아래 사진에서 동그라미로 표시된 정보가 나타나면 다음 단계로 진행할 수 있습니다.
    * 어떠한 정보도 반환되지 않는다면, 이는 상위 컴퓨터의 시스템 문제이며, 이쪽에서는 어쩔 수 없습니다. 정상인 시스템으로 교체하거나 상위 컴퓨터를 교체해야 합니다.
    * `lsusb` 명령이 없다는 메시지가 나오면, 다음 명령을 실행하여 설치할 수 있습니다.
        ```bash
        sudo apt-get install usbutils
        ```

* `1d50:6177`는 이번에 사용해야 할 장치입니다. `OpenMoko, Inc. stm32f407xx` 접미사는 어느 정도 참고 가치가 있지만 판단 근거로 삼지 않습니다.
    * 일부 상위 컴퓨터는 시스템 문제로 인해 정보가 완전히 표시되지 않거나 표시되지 않을 수 있습니다.
    * `lsusb` 명령을 입력하여 장치가 표시되지만 `1d50:6177`가 없다면 데이터선을 교체하거나 메인보드를 상위 컴퓨터의 다른 USB 포트에 연결해보세요.
