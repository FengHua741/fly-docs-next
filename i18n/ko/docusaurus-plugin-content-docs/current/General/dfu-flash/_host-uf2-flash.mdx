1. 번플링 도구 설치

    * 아래 명령어를 SSH에 입력합니다.
    * 처음 번플링할 때만 실행하고, 이후에는 다시 실행하지 않습니다.
    ```bash
    cd ~/klipper/lib/rp2040_flash/ && make
    ```
    <ImageView image={require('@site/docs/General/dfu-flash/img/2040.webp').default} size="100%" align="left" />

2. 다음 명령어를 실행하여 연결이 성공했는지 확인하세요.

    * `lsusb` 입력
    * `2e8a:0003` 장치가 없으면 다시 **번플링 모드**로 들어가십시오.
    * 시스템 버전 문제로 인해 다른 이름으로 표시될 수 있습니다. `2e8a:0003`이 표시되면 번플링 모드로 진입한 것입니다.

        <ImageView image={require('@site/docs/General/get-id/img/2e8a.webp').default} size="80%" align="left" /> 

    :::warning 주의사항

    `2e8a:0003`이 있을 때만 다음 단계로 진행하십시오.

    :::

3. 사전 컴파일된 펌웨어 가져오기

    * 여기에는 사전 컴파일된 펌웨어가 포함되어 있습니다.
    * 주의: 상위 컴퓨터가 인터넷에 연결되어 있어야 설치할 수 있습니다.
    * 상위 컴퓨터는 다음 명령어를 실행하여 펌웨어 패키지를 다운로드해야 합니다.

    ```bash
    cd && git clone https://github.com/FengHua741/FLY_Katapult.git
    ```

    * 펌웨어 패키지를 이미 다운로드한 경우 업데이트한 후 나머지 작업을 계속 진행하십시오.

    ```bash
    cd ~/FLY_Katapult && git pull
    ```
