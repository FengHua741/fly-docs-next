## 라즈베리 파이 5와 MIPI

:::tip
FLY_MIPI는 라즈베리 파이 5를 위해 설계된 어댑터 모듈로, 이 어댑터를 통해 DSI 및 CSI 장치를 직접 사용할 수 있습니다.
:::

### 라즈베리 파이 5와 MIPI 연결

    :::danger 주의사항

    * 라즈베리 파이 5의 DSI 및 CSI 인터페이스는 이제 MIPI라고 불립니다.
    * MIPI 인터페이스는 구성에 따라 CSI 카메라 또는 DSI 디스플레이를 직접 사용할 수 있습니다.
    * 금손 방향을 유의해야 합니다.

    :::

    * 라즈베리 파이 5에는 두 개의 MIPI 인터페이스가 있으며, 앞에서 보면 1과 0으로 표시되어 있습니다.

## DSI 디스플레이 연결

    * 1번 인터페이스를 사용하는 경우 아래와 같이 화면을 구성해야 합니다.

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * 0번 인터페이스를 사용하는 경우 아래와 같이 화면을 구성해야 합니다.

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## CSI 카메라 연결

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * 카메라 모델은 구매한 모델에 따라 구성해야 합니다. 이 튜토리얼에서는 imx219를 참조 모델로 사용합니다.

    * 왼쪽 MIPI 인터페이스를 사용하는 경우 아래와 같이 카메라를 구성해야 합니다.
    ```bash
    dtoverlay=imx219,cam1
    ```

    * 오른쪽 MIPI 인터페이스를 사용하는 경우 아래와 같이 카메라를 구성해야 합니다.
    ```bash
    dtoverlay=imx219,cam0
    ```
