## CSI 사용 방법

    * 미러링 플래싱이 완료된 후 `/boot/config.txt`에서 `camera_auto_detect=1`을 찾아서 `camera_auto_detect=0`으로 수정하고, 카메라 모델에 맞게 설정을 추가합니다.
    * 아래는 참조 설정입니다.

    ```bash
    dtoverlay=imx219,cam1
    ```
