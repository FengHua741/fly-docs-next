## 중요 경고
    :::tip 필독

    - 구성 파일의 `부분` 설정은 시스템을 처음 시작할 때만 적용됩니다.
    - 본 가이드는 수정 가능한 옵션만 제공합니다.

    :::

    :::warning 주의

    - 가능한 한 `mainsail` 또는 `fluidd` 웹 페이지를 사용하여 구성 변경을 수행하세요.
    - 웹 번역기를 사용하지 마세요. 그렇지 않으면 저장 후 시스템이 시작되지 않을 수 있습니다.
    - 구성 파일에 중국어나 특수 문자를 입력하지 마세요.
    - 구성 파일을 수정한 후에는 시스템을 재시작해야 적용됩니다.

    :::

    :::danger

    - 구성 파일에서 `######### PLEASE DO NOT EDIT THE FOLLOWING #########` 아래의 내용은 수정하지 마세요.

    :::

## 구성 파일 수정 방법

### 시스템 구동 후 구성 파일 수정

- 최초 구동 시 적용되는 구성 항목은 시스템 구동 후 전문 텍스트 편집기를 사용하여 이동식 디스크 `FlyOS-Conf`의 구성 파일 `config.txt`를 수정해야 합니다.

### 시스템 실행 중 온라인 구성 수정

- 일반적인 구성 항목은 시스템 웹 구성 페이지에서 수정한 후 시스템을 재시작하면 적용됩니다.
    - 브라우저에서 장치 IP 주소로 접속합니다. 예: `http://192.168.1.2/`
    - fluidd는 아래 왼쪽 그림처럼 `선택 해제` -> `숨겨진 파일 및 폴더 필터링`을 해제합니다.
    - mainsail은 아래 오른쪽 그림처럼 `선택` -> `숨겨진 파일 표시`를 선택합니다.

    - 이제 `.flyos-config` 폴더를 볼 수 있으며 이 폴더 내에 `sys-config.conf` 파일이 있습니다.
    - `sys-config.conf` 파일은 이동식 디스크 `FlyOS-Conf`의 구성 파일 `config.txt`의 소프트 링크입니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config4.webp').default} size="60%" align="center" />

    - 주의: `sys-config.conf` 파일의 구성 항목 형식은 `key=value`여야 하며 `key: value`로 변경하지 마세요.
    - 저장 후 시스템을 재시작하면 적용됩니다.

## 구성 항목

### 무선 네트워크 구성

- 아래 구성 항목은 시스템 시작 시 항상 적용됩니다.

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | `공백` | 임의의 WIFI 이름 | WIFI 이름, 입력하면 시스템 시작 시 우선적으로 연결됩니다. |
| `WIFI_PASS` | `공백` | 비밀번호가 없으면 공백 유지 | WIFI 비밀번호, 입력하면 비밀번호를 사용하여 연결합니다. |

----

### 프린터 기본 구성

:::tip
- 아래 구성 항목은 최초 시작 시만 적용됩니다.
:::

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :---: | :--- | :--- |
| `language` | `zh-CN` | <details><summary>자세히 보기</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | 최초 시작 시 fluidd와 mainsail의 기본 언어 설정 |
| `printer_name` | `공백` | 임의의 문자열 | 최초 시작 시 fluidd와 mainsail의 기본 프린터 이름 설정 |

----

### 시스템 구성

- 아래 구성 항목은 시스템 시작 시 항상 적용됩니다.
- 아래는 수정 가능한 옵션입니다.

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :-------: | :--- | :--- |
| `board` |  <code>{props.board}</code> |   <code>{props.board}</code> | 보드 모델, 사용 가능한 값 외에는 수정하지 마세요 |
| `localtime` | `Asia/Shanghai` | [세계 타임존 대조표](/docs/DebugDoc/flyos-fast/timezone) | 시스템 기본 타임존 설정, 시스템이 연결된 후 자동으로 시간을 조정합니다 |
| `canbus_bitrate` | `1000000` | `1000000`<br/> `500000`<br/> `250000` | FAST 시스템 모든 CAN 장치의 기본 속도 설정, 여기서만 수정 가능, 기본값은 1M |

----

### 화면 표시 구성

- 아래 구성 항목은 시스템 시작 시 항상 적용됩니다.

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :-------: | :--- | :--- |
| `screen` | `hdmi` | <code>nano</code>,<code>{props.screen}</code> <br/> <code>{props.screen1}</code><code>{props.screen2}</code> | 화면 설정, none일 경우 모든 표시를 비활성화 |
| `rotate` | `90` | `0`, `90`, `180`, `270` | FLY-TFT 회전 각도 설정, hdmi는 무효 |
| `klipper_screen` | `0` | `0`, `1` | KlipperScreen 활성화 여부 설정<br/>1: 활성화<br/>0: 비활성화<br/>`printer=klipper`일 경우 유효 |
| `moon2uart` | `0` | `0`, `1` | moon2uart 활성화 여부 설정<br/>1: 활성화<br/>0: 비활성화<br/>`printer=klipper`일 경우 유효 |

----

### Klipper 관련 구성

- 아래 구성 항목은 시스템 시작 시 항상 적용됩니다.

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :-------: | :--- | :--- |
| `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | Klipper 기본 UI 설정<br/>온라인 한 번 클릭 전환 기능에는 영향을 미치지 않습니다<br/>`printer=klipper`일 경우 유효 |
| `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | FAST 시스템 모든 CAN 장치의 기본 속도 설정, 여기서만 수정 가능, 기본값은 1M |
| `crowsnest` | `0` | `0`, `1` | Crowsnest 활성화 여부 설정<br/>1: 활성화<br/>0: 비활성화<br/>Klipper와 RRF 모드 모두 사용 가능 |

----

### 프린터 유형 구성

- 아래 구성 항목은 시스템 시작 시 항상 적용됩니다.

### 정전 시 자동 꺼짐

- 아래 구성 항목은 시스템 시작 시 항상 적용됩니다.
- 전원 차단 후 계속 인쇄하기 위해 정전 자동 꺼짐 기능을 비활성화해야 합니다.

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :-------: | :--- | :--- |
| `shutdown_pin_state` | `0` | `0`, `1` | 종료 버튼의 전압 상태 설정<br/> 1: 낮은 전압 반복<br/> 0: 높은 전압 트리거 |
| `shutdown_pin` | <code>{props.shutdown_pin}</code>| <code>{props.shutdown_pin}</code> | 종료 버튼의 트리거 핀 설정, none 설정 시 비활성화 |

----
