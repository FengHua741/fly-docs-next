## 중요한 경고
    :::팁 필수 읽기

    - 구성 파일의 `파트` 설정은 번들 후 첫 번째 시스템 시작 시에만 적용됩니다.
    - 구성 과정이 상당히 복잡하므로, 네트워크를 통해 기기를 연결한 후 브라우저로 구성하는 것을 강력히 권장합니다.
    - 이 가이드는 수정 가능한 옵션만 제공합니다.

    :::

    :::주의

    - 가능한 한 `mainsail` 또는 `fluidd` 웹페이지를 사용하여 구성 변경을 수행하세요.
    - 웹페이지 번역을 열지 마세요. 그렇지 않으면 저장 후 시스템이 시작되지 않을 수 있습니다.
    - 구성 파일에는 중국어나 특수 문자를 입력하지 않는 것이 좋습니다.
    - 구성 파일을 수정한 후에는 시스템을 재시작해야 변경 사항이 적용됩니다.

    :::

    :::경고

    - 구성 파일에서 `######### PLEASE DO NOT EDIT THE FOLLOWING #########` 아래의 내용은 수정하지 마세요.

    :::

## 구성 파일 수정 방법

### 번들 후 구성 파일 수정

- 첫 번째 시작 시만 적용되는 구성 항목은 번들 후 전문 텍스트 편집기를 사용하여 이동식 디스크 `FlyOS-Conf`의 구성 파일 `config.txt`를 수정해야 합니다.

### 시스템 시작 후 온라인 구성 수정

- 일반적인 구성 항목은 시스템 웹 구성 페이지에서 수정한 후 시스템을 재시작하면 적용됩니다.
    - 장치 IP 주소를 브라우저에서 열기, 예: `192.168.1.2/`
    - fluidd는 아래 왼쪽図와 같이 `체크박스 해제` -> `숨김 파일 및 폴더 필터링`
    - mainsail은 아래 오른쪽図와 같이 `체크박스 선택` -> `숨김 파일 표시`

    <table>
      <tr>
        <td>
          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config1.webp').default} size="100%" align="center" />
        </td>
        <td>
          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />
        </td>
      </tr>

    - 이제 `.flyos-config` 폴더를 볼 수 있으며, 이 폴더 내부에는 `sys-config.conf` 파일이 있습니다.
    - `sys-config.conf` 파일은 이동식 디스크 `FlyOS-Conf`의 구성 파일 `config.txt`의 소프트 링크입니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - `sys-config.conf` 파일을 열어 필요한 구성 항목을 수정한 후 `저장` -> `닫기`하고 재시작하면 됩니다.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config4.webp').default} size="60%" align="center" />

    - 주의: `sys-config.conf` 파일의 구성 항목 형식은 `key=value`이며, `key: value`로 수정하지 마세요.
    - 수정 후 저장하고 시스템을 재시작하면 적용됩니다.

## 구성 항목

### 무선 네트워크 구성

- 다음 구성 항목은 시스템 시작 시마다 적용됩니다.

| 구성 항목 | 기본값 | 사용 가능한 값 | 설명 |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | `공백` | 임의의 WIFI 이름 | WIFI 이름, 입력하면 시스템 시작 시 자동으로 우선적으로 연결됩니다 |
| `WIFI_PASS` | `공백` | 비밀번호가 없는 경우 공백 유지 | WIFI 비밀번호, 입력하면 비밀번호로 WIFI에 연결 |
