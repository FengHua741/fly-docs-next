## 설정

### 전원이 끊기면 종료 비활성화
    :::danger 주의
    - 전원이 끊기면 종료 기능을 비활성화해야 하며, 그렇지 않으면 중단된 상태에서 다시 시작할 수 없습니다.
    - 중단된 상태에서 다시 시작할 때 자동으로 전원이 꺼지는 기능이 포함되어 있습니다.
    :::

- 브라우저의 주소 표시줄에 장치의 IP 주소를 입력합니다. 예: `http://192.168.6.179`
- 구성 페이지로 이동합니다.
    - 장치의 IP 주소를 브라우저에서 열기, 예: `http://192.168.1.2/`
    - Fluidd는 아래 왼쪽 이미지와 같이 `선택 해제` -> `숨겨진 파일 및 폴더 표시`를 수행합니다.
    - Mainsail은 아래 오른쪽 이미지와 같이 `선택` -> `숨겨진 파일 표시`를 수행합니다.

### plr.cfg 구성 파일
    :::danger 주의
    * 구성 파일에서: (PINS)  
    * `<code>{props.power_pine}</code>`로 대체해주세요.

    :::

    - 프린터 구성 페이지에서 `plr.cfg` 파일을 찾아서 엽니다.
    - 파일 내용을 비운 다음 아래의 구성 내용을 붙여넣습니다.
    - 구성 파일 내용은 다음과 같습니다:

    ```cfg
    [mcu host]   
    serial: /tmp/klipper_host_mcu
    
    [power_loss_resume]
    power_pin: (PINS)
    is_shutdown: True # 셧다운 여부, 기본값은 활성화
    paused_recover_z: -2.0 # 일시정지 상태에서 중단되었을 때 복구 시 Z축 이동 거리, 기본값은 이동하지 않음
    start_gcode:
        # 재개하기 전에 실행할 G-code
        # 셧다운 전에 저장된 모든 매개변수는 {PLR}를 통해 접근 가능
        # M118 {PLR}를 사용하여 모든 사용 가능한 매개변수 출력
        M118 시작 재개: {PLR.print_stats.filename}
        M118 중단 위치: X:[{PLR.POS_X}] Y:[{PLR.POS_Y}] Z:[{PLR.POS_Z}] E:[{PLR.POS_E}]
        M140 S{PLR.bed.target}      ; 열판 온도 설정
        M104 S{PLR.extruder.target-10}  ; 노즐 온도 설정
        M109 S{PLR.extruder.target-10}  ; 노즐이 설정된 온도까지 가열 대기
        G91                             ; 상대 좌표
        G1 Z2 F100                      ; Z축 상승, X,Y 리셋 준비
        G90                             ; 절대 좌표
        G28 X Y                         ; X,Y 리셋
        M140 S{PLR.bed.target}          ; 열판 온도 설정
        M104 S{PLR.extruder.target}     ; 노즐 온도 설정
        M190 S{PLR.bed.target}          ; 열판이 설정된 온도까지 가열 대기
        M109 S{PLR.extruder.target}     ; 노즐이 설정된 온도까지 가열 대기
        M83                             ; 상대적 배출
        # G1 E0.5 F400                  ; 약간 배출
    layer_count: 2 # 특정 층 수만큼 재개 후 layer_change_gcode 실행
    layer_change_gcode:                
        # {layer_count}층만큼 재개 후 실행할 G-code
        M118 재개 속도 복구
        M106 S{PLR.fan_speed}             ; 블로워 팬 켜기
        M220 S{PLR.move_speed_percent}    ; 요청 속도 퍼센트 설정
        M221 S{PLR.extrude_speed_percent} ; 요청 배출 속도 퍼센트 설정
    shutdown_gcode:
        # 셧다운 전에 실행할 G-code
        M118 전원 전압 저하로 인한 셧다운
        # M112 ; 긴급 정지

    ```

    :::warning 주의

    - 위 구성 파일의 `start_gcode` 매크로는 실제 장비 사양에 따라 수정해야 할 수 있습니다.
    - `[homing_override]`를 사용하는 경우 무작위로 리미트 스위치 위치를 설정하지 마십시오. 이로 인해 발생하는 중단된 상태에서 다시 시작 실패는 책임지지 않습니다.

    :::

- 위 구성 파일을 저장한 후
- `printer.cfg` 파일을 열고 파일 맨 위에 다음 내용을 추가합니다:

    ```cfg

    [include plr.cfg]

    ```

- 우측 상단의 저장 및 재부팅 버튼을 클릭합니다.
- 이제 Klipper의 전원이 끊기면 다시 시작 기능이 설정되었습니다.

## 테스트

- 임의의 파일을 인쇄하고 인쇄 도중 `긴급 정지` 버튼을 클릭하여 전원이 끊긴 것처럼 시뮬레이션합니다.
- 다음으로 `펌웨어 재시작`을 클릭하고 Klipper가 정상적으로 연결될 때까지 기다립니다.
- 웹 페이지나 KlipperScreen에서 팝업 메시지가 나타나면 전원이 끊기면 다시 시작 기능이 정상 작동한다는 것을 의미합니다.
- 이후 실제 전원 끊김 상황에서도 테스트할 수 있습니다.
