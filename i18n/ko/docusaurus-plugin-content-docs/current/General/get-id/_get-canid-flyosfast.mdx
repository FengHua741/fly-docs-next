## CAN ID 입력하기

    1. `DEVICES`를 닫고 ID 입력
    :::danger 주의事项

    * 첫 번째 붉은 박스 앞의 #을 삭제해야 합니다. 공백 포함해서 삭제해야 하며, `serial:`을 `canbus_uuid:`으로 변경해야 합니다.
    * FLY 시스템 기본 설정에는 [MCU]가 있을 수 있으나 아래 ID가 `serial: /tmp/klipper_host_mcu`라면 `[MCU]`를 `[mcu HOST]`로 변경하거나 해당 두 줄을 삭제해야 합니다.
    * 아래에서 `canbus_uuid:`를 복사할 수 있습니다.
    :::

    ```cfg
    [mcu]
    canbus_uuid:
    ```

    3. Klipper가 `ADC out of range`를 나타내면 이것은 정상적인 현상입니다. 열판과 온도 센서를 메인보드에 연결하고 노즐, 열판의 온도 센서 핀을 설정한 후 저장 및 재시작하면 됩니다.

:::
