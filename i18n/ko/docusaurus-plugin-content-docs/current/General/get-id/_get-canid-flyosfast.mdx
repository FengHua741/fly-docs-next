## CAN ID 입력하기

    1. `DEVICES`를 닫고 ID를 입력합니다
    :::danger 주의사항
    * 구성 파일에서 다음 부분을 수정해야 합니다:  
    ```
    [mcu] 
    serial: /tmp/klipper_host_mcu
    ```  
    수정된 내용은 다음과 같습니다:
    ```
    [mcu host] 
    serial: /tmp/klipper_host_mcu
    ```
    * 추가적으로 다음 내용을 입력합니다:
    ```
    [mcu]
    canbus_uuid: <여기에 방금 조회한 ID를 입력하세요>
    ```
    :::

    3. Klipper에서 `ADC out of range`라는 메시지가 나타나는 경우, 이는 정상적인 현상입니다. 핫베드와 히터를 메인보드에 연결하고 노즐과 핫베드의 히터 핀을 설정한 후 다시 저장하고 재시작하면 됩니다.

:::danger
주의: 문서에서 나온 모든 ID는 예제입니다. 각 메인보드의 ID는 모두 다릅니다. 실제로 얻은 ID를 입력해야 합니다.
:::
