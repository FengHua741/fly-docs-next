1. 브라우저를 열고 주소 표에 호스트 컴퓨터의 IP 주소를 입력합니다. 예를 들어, 호스트 컴퓨터의 IP가 `192.168.101.179`라면 이 주소를 입력하고 Enter 키를 누릅니다.

## USB ID 입력하기

    1. `DEVICES`를 닫고 ID를 입력하세요.

    :::danger 주의사항
      * 구성 파일에서 다음 부분을 수정하세요:
      ```
      [mcu] 
      serial: /tmp/klipper_host_mcu
      ```  
      수정된 내용은 다음과 같습니다:
      ```
      [mcu host] 
      serial: /tmp/klipper_host_mcu
      ```
      * 다음 항목을 추가하세요:
      ```
      [mcu]
      serial: <여기에는 방금 검색한 ID를 입력하세요>
      ```
      :::
    <table>

        <td>
          - fluidd에서 [MCU]를 찾습니다.

        </td>

    2. ID를 구성 파일에 입력하세요.
    :::tip

    * ID를 입력한 후 오른쪽 상단의 `SAVE & RESTART`를 클릭하세요.
    :::

      <tr>

    3. Klipper에서 `ADC out of range`라는 메시지가 나타나면 이는 정상적인 현상입니다. 핫베드와 센서를 메인보드에 연결하고 헤드, 핫베드의 센서 핀을 구성한 후 다시 저장하고 재부팅하세요.

:::danger
주의: 문서에서 나온 모든 ID는 예제입니다. 각 메인보드의 ID는 모두 다르므로 실제로 얻은 ID를 입력하세요.
:::
