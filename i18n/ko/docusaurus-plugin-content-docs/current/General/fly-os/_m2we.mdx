# M2WE에 USB를 사용하여 부팅하기

### 준비 작업

### 준비 작업

    * 먼저 `SDFormatter` 및 `Rufus` 소프트웨어를 설치합니다. <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">포맷 도구</Button>
    * [M2WE 부트 드라이버](/docs/ResDownload/auxiliary_software/ "클릭하여 바로 이동") 다운로드
    * 시스템을 압축 해제한 후 파일 이름의 확장자가 `.img`로 끝나야 합니다.
    * `balenaEtcher` 또는 `Raspberry Pi`를 사용한 시스템 번들링은 권장하지 않습니다.
    * M2WE 또는 SD 카드를 M2SD에 장착한 후 컴퓨터의 USB 인터페이스에 M2SD를 연결합니다.
    * M2WE에 데이터가 있는 경우 백업하시기 바랍니다.

### 드라이버 설치

    * `Zadig` 소프트웨어를 실행합니다.
    * 첫 번째 `Zadig`는 설치 없이 바로 실행할 수 있고, 나머지 두 개는 설치해야 합니다. 설치 과정에서는 모두 `다음` 또는 `Next`를 클릭하면 됩니다.
    * `Zadig`를 열고 상단 메뉴의 `Options`에서 `모든 장치 표시`를 선택합니다.

    * 위의 그림에서 두 번째 항목은 반드시 `WinUSB(vxxxxxxx)`를 선택해야 합니다. 버전 번호는 중요하지 않습니다.
    * `드라이버 설치`를 클릭하여 드라이버를 설치하세요.

    * USB 드라이버 설치가 성공했습니다 (드라이버는 한 번만 설치하면 됩니다. 이후 다시 번들을 할 때는 드라이버를 재설치할 필요가 없습니다).
    * 설치된 FLY-BOOT를 열고 `장치 검사`를 클릭합니다.
    * 앞의 단계들이 정확히 이루어졌다면 장치가 사용 가능하다는 메시지를 받게 될 것입니다.
    * `BOOT`를 시작하면 정상적으로 작동하면 `BOOT`가 성공적으로 활성화되었다는 메시지를 받습니다.
:::
### M2WE 포맷하기

    * `TFFormatter.exe` 소프트웨어를 사용하여 `BOOT` 디스크를 포맷합니다.

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * 또는 Windows 내장 도구인 디스크 관리에서 파티션을 삭제한 후 새로 파티션을 생성하면 됩니다.
    * 이제 파일 탐색기를 열면 14.5GB 용량의 U-disk가 나타납니다 (16GB eMMC는 14.5GB로 표시됨).
    * 이렇게 하면 Win32DiskImager를 사용하여 시스템 이미지를 번들링할 수 있습니다. 디스크 선택은 이 14.5GB U-disk의 드라이브를 선택하시기 바랍니다.
