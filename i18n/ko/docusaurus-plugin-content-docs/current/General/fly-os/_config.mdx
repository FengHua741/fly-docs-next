## 구성 파일 수정 방법

### 구성 파일 수정 방법 (번개 OS 이미지를 번들링한 후)

- 최초 시작 시 적용되는 구성 항목은 번들링 작업을 완료한 후, 전문 텍스트 편집기로 이동식 디스크 `FlyOS-Conf`의 구성 파일 `config.txt`를 열고 수정해야 합니다.

### 시스템 실행 중 온라인 구성 수정

- 일반 구성 항목은 시스템 웹 구성 페이지에서 수정한 후 시스템을 재시작하면 적용됩니다.
    - 장치의 `ip+:9999` 주소를 브라우저에서 열어주세요. 예: `192.168.1.2:9999`
    - 호스트 이름 변경이 지원됩니다.
    - 시스템 시간대 변경이 지원됩니다.
    - 상위기기 40핀 확장을 활성화할 수 있습니다. (PWM 활성화 후에는 상위기기와의 시리얼 연결이 불가능하며, 화면 활성화 후에는 상위기기 가속도 센서 포트를 사용할 수 없음)
    - 메인보드 모델 변경이 지원됩니다.
    - Klipper 웹 페이지 수정이 지원됩니다. Fluid 및 Mainsail도 지원됩니다.
    - KlipperScreen을 켜거나 끌 수 있습니다.
    - Klipper 프로세스를 특정 CPU 코어에서 강제로 실행하도록 설정할 수 있으며, Klipper가 선택한 코어에서 실행되도록 지정할 수 있습니다.
