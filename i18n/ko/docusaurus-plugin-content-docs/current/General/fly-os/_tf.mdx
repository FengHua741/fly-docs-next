# 시스템 TF 카드 플래싱

### 준비 작업

    * 먼저 `SDFormatter`와 `Rufus` 소프트웨어를 설치하세요.
    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">포맷 도구</Button>
    * 시스템을 압축 해제한 후 파일명의 확장자가 `.img`로 끝나야 합니다.
    * `balenaEtcher` 또는 `Raspberry Pi`를 사용한 시스템 플래싱은 권장하지 않습니다.
    * TF 카드를 리더기에 삽입하고 리더기를 컴퓨터의 USB 포트에 연결합니다.
    * TF 카드에 데이터가 있다면 백업하십시오. 시스템 카드는 플래싱이 완료된 후에도 분리하지 않는 것이 좋습니다. TF 카드를 M2WE로 플래싱해야만 다시 분리할 수 있습니다.

### 메모리 카드 포맷하기

    * `SDFormatter.exe` 소프트웨어를 사용하여 TF 카드를 포맷합니다.
    * 포맷할 디스크를 선택하세요.

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * 또는 Windows 기본 제공 "디스크 관리"에서 파티션을 삭제한 후 새 파티션을 생성할 수 있습니다.
