# 시스템 부트 TF 카드

### 준비 작업

    * 먼저 `Win32DiskImager` 소프트웨어를 설치합니다.
    * 시스템을 압축 해제하면, 압축 해제된 파일의 확장자가 `.img`가 됩니다.
    * 시스템 부트에 `balenaetcher`를 사용하지 않는 것이 좋습니다.
    * TF 카드를 리더기에 삽입하고, 리더기를 컴퓨터의 USB 포트에 연결합니다.
    * TF 카드에 데이터가 있다면 백업하십시오. 시스템 카드는 시스템 부트 후 분리하지 않는 것이 좋습니다. M2WE로 다시 부트해야 하는 경우에만 TF 카드를 분리할 수 있습니다.

### 메모리 카드 포맷

    * `TFFormatter.exe` 소프트웨어를 사용하여 TF 카드를 포맷합니다.
    * 포맷할 디스크를 선택합니다.

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * 또는 Windows 자체의 디스크 관리에서 파티션을 삭제한 후, 파티션을 다시 생성할 수 있습니다.
