# TF 카드를 통해 M2WE로 부팅

:::info
시스템 이전은 TF 카드에 있는 기존 이미지를 M2WE로 완전히 복사하는 것을 가능하게 합니다. 이 과정이 완료되면 M2WE의 내용은 원래 TF 카드의 내용과 동일하게 유지됩니다.
:::

    * M2WE를 설치한 후 전원을 켜고 부팅합니다.
    * 먼저 [SSH 연결](/docs/DebugDoc/BasicTutorial/index.mdx "클릭하여 바로 이동")을 사용해 연결하세요.
    * 아래 명령어를 실행하면 비밀번호 입력이 요청될 수 있습니다. `mellow`를 입력하고 Enter 키를 누릅니다. (비밀번호는 화면에 표시되지 않습니다)

    ```cfg
    sudo nand-sata-install
    ```

    * `Boot from eMMC - system on eMMC` 옵션을 선택하고 Enter 키를 누릅니다. (일반적으로 기본값이므로 Enter 키만 누르면 됩니다.)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_1.webp').default} size="80%" align="left" />

    * `This script will erase your eMMC. Continue?`가 나타날 때 `Yes`를 선택하고 Enter 키를 누릅니다. (일반적으로 기본값이므로 Enter 키만 누르면 됩니다.)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_2.webp').default} size="80%" align="left" />

    * 파일 시스템 유형을 `ext4`로 설정하고 Enter 키를 누릅니다. (일반적으로 기본값이므로 Enter 키만 누르면 됩니다.)

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_3.webp').default} size="80%" align="left" />

    * 진행 상황 바가 나타날 때까지 기다립니다. (2분 이상 대기 후에도 다음 그림의 진행 상황 바가 나타나지 않는 경우 Ctrl+C를 눌러 작업을 중단하고 장치를 재부팅한 후 다시 시도하세요.)
    
    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_4.webp').default} size="80%" align="left" />
    * 시스템 이전 도중 전원을 끄거나 다른 작업을 하지 마세요. 이전이 완료될 때까지 기다리세요.
    * 이전이 완료되면 Enter 키를 누르고 장치가 완전히 꺼질 때까지 기다립니다.

    <ImageView image={require('@site/docs/General/fly-os/img/emmc/emmc_install_5.webp').default} size="80%" align="left" />


    * 전원을 끄고 TF 카드를 제거한 후 다시 전원을 켭니다. 이제 eMMC의 시스템으로 부팅할 수 있습니다.
