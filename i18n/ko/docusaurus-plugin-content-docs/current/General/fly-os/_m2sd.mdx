### M2WE 또는 SD 카드를 M2WE에 설치하기

    :::danger 주의사항

    * 본 제품은 USB3.0을 지원합니다.
    * 본 제품은 TF와 M2WE를 동시에 설치한 후 사용할 수 없습니다.
    * M2WE 또는 SD 카드를 설치한 후 컴퓨터의 USB 포트에 연결하면 포맷할 수 있습니다.
    * NVME 등의 다른 장치를 설치하지 마십시오. 그렇지 않으면 장치가 손상될 수 있습니다!!!

### 준비 작업

    * 먼저 `Win32DiskImager` 소프트웨어를 설치하십시오.
    * 시스템 브닝에 `balenaetcher`를 사용하는 것은 권장하지 않습니다.
    * 시스템을 압축 해제하고, 압축 해제된 파일의 확장자는 `.img`입니다.
    * TF 카드를 리더기에 삽입하고, 리더기를 컴퓨터의 USB 단자에 연결하십시오.
    * TF 카드에 데이터가 있는 경우 백업하십시오. 시스템 카드는 시스템을 브닝한 후 분리하지 않는 것이 좋습니다. M2WE로 다시 브닝해야 할 경우에만 TF 카드를 분리하십시오.

### 메모리 카드 포맷하기

    * `TFFormatter.exe` 소프트웨어를 사용하여 포맷하십시오.
    * 포맷할 디스크를 선택하십시오.

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * 아니면 Windows 자체의 디스크 관리에서 파티션을 삭제한 후 다시 파티션을 생성하십시오.
