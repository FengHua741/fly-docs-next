## FLY 상위기기 사용 설명서

:::tip
브라우저에서 IP:9999을 열어 FLY-Tools에 액세스하세요.
:::

:::tip
FLY 상위기기의 내장 HDMI 포트는 마이크로 HDMI입니다. HDMI 케이블을 구매할 때 유의하시기 바랍니다!!!
:::

:::tip
FLY 상위기기는 `FLY-PI-V2`, `FLY-C8` 및 `FLY-PRO-X10`에서 FPC 원터치를 지원합니다.
:::

:::tip
FPC에서 HDMI 모듈을 사용하면 Raspberry Pi와 같은 상위기기를 호환할 수 있습니다.
:::

    * 화면을 사용하려면 HDMI 아래의 옵션을 켜야 합니다.
