::: 팁

- 상위 컴퓨터가 정상적으로 인터넷에 연결되어 있는지 확인하세요.
- 펌웨어를 컴파일하기 전에 SSH를 통해 네트워크를 통해 상위 컴퓨터에 로그인해야 합니다.
- 상위 컴퓨터에 시리얼 포트 도구를 사용하지 마세요.

:::

### Klipper 펌웨어 컴파일 및 구성 페이지 조작 설명

::: 팁

키보드 입력 모드가 반각 모드, 즉 영문 모드인지 확인하세요.

:::

1. 키보드의 `↑` 키는 커서를 위로 이동시키고, `↓` 키는 아래로 이동시켜 메뉴 항목을 선택합니다.
2. 확인 키 `Enter` 또는 공백 키 `Space`는 메뉴를 체크하거나 하위 메뉴로 진입합니다.
3. 종료 키 `ESC`는 상위 메뉴로 돌아갑니다.
4. `Q` 키는 Klipper 펌웨어 구성 페이지를 종료합니다.
5. `Y` 키는 Klipper 펌웨어 구성 페이지를 종료할 때, 프롬프트가 표시되면 `Y` 키를 눌러 구성 파일을 저장합니다.

::: 주의

옵션 수가 적은 경우 `[ ] Enable extra low-level configuration options`를 체크하여 일부 숨겨진 옵션이 표시되도록 합니다.

:::
