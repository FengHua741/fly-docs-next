# 라즈베리파이 이미지


:::warning

SD 카드 주의사항:

* 용량은 `16GB` 이상 또는 `128GB` 이하의 TF 카드여야 합니다.
* TF 카드의 전송 속도는 `class10`급 또는 그 이상이어야 하며, SanDisk 등의 브랜드를 권장합니다.

:::

:::danger 주의

* 서버 문제로 임의로 시스템을 업데이트하거나 Klipper를 업데이트하지 마십시오.
* Raspberry Pi 3B/3B+/4B/CM4/5B에 적합한 시스템 이미지입니다.
* 이 이미지는 다양한 서비스를 `bookworm` 기반으로 설치합니다.
* 이 이미지는 기본적으로 TFT 스크린을 설치하고 활성화하며, 필요하지 않다면 수동으로 `/boot/config.txt`를 수정해야 합니다.

:::

:::info 중요한 안내

* 다운로드한 시스템은 압축을 풀어야 하며, 단순히 확장자를 변경하거나 바로 번역할 수 없습니다!!!!
* 압축을 풀지 않고 번역하면 시작할 수 없습니다!!!!

:::
