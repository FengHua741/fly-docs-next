# {props.name} 시스템 다운로드

:::warning

SD 카드 주의사항:

* 용량은 `16GB` 이상 또는 `128GB` 이하의 TF 카드여야 합니다.
* TF 카드의 전송 속도는 `class10`급 또는 그 이상이어야 하며, 권장 브랜드는 SanDisk 등입니다.

:::

:::danger 주의

* 서버 문제로 인해 임의로 시스템이나 klipper를 업데이트하지 마십시오.
* FLYOS-Armbian 시스템을 다운로드하십시오. 기본 Armbian 설치는 사용하기 어려우므로 설치하지 않는 것이 좋습니다.

:::

:::info 중요한 안내

* 다운로드한 시스템은 압축을 풀어야 하며, 확장자를 변경하거나 바로 번역하지 마십시오!!!!!
* 압축을 풀지 않고 번역하면 부팅이 되지 않습니다.

:::
