            ## v2
                * 2022-1
                * 배포 버전을 Bullseye로 변경
                * 커널을 5.10.85로 업데이트
                * Klipper, Moonraker, Fluidd, Mainsail, KlipperScreen 사전 설치
                * Klipper가 Python3 환경을 사용하여 한글 파일명 문제 해결
                * boot 파티션을 fat로 수정하여 Windows 시스템에서 구성 파일을 직접 수정할 수 있음
                * FLY-Config, 새로운 구성 시스템 추가
                * CatSerial 추가, ```http://ip:9999```에 액세스하여 모든 연결된 시리얼 장치를 직접 확인 가능

            ## v2.1

            ## v2.9.3
                * FLY-Π 호환성 추가

                * udev 규칙 충돌 문제 수정

                * CAN 전송 버퍼 크기를 1024로 수정
