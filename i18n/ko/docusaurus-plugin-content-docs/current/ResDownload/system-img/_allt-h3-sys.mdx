# {props.name} 시스템 다운로드

:::warning

SD 카드 주의사항:

* 용량은 `16GB` 이상 또는 `128G` 이하의 TF 카드여야 합니다.
* TF 카드의 전송 속도는 `class10`급 또는 그 이상이어야 하며, SanDisk 등의 브랜드를 권장합니다.

:::

:::danger 주의

* 서버 문제로 인해 시스템을 무작위로 업데이트하거나 klipper를 업데이트하지 마십시오.
* FLY OS 시스템을 다운로드하십시오. 원판 Armbian은 설치가 어려워 권장하지 않습니다.

:::

:::info 중요 안내

* 다운로드한 시스템은 압축을 풀어야 하며, 확장자를 변경하거나 바로 번역해서는 안 됩니다!!!!
* 압축을 풀지 않고 번역하면 부팅할 수 없습니다!!!!

:::
