# Скачивание ресурсов

:::tip

Пожалуйста, выберите соответствующий тип ресурса в навигационном меню слева для загрузки.

:::

:::warning Важно

Обратите внимание на имя файла и подсказки на странице при его скачивании, чтобы избежать загрузки неправильного файла.

:::

:::danger Предупреждение

Некоторые файлы могут находиться на Github или в облачных хранилищах. При скачивании убедитесь, что интернет-соединение работает нормально.

:::
