# Возобновление печати после отключения питания в Klipper

- Возобновление печати после отключения питания в Klipper означает, что после восстановления питания Klipper автоматически восстанавливает состояние печати.
- ПО FLY, за исключением материнской платы C5, должно использоваться вместе с модулем отключения питания KPPM.
- Не подходит для моделей принтеров, где при отключении питания Z-ось смещается.
