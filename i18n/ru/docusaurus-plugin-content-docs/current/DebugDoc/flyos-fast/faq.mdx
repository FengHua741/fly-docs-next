# Часто задаваемые вопросы о FlyOS-Fast

## Вопрос: При выполнении команды появляется сообщение `command not found`
- **Причина**: Система не содержит эту команду.
- **Решение**: В системе уже предустановлены многие часто используемые пакеты. Если нужен другой пакет, отправьте его нам. При необходимости он будет предустановлен в следующей версии.

## Вопрос: Почему нельзя выполнить команду `apt update`?
- **Причина**: FlyOS-Fast является лёгкой быстрой системой и не имеет инструмента управления пакетами apt, поэтому команда `apt update` не может быть выполнена.
- **Решение**: В системе уже предустановлены многие часто используемые пакеты. Если нужен другой пакет, отправьте его нам. При необходимости он будет предустановлен в следующей версии.

## Вопрос: При выполнении команды для прошивки появляется сообщение `-bash: ~/klippy-env/bin/python: No such file or directory`
- **Причина**: FlyOS-Fast не использует виртуальное окружение venv.
- **Решение**: Замените `~/klippy-env/bin/python` на `python`.

## Вопрос: Почему нельзя войти как пользователь `fly`?
- **Причина**: FlyOS-Fast имеет только одного пользователя root, все операции выполняются под этим пользователем.
- **Решение**: В системе нет пользователя `fly`. Для входа используйте пользователя `root`.

## Вопрос: Почему не открывается fly-tools на порту 9999?
- **Причина**: FlyOS-Fast не установлен fly-tools.
- **Решение**: Fluidd и mainsail уже поддерживают запрос ID, настройка экранов находится в [системной конфигурации](/docs/DebugDoc/flyos-fast/config).

## Вопрос: Почему на экране показывается только логотип, но нет интерфейса KlipperScreen?
- **Причина**: KlipperScreen не запущен.
- **Решение**: Включите KlipperScreen в [системной конфигурации](/docs/DebugDoc/flyos-fast/config), установив значение `klipper_screen` равным `1`.

## Вопрос: Почему система не предоставляет веб-интерфейс для обновления компонентов klipper и других?
- **Причина**: Встроенные в FlyOS-Fast компоненты klipper, moonraker и другие являются версиями с доработками.
- **Решение**: При необходимости можно самостоятельно обновить их через командную строку, однако это не рекомендуется. Ожидайте обновлений системы.
