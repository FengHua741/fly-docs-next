# Инсталляция системного образа

## Подготовка

1. Завершите загрузку системы на предыдущей странице [Система для быстрой отладки](/docs/DebugDoc/flyos-fast/).
2. Скачайте инструмент для записи образов, в примере будет использоваться `Rufus`. На самом деле можно использовать другие инструменты, такие как `Etcher`, `Win32 Disk Imager` и т. д.
    - Ссылка для скачивания Rufus на китайском языке: [скачать](https://rufus.ie/zh/#download) или английском языке: [скачать](https://rufus.ie/en/#download) (поддерживает только Windows).
    - Ссылка для скачивания Etcher: [скачать](https://etcher.balena.io/#download-etcher) (поддерживает Windows, MacOS, Linux).
    - Ссылка для скачивания Win32 Disk Imager: [скачать](https://sourceforge.net/projects/win32diskimager/) (поддерживает только Windows).
3. Скачайте системный образ, в примере будет использоваться `FlyOS-Fast-H5.exe`.

4. Распакуйте системный образ, чтобы получить файл `.img`.
    - Дважды щелкните или откройте правой кнопкой мыши `FlyOS-Fast-H5.exe`, и через несколько секунд появится следующее окно (если система предупреждает о небезопасности, выберите «Подробнее» -> «Запустить»).
    - Выберите директорию для распаковки, затем нажмите кнопку `Extract`, подождите, пока процесс завершится.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - После завершения распаковки в директории для распаковки появится файл `FlyOS-Fast-H5-vx.x.x.img`. `x.x.x` — это номер версии системы.
5. Подготовьте карту памяти для записи.
    - Карта памяти должна быть больше 4 ГБ, рекомендуется использовать TF-карту SanDisk 32 ГБ C10 и выше (во время тестирования было обнаружено, что многие дешевые карты не могут запуститься после записи).
    - Карта памяти должна быть использована с кардридером SD-карт, вставьте карту памяти в кардридер, затем вставьте кардридер в компьютер.

## Запись системы

1. Откройте Rufus, выберите устройство, которое должно отображать объем, соответствующий объему карты памяти, вставленной в компьютер.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="30%" align="center" />

2. Выберите тип загрузки, по умолчанию выберите `образ файла`.
3. Выберите файл образа, нажмите кнопку `Выбрать` справа от `тип загрузки`, выберите распакованный файл `.img`.
4. После выбора файла образа, `тип загрузки` изменится на имя файла образа.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="30%" align="center" />

5. Нажмите кнопку `Начать`, если появится окно с предупреждением о необходимости очистки дисковых данных, нажмите `ОК`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />

6. После завершения записи компьютер издаст звуковой сигнал, и полоса прогресса будет показывать `Готово`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="30%" align="center" />

7. Не торопитесь вытаскивать карту памяти после завершения записи.
8. Откройте проводник, проверьте наличие флешки или портативного устройства с именем `FlyOS-Conf`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />

9. Откройте `FlyOS-Conf`, если присутствуют файлы, показанные на рисунке ниже, это указывает на успешную запись.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

10. Файл `config.txt` является файлом конфигурации системы, пожалуйста, ознакомьтесь с содержимым на следующей странице. Измените файл конфигурации.
