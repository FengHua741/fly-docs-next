# Таблица соответствия часовых поясов мира

:::tip Важно

- Для настройки часового пояса укажите идентификатор часового пояса в параметре `localtime`, например: `Asia/Shanghai`.
- После перезагрузки системы настройка вступит в силу.
- До подключения к сети время будет неправильным, после подключения оно автоматически синхронизируется.

:::
