# Встроенные компоненты системы

:::danger

- Все компоненты и службы, встроенные в систему FlyOS-Fast, не поддерживают самостоятельную модификацию и обновление пользователей.
- Изменения, внесенные вами самостоятельно, могут привести к проблемам с доступом к веб-страницам или невозможности запуска системы.
- Пожалуйста, перепрошить систему.

:::

## Klipper

:::warning

- Вы можете попробовать добавить сторонние дополнения к существующему Klipper, но не изменяйте существующие модули дополнений.

:::

### Автоматическая приостановка при возникновении ошибок

- При печати, если у Klipper возникают ошибки, такие как аномалии температуры или драйвера, он автоматически приостанавливает печать, но не выключается.

### Оптимизация реального времени

- Оптимизация реального времени для Klipper достигается путем изменения ядра и кода Klipper, что позволяет улучшить реальное время Klipper.
- Эффективно снижает ошибку `Timer too close`.

### Поддержка продолжения печати после отключения питания или возникновения ошибок

- Поддерживает продолжение печати после отключения питания.
- Поддерживает возобновление печати после ошибок.

### Поддержка TCP сокетов

- Поддерживает одновременное использование Unix сокетов и TCP сокетов.
- Используйте параметр -p для указания порта TCP для включения TCP сокетов.

### Автоматическая обработка файлов с USB-накопителя

- Поддерживает автоматическую обработку файлов с USB-накопителя, чтобы избежать прерываний печати.

## Moonraker

### Поддержка TCP сокетов

- Поддерживает подключение к klippy через TCP сокеты.
- Параметр `klippy_tcp_address` указывает адрес TCP.
- Параметр `klippy_tcp_port` указывает порт TCP.

### Оптимизация загрузки больших файлов

- Решает проблемы с ошибками при последовательной загрузке больших файлов.

### Управление сетью - для экранов FLY5,7

- Поддерживает управление беспроводной сетью, подходящее для экранов FLY5,7.

### Поддержка экранов FLY

- Поддерживает экраны FLY.
- Поддерживает генерацию изображений первого слоя модели.
