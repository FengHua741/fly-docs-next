# Запись выпусков системы

## V1.2.39

### Изменения

- Устранена проблема с невозможностью запуска после прошивки на систему Windows11 24H2.

### Добавления

- Добавлена система конфигурации `canbus_bitrate`, для установки скорости по умолчанию канала can0, значение по умолчанию `1000000` (то есть `1M`).
- Добавлена система конфигурации `localtime`, для установки часового пояса по умолчанию, значение по умолчанию `Asia/Shanghai`.
