# Запись о выпуске системы

## V1.2.40

### Изменения

- Исправлены известные проблемы
- Добавлена поддержка новых параметров для восстановления печати после отключения питания в klipper: `paused_recover_z`, используется для установки компенсации высоты Z-оси при возобновлении печати после паузы и отключения питания
- klipper обновлен до последней версии
- moonraker обновлен до последней версии
- KlipperScreen обновлен до последней версии
- katapult обновлен до последней версии
- fluidd обновлен до последней версии
- mainsail обновлен до последней версии

### Новые функции

- Добавлен новый расширение klipper [klipper-led_effect](https://github.com/julianschill/klipper-led_effect.git)

----

## V1.2.39

### Изменения

- Устранена проблема с невозможностью запуска системы при прошивке на Windows 11 24H2

### Новые функции

- Добавлена новая конфигурация системы `canbus_bitrate`, для установки скорости по умолчанию канала can0, значение по умолчанию `1000000` (то есть `1M`)
- Добавлена новая конфигурация системы `localtime`, для установки часового пояса по умолчанию, значение по умолчанию `Asia/Shanghai`
