# Введение в систему и загрузка

## Введение

* FlyOS-Fast — это облегченная система на базе Linux, специально адаптированная для 3D-печати и поддерживающая все платформы верхнего уровня Fly.
* FlyOS-Fast облегченная и быстро запускается.
* Ядро оптимизировано для обеспечения реального времени Klipper.
* Оптимизация файловой системы снижает риск неработоспособности системы из-за внезапного отключения питания.
* Быстрый старт, запуск Klipperscreen за 30 секунд.
* Предустановлены службы Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen и другие.
* Поддержка переключения между системами Klipper и RRF.
* Поддержка онлайн-переключения между Mainsail и Fluidd.
* Поддержка печати с USB-флешки.
* Поддержка продолжения печати после отключения питания.
* Поддержка экранов на основе последовательного порта.
* Поддержка автоматической прошивки firmware при запуске (fly-c8, fly-geminiv3).
* Поддержка предварительной установки языка по умолчанию для интерфейсов UI, таких как Fluidd, а также имени принтера по умолчанию.
* Для получения дополнительных функций OEM-настройки, таких как логотип при запуске, предустановленные конфигурационные файлы, скрытие вкладок Mainsail, пожалуйста, свяжитесь с нами.

## Назначение имен файлов образов системы

* Правило именования файлов системы: `FlyOS-Fast-[серия_чипов]-[номер_версии].img`, например: `FlyOS-Fast-H5-v1.0.0.img`.
* Исходный файл имеет формат `.img`, в ссылках на загрузку предоставляются только сжатые файлы формата `.exe` и `.7z` для удобства загрузки.
* Для систем на базе Windows скачивайте файл `.exe`, который является сжатым архивом 7z, после скачивания дважды щелкните по нему, чтобы разархивировать и получить файл `.img` (можно распаковать только на системах Windows).
* Для других систем, таких как MacOS и Linux, скачивайте файл `.7z`, который является сжатым архивом 7z, после скачивания используйте инструмент, поддерживающий формат 7z, для распаковки и получения файла `.img`.
* Из-за большого размера файлов системы временно не предоставляются другие форматы сжатия и исходные образы.

:::tip Выбор системы

* Серии fly-pi, fly-c8, fly-gemini и другие, все продукты Fly, использующие чипы Allwinner H5, выбирайте `FlyOS-Fast-H5.exe` или `FlyOS-Fast-H5.7z`.
* Серии fly-pi-lite2, fly-miniPad и другие, все продукты Fly, использующие чипы Allwinner H3, выбирайте `FlyOS-Fast-H3.exe` или `FlyOS-Fast-H3.7z`.
* Модели чипов отличаются, поэтому и файлы системы различны. Пожалуйста, выбирайте в соответствии с вашими потребностями.

:::

## Скачивание образов системы

* Официальный адрес для скачивания FlyOS-Fast: [Библиотека FlyOS-Fast](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/packages)

- Имя пользователя по умолчанию: `root`
- Пароль по умолчанию: `mellow`

:::tip Примечание

Эта система является облегченной пользовательской системой, компоненты и службы, такие как Klipper, встроенные в нее, не являются официальными версиями и не поддерживают самостоятельную модификацию и обновление пользователей.

:::

## Процесс скачивания образа системы

* Процесс скачивания образа FlyOS-Fast:

    1. Откройте браузер и перейдите на вышеупомянутую библиотеку FlyOS-Fast.

    2. Выберите нужный формат сжатия и перейдите на страницу файла.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down1.webp').default} size="50%" align="center" />

    3. Выберите версию (по умолчанию последняя версия).

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down2.webp').default} size="50%" align="center" />

    4. Если кнопка загрузки не видна, переместите курсор в область списка файлов, удерживайте клавишу `Shift` и прокрутите колесико мыши, чтобы прокручивать список файлов слева направо.
    5. Также можно переместить курсор на горизонтальный ползунок внизу и, удерживая левую кнопку мыши, потянуть его вправо, чтобы увидеть кнопку загрузки.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down3.webp').default} size="50%" align="center" />

    6. Нажмите кнопку загрузки, чтобы скачать соответствующий файл системы.
