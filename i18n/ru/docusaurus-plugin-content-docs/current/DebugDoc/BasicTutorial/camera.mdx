# Добавление USB-камеры


:::danger Важно

* Требуется установка службы `crowsnest`, если она не установлена в системах lite2 и mini pad.
* Требуется камера без драйверов, которая поддерживается plug-and-play, такая как в системах Linux, Armbian и т. д., для klipper, драйверы не требуются.
* В данном руководстве используется последняя версия klipper, crowsnest и fuidd; для других методов требуется самостоятельное изучение.

:::

## Открытие конфигурации

:::tip

* ID камеры может отличаться в зависимости от системы, важно определить стабильный ID.

:::

    * В Fuidd найдите файл конфигурации и откройте `crowsnest.conf`
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf.webp').default} size="100%" align="center" />

    * Найдите `device: ` и удалите настроенный ID
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf1.webp').default} size="100%" align="center" />

## Поиск ID камеры

:::tip

* Убедитесь, что камера подключена к хост-компьютеру.

:::

    * Нажмите на верхнюю кнопку `DEVICES`, затем `VIDEO` и наконец обновите
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf2.webp').default} size="100%" align="center" />
    * Найдите `path_by_id` и скопируйте следующую строку
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf3.webp').default} size="100%" align="center" />

## Добавление ID

    * Добавьте ID и сохраните конфигурацию
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf4.webp').default} size="100%" align="center" />

## Активация камеры

    * Перейдите в настройки, выберите камеру и нажмите "Добавить камеру"
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf5.webp').default} size="100%" align="center" />
    * Введите желаемое имя и нажмите "Сохранить"
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf6.webp').default} size="100%" align="center" />

## Проверка активации

    * Нажмите на панель инструментов, где должна появиться картинка с камеры
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/cam/conf7.webp').default} size="100%" align="center" />
    * Если изображение не отображается, перезагрузите систему
