# Через SSH подключиться к сети


* Убедитесь, что система уже запущена и подключена через SSH к хост-компьютеру, рекомендуется использовать **MobaXterm_Personal**
* Убедитесь, что на хост-компьютере установлены **M2WE** или **2.4G_WiFi модуль**, если это **Lite2** или **MiniPad**, то установка модуля не требуется
* Обратите внимание, что **2.4G_WiFi модуль**, **Lite2** и **MiniPad** могут использовать только 2.4G Wi-Fi и не поддерживают 5G Wi-Fi
* Обратите внимание, что сеть не может использовать объединение диапазонов, иначе подключение будет невозможным
* Убедитесь, что антенна установлена!!!

 1. Введите команду **nmtui** в программе SSH и нажмите Enter, появится интерфейс, как показано на рисунке ниже

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. Нажмите Enter, затем выберите **Activate a connection** и нажмите Enter

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. Выберите нужную сеть для подключения, нажмите Enter, введите пароль WiFi, а затем нажмите OK для сохранения и подключения

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. После успешного подключения к Wi-Fi нажмите несколько раз клавишу Esc на клавиатуре, чтобы выйти из текущего интерфейса, затем введите команду **ip a** и нажмите Enter

 * В изображении IP-адрес хост-компьютера — 192.168.2.136

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
