# Подключение к SSH на главном компьютере

:::warning

* Данная документация применима только для главного компьютера FLY, остальные устройства предоставляют только справочные данные.
* Если вы занимаетесь компиляцией и записью прошивки, обратитесь к руководству по сетевому подключению, а не к руководству по последовательному подключению.

:::

* Вы можете подключиться к устройству через последовательный порт или Wi-Fi.
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другой инструмент для SSH.
* Убедитесь, что карта SD или M2WE прошиты **соответствующей прошивкой для главного компьютера**.
* **Если главный компьютер не запускается корректно, отключите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**
* **Всегда подключайтесь к SSH через сеть для удобства компиляции прошивки!**
* **Для компиляции прошивки см. руководство по подключению к главному компьютеру через Wi-Fi или кабель.**

        ## Подключение к SSH главного компьютера через IP сети

        * Убедитесь, что система уже **нормально запущена**.
        * Пожалуйста, войдите в веб-интерфейс маршрутизатора, если вы не знаете IP адрес главного компьютера.
        * Убедитесь, что вы **знаете имя пользователя и пароль главного компьютера**. По умолчанию пользователь FLY — **fly**, пароль — **mellow**.
        * Если материнская плата уже запущена и подключена к сети, используйте приведённый ниже метод для подключения к SSH.
        * Откройте MobaXterm и выполните следующие настройки, указав IP адрес, который вы видели ранее (его можно также найти в веб-интерфейсе маршрутизатора).

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: На шаге 4 введите имя пользователя главного компьютера.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится окно предупреждения, нажмите `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Введите пароль, после чего нажмите Enter.

        **Внимание: Вводимый пароль не отображается, просто нажмите Enter после ввода.**

        По умолчанию пользователь FLY — **fly**, пароль — **mellow**.
