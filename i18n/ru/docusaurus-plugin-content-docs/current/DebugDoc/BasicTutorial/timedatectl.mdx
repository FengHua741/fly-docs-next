# Изменение часового пояса системы

    Обратите внимание, что если FLY ПК использует версию FLY OS 3.3.4, можно напрямую использовать FLY-Tools для изменения часового пояса.
    Для других ПК см. ниже инструкции.


    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    Замените `<TIMEZONE>` на код часового пояса, который вы хотите использовать. Вы можете найти коды часовых поясов на веб-сайте [iana.org](https://www.iana.org/time-zones). Например, если вы хотите изменить системный часовой пояс на Пекинское время (Asia/Shanghai), введите следующую команду:

    ```bash
    sudo timedatectl set-timezone Asia/Shanghai
    ```

    Теперь ваш системный часовой пояс успешно изменен на Шанхайское время. Вы можете проверить настройки часового пояса с помощью следующей команды:

```bash
date
```
