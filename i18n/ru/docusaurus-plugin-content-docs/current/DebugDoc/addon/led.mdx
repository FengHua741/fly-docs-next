# Расширение управления светодиодами через плагины

Оригинальный документ

```cfg
https://github.com/julianschill/klipper-led_effect
```

## Установка

Пожалуйста, используйте инструмент SSH, такой как **MobaXterm_Personal**, подключитесь по **Wi-Fi** к вашему хост-компьютеру и убедитесь в следующем:

1. **Убедитесь, что на хост-компьютере установлены фиксации Klipper**
2. **Убедитесь, что пользователь, с которым вы вошли, является пользователем, установившим Klipper**
3. **Убедитесь, что ваша раскладка клавиатуры в английском языке**
4. **Убедитесь, что ваш хост-компьютер может нормально находить устройство**
5. **Убедитесь, что все вышеупомянутые注意事项 выполнены, иначе дальнейшие шаги невозможны**

Этот модуль можно установить в существующую установку Klipper с помощью установочного скрипта.

```cfg
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Пример конфигурации

```cfg
[neopixel TP_led]
pin: PB7
chain_count: 25
# Количество светодиодов
# Количество лампочек
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
