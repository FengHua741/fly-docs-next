:::info
Разделено на два маленьких шага

* Первый шаг
* Проверьте, является ли макрос возврата домой `safe_z_home`. Если он уже настроен, продолжайте дальше.
```cfg
[safe_z_home]                
home_xy_position:150,150     # Координаты центра по xy (вычисляется путем деления максимального значения печатного стола пополам; можно скорректировать самостоятельно)
speed:300                    # Скорость возврата домой
z_hop:10                     # Высота подъема перед возвратом домой
```

* Через консоль опустите головку Z до соприкосновения с печатным столом и продолжайте пошаговые инструкции

### Шаг 2: Температурная компенсация
:::info
* Перед проведением температурной компенсации не нагревайте печатный стол и сопло
:::
* Выполните макрос, показанный на следующих изображениях, `TEMP_COMPENSATION`, и нажмите на него для запуска
<ImageView image={require('@site/docs/DebugDoc/faq/img/temp.webp').default} size="50%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy6.webp').default} size="50%" align="left" />

1. После выполнения данной операции на интерфейсе появится окно корректировки Z-оси. Используйте вышеупомянутый ручной метод [Калибровки смещения Z (Метод проверки бумаги)](https://www.klipper3d.org/Bed_Level.html#the-paper-test), чтобы поместить бумагу между соплом и поверхностью печатного стола, затем подтвердите это значение.
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
2. После принятия значения автоматически установятся температуры: печатный стол — 80°C, сопло — 250°C.
3. Если вы находитесь в комнате с кондиционером или开着 окна, для обеспечения стабильного повышения температуры Eddy рекомендуется отключить кондиционер или закрыть окна, так как поток воздуха может повлиять на процесс нагрева.
4. По мере повышения температуры Eddy система автоматически будет предлагать вам каждые 3°C выполнить ручную калибровку смещения Z. Температура печатного стола высока, будьте осторожны!
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
5. Повторяйте ручную калибровку смещения Z (Метод проверки бумаги) до завершения. Если температура Eddy перестанет повышаться, используйте следующие команды G-кода для преждевременного завершения калибровки.
   
* Доступные дополнительные команды G-кода во время дрейфовой калибровки включают:
   * `TEMPERATURE_PROBE_NEXT` — используется для принудительной выборки новых данных EDDy до достижения целевой температуры `80°C`.
   * `TEMPERATURE_PROBE_COMPLETE` — используется для завершения калибровки до достижения заданной температуры `80°C`.
   * `ABORT` — используется для прерывания калибровки и игнорирования результатов.
   * После завершения калибровки используйте `SAVE_CONFIG`, чтобы сохранить настройки смещения температуры!
   * Как уже упоминалось выше, этот процесс калибровки более сложный и требует больше времени, чем большинство других программ. Возможно, потребуется практика и несколько попыток для достижения отличных результатов при печати первого слоя в широком диапазоне температур!

## Как откалибровать z-offset
* Первый шаг: используйте slicer Pursa/OrcaSlicer/cura/Bambu Studio, чтобы напечатать прямоугольник размером 200x200 мм с высотой 0,2 мм для микроКалибровки z-offset
* Второй шаг: через IP принтера войдите в `fluidd` или `mainsail`, найдите панель управления "Движение", а затем отрегулируйте смещение z-axis, чтобы настроить z_OFFSET
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* После завершения калибровки нажмите `SAVE_CONFIG`, чтобы сохранить параметры

* Найдите соответствующие файлы по изображению и измените их

* После завершения калибровки нажмите `SAVE_CONFIG`, чтобы сохранить параметры

:::
