:::info
* Для принтеров с функциями `z_tilt` или `quad_gantry_level (QGL)` не забудьте выполнить многоуровневую настройку Z.
:::

### Шаг 2: Температурная компенсация
:::info
* Перед проведением температурной компенсации не нагревайте нагревательное ложе и сопло.
:::

* Выполните следующую макрокоманду `TEMP_COMPENSATION`, как показано на изображении, и нажмите "Выполнить"
<ImageView image={require('@site/docs/DebugDoc/faq/img/temp.webp').default} size="50%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy6.webp').default} size="50%" align="left" />

1. После выполнения этой операции пользовательский интерфейс отобразит диалоговое окно для настройки Z-оси. Используйте вышеупомянутый ручной метод калибровки Z-offset (Paper Test), вставив бумагу между соплом и рабочей поверхностью стола, затем подтвердите это значение.
2. После принятия значения установите температуру нагревательного ложа на максимум, а температуру сопла на 250°C.
3. Если вы находитесь в комнате с кондиционером или открытыми окнами, рекомендуется закрыть кондиционер или окна, так как ветер может повлиять на повышение температуры.
4. По мере повышения температуры Eddy система автоматически запросит у вас выполнить ручную калибровку Z-offset каждые 3°C. Будьте осторожны, так как нагревательное ложе имеет высокую температуру!
5. Повторите ручную калибровку Z-offset (Paper Test) до завершения калибровки. Если температура Eddy перестанет повышаться, вы можете использовать следующие команды для досрочного завершения калибровки.

* В дополнение к основным командам калибровки во время дрейфа доступны следующие команды G-кода:
* `TEMPERATURE_PROBE_NEXT` используется для принудительного сбора новых данных Eddy перед достижением целевой температуры Eddy `80°C`.
* `TEMPERATURE_PROBE_COMPLETE` используется для завершения калибровки до достижения целевой температуры Eddy `80°C`.
* `ABORT` используется для прекращения калибровки и игнорирования результатов.
* При завершении калибровки используйте команду `SAVE_CONFIG` для сохранения смещений температуры.
* Как было указано выше, этот процесс калибровки более сложный и трудоемкий по сравнению с большинством других программ. Возможно, потребуется практика и несколько попыток, чтобы достичь отличных результатов при печати первого слоя в широком диапазоне температур!

## Настройка z-offset/z-компенсация
* Используйте следующую команду для настройки Z-offset с использованием листа формата А4. Перед настройкой значения `z-offset` ознакомьтесь с руководством по настройке `paper test` в документации Klipper по ссылке [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)

G28
PROBE_CALIBRATE METHOD=AUTO

## Как отрегулировать z-offset с точностью
* Шаг 1: Используйте Purusa/OrcaSlicer/Cura/Bambu Studio, чтобы нарезать прямоугольник размером 200x200 мм с высотой 0,2 мм для печати и настройки z-offset
* Шаг 2: Через IP-адрес принтера войдите в `fluidd` или `mainsail`, найдите панель управления `управление движением` и настройте смещение Z с помощью Z-axis offset. Запомните значение настройки z_OFFSET
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* Шаг 3: Перезапустите Klipper с помощью следующей команды
```bash
FIRMWARE_RESTART
```
* Шаг 4: Используйте следующую команду для настройки Z-offset, как показано в интерфейсе пользователя. Значение, на которое вы настроили Z-offset, должно быть таким же, как и при печати, используя метод `paper test` [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
```bash
G28
PROBE_CALIBRATE METHOD=AUTO  
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* После завершения калибровки сохраните параметры с помощью команды `SAVE_CONFIG`

* Если после выполнения всех вышеуказанных действий сканирование стола не прошло успешно, измените z-лимит на виртуальный, заменив исходное значение endstop_pin на `endstop_pin: probe:z_virtual_endstop` 
:::
