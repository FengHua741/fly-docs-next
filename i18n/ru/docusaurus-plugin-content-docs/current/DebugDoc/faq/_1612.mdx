## Инструкции по эксплуатации

    :::warning Пункты, требующие внимания

    * Обратите внимание, что сначала необходимо выполнить калибровку тока, а затем калибровку частоты катушки.
    * Машина с несколькими осями Z требует одного ручного выравнивания. После завершения всех калибровок выполните еще одно выравнивание и затем снова проведите калибровку.
    * Прошивка нижнего уровня должна полностью соответствовать версии Kliiper.

    * Используйте веб-страницу `FUIDD`, чтобы добавить нижеследующую конфигурацию, а затем выполнить последующие операции.
        ```cfg
        [force_move]
        enable_force_move: true
        ```

    ### Второй шаг: Калибровка тока EDDY

        * 1: Введите в консоль следующую команду.
            ```bash
            SET_KINEMATIC_POSITION z=80
            ```
        * 2: Верните X и Y, переместите головку инструмента в центр нагревательной платформы.
            ```bash
            G28 X Y
            ```
        * После завершения вышеуказанных шагов требуется калибровка тока Eddy. Сначала переместите Eddy в центр нагревательной платформы, а затем соприкосните сопло с платформой. Затем в консоли выполните следующую команду для автоматического получения тока, и наконец, выполните `SAVE_CONFIG`, чтобы сохранить параметры. Нормальное значение обратной связи должно быть около `15`. Если обратная связь равна `0` или `30`, проверьте, правильно ли подключено (плохой контакт).
            ```bash
            LDC_CALIBRATE_DRIVE_CURRENT CHIP=fly_eddy_probe
            ```
        * Нормальное возвращаемое значение должно быть примерно `15`.
        * После завершения калибровки выполните `SAVE_CONFIG`, чтобы сохранить параметры.

    ### Второй шаг: Калибровка частоты Eddy


        :::info

        * При установке Eddy плата eddy ocb должна быть выше сопла не менее чем на 1 - 2 мм.
        * При калибровке частоты Eddy плата катушки должна быть удалена от нагревательной платформы не менее чем на 5 мм.
        * Отдаление на 5 мм здесь необходимо для удобства проведения следующего шага - получения `z_offset` с помощью бумаги.

        :::

        * Затем переместите сопло в центр платформы (обратите внимание: на этом шаге убедитесь, что машина не загружена heightmap).
        * Начните ручную калибровку z-отклонения [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test).
            ```bash
            PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
            ```
        * После завершения калибровки выполните `SAVE_CONFIG`, чтобы сохранить параметры.
        :::info
        * Для принтера с функцией `z_tilt` или `quad_gantry_level (QGL)`, сначала отрегулируйте параллельность. Для `i3` отрегулируйте параллельность оси X, для三叉戟 отрегулируйте параллельность нагревательной платформы, для voron 2.4 отрегулируйте параллельность портала, а затем снова проведите калибровку частоты Eddy.
        :::
    ### Третий шаг: Компенсация температуры
        :::info
        * При выполнении компенсации температуры не нагревайте `нагревательную платформу, сопло`.
        :::
        
        * Третий шаг: Компенсация температуры должна быть выполнена успешно, прежде чем начинать.

        * Верните все оси.
       ```bash
       G28
       ```

        * Если есть `[quad_gantry_level]` или `[z_tilt]`, сначала выполните `Z_TILT_ADJUST` или `QUAD_GANTRY_LEVEL` один раз, а затем снова верните оси.
        * Введите в консоль G0 Z5 или с помощью веб-страницы и других способов переместите ось Z на `5 мм` над нагревательной платформой.
        * Выполните следующий код, чтобы увеличить время бездействия машины, чтобы избежать timeout во время процесса нагрева.
            ```bash
            SET_IDLE_TIMEOUT TIMEOUT=65000
            ```
        * Выполните следующую команду для начала калибровки.
            ```bash
            TEMPERATURE_PROBE_CALIBRATE PROBE=fly_eddy_probe TARGET=56 STEP=4
            ```
        :::warning Пункты, требующие внимания

        * Примечание: В вышеуказанной команде целевая температура EDDy установлена на `56°C`, что подходит для большинства устройств.
        * Если фактическая температура камеры вашего 3D-принтера выше, вы можете соответствующим образом повысить целевую температуру.
        * Обратите внимание, что чем выше целевая температура, тем дольше потребуется времени для достижения температурного баланса Eddy, и соответственно, дольше будет процесс калибровки.

        :::
    
        1. После выполнения этой операции на UI будет отображено окно для корректировки оси Z. Используйте вышеупомянутую ручную [метод калибровки отклонения оси Z (Paper Test)](https://www.klipper3d.org/Bed_Level.html#the-paper-test), вставьте бумагу между соплом и платформой, а затем подтвердите значение.
        2. После принятия значения установите температуру нагревательной платформы на максимальную, а температуру сопла - на 220°C.
        3. Если вы находитесь в комнате с кондиционером или открытым окном, для обеспечения повышения температуры Eddy рекомендуется выключить кондиционер или закрыть окно, так как ветер влияет на повышение температуры.
        4. По мере повышения температуры Eddy система автоматически будет предлагать вам выполнять ручную калибровку отклонения оси Z каждые 4°C. Температура нагревательной платформы очень высокая, будьте осторожны, чтобы не обжечься!!!
        5. Повторяйте ручную калибровку отклонения оси Z (Paper Test) до завершения калибровки. Если обнаружено, что температура Eddy больше не повышается, вы можете использовать следующие соответствующие команды для досрочного завершения калибровки.
        
        * Дополнительные команды gcode, доступные во время калибровки дрейфа, включают: TEMPERATURE_PROBE_NEXT
            * `TEMPERATURE_PROBE_NEXT` используется для принудительного сбора новых данных EDDy до достижения целевой температуры `56°C`, установленной для EDDy.
            * `TEMPERATURE_PROBE_COMPLETE` используется для завершения калибровки EDDy до достижения заданной температуры `56°C`.
            * `ABORT` может быть использован для прекращения калибровки и игнорирования результатов.
        * После завершения этих шагов Eddy сможет обеспечить превосходный эффект печати первого слоя в широком диапазоне температур!


## Как тонко настроить z - offset
* Первый шаг: Используйте pursa / OrcaSlicer / cura / Bambu Studio, чтобы вырезать квадрат размером 200x200 и высотой 0,2 для печати и тонкой настройки z - offset.
* Второй шаг: Войдите в `fluidd` `mainsail` через IP-адрес принтера, найдите панель управления `движение` и z- ось смещения, продолжайте тонкую настройку z_OFFSET и запомните значение тонкой настройки.
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* Третий шаг: Перезапустите klippper с помощью следующей команды.
```bash
FIRMWARE_RESTART
```
* Четвертый шаг: Отрегулируйте Z - offset с помощью следующей команды. На появившемся интерфейсе UI отрегулируйте `z - offset`. Значение, которое вы отрегулируете, - это значение, которое вы отрегулировали во время печати.
```bash
G28
PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />

:::info
* Если возникает ошибка `Eddy current sensor error`, увеличьте значение z_offset в конфигурации eddy. Посмотрите на иллюстрацию.

<ImageView image={require('@site/docs/DebugDoc/faq/img/EDDYZOFF.webp').default} size="50%" align="left" />
:::

:::info
 * Если после завершения всех вышеперечисленных операций начальная калибровка поверхности не удалась, измените ограничение оси Z на виртуальное ограничение.
:::
