:::info
* Для принтеров с функциями `z_tilt` или `quad_gantry_level (QGL)` не забудьте выполнить многоуровневую настройку Z.
:::

### Первый этап калибровки EDDY
* Нажмите на макрокоманду `CALIBRATE_EDDY`, как показано на изображении, и выполните её
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy5.webp').default} size="50%" align="left" />
* После этого появится следующий интерфейс пользователя
* Используйте лист формата A4 для настройки Z-offset, используя появившийся интерфейс пользователя. Перед изменением значения `z-offset` ознакомьтесь с руководством по инструкциям в документации Klipper по методу "Проверка бумаги" [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
* После выбора подходящего значения нажмите `ACCEPT`
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* После завершения калибровки сохраните параметры с помощью команды `SAVE_CONFIG`   

### Второй этап: температурная компенсация
:::info
* Перед выполнением температурной компенсации не нагревайте нагревательный стол и сопло.
:::

* Выполните макрокоманду `TEMP_COMPENSATION`, как показано на изображении, и нажмите выполнить
<ImageView image={require('@site/docs/DebugDoc/faq/img/temp.webp').default} size="50%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy6.webp').default} size="50%" align="left" />

1. После выполнения этой операции на экране появится окно для настройки Z-оси. Используйте вышеупомянутый ручной метод калибровки Z-смещения (Проверка бумаги) [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test), чтобы проверить значение, вставив лист бумаги между соплом и рабочей поверхностью, а затем подтвердите это значение.
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
2. После подтверждения значения автоматически установите температуру нагревательного стола до 80°C, а сопла до 250°C.
3. Если вы находитесь в комнате с кондиционером или открытыми окнами, рекомендуется закрыть кондиционер или окна, так как ветер может влиять на повышение температуры.
4. По мере увеличения температуры Eddy система будет автоматически предлагать вам выполнять ручную калибровку Z-смещения каждые 3°C. Будьте осторожны, так как температура нагревательного стола очень высокая!
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="30%" align="left" />
5. Повторяйте ручную калибровку Z-смещения (Проверка бумаги) до завершения калибровки. Если вы заметили, что температура Eddy больше не повышается, вы можете использовать приведенные ниже команды для завершения калибровки заранее.

* Доступные дополнительные команды G-кода во время калибровки включают: `TEMPERATURE_PROBE_NEXT`
* `TEMPERATURE_PROBE_NEXT` используется для принудительной выборки новых данных EDDY перед достижением целевой температуры `80°C`.
* `TEMPERATURE_PROBE_COMPLETE` используется для завершения калибровки до достижения целевой температуры `80°C`.
* `ABORT` используется для прекращения калибровки и игнорирования результатов.
* При завершении калибровки используйте `SAVE_CONFIG` для сохранения настроек смещения температуры!
* Как уже упоминалось, вышеописанный процесс калибровки более сложен и требует больше времени, чем большинство других процедур. Возможно, потребуется практика и несколько попыток, чтобы достичь отличных результатов при печати первого слоя на широком диапазоне температур!

## Как настроить z-offset
* Шаг 1: Используйте Slic3r/Pursa/OrcaSlicer/Cura/Bambu Studio для создания файла STL размером 200x200 мм с высотой 0,2 мм для калибровки z-offset.
* Шаг 2: Через IP принтера войдите в `fluidd` или `mainsail`, найдите панель управления и настройте смещение Z-оси с помощью микроподстройки z_OFFSET
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* После завершения калибровки сохраните параметры с помощью команды `SAVE_CONFIG`  

## Команды для изменения Z-offset
* Используйте следующие команды для изменения значения Z-offset, как показано на появившемся интерфейсе пользователя. Значение, которое вы изменяете, будет использоваться в процессе печати. [Paper test](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
```bash
G28
PROBE_CALIBRATE METHOD=AUTO  
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />
* После завершения калибровки сохраните параметры с помощью команды `SAVE_CONFIG`

:::info
* Все указанные выше действия выполнены, но если сканирование стола не прошло успешно, измените ограничение Z на виртуальное, заменив исходное значение endstop_pin на `endstop_pin: probe:z_virtual_endstop`
:::
