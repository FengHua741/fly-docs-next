    * Обратите внимание, что необходимо сначала выполнить калибровку тока, а затем калибровку частоты катушки
    * Машины с несколькими осями Z требуют ручной регулировки один раз после выполнения всех калибровок, затем повторной регулировки и калибровки
    * Версия прошивки нижнего уровня должна полностью соответствовать версии Kliiper

    ### Шаг 2: Калибровка тока EDDY

        * 1: Введите следующую команду в консоль
            ```bash
            SET_KINEMATIC_POSITION z=80
            ```
        * 2: Верните X и Y, переместите головку инструмента в центр стола
            ```bash
            G28 X Y
            ```
        * После завершения вышеуказанных шагов, необходимо калибровать ток Eddy. Сначала переместите Eddy в центр стола, а затем сопло должно едва касаться стола. Затем выполните следующую команду в консоли для автоматического получения тока, а затем `SAVE_CONFIG` для сохранения параметров. Нормальное значение обратной связи составляет около 15, если обратная связь равна 0 или 30, проверьте подключение (плохой контакт)
            ```bash
            LDC_CALIBRATE_DRIVE_CURRENT CHIP=fly_eddy_probe
            ```
        * Нормальное возвращаемое значение должно быть около `15`
        * После завершения калибровки сохраните параметры с помощью `SAVE_CONFIG`

    ### Шаг 2: Калибровка частоты Eddy


        :::info

        * При установке Eddy, плата Eddy ocb должна быть выше сопла на 1-2 мм
        * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от стола
        * Здесь расстояние в 5 мм необходимо для удобства следующего шага по извлечению бумаги для получения `z_offset`

        :::

        * Затем переместите экструдер в центр платформы (обратите внимание: на этом этапе убедитесь, что машина не загружает heightmap)
        * Начните ручную калибровку смещения Z [Тест бумаги](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
            ```bash
            PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
            ```
        * После завершения калибровки сохраните параметры с помощью `SAVE_CONFIG`
        :::info
        * Принтеры с функцией `z_tilt` или `quad_gantry_level（QGL）`, сначала отрегулируйте параллельность `i3 отрегулируйте параллельность оси X, трезубец отрегулируйте параллельность стола, Voron 2.4 отрегулируйте параллельность портала, а затем еще раз откалибруйте частоту Eddy
        :::
    ### Шаг 3: Компенсация температуры
        * Шаг 3: Компенсация температуры должна начинаться после успешного выполнения всех шагов

        * Верните все оси
       ```bash
       G28
       ```

        * Если у вас есть `[quad_gantry_level]` или `[z_tilt]`, сначала выполните `Z_TILT_ADJUST` или `QUAD_GANTRY_LEVEL`, а затем верните оси еще раз
        * Введите в консоль G0 Z5 или через веб-интерфейс переместите ось Z на 5 мм над столом
        * Выполните следующий код, чтобы установить `idle timeout` для машины на более длительный период, чтобы избежать тайм-аута во время нагрева
            ```bash
            SET_IDLE_TIMEOUT TIMEOUT=65000
            ```
        * Выполните следующую команду для начала калибровки
            ```bash
            TEMPERATURE_PROBE_CALIBRATE PROBE=fly_eddy_probe TARGET=56 STEP=4
            ```
        :::warning Важная информация

        * Подсказка: В вышеупомянутой команде целевая температура EDDy установлена на `56℃`, что подходит для большинства устройств.
        * Если фактическая температура камеры вашего 3D-принтера выше, вы можете соответствующим образом повысить целевую температуру.
        * Обратите внимание, что чем выше целевая температура, тем больше времени потребуется на калибровку, так как Eddy потребуется больше времени для достижения температурного равновесия.

        :::
    
        1. После выполнения этой операции, UI покажет окно для настройки Z. Пожалуйста, используйте упомянутый выше метод ручной калибровки смещения Z (тест с бумагой), чтобы зажать бумагу между соплом и столом, а затем подтвердите значение.
        2. После принятия значения, установите температуру стола на максимальную, а температуру сопла на 220℃.
        3. Если вы находитесь в помещении с кондиционером или открытыми окнами, рекомендуется выключить кондиционер или закрыть окна, чтобы обеспечить подъем температуры Eddy, так как ветер может влиять на повышение температуры.
        4. По мере повышения температуры Eddy, система будет автоматически предлагать вам каждые 4℃ выполнять ручную калибровку смещения Z. Температура стола очень высока, будьте осторожны, чтобы не обжечься!!!
        5. Повторяйте ручную калибровку смещения Z (тест с бумагой) до завершения калибровки. Если вы заметили, что температура Eddy больше не повышается, вы можете использовать следующие команды для досрочного завершения калибровки.
        
        * Дополнительные команды gcode, доступные во время калибровки дрейфа, включают: TEMPERATURE_PROBE_NEXT
            * `TEMPERATURE_PROBE_NEXT`    Используется для принудительной выборки новых данных Eddy до достижения установленной целевой температуры `56℃`.
            * `TEMPERATURE_PROBE_COMPLETE`    Используется для завершения калибровки, если Eddy не может достичь установленной температуры `56℃`.
            * `ABORT`  Может использоваться для прерывания калибровки и игнорирования результатов.
        * После выполнения этих шагов, Eddy сможет обеспечить отличное качество первой слоя печати в широком диапазоне температур!

## Как настроить z—offset
* Шаг 1: Используйте purusa/OrcaSlicer/cura/Bambu Studio, чтобы нарезать квадрат 200x200 мм высотой 0.2 мм для настройки z—offset
* Шаг 2: Через IP-адрес принтера войдите в `fluidd` `mainsail`, найдите панель управления, выберите `Движение`, введите z—offset и продолжайте настройку, запомните настроенное значение
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* Шаг 3: Используйте следующую команду для перезапуска klippper
```bash
FIRMWARE_RESTART
```
* Шаг 4: Используйте следующую команду для настройки z—offset, в открывшемся UI интерфейсе настройте `z—offset` на значение, которое было настроено во время печати
```bash
G28
PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />

:::info
* Если появляется ошибка `Eddy current sensor error`, увеличьте значение z_offset в конфигурации eddy, смотрите пример

<ImageView image={require('@site/docs/DebugDoc/faq/img/EDDYZOFF.webp').default} size="50%" align="left" />
:::

 * Если после выполнения вышеуказанных действий происходит сбой при сканировании кровати, измените z-ограничение на виртуальное ограничение
:::
