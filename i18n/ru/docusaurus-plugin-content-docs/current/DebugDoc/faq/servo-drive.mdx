# Инструкция по использованию внешних драйверов

## Важные замечания

:::danger Важно

* Данная инструкция основана на Klipper для управления
* Подключение модуля осуществляется через анодную схему подключения

:::

## Настройка перемычек драйвера

    * Никакие перемычки для драйвера не требуются!!!

## Встроенные переключатели драйвера
    * Рекомендуется использовать значение `3200` для встроенных переключателей драйвера
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/3200.webp').default} size="80%" align="center" />

## Пример конфигурации Klipper

    * Необходимо отключить соответствующую конфигурацию драйвера, например `[tmc5160 stepper_x]` или `[tmc2209 stepper_x]`
    * Установите значение `microsteps:` равным `16`
    * Добавьте параметр `step_pulse_duration: 0.000004`
    * Включение двигателя: удалите символ `!` в параметре `enable_pin:!`
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/cfg.webp').default} size="50%" align="center" />

## Информация о импульсах драйвера

    :::info

    * Возможны смещения при печати модели
    * Мотор может вибрировать, издавать посторонние звуки, возникать разнообразные проблемы
    * Можно изменить значение `step_pulse_duration: 0.000004`
    * Увеличить это значение, например, до `0.000009`, `0.00001` или `0.00002` и т.д.

    :::
