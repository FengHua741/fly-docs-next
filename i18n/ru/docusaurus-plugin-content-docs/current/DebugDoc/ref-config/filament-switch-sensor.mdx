# Датчик отсутствия filaments

```cfg title="printer.cfg"
[filament_switch_sensor Мониторинг отсутствия filaments]
pause_on_runout: True
switch_pin: ^PC2
runout_gcode: 
        PAUSE
        M118 Отсутствие filaments, печать приостановлена
insert_gcode:
        M118 Восстановление filaments

```
