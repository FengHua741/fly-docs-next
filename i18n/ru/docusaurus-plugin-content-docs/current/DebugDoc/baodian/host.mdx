# Главная станция

## Как запустить новую главную станцию

  1. Сначала определите модель вашей главной станции — Lite 2 или PI V2 и т.д.
  2. Готовы ли вы к следующему набору инструментов?
        * Программное обеспечение для записи
        * Соответствующий образ системы
        * Читалка карт, TF-карта, тип-C с функцией передачи данных

  Примечания перед записью системы:
        * Убедитесь, что объем карты памяти не менее `16 ГБ` или не более `128 ГБ`, а скорость передачи должна быть `class10` или выше, рекомендуется использовать TF-карты брендов, таких как SanDisk
        * Не обновляйте систему произвольно! Обновление из-за сервера за рубежом может вызвать проблемы
        * Рекомендуется использовать рекомендованное программное обеспечение
        * Пожалуйста, постарайтесь выучить немного знаний о LINUX; я рекомендую <Button variant="contained" disableElevation component="a" rel="noopener" href="https://item.jd.com/12443890.html">«Личные рецепты Бирдэна по Linux»</Button>, основной учебник

  3. Образ системы FLY <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/category/система-образов">Образ системы</Button>
  4. Необходимые инструменты <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">Инструменты для записи</Button>

## После скачивания всех необходимых инструментов и образов системы

  * Разархивируйте образ системы, распакованный файл имеет расширение `.img`
  * Необходимо использовать SDFormatter для форматирования карты памяти
  * Используйте `win32DiskImager` для записи в систему
  * Не рекомендуется использовать `balenaetcher` для записи системы
  * Если используется главная станция FLY, откройте `FlyOS-Env.txt` для настройки
  * После записи системы не рекомендуется форматировать карту!!!

## Запуск системы

  * После записи системы установите карту системы на главную станцию и включите питание
  * В этот момент система уже записана и настроена, используйте тип-C с функцией передачи данных для подключения к компьютеру через последовательный порт
  * Обратите внимание, что при нормальной работе других главных станций отображаются сообщения журнала загрузки, если они не отображаются, нажмите два раза клавишу Enter или повторите запись системы
  * После успешного запуска можно подключиться к Wi-Fi


## Инструкции по использованию M2WE и карты памяти

  * Если установлены M2WE и карта памяти, система по умолчанию будет запускаться с карты памяти
  * Рекомендуется использовать SD-карту для записи системы на M2WE; использование USB для записи может привести к неудаче из-за совместимости драйверов, рекомендуется использовать карту памяти напрямую для записи на M2WE
  * Убедитесь, что загруженный образ системы соответствует требованиям, в противном случае система не запустится
  * Обратите внимание, что Lite2 и MINI PAD могут использовать только карту памяти


## Ежедневные注意事项 при эксплуатации

  * Не отключайте питание напрямую, требуется нормальное выключение системы, иначе могут возникнуть потери данных
  * Так как серверы GitHud находятся за рубежом, не рекомендуется самостоятельно обновлять или устанавливать различные плагины
