# Инструкция по прошивке

    :::danger Важно

    * Лучше всего использовать свой собственный компилятор для сборки прошивки Klipper
    * Использование чужой прошивки или прошивки с другим версиями может привести к различным проблемам
    * Стремитесь избегать использования модифицированных версий прошивки

    :::

## Особые указания

    :::info Особые указания
    * Для большинства материнских плат FLY прошивка `Katapult` уже установлена на заводе
    * Для материнских плат FLY в основном используется прошивка `Katapult` для записи прошивки Klipper
    * Для материнских плат DP5, D5, D7 требуется ручная запись соответствующей прошивки `Katapult`
    * Из-за предыдущих прошивок после записи прошивки `Katapult`, если необходимо искать ID материнской платы, нужно нажать кнопку сброса два раза, иначе запись прошивки невозможна
    :::

## Инструкция по компиляции прошивки

    :::tip

    * Для материнских плат FLY используется загрузчик Katapult с карты памяти
    * Преимуществом прошивки Katapult является возможность обновления прошивки без физического контакта с материнской платой
    * Преимущество загрузчика заключается в том, что не нужно беспокоиться о неправильной прошивке
    * Рекомендуется использовать `MobaXterm` для настройки прошивки, так как некоторые специфические SSH-программы могут вызывать ошибки компиляции
    
    :::

    * Обратите внимание, что независимо от типа прошивки, важно убедиться, что прошивка скомпилирована правильно, чтобы избежать проблем с поиском печати или обнаружения устройства

## Инструкция по компиляции прошивки

    :::tip
    * Необходимо отметить опцию `Enable extra low-level configuration options`, чтобы показывались подробные настройки
    :::

### STM32

    * При компиляции прошивки STM32 обратите внимание на то, что при конфигурации `GPIO pins to set at micro-controller startup` все буквы должны быть заглавными

### RP2040

    * При компиляции прошивки RP2040 обратите внимание на то, что при конфигурации `GPIO pins to set at micro-controller startup` все буквы должны быть строчными

## Инструкция по записи прошивки

    :::danger Важно

    * При записи прошивки Klipper через прошивку `Katapult` необходимо убедиться, что устройство можно найти; если устройство не найдено, нужно нажать кнопку сброса два раза
    * Не используйте методы записи, отличные от указанных в руководстве
    * Проверьте, есть ли ошибки в компиляции прошивки; если есть ошибки, запись прошивки будет невозможна

    :::

### Katapult прошивка

    * Решение проблемы с компиляцией ошибочной прошивки с использованием прошивки Katapult
    * Нажмите кнопку сброса два раза; обычно после этого загорится мигающий светодиод
    * Очистите существующую прошивку материнской платы, затем повторно запишите прошивку Katapult

### Bootloader прошивка

    * Решение проблемы с компиляцией ошибочной прошивки с использованием прошивки bootloader
    * Просто перезапишите прошивку с помощью карты памяти TF
    * Если запись не удалась, отключите питание, установите карту памяти на материнскую плату, затем подайте питание; через пять секунд прошивка будет выполнена
