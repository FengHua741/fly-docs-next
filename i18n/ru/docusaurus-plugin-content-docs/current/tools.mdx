# Полезные инструменты

:::tip

Данный документ содержит различные полезные инструменты, связанные с настройкой Klipper.

:::

:::warning Важно

Некоторые маленькие инструменты, такие как "Парсер прошивки Klipper", являются онлайн-инструментами, в то время как "Fly-Tools" требуют установки перед использованием. Пожалуйста, учитывайте это различие.

:::

:::danger Важно знать

Если во время использования возникнут проблемы, пожалуйста, своевременно создайте Issue.

:::
