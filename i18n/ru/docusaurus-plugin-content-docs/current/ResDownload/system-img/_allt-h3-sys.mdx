# Система {props.name} для скачивания

:::предупреждение

Важно о SD-карте:

* Объем должен быть не менее `16 ГБ` и не более `128 ГБ` TF-карты
* Скорость передачи TF-карты должна быть не ниже `class10` или выше, рекомендуется использовать карты памяти SanDisk и т.д.

:::

:::опасность

* Из-за проблем с серверами не рекомендуется произвольно обновлять систему или klipper
* Пожалуйста, скачивайте систему FLY OS, использование оригинальной версии Armbian требует слишком много усилий и не рекомендуется для установки

:::

:::информация Важное уведомление

* Обратите внимание, что скачанная система требует распаковки, а не просто изменения расширения файла или прямого записи на диск!!
* Без распаковки и записи невозможно запустить систему!!

:::
