# Система {props.name} для скачивания

:::предупреждение

Важно о SD-карте:

* Объем должен быть не менее `16 ГБ` и не более `128 ГБ` TF-карты
* Скорость передачи TF-карты должна быть класса `10` или выше, рекомендуется использовать TF-карты производителя SanDisk

:::

:::опасность Важно!

* Из-за проблем с серверами не рекомендуется самостоятельно обновлять систему или klipper
* Пожалуйста, скачивайте FLYOS-Armbian, использование оригинального Armbian требует большего опыта и не рекомендуется

:::

:::информация Важное уведомление

* Обратите внимание, что скачанная система требует распаковки, а не просто изменения расширения файла или прямого записи на карту памяти!!!
* Если система не будет распакована перед записью, она не запустится!!!

:::
