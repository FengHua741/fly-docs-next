        <TabItem value="H5" label="Журнал обновлений систем PI, PI V2, Gemini, C8">

            ## v2
                * 2022-1
                * Смена версии релиза на Bullseye
                * Обновление ядра до 5.10.85
                * Предустановлены Klipper, Moonraker, Fluidd, Mainsail, KlipperScreen
                * Klipper использует среду Python3 для решения проблемы с именами файлов на китайском языке
                * Изменение раздела boot на fat для возможности редактирования конфигурационных файлов непосредственно в системе Windows
                * Добавлена FLY-Config —全新 система конфигурации
                * Добавлен CatSerial — просмотр всех подключенных последовательных устройств по адресу ```http://ip:9999```

            ## v2.1
                * Исправлена ошибка CatSerial при отсутствии подключенных устройств

            ## v2.2
                * Добавлена прошивка BL для серийных материнских плат Gemini
                * Инструкция по прошивке доступна

            ## v2.4
                * Исправлено отсутствие некоторых окружений для прошивки Klipper
                * Изменены шаблоны конфигурации

                * Добавлены конфигурации пинов для материнских плат V1 и V1.1. Включение через FLY_Config

            ## v2.9
                * Поддержка материнской платы Gemini v2

                * Изменен модуль ядра gs_usb

            ## v2.9.3
                * Совместимость с FLY-Π

                * Установлены правила PolicyKit для Moonraker
                * Исправлены конфликты прав udev
                * Исправлена ошибка с отсутствием библиотеки libusb при попытке прошивки klipper
                * Исправлена проблема с недоступностью статических ресурсов FLY-Tools
                * Изменена модель материнской платы по умолчанию на Gemini v2
                * Изменен параметр serial в printer.cfg на /dev/ttyACM0
                * Изменен размер буфера отправки данных CAN на 1024

            ## v2.9.4
                * Отменена предустановка исходников ядра и заголовочных файлов (в дальнейшем будут распространяться через deb-пакеты)
                * Исправлено неисправимое подключение RTL88XXau

            ## v2.9.5
                * Добавлена управляемость Wi-Fi сетью в KlipperScreen
                * Исправлены другие проблемы

                * Исправлены другие проблемы

                * Исправлено непроизводительное запускание KlipperScreen
