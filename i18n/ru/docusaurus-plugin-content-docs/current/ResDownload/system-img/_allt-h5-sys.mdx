# Система {props.name} для скачивания

:::предупреждение

Важно при работе с картой памяти:

* Объем должен быть не менее `16 ГБ` и не более `128 ГБ`
* Класс скорости карты памяти должен быть не ниже `class10`, рекомендуется использовать карты памяти SanDisk и выше

:::

:::опасность

* Из-за проблем с серверами не рекомендуется обновлять систему или klipper без необходимости
* Рекомендуется скачивать систему FLY OS, установка оригинального Armbian считается сложной и не рекомендуемой

:::

:::информация Важное уведомление

* Обратите внимание, что загруженная система требует распаковки, а не просто изменения расширения файла или прямого копирования на карту памяти
* Без предварительной распаковки система не запустится

:::
