# Система {props.name} для скачивания

:::предупреждение

Важно о SD-карте:

* Емкость должна быть не менее `16 ГБ` и не более `128 ГБ` TF-карты
* Скорость записи TF-карты должна быть класса `10` или выше, рекомендуется использовать карты памяти от SanDisk и других известных брендов

:::

:::опасность Важно знать

* Из-за проблем на сервере не рекомендуется самостоятельно обновлять систему или klipper
* Пожалуйста, скачивайте систему FLYOS-Armbian, использование оригинальной системы Armbian требует большего опыта и не рекомендуется

:::

:::информация Важное примечание

* Обратите внимание, что скачанная система требует распаковки, а не просто изменения расширения файла или прямого копирования на устройство
* Без предварительной распаковки система не запустится

:::
