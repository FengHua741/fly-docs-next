# Изображение Raspberry Pi


  :::предупреждение

  Примечания к SD-карте:

  * Объем должен быть не менее `16 ГБ` или не более `128 ГБ` карты памяти TF
  * Скорость передачи карточки памяти должна быть класса `10` или выше, рекомендуется использовать карты памяти бренда SanDisk и выше

  :::

  :::опасность Важно

  * Из-за проблем с сервером не обновляйте систему или klipper без необходимости
  * Системный образ совместим с Raspberry Pi 3B/3B+/4B/CM4/5B
  * Этот образ системы основан на `bookworm` с установкой различных служб
  * В этом образе по умолчанию установлено и включено TFT-экран, если TFT-экран не требуется, его необходимо вручную отключить через `/boot/config.txt`

  :::

  :::информация Важно

  * Обратите внимание, что загруженная система требует распаковки, а не просто изменения расширения файла или прямого записи! !!
  * Без распаковки невозможно запустить систему!

  :::
