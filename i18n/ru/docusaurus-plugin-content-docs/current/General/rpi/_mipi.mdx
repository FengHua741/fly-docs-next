## Рaspberry Pi 5 с MIPI

:::tip
FLY_MIPI — это переходная плата, специально разработанная для Raspberry Pi 5, которая позволяет использовать DSI и CSI устройства напрямую через переходник.
:::

### Подключение MIPI к Raspberry Pi 5

    :::danger Важно

    * Интерфейсы DSI и CSI на Raspberry Pi 5 теперь называются MIPI.
    * Интерфейс MIPI можно использовать напрямую для подключения CSI-камеры или экрана DSI.
    * Обратите внимание на направление контактов.

    :::

    * На Raspberry Pi 5 теперь два интерфейса MIPI. При взгляде сверху можно увидеть маркировку 1 и 0.


## Подключение экрана DSI

    * Если используется интерфейс 1, необходимо сконфигурировать экран согласно следующему примеру:

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * Если используется интерфейс 0, необходимо сконфигурировать экран согласно следующему примеру:

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## Подключение камеры CSI

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * Обратите внимание, что модель камеры должна быть настроена в соответствии с приобретенной моделью. В данном руководстве используется камера imx219 как пример.

    * Если используется левый интерфейс MIPI, необходимо сконфигурировать камеру следующим образом:
    ```bash
    dtoverlay=imx219,cam1
    ```

    * Если используется правый интерфейс MIPI, необходимо сконфигурировать камеру следующим образом:
    ```bash
    dtoverlay=imx219,cam0
    ```
