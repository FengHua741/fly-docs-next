## Использование CSI

    * После записи образа введите /boot/config.txt и найдите `camera_auto_detect=1`, измените его на `camera_auto_detect=0`, а затем добавьте конфигурацию в соответствии с моделью камеры
    * Ниже приведена примерная конфигурация

    ```bash
    dtoverlay=imx219,cam1
    ```
