### Начало прошивки

    :::предупреждение
       * **Не настройте ПК Fly** обязательно выполните следующую команду для установки зависимостей прошивки! В противном случае прошивка не будет выполнена!
       ```
      cd && git clone https://github.com/Arksine/katapult.git
       ```
      <ImageView image={require('@site/docs/General/dfu-flash/img/get_katapult.webp').default} size="80%" align="center" />
    :::
    * Используйте следующую команду для поиска ID вашего устройства, результат должен совпадать с показанным ниже (примечание: ID на изображении является примером, каждый материнская плата имеет уникальный ID)

        ```bash
        ls /dev/serial/by-id/*
        ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/2040.webp').default} size="80%" align="center" />


    * Для прошивки используйте следующую команду, убедитесь, что прошивка успешно скомпилирована, *замените `<ID_вашей_материнской_платы>` на ID, полученный на предыдущем шаге*

        ```bash
        ~/klippy-env/bin/python ~/katapult/scripts/flashtool.py -d /dev/serial/by-id/<ID_вашей_материнской_платы>
        ```

    * Пример ниже
        <ImageView image={require('@site/docs/General/dfu-flash/img/kaulut7.webp').default} size="100%" align="center" />

    * Результат успешной прошивки

## Обновление прошивки

<Tabs>
    <TabItem value="w" label="Обновление USB-прошивки" default>
        * Используйте следующую команду для поиска ID вашей материнской платы, результат должен совпадать с показанным ниже (примечание: ID на изображении является примером, каждый материнская плата имеет уникальный ID)

        ```bash
        ls /dev/serial/by-id/*
        ```

        :::предупреждение
            На изображении показан ID `/dev/serial/by-id/usb-katapult_rp2040_E662549553642032-if00`
        :::

        <ImageView image={require('@site/docs/General/dfu-flash/img/2040.webp').default} size="80%" align="center" />

        * Для обновления USB-прошивки после компиляции новой прошивки введите следующую команду, *замените `<ID_вашей_материнской_платы>` на ID, полученный на предыдущем шаге*

        ```bash
        cd ~/klipper/ && make flash FLASH_DEVICE=<ID_вашей_материнской_платы>
        ```

        <ImageView image={require('@site/docs/General/dfu-flash/img/katapult5.webp').default} size="100%" align="center" />

## Метод решения ошибочной прошивки

    * Отключите питание материнской платы, затем включите ее и быстро нажмите два раза кнопку сброса
    * Или используйте очистку прошивки в WINDOWS и перепишите BL заново
