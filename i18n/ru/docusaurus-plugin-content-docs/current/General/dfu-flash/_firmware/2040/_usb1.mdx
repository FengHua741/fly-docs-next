Чтобы скомпилировать прошивку, выполните следующие шаги:
* Подключитесь по SSH и введите следующую команду:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Команда `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, а команда `make menuconfig` используется для компиляции прошивки. После выполнения команды должен появиться следующий интерфейс:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Micro-controller Architecture** и выберите **Raspberry Pi RP2040**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, затем выберите: **16KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* В разделе **Communication interface** выберите: **USBSERIAL**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb1.webp').default} size="100%" align="Center" />
