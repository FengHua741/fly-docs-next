* Обратите внимание, что текущая панель инструментов находится в режиме прошивки, и может прошиваться только固件 с расширением uf2.
* Для записи固件 на панель инструментов используйте следующую команду:

        ```bash
        cd && cd ~/klipper/
        make flash FLASH_DEVICE=2e8a:0003
        ```
        :::tip
           Если это первая запись системы после её создания на хост-компьютере, возможно появление подсказок, аналогичных красному кругу на нижеследующем изображении — наличие частично красных сообщений является нормой и не означает ошибку.
        :::
