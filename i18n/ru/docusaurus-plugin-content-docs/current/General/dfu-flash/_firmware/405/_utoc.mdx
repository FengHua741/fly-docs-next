Вот описание процесса компиляции прошивки:
* Подключитесь к SSH и введите следующую команду и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` — это удаление предыдущих данных компиляции и прошивки, а `make menuconfig` — это команда для компиляции прошивки. После выполнения команды должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Micro-controller Architecture** и выберите **STMicroelectronics STM32**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

Изображения обновляются
* Перейдите в меню **Processor model**, выберите **STM32F405** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, затем выберите: **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.webp').default} size="100%" align="Center" />

* Выберите **Communication interface**, затем выберите: **USB to CAN bus bridge (USB on PA11/PA12)**
* Следуйте изображению, чтобы выбрать **CAN bus interface (CAN bus (on PB8/PB9))** --->

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/utoc.webp').default} size="100%" align="Center" />
