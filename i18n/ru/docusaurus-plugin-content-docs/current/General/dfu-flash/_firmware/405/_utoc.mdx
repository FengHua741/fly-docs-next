Ниже приведены инструкции по компиляции прошивки:
* После подключения к SSH введите следующие команды и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляет предыдущие данные компиляции и прошивку, `make menuconfig` компилирует прошивку, после выполнения должно появиться следующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Micro-controller Architecture** и выберите **STMicroelectronics STM32**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

Картинки обновляются
* Перейдите в меню **Processor model**, выберите **STM32F405** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, выберите: **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.webp').default} size="100%" align="Center" />

* Выберите **Communication interface**, выберите: **USB to CAN bus bridge (USB on PA11/PA12)**
* Согласно картинке выберите **CAN bus interface （CAN bus （on PB8/PB9）） --->**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/utoc.webp').default} size="100%" align="Center" />
