Чтобы скомпилировать прошивку, выполните следующие шаги:
* Подключитесь по SSH и введите следующую команду:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Команды `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` удаляют предыдущие данные компиляции и прошивку, а команда `make menuconfig` начинает процесс компиляции прошивки. После выполнения команды должен появиться следующий интерфейс:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Micro-controller Architecture** и выберите **STMicroelectronics STM32**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Перейдите в меню **Processor model** и выберите **STM32F072**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, затем выберите: **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* Выберите **Communication interface**, затем выберите: **CAN bus (on PB8/PB9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/can.webp').default} size="100%" align="Center" />
