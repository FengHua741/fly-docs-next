Вот описание процесса компиляции прошивки:
* Подключитесь к SSH и введите следующую команду и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` — это удаление предыдущих данных компиляции и прошивки, а `make menuconfig` — это команда для компиляции прошивки, после выполнения которой должен появиться нижеследующий интерфейс:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Enable extra low-level configuration options** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Micro-controller Architecture** и выберите **STMicroelectronics STM32**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Перейдите в меню **Processor model**, выберите **STM32F072** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.webp').default} size="100%" align="Center" />

* Выберите **Bootloader offset**, выберите: **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* В разделе **Communication interface** выберите: **USB (on PA11/PA12)** и нажмите Enter
    
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb.webp').default} size="100%" align="Center" />
* Проверьте соответствие следующей картинке, убедитесь, что все настройки совпадают с изображением, прежде чем продолжить:
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb2.webp').default} size="100%" align="Center" />
