Ниже описан процесс компиляции прошивки:
* Подключитесь к SSH и введите следующую команду и нажмите Enter:

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Где `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` — это удаление предыдущих данных компиляции и прошивки, а `make menuconfig` — это команда для компиляции прошивки, после выполнения которой должно появиться нижеследующее окно:

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Выберите **Включить дополнительные настройки низкого уровня** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Перейдите в меню **Архитектура микроконтроллера** и выберите **STMicroelectronics STM32**, затем нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Перейдите в меню **Модель процессора**, выберите **STM32H723** и нажмите Enter

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.webp').default} size="100%" align="Center" />

* Выберите **Offset загрузчика**, выберите: **128KiB загрузчик**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.webp').default} size="100%" align="Center" />

* Выберите **Частота тактирования (8 МГц кристалл)**, выберите: **25 МГц кристалл**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.webp').default} size="100%" align="Center" />

* В разделе **Коммуникационный интерфейс**, выберите: **USB (на PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/usb.webp').default} size="100%" align="Center" />
