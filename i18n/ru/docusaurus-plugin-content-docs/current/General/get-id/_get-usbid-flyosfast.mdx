## Заполнение USB ID

    1. Закройте «DEVICES» и заполните ID

    :::danger Важно знать
      * Измените в файле конфигурации:
      ```
      [mcu] 
      serial: /tmp/klipper_host_mcu
      ```  
      на
      ```
      [mcu host] 
      serial: /tmp/klipper_host_mcu
      ```
      * Добавьте:
      ```
      [mcu]
      serial: <вставьте здесь только что полученный ID>
      ```
      :::
    <table>
      <tr>
        <td>
          - В fluidd найдите [MCU]
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_cfg.webp').default} size="100%" align="center" />
        </td>
        <td>
          - В mainsail найдите [MCU]
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_cfg.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>


    2. Заполните ID в конфигурации
    :::tip

    * После заполнения ID нажмите в правом верхнем углу на «SAVE & RESTART »
    :::
    <table>
      <tr>
        <td>
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_save.webp').default} size="100%" align="center" />
        </td>
        <td>
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_save.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>
    3. Если Klipper показывает сообщение «ADC out of range», это нормально. Подключите нагревательный стол и термисторы к материнской плате и настройте выводы датчиков температуры для сопла и нагревательного стола, затем сохраните и перезагрузите.

:::danger
Внимание: Все ID, указанные в документации, являются примерами. У каждой материнской платы ID различаются. Пожалуйста, используйте фактические полученные ID.
:::
