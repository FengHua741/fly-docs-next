    :::tip

    * Убедитесь, что хост-компьютер может нормально подключаться к сети
    * Перед компиляцией прошивки используйте SSH для подключения к хост-компьютеру через сеть
    * Не используйте инструменты последовательного порта для входа в хост-компьютер

    :::

### Инструкция по компиляции прошивки и работе с конфигурационной страницей Klipper

    :::tip

    Убедитесь, что режим ввода клавиатуры установлен на полуточный режим, то есть английский режим

    :::

    1. Клавиша `↑`, клавиша `↓` на клавиатуре используется для перемещения курсора вверх или вниз для выбора пунктов меню
    2. Клавиша подтверждения `Enter` или пробел `Space` используется для выбора пунктов меню или перехода к подменю
    3. Клавиша выхода `ESC` используется для возврата на уровень выше
    4. Клавиша `Q` используется для выхода из конфигурационной страницы прошивки Klipper
    5. При выходе из конфигурационной страницы прошивки Klipper, если появляется запрос, нажмите клавишу `Y`, чтобы сохранить файл конфигурации

    :::warning

    Если количество опций на странице конфигурации меньше, сначала отметьте `[ ] Enable extra low-level configuration options`, чтобы отобразить некоторые скрытые параметры

    :::
