## Использование верхнего уровня FLY

:::подсказка
Откройте браузер и введите ваш IP-адрес:9999, чтобы получить доступ к FLY-Tools.
:::

:::подсказка
На верхнем уровне FLY интерфейс HDMI встроенного является Micro HDMI, обратите внимание при покупке кабеля HDMI!!!
:::

:::подсказка
Верхний уровень FLY поддерживает "FLY-PI-V2", "FLY-C8" и "FLY-PRO-X10" с подключением FPC "одним проводом".
:::

:::подсказка
Используя модуль преобразования FPC в HDMI, можно обеспечить совместимость с такими верхними уровнями, как Raspberry Pi.
:::

    * Для использования HDMI необходимо включить соответствующие параметры, чтобы экран мог быть запущен
