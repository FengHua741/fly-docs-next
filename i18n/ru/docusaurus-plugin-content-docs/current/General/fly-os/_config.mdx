## Важное уведомление
    :::tip Обязательно прочитать

    - Настройки в разделе файла конфигурации будут действовать только при первом запуске системы после прошивки.
    - Так как процесс настройки может быть сложным, настоятельно рекомендуется подключиться к машине через сеть и выполнить настройку через браузер.
    - Данная инструкция предоставляет доступные для изменения параметры.

    :::

## Как изменить файл конфигурации

### Изменение файла конфигурации после прошивки

- Конфигурационные элементы, которые действуют только при первом запуске после прошивки, необходимо изменять после прошивки с использованием профессионального текстового редактора для открытия файла конфигурации `config.txt` на мобильном диске `FlyOS-Conf`.

### Онлайн изменение конфигурации во время работы системы

- Если речь идет о обычных параметрах конфигурации, их можно изменить на странице веб-конфигурации системы и они вступят в силу после перезапуска системы.
    - Откройте в браузере адрес устройства `ip+:9999`, например: `192.168.1.2:9999`
    - Поддерживается изменение имени хоста
    - Поддерживается изменение часового пояса системы
    - Поддерживается включение расширения 40PIN верхнего уровня (обратите внимание, что после включения PWM невозможно использовать последовательный порт для подключения верхнего уровня, а после включения экрана может быть невозможна связь с верхним уровнем через интерфейс ускорения)
    - Поддерживается изменение модели материнской платы
    - Поддерживается изменение веб-интерфейса Klipper, включая Fuidd и Mainsail
    - Поддерживается включение или отключение KlipperScreen
    - Поддерживается принудительное выполнение процесса Klipper на указанный ядро CPU, можно указать, чтобы Klipper работал на выбранном ядре
