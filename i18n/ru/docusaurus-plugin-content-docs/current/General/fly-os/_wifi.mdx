    ## Система FLYOS-Armbian подключается к Wi-Fi

    1. Сначала подключитесь к хост-компьютеру через последовательный порт: <Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-serial-port-connection-to-host-ssh">Подключение через последовательный порт к хост-компьютеру SSH</Button>

    2. Введите команду `sudo nmtui`, возможно, потребуется ввести пароль, введите пароль `mellow`, затем нажмите клавишу `Enter`. На экране появится интерфейс, как показано на изображении ниже (обратите внимание: вводимый пароль не виден)  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi2.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi3.webp').default} size="60%" align="center" />

    3. Выберите опцию `Activate a connection`, затем нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi4.webp').default} size="60%" align="center" />

    4. Если ваш роутер работает нормально, то вы увидите список доступных сетей Wi-Fi. Знак `*` после имени сети указывает на силу сигнала
    :::tip
    - **Не поддерживаются сети Wi-Fi с именем на китайском языке!!!**
    - Поддерживается Wi-Fi 2.4G, а поддержка Wi-Fi 5G зависит от того, поддерживает ли ваш модуль Wi-Fi 5G
    - Модуль `M2WE` поддерживает Wi-Fi 5G
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi5.webp').default} size="60%" align="center" />

    5. Выберите нужную вам сеть Wi-Fi, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi6.webp').default} size="60%" align="center" />

    6. После этого появится следующее окно. Правильно введите пароль для вашей сети Wi-Fi, затем нажмите клавишу `Enter`, чтобы подключиться к Wi-Fi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi7.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi8.webp').default} size="60%" align="center" />

    7. Если пароль верный, перед соответствующей сетью Wi-Fi появится символ `*`, что означает успешное подключение к Wi-Fi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi9.webp').default} size="60%" align="center" />

    8. Затем дважды нажмите клавишу `Esc`, чтобы вернуться в командную строку
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi10.webp').default} size="60%" align="center" />
    
    9. В SSH введите команду `ip a | grep inet`, затем нажмите клавишу `Enter`. В выводе найдите IP-адрес, совпадающий с вашим маршрутизатором, чтобы получить доступ
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi11.webp').default} size="90%" align="center" />

    10. Откройте браузер, введите в поле поиска найденный IP-адрес хост-компьютера, затем нажмите клавишу `Enter`, чтобы войти в веб-интерфейс управления fluidd/mainsail
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi12.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi13.webp').default} size="90%" align="center" />

    11. В поле поиска браузера введите `IP:9999`, затем нажмите клавишу `Enter`, чтобы войти в интерфейс `fly tools`. Через этот интерфейс можно активировать различные функции хост-компьютера
    :::tip
    - Интерфейс `fly tools` доступен только в системе `FLYOS-Armbian`. В системе `FLYOS-FAST` интерфейс `fly tools` отсутствует!!!
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi14.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi15.webp').default} size="90%" align="center" />
