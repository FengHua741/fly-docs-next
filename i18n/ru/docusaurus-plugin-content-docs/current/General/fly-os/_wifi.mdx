:::warning Важно

* Для работы FLY-приложения необходим модуль M2WE или 2.4G Wi-Fi, иначе подключение к Wi-Fi невозможно.
* Обратите внимание, что не поддерживаются сети Wi-Fi с русскими названиями и сети Wi-Fi 2.4G/5G в режиме объединения.
* Если у вас mini pad или lite2, игнорируйте данное предупреждение.

:::

## Подключение к ПО верхнего уровня

* Если настройка Wi-Fi через BOOT-диск завершилась неудачей, используйте следующую конфигурацию для настройки соединения с Wi-Fi. Обратите внимание, что необходимо использовать последовательный порт для подключения к ПО верхнего уровня.
* Сначала подключитесь к SSH через последовательный порт ([нажмите здесь, чтобы перейти](/docs/DebugDoc/BasicTutorial/index.mdx "щелкните для перехода"))
* Выполните следующую команду, возможно, потребуется ввести пароль. Введите пароль `mellow` и нажмите Enter (пароль не отображается при вводе):

```cfg
sudo nmtui
```

    <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

* Выберите `Activate a connection` и нажмите Enter

    <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

* После входа в интерфейс используйте клавиши `↑ ↓` для выбора нужной сети Wi-Fi, затем нажмите Enter, введите пароль и дождитесь успешного подключения

    <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.webp').default} size="80%" align="left" />


    <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.webp').default} size="80%" align="left" />

* После успешного подключения вы увидите изображение, как показано ниже

    <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.webp').default} size="80%" align="left" />

* Нажмите клавишу ESC для выхода в командную строку. Введите `ip a`, IP-адрес верхнего уровня 192.168.2.126 будет показан на изображении
