:::warning Важно

* Для работы с ПО верхнего уровня для FLY необходим модуль M2WE или модуль Wi-Fi на 2,4 ГГц, иначе будет использоваться Wi-Fi.
* Обратите внимание, что не поддерживаются сети Wi-Fi с китайскими названиями и сети Wi-Fi, работающие в диапазонах 2,4 ГГц и 5 ГГц совместно.
* Если у вас mini pad или lite2, проигнорируйте это предупреждение.

:::

## Подключение к ПО верхнего уровня

* Если настройка Wi-Fi через BOOT дисков не удалась, можно воспользоваться этим руководством для подключения к Wi-Fi. Обратите внимание, что требуется использование последовательного порта для подключения к ПО верхнего уровня.
* Сначала подключитесь к SSH через последовательный порт ([щелкните здесь, чтобы перейти](/docs/DebugDoc/BasicTutorial/index.mdx "Щелкните для перехода"))
* Выполните следующую команду, возможно, потребуется ввести пароль. Введите пароль `mellow` и нажмите Enter (пароль не отображается).

```cfg
sudo nmtui
```
