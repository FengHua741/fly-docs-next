    ## Система FLYOS-Armbian подключается к WiFi

    1. Сначала подключитесь к хост-компьютеру через последовательный порт: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-serial-connection-to-host-ssh">Подключение FLYOS-Armbian к хост-компьютеру через SSH</Button>

    2. Введите команду `sudo nmtui`, возможно, потребуется ввести пароль, введите пароль `mellow`, затем нажмите клавишу `Enter`. Появится интерфейс, показанный на следующем изображении (обратите внимание: вводимый пароль не отображается)  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi2.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi3.webp').default} size="60%" align="center" />

    3. Выберите вариант `Activate a connection`, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi4.webp').default} size="60%" align="center" />

    4. Если ваш роутер WiFi работает нормально, вы увидите список WiFi, показанный на следующем изображении. Знак `*` после имени WiFi указывает на силу сигнала
    :::tip
    - **Не поддерживаются сети WiFi с именами на китайском языке !!!**
    - Поддерживается WiFi на частоте 2.4 ГГц, но поддержка WiFi на частоте 5 ГГц зависит от того, поддерживает ли приобретенный модуль WiFi частоту 5 ГГц
    - Модуль `M2WE` поддерживает WiFi на частоте 5 ГГц
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi5.webp').default} size="60%" align="center" />

    5. Выберите нужную вам сеть WiFi, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi6.webp').default} size="60%" align="center" />

    6. Далее появится следующий интерфейс, правильно введите пароль для вашей сети WiFi, затем нажмите клавишу `Enter`, чтобы подключиться к WiFi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi7.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi8.webp').default} size="60%" align="center" />

    7. Если пароль правильный, перед соответствующей сетью WiFi появится символ `*`, что означает успешное подключение к WiFi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi9.webp').default} size="60%" align="center" />

    8. Затем дважды нажмите клавишу `Esc`, чтобы вернуться к командной строке
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi10.webp').default} size="60%" align="center" />
    
    9. В SSH введите команду `ip a | grep inet`, затем нажмите клавишу `Enter`. В возвращаемом содержимом найдите IP-адрес, совпадающий с вашим роутером, чтобы получить доступ
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi11.webp').default} size="90%" align="center" />

    10. Откройте браузер, введите IP-адрес хост-компьютера, который вы только что запросили, затем нажмите клавишу `Enter`, чтобы войти в веб-интерфейс fluidd/mainsail
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi12.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi13.webp').default} size="90%" align="center" />

    11. В поисковой строке браузера введите `IP:9999`, затем нажмите клавишу `Enter`, чтобы войти в интерфейс `fly tools`. Через этот интерфейс можно активировать различные функции хост-компьютера
    :::tip
    - Интерфейс `fly tools` доступен только в системе `FLYOS-Armbian`. В системе `FLYOS-FAST` интерфейса `fly tools` нет !!!
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi14.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi15.webp').default} size="90%" align="center" />
