# Через подготовку USB запись на M2WE

## Подготовительные работы

    * Сначала установите программное обеспечение `Win32DiskImager`
    * Рекомендуется не использовать `balenaetcher` для записи системы
    * Распакуйте систему, после распаковки файлы будут иметь расширение `.img`
    * Скачайте [драйвер M2WE BOOT](/docs/ResDownload/auxiliary_software/ "Щелкните для перехода")

## Установка драйверов

    * Запустите программу `Zadig`
    * За исключением первого `Zadig`, который можно запустить без установки, остальные требуют установки. В процессе установки все время нажимайте `Далее` или `Next`
    * Откройте `Zadig`, в верхнем меню выберите `Options` и отметьте `List All Devices`

    <ImageView image={require('@site/docs/General/fly-os/img/zadig.webp').default} size="80%" align="left" />

:::

    * Запустите установленную программу FLY-BOOT, нажмите `Проверить устройство`
    * Если предыдущие шаги выполнены правильно, система сообщит о доступном устройстве
    * Нажмите `Запустить BOOT`, если всё нормально, будет показано успешное включение BOOT

## Форматирование M2WE

    * Используйте программное обеспечение `TFFormatter.exe` для форматирования диска BOOT

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * Или используйте стандартную функцию Windows для удаления разделов, после удаления разделов создайте их заново
    * Теперь можно открыть проводник, там появится флешка размером 14.5 ГБ (16 ГБ eMMC отображаются как 14.5 ГБ)
    * Таким образом можно использовать Win32diskimager для записи системного образа, указав диск букву этого флеш-накопителя размером 14.5 ГБ
