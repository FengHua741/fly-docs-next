## Важное уведомление
    :::tip Обязательно к прочтению

    - Настройки в разделе `part` файла конфигурации будут действовать только при первой загрузке системы после прошивки.
    - Из-за сложности процесса настройки настоятельно рекомендуется производить настройку через браузер после подключения к устройству по сети.
    - Данная инструкция предоставляет доступные для изменения параметры.

    :::

    :::warning Внимание

    - Лучше использовать веб-интерфейсы `mainsail` или `fluidd` для модификации настроек.
    - Не используйте перевод страницы, иначе после сохранения система может не запуститься.
    - В файле конфигурации избегайте ввода китайских символов или специальных символов.
    - После модификации файла конфигурации необходимо перезагрузить систему для применения изменений.

    :::

    :::danger

    - Ниже в файле конфигурации, в разделе `######### PLEASE DO NOT EDIT THE FOLLOWING #########`, ничего не изменяйте.

    :::

## Как изменить файл конфигурации

### Изменение конфигурационного файла после прошивки

- Настройки, которые действуют только при первом запуске системы после прошивки, можно изменить с помощью профессионального текстового редактора, открыв файл конфигурации `config.txt` на мобильном диске `FlyOS-Conf`.

### Онлайн изменение конфигурации во время работы системы

- Если речь идет о обычных настройках, их можно изменить через веб-интерфейс системы после перезагрузки.
    - Откройте IP-адрес устройства в браузере, например: `192.168.1.2/`.
    - Для fluidd отключите фильтр скрытых файлов и папок, как показано слева на картинке ниже.
    - Для mainsail активируйте отображение скрытых файлов, как показано справа на картинке ниже.

## Параметры конфигурации

### Настройка беспроводной сети

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | `пусто` | любое имя Wi-Fi | Имя Wi-Fi; если указано, система будет автоматически пытаться подключиться при загрузке |
| `WIFI_PASS` | `пусто` | оставьте пустым, если пароль отсутствует | Пароль Wi-Fi; если указано, система будет использовать пароль для подключения к Wi-Fi |

----

### Настройка принтера по умолчанию

:::tip
- Данные параметры применяются только при первой загрузке системы.
:::

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :---: | :--- | :--- |
| `language` | `zh-CN` | <details><summary>развернуть подробности</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | Устанавливает язык по умолчанию для fluidd и mainsail при первой загрузке |
| `printer_name` | `пусто` | произвольная строка | Устанавливает имя принтера по умолчанию для fluidd и mainsail |

----

### Настройки системы

- Данные параметры применяются при каждой загрузке системы.
- Ниже приведены настраиваемые опции.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `board` |  <code>{props.board}</code> |   <code>{props.board}</code> | Модель материнской платы, не изменяйте значения за пределами возможных |
| `localtime` | `Asia/Shanghai` | [Справочник часовых поясов](/docs/DebugDoc/flyos-fast/timezone) | Устанавливает часовой пояс по умолчанию; система автоматически синхронизирует время при подключении к интернету |
| `canbus_bitrate` | `1000000` | `1000000`<br/> `500000`<br/> `250000` | Устанавливает скорость по умолчанию для всех CAN-устройств в FAST-системе, изменение возможно только здесь, значение по умолчанию — 1М |

----

### Настройка экрана

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `screen` | `hdmi` | <code>nano</code>,<code>{props.screen}</code> <code>{props.screen3}</code> <br/> <code>{props.screen1}</code><code>{props.screen2}</code><br/> <code>{props.screen4}</code><code>{props.screen5}</code>| Устанавливает тип экрана, none — отключение всех дисплеев |
| `rotate` | `90` | `0`, `90`, `180`, `270` | Устанавливает угол поворота FLY-TFT, для HDMI неактивно |
| `klipper_screen` | `0` | `0`, `1` | Включить/выключить интерфейс KlipperScreen<br/>1: включить<br/>0: выключить<br/>действительно при `printer=klipper` |
| `moon2uart` | `0` | `0`, `1` | Включить/выключить moon2uart<br/>1: включить<br/>0: выключить<br/>действительно при `printer=klipper` |

----

### Настройки Klipper

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | Устанавливает интерфейс по умолчанию для Klipper<br/>не влияет на функцию онлайн-переключения<br/>действительно при `printer=klipper` |
| `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | Устанавливает скорость по умолчанию для всех CAN-устройств в FAST-системе, изменение возможно только здесь, значение по умолчанию — 1М |
| `crowsnest` | `0` | `0`, `1` | Включить/выключить Crowsnest<br/>1: включить<br/>0: выключить<br/>действительно для режимов klipper и rrf |

----

### Настройки типа принтера

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `printer` | `klipper` | `klipper`, `rrf` | Конфигурация системы для запуска службы<br/>`klipper`: klipper, moonraker и другие службы<br/>`rrf`: RRF DSF, DWC и другие службы |

----

### Выключение питания при выключении

- Данные параметры применяются при каждой загрузке системы.
- Для работы с автономным возобновлением печати отключите функцию выключения питания.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `shutdown_pin_state` |<code>{props.state}</code>| `0`, `1` | Устанавливает уровень электрического состояния кнопки выключения<br/> 1: низкий уровень повторяющийся<br/> 0: высокий уровень триггерный  |
| `shutdown_pin` | <code>{props.shutdown_pin}</code>| <code>{props.shutdown_pin}</code> | Устанавливает триггерный вывод кнопки выключения, none — отключение функции |

----
