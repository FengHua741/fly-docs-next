## Важное уведомление
    :::tip Обязательно прочитать

    - Настройки в разделе `part` файла конфигурации действуют только при первом запуске системы после прошивки.
    - Данная инструкция предоставляет возможность изменять только определенные параметры.

    :::

    :::warning Внимание

    - Старайтесь использовать веб-интерфейсы `mainsail` или `fluidd` для внесения изменений в настройки.
    - Не открывайте страницы перевода веб-сайтов, иначе после сохранения система может не запуститься.
    - В файле конфигурации лучше избегать использования китайских символов или специальных символов.
    - После изменения файла конфигурации необходимо перезагрузить систему для применения изменений.

    :::

    :::danger

    - Содержимое файла конфигурации ниже раздела `######### PLEASE DO NOT EDIT THE FOLLOWING #########` не подлежит редактированию.

    :::

## Как изменить файл конфигурации

### Изменение конфигурационного файла после прошивки

- Настройки, которые действуют только при первом запуске системы, должны быть изменены с использованием профессионального текстового редактора. Откройте файл конфигурации `config.txt` на мобильном диске `FlyOS-Conf`.

### Онлайн изменение конфигурации при запущенной системе

- Если это обычные настройки, их можно изменить через веб-интерфейс системы и применить изменения после перезагрузки.
    - Откройте IP-адрес устройства в браузере, например: `http://192.168.1.2/`
    - Для fluidd отключите фильтр скрытых файлов, как показано слева на картинке.
    - Для mainsail включите отображение скрытых файлов, как показано справа на картинке.

## Параметры конфигурации

### Настройка беспроводной сети

- Данные параметры применяются при каждом запуске системы.

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | пусто | любое имя Wi-Fi | Имя сети Wi-Fi; если указано, система автоматически попробует подключиться при загрузке |
| `WIFI_PASS` | пусто | оставьте пустым, если нет пароля | Пароль Wi-Fi; если указано, используется для подключения с паролем |

----

### Настройка принтера по умолчанию

:::tip
- Данные параметры действуют только при первом запуске.
:::

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :---: | :--- | :--- |
| `language` | `zh-CN` | <details><summary>Развернуть подробности</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | Устанавливает язык по умолчанию для fluidd и mainsail при первом запуске |
| `printer_name` | пусто | любая строка | Устанавливает имя принтера по умолчанию для fluidd и mainsail |

----

### Настройки системы

- Данные параметры применяются при каждом запуске системы.
- Ниже приведены доступные для изменения параметры.

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :-------: | :--- | :--- |
| `board` |  <code>{props.board}</code> |   <code>{props.board}</code> | Модель платы, не изменяйте значения вне списка |
| `localtime` | `Asia/Shanghai` | [Список часовых поясов](/docs/DebugDoc/flyos-fast/timezone) | Устанавливает часовой пояс системы, система автоматически синхронизирует время при подключении к сети |
| `canbus_bitrate` | `1000000` | `1000000`<br/> `500000`<br/> `250000` | Устанавливает скорость всех CAN-устройств в FAST-системе, может быть изменено только здесь, значение по умолчанию 1 Мбит/с |

----

### Настройка экрана

- Данные параметры применяются при каждом запуске системы.

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :-------: | :--- | :--- |
| `screen` | `hdmi` | <code>nano</code>,<code>{props.screen}</code> <br/> <code>{props.screen1}</code><code>{props.screen2}</code> | Устанавливает экран, none отключает все выводы |
| `rotate` | `90` | `0`, `90`, `180`, `270` | Устанавливает угол поворота FLY-TFT, не действует для HDMI |
| `klipper_screen` | `0` | `0`, `1` | Устанавливает, включать ли интерфейс KlipperScreen<br/>1: включить<br/>0: выключить<br/>Эффективно при `printer=klipper` |
| `moon2uart` | `0` | `0`, `1` | Устанавливает, включать ли moon2uart<br/>1: включить<br/>0: выключить<br/>Эффективно при `printer=klipper` |


----

### Настройки Klipper

- Данные параметры применяются при каждом запуске системы.

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :-------: | :--- | :--- |
| `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | Устанавливает стандартный пользовательский интерфейс Klipper<br/>Не влияет на функцию быстрой смены интерфейса онлайн<br/>Эффективно при `printer=klipper` |
| `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | Устанавливает скорость всех CAN-устройств в FAST-системе, может быть изменено только здесь, значение по умолчанию 1 Мбит/с |
| `crowsnest` | `0` | `0`, `1` | Устанавливает, включать ли Crowsnest<br/>1: включить<br/>0: выключить<br/>Доступно для режимов klipper и rrf |

----

### Настройка типа принтера

- Данные параметры применяются при каждом запуске системы.

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :-------: | :--- | :--- |
| `printer` | `klipper` | `klipper`, `rrf` | Устанавливает тип службы, с которой система будет запускаться<br/>`klipper`: klipper, moonraker и другие службы<br/>`rrf`: RRF DSF, DWC и другие службы |

----

### Выключение питания при выключенном питании

- Данные параметры применяются при каждом запуске системы.
- Для использования функции автономного продолжения печати необходимо отключить функцию выключения питания.

| Параметр | Значение по умолчанию | Допустимые значения | Описание |
| :---: | :-------: | :--- | :--- |
| `shutdown_pin_state` | `0` | `0`, `1` | Устанавливает логическое состояние кнопки выключения<br/> 1: повторяющийся низкий уровень<br/> 0: высокий уровень триггер |
| `shutdown_pin` | <code>{props.shutdown_pin}</code>| <code>{props.shutdown_pin}</code> | Устанавливает пин кнопки выключения, установите none для отключения |

----

| :---: | :-------: | :--- | :--- |
