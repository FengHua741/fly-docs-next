## Важное уведомление
    :::tip Обязательно прочитать

    - Настройки в разделе `part` файла конфигурации будут применяться только при первой загрузке системы после прошивки.
    - Из-за сложности процесса настройки настоятельно рекомендуется подключиться к машине через сеть и выполнить настройку через браузер.
    - Данная инструкция предоставляет доступные для изменения параметры.

    :::

    :::warning Внимание

    - Лучше использовать веб-интерфейсы `mainsail` или `fluidd` для модификации настроек.
    - Не открывайте страницы переводчиков, так как после сохранения система может не запуститься.
    - В файле конфигурации не рекомендуется использовать китайские символы или специальные символы.
    - Для применения изменений в файле конфигурации необходимо перезагрузить систему.

    :::

    :::danger

    - Контент ниже раздела `######### PLEASE DO NOT EDIT THE FOLLOWING #########` в файле конфигурации не следует редактировать.

    :::

## Как изменить файл конфигурации

### Изменение конфигурационного файла после завершения прошивки

- Конфигурационные параметры, которые действуют только при первой загрузке системы после прошивки, должны быть изменены с использованием профессионального текстового редактора в файле конфигурации `config.txt` на мобильном диске `FlyOS-Conf`.

### Онлайн-изменение конфигурации при уже запущенной системе

- Если речь идет о обычных параметрах, их можно изменить через веб-интерфейс системы, после чего перезагрузить систему для применения изменений.
    - Откройте в браузере IP-адрес устройства, например: `192.168.1.2/`.
    - В `fluidd` снимите флажок, как показано слева на рисунке -> «Отключить фильтр скрытых файлов и папок».
    - В `mainsail` установите флажок -> «Показывать скрытые файлы».

## Параметры конфигурации

### Настройка беспроводной сети

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :---: | :--- | :--- |
| `WIFI_SSID` | Пустое | Любой SSID Wi-Fi | Имя Wi-Fi; если указано, система автоматически попробует подключиться при загрузке |
| `WIFI_PASS` | Пустое | Оставьте пустым, если пароль отсутствует | Пароль Wi-Fi; если указано, используется для подключения к Wi-Fi |

----

### Настройки принтера по умолчанию

:::tip
- Данные параметры действуют только при первой загрузке системы.
:::

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :---: | :--- | :--- |
| `language` | `zh-CN` | <details><summary>Развернуть подробности</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | Устанавливает язык по умолчанию для fluidd и mainsail при первой загрузке |
| `printer_name` | Пустое | Любой строковой символ | Устанавливает имя принтера по умолчанию для fluidd и mainsail |

----

### Системная настройка

- Данные параметры применяются при каждой загрузке системы.
- Ниже приведены доступные для изменения опции.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `board` |  <code>{props.board}</code> |   <code>{props.board}</code> | Модель платы, не рекомендуется изменять значения вне указанных |
| `localtime` | `Asia/Shanghai` | [Таблица часовых поясов](/docs/DebugDoc/flyos-fast/timezone) | Устанавливает часовой пояс системы, система автоматически синхронизирует время при подключении к сети |
| `canbus_bitrate` | `1000000` | `1000000`<br/> `500000`<br/> `250000` | Устанавливает стандартную скорость всех CAN-устройств в FAST-системе, изменение доступно только здесь, значение по умолчанию — 1М |

----

### Настройка экрана

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `screen` | `hdmi` | <code>nano</code>,<code>{props.screen}</code> <code>{props.screen3}</code> <br/> <code>{props.screen1}</code><code>{props.screen2}</code>| Устанавливает экран, none отключает все выводы |
| `rotate` | `90` | `0`, `90`, `180`, `270` | Устанавливает угол поворота экрана FLY-TFT, параметр hdmi игнорируется |
| `klipper_screen` | `0` | `0`, `1` | Устанавливает использование интерфейса KlipperScreen<br/>1: Включить<br/>0: Выключить<br/>Действительно при `printer=klipper` |
| `moon2uart` | `0` | `0`, `1` | Устанавливает использование moon2uart<br/>1: Включить<br/>0: Выключить<br/>Действительно при `printer=klipper` |

----

### Настройки Klipper

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | Устанавливает интерфейс по умолчанию для Klipper<br/>Не влияет на функцию переключения в один клик<br/>Действительно при `printer=klipper` |
| `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | Устанавливает стандартную скорость всех CAN-устройств в FAST-системе, изменение доступно только здесь, значение по умолчанию — 1М |
| `crowsnest` | `0` | `0`, `1` | Устанавливает использование Crowsnest<br/>1: Включить<br/>0: Выключить<br/>Поддерживается как в режиме klipper, так и в режиме rrf |

----

### Настройка типа принтера

- Данные параметры применяются при каждой загрузке системы.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `printer` | `klipper` | `klipper`, `rrf` | Устанавливает тип запуска системы<br/>`klipper`: Запускаются службы klipper, moonraker и т.д.<br/>`rrf`: Запускаются службы RRF DSF, DWC и т.д. |

----

### Автоматическое выключение при выключении питания

- Данные параметры применяются при каждой загрузке системы.
- Для использования функции автономной печати при обесточивании требуется отключить функцию автоматического выключения.

| Параметр | Значение по умолчанию | Возможные значения | Описание |
| :---: | :-------: | :--- | :--- |
| `shutdown_pin_state` |<code>{props.state}</code>| `0`, `1` | Устанавливает уровень триггера кнопки выключения<br/> 1: Низкий уровень повторно<br/> 0: Высокий уровень триггер |
| `shutdown_pin` | <code>{props.shutdown_pin}</code>| <code>{props.shutdown_pin}</code> | Устанавливает пин триггера кнопки выключения, установка none отключает функцию |

----
