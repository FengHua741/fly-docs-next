### Инструкция по компиляции и конфигурации фиксированной части Klipper

    :::tip

    Убедитесь, что режим ввода клавиатуры установлен на полуширинный (английский режим)

    :::

    1. Клавиши `↑` и `↓` на клавиатуре используются для перемещения курсора вверх и вниз для выбора пунктов меню
    2. Клавиша подтверждения `Enter` или пробел `Space` используется для выбора пункта меню или входа в подменю
    3. Клавиша выхода `ESC` используется для возврата на уровень выше в меню
    4. Клавиша `Q` используется для выхода из страницы конфигурации фиксированной части Klipper
    5. При выходе из страницы конфигурации фиксированной части Klipper с подсказкой нажмите клавишу `Y`, чтобы сохранить файл конфигурации

    :::warning

    Если в меню доступно мало опций, сначала выберите `[ ] Enable extra low-level configuration options`, чтобы отобразить некоторые скрытые параметры

    :::
