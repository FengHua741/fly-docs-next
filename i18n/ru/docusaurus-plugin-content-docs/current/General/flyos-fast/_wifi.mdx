    ## СОЕДИНЕНИЕ СИСТЕМЫ FLYOS-FAST С WiFi
    
    1. В SSH введите команду `nmtui`, нажмите клавишу `Enter`, после чего появится интерфейс, показанный на следующем изображении
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi2.webp').default} size="40%" align="center" />

    2. Выберите опцию `Activate a connection`, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi3.webp').default} size="40%" align="center" />

    3. Если ваш роутер работает нормально, то вы увидите список WiFi, представленный на следующем изображении. Звездочка (`*`) рядом с названием WiFi указывает на силу сигнала
    :::tip
    - **Не поддерживаются сети WiFi с китайскими названиями!!**
    - Поддерживается WiFi 2.4G, но поддержка WiFi 5G зависит от модуля WiFi, который вы приобрели
    - Модуль `M2WE` поддерживает WiFi 5G
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi4.webp').default} size="40%" align="center" />

    4. Выберите нужную вам сеть WiFi, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi5.webp').default} size="40%" align="center" />

    5. После этого появится следующее окно. Введите правильный пароль для WiFi и нажмите клавишу `Enter`, чтобы подключиться
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi6.webp').default} size="40%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi7.webp').default} size="40%" align="center" />

    6. Если пароль верный, перед соответствующей сетью WiFi появится символ `*`, что означает успешное подключение
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi8.webp').default} size="40%" align="center" />

    7. Далее нажмите клавишу `Esc` дважды, чтобы вернуться в командную строку
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi9.webp').default} size="60%" align="center" />
    
    8. В SSH введите команду `ip a | grep inet`, нажмите клавишу `Enter`. В результате найдите IP-адрес, совпадающий с подсетью вашего роутера, чтобы получить доступ
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi10.webp').default} size="90%" align="center" />

    9. Откройте браузер, введите в поисковую строку IP-адрес, полученный ранее, нажмите клавишу `Enter`, чтобы войти в веб-консоль fluidd/mainsail
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi11.webp').default} size="60%" align="center" />
    
    :::tip
    - В системе `FLYOS-FAST` можно в любой момент переключаться между веб-консолями fluidd/mainsail, щелкнув на значок mainsail/fluidd в правом верхнем углу!
    :::
