    ## Соединение FLYOS-FAST с WiFi
    :::tip
    - Если конфигурация подключения к WiFi в диске `FlyOS-Conf` не удалась, можно выполнить этот учебник, чтобы через последовательный порт подключиться к хост-компьютеру и подключиться к WiFi
    :::

    1. Сначала подключитесь к хост-компьютеру через последовательный порт: <Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-serial-port-connection-to-host-ssh">Соединение FLYOS-FAST с хост-компьютером через последовательный порт SSH</Button>

    2. В SSH введите команду `nmtui`, нажмите клавишу `Enter`, после чего появится интерфейс, как показано на рисунке ниже
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi2.webp').default} size="40%" align="center" />

    3. Выберите вариант `Activate a connection`, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi3.webp').default} size="40%" align="center" />

    4. Если ваш маршрутизатор работает нормально, то вы увидите список WiFi, как показано на рисунке ниже. Звездочка (*) рядом с именем WiFi указывает на силу сигнала
    :::tip
    - **Не поддерживаются сети WiFi с китайскими названиями!!**
    - Поддерживается WiFi на частоте 2.4 ГГц, но поддержка WiFi на 5 ГГц зависит от модуля WiFi, который вы приобрели. Модуль `M2WE` поддерживает WiFi на 5 ГГц.
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi4.webp').default} size="40%" align="center" />

    5. Выберите нужную вам сеть WiFi, нажмите клавишу `Enter`, чтобы перейти на следующий уровень меню
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi5.webp').default} size="40%" align="center" />

    6. Далее появится следующий интерфейс, правильно введите пароль WiFi, нажмите клавишу `Enter`, чтобы подключиться к WiFi
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi6.webp').default} size="40%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi7.webp').default} size="40%" align="center" />

    7. Если пароль верный, перед соответствующей сетью WiFi появится символ `*`, что означает успешное подключение к WiFi
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi8.webp').default} size="40%" align="center" />

    8. Затем дважды нажмите клавишу `Esc`, чтобы вернуться в командную строку
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi9.webp').default} size="60%" align="center" />
    
    9. В SSH введите команду `ip a | grep inet`, нажмите клавишу `Enter`, в результатах поиска найдите IP-адрес, совпадающий с вашим маршрутизатором, чтобы получить доступ
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi10.webp').default} size="90%" align="center" />

    10. Откройте браузер, введите в поле поиска IP-адрес хост-компьютера, который вы только что нашли, нажмите клавишу `Enter`, чтобы перейти к веб-интерфейсу fluidd/mainsail
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi11.webp').default} size="60%" align="center" />
    
    :::tip
    - В системе FLYOS-FAST можно в любое время переключаться между веб-интерфейсами fluidd/mainsail, щелкнув значок mainsail/fluidd в правом верхнем углу!
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi12.webp').default} size="90%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi13.webp').default} size="90%" align="center" />
