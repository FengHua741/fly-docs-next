::::предупреждение Операционное руководство для страницы настройки Wi-Fi
* Убедитесь, что режим ввода клавиатуры установлен на полуэкранный режим, то есть английский режим
* Клавиша `↑` на клавиатуре и клавиша `↓` используются для перемещения курсора вверх и вниз для выбора пунктов меню
* Клавиша подтверждения `Enter` используется для выбора пункта меню или входа в подменю
* Клавиша выхода `ESC` используется для возврата на предыдущий уровень меню
::::
