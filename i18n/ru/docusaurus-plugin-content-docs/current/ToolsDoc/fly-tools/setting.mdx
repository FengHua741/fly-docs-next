# Демонстрация функций

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/setting.webp').default} size="100%" align="left" />


    * Поддержка изменения имени хоста
    * Поддержка изменения часового пояса системы
    * Поддержка включения расширения 40PIN мotherboard (обратите внимание, что после включения PWM невозможно использовать последовательный порт для подключения к материнской плате, а при включении экрана может быть недоступен порт акселерометра матьшиной платы)
    * Поддержка изменения модели материнской платы
    * Поддержка изменения веб-интерфейса Klipper, поддержка Fuidd и Mainsail
    * Поддержка включения или отключения KlipperScreen
    * Поддержка принудительного запуска процесса Klippy на указанный CPU ядро, можно указать, чтобы Klipper работал на выбранном ядре
