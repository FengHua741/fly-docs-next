# Установка FLY-Tools

    1. Скачать последнюю версию Klipper

    ```bash
    cd ~/klipper
    git pull
    ```

    * Если вы вносили изменения в локальный исходный код Klipper, что вызывает ошибку при скачивании, используйте следующую команду. (Это действие приведет к потере ваших предыдущих изменений)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. Скачать последнюю версию Katapult (ранее назывался Canboot)

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Установить службу Fly-Tools

        * Процесс установки зависит от поставщика Git. Если процесс загрузки застрянет, нажмите `CTRL+C`, чтобы выйти и повторить попытку.
        * Максимальное количество запросов в час на Github — 60, неизвестно для Gitee.
        * Рекомендуется пользователям из Китая использовать Gitee для получения скрипта установки
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Получить скрипт установки через Github

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
