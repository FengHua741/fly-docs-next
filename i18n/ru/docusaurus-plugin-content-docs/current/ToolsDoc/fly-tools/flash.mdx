# Функциональная демонстрация

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/home.gif').default} size="100%" align="left" />

    * Поддержка一键 копирования идентификатора устройства, автоматическое распознавание ID и добавление перед соответствующей конфигурацией **serial:** , **canbus_uuid:** , **device:**
    * Поддержка изменения скорости CAN и буферизации CAN, что позволяет пользователям удобно настраивать CAN
    * Поддержка поиска устройств камеры, избегая неудобств для пользователей при поиске камер

    :::tip Подсказка

    * Обратите внимание, что FLY-Tools не будет автоматически создавать конфигурацию CAN, необходимо самостоятельно создать конфигурацию CAN для использования функции изменения скорости и буферизации CAN

    :::

    * Пожалуйста, предоставьте SSH использование сетевого подключения к вашему пользователю для выполнения команд ниже
    * Ubuntu временно не поддерживает эту функцию

    ```bash
    sudo /bin/sh -c "cat > /etc/network/interfaces.d/can0" << EOF
    allow-hotplug can0
    iface can0 can static
        bitrate 1000000
        up ifconfig $IFACE txqueuelen 1024
        pre-up ip link set can0 type can bitrate 1000000
        pre-up ip link set can0 txqueuelen 1024
    EOF
    ```
