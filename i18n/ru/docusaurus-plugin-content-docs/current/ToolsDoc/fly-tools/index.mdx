# О简介 FLY-Tools

    FLY-Tools — это веб-инструмент отладки, разработанный компанией Guangzhou Miling Electronic Technology Co., Ltd. специально для Klipper. Он совместим не только с Fly Host (входит в систему по умолчанию, без необходимости установки), но и с большинством других хостов, на которых установлены службы Klipper.

    :::tip Примечание

    * FLY-Tools работает только на Linux-системах с установленным klipper

    :::
        

## Особенности FLY-Tools

    * Поддержка нескольких языков, темная тема
    * Возможность запросить и одномоментно скопировать USB Serial ID, USB Device ID, CANbus UUID, Video Device ID
    * Возможность одномоментного включения или отключения определенных CAN-устройств
    * Возможность онлайн-изменения скорости и размера буфера отправки для CAN-устройств
    * Возможность онлайн-компиляции прошивки klipper для материнской платы FLY (компиляция Katapult находится в процессе адаптации)
    * Поддержка онлайн-загрузки, включая методы загрузки DFU, HID, CAN
    * Возможность скачивания всех скомпилированных файлов прошивок
    * Возможность онлайн-создания графиков нагрузки Klipper
    * Возможность одномоментной автоматической измерения и создания резонансных диаграмм
    * Поддержка веб-SSH-соединения
    * Поддержка веб-страниц SSH (на основе [ttyd](https://github.com/tsl0922/ttyd))
    * Совместимость с системой FLYOS 3.3.4, поддерживающая настройку экрана, отображение веб-страниц и т. д.
