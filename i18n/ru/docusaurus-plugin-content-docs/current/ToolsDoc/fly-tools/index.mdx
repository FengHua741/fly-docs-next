# Введение в FLY-Tools

    FLY-Tools — это веб-инструмент отслеживания и настройки, разработанный компанией Guangzhou Mailing Electronic Technology Co., Ltd. специально для Klipper. Он работает как на родном Fly Host PC (в системе по умолчанию, без необходимости установки), так и совместим с большинством существующих систем Host PC с установленным сервисом Klipper.

    :::tip Подсказка

    * FLY-Tools применяется только к Linux-системам с установленным klipper

    :::

## Особенности FLY-Tools

    * Поддержка нескольких языков, тема темной темы
    * Возможность запроса и одного щелчка для копирования ID USB-порта, ID USB-устройства, UUID CANbus и ID устройства видео
    * Однощелчковая активация или деактивация определенного CAN-устройства
    * Возможность изменения скорости и размера буфера отправки CAN-устройства онлайн
    * Возможность онлайн-компиляции прошивки klipper для материнской платы FLY (компиляция Katapult находится в стадии адаптации)
    * Поддержка онлайн-записи, включая методы записи DFU, HID, CAN и другие
    * Возможность скачивания всех скомпилированных файлов прошивок
    * Возможность онлайн-создания нагрузочных графиков Klipper
    * Возможность автоматического измерения и генерации резонансных диаграмм одним щелчком мыши
    * Поддержка веб-SSH-подключения
    * Поддержка веб-страниц SSH (на основе [ttyd](https://github.com/tsl0922/ttyd))
    * Поддержка системы FLYOS 3.3.4, где можно настроить экран, отображение веб-страницы и другое
