# Демонстрация функций

    <ImageView image={require('@site/docs/ToolsDoc/fly-tools/img/utils.gif').default} size="100%" align="left" />

    * Поддерживает генерацию стандартных графиков нагрузки Klipper
    * Поддерживает генерацию графиков частоты MCU
    * Поддерживает генерацию графиков системной нагрузки
    * Поддерживает генерацию графиков температуры
    * Поддерживает генерацию графиков частотной характеристики, также можно создавать графики по существующим файлам


# Если возникает ошибка об отсутствии `matplotlib`, выполните следующую команду

    ```bash
    ~/klippy-env/bin/pip install -v matplotlib
    ```
