## Подключение

    * Подключается к разъему PWM основной платы, для других плат необходимо обратиться к соответствующей документации.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-7040/img/wiring.webp').default} size="80%" align="Center" />

## Пример конфигурации

    ```cfg
    [fan]
    ##    Вентилятор охлаждения печати 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## Решение проблемы автоматического вращения при включении

    :::info
    * Здесь приводится только краткое объяснение
    * При компиляции прошивки добавьте сигнал вывода PWM и инвертируйте его состояние
    * При тестировании использовалась платформа SUPER8PRO, вывод PF8; при компиляции прошивки выберите `GPIO pins to set at micro-controller startup` и добавьте `!PF8`
    * Обратите внимание, что используется английская восклицательная точка
    * Аналогично для других плат
    :::
