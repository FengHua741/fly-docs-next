# Подключение к Klipper

    :::info Важное замечание
    * AirClick должен быть подключен к той же сети, что и целевая принтер
    :::

1. Нажмите на системные настройки слева
   
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. Выберите `Klipper` в разделе `Printer Type`
3. В поле `Moonraker IP` введите IP-адрес машины Klipper
4. В поле `Moonraker Port` введите порт управляющей машины Klipper, по умолчанию используется `7125`. Если используется другой порт, введите соответствующий порт
5. Поле `Apikey` игнорируйте, временно не поддерживается Apikey

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. После настройки как показано на рисунке выше, нажмите "сохранить" в правом нижнем углу для сохранения конфигурации
7. При успешном подключении будет показано уведомление в верхней части веб-страницы

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. Если подключение не удалось:
    - Проверьте правильность IP-адреса и порта
    - Убедитесь, что принтер и AirClick подключены к одной сети
    - Проверьте, работает ли управляющая машина Klipper нормально
    - Попробуйте открыть веб-страницу управляющей машины Klipper в браузере
