# Обновление прошивки и перепрошивка

    :::danger Важно

    * Предустановленная прошивка уже включена при выпуске устройства

    :::

## Скачивание прошивки

    * Описание прошивки
    * `wbtn-vX.X.X-X-XXXXXX-full.bin` — полная прошивка, должна записываться с адреса `0x0`, после записи все настройки будут потеряны
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin` — инкрементальная прошивка, должна записываться с адреса `0x10000`, обновляет только прошивку без влияния на существующие настройки
    * `wbtn-vX.X.X-X-XXXXXX-www.bin` — прошивка интерфейса, должна записываться с адреса `0x290000`, обновляет только веб-интерфейс без влияния на существующую прошивку и настройки
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/airclick/packages">Скачать пакет прошивки</Button>

## Обновление прошивки
    * Необходимо отключить питание устройства `AirClick` и подключить разъем для перепрошивки ESP32 к компьютеру по BOOT
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.webp').default} size="80%" align="Center" />

    1. Скачайте и распакуйте инструмент для перепрошивки USB <Button variant="contained" disableElevation component="a" rel="noopener" href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">Щелкнуть для скачивания</Button>
    2. Откройте ``flash_download_tool_3.9.4.exe``, обратите внимание: после запуска инструмента появится окно CMD, а затем интерфейс инструмента, обязательно не закрывайте окно CMD
    3. В инструменте перепрошивки выберите в ``ChipType:`` вариант ``ESP32-C3``
    4. В инструменте перепрошивки выберите в ``WorkMode:`` вариант ``Develop``
    5. В инструменте перепрошивки выберите в ``LoadMode:`` вариант ``USB``
    6. Нажмите ``OK``, см. изображение ниже
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.webp').default} size="50%" align="Center" />
    7. После нажатия появится изображение ниже
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.webp').default} size="50%" align="Center" />

    :::tip

    * Зеленые выделенные области на изображении выше должны быть точно такими же, как на рисунке, изменять их нельзя

    :::

    8. Подключите AirClick к компьютеру через USB Type-C кабель
    9. Щелкните синюю область 1 на изображении выше, чтобы выбрать прошивку для записи
    10. Щелкните синюю область 2 на изображении выше, чтобы ввести адрес записи
    11. Щелкните синюю область 3 на изображении выше, чтобы выбрать COM-порт AirClick (если нет, удерживайте кнопку перепрошивки AirClick и подключите устройство к компьютеру)
    12. Щелкните синюю область 4 на изображении выше, чтобы установить максимальную скорость波特ности `921600`
    13. Щелкните синюю область 5 на изображении выше, чтобы начать запись, дождитесь завершения записи, после чего AirClick будет обновлена (отсоединяйте кабель от AirClick только после получения подтверждения о завершении записи)
