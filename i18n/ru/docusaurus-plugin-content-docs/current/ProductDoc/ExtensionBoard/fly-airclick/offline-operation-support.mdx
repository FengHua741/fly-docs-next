# Настройка подключения Klipper через кабель

:::danger Важно

* Необходимо прошить STM32F072, чтобы подключить его к Klipper, иначе операции с проводным подключением не поддерживаются.

:::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции (время займет немного)
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Переход в режим прошивки

    * При отключенном питании AirClick удерживайте кнопку BOOT на STM32, чтобы подключить разъем прошивки STM32 через Type-C к хост-компьютеру
