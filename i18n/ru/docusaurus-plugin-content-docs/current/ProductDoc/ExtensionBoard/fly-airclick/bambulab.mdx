# Подключение к BambuLab

    :::info Важное примечание
    * AirClick должен быть подключен к той же сети, что и целевая принтер
    * Поддерживается только подключение через локальную сеть (LAN) к принтеру BambuLab
    :::

1. Включите режим локальной сети (LAN Only Mode) на принтере BambuLab

    - Включите экран принтера BambuLab
    - `Настройки` -> `Режим локальной сети (LAN Only Mode)` -> `Включить`
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-1.webp').default} size="40%" align="Center" />

2. Нажмите на системные настройки слева

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="60%" align="Center" />

3. Выберите `BambuLab` в разделе `Тип принтера`
4. Нажмите кнопку обновления списка устройств

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-2.webp').default} size="60%" align="Center" />

5. Выберите принтер

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-3.webp').default} size="60%" align="Center" />

    - Если принтер не появляется в списке после обновления, можно выбрать пользовательский ввод. Вручную введите IP-адрес и серийный номер принтера

6. Заполните код доступа
    - Код доступа — это код доступа в режиме локальной сети принтера, который можно найти в настройках принтера, следуя шагам из пункта 1

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/bambu-4.webp').default} size="60%" align="Center" />

5. После настройки выше нажмите `Подтвердить` в правом нижнем углу для сохранения конфигурации
6. При успешном подключении появится уведомление в верхней части веб-страницы

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

7. Если подключение не удалось
    - Проверьте правильность IP-адреса
    - Убедитесь, что принтер и AirClick подключены к одной сети
    - Проверьте, включен ли режим локальной сети на принтере
    - Проверьте, нет ли ошибки в коде доступа
