# FLY VoltUp Power

    Модуль повышения напряжения FLY-VoltUp Power разработан специально для высоковольтного драйвера, позволяя пользователям реализовать высоковольтное питание без необходимости покупки дополнительного источника питания высокого напряжения, что помогает экономить место при проектировании 3D-принтеров.

## Основные характеристики

    * Выходное напряжение:可以选择 48В или 60В
    * Разъем для вентилятора на плате с фиксатором для вентилятора 4010, выходное напряжение равно входному
    * Двойной разъем для расширения питания: предоставляет один разъем питания, который позволяет одновременно расширять два выходных разъема питания и индивидуально устанавливать напряжение одного из них на 48В или 60В
    * Легкость установки: стандартный размерный дизайн подходит для различных условий использования, не требует дополнительной установки драйверов.
    * Компактные размеры: размер расширительной платы рационально спроектирован, не занимает много места, удобен для интеграции в различные проекты.

## Технические характеристики:

    * Разъем питания: 12В/24В
    * Выходные разъемы: два выходных разъема с напряжением 48В/60В (выбор осуществляется через перемычку)
    * Разъемы для вентилятора: два XH2.54 2P VCC FAN


## Примечания:

    * Перед установкой отключите питание и выключите все устройства
    * При выводе 48В максимальный ток двигателя может быть установлен на 2А, а при выводе 60В — на 1А
    * Напряжение на двух разъемах VCC FAN соответствует входному напряжению, выходное напряжение совпадает с входным
    * На модуле предусмотрены места для крепления вентилятора 4010, так как модуль повышения напряжения выделяет значительное количество тепла, необходимо установить вентилятор для охлаждения
    * Встроена защитная микросхема от неправильного подключения и короткого замыкания, предотвращающая повреждение деталей из-за ошибок пользователя.
    * Не вставляйте перемычки напряжения одновременно, чтобы избежать повреждения элементов
    * Подключайте правильную полярность, чтобы избежать повреждения элементов.
