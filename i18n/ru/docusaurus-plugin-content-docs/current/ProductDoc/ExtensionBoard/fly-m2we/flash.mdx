## Подготовка перед записью системы

    * Программное обеспечение для чтения карт
    * Карта памяти TF объемом не менее 16 ГБ или не более 128 ГБ, скорость передачи карточки должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Скачайте необходимые <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструменты записи</Button>

## Запись системы

:::info
M2WE поддерживает три режима, я рекомендую использовать запись системы через карту TF на M2WE или запись через накопитель M2SD на M2WE
:::

:::info
* Если в устройстве установлен M2WE и карта памяти, система по умолчанию будет запускаться с карты памяти
* Для M2WE рекомендуется использовать запись системы через карту TF, запись через USB может завершиться неудачей из-за совместимости драйверов и т.д., рекомендуется использовать прямую запись [карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие загружаемого образа системы, если он не соответствует, система не запустится
:::
