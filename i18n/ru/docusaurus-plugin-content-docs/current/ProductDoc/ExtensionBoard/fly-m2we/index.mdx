## Внимание

:::danger Внимание

* На M2WE предустановлен FLYOS и он готов к запуску сразу после выпуска из производства
* M2WE можно использовать только с продуктами FLY, несовместим с другими устройствами
* Антеночный разъем M2WE — `IPEX1`
* M2WE не поддерживает объединение 2,4ГГц и 5ГГц, Wi-Fi с китайскими символами, специальными символами или скрытым по умолчанию Wi-Fi

:::
