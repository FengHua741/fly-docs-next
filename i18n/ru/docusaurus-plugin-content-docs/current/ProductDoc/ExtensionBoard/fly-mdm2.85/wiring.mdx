# Схема подключения

:::warning Важно
    - Питание может подаваться только напряжением DC5V, при подключении убедитесь в правильности polarity, подключение должно производиться в отключенном состоянии, проверьте правильность подключения перед включением питания; компания не несет ответственности за возможные убытки, вызванные неправильным подключением.
:::
