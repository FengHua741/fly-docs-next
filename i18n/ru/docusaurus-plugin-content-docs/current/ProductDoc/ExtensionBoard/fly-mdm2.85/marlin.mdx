# Настройка Marlin

## Настройка функций обрыва и засорения проволоки

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/marlin.webp').default} size="70%" align="center" />

1. Откройте файл конфигурации Marlin `Configuration.h` и найдите строку `#define FILAMENT_RUNOUT_SENSOR`, уберите перед ней символ комментария.
2. Найдите строку `#define FILAMENT_RUNOUT_DISTANCE_MM`, уберите перед ней символ комментария и установите значение `3.7` (значение можно изменить в зависимости от используемого диска; если происходит ложное срабатывание, увеличьте это значение).
3. Найдите строку `#define FILAMENT_MOTION_SENSOR`, уберите перед ней символ комментария.
4. Для функции датчика проволоки Marlin также необходимо активировать строки `#define NOZZLE_PARK_FEATURE` и `#define ADVANCED_PAUSE_FEATURE`. Удалите символы комментария и настройте конкретные параметры для вашего принтера.
