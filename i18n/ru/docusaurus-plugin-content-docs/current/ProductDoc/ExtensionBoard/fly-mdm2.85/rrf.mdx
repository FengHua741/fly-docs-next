# Настройка RRF

## Настройка функций обнаружения обрыва и затора нити

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} size="70%" align="center" />

1. Откройте веб-интерфейс принтера RRF, перейдите в `Files` -> `System`, найдите файл `config.g` и нажмите на него для открытия и редактирования файла.
2. В файле добавьте следующий код для включения функции обнаружения обрыва нити.

```gcode
M591 D0 P2 C"PA9" S1
```

3. В файле добавьте следующий код для включения функции обнаружения затора нити. (`L` параметр изменяется в зависимости от используемого диска; если происходит ложное срабатывание, увеличьте это значение)


```gcode
M591 D0 P7 C"PA10" L3.7 R75:125 S1
```
