# Краткое описание продукта

    * Главный контроллер использует ESP32-S3R8, который имеет более высокую производительность по сравнению с ESP32. Он оснащен 8 МБ PSRAM и 16 МБ флеш-памятью. Экран — IPS, с высокой четкостью изображения, высокой яркостью, широким углом обзора и细腻ыми цветами. Поддерживает емкостное касание.

## Особенности продукта

    :::tip

    * Всего 6 страниц, листайте влево, вправо, вверх или вниз для переключения между ними

    :::

    * Главная (отображает состояние ожидания, печати, остановки, ошибки и т. д.)

    * Экструзия (управление экструзией, откатом, температурой экструдера)

    * Системные службы (управление запуском, остановкой и перезапуском услуг хоста, klipper, moonraker и т. д.)

    * Настройки (используется для настройки различных параметров halo, wifi, ip, языка, яркости подсветки и т. д.)

    * Информация о устройстве (отображает информацию о halo, версию прошивки. Проверка обновлений и т. д.)

    * Детектирование расходных материалов: детектирование разрыва и затора расходных материалов, измерение длины расходных материалов, автоматическая приостановка и повышение GPIO47 после достижения указанной длины затора (все эти функции по умолчанию выключены и могут быть включены в настройках)

    * Выключение после завершения печати: поддерживает задержку выключения хоста и повышение GPIO48 после завершения печати (все эти функции по умолчанию выключены и могут быть включены в настройках)
