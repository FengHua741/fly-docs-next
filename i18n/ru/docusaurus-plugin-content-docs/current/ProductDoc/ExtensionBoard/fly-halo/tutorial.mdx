# Руководство по использованию

## Важные замечания

    * При первом использовании необходимо подключиться к сети. Если ваш Wi-Fi не может подключиться к интернету, Halo не сможет получить начальные настройки при первом включении
    * Все настройки в меню настроек вступят в силу только после нажатия **Сохранить настройки** и перезагрузки устройства
    * Функция обнаружения заторов по умолчанию отключена; чтобы получить доступ к ней с главной страницы, нужно сделать свайп вверх после её активации в настройках и перезагрузки устройства
    * Функция автоматического выключения при завершении печати по умолчанию отключена; её можно включить в настройках
    * Все кнопки отмены печати или паузы печати в всплывающих окнах требуют длительного нажатия (для предотвращения случайных нажатий)
    * В разделе "Системные службы" Host-сервис является хост-компьютером верхнего уровня
    * При использовании официальной прошивки не рекомендуется пытаться заменить основной контроллер или чип FLASH

    :::tip

    * Важно: если размер файла конфигурации `moonraker.conf` превышает 4 МБ, вы можете столкнуться с проблемами использования. При возникновении аномалий рекомендуется уменьшить размер конфигурационного файла `moonraker.conf`

    :::

## Обновление системной прошивки

    * Обновления версии Halo бывают двух типов: полное обновление и OTA-обновление (частичное)
        * Полное обновление: затрагивает базовые изменения и требует использования USB для записи полной прошивки
        * Частичное обновление: стандартное OTA-обновление; проверку обновлений можно выполнить на странице информации о устройстве

## Использование мобильного сканирования для настройки Wi-Fi

    :::tip

    * Некоторые мобильные устройства могут не поддерживать функцию сканирования и подключения к Wi-Fi через QR-код. В этом случае можно подключиться к точке доступа с названием **FLY-halo-xxxxxxxx** в настройках Wi-Fi телефона

    :::

    :::tip

    * Если после подключения к точке доступа не появляется веб-страница, необходимо вручную открыть ссылку в браузере

    :::

    :::tip

    * Обратите внимание, что HALO может работать только с 2,4 ГГц Wi-Fi

    :::

    :::tip

    * Экран HALO не может обнаружить скрытые точки доступа Wi-Fi

    :::
