# FLY_halo

## 1. Домашняя страница

    Щелкните на горячем столе или конце нагревательного элемента, затем проведите пальцем с экрана сверху по часовой стрелке до нужной температуры и нажмите на горячий стол или конец нагревательного элемента для установки температуры.

    Если нужно отображать температуру MCU и компьютера верхнего уровня, добавьте следующую конфигурацию

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # Связь с MCU (по умолчанию)

    [temperature_sensor host]        # Температура компьютера верхнего уровня
    sensor_type: temperature_host     # Связь с компьютером верхнего уровня
    ```


    :::warning Важные замечания

    * Обратите внимание на максимальную температуру, которая определяется в конфигурации

    :::
