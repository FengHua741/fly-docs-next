# Инструкция по работе с страницей Halo

## Структура страницы

```bash
  Страница печати файлов          Домашняя страница            Страница экструзии            Страница системных служб             Страница настроек              Страница информации о устройстве
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Печать файлов |<-->|     Главная    |<-->|   Экструзия   |<-->| Службы системы |<-->| Настройки   |<-->| Информация об устройстве |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Обнаружение заторов |
                   |             |
                   +-------------+
                     Страница обнаружения заторов
            (Доступна только при включенном обнаружении заторов в настройках)
```

    2. Нажатие: Переход к настройке температуры нагревательного стола, регулировка температуры с помощью круговой полосы, повторное нажатие области 2 подтверждает установленную температуру

## Страница экструзии

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.webp').default} size="50%" align="left" />
    * На рисунках выше, области для взаимодействия соответствуют нижеуказанным номерам
    1. Нажатие: Увеличение целевой температуры экструдера на 1 градус
    2. Долгое нажатие: Переход к настройке температуры экструдера, регулировка температуры с помощью круговой полосы, повторное нажатие подтверждает установленную температуру
    3. Нажатие: Уменьшение целевой температуры экструдера на 1 градус
    4. Нажатие: Экструзия
    5. Вертикальное движение: Выбор длины экструзии или возврата, единица измерения — мм
    6. Нажатие: Возврат
    7. Долгое нажатие: Переход к настройке скорости экструзии на рисунке 2, вертикальное движение для выбора скорости экструзии, единица измерения — мм/с, после выбора нажмите пустую область вокруг для выхода
