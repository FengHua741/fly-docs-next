## Инкрементальное обновление (OTA)

    * В разделе информации о устройстве Halo, в поле версии можно нажать «Проверить обновления». Если доступно обновление, появится уведомление,
        после чего нажмите «Обновить», чтобы начать процесс обновления. Во время обновления не отключайте питание, дождитесь прогресса 100%, затем устройство автоматически перезагрузится.


## Полное обновление (USB)

    * Если публикуется полная версия прошивки для обновления, то для её установки необходимо подключить Halo к компьютеру через USB
    :::tip

    * При полном обновлении все настройки устройства будут сброшены до заводских настроек

    :::

    1. Скачайте и распакуйте инструмент записи USB [flash_download_tool_3.9.4.zip (скачать)](https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip)
    2. Скачайте прошивку [Прошивка Halo](docs/ProductDoc/ExtensionBoard/fly-halo/firmware.mdx)
    3. Откройте ``flash_download_tool_3.9.4.exe``. Примечание: после запуска инструмента записи сначала появится окно командной строки, а затем интерфейс программы. Не закрывайте окно командной строки.
    4. В инструменте записи выберите в поле ``ChipType:`` значение ``ESP32-S3``
    5. В инструменте записи выберите в поле ``WorkMode:`` значение ``Develop``
    6. В инструменте записи выберите в поле ``LoadMode:`` значение ``USB``
    7. Нажмите ``OK``, см. рисунок 1 ниже

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/flash-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/flash-2.webp').default} size="50%" align="left" />

    :::tip

    * На рисунке 2 зеленая выделенная область должна быть точно такой же, как показано на изображении, ее нельзя изменять самостоятельно.

    :::

    8. Подключите Halo к компьютеру через USB-TypeC кабель
    9. Нажмите кнопку «Синего регион 1» на рисунке 2, чтобы выбрать скачанный файл прошивки
    10. Нажмите кнопку «Синего региона 2» на рисунке 2, чтобы выбрать COM-порт Halo (если он недоступен, удерживайте кнопку «B» на обратной стороне Halo и подключите устройство к компьютеру)
    11. Нажмите кнопку «Синего региона 3» на рисунке 2, чтобы начать запись. После завершения записи перезагрузите Halo для завершения полного обновления (не отключайте линию Halo до появления уведомления о завершении записи)
