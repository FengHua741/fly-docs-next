## Руководство по установке

## Установка на Voron Stealthburner

    * Для установки Halo на Stealthburner требуется использовать совместимые аксессуары, предоставляемые нами
    * Сначала скачайте два STL-файла, предоставленных нами, и распечатайте их самостоятельно
        * [Stealthburner_main_body(Adapt_to_Halo).stl (скачать)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl (скачать)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step (скачать)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### Установка переходной платы и соединительного кабеля
    * Положение выхода можно выбрать по своему усмотрению, не обязательно приклеивать к задней части вентилятора
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.webp').default} size="50%" align="left" />


    :::warning Важные замечания

    * Цветовая маркировка выходов: **черный — минус GND**, **красный — плюс VCC (рабочее напряжение DC12-24V)**, **белый — GPIO47 (может использоваться для обнаружения заклинивания)**, **желтый — GPIO48 (может использоваться для остановки после завершения печати)**

    :::
