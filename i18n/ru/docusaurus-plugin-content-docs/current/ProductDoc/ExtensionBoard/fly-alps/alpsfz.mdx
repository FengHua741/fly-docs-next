# Как отрегулировать порог срабатывания модуля

## Открытие плоскостности
    В консоли `Klipper` отправьте команду `SET_PIN PIN=_probe_ready VALUE=1`, наблюдайте за индикатором срабатывания, изменяйте пороговое значение с малого до большего до тех пор, пока индикатор не погаснет. Введите следующую команду:
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```

## Настройка давления
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">Инструмент помощник для последовательного соединения</Button>

    * Подключите модуль к компьютеру с помощью `USB` - кабеля, откройте инструмент помощника для последовательного соединения, выберите соответствующий порт, установите скорость передачи данных 115200 и нажмите "Подключить".
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    В окне последовательного соединения введите следующую команду для настройки порогового значения:
    ```cfg
    set порог_значение
    ```
    Значение по умолчанию: 20000
    Чем меньше число, тем более чувствительным является устройство, чем больше число, тем менее чувствительным.
    * Например: `set 20000`, установите пороговое значение в 20000, чем меньше число, тем более чувствительным является устройство, чем больше число, тем менее чувствительным.
    * Команды для использования в инструменте последовательного соединения:

    * Последовательная отправка `v`, в реальном времени выводятся значения abc. a: до фильтрации; b: после фильтрации.
    * Последовательная отправка `uv`, отключает вывод.
    * Последовательная отправка `rt`, проверяет текущее значение порога.
