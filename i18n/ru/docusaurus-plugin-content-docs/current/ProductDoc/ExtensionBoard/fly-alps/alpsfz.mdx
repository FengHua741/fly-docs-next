# Как отрегулировать порог триггера модуля
<Button variant="contained" disableElevation href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">Получить инструмент помощник для последовательного порта</Button>

Используя инструмент последовательного порта, можно использовать следующую команду:
`Консоль Klipper` отправляет "SET_PIN PIN=_probe_ready VALUE=1", наблюдая за индикатором триггера, настраивайте пороговое значение с малых значений до больших, пока индикатор не перестанет светиться. Введите следующую команду:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
* Подключите модуль к компьютеру с помощью USB-кабеля, откройте инструмент помощника последовательного порта, выберите соответствующий номер порта, скорость波特率为 115200, затем нажмите кнопку "Подключить".

В инструменте последовательного порта введите следующую команду для настройки значения триггера:
```cfg
set значение_триггера
```
Значение триггера по умолчанию: 20000
Чем меньше значение, тем более чувствительным является устройство; чем больше значение, тем менее чувствительным.
* Например: `set 20000`, установка значения триггера в 20000, чем меньше число, тем более чувствительным является устройство; чем больше число, тем менее чувствительным.
* Команды для использования в инструменте последовательного порта:

* Последовательная передача `v`, в реальном времени выводятся значения abc. a: до фильтрации; b: после фильтрации.
* Последовательная передача `uv`, отключает вывод.
* Последовательная передача `rt`, проверяет текущее значение порога.
