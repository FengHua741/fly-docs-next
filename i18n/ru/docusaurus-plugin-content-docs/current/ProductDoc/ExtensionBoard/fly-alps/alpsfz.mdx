# Как отрегулировать порог срабатывания модуля
Используя инструмент последовательного порта, можно использовать следующую команду:
Отправьте в консоль Klipper команду "SET_PIN PIN=_probe_ready VALUE=1", наблюдайте за индикатором срабатывания, изменяйте пороговое значение постепенно от меньшего к большему до тех пор, пока индикатор не перестанет гореть. Введите следующую команду:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
В инструменте последовательного порта введите следующую команду для настройки порогового значения:
```cfg
set пороговое_значение
```
Значение по умолчанию: 20000
* Команды для использования в инструменте последовательного порта:

* Последовательная отправка `v`, для вывода в реальном времени значений adc. a: до фильтрации; b: после фильтрации.
* Последовательная отправка `uv`, для отключения вывода.
* Последовательная отправка `rt`, для просмотра текущего порога.
