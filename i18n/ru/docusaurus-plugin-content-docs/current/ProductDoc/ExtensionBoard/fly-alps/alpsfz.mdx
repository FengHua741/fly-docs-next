# Как отрегулировать порог срабатывания модуля

## Открытие модуля выравнивания
    В консоли `Klipper` отправьте команду `SET_PIN PIN=_probe_ready VALUE=1`, наблюдайте за индикатором срабатывания, регулируйте пороговое значение от малого к большему до тех пор, пока индикатор не погаснет. Введите следующую команду:
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```

## Настройка давления
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">Помощник последовательного соединения</Button>

    * Используйте USB-кабель для подключения модуля к компьютеру, откройте помощника последовательного соединения, выберите соответствующий порт, скорость波特ности 115200, нажмите "Подключить".
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    В последовательном соединении введите следующую команду для настройки порогового значения:
    ```cfg
    set порог_значение
    ```
    Значение по умолчанию: 20000
    Чем меньше значение, тем более чувствительным оно является, чем больше значение, тем менее чувствительным.
    * Например: `set 20000`, установите пороговое значение в 20000, чем меньше значение, тем более чувствительным оно является, чем больше значение, тем менее чувствительным.
    * Команды для использования в последовательном соединении:

    * Посылка команды `v` через последовательный порт выводит в реальном времени значения abc. a: до фильтрации; b: после фильтрации.
    * Посылка команды `uv` через последовательный порт отключает вывод.
    * Посылка команды `rt` через последовательный порт показывает текущее значение порога.
