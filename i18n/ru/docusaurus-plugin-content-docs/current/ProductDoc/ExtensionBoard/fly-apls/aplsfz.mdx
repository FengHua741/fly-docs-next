# Как отрегулировать порог срабатывания модуля
Используя инструмент последовательного порта, можно использовать следующие команды:
В Klipper консоли отправьте "SET_PIN PIN=_probe_ready VALUE=1", наблюдайте за индикатором срабатывания, регулируйте порог от малого к большему до тех пор, пока индикатор не погаснет. Введите следующую команду:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
В инструменте последовательного порта введите следующую команду для настройки порога срабатывания:
```cfg
set порог_срабатывания
```
* Команды инструмента последовательного порта:

* Последовательная передача `v`, реальное время вывода значений adc. a: до фильтрации; b: после фильтрации.
* Последовательная передача `uv`, отключение вывода.
* Последовательная передача `rt`, просмотр текущего порога.
