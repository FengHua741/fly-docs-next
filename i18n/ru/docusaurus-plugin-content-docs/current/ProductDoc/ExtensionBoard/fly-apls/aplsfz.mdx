# Как отрегулировать порог срабатывания модуля
Используя инструмент последовательного порта, можно использовать следующую команду:
В консоль Klipper отправьте "SET_PIN PIN=_probe_ready VALUE=1", наблюдая за индикатором срабатывания, увеличивайте порог постепенно от малого значения до тех пор, пока индикатор не погаснет. Введите следующую команду:
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
В инструменте последовательного порта введите следующую команду для настройки порога срабатывания:
```cfg
set порог_срабатывания
```
Значение порога по умолчанию: 20000
* Команды для инструмента последовательного порта:

* Последовательно отправьте `v`, чтобы в реальном времени выводить значения adc. a: до фильтрации; b: после фильтрации.
* Последовательно отправьте `uv`, чтобы отключить вывод.
* Последовательно отправьте `rt`, чтобы проверить текущее значение порога.
