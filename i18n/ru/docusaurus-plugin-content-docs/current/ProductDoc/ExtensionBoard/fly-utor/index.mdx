# FLY-UTOR

    FLY-UTOR представляет собой высокопроизводительное, многофункциональное решение для расширения интерфейсов, предназначенное для предоставления пользователям дополнительных портов USB и возможностей расширения питания, а также поддержки передачи данных UART в RS232. Расширяемая плата имеет компактный дизайн и легко устанавливается, что делает её подходящей для различных сценариев применения, где требуется дополнительное количество интерфейсов и источников питания.

## Основные характеристики

    * Двойной расширенный порт USB: Плата расширения предоставляет два дополнительных порта USB, поддерживающих стандарт USB 2.0, что обеспечивает совместимость с различными устройствами USB и высокоскоростную передачу данных.
    * Поддержка преобразования UART в RS232: Встроенный преобразователь UART в RS232 позволяет пользователю подключать последовательные устройства к системе, расширяя возможности коммуникации устройств.
    * Двойное расширение портов питания: Предоставляются два порта питания, позволяющие одновременно расширять два внешних источника питания для удовлетворения дополнительных потребностей в питании.
    * Стабильная система питания: Плата расширения оснащена стабильной цепью питания, гарантирующей стабильную работу подключенных устройств.
    * Легкость установки: Имеет стандартные размеры, что позволяет использовать её в различных средах без необходимости установки дополнительного драйвера.
    * Компактные размеры: Размер платы удобен и не занимает слишком много места, что облегчает интеграцию в различные проекты.

## Технические характеристики:

    * Доступные интерфейсы: USB X2, UART to RS232 X2, порты питания X2
    * Порты питания: 12В/24В расширение
    * Совместимость: хост-устройства с интерфейсом USB

## Примечания:

    * Перед установкой отключите питание и выключите все устройства.
    * Вставьте многофункциональную плату расширения USB в разъем USB материнской платы или другой разъем расширения USB.
    * Подключите необходимые USB-устройства и/или источники питания к плате расширения.
    * Для использования функции преобразования UART в RS232 подключите соответствующие последовательные устройства.
    * Подключите питание снова и запустите устройство.
    * Система должна автоматически обнаружить расширяемую плату и выполнить настройку. При необходимости обратитесь к разделу "Примечания по конфигурации после операции".
