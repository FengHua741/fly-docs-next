# FLY-UTOR

    FLY-UTOR представляет собой высокопроизводительное и универсальное решение для расширения интерфейсов, предназначенное для предоставления дополнительных портов USB и возможностей расширения питания, а также поддерживающее преобразование UART в RS232. Данный расширительный модуль имеет компактный дизайн и прост в установке, что делает его подходящим решением для различных сценариев применения, где требуется дополнительный интерфейс и выход питания.

## Основные характеристики

    * Расширение двух портов USB: на плате предусмотрено два дополнительных порта USB, поддерживающих стандарт USB 2.0, что обеспечивает совместимость с различными устройствами USB и высокоскоростную передачу данных.
    * Поддержка преобразования UART в RS232: встроенный преобразователь UART в RS232 позволяет пользователю подключать последовательные устройства к системе, расширяя возможности коммуникации оборудования.
    * Расширение двух портов питания: предоставляются два порта питания, позволяющие одновременно расширить два внешних источника питания, чтобы удовлетворить дополнительные потребности в энергии.
    * Стабильная схема питания: на плате разработана стабильная схема питания, гарантирующая стабильную работу подключенных устройств.
    * Простота установки: стандартный размер платы позволяет использовать её в разных средах без необходимости установки дополнительного программного обеспечения.
    * Компактные размеры: плата имеет оптимальные размеры, не занимая много места, что облегчает интеграцию в различные проекты.

## Технические характеристики:

    * Доступные интерфейсы: USB X2, Преобразование UART в RS232 X2, Порты питания X2
    * Порты питания: 12В/24В расширение
    * Совместимость: компьютеры с портом USB

## Примечания:

    * Перед установкой отключите питание и выключите все устройства.
    * Вставьте мультисистемный USB-адаптер в порт USB материнской платы или другой USB-порт расширения.
    * Подключите необходимые устройства USB и/или источники питания к расширительному модулю.
    * Для использования функции преобразования UART в RS232 подключите соответствующее последовательное устройство.
    * После этого подключите питание и запустите устройство.
    * Система должна автоматически обнаружить расширительный модуль и произвести настройку. Если это необходимо, обратитесь к разделу с рекомендациями по настройке после установки.
