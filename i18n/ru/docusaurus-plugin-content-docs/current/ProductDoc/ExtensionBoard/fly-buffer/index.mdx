# Введение

# Использование буферов LLL

    Буфер для расходных материалов LLL разработан специально для удаленной подачи нити на 3D-принтер и предназначен для упрощения процесса загрузки и выгрузки нити. Этот буфер оснащен двумя кнопками, которые предназначены для подачи и возврата нити, позволяя пользователю выполнять эти операции электрически.

### Функция буфера:

    Когда экструдер 3D-принтера подает нить, если возникает сопротивление при拉动 к дальнему источнику нити, буфер обнаруживает это действие и автоматически подает нить в направлении нагревательного элемента. При каждом таком действии буфер повторяет эту операцию подачи нити. Это позволяет эффективно снижать трение экструдера о нить, сохраняя стабильное давление экструдера, что в свою очередь повышает стабильность качества печати.

### Функция обнаружения обрыва нити:

    Встроенный детектор обрыва нити буфера отправляет сигнал 3D-принтеру для приостановки печати при обнаружении отсутствия нити в буфере. Для использования этой функции необходимо подключить сигнальные выводы и изменить конфигурацию 3D-принтера. Однако функция буфера может использоваться без изменения конфигурации. 

    С помощью вышеперечисленных функциональных возможностей устройство LLL не только упрощает работу с нитями, но и повышает стабильность процесса печати и качество печати.

* Обратите внимание, что перед использованием функций необходимо подключить нить, иначе можно использовать кнопки для управления двигателем.

## Конфигурационный пример

    ```cfg
    [filament_switch_sensor Обнаружение обрыва нити]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 Недостаток нити
    insert_gcode:
            M118 Загрузка нити
    ```
