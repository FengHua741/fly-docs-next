# BDsensor

    Датчик расстояния BDsensor представляет собой устройство, которое может работать в паре с 3D-принтером для выполнения быстрой настройки уровня стола и регулировки оси Z в реальном времени.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.webp').default} size="50%" align="left" />

## Функции

    * Замена механического ограничителя оси Z
    * Во время сканирования не требуется подъем оси Z, что повышает скорость печати
    * Сканирование по X и Y позволяет проводить плотное детектирование поверхности стола



## Подключение

    * Питание: 5В
    * Фирменное обеспечение: Klipper/Marlin
    * Подключение: инструментальная панель/основная плата (BDsensor будет использовать один канал сервопривода и канал ограничителя)
    * Теплый стол: обычная поверхность из PEI, алюминиевый стол с нагревом и другие металлические поверхности, поддерживающие нагрев, не поддерживаются магнитные поверхности с отверстиями и неметаллические поверхности


## Примечания

    * Для работы с Klipper необходимо загрузить программное обеспечение klipper и последнюю правку для BD-SENSOR, а также перепрошить основную плату
    * При настройке держите экстренное остановку в доступном месте, чтобы избежать царапин на PEI
    * Рекомендуемая скорость сканирования стола — 200!
    * Скорость возврата оси Z не рекомендуется превышать 5
