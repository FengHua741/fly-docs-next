# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />



## Особенности продукта

    * Подходит для мест с 36-моторами
    * Может подключаться к Raspberry Pi, FLY PI, Orange Pi, материнским платам и другим устройствам через интерфейс SPI

## Технические характеристики

    * Интерфейс на плате: ZH1.5 6P разъем
    * Способ связи: SPI
    * Рабочее напряжение: 5В

## Важные замечания

    :::warning Важно

    * Рабочее напряжение 5В, не подключайте неправильно
    * Подходят различные прошивки, убедитесь, что они могут использоваться с подключением SPI

    :::
