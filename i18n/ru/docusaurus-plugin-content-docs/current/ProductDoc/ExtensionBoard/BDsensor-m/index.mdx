# BDsensor-m

    BD-сенсор - это датчик расстояния, который можно использовать вместе с 3D-принтером для быстрой настройки уровня платформы и корректировки оси Z в реальном времени.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## Функции
    * Установка и обзор
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Замена концевика оси Z
    * Сканирование без поднятия или опускания оси Z, что повышает скорость печати
    * Сканирование с перемещением по осям XY, возможность плотного сканирования поверхности
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## Подключение

    * Питание: 5V
    * Прошивка: Klipper/Marlin
    * Подключение: к материнской плате (BDsensor будет использовать интерфейс exp1 материнской платы)
    * Нагревательная платформа: поддерживает: обычная PEI, алюминиевые платы и другие металлические поверхности, не поддерживает магнитные перфорированные платформы и неметаллические поверхности


## Важные замечания

    * Для Klipper необходимо скачать klipper и последний патч BD-SENSOR-m, а также прошить заново прошивку материнской платы
    * При отладке убедитесь, что кнопка аварийной остановки находится в легкодоступном месте, чтобы избежать повреждения PEI
    * Рекомендуемая скорость сканирования платформы - 200!
    * Скорость возврата оси Z не должна превышать 5
