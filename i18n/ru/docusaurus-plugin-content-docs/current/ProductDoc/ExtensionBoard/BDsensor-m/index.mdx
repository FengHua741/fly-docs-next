# BDsensor-m

    BD сенсор - это датчик расстояния, который может использоваться вместе с 3D-принтером для быстрого выравнивания платформы и регулировки оси Z в реальном времени.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## Функции
    * Установка и введение
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Замена концевого выключателя оси Z
    * Нет необходимости поднимать или опускать ось Z при сканировании, что повышает скорость печати
    * Сканирование при перемещении по осям XY, возможность плотного сканирования сетки платформы
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## Подключение

    * Питание: 5V
    * Прошивка: Klipper/Marlin
    * Подключение: Материнская плата (BDsensor будет использовать интерфейс exp1 на материнской плате)
    * Тепловая платформа: Поддерживаются обычная PEI, алюминиевые тепловые платформы и другие металлические поверхности, не поддерживаются магнитные платформы с отверстиями и неметаллические поверхности


## Предостережения

    * Для Klipper необходимо скачать klipper и последний патч BD-SENSOR-m, а также перепрошить прошивку материнской платы
    * При настройке держите аварийный стоп в легкодоступном месте, чтобы избежать царапин на PEI
    * Рекомендуемая скорость сканирования платформы - 200!
    * Скорость возвращения оси Z не должна превышать 5
