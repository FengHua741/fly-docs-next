# BDsensor-m

    Датчик BD - это датчик расстояния, который можно использовать вместе с 3D-принтером для быстрой настройки уровня стола и регулировки оси Z в реальном времени.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.webp').default} size="50%" align="left" />

## Функции

    * Замена концевого выключателя оси Z
    * Сканирование без поднятия и опускания оси Z, повышение скорости печати
    * Сканирование при перемещении по осям XY, возможность плотного сканирования поверхности стола

## Подключение

    * Питание: 5В
    * Прошивка: Klipper/Marlin
    * Подключение: материнская плата (BDsensor будет использовать интерфейс exp1 материнской платы)
    * Стол: поддерживаются: обычный PEI, алюминиевый стол и другие металлические поверхности стола, не поддерживаются магнитные столы с отверстиями и неметаллические поверхности столов


## Внимание

    * Для Klipper требуется скачать klipper и последний патч BD-SENSOR-m, а также перепрошить прошивку материнской платы
    * При отладке держите аварийный выключатель в доступном месте, чтобы избежать царапин на PEI
    * Рекомендуемая скорость сканирования стола - 200!
    * Скорость возврата оси Z не должна превышать 5
