# BDsensor-m

    Датчик расстояния BD - это датчик, который можно использовать совместно с 3D-принтером для реального времени быстрой выравнивания стола и настройки z-оси.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## Функции
    * Инструкция по установке
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Замена ограничителя z-оси
    * При сканировании не требуется подъем z-оси, что повышает скорость печати
    * Сканирование в движении по XY-плоскости позволяет проводить интенсивное детектирование поверхности стола
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## Подключение

    * Питание: 5В
    * Фirmware: Klipper/Marlin
    * Подключение: материнская плата (BDsensor будет использовать разъем exp1 материнской платы)
    * Теплоотвод: поддерживает обычный PEI, алюминиевые панели и другие металлические поверхности, но не поддерживает магнитные панели с отверстиями и неметаллические поверхности

## Примечания

    * Для Klipper необходимо загрузить klipper и последнюю версию патча BD-SENSOR-m и перепрошить прошивку материнской платы
    * Во время настройки убедитесь, что аварийная остановка находится там, где ее можно легко активировать, чтобы избежать царапин на PEI
    * Рекомендуемая скорость сканирования стола: 200
    * Скорость возврата z-оси не рекомендуется превышать 5
