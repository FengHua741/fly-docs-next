# BDsensor-m

    BDsensor - это датчик расстояния, который можно использовать с 3D-принтером для быстрой и автоматической настройки уровня стола и регулировки оси Z в реальном времени.
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/bdsensonm.webp').default} size="100%" align="left" />

## Функции
    * Введение в установку
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/js_webp.webp').default} size="100%" align="left" />
    * Замена ограничителя оси Z
    * Сканирование без необходимости подъема и опускания оси Z, увеличение скорости печати
    * Перемещение по осям XY для сканирования, возможность плотного сканирования стола
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/BDsensor-m/img/z.webp').default} size="100%" align="left" />

## Подключение

    * Питание: 5V
    * Прошивка: Klipper/Marlin
    * Подключение: к материнской плате (BDsensor использует интерфейс exp1 на материнской плате)
    * Нагревательный стол: Поддерживается: обычный PEI, алюминиевый нагревательный стол и другие металлические поверхности, не поддерживаются: магнитные столы с отверстиями и неметаллические поверхности


## Меры предосторожности

    * Для Klipper необходимо загрузить последние версии klipper и патч BD-SENSOR-m, а также перепрошить прошивку материнской платы
    * Во время настройки держите аварийный стоп в доступном месте, чтобы избежать царапин на PEI
    * Рекомендуемая скорость сканирования стола - 200!
    * Не рекомендуется превышать скорость возврата оси Z более 5
