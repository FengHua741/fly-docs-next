# Введение

# Использование буферов LLL PLUS

    Буфер для расходных материалов LLL разработан специально для удаленной подачи нити в 3D-принтерах и предназначен для упрощения процесса загрузки и выгрузки нити. Этот буфер оснащен двумя кнопками, одна из которых используется для подачи материала, а другая — для его возврата, позволяя пользователям выполнять эти операции электрическим способом.

### Функция буферизации:

    Когда экструдер 3D-принтера подает материал, и если возникает тянущее действие на удаленном стойке с материалом, буфер обнаруживает это движение и автоматически подает нить в сторону нагревательного элемента. При каждом таком действии буфер повторяет этот процесс подачи материала. Это позволяет значительно снизить воздействие на материал со стороны экструдера, сохраняя стабильное давление экструдера, что способствует повышению стабильности качества печати.

### Функция обнаружения отсутствия материала:

    Встроенный датчик обнаружения отсутствия материала в буфере отправляет сигнал принтеру при обнаружении отсутствия материала, что приводит к остановке печати. Для использования этой функции необходимо подключить сигнальные выводы и изменить конфигурацию принтера. Однако функция буферизации может использоваться без изменения конфигурации.

    Благодаря данным функциям буфер для расходных материалов LLL не только упрощает работу с материалами, но и повышает стабильность процесса печати и качество печати.

* Обратите внимание, что перед использованием функций необходимо подключить материал, иначе можно использовать только кнопки для управления моторами

## Пример конфигурации

    ```cfg
    [filament_switch_sensor Обнаружение отсутствия материала]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 Отсутствие материала
    insert_gcode:
            M118 Подача материала
    ```
