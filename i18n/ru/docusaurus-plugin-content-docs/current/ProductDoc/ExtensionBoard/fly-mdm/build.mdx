# Скачивание модели и её сборка

### Описание файлов модели

| Название файла модели | Описание |
| :---: | :--- |
| pcb_cover.stp | Покрытие платы |
| switch_sensor_trigger_block.stp | Триггерный блок обнаружения обрыва материала |
| rear_cover.stp | Задняя крышка |
| upper_cover.stp | Верхняя крышка |
| main_body.stp | Основной корпус |
| encoder_3_5.8mm.stp | Кодовый диск энкодера, 3 полосы, минимальная чувствительная дистанция 5.8 мм <br/> `Для высокопроизводительной печати (более 20 мм³/с) рекомендуется использовать этот вариант` |
| encoder_5_3.5mm.stp | Кодовый диск энкодера, 5 полос, минимальная чувствительная дистанция 3.5 мм <br/> `Подходит для средней скорости, рекомендуется использовать` |
| encoder_10_1.73mm.stp | Кодовый диск энкодера, 10 полос, минимальная чувствительная дистанция 1.73 мм <br/> `Для низкоскоростной точной печати рекомендуется использовать этот вариант` <br/> Ограничен размером выборки firmware принтера при обнаружении расходного материала |


## Сборка

:::warning Важно при печати модели

- Корпус не рекомендуется печатать из `PETG` и прозрачных материалов, так как это может повлиять на работу датчика.

:::
