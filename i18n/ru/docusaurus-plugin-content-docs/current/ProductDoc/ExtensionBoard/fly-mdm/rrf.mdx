# Настройка RRF

## Настройка функций обнаружения обрыва и затора материала

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} size="70%" align="center" />

1. Откройте веб-интерфейс принтера RRF, перейдите в `Файлы` -> `Система`, найдите файл `config.g` и нажмите кнопку для его открытия и редактирования.
2. В файле добавьте следующий код для включения функции обнаружения обрыва материала.

```gcode
M591 D0 P2 C"PA9" S1
```

3. В файле добавьте следующий код для включения функции обнаружения затора материала. (`L` параметр изменяется в зависимости от используемого дискового механизма; если возникают ложные срабатывания, увеличьте это значение постепенно)


```gcode
M591 D0 P7 C"PA10" L3.5 R75:125 S1
```
