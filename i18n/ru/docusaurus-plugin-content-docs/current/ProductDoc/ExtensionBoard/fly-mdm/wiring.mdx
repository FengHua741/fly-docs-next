# Схема подключения

:::warning Важно
    - Питание может подаваться только напряжением DC5V. При подключении убедитесь в правильности полярности подключения, а также выполняйте подключение в отключенном состоянии источника питания. Подтвердите правильность подключения перед включением питания; любые убытки, вызванные ошибками подключения, не покрываются нашей компанией.
:::
