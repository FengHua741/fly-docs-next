# Настройка Marlin

## Настройка функций обрыва и затора проволоки

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/marlin.webp').default} size="70%" align="center" />

1. Откройте файл конфигурации Marlin `Configuration.h` и удалите комментарий с строки `#define FILAMENT_RUNOUT_SENSOR`.
2. Найдите строку `#define FILAMENT_RUNOUT_DISTANCE_MM`, удалите комментарий и установите значение `3.5` (значение можно изменить в зависимости от используемого диска; если происходит ложное срабатывание, увеличьте это значение).
3. Найдите строку `#define FILAMENT_MOTION_SENSOR` и удалите комментарий.
4. Для работы датчика проволоки в Marlin также необходимо активировать функции `#define NOZZLE_PARK_FEATURE` и `#define ADVANCED_PAUSE_FEATURE`. Удалите комментарии и настройте параметры под ваш принтер.
