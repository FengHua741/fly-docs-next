# Расширение функционала драйвера для управления светодиодами через плагины

Для установки и настройки необходимо подключиться к хост-компьютеру по SSH в соответствии с приведенной ниже ссылкой:

```
https://github.com/julianschill/klipper-led_effect
```

## Установка

Подключитесь к вашему хост-компьютеру через **MobaXterm_Personal** или другую **SSH-утилиту** по Wi-Fi и убедитесь в следующем:

1. **Убедитесь, что на хост-компьютере установленыfirmware Klipper**
2. **Убедитесь, что используемый пользователь уже установил Klipper**
3. **Убедитесь, что раскладка клавиатуры в английском языке**
4. **Убедитесь, что хост-компьютер может нормально обнаруживать устройство**
5. **Убедитесь, что все вышеперечисленные пункты выполнены, иначе дальнейшие шаги невозможны**

Модуль можно установить в существующую установку Klipper с помощью установочного скрипта.

```
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Пример конфигурации

```
[neopixel TP_led]
pin: PB7
chain_count: 25
# Количество светодиодов
# Число RGB-точек
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)
```
