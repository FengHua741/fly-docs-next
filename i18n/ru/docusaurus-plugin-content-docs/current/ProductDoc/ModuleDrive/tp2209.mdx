# TP 2209

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/tp2209.webp').default} size="60%" align="center" />


## Основные параметры

    * Напряжение входа: 12V-24V
    * Подходит для: прошивок Marlin/Klipper/RRF
    * Моторы практически не теряют шаги
    * Поддерживает тихий режим работы
    * Большая площадь вентиляционного подогревателя, снижает температуру работы драйвера
    * Режимы работы драйвера: TMC: UART
    * Поддерживает обнаружение застревания двигателя


## Настройка платы FLY через перемычки

    :::tip

    * Для TP2209 используется режим UART
    * Для TP2209 не требуется накладывать перемычки на драйвер, достаточно только конфигурации
    
    :::


## Ссылка на рекомендуемую конфигурацию TP2209

    :::danger

    Значение шунтового резистора драйвера равно 0.110, пожалуйста, не устанавливайте его ошибочно

    :::

    ```cfg
    [tmc2209 stepper_x]
    uart_pin: 
    interpolate: False
    run_current: 0.8                    # Значение тока при работе двигателя (единицы: мА)
    sense_resistor: 0.110               # Шунтовый резистор драйвера не изменять
    stealthchop_threshold: 0
    ```



## Инструкция по использованию без концевиков

    <DIAG />

    :::tip Совет по использованию без концевиков

    * Исходную строку `endstop_pin:` необходимо отключить или удалить, а добавить `endstop_pin: tmc2209_stepper_x:virtual_endstop`
    * После настройки `diag_pin:` состояние концевика будет показано как не активированное
    * Для настройки `driver_SGTHRS:` необходимо провести самостоятельное тестирование подходящего значения

    ::


    * Пример конфигурации

    ```cfg
    [stepper_x]
    # endstop_pin:PF3
    endstop_pin: tmc2209_stepper_x:virtual_endstop
    homing_retract_dist: 0        # При отступлении назад не изменять значение 0, иначе может произойти сбой при возврате домой

    [tmc2209 stepper_x]
    diag_pin: ^
    driver_SGTHRS: 100
    
    ```
