# Введение

    * Раздельный дизайн, поддерживает различные модули DSI и HDMI
    * Различные размеры, 4,3 дюйма, 4 дюйма, 7 дюймов
    * Высокое разрешение, не пропускает ни одной детали
    * Поддерживает регулировку яркости на 10 уровнях
    * Поддерживает поворот экрана на 180°

:::warning Примечание
Для экранов семи дюймов неприменимо использование модуля HDMI, модуль DSI работает нормально
:::
