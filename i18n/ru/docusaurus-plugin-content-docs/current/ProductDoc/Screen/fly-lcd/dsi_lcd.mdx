## Краткое описание продукта

    * Размер экрана: 120.7мм*76.0мм
    * Площадь экрана: 108мм*66.2мм
    * Разрешение: 800*480
    * Тип экрана: IPS
    * Способ коммуникации: DSI/Mipi
    * Интерфейс на плате: 15-pin DSI / 22-pin Mipi
    * Рабочая температура: 0°-60°C
    * Тип сенсора: Capacitive (емкостный)
    * Логическое напряжение: 3.3V
    * Вес: 135.5г

        :::danger Необходимо учесть

        * Интерфейсы DSI и CSI на Raspberry Pi 5 теперь называются MIPI
        * Интерфейс MIPI может быть настроен для использования как с камерой CSI, так и с экраном DSI
        * Обратите внимание на направление контактов

        :::

        * На Raspberry Pi 5 сейчас два интерфейса MIPI, если посмотреть спереди можно увидеть маркировку 1 и 0

        * Подключение экрана DSI


            * Если используется порт 1, необходимо выполнить следующую конфигурацию экрана

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * Если используется порт 0, необходимо выполнить следующую конфигурацию экрана

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
