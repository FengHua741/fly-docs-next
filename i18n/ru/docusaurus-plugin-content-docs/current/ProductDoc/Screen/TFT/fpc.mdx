# Руководство по изменению параметров экрана
:::tip
Откройте ваш IP:9999 в браузере, чтобы получить доступ к FLY-Tools
:::

:::
    * Для использования TFT необходимо включить опцию ниже, чтобы активировать экран
    * После входа в FLY Tools
* На первом шаге необходимо войти в fly_TOOLS для изменения параметров

* Вам нужно настроить экран в FLY TOOLS и включить KS
