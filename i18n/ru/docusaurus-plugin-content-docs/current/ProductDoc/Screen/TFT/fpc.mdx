# Руководство по изменению параметров экрана
:::tip
Откройте в браузере ваш IP:9999, чтобы получить доступ к FLY-Tools
:::

:::
    * Для использования tft необходимо включить нижние опции, чтобы экран работал
    * После входа в FLY Tools
* Первым шагом нужно зайти в fly_TOOLS и изменить параметры

* В FLY TOOLS необходимо настроить экран и включить KS согласно инструкции
