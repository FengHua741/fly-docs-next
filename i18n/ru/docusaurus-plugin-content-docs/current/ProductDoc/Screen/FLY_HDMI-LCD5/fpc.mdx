# Руководство по изменению параметров экрана
:::tip
Откройте браузер по вашему IP:9999 для доступа к FLY-Tools
:::

:::tip
Интерфейс HDMI на плате FLY является Micro HDMI, обратите внимание при покупке кабеля HDMI!!!
:::

:::tip
FLY поддерживает однокабельное подключение FPC для моделей `FLY-PI-V2`, `FLY-C8` и `FLY-PRO-X10` или `FLY_LITE2/2.1`
:::
    * Чтобы включить экран, необходимо активировать соответствующую опцию
    * После входа в FLY Tools
* Первым шагом нужно войти в fly_TOOLS для изменения параметров
