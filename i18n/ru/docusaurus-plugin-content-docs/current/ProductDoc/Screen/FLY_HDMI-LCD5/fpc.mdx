# Инструкция по изменению параметров экрана
:::tip
Откройте браузер и введите ваш IP:9999 для доступа к FLY-Tools
:::

:::tip
На борту верхнего уровня FLY интерфейс HDMI представляет собой Micro HDMI, обратите внимание при покупке HDMI-кабеля!!!
:::

:::tip
Верхний уровень FLY поддерживает технологию FPC One-Line для следующих моделей: `FLY-PI-V2`, `FLY-C8` и `FLY-PRO-X10` или `FLY_LITE2/2.1`

:::
    * Для использования HDMI необходимо включить нижеследующие параметры, чтобы активировать экран
    * После входа в FLY Tools
* Первым шагом необходимо зайти в fly_TOOLS для изменения параметров
