# Руководство по изменению параметров экрана
:::tip
Откройте браузер по вашему IP:9999, чтобы получить доступ к FLY-Tools
:::

:::tip
Интерфейс HDMI на плате FLY верхнего уровня является Micro HDMI, пожалуйста, обратите внимание при покупке HDMI кабеля!!!
:::

:::tip
Верхнеуровневая плата FLY поддерживает FPC с одной линией для `FLY-PI-V2`, `FLY-C8` и `FLY-PRO-X10` или `FLY_LITE2/2.1`

:::
    * Для использования HDMI нужно включить нижеследующую опцию, чтобы активировать экран
    * После входа в FLY Tools
* Первый шаг - войти в fly_TOOLS и изменить параметры
