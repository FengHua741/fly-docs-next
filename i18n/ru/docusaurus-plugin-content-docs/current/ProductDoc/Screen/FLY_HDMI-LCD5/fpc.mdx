# Руководство по изменению параметров экрана
:::tip
Откройте ваш IP:9999 в браузере, чтобы получить доступ к FLY-Tools
:::

:::tip
Интерфейс HDMI на верхней панели FLY — это Micro HDMI, пожалуйста, обратите внимание при покупке HDMI кабеля!!!
:::

:::tip
Верхняя панель FLY поддерживает FPC подключение для `FLY-PI-V2`, `FLY-C8` и `FLY-PRO-X10` или `FLY_LITE2/2.1`

:::
    * Для использования HDMI необходимо включить соответствующий параметр, чтобы активировать экран
    * После входа в FLY Tools
* На первом шаге необходимо войти в fly_TOOLS для изменения параметров
