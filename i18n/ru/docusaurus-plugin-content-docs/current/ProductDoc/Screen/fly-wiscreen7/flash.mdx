# Изменение заставки загрузки
:::tip
    Пожалуйста, строго соблюдайте `разрешение` и `имя файла` при создании изображений, иначе система не сможет их распознать.

    Готовые изображения поместите в корневую директорию SD-карты/флешки и, находясь в состоянии загрузки экрана, вставьте экран и нажмите кнопку обновления.
:::
## Изменение логотипа загрузки

    * Требуемое разрешение изображения `1024*600`
    * Требуемое имя файла `boot_logo.JPG`
    * Размер не должен превышать `105KB`



## Изменение изображения первого интерфейса

    * Требуемое разрешение изображения `736*563`
    * Требуемое имя файла `print.png`
    * Размер не должен превышать `45KB`



## Изменение изображения интерфейса печати

    * Требуемое разрешение изображения `355*565`
    * Требуемое имя файла `print_smail.png`
    * Размер не должен превышать `45KB`
