## Настройка Klipper

### Настройка светодиодов

    ```cfg
    [led LED]
    white_pin:PA15
    ```

### Настройка PTC

    ```cfg
    [heater_generic warehouse]
    heater_pin:E_CS
    sensor_type:ATC Semitec 104GT-2
    sensor_pin:PA1
    min_temp:-270
    max_temp:600
    control = pid
    pid_kp = 16.804
    pid_ki = 0.794
    pid_kd = 88.854
    ```

### Настройка вентилятора

    ```cfg
    [fan_generic fan1]
    pin: FAN1
    ```


    ```cfg
    [fan_generic fan2]
    pin: FAN2
    ```

 

### Пользовательские G-коды

    * При выдвижении меню появляются пользовательские G-коды,
    * Одна кнопка отправляет несколько команд в формате G-кода:
    * Пример:

    ```cfg
    G91 
    G1 X10 F6000
    G90
    ```

    * Ввод с клавиатуры выглядит так (начиная со второй строки, каждая команда начинается с \n):

    ```cfg
    G91\nG1 X10 F6000\nG90
    ```



## Использование объектов исключения

    * Необходимо заменить moonraker

    * Сначала подключитесь через SSH и убедитесь, что есть доступ к интернету

    1. Сделайте резервную копию moonraker

    ```bash
    mv ~/moonraker ~/moonraker-bak
    ```

    2. Остановите службу moonraker

    ```bash
    sudo systemctl restart moonraker
    ```

    3. Склонируйте moonraker

    ```bash
    git clone https://e.coding.net/g-ofpa1390/3D-Printers/moonraker-dev.git -b flylcd-dev moonraker
    ```

    4. В браузере найдите файл `moonraker.conf` на верхнем уровне системы и добавьте его, затем перезагрузите систему

    ```bash
    [network]
    ```

    5. Затем в браузере введите следующий адрес, где IP нужно заменить на IP верхнего уровня

    ```bash
    http://ip:7125/server/network/wifi/scan
    ```
