##   Обновление
    * 1. Переименуйте скачанный файл в `update.img`
    * 2. Поместите файл `update.img` на SD-карту/USB-накопитель и вставьте его в экран, можно вставить во время работы экрана.
    * 3. Выберите экран и нажмите обновить для завершения обновления
