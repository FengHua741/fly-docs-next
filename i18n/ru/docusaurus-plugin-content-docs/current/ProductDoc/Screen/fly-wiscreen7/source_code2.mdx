## Настройка функции кнопки
    <ImageView image={require('@site/docs/ProductDoc/Screen/fly-wiscreen7/img/ide10.gif').default} size="100%" align="left" />

* Используйте функцию для отправки одного командного файла G-code  
    ```cfg
    gcode_send2("","G28");
    ```
* Используйте функцию для одновременной отправки нескольких команд G-code:
    ```cfg
    gcode_send2("","G91\\nG1 X10 F3000\\nG90");
    ```
* Или используйте такой метод для одновременной отправки нескольких команд G-code:
    ```cfg
    gcode_send2("","G91");
    gcode_send2("","G1 X10 F3000");
    gcode_send2("","G90");
    ```
