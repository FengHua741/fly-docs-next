# Введение

      Fly_WiScreen — это премиальная панель управления для 3D-принтеров на базе Fly3D от команды разработчиков материнской платы компании Guangzhou Meilun Electronic Technology Co., Ltd. Панель позволяет управлять 3D-принтером по Wi-Fi или через проводное подключение, поддерживает передачу файлов печати через SD-карту и флешку, обеспечивая быстрый отклик и отличный用户体验. Панель поддерживает режимы главного и вспомогательного устройства для синхронизации работы нескольких принтеров, что облегчает управление устройствами. Встроенный порт питания 5В можно использовать для подключения модуля зарядки для организации мобильной версии.

## Функциональные возможности

    * Поддержка подключения к другим компьютерам с установленным Klipper через Wi-Fi или Ethernet
    * Поддержка поиска доступных компьютеров после подключения к Wi-Fi
    * Поддержка управления несколькими принтерами одновременно
    * Возможность загрузки файлов печати через USB и TF-карту
    * Настройка макросов для удобства управления и отладки

## Технические характеристики

    * Процессор: Allwinner T113-S3, двуядерный Cortex-A7 с максимальной частотой 1.2 ГГц
    * Оперативная память: интегрирована 128 МБ DDR3
    * Память: 128 МБ NAND Flash, слот для TF-карты
    * Экран: 1024x800 емкостный экран
    * Внешние интерфейсы: один порт USB 2.0, один слот для TF-карты
    * Протокол Wi-Fi: 802.11b/g/n
    * Тип экрана: IPS
    * Размер экрана: 7 дюймов
    * Поддержка multitouch: есть
    * Напряжение питания: 5 В
