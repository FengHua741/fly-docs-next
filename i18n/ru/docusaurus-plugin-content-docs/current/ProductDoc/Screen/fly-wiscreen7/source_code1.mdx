# Инструкция по структуре проекта

## Введение
#### Папка Logic
* 99% кода программирования пользовательского интерфейса находится в папке Logic, остальные папки можно игнорировать
* gcodeLogic.cc похожа на main.c, 99% кода пользовательского интерфейса находится здесь, другие файлы не требуют внимания

#### Папка i18n
* Хранит таблицы перевода на разные языки

#### Папка resources
* Хранит ресурсы изображений, используемых в пользовательском интерфейсе

#### Папка ui
* Хранит файлы интерфейса пользовательского экрана, при двойном клике открывается
* При двойном клике на стрелку внизу слева, показывающую "глаз", можно открыть скрытый интерфейс
