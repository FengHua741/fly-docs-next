# Изменение заставки загрузки
:::tip
    Пожалуйста, строго соблюдайте `разрешение` и `имя файла` при создании изображения, иначе система не сможет его распознать.

    Готовое изображение поместите в корень SD-карты/флешки и, находясь в состоянии загрузки экрана, вставьте экран и нажмите кнопку обновления.
:::
## Изменение логотипа загрузки

    * Требуемое разрешение изображения: `800*480`
    * Требуемое имя файла: `boot_logo.JPG`
    * Размер не должен превышать: `105KB`



## Изменение изображения первого интерфейса

    * Требуемое разрешение изображения: `500*400`
    * Требуемое имя файла: `print.png`
    * Размер не должен превышать: `45KB`



## Изменение изображения интерфейса печати

    * Требуемое разрешение изображения: `311*480`
    * Требуемое имя файла: `print_smail.png`
    * Размер не должен превышать: `45KB`
