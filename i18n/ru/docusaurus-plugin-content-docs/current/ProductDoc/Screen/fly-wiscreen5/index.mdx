# Введение

      Fly_WiScreen — это премиальная панель управления для 3D-принтеров, разработанная командой Fly3D основной платы компании Guangzhou Meilun Electronic Technology Co., Ltd. Данная панель позволяет управлять 3D-принтером по Wi-Fi, поддерживает передачу файлов печати через SD-карту и флешку, обеспечивая высокую скорость отклика и отличный пользовательский опыт. Панель поддерживает режим главного/вспомогательного устройства для синхронизации работы нескольких принтеров, что значительно облегчает управление оборудованием. Также в комплекте предусмотрен порт 5В для подключения модуля зарядки, что позволяет адаптировать устройство под использование аккумулятора для мобильности.

## Функциональные возможности

    * Поддержка подключения к другим компьютерам с установленным Klipper через Wi-Fi
    * Поддержка сканирования сетей Wi-Fi; после подключения к сети можно самостоятельно находить компьютеры
    * Поддержка управления несколькими принтерами одновременно
    * Поддержка загрузки файлов печати через USB и TF-карту
    * Возможность настройки макросов для удобства управления и отладки

## Технические характеристики

    * Процессор: Allwinner T113-S3, двуядерный Cortex-A7 с максимальной частотой 1.2 ГГц
    * Оперативная память: интегрирована 128 МБ DDR3
    * Накопитель: 128 МБ NAND Flash, слот для TF-карты
    * Экран: емкостный экран 800x480
    * Внешние интерфейсы: один порт USB 2.0, слот для TF-карты
    * Протокол Wi-Fi: 802.11b/g/n
    * Тип экрана: IPS
    * Размер экрана: 5 дюймов
    * Многоточечный сенсорный ввод: поддерживается
    * Номинальное напряжение: 5 В
