# Введение

    Fly-MINI-Pad является начальным уровнем компьютера верхнего уровня для Klipper.

## Особенности

    * CPU: Высокоэффективный чип Allwinner H3 с четырьмя ядрами 64-битного Cortex-A7
    * GPU: Mali-400 MP2
    * ОЗУ: 512 МБ DDR3
    * ПЗУ: Поддерживает максимальную карту microSD объемом 128 ГБ
    * WIFI: Интегрированный Wi-Fi 2,4 ГГц
    * Периферия: SPI x1, USB x2, Type-C x1
    * Интерфейсы: Три интерфейса XH2.54-2P, поддерживающие POWER-IN (максимальное входное напряжение 24В), CAN-BUS (поддержка подключения к шине CAN), LI-PO (поддержка питания/зарядки LiPo напряжением 3,7В с защитой)
    * Предустановленная система: klipper, moonraker, mainsail, fulidd, klipper-screen, плагин компенсации резонанса, numpy и FLY-Tools (необходимо покупать TF-карту)
    * Экран: емкостный/резистивный экран, разрешение 320×480
