# Использование встроенного UTOC

:::warning Важно

Не рекомендуется использовать два UTOIC, так как процесс записи и подключения будет очень сложным

:::
    * Если вам не нужен встроенный UTOC, выполните следующую команду, которая не даст никаких подсказок
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * Метод восстановления, который также не даст никаких подсказок
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * Устройство MINI Pad уже оснащено встроенным UTOC
