# Использование акселерометра

    :::tip

    * Mini Pad может подключать внешний акселерометр напрямую через интерфейс на плате или использовать встроенный акселерометр с помощью инструментальной платы и т.д.

    :::

## Подключение внешнего акселерометра через интерфейс на плате

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.webp').default} size="80%" align="left" />

    * Пример конфигурации

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # пример
    ```
