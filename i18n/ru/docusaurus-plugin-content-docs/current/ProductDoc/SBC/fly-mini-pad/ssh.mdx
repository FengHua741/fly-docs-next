# Подключение к ПК через SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другой **SSH-терминальный инструмент**
* Убедитесь, что SD-карта или M2WE загружены с **соответствующей операционной системой ПК**
* **Если ПК не запускается корректно, удалите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т.д.!**
* **В конце подключитесь к SSH через сеть для удобства компиляции прошивки!**
* **Для компиляции прошивки обратитесь к руководству по подключению ПК через Wi-Fi или сетевой кабель**

        ## Подключение к ПК через IP-адрес сети

        * Войдите в веб-интерфейс маршрутизатора, к которому подключен ПК
        * Убедитесь, что система уже **запущена корректно**
        * Убедитесь, что **вы знаете имя пользователя и пароль ПК**, по умолчанию для системы FLY: пользователь — **fly**, пароль — **mellow**
        * Если материнская плата запущена и подключена к сети, можно подключиться к SSH через IP-адрес
        * Откройте MobaXterm и выполните следующие настройки, указав ранее просмотренный IP-адрес или проверив его в веб-интерфейсе маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: на этом шаге введите имя пользователя ПК**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится окно безопасности, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Введите пароль, после чего нажмите Enter

        **Внимание: при вводе пароля символы не отображаются, просто нажмите Enter после завершения ввода**

        По умолчанию для системы FLY: пользователь — **fly**, пароль — **mellow**
