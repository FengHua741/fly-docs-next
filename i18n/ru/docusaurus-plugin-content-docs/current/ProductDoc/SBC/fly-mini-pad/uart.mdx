# Метод настройки

:::danger Важно
Обратите внимание, что после использования этого метода невозможно будет подключить внешний акселерометр к интерфейсу акселерометра mini pad.
:::

    * Откройте BOOT-диск и найдите файл `armbianEnv.txt`, затем откройте его
    * Измените в строке `overlays=usbhost2 usbhost3 uart2` слово **uart2** на **uart3** и сохраните изменения

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart.webp').default} size="80%" align="center" />

## Метод подключения

    * Обратите внимание, что TX mini-pad должен быть подключен к RX материнской платы с прошивкой последовательного порта
    * Обратите внимание, что RX mini-pad должен быть подключен к TX материнской платы с прошивкой последовательного порта
    * Обратите внимание, что необходимо обеспечить общий земляной провод
    * Иначе подключение к материнской плате с прошивкой последовательного порта будет невозможно

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/uart1.webp').default} size="80%" align="center" />

## Справочная конфигурация для Klipper

    ```cfg
    [mcu]
    serial: /dev/ttyS3
    baud：250000
    #   Используемая скорость передачи данных. По умолчанию 250000
    restart_method:command
    ```
