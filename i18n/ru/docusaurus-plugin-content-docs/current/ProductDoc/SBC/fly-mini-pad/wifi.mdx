# Подключение к Wi-Fi

## Установка антенны на MINI PAD

    * Точка с красной стрелкой указывает на разъем Wi-Fi антенны
    * Разъем Wi-Fi антенны — это `ipex1`
    * Установите антенну на MINI PAD, иначе может произойти сбой подключения из-за слабого сигнала
