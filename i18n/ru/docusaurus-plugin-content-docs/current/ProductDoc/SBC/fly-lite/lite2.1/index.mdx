# Введение

    Fly-Pi-lite2.1 является начальным уровнем компьютером для Klipper
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.webp').default} size="80%" align="center" />

## Особенности

    * Процессор: высокопроизводительный чип Allwinner H3 с четырьмя ядрами 64-битного Cortex-A7
    * Графический процессор: Mali-400 MP2
    * ОЗУ: 512 МБ DDR3
    * ПЗУ: поддерживает максимум 128 ГБ карты microSD
    * WI-FI: встроенный 2,4 ГГц Wi-Fi
    * Периферия: Spi x1, URAT x2, USB x2, HDMI x1
    * Интерфейсы:预留FPC, HDMI + USB FPC "одна линия"
    * Система предустановлена klipper, moonraker, mainsail, fulidd, klipper-screen, плагин компенсации резонанса numpy и FLY-Tools, не требует сложных операций, таких как замена источника, что делает его более доступным для новичков (требуется покупка карты памяти)


## Новые функции

    * Обновление интерфейса TFT V1 до TFT V2
    * Добавлены выводы для сброса и выключения
    * Добавлена индикация включения питания
    * Удалена кнопка BOOT, заменена на кнопку сброса
    * Оптимизация проводки
