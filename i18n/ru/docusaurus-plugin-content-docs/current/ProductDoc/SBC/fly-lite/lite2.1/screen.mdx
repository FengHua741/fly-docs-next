# Подключение и настройка экрана

:::info

* lite2.1 требует подключения по SSH к системе и ввода нижеуказанной команды для использования Fly Tools для настройки экрана
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::

:::info

* Fly Tools требует подключения к сети и ввода IP-адреса хост-машины с добавлением `:9999` для доступа
* Например, `192.168.6.110:9999`

## Метод 1    
         * Для настройки емкостного экрана используйте следующую команду, чтобы настроить на `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
    * Проверьте, удалось ли изменить настройки с помощью следующей команды
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Изменить настройки с емкостного на резистивный экран можно с помощью следующей команды

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
```
