# Настройка экрана и его подключения

:::info

* Для lite2.1 необходимо подключиться к системе через SSH и ввести указанные ниже команды, чтобы использовать Fly Tools для настройки экрана:
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::


* Для работы с Fly Tools необходимо, чтобы верхний компьютер был подключен к сети и можно было получить доступ по адресу `IP+:9999`
* Например: `192.168.6.110:9999`

## Метод один    
         * Для настройки емкостного экрана используйте следующую команду для изменения настройки на `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Убедитесь, что переключатель на обратной стороне экрана установлен в положение `Cap`
    * Проверьте успешность изменений с помощью следующей команды
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Для изменения настроек емкостного экрана на резистивный используйте следующую команду

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
```

* Вставьте SD/TF-карту в кардридер и подключите его к USB порту компьютера. В этом случае на компьютере появится диск boot
