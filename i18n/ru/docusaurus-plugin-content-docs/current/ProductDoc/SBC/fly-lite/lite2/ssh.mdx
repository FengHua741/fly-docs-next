# Подключение к ПК по SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другую **SSH-терминальную утилиту**
* Убедитесь, что карта SD или M2WE загружена с **соответствующей операционной системой для ПК**
* Если ПК не запускается нормально, удалите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!!!
* В конце подключитесь через сеть к SSH для удобства компиляции прошивки!!
* Для компиляции прошивки см. руководство по подключению к ПК через Wi-Fi или проводной интерфейс.

        ## Подключение к ПК по SSH через сеть

        * Авторизируйтесь в веб-интерфейсе маршрутизатора, к которому подключен ПК
        * Убедитесь, что система уже **нормально запущена**
        * Убедитесь, что вы знаете **имя пользователя и пароль верхнего ПК**, FLY-система по умолчанию имеет имя пользователя **fly**, пароль **mellow**
        * Если материнская плата уже запущена и подключена к сети, можно подключиться к SSH по IP-адресу
        * Откройте MobaXterm и выполните следующие настройки, введя ранее просмотренный IP-адрес, также его можно найти в веб-интерфейсе маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на 4-м шаге вводится имя пользователя ПК**

        По умолчанию для FLY-системы имя пользователя: `fly`, пароль: `mellow`
