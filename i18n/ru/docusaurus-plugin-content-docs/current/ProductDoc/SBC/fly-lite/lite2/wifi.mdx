# Подключение к Wi-Fi

## Установка антенны на LITE 2

    * Красная стрелка указывает на разъем антенны Wi-Fi
    * Разъем антенны — это `ipex1`
    * Установите антенну на LITE 2, иначе может возникнуть проблема с сигналом, из-за которой подключение к сети будет невозможно

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="100%" align="left" />  

* Если подключение к Wi-Fi не произошло, проверьте, установлена ли антенна, а затем используйте приведенный ниже метод для подключения к Wi-Fi
