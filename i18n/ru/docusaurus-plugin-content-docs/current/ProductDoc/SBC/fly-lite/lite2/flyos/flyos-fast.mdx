# Запись системного образа

## Установка системы на карту

    * После записи системы на TF-карту, её необходимо подключить к задней части LITE2, иначе система не запустится
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/tf.webp').default} size="100%" align="left" />  

## Установка антенны

    1. Установите антенну
    :::warning
    * Обратите внимание, что ПК верхнего уровня не поддерживает сети Wi-Fi с именами на китайском языке и сети Wi-Fi, сочетающие 2.4 ГГц и 5 ГГц
    :::
    * Место, указанное красной стрелкой, является разъёмом антенны Wi-Fi
    * Разъём Wi-Fi — это `ipex1`
    * Установите антенну, иначе возможна потеря сигнала, из-за чего устройство не сможет подключиться к роутеру
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После установки системы можно перейти к следующему шагу: подключение к ПК через SSH
    :::

    2. Подайте питание на ПК.  
    **Ваш ПК уже может быть запущен!**  
    Перейдите к следующему шагу: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2/ssh">Подключение к ПК через SSH</Button>
