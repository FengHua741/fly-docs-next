# Введение

    Fly-Pi-lite2 является начальным уровнем контроллера для Klipper.

## Особенности

    * Процессор: высокопроизводительный чип Allwinner H3 с 4-ядерным 64-битным Cortex-A7
    * Графический процессор: Mali-400 MP2
    * ОЗУ: 512 МБ DDR3
    * ПЗУ: поддерживает максимум 128 ГБ карты microSD
    * WI-FI: встроенный 2,4 ГГц Wi-Fi
    * Периферия: Spi x1, URAT x2, USB x2, HDMI x1
    * Интерфейс:预留FPC, HDMI + USB FPC "одной лентой"
    * Предустановленная система: Klipper, Moonraker, Mainsail, Fulld, Klipper-Screen, плагин компенсации резонанса, numpy и FLY-Tools, не требует сложных операций типа замены источника, что делает его более доступным для новичков (требуется приобретение карточки памяти)
