# Введение

      Fly-CM4 полностью совместим с Raspberry Pi 3B и 4B и специально улучшен для расширения модуля CM4. Этот адаптерный модуль предоставляет целый набор функций, оптимизированных для 3D-принтеров. Питание от 12 до 24 В постоянного тока обеспечивает стабильность питания, что критически важно для точного управления и непрерывной работы 3D-принтера. Поддержка FLY-TFT и Fly-HDMI экранов через FPC-соединение обеспечивает более удобный способ прокладки проводов. Предоставляется порт Can, позволяющий верхнему уровню оборудования напрямую подключаться к нижнему уровню через протокол Can, что является популярным методом на данный момент.

## Особенности

    * Совместимость с модулем Raspberry Pi CM4
    * Поддержка питания 12В–24В постоянного тока
    * Наличие модуля UTOC на плате
    * Поддержка TFT и HDMI "один провод"
    * Поддержка интерфейсов DSI/spi экранов
    * Поддержка интерфейсов CSI/USB камер
    * 40PIN расширенный разъем, совместимый с Raspberry Pi
