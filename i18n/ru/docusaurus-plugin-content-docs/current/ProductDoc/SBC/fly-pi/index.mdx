# Введение

        Мы рады объявить о выпуске новой материнской платы FLY-π. Данная плата может заменить Raspberry Pi в качестве главного контроллера и работает с полным десктопом на базе Armbian (оптимизированный под Linux). Платформа совместима с Klipper и RRF прошивками, а также может работать с экранами Klipper при использовании SHT toolboard.

Целью разработки данной платы было сделать использование Klipper или RRF прошивок более доступным для пользователей. Мы оптимизировали системный образ Linux для использования в 3D принтерах, сделав установку проще и понятнее даже для начинающих. Снижаем стоимость обучения и позволяете сосредоточиться на получении удовольствия от 3D печати (предустановленная Klipper на TF карте "plug and play"). Установка простая, даже начинающий сможет быстро освоить работу с ней. ## Особенности

    * Процессор: Высокоэффективный чип H5 Allwinner, 4 ядра 64-битный Cortex-A53
    * Графический процессор: Высокопроизводительный Mali 450 с 6 ядрами, частота заполнения пикселей > 2.7 gpixels/s
    * ОЗУ: 1 ГБ DDR3 (общий объем с GPU) (512+512)
    * ПЗУ: Поддерживает карты SD до 128 ГБ
    * Периферия: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100 Мбит/с), CAN x1 (модуль CAN приобретается отдельно)
    * Разъемы: М.2 интерфейс для расширения Wi-Fi, eMMC и т.д. (приватный протокол, не подключайте несовместимое оборудование)
    * 40Pin разъем, совместимый с Raspberry Pi
    * Поддержка питания 12-24 В постоянного тока, более стабильное питание
    * Предусмотрен разъем для вентилятора 5В с управлением, для охлаждения платы
    * Предусмотрен разъем расширения CAN
    * Предустановлены klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, компенсатор резонанса, библиотека numpy и FLY-Tools, без необходимости сложных операций типа перенастройки источников, что упрощает работу для новичков (необходимо приобрести TF карточку или M2WE)
