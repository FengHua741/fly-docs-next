## Установка антенны

    1. Установите антенну на модуль `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы Fly-компьютера с беспроводной связью требуется модуль `M2WE` или `2.4G WIFI`, иначе WiFi не будет работать
    * Обратите внимание, что компьютер верхнего уровня не поддерживает сети Wi-Fi с русскими названиями, а также сети, объединяющие 2.4G и 5G
    :::
    * Место, помеченное красной стрелкой, является разъемом антенны Wi-Fi
    * Разъем антенны Wi-Fi — это `ipex1`
    * Антенну необходимо установить на M2WE, иначе возможна потеря сигнала, из-за чего подключение к маршрутизатору может быть невозможно
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После завершения установки системы карта может быть готова к следующему шагу: подключение через SSH к верхнему уровню
    :::

    2. Подайте питание на компьютер верхнего уровня.  
    **Ваш верхний уровень уже может быть запущен!**  
    Выполните следующий шаг: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-pi/ssh">Подключение через SSH к верхнему уровню</Button>
