## Прошивка системы

:::info
Прошивка TF-карты или прошивка через TF-карту в M2WE требует использования только одного из методов прошивки.
:::

:::info
* Если в машине установлен M2WE и карта памяти, система по умолчанию будет загружаться с карты памяти
* Рекомендуется использовать TF-карту для прошивки системы в M2WE; использование USB для прошивки в M2WE может завершиться неудачей из-за совместимости драйверов и т.д.; рекомендуется использовать прямую прошивку [карты памяти в M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие скачанного образа системы; если он не соответствует, система не запустится
:::

## Установка антенны

    1. Установите антенну для модуля `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы FLY-устройства требуется модуль `M2WE` или `2.4G WiFi`, иначе Wi-Fi не будет работать
    * Обратите внимание, что устройство не поддерживает сети Wi-Fi с русскими названиями, а также сети Wi-Fi, поддерживающие одновременную работу на частотах 2.4G и 5G
    :::
    * Место, куда указывает красная стрелка, является разъемом для антенны Wi-Fi
    * Разъем для антенны Wi-Fi — это `ipex1`
    * Убедитесь, что антенна установлена на PI V2, чтобы избежать проблем с соединением с роутером из-за слабого сигнала
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi/img/wifi.webp').default} size="80%" align="left" />
