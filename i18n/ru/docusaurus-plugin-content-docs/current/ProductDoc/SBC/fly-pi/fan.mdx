# Вентиляторное гнездо 5В на плате

## На плате Pi установлен контролируемый разъем 5В, предназначенный для охлаждения основного процессора. Схема подключения показана на следующем рисунке:

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi/img/corefan.webp').default} size="80%" align="center" />

## Пример конфигурации

    ```cfg
    [mcu host]                      # FlyOS автоматически сгенерировал для вас linux MCU; достаточно просто определить его
    serial: /tmp/klipper_host_mcu 

    [temperature_sensor FLY-π]      # Температура верхнего уровня
    sensor_type: temperature_host


    [temperature_fan core_fan]      # Вентилятор верхнего уровня
    pin: host:gpiochip1/gpio103     # Pin вентилятора верхнего уровня
    max_power: 1.0
    sensor_type: temperature_host   # Установлено как температура контроллера верхнего уровня
    control:watermark               # Способ управления
    target_temp: 48                 # Температура включения вентилятора верхнего уровня
    min_temp: 0                     # Минимальная температура, ниже которой будет выдана ошибка
    max_temp: 90                    # Максимальная температура, выше которой будет выдана ошибка
    off_below: 0.10
    kick_start_time: 0.50
    max_speed: 0.8                  # Максимальная скорость, составляющая 80% от полной мощности
    min_speed: 0.3                  # Минимальная скорость, составляющая 30% от полной мощности
    ```
