# Подключение к ПК по SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другой **SSH-терминальный инструмент**
* Убедитесь, что карта SD или M2WE прошиты **соответствующей системой ПК**
* **Если ПК не запускается нормально, удалите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**
* **Наконец, подключитесь к SSH через сеть для удобства компиляции прошивки!**
* **Для компиляции прошивки см. руководство по подключению ПК через Wi-Fi или сетевой кабель**

        ## Использование сети для подключения к ПК по SSH

        * Войдите в веб-интерфейс маршрутизатора, к которому подключен ПК
        * Убедитесь, что система уже **нормально запущена**
        * Убедитесь, что вы знаете **имя пользователя и пароль верхнего компьютера**, по умолчанию для системы FLY имя пользователя — **fly**, пароль — **mellow**
        * Если материнская плата уже запущена и подключена к сети, можно использовать IP для подключения к SSH
        * Откройте MobaXterm и выполните следующие настройки, введите IP, который вы видели ранее, либо просмотрите его в веб-интерфейсе маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на четвёртом шаге вводится имя пользователя верхнего компьютера**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Запрос пароля, введите пароль верхнего компьютера и нажмите Enter

        **Внимание: здесь вводимый пароль не будет отображаться, просто нажмите Enter после ввода**

        По умолчанию для системы FLY имя пользователя — **fly**, пароль — **mellow**
