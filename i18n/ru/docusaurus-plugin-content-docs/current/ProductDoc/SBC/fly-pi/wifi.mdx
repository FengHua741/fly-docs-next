# Подключение к Wi-Fi


## Установка антенны на PI

    * Точка, указываемая красной стрелкой, является разъемом антенны Wi-Fi
    * Разъем антенны — это `ipex1`
    * Установите антенну на PI, иначе может возникнуть проблема с плохим сигналом, из-за чего подключение к сети будет невозможно
