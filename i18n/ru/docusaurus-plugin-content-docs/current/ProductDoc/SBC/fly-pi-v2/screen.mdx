## Метод 1    
         * Для настройки сенсорного экрана используйте следующую команду для установки в режим `fly-tft-v1`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v1/' -i /boot/FlyOS-Env.txt
```
         * Убедитесь, что переключатели на обратной стороне экрана установлены в положение `Cap`
    * Для проверки успешности изменений используйте следующую команду
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```

 * Для изменения конфигурации сенсорного экрана на резистивный используйте следующую команду

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v1/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

## Метод 1        
           * Для настройки резистивного экрана используйте следующую команду для установки в режим `fly-tft-v1` 
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v1/' -i /boot/FlyOS-Env.txt
```
                                * Убедитесь, что переключатели на обратной стороне экрана установлены в положение `Resi`
        * Для проверки успешности изменений используйте следующую команду
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```

 * Для изменения конфигурации резистивного экрана на сенсорный используйте следующую команду

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v1/}' /boot/FlyOS-Env.txt
```
