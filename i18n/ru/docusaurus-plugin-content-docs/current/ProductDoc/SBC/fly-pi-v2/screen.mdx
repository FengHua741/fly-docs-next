## Настройка экрана

<Tabs>
    <TabItem value="flyos-fast" label="Настройка TFT экрана в системе FLYOS-FAST">

    ### Настройка TFT экрана в системе FLYOS-FAST
    1. Откройте браузер и введите IP-адрес устройства, например: `http://192.168.6.110/`
       <table>
         <tr>
           <td>
             В `fluidd`: отмените выбор пункта `Фильтровать скрытые файлы и папки`.
             <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config1.webp').default} size="100%" align="center" />
           </td>
           <td>
             В `mainsail`: установите галочку `Показывать скрытые файлы`.
             <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />
           </td>
         </tr>
       </table>
  
    2. После этого вы увидите папку `.flyos-config`, перейдите в неё, там будет файл `sys-config.conf`.

     <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />
    
    3. Откройте файл `sys-config.conf`, внесите необходимые изменения и сохраните его.
    :::warning
        - Для резистивного экрана укажите `fly-tft-v1`, обратите внимание, что после знака `=` **не должно быть пробелов!!!**, убедитесь, что переключатель на задней стороне экрана установлен в положение `Resi`
        - Для емкостного экрана укажите `fly-tft-v2-c`, обратите внимание, что после знака `=` **не должно быть пробелов!!!**, убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
        - Обратите внимание: формат файла `sys-config.conf` — это `ключ=значение`, запрещено произвольно изменять формат, иначе настройки не будут применяться!!!
    :::
    
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_tft_conf.webp').default} size="90%" align="center" />
    
    4. После сохранения изменений перезагрузите систему для применения настроек.
    </TabItem>


    <TabItem value="flyos-armbian" label="Настройка TFT экрана в системе FLYOS-Armbian" default>
    ### Настройка TFT экрана в системе FLYOS-Armbian
    :::info
    - Подключитесь к верхнему устройству по Wi-Fi, получите IP-адрес верхнего устройства, затем в браузере введите `IP+:9999`, чтобы получить доступ к FLY TOOLS для настройки экрана
    - Например, если IP вашего верхнего устройства — `192.168.6.110`, то в браузере введите `192.168.6.110:9999`
    :::
    1. Откройте FLY TOOLS и следуйте инструкциям на изображении ниже, чтобы включить `Klipper Screen`, выбрать модель экрана, нажмите правую нижнюю кнопку «Сохранить конфигурацию», и после этого перезагрузите верхнее устройство — настройки вступят в силу!
      :::warning
      - Для резистивного экрана выберите: `fly-tft-v1`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Resi`
      - Для емкостного экрана выберите: `fly-tft-v2-c`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
      :::

    2. Также можно выполнить настройку через SSH с помощью командной строки (этот метод не рекомендуется!)
        * Для настройки резистивного экрана используйте следующую команду, чтобы установить значение `fly-tft-v1`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Resi`
        ```cfg
        sudo sed '11s/screen=none/screen=fly-tft-v1/' -i /boot/FlyOS-Env.txt
        ```

        * Для настройки емкостного экрана используйте следующую команду, чтобы установить значение `fly-tft-v2-r`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
        ```cfg
        sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
        ```

        * Используйте следующую команду для проверки успешности настройки

        ```cfg
        sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
        ```

        * Чтобы изменить резистивный экран на емкостный, используйте следующую команду
        ```cfg
        sudo sed -i '11{/^#/!s/screen=fly-tft-v1/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
        ```
        * Чтобы изменить емкостный экран на резистивный, используйте следующую команду
        ```cfg
        sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v1/}' /boot/FlyOS-Env.txt
        ```
