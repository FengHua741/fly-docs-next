## Настройка экрана

<Tabs>
    <TabItem value="flyos-fast" label="Настройка экрана в системе FLYOS-FAST">

    ### Настройка экрана в системе FLYOS-FAST
    1. Откройте браузер и введите IP-адрес устройства, например: `http://192.168.6.110/` 
       <table>
         <tr>
           <td>
             В `fluidd` отключите галочку -> `Скрыть скрытые файлы и папки`.
             <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config1.webp').default} size="100%" align="center" />
           </td>
           <td>
             В `mainsail` установите галочку -> `Показать скрытые файлы`.
             <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />
           </td>
         </tr>
       </table>
  
    2. После этого вы увидите папку `.flyos-config`, откройте её и найдите файл `sys-config.conf`.

     <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />
    
    3. Откройте файл `sys-config.conf`, сделайте необходимые изменения и сохраните его.
    :::warning

        - Для сенсорного экрана установите конфигурацию: `fly-tft-v1`, обратите внимание, что **после знака `=` не должно быть пробелов!!**, убедитесь, что переключатель на задней стороне экрана установлен в положение `Resi`
        - Для емкостного экрана установите конфигурацию: `fly-tft-v2-c`, обратите внимание, что **после знака `=` не должно быть пробелов!!**, убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
        - Для настройки интерфейса klipper_screen установите значение: `1`, обратите внимание, что **после знака `=` не должно быть пробелов!!**
        - Обратите внимание: формат файла `sys-config.conf` должен быть `ключ=значение`, изменение формата недопустимо, иначе настройки не будут применяться!!
    :::
    
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_tft_conf.webp').default} size="90%" align="center" />
    
    4. После сохранения изменений, перезагрузите систему для применения настроек.
    </TabItem>


    <TabItem value="flyos-armbian" label="Настройка экрана в системе FLYOS-Armbian" default>
    ### Настройка экрана в системе FLYOS-Armbian
    :::info
    - Подключите верхний компьютер к Wi-Fi, получите IP-адрес верхнего компьютера, затем в браузере введите `IP+:9999`, чтобы получить доступ к FLY TOOLS и выполнить настройку экрана
    - Например, если IP-адрес вашего верхнего компьютера — `192.168.6.110`, в браузере введите `192.168.6.110:9999`
    :::
    1. Откройте FLY TOOLS, следуйте инструкциям на рисунках ниже, чтобы включить интерфейс Klipper Screen, выбрать модель экрана и нажмите правую нижнюю кнопку «Сохранить конфигурацию». После этого перезагрузите верхний компьютер, чтобы настройки вступили в силу!
      :::warning
      - Для сенсорного экрана выберите: `fly-tft-v1`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Resi`
      - Для емкостного экрана выберите: `fly-tft-v2-c`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
      :::

    2. Также можно настроить через SSH командами в командной строке (не рекомендуется!)
        * Для сенсорного экрана используйте следующую команду, чтобы настроить как `fly-tft-v1`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Resi`
        ```cfg
        sudo sed '11s/screen=none/screen=fly-tft-v1/' -i /boot/FlyOS-Env.txt
        ```

        * Для емкостного экрана используйте следующую команду, чтобы настроить как `fly-tft-v2-r`, убедитесь, что переключатель на задней стороне экрана установлен в положение `Cap`
        ```cfg
        sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
        ```

        * Используйте следующую команду для проверки успешности настройки

        ```cfg
        sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
        ```

        * Для изменения сенсорного экрана на емкостный используйте следующую команду
        ```cfg
        sudo sed -i '11{/^#/!s/screen=fly-tft-v1/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
        ```
        * Для изменения емкостного экрана на сенсорный используйте следующую команду
        ```cfg
        sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v1/}' /boot/FlyOS-Env.txt
        ```
