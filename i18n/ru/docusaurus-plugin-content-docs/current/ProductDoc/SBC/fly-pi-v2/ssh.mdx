# Подключение к ПК по SSH

* Можно использовать **последовательный порт** или **Wi-Fi** для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другой **инструмент для SSH-подключений**
* Убедитесь, что карта SD или M2WE прошиты **соответствующей системой для вашего ПК**
* **Если ПК не запускается корректно, отключите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т.д.!**

:::warning Внимание
- Если ваш ПК запускается впервые, обязательно используйте **последовательное подключение для SSH на ПК**
- При компиляции прошивки обязательно используйте **Wi-Fi для подключения к SSH на ПК**, чтобы упростить процесс компиляции и прошивки!
- Операции подключения к SSH для системы **FLYOS-Armbian** и системы **FLYOS-FAST** отличаются. Выберите соответствующую инструкцию в зависимости от прошитой системы!
::>

        ## Подключение через Wi-Fi к SSH на ПК FLYOS-Armbian

        * Убедитесь, что система ПК уже **корректно запущена**
        * Убедитесь, что вы **знаете имя пользователя и пароль ПК**. По умолчанию для системы FLY имя пользователя: `fly`, пароль: `mellow`
        * Для подключения через Wi-Fi к SSH на ПК необходимо, чтобы ваш **ПК уже был подключен к роутеру, и вы знаете его IP**

        * Откройте MobaXterm и установите настройки, как показано на рисунке ниже, в поле `Remote host` укажите IP ПК, который вы нашли при подключении через последовательный порт, либо можете найти его через интерфейс роутера

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Внимание: здесь на четвёртом шаге вводится имя пользователя ПК**
        :::tip
        - На четвёртом шаге введите имя пользователя ПК. Для fly ПК с системой FLYOS-Armbian введите `fly`, для других ПК укажите правильное имя пользователя в зависимости от实际情况
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * При первом подключении появится окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Введите пароль, нажмите Enter
        :::tip
        - При вводе пароля символы не будут видны, просто нажмите Enter после завершения ввода
        :::
        
        **По умолчанию для системы FLYOS-Armbian имя пользователя: `fly`, пароль: `mellow`**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        **Если появится окно, как показано ниже, оно предупреждает о сохранении пароля пользователя. Нажмите `No`**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Если появится следующий экран, подключение успешно выполнено!
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
