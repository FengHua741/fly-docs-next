# Подключение к ПК по SSH

* Можно использовать **последовательный порт** или **Wi-Fi** для подключения к устройству.
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другое **SSH-терминальное приложение**.
* Убедитесь, что карта SD или M2WE записаны с **системой, совместимой с вашим ПК**.
* **Если ПК не запускается корректно, удалите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**

:::warning Внимание
- Если это первая загрузка вашего ПК, обязательно используйте **последовательное соединение для подключения к SSH ПК**.
- При компиляции прошивки используйте обязательно **Wi-Fi-подключение к SSH ПК**, чтобы упростить процесс компиляции и перезагрузки!
- Существуют различия в подключении к SSH для системы **FLYOS-Armbian** и системы **FLYOS-FAST**. Выберите соответствующую инструкцию в зависимости от установленной на вашем устройстве системы!
::>

        * Откройте `MobaXterm` и выполните следующие настройки, выбрав ранее виденный в диспетчере устройств порт.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-2.webp').default} size="60%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-3.webp').default} size="60%" align="center" />

        * Подождите немного, и если появится этот экран, значит, устройство запущено.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="60%" align="center" />

        * **Если через несколько минут интерфейс не показывает ничего, нажмите несколько раз клавишу Enter.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-5.webp').default} size="60%" align="center" />

        * **После нажатия клавиши Enter вы увидите следующее содержимое.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-6.webp').default} size="60%" align="center" />

        :::warning
        - В данный момент вы вошли как пользователь root. Введите: `su fly`, чтобы переключиться на обычного пользователя и продолжить работу.
        :::
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-13.webp').default} size="60%" align="center" />

        * Если нужно проверить IP, введите в SSH: `ip a | grep inet` и найдите IP возвращаемого содержимого, который принадлежит вашему маршрутизатору.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-8.webp').default} size="70%" align="center" />

        ## Подключение по Wi-Fi к SSH ПК FLYOS-Armbian

        * Убедитесь, что система ПК уже успешно запущена.
        * Убедитесь, что вы знаете **имя пользователя и пароль ПК**. По умолчанию для системы FLY имя пользователя — fly, пароль — mellow.
        * Предварительным условием подключения по Wi-Fi к SSH является подключение ПК к роутеру и знание его IP-адреса.

        * Откройте MobaXterm и установите настройки, как показано на рисунке ниже. В поле «Remote host» введите IP-адрес ПК, полученный при подключении через последовательный порт, или получите IP через интерфейс роутера.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Внимание: здесь в четвертом шаге вводится имя пользователя ПК**
        :::tip
        - В четвертом шаге введите имя пользователя ПК. Для fly ПК с системой FLYOS-Armbian введите «fly», для других ПК используйте правильное имя пользователя в соответствии с фактическими условиями.
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * При первом подключении появится окно, нажмите «Accept».

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Введите пароль ПК, после чего нажмите Enter.
        :::tip
        - Здесь пароль не будет виден при вводе, просто нажмите Enter после ввода.
        :::
        
        По умолчанию для системы FLYOS-Armbian имя пользователя — fly, пароль — mellow.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        **Если появится окно, как показано на изображении ниже, это предупреждение о сохранении пароля пользователя. Нажмите «No».**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * После этого появится изображение, как показано ниже, что означает успешный вход!

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
