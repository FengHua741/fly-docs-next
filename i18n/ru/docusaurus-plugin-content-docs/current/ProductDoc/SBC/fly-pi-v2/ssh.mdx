# Подключение к ПК через SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другую **SSH-терминальную утилиту**
* Убедитесь, что на SD-карте или M2WE записана **система соответствующего ПК**
* **Если ПК не запускается нормально, удалите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**
* **В конце подключитесь к SSH через сеть для удобства компиляции прошивки!**
* **Для компиляции прошивки см. руководство по подключению ПК через Wi-Fi или проводной интерфейс**

        ## Подключение к ПК через сетевой IP SSH

        * Войдите в панель управления маршрутизатора, к которому подключен ПК
        * Убедитесь, что система уже **нормально запущена**
        * Убедитесь, что вы знаете **имя пользователя и пароль ПК**, для системы FLY имя пользователя по умолчанию — **fly**, пароль по умолчанию — **mellow**
        * Если материнская плата уже запущена и подключена к сети, можно подключиться к SSH через IP
        * Откройте MobaXterm и выполните следующие настройки, указав ранее просмотренный IP или найдя его в панели управления маршрутизатором

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на четвёртом шаге вводится имя пользователя ПК**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Введите пароль, когда система запросит его, и нажмите Enter

        **Внимание: при вводе пароля символы не будут отображаться, просто нажмите Enter после завершения ввода**

        По умолчанию для системы FLY имя пользователя — `fly`, пароль — `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />
