# Подключение к ПК по SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другую **SSH-терминальную утилиту**
* Убедитесь, что карта SD или M2WE записана **соответствующей системой ПК**
* **Если ПК не запускается корректно, отключите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**
* **Наконец, подключитесь к SSH через сеть для удобства компиляции фиксированных версий!**
* **Для компиляции фиксированной версии обратитесь к руководству по подключению ПК через Wi-Fi или сетевой кабель**

        ## Подключение к ПК через IP-адрес по SSH

        * Пожалуйста, войдите в веб-интерфейс маршрутизатора, к которому подключен ПК
        * Убедитесь, что система уже **нормально запущена**
        * Убедитесь, что **вы знаете имя пользователя и пароль верхнего компьютера**, FLY по умолчанию использует имя пользователя **fly**, пароль **mellow**
        * Если материнская плата уже запущена и подключена к сети, можно использовать IP для подключения к SSH
        * Откройте MobaXterm и выполните следующие настройки, IP заполняется тем, что было просмотрено ранее, также можно найти его в веб-интерфейсе маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на 4-м шаге вводится имя пользователя верхнего компьютера**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        * Запрос на ввод пароля, после ввода пароля ПК нажмите Enter

        **Внимание: здесь пароль не отображается при вводе, просто нажмите Enter после ввода**

        По умолчанию для FLY-системы пользователь: `fly`, пароль: `mellow`
