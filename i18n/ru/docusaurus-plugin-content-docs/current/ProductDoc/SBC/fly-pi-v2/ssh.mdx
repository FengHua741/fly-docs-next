# Подключение к ПК через SSH

* Можно использовать **последовательный порт** или **Wi-Fi** для подключения к устройству
* Пожалуйста, подготовьте заранее **MobaXterm_Personal** или другой **SSH-терминал**
* Убедитесь, что SD-карта или M2WE уже записаны **системой, совместимой с ПК**
* **Если ПК не запускается корректно, удалите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т.д.!**

:::warning Внимание
- Если это первый запуск ПК, обязательно используйте **последовательное соединение для подключения к SSH ПК**
- При компиляции прошивки обязательно используйте **Wi-Fi для подключения к SSH ПК**, чтобы упростить процесс компиляции и записи!
:::

        * Подождите немного, пока появится этот экран — это означает, что система запущена

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="60%" align="center" />

        * **Если через несколько минут экран не отображается, нажмите несколько раз клавишу Enter**

        :::warning

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-13.webp').default} size="60%" align="center" />

    <TabItem value="net" label="Подключение через Wi-Fi FLYOS-Armbian" default>
        ## Подключение через Wi-Fi FLYOS-Armbian к SSH ПК

        * Убедитесь, что система ПК уже **запущена корректно**
        * Убедитесь, что **вы знаете имя пользователя и пароль ПК**. По умолчанию для FLY системы имя пользователя — `fly`, пароль — `mellow`
        * Для подключения через Wi-Fi к SSH ПК необходимо: ваше устройство уже должно быть подключено к маршрутизатору, и вы должны знать IP ПК  

        * Откройте MobaXterm и установите параметры, как показано на рисунке. В поле `Remote host` укажите IP ПК, который вы нашли при подключении через последовательный порт, либо запросите IP через интерфейс маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Важно: на четвертом шаге введите имя пользователя ПК**
        :::tip
        - На четвертом шаге введите имя пользователя ПК. Для fly ПК с системой FLYOS-Armbian введите `fly`, для других ПК укажите правильное имя пользователя
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * При первом подключении появится окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Запрос на ввод пароля, после ввода пароля ПК нажмите Enter
        :::tip
        - При вводе пароля символы не отображаются, введите пароль и нажмите Enter
        :::
        
        По умолчанию для системы FLYOS-Armbian имя пользователя — `fly`, пароль — `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        **Если появится окно, как показано на изображении ниже, это окно спрашивает, хотите ли вы сохранить пароль пользователя, нажмите `No`**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Если появится следующий экран, подключение выполнено успешно!

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
