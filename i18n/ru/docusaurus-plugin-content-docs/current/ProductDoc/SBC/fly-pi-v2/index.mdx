# Введение

        Мы рады объявить о выпуске обновленной версии FLY-Pi V2. Плата имеет те же размеры установки, что и Raspberry Pi, и может заменить Raspberry Pi в качестве главного компьютера. На борту имеется 1 ГБ оперативной памяти DDR3, позволяющей запускать полный рабочий стол LINUX (на основе оптимизированной версии Armbian). Эта плата может использоваться совместно с фиксированными прошивками Klipper и RRF, а также может работать с экраном Klipper при использовании инструментальной панели SHT.
    Основная цель разработки этой платы - сделать использование прошивок Klipper или RRF более доступным для клиентов. Мы оптимизировали систему Linux для работы с 3D-принтерами, сделав установку и настройку проще, даже для пользователей с минимальными знаниями. Это снижает затраты на обучение, позволяя сосредоточиться на удовольствии от использования 3D-печати (пакеты с предустановленной прошивкой Klipper на SD-карте готовы к работе сразу после подключения питания).

## Особенности

    * Совместимость с Raspberry Pi 3B и 4B по размерам
    * Процессор: высокопроизводительный чип Allwinner H5, 4-ядерный 64-битный Cortex-A53
    * Графический процессор: высокопроизводительный 6-ядерный Mali 450, производительность растровых операций более 2,7 gpixel/s
    * ОЗУ: 1 ГБ DDR3 (общая с GPU) (512+512)
    * ПЗУ: поддерживает максимальную карту памяти microSD объемом до 128 ГБ
    * Периферия: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100 Мбит/с), CAN (встроен модуль USB-to-CAN, то есть встроенный UTOC)
    * Интерфейсы: встроен M.2 интерфейс для расширения Wi-Fi, eMMC и т. д. (протокол частный, не подключайте несовместимые с FLY Pi устройства)
    * 40-пиновый разъем, совместимый с Raspberry Pi
    * Поддержка питания от 12 до 24 В постоянного тока, более стабильное питание
    * Предусмотрен разъем для вентилятора 5В ZH1.5
    * Система предустановлена Klipper, Moonraker, Mainsail, Fluidd, Klipper Screen, Crowsmest, компоненты для компенсации резонанса, numpy и FLY-Tools, без необходимости сложных операций, таких как изменение источников, что делает работу новичков проще (необходимо приобрести TF-карту или M2WE)
