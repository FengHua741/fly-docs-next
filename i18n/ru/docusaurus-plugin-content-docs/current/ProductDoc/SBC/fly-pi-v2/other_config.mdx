        ## Система FLYOS-FAST - другие настройки
        :::tip Важно прочитать
        - Некоторые параметры в файле конфигурации будут действовать только при первой загрузке системы после прошивки.
        - Поэтому перед тем как установить карту памяти в плату, необходимо сначала отредактировать файл config.txt в каталоге FlyOS-Conf на сменном диске.
        - Затем вставьте карту памяти в плату и запустите систему.
        :::
        
        :::warning Внимание
        
        - Старайтесь не использовать встроенный в Windows редактор заметок для редактирования файла конфигурации, лучше используйте VSCode или другой профессиональный текстовый редактор.
        - В файле конфигурации не рекомендуется вводить китайские символы или специальные символы.
        - После изменения файла конфигурации необходимо перезагрузить систему, чтобы изменения вступили в силу.
        
        :::
        
        :::danger
        
        - Пожалуйста, не изменяйте содержимое ниже `######### PLEASE DO NOT EDIT THE FOLLOWING #########`.
        
        :::
        
        ## Как изменить файл конфигурации
        
        ### Изменение файла конфигурации после прошивки
        
        - Параметры, которые должны действовать только при первой загрузке, можно изменить только после прошивки, открыв файл config.txt в каталоге FlyOS-Conf на сменном диске с помощью профессионального текстового редактора.
        
        ### Онлайн-изменение конфигурации во время работы системы
        
        - Если это обычные параметры, их можно изменить через веб-интерфейс системы и перезагрузить систему для применения изменений.
            - Откройте браузер и введите IP-адрес устройства, например: `http://192.168.1.2/`
            - В Fluidd снимите флажок -> «Скрыть скрытые файлы и папки» (как показано слева на рисунке).
            - В Mainsail поставьте галочку -> «Показать скрытые файлы» (как показано справа на рисунке).

              <tr>
                <td>

                </td>

            - Теперь вы увидите папку `.flyos-config`. Перейдите в эту папку, где находится файл `sys-config.conf`.
            - Файл `sys-config.conf` является мягким ссылкой на файл конфигурации `config.txt` в каталоге FlyOS-Conf на сменном диске.
        
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />
        
            - Откройте файл `sys-config.conf`, сделайте необходимые изменения, затем сохраните и закройте его.
        
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config4.webp').default} size="60%" align="center" />
        
            - Обратите внимание: формат строк в файле `sys-config.conf` должен быть `ключ=значение`, а не `ключ: значение`.
            - После сохранения изменений перезагрузите систему для их применения.

        ## Параметры конфигурации
        
        ### Настройка беспроводной сети
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :---: | :--- | :--- |
        | `WIFI_SSID` | пусто | любое имя Wi-Fi | Имя Wi-Fi; если указано, система будет автоматически подключаться к этому Wi-Fi при запуске |
        | `WIFI_PASS` | пусто | оставить пустым, если пароль отсутствует | Пароль Wi-Fi; если указано, система будет подключаться с использованием этого пароля |
        
        ----
        
        ### Настройка принтера по умолчанию
        
        :::tip
        - Данные параметры будут применяться только при первой загрузке.
        :::
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :---: | :--- | :--- |
        | `language` | `zh-CN` | <details><summary>Развернуть подробности</summary>`zh-CN`, `zh-HK`, `en`, `de`, `cz`, `es`, `hu`, `ja`, `nl`, `pt`, `se`, `uk`, `da`, `fr`, `it`, `ko`, `pl`, `ru`, `tr`</details> | Установка языка по умолчанию для fluidd и mainsail при первой загрузке |
        | `printer_name` | пусто | произвольная строка | Установка имени принтера по умолчанию для fluidd и mainsail при первой загрузке |
        
        ----
        
        ### Системные настройки
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `console` | `serial` | `serial`, `both` | Данный параметр не должен изменяться; только serial эффективен |
        | `board` | H5: `fly-pi-v2`<br/>H3: `fly-lite2.1` | <details><summary>Развернуть H5 серии</summary>`fly-pi`, `fly-pi-v2`, `fly-gemini-v1`, `fly-gemini-v2`, `fly-gemini-v3`, `fly-c8`</details><br/><details><summary>Развернуть H3 серии</summary>`fly-lite2`, `fly-lite2.1`, `fly-minipad`</details> | Модель основной платы, используется главным образом для автоматической прошивки прошивок и т.д. |
        | `shutdown_pin_state` | `1` | `0`, `1` | Устанавливает логическое состояние кнопки выключения<br/>1: выключение на низком уровне<br/>0: выключение на высоком уровне |
        | `shutdown_pin` | `none` | `PA21`, `PA4`... | Устанавливает триггерную ножку кнопки выключения, по умолчанию none — функция выключения отключена |
        | `spidev` | `spidev0.0 spidev1.0` | `spidev0.0`, `spidev0.1`, `spidev1.0`, `spidev1.1` | Устанавливает порты для активации spidev<br/>Для активации нескольких портов используйте пробел для разделения |
        | `localtime` | `Asia/Shanghai` | [Таблица часовых поясов](/docs/DebugDoc/flyos-fast/timezone) | Устанавливает часовой пояс системы, система автоматически синхронизируется времени при подключении к сети |
        | `canbus_bitrate` | `1000000` | `1000000`, `500000`, `250000` | Устанавливает стандартную скорость всех can устройств в системе, по умолчанию 1М |
        
        ----
        
        ### Настройки типа принтера
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `printer` | `klipper` | `klipper`, `rrf` | Настройка запуска службы системы<br/>`klipper`: klipper, moonraker и другие службы<br/>`rrf`: RRF DSF, DWC и другие службы |
        
        ### Настройки экрана
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `screen` | `hdmi` | <details><summary>Развернуть H5 серии</summary>`none`, `hdmi`, `fly-tft-v1`, `fly-tft-v2-r`, `fly-tft-v2-c`</details><br/><details><summary>Развернуть H3 серии</summary>`none`, `hdmi`, `fly-tft-v1`, `fly-minipad-r`, `fly-minipad-c`, `fly-tft-v2-r`, `fly-tft-v2-c`</details> | Настройка экрана, none отключает все отображения |
        | `rotate` | `90` | `0`, `90`, `180`, `270` | Настройка угла поворота экрана FLY-TFT, не работает для HDMI |
        
        ----
        
        ### Настройки Klipper
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `klipper_screen` | `0` | `0`, `1` | Настройка использования KlipperScreen<br/>1: включить<br/>0: выключить<br/>Эффективно при `printer=klipper` |
        | `moon2uart` | `0` | `0`, `1` | Настройка использования moon2uart<br/>1: включить<br/>0: выключить<br/>Эффективно при `printer=klipper` |
        | `m2u_port` | `/dev/ttyS1` | допустимые последовательные порты | Порт UART для moon2uart, эффективно при `moon2uart=1` |
        | `m2u_baud` | `115200` | допустимые скорости передачи данных | Скорость передачи данных для moon2uart, эффективно при `moon2uart=1` |
        | `klipper_webui` | `fluidd` | `fluidd`, `mainsail` | Настройка стандартного интерфейса пользовательского веб-интерфейса Klipper<br/>Не влияет на функцию онлайн-переключения<br/>Эффективно при `printer=klipper` |
        
        ----
        
        ### Настройки RRF
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `rrf2uart` | `0` | `0`, `1` | Пока неактивно, доступность будет добавлена в будущих версиях |
        
        ----
        
        ### Другие настройки
        
        - Данные параметры будут применяться при каждой загрузке системы.
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `crowsnest` | `0` | `0`, `1` | Настройка использования Crowsnest<br/>1: включить<br/>0: выключить<br/>Работает в режимах klipper и rrf |
        
        :::tip
        - Данные параметры будут применяться только при первой загрузке.
        :::
        
        | Параметр | Значение по умолчанию | Допустимые значения | Описание |
        | :---: | :-------: | :--- | :--- |
        | `firmware` | пусто | допустимые пути к файлам прошивки | Настройка автозагрузки прошивки при старте, на данный момент доступно только для C8 и Gemini-V3 |
        
        ----
        
        :::danger
        
        - Пожалуйста, не изменяйте содержимое ниже `######### PLEASE DO NOT EDIT THE FOLLOWING #########`.
