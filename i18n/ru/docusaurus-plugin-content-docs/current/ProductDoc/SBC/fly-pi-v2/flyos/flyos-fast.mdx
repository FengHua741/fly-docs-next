## Краткое описание системы

* FlyOS-Fast — это лёгкая версия Linux-системы, специально адаптированная для работы с 3D-принтерами, с быстрым запуском (всего 30 секунд). Поддерживает все платформы Fly.
* Оптимизирован ядро системы для повышения реального времени отклика Klipper.
* Оптимизирован файловая система для предотвращения повреждений системы при внезапном выключении питания.
* Предустановлены службы Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen.
* Поддерживается переключение между Klipper и RRF.
* Поддерживается переключение между Mainsail и Fluidd в режиме онлайн.
* Поддерживается печать с USB-накопителя.
* В сочетании с модулем KPPM возможна функция продолжения печати после отключения питания.
* Поддерживается предварительная настройка языка интерфейса Fluidd и имени принтера.
* Дополнительные OEM-функции: логотип при старте, предварительно установленные конфигурационные файлы, скрытие вкладок Mainsail и другие — свяжитесь с нами за подробностями.

## Подготовка

1. Скачайте образ системы
    - Ссылка на загрузку образа: [Скачать образ системы Piv2](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "Щелкните здесь для перехода") (поддерживает только Windows)  
    - Ссылка на загрузку образа: [Скачать образ системы Piv2](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "Щелкните здесь для перехода") (поддерживает Windows, MacOS, Linux)

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. Скачайте инструмент для записи образа. В качестве примера будет использоваться `Rufus`. На самом деле можно использовать другие инструменты, такие как `Etcher`, `Win32 Disk Imager`.
    - Скачать Rufus: [скачать на русском](https://rufus.ie/zh/#download) или [скачать на английском](https://rufus.ie/en/#download) (поддерживает только Windows).
    - Скачать Etcher: [скачать](https://etcher.balena.io/#download-etcher) (поддерживает Windows, MacOS, Linux).
    - Скачать Win32 Disk Imager: [скачать](https://sourceforge.net/projects/win32diskimager/) (поддерживает только Windows).

3. В качестве примера будет использоваться `FlyOS-Fast-H5.exe`.

4. Разархивируйте файл образа системы, чтобы получить `.img` файл.
    - Щелкните дважды или щелкните правой кнопкой мыши на `FlyOS-Fast-H5.exe`, подождите несколько секунд, появится окно (если система предупреждает о небезопасности, выберите «Дополнительные сведения» -> «Запустить»).
    - Можно выбрать директорию для распаковки, затем нажмите кнопку `Extract`, дождитесь завершения распаковки.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - После завершения распаковки в директории появится файл `FlyOS-Fast-H5-vx.x.x.img`, где `x.x.x` — номер версии системы.

5. Запишите образ.

            2. Откройте `Rufus`, выберите устройство. Выбранное устройство должно показывать емкость, совпадающую с емкостью подключенной к компьютеру TF-карты.

            4. После выбора файла образа, `Тип загрузочного устройства` изменится на имя файла образа.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="40%" align="center" />
            
            5. Нажмите кнопку `Начать`, если появится окно с предупреждением о необходимости очистки данных на диске, нажмите `ОК`.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />
            
            6. После завершения записи, компьютер издаст звуковой сигнал, и прогресс-бар покажет статус `Готов`.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="40%" align="center" />

            8. Откройте Проводник, проверьте наличие флешки или мобильного устройства с именем `FlyOS-Conf`.
            
                <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />
