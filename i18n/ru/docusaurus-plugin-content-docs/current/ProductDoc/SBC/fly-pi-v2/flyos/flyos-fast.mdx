## УСТАНОВКА АНТЕННЫ

    1. Установите антенну для модуля `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы с FLY манипулятором требуется модуль `M2WE` или `2.4G WIFI`, иначе WiFi не будет работать
    * Обратите внимание, что манипулятор не поддерживает Wi-Fi с именем на китайском языке и не поддерживает объединение 2.4G и 5G в одно頻о
    :::
    * Место с красной стрелкой указывает на разъем Wi-Fi антенны
    * Разъем Wi-Fi антенны - `ipex1`
    * Убедитесь, что антенна установлена на M2WE, чтобы избежать слабого сигнала и невозможности подключения к роутеру
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После завершения установки системной карты можно перейти к следующему шагу: подключению манипулятора через SSH
    :::

    2. Подайте питание на манипулятор.  
    **На этом ваш манипулятор уже может быть запущен!!!**  
    Продолжите выполнение следующего шага: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-pi-v2/ssh">Подключение к манипулятору через SSH</Button>
