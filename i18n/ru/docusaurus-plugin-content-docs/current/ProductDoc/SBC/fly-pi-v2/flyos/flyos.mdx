## Подготовка перед инсталляцией системы

    * Чтение карты
    * Карта памяти TF с объемом не менее 16 ГБ или не более 128 ГБ, скорость передачи карточки должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Скачайте необходимые <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструменты для записи</Button>

* Если в машине установлен M2WE и карта памяти, система по умолчанию будет загружаться с карты памяти
* Рекомендуется использовать карту TF для записи системы на M2WE; использование USB для записи на M2W может вызвать неудачу из-за совместимости драйверов и т.д.; рекомендуется использовать прямую запись [карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)

:::
