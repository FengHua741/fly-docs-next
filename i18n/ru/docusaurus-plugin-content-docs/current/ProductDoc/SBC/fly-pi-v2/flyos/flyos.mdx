## Подготовка перед просадкой системы

    * Программатор
    * Карта памяти TF, объем которой должен быть не менее 16 ГБ или не более 128 ГБ, скорость передачи карты памяти должна быть класса 10 или выше
    * Скачать соответствующую <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">системную картинку</Button>
    * Скачать необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент для просадки</Button>

## Просадка системы

:::info
Просадка TF-карты и просадка через TF-карту в M2WE может выполняться только одним из указанных способов
:::

:::info
* Если в машине установлены M2WE и карта памяти, система по умолчанию будет запускаться с карты памяти
* Рекомендуется использовать TF-карту для просадки системы; использование USB для просадки в M2W может привести к неудаче из-за совместимости драйверов; рекомендуется использовать прямую просадку [карты памяти в M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Убедитесь, что скачанная системная картинка соответствует вашей системе. В противном случае система не запустится
:::
