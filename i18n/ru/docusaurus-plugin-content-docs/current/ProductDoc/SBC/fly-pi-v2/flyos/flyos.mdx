## Установка антенны

    1. Установите антенну для модуля `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы Wi-Fi на FLY-компьютере необходимо использовать модуль `M2WE` или `2.4G WiFi`, иначе Wi-Fi будет недоступен
    * Обратите внимание, что компьютер не поддерживает Wi-Fi с именем на китайском языке и не поддерживает объединение 2.4G и 5G
    :::
    * Красная стрелка указывает на разъем антенны Wi-Fi
    * Разъем антенны Wi-Fi - `ipex1`
    * Убедитесь, что антенна установлена на PI V2, иначе возможны проблемы с сигналом, из-за чего невозможно будет подключиться к роутеру
