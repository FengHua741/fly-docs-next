## Подготовка перед записью системы

    * Читалка карт
    * Карта памяти TF, объем которой должен быть не менее 16 ГБ или не более 128 ГБ, скорость передачи карты памяти должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Скачайте необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент записи</Button>

## Запись системы

:::info
Запись системы на TF-карту и запись системы через TF-карту на M2WE можно выполнить одним из этих способов
:::

:::info
* Если в устройстве установлены M2WE и карта памяти, система по умолчанию будет загружаться с карты памяти
* Рекомендуется использовать TF-карту для записи системы на M2WE; использование USB для записи на M2WE может привести к неудаче из-за совместимости драйверов; рекомендуется использовать [запись системы на M2WE с карты памяти](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Убедитесь, что скачанный образ системы соответствует вашему устройству; если он не соответствует, система не сможет загрузиться
:::
