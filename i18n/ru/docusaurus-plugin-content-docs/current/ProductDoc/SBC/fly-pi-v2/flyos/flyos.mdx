## Установка антенны

    1. Установите антенну для модуля `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы Wi-Fi на FLY-компьютере требуется модуль `M2WE` или `2.4G WiFi`, иначе Wi-Fi не будет работать
    * Обратите внимание, что компьютер не поддерживает сети Wi-Fi с именем на китайском языке, а также сети Wi-Fi, поддерживающие одновременную работу в диапазонах 2.4G и 5G
    :::
    * Красная стрелка указывает место подключения антенны Wi-Fi
    * Разъем Wi-Fi — это `ipex1`
    * Антенну необходимо установить на M2WE, иначе может возникнуть проблема слабого сигнала при подключении к роутеру
