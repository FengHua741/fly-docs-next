## Подготовка перед прошивкой

    * Считыватель карт
    * Карта памяти TF, объем которой должен быть не менее 16 ГБ и не более 128 ГБ, скорость передачи данных карты должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Скачайте необходимые <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">инструменты для записи</Button>

## Прошивка системы

:::info
Прошивка карты TF и прошивка через карту TF на M2WE требуют использования только одного из методов прошивки
:::

:::info
* Если в устройстве установлен M2WE и карта памяти, система по умолчанию будет запускаться с карты памяти
* Рекомендуется использовать карту TF для записи системы, запись через USB может завершиться неудачей из-за совместимости драйверов и т.д.; рекомендуется использовать прямую запись [карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на то, что скачанный образ системы должен быть соответствующим; если он не соответствует, система не запустится
:::
