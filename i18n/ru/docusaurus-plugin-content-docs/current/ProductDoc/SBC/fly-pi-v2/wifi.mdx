# Подключение к Wi-Fi
:::tip
- Существуют незначительные отличия в шагах и методах подключения к Wi-Fi для систем `FLYOS-FAST` и `FLYOS-Armbian`. Пожалуйста, выберите соответствующий учебник для системы, которую вы записали!
:::

:::warning Предупреждение
    * Для работы с FLY требуется использовать модуль `M2WE`, модуль `2.4G Wi-Fi` или беспроводную USB-карту без драйверов для Linux; в противном случае подключение к Wi-Fi невозможно.
    * Обратите внимание, что настройка FLY не поддерживает сети Wi-Fi с именем на китайском языке и сети Wi-Fi, сочетающие 2.4G и 5G диапазоны.
    * Если у вас `mini pad` или `lite2/lite2.1`, проигнорируйте это предупреждение.
::>
