# Подключение к Wi-Fi
:::tip
- Шаги и методы подключения к Wi-Fi в системе `FLYOS-FAST` немного отличаются от системы `FLYOS-Armbian`. Пожалуйста, выберите соответствующий учебник для системы, которую вы записали!
:::

:::warning Предупреждение
    * Для работы с FLY требуется модуль `M2WE` или модуль `2.4G Wi-Fi` или беспроводная сетевая карта USB для Linux без драйверов, иначе подключение к Wi-Fi невозможно.
    * Обратите внимание, что позиционный компьютер не поддерживает подключение к сети Wi-Fi с именем на китайском языке и не поддерживает объединенные сети Wi-Fi 2.4G и 5G.
    * Если у вас `mini pad` или `lite2/lite2.1`, проигнорируйте это предупреждение.
:::
