# Возобновление печати после отключения питания в Klipper

## Общие сведения

- Возобновление печати после отключения питания в Klipper означает, что после повторного включения Klipper может автоматически восстановить состояние печати.
- Для использования функции возобновления печати требуется система FAST.
- PI-V2 требует использования модуля отключения питания KPPM.
- Не подходит для моделей принтеров, где после отключения электропитания Z-ось смещается.
