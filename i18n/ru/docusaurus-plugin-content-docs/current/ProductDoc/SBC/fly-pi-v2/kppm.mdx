# Функция возобновления печати после отключения питания в Klipper

## Общие сведения

- Функция возобновления печати после отключения питания в Klipper позволяет восстановить состояние печати Klipper после повторного включения питания.
- Для использования функции возобновления печати необходима система FAST.
- PI-V2 требует использования модуля отключения питания KPPM.
- Не подходит для моделей принтеров, где после отключения электроэнергии Z-ось смещается.
