# Компиляция прошивки для USB

    :::tip

    * Для Micro4 прошивка для USB уже предварительно установлена, повторная перезаливка не требуется

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO пины для настройки при старте микроконтроллера, введите: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/usb.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (процесс может занять некоторое время):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода должно быть:
        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
