# Micro4

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/micro.webp').default} size="80%" align="center" />

## Основные характеристики

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Поддержка моста CAN
    * На板ная интеграция 4 x 2209
    * Наboard функциональная ножка DIAG, для использования без ограничений возврата в исходное положение достаточно простого подключения/отключения переходника



## Технические параметры

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Фирменное ПО: Klipper
    * Поддержка питания 12-24В постоянного тока, более стабильное питание, без необходимости понижающего модуля
    * Режим работы драйверов: UART, STEP/DIR
    * Расширительный разъем: BLTouch (Servos, Probe), приближения X1
    * Вентиляторы: управляемые двухпроводные вентиляторы X2, постоянно включенный вентилятор X1
    * Датчики температуры: 2 канала NTC 100K
    * Интерфейсы нагрева: 2 управляемых интерфейса с поддержкой до 10А
