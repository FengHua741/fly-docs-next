    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (время займет немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последним выводом будет следующее, компиляция успешна:

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
