# Справочное подключение материнской платы

    :::warning Важно

    Из-за высокого разрешения изображения, просмотр на веб-странице может быть нечетким. Пожалуйста, наведите курсор мыши на схему подключения, щелкните правой кнопкой мыши и выберите "Сохранить изображение как..." для сохранения изображения на рабочий стол и более четкого просмотра!!!!

    :::
