# Справочное подключение материнской платы

    :::warning Важно

    Из-за высокого разрешения изображения, просмотреть его на веб-странице может быть сложно. Пожалуйста, наведите курсор мыши на схему подключения, нажмите правую кнопку мыши и выберите "Сохранить изображение как...", чтобы сохранить изображение на рабочий стол и просмотреть его более детально!!!!

    :::
