# Инструкция по прошивке прошивки

:::danger Важно знать
* Для прошивки материнской платы требуется прошить Klipper через Katapult
* Если вы уверены, что материнская плата уже прошита Katapult, то нет необходимости выполнять последний шаг прошивки прошивки Katapult — это просто мера на случай неудачной прошивки материнской платы
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли один из светодиодов на материнской плате. Если да, то это указывает на то, что материнская плата уже находится в режиме Katapult

:::

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно установить только одну из двух прошивок

    :::
    
    * E3 V2 может быть подключена к ПК следующим образом:
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная передача данных
        * Недостаток: использование нескольких USB-устройств может повлиять на передачу данных
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB-мостовая прошивка</Button>
        * Преимущество: может подключаться к нескольким устройствам без использования портов USB
        * Недостаток: совместное использование с `KlipperScreen` может привести к ошибкам, таким как превышение времени ожидания при возврате домой.
