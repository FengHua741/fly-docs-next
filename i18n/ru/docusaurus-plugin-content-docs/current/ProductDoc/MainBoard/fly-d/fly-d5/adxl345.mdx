## Рекомендуемые настройки

    ```cfg
    [adxl345]
    cs_pin: PB12
    spi_bus: spi2 # Жесткий SPI, выберите один из двух: жесткий SPI или мягкий SPI. По умолчанию используется жесткий SPI
    # spi_software_mosi_pin: PB15 # Мягкий SPI, выберите один из двух: жесткий SPI или мягкий SPI. По умолчанию используется жесткий SPI
    # spi_software_miso_pin: PB14
    # spi_software_sclk_pin: PB13

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # Обычно это центральная точка горячего стола, пожалуйста, отредактируйте в соответствии с реальной конфигурацией вашей машины
    ```
