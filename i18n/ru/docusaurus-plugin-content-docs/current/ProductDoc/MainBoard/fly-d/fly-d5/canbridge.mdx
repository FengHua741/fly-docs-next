# Мост CAN

CAN — это протокол последовательной коммуникации, используемый в реальных приложениях и передающий сигналы по двойной витой паре. Это один из самых широко распространенных полей связи в мире. Протокол CAN используется для коммуникации между разными элементами в автомобилях, заменяя дорогие и громоздкие проводные harnesses. Стойкость этого протокола позволяет использовать его в других автоматизированных и промышленных применениях. Основные характеристики протокола CAN включают целостную передачу данных по последовательному каналу, поддержку реального времени, скорость передачи данных до 1 Мбит/с, а также возможность адресации с 11 битами и функцию обнаружения ошибок.
В 3D-принтерах используется CAN-шина для уменьшения количества проводов, соединяющих печатающую головку с материнской платой. Ранее требовалось более десяти проводов, но с использованием CAN их количество сокращается до четырех, что значительно снижает количество проводов и упрощает их распределение.
В данном разделе кратко описано использование моста CAN.
