# Компиляция прошивки через USB


## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится запрос **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Сейчас конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее содержимое, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Переключение в режим записи

    :::tip Совет
        После 1 сентября 2024 года выпущенные D5 уже имеют прошивку Katapult
    :::
    * Подключите основную плату к компьютеру с помощью USB Type-C, затем **дважды нажмите кнопку reset**, при этом на плате загорится мигающий индикатор.
    :::warning Предупреждение
       * Если индикатор не мигает, пожалуйста, перепишите прошивку Katapult: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-flash-mode">Записать прошивку Katapult</Button>
       * Если индикатор не мигает, после записи прошивки Katapult выполните следующий шаг
