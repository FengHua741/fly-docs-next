:::tip Важные замечания
* D5 уже предварительно прошит Katapult-прошивкой при выпуске 1 сентября 2024 года.
* После единичной прошивки BL последующая прошивка BL не требуется (BL D5 является Katapult-прошивкой).
* BL - это лишь загрузчик, который используется для загрузки Katapult-прошивки, поэтому необходимо прошить также Klipper-прошивку.
::>



## Определение нужной вам прошивки

    :::tip

    * Обратите внимание, что нужно прошивать только одну из двух доступных прошивок.

    :::

    * D5 можно подключить к ПК двумя способами:
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка (самый распространенный вариант)</Button>
        * Преимущество: надежная и стабильная передача данных.
        * Недостаток: может быть влияние на передачу данных при наличии нескольких USB-устройств.
    * <Button variant="contained" disableElevation href="./can">USB-мостовая прошивка (обычно используется при работе с CAN-платой)</Button>
        * Преимущество: может подключать несколько устройств без использования порта USB.
        * Недостаток: возможны ошибки, такие как превышение времени ожидания при использовании совместно с `KlipperScreen`.
