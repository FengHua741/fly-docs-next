# Прошивка Katapult

## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    * Устройства D5, выпущенные с 1 сентября 2024 года, уже предустановлены прошивкой Katapult
    :::

## Вход в режим DFU для прошивки

    :::tip

    Слева — кнопка «BOOT», справа — кнопка «RESET»

    :::

    * Есть два способа войти в режим прошивки
    * Первый: при полном отключении питания, удерживайте кнопку BOOT и подключите кабель передачи данных к материнской плате и вашему устройству
    * Второй: после подключения кабеля передачи данных к материнской плате и вашему устройству, удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST, затем отпустите кнопку BOOT
