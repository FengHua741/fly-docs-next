    * Нажмите клавишу `Q`, чтобы появилось окно **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (процесс может занять некоторое время)
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Переключение в режим прошивки

    :::tip Подсказка
        После 1 сентября 2024 года出厂ные D5 уже имеют прошивку Katapult
    :::
    * Подключите основную плату к хост-компьютеру с помощью Type-C кабеля, затем **дважды нажмите кнопку reset**; при этом на плате загорится мигающий светодиод.
    :::warning Предупреждение
       * Если светодиод не мигает, перепрошлите прошивку Katapult: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-flashing-mode">Прошивка Katapult</Button>
       * Если светодиод не мигает, после прошивки Katapult выполните следующий шаг
    :::
