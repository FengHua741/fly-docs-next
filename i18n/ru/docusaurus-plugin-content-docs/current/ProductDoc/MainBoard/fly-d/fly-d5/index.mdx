# Введение

> Mellow Fly-D5 — это высокопроизводительная плата для 3D-принтера, разработанная командой по разработке материнских плат Fly3D компании Guangzhou Meilun Electronics Technology Co., Ltd., поддерживающая до 5 моторов шагового двигателя.

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/d5.webp').default} size="60%" align="center" />

## **1. Основные особенности**

* Использует 32-битный контроллер STM32F072RBT6 с тактовой частотой 48 МГц из серии ARM Cortex-M0+
* Поддерживает подключение CAN-шины для более стабильной передачи данных с меньшей задержкой
* Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и других
* На борту функция диагностики через вывод сигнальных ножек DIAG, позволяющая использовать бесконечную настройку без позиционирования через простое подключение/отключение переходников

## 2. Технические характеристики

* Микроконтроллер: STM32F072RBT6
* Фирменное ПО: Klipper/Marlin
* Поддерживает питание от 12 до 24В постоянного тока для более стабильной работы без необходимости использования понижающего модуля
* Режимы работы драйверов: SPI, UART, STEP/DIR
* Поддерживаемые драйверы: внешние TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и другие
* Расширительные интерфейсы: BLTouch (Servos, Probe), UART X1, 12864 X1
* Вентиляторы: два управляемых двухпроводных вентилятора X2, два параллельных вентилятора X2, два постоянно включенных вентилятора X2
* Датчики температуры: два канала NTC 100К
* Интерфейсы нагрева: два управляемых канала с поддержкой до 10А
* Поддерживаемые конструкции принтеров: Cartesian, Delta, Kossel, Ultimaker, CoreXY
