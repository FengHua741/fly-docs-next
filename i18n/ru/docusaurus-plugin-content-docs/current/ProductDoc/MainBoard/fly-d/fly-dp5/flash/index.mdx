# Инструкция по прошивке прошивки

:::danger Важно знать
* Прошивка материнской платы требует использования Katapult для прошивки Klipper
* Если вы уверены, что материнская плата уже прошита Katapult, то последующая прошивка Katapult не требуется — это просто мера предосторожности на случай неудачной прошивки
* Вы можете быстро дважды нажать кнопку сброса и проверить, загорается ли светодиод на материнской плате. Если да, то материнская плата находится в режиме Katapult
:::

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что достаточно прошить только одну из доступных прошивок

    :::
    
    * DP5 можно подключить к компьютеру следующим способом
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если имеется несколько USB-устройств, может происходить влияние на передачу данных
    * <Button variant="contained" disableElevation href="./can">Программная прошивка через CAN-мост</Button>
        * Преимущество: возможность подключения нескольких устройств без занятия портов USB
        * Недостаток: возможны ошибки, такие как превышение времени сброса при совместном использовании с `KlipperScreen`.
