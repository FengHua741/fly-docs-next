## Вход в режим прошивки

    :::tip

    Слева - кнопка `BOOT`, справа - кнопка `RESET`

    :::

    * Есть два способа входа в режим прошивки
    * Первый: при полном отсутствии питания на материнской плате, удерживайте кнопку BOOT и подключите передатчик данных между материнской платой и вашим устройством
    * Второй: после подключения передатчика данных между материнской платой и вашим устройством, удерживайте кнопку BOOT и нажмите кнопку RST, затем отпустите кнопку RST, а затем отпустите кнопку BOOT
