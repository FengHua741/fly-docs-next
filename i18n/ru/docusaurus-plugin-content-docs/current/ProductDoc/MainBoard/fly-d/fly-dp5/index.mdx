# Введение

> Mellow Fly-DP5 — это высокопроизводительная плата для 3D-принтера, разработанная командой Fly3D компании Guangzhou Meilun Electronic Technology Co., Ltd., которая поддерживает до 5 моторов шагового двигателя.

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-dp5/img/dp5.webp').default} size="60%" align="center" />

## **1. Основные особенности продукта**

* Используется 32-битный микроконтроллер STM32F072RBT6 с тактовой частотой 48 МГц из серии ARM Cortex-M0+
* Поддерживает подключение по CAN-шифру для более стабильной передачи данных с меньшей задержкой
* Драйверы: DP2209 и DP560
* На борту функциональная контактная ножка DIAG, позволяющая использовать безграниченную поддержку возврата на ноль путем простого подключения или отключения переходников

## 2. Технические характеристики

* Микроконтроллер: STM32F072RBT6
* Фирменное ПО: Klipper/Marlin
* Поддерживает питание от 12 до 24В постоянного тока для более стабильного питания без необходимости использования понижающего модуля
* Режим работы драйверов: SPI, UART, STEP/DIR
* Поддерживаемые драйверы: внешние TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и другие
* Расширительные интерфейсы: BLTouch (Servos, Probe), UART X1, 12864 X1
* Вентиляторы: два двухпроводных управляемых вентилятора X2, два параллельных вентилятора X2, два постоянно включенных вентилятора X2
* Датчики температуры: 2 канала NTC 100К
* Интерфейсы нагрева: два управляемых интерфейса с поддержкой до 10А
* Поддерживаемые конструкции принтеров: Cartesian, Delta, Kossel, Ultimaker, CoreXY
