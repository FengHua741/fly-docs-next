# Расширение возможностей управления светодиодами через плагин

Для установки и настройки требуется подключение по SSH к главному компьютеру в соответствии с приведенной ниже ссылкой

```cfg
https://github.com/julianschill/klipper-led_effect
```

## Установка

Подключитесь через **MobaXterm_Personal** или любое другое **SSH-приложение** по Wi-Fi к вашему главному компьютеру и убедитесь в следующем:

1. **Убедитесь, что на главном компьютере установлен фиксатор Klipper**
2. **Убедитесь, что вошедший пользователь является пользователем, установившим Klipper**
3. **Убедитесь, что ваша раскладка клавиатуры английская**
4. **Убедитесь, что ваш главный компьютер может нормально находить устройство**
5. **Убедитесь, что все вышеперечисленные注意事项 выполнены, иначе дальнейшие шаги невозможны**

Этот модуль можно установить в существующую установку Klipper с помощью установочного скрипта.

```cfg
cd ~
git clone https://github.com/julianschill/klipper-led_effect.git
cd klipper-led_effect
./install-led_effect.sh
```

## Примеры конфигурации

```cfg
[neopixel TP_led]
pin: PB7
chain_count: 25
# Количество светодиодов
# Количество светодиодов
color_order: GRB
initial_RED: 0.4    
initial_GREEN: 0.8
initial_BLUE: 1
initial_WHITE: 0.0
#66CCFF 

[led_effect sb_nozzle_cooling]
autostart:              false
frame_rate:             24
leds:
    neopixel:TP_led (9,10)
layers:
        breathing  3 1 top (0.0, 0.0, 1.0, 0.1)

[led_effect rainbow]
leds:
    neopixel:TP_led
autostart:                          true
frame_rate:                         24
layers:
    gradient  0.3  1 add (0.3, 0.0, 0.0),(0.0, 0.3, 0.0),(0.0, 0.0, 0.3)

```
