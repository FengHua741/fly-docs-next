# Описание PIN-ов

:::warning Важное замечание
Из-за высокого разрешения изображений просмотреть их на веб-странице невозможно. Пожалуйста, наведите курсор мыши на схему подключения, щелкните правой кнопкой мыши и сохраните изображение на рабочий стол для более детального просмотра!

### Переключатели на материнской плате драйвера

* **Режим STEP/DIR.** В этом режиме установка деления осуществляется с помощью перемычек, а регулировка тока производится с помощью потенциометра. Самые распространенные драйверы A4988 и 8825 требуют проверки таблицы деления, предоставленной производителем, а затем настройки деления с помощью перемычек.
* **Режим UART.** Самыми распространенными драйверами, использующими этот режим, являются TMC2208, TMC2209, TMC2226 и другие. Эти чипы могут взаимодействовать с контроллером через асинхронную последовательную связь UART, а параметры драйвера (деление, ток, тихий режим и др.) можно настроить через файл конфигурации.
* **Режим SPI.** К наиболее распространенным драйверам, использующим этот режим, относятся TMC5160, TMC2130, TMC2240 и другие. Параметры драйвера (деление, ток, тихий режим и др.) также можно настроить через файл конфигурации.
