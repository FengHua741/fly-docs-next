# Инструкция по прошивке прошивки

:::danger Важное примечание
* Прошивка основной платы требует прошивки Klipper через Katapult
* Если вы уверены, что плата уже прошита Katapult, то не нужно выполнять последний шаг прошивки прошивки Katapult — это просто на случай, если прошивка Katapult не удалась
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли один светодиод на плате, если да, то это означает, что плата уже находится в прошивке Katapult
:::

## Как определить нужную вам прошивку

    :::tip

    * Обратите внимание, что вам нужно прошить только одну из прошивок

    :::
    
    * Для D8 можно использовать следующие способы подключения к хост-компьютеру
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если есть несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Программатор CAN-шины</Button>
        * Преимущество: можно подключить несколько устройств без использования портов USB
        * Недостаток: совместное использование с `KlipperScreen` может вызывать ошибки, такие как превышение времени ожидания при возврате в исходное положение.
:::
