# Введение

> Mellow Fly-D8 — это высокопроизводительная плата для 3D-принтера, разработанная командой основной платы Fly3D компании Guangzhou Meilun Electronic Technology Co., Ltd. Данная плата поддерживает до 8 линий шаговых двигателей.

## Основные характеристики продукта

    * Интерфейсы платы с ограничителями и другими устройствами имеют изолированное питание 5В; если не подключить источник питания POWER, то питание не будет выдаваться
    * Использует 32-битный процессор ARM Cortex-M7 с тактовой частотой 550 МГц STM32H723
    * Поддерживает CAN-шина и RS232-соединение, что обеспечивает более стабильную передачу данных, меньшую задержку и более надежное подключение
    * На плате предусмотрены кнопки BOOT и RESET, пользователи могут через USB войти в режим DFU для обновления прошивки
    * Питание имеет защиту от подключения polarity inversion, чтобы предотвратить повреждение платы при неправильном подключении
    * Предоставляются два порта USB2.0, расширяя возможности подключения; один из портов USB может быть переключен на интерфейс RS232 через переключатель, чтобы соответствовать различным требованиям подключения устройств

# Технические характеристики

    * Центральный процессор: 32-битный процессор ARM Cortex-M7 с тактовой частотой 550 МГц STM32H723
    * Прошивка: Marlin 2.0 / Klipper
    * Поддержка драйверов: TMC: UART / SPI
    * Поддержка восьми драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и другие
    * Дисплей: сенсорный экран по последовательному интерфейсу, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Поддержка автоматической системы сенсоров для ровной печатной поверхности: BLTouch, PL08N и другие близкие сенсоры, Klicky, Voron Tap
    * Напряжение питания: поддерживает 12В / 24В
    * 6 интерфейсов ограничителей, обеспечивающих точное ограничение диапазона и управление положением
    * 2 RGB-интерфейса, которые могут предоставлять разнообразные цветовые эффекты
    * 3 интерфейса нагревательных элементов для независимого управления экструдером и теплым столом
    * 4 термисторных интерфейса, способные одновременно контролировать до четырех групп температур для точного контроля температуры
    * Интерфейсы USB: поддерживаются два порта USB2.0, один из которых может быть переключен на интерфейс RS232 через переключатель
    * Предоставляются два порта USB2.0, расширяя возможности подключения; один из портов USB может быть переключен на интерфейс RS232 через переключатель, чтобы соответствовать различным требованиям подключения устройств
