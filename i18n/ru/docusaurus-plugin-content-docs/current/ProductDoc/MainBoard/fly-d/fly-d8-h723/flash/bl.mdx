## Вход в режим программирования

    :::tip

    Слева — кнопка «RST», справа — кнопка «BOOT»

    :::

    * Есть два способа входа в режим программирования
    * Первый: полностью отключите электропитание материнской платы, удерживайте кнопку BOOT и подключите передатчик данных (одно из его концов к материнской плате, другое — к вашему устройству)
    * Второй: после подключения передатчика данных (одно из его концов к материнской плате, другое — к вашему устройству), удерживайте кнопку BOOT, нажмите кнопку RST, затем отпустите кнопку RST, а затем отпустите кнопку BOOT
