# RS232-подключение

Стандартный интерфейс RS232 (также известный как EIA RS-232) является одним из commonly используемых стандартов последовательной коммуникации, широко применяемым для передачи данных между различными устройствами. Его стабильность и надежность делают его предпочтительным вариантом коммуникации для многих устройств. Протокол использует сигналы электрического потенциала, отличные от стандартных 5В, для передачи данных по проводам, что позволяет максимально снизить влияние помех. В системах 3D-принтеров RS232 имеет более высокую скорость передачи данных, лучшую стабильность, более простую настройку и лучшую совместимость с ПК по сравнению с CAN.

## Использование инструментальной платы с подключением RS232 к основной плате

    * Обратите внимание, что питание инструментальной платы не подключается к проводам RS232, а подключается к линии питания 12V-24V

    * Фирменная прошивка D8 для материнской платы не требует перекомпиляции
