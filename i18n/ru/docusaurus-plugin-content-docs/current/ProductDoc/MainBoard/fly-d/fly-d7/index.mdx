# Введение

> Fly-D7 — это высокопроизводительная плата для 3D-принтера, разработанная командой по созданию основной платы Fly3D компании Guangzhou Mailing Electronic Technology Co., Ltd. Она поддерживает до 7 моторов шагового двигателя.

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d7/img/d7.webp').default} size="60%" align="center" />

## **1. Основные особенности продукта**

    * Использует 32-битный контроллер STM32F072RBT6 с тактовой частотой 48 МГц из серии ARM Cortex-M0+
    * Поддерживает подключение CAN-шины для более стабильной передачи данных с меньшей задержкой и более надежным соединением
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и других
    * Функция DIAG на плате позволяет использовать поддержку без ограничений при возврате к нулевому положению путем простого подключения или отключения jumper'а

## 2. Технические характеристики

    * MCU: STM32F072RBT6
    * Фирменное ПО: Klipper/Marlin
    * Поддержка питания 12-24В постоянного тока для более стабильной работы без необходимости установки понижающего модуля
    * Режимы работы драйверов: SPI, UART, STEP/DIR
    * Поддерживаемые драйверы: внешний TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и другие
    * Расширение интерфейсов: BLTouch (Servos, Probe), UART X1
    * Вентиляторы: два двухпроводных управляемых вентилятора X2, два параллельных вентилятора X2, два постоянно включенных вентилятора X2
    * Датчики температуры: три датчика NTC 100К
    * Интерфейсы нагрева: три управляемых интерфейса с поддержкой тока до 10А
    * Поддерживаемые конструкции машин: Cartesian, Delta, Kossel, Ultimaker, CoreXY
