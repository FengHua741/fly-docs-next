    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация сохранена, и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (время может занять较多):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующие строки, то компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Переключение в режим записи

    :::tip Совет
        После 1 сентября 2024 года出厂ные D7 уже имеют записанную прошивку Katapult
    :::
    * Подключите основную плату к компьютеру с помощью Type-C кабеля, затем **дважды нажмите кнопку reset**, при этом на плате загорится мигающий светодиод.
    :::warning Предупреждение
       * Если светодиод не горит, перепишите прошивку Katapult: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#вход_в_dfu_режим">Записать прошивку Katapult</Button>
       * Если светодиод не горит, выполните запись прошивки Katapult, а затем повторите шаг
    :::
