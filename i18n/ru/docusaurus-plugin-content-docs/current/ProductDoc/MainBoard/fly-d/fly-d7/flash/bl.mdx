# Прошивка Katapult


## Описание прошивки Katapult

    <CANBOOT />

    :::tip

    :::

## Вход в режим DFU для прошивки

    :::tip

    Слева — кнопка `BOOT`, справа — кнопка `RESET`.

    :::

    * Есть два метода входа в режим прошивки:
    * Первый: при полном отключении питания, удерживайте кнопку BOOT и подключите данные провод к материнской плате и вашему устройству.
    * Второй: после подключения данных проводов к материнской плате и вашему устройству, удерживайте кнопку BOOT и нажмите кнопку RST, затем отпустите кнопку RST, а затем отпустите кнопку BOOT.
