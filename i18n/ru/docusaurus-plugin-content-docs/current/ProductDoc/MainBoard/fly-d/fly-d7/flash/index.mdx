    Перед компиляцией и записью прошивки обязательно прочитайте этот раздел!!!!

:::tip Важные замечания
* D5 уже предустановлена прошивка Katapult с 1 сентября 2024 года
* После первой записи BL последующая запись BL не требуется (BL D5 — это Katapult-прошивка)
* BL — это только загрузчик, который используется для загрузки Katapult-программного обеспечения, нужно также записать прошивку Klipper
:::

## Определение необходимой прошивки

    :::tip

    * Обратите внимание, что вам нужно установить только одну из прошивок

    :::
    
    * D7 можно подключить к компьютеру следующими двумя способами:
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка (самый распространенный)</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если несколько USB-устройств, могут возникнуть проблемы с передачей данных
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB-мостовая прошивка (обычно используется при работе с CAN-платами)</Button>
        * Преимущество: может подключаться к нескольким устройствам без использования портов USB
        * Недостаток: возможны ошибки, такие как превышение времени ожидания при использовании вместе с `KlipperScreen`.
