## Начало компиляции

    <SSH />

    <USB />

    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (время может занять длительное)
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последним выводом будет следующее, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Переключение в режим прошивки

    :::tip Подсказка
        После 1 сентября 2024 года出厂ная D7 уже прошита Katapult-прошивкой
    :::
    * Подключите материнскую плату к хост-компьютеру с помощью Type-C-кабеля, затем **дважды нажмите кнопку reset**, при этом на материнской плате загорятся светодиоды.
    :::warning Предупреждение
       * Если светодиоды не загорелись, повторно выполните прошивку Katapult-прошивки: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#вход-в-режим-прошивки-dfu">Прошивка Katapult-прошивки</Button>
       * Если светодиоды не загорелись, после завершения прошивки Katapult-прошивки повторите следующий шаг
    :::
