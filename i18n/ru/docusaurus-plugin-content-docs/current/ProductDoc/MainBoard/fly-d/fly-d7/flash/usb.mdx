# Компиляция прошивки через USB


## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится запрос **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация сохранена, и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод завершения содержит следующее, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
## Переход в режим записи

    :::tip Подсказка
        После 1 сентября 2024 года выпущенные D7 уже имеют записанную прошивку Katapult
    :::
    * Подключите материнскую плату к хост-компьютеру с помощью Type-C кабеля, затем **дважды нажмите кнопку reset**, при этом на материнской плате загорится мигающий светодиод.
    :::warning Предупреждение
       * Если светодиод не мигает, выполните повторную запись прошивки Katapult: <Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#enter-dfu-flashing-mode">Записать прошивку Katapult</Button>
