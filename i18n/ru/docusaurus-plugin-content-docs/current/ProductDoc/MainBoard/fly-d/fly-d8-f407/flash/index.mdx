# Инструкция по прошивке прошивки

:::danger Важное примечание
* Прошивка материнской платы требует использования Katapult для прошивки Klipper
* Если вы уверены, что материнская плата уже прошита Katapult, то нет необходимости выполнять последний шаг по прошивке прошивки Katapult — это просто мера предосторожности на случай неудачной прошивки.
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли один светодиод на материнской плате. Если да, то это указывает на то, что материнская плата уже находится в режиме Katapult.
:::

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно установить только одну из этих прошивок

    :::
    
    * D8 можно подключить к компьютеру следующими способами
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная передача данных
        * Недостаток: если несколько USB-устройств могут повлиять на передачу данных
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка через CAN-мост</Button>
        * Преимущество: может подключаться несколько устройств без использования портов USB
        * Недостаток: при совместном использовании с `KlipperScreen` может вызывать ошибки, такие как превышение времени ожидания при возврате на исходную позицию.
