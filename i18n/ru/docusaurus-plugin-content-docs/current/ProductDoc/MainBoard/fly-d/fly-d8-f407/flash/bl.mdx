## Вход в режим программирования

    :::tip

    Слева — кнопка «RESET», справа — кнопка «BOOT»

    :::

    * Есть два способа входа в режим программирования
    * Первый: при полном отсутствии питания на материнской плате нажмите и удерживайте кнопку BOOT, затем подключите данные кабель с одной стороны к материнской плате, а с другой стороны к вашему устройству
    * Второй: после подключения данных кабеля с одной стороны к материнской плате, а с другой стороны к вашему устройству нажмите и удерживайте кнопку BOOT, затем нажмите кнопку RESET, отпустите кнопку RESET, затем отпустите кнопку BOOT
