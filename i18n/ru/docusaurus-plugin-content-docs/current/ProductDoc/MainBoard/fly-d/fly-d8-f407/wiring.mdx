# Справочник по подключению
## Описание PIN-ов

:::warning Важное замечание
Из-за высокого разрешения изображений просмотр веб-страницы может сделать их недоступными для четкого просмотра. Пожалуйста, наведите курсор мыши на схему подключения, щелкните правой кнопкой мыши и сохраните изображение на рабочий стол для более детального просмотра!

## Подключение основной платы

:::warning Важное замечание
Из-за высокого разрешения изображений просмотр веб-страницы может сделать их недоступными для четкого просмотра. Пожалуйста, наведите курсор мыши на схему подключения, щелкните правой кнопкой мыши и сохраните изображение на рабочий стол для более детального просмотра!
:::

:::warning Предупреждение
Не используйте готовые покупные трехжильные концевые выключатели! Такие выключатели могут вызвать короткое замыкание основной платы и повредить её!!! Подключайте провода строго в соответствии с указанными в схеме способами!!!
:::
