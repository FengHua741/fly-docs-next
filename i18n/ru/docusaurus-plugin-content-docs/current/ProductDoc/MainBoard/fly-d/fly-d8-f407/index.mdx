# Введение

> Mellow Fly-D8 — это высокопроизводительная плата для 3D-принтера, разработанная командой по разработке материнских плат Fly3D компании Guangzhou Meilun Electronic Technology Co., Ltd. Она поддерживает до 8 моторов шагового двигателя.

## Особенности продукта

    * Интерфейсы платы для ограничителей и т. д. работают на напряжении 5В с изоляцией и защитой. Если не подключить питание POWER, то питание не будет выводиться
    * Используется 32-битный процессор ARM Cortex-M4 серии с частотой 168 МГц, STM32F407-芯
    * Поддерживает CAN-шину и соединение RS232 для более стабильной передачи данных, меньшей задержки и более надежного подключения
    * На плате предусмотрены кнопки BOOT и RESET, пользователи могут обновлять прошивку через USB в режиме DFU
    * Есть защита от подключения питания в обратном направлении на разъеме питания, чтобы предотвратить повреждение платы при неправильном подключении проводов во время DIY
    * Предоставлено два интерфейса USB2.0, что расширяет возможности подключения, один из портов USB может переключаться на интерфейс RS232 с помощью выключателя, чтобы соответствовать различным требованиям подключения устройств

# Параметры продукта

    * Центральный процессор: 32-битный процессор ARM Cortex-M4 серии с частотой 168 МГц, STM32F407ZGT6
    * Фирменное программное обеспечение: Marlin 2.0/Klipper
    * Поддержка режимов управления драйверами: TMC: UART/SPI
    * Поддержка восьми драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и т. д.
    * Дисплей: сенсорный экран последовательной связи, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * Поддержка автоматической системы детектора уровня стола: BLTouch, PL08N и другие близкие датчики, Klicky, Voron Tap и т. д.
    * Входное напряжение: поддерживает 12В/24В
    * 6 интерфейсов ограничителей, обеспечивающих точное ограничение диапазона и управление положением
    * 2 RGB-интерфейса, способные обеспечивать многоцветные визуальные эффекты
    * 3 интерфейса нагревательных элементов для независимого управления экструдером и теплым столом
    * 4 термочувствительных интерфейса, позволяющие одновременно контролировать до четырех групп температур для точного контроля температуры
    * Интерфейсы USB: поддержка двух портов USB2.0, один из которых может переключаться на интерфейс RS232 с помощью выключателя
    * Предоставлено два интерфейса USB2.0, что расширяет возможности подключения, один из портов USB может переключаться на интерфейс RS232 с помощью выключателя, чтобы соответствовать различным требованиям подключения устройств
