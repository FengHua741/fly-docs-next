# CDY V3_V2

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-cdy/fly-cdy-v3/img/cdy.webp').default} size="80%" align="center" />

## Основные особенности

    * 32-битный ARM Cortex-M4 серии с тактовой частотой 168 МГц, чип STM32F407ZGT6
    * Поддержка выбора напряжения питания, по умолчанию 5В
    * Поддержка прошивки RRF



## Технические характеристики

    * 32-битный ARM Cortex-M4 серии с тактовой частотой 168 МГц, чип STM32F407ZGT6
    * Поддержка подключения CAN-шины для более стабильной передачи данных, меньшей задержки и более надежного соединения
    * Прошивка: Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и т.д.
    * Режимы работы драйверов: TMC: UART/SPI
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * При использовании прошивки Repap можно использовать расширяемый модуль ESP32 WiFi с поддержкой скорости до 1.6 м/с
    * Уникальная конструкция платы с MOS-транзистором для вентилятора PWM, что позволяет легко заменять поврежденные элементы и снижает сложность ремонта для клиентов
    * Платы выполнены из меди толщиной 2OZ, что обеспечивает поддержку больших токов
    * 3 интерфейса ADC, поддерживающие до трех термисторов
    * 6 цифровых интерфейсов IO, которые могут использоваться для выравнивания и ограничений
    * 4 интерфейса нагревательных элементов, поддерживающих до четырех нагревательных элементов
