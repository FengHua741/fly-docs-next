# ПО загрузчика

## Вход в режим программирования

    :::tip
 
    Справа — кнопка BOOT, слева — кнопка RESET

    :::

    * Есть два способа входа в режим программирования
    * Первый: при полностью отключенном питании платы, удерживайте кнопку BOOT и подключите USB-кабель с возможностью передачи данных к плате и устройству
    * Второй: после подключения USB-кабеля с возможностью передачи данных к плате и устройству, удерживайте кнопку BOOT, нажмите кнопку RST, затем отпустите кнопку RST и последним отпустите кнопку BOOT
