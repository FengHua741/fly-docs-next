# Компиляция и запись прошивки

    :::tip

    * По умолчанию CDV использует TF-карту для записи прошивки
    * Подготовьте TF-карту объемом не более `32 ГБ`
    * Использование неизвестных брендов может привести к неудачной записи, рекомендуется использовать следующие рекомендованные TF-карты для обновления прошивки
    * Скорость передачи данных TF-карты должна быть как минимум `class10`, рекомендуется использовать TF-карты от SanDisk и других известных брендов

    :::
