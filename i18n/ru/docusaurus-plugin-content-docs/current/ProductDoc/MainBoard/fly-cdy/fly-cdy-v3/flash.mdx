# Компиляция и запись прошивки

    :::tip

    * По умолчанию CDV использует TF-карту для записи прошивки
    * Подготовьте TF-карту объемом не более `32 ГБ`
    * Некачественные карты памяти могут привести к неудачной записи, поэтому рекомендуется использовать следующие бренды TF-карт для обновления прошивки
    * Лучше всего использовать TF-карты класса `10` или выше, например, от производителя SanDisk

    :::
