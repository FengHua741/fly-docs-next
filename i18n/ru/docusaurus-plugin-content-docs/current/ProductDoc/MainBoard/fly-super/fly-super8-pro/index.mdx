# Параметры продукта

    * Используется 32-битный контроллер STM32H723ZGT6 с тактовой частотой 550 МГц из серии ARM Cortex-M7
    * Прошивка: Marlin 2.0/Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и других
    * Режимы работы драйвера TMC: UART/SPI
    * 8 гнезд высокого напряжения, поддерживают управление 8 независимыми моторами, 5 экструдерами и 10 PWM вентиляторами
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 вход HIGH VOLTAGE, поддерживает автоматическую плоскостность сенсоров: BLTouch, PL08N и других близких сенсоров, Klicky, Voron Tap и т.д.
    * Необязательное питание концевых выключателей: 5В/12В/24В
    * При использовании прошивки Reprap можно использовать встроенный модуль WiFi ESP32, скорость до 1.6 м/с
    * Уникальный дизайн платы MOS для PWM вентиляторов, что позволяет легко заменять поврежденные части и упрощает ремонт клиентов; два разъема 2-pin могут поддерживать 4-pin PWM вентиляторы
    * PCB изготовлено из меди толщиной 2 OZ, что позволяет поддерживать больший ток
    * 6 интерфейсов ADC, поддерживающих до шести термисторов
    * 8 интерфейсов IO, которые можно использовать для выравнивания и ограничений
    * 5 интерфейсов нагревательных элементов, поддерживающих до пяти нагревательных элементов
    * 2 интерфейса PMW
