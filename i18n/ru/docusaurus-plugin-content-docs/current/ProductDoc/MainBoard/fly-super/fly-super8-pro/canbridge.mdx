# CAN-мост

CAN-шина - это последовательный протокол связи, используемый для реального времени, который может передавать сигналы по витой паре и является одним из наиболее распространенных в мире полевых шин. Протокол CAN используется для связи между различными компонентами в автомобиле, заменяя дорогие и громоздкие распределительные жгуты проводов. Благодаря своей надежности протокол нашел применение и в других автоматизированных и промышленных приложениях. Характеристики протокола CAN включают полную последовательную передачу данных, поддержку реального времени, скорость передачи до 1Mb/s, 11-битную адресацию и возможность обнаружения ошибок.
В 3D-принтерах CAN-шина используется для уменьшения количества проводов, необходимых для подключения экструдера к основной плате. Вместо десятка проводов с использованием CAN достаточно четырех, что значительно уменьшает количество проводки и облегчает установку.
В данной главе дается краткий обзор использования CAN-моста.

## Подключение инструментальной платы к основной плате через CAN

    * Обратите внимание, что питание инструментальной платы не подключается к CAN-порту, а к линии питания 12V-24V
