# Фирменное ПО BL

    :::tip

    * Обратите внимание, что при производстве уже предварительно записан загрузчик `bootloader`, поэтому повторная запись не требуется.

    :::
## Вход в режим программирования


    * Существует два способа входа в режим программирования
    * Первый: полностью отключите питание материнской платы, затем соедините BOOT, а затем подключите передатчик данных с одной стороны к материнской плате, а с другой стороны к вашему устройству.
    * Второй: после подключения передатчика данных с одной стороны к материнской плате, а с другой стороны к вашему устройству, коротко замкните BOOT, затем нажмите кнопку RST, затем отпустите кнопку RST и в конце отсоедините короткое замыкание на BOOT jumper.
