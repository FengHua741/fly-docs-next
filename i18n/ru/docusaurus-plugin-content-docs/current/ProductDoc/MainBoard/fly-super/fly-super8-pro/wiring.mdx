# Подключение материнской платы

:::warning Важно
Из-за высокого разрешения изображения, просмотр на веб-странице может быть затруднен. Пожалуйста, поместите курсор мыши на схему подключения, щелкните правой кнопкой мыши и выберите "Сохранить изображение как" на рабочий стол для просмотра!
:::

:::warning Напоминание
Пины Super8 pro идентичны пинам super8, поэтому здесь мы напрямую ссылаемся на схему подключения super8, без дополнительных пояснений.
:::
