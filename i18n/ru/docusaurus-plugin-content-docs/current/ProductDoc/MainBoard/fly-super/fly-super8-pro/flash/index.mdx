    :::tip

    * По умолчанию Super8 PRO использует microSD-карту для прошивки прошивки
    * Подготовьте microSD-карту емкостью не более `32 ГБ`
    * Использование поддельных карт памяти может привести к неудачной прошивке, постарайтесь использовать рекомендованные ниже карты памяти для обновления прошивки
    * Лучше использовать карту памяти класса `10` или выше, рекомендуется использовать карты памяти SanDisk и т. д.

    :::

# Инструкция по прошивке прошивки

    :::tip

    * При выдаче Super8 Pro уже предварительно записана загрузочная программа на microSD-карту, повторное записывание не требуется

## Как определить нужную вам прошивку

    :::tip

    * Обратите внимание, что нужно установить только одну из прошивок

    :::
    
    * Super8 PRO можно подключить к ПК следующим образом
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Преимущество: надежная связь, стабильность
        * Недостаток: если несколько USB-устройств могут повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мост прошивки</Button>
        * Преимущество: можно подключить несколько устройств без использования порта USB
        * Недостаток: возможны ошибки, такие как превышение времени ожидания при использовании вместе с `KlipperScreen`.
