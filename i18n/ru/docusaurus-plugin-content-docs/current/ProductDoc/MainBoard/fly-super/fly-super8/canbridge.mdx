# CAN мост

CAN-шина - это серийный протокол связи, используемый для приложений реального времени. Она может использовать витую пару для передачи сигналов и является одной из самых распространенных полевых шин в мире. CAN-протокол применяется для общения между различными компонентами в автомобиле, заменяя дорогие и громоздкие кабельные жгуты. Его надежность позволила расширить его использование на другие автоматизированные и промышленные приложения. Характеристики CAN-протокола включают в себя целостность серийной передачи данных, поддержку в реальном времени, скорость передачи до 1Mb/s, а также 11-битную адресацию и возможность обнаружения ошибок.
3D-принтеры используют CAN-шину для уменьшения количества проводов, подключающих печатающую головку к материнской плате. Вместо десятков проводов, с использованием CAN требуется только четыре провода, что значительно снижает количество проводов и упрощает прокладку кабелей.
В этой главе дается краткий обзор использования моста CAN.
