# ПО для BL

    :::tip

    * Обратите внимание, что на заводе уже предварительно записаны `bootloader`, поэтому не нужно повторно записывать его

    :::

## Вход в режим программирования


    * Существует два способа входа в режим программирования
    * Первый: при полном отключении питания от материнской платы коротко замкните BOOT, а затем подключите один конец с данными к материнской плате, а другой конец к вашему устройству
    * Второй: после подключения одного конца с данными к материнской плате, а другого к вашему устройству, коротко замкните BOOT, нажмите кнопку RST, затем отпустите кнопку RST и finally отсоедините короткое замыкание BOOT
