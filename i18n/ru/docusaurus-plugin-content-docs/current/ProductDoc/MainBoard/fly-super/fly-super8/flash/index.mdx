    :::tip

    * По умолчанию Super8 использует карту microSD для прошивки прошивки
    * Приготовьте карту microSD емкостью не более `32 ГБ`
    * Использование карт памяти неизвестных брендов может привести к неудачной прошивке, поэтому рекомендуется использовать рекомендованные ниже карты памяти для обновления прошивки
    * Лучше всего использовать карты памяти класса `10` или выше, например, карты памяти SanDisk

    :::

# Инструкция по прошивке прошивки


## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только одну из прошивок

    :::
    
    * Super8 можно подключить к ПК следующими способами
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если есть несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мостовая прошивка</Button>
        * Преимущество: может подключать несколько устройств без использования USB-портов
        * Недостаток: при совместном использовании с `KlipperScreen` могут возникать ошибки, такие как превышение времени ожидания при возвращении домой.
