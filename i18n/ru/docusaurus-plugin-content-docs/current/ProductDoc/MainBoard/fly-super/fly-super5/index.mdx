# Параметры продукта

    * Используется 32-битный контроллер STM32H723ZGT6 с тактовой частотой 550 МГц из серии ARM Cortex-M7
    * Прошивка: Marlin 2.0/Reprap/Klipper
    * Поддерживаемые драйверы: A4988, LV8729, DRV8225, TMC2208/2209/5160 и т. д.
    * Режимы работы драйверов TMC: UART/SPI
    * 5 высоковольтных выходов драйвера, поддерживают управление 5 независимыми двигателями
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 вход высокого напряжения, поддерживает автоматические сенсоры плоскости: BLTouch, PL08N и другие сенсоры близкого расстояния, Klicky, Voron Tap и т. д.
    * Выборочное питание ограничителя хода: 5В/12В/24В
    * При использовании прошивки Repetier можно использовать встроенный модуль WiFi ESP32 на плате, поддерживающий скорость до 1.6 м/с
    * Уникальная конструкция платы PWM вентилятора с MOSFET, легко заменяемая при повреждении, снижает сложность ремонта для клиентов; два разъема 2-pin могут работать с 4-pin PWM вентиляторами
    * Платы PCB изготовлены из меди толщиной 2 OZ, способны выдерживать большие токи
    * 3 интерфейса ADC, поддерживают до трех термисторов
    * 7 цифровых интерфейсов IO, могут использоваться для выравнивания и ограничений хода
    * 3 интерфейса нагревательных элементов, поддерживают до трех нагревательных элементов
