# Фирменное ПО BL

    :::tip

    * Обратите внимание, что на заводе уже предварительно записан `bootloader`, повторная запись не требуется.

    :::

## Вход в режим программирования

    :::tip

    Справа находится `переключатель BOOT`, слева — `кнопка RESET`.

    :::

    * Есть два способа входа в режим программирования
    * Первый: при полностью отключенном питании устройства, удерживайте кнопку BOOT и подключите USB-кабель с функцией передачи данных к материнской плате и вашему устройству.
    * Второй: после подключения USB-кабеля с функцией передачи данных к материнской плате и вашему устройству, удерживайте кнопку BOOT и нажмите кнопку RST, затем отпустите кнопку RST, а затем отпустите кнопку BOOT.
