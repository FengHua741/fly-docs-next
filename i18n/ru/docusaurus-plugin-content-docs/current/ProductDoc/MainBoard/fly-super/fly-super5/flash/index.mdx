    :::tip

    * Super5 Pro по умолчанию использует microSD-карту для прошивки прошивки
    * Подготовьте microSD-карту емкостью не более `32 ГБ`
    * Неоригинальные карты памяти могут привести к сбоям при прошивке, постарайтесь использовать рекомендованные ниже карты памяти для обновления прошивки
    * Скорость записи microSD-карты должна быть как минимум `class10` или выше, рекомендуется использовать карты памяти от SanDisk и других надежных брендов

    :::

# Инструкция по прошивке прошивки

    :::tip

    * При производстве Super5 Pro уже предварительно записан загрузчик на microSD-карту, поэтому вам не нужно повторно его прошивать

    :::


## Как определить нужную вам прошивку

    :::tip

    * Обратите внимание, что вам нужно установить только одну из доступных прошивок

    :::
    
    * Super5 Pro может подключаться к ПК следующими способами
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если подключено несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB-мостовая прошивка</Button>
        * Преимущество: может подключать несколько устройств без использования порта USB
        * Недостаток: совместное использование с `KlipperScreen` может вызывать ошибки, такие как превышение времени ожидания при возврате домой.
