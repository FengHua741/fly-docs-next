# Фирменное ПО BL

    :::tip

    * Обратите внимание, что при出厂 поставке уже предварительно записан `bootloader`, поэтому повторная запись не требуется.

    :::

## Вход в режим записи

    :::tip

    Справа находится кнопка `BOOT`, слева — кнопка `RESET`.

    :::

    * Есть два способа входа в режим записи
    * Первый: при полном отключении питания от главной платы, удерживайте кнопку BOOT и подключите кабель передачи данных к плате и вашему устройству.
    * Второй: после подключения кабеля передачи данных к плате и вашему устройству, удерживайте кнопку BOOT и нажмите кнопку RST, затем отпустите кнопку RST, а затем отпустите кнопку BOOT.
