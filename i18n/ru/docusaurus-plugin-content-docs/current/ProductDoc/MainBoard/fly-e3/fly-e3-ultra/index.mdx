## Технические характеристики

* 32-битный ARM Cortex-M4 с тактовой частотой 550 МГц, чип STM32H723VGT6
* Фиксированное ПО: Reprap/Klipper
* На борту пять драйверов TMC2209, три драйвера XYZ поддерживают режим бесконечной подачи
* Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
* 1 высоковольтный вход, поддерживает автоматические датчики плоскости: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и т.д.
* При использовании фиксированного ПО Repap можно использовать модуль ESP32 Wi-Fi с поддержкой скорости до 1.6 м/с
* Уникальный дизайн платы PWM вентилятора с MOS, который можно заменить непосредственно, снижая сложность ремонта клиентов, четыре интерфейса вентилятора на 2 пина
* 3 интерфейса ADC для датчиков температуры
3 интерфейса ограничения для подключения переключателей ограничения
* 4 интерфейса IO, могут использоваться для датчиков приближения, ограничений, лазерных головок, RGB-ламп и т.д.
* 3 интерфейса нагрева
* 1 интерфейс последовательного экрана для RRF
* Версия Max поддерживает 1 K-type термопару и 1 датчик температуры PT100
* Новая функция переключателя拨码可用于 выбора режима 2/3/4 проводов датчика температуры PT100
* Новая интегрированная散热片 увеличивает площадь охлаждения драйверов
* Новый интерфейс CAN
