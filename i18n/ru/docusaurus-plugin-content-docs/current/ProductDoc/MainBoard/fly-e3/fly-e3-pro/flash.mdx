# Компиляция и запись прошивки

    :::tip

    * По умолчанию E3 Pro использует microSD-карту для записи прошивки
    * Приготовьте microSD-карту объемом не более `32 ГБ`
    * Неисправные карты памяти могут привести к неудачной записи, поэтому рекомендуется использовать карты памяти из рекомендованного списка
    * Лучше всего использовать карты памяти класса `10` или выше, такие как SanDisk

    :::
