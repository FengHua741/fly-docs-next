# Компиляция и запись прошивки

    :::tip

    * E3 Pro по умолчанию использует TF-карту для записи прошивки
    * Приготовьте TF-карту емкостью не более `32 ГБ`
    * Использование неизвестных брендов карт может привести к сбоям при записи, поэтому рекомендуется использовать рекомендованные ниже бренды TF-карт
    * Лучше всего использовать карты класса `10` или выше, например, карты SanDisk

    :::


    * Если вывод содержит следующее, компиляция успешна

        ```bash
        Linking out/klipper.elf

        ```
