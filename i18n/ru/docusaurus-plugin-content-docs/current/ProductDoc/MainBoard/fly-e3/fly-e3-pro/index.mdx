## Основные характеристики

    * 32-битный ARM Cortex-M4 с частотой 168 МГц, чип STM32F407ZGT6
    * Поддержка прошивки RRF



## Технические параметры

    * 32-битный ARM Cortex-M4 с частотой 168 МГц, чип STM32F407ZGT6
    * Фирменное ПО: Reprap/Klipper
    * На борту пять драйверов TMC2209
    * Дисплей: последовательный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 высоковольтный вход, поддерживает автоматические датчики плоскости: BLTouch, PL08N и другие сенсоры приближения, Klicky, Voron Tap и т.д.
    * При использовании прошивки repap возможно использование модуля ESP32 Wi-Fi, скорость до 1.6 м/с
    * Уникальный дизайн платы PWM вентилятора с MOS, который можно легко заменить, снижая сложность ремонта для клиентов, четыре интерфейса вентилятора формата 2 pin
    * PCB с медью толщиной 2OZ, способная поддерживать больший ток
    * 2 интерфейса ADC, поддерживающие до двух термисторов
    * 4 цифровых интерфейса IO, могут быть использованы для выравнивания и ограничений
    * 3 интерфейса нагревательных элементов, поддерживают до трех нагревательных элементов
