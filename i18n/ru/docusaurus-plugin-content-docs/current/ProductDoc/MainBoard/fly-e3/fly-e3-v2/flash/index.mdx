    :::tip

    * По умолчанию для E3 V2 используется карта памяти TF для прошивки прошивки
    * Подготовьте карту памяти TF с объемом не более `32 ГБ`
    * Неисправные карты памяти могут привести к неудачной прошивке, рекомендуется использовать следующие рекомендованные карты памяти для обновления прошивки
    * Лучше всего использовать карту памяти класса `10` или выше, рекомендуется использовать карты памяти SanDisk и т. д.

    :::

# Инструкция по прошивке прошивки


## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только одну из прошивок

    :::
    
    * E3 V2 может подключаться к компьютеру следующими способами
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если есть несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB-мост прошивки</Button>
        * Преимущество: можно подключать несколько устройств без использования портов USB
        * Недостаток: возможно возникновение ошибок, таких как превышение времени ожидания при использовании совместно с `KlipperScreen`.
