## Переход в режим программирования

    :::tip

    Справа — кнопка `BOOT`, слева — кнопка `RESET`

    :::

    * Есть два способа входа в режим программирования
    * Первый: при полностью отключенном питании, удерживайте кнопку BOOT и подключите передатчик данных к материнской плате и вашему устройству
    * Второй: после подключения передатчика данных к материнской плате и вашему устройству, удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST, затем отпустите кнопку BOOT
