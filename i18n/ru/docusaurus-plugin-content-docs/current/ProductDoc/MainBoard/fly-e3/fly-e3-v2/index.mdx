## Параметры продукта

    * 32-битный процессор ARM Cortex-M4 серии с тактовой частотой 168 МГц, чип STM32F407ZGT6
    * Поддержка CAN шины для более стабильной передачи данных с меньшей задержкой и более надежным подключением
    * Прошивка: Reprap/Klipper
    * Поддержка драйверов: A4988, LV8729, DRV8225, TMC2208/2209/5160 и т.д.
    * Режим работы драйверов: TMC: UART/SPI
    * Дисплей: серийный сенсорный экран, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 вход высокого напряжения, поддерживающий автоматические сенсоры плоскости: BLTouch, PL08N и другие датчики приближения, Klicky, Voron Tap и т.д.
    * При использовании прошивки RepRap можно использовать расширенный модуль ESP32 Wi-Fi с поддержкой скорости до 1.6 м/с
    * Уникальная конструкция платы PWM вентилятора с MOSFET, которая позволяет легко заменять поврежденные элементы, снижая сложность ремонта клиентов; четыре разъема 2-pin для вентиляторов
    * Платы PCB выполнены из меди толщиной 2 OZ, что позволяет выдерживать большие токи
    * 2 интерфейса ADC, поддерживающие до двух термисторов
    * 4 цифровых порта, которые могут использоваться для выравнивания поверхности и ограничений
    * 3 интерфейса нагревательных элементов, поддерживающих до трех нагревательных элементов
