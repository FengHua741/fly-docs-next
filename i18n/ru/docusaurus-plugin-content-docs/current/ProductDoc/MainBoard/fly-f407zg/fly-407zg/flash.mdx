# Компиляция и запись прошивки

    :::tip

    * По умолчанию для F407ZG используется запись прошивки через TF-карту
    * Приготовьте TF-карту объемом не более `32 ГБ`
    * Нестандартные карты памяти могут привести к неудаче записи, поэтому рекомендуется использовать рекомендованные ниже карты памяти для обновления прошивки
    * Лучше всего использовать TF-карты класса `10` или выше, например, от производителя SanDisk

    :::

    * Теперь конфигурация должна быть сохранена, и вы должны вернуться в командную строку
    * Введите `make -j4`, чтобы начать компиляцию (время займет достаточно долго)
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, то компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
