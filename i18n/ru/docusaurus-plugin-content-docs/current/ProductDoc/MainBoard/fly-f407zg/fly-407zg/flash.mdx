# Компиляция и запись прошивки

    :::tip

    * По умолчанию для 407ZG используется запись прошивки через TF-карту
    * Приготовьте TF-карту объемом не более `32 ГБ`
    * Неисправные карты памяти могут привести к неудачной записи прошивки, постарайтесь использовать рекомендованные ниже карты памяти для обновления прошивки
    * Лучше всего использовать TF-карты класса `class10` или выше, рекомендуется использовать карты памяти производства SanDisk

    :::

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернетесь в командную строку
    * Введите следующую команду для начала компиляции (процесс займет некоторое время)
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, компиляция успешна

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
