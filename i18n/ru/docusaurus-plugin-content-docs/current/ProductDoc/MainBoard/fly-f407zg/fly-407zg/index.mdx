## Параметры продукта

    * 32-битный ARM Cortex-M4 серии с тактовой частотой 168 МГц, чип STM32F407ZGT6
    * Поддержка подключения CAN-шины для более стабильной передачи данных с меньшей задержкой и более надежным соединением
    * Прошивка: Reprap/Klipper
    * Поддерживаемые драйверы: A4988, LV8729, DRV8225, TMC2208/2209/5160 и другие
    * Режим работы драйверов: TMC: UART/SPI
    * Дисплей: сенсорный экран по последовательному интерфейсу, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * При использовании прошивки Repap можно использовать расширяемый модуль ESP32 Wi-Fi с поддержкой скорости до 1.6 м/с
    * Уникальный дизайн платы PWM вентилятора с MOS-транзистором, который можно заменить непосредственно при выходе из строя, снижая сложность ремонта для клиентов
    * Плата PCB с медью толщиной 2OZ, способная поддерживать больший ток
    * 6 интерфейсов ADC, поддерживающих до шести термодатчиков
    * 7 интерфейсов IO, которые могут использоваться для выравнивания и ограничений
    * 7 интерфейсов нагревательных элементов, поддерживающих до семи нагревательных элементов
