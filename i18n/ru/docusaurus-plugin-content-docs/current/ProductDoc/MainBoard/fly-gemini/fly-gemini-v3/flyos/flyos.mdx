## Установка антенн

    1. Установите антенну для модуля `M2WE` или модуля `2,4G WiFi`
    :::warning
    * Для работы Wi-Fi на FLY-компьютере требуется использование модуля `M2WE` или `2,4G WiFi`; без них Wi-Fi не будет работать
    * Обратите внимание, что компьютер не поддерживает сети Wi-Fi с русскими названиями и сети Wi-Fi, поддерживающие одновременно частоты 2,4 ГГц и 5 ГГц
    :::
    * Красная стрелка указывает на разъем антенны Wi-Fi
    * Разъем антенны Wi-Fi — `ipex1`
    * Установите антенну на M2WE, чтобы избежать слабого сигнала при подключении к маршрутизатору
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После установки системы карта можно перейти к следующему шагу — подключению к компьютеру через SSH
    :::

    2. Подайте питание на компьютер.
    **На этом этапе ваш компьютер уже может быть запущен!**
    Выполните следующий шаг: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">Подключение к компьютеру через SSH</Button>
