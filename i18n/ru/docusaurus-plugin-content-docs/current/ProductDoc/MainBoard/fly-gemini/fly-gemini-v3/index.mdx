# Гемини V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.webp').default} size="80%" align="center" />

## Особенности продукта
    * Наboard компьютер, может заменить Raspberry Pi в качестве компьютера верхнего уровня; оснащен 1 ГБ встроенной памяти DDR3.
    * Усиленная защита питания 5В, снижает вероятность повреждения чипов.
    * Отменена настройка карты памяти для микроконтроллера, что упрощает процесс прошивки.
    * Система предустановлена klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, модуль компенсации резонанса, numpy и FLY-Tools, не требует сложных операций типа замены источников, что делает его более доступным для новичков (требуется покупка карты памяти TF или M2WE).


## Технические характеристики

    * CPU: Высокоэффективный чип H5 Allwinner, 4-ядерный 64-битный Cortex-A53
    * GPU: Высокопроизводительный 6-ядерный Mali 450, производительность пиксельной шины превышает 2,7 gpixel/s
    * ОЗУ: 1 ГБ DDR3 (общий доступ с GPU)
    * ПЗУ: Поддерживает карты памяти SDCARD до 128 ГБ
    * Периферия: Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100М), CAN x1
    * Интерфейсы: Наboard M.2 интерфейс, расширяемый Wi-Fi, eMMC и т. д. (частная протокол, не подключайте не специализированные для FLY π устройства).
    * Поддерживает питание от 12-24В постоянного тока, более стабильное питание, исключая необходимость понижающего модуля
    * Предусмотрен управляемый разъем для вентилятора 5В, может использоваться для охлаждения компьютера верхнего уровня
    * Предусмотрен расширяемый интерфейс CAN
    * Система предустановлена klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, модуль компенсации резонанса, numpy и FLY-Tools, не требует сложных операций типа замены источников, что делает его более доступным для новичков (требуется покупка карты памяти TF или M2WE)
    * Драйвер: высоковольтный драйвер *4
    * MCU: STM32F405
