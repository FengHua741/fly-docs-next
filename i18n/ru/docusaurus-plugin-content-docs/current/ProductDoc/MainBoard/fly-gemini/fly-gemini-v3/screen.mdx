* FLY-TOOLS требует подключения к интернету через ПК для доступа по адресу `IP+:9999`
* Например: `192.168.6.110:9999`

## Метод первый    
         * Для настройки емкостного экрана используйте следующую команду для установки конфигурации `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Убедитесь, что переключатели на обратной стороне экрана установлены в положение `Cap`
    * Используйте следующую команду для проверки успешности изменения
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Для изменения конфигурации емкостного экрана на резистивный используйте следующую команду

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

           * Для настройки резистивного экрана используйте следующую команду для установки конфигурации `fly-tft-v2-r`


        * Используйте следующую команду для проверки успешности изменения
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
