# Инструкция по прошивке прошивки

## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только одну из прошивок

    :::
    
    * Для Gemini V3 доступны следующие способы подключения к ПК
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: может повлиять на передачу данных, если есть несколько USB-устройств
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB-мост прошивка</Button>
        * Преимущество: можно подключить несколько устройств без использования USB-портов
        * Недостаток: может вызвать ошибки, такие как превышение времени ожидания, при совместном использовании с `KlipperScreen`
