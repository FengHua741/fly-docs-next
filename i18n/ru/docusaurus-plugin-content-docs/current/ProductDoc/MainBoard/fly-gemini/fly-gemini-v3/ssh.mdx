# Подключение к ПК по SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другую **SSH-терминальную утилиту**
* Убедитесь, что карта SD или M2WE прошиты **соответствующей операционной системой ПК**
* **Если ПК не запускается нормально, отключите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**
* **В конце подключитесь к SSH через сеть для компиляции прошивки!**
* **Для компиляции прошивки см. руководство по подключению к ПК через Wi-Fi или проводной интерфейс**

        ## Подключение к ПК по SSH через сеть

        * Войдите в веб-интерфейс маршрутизатора
        * Убедитесь, что система уже **нормально запущена**
        * Убедитесь, что **вы знаете имя пользователя и пароль верхнего уровня ПК**, FLY по умолчанию: пользователь **fly**, пароль **mellow**

        * Откройте MobaXterm, настройте следующим образом, IP введите предыдущий просмотренный или посмотрите в веб-интерфейсе маршрутизатора
