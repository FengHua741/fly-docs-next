## Подготовка перед прошивкой системы

    * Чтение карты
    * TF-карта объемом не менее 16 ГБ и не более 128 ГБ, скорость передачи данных должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-Gemini">образ системы</Button>
    * Скачайте необходимый <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">инструмент для прошивки</Button>

## Прошивка системы

:::info
Прошивка TF-карты или прошивка через TF-карту в M2WE может выполняться любым из указанных способов
:::

:::info
* Если в машине установлены M2WE и карта памяти, система по умолчанию будет загружаться с карты памяти
* Для M2WE рекомендуется использовать TF-карту для прошивки, использование USB для прошивки может привести к несовместимости драйверов и возможной неудаче прошивки; рекомендуется использовать [прошивку с карточки памяти в M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Убедитесь, что скачанный образ системы совместим, иначе система не запустится
:::
