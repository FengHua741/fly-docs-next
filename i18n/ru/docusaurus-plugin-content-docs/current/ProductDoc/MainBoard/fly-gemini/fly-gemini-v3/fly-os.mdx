## Подготовка к записи системы

    * Картридер
    * Карта TF объемом не менее 16 ГБ и не более 128 ГБ, скорость передачи данных должна быть класса 10 или выше
    * Загрузить соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">образ системы</Button>
    * Загрузить необходимые <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструменты для записи</Button>

## Запись системы

:::info
Запись на TF карту и запись через TF карту на M2WE требует только один из этих методов
:::

:::info
* Если в устройстве установлены M2WE и карта памяти, система по умолчанию загружается с карты памяти
* Рекомендуется использовать TF карту для обновления системы на M2WE, так как запись через USB может не удаться из-за проблем с драйверами; рекомендуется использовать [запись с TF карты на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Убедитесь, что загруженный образ системы соответствует вашему устройству, иначе система не запустится
:::
