## Подготовка к системной записи

    * Картридер
    * Карта TF объемом не менее 16 ГБ и не более 128 ГБ, скорость передачи данных должна быть класса 10 или выше
    * Загрузите соответствующий <Button variant="contained" disableElevation href="/docs/ResDownload/system-img/fly-Gemini">системный образ</Button>
    * Загрузите необходимое <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">программное обеспечение для записи</Button>

## Системная запись

:::info
Запись на карту TF и запись на M2WE через карту TF - выбирайте только один метод
:::

:::info
* Если в устройстве установлены M2WE и карта памяти, система по умолчанию загружается с карты памяти
* Рекомендуется использовать карту TF для установки системы на M2WE, так как запись через USB на M2WE может завершиться неудачей из-за проблем с совместимостью драйверов. Рекомендуется использовать [запись с карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на то, соответствует ли загруженный системный образ, иначе система не запустится
:::
