# Сборка прошивки через USB

## Начало сборки

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, чтобы появилось **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы должны вернуться к командной строке
    * Введите `make -j4` для начала сборки, это может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце выводится следующий текст, значит сборка завершена успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Загрузка прошивки

:::tip
Выполните следующую команду для загрузки прошивки
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
