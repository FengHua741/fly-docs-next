# Компиляция прошивки через USB

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, чтобы появилось предложение **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (процесс может занять некоторое время)
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Запись прошивки

:::tip
Для записи прошивки выполните следующую команду
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
