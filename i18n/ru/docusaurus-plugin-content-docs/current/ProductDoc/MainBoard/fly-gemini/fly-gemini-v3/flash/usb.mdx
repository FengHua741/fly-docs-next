# Компиляция прошивки

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится предложение **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции (время может занять немного)
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого означает успешную компиляцию

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Загрузка прошивки

:::tip
Выполните следующую команду для загрузки прошивки
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
