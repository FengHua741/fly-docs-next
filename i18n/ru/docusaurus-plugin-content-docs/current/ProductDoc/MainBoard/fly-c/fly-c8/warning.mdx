# Заметки по материнской плите

## Индикаторы питания
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/power.webp').default} size="80%" align="center" />
    * При подаче питания только через TYPE-C, без подключения других источников питания материнской платы, должны загораться хотя бы два индикатора: `3.3V` и `5V`.
    * После подключения основного питания материнской платы индикаторы питания должны светиться на `3.3V`, `5V` и `VCC`.

### Устранение неполадок


    :::danger Важно

    * При отсутствии подключенных внешних устройств (драйверов, ограничителей, дисплеев 12864 и т. д.)
    * При подаче питания только через TYPE-C индикатор `3.3V` должен постоянно гореть.

    :::

    * Если после подключения основного питания материнской платы и питания драйвера индикатор `5V` не загорается, обратитесь в службу поддержки.

## Разъём для ограничителя

    * Обратите внимание на последовательность подключения. Для трёхпроводного ограничителя важно правильно подключить 5V, GND и S; если подключение выполнено неправильно, это может привести к повреждению материнской платы.
    * Если используется двухпроводной вариант, достаточно подключить S и GND.
    * Для PL08 и других близкорасположенных датчиков необходимо снаружи сигнального разъёма подключить диод для понижения напряжения до уровня ниже 3.3V.
