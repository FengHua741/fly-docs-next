## Установка антенны

    1. Установите антенну для модуля `M2WE` или `2.4G WiFi`
    :::warning
    * ПК верхнего уровня FLY требует использования модуля `M2WE` или `2.4G WIFI`, иначе невозможно использование Wi-Fi
    * Обратите внимание, что ПК верхнего уровня не поддерживает сети Wi-Fi с русскими именами, а также сети Wi-Fi, объединяющие 2.4G и 5G
    :::
    * Показанная красным стрелкой часть является разъемом для антенны Wi-Fi
    * Разъем Wi-Fi — это `ipex1`
    * Пожалуйста, установите антенну на M2WE, иначе из-за слабого сигнала может возникнуть проблема с подключением к маршрутизатору
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После установки системной карты можно перейти к следующему шагу, подключившись к ПК через SSH
    :::

    2. Подайте питание на ПК верхнего уровня.  
    **На этом ваш ПК верхнего уровня уже готов к работе!!!**  
    Перейдите к следующему шагу: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">Подключение к ПК через SSH</Button>
