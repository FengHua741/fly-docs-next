# Фирменное ПО BL

    :::tip

    * Обратите внимание, что при выпуске уже предварительно записан загрузчик `bootloader`, поэтому его не нужно переписывать

    :::

## Вход в режим программирования


    * Существует два способа входа в режим программирования
    * Первый: при полном отключении питания от主板, удерживайте BOOT и подключите шину передачи данных одной стороной к motherboard, другой стороной к вашему устройству, подходящий для WINDOWS
    * Второй: после подключения шины передачи данных одной стороной к motherboard, другой стороной к вашему устройству, удерживайте BOOT и введите команду, подходящий для позиционного устройства
