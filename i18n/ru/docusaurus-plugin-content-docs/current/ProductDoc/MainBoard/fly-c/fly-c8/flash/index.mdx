# Инструкция по прошивке прошивки

    
## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошить только одну из этих прошивок

    :::
    
    * C8 можно подключить к компьютеру следующим образом
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если есть несколько USB-устройств, это может повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мост прошивка</Button>
        * Преимущество: можно подключать несколько устройств без использования портов USB
        * Недостаток: при совместном использовании с `KlipperScreen`可能出现 сбои, такие как превышение времени ожидания и т. д.
