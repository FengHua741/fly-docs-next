# Запись прошивки USB-CAN моста

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4`, чтобы начать компиляцию, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее содержимое, то компиляция завершена успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Запись прошивки

:::tip
Для записи прошивки выполните следующую команду:
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
