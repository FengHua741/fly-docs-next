    * Чтение карты

## Прошивка системы

:::info
Прошивка TF-карты или прошивка через TF-карту на M2WE требует использования только одного из способов.
:::

:::info
* Если в устройстве установлен M2WE и карта памяти, система по умолчанию будет загружаться с карты памяти
* Рекомендуется использовать TF-карту для прошивки системы на M2WE, так как использование USB может привести к неудачной прошивке из-за совместимости драйверов; рекомендуется использовать [прошивку карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на соответствие скачанного образа системы; если он не соответствует, система не запустится
:::

## Установка антенны

    1. Установите антенну на модуль `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы Wi-Fi требуется использование модуля `M2WE` или `2.4G WIFI`, иначе Wi-Fi недоступен
    * Обратите внимание, что устройство не поддерживает сети Wi-Fi с русскими названиями и сети Wi-Fi, сочетающие 2.4G и 5G
    ::
    * Место с красной стрелкой указывает на разъем антенны Wi-Fi
    * Разъем антенны Wi-Fi — это `ipex1`
    * Убедитесь, что антенна установлена на M2WE, иначе возможны проблемы с сигналом, из-за чего невозможно будет подключиться к роутеру
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После установки системы карта может быть использована для следующего шага — подключения к ПК через SSH
    :::

    2. Подайте питание на устройство.  
    **Ваше устройство уже может быть запущено!**  
    Продолжите выполнение следующих шагов: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">Подключение к ПК через SSH</Button>
