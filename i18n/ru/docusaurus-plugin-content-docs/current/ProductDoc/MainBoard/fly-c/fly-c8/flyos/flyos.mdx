## Прошивка системы

:::info
Прошивка TF-карты или прошивка через TF-карту на M2WE требует использования только одного из методов прошивки
:::

:::info
* Если в устройстве установлен M2WE и карта памяти, система по умолчанию будет загружаться с карты памяти
* Рекомендуется использовать TF-карту для прошивки системы; прошивка через USB может завершиться неудачей из-за совместимости драйверов и т.д.; рекомендуется использовать прямую прошивку [карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Убедитесь, что скачанный образ системы соответствует вашему устройству, иначе система не сможет запуститься
:::

## Установка антенны

    1. Установите антенну для модуля `M2WE` или модуля `2.4G WiFi`
    :::warning
    * Для работы Wi-Fi на FLY-компьютере требуется использование модуля `M2WE` или `2.4G WiFi`, иначе Wi-Fi не будет работать
    * Обратите внимание, что на FLY-компьютере не поддерживаются сети Wi-Fi с русскими названиями, а также сети Wi-Fi, работающие одновременно на частотах 2.4G и 5G
    :::
    * Красная стрелка указывает на разъем антенны Wi-Fi
    * Разъем антенны Wi-Fi - это `ipex1`
    * Убедитесь, что антенна установлена на M2WE, иначе возможны проблемы с сигналом, что приведет к невозможности подключения к роутеру
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После установки системы карта может быть подключена к следующему шагу через SSH
    :::

    2. Подайте питание на FLY-компьютер.  
    **На этом этапе ваш FLY-компьютер уже готов к запуску!!**  
    Перейдите к следующему шагу: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">Подключение к FLY-компьютеру через SSH</Button>
