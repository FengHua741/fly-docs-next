# Подключение к ПК через SSH


* Можно использовать последовательный порт или Wi-Fi для подключения к устройству
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другой **SSH терминальный инструмент**
* Убедитесь, что SD-карта или M2WE записаны **соответствующей системой для ПК**
* **Если ПК не запускается нормально, отключите все внешние устройства, такие как драйверы, ограничители, вентиляторы и т. д.!**
* **В конце подключитесь к SSH через сеть для удобства компиляции прошивки!**
* **Для компиляции прошивки обратитесь к руководству по подключению ПК через Wi-Fi или проводной интерфейс**

        ## Подключение к ПК через IP сети SSH

        * Авторизуйтесь в бэкэнде маршрутизатора, к которому подключен ПК
        * Убедитесь, что система уже **нормально запущена**
        * Убедитесь, что вы знаете **имя пользователя и пароль ПК**, для системы FLY имя пользователя по умолчанию **fly**, пароль **mellow**
        * Если материнская плата уже запущена и подключена к сети, можно подключиться через IP к SSH
        * Откройте MobaXterm и выполните следующие настройки, указав IP, который видели ранее, либо найдите его в бэкэнде маршрутизатора

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на 4-м шаге вводится имя пользователя ПК**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении появится это окно, нажмите `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Запрос на ввод пароля, после ввода пароля ПК нажмите Enter

        **Внимание: здесь введённый пароль не отображается, просто нажмите Enter после ввода**

        По умолчанию для системы FLY пользователь: `fly`, пароль: `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />
