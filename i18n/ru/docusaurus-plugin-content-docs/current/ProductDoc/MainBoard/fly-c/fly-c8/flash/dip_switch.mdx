# Введение в переключатели кодов

:::tip
FLY Gemini подключается к хост-компьютеру через переключатель кодов. Если положение переключателей не соответствует требуемому, устройство не сможет нормально функционировать.
:::

:::danger Важно
Обратите внимание, что переключатели кодов необходимо установить в указанные позиции для правильного подключения к хост-компьютеру.
:::

## Подключение встроенной хост-системы

    * Переключите переключатели кодов `1` и `2` вправо, чтобы соединить нижний уровень с хост-компьютером через встроенное USB-соединение.
    * Переключите переключатели кодов `3` и `4` влево, иначе это может повлиять на соединение между нижним уровнем и хост-компьютером.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * После выполнения данной процедуры нижний порт USB будет занят нижним уровнем. Не подключайте другие устройства.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Подключение внешней хост-системы

    * Переключите переключатели кодов `3` и `4` вправо, чтобы соединить нижний уровень с хост-компьютером через встроенное USB-соединение.
    * Переключите переключатели кодов `1` и `2` влево, иначе это может повлиять на соединение между нижним уровнем и хост-компьютером.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
