## Подготовка к прошивке системы

    * Чтение карты
    * TF-карта с объемом не менее 16 ГБ или не более 128 ГБ, скорость передачи данных должна быть класса 10 или выше
    * Скачайте соответствующий <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">образ системы</Button>
    * Скачайте необходимый <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">инструмент для записи</Button>

## Прошивка системы

:::info
Прошивка TF-карты и прошивка системы через TF-карту на M2WE может выполняться одним из этих способов
:::

:::info
* Если в машине установлены M2WE и карта памяти, система по умолчанию будет запускаться с карты памяти
* Для M2WE рекомендуется использовать TF-карту для прошивки, использование USB для прошивки на M2W может вызвать неудачу из-за проблем совместимости драйверов. Рекомендуется использовать прямую прошивку [карты памяти на M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Обратите внимание на то, что скачанный образ системы должен соответствовать, в противном случае система не запустится
:::
