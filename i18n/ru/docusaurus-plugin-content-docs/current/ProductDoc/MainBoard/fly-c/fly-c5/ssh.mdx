# Подключение к хост-компьютеру через SSH

* Можно использовать последовательный порт или Wi-Fi для подключения к устройству.
* Пожалуйста, заранее подготовьте **MobaXterm_Personal** или другое **SSH-терминальное средство**.
* Убедитесь, что карта SD или M2WE имеет записанную **систему соответствующего хост-компьютера**.
* **Если хост-компьютер не запускается нормально, удалите все внешние устройства, такие как драйверы, ограничители и вентиляторы!**
* **В конце подключитесь через сеть к SSH для удобства компиляции прошивки!**
* **Для компиляции прошивки см. руководство по подключению к хост-компьютеру через Wi-Fi или сетевой кабель.**

        ## Использование IP-сетевого подключения к хост-компьютеру через SSH

        * Пожалуйста, войдите в веб-интерфейс **маршрутизатора**, к которому подключен хост-компьютер.
        * Убедитесь, что система уже **запущена нормально**.
        * Убедитесь, что вы знаете **имя пользователя и пароль хост-компьютера**, по умолчанию для системы FLY имя пользователя — **fly**, пароль — **mellow**.
        * Если материнская плата уже запущена и подключена к сети, можно подключиться через IP к SSH.
        * Откройте MobaXterm и выполните следующие настройки, указав IP, который вы видели ранее, либо проверьте его в веб-интерфейсе маршрутизатора.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Внимание: здесь на 4-м шаге вводится имя пользователя хост-компьютера.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * При первом подключении может появиться это окно, просто нажмите `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Введите пароль хост-компьютера, когда система запросит его, и нажмите Enter.

        **Внимание: при вводе пароля ничего не будет отображаться, просто нажмите Enter после завершения ввода.**

        По умолчанию для системы FLY имя пользователя — `fly`, пароль — `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />
