# Функция возобновления печати после отключения питания в Klipper

- Функция возобновления печати после отключения питания в Klipper означает, что после восстановления питания Klipper автоматически может восстановить состояние печати.
- C5 можно использовать без KPPM
- Не применяется для моделей принтеров, где после отключения питания Z-ось смещается.
