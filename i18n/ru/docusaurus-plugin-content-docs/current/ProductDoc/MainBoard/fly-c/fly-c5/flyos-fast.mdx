## Установка антенн

    1. Установите антенну для модуля `M2WE` или `2.4G WiFi`
    :::warning
    * Для работы FLY с Wi-Fi требуется модуль `M2WE` или `2.4G WiFi`, иначе подключение к Wi-Fi невозможно
    * Обратите внимание, что приложение не поддерживает сети Wi-Fi с русскими названиями и сети, поддерживающие одновременную работу в диапазонах 2.4G и 5G
    :::
    * Место с красной стрелкой указывает на разъем антенны Wi-Fi
    * Разъем антенны Wi-Fi — `ipex1`
    * Антенну необходимо установить на M2WE, иначе из-за слабого сигнала может возникнуть проблема соединения с маршрутизатором
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - После установки системы карта уже готова к следующему шагу: подключитесь к приложению через SSH
    :::

    2. Подайте питание на приложение.  
    **Ваше приложение уже готово к запуску!**  
    Перейдите к следующему шагу: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh">Подключение через SSH</Button>
