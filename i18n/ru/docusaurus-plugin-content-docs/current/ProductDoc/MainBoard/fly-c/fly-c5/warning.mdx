# Заметки по материнской плите

## Индикатор питания
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/power.webp').default} size="80%" align="center" />
* При подаче питания только через TYPE-C без подключения других источников питания на материнскую плату должны гореть как минимум два индикатора: `3.3V` и `5V`.
* После подключения основного питания материнской платы индикатор питания должен светиться для `3.3V`, `5V` и `VCC`.

### Устранение неполадок

:::danger Важно

* При отсутствии подключенных внешних устройств (драйверов, ограничителей, 12864 и т. д.)
* При питании только через TYPE-C индикатор `3.3V` должен постоянно светиться.

:::

* Если после подключения основного питания материнской платы и питания драйвера индикатор `5V` не загорается, необходимо обратиться в службу поддержки.

## Разъем ограничителя

* Обратите внимание на последовательность подключения проводов. Для трехжильных ограничителей важно правильно подключать 5V, GND и S; ошибочное подключение может привести к повреждению материнской платы.
* Если используется двухжильный вариант, достаточно подключить S и GND.
* Для PL08 и других близкорасположенных датчиков требуется внешнее подключение диода для понижения напряжения до уровня ниже 3.3V.
