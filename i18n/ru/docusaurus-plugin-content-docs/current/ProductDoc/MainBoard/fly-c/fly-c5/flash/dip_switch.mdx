# Введение в переключатель кодов

:::tip
FLY c8 подключается к хост-компьютеру через переключатель кодов, если положение переключателя неправильное, то устройство не сможет работать нормально.
:::

:::danger Важно знать
Обратите внимание, что переключатель кодов должен быть установлен в указанное положение для нормального подключения к хост-компьютеру.
:::

## Подключение встроенному хост-компьютеру

    * Переключите переключатели кодов `1` и `2` вправо, чтобы соединить нижнее устройство с хост-компьютером через встроенную USB-связь
    * Переключите переключатели кодов `3` и `4` влево, иначе это может повлиять на соединение между нижним устройством и хост-компьютером
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * После выполнения этой операции нижний порт USB будет занят нижним устройством, не подключайте другие устройства
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Подключение внешнего хост-компьютера

    * Переключите переключатели кодов `3` и `4` вправо, чтобы соединить нижнее устройство с хост-компьютером через встроенную USB-связь
    * Переключите переключатели кодов `1` и `2` влево, иначе это может повлиять на соединение между нижним устройством и хост-компьютером
