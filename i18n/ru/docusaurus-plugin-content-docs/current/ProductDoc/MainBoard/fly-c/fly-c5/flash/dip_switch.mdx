# Введение в переключатели

:::tip
FLY c8 подключается к хост-компьютеру через переключатель, и если положение переключателей неправильное, он не сможет нормально работать.
:::

:::danger Важно отметить
Обратите внимание, что переключатели должны быть установлены в указанное положение для нормального подключения к хост-компьютеру.
:::
