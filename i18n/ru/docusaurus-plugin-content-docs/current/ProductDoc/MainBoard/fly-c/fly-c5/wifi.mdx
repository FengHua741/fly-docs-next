# Подключение к Wi-Fi

## Установка антенны на C5

* Точка с красной стрелкой указывает на разъем Wi-Fi антенны
* Разъем Wi-Fi антенны — это `ipex1`
* Установите антенну на C5, иначе может возникнуть проблема с сигналом, из-за которой подключение к сети будет невозможно
