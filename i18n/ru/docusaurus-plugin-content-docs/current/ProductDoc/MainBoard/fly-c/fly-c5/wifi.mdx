# Подключение к Wi-Fi

## Установка антенны на C5

* Показанное красной стрелкой место является разъемом антенны Wi-Fi
* Разъем антенны Wi-Fi — это `ipex1`
* Установите антенну на C5, иначе может возникнуть проблема с отсутствием сигнала, из-за чего подключение к сети будет невозможно
