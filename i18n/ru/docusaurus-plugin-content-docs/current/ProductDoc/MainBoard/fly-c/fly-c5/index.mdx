## Особенности продукта

- 32-битный ARM Cortex-M4, 550 МГц, чип STM32H723VGT6
- Поддержка режимов работы SPI, UART, STEP/DIR с драйверами TMC
- Пин DIAG не占用 лимиты, после настройки может обеспечивать бесконечное возвращение домой без лимитов
- Цифровые вентиляторы поддерживают напряжение 24В, 12В, 5В, без необходимости дополнительных модулей понижения напряжения
- Поддержка автовыключения при отключении питания и продолжения печати после восстановления (автовыключение по умолчанию включено)
- Предустановленная прошивка USB Klipper при出厂е
- Многофункциональное управление питанием, короткие замыкания портов, концевиков и драйверов не влияют на работу компьютера верхнего уровня
- Зарезервирован разъем 5V RGB
- Поддержка TFT и HDMI экранов через один провод
- Система предустановлена klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, пакет numpy для компенсации резонанса, без необходимости изменения источника, проста в освоении (требуется покупка карты TF или M2WE)

## Технические параметры

### Верхний контроллер

- **CPU**: Allwinner H618, 4-ядерный 64-битный Cortex-A53
- **GPU**: Arm Mali-G31
- **ОЗУ**: 1 ГБ DDR4 3200МГц
- **ПЗУ**: поддерживает максимум 128 ГБ карты памяти SD
- **Периферия**:
    - TFT x1
    - USB 2.0 x2
    - Micro HDMI x1
    - Порт Ethernet (Eth) x1 (100Мб/с)
    - FPC-HDMI x1
    - UART x1
- **Интерфейсы**: интегрированный M.2, расширяемый Wi-Fi, eMMC и т.д. (частная протоколизация, не подключать устройства M.2 не из серии FLY)
- **Поддержка**: TFT и HDMI экраны поддерживают передачу сигнала через один FPC-провод
- **Предустановленная система**: klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, пакет numpy для компенсации резонанса, без необходимости изменения источника, проста в освоении (требуется покупка карты TF или M2WE)

### Нижний контроллер

- **MCU**: STM32H723VGT6
- **Прошивка**: KLIPPER
- **Питание**: поддерживает питание 12-24 В постоянного тока, без необходимости модуля понижения напряжения
- **Максимальный ток нагревательной платформы**: 15А
- **Максимальный ток нагревательного элемента**: 6А
- **Максимальный ток вентилятора**: 1А
- **Режим работы драйвера**: SPI, UART, STEP/DIR
- **Поддерживаемые драйверы**: TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988 и т.д.
- **Расширение интерфейсов**: BLTouch (Servos, Probe), RGB x1, UART x1, CAN x1
- **Вентиляторы**: два цифровых вентилятора x4 (поддерживаются напряжения 24В, 12В, 5В)
- **Датчики температуры**: 3 канала NTC 100К
- **Поддерживаемые конструкции**: Cartesian, Delta, Kossel, Ultimaker, CoreXY и т.д.
