# Фирменное ПО BL

    :::tip

    * Обратите внимание, что при производстве уже предварительно записан `bootloader`, поэтому нет необходимости в повторной записи.

    :::

## Вход в режим программирования


    * Существует два способа входа в режим программирования
    * Первый: при полном отключении питания от главной платы, удерживайте кнопку BOOT и подключите соединительный кабель с возможностью передачи данных к главной плате и вашему устройству, рекомендуется для WINDOWS
    * Второй: после подключения соединительного кабеля с возможностью передачи данных к главной плате и вашему устройству, удерживайте кнопку BOOT и введите команду, рекомендуется для верхнего устройства
