# Фирменное ПО BL

    :::tip

    * Обратите внимание, что на заводе уже предварительно записан `bootloader`, повторная запись не требуется

    :::

## Вход в режим программирования

    :::tip

    * Слева — это кнопка RST материнской платы, справа — это кнопка BOOT материнской платы

    :::

    * Есть два способа войти в режим программирования
    * Первый: при полном отключении питания материнской платы длительное нажатие кнопки BOOT, затем подключение одного конца передающего данных кабеля к материнской плате, а другой конец к вашему устройству, подходящий для WINDOWS
    * Второй: после подключения одного конца передающего данных кабеля к материнской плате, а другого конца к вашему устройству, длительное нажатие кнопки BOOT, затем ввод команды, подходящей для ПК
