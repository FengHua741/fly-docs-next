# Фирменное ПО BL

    :::tip

    * Обратите внимание, что при выпуске уже предварительно записан `bootloader`, поэтому нет необходимости повторно записывать его.
    * C5 можно использовать только с родной программой для записи.
    :::

## Подключение по SSH к программе верхнего уровня

    :::danger Предупреждение
    - Перед записью ФП BL необходимо подключиться по `wi-fi` к программе верхнего уровня через SSH.
    :::

 * Сначала выполните: <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh#использование-wifi-для-подключения-ssh-к-программе-верхнего-уровня">Использование wi-fi для подключения SSH к программе верхнего уровня</Button>

## Запись ФП


    * Переведите переключатели в верхнее положение, подключите нижний уровень к USB верхнему уровню
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />

    * Введите следующую команду для начала записи BL


        ```bash
        fly-flash -d c5 -u -f /usr/lib/firmware/bootloader/hid_bootloader_h723_v1.0.bin
        ```
