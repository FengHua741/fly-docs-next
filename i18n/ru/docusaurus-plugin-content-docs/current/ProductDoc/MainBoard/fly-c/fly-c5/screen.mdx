# Настройка и подключение экрана

## Настройка и подключение экрана C5
Подключение tft-экрана

:::tip
- Перед подключением экрана отключите питание, чтобы избежать повреждения экрана.
- При подключении экрана убедитесь в правильности направления fpc-линии, чтобы избежать повреждений экрана.
:::

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/screen.webp').default} size="80%" align="left" />

- Для доступа к устройству используйте IP-адрес, например: `http://192.168.1.1`
- Перейдите на страницу конфигурации
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/screen1.webp').default} size="80%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/screen2.webp').default} size="80%" align="left" />
