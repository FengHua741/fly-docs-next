# Настройка экрана и его подключения

## Способы настройки


- В адресной строке браузера введите IP-адрес устройства, например: `http://192.168.6.179`
- Перейдите на страницу настройки
    - В браузере откройте IP-адрес устройства, например: `http://192.168.1.2/`
    - Для fluidd нажмите, как показано слева на рисунке, `снять галочку` -> `скрыть скрытые файлы и папки`.
    - Для mainsail нажмите, как показано справа на рисунке, `поставить галочку` -> `показать скрытые файлы`.

      <tr>

        </td>

          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />
        </td>

    - Теперь вы увидите папку `.flyos-config`, войдите в неё, где есть файл `sys-config.conf`.
    - Файл `sys-config.conf` является ссылкой на конфигурационный файл `config.txt` в мобильном накопителе `FlyOS-Conf`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - Откройте файл `sys-config.conf`, найдите `screen=` для изменения типа экрана.
    - Измените номер ⓪ после записи `klipper_screen` на 1, чтобы активировать службу `klipper_screen`.
    - Измените параметр после записи `screen` на требуемый тип экрана, затем сохраните и закройте файл, после чего перезагрузите устройство.
    <ImageView image={require('@site/docs/General/flyos-fast/img/screen.webp').default} size="90%" align="center" />

    - Примечание: формат настроек в файле `sys-config.conf` должен быть `ключ=значение`, а не `ключ: значение`.
    - После сохранения изменений перезагрузите систему для применения настроек.

## Подключение экрана C5

    :::danger Важно
    - Перед подключением экрана выключите питание, чтобы избежать повреждения экрана.
    - При подключении экрана убедитесь в правильности направления fpc-кабеля, чтобы избежать повреждений.
    :::
