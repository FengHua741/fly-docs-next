# Настройка экрана и его подключение

## Способы настройки


- Через адресную строку браузера введите IP-адрес устройства, например: `http://192.168.6.179`
- Перейдите на страницу настроек
    - В браузере откройте IP-адрес устройства, например: `http://192.168.1.2/`
    - Для Fluidd нажмите, как показано слева на рисунке, `снять галочку` -> `скрыть скрытые файлы и папки`.
    - Для Mainsail нажмите, как показано справа на рисунке, `поставить галочку` -> `показать скрытые файлы`.

    - Теперь вы увидите папку `.flyos-config`, перейдите в эту папку — там будет файл `sys-config.conf`.
    - Файл `sys-config.conf` является символической ссылкой на файл конфигурации `config.txt` в съемном накопителе `FlyOS-Conf`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - Откройте файл `sys-config.conf` и найдите `screen=`, чтобы изменить тип экрана.
    - Измените номер ⓪ после записи «klipper_screen» на 1, чтобы включить службу «klipper_screen».
    - Измените параметр после записи «screen=» на нужный экран, затем сохраните и закройте файл, а затем перезагрузите устройство.
    <ImageView image={require('@site/docs/General/flyos-fast/img/screen.webp').default} size="90%" align="center" />

    - Внимание: формат записей в файле `sys-config.conf` должен быть `ключ=значение`, не изменяйте на `ключ: значение`.
    - После сохранения изменений перезагрузите систему для применения.

## Подключение экрана C5

    :::danger Важно
    - При подключении экрана отключите питание, чтобы избежать повреждения экрана.
    - При подключении экрана обратите внимание на направление FPC-проводника, чтобы избежать повреждения экрана.
