# Инструкция по прошивке прошивки

    
## Как определить, какую прошивку вам нужно

    :::tip

    * Обратите внимание, что вам нужно прошивать только одну из этих прошивок

    :::
    
    * C5 можно подключить к компьютеру следующим образом
    * <Button variant="contained" disableElevation href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: если несколько USB-устройств могут повлиять на передачу данных
    * <Button variant="contained" disableElevation href="./can">USB-мост прошивка</Button>
        * Преимущество: может подключать несколько устройств без использования порта USB
        * Недостаток: возможны ошибки, такие как превышение времени ожидания при совместной работе с `KlipperScreen`.
