# Инструкция по прошивке прошивки

    :::warning Важные замечания

    * Для PRO X10 предварительно установлен Katapult
    * После прошивки BL дальнейшая прошивка BL не требуется
    * У PRO X10 есть два типа BL: Katapult-прошивка и BL-прошивка, записанная с TF-карты
    * При необходимости можно выбрать подходящий вариант и скачать заново <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">BL-прошивку</Button>

    :::
    
    :::danger Необходимо знать
    * Прошивка материнской платы осуществляется через Katapult для прошивки Klipper
    * Если материнская плата уже содержит Katapult, то последний этап прошивки Katapult-прошивки не требуется, это лишь мера на случай неудачной прошивки
    * Можно быстро дважды нажать кнопку сброса, чтобы проверить, загорается ли один светодиод на материнской плате; если да, то материнская плата находится в режиме Katapult

    :::

## Как определить нужную вам прошивку

    :::tip

    * Обратите внимание, что достаточно установить только одну из прошивок

    :::
    
    * Для PRO X10 возможны следующие способы подключения к ПК
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB-прошивка</Button>
        * Преимущество: надежная и стабильная связь
        * Недостаток: возможное влияние на передачу данных при наличии нескольких USB-устройств
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB-мостовая прошивка</Button>
        * Преимущество: возможность подключать несколько устройств без использования портов USB
        * Недостаток: возможны ошибки вроде превышения времени сброса при совместном использовании с `KlipperScreen`.
