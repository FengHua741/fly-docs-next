## Основные характеристики

    * Используется 32-битный контроллер STM32H723ZGT6 с тактовой частотой 550 МГц из серии ARM Cortex-M7
    * Связь между материнской платой и основной платой осуществляется через соединение BTB, что позволяет использовать основные платы Orange Pi CM4 и Raspberry Pi CM4
    * На материнской плите предусмотрена кнопка BOOT, которая позволяет обновлять загрузчик материнской платы через режим DFU
    * Десять выдвижных разъемов для управления драйверами, не требующих никаких перемычек для переключения режимов работы
    * Предусмотрен функциональный вывод DIAG, который позволяет использовать бесконечное возвращение в исходное положение простым подключением переходника
    * Питание драйвера имеет возможность выбора источников питания HVIN и VIN
    * Обновление микроконтроллера (MCU) возможно как через MicroSD-карту, так и через команду make flash Klipper через DFU
    * Шесть двухпроводных управляемых вентиляторов с возможностью замены MOSFET, два четырехпроводных вентилятора и один вентилятор VCC
    * Цифровое управление скоростью вентиляторов с возможностью выбора напряжений 24В, 12В и 5В, что исключает необходимость внешних трансформаторов и снижает риск повреждения платы
    * Использование сменных предохранителей для удобства замены;

## Технические характеристики

    * MCU: STM32H723ZGT6
    * Напряжение питания драйвера: VIN(12V-24V), HVIN(12V-48V)
    * Напряжение питания материнской платы: VIN(12V-24V)
    * Напряжение питания горячего стола: BED IN(12V-24V)
    * Интерфейсы нагрева: горячий стол (BED OUT), нагревательные стержни (HETA0, HETA1, HETA2, HETA3)
    * Максимальный ток на разъеме горячего стола: 15А
    * Максимальный ток на интерфейсе нагревательных стержней: 6А
    * Максимальный ток на интерфейсе вентиляторов: 1А
    * Поддерживаемые режимы работы драйверов: SPI, UART, STEP/DIR
    * Расширительные интерфейсы: BLTouch (Servos, Probe), UART X1, высоковольтный ограничитель X1
