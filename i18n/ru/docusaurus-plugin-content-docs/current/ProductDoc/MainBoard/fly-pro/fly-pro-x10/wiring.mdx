# Основные сведения о подключении материнской платы

    :::warning Важно

    Из-за высокого разрешения изображений при просмотре на веб-странице они могут быть недостаточно четкими. Пожалуйста, наведите курсор мыши на схему подключения и щелкните правой кнопкой мыши, выберите "Сохранить изображение как..." и сохраните изображение на рабочий стол для более детального просмотра!!!

    :::
