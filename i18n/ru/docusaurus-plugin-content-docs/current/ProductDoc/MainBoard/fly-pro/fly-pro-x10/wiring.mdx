# Подключение материнской платы

:::warning Важно
  Из-за высокого разрешения изображений просмотреть их на веб-странице может быть сложно. Пожалуйста, наведите курсор мыши на схему подключения и щелкните правой кнопкой мыши, выберите "Сохранить изображение как..." и сохраните его на рабочий стол для более детального просмотра!!!
:::

## Инструкции по подключению материнской платы

:::warning Предупреждение
  Не используйте готовые заводские **трехжильные датчики ограничителя перемещения**. Такие датчики могут вызвать **короткое замыкание материнской платы** и **повредить её**!!! Подключайтесь строго в соответствии с указанным способом на схеме подключения!!!
:::
