# Подключение материнской платы

:::warning Важно
  Из-за высокого разрешения изображений, просмотреть их на веб-странице может быть сложно. Пожалуйста, наведите курсор мыши на схему подключения, щелкните правой кнопкой мыши и выберите "Сохранить изображение как..." для сохранения изображения на рабочий стол и более детального просмотра!!!
:::

## Инструкции по подключению материнской платы

:::warning Предупреждение
  Не используйте готовые заводские трехжильные ограничители перемещения. Такие ограничители могут вызвать короткое замыкание материнской платы и повредить её!!! Подключайтесь строго в соответствии с указанной в схеме подключения схемой подключения!!!
:::
