# Заметки по материнской плите

## Индикатор питания
    * При использовании Type-C для подачи питания без подключения дополнительного источника питания на материнскую плату индикаторы должны светиться `SYS_5V` и SYS_3.3V
    * После подключения питания материнской платы (`HVIN`, `VIN`, `BOARO`, `POWER`) индикатор питания платы должен гореть постоянно

## Драйвер

    * TP5160 и TP2209 имеют разные названия, но выбор режима драйвера можно сделать без перемычек
    * Разъем драйвера предназначен только для установки драйверов TP и не может быть использован для других устройств

## Гнездо ограничителя
    * ProX10 поддерживает выбор между 5В и 12В напряжением, убедитесь, что при использовании датчиков близости установлен диод для понижения напряжения

## Управляющее ПО
    * ProX10 поддерживает использование внешнего управляющего ПО, переключение на подключение через Type-C позволяет использовать его

## Прошивка

    * Материнская плата поддерживает прошивку с помощью TF-карты, однако по умолчанию используется `Katapult`
    * Для использования TF-карты для прошивки необходимо повторно прошить BL
