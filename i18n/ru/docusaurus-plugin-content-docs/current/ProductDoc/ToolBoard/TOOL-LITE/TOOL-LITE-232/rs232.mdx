# Компиляция прошивки RS232

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Выберите **GPIO пины для установки при старте микроконтроллера**, введите: **!gpio18**
    :::warning
        Введите `!gpio18` в английской раскладке клавиатуры
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/TOOL-LITE/TOOL-LITE-232/img/rs232.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="65%" align="center" />
    * Если последний вывод содержит следующее, то компиляция успешна
       <ImageView image={require('@site/docs/General/dfu-flash/img/make_uf2_finish.webp').default} size="65%" align="center" />




## Переключение в режим прошивки

    :::tip

    Стрелка указывает на кнопку `BOOT`

    :::

    * При полностью отключенном инструментальном щите, удерживайте кнопку `BOOT`, используйте USB Type-C кабель с функцией передачи данных, чтобы подключить **инструментальный щит** к **USB порту хост-компьютера**, после правильного подключения отпустите кнопку `BOOT`
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/TOOL-LITE/TOOL-LITE-232/img/boot.webp').default} size="80%" align="center" />
