# Введение

# 1. Общее описание продукта

FLY TOOL-LITE-232 — это платформа для инструментов, разработанная компанией Guangzhou Meilun Electronic Technology Co., Ltd. специально для 3D-принтеров. Данная плата предоставляет интерфейсы xt30/kf, позволяя заменить сложные соединения между материнской платой и печатающей головкой четырьмя проводами, что оптимизирует расположение. TOOL-LITE-232 совместима с шаговыми двигателями 36/42, интегрируя в себя трансивер RS232, порт TypeC, драйвер TMC2209, акселерометр LIS2DW, два управляемых вентилятора и другие функциональные элементы. Эта плата предназначена для использования с прошивкой Klipper.

## 1.1 Особенности продукта

    * Предусмотрена кнопка Boot, пользователи могут войти в режим Boot через порт TypeC для обновления прошивки.
    * Поддерживает прошивку Klipper.
    * Поддерживает режим S232.
    * В режиме RS232 обеспечивается более высокая скорость передачи данных и большая стабильность.
    * Поддержка датчиков температуры: термисторы ntc100k и PT1000.
    * Поддерживает два управляемых вентилятора, максимальный ток выхода составляет 1 А.
    * На борту установлен акселерометр LIS2DW.
    * Предусмотрен высоковольтный разъем probe, на борту установлен драйвер 2209, поддерживает различные датчики выравнивания, например, BLtouch, klicky, Voron Tap, PL08n и т. д.
    * Поддерживает входное напряжение 12-24В, ток до 15А, увеличенный разъем нагревательного элемента, номинальный ток 10А.

## 1.2 Новые функции

    * Добавлен режим RS232, обеспечивающий более высокую скорость передачи данных и большую стабильность, а также упрощает процесс отладки.
    * Добавлена 1-метровая кабельная сборка питания/сигнала, выполненная по технологии двойной витой пары, что повышает антишумовые свойства.

## 1.3 Технические характеристики

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+ @ 133MHz
    * Соединительные интерфейсы используют XT30 (2+2) соединители, поддерживающие ток до 15А, пиковый ток до 30А.
    * Прошивка: Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Датчики на борту: LIS2DW
    * Разъем нагревательного элемента: HEAT0, 10A
    * Расширительные интерфейсы: высоковольтный разъем probe, EndStop, PT100/PT1000, XH2.54-2P CAN расширительный разъем
    * Интерфейсы вентиляторов: два управляемых вентилятора DC
    * Драйвер двигателя: на борту установлен TMC2209
    * Датчик температуры: один термистор 100K NTC или PT1000 (TH0)
