# Компиляция прошивки RS232

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись к командной строке
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, компиляция успешна

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переключение в режим прошивки

    :::tip

    Закругленное место — это кнопка `BOOT`

    :::

    * Для входа в DFU требуется полный разряд питания материнской платы. Удерживайте кнопку BOOT, а затем подключите данные проводники между материнской платой и хост-компьютером
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.webp').default} size="100%" align="center" />
