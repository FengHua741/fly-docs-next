# Программирование прошивки RS232

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Выбрать GPIO пины для установки при старте микроконтроллера, ввести: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * Нажать клавишу `Q`, появится **Save configuration**, затем нажать `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Ввести следующую команду для начала компиляции (время может занять немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого означает успешную компиляцию:

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переключение в режим программирования

    :::tip

    Отмечено кругом как `BOOT кнопка`

    :::

    * Для входа в DFU необходимо полностью отключить питание платы, удерживать кнопку BOOT и подключить кабель данных к материнской плате и хост-компьютеру
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.webp').default} size="100%" align="center" />
