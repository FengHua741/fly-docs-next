# Фирменное ПО Katapult

## Описание фирменного ПО Katapult

    <CANBOOT />

    :::tip

    * Версия ERCF V2 поддерживает CANR, S232, USB
    * Здесь предоставляются только фирменные версии для CAN с скоростью 500K и 1M

    :::

## Вход в режим прошивки

    :::tip

    Круг выделен как `КНОПКА BOOT`

    :::

    * Для входа в режим прошивки необходимо полностью отключить питание материнской платы, удерживать кнопку BOOT и подключить шину передачи данных одной конечностью к материнской плате, а другой конечностью к компьютеру
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/boot.webp').default} size="100%" align="center" />

            <Katapult />

            4. Прошивка прошивки

            :::tip

            Требуется прошить только одну из прошивок

            :::

            * Ниже приведена команда для прошивки прошивки с CAN-скоростью 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_500K.uf2
                ```
                       
            * Ниже приведена команда для прошивки прошивки с CAN-скоростью 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCFV2_CANBOOT_CAN_1M.uf2
                ```
            * Ниже приведена команда для прошивки прошивки с интерфейсом USB
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCFV2_CANBOOT_USB.uf2
                ```

            5. Если появляется нижеследующее уведомление, то прошивка успешно завершена

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            После завершения прошивки требуется полностью отключить питание материнской платы.

            :::
