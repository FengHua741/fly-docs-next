# Справочное подключение материнской платы

    :::warning Важно

    Из-за высокого разрешения изображений просмотреть их на веб-странице невозможно. Пожалуйста, наведите курсор мыши на схему подключения и щелкните правой кнопкой мыши, выберите «Сохранить изображение как...» и сохраните изображение на рабочий стол для более подробного просмотра!!!

    :::
