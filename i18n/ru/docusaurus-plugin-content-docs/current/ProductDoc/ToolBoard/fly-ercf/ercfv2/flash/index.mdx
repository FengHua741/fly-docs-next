# Инструкция по прошивке прошивки

    * V2 ERFC может подключаться к компьютеру через два способа: RS232 и CAN
    * Для подключения RS232 и CAN требуется использовать кабель `XT30(2+2)-F`
    * Ниже приведен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Прошивка основной платы производится через Katapult для прошивки Klipper
* Если вы уверены, что основная плата уже прошита Katapult, то не нужно выполнять последний шаг прошивки прошивки Katapult — это просто мера предосторожности на случай неудачного прошивания
* Можно дважды быстро нажать кнопку сброса и проверить, загорается ли светодиод на плате, если да, то это указывает на успешную загрузку в прошивку Katapult
:::

## Как определить, какую прошивку вам нужна

    * V2 ERFC может подключаться к компьютеру следующим образом:
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка CAN-прошивки</Button>
        * Преимущество: если установлена прошивка Katapult, можно обновлять прошивку без физического контакта с платой, а также поддерживаются шины связи
        * Недостаток: возможны ошибки, такие как превышение времени сброса, при совместном использовании с KlipperScreen
        * Примечание: перед использованием CAN убедитесь, что ваш компьютер поддерживает CAN, и используйте его совместно с UTOC или материнской платой с прошивкой шлюза
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Прошивка прошивки RS232</Button>
        * Преимущество: не требует специального программного обеспечения, надежная и стабильная связь
        * Недостаток: для повторной прошивки требуется вход в режим DFU вручную или двойное нажатие кнопки сброса для входа в Katapult для обновления прошивки
        * Примечание: для использования необходимы G2T или UTOR
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Прошивка USB-прошивки</Button>
        * Преимущество: если установлена прошивка Katapult, можно обновлять прошивку без физического контакта с платой, для прошивки через карту памяти или DFU требуется прежний метод обновления, надежная и стабильная связь
        * Недостаток: возможны проблемы с передачей данных при наличии нескольких устройств USB
