    <CAN />

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции, процесс может занять некоторое время
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода, если компиляция успешна, должно появиться следующее:
        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```

## Программирование firmware

    * Перед программированием подключите Tool Board к UTOC или материнской плате с прошивкой Bridge через XT30 кабель, ниже приведен пример соединения
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L составляет около `60 Ом` (в отключенном состоянии), если показывает `120 Ом`, то необходимо добавить резистор `120 Ом`, если около `40 Ом`, то нужно снять резистор
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
