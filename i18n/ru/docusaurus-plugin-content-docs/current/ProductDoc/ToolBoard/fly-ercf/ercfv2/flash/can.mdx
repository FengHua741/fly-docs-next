    <CAN />

    * Выберите GPIO пины для настройки при старте микроконтроллера, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого, компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки используйте кабель XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведена схема подключения
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется мультиметром при отключенном питании и составляет около `60 Ом`. Если показатель `120 Ом`, подключите резистор `120 Ом`, если около `40 Ом`, удалите резистор
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
