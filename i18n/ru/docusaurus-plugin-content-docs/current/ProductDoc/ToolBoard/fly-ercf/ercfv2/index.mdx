# Продуктное описание

    FLY-ERCF — это многоцветный контроллер для 3D-принтера на базе RP2040, предназначенный в основном для проекта Enraged Rabbit. Он также может использоваться для управления другими двухосевыми шаговыми двигателями.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/img/ercf.webp').default} size="100%" align="left" />

## Особенности продукта

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Поддержка режимов CAN и RS232
    * Два драйверных разъема, поддерживающих UART и SPI
    * Совместимость с оригинальной ERCF EASY Board
    * Компактность конструкции, использование четырехслойного дизайна
    * Замена 14PIN разъема на XHD2.54, что обеспечивает более надежное подключение для предотвращения обрыва проводов
    * Поддержка напряжений от 12 до 24В
    * Поддержка CAN-шины и RS232, обеспечивающая более стабильную передачу данных с меньшей задержкой и более надежным соединением
    * CAN-порт использует XT30 (2+2), поддерживающий ток до 15А

## Технические характеристики

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Фирменное ПО: Klipper CAN/RS232
    * Входное напряжение: DC12V-24V
    * Разъемы: XHD2.54 двойной 7PIN расширительный разъем, SERVO X1, ENDSTOP X2, EXTRA X1
    * Драйвер: поддержка UART/SPI и STEP/DIR
    * Поддержка напряжений от 12 до 24В
    * Поддержка CAN-шины и RS232, обеспечивающая более стабильную передачу данных с меньшей задержкой и более надежным соединением
    * CAN-порт использует XT30 (2+2), поддерживающий ток до 15А
