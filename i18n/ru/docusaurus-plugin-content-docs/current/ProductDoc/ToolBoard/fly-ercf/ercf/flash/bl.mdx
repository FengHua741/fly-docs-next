            4. Прошивка прошивки

            :::tip

            Необходимо прошить только одну из прошивок

            :::

            * Ниже приведена команда для прошивки прошивки с скоростью CAN 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * Ниже приведена команда для прошивки прошивки с скоростью CAN 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * Ниже приведена команда для прошивки прошивки USB
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. Если появляется следующее уведомление, то прошивка успешно завершена

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            После завершения прошивки требуется полностью отключить питание платы.
