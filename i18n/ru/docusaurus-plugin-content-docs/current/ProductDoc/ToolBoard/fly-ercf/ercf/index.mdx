# Общее описание продукта

    FLY-ERCF — это мультицветный контроллер для 3D-принтера на базе RP2040, разработанный для проекта Enraged Rabbit. Он также может использоваться в других ситуациях с управлением двухосного шагового двигателя.
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/img/ercf.webp').default} size="100%" align="left" />

## Особенности продукта

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Два драйверных разъема, поддерживающих UART и SPI
    * Совместимость с оригинальной платой ERCF EASY Board
    * Минимальные размеры, выполнена из 4-х слойной платы
    * На борту есть разъем 2.54MM двойной ряд 7PIN для расширения функционала (например, для установки лимитов и т.д.)
    * Поддержка напряжений 12-24В
    * Поддержка CAN-шины и RS232, более стабильная передача данных, меньшая задержка, более надежное подключение
    * Разъем CAN использует XT30 (2+2), поддерживает ток до 15А

## Технические характеристики

    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Фирменное ПО: Klipper
    * Входное напряжение: DC12V-24V
    * Интерфейсы: 2.54MM двойной ряд 7PIN, SERVO X1, ENDSTOP X2, EXTRA X1
    * Драйвер: поддерживает UART/SPI и STEP/DIR

    * Поддержка напряжений 12-24В
    * Поддержка CAN-шины, более стабильная передача данных, меньшая задержка, более надежное подключение
    * Разъем CAN использует XT30 (2+2), поддерживает ток до 15А
