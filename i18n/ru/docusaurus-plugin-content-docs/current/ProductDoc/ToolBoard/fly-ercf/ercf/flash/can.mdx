## Начало компиляции

    <SSH />

    <CAN />

    * Выберите GPIO пины для настройки при старте микроконтроллера, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись к командной строке
    * Введите следующую команду для начала компиляции (время займет немало):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, компиляция успешна:

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите инструментальную плату к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30. Ниже приведен пример соединения:
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется около `60 Ом` в отключенном состоянии; если оно составляет `120 Ом`, установите перемычку `120 Ом`; если около `40 Ом`, уберите перемычку
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/120.webp').default} size="50%" align="center" />
