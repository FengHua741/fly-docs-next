# Компиляция прошивки CAN

## Начало компиляции

    <SSH />

    <CAN />

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись к командной строке
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее содержимое, компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите инструментальную плату к UTOC или материнской плате с прошивкой моста через кабель XT30, ниже приведена схема соединения
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что перед питанием CAN H и CAN L измеряются мультиметром сопротивление около `60 Ом`, если оно составляет `120 Ом`, подключите переходник `120 Ом`, если около `40 Ом`, удалите переходник
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/120.webp').default} size="50%" align="center" />
