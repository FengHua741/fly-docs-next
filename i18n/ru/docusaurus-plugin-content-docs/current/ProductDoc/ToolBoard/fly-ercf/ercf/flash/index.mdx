# Инструкция по прошивке прошивки

    * ERCF можно подключать к ПК через CAN
    * Для подключения CAN требуется использовать кабель `XT30(2+2)-F`
    * Ниже приведён вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно знать
* Для прошивки основной платы необходимо использовать Katapult для прошивки Klipper
* Если вы уверены, что основная плата уже прошита Katapult, нет необходимости выполнять последний шаг прошивки прошивки Katapult; это просто мера предосторожности на случай неудачной прошивки
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли светодиод на плите; если да, то плата находится в режиме Katapult
:::

## Как определить нужную прошивку

    * ERCF можно подключаться к ПК следующим образом
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка прошивки CAN</Button>
        * Преимущество: если установлена прошивка Katapult, её можно обновлять без контакта с платой, при этом поддерживаются bus-соединения
        * Недостаток: совместное использование с KlipperScreen может вызвать ошибки таймаута при возврате домой
        * Совет: если вы хотите использовать CAN, убедитесь, что ПК поддерживает CAN, и используйте его совместно с UTOC или материнской платой с прошивкой моста
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Прошивка прошивки USB</Button>
        * Преимущество: если установлена прошивка Katapult, её можно обновлять без контакта с платой, при этом если используется карта памяти или DFU-прошивка, всё равно придётся обновлять как обычно; надёжность и стабильность связи
        * Недостаток: если подключено несколько USB-устройств, это может повлиять на передачу данных
