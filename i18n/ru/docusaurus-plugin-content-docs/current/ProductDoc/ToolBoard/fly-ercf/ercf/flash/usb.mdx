# Компиляция прошивки через USB

    :::tip

    * Предустановленная на заводе Katapult с поддержкой CAN
    * Если используется загрузка через USB, необходимо загрузить прошивку Katapult для USB
    * После первой загрузки повторная загрузка не требуется
    * <Button variant="contained" disableElevation href="./bl">Загрузка Katapult-прошивки</Button>

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/usb.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
