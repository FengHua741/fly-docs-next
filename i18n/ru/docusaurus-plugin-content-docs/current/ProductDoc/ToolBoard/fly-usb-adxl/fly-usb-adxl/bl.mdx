# Фирменное ПО Katapult

## Переход в режим прошивки

    :::tip

    Обведено — `КНОПКА BOOT`

    :::

    * Для входа в режим прошивки необходимо, чтобы материнская плата была полностью отключена от питания, затем удерживать кнопку BOOT, пока не подключите шину передачи данных с одной стороны к материнской плате, а с другой — к вашему устройству.
