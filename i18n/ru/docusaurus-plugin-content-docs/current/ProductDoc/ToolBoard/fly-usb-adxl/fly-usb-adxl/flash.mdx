# Компиляция прошивки USB

    :::tip

    * При производстве устанавливают предварительно прошитый USB-совместимый Katapult
    * Если прошивка не установлена, вы можете её самостоятельно прошить
    * [Прошивка Katapult](./bl.mdx "Щелкните для перехода")

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, чтобы появилось **Save configuration**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна сохраниться конфигурация и вернуться к командной строке
    * Введите `make -j4` для начала компиляции (время может занять немного)
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого компиляция успешна

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
