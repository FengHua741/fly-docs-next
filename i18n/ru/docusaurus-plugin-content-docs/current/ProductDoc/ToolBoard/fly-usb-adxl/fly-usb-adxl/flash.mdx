# Компиляция прошивки USB

    :::tip

    * При производстве USB-акселерометра предварительно прошита USB-подключаемая прошивка klipper
    * Требуется только обновление прошивки
    * Можно подключить компьютер и USB-акселерометр через USB-кабель

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы должны вернуться в командную строку
    * Введите `make -j4`, чтобы начать компиляцию, это займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, то компиляция успешна

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
