# Компиляция прошивки USB

    :::tip

    * При производстве USB-акселерометра предварительно прошита USB-подключаемая прошивка Klipper
    * Достаточно обновить прошивку
    * Можно подключить компьютер и USB-акселерометр через USB-кабель

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце выводится следующее содержимое, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
