# Компиляция прошивки USB

    :::tip

    * При выпуске USB-акселерометра предварительно записана прошивка Katapult с поддержкой USB
    * Если прошивка не установлена, ее можно записать самостоятельно
    * [Прошивка Katapult](./bl.mdx "Щелкните для перехода")

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
