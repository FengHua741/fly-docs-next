# Компиляция прошивки для USB

    :::tip

    * При производстве USB-акселерометра предварительно записывается прошивка для подключения через USB к Klipper
    * Достаточно обновить прошивку
    * Можно подключить компьютер и USB-акселерометр с помощью USB-кабеля

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и выход в командную строку
    * Введите `make -j4` для начала компиляции, процесс занимает некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее, то компиляция успешна

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
