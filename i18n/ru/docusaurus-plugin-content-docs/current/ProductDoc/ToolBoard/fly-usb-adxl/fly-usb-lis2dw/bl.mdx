# Фирменное ПО Katapult


## Переход в режим прошивки

    :::tip

    Обозначенное кругом — это `BUTTON BOOT`

    :::

    * Для входа в режим прошивки необходимо, чтобы материнская плата была полностью разрядилась, затем нажать и удерживать кнопку BOOT, пока не подключите кабель данных с одной стороны к материнской плате, а с другой стороны к вашему устройству.
