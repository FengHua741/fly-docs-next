# Компиляция прошивки USB

    :::tip

    * При выпуске USB-акселерометра предварительно записана прошивка Katapult для подключения через USB
    * Если прошивка не записана, её можно записать самостоятельно
    * [Прошивка Katapult](./bl.mdx "Щёлкните для перехода")

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />
    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, процесс займёт некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце выводится следующее содержимое — компиляция успешно завершена

        ```bash
        Сборка out/klipper.elf
        Создание hex-файла out/klipper.bin
        ```
