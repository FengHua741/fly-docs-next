# Компиляция прошивки USB

    :::tip

    * При производстве USB-акселерометра предварительно установлено USB-подключаемое программное обеспечение Klipper
    * Требуется только обновление прошивки
    * Можно подключить компьютер и USB-акселерометр через USB-кабель

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />
    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio8
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите клавишу `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее, то компиляция успешно завершена

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
