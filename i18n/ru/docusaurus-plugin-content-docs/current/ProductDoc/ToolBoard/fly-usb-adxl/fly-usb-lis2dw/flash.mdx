# Компиляция прошивки для USB

    :::tip

    * При производстве ускоритель USB поставляется с предварительно записанной прошивкой Katapult для подключения через USB
    * Если прошивка не установлена, вы можете её самостоятельно записать
    * [Прошивка Katapult](./bl.mdx "Щелкните для перехода")

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />
    * Выберите GPIO пины, которые необходимо установить при старте микроконтроллера, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, чтобы появилось **Сохранение конфигурации**, затем нажмите клавишу `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, это займет некоторое время
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце выводится следующее содержимое, что указывает на успешную компиляцию:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
