# Инструкция по прошивке прошивки

    * MMU может подключаться к ПК тремя способами: RS232, CAN и USB.
    * Для подключения через RS232 и CAN используется кабель `XT30(2+2)-F`, а для USB — кабель с функцией передачи данных типа `TYPE-C`.
    * Ниже приведен образец кабеля `XT30(2+2)-F`:
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Прошивка основной платы требует использования Katapult для записи Klipper.
* Если вы уверены, что основная плата уже прошита Katapult, нет необходимости выполнять последующую запись прошивки Katapult — это только на случай, если предыдущая запись не удалась.
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли светодиод на плите. Если да, то плата находится в режиме Katapult.

:::

## Как определить нужную прошивку

    * MMU можно подключать к ПК следующими способами:
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка прошивки CAN</Button>
        * Преимущество: если установлена прошивка `Katapult`, можно обновлять прошивку без физического контакта с платой, и поддерживаются сетевые подключения.
        * Недостаток: возможны ошибки, такие как превышение времени ожидания, при совместном использовании с `KlipperScreen`.
        * Совет: если планируется использовать CAN, убедитесь, что ПК поддерживает CAN, и используйте совместно с UTOC или основной платой с прошивкой моста.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Прошивка прошивки RS232</Button>
        * Преимущество: не требуется специального программного обеспечения, надежное и стабильное соединение.
        * Недостаток: для перепрошивки требуется войти вручную в DFU или дважды нажать кнопку сброса, чтобы войти в `Katapult` для обновления прошивки.
        * Совет: потребуется G2T или UTOR для работы.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Прошивка прошивки USB</Button>
        * Преимущество: если установлена прошивка `Katapult`, можно обновлять прошивку без физического контакта с платой; если используется карта памяти или DFU, необходимо использовать прежний метод обновления прошивки, надежное и стабильное соединение.
        * Недостаток: возможное влияние на передачу данных при наличии нескольких устройств USB.
