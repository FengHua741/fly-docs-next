# Компиляция прошивки для USB

    :::tip

    * Предустановленная в заводских условиях CAN-связь с использованием фиксированной прошивки Katapult
    * Для использования записи прошивки через USB требуется прошивка USB Katapult
    * После записи больше не требуется повторная запись
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Запись прошивки Katapult</Button>

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO пины для настройки при старте микроконтроллера, введите: !PA15 (обратите внимание, что это прописные буквы)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись к командной строке
    * Введите следующую команду для начала компиляции (процесс может занять некоторое время):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующие строки, компиляция успешно завершена:
        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
