# Компиляция прошивки через USB

    :::tip

    * По умолчанию предустановленная прошивка Katapult с CAN-интерфейсом
    * Для использования USB-записи требуется прошивка Katapult для USB
    * После записи прошивка не требует повторной перезаписи
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Запись прошивки Katapult</Button>

    :::

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <USB />

    * Выберите GPIO пины, которые будут установлены при старте микроконтроллера, введите: !PA15 (обратите внимание, что это должно быть заглавное !)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее, компиляция успешна

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
