# Подключение материнской платы

    :::warning Важно

    Из-за высокого разрешения изображений просмотр на веб-странице может быть нечетким. Пожалуйста, **наведите курсор мыши** на схему подключения и **правой кнопкой мыши** выберите **Сохранить изображение как...**, чтобы сохранить изображение на рабочий стол и просмотреть его более четко!!!!

    :::
