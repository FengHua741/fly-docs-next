# 1. Общее описание

FLY SHT36 V3 — это платформа для инструментов, разработанная компанией Guangzhou Mellow Electronics Technology Co., Ltd. специально для головки экструдера 3D-принтера. Используя эту платформу, можно заменить сложные соединения между основной платой и головкой печати четырьмя проводами, оптимизируя расположение. SHT36 V3 совместим с шаговыми двигателями 36, интегрируя CAN-трансивер, трансивер RS232, TypeC-порт, драйвер TMC2209, акселерометр LIS2DW и два управляемых вентилятора, а также другие функциональные возможности. Данная плата совместима с прошивками Klipper и RRF.

## 1.1 Основные характеристики продукта

    * Предусмотрена кнопка Boot, пользователи могут войти в режим Boot через TypeC для обновления прошивки
    * Поддерживает прошивки Klipper и RRF
    * Поддерживает режим CAN, режим RS232, режим CAN RRF,
    * В режиме RS232 имеет более высокую скорость передачи данных и большую стабильность. В режиме CAN поддерживает расширение CAN, предусмотрены разъемы для расширения CAN
    * Датчик температуры: на борту установлен чип MAX31865, совместим с датчиками температуры PT100 и PT1000.
    * Поддерживает два управляемых вентилятора, выбор напряжения, возможность замены маленькой платы MOS, что упрощает ремонт и замену, максимальный выходной ток до 1A
    * На борту установлен акселерометр LIS2DW и датчик LDC1612
    * Предусмотрен разъем RGB, на борту установлен драйвер 2209, поддерживает различные датчики выравнивания, такие как BLtouch, klicky, Voron Tap, PL08n и т.д.
    * Поддерживает входное напряжение от 12 до 24В, ток до 15А, увеличенные разъемы нагревательных элементов, номинальный ток 10А

## 1.2 Новые функции

    * Поддерживает переключение между режимами CAN Klipper, RS232 Klipper, RRF CAN
    * Добавлен режим RS232, который обеспечивает более высокую скорость передачи данных и большую стабильность, а также упрощает отладку
    * Добавлен режим RRF CAN, может быть использована как платформа RRF, повышая универсальность
    * Добавлен разъем для расширения CAN, что упрощает подключение устройств CAN
    * Добавлен переключатель резистора конечного положения, его резистор 120R можно выбрать с помощью переключателя, без необходимости использования перемычек
    * Добавлена кабельная сборка питания/сигнала длиной 1 метр, кабель выполнен по технологии двойной спирали, что обеспечивает лучшую защиту от электромагнитных помех

## 1.3 Технические параметры

    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Разъем подключения использует XT30 (2+2) со способностью выдерживать ток до 15А, пиковый ток до 30А
    * Прошивка: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Датчики на борту: LIS2DW и LDC1612
    * Разъем для нагревательного элемента: HEAT0 (10A)
    * Расширительный разъем: RGB, EndStop, PT100/PT1000, XH2.54-2P разъем для расширения CAN
    * Разъемы для вентиляторов: два управляемых вентилятора DC
    * Драйвер двигателя: на борту TMC2209
    * Датчик температуры: один NTC 100K или PT1000 (TH0), один PT100/PT1000 на выбор
