# Использование Eddy

    :::info
  * Если ранее была настроена конфигурация eddy, удалите ее
  * При установке Eddy катушка должна находиться над соплом и быть на расстоянии 0,6-0,8 см от него
  * Во время тестирования не включайте нагревательный стол
  * Перед использованием удалите из конфигурации параметр `probe`
    :::

    :::warning Внимание
    * Для машин с несколькими Z-осями необходимо выполнить ручную выравнивание один раз
    :::

    # Для использования за рубежом используйте следующую команду
    ### Начинаем установку全自动脚本 eddy
    * Откройте скачанный инструмент ssh, введите следующие команды и нажмите Enter, затем нажмите кнопку копирования
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * В файле конфигурации `moonraker.conf` веб-страницы `fluidd` добавьте следующие настройки для удобства обновления
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Если обновили, используйте инструмент ssh для выполнения следующих команд
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* Выполните команду, и если появится изображение, введите пароль вашего верхнего компьютера

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* Если появится изображение, введите соответствующий номер принтера, есть выбор между китайским и английским языком, после ввода нажмите Enter
* Выберите `3z`, чтобы убедиться, что в конфигурации `printer.cfg` есть параметр `z_tilt` / Выберите `4z`, чтобы убедиться, что в конфигурации `printer.cfg` есть параметр `quad_gantry_level`, для независимого двойного Z выберите `3z`

* Если вы используете `sht36_v3`, введите `N` и нажмите Enter
