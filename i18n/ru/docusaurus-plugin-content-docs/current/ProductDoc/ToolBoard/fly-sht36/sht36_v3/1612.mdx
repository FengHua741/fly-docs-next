# Eddy Использование

    :::info
  * Если ранее была настроена конфигурация eddy, удалите её
  * При установке Eddy катушка должна находиться над соплом и находиться на расстоянии 0,6-0,8 см от него
  * Во время тестирования не включайте горячую платформу
  * Перед использованием удалите конфигурацию `probe` из конфигурации
    :::

    :::warning Внимание
    * Для машин с несколькими Z-осями необходимо ручной уровень
    :::

    # За рубежом используйте следующие команды
    ### Начинаем установку全自动脚本 eddy
    * Откройте скачанный SSH-инструмент, введите следующие команды, нажмите Enter, затем нажмите кнопку «Копировать»
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * В файле конфигурации `moonraker.conf` в веб-интерфейсе `fluidd` добавьте следующие настройки для удобства обновления
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Если обновили, используйте SSH-инструмент для выполнения следующих команд
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```

* Выполните команды, как показано на рисунке, введите пароль вашего компьютера

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* Как показано на рисунке, введите соответствующий номер принтера, если есть выбор между китайским и английским, введите его и нажмите Enter
* Выберите `3z`, чтобы убедиться, что в конфигурации `printer.cfg` есть настройка `z_tilt`, или выберите `4z`, чтобы убедиться, что в конфигурации `printer.cfg` есть настройка `quad_gantry_level`, для «независимой двойной Z» выберите `3z` [Ссылка на справочник конфигурации](https://mellow.klipper.cn/docs/category/klipper-%E5%8F%82%E8%80%83%E9%85%8D%E7%BD%AE)
