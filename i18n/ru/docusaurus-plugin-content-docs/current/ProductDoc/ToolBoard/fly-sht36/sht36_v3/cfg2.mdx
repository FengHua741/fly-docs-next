# Справочная конфигурация

```cfg
[probe_eddy_current fly_eddy_probe]
sensor_type: ldc1612
i2c_address: 43
i2c_mcu: SHT36
i2c_bus: i2c1e
x_offset: 0 # Не забудьте установить смещение по x
y_offset: 0 # Не забудьте установить смещение по y
speed:10
lift_speed: 15.0
i2c_speed: 4000000

[temperature_probe fly_eddy_probe]
sensor_type: Generic 3950
sensor_pin:SHT36:gpio28

[force_move]
enable_force_move: true
```
