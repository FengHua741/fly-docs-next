# Инструкция по прошивке прошивки
    
    * Для компиляции прошивки и её прошивки требуется ПО верхнего уровня
    * V3 SHT36 может подключаться к ПО верхнего уровня через RS232 или CAN
    * Для подключения RS232 и CAN используется кабель `XT30(2+2)-F`
    * Ниже приведён вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Прошивка основной платы требует записи Katapult для записи Klipper
* Если вы уверены, что основная плата уже прошита Katapult, то не нужно выполнять последний шаг по записи прошивки Katapult — это просто мера на случай неправильной записи прошивки
* Можно быстро дважды нажать кнопку Reset, чтобы проверить, загорается ли светодиод на материнской плате. Если да, то это означает, что материнская плата находится в режиме прошивки Katapult
:::

## Как определить, какую прошивку вам нужно

    * V3 SHT36 может подключаться к ПО верхнего уровня следующим образом
    * <Button variant="contained" disableElevation href="./can">CAN-прошивка</Button>
        * Преимущество: если установлена прошивка Katapult, прошивку можно обновлять без физического контакта с материнской платой, а также поддерживаются сетевые соединения
        * Недостаток: совместное использование с `KlipperScreen` может вызывать ошибки таймаута при возврате домой и т. д.
        * Примечание: если вы хотите использовать CAN, убедитесь, что ПО верхнего уровня поддерживает CAN, и необходимо использовать UTOC или материнскую плату с прошивкой шлюза
    * <Button variant="contained" disableElevation href="./rs232">RS232-прошивка</Button>
        * Преимущество: не требует специального ПО для использования, надёжная и стабильная связь
        * Недостаток: для повторной прошивки необходимо войти в DFU вручную или дважды щёлкнуть кнопку RST для входа в `Katapult` для обновления прошивки
        * Примечание: требуется комбинация с G2T или UTOR для использования
