# Инструкция по прошивке прошивки
    
    * Для компиляции прошивки и её записи необходим компьютерный интерфейс
    * SHT36 V3 может подключаться к компьютерному интерфейсу через RS232 или CAN
    * Для подключения RS232 и CAN используется переходник `XT30(2+2)-F`
    * Ниже приведен образец переходника `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Прошивка основной платы выполняется через Katapult для записи Klipper
* Если основная плата уже прошита Katapult, то нет необходимости выполнять последний этап записи прошивки Katapult — это только на случай, если прошивка Katapult не была успешно записана
* Можно быстро дважды нажать кнопку сброса, чтобы проверить, загорается ли светодиод на плите. Если да, то это означает, что плата находится в режиме Katapult
:::

## Как определить, какую прошивку вам нужно

    * SHT36 V3 может подключаться к компьютерному интерфейсу следующим образом
    * <Button variant="contained" disableElevation href="./can">CAN-прошивка</Button>
        * Преимущество: если установлено прошивка Katapult, обновление прошивки возможно без физического контакта с платой, при этом поддерживаются подключения по шине
        * Недостаток: возможны ошибки, такие как превышение времени возврата при совместном использовании с KlipperScreen
        * Примечание: если требуется использование CAN, убедитесь, что компьютерный интерфейс поддерживает CAN, и потребуется UTOC или плата с прошивкой моста
    * <Button variant="contained" disableElevation href="./rs232">RS232-прошивка</Button>
        * Преимущество: не требует специального программного обеспечения, связь надежна и стабильна
        * Недостаток: для обновления прошивки требуется вход в DFU вручную или двойное нажатие кнопки сброса для входа в Katapult
        * Примечание: требуется G2T или UTOR для использования
