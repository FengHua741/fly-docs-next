# Компиляция прошивки RS232

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Выберите **GPIO pins для установки при старте микроконтроллера**, введите: **!gpio5**
    :::warning
        Введите `!gpio5` в английской раскладке
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="65%" align="center" />
    * Если последний вывод содержит следующее изображение, компиляция успешно завершена
       <ImageView image={require('@site/docs/General/dfu-flash/img/make_uf2_finish.webp').default} size="65%" align="center" />




## Переход в режим записи

    :::tip

    Стрелка указывает на кнопку `BOOT`

    :::

    * При полностью отключенном инструментальном щите, удерживайте кнопку `BOOT`, подключите щит к интерфейсу USB вашего компьютера с помощью USB Type-C кабеля с функцией передачи данных, после правильного подключения отпустите кнопку `BOOT`
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="80%" align="center" />
