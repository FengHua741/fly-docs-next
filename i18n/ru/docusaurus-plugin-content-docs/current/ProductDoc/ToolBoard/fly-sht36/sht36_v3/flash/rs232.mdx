# Компиляция прошивки RS232

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Выберите **GPIO пины для установки при старте микроконтроллера**, введите: **!gpio5**
    :::warning
        Введите `!gpio5` в английской раскладке.
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции (время может занять немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее содержимое, то компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переключение в режим записи

    :::tip

    Стрелка указывает на кнопку `BOOT`.

    :::

    * Отключите питание инструментальной платы, удерживая кнопку `BOOT`, подключите плату инструментов к интерфейсу USB вашего основного устройства с помощью Type-C кабеля с функцией передачи данных, после правильного подключения отпустите кнопку `BOOT`.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/flash/img/boot.webp').default} size="80%" align="center" />
