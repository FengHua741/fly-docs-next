# Фирменное ПО Katapult

## Описание фирменного ПО Katapult

            4. Прошивка прошивки

            :::tip

            Необходимо прошить только одну из прошивок

            :::

            * Ниже приведена команда для прошивки прошивки с скоростью CAN 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Ниже приведена команда для прошивки прошивки с скоростью CAN 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. Если появляется нижеследующее уведомление, то прошивка успешна
