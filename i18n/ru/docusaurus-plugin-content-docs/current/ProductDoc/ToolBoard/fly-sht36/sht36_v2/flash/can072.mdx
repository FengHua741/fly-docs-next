# Компиляция прошивки версии 072 для CAN

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите **GPIO pins to set at micro-controller startup**, введите: !PC13 (обратите внимание на заглавные буквы)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/072.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена, и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, время может занять немного
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки используйте провод XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведен график соединения
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что перед питанием измеряется сопротивление между CAN H и CAN L, которое должно быть около `60 Ом`; если `120 Ом`, установите резистор `120 Ом`; если около `40 Ом`, отключите резистор
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/120.webp').default} size="50%" align="center" />
