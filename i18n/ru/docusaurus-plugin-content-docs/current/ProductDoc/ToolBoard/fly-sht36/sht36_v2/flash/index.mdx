# Инструкция по прошивке прошивки

    * Для компиляции прошивки и её прошивки необходим компьютерный интерфейс.
    * Устройство SHT36 V2 может подключаться к компьютерному интерфейсу через соединение CAN.
    * Для соединения CAN используйте кабель `XT30(2+2)-F`.
    * Ниже приведен вид кабеля `XT30(2+2)-F`:
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно знать
* Прошивка основной платы требует использования Katapult для прошивки Klipper.
* Если вы уверены, что основная плата уже прошита Katapult, то нет необходимости выполнять последний шаг прошивки прошивки Katapult — это только на случай, если прошивка не удалась.
* Вы можете быстро дважды нажать кнопку сброса, чтобы проверить, загорается ли светодиод на плате. Если да, то это означает, что устройство находится в прошивке Katapult.
:::

## Компиляция CAN-прошивки

    * Устройство SHT36 V2 может подключаться к компьютерному интерфейсу только через соединение CAN.
    * Для STM 103 см. <Button variant="contained" disableElevation component="a" rel="noopener" href="./can103">Прошивка прошивки CAN для 103</Button>.
    * Для APM 072 и STM 072 см. <Button variant="contained" disableElevation component="a" rel="noopener" href="./can072">Прошивка прошивки CAN для 072</Button>.
