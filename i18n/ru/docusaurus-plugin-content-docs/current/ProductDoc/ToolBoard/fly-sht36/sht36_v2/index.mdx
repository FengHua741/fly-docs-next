# Введение

## Описание продукта

    FLY SHT36 V2 — это инструментальная плата для головки экструдера 3D-принтера, разработанная компанией Guangzhou Miling Electronic Technology Co., Ltd. Использование этой платы позволяет заменить сложные соединения между основной платой и печатающей головкой четырьмя проводами, оптимизируя расположение. SHT36 V2 совместим с моторами на 36 шагов и включает в себя трансивер CAN, порт USB, драйвер TMC2209, акселерометр ADXL345, два управляемых вентилятора и другие функции. Данная плата предназначена для использования с прошивкой Klipper.

    * Компактность конструкции, выполненная по шестислойной технологии
    * Поддержка подключения по CAN-шинах для более стабильной передачи данных, меньшей задержки и надежного соединения
    * Наboard драйвер TMC2209, работающий в UART-режиме, может использоваться для экструдера
    * Наboard акселерометр ADXL345
    * Поддержка PT100 (MAX31865)
    * Возможность замены MOS-типа вентилятора
    * Совместимость с датчиками выравнивания BLTOUCH, PL08N и интеграция XY-ограничителей
    * Поддержка входного напряжения от 12 до 24 В
    * MCU: STM32F072
    * Коммуникация: CAN
    * Поддержка датчиков выравнивания BLTOUCH, PL08N и интеграция XY-ограничителей

## Новые функции

    * Добавлена возможность переключения между резисторами 4.7К (для термочувствительных элементов NTC) или 1.1К (для PT1000)
    * CAN-порт больше не использует MX3.0 разъем; новый разъем поддерживает ток до 15 А, пиковый ток — до 30 А
    * Добавлен выбор напряжений для вентиляторов: 5/12/24 В
    * Добавлен наboard резистор NTC100k для измерения температуры камеры
    * Добавлена защита от обратного подключения питания
    * Добавлен вывод INT1 акселерометра ADXL345 на MCU для некоторых пользователей, использующих его для выравнивания
    * Увеличен размер разъема нагревательного элемента, рассчитанный на ток 10 А
    * Добавлен предохранитель высокого напряжения
    * Заменен основной контроллер на национальный MCU
    * Поддержка магнитных энкодеров (as5047d), используемых Klipper для анализа движения (в настоящее время Klipper не поддерживает эту функцию)
    * Добавлена 1-метровая CAN-проводка, чтобы избежать проблем с потерей сигнала из-за неправильного соединения
    * Размер установки: расстояние между отверстиями 43,84 мм, диаметр установочных отверстий M3
    * Процессор: GD32F103 (до 28 октября 2022 года), APM32F072 (после 28 октября 2022 года)
