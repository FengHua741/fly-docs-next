# Описание продукта

    FLY-SHT36-LIS3DH — это инструментальная плата для головки экструдера, разработанная компанией Guangzhou Mellow Electronics Technology Co., Ltd. для 3D-принтеров. Используя эту плату, можно заменить сложные соединения между материнской платой и головкой печати четырьмя проводами, оптимизировав расположение. Плата FLY-SHT36-LIS3DH совместима с 36-ступенчатыми двигателями stepper и включает в себя трансивер CAN, порт USB, драйвер TMC2209, акселерометр LIS3DH, два управляемых вентилятора и другие функции. Данная плата предназначена для использования с прошивкой Klipper.

    * Компактный дизайн с极致ной минимизацией размеров, выполненная по шестислойной технологии
    * Поддержка CAN-шины для более стабильной передачи данных, меньшей задержки и более надежного подключения
    * Встроен драйвер TMC2209, работающий в режиме UART, может использоваться для экструдера
    * Встроенный акселерометр ADXL345
    * Поддерживает PT100 (MAX31865)
    * Дизайн MOSFET для замены вентиляторов
    * Совместимость с датчиками выравнивания BLTOUCH, PL08N и интеграция XY-ограничителей
    * Поддержка входного напряжения от 12 до 24 В
    * MCU: STM32F072
    * Коммуникации: CANBUS, USB
