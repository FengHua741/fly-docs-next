# Инструкция по прошивке прошивки

    * SHT36 LIS3DH может подключаться к ПК только через CAN
    * Для подключения CAN необходимо использовать кабель `XT30(2+2)-F`
    * Ниже приведен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно знать
* Для прошивки материнской платы требуется прошить Katapult через Katapult
* Если вы уверены, что на материнской плате уже установлена прошивка Katapult, то нет необходимости выполнять последующую прошивку прошивки Katapult, это лишь мера предосторожности на случай неудачной прошивки
* Можно быстро дважды нажать на кнопку сброса и проверить, загорается ли светодиод на материнской плате. Если да, то это означает, что материнская плата находится в режиме прошивки Katapult
:::

## Как определить, какая прошивка вам нужна

    * SHT36 LIS3DH можно подключаться к ПК следующими способами
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка прошивки CAN</Button>
        * Преимущество: если установлена прошивка `Katapult`, то можно обновлять прошивку без физического контакта с материнской платой, при этом поддерживаются сетевые подключения
        * Недостаток: возможно возникновение ошибок, таких как превышение времени возврата при использовании вместе с `KlipperScreen`
        * Примечание: если вы используете CAN, убедитесь, что ПК поддерживает CAN, и обязательно используйте UTOC или материнскую плату с прошивкой шлюза
