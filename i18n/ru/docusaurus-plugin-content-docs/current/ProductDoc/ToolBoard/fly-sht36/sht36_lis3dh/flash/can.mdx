    * Выберите GPIO пины для настройки при старте микроконтроллера, введите: `!gpio20`. Обратите внимание, что `!` — это заглавная буква.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите следующую команду для начала компиляции (время может занять немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце выводится следующее содержимое, компиляция прошла успешно:

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```

## ПРОГРАММИРОВАНИЕ ФИРМЕНТЫ

    * Перед программированием используйте кабель XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведен пример соединительной схемы
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что при отключенном питании измерение сопротивления между CAN H и CAN L составляет около `60 Ом`. Если показатель равен `120 Ом`, подключите переходник `120 Ом`; если около `40 Ом`, уберите переходник
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/120.webp').default} size="50%" align="center" />
