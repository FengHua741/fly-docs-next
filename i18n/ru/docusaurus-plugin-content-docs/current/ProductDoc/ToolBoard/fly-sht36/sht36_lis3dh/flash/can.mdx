        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />

    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующее, компиляция успешно завершена:

        ```bash
        Связывание out/klipper.elf
        Создание hex-файла out/klipper.bin
        ```

## Программирование прошивки

    * Перед программированием подключите инструментальную плату к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30. Ниже приведена схема соединения
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v2/flash/img/sht36v2_utoc.webp').default} size="80%" align="left" />
    * Проверьте сопротивление между CAN H и CAN L мультиметром при отключенном питании. Оно должно быть около `60 Ом`. Если оно составляет `120 Ом`, установите резистор `120 Ом`. Если оно около `40 Ом`, снимите резистор
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/120.webp').default} size="50%" align="center" />
