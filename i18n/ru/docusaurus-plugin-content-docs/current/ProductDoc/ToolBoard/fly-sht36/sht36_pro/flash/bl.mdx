# Фирменное ПО Katapult

## Описание фирменного ПО Katapult

    <CANBOOT />

    :::tip

    * SHT36 PRO поддерживает CAN
    * Здесь доступны только версии ПО CAN с скоростью 500K и 1M
    :::

## Вход в режим программирования

    :::tip

    Круглым выделен `BOOT кнопка`

    :::

    * При полном отключении питания платы, коротко замкните BOOT кнопку, а затем подключите кабель передачи данных к плате и вашему устройству
    * Затем отпустите кнопку BOOT

            4. Прошивка прошивки

            :::tip

            Необходимо прошить только одну из прошивок

            :::

            * Ниже приведена команда для прошивки прошивки с скоростью CAN 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Ниже приведена команда для прошивки прошивки с скоростью CAN 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. Если появляется нижеследующее уведомление, то прошивка успешна

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            После завершения прошивки необходимо полностью отключить питание платы.

            :::
