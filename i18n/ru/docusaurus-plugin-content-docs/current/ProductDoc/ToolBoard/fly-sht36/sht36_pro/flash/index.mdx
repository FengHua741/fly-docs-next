# Инструкция по прошивке прошивки

    * SHT36 PRO может подключаться к компьютеру через CAN-интерфейс
    * Для подключения CAN требуется использовать кабель `XT30(2+2)-F`
    * Ниже представлен образец кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Для прошивки основной платы необходимо использовать Katapult для прошивки Klipper
* Если основная плата уже прошита Katapult, то необязательно выполнять последний шаг по прошивке прошивки Katapult — это просто на случай, если прошивка Katapult на главной плате не удалась
* Можно быстро дважды нажать кнопку сброса и проверить, загорается ли светодиод на плате. Если да, то это означает, что плата находится в режиме Katapult
:::

## Компиляция CAN-прошивки

    * SHT36 PRO может подключаться к компьютеру только через CAN-интерфейс
    * <Button variant="contained" disableElevation href="./can">Прошивка CAN-прошивки</Button>
        * Преимущество: если установлено прошивка Katapult, можно обновить прошивку без физического контакта с платой и поддерживаются сетевые подключения
        * Недостаток: возможно возникновение ошибок типа превышения времени ожидания при совместном использовании с KlipperScreen
        * Совет: перед использованием CAN убедитесь, что ваш компьютер поддерживает CAN, и используйте его совместно с UTOC или материнской платой с прошивкой шлюза
