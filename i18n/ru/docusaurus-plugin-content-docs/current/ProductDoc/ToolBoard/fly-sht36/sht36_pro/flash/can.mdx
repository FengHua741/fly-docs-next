    * Выберите GPIO пины, которые нужно установить при старте микроконтроллера, введите: !gpio13 Обратите внимание, что восклицательный знак должен быть заглавным
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого означает успешную компиляцию:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста через XT30 кабель, ниже приведен пример соединения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/sht36pro_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что перед питанием CAN H и CAN L имеют сопротивление около `60Ом`. Если показывает `120Ом`, подключите переходник `120Ом`, если около `40Ом`, удалите переходник
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/120.webp').default} size="80%" align="center" />
