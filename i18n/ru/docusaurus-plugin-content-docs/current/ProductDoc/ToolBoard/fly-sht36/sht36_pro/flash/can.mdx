# Компиляция CAN-программного обеспечения

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите GPIO пины для настройки при старте микроконтроллера, введите: !gpio13 (обратите внимание, что восклицательный знак должен быть заглавным)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна сохраниться конфигурация и вы вернетесь в командную строку
    * Введите `make -j4` для начала компиляции, процесс займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого — компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка программного обеспечения

    * Перед прошивкой используйте провод XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведен пример соединения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/sht36pro_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что перед питанием CAN H и CAN L измеряются мультиметром сопротивление около `60 Ом`, если оно `120 Ом`, подключите резистор `120 Ом`, если около `40 Ом`, удалите резистор
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/120.webp').default} size="80%" align="center" />
