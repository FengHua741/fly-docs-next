# Компиляция прошивки CAN

## Начало компиляции

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: `!gpio13`. Обратите внимание, что `!` пишется большими буквами
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернулись к командной строке
    * Введите следующую команду для начала компиляции (время может занять немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод последней строки содержит:
        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
        то компиляция успешно завершена.

## Запись прошивки

    * Перед записью прошивки подключите инструментальную плату к UTOC или материнской плате с прошивкой моста через XT30 кабель. Ниже приведена схема подключения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/sht36pro_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется около `60 Ом` в отключенном состоянии. Если показатель составляет `120 Ом`, подключите переходник `120 Ом`; если около `40 Ом`, снимите переходник
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/120.webp').default} size="80%" align="center" />

    <CANFLASH />
