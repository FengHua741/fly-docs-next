# Описание продукта

    FLY-SHT36-PRO — это платформа для головки экструдера, разработанная компанией Guangzhou Mellen Electronics Technology Co., Ltd. для 3D-принтеров. Используя данную платформу, можно заменить сложные соединения между основной платой и головкой принтера четырьмя проводами, что оптимизирует расположение. FLY-SHT36-PRO совместим с моторами на 36 шагов и интегрирует трансивер CAN, разъем USB, драйвер TMC5160, акселерометр ADXL345, два управляемых вентилятора и другие функциональные компоненты.

    * Экстремально компактный размер, выполнена по шестислойной технологии
    * Поддерживает подключение по CAN-шиине, обеспечивающее более стабильную передачу данных, меньшее время задержки и надежное соединение
    * На борту установлен драйвер TMC5160 для экструдера
    * На борту установлен акселерометр ADXL345
    * Поддерживает PT100 (MAX31865)
    * Разъем MOS для замены вентиляторов
    * Совместим с датчиками выравнивания BLTOUCH, PL08N и интегрированы XY-ограничители
    * Поддерживает входное напряжение 12-24В
    * MCU: STM32F072
    * Коммуникация: CANBUS, USB

## Технические характеристики

    1. Поддерживает режимы Klipper CAN, RS232, RRF_CAN
    2. Разъем подключения использует XT30 (2+2) со способностью пропускать до 15А, пиковый ток до 30А
    3. MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    4. Входное напряжение: DC12-24V 5A
    5. Максимальный выходной ток нагревательного элемента: 5A
    6. Максимальный выходной ток вентилятора: 1A
