# Общее описание продукта

    FLY SHT36/42 — это платформа для головки экструдера, разработанная компанией Guangzhou Mellen Electronics Technology Co., Ltd. для 3D-принтеров. Используя данную плату, можно заменить сложные соединения между материнской платой и головкой печати четырьмя проводами, что улучшает структуру подключения. SHT36 совместим с шаговыми двигателями 36, а SHT42 — с двигателями 42. Платформа интегрирует трансивер CAN, порт USB, драйвер TMC2209, акселерометр ADXL345, два управляемых вентилятора и другие функциональные компоненты. Данная платформа совместима с прошивкой Klipper.

    * Компактный дизайн, 6-слойная конструкция
    * Поддержка подключения по CAN-шины для более стабильной передачи данных, меньшей задержки и надежного подключения
    * Наboard установлен драйвер TMC2209, использующий UART-режим для работы с экструдером
    * Наboard установлен акселерометр ADXL345
    * Поддержка PT100 (MAX31865)
    * Дизайн MOS-переключателей для вентиляторов
    * Совместимость с датчиками выравнивания BLTOUCH, PL08N и интеграция XY-ограничителей
    * Поддержка входного напряжения от 12 до 24 В
    * MCU: STM32F072
    * Коммуникации: CANBUS, USB
    * Интерфейсы: Управляемые вентиляторы*2, RGB*1, Limit*3, Probe*1, Servo*1, нагревательная трубка*1, термистор*1, PT100*1

## Технические характеристики

    1. Габаритные размеры: 51.27мм×36.26мм (SHT36), 42мм×42мм (SHT42)
    2. Размер крепления: расстояние между отверстиями 43.84мм, диаметр M3 установочных отверстий
    3. Процессор: STM32F072C8T6
    4. Входное напряжение: DC12-24В 5А
    5. Максимальный ток нагревательной спирали: 5А
    6. Максимальный ток на интерфейсе вентилятора: 1А
