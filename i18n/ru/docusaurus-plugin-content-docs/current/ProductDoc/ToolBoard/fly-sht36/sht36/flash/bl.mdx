            4. Программирование прошивки

            :::tip

            Необходимо программировать только одну из прошивок

            :::

            * Ниже приведена команда для записи прошивки со скоростью CAN 500K
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_500K.BIN
                ```
                       
            * Ниже приведена команда для записи прошивки со скоростью CAN 1M
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_1M.BIN
                ```


            5. При появлении указанного на изображении содержимого ``File downloaded successfully`` прошивка успешно выполнена

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::tip

            После завершения программирования необходимо полностью отключить питание материнской платы.

            :::
