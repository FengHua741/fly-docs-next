# Инструкция по прошивке прошивки

    * SHT36/42 можно подключать к ПК через подключение по CAN
    * Для подключения по CAN требуется использование металлических наконечников MX3.0
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />

## Как определить, какую прошивку вам нужна

    * SHT36/42 может подключаться к ПК через подключение CAM
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка прошивки CAN</Button>
        * Преимущество: если установлено прошивка `Katapult`, можно обновлять прошивку без необходимости физического контакта с материнской платой, а также поддерживаются подключения по шине
        * Недостаток: возможно возникновение ошибок типа превышения времени ожидания при совместном использовании с `KlipperScreen`
        * Примечание: если вы хотите использовать CAN, убедитесь, что ваш ПК поддерживает CAN, и используйте UTOC или материнскую плату с прошивкой моста
