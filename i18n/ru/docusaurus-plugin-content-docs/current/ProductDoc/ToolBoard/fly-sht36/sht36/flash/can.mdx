    <CAN />

    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце выводится следующее содержимое, то компиляция прошла успешно

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Прошивка программного обеспечения

    * Перед прошивкой используйте XT30-кабель для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведена схема подключения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется при отключенном питании и составляет около `60 Ом`; если оно равно `120 Ом`, подключите резистор на `120 Ом`; если около `40 Ом`, удалите резистор
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/flash/img/120.webp').default} size="100%" align="center" />
