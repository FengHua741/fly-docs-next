    * В конце вывода следующего содержимого означает успешную компиляцию:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки используйте кабель XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведена схема подключения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется мультиметром при отключенном питании и составляет около `60 Ом`; если показывает `120 Ом`, подключите переходник на `120 Ом`, если около `40 Ом`, удалите переходник
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/flash/img/120.webp').default} size="100%" align="center" />
