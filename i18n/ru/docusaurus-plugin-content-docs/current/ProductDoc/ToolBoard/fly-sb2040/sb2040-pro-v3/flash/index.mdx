# Инструкция по прошивке прошивки

    * SB2040 PRO V3 может быть подключено к ПК двумя способами: через RS232 или CAN
    * Для подключения RS232 и CAN используется переходник `XT30(2+2)-F`
    * Ниже показан вид переходника `XT30(2+2)-F`
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Для прошивки материнской платы требуется прошить Katapult для прошивки Klipper
* Если вы уверены, что материнская плата уже прошита Katapult, то не нужно выполнять последнюю процедуру прошивки прошивки Katapult — это только мера предосторожности на случай неудачной прошивки
* Можно быстро дважды нажать кнопку Reset, чтобы проверить, загорается ли светодиод на материнской плате; если да, то это означает, что материнская плата уже в режиме Katapult
:::

## Как определить нужную вам прошивку

    * SB2040 PRO V3 может подключаться к ПК следующими способами:
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка CAN-программного обеспечения</Button>
        * Преимущество: если установлена прошивка `Katapult`, можно обновлять программное обеспечение без необходимости физического контакта с материнской платой, а также поддерживаются шины подключения
        * Недостаток: при совместном использовании с `KlipperScreen` могут возникать ошибки, такие как превышение времени ожидания
        * Примечание: если вы используете CAN, убедитесь, что ПК поддерживает CAN, и используйте его вместе с UTOC или материнской платой с прошивкой шлюза
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Прошивка RS232-программного обеспечения</Button>
        * Преимущество: не требует специального программного обеспечения, надежная и стабильная связь
        * Недостаток: для повторной прошивки необходимо войти в DFU вручную или дважды нажать RST, чтобы войти в `Katapult` для обновления программного обеспечения
        * Примечание: для использования требуется G2T или UTOR
