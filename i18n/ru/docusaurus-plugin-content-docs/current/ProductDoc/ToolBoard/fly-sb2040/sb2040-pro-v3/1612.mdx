# Использование Eddy

    :::info

    * Обратите внимание, что для сканирования платформы рекомендуется использовать отдельный Z-концевик, не рекомендуется использовать сканирование платформы как Z-концевик
    * При установке Eddy, дно должно находиться на расстоянии 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от нагревательной платформы
    * Пожалуйста, не проводите тестирование на нагревательной платформе при комнатной температуре, и не включайте нагревательную платформу во время тестирования

    :::

## Рекомендации по настройке и использованию

### Изменение стандартной частоты ldc1612 1612
    Этот метод используется только для sht36-v3/SB2040-v3 или SB2040-Pro-v3
    Можно использовать инструмент Mobaxterm для выполнения команд

    Скачайте необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент Mobaxterm</Button>
    
    [Подробные инструкции смотрите здесь](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    Выполните первую команду и нажмите Enter

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    Выполните вторую команду и нажмите Enter
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    Если замена успешна, будет возвращена строка с `LDC1612_FREQ = 40000000`. Если нет возврата, значит замена не удалась.
    Или используйте эту команду для проверки успешности
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```

   Примечание: После изменения используйте эту команду для перезапуска службы Klipper и повторной калибровки частоты Eddy
   ```cfg
   systemctl restart klipper
   ```
   Команда калибровки частоты Eddy
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### Настройка сканирования платформы

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что в настоящее время внешний модуль 1612 еще не выпущен, эта конфигурация временно не может быть использована
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Настройка температурной компенсации

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Настройка bed_mesh

    * В [bed_mesh] параметр `horizontal_move_z` рекомендуется установить на 2, чтобы Eddy был как можно ближе к платформе во время сканирования

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
