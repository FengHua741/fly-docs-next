# Обзор джамперов на плате

:::warning Важное замечание

Из-за высокого разрешения изображений просмотреть их на веб-странице невозможно. Пожалуйста, наведите **курсор мыши** на схему подключения и выберите **сохранить изображение как...** , чтобы сохранить изображение на рабочий стол и рассмотреть его более детально!!

:::
