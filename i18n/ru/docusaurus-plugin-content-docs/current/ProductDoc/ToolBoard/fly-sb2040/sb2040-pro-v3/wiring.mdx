# Схема подключения материнской платы

    :::warning Важное примечание

    Из-за высокого разрешения изображения, при просмотре на веб-странице невозможно рассмотреть детали. Пожалуйста, поместите курсор мыши над схемой подключения, щелкните правой кнопкой мыши и выберите "Сохранить изображение как", чтобы сохранить изображение на рабочий стол и затем просмотреть его!!!

    :::
