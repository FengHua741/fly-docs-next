        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```

## Переход в режим прошивки

    :::tip

    Слева — кнопка `RESET`, справа — кнопка `BOOT`

    :::

    * Есть два способа войти в DFU
    * Первый: полностью отключите питание платы, удерживая кнопку BOOT, подключите кабель данных к плате и компьютеру
    * Второй: подключите кабель данных к плате и компьютеру, удерживайте кнопку BOOT, нажмите кнопку RST, затем отпустите кнопку RST и последовательно отпустите кнопку BOOT
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
