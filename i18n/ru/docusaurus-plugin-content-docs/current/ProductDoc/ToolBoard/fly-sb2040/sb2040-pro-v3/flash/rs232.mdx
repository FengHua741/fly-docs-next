    * Если выводится следующее содержимое, то компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Создание файла uf2 out/klipper.uf2
        ```


## Переключение в режим прошивки

    :::tip

    Слева — кнопка RESET, справа — кнопка BOOT

    :::

    * Есть два способа переключиться в DFU-режим
    * Первый: полностью отключите питание платы, удерживайте кнопку BOOT, затем подключите данные передачи одной концовкой к плата, другой к компьютеру
    * Второй: подключите данные передачи одной концовкой к плата, другой к компьютеру, удерживайте кнопку BOOT, затем нажмите кнопку RST, затем отпустите кнопку RST, последовательно отпустите кнопку BOOT
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
