    <CAN />

    * Выберите **GPIO пины, установленные при старте микроконтроллера**, введите: **!gpio5,!gpio19,!gpio24**
    :::warning
        Пожалуйста, вводите в режиме английской раскладки клавиатуры `!gpio5,!gpio19,!gpio24`
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись к командной строке
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, то компиляция успешна:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30. Ниже приведен пример схемы подключения
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/sb2040v3_utoc.webp').default} size="80%" align="center" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется вольтметром около `60 Ом`. Если показывает `120 Ом`, включите переключатель на `120 Ом`. Если около `40 Ом`, отключите переключатель
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.webp').default} size="100%" align="center" />
