# Компиляция CAN-программного обеспечения

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите **GPIO пины, которые должны быть установлены при запуске микроконтроллера**, введите: **!gpio5**
    :::warning
        Пожалуйста, вводите в английской раскладке клавиатуры `!gpio5,!gpio19,!gpio24`
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите `Q`, появится **Save configuration**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если в конце выводится следующее содержимое, компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста через кабель XT30. Ниже приведена ссылка на схему подключения
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/sb2040v3_utoc.webp').default} size="80%" align="center" />
    * Убедитесь, что перед питанием измеряется сопротивление между CAN H и CAN L на уровне около `60 Ом`. Если сопротивление составляет `120 Ом`, включите переключатель `120 Ом`. Если оно около `40 Ом`, отключите переключатель
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.webp').default} size="100%" align="center" />
