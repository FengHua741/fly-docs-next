# Компиляция CAN-программного обеспечения

## Начало компиляции

    <SSH />

    <CAN />

    * Выберите **GPIO пины для установки при старте микроконтроллера**, введите: **!gpio5,!gpio19,!gpio24**
    :::предупреждение
        Пожалуйста, вводите в режиме английской раскладки клавиатуры `!gpio5,!gpio19,!gpio24`
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь конфигурация должна быть сохранена и вы должны вернуться к командной строке
    * Введите следующую команду для начала компиляции (время может занять длительное):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод содержит следующие строки, то компиляция успешна:

        ```bash
        Связывание out/klipper.elf
        Создание файла bin out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки используйте кабель XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведен пример соединительной схемы
    * Убедитесь, что переключатели установлены правильно
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/sb2040v3_utoc.webp').default} size="80%" align="center" />
    * Убедитесь, что перед питанием измеряется сопротивление между CAN H и CAN L на уровне около `60 Ом`. Если показатель равен `120 Ом`, то переключатель для `120 Ом` должен быть включен. Если около `40 Ом`, то переключатель должен быть выключен
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.webp').default} size="100%" align="center" />
