# Фирменное ПО Katapult

## Описание фирменного ПО Katapult

    <CANBOOT />

    :::tip

    * SB2040_PRO_V3 поддерживает CAN и RS232
    * Здесь предоставляется только CAN-программное обеспечение с скоростями 500K и 1M
    :::

## Вход в режим программирования

    :::tip

    Слева - кнопка "RESET", справа - кнопка "BOOT"

    :::

    * Есть два способа войти в режим программирования
    * Первый: при полном отключении питания нажимаем и удерживаем кнопку BOOT, затем подключаем кабель передачи данных к материнской плате и вашему устройству
    * Второй: после подключения кабеля передачи данных к материнской плате и вашему устройству, удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST, а затем отпустите кнопку BOOT
