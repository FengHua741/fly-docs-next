# Инструкция по прошивке прошивки

    * SB2040 может подключаться к ПК через соединение CAN
    * Для подключения CAN используется кабель `XT30(2+2)-F`
    * Ниже приведен образец кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно знать
* Для прошивки основной платы требуется использовать Katapult для прошивки Klipper
* Если основная плата уже прошита Katapult, то нет необходимости выполнять последующую прошивку прошивки Katapult, это просто мера на случай неудачной прошивки
* Можно быстро дважды нажать кнопку сброса, чтобы проверить, загорится ли светодиод на плите. Если да, то плата уже находится в прошивке Katapult
:::

## Как определить, какая прошивка вам нужна

    * SB2040 может подключаться к ПК только через соединение CAN
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка прошивки CAN</Button>
        * Преимущество: если установлена прошивка Katapult, можно обновлять прошивку без физического контакта с платой и поддерживаются шинные соединения
        * Недостаток: возможно возникновение ошибок типа сброса в случае совместного использования с KlipperScreen
        * Подсказка: если необходимо использовать CAN, убедитесь, что ПК поддерживает CAN, и используйте его вместе с UTOC или материнской платой с прошивкой шлюза
