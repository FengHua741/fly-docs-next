## Начало компиляции

    <CAN />

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции, процесс может занять некоторое время
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если вывод последних строк содержит:
        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
        то компиляция успешно завершена.

## Запись прошивки

    * Перед записью прошивки используйте кабель XT30 для подключения инструментальной платы к UTOC или материнской плате с прошивкой моста, ниже приведена схема соединения
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/sb2040_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется около `60 Ом` в отключенном состоянии. Если показывает `120 Ом`, подключите переходник `120 Ом`; если около `40 Ом`, удалите переходник
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/120.webp').default} size="50%" align="center" />
