# Фирменное ПО Katapult

## Описание Фирменного ПО Katapult

    <CANBOOT />

    :::tip

    * SB2040 поддерживает CAN
    * Здесь доступны только прошивки CAN с скоростью 500K и 1M
    :::

## Вход в режим программирования

    :::tip

    Обведённым является `BOOT-кнопка`

    :::

    * При полном отключении питания, удерживайте BOOT-кнопку и подключите данные провода к материнской плате и вашему устройству

            4. Программирование прошивки

            :::tip

            Необходимо запрограммировать только одну из прошивок

            :::

            * Ниже приведена команда для записи прошивки с CAN-скоростью 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_500K.uf2
                ```
                       
            * Ниже приведена команда для записи прошивки с CAN-скоростью 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040_CANBOOT_1M.uf2
                ```


            5. Если появляется нижеследующее уведомление, то процесс программирования завершён успешно

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            После завершения программирования необходимо полностью отключить питание материнской платы.

            :::

        </TabItem>
    </Tabs>
