# 1. Общее описание

FLY SB2040 V3 — это инструментальная плата, разработанная компанией Guangzhou Mellen Electronics Technology Co., Ltd. для принтерной головы VORON Stealthburber. Использование этой платы позволяет заменить сложные соединения между основной платой и печатной головкой четырьмя проводами, оптимизируя расположение. SB2040 совместим с stealthburner и afterburner, включает в себя CAN-трансивер, трансивер RS232, USB-порт, драйвер TMC2209, акселерометр LIS2DW и три управляемых вентилятора, а также другие функции. Данная плата поддерживает прошивки Klipper и RRF.

## 1.1 Особенности продукта

    * Предусмотрена кнопка Boot, пользователи могут войти в режим Boot через TypeC для обновления прошивки
    * Поддерживает прошивки Klipper и RRF
    * Поддерживает режим CAN, режим RS232, режим RRF CAN,
    * В режиме RS232 обеспечивается более высокая скорость передачи данных и большая стабильность. В режиме CAN поддерживаются расширения CAN, предусмотрены разъемы для расширения CAN
    * Датчики температуры: интегрирован чип MAX31865, совместим с датчиками температуры PT100 и PT1000
    * Интегрирован акселерометр LIS2DW и датчик LDC1612
    * Предусмотрен разъем RGB, интегрирован драйвер 2209, поддерживает различные датчики выравнивания, например, klicky, Voron Tap, PL08n и т. д.
    * Поддерживает входное напряжение от 12 до 24В, ток до 15А, увеличен разъем нагревательного элемента, номинальный ток 10А

## 1.2 Новые функции

    * Поддержка переключения между режимами Klipper CAN, RS232, RRF CAN
    * Добавлен режим RS232 с более высокой скоростью передачи данных и лучшей стабильностью, проще в настройке
    * Добавлен режим RRF CAN, может быть использована как плата RRF, что повышает универсальность
    * Добавлен разъем для расширения CAN, упрощает подключение устройств CAN
    * Добавлен переключатель резисторов терминатора, резистор терминатора 120R可以选择 через переключатель, без необходимости пайки
    * Добавлена кабельная сборка питания/сигнала длиной 1 метр, с двойной спиральной конструкцией, обеспечивающая лучшую защиту от помех

## 1.3 Технические характеристики


    * Микроконтроллер: Raspberry Pi rp2040, Dual core ARM Cortex-M0+ @ 133MHz
    * Разъемы подключения используют XT30 (2+2), поддерживают ток до 15А, пиковый ток до 30А
    * Прошивки: Reprap/Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Интегрированные датчики: LIS2DW и опционально ldc1612 и max31865
    * Разъем для нагревательного элемента: HEAT0
    * Расширяемые разъемы: RGB, EndStop, PT100/PT1000, XH2.54-2P CAN
    * Разъемы для вентиляторов: один канал DC вентилятора, два канала DC вентиляторов или PWM вентиляторов
    * Драйвер двигателя: интегрированный TMC2209
    * Датчики температуры: один канал 100K NTC или PT1000 (TH0), один канал PT100/PT1000 по выбору
