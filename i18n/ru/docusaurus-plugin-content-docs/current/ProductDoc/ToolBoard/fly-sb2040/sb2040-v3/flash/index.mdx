# Инструкция по прошивке прошивки

    * V3 SB2040 может подключаться к ПК двумя способами: через RS232 или CAN
    * Для подключения через RS232 или CAN требуется использовать кабель `XT30(2+2)-F`
    * Ниже приведен вид кабеля `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно
* Прошивка материнской платы осуществляется через Katapult для записи Klipper
* Если вы уверены, что материнская плата уже прошита Katapult, то нет необходимости выполнять последующую запись прошивки Katapult — это только на случай неудачной предыдущей попытки
* Можно быстро дважды нажать кнопку Reset и проверить, загорается ли светодиод на материнской плате (если загорается, значит, материнская плата находится в прошивке Katapult)
:::

## Как определить, какую прошивку вам нужно

    * V3 SB2040 может подключаться к ПК следующими способами
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Прошивка прошивки CAN</Button>
        * Преимущество: если установлено прошивка Katapult, обновление прошивки возможно без физического контакта с материнской платой, а также поддерживаются шинные подключения
        * Недостаток: совместное использование с `KlipperScreen` может вызывать ошибки, такие как превышение времени ожидания при возврате домой
        * Примечание: если вы хотите использовать CAN, убедитесь, что ваш ПК поддерживает CAN, и используйте его вместе с UTOC или материнской платой с прошивкой моста
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Прошивка прошивки RS232</Button>
        * Преимущество: не требует специального программного обеспечения для использования, надежная и стабильная связь
        * Недостаток: для повторной записи прошивки необходимо войти в режим DFU вручную или дважды нажать кнопку Reset для входа в Katapult для обновления прошивки
        * Примечание: требуется использовать G2T или UTOR
