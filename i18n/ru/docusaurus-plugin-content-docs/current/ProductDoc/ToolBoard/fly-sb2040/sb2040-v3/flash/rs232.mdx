    <RS232 />

    * Выберите **GPIO pins для установки при старте микроконтроллера**, введите: **!gpio5**
    :::warning
        Пожалуйста, вводите `!gpio5` в режиме английской раскладки клавиатуры.
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/rs232.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Save configuration**, затем нажмите клавишу `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернулись к командной строке
    * Введите `make -j4` для начала компиляции, процесс может занять некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, то компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переключение в режим загрузки

    :::tip

    Слева находится кнопка **RESET**, справа — **BOOT**

    :::

    * Есть два способа переключиться в DFU-режим
    * Первый: полностью отключите питание платы, удерживая кнопку BOOT, подключите шнур данных к материнской плате и компьютеру
    * Второй: подключите шнур данных к материнской плате и компьютеру, удерживайте кнопку BOOT, затем нажмите кнопку RST, отпустите кнопку RST, затем отпустите кнопку BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.webp').default} size="100%" align="center" />
