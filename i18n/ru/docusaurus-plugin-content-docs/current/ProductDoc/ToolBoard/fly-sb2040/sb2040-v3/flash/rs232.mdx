    * Теперь должна быть сохранена конфигурация и вы вернулись в командную строку
    * Введите следующую команду для начала компиляции, процесс может занять некоторое время
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит следующее, компиляция успешно завершена

        ```bash
        Linking out/klipper.elf
        Creating uf2 file out/klipper.uf2
        ```


## Переключение в режим прошивки

    :::tip

    Слева — кнопка `RESET`, справа — кнопка `BOOT`

    :::

    * Есть два способа переключиться в DFU-режим
    * Первый: полностью отключите питание платы, удерживая кнопку BOOT, подключите переходник с данными к плате и компьютеру
    * Второй: подключите переходник с данными к плате и компьютеру, удерживая кнопку BOOT, нажмите кнопку RST, затем отпустите кнопку RST, а затем кнопку BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/boot.webp').default} size="100%" align="center" />
