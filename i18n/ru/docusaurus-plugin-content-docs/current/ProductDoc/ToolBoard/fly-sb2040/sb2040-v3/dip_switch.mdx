# Обзор переключателей на плате

:::warning Важное замечание

Из-за высокого разрешения изображений просмотреть их на веб-странице невозможно. Пожалуйста, **наведите курсор мыши** на схему подключения, **кликните правой кнопкой мыши** и выберите **Сохранить изображение как...**, чтобы сохранить изображение на рабочий стол и просмотреть его там!!

:::
