# Использование Eddy

    :::info

    * Обратите внимание, что для сканирования кровати рекомендуется использовать отдельный Z-ограничитель, не рекомендуется использовать сканирование кровати как Z-ограничитель
    * При установке Eddy, положение основания относительно сопла должно быть на 1-2 мм выше сопла
    * При калибровке частоты Eddy, катушка PCB должна быть на расстоянии не менее 5 мм от нагревательной кровати
    * Пожалуйста, не включайте нагревательную кровать при тестировании на обычной температуре

    :::

## Справочная конфигурация и инструкции по использованию

### Изменение стандартной частоты ldc1612 1612
    Этот метод применим только к sht36-v3/SB2040-v3 или SB2040-Pro-v3
    Можно использовать инструмент Mobaxterm для выполнения команд

    Загрузите необходимый <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">инструмент Mobaxterm</Button>

    
    [Для подробного использования обратитесь к](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    Выполните первую команду и нажмите Enter

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    Выполните вторую команду и нажмите Enter
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    Если замена прошла успешно, будет возвращена строка с `LDC1612_FREQ = 40000000`, тогда замена успешна. Если ничего не возвращено, значит замена не удалась.
    Или используйте эту команду для проверки успеха
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   Подсказка: После изменения используйте эту команду для перезапуска службы klipper и повторной калибровки частоты Eddy
   ```cfg
   systemctl restart klipper
   ```
   Команда калибровки частоты Eddy
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### Конфигурация сканирования кровати

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Обратите внимание, что пока нет внешнего модуля 1612, эта конфигурация пока не работает
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Конфигурация температурной компенсации

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Конфигурация bed_mesh

    * Параметр `horizontal_move_z` в [bed_mesh] рекомендуется установить на 2, чтобы при сканировании платформы Eddy был как можно ближе к платформе

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
