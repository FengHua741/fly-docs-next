# Использование Eddy

    :::info
  * Если ранее была настроена конфигурация eddy, удалите её
  * При установке Eddy катушка должна находиться над соплом и быть на расстоянии 0,6-0,8 см от него
  * Во время тестирования не включайте нагревательный стол
  * Перед использованием удалите конфигурацию `probe` из конфигурации
    :::

    :::warning Внимание
    * Для машин с несколькими Z-осями необходимо выполнить ручную регулировку один раз
    :::

    # Используйте следующую команду для международного использования
    ### Начинаем установку全自动脚本 eddy
    * Откройте скачанный ssh-инструмент, введите следующие команды, нажмите Enter, затем нажмите кнопку "Копировать"
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * В файле конфигурации `moonraker.conf` в веб-интерфейсе `fluidd` добавьте следующую конфигурацию, чтобы облегчить обновление в будущем
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Если обновили, выполните следующую команду
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```

* При выполнении появится сообщение, введите пароль вашего компьютера

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* Появится сообщение "Выберите ввод", введите номер принтера, соответствующий вашему принтеру, есть выбор на китайском и английском языках, после ввода нажмите Enter
* Выберите `3z`, чтобы убедиться, что в файле конфигурации `printer.cfg` есть конфигурация `z_tilt`. Выберите `4z`, чтобы убедиться, что в файле конфигурации `printer.cfg` есть конфигурация `quad_gantry_level`. Для независимых двойных Z-осей выберите `3z`. [Ссылка на справочник конфигураций](https://mellow.klipper.cn/docs/category/klipper-%E5%8F%82%E8%80%83%E9%85%8D%E7%BD%AE)
