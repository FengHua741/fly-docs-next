# Использование Eddy

    :::info
  * Если ранее была настроена конфигурация eddy, удалите её
  * При установке Eddy катушка должна быть над соплом и находиться на расстоянии 0,6-0,8 см от него
  * Во время тестирования не включайте горячую плату
  * Перед использованием удалите конфигурацию `probe` из конфигурации
    :::

    :::warning Внимание
    * Для машин с несколькими Z-осью необходимо выполнить ручную выравнивание один раз
    :::

    # Используйте следующие команды для международного использования
    ### Начало установки автоматического скрипта eddy
    * Откройте загруженный инструмент SSH, введите следующие команды и нажмите Enter, затем нажмите копировать
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * В файле конфигурации `moonraker.conf` веб-страницы `fluidd` добавьте следующие настройки для удобства обновления
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Если обновили, выполните следующую команду
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```

* Выполните команду, и если появится изображение, введите пароль вашего верхнего компьютера

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* Если появится изображение, введите `Выбор ввода`, введите номер принтера, который соответствует, есть выбор на китайском и английском языках, после ввода нажмите Enter
* Выберите `3z`, чтобы убедиться, что в конфигурации `printer.cfg` есть конфигурация `z_tilt` / выберите `4z`, чтобы убедиться, что в конфигурации `printer.cfg` есть конфигурация `quad_gantry_level`, для независимого двойного Z выберите `3z`
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy2.webp').default} size="100%" align="left" />

* Если появится изображение, вам нужно ввести `Y` и нажать Enter
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/eddy4.webp').default} size="100%" align="left" />

* Если вы используете `sb2040-v3-PROV3`, введите y и нажмите Enter
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/img/eddy7.webp').default} size="100%" align="left" />

* Если появится изображение, значит установка прошла успешно
