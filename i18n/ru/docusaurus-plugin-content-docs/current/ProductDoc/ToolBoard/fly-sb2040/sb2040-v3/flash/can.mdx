# Компиляция CAN-программного обеспечения

## Начало компиляции

    <SSH />

    <MakeOperate/>

    <CAN />

    * Выберите **GPIO пины, установленные при старте микроконтроллера**, введите: **!gpio5,!gpio19,!gpio24**
    :::warning
        Пожалуйста, вводите в режиме английской раскладки клавиатуры `!gpio5,!gpio19,!gpio24`
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите клавишу `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы должны вернуться к командной строке
    * Введите следующую команду для начала компиляции (время может занять немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит:
        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
    то компиляция прошла успешно.

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30. Ниже приведен пример схемы подключения:
    * Убедитесь, что переключатели установлены на нужное положение
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/sb2040v3_utoc.webp').default} size="80%" align="center" />
    * Убедитесь, что между CAN H и CAN L измеряется сопротивление около `60 Ом` в отключенном состоянии. Если показывает `120 Ом`, то переключатель `120 Ом` должен быть включен, если около `40 Ом`, то переключатель должен быть выключен.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-v3/flash/img/120.webp').default} size="100%" align="center" />
