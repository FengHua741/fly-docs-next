# Фирменное ПО Katapult

## Описание Фирменного ПО Katapult

    <CANBOOT />

    :::tip

    * SB2040_PRO поддерживает CAN
    * Здесь предоставляются только два варианта CAN-программного обеспечения с скоростью 500K и 1M
    :::

## Вход в режим прошивки

    :::tip

    Выделенное кругом — это кнопка `BOOT`

    :::

            <Katapult />

            4. Прошивка Фирменного ПО

            :::tip

            Необходимо прошивать только одну из программ

            :::

            * Ниже приведена команда для прошивки программного обеспечения с CAN-скоростью 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_500K.uf2
                ```
                       
            * Ниже приведена команда для прошивки программного обеспечения с CAN-скоростью 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```


            5. Если появляется нижеследующее уведомление, то прошивка успешно завершена

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            После завершения прошивки необходимо полностью отключить питание материнской платы.

            :::

        </TabItem>
    </Tabs>
