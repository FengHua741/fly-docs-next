# Способы решения проблемы недостаточного напряжения драйвера

## Метод первый

    :::danger

    * Заменить источник питания
    * Добавить фильтр источника питания
    * Проверить работу двигателя
    * Подключить корпус двигателя к GND платы
    * В конфигурации драйвера экструдера добавить `driver_TPFD: 0`

    :::

## Метод второй
    * Подключиться по SSH к хост-компьютеру
    * Ввести следующую команду
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * После обновления klipper этот шаг может быть восстановлен
    :::
