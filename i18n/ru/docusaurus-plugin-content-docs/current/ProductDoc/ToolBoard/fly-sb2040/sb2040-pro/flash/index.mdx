# Инструкция по прошивке прошивки

    * SB2040 PRO может подключаться к ПК через соединение CAN
    * Для подключения CAN используется соединительный провод `XT30(2+2)-F`
    * Ниже приведен вид соединительного провода `XT30(2+2)-F`
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Важно знать
* Прошивку основной платы необходимо производить с помощью Katapult для прошивки Klipper
* Если основная плата уже прошита Katapult, то нет необходимости выполнять последний шаг прошивки прошивки Katapult, так как это просто мера на случай неудачи прошивки
* Можно быстро дважды нажать кнопку сброса, чтобы проверить, загорается ли светодиод на плате; если да, это означает, что плата находится в режиме Katapult
:::

## Как определить нужную прошивку

    * SB2040 PRO может подключаться к ПК следующим способом
    * [CAN-прошивка](./can.mdx "Щелкните для перехода")
        * Преимущество: если установлена прошивка `Katapult`, можно обновлять прошивку без физического контакта с платой, а также поддерживаются шины подключения
        * Недостаток: возможно возникновение ошибок типа срыва позиционирования при совместном использовании с `KlipperScreen`
        * Примечание: если требуется использовать CAN, убедитесь, что ПК поддерживает CAN, и необходимо использовать UTOC или плату с прошивкой моста


## Информация о прошивке Katapult

    * У SB2040 PRO по умолчанию предустановлена прошивка Katapult для подключения CAN со скоростью 1М
