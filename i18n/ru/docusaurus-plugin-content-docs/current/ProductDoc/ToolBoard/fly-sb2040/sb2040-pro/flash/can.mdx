    <CAN />

    * Выберите GPIO pins для установки при старте микроконтроллера, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы должны вернуться к командной строке
    * Введите следующую команду для начала компиляции (время займет немного):
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Если последний вывод содержит:
        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
    то компиляция прошла успешно.

## ЗАЖИГАНИЕ ФIRMWARE

    * Перед загрузкой firmware подключите Tool Board к UTOC или материнской плате с прошивкой bridge, используя XT30 кабель. Ниже приведена схема подключения.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/sb2040_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что перед питанием измерьте сопротивление CAN H и CAN L на萬итом тестере, оно должно быть около `60 Ом`. Если показывает `120 Ом`, подключите резистор `120 Ом`, если около `40 Ом`, отсоедините резистор.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/120.webp').default} size="50%" align="center" />
