    <CAN />

    * Выберите GPIO пины для установки при старте микроконтроллера, введите: gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/can.webp').default} size="100%" align="center" />
    * Нажмите `Q`, появится **Сохранить конфигурацию**, затем нажмите `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Теперь должна быть сохранена конфигурация и вы вернулись в командную строку
    * Введите `make -j4` для начала компиляции, это займет некоторое время
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * В конце вывода следующего содержимого компиляция успешно завершена:

        ```bash
        Linking out/klipper.elf
        Создание hex-файла out/klipper.bin
        ```

## Запись прошивки

    * Перед записью прошивки подключите плату инструментов к UTOC или материнской плате с прошивкой моста с помощью кабеля XT30. Ниже приведен пример соединения:
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040/flash/img/sb2040_utoc.webp').default} size="80%" align="left" />
    * Убедитесь, что сопротивление между CAN H и CAN L измеряется около `60 Ом` в отключенном состоянии. Если показывает `120 Ом`, подключите резистор на `120 Ом`. Если около `40 Ом`, отключите резистор.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/120.webp').default} size="50%" align="center" />
