# Продукт

    FLY SB2040 PRO — это плата инструментов, разработанная компанией Guangzhou Mellow Electronic Technology Co., Ltd. для принтерной головы VORON Stealthburber. Использование этой платы позволяет заменить сложные соединения между основной платой и печатной головой четырьмя проводами, оптимизируя расположение. SB2040 совместим с stealthburner и afterburner, интегрируя CAN-трансивер, порт USB, драйвер TMC2240, акселерометр ADXL345, три управляемых вентилятора и другие функции. Данная плата подходит для прошивок Klipper и RRF.

## Особенности продукта

    * Предусмотрена кнопка Boot, пользователь может войти в режим Boot через Type-C для обновления прошивки
    * Поддерживает прошивку Klipper
    * Датчик температуры: встроен чип MAX31865, совместимый с PT100
    * Встроен акселерометр ADXL345
    * Предусмотрен разъем RGB, встроенный драйвер 2240, поддерживает множество датчиков выравнивания, например, klicky, Voron Tap, PL08n и т. д.
    * Поддерживает входное напряжение 12-24В, ток до 15А, увеличенный разъем нагревательного элемента, номинальный ток 10А


## Параметры продукта


    * MCU: Raspberry Pi rp2040, Dual core ARM Cortex-M0+@133MHz
    * Разъемы: XT30 (2+2), поддерживают ток до 15А, пиковый ток 30А
    * Прошивка: Klipper
    * Входное напряжение: DC12V-24V
    * Логическое напряжение: DC3.3V
    * Встроенные датчики: ADXL345 и опциональный MAX31865
    * Разъем нагревательного элемента: HEAT0
    * Расширение: три управляемых вентилятора, RGB, три концевика, один нагревательный элемент, один термодатчик
    * Разъемы вентиляторов: три управляемых вентилятора
    * Драйвер двигателя: встроенный TMC2240
