# Image de Raspberry Pi

:::warning

Astuces sur la carte SD :

* La capacité doit être d'au moins `16GB` et au plus `128GB`
* La vitesse de transmission de la carte TF doit être de classe `10` ou supérieure, il est recommandé d'utiliser des cartes TF de marque SanDisk par exemple

:::

:::danger À savoir

* Ne mettez pas à jour le système ou Klipper en raison de problèmes serveur
* L'image du système est compatible avec les Raspberry Pi 3B/3B+/4B/CM4/5B
* Cette image est basée sur `bookworm` pour installer divers services
* Cette image installe et active par défaut l'écran TFT. Si vous n'avez pas besoin d'utiliser l'écran TFT, vous devez modifier manuellement `/boot/config.txt`

:::

:::info Important

* Veuillez noter que le système téléchargé nécessite d'être décompressé, pas simplement renommé ou gravé directement !!!!!
* Sans décompression avant la gravure, le système ne démarrera pas !!!!!

:::
