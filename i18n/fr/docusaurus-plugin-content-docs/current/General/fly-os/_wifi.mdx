:::danger À savoir

* Le logiciel supérieur sur FLY doit utiliser le module M2WE ou 2,4G WIFI, sinon utiliser le WiFi
* Veuillez noter que les réseaux WiFi avec des noms en chinois et ceux avec double bande 2,4G et 5G ne sont pas pris en charge
* Si vous utilisez un mini pad ou lite2, ignorez cet avertissement

---

## Connexion au logiciel supérieur

* Si la configuration du WiFi dans le dossier BOOT échoue, veuillez suivre cette configuration pour connecter le WiFi, veuillez noter qu'une connexion série est nécessaire pour le logiciel supérieur
* Connectez-vous d'abord à SSH via la connexion série [ici](/docs/DebugDoc/BasicTutorial/index.mdx "Cliquez pour sauter")
* Exécutez la commande ci-dessous, une invite de saisie de mot de passe peut apparaître, tapez le mot de passe `mellow` puis appuyez sur Entrée, notez que le mot de passe n'est pas visible

```cfg
sudo nmtui
```

<ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

* Après avoir appuyé sur Entrée après avoir sélectionné `Activation d'une connexion`

<ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

* Entrez dans l'interface suivante, utilisez les touches fléchées `↑ ↓` du clavier pour sélectionner le WiFi auquel vous voulez vous connecter, appuyez sur Entrée, entrez le mot de passe et attendez que la connexion soit réussie

<ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.webp').default} size="80%" align="left" />

<ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.webp').default} size="80%" align="left" />

* Une fois la connexion réussie, comme illustré dans l'image suivante

<ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.webp').default} size="80%" align="left" />

* Appuyez sur la touche Echap pour sortir, revenez à l'interface de ligne de commande, entrez `ip a`, l'adresse IP `192.168.2.126` indiquée dans l'image suivante est l'adresse IP du logiciel supérieur
