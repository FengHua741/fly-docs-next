    ## CONNECTION À UN RÉSEAU WIFI DANS LE SYSTÈME FLYOS-ARMBIAN

    1. Connectez-vous d'abord à la station de travail via un port série :<Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-connexion-port-série-à-la-station-de-travail-ssh">CONNEXION PORT-SÉRIE À LA STATION DE TRAVAIL SSH</Button>

    2. Tapez la commande `sudo nmtui`, vous pourriez être invité à entrer un mot de passe, entrez le mot de passe `mellow`, puis appuyez sur la touche `Entrée`. Cela affichera l'interface suivante (remarque : le mot de passe entré ne sera pas visible)  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi2.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi3.webp').default} size="60%" align="center" />

    3. Sélectionnez l'option `Activer une connexion`, appuyez sur la touche `Entrée` pour accéder au menu suivant
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi4.webp').default} size="60%" align="center" />

    4. Si votre routeur WiFi fonctionne correctement, vous verrez la liste suivante des réseaux WiFi. Le caractère `*` après le nom du réseau indique la force du signal
    :::tip
    - **Les réseaux WiFi avec des noms en chinois ne sont PAS pris en charge !!!**
    - Les réseaux WiFi 2,4 GHz sont supportés, mais la compatibilité avec les réseaux WiFi 5G dépend de la compatibilité de votre module WiFi avec les réseaux 5G
    - Le module `M2WE` prend en charge les réseaux 5G
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi5.webp').default} size="60%" align="center" />

    5. Sélectionnez le réseau WiFi auquel vous souhaitez vous connecter, appuyez sur la touche `Entrée` pour accéder au menu suivant
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi6.webp').default} size="60%" align="center" />

    6. Une interface similaire à celle ci-dessous apparaîtra, entrez correctement votre mot de passe WiFi, puis appuyez sur la touche `Entrée` pour vous connecter au WiFi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi7.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi8.webp').default} size="60%" align="center" />

    7. Si le mot de passe est correct, un symbole `*` apparaîtra devant le réseau WiFi correspondant, indiquant que la connexion a réussi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi9.webp').default} size="60%" align="center" />

    8. Appuyez deux fois sur la touche `Esc` pour revenir à l'interface de ligne de commande
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi10.webp').default} size="60%" align="center" />
    
    9. Dans SSH, tapez `ip a | grep inet`, puis appuyez sur la touche `Entrée`. Recherchez dans les résultats l'adresse IP qui appartient à la même plage que votre routeur pour y accéder
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi11.webp').default} size="90%" align="center" />

    10. Ouvrez un navigateur web, saisissez l'adresse IP de la station de travail que vous avez trouvée dans la barre de recherche, puis appuyez sur la touche `Entrée` pour accéder à la console WEB fluidd/mainsail
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi12.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi13.webp').default} size="90%" align="center" />

    11. Dans la barre de recherche du navigateur, saisissez `IP:9999`, puis appuyez sur la touche `Entrée` pour accéder à l'interface `fly tools`. À travers cette interface, vous pouvez activer diverses fonctions de la station de travail
    :::tip
    - L'interface `fly tools` n'est disponible que dans le système `FLYOS-ARMBIAN` !!! Elle n'est pas présente dans `FLYOS-FAST` !!!
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi14.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi15.webp').default} size="90%" align="center" />
