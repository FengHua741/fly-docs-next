    ## Connexion à un réseau WiFi avec le système FLYOS-Armbian

    1. Veuillez d'abord vous connecter via un port série au PC principal :<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-connexion-port-série-au-pc-ssh">Connexion port série au PC principal ssh</Button>

    2. Entrez la commande `sudo nmtui`, une invite pour entrer le mot de passe peut apparaître, entrez le mot de passe `mellow`, puis appuyez sur la touche `Entrée`. Une interface comme celle illustrée ci-dessous s'affichera (remarque : les caractères saisis ne sont pas visibles)  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi2.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi3.webp').default} size="60%" align="center" />

    3. Sélectionnez l'option `Activer une connexion`, puis appuyez sur la touche `Entrée` pour accéder au menu suivant
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi4.webp').default} size="60%" align="center" />

    4. Si votre routeur WiFi fonctionne correctement, vous verrez alors la liste des réseaux WiFi suivante. Le caractère `*` après le nom du WiFi indique la force du signal
    :::tip
    - **Les réseaux WiFi avec un nom en chinois ne sont pas pris en charge !!!**
    - Les réseaux WiFi 2,4G sont pris en charge, mais la prise en charge du WiFi 5G dépend de la compatibilité de votre module WiFi avec le WiFi 5G
    - Le `M2WE` prend en charge le WiFi 5G
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi5.webp').default} size="60%" align="center" />

    5. Sélectionnez le WiFi auquel vous souhaitez vous connecter, puis appuyez sur la touche `Entrée` pour accéder au menu suivant
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi6.webp').default} size="60%" align="center" />

    6. Une interface similaire à celle ci-dessous apparaîtra, entrez correctement le mot de passe de votre WiFi, puis appuyez sur la touche `Entrée` pour vous connecter au WiFi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi7.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi8.webp').default} size="60%" align="center" />

    7. Si le mot de passe est correct, un symbole `*` apparaîtra devant le nom du WiFi correspondant, ce qui indique que la connexion WiFi a réussi
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi9.webp').default} size="60%" align="center" />

    8. Ensuite, appuyez deux fois sur la touche `Échap` pour revenir à l'interface de ligne de commande
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi10.webp').default} size="60%" align="center" />
    
    9. Dans SSH, entrez la commande `ip a | grep inet`, puis appuyez sur la touche `Entrée`. Recherchez dans le contenu retourné une IP qui appartient à la même plage que celle de votre routeur pour y accéder
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi11.webp').default} size="90%" align="center" />

    10. Ouvrez un navigateur web, dans la barre de recherche, entrez l'IP du PC principal que vous avez trouvé précédemment, puis appuyez sur la touche `Entrée` pour accéder au tableau de bord WEB de fluidd/mainsail
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi12.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi13.webp').default} size="90%" align="center" />

    11. Dans la barre de recherche du navigateur, entrez `IP:9999`, puis appuyez sur la touche `Entrée` pour accéder à l'interface `fly tools`. À travers cette interface, vous pouvez activer diverses fonctions du PC principal
    :::tip
    - L'interface `fly tools` n'est disponible que pour le système `FLYOS-Armbian`. Il n'est pas présent dans `FLYOS-FAST` !!!
    :::
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi14.webp').default} size="60%" align="center" />  
    <ImageView image={require('@site/docs/General/fly-os/img/flyos_wifi15.webp').default} size="90%" align="center" />
