:::danger Avertissement

* L'ordinateur de bord FLY nécessite l'utilisation du module M2WE ou de la connectivité WiFi en 2,4 GHz, sinon la connexion WiFi ne fonctionnera pas.
* Veuillez noter que les réseaux WiFi avec un nom en chinois et les réseaux WiFi dual-band (2,4 GHz et 5 GHz) ne sont pas pris en charge.
* Si vous utilisez un mini pad ou un lite2, ignorez cet avertissement.

---

## Connexion à l'ordinateur de bord

* Si la configuration WiFi échoue dans le mode BOOT, vous pouvez suivre cette procédure pour configurer la connexion WiFi. Veuillez noter qu'une connexion série est nécessaire pour l'ordinateur de bord.
* Connectez-vous d'abord à SSH via une connexion série en cliquant [ici](/docs/DebugDoc/BasicTutorial/index.mdx "cliquez pour accéder").
* Exécutez la commande suivante, il se peut que vous soyez invité à entrer un mot de passe ; entrez `mellow` puis appuyez sur Entrée. Veuillez noter que le mot de passe n'est pas visible.

```cfg
sudo nmtui
```

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui1.webp').default} size="80%" align="left" />

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui2.webp').default} size="80%" align="left" />

* Sélectionnez `Activer une connexion` puis appuyez sur Entrée.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui3.webp').default} size="80%" align="left" />

* Une fois dans l'écran suivant, utilisez les touches fléchées du clavier (`↑ ↓`) pour sélectionner le réseau WiFi auquel vous souhaitez vous connecter, puis appuyez sur Entrée. Entrez ensuite le mot de passe et attendez la connexion réussie.

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui4.webp').default} size="80%" align="left" />


        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui5.webp').default} size="80%" align="left" />

* Une fois la connexion réussie, vous verrez l'image suivante :

        <ImageView image={require('@site/docs/General/fly-os/img/nmtui/nmtui6.webp').default} size="80%" align="left" />

* Appuyez sur la touche Echap (ESC) pour sortir, revenez à l'interface de ligne de commande, puis tapez `ip a`. L'adresse IP de l'ordinateur de bord, `192.168.2.126`, sera indiquée dans l'image suivante.
