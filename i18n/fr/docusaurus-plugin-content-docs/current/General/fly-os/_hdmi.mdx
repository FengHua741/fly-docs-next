## Utilisation de l'interface supérieure FLY

:::tip
Ouvrez votre IP:9999 dans le navigateur pour accéder à FLY-Tools
:::

:::tip
L'interface HDMI embarquée sur l'interface supérieure FLY est un Micro HDMI, veuillez faire attention lors de l'achat du câble HDMI !!!
:::

:::tip
L'interface supérieure FLY prend en charge FPC One-Line pour `FLY-PI-V2`, `FLY-C8` et `FLY-PRO-X10`
:::

:::tip
L'utilisation d'un module de conversion FPC vers HDMI est compatible avec des interfaces supérieures telles que Raspberry Pi
:::
