## Installation du système

1. Ouvrir Rufus, choisir le périphérique, la capacité affichée du périphérique choisi doit être identique à celle de la carte de démarrage insérée dans l'ordinateur.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="30%" align="center" />

2. Sélectionner le type de démarrage, par défaut sélectionnez `fichier d'image`.
3. Sélectionner le fichier d'image, cliquez sur le bouton `Sélectionner` à droite du type de démarrage, choisissez le fichier `.img` décompressé.
4. Après avoir sélectionné le fichier d'image, le type de démarrage changera pour afficher le nom du fichier d'image.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="30%" align="center" />

5. Cliquez sur le bouton `Démarrer`, si une fenêtre de confirmation s'affiche pour effacer les données du disque, cliquez sur `OK`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />

6. Une fois l'écriture terminée, l'ordinateur émettra un signal sonore et la barre de progression affichera `Prêt`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="30%" align="center" />

7. Ne retirez pas immédiatement la carte de démarrage après l'écriture.
8. Ouvrez l'explorateur de fichiers, vérifiez qu'il existe un lecteur amovible ou une clé USB nommée `FlyOS-Conf`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />

9. Ouvrez `FlyOS-Conf`, si les deux fichiers suivants sont présents, cela signifie que l'installation a réussi.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

10. Installez la carte système sur la carte mère.
