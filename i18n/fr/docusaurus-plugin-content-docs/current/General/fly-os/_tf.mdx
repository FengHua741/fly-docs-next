# Installation du système sur la carte TF

### Préparation

* Installez d'abord le logiciel `Win32DiskImager`
* Décompressez le système, les fichiers décompressés ont une extension `.img`
* Il est déconseillé d'utiliser `balenaetcher` pour l'installation du système
* Insérez la carte TF dans un lecteur et branchez le lecteur à une prise USB de l'ordinateur
* Si la carte TF contient des données, veuillez effectuer une sauvegarde. Après l'installation du système, il n'est pas recommandé de retirer la carte TF sauf si nécessaire pour la reprogrammer vers M2WE puis la retirer

### Formattage de la carte mémoire

* Utilisez le logiciel `TFFormatter.exe` pour formater la carte TF
* Sélectionnez le disque que vous souhaitez formater

<ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

* Ou utilisez la gestion des disques intégrée à Windows pour supprimer les partitions, puis récréez-les après.
