# Installation du système sur une carte TF

### Préparation

    * Installez d'abord le logiciel `SDFormatter` et `Rufus`
    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">Outils de formatage</Button>
    * Décompressez le système, après décompression le fichier aura une extension `.img`
    * Il est déconseillé d'utiliser `balenaetcher` ou `Raspberry Pi` pour l'installation du système sur la carte TF
    * Insérez la carte TF dans un lecteur de cartes, puis branchez le lecteur à une interface USB du PC
    * Si la carte TF contient des données, veillez à faire une sauvegarde. Après installation du système, il n'est pas recommandé de retirer la carte TF, sauf si elle doit être réinstallée sur M2WE

### Formattage de la carte mémoire

    * Utilisez le logiciel `SDFormatter.exe` pour formater la carte TF
    * Sélectionnez le disque à formatter

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * Vous pouvez également utiliser la gestion des disques intégrée à Windows pour supprimer les partitions, puis recréer une partition après suppression.
