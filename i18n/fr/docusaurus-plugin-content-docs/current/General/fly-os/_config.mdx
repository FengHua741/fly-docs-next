## Important提示
    :::astuce À lire absolument

    - Les paramètres configurés dans la section du fichier de configuration ne prendront effet qu'à la première démarrage après le flashage.
    - En raison de la complexité du processus de configuration, il est fortement recommandé de configurer via le navigateur en se connectant à la machine via un réseau après le flashage.
    - Ce tutoriel ne fournit que les options modifiables.

    :::

## Comment modifier le fichier de configuration

### Modification du fichier de configuration après le flashage

- Les paramètres configurés uniquement lors du premier démarrage doivent être modifiés en ouvrant le fichier `config.txt` du disque amovible `FlyOS-Conf` avec un éditeur de texte professionnel après le flashage.

### Modification en ligne des paramètres du système

- Si vous modifiez les paramètres généraux via la page de configuration web du système et redémarrez le système, les modifications prendront effet.
    - Ouvrir l'adresse IP du périphérique `ip+:9999` dans le navigateur, par exemple : `192.168.1.2:9999`
    - La modification du nom d'hôte est prise en charge
    - La modification du fuseau horaire est prise en charge
    - L'activation de l'extension 40PIN du PC supérieur est prise en charge (remarque : l'utilisation du PWM rendra impossible la connexion série au PC supérieur, et l'activation de l'écran pourrait rendre impossible l'utilisation de l'accéléromètre sur le PC supérieur)
    - La modification du modèle de la carte mère est prise en charge
    - La modification de Klipper via le web est prise en charge, y compris Fluid et Mainsail
    - L'activation ou la désactivation de KlipperScreen est prise en charge
    - Le fonctionnement forcé de Klipper sur un cœur de processeur spécifique est pris en charge, permettant de faire tourner Klipper sur le cœur choisi
