## Recherche d'ID USB
1. Ouvrir le navigateur, taper l'adresse IP de la station supérieure dans la barre d'URL, par exemple, si mon adresse IP de la station supérieure est `192.168.101.179`, taper et appuyer sur Entrée directement.
  <ImageView image={require('@site/docs/General/get-id/img/wed/ip.webp').default} size="80%" align="center" /> 

2. Ouvrir l'interface WEB de la station supérieure, trouver dans le menu latéral de configuration à gauche :
  <table>
    <tr>
      <td>
        - Pour fluidd : `{…}`, cliquer pour entrer, puis trouver `printer.cfg`
        <ImageView image={require('@site/docs/General/get-id/img/fluidd-cfg.webp').default} size="100%" align="center" />
      </td>
      <td>
        - Pour mainsail : `machine`, cliquer pour entrer, puis trouver `printer.cfg`
        <ImageView image={require('@site/docs/General/get-id/img/mainsail-cfg.webp').default} size="100%" align="center" />
      </td>
    </tr>
  </table>

  3. Cliquez sur `printer.cfg`, puis cliquez sur le bouton `DEVICES` en haut à droite
    <table>
      <tr>
        <td>
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_devices.webp').default} size="100%" align="center" />
        </td>
        <td>
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_devices.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>

  4. Cliquez sur `SERIAL`, puis rafraîchir
    <table>
      <tr>
        <td>
          - Rafraîchissement de fluidd
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_refresh.webp').default} size="100%" align="center" />
        </td>
        <td>
          - Rafraîchissement de mainsail
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_refresh.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>

  5. Copier l'ID, cliquer sur la flèche pointée pour copier
    <table>
      <tr>
        <td>
          - Pour fluidd, vérifier la présence de `ttyAMC` et s'assurer que Path by ID existe, c'est cet ID USB
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_id.webp').default} size="100%" align="center" />
        </td>
        <td>
          - Pour mainsail, vérifier la présence de `ttyAMC` et s'assurer que Path by ID existe, c'est cet ID USB
          <ImageView image={require('@site/docs/General/get-id/img/wed/mainsail_id.webp').default} size="100%" align="center" />
        </td>
      </tr>
    </table>

## Remplir l'ID USB

    1. Fermer `DEVICES` et remplir l'ID

    :::danger À savoir
      * Modifier le fichier de configuration comme suit :  
      ```
      [mcu] 
      serial: /tmp/klipper_host_mcu
      ```  
      en
      ```
      [mcu host] 
      serial: /tmp/klipper_host_mcu
      ```
      * Ajouter :
      ```
      [mcu]
      serial: <remplacer ici par l'ID que vous avez obtenu>
      ```
      :::
    <table>
      <tr>
        <td>
          - Trouver [MCU] dans fluidd
          <ImageView image={require('@site/docs/General/get-id/img/wed/fluidd_cfg.webp').default} size="100%" align="center" />

    2. Remplir l'ID dans la configuration
    :::tip

    * Après avoir rempli l'ID, cliquer sur `SAVE & RESTART` en haut à droite
    :::
    <table>
      <tr>
        <td>

    3. Si Klipper affiche `ADC out of range`, il s'agit d'un phénomène normal. Connectez le lit chauffant et les thermistances au contrôleur mère, configurez correctement les broches des thermistances du jet et du lit, puis enregistrez et redémarrez.

:::danger
Attention : Tous les ID mentionnés dans le document sont des exemples, chaque carte mère a un ID différent, veuillez utiliser l'ID réellement obtenu.
:::
