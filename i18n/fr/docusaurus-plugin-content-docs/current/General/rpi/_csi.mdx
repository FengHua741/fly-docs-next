## Utilisation du CSI

    * Après la gravure de l'image, allez dans /boot/config.txt et trouvez `camera_auto_detect=1`, puis modifiez-le en `camera_auto_detect=0` et ajoutez les configurations selon le modèle de la caméra.
    * Ci-dessous se trouve une configuration de référence

    ```bash
    dtoverlay=imx219,cam1
    ```
