## Raspberry Pi 5 avec MIPI

:::tip
FLY_MIPI est un module de conversion spécialement conçu pour le Raspberry Pi 5, permettant d'utiliser directement des périphériques DSI et CSI via une adaptation.
:::

### Connexion du Raspberry Pi 5 avec MIPI

    :::danger À savoir

    * Les interfaces DSI et CSI du Raspberry Pi 5 sont maintenant appelées MIPI
    * Les interfaces MIPI peuvent être utilisées directement avec des caméras CSI ou des écrans DSI via une configuration
    * Veuillez prêter attention à la direction des broches

    :::

    * Le Raspberry Pi 5 dispose maintenant de deux interfaces MIPI, si vous regardez de face, vous pouvez voir qu'elles sont marquées 1 et 0

## Connexion de l'écran DSI

    * Si vous utilisez l'interface 1, veuillez configurer l'écran selon la configuration suivante

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi1
    ```

    * Si vous utilisez l'interface 0, veuillez configurer l'écran selon la configuration suivante

    ```bash
    dtoverlay=vc4-kms-dsi-7inch,dsi0
    ```

## Connexion de la caméra CSI

    <ImageView image={require('@site/docs/General/rpi/img/csi1.webp').default} size="100%" align="left" />
    * Veuillez noter que le modèle de caméra doit être configuré en fonction de celui que vous avez acheté, ce tutoriel fait référence à l'utilisation de imx219

    * Si vous utilisez l'interface MIPI de gauche, veuillez configurer la caméra selon la configuration suivante
    ```bash
    dtoverlay=imx219,cam1
    ```

    * Si vous utilisez l'interface MIPI de droite, veuillez configurer la caméra selon la configuration suivante
    ```bash
    dtoverlay=imx219,cam0
    ```
