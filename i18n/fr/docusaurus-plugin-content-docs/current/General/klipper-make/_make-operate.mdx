### Instructions d'exploitation pour la compilation et la configuration du firmware Klipper

:::tip

Assurez-vous que le mode d'entrée du clavier est en mode demi-largeur, c'est-à-dire en mode anglais

:::

1. La touche `↑` du clavier, la touche `↓`, est utilisée pour déplacer le curseur vers le haut ou le bas pour sélectionner les éléments de menu
2. La touche `Entrée` ou la touche `Espace`, est utilisée pour cocher les éléments de menu ou entrer dans les sous-menus
3. La touche `ESC`, est utilisée pour revenir au menu précédent
4. La touche `Q`, est utilisée pour quitter la page de configuration du firmware Klipper
5. La touche `Y`, lorsque vous quittez la page de configuration du firmware Klipper et qu'une提示信息被截断了，完整的应该是“La touche `Y`, lorsque vous quittez la page de configuration du firmware Klipper et qu'une提示信息被截断了，完整的应该是“La touche `Y`, lorsque vous quittez la page de configuration du firmware Klipper et qu'une demande de confirmation s'affiche, appuyez sur `Y` pour sauvegarder le fichier de configuration.”
