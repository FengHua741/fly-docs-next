* Un ordinateur hôte avec le service « klipper » installé est nécessaire. <Button variant="contained" disableElevation href="/docs/DebugDoc/BasicTutorial/">Connexion SSH via le réseau</Button>
* Et le dispositif doit être bien connecté à l'ordinateur hôte.
* Attention : Veuillez utiliser un ordinateur hôte ordinaire, tels que Raspberry Pi, FLY Pi, etc. Si c'est un ordinateur hôte modifié (par exemple, WiFi Stick, Redmi, etc.), en raison de nombreux problèmes, nous ne pouvons pas fournir de support technique.

:::warning Consignes à prendre en compte

* Veuillez ne pas utiliser l'utilisateur « ROOT ».

:::

su fly
