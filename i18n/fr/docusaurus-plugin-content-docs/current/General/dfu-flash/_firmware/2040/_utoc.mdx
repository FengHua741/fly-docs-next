Voici comment compiler le firmware :
* Connectez-vous à SSH puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` est utilisé pour supprimer les données et le firmware de compilation précédents. `make menuconfig` est utilisé pour compiler le firmware. Après l'exécution, il devrait apparaître l'interface ci-dessous :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Entrez dans le menu **Microcontrôleur Architecture** puis sélectionnez **Raspberry Pi RP2040** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, choisissez : **16KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, choisissez : **Pont USB vers bus CAN**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/utoc.webp').default} size="100%" align="Center" />
