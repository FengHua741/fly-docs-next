Voici comment compiler le firmware :
* Connectez-vous via SSH puis tapez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Parmi les commandes `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`, il s'agit de supprimer les données de compilation précédentes et le firmware. `make menuconfig` est utilisé pour compiler le firmware, après quoi l'interface suivante devrait apparaître

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Accédez au menu **Microcontrôleur Architecture** puis sélectionnez **Raspberry Pi RP2040** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Décalage du bootloader**, choisissez : **Aucun bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, choisissez : **UART0 sur GPIO0/GPIO1**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/rs232.webp').default} size="100%" align="Center" />
