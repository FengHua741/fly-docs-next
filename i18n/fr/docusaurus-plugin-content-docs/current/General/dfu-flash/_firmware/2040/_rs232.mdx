Voici comment compiler le firmware :
* Connectez-vous via SSH, puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Ici, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware, tandis que `make menuconfig` compile le firmware. Après exécution, vous devriez obtenir l'interface ci-dessous :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration supplémentaires de bas niveau** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Accédez au menu **Micro-contrôleur Architecture** puis sélectionnez **Raspberry Pi RP2040/RP235x** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Bootloader offset**, puis choisissez : **Aucun bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/0kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, puis choisissez : **UART0 sur GPIO0/GPIO1**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/rs232.webp').default} size="100%" align="Center" />
