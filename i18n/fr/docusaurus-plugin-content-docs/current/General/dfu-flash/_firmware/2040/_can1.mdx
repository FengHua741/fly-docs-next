Voici comment compiler le firmware :
* Connectez-vous via SSH, puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Où `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` est la suppression des données de compilation précédentes et du firmware, et `make menuconfig` est la compilation du firmware ; après exécution, l'interface ci-dessous devrait apparaître :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Architecture du microcontrôleur** puis sélectionnez **Raspberry Pi RP2040/RP235x** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, puis choisissez : **16KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, puis choisissez : **BUS CAN**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/can.webp').default} size="100%" align="Center" />

* Sélectionnez `(4) Numéro de broche RX CAN`, effacez `4` et tapez `1`
* Sélectionnez `(5) Numéro de broche TX CAN`, effacez `5` et tapez `0`

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/can1.webp').default} size="100%" align="Center" />
