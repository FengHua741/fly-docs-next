Voici comment compiler le firmware :
* Connectez-vous via SSH, puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Ici, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware, tandis que `make menuconfig` compile le firmware. Une fois exécuté, cela devrait afficher l'interface suivante :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Accédez au menu **Architecture du microcontrôleur**, puis sélectionnez **Raspberry Pi RP2040/RP235x** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, choisissez : **Pas de bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/0kb.webp').default} size="100%" align="Center" />

* Pour **Interface de communication**, choisissez : **USBSERIAL**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb.webp').default} size="100%" align="Center" />
