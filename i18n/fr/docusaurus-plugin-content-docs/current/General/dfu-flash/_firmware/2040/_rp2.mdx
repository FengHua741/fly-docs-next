* Veuillez noter que le panneau d'outils est actuellement en mode de gravure du firmware et ne peut graver que des firmwares avec l'extension uf2.
* À l'aide de la commande ci-dessous, gravez le firmware au panneau d'outils.

        ```bash
        cd && cd ~/klipper/
        make flash FLASH_DEVICE=2e8a:0003
        ```
        :::tip
           Si c'est la première fois que vous gravez après avoir configuré le système à partir de l'ordinateur principal, vous pourriez voir un message similaire à celui entouré en rouge dans l'image ci-dessous - une partie des messages sera en rouge, ce qui est normal et n'est pas une erreur.
        :::
