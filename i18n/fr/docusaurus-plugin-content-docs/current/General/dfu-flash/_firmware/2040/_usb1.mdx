Voici la description de la compilation du firmware :
* Après avoir établi une connexion SSH, entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Où `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` est utilisé pour supprimer les données et le firmware de la compilation précédente et `make menuconfig` sert à compiler le firmware. Une fois exécuté, cela devrait afficher l'interface ci-dessous :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration supplémentaires de niveau bas** puis appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Architecture du microcontrôleur** puis sélectionnez **Raspberry Pi RP2040** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, choisissez : **16KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/16kb.webp').default} size="100%" align="Center" />

* Pour **Interface de communication**, choisissez : **USBSERIAL**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb1.webp').default} size="100%" align="Center" />
