Voici la procédure pour compiler le firmware :
* Connectez-vous via SSH, puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Ici, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware, tandis que `make menuconfig` est utilisé pour compiler le firmware. Une fois exécuté, une interface similaire à celle ci-dessous doit apparaître :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Architecture du microcontrôleur**, puis sélectionnez **Raspberry Pi RP2040** et appuyez sur Entrée.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/2040.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, puis choisissez : **Aucun bootloader**.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/0kb.webp').default} size="100%" align="Center" />

* Pour **Interface de communication**, choisissez : **USBSERIAL**.

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/2040/img/usb1.webp').default} size="100%" align="Center" />
