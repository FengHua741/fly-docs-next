Voici les étapes pour compiler le firmware :
* Connectez-vous à SSH, puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Parmi les commandes `rm -rf ~/klipper/.config && rm -rf ~/klipper/out`, il s'agit de supprimer les données de compilation précédentes et le firmware. `make menuconfig` est la commande de compilation du firmware. Après son exécution, l'interface suivante devrait apparaître :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration supplémentaires de bas niveau** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Accédez au menu **Architecture du microcontrôleur**, puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Accédez au menu **Modèle de processeur**, sélectionnez **STM32F405** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/405.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, choisissez : **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/32kb.webp').default} size="100%" align="Center" />

* Pour **Interface de communication**, choisissez : **USB (sur PA11/PA12)**

   <ImageView image={require('@site/docs/General/dfu-flash/_firmware/405/img/usb.webp').default} size="100%" align="Center" />
