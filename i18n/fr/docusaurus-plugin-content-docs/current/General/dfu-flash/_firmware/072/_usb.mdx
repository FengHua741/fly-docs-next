Voici comment compiler le firmware :
* Connectez-vous à SSH puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Où `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` est la suppression des données de compilation précédentes et du firmware `make menuconfig` est la compilation du firmware, après l'exécution, une interface similaire à celle ci-dessous devrait apparaître

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Enable extra low-level configuration options** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Accédez au menu **Micro-controller Architecture** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Accédez au menu **Processor model**, sélectionnez **STM32F072** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.webp').default} size="100%" align="Center" />

* Sélectionnez **Bootloader offset**, choisissez : **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* **Communication interface**, choisissez : **USB (on PA11/PA12)** et appuyez sur Entrée
    
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb.webp').default} size="100%" align="Center" />
* **Veuillez vérifier avec l'image ci-dessous pour vous assurer qu'elle correspond, avant de passer à l'étape suivante**
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/usb2.webp').default} size="100%" align="Center" />
