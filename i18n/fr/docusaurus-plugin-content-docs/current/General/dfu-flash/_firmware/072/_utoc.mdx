Voici comment compiler le firmware :
* Connectez-vous à SSH puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Où `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` est la suppression des données de compilation précédentes et du firmware, et `make menuconfig` est la compilation du firmware. Après l'exécution, il devrait apparaître l'interface ci-dessous

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Microcontrôleur Architecture** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Allez dans le menu **Modèle de processeur**, sélectionnez **STM32F072** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, choisissez : **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, choisissez : **Pont USB vers bus CAN (USB sur PA11/PA12)** appuyez sur Entrée
* Après avoir choisi **Pont USB vers bus CAN (USB sur PA11/PA12)**, ce paramètre par défaut ne doit pas être modifié **Interface CAN bus (CAN bus (sur PB8/PB9)) -- >**
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc.webp').default} size="100%" align="Center" />   

* **Veuillez vérifier avec l'image ci-dessous pour vous assurer qu'elle correspond avant de compiler**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc2.webp').default} size="100%" align="Center" />
