Voici comment compiler le firmware :

* Connectez-vous à SSH et entrez les commandes suivantes :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Où `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données et le firmware précédemment compilés, et `make menuconfig` compile le firmware. Après l'exécution, l'interface suivante devrait apparaître :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Enable extra low-level configuration options** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Micro-controller Architecture** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Allez dans le menu **Processor model**, sélectionnez **STM32F072** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/072.webp').default} size="100%" align="Center" />

* Sélectionnez **Bootloader offset**, choisissez : **8KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/8kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Communication interface**, choisissez : **USB to CAN bus bridge (USB on PA11/PA12)** et appuyez sur Entrée
* Après avoir sélectionné **USB to CAN bus bridge (USB on PA11/PA12)**, ne modifiez pas le paramètre par défaut **CAN bus interface （CAN bus (on PB8/PB9)）--->** 
    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc.webp').default} size="100%" align="Center" />   

* Veuillez vérifier l'image ci-dessous avant de compiler :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/072/img/utoc2.webp').default} size="100%" align="Center" />
