Voici la procédure pour compiler le firmware :
* Connectez-vous via SSH puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Parmi les commandes ` rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware `make menuconfig` compile le firmware, après quoi l'interface suivante devrait apparaître

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Micro-contrôleur Architecture** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Allez dans le menu **Modèle de processeur**, sélectionnez **STM32H723** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.webp').default} size="100%" align="Center" />

* Sélectionnez **Décalage du bootloader**, choisissez : **128KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/128kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Référence d'horloge (cristal de 8 MHz)**, choisissez : **cristal de 25 MHz**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.webp').default} size="100%" align="Center" />

* Sélectionnez **Interface de communication**, choisissez : **Pont USB vers bus CAN (USB sur PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/utoc.webp').default} size="100%" align="Center" />
