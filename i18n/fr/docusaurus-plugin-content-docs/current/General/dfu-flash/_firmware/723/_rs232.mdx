Voici comment compiler le firmware :
* Connectez-vous à SSH puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Parmi lesquelles ` rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware `make menuconfig` compile le firmware, une fois exécutée, l'interface ci-dessous devrait apparaître

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Accédez au menu **Microcontrôleur Architecture** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Accédez au menu **Modèle de processeur**, sélectionnez **STM32H723** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/723.webp').default} size="100%" align="Center" />

* Sélectionnez **Offset du bootloader**, sélectionnez : **Aucun bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/0kb.webp').default} size="100%" align="Center" />

* Sélectionnez **Référence d'horloge (cristal de 8 MHz)** , sélectionnez : **Cristal de 25 MHz**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/25mhz.webp').default} size="100%" align="Center" />

* **Interface de communication**, sélectionnez : **Série (sur USART1 PA10/PA9)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/723/img/uart.webp').default} size="100%" align="Center" />
