Voici la procédure pour compiler le firmware :
* Connectez-vous à SSH puis entrez la commande suivante et appuyez sur Entrée :

    ```bash
    cd ~/klipper && rm -rf ~/klipper/.config && rm -rf ~/klipper/out && make menuconfig
    ```

* Parmi les commandes, `rm -rf ~/klipper/.config && rm -rf ~/klipper/out` supprime les données de compilation précédentes et le firmware, tandis que `make menuconfig` compile le firmware. Après l'exécution, l'interface suivante devrait apparaître :

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make1.webp').default} size="100%" align="Center" />

* Sélectionnez **Activer les options de configuration de bas niveau supplémentaires** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make2.webp').default} size="100%" align="Center" />

* Allez dans le menu **Architecture du microcontrôleur** puis sélectionnez **STMicroelectronics STM32** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/img/make3.webp').default} size="100%" align="Center" />

* Allez dans le menu **Modèle de processeur**, sélectionnez **STM32F407** et appuyez sur Entrée

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/407.webp').default} size="100%" align="Center" />

* Sélectionnez **Décalage du bootloader**, choisissez : **32KiB bootloader**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/32kb.webp').default} size="100%" align="Center" />

* Pour **Interface de communication**, choisissez : **USB (sur PA11/PA12)**

    <ImageView image={require('@site/docs/General/dfu-flash/_firmware/407/img/usb.webp').default} size="100%" align="Center" />
