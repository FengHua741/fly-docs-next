## Recherche d'appareils
* Assurez-vous que l'ordinateur principal est correctement connecté à Internet et que le câble de connexion du périphérique inférieur à l'ordinateur principal a la fonction de transmission de données.
* Après avoir établi une connexion SSH avec l'ordinateur principal, entrez `lsusb` puis appuyez sur Entrée. Si les informations entourées dans l'image ci-dessous apparaissent, vous pouvez passer à l'étape suivante.
    * Si aucune information n'est renvoyée, il s'agit d'un problème du système de l'ordinateur principal. Nous ne pouvons rien faire dans ce cas. Vous devez remplacer le système confirmé comme étant normal ou changer l'ordinateur principal.
    * Si l'indication "aucune commande lsusb" apparaît, vous pouvez exécuter cette commande pour installer :
        ```bash
        sudo apt-get install usbutils
        ```

* L'ID affiché dans le cadre rouge peut être utilisé comme référence mais n'est pas un ID "fixe", il est variable. Le suffixe `OpenMoko, Inc. Geschwister Schneider CAN adapter` a une valeur de jugement.
* Certains ordinateurs principaux peuvent ne pas afficher ou afficher incomplètement en raison de problèmes de système.
