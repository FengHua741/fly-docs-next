        * Utilisez la commande ci-dessous pour rechercher l'ID de votre carte mère, il devrait apparaître l'ID comme indiqué dans l'image ci-dessous (remarque : l'ID de l'image est donné à titre indicatif, chaque carte mère a un ID différent)

        ```bash
        ls /dev/serial/by-id/*
        ```

        :::tip Remarque
            L’ID de référence dans l’image suivante `/dev/serial/by-id/usb-Klipper_rp2040_12345-if00` est l'ID de la carte mère.
        :::

        <ImageView image={require('@site/docs/General/dfu-flash/img/ck.webp').default} size="80%" align="center" />

        * Pour mettre à jour le firmware USB, après avoir compilé le dernier firmware, entrez la commande ci-dessous pour mettre à jour, *remplacez `<votre-ID-de-carte-mère>` par l'ID trouvé à l'étape précédente*

        ```bash
        cd ~/klipper/ && make flash FLASH_DEVICE=<votre-ID-de-carte-mère>
        ```

## Méthode de correction en cas d'erreur de programmation
    * Appuyez sur le bouton pour entrer en DFU, un lecteur flash apparaîtra sur l'ordinateur, déposez-y le firmware
    * Utilisez WINDOWS pour nettoyer le firmware puis rechargez-le
