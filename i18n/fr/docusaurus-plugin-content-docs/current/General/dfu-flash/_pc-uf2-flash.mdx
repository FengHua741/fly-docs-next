* Préparez un câble de données avec fonction de transmission de données
* Téléchargez le firmware sur l'ordinateur
* Branchez ou maintenez enfoncé le sautateur BOOT puis reliez-le au ordinateur, cela fera apparaître le disque `RPI-RP2`
    <ImageView image={require('@site/docs/General/dfu-flash/img/uf2.webp').default} size="100%" align="center" />
* Désbranchez ou relâchez le sautateur BOOT !!!
* Copiez le firmware à graver dans le dossier, veillez à ne copier que des fichiers avec l'extension `.uf2` !!! (Ne modifiez pas l'extension du fichier, cela n'a aucun effet)
* Référez-vous
    <ImageView image={require('@site/docs/General/dfu-flash/img/uf21.webp').default} size="100%" align="center" />
* Une fois le firmware copié, le disque `RPI-RP2` disparaîtra automatiquement, la gravure du firmware est terminée !
