### Notice sur les opérations de compilation et de configuration du firmware Klipper

    :::warning

    Assurez-vous que le mode d'entrée du clavier est en mode demi-largeur, c'est-à-dire en mode anglais

    :::

    1. Les touches `↑` et `↓` du clavier servent à déplacer le curseur vers le haut ou le bas pour sélectionner les éléments du menu
    2. La touche `Entrée` ou la barre d'espace `Espace` sert à cocher un élément du menu ou à accéder au sous-menu
    3. La touche `Échap` sert à revenir au menu précédent
    4. La touche `Q` sert à quitter la page de configuration du firmware Klipper
    5. Lors de la sortie de la page de configuration du firmware Klipper avec une invitation, appuyez sur la touche `Y` pour enregistrer le fichier de configuration

    :::warning

    Si le nombre d'options dans la page de configuration est faible, activez d'abord `[ ] Activer les options de configuration de niveau inférieur supplémentaires` pour afficher certaines options masquées

    :::
