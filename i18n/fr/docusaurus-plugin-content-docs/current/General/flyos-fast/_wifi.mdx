    :::astuce
    - Si la connexion WiFi configurée dans le disque `FlyOS-Conf` échoue, vous pouvez suivre ce tutoriel pour connecter le périphérique via un port série à l'ordinateur principal, puis connecter le WiFi.
    :::

    1. Veuillez d'abord utiliser un port série pour vous connecter à l'ordinateur principal : <Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh#flyos-fast-serial-connect-to-host-ssh">Connexion FLYOS-FAST au port série à l'ordinateur principal ssh</Button>

    2. Dans SSH, tapez la commande `nmtui`, appuyez sur la touche `Entrée`, une interface comme celle de l'image ci-dessous s'affichera
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi2.webp').default} size="40%" align="center" />

    3. Sélectionnez l'option `Activer une connexion`, appuyez sur la touche `Entrée`, puis entrez dans le menu suivant
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi3.webp').default} size="40%" align="center" />

    4. Si votre routeur WiFi fonctionne correctement, vous verrez la liste des réseaux Wi-Fi telle que celle de l'image ci-dessous. Le signe `*` après le nom du réseau indique la force du signal
    :::astuce
    - **Les réseaux Wi-Fi avec des noms en chinois ne sont pas pris en charge !!!**
    - Les réseaux Wi-Fi 2,4G sont supportés, mais la prise en charge du Wi-Fi 5G dépend de la compatibilité de votre module Wi-Fi avec le Wi-Fi 5G
    - `M2WE` prend en charge le Wi-Fi 5G
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi4.webp').default} size="40%" align="center" />

    5. Sélectionnez le réseau Wi-Fi auquel vous souhaitez vous connecter, appuyez sur la touche `Entrée`, puis entrez dans le menu suivant
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi5.webp').default} size="40%" align="center" />

    6. Une interface comme celle-ci apparaîtra ensuite. Entrez correctement votre mot de passe Wi-Fi, puis appuyez sur la touche `Entrée` pour vous connecter au WiFi
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi6.webp').default} size="40%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi7.webp').default} size="40%" align="center" />

    7. Si le mot de passe est correct, un symbole `*` apparaîtra devant le nom du réseau Wi-Fi correspondant, indiquant une connexion réussie
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi8.webp').default} size="40%" align="center" />

    8. Ensuite, appuyez deux fois sur la touche `Échap` pour retourner à l'interface de ligne de commande
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi9.webp').default} size="60%" align="center" />
    
    9. Dans SSH, tapez la commande `ip a | grep inet`, appuyez sur la touche `Entrée`. Dans les résultats retournés, trouvez l'adresse IP dans la même plage que votre routeur pour accéder
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi10.webp').default} size="90%" align="center" />

    10. Ouvrez un navigateur, entrez l'adresse IP de l'ordinateur principal que vous avez recherchée dans la barre de recherche, appuyez sur la touche `Entrée`, vous pourrez accéder au tableau de bord de contrôle WEB fluidd/mainsail
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi11.webp').default} size="60%" align="center" />
    
    :::astuce
    - Dans le système FLYOS-FAST, vous pouvez basculer en temps réel entre les tableaux de bord de contrôle fluidd/mainsail en cliquant sur l'icône située en haut à droite !
    :::
