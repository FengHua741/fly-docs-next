    ## Connexion au WiFi dans le système FLYOS-FAST
    
    1. Dans SSH, tapez la commande `nmtui` puis appuyez sur la touche `Entrée`. Une interface similaire à celle de l'image ci-dessous apparaîtra.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi1.webp').default} size="60%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi2.webp').default} size="40%" align="center" />

    2. Sélectionnez l'option `Activer une connexion`, puis appuyez sur la touche `Entrée` pour accéder au menu suivant.
    <NMTUI_OPERATE />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi3.webp').default} size="40%" align="center" />

    3. Si votre routeur WiFi fonctionne correctement, vous verrez la liste des réseaux Wi-Fi suivante. Le symbole `*` après le nom du réseau indique la force du signal.
    :::tip
    - **Les réseaux Wi-Fi avec un nom en chinois ne sont pas pris en charge !!!**
    - Les réseaux Wi-Fi 2,4 GHz sont pris en charge, mais la compatibilité avec les réseaux Wi-Fi 5 GHz dépend de la compatibilité de votre module Wi-Fi. Le modèle `M2WE` prend en charge le Wi-Fi 5 GHz.
    :::
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi4.webp').default} size="40%" align="center" />

    4. Sélectionnez le réseau Wi-Fi auquel vous souhaitez vous connecter, puis appuyez sur la touche `Entrée` pour accéder au menu suivant.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi5.webp').default} size="40%" align="center" />

    5. Un écran similaire apparaîtra, entrez correctement le mot de passe de votre réseau Wi-Fi, puis appuyez sur la touche `Entrée` pour vous connecter.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi6.webp').default} size="40%" align="center" />
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi7.webp').default} size="40%" align="center" />

    6. Si le mot de passe est correct, un symbole `*` apparaîtra devant le nom du réseau Wi-Fi, indiquant que la connexion a réussi.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi8.webp').default} size="40%" align="center" />

    7. Appuyez deux fois sur la touche `Échap` pour revenir à l'interface de ligne de commande.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi9.webp').default} size="60%" align="center" />
    
    8. Dans SSH, tapez la commande `ip a | grep inet`, puis appuyez sur la touche `Entrée`. Recherchez l'adresse IP dans le retour correspondant au même segment que celui de votre routeur pour y accéder.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi10.webp').default} size="90%" align="center" />

    9. Ouvrez un navigateur web, entrez l'adresse IP du contrôleur dans la barre de recherche, puis appuyez sur la touche `Entrée` pour accéder au panneau de contrôle WEB fluidd/mainsail.
    <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_wifi11.webp').default} size="60%" align="center" />
    
    :::tip
    - Dans le système FLYOS-FAST, vous pouvez basculer en temps réel entre les interfaces fluidd et mainsail en cliquant sur l'icône située en haut à droite !
    :::
