# Introduction à FLY-Tools

    FLY-Tools est un outil de débogage Web personnalisé pour Klipper développé par Guangzhou Miling Electronic Technology Co., Ltd. Il est non seulement adapté au système embarqué de Fly (prêt à l'emploi, sans installation supplémentaire), mais est également compatible avec la plupart des systèmes de contrôle équipés du service Klipper.

    :::astuce Astuce

    * FLY-Tools n'est compatible qu'avec les systèmes Linux ayant installé klipper

    :::

## Caractéristiques de FLY-Tools

    * Prise en charge de plusieurs langues et thème sombre
    * Permet de rechercher et copier automatiquement en un clic l'ID du port USB, l'ID de périphérique USB, l'UUID de CANbus et l'ID de périphérique vidéo
    * Permet d'activer ou désactiver un périphérique CAN spécifique en un clic
    * Permet de modifier en ligne le taux de transmission et la taille du tampon d'envoi des périphériques CAN
    * Permet de compiler en ligne le firmware Klipper pour la carte mère FLY (La compilation du firmware Katapult est en cours d'adaptation)
    * Permet de programmer en ligne via DFU, HID, CAN, etc.
    * Permet de télécharger tous les fichiers de firmware compilés
    * Permet de générer en ligne des graphiques de charge Klipper
    * Permet de mesurer automatiquement et générer un graphique de résonance en un clic
    * Permet de se connecter via SSH sur une page web
    * Prend en charge la connexion SSH basée sur [ttyd](https://github.com/tsl0922/ttyd)
    * Compatible avec le système FLYOS 3.3.4, permettant de configurer l'écran et l'affichage web
