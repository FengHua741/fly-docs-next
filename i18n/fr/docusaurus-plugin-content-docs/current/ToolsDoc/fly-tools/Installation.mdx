# Installation de FLY-Tools

    1. Cloner la dernière version de Klipper

    ```bash
    cd ~/klipper
    git pull
    ```

    * Si vous avez modifié le code source local de Klipper, ce qui provoque un échec du clonage, utilisez la commande suivante. (Cette opération jettera les fichiers que vous avez précédemment modifiés)

    ```bash
    cd ~/klipper
    git checkout .
    git pull
    ```

    2. Cloner la dernière version de Katapult (anciennement Canboot)

    ```
    cd ~/
    git clone https://github.com/Arksine/katapult
    ```

    3. Installer le service Fly-Tools

        * Le processus d'installation dépend du fournisseur de services Git. Si le téléchargement est bloqué, appuyez sur `CTRL+C` pour sortir et réessayer.
        * Github a une limite de 60 requêtes par heure, Gitee inconnue.
        * Il est recommandé aux utilisateurs en Chine d'obtenir le script d'installation via Gitee
        ```bash
        curl -kfsSL https://gitee.com/zxkxz/FLY-Tools/raw/main/scripts/install.sh | sudo bash
        ```

        * Obtenir le script d'installation via Github

        ```bash
        curl -kfsSL https://raw.githubusercontent.com/kluoyun/FLY-Tools/main/scripts/install.sh | sudo bash
        ```
