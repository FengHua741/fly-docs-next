# Téléchargement des ressources

:::tip

Veuillez sélectionner le type de ressource correspondant dans la barre de navigation à gauche pour le téléchargement.

:::

:::warning Attention

Lors du téléchargement de fichiers, veillez à bien lire le nom du fichier et les indications sur la page afin d'éviter de télécharger un fichier incorrect.

:::

:::danger À savoir

Certains fichiers peuvent être stockés sur Github ou dans un service de stockage en ligne. Assurez-vous que votre connexion Internet est stable lors du téléchargement.

:::
