# Modification de la zone horaire du système

    Veuillez noter que si vous utilisez FLY OS 3.3.4, l'application supérieure FLY peut utiliser directement FLY-Tools pour modifier la zone horaire.
    Pour les autres applications supérieures, veuillez suivre les instructions ci-dessous.

    ```bash
    sudo timedatectl set-timezone <TIMEZONE>
    ```

    Remplacez `<TIMEZONE>` par le code de la zone horaire que vous souhaitez utiliser. Vous pouvez rechercher le code de la zone horaire sur le site [iana.org](https://www.iana.org/time-zones). Par exemple, si vous souhaitez changer la zone horaire du système en temps de Shanghai (Asia/Shanghai), vous pouvez entrer la commande suivante :

    ```bash
    sudo timedatectl set-timezone Asia/Shanghai
    ```

    Maintenant, votre zone horaire système a été modifiée avec succès en temps de Shanghai. Vous pouvez vérifier la configuration de la zone horaire avec la commande suivante :

```bash
date
```
