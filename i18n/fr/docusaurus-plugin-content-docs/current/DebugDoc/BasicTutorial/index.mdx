# Connexion SSH au PC supérieur

:::danger

* Ce document ne s'applique qu'au PC supérieur FLY, les autres appareils ne sont mentionnés que pour référence.
* Si vous compilez et brûlez le firmware, veuillez consulter le tutoriel de connexion réseau plutôt que le tutoriel de connexion série.

:::

* Vous pouvez vous connecter à l'appareil via le port série ou le WiFi.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou tout autre **outil de terminal SSH**.
* Assurez-vous que la carte SD ou M2WE a été gravée avec le système correspondant au PC supérieur.
* **Si le PC supérieur ne démarre pas correctement, retirez tous les périphériques externes tels que les pilotes, les limitateurs, les ventilateurs, etc. !!!**
* **Enfin, connectez-vous au SSH via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, consultez le tutoriel de connexion au PC supérieur via le WiFi ou le câble réseau.**

        ## Utilisation de l'adresse IP réseau pour se connecter au SSH du PC supérieur

        * Assurez-vous que le système est démarré correctement.
        * Connectez-vous à l'interface arrière du routeur auquel le PC supérieur est connecté, sauf si vous connaissez l'adresse IP du PC supérieur.
        * Assurez-vous que vous connaissez le nom d'utilisateur et le mot de passe du PC supérieur, le nom d'utilisateur par défaut est **fly**, et le mot de passe par défaut est **mellow**.
        * Si la carte mère est déjà démarrée et est connectée au réseau, vous pouvez utiliser la méthode suivante pour vous connecter au SSH.
        * Ouvrez MobaXterm et effectuez les paramétrages suivants, entrez l'adresse IP que vous avez vue précédemment, ou consultez-la dans l'interface arrière du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Attention : Dans cette étape 4, entrez le nom d'utilisateur du PC supérieur.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * La première connexion peut faire apparaître cette fenêtre, cliquez sur `Accept` pour continuer.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Il vous sera demandé d'entrer le mot de passe, tapez le mot de passe du PC supérieur et appuyez sur Entrée.

        **Attention : Ici, le mot de passe n'est pas affiché lors de la saisie, appuyez simplement sur Entrée une fois terminé.**

        Le nom d'utilisateur par défaut est : `fly` , et le mot de passe par défaut est : `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />
