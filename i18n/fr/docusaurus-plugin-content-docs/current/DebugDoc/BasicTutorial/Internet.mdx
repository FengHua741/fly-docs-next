# Connexion réseau via SSH


* Assurez-vous que le système est démarré et connecté à l'hôte supérieur via SSH en utilisant un port série, il est recommandé d'utiliser **MobaXterm_Personal**
* Assurez-vous que l'hôte supérieur est installé avec **M2WE** ou **2.4G_WiFi module**, si c'est **Lite2** ou **MiniPad**, vous n'avez pas besoin d'installer de module
* Notez que **2.4G_WiFi module**, **Lite2** et **MiniPad** ne peuvent utiliser que le WiFi 2.4G et non le WiFi 5G
* Veuillez noter que le réseau ne peut pas utiliser la fusion des deux fréquences sinon la connexion ne fonctionnera pas correctement
* Assurez-vous que les antennes sont installées correctement !!!

 1. Dans le logiciel SSH, tapez la commande **nmtui** puis appuyez sur Entrée, une interface comme celle ci-dessous apparaîtra

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="80%" align="center" />

 2. Appuyez sur Entrée puis sélectionnez **Activate a connection** et appuyez à nouveau sur Entrée

  <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui.webp').default} size="80%" align="center" />

 3. Sélectionnez le réseau auquel vous souhaitez vous connecter, appuyez sur Entrée, entrez le mot de passe WiFi, puis sélectionnez OK pour enregistrer la connexion

   <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-1.webp').default} size="80%" align="center" />

 4. Après avoir connecté le WIFI, appuyez plusieurs fois sur la touche **Esc** du clavier pour sortir de l'interface actuelle, puis tapez **ip a** et appuyez sur Entrée

 * L'adresse IP de l'hôte supérieur est 192.168.2.136

    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/nmtui-3.webp').default} size="80%" align="center" />
