# Fonctionnalités étendues de KlipperScreen

## Méthode pour activer le curseur de la souris dans KlipperScreen

    * Ouvrez `KlipperScreen.conf` dans les paramètres de l'imprimante, ce fichier est généré automatiquement après l'installation de KlipperScreen. Si vous ne l'avez pas, créez-le vous-même.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.webp').default} size="100%" align="center" />
    * Ajoutez le code ci-dessous au début de la configuration
    ```cfg
    [main]
    show_cursor: True
    ```
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf-1.webp').default} size="80%" align="center" />
    * Enregistrez et redémarrez pour que cela prenne effet

## Connexion à distance à d'autres ordinateurs de contrôle

    * Ouvrez `KlipperScreen.conf` dans les paramètres de l'imprimante, ce fichier est généré automatiquement après l'installation de KlipperScreen. Si vous ne l'avez pas, créez-le vous-même.
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/ks/conf.webp').default} size="100%" align="center" />
    * Ajoutez le code ci-dessous au début de la configuration
    * Vous pouvez modifier le nom de l'imprimante `VORON-300` et l'adresse IP du serveur `192.168.6.217` à connecter.
    * Notez que la deuxième `[printer KlipperScreen]` est pour se connecter à une machine distante étendue, vous pouvez utiliser cette configuration pour déboguer ou éteindre.
    :::danger À savoir

    * Assurez-vous d'être dans un même réseau local

    :::

    ```cfg
    # Définissez l'imprimante et le nom. Le nom est tout ce qui suit le premier mot imprimante
    [printer VORON-300]
    # Définissez l'hôte/moonraker port si différent de 127.0.0.1 et 7125
    moonraker_host: 192.168.6.217
    # Les ports 443 et 7130 utiliseront https/wss
    moonraker_port: 7125
    # Clé API Moonraker si ce n'est pas connecté depuis une adresse IP client fiable
    moonraker_api_key: False

    # Définissez l'imprimante et le nom. Le nom est tout ce qui suit le premier mot imprimante
    [printer KlipperScreen]
    # Définissez l'hôte/moonraker port si différent de 127.0.0.1 et 7125
    moonraker_host: 127.0.0.1
    # Les ports 443 et 7130 utiliseront https/wss
    moonraker_port: 7125
    # Clé API Moonraker si ce n'est pas connecté depuis une adresse IP client fiable
    moonraker_api_key: False

    ```
