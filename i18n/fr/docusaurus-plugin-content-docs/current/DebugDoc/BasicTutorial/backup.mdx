# Méthodes de configuration de sauvegarde

:::danger À savoir

* Ce tutoriel s'applique aux FLY-Pi, Raspberry Pi, Orange Pi et autres ordinateurs supérieurs.
* Si vous utilisez EMMC, vous devez pouvoir entrer en mode de gravure, sinon ce tutoriel ne s'applique pas.
* Ce tutoriel est basé sur des opérations sous le système Windows, veuillez ne pas formater les disques inconnus, etc.!!!
* Ne formatez PAS !!! Ne formatez PAS !!! Ne formatez PAS !!!

:::

## Téléchargement du logiciel nécessaire

* Cliquez pour accéder au téléchargement, la page web vous indiquera la version appropriée pour votre ordinateur.
* Le logiciel est une version portable, pas besoin de l'installer, décompressez-le après le téléchargement pour l'utiliser.
* <Button variant="contained" disableElevation component="a" rel="noopener" href="https://www.diskgenius.cn/download.php">Outils de disque DG</Button>

## Démarrage de la sauvegarde

:::info
* Ne formatez PAS !!!
* Après la formatage, il sera impossible de récupérer !!!
* Si c'est une carte TF, branchez-la directement sur l'ordinateur.
* Si c'est un EMMC, vous devez entrer manuellement en mode de gravure !!!
:::

### Connexion
* Branchez le système à sauvegarder sur l'ordinateur.
* Ouvrez `DiskGenius.exe`.
* S'il est normal, il affichera le disque que vous devez sauvegarder.
    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/boot.webp').default} size="80%" align="center" />

### Recherche
* Cliquez sur la partition principale à gauche, puis cliquez sur Parcourir les fichiers en haut, cela affichera enfin les fichiers ci-dessous.
    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/home.webp').default} size="80%" align="center" />
* Trouvez `home` et double-cliquez pour y entrer.
* Une fois entré, il y a généralement qu'un seul dossier, et son nom est celui de votre utilisateur système.
* Si deux ou plus, vous devez déterminer par vous-même lequel contient vos configurations système.
* Double-cliquez ensuite pour y entrer.
* Cherchez `printer_data`.
    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/printer-data.webp').default} size="80%" align="center" />
* Le dossier `printer_data` contient vos fichiers d'impression, les configurations Klipper, les journaux Klipper, etc., ils peuvent être exportés directement.

### Exportation des configurations
* Cliquez avec le bouton droit sur `printer_data`, un message apparaîtra ci-dessous.
    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export.webp').default} size="40%" align="center" />
* Sélectionnez le répertoire où vous souhaitez enregistrer ou exportez directement.
* Confirmez l'exportation.
    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export1.webp').default} size="80%" align="center" />
* Ouvrez le fichier exporté et trouvez `config`, qui contient vos configurations système.
    <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/backup/export2.webp').default} size="80%" align="center" />
* L'exportation des configurations est terminée.
* Veuillez noter que le fichier `moonraker` ne doit pas être téléchargé sur d'autres systèmes, car cela peut entraîner une impossibilité de connexion de Klipper en raison de différences de système.
