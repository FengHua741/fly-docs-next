# Tutoriel d'utilisation
# Ce script nécessite une connexion Internet pour être utilisé
## 1. Installation automatique
```cfg
cd ~
git clone https://github.com/xishangyue/Semi-automatic-burning.git
cd Semi-automatic-burning
chmod +x Semi-automatic-burning.sh
./Semi-automatic-burning.sh
```
## Exécution à nouveau
```cfg
./Semi-automatic-burning/Semi-automatic-burning.sh
```
* La première exécution installera automatiquement les dépendances

* Si vous avez un firmware BL et un ID USB, choisissez le mode ID USB Si vous utilisez USB to CAN, choisissez le mode CANID
* Si vous avez une carte CAN, choisissez directement le mode CANID

### Vous entrerez automatiquement dans l'interface de configuration du firmware
* Configurez le firmware de la carte mère puis appuyez sur Q Y et appuyez sur Entrée
* Le firmware sera compilé automatiquement
