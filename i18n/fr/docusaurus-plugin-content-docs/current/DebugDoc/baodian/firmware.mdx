# Notice du firmware

    :::danger Notice

    * Il est préférable d'utiliser le firmware de l'ordinateur principal pour compiler le firmware Klipper
    * L'utilisation d'un firmware compilé par quelqu'un d'autre ou d'une version différente du firmware peut entraîner divers problèmes
    * Essayez d'éviter d'utiliser un ordinateur principal modifié

    :::

## Notice spéciale

    :::info Notice spéciale
    * La plupart des cartes mères FLY ont été programmées avec le firmware `Katapult` lors de la fabrication
    * Les cartes mères FLY utilisent principalement le firmware `Katapult` pour écrire le firmware Klipper
    * Les cartes mères DP5, D5 et D7 nécessitent une programmation manuelle du firmware `Katapult` correspondant
    * En raison de l'influence de l'ancien firmware, après avoir programmé le firmware `Katapult`, si vous devez rechercher l'ID de la carte mère, vous devez appuyer deux fois sur le réinitialisation de la carte mère, sinon la programmation du firmware ne peut pas être effectuée
    :::

## Explications sur la compilation du firmware

    :::tip

    * Les cartes mères FLY utilisent le bootloader programmé avec Katapult et la carte mémoire
    * L'avantage du firmware Katapult est qu'il permet une mise à jour du firmware sans contact avec la carte mère
    * L'avantage du firmware bootloader est qu'il évite de flasher le mauvais firmware
    * Il est recommandé d'utiliser `MobaXterm` pour la configuration du firmware, car certains logiciels SSH spéciaux signalés par les membres de la communauté peuvent entraîner des échecs de compilation
    
    :::

    * Veuillez noter que quelle que soit la version du firmware, il est essentiel de s'assurer que la compilation du firmware est correcte, sinon cela peut entraîner des problèmes de recherche d'impression normale ou même la non-détection de l'appareil

## Explications sur la compilation du firmware

    :::tip
    * Il est important de cocher l'option `Activer les options de configuration de bas niveau supplémentaires` sinon le contenu détaillé ne sera pas affiché
    :::

### STM32

    * Si la compilation du firmware STM32 nécessite la configuration des `broches GPIO à configurer au démarrage du microcontrôleur`, veuillez noter qu'il faut utiliser des majuscules

### RP2040

    * Si la compilation du firmware RP2040 nécessite la configuration des `broches GPIO à configurer au démarrage du microcontrôleur`, veuillez noter qu'il faut utiliser des minuscules

## Explications sur la programmation du firmware

    :::danger Notice

    * Pour programmer le firmware Klipper via le firmware Katapult, assurez-vous que le périphérique peut être trouvé. Si le périphérique ne peut pas être trouvé, appuyez deux fois sur la réinitialisation de la carte mère
    * N'utilisez pas de méthodes autres que celles du tutoriel pour programmer le firmware
    * Vérifiez s'il y a des erreurs de compilation du firmware. Si des erreurs de compilation existent, la programmation du firmware ne peut pas être effectuée correctement

    :::

### Firmware Katapult

    * Méthode pour résoudre les problèmes de compilation d'un firmware erroné avec le firmware Katapult
    * Appuyez deux fois sur la réinitialisation de la carte mère, normalement une LED clignotera après le réinitialisation
    * Effacez le firmware de la carte mère, puis reprogrammez le firmware Katapult

### Firmware bootloader

    * Méthode pour résoudre les problèmes de compilation d'un firmware erroné avec le firmware bootloader
    * Réessayez d'écrire le firmware avec la carte SD
    * Si cela ne fonctionne pas, débranchez l'alimentation, installez la carte SD dans la carte mère, puis rallumez-la, cela prendra cinq secondes pour écrire
