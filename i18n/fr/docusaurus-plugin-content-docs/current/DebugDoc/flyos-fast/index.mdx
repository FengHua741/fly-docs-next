# Présentation du système et téléchargement

## Présentation

* FlyOS-Fast est un système Linux allégé, personnalisé pour l'impression 3D, compatible avec toutes les plateformes de contrôle Fly.
* FlyOS-Fast est allégé et démarre rapidement.
* Le noyau est optimisé pour garantir la réactivité de Klipper.
* Le système de fichiers est optimisé pour réduire le risque d'impossibilité de démarrage en cas de coupure de courant.
* Démarrage rapide, 30 secondes pour démarrer Klipperscreen.
* Préinstallé avec Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen, etc.
* Supporte le passage entre les systèmes Klipper et RRF.
* Supporte le passage en ligne entre Mainsail et Fluidd.
* Supporte l'impression via une clé USB.
* Supporte la reprise d'impression après coupure de courant.
* Supporte les écrans à liaison série.
* Supporte le flashage automatique au démarrage des firmwares (fly-c8, fly-geminiv3).
* Supporte la configuration avancée des langues par défaut et des noms d'imprimantes pour Fluidd, etc.
* Pour plus de fonctionnalités OEM sur mesure : tels que le logo au démarrage, les fichiers de configuration préparamétrés, la possibilité de masquer les onglets Mainsail, etc., veuillez nous contacter.

## Présentation des noms des fichiers d'image du système

* La règle de nommage des fichiers système est : `FlyOS-Fast-[série de puces]-[numéro de version].img`, par exemple : `FlyOS-Fast-H5-v1.0.0.img`.
* Le fichier source est au format `.img`, les liens de téléchargement ne fournissent que des fichiers compressés au format `.exe` et `.7z`, pour faciliter le téléchargement.
* Pour les systèmes Windows, téléchargez le fichier `.exe`, qui est un fichier de compression 7z auto-extractible. Une fois le téléchargement terminé, double-cliquez dessus pour extraire le fichier `.img` (seul le système Windows peut extraire ce fichier).
* Pour les autres systèmes comme MacOS et Linux, téléchargez le fichier `.7z`, qui est un fichier de compression 7z. Une fois le téléchargement terminé, utilisez un outil compatible avec le format 7z pour extraire le fichier `.img`.
* En raison de la taille importante des fichiers système, d'autres formats de compression et le téléchargement de l'image source sont temporairement indisponibles.

:::tip Sélection du système

* Les séries fly-pi, fly-c8, fly-gemini, etc., tous les produits Fly utilisant le puces H5, choisissez `FlyOS-Fast-H5.exe` ou `FlyOS-Fast-H5.7z`.
* Les séries fly-pi-lite2, fly-miniPad, etc., tous les produits Fly utilisant le puces H3, choisissez `FlyOS-Fast-H3.exe` ou `FlyOS-Fast-H3.7z`.
* Les modèles de puces étant différents, les fichiers système sont également différents, veuillez choisir en fonction de vos besoins réels.

:::

## Téléchargement des images du système

* L'adresse officielle de téléchargement de FlyOS-Fast : [FlyOS-Fast dépôt d'artefacts](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/packages)

- Nom d'utilisateur par défaut : `root`
- Mot de passe par défaut : `mellow`

:::tip À noter

Ce système est un système personnalisé allégé, les composants et services intégrés comme Klipper ne sont pas des versions officielles et ne supportent pas la modification ou la mise à jour par l'utilisateur.

:::

## Processus de téléchargement de l'image du système

* Processus de téléchargement de l'image FlyOS-Fast :

    1. Accédez au dépôt d'artefacts FlyOS-Fast dans votre navigateur web.

    2. Choisissez le format de compression souhaité, puis cliquez pour accéder à la page du fichier.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down1.webp').default} size="50%" align="center" />

    3. Choisissez la version (version par défaut : la dernière).

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down2.webp').default} size="50%" align="center" />

    4. Si vous ne voyez pas le bouton de téléchargement, placez la souris dans la liste de fichiers, appuyez sur la touche `Shift` et roulez la molette de la souris pour faire défiler la liste de fichiers horizontalement.
    5. Vous pouvez également placer la souris sur la barre de défilement en bas, maintenez le bouton gauche de la souris enfoncé et faites glisser vers la droite pour voir le bouton de téléchargement.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/down3.webp').default} size="50%" align="center" />

    6. Cliquez sur le bouton de téléchargement pour obtenir le fichier d'image du système correspondant.
