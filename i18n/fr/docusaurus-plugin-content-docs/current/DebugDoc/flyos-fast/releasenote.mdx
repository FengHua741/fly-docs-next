# Historique des publications du système

## V1.2.39

### Modifications

- Résolution du problème empêchant le démarrage après gravure sur le système Windows11 24H2.

### Ajouts

- Ajout de la configuration système `canbus_bitrate`, pour définir le taux de transmission par défaut du can0, valeur par défaut `1000000` soit `1M`.
- Ajout de la configuration système `localtime`, pour définir le fuseau horaire par défaut, valeur par défaut `Asia/Shanghai`.
