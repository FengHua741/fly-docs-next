## Instructions d'utilisation

    :::warning Notes importantes

    * Veuillez noter qu'il faut d'abord effectuer l'étalonnage du courant avant l'étalonnage de la fréquence de la bobine.
    * Les machines à plusieurs axes Z nécessitent un nivellement manuel une fois que toutes les étalonnages ont été effectuées, puis un nouvel étalonnage.
    * Le firmware de la machine inférieure doit être parfaitement compatible avec la version de Klipper.

    :::

    * Veuillez utiliser la page web `FUIDD` pour ajouter la configuration ci-dessous avant d'exécuter les opérations suivantes
        ```cfg
        [force_move]
        enable_force_move: true
        ```

    ### Étape 2 : Étalonner le courant EDDY

        * 1: Entrez la commande ci-dessous dans la console
            ```bash
            SET_KINEMATIC_POSITION z=80
            ```
        * 2: Renvoyez à l'origine X et Y, déplacez la tête d'outil au centre du lit
            ```bash
            G28 X Y
            ```
        * Après avoir complété les étapes ci-dessus, il est nécessaire d'étalonner le courant Eddy. D'abord, déplacez Eddy au centre du lit puis ajustez la buse pour qu'elle touche juste le lit. Ensuite, exécutez la commande ci-dessous dans la console pour obtenir automatiquement le courant, puis `SAVE_CONFIG` pour sauvegarder les paramètres. Une valeur de retour d'environ 15 est normale, une valeur de 0 ou 30 indique un problème de connexion (mauvais contact).
            ```bash
            LDC_CALIBRATE_DRIVE_CURRENT CHIP=fly_eddy_probe
            ```
        * La valeur de retour normale devrait être autour de `15`
        * Après l'étalonnage, sauvegardez les paramètres avec `SAVE_CONFIG`

    ### Étape 2 : Étalonner la fréquence Eddy


        :::info

        * Lors de l'installation d'Eddy, la plaque ocb doit être au moins 1-2 mm plus haute que la buse.
        * Lors de l'étalonnage de la fréquence Eddy, la bobine PCB doit être éloignée d'au moins 5 mm du lit.
        * Cette distance de 5 mm est pour faciliter l'étape suivante d'obtention du `z_offset` avec du papier.

        :::

        * Ensuite, déplacez la tête d'impression au centre de la plateforme (remarque : assurez-vous que la machine n'a pas chargé de heightmap à cette étape)
        * Commencez l'étalonnage manuel de l'offset z [Test du papier](https://www.klipper3d.org/Bed_Level.html#the-paper-test)
            ```bash
            PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
            ```
        * Après l'étalonnage, sauvegardez les paramètres avec `SAVE_CONFIG`
        :::info
        * Pour les imprimantes avec la fonction `z_tilt` ou `quad_gantry_level (QGL)`, ajustez d'abord la parallélisme. Pour i3, ajustez le parallélisme de l'axe X, pour les machines à trois axes ajustez le parallélisme du lit, pour voron 2.4 ajustez le parallélisme du portique avant de recalibrer la fréquence Eddy.
        :::
    ### Étape 3 : Compensation de température
        * Étape 3 : La compensation de température doit être effectuée après avoir réussi à retourner à l'origine.

        * Retournez tous les axes à l'origine
       ```bash
       G28
       ```

        * Si vous avez `[quad_gantry_level]` ou `[z_tilt]`, exécutez d'abord `Z_TILT_ADJUST` ou `QUAD_GANTRY_LEVEL`, puis retournez à l'origine une fois de plus.
        * Dans la console, entrez G0 Z5 ou utilisez un autre moyen pour déplacer l'axe Z à 5 mm au-dessus du lit.
        * Exécutez le code ci-dessous pour prolonger le `idle timeout` de la machine, pour éviter un timeout pendant le chauffage.
            ```bash
            SET_IDLE_TIMEOUT TIMEOUT=65000
            ```
        * Exécutez la commande ci-dessous pour commencer l'étalonnage
            ```bash
            TEMPERATURE_PROBE_CALIBRATE PROBE=fly_eddy_probe TARGET=56 STEP=4
            ```
        :::warning Notes importantes

        * Remarque : Dans la commande ci-dessus, la température cible pour EDDy est fixée à `56℃`, ce qui convient à la plupart des dispositifs.
        * Si la température réelle de la cavité de votre imprimante 3D est plus élevée, vous pouvez augmenter la température cible en conséquence.
        * Veuillez noter que plus la température cible est élevée, plus le temps d'étalonnage sera long, car Eddy prendra plus de temps pour atteindre l'équilibre thermique.

        :::
    
        1. Après avoir exécuté cette opération, l'interface utilisateur affichera une boîte de dialogue pour ajuster l'axe Z. Utilisez la méthode manuelle d'étalonnage de l'offset Z (Test du papier) mentionnée ci-dessus pour placer une feuille de papier entre la buse et le lit, puis confirmez cette valeur.
        2. Après avoir accepté la valeur, réglez la température du lit à la plus élevée, et la buse à 220℃.
        3. Si vous êtes dans une pièce climatisée ou avec des fenêtres ouvertes, pour garantir la montée en température d'Eddy, il est recommandé de fermer la climatisation ou les fenêtres, car le vent peut affecter la montée en température.
        4. À mesure que la température d'Eddy augmente, le système vous invitera automatiquement à effectuer un étalonnage manuel de l'offset Z tous les 4℃. Attention, la température du lit est très élevée, évitez les brûlures !!!
        5. Répétez l'étalonnage manuel de l'offset Z (Test du papier) jusqu'à ce que l'étalonnage soit terminé. Si la température d'Eddy ne monte plus, vous pouvez utiliser les commandes suivantes pour terminer l'étalonnage plus tôt.
        
        * Pendant la calibration de dérive, les commandes gcode supplémentaires disponibles incluent : TEMPERATURE_PROBE_NEXT
            * `TEMPERATURE_PROBE_NEXT`    Pour forcer Eddy à échantillonner de nouvelles données avant d'atteindre la température cible de `56℃`.
            * `TEMPERATURE_PROBE_COMPLETE`    Pour compléter l'étalonnage si Eddy ne peut pas atteindre la température cible de `56℃`.
            * `ABORT`  Pour annuler l'étalonnage et ignorer les résultats.
        * Après avoir terminé ces étapes, Eddy sera capable de fournir d'excellents résultats d'impression de la première couche sur une large gamme de températures !


## Comment ajuster finement l'offset z
* Étape 1 : Utilisez Prusa/OrcaSlicer/Cura/Bambu Studio pour trancher une pièce carrée de 200x200 de hauteur 0.2 pour ajuster finement l'offset z.
* Étape 2 : Accédez à l'interface de la machine via l'IP de l'imprimante, trouvez le tableau de bord `Contrôle de déplacement` pour ajuster finement l'offset z, puis notez la valeur ajustée.
<ImageView image={require('@site/docs/DebugDoc/faq/img/Zoff.webp').default} size="80%" align="left" />
* Étape 3 : Redémarrez Klipper avec la commande suivante
```bash
FIRMWARE_RESTART
```
* Étape 4 : Utilisez la commande ci-dessous pour ajuster l'offset z, la valeur ajustée dans l'interface UI doit correspondre à celle ajustée pendant l'impression.
```bash
G28
PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
```
<ImageView image={require('@site/docs/DebugDoc/faq/img/UI.webp').default} size="50%" align="left" />

:::info
* Si l'erreur `Erreur de capteur de courant Eddy` apparaît, augmentez la valeur de z_offset dans la configuration d'Eddy, regardez l'exemple.

<ImageView image={require('@site/docs/DebugDoc/faq/img/EDDYZOFF.webp').default} size="50%" align="left" />
:::

:::info
 * Si toutes les opérations ci-dessus sont terminées mais que le balayage du lit échoue, changez le z en limite virtuelle.
:::
