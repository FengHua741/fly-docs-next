# Erreurs liées à TMC



* `Impossible de lire l'enregistrement IFCNT du port UART TMC 'stepper_x'` ou `Impossible de lire l'enregistrement de l'enregistrement SPI TMC 'stepper_x'`

    1. Détails de l'erreur

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart.webp').default} size="80%" align="center" />

    2. Cause de l'erreur

        * Impossibilité de communiquer avec le pilote

    3. Méthodes principales de résolution

        * Vérifiez si la configuration des broches UART du pilote correspondante est correcte
        * Les sauts UART sont-ils corrects ?
        * Le pilote dispose-t-il d'alimentation ?
        * Remplacez le pilote ou configurez la prise en charge pour effectuer un diagnostic
        * Si vous utilisez un autre pilote, vous devez déterminer si les broches UART du pilote sont compatibles avec FLY. Les broches UART du pilote FLY sont la quatrième broche en partant du haut à gauche
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/2209uart.webp').default} size="80%" align="center" />
        * Si les broches UART de votre pilote sont sur la cinquième broche, vous pouvez résoudre le problème selon le saut ci-dessous
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/uart1.webp').default} size="80%" align="center" />

* `Le pilote TMC XXXX signale une erreur : GSTAT: 00000001 reset=1(Réinitialisation)`

    1. Détails de l'erreur

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/reset.webp').default} size="80%" align="center" />

    2. Cause de l'erreur

        * Réinitialisation du pilote


    3. Méthodes principales de résolution

        * Les connexions d'alimentation sont mal branchées
        * Le fil du moteur est sectionné
        * Vous pouvez connecter la coque métallique du moteur au GND du contrôleur principal ou de la plaque de test
---
<br/>
<br/>

* `Le pilote TMC 'extrudeur' signale une erreur : DRV_STATUS: c00d0103 otpw=1(Avertissement de surchauffe !)`

    1. Détails de l'erreur

        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/overtemperror.webp').default} size="80%" align="center" />

    2. Cause de l'erreur

        * Le pilote est surchauffé !
        * Ce n'est pas le moteur qui surchauffe !

    3. Méthodes principales de résolution

        * Diminuer le courant de fonctionnement
        * Supprimer le courant de maintien
        * Vérifiez si la résistance d'échantillonnage du pilote est correcte
        * Améliorez la dissipation thermique du pilote, par exemple en ajoutant un ventilateur
---
<br/>
<br/>

* `Le pilote TMC 'extrudeur' signale une erreur : DRV_STATUS: 601f1000 s2vsa=1(Courant court-circuit vers l'alimentation A!) csactual=31 ola=1(Charge ouverte A!) olb=1(Charge ouverte B!)`

    1. Détails de l'erreur

        * Aucune image disponible

    2. Cause de l'erreur

        * Le courant du pilote est incorrectement vérifié


    3. Méthodes principales de résolution
        
        * Vérifiez si les valeurs de résistance des deux ensembles de bobines du moteur pas à pas sont identiques
        * Vérifiez si les fils internes du moteur pas à pas ou des composants du moteur sont relâchés ou courts-circuités
        * Désactivez le mode silencieux
---
<br/>
<br/>

        * Vérifiez si les valeurs de résistance des deux ensembles de bobines du moteur pas à pas sont identiques

        * Vérifiez l'ordre des fils du moteur, s'il est composé d'un groupe à gauche et d'un groupe à droite
