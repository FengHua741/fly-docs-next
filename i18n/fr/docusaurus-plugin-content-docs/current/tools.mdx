# Outils pratiques

:::tip

Ce document contient divers outils utiles pour l'adaptation de Klipper.

:::

:::warning Attention

Certains petits outils, comme le « Décodeur de firmware Klipper » sont en ligne, tandis que « Fly-Tools » doivent être installés avant utilisation. Veuillez faire la distinction.

:::

:::danger À savoir

Si vous rencontrez des problèmes lors de l'utilisation, veuillez soumettre une demande d'assistance à temps.

:::
