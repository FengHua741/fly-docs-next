## Entrer en mode de programmation

    :::tip

    Au-dessus se trouve le bouton `RESET`, en dessous se trouve le bouton `BOOT`

    :::

    * Il y a deux méthodes pour entrer en mode de programmation
    * Première méthode : dans une situation où la carte mère est complètement déconnectée de l'alimentation, appuyez sur le bouton BOOT puis branchez le câble de transmission de données d'une extrémité à la carte mère et de l'autre extrémité à votre appareil
    * Deuxième méthode : après avoir branché le câble de transmission de données d'une extrémité à la carte mère et de l'autre extrémité à votre appareil, appuyez sur le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST enfin relâchez le bouton BOOT

            4. Flasher le firmware

            :::tip

            Vous n'avez besoin de flasher qu'un seul firmware

            :::

            * Ci-dessous est la commande pour flasher le firmware CAN avec un taux de 500K
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```
                       
            * Ci-dessous est la commande pour flasher le firmware CAN avec un taux de 1M
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_CAN_500K.bin
                ```

            * Ci-dessous est la commande pour flasher le firmware USB
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/MMU/FLY_MMU_CANBOOT_USB.bin
                ```

            5. Si vous voyez le contenu indiqué par la flèche ci-dessous ``File downloaded successfully`` alors le flashage a réussi

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::tip

            Après le flashage, vous devez débrancher complètement l'alimentation de la carte mère.

            :::
