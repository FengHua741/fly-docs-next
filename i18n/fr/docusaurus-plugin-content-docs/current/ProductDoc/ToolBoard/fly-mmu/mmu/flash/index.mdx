# Explications sur la programmation de la mise à jour du firmware

    * MMU peut se connecter à l'ordinateur principal via trois méthodes : RS232, CAN, USB
    * Parmi elles, RS232 et CAN nécessitent l'utilisation de `XT30(2+2)-F` pour la connexion, tandis que USB nécessite l'utilisation d'un câble de données USB Type-C avec fonction de transmission de données
    * Ci-dessous est le modèle de câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La mise à jour du firmware de la carte mère doit être effectuée en programmant Klipper via Katapult
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, il n'est pas nécessaire de programmer le firmware Katapult à la fin, car cela ne serait qu'une mesure de précaution au cas où la programmation de la carte mère aurait échoué
* Vous pouvez vérifier rapidement si la carte mère est entrée dans le firmware Katapult en appuyant deux fois rapidement sur le bouton de réinitialisation. Si un LED clignote, cela signifie que la carte mère est dans le firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * MMU peut se connecter à l'ordinateur principal via les méthodes suivantes
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Programmation du firmware CAN</Button>
        * Avantages : Si vous avez installé le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher la carte mère et il supporte la connexion en bus
        * Inconvénients : Il peut y avoir des erreurs comme un dépassement de délai de retour à zéro en utilisant `KlipperScreen` en même temps
        * Conseil : Si vous devez utiliser CAN, assurez-vous que l'ordinateur principal prend en charge CAN et qu'il doit être utilisé avec UTOC ou une carte mère avec un firmware de pont
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Programmation du firmware RS232</Button>
        * Avantages : Il n'est pas nécessaire d'utiliser un logiciel spécial pour l'utiliser, la communication est fiable et stable
        * Inconvénients : Pour mettre à jour le firmware, vous devez entrer en mode DFU manuel ou appuyer deux fois sur le bouton RST pour entrer dans `Katapult` afin de mettre à jour le firmware
        * Conseil : Il faut utiliser G2T ou UTOR pour l'utiliser
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Programmation du firmware USB</Button>
        * Avantages : Si vous avez installé le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher la carte mère, et si vous utilisez une carte mémoire ou une programmation DFU, vous devez toujours suivre la méthode originale pour mettre à jour le firmware. La communication est fiable et stable
        * Inconvénients : Si plusieurs périphériques USB sont présents, cela peut affecter la transmission de données
