# Compilation du firmware USB

    :::tip

    * Le firmware Katapult avec connexion CAN préinstallé à la sortie d'usine
    * Si vous utilisez le programmeur USB, il est nécessaire de flasher le firmware USB de Katapult
    * Une fois flashé, il n'est plus nécessaire de refaire le processus
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Flashing du firmware Katapult</Button>

    :::

## Démarrage de la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : !PA15 (remarque : c'est en majuscules)
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * Appuyez sur `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous êtes revenu à l'interface de ligne de commande
    * Entrez la commande suivante pour commencer la compilation, cela peut prendre un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Lorsque la sortie finale ressemble à ce qui suit, la compilation est réussie

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
