# Compilation du firmware USB

    :::tip

    * Le firmware CAN préinstallé à la sortie d'usine pour le MMU
    * Si vous utilisez la compilation via USB, vous devez brûler le firmware USB pour Katapult
    * Une fois brûlé, il n'est pas nécessaire de le brûler à nouveau
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Brûlage du firmware Katapult</Button>

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : !PA15 Veuillez noter que ! est en majuscule
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-mmu/mmu/flash/img/usb.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * La fin de la sortie avec le contenu suivant indique que la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```
