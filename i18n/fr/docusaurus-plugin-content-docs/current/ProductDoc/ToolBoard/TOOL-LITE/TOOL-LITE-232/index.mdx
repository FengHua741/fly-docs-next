# Présentation

# 1. Présentation du produit

FLY TOOL-LITE-232 est une carte d'outils conçue par Guangzhou Melun Electronic Technology Co., Ltd pour les imprimantes 3D, qui offre des interfaces xt30/kf. Elle peut remplacer les câbles complexes entre la carte mère et la tête d'impression avec seulement quatre fils, optimisant ainsi l'aménagement. La TOOL-LITE-232 convient aux moteurs pas à pas de 36/42 étapes et intègre un transmetteur-récepteur RS232, un port TypeC, un pilote TMC2209, un capteur d'accélération LIS2DW, deux ventilateurs contrôlables et d'autres fonctionnalités. Cette carte d'outils convient au firmware Klipper.

## 1.1 Caractéristiques du produit

    * Bouton Boot réservé, l'utilisateur peut entrer en mode Boot via le port TypeC pour mettre à jour le firmware
    * Prise en charge du firmware Klipper
    * Prise en charge du mode S232
    * Dans le mode RS232, il offre un taux de transmission plus élevé et une meilleure stabilité.
    * Capteurs de température pris en charge : capteurs de température ntc100k et PT1000.
    * Deux ventilateurs contrôlables pris en charge, avec un courant de sortie maximal de 1A
    * Capteur d'accélération LIS2DW embarqué
    * Interface probe haute tension réservée, pilote 2209 embarqué, supporte divers capteurs de nivellement, tels que BLtouch, klicky, Voron Tap, PL08n, etc.
    * Supporte une tension d'entrée de 12-24V, un courant d'entrée de 15A, une augmentation de la prise de câble chauffant, avec un courant nominal de 10A

## 1.2 Nouvelles fonctionnalités

    * Nouveau mode RS232, offrant un taux de transmission plus élevé et une meilleure stabilité, plus simple à utiliser pour le débogage
    * Nouvelle connexion d'alimentation/signal d'un mètre, avec un design à double torsion, offrant une meilleure performance anti-interférences

## 1.3 Paramètres du produit

    * MCU : Raspberry Pi rp2040, Double cœur ARM Cortex-M0+ @ 133MHz
    * Interfaces de connexion utilisant des connecteurs XT30 (2+2) supportant un courant de 15A, pic à 30A
    * Firmware : Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs embarqués : LIS2DW
    * Prises chauffantes : Chauffant (HEAT0), 10A
    * Interfaces d'extension : interface probe haute tension, EndStop, PT100/PT1000, XH2.54-2P CAN extension port
    * Interfaces de ventilateur : deux ventilateurs DC contrôlables
    * Pilotes de moteur : pilote TMC2209 embarqué
    * Capteurs de température : un capteur de température de 100K NTC ou PT1000 (TH0)
