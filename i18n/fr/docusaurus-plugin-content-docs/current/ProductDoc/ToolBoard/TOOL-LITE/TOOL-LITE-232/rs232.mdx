# Compilation du firmware RS232

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <RS232 />

    * Sélectionnez **GPIO pins à définir au démarrage du microcontrôleur**, entrez : **!gpio18**
    :::warning
        Veuillez entrer `!gpio18` en mode d'entrée anglaise
    :::
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/TOOL-LITE/TOOL-LITE-232/img/rs232.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration doit maintenant être enregistrée et vous devez être revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="65%" align="center" />
    * Si la sortie finale est comme ci-dessous, la compilation a réussi
       <ImageView image={require('@site/docs/General/dfu-flash/img/make_uf2_finish.webp').default} size="65%" align="center" />




## Entrer en mode de flashage

    :::tip

    L'indicateur pointé par la flèche est le bouton `BOOT`

    :::

    * Le boîtier doit être complètement déconnecté avant d'appuyer sur le bouton `BOOT`. Utilisez un câble de données avec fonction de transfert de données USB-C pour connecter le boîtier à l'interface USB de votre ordinateur hôte. Après une connexion correcte à l'ordinateur hôte, relâchez le bouton `BOOT`
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/TOOL-LITE/TOOL-LITE-232/img/boot.webp').default} size="80%" align="center" />
