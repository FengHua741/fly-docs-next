# Introduction

# 1. Product Introduction

FLY TOOL-LITE-CAN est une carte outil conçue par Guangzhou Melun Electronic Technology Co., Ltd pour les imprimantes 3D. Cette carte outil fournit des interfaces xt30/kf, permettant de remplacer les câbles complexes entre la carte mère et l'extrudeur avec seulement quatre fils, optimisant ainsi le layout. TOOL-LITE-CAN est compatible avec les moteurs pas à pas 36/42. Elle intègre un transmetteur-récepteur CANBus, un port TypeC, un driver TMC2209, un capteur d'accélération LIS2DW, deux ventilateurs contrôlables et d'autres fonctionnalités. Cette carte outil est compatible avec le firmware Klipper.

## 1.1 Caractéristiques du produit

    * Bouton Boot prévu, l'utilisateur peut entrer en mode Boot via le port TypeC pour mettre à jour le firmware.
    * Prise en charge du firmware Klipper.
    * Prise en charge du mode CANBus.
    * En mode CANBus, elle offre un taux de transmission plus élevé et une meilleure stabilité.
    * Capteurs de température pris en charge : capteurs de température ntc100k et PT1000.
    * Deux ventilateurs contrôlables, avec une sortie maximale de courant de 1A.
    * Capteur d'accélération LIS2DW monté sur la carte.
    * Interface probe haute tension prévue, driver TMC2209 monté sur la carte, supporte divers capteurs de nivellement tels que BLtouch, klicky, Voron Tap, PL08n, etc.
    * Prise en charge de tensions d'entrée de 12-24V et de courants jusqu'à 15A, interface chauffage élargie avec un courant nominal de 10A.

## 1.2 Nouveautés

    * Nouvelle connexion d'alimentation/signaux de 1m, les câbles utilisent une conception torsadée, offrant des performances antienregistrement accrues.

## 1.3 Paramètres du produit

    * MCU : Raspberry Pi rp2040, double cœur ARM Cortex-M0+@133MHz
    * Interfaces de connexion utilisant des connecteurs XT30 (2+2) prenant en charge un courant de 15A, pointeau 30A
    * Firmware : Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs montés sur la carte : LIS2DW
    * Interfaces de chauffage : chauffage (HEAT0), 10A
    * Interfaces d'extension : interface probe haute tension, fin de course, PT100/PT1000, XH2.54-2P interface CAN d'extension
    * Interfaces de ventilateur : deux ventilateurs DC contrôlables
    * Drivers de moteur : driver TMC2209 monté sur la carte
    * Capteurs de température : capteur de température 100K NTC ou PT1000 (TH0)
