<Tabs>
    <TabItem value="can" label="Obtention de l'UUID CANBus">

    * Avant de rechercher l'ID, connectez le panneau d'outils à UTOC ou à une carte mère avec un firmware de pont flashé en utilisant un câble de données XT30. Ci-dessous est un schéma de câblage de référence
    * Veuillez vérifier que les commutateurs sont positionnés correctement
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Assurez-vous que la résistance entre CAN H et CAN L est d'environ `60Ω` en mesure avec un multimètre dans un état déchargé. Si c'est `120Ω`, vous devez brancher un câble de saut de `120Ω`. Si c'est environ `40Ω`, vous devez retirer le câble de saut
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/120.webp').default} size="100%" align="center" />
        <GetCanbusUUID />

    </TabItem>
    <TabItem value="rs232" label="Obtention de l'ID RS232">

    * Avant de rechercher l'ID, connectez le panneau d'outils à UTOR ou G2T en utilisant un câble de données XT30. Ci-dessous est un schéma de câblage de référence
    * Veuillez vérifier que les commutateurs sont positionnés correctement
    <ImageView image={require('@site/docs/General/get-id/img/utor.webp').default} size="80%" align="left" />              
        <GetRS232ID />

    </TabItem>
    <TabItem value="usb" label="Obtention de l'ID USB" default>
        
    * Préparez un câble de données capable de transférer des données. Si la longueur est très longue, il est recommandé d'utiliser un câble avec un aimant
        <GetUsbID />
    </TabItem>
