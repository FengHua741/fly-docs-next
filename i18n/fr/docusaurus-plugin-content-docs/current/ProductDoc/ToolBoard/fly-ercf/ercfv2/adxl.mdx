# Utilisation de l'accéléromètre


import ADXL from "/docs/DebugDoc/faq/kipper-adxl.mdx";

## Basculer vers klipper

    * Car il n'a pas encore été fusionné, vous devez basculer vers klipper pour pouvoir l'utiliser

    ```bash
    sudo cp -r klipper klipperdev
    sudo rm -rf ~/klipper
    git clone https://github.com/FLYmaker/klipper.git
    sudo systemctl restart klipper
    ```

    * Configuration de référence
    ```cfg
    #####################################################################
    #                       Configuration de l'accéléromètre LIS3DH (activer si nécessaire)
    #####################################################################
    [lis2dw]
    is_lis3dh:True
    cs_pin: SHT36:gpio9
    spi_software_sclk_pin: SHT36:gpio10
    spi_software_mosi_pin: SHT36:gpio11
    spi_software_miso_pin: SHT36:gpio12

    [resonance_tester]
    accel_chip: lis2dw
    probe_points:
        125, 125, 20  # Définissez ces coordonnées lorsque vous effectuez des mesures de résonance avec la buse
    ```
