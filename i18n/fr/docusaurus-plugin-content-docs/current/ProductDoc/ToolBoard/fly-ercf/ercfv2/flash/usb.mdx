# Compilation de la firme USB

    :::tip

    * La firme Katapult préinstallée pour l'ERCF avec la connexion CAN
    * Si vous utilisez la compilation via USB, vous devez brûler la firme USB de Katapult
    * Une seule fois de brûlage est nécessaire après le premier brûlage
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Brûlage de la firme Katapult</Button>

    :::

## Démarrage de la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Sélectionnez les GPIO pins à initialiser au démarrage du microcontrôleur, entrez : gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/usb.webp').default} size="100%" align="center" />
    * Appuyez sur `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient les lignes suivantes, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```
