# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::tip

    * ERCF  prend en charge CAN et USB
    * Cette version fournit le firmware CAN à 500K et 1M
    :::

## Entrer en mode de programmation

    :::tip

    Ce qui est entouré est la `touche BOOT`

    :::

    * Pour entrer en mode de programmation, il faut que la carte mère soit complètement débranchée, puis maintenir enfoncée la touche BOOT et connecter le câble de transmission de données entre la carte mère et l'ordinateur de contrôle.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/boot.webp').default} size="100%" align="center" />

            4. Programmer le firmware

            :::tip

            Il n'est nécessaire de programmer qu'un seul firmware

            :::

            * Ci-dessous, la commande pour programmer le firmware à 500K CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_500K.uf2
                ```
                       
            * Ci-dessous, la commande pour programmer le firmware à 1M CAN
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/ERCF/FLY_ERCF_CANBOOT_CAN_1M.uf2
                ```
            * Ci-dessous, la commande pour programmer le firmware USB
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/FLY_ERCF_CANBOOT_USB.uf2
                ```

            5. Si vous voyez ci-dessous, la programmation est réussie.

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />

            :::tip

            Après la programmation, il faut débrancher complètement la carte mère une fois.

            :::
