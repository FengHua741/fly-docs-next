# Instructions de gravage du firmware

    * ERCF peut être connecté à l'ordinateur principal via la méthode CAN
    * CAN doit être connecté avec `XT30(2+2)-F`
    * Ci-dessous est l'apparence du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* Le gravage du firmware de la carte mère doit passer par Katapult pour graver Klipper
* Si la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire d'effectuer le dernier gravage du firmware Katapult, cela ne sert qu'à prévenir une éventuelle erreur de gravage
* Il est possible de vérifier si la carte mère est entrée dans le firmware Katapult en appuyant rapidement deux fois sur le bouton de réinitialisation, si une LED clignote, c'est un bon signe
:::

## Comment déterminer quel firmware vous avez besoin

    * ERCF peut se connecter à l'ordinateur principal de la manière suivante
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Gravage du firmware CAN</Button>
        * Avantage : Si le firmware Katapult est installé, il est possible de mettre à jour le firmware sans toucher la carte mère et il prend en charge les connexions en bus
        * Inconvénient : L'utilisation conjointe de KlipperScreen peut entraîner des erreurs telles que des dépassements de délai lors des recalibrages
        * Astuce : Si vous devez utiliser CAN, assurez-vous que l'ordinateur principal supporte CAN et qu'il est nécessaire d'utiliser UTOC ou une carte mère avec un firmware de pont
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Gravage du firmware USB</Button>
        * Avantage : Si le firmware Katapult est installé, il est possible de mettre à jour le firmware sans toucher la carte mère, si le firmware est installé via une carte SD ou DFU, il est nécessaire de suivre la méthode d'origine pour mettre à jour le firmware, la communication est fiable et stable
        * Inconvénient : Si plusieurs périphériques USB sont présents, ils peuvent affecter la transmission des données
