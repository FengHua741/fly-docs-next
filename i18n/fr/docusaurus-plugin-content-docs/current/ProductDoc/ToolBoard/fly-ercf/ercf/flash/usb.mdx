# USB 固件编译

    :::tip

    * Le firmware Katapult préinstallé sur l'ERCF via la connexion CAN
    * Si vous utilisez le firmware USB pour le flashage, vous devez flasher le firmware USB de Katapult
    * Une fois flashé, il n'est pas nécessaire de le refaire
    * <Button variant="contained" disableElevation href="./bl">Flashage du firmware Katapult</Button>

    :::

## Début du processus de compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Sélectionnez les GPIO pins à configurer au démarrage du microcontrôleur, entrez : gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercf/flash/img/usb.webp').default} size="100%" align="center" />
    * Appuyez sur `Q`, apparaît **Enregistrer la configuration**, appuyez alors sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * La sortie finale doit ressembler à ceci pour que la compilation soit réussie

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```
