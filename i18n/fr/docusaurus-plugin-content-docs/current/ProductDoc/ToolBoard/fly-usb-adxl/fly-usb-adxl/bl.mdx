# Firmware Katapult

## Entrer en mode de flashage

    :::tip

    Souligné est `le bouton BOOT`

    :::

    * Pour entrer en mode de flashage, il faut que la carte mère soit complètement débranchée, maintenir enfoncé le bouton BOOT puis connecter un câble de transmission de données entre la carte mère et votre appareil.
