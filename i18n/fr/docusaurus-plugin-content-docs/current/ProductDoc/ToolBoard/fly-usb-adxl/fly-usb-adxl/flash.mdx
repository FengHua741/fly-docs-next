# Compilation du firmware USB

    :::tip

    * Le firmware Klipper pour la connexion USB est préinstallé sur l'accéléromètre USB à l'usine
    * Il suffit de mettre à jour le firmware
    * Il est possible de connecter l'ordinateur et l'accéléromètre USB via un câble USB

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur la touche `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration doit maintenant être enregistrée et vous êtes revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour démarrer la compilation, cela prend un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient les lignes suivantes, la compilation a réussi :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
