# Compilation du firmware USB

    :::tip

    * Le capteur d'accélération USB est pré-rempli avec le firmware Klipper pour la connexion USB à l'usine
    * Il suffit de mettre à jour le firmware
    * Vous pouvez connecter votre ordinateur et le capteur d'accélération USB via un câble USB

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur `Q`, puis apparaît **Save configuration**, appuyez ensuite sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir sauvegardé la configuration et être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prendra un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble au texte suivant, la compilation a réussi

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
