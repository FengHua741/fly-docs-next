# Firmware Katapult

## Entrer en mode de flashage

    :::tip

    Marqué en rond est le `bouton BOOT`

    :::

    * Pour entrer en mode de flashage, il faut que la carte mère soit complètement débranchée, maintenir enfoncé le bouton BOOT puis brancher le câble de données avec une transmission de données entre la carte mère et votre appareil.

                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/USBADXL/FLY_USBADXL.uf2
                ```
