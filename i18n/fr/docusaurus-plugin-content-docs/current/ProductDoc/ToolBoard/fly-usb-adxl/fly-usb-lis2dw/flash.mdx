# Compilation du firmware USB

    :::tip

    * Le firmware Klipper pour la connexion USB est préinstallé sur l’accéléromètre USB à la sortie d’usine
    * Il suffit de mettre à jour le firmware
    * Il est possible de connecter l’ordinateur et l’accéléromètre USB via un câble USB

    :::

## Démarrage de la compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Appuyez sur `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration doit maintenant être enregistrée et vous êtes revenu à l'interface de commande
    * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble au texte ci-dessous, la compilation a réussi

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
