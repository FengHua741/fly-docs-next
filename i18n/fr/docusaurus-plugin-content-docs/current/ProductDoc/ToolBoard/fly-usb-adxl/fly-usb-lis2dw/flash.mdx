# Compilation du firmware USB

    :::tip

    * Le capteur d'accélération USB est préinstallé avec le firmware Katapult connecté via USB à la sortie de l'usine
    * Si le firmware n'est pas flashé, vous pouvez le faire manuellement
    * [Flashing du firmware Katapult](./bl.mdx "Cliquez pour sauter")

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, apparaîtra **Enregistrer la configuration**, puis appuyez sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Enfin, si la sortie suivante s'affiche, la compilation a réussi :

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```
