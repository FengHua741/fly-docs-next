# Compilation du firmware USB

    :::tip

    * Le firmware USB est préinstallé à la fabrication sur l'accéléromètre USB avec la connectivité Klipper
    * Il suffit de mettre à jour le firmware
    * Vous pouvez utiliser un câble USB pour connecter l'ordinateur à l'accéléromètre USB

    :::

## Démarrage de la compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : gpio8
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Appuyez sur `Q`, une fenêtre **Save configuration** apparaît, puis appuyez sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être enregistrée et vous êtes revenu à l'interface de commande
    * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble au contenu ci-dessous, la compilation a réussi

```bash
  Linking out/klipper.elf
  Building out/lib/elf2uf2/elf2uf2
  Creating uf2 file out/klipper.uf2
```
