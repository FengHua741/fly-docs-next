# Compilation du firmware USB

    :::tip

    * Le capteur d'accélération USB est préinstallé avec le firmware Katapult connecté via USB à la sortie de l'usine
    * Si vous constatez qu'il n'a pas été flashé, vous pouvez le faire vous-même
    * [Flashage du firmware Katapult](./bl.mdx "Cliquez pour accéder")

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Sélectionnez les broches GPIO à initialiser au démarrage du microcontrôleur, entrez : gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/usb.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, apparaît **Enregistrer la configuration**, appuyez ensuite sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * La compilation a réussi si vous obtenez le message suivant en fin de sortie :

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```
