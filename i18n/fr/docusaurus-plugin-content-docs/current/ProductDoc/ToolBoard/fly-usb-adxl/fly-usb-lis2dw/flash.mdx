# Compilation de la firmware USB

    :::tip

    * Le capteur d'accélération USB est préinstallé avec la firmware Katapult connectée via USB à la sortie d'usine
    * Si la firmware n'est pas installée, vous pouvez la réinstaller vous-même
    * [Installation de la firmware Katapult](./bl.mdx "Cliquez pour accéder")

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />
    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : gpio24
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-usb-adxl/fly-usb-lis2dw/img/24.webp').default} size="100%" align="center" />
    * Appuyez sur `Q`, une fenêtre **Save configuration** apparaît, puis appuyez sur `Y`
    <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir sauvegardé la configuration et être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
    <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * La fin des messages suivants indique que la compilation a réussi :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
