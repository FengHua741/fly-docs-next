# Instructions de gravage du firmware

    * La compilation du firmware et le gravage nécessitent un logiciel de haut niveau
    * SHT36 V2 peut utiliser la méthode de connexion CAN pour se connecter au logiciel de haut niveau
    * CAN doit être connecté en utilisant `XT30(2+2)-F`
    * Ci-dessous est l'apparence du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* Le flashage du firmware de la carte mère doit passer par Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin de faire le flashage final de Katapult, c'est juste une précaution si le flashage précédent n'a pas réussi
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère, si oui, cela signifie que la carte mère est déjà dans le firmware Katapult
:::

## Gravage du firmware CAN

    * SHT36 V2 ne peut utiliser que la méthode de connexion CAN pour se connecter au logiciel de haut niveau
    * Pour STM 103, veuillez consulter   <Button variant="contained" disableElevation href="./can103">Gravage du firmware CAN 103</Button>
    * Pour APM 072 et STM 072, veuillez consulter   <Button variant="contained" disableElevation href="./can072">Gravage du firmware CAN 072</Button>
