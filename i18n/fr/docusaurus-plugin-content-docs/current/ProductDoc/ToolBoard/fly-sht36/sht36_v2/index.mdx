# Présentation

## Présentation du produit

    FLY SHT36 V2 est une carte d'outils conçue par Guangzhou Merlen Electronic Technology Co., Ltd pour l'extrudeur d'imprimante 3D, utilisée pour remplacer les câbles complexes entre la carte mère et l'extrudeur avec quatre fils, optimisant ainsi le layout. SHT36 V2 est compatible avec les moteurs pas à pas de 36 étapes, intégrant un transmetteur-récepteur CAN, un port USB, un pilote TMC2209, un capteur d'accélération ADXL345, deux ventilateurs contrôlables et d'autres fonctionnalités. Cette carte d'outils est compatible avec le firmware Klipper.

    * Compression extrême du volume, conçue avec une carte à 6 couches
    * Supporte la connexion CAN, offrant une transmission de données plus stable, une latence réduite et une connexion plus stable
    * Intègre un pilote TMC2209, utilisable en mode UART, pouvant être utilisé pour l'extrudeur
    * Intègre un capteur d'accélération ADXL345
    * Supporte PT100 (MAX31865)
    * Conception de MOS interchangeable pour les ventilateurs
    * Supporte des capteurs de nivellement tels que BLTOUCH, PL08N, intégration des interrupteurs de limite XY
    * Supporte une entrée de tension de 12-24V 
    * MCU: STM32F072
    * Communication: CAN
    * Supporte des capteurs de nivellement tels que BLTOUCH, PL08N, intégration des interrupteurs de limite XY

## Nouveautés

    * Ajout du support pour une résistance de 4,7kΩ (pour le thermistor NTC) ou 1,1kΩ (pour le PT1000) pour basculer
    * L'interface CAN n'utilise plus les connecteurs MX3.0, les nouveaux connecteurs supportent un courant de 15A, avec un pic à 30A
    * Ajout de la possibilité de choisir entre 5/12/24V pour les ventilateurs
    * Ajout d'une résistance NTC100k intégrée sur la carte pour mesurer la température du bac
    * Ajout d'une protection contre la polarité inversée de la source
    * Ajout de l'interrupteur INT1 du ADXL345 connecté au mcu, pour certains utilisateurs utilisant ce capteur pour le nivellement
    * Augmentation de la taille des bornes de la résistance chauffante, avec un courant nominal de 10A
    * Ajout d'une limite de tension élevée
    * Remplacement du contrôleur principal par un MCU national
    * Supporte les encodeurs magnétiques (as5047d), utilisés pour l'analyse de mouvement dans Klipper (Klipper ne supporte pas actuellement cette fonctionnalité)
    * Nouvelle rallonge CAN de 1m, évitant aux utilisateurs le problème de perte de signal causé par une mauvaise compression des connecteurs
    * Dimensions d'installation : distance entre les trous de 43,84mm, trous d'installation M3
    * Processeur : GD32F103 (pour les achats avant le 28 octobre 2022), APM32F072 (pour les achats après le 28 octobre 2022)
