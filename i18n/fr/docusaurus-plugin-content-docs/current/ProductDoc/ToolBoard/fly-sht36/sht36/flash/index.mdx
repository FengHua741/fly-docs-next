# Notice de programmation de la mémoire interne

    * Les SHT36/42 peuvent être connectés à une station supérieure via une connexion CAN.
    * Le câblage CAN nécessite l'utilisation de brides métalliques MX3.0.


## Comment déterminer quel firmware vous avez besoin

    * Les SHT36/42 peuvent être connectés à une station supérieure via une méthode CAM.
    * <Button variant="contained" disableElevation href="./can">Programmation du firmware CAN</Button>
        * Avantage : Si vous installez le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher au circuit imprimé principal et il supporte les connexions en bus.
        * Inconvénient : L'utilisation conjointe de KlipperScreen peut entraîner des erreurs comme des retards de positionnement.
        * Remarque : Si vous souhaitez utiliser CAN, assurez-vous que la station supérieure prend en charge CAN et qu'elle est équipée d'un UTOC ou d'une carte mère avec un firmware pont.
