# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::tip

    * Support CAN pour SHT36/42
    * Seule la version CAN est fournie ici, avec des vitesses de 500K et 1M
    :::

## Entrer en mode de flashage

    :::tip

    Ce qui est entouré est le `bouton BOOT`

    :::

    * Avec la carte mère complètement débranchée, court-circuitez le bouton BOOT puis branchez un câble de données entre la carte mère et votre appareil

            4. Flasher le firmware

            :::tip

            Il n'est nécessaire de flasher qu'un seul firmware

            :::

            * Ci-dessous, la commande pour flasher le firmware à 500K CAN
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_500K.BIN
                ```
                       
            * Ci-dessous, la commande pour flasher le firmware à 1M CAN
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/developer/SHT36/FLY_SHT36_072_1M.BIN
                ```


            5. Si vous voyez le contenu indiqué par les flèches ci-dessous ``File downloaded successfully``, alors le flashage a réussi.

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />

            :::tip

            Après le flashage, il faut débrancher complètement la carte mère.

            :::
