# Firmware Katapult

## Explications du firmware Katapult

    <CANBOOT />

    :::conseil

    * SHT36_V3 prend en charge CAN et RS232
    * Seule la version CAN du firmware est fournie ici, avec des vitesses de 500K et 1M
    :::

## Entrer en mode de programmation

    :::conseil

    L'indicateur pointe vers le bouton `BOOT`

    :::

    * Éteignez complètement la carte mère, puis appuyez sur le bouton BOOT tout en connectant un câble de transmission de données d'un côté à la carte mère et de l'autre côté à votre appareil.

            4. Programmation du firmware

            :::conseil

            Il n'est nécessaire de programmer qu'un seul firmware.

            :::

            * Ci-dessous est la commande pour écrire le firmware à 500K CAN
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_500K.uf2
            ```
                       
            * Ci-dessous est la commande pour écrire le firmware à 1M CAN
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36V3_katapult_CAN_1M.uf2
            ```


            5. Si vous voyez ci-dessous, cela signifie que la programmation a réussi.

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::conseil

            Après la programmation, il faut débrancher complètement la carte mère.
