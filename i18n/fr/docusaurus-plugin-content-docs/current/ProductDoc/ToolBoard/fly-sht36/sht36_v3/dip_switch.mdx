# Présentation des commutateurs montés sur le circuit imprimé

:::warning Remarques

En raison de la résolution d'image trop élevée, il est impossible de voir clairement en parcourant la page web. Veuillez placer **le curseur de la souris sur le schéma de câblage**, **faire un clic droit**, puis sélectionner **Enregistrer l'image sous** pour enregistrer l'image sur le bureau et la consulter !!!

:::

## Sélection du mode de fonctionnement

:::warning Remarques

Veuillez noter que SHT36 V3 prend en charge trois modes de connexion. Vous devez modifier les commutateurs et reprogrammer le firmware pour une connexion correcte.
