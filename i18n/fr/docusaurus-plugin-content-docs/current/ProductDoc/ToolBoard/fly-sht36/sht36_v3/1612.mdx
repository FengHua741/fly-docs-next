# Utilisation d'Eddy

    :::info
  * Si vous avez configuré précédemment l'édition d'Eddy, veuillez supprimer la configuration.
  * Lors de l'installation d'Eddy, la bobine doit être au-dessus de la buse et à une distance de 0,6 à 0,8 cm de celle-ci.
  * Ne pas allumer le lit chauffant lors des tests.
  * Supprimez la configuration de `probe` avant utilisation.
    :::

    :::warning Remarques
    * Les machines multi-Z nécessitent un nivellement manuel une fois effectué.
    :::

## Explications d'utilisation
   * Connectez-vous à l'interface SSH via IP pour accéder au terminal et exécutez le script.
    Téléchargez les outils SSH nécessaires <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Outils Mobaxterm</Button>


<Tabs>
  <TabItem value="eddy1" label="Utilisation internationale">
    # Veuillez utiliser la commande suivante pour l'international
    ### Commencez l'installation du script automatique d'Eddy
    * Ouvrez l'outil SSH téléchargé, entrez la commande suivante et appuyez sur Entrée, puis cliquez sur Copier.
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Ajoutez la configuration suivante dans le fichier de configuration `moonraker.conf` de la page web de `fluidd` pour faciliter les mises à jour ultérieures.
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Si vous avez mis à jour, utilisez l'outil SSH pour exécuter les commandes suivantes.
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
