# Utilisation d'Eddy

    :::info
  * Si vous avez précédemment configuré l'utilisation d'Eddy, veuillez supprimer la configuration.
  * Au moment de l'installation d'Eddy, le bobineau doit être placé au-dessus de l'embout et à une distance de 0,6-0,8 cm de cet embout.
  * Pendant les tests, n'allumez pas la table chauffante.
  * Avant utilisation, supprimez la configuration du `sondeur` dans la configuration.
    :::

    :::warning Remarques importantes
    * Pour les machines à plusieurs axes Z, un nivellement manuel doit être effectué une fois.
    :::

## Explications d'utilisation
   * Connectez-vous à l'aide d'un outil ssh via l'adresse IP pour accéder au terminal et exécutez le script.
    Téléchargez l'outil ssh nécessaire <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">Outil MobaXterm</Button>


<Tabs>
  <TabItem value="eddy1" label="Utilisation internationale">
    # Veuillez utiliser la commande ci-dessous pour l'international
    ### Voici le début de l'installation du script automatique d'Eddy
    * Ouvrez l'outil ssh que vous avez téléchargé, entrez la commande ci-dessous et appuyez sur Entrée pour copier.
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Ajoutez la configuration suivante dans le fichier de configuration `moonraker.conf` de la page web de `fluidd` pour faciliter les mises à jour ultérieures.
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Si vous avez mis à jour, utilisez l'outil ssh pour exécuter la commande ci-dessous.
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* Exécutez-le et entrez votre mot de passe de la station supérieure comme indiqué dans l'image suivante.

<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />

* Si vous voyez l'image suivante, entrez le numéro correspondant à votre imprimante et appuyez sur Entrée. Il y a des options en chinois et en anglais. Après avoir entré le numéro, appuyez sur Entrée.
* Sélectionnez `3z` pour vous assurer que la configuration `z_tilt` est présente dans le fichier `printer.cfg` ou sélectionnez `4z` pour vous assurer que la configuration `quad_gantry_level` est présente. Pour un double Z indépendant, choisissez `3z`. [Référence de configuration](https://mellow.klipper.cn/docs/category/klipper-%E5%8F%82%E8%80%83%E9%85%8D%E7%BD%AE)
