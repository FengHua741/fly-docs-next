# Explications sur la programmation de la mémoire vive
    
    * La compilation du firmware et la programmation du firmware nécessitent l'utilisation d'un logiciel supérieur
    * La version V3 de SHT36 peut se connecter à un logiciel supérieur via RS232 ou CAN
    * RS232 et CAN nécessitent l'utilisation de `XT30(2+2)-F` pour la connexion
    * Ci-dessous se trouve l'apparence du câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La programmation de la mémoire vive de la carte mère doit passer par Katapult pour programmer Klipper
* Si la carte mère a déjà été programmée avec Katapult, il n'est pas nécessaire de procéder à la dernière étape de programmation de Katapult, cela n'est fait que pour prévenir une éventuelle erreur de programmation de la mémoire vive de Katapult
* Il est possible de vérifier rapidement si la carte mère est entrée en mode Katapult en appuyant deux fois rapidement sur le bouton de réinitialisation, si une LED clignote alors la carte mère est entrée en mode Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * La version V3 de SHT36 peut se connecter à un logiciel supérieur via les méthodes suivantes
    * <Button variant="contained" disableElevation href="./can">Firmware CAN</Button>
        * Avantage : Si vous avez installé le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher la carte mère et il prend en charge la connexion en bus
        * Inconvénient : L'utilisation de KlipperScreen peut entraîner des erreurs comme un dépassement de délai lors du retour à zéro
        * Conseil : Si vous souhaitez utiliser CAN, assurez-vous que votre logiciel supérieur prend en charge CAN et qu'il doit être utilisé avec UTOC ou une carte mère avec un firmware de pont
    * <Button variant="contained" disableElevation href="./rs232">Firmware RS232</Button>
        * Avantage : Il n'est pas nécessaire d'utiliser un logiciel spécial pour l'utiliser, la communication est fiable et stable
        * Inconvénient : Une nouvelle programmation de firmware nécessite d'entrer en DFU manuel pour mettre à jour le firmware ou appuyer deux fois rapidement sur RST pour entrer Katapult afin de mettre à jour le firmware
        * Conseil : Vous devez utiliser G2T ou UTOR pour l'utiliser
