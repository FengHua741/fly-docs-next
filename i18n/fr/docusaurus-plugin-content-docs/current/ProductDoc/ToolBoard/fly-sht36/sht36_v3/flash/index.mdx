# Notice sur la gravure du firmware
    
    * La compilation et la gravure du firmware nécessitent l'utilisation d'un ordinateur supérieur.
    * SHT36 V3 peut se connecter à l'ordinateur supérieur via les modes RS232 ou CAN.
    * RS232 et CAN nécessitent l'utilisation de câbles `XT30(2+2)-F`.
    * Voici un exemple de câble `XT30(2+2)-F` :
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La gravure du firmware de la carte mère doit passer par Katapult pour graver Klipper.
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire de graver à nouveau le firmware Katapult à la fin, car cela ne fait que prévenir une éventuelle erreur de gravage de la partie Katapult de la carte mère.
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère. Si c'est le cas, cela signifie que la carte mère est dans le firmware Katapult.
:::

## Comment déterminer quel type de firmware vous avez besoin

    * SHT36 V3 peut se connecter à l'ordinateur supérieur de la manière suivante :
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware CAN</Button>
        * Avantage : Si vous installez le firmware Katapult, vous pouvez mettre à jour le firmware sans toucher la carte mère, et il prend en charge la connexion en bus.
        * Inconvénient : Peut provoquer des erreurs comme un dépassement de délai lors de l'utilisation conjointe de KlipperScreen.
        * Conseil : Si vous utilisez CAN, assurez-vous que votre ordinateur supérieur prend en charge CAN, et qu'il est utilisé avec UTOC ou une carte mère avec firmware de pont.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Firmware RS232</Button>
        * Avantage : Nécessite aucun logiciel spécial pour être utilisé, communication fiable et stable.
        * Inconvénient : Pour mettre à jour le firmware, vous devez entrer en mode DFU manuel ou double-cliquer sur RST pour entrer dans Katapult.
        * Conseil : Nécessite G2T ou UTOR pour être utilisé.
