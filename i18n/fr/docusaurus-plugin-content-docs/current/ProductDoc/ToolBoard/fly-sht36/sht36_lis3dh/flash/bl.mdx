# Firmware Katapult

## Description du firmware Katapult

    <CANBOOT />

    :::astuce

    * SHT36 LIS3DH prend en charge CAN
    * Seule la version CAN du firmware est fournie ici, avec des vitesses de 500K et 1M
    :::

## Entrer en mode de programmation

    :::astuce

    Ce qui est entouré est le `bouton BOOT`

    :::

    * Éteignez complètement la carte mère, puis court-circuitez le bouton BOOT et branchez un câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur votre appareil
    * Ensuite, retirez le court-circuit du bouton BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_lis3dh/flash/img/boot.webp').default} size="50%" align="center" />

            4. Flasher le firmware

            :::astuce

            Il n'est nécessaire de flasher qu'un seul firmware

            :::

            * Ci-dessous se trouve la commande pour flasher le firmware à une vitesse de CAN de 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_500K.uf2
                ```
                        
                * Ci-dessous se trouve la commande pour flasher le firmware à une vitesse de CAN de 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36LIS3DH_CANBOOT_CAN_1M.uf2
                ```


            5. Si vous voyez le message ci-dessous, cela signifie que le flashage a réussi

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::astuce

            Après le flashage, il est nécessaire de débrancher complètement la carte mère.
