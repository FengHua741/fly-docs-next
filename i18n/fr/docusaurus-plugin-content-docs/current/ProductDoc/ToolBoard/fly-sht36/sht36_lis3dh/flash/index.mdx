# Instructions de gravage du firmware

    * SHT36 LIS3DH ne peut utiliser que la connexion CAN pour se connecter à l'ordinateur supérieur
    * La connexion CAN doit utiliser `XT30(2+2)-F`
    * Ci-dessous est le style de câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger Avertissement
* Le gravage du firmware de la carte mère doit passer par Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin d'effectuer le dernier gravage du firmware Katapult, cela n'est qu'une précaution au cas où le firmware Katapult ne serait pas gravé avec succès
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère, si c'est le cas, cela signifie que la carte mère est déjà dans le firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * SHT36 LIS3DH peut utiliser la méthode suivante pour se connecter à l'ordinateur supérieur
    * <Button variant="contained" disableElevation href="./can">Gravage du firmware CAN</Button>
        * Avantage : Si vous avez flashé le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher la carte mère et il supporte les connexions en bus
        * Inconvénient : L'utilisation de `KlipperScreen` peut entraîner des erreurs telles qu'un délai d'attente lors du retour à zéro
        * Conseil : Si vous devez utiliser CAN, assurez-vous que l'ordinateur supérieur prend en charge CAN et qu'il doit être associé à UTOC ou une carte mère avec un firmware de pont
