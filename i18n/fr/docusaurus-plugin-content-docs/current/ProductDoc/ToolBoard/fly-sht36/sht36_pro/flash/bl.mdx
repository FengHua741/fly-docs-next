# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::conseil

    * Le SHT36 PRO prend en charge CAN
    * Seule la version CAN du firmware est fournie ici, avec des vitesses de 500K et 1M
    :::

## Entrer en mode de programmation

    :::conseil

    Ce qui est entouré est le `bouton BOOT`

    :::

    * Avec le circuit principal complètement débranché, court-circuitez le bouton BOOT puis branchez un câble de transmission de données d'une extrémité au circuit principal et de l'autre à votre appareil
    * Ensuite, retirez le court-circuit du bouton BOOT
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_pro/flash/img/boot.webp').default} size="80%" align="center" />

            4. Flasher le firmware

            :::conseil

            Vous n'avez besoin de flasher qu'un seul firmware

            :::

            * Ci-dessous est la commande pour flasher le firmware avec une vitesse de CAN de 500K
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_500K.uf2
            ```
                       
            * Ci-dessous est la commande pour flasher le firmware avec une vitesse de CAN de 1M
            ```bash
            sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SHT36/FLY_SHT36PRO_CANBOOT_1M.uf2
            ```


            5. Si vous voyez ci-dessous, cela signifie que le flashage a réussi.

            <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::conseil

            Après le flashage, il est nécessaire de couper l'alimentation au circuit principal une fois.
