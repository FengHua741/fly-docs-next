# Présentation du produit

    FLY-SHT36-PRO est une carte outil conçue par la société Guangzhou Mellen Electronics Technology Co., Ltd pour l'extrudeur d'imprimante 3D. En utilisant cette carte, quatre fils peuvent remplacer les connexions complexes entre la carte mère et la tête d'impression, optimisant ainsi le câblage. La FLY-SHT36-PRO est compatible avec les moteurs pas à pas 36. Elle intègre un transmetteur-récepteur CAN, un port USB, un driver TMC5160, un capteur d'accélération ADXL345 et d'autres fonctionnalités comme deux ventilateurs contrôlables.

    * Compression de volume extrême, conception à 6 couches
    * Support des bus CAN pour une transmission de données plus stable, avec moins de latence et une connexion plus stable
    * Le driver TMC5160 intégré peut être utilisé pour l'extrudeur
    * Le capteur d'accélération ADXL345 est intégré sur la carte
    * Support pour PT100 (MAX31865)
    * Conception de MOS pour les ventilateurs amovibles
    * Support des capteurs de nivellement tels que BLTOUCH, PL08N, et des interrupteurs de limite XY intégrés
    * Entrée de tension : 12-24V
    * MCU: STM32F072
    * Communication: CANBUS, USB

## Paramètres du produit

    1. Support des modes Klipper CAN, RS232 et RRFCAN pour un changement mutuel
    2. Les connecteurs d'interface utilisent des connecteurs XT30 (2+2) supportant un courant de 15A, avec un pic de 30A
    3. MCU : Raspberry Pi rp2040, double cœur ARM Cortex-M0+@133MHz
    4. Tension d'entrée : DC12-24V 5A
    5. Courant maximal de sortie pour la résistance de chauffage : 5A
    6. Courant maximal de sortie pour les ventilateurs : 1A
