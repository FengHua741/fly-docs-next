# Impossible d'utiliser le pilote 5160 sur la carte mère

    ```bash
    Unable to write tmc spi 'extruder' register IHOLD_IRUN
    ```

## Raisons spécifiques

    * Si vous utilisez 5160, vous obtiendrez une erreur ci-dessous en raison du problème de la chaîne d'outils Arm `arm-none-eabi-gcc`

## Méthodes de résolution
    Il existe trois méthodes de résolution

    * Changer la chaîne d'outils `arm-none-eabi-gcc` de `gcc10` à `gcc8`
    * Changer le système de l'ordinateur principal
    * Écrire un firmware correct

    Télécharger et graver le firmware, veuillez noter que le panneau de commande doit être connecté normalement au CAN et doit disposer de CANBOOT

    * Télécharger

    ```bash
    cd ~/ && git clone https://github.com/FengHua741/FLY_Katapult.git
    ```
    * Graver

    ```
    python3 ~/klipper/lib/canboot/flash_can.py -f ~/FLY_Katapult/Klipper/SHT36PRO/SHT36PRO_KLIPPER_CAN-1M_v0.12.0-229-g589bd64c.bin -u <SHT36PRO_canid>
    ```
