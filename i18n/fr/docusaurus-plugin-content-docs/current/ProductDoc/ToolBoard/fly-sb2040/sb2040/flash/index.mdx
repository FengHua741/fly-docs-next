# Instructions de gravage du firmware

    * SB2040 peut être connecté à l'ordinateur hôte via la connexion CAN
    * La connexion CAN doit utiliser `XT30(2+2)-F`
    * Ci-dessous se trouve le câble de données `XT30(2+2)-F`
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* Le firmware de la carte mère doit être gravé via Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin d'effectuer le dernier étape de gravage du firmware Katapult, cela n'est fait que pour prévenir un échec de gravage
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour vérifier si une LED clignote sur la carte mère, si c'est le cas, cela signifie que la carte mère est dans le firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * SB2040 ne peut être connecté à l'ordinateur hôte que via la connexion CAN
    * <Button variant="contained" disableElevation href="./can">Gravage du firmware CAN</Button>
        * Avantages : Si vous installez le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher à la carte mère et il supporte la connexion en bus
        * Inconvénients : L'utilisation conjointe de `KlipperScreen` peut entraîner des erreurs telles qu'un dépassement de délai lors du retour à zéro
        * Remarque : Si vous souhaitez utiliser CAN, assurez-vous que l'ordinateur hôte prend en charge CAN et qu'il doit être associé à UTOC ou à une carte mère avec un firmware de pont installé
