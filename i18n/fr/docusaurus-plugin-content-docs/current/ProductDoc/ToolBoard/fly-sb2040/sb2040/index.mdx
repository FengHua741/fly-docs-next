## Présentation du produit

    FLY SB2040 est une carte outil conçue par Guangzhou Mellen Electronic Technology Co., Ltd pour l'imprimante à tête d'impression Stealthburber de VORON. En utilisant cette carte outil, quatre fils peuvent remplacer les connexions complexes entre la carte mère et la tête d'impression, optimisant ainsi le layout. SB2040 est compatible avec les moteurs pas à pas de 36 étapes. Elle intègre un transmetteur-récepteur CAN, un port USB, un driver TMC2209, un capteur d'accélération ADXL345, deux ventilateurs contrôlables et d'autres fonctionnalités. Cette carte outil convient aux firmwares Klipper.

    * Supporte la connexion CAN, avec des transferts de données plus stables, des délais plus faibles et une connexion plus stable.
    * Intègre un driver TMC2209 sur la carte, utilisable en mode UART, pour les extrudeurs.
    * Intègre un capteur d'accélération ADXL345 sur la carte.
    * Supporte les capteurs de nivellement BLTOUCH, PL08N et intègre des interrupteurs limites XY.
    * Supporte une entrée de tension de 12-24V.
    * MCU : Raspberry Pi rp2040, double cœur ARM Cortex-M0+ @ 133MHz.
    * Supporte la résistance de 4,7K (pour les thermistances NTC) ou de 1K (pour les PT1000) pour le commutateur.
    * L'interface CAN utilise un connecteur XT30 (2+2), supportant un courant de 15A, avec un pic de 30A.
    * Intègre une résistance NTC100k sur la carte pour mesurer la température de la chambre.
    * Ajoute les broches INT1 et INT2 du capteur ADXL345 connectées au MCU pour certains utilisateurs pour le nivellement.
    * Supporte les limites de tension élevée.
    * Offre un câble de connexion CAN de 3m inclus, évitant les problèmes de perte de signal dus à une connexion incorrecte des connecteurs.
    * Supporte les ventilateurs de 2/3/4 lignes.
    * Conception modulaire, la tête Stealthburner peut être facilement démontée.
    * Compatible avec la carte outil tête Stealthburner de Hartk (le petit ventilateur de la carte n'a pas besoin d'être changé).
    * Interfaces : 3 ventilateurs contrôlables, RGB*1, limiteurs*3, résistance chauffante*1, thermistor*1.

## Spécifications du produit

    * Supporte le mode CAN Klipper, RS232, RRFCAN pour le basculement mutuel.
    * Les interfaces de connexion utilisent un connecteur XT30 (2+2) supportant un courant de 15A, avec un pic de 30A.
    * MCU : Raspberry Pi rp2040, double cœur ARM Cortex-M0+ @ 133MHz.
    * Tension d'entrée : DC12-24V.
    * Courant maximal de sortie pour la résistance chauffante : 5A.
    * Courant maximal de sortie pour les interfaces ventilateur : 1A.
