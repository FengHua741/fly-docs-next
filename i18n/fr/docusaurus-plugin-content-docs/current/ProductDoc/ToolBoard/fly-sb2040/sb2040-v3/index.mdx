# 1. Présentation du produit

    FLY SB2040 V3 est une carte outil conçue par la société Guangzhou Mellen Electronic Technology Co., Ltd pour l'imprimante Voron Stealthburber. En utilisant cette carte outil, quatre fils peuvent remplacer les câbles complexes entre la carte mère et l'imprimante, optimisant ainsi le câblage. SB2040 convient aux modèles stealthburner et afterburner. Elle intègre un transmetteur-récepteur CAN, un transmetteur-récepteur RS232, un port USB, un pilote TMC2209, un capteur d'accélération LIS2DW et trois ventilateurs contrôlables, entre autres fonctionnalités. Cette carte outil convient aux firmwares Klipper et RRF.

## 1.1 Caractéristiques du produit

    * Bouton Boot réservé, permettant à l'utilisateur de passer en mode Boot via le port Type-C pour mettre à jour le firmware
    * Support des firmwares Klipper et RRF
    * Support des modes CAN, RS232, RRF CAN,
    * En mode RS232, il offre un taux de transmission plus élevé et une meilleure stabilité. En mode CAN, il supporte l'extension CAN, avec une interface d'extension CAN réservée
    * Capteurs de température : puce MAX31865 embarquée, compatible avec les capteurs de température PT100 et PT1000.
    * Capteurs embarqués LIS2DW et LDC1612
    * Interface RGB réservée, pilote 2209 embarqué, supporte divers capteurs de nivellement, tels que klicky, Voron Tap, PL08n, etc.
    * Support des tensions d'entrée de 12-24V, courant d'entrée de 15A, port chauffant agrandi, courant nominal 10A

## 1.2 Nouveautés

    * Support des modes CAN de Klipper, RS232 de Klipper, et RRFCAN de Klipper, avec possibilité de basculer entre eux
    * Nouveau mode RS232, offrant un taux de transmission plus élevé et une meilleure stabilité, plus simple à configurer
    * Nouveau mode RRF CAN, pouvant être utilisé comme carte outil RRF, augmentant la polyvalence
    * Interface d'extension CAN réservée, facilitant l'extension d'appareils CAN
    * Nouvelle commutation de résistance terminale par dip-switch, la résistance terminale de 120R peut être sélectionnée par dip-switch sans utiliser de jumpers
    * Nouvelle rallonge de 1 mètre pour alimentation/signal, utilisant un design en double torsion, offrant une meilleure résistance aux interférences

## 1.3 Paramètres du produit


    * MCU : Raspberry Pi rp2040, Double cœur ARM Cortex-M0+ @ 133 MHz
    * Interfaces de connexion utilisant des connecteurs XT30 (2+2) supportant 15A de courant, pic de 30A
    * Firmware : Reprap/Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs embarqués : LIS2DW et optionnels ldc1612 et max31865
    * Interface chauffant : chauffant (HEAT0)
    * Interfaces d'extension : RGB, EndStop, PT100/PT1000, connecteur CAN XH2.54-2P
    * Interfaces ventilateur : un ventilateur DC, deux ventilateurs DC ou ventilateurs PWM
    * Pilote moteur : pilote TMC2209 embarqué
    * Capteurs de température : un capteur NTC de 100K ou PT1000 (TH0), un capteur PT100/PT1000 optionnel
