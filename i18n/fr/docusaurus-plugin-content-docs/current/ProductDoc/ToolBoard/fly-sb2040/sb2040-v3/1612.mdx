# Utilisation d'Eddy

    :::info
  * Si vous avez configuré précédemment l'édition d'Eddy, veuillez supprimer
  * Lors de l'installation d'Eddy, la position du plateau inférieur par rapport à l'embout doit être au moins 1-2 mm au-dessus de l'embout
  * Ne pas allumer le lit chauffant lors des tests
  * Supprimez la configuration `probe` dans les paramètres avant utilisation
    :::

    :::warning Remarques
    * Les machines à plusieurs axes Z nécessitent un niveau manuel une fois réglé correctement
    :::

    # Veuillez utiliser la commande ci-dessous pour l'utilisation internationale
    ### Installation automatique d'Eddy commençant maintenant
    * Ouvrez l'outil SSH téléchargé, entrez la commande suivante et appuyez sur Entrée, puis cliquez sur Copier
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Ajoutez la configuration suivante dans le fichier de configuration `moonraker.conf` de la page web de `fluidd` pour faciliter les mises à jour ultérieures
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Si vous mettez à jour, exécutez la commande suivante
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
