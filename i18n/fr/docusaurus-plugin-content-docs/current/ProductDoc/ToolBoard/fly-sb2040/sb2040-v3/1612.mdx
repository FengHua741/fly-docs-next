# Utilisation d'Eddy

    :::info
  * Si vous aviez précédemment configuré la configuration d'Eddy, veuillez la supprimer
  * Lors de l'installation d'Eddy, le bobineau doit être au-dessus de l'embout et à une distance de 0,6-0,8 cm de l'embout
  * Ne pas allumer la plaque chauffante lors des tests
  * Supprimez la configuration `probe` dans la configuration avant utilisation
    :::

    :::warning Remarques
    * Les machines à plusieurs axes Z doivent être nivelées manuellement une fois
    :::


## Schéma de câblage

    * Le logo doit être orienté vers la plaque chauffante lors de l'installation

## Explications d'utilisation
   * Connectez-vous à la console via SSH en utilisant l'adresse IP, puis exécutez le script
    Téléchargez l'outil SSH nécessaire <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">Outil Mobaxterm</Button>


<Tabs>
  <TabItem value="eddy1" label="Utilisation internationale">
    # Veuillez utiliser la commande suivante pour l'international
    ### Voici le début de l'installation du script automatique d'Eddy
    * Ouvrez l'outil SSH que vous avez téléchargé, entrez la commande ci-dessous, appuyez sur Entrée, puis cliquez sur Copier
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Ajoutez la configuration suivante dans le fichier de configuration `moonraker.conf` de la page `fluidd` pour faciliter les mises à jour ultérieures
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Si vous avez mis à jour, exécutez cette commande
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>
