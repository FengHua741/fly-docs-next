# Utilisation de Eddy

    :::info

    * Veuillez noter que pour le balayage du lit, il est recommandé d'utiliser une limite Z séparée, et il n'est pas conseillé d'utiliser le balayage comme limite Z
    * Lors de l'installation de Eddy, la position du fond par rapport à la buse doit être au moins 1-2 mm au-dessus de la buse
    * Lors du calibrage de la fréquence Eddy, le PCB de la bobine doit être à au moins 5 mm du lit chauffant
    * Veuillez ne pas tester le lit chauffant à température ambiante et ne pas l'allumer pendant le test

    :::

## Configuration de référence et guide d'utilisation

### Modification de la fréquence par défaut de ldc1612 1612
    Cette méthode est uniquement utilisée pour sht36-v3/SB2040-v3 ou SB2040-Pro-v3
    Vous pouvez utiliser l'outil Mobaxterm pour exécuter les commandes

    Téléchargez l'outil Mobaxterm nécessaire <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Mobaxterm</Button>

    
    [Veuillez consulter l'utilisation détaillée](/docs/ProductDoc/SBC/fly-pi-v2/ssh)


    Exécutez la première commande et appuyez sur Entrée

    ```cfg
sed -i 's/LDC1612_FREQ = 12000000/LDC1612_FREQ = 40000000/g' ~/klipper/klippy/extras/ldc1612.py
    ```

    Exécutez la deuxième commande et appuyez sur Entrée
  ```cfg
grep "LDC1612_FREQ = 40000000" ~/klipper/klippy/extras/ldc1612.py
  ```
    Si la substitution est réussie, une ligne contenant `LDC1612_FREQ = 40000000` sera renvoyée. Si rien n'est renvoyé, cela signifie que la substitution a échoué.
    Ou utilisez cette commande pour vérifier si la substitution a réussi
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/1612.webp').default} size="110%" align="left" />
```cfg
sed -n '10,15p' ~/klipper/klippy/extras/ldc1612.py
```
   <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/ldc161240mhz.webp').default} size="30%" align="left" />

   Conseil : Après la modification, veuillez utiliser cette commande pour redémarrer le service klipper et recalibrer la fréquence eddy
   ```cfg
   systemctl restart klipper
   ```
   Commande de calibration de la fréquence eddy
   ```cfg
   PROBE_EDDY_CURRENT_CALIBRATE CHIP=fly_eddy_probe
   ```

    ### Configuration du balayage du lit

        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 43
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

        * Veuillez noter qu'il n'y a actuellement pas de module 1612 externe disponible, cette configuration ne peut pas être utilisée pour le moment
        ```cfg
        [probe_eddy_current fly_eddy_probe]
        sensor_type: ldc1612
        z_offset: 1.0
        i2c_address: 42
        i2c_mcu: SB2040
        i2c_bus: i2c1b
        x_offset: 0
        y_offset: 0
        speed:40
        lift_speed: 5
        ```

    ### Configuration de la compensation de température

        ```cfg
        [temperature_probe fly_eddy_probe]
        sensor_type:Generic 3950
        sensor_pin:SB2040:gpio28
        ```

### Configuration de bed_mesh

    * Dans [bed_mesh], le paramètre `horizontal_move_z` est recommandé à 2 pour que Eddy soit le plus proche possible de la plateforme lors du balayage

    ```cfg
    [bed_mesh]
    horizontal_move_z: 2
    ```
