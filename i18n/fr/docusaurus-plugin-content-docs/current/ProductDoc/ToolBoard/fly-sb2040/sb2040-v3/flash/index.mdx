# Instructions de gravage du firmware

    * La version V3 de SB2040 peut être connectée à l'ordinateur hôte via les méthodes RS232 et CAN
    * RS232 et CAN nécessitent l'utilisation d'un câble de données `XT30(2+2)-F`
    * Voici le style du câble de données `XT30(2+2)-F` ci-dessous
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* Le gravage du firmware de la carte mère doit passer par Katapult pour graver Klipper
* Si la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire de faire le dernier gravage de Katapult, c'est juste une précaution au cas où le firmware Katapult ne serait pas correctement gravé
* Il est possible de vérifier rapidement en appuyant deux fois rapidement sur le bouton de réinitialisation si une LED clignote sur la carte mère, ce qui indiquerait que la carte mère est dans le firmware Katapult
:::

## Comment déterminer quel type de firmware vous avez besoin

    * La version V3 de SB2040 peut être connectée à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Gravage du firmware CAN</Button>
        * Avantage : Si vous avez flashé le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher la carte mère et il prend en charge la connexion en bus
        * Inconvénient : L'utilisation conjointe de `KlipperScreen` peut entraîner des erreurs comme des dépassements de temps lors de l'homologation
        * Remarque : Si vous souhaitez utiliser CAN, assurez-vous que l'ordinateur hôte supporte CAN et qu'il faut utiliser UTOC ou une carte mère avec un firmware de pont
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Gravage du firmware RS232</Button>
        * Avantage : Il n'est pas nécessaire d'utiliser un logiciel spécial, la communication est fiable et stable
        * Inconvénient : Pour reprogrammer le firmware, il faut entrer en mode DFU manuel ou double-cliquer sur RST pour entrer dans `Katapult` afin de mettre à jour le firmware
        * Remarque : Vous devez utiliser G2T ou UTOR pour pouvoir utiliser cela
