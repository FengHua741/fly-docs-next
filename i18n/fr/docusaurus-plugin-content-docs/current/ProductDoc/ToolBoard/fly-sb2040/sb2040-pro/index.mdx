# Présentation du produit

    FLY SB2040 PRO est une carte d'outils développée par la société Guangzhou Mellen Electronics Technology Co., Ltd pour l'imprimante Voron Stealthburber. En utilisant cette carte d'outils, quatre câbles peuvent remplacer le câblage complexe entre la carte mère et l'imprimante, optimisant ainsi la disposition. SB2040 convient aux stealthburner et afterburner, elle intègre un transmetteur-récepteur CAN, un port USB, un pilote TMC2240, un capteur d'accélération ADXL345 et d'autres fonctionnalités telles que trois ventilateurs contrôlables. Cette carte d'outils convient aux firmwares Klipper et RRF.

## Caractéristiques du produit

    * Un bouton Boot est prévu, l'utilisateur peut entrer en mode Boot via le port Type-C pour mettre à jour le firmware
    * Support des firmwares Klipper
    * Capteurs de température : puce MAX31865 embarquée, compatible PT100
    * Capteur d'accélération ADXL345 embarqué
    * Interface RGB prévue, pilote 2240 embarqué, supportant divers capteurs de niveau, tels que klicky, Voron Tap, PL08n, etc.
    * Support de tension d'entrée de 12-24V, 15A de courant, augmentation de l'interrupteur de résistance chauffante, courant nominal 10A


## Paramètres du produit


    * MCU : Raspberry Pi rp2040, double cœur ARM Cortex-M0+ @ 133MHz
    * Interfaces de connexion : connecteur XT30 (2+2) supportant 15A de courant, pointeau 30A
    * Firmware : Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs embarqués : ADXL345 et MAX31865 optionnel
    * Interface de résistance chauffante : résistance chauffante (HEAT0)
    * Interfaces d'extension : 3 ventilateurs contrôlables, RGB *1, limiteurs *3, résistance chauffante *1, thermistor *1
    * Interfaces de ventilateur : trois ventilateurs contrôlables
    * Pilote de moteur : pilote TMC2240 embarqué
