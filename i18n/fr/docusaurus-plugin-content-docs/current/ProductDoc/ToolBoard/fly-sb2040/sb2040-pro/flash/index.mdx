# Instructions de gravage du firmware

    * SB2040 PRO peut se connecter à l'ordinateur hôte via la méthode de connexion CAN
    * Le CAN doit être connecté avec `XT30(2+2)-F`
    * Ci-dessous est l'apparence du câble de données `XT30(2+2)-F`
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* Le gravage du firmware de la carte mère doit passer par Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, il n'est pas nécessaire de graver le firmware Katapult à la fin, c'est juste une précaution au cas où la gravure du firmware Katapult n'aurait pas réussi
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si un LED clignote sur la carte mère, s'il y en a un, cela signifie que la carte mère est déjà dans le firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    * SB2040 PRO peut se connecter à l'ordinateur hôte via les méthodes suivantes
    * [Firmware CAN](./can.mdx "Cliquez pour sauter")
        * Avantages : Si vous installez le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher la carte mère et il prend en charge la connexion en bus
        * Inconvénients : Il peut y avoir des erreurs comme le dépassement de temps de retour à zéro lors de l'utilisation conjointe avec `KlipperScreen`
        * Astuce : Si vous avez besoin d'utiliser CAN, assurez-vous que l'ordinateur hôte prend en charge CAN et qu'il faut associer UTOC ou une carte mère avec un firmware de pont installé


## Instructions relatives au firmware Katapult

    * SB2040 PRO est livré avec un firmware Katapult pour la connexion CAN à une vitesse de 1M
