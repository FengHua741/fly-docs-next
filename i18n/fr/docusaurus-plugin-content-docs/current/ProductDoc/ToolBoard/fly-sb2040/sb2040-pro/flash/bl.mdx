# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::tip

    * SB2040_PRO prend en charge CAN
    * Seule la version CAN du firmware est fournie ici, à des vitesses de 500K et 1M
    :::

## Entrer en mode de programmation

    :::tip

    Ce qui est entouré est le `bouton BOOT`

    :::

    * En maintenant enfoncé le bouton BOOT, branchez le câble de transmission de données au motherboard et à votre appareil lorsque le motherboard est complètement déchargé.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro/flash/img/boot.webp').default} size="50%" align="center" />

            4. Programmation du firmware

            :::tip

            Il n'est nécessaire de programmer qu'un seul firmware

            :::

            * Ci-dessous est la commande pour écrire le firmware à une vitesse de CAN de 500K
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_500K.uf2
                ```
                       
            * Ci-dessous est la commande pour écrire le firmware à une vitesse de CAN de 1M
                ```bash
                sudo ~/klipper/lib/rp2040_flash/rp2040_flash ~/FLY_Katapult/BL/SB2040/FLY_SB2040PRO_CANBOOT_1M.uf2
                ```


            5. Si vous voyez ci-dessous, la programmation a réussi.

                <ImageView image={require('@site/docs/General/dfu-flash/img/flash3.webp').default} size="100%" align="center" />


            :::tip

            Après la programmation, il est nécessaire de couper l'alimentation au motherboard une fois.

            :::
