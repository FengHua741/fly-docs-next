# Méthodes de résolution du sous-tensionnement du pilote

## Méthode un

    :::danger

    * Remplacer l'alimentation
    * Ajouter un filtre d'alimentation
    * Vérifier le moteur
    * Branchez la carcasse du moteur sur la GND du panneau de contrôle
    * Ajoutez `driver_TPFD: 0` à la configuration du moteur d'extrusion

    :::

## Méthode deux
    * Connectez-vous via SSH au contrôleur supérieur
    * Entrez la commande ci-dessous
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * Cette étape pourrait être réinitialisée après une mise à jour de klipper
    :::
