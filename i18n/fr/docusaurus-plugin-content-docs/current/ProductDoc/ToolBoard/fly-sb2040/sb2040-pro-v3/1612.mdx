# Utilisation d'Eddy

    :::info
  * Si vous aviez précédemment configuré l'installation d'Eddy, veuillez la supprimer
  * Lors de l'installation d'Eddy, le bobineau doit être au-dessus de la buse et à une distance de 0,6-0,8 cm de celle-ci
  * Ne pas allumer la plaque chauffante lors des tests
  * Supprimez la configuration `probe` dans les paramètres avant utilisation
    :::

    :::warning Remarques
    * Les machines multi-Z nécessitent un niveau manuel une fois effectué
    :::


## Schéma de câblage

    * Le logo doit être orienté vers la plaque chauffante lors de l'installation

## Explications d'utilisation
   * Connectez-vous à l'interface de ligne de commande via SSH en utilisant l'adresse IP, puis exécutez le script
    Téléchargez l'outil SSH nécessaire <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Outil Mobaxterm</Button>


<Tabs>
  <TabItem value="eddy1" label="Utilisation internationale">
    # Veuillez utiliser les commandes suivantes pour l'utilisation internationale
    ### Commencez l'installation du script automatique d'Eddy
    * Ouvrez l'outil SSH que vous avez téléchargé, entrez la commande suivante, appuyez sur Entrée, puis cliquez sur Copier
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Ajoutez la configuration suivante dans le fichier de configuration `moonraker.conf` de la page web de `fluidd` pour faciliter les mises à jour ultérieures
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Si vous mettez à jour, exécutez la commande suivante
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* Si vous voyez l'image suivante, tapez `choisir l'entrée` et appuyez sur Entrée avec le numéro correspondant à votre imprimante, qu'il s'agisse de caractères chinois ou anglais
