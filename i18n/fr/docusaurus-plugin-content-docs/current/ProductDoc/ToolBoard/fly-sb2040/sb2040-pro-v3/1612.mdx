# Utilisation d'Eddy

    :::info
  * Si vous aviez précédemment configuré l'utilisation d'Eddy, veuillez supprimer la configuration.
  * Lors de l'installation d'Eddy, la bobine doit être placée au-dessus de l'embout et à une distance de 0,6-0,8 cm du même.
  * Ne pas allumer le lit chaud lors des tests.
  * Supprimez la configuration `probe` dans les paramètres avant utilisation.
    :::

    :::warning Remarques importantes
    * Pour les machines avec plusieurs axes Z, il est nécessaire de calibrer manuellement une fois.
    :::

## Explications d'utilisation
   * Connectez-vous à l'interface de ligne de commande via SSH en utilisant l'adresse IP, puis exécutez le script.
    Téléchargez l'outil SSH requis <Button variant="contained" disableElevation href="/docs/ResDownload/auxiliary_software/">Outil Mobaxterm</Button>


<Tabs>
  <TabItem value="eddy1" label="Utilisation internationale">
    # Veuillez utiliser la commande ci-dessous pour l'utilisation internationale
    ### Voici la procédure d'installation du script d'automatisation d'Eddy
    * Ouvrez l'outil SSH que vous avez téléchargé, entrez la commande suivante et appuyez sur Entrée pour copier.
    
    ```bash
    rm -rf ./Eddy-Automated-Configuration-Script
    git clone https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
   * Ajoutez la configuration suivante dans le fichier de configuration `moonraker.conf` de la page web de `fluidd`, pour faciliter les mises à jour ultérieures.
    ```bash
    [update_manager Eddy-Automated-Configuration-Script]
    type: git_repo
    channel: dev
    path: ~/Eddy-Automated-Configuration-Script
    origin: https://github.com/xishangyue/Eddy-Automated-Configuration-Script.git
    primary_branch: main
    install_script: eddy1.sh
    managed_services: klipper
    ```
   * Si vous avez mis à jour, exécutez la commande suivante
    ```bash
    sudo chmod +x ./Eddy-Automated-Configuration-Script/eddy1.sh
    ./Eddy-Automated-Configuration-Script/eddy1.sh
    ```
</TabItem>

* Exécutez et appuyez sur Entrée lorsque vous voyez le message comme illustré ci-dessous pour entrer votre mot de passe principal.
<ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36_v3/img/EDDY1.webp').default} size="100%" align="left" />
