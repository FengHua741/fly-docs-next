        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale est la suivante, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier uf2 out/klipper.uf2
        ```

## Entrer en mode de programmation

    :::astuce

    À gauche se trouve le bouton **RESET**, à droite le bouton **BOOT**

    :::

    * Il existe deux méthodes pour entrer en DFU
    * Première méthode : Totalement déconnectez le circuit principal, maintenez le bouton BOOT puis branchez le câble de transmission de données entre le circuit principal et l'hôte
    * Seconde méthode : Branchez le câble de transmission de données entre le circuit principal et l'hôte, maintenez le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST et enfin relâchez le bouton BOOT
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
