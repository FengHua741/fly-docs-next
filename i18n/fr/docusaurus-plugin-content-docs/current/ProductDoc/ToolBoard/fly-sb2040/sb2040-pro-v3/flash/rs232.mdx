    * Si la sortie finale ressemble au contenu ci-dessous, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier uf2 out/klipper.uf2
        ```


## Entrer en mode de mise à jour

    :::tip

    À gauche se trouve le bouton **RESET**, à droite se trouve le bouton **BOOT**

    :::

    * Il existe deux méthodes pour entrer en DFU
    * Première méthode : Dans le cas où la carte mère est complètement débranchée, maintenez enfoncé le bouton BOOT, puis branchez le câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur l'hôte
    * Deuxième méthode : Branchez le câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur l'hôte, maintenez enfoncé le bouton BOOT, puis appuyez sur le bouton RST, relâchez ensuite le bouton RST et enfin relâchez le bouton BOOT
    <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/boot.webp').default} size="100%" align="center" />
