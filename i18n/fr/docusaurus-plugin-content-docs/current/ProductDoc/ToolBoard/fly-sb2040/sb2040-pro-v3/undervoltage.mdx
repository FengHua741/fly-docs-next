# Méthodes de résolution pour la tension d'alimentation insuffisante du driver

## Méthode un

    :::danger

    * Remplacer l'alimentation
    * Ajouter un filtre d'alimentation
    * Vérifier le moteur
    * Branchez la carcasse du moteur sur la GND du panneau de contrôle
    * Ajoutez `driver_TPFD: 0` dans la configuration du driver de l'extrudeuse

    :::

## Méthode deux
    * Connectez-vous via SSH à la machine supérieure
    * Entrez la commande suivante
    ```bash
    sed -i 's/"uv_cp":                    0x01 << 2/#&/' ~/klipper/klippy/extras/tmc2240.py
    ```

    :::danger
    * La réinitialisation de Klipper pourrait annuler cette étape
    :::
