# 1. Présentation du produit

    FLY SB2040 PRO V3 est une carte outil conçue par Guangzhou Meilun Electronics Technology Co., Ltd pour l'imprimante Voron Stealthburber. En utilisant cette carte outil, quatre fils peuvent remplacer les câbles complexes entre la carte mère et l'imprimante, optimisant ainsi la disposition. SB2040 convient aux modes stealthburner et afterburner. Elle intègre un transmetteur-récepteur CAN, un transmetteur-récepteur RS232, un port USB, un pilote TMC2240, un capteur d'accélération LIS2DW et trois ventilateurs contrôlables, entre autres fonctionnalités. Cette carte outil convient aux firmwares Klipper et RRF.

## 1.1 Caractéristiques du produit

    * Un bouton Boot est prévu, permettant aux utilisateurs d'entrer en mode Boot via le port Type-C pour mettre à jour le firmware
    * Prise en charge des firmwares Klipper et RRF
    * Prise en charge des modes CAN, RS232, RRF CAN
    * Dans le mode RS232, elle offre une vitesse de transmission plus élevée et une meilleure stabilité. Dans le mode CAN, elle prend en charge l'extension CAN, avec une interface d'extension CAN réservée
    * Les capteurs de température prennent en charge le module MAX31865, compatible avec les capteurs de température PT100 et PT1000
    * Intègre un capteur d'accélération LIS2DW et un capteur LDC1612
    * Une interface RGB est prévue, avec un pilote 2240 sur la carte, supportant divers capteurs de nivellement, tels que klicky, Voron Tap, PL08n, etc.
    * Prise en charge des tensions d'entrée de 12-24V, 15A, avec une augmentation de la prise de chauffe-buse, courant nominal de 10A

## 1.2 Nouveautés

    * Prise en charge du mode CAN Klipper, RS232 Klipper, RRFCAN
    * Ajout du mode RS232, offrant une vitesse de transmission plus élevée et une meilleure stabilité, facilitant également le débogage
    * Ajout du mode RRF CAN, pouvant être basculé vers une carte outil RRF, améliorant ainsi l'adaptabilité
    * Ajout d'une interface d'extension CAN, facilitant l'extension d'appareils CAN
    * Ajout d'un commutateur de code de dipôle pour la résistance terminale, dont la résistance terminale de 120R peut être choisie via le commutateur, sans nécessité de jumpers
    * Ajout d'un câble de connexion d'alimentation/signal de 1 mètre, conçu en torsadé, offrant une meilleure performance anti-interférence

## 1.3 Spécifications techniques


    * MCU : Raspberry Pi rp2040, double cœur ARM Cortex-M0+ @ 133MHz
    * Interfaces de connexion : connecteur XT30 (2+2) supportant 15A de courant, pointeau 30A
    * Firmware : Reprap/Klipper
    * Tension d'entrée : DC12V-24V
    * Tension logique : DC3.3V
    * Capteurs sur la carte : LIS2DW et optionnels ldc1612 et max31865
    * Interface chauffe-buse : chauffe-buse (HEAT0)
    * Interfaces d'extension : RGB, EndStop, PT100/PT1000, XH2.54-2P CAN
    * Interfaces ventilateurs : un ventilateur DC, deux ventilateurs DC ou ventilateurs PWM
    * Pilote moteur : pilote TMC2240 sur la carte
    * Capteurs de température : un capteur NTC de 100K ou PT1000 (TH0), un capteur PT100/PT1000 optionnel
