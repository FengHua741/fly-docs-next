<Tabs>
    <TabItem value="can" label="Obtention du CANBus UUID">

    * Avant de rechercher l'ID, connectez le panneau d'outils à UTOC ou à une carte mère avec un firmware de pont flashé en utilisant un câble de données XT30. Ci-dessous se trouve un schéma de câblage de référence
    * Vérifiez si les commutateurs sont positionnés correctement
        <ImageView image={require('@site/docs/General/get-id/img/utoc.webp').default} size="80%" align="left" />
    * Assurez-vous que la résistance entre CAN H et CAN L est d'environ `60Ω` en utilisant un multimètre dans une situation sans alimentation. Si c'est `120Ω`, veillez à activer le commutateur de `120Ω`. Si c'est environ `40Ω`, veillez à désactiver le commutateur.
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sb2040/sb2040-pro-v3/flash/img/120.webp').default} size="100%" align="center" />

    * Vérifiez si les commutateurs sont positionnés correctement
