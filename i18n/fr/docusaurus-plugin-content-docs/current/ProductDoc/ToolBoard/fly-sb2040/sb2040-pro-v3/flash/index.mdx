# Instructions pour la programmation de la mise à jour du firmware

    * SB2040 PRO V3 peut être connecté à l'ordinateur hôte via RS232 et CAN.
    * RS232 et CAN nécessitent l'utilisation d'un câble de données `XT30(2+2)-F`.
    * Voici le style du câble de données `XT30(2+2)-F` en dessous.
    <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger À savoir
* La mise à jour du firmware de la carte mère doit passer par la programmation de Katapult pour Klipper.
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, vous n'avez pas besoin de faire la dernière étape de la programmation du firmware Katapult, cela ne sert qu'à prévenir une éventuelle erreur de programmation.
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère. Si c'est le cas, cela signifie que la carte mère est déjà dans le firmware Katapult.
:::

## Comment déterminer quel type de firmware vous avez besoin

    * SB2040 PRO V3 peut être connecté à l'ordinateur hôte de la manière suivante :
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Programmation du firmware CAN</Button>
        * Avantages : Si vous installez le firmware `Katapult`, vous pouvez mettre à jour le firmware sans toucher à la carte mère et il prend en charge la connexion en bus.
        * Inconvénients : L'utilisation conjointe de `KlipperScreen` peut entraîner des erreurs comme des retards de retour à zéro.
        * Remarque : Si vous devez utiliser CAN, assurez-vous que l'ordinateur hôte prend en charge CAN et qu'il faut associer UTOC ou une carte mère avec un firmware de pont.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">Programmation du firmware RS232</Button>
        * Avantages : Il n'est pas nécessaire d'utiliser un logiciel spécial, la communication est fiable et stable.
        * Inconvénients : Pour réinstaller le firmware, vous devez entrer en mode DFU manuellement ou appuyer deux fois sur RST pour entrer dans `Katapult` afin de mettre à jour le firmware.
        * Remarque : Vous devez associer G2T ou UTOR pour l'utiliser.
