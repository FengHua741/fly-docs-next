# A4988

    <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988.webp').default} size="50%" align="center" />

## Paramètres de base

    * Convient pour : Firmware Marlin/Klipper/RRF
    * Protection contre l'arrêt en cas de surchauffe
    * Protection contre les courts-circuits au sol
    * Protection contre les courts-circuits sous charge

## Installation sur la carte mère FLY
    * Veuillez noter qu'il faut aligner EN vers le coin supérieur gauche pour l'installer
    * Les DP5, PRO X10, MMU de FLY ne peuvent utiliser que des pilotes TP
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />

## Explication de l'ajustement du courant et de la subdivision du moteur
    * Explication de l'ajustement du courant du moteur
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/4988-2.webp').default} size="100%" align="center" />
    * Le 4988 utilise le mode de casquette de saut normal, il est possible de configurer 16 subdivisions après avoir mis quatre sauts
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-dip.webp').default} size="100%" align="center" />

## Configuration de référence Klipper
    * Le 4988 n'a pas besoin de configuration de pilote comme `[tmc5160 stepper_x]` ou `[tmc2209 stepper_x]`, il n'est pas nécessaire, la configuration après cela rendra son utilisation impossible
    * Définissez `microsteps:` à `16`
        <ImageView image={require('@site/docs/DebugDoc/faq/img/tmc/cfg.webp').default} size="50%" align="center" />
