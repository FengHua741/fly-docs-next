# Présentation

    * Certains points à prendre en compte concernant les pilotes


## Installation des pilotes

    * Veuillez noter que l'EN doit être aligné avec le coin supérieur gauche pour l'installation
    * Le pied EN de la prise de pilotage du motherboard FLY est situé en haut à gauche
    * Les DP5, PRO X10, MMU de FLY ne peuvent utiliser que les pilotes TP
        <ImageView image={require('@site/docs/ProductDoc/ModuleDrive/img/drive-install.webp').default} size="80%" align="center" />

## Configuration des pilotes

    * La méthode de configuration des pilotes TP et des pilotes standards est la même
    * Le pilote FLY-2240 supporte la connexion UART, mais nécessite une modification manuelle de la résistance
    * Il faut faire attention à la résistance d'échantillonnage du pilote. Une configuration incorrecte pourrait endommager le moteur ou le pilote
