## Introduction
#### Dossier Logic
* Nous faisons 99% de notre programmation de code UI dans le dossier Logic, les autres dossiers ne nécessitent pas d'attention particulière
* gcodeLogic.cc est similaire à main.c, 99% du code UI se trouve ici, les autres fichiers n'ont pas besoin d'être beaucoup regardés

#### Dossier i18n
* Contient les tableaux de traduction pour chaque pays

#### Dossier resources
* Contient les ressources d'image utilisées par l'interface utilisateur

#### Dossier ui
* Contient les fichiers d'interface utilisateur, en double-cliquant dessus ils s'ouvrent
* En double-cliquant sur la flèche en bas à gauche pointant vers "l'œil" caché, on peut ouvrir l'interface cachée
