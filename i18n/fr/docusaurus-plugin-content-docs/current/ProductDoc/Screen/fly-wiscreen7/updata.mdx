##   Mise à jour
    * 1. Renommez le fichier téléchargé en `update.img`
    * 2. Placez le fichier `update.img` dans une carte SD/clé USB et insérez-le dans l'écran. Il peut être inséré pendant que l'écran est allumé.
    * 3. Cochez la case de l'écran, cliquez sur mettre à jour pour terminer la mise à jour
