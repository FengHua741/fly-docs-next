## Configuration Klipper

### Configuration LED

    ```cfg
    [led LED]
    white_pin:PA15
    ```

### Configuration PTC

    ```cfg
    [heater_generic warehouse]
    heater_pin:E_CS
    sensor_type:ATC Semitec 104GT-2
    sensor_pin:PA1
    min_temp:-270
    max_temp:600
    control = pid
    pid_kp = 16.804
    pid_ki = 0.794
    pid_kd = 88.854
    ```

### Configuration FAN

    ```cfg
    [fan_generic fan1]
    pin: FAN1
    ```

    ```cfg
    [fan_generic fan2]
    pin: FAN2
    ```

 

### Gcode personnalisé

    * Le Gcode personnalisé apparaît après avoir fait glisser l'écran vers le bas,
    * Un bouton peut envoyer plusieurs commandes au format Gcode :
    * Exemple :

    ```cfg
    G91 
    G1 X10 F6000
    G90
    ```

    * Format d'entrée sur l'écran : (à partir de la deuxième ligne, ajoutez \n devant chaque commande)

    ```cfg
    G91\nG1 X10 F6000\nG90
    ```



## Utilisation de l'exclusion d'objets

    * Il est nécessaire de remplacer moonraker

    * Veuillez d'abord vous connecter via SSH et vous assurer que vous pouvez accéder à Internet normalement

    1. Sauvegardez les fichiers moonraker

    ```bash
    mv ~/moonraker ~/moonraker-bak
    ```

    2. Arrêtez le service moonraker

    ```bash
    sudo systemctl restart moonraker
    ```

    3. Clonez moonraker

    ```bash
    git clone https://e.coding.net/g-ofpa1390/3D-Printers/moonraker-dev.git -b flylcd-dev moonraker
    ```

    4. Dans le navigateur, accédez à la station supérieure pour trouver `moonraker.conf`, puis redémarrez le système

    ```bash
    [network]
    ```

    5. Ensuite, dans le navigateur, entrez l'adresse ci-dessous, où l'IP doit être remplacée par celle de la station supérieure

    ```bash
    http://ip:7125/server/network/wifi/scan
    ```

* Ceci indique que l'opération est terminée
