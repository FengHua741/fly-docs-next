# Tutoriel pour la modification des paramètres de l'écran
:::tip
Ouvrez votre navigateur et accédez à IP:9999 pour utiliser FLY-Tools.
:::

:::tip
L'interface HDMI embarquée sur la carte mère FLY est un Mini HDMI, veuillez faire attention lors de l'achat du câble HDMI !!!
:::

:::tip
Le logiciel de bord FLY prend en charge FPC one-wire pour `FLY-PI-V2`, `FLY-C8` et `FLY-PRO-X10` ou `FLY_LITE2/2.1`.

:::
    * Pour activer l'écran, vous devez activer l'option HDMI ci-dessous
    * Après avoir accédé à FLY Tools
* La première étape consiste à entrer dans fly_TOOLS pour modifier les paramètres
