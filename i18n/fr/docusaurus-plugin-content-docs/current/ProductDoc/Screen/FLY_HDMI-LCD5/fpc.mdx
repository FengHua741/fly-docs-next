# Tutoriel de modification des paramètres de l'écran
:::tip
Ouvre ton navigateur à l'adresse IP:9999 pour accéder à FLY-Tools
:::

:::tip
L'interface HDMI embarquée sur la carte de contrôle FLY est de type Micro HDMI, fais attention lors de l'achat du câble HDMI !!!
:::

:::tip
Le contrôleur FLY prend en charge la connexion FPC avec `FLY-PI-V2`, `FLY-C8`, `FLY-PRO-X10` ou `FLY_LITE2/2.1`

:::
    * Pour utiliser HDMI, il faut activer l'option ci-dessous pour activer l'écran
    * Une fois dans FLY Tools
* La première étape consiste à entrer dans fly_TOOLS pour modifier les paramètres
