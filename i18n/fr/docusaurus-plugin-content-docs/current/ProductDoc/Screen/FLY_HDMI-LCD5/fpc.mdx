# Tutoriel de modification des paramètres de l'écran
:::tip
Ouvrez votre navigateur à votre IP:9999 pour accéder à FLY-Tools
:::

:::tip
L'interface HDMI intégrée de la carte mère FLY est de type Micro HDMI, faites attention lors de l'achat du câble HDMI !!!
:::

:::tip
Les cartes mères FLY supportant la connexion FPC en une seule ligne sont `FLY-PI-V2`, `FLY-C8`, `FLY-PRO-X10` ou `FLY_LITE2/2.1`

:::
    * Pour utiliser l'HDMI, il faut activer l'option ci-dessous pour activer l'écran
    * Après avoir accédé à FLY Tools
* La première étape consiste à entrer dans fly_TOOLS pour modifier les paramètres
