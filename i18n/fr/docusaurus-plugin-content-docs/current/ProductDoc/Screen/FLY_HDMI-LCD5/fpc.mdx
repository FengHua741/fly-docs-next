# Tutoriel pour modifier les paramètres de l'écran
:::tip
Ouvrez votre navigateur et accédez à votre IP:9999 pour accéder à FLY-Tools
:::

:::tip
L'interface HDMI intégrée à la carte mère de FLY est de type Micro HDMI, veuillez faire attention lors de l'achat d'un câble HDMI !!!
:::

:::tip
Les machines hôtes FLY supportant la connexion FPC sont `FLY-PI-V2`, `FLY-C8`, `FLY-PRO-X10` et `FLY_LITE2/2.1`

:::
    * Pour utiliser HDMI, il est nécessaire d'activer l'option ci-dessous pour activer l'écran
    * Une fois dans FLY Tools
* La première étape consiste à entrer dans fly_TOOLS pour modifier les paramètres
