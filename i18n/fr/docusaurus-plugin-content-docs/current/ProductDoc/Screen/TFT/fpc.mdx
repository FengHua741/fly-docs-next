# Tutoriel pour modifier les paramètres de l'écran
:::tip
Ouvrez votre navigateur sur votre IP:9999 pour accéder à FLY-Tools
:::

:::
    * Pour utiliser l'écran tft, vous devez activer l'option ci-dessous
    * Après avoir accédé à FLY Tools
* La première étape consiste à entrer dans fly_TOOLS pour modifier les paramètres

* Vous devez configurer l'écran dans FLY TOOLS et activer KS
