## Présentation du produit

    * Taille de l'écran : 120.7mm*76.0mm
    * Zone d'affichage de l'écran : 108mm*66.2mm
    * Résolution : 800*480
    * Type d'écran : IPS
    * Méthode de communication : DSI/Mipi
    * Interfaces embarquées : 15 broches DSI / 22 broches Mipi
    * Température de fonctionnement stable : 0°-60°C
    * Type de tactile : Capacitive (capacitif)
    * Tension logique : 3.3V
    * Poids : 135.5g

        :::danger Informations importantes

        * Les interfaces DSI et CSI du Raspberry Pi 5 ont maintenant été renommées en MIPI
        * L'interface MIPI peut être utilisée directement pour utiliser une caméra CSI ou un écran DSI
        * Veuillez noter la direction des broches dorées

        :::

        * Le Raspberry Pi 5 dispose maintenant de deux interfaces MIPI, si vous regardez de face, vous pouvez voir qu'il y a des marques 1 et 0

        * Câblage de l'écran DSI


            * Si vous utilisez l'interface 1, veuillez configurer l'écran comme suit

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi1
            ```

            * Si vous utilisez l'interface 0, veuillez configurer l'écran comme suit

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
