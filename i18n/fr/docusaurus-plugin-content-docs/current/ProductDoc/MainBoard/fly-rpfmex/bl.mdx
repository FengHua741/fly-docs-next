# Firmware Katapult



## Entrer en mode de flashage

    :::tip

    Souligné est `BOUTON BOOT`

    :::

    * Pour entrer en mode de flashage, il faut que la carte mère soit complètement débranchée. Ensuite, maintenez enfoncée la touche BOOT puis branchez le câble de transfert de données d'un côté sur la carte mère et de l'autre côté sur votre appareil.
