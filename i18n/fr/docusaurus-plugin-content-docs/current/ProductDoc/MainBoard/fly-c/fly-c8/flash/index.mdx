# Notice sur la programmation de la mise à jour du firmware
    
## Comment savoir quel type de firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware.

    :::
    
    * C8 peut être connecté à l'hôte via les méthodes suivantes :
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont présents, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware de pont USB</Button>
        * Avantage : il est possible de connecter plusieurs appareils sans occuper de ports USB
        * Inconvénient : l'utilisation conjointe avec `KlipperScreen` peut entraîner des erreurs telles que le dépassement du délai de retour à zéro.
