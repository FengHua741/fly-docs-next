# BL firmware

    :::tip

    * Veuillez noter que le bootloader est préinstallé en usine et n'a pas besoin d'être réinstallé.

    :::

## Entrée en mode de programmation


    * Il existe deux méthodes pour entrer en mode de programmation.
    * Première méthode : Désactivez complètement l'alimentation du circuit mère, maintenez enfoncée la touche BOOT, puis branchez un câble de transmission de données entre la carte mère et votre appareil, adapté à WINDOWS.
    * Deuxième méthode : Après avoir branché un câble de transmission de données entre la carte mère et votre appareil, maintenez enfoncée la touche BOOT, puis saisissez une commande, adapté à l'appareil supérieur.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.webp').default} size="100%" align="center" />

            * Déplacez les commutateurs `1`, `2` vers la droite pour connecter le périphérique inférieur avec le périphérique supérieur via USB intégré.
            * Déplacez les commutateurs `3`, `4` vers la gauche, sinon cela affectera la connexion entre le périphérique inférieur et le périphérique supérieur.
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
            * Après avoir déplacé les commutateurs selon cette méthode, le port USB en bas sera occupé par le périphérique inférieur, veuillez ne pas y brancher d'autres appareils.
                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />

            * Assurez-vous que le périphérique supérieur peut accéder à Internet


                ```bash
                sudo wget https://cdn.mellow.klipper.cn/BL/HIDBL.bin -O ~/HIDBL.bin && sudo fly-flash -d c8 -u -f ~/HIDBL.bin
                ```
            * Lorsque vous voyez le message suivant, appuyez longtemps sur le bouton BOOT du périphérique inférieur puis appuyez sur Entrée.

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.webp').default} size="80%" align="center" />
