## Préparation avant la gravure du système

    * Lecteur de cartes
    * Carte TF d'une capacité minimale de 16 Go ou maximale de 128 Go, la vitesse de transmission de la carte TF doit être de classe 10 ou supérieure
    * Télécharger l'image système correspondante <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-c8">image système</Button>
    * Télécharger l'outil de gravure nécessaire <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">outil de gravure</Button>

## Gravure du système

:::info
La gravure sur carte TF et la gravure via carte TF sur M2WE nécessitent seulement l'utilisation d'une méthode de gravure.
:::

:::info
* Si le machine est installé avec M2WE et une carte mémoire, le système démarre automatiquement à partir de la carte mémoire.
* Il est recommandé d'utiliser une carte TF pour graver le système sur M2WE, car la gravure via USB peut échouer en raison de conflits de pilotes ; il est conseillé d'utiliser directement [gravure de carte mémoire sur M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os).
* Veuillez vérifier que l'image système téléchargée correspond bien, sinon elle ne pourra pas démarrer.
:::

## Installation des antennes

    1. Installer les antennes pour le module `M2WE` ou le module `WiFi 2.4G`
    :::warning
    * L'ordinateur principal FLY nécessite l'utilisation du module `M2WE` ou du module `WiFi 2.4G`, sans quoi le WiFi ne fonctionnera pas.
    * Veuillez noter que l'ordinateur principal ne supporte pas les noms de réseau WiFi en chinois, ni les réseaux WiFi dual-band (2.4G et 5G).
    :::
    * La flèche rouge indique l'emplacement de l'interface de l'antenne WiFi
    * L'interface de l'antenne WiFi est de type `ipex1`
    * Veuillez installer l'antenne sur M2WE, sinon une mauvaise connexion au routeur pourrait résulter d'un signal faible
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - Une fois la carte système installée, vous pouvez passer à l'étape suivante : connexion SSH à l'ordinateur principal
    :::

    2. Alimenter l'ordinateur principal.  
    **Ici, votre ordinateur principal peut maintenant démarrer !!!**  
    Veuillez passer à l'étape suivante : <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">Connexion SSH à l'ordinateur principal</Button>
