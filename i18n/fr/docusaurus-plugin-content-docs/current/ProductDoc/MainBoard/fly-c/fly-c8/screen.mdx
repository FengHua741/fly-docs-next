    <TabItem value="tft" label="Configuration de l'écran tactile TFT" default>
## Méthode un    
         * La configuration tactile nécessite d'utiliser la commande suivante pour configurer en `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Vérifiez que le commutateur à glissière sur l'arrière de l'écran est positionné sur `Cap`
    * Utilisez la commande suivante pour vérifier si la modification a été effectuée avec succès
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Pour modifier la configuration tactile en résistive, utilisez la commande suivante

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

        <TabItem value="hdmi" label="Configuration de l'écran résistif TFT">
## Méthode un        
           * La configuration résistive nécessite d'utiliser la commande suivante pour configurer en `fly-tft-v2-r` 
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
```
                                * Vérifiez que le commutateur à glissière sur l'arrière de l'écran est positionné sur `Resi`
        * Utilisez la commande suivante pour vérifier si la modification a été effectuée avec succès
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
                                    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-r.webp').default} size="80%" align="center" />
 * Pour modifier la configuration résistive en tactile, utilisez la commande suivante

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v2-c/}' /boot/FlyOS-Env.txt
```
