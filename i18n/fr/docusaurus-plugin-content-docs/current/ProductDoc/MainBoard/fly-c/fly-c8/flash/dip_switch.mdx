# Présentation des commutateurs dip

:::tip
FLY Gemini se connecte à la station de travail supérieure via les commutateurs dip. Si leur position est incorrecte, elle ne peut pas être utilisée correctement.
::

:::danger À savoir
Veuillez noter que les commutateurs dip doivent être placés à des positions spécifiques pour une connexion correcte avec la station de travail supérieure.
::

## Connexion à la station de travail embarquée

    * Les commutateurs dip `1`, `2` vers la droite, pour connecter le sous-système inférieur et le supérieur via USB intégré
    * Les commutateurs dip `3`, `4` vers la gauche, sinon cela affectera la connexion entre le sous-système inférieur et le supérieur
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * Après avoir déplacé les commutateurs dip selon cette méthode, le port USB en bas sera occupé par le sous-système inférieur. Veuillez ne pas connecter d'autres appareils.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Connexion à une station de travail externe

    * Les commutateurs dip `3`, `4` vers la droite, pour connecter le sous-système inférieur et le supérieur via USB intégré
    * Les commutateurs dip `1`, `2` vers la gauche, sinon cela affectera la connexion entre le sous-système inférieur et le supérieur
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
