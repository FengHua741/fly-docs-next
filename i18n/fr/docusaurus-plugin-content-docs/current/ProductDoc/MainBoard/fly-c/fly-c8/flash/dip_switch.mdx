# Introduction aux commutateurs DIP

:::tip
FLY c8 se connecte à l'ordinateur principal via les commutateurs DIP ; si la position des commutateurs n'est pas correcte, il ne pourra pas être utilisé normalement.
:::

:::danger À savoir
Veuillez noter que les commutateurs DIP doivent être placés dans la position spécifiée pour une connexion correcte avec l'ordinateur principal.
:::

## Connexion de l'ordinateur principal embarqué

    * Placer les commutateurs DIP `1` et `2` vers la droite pour connecter le sous-système et l'ordinateur principal via USB intégré.
    * Placer les commutateurs DIP `3` et `4` vers la gauche, sinon cela affectera la connexion entre le sous-système et l'ordinateur principal.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * Après avoir ajusté les commutateurs DIP selon cette méthode, le port USB en dessous sera occupé par le sous-système ; veuillez ne pas brancher d'autres appareils.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Connexion d'un ordinateur principal externe

    * Placer les commutateurs DIP `3` et `4` vers la droite pour connecter le sous-système et l'ordinateur principal via USB intégré.
    * Placer les commutateurs DIP `1` et `2` vers la gauche, sinon cela affectera la connexion entre le sous-système et l'ordinateur principal.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
