# Flashage du firmware CAN via un pont USB

## Début de la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Appuyez sur `Q`, apparaît **Save configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Appuyez sur `Q`, apparaît **Save configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous êtes revenu à l'interface de commande
    * Entrez la commande ci-dessous pour commencer la compilation, cela prend un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Une sortie similaire au texte suivant indique que la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```



## Flashage du firmware

:::tip
Exécutez la commande ci-dessous pour flasher le firmware
:::

    ```bash
    sudo fly-flash -d c8 -h -f ~/klipper/out/klipper.bin
    ```
