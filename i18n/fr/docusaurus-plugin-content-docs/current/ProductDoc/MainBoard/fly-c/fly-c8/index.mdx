## Caractéristiques du produit

    * Utilise un contrôleur principal STM32F407VGT6 de série ARM Cortex-M0+ cadencé à 168 MHz
    * Intègre des modes de fonctionnement SPI et UART avec des pilotes TMC
    * Intègre une broche DIAG, permettant d'utiliser l'ajustement sans fin simplement en branchant ou débranchant un capot de saut
    * Ventilateur numérique avec tension de 24V, 12V et 5V disponibles, évitant ainsi l'achat de modules de dérivation
    * Nouvelle méthode d'écriture Flash, permettant de mettre à jour le firmware de la carte mère via FLY-Tools
    * Gestion multi-courant, les courts-circuits sur les ports, les limites ou les moteurs n'affectent pas le fonctionnement de l'ordinateur supérieur
    * Interface RGB réservée avec tension de 12V ou 5V disponible
    * Interface SPI réservée, permettant à l'utilisateur d'ajouter un capteur d'accélération externe pour une compensation d'accélération
    * Prise en charge d'un écran TFT et d'un écran HDMI en mode "one-wire"
    * Le système préinstalle klipper, moonraker, mainsail, fulidd, klipper-screen, crowsmest, plugins de compensation de résonance, numpy et FLY-Tools, sans nécessiter des opérations complexes comme le changement de source, facilitant l'utilisation pour les débutants (nécessite l'achat d'une carte SD ou M2WE)

## Spécifications du produit

### Ordinateur supérieur
    * CPU : Processeur Allwinner H5 à haute performance, 4 cœurs 64-bit Cortex-A53
    * GPU : GPU à haute performance 6 cœurs Mali 450, taux de remplissage de pixels supérieur à 2,7 gpixel/s
    * RAM : 1 Go DDR3 (partagé avec le GPU)
    * ROM : Supporte jusqu'à 128 Go de carte SD
    * Périphériques : Spi x1, TFT x1, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1, interface audio 3,5 mm x1
    * Interfaces : Connecteur M.2 embarqué, pouvant être étendu pour le WIFI, eMMC, etc. (protocole privé, ne pas connecter des appareils M.2 non spécifiquement conçus pour FLY π)
    * Réservé une interface de ventilateur 5V contrôlable ZH1.5, utilisable pour le refroidissement de l'ordinateur supérieur
    * Prise en charge d'un écran TFT et d'un écran HDMI en mode "one-wire"
    * Interface CAN réservée pour le pontage
    * Le système préinstalle klipper, moonraker, mainsail, fulidd, klipper-screen, crowsmest, plugins de compensation de résonance, numpy et FLY-Tools, sans nécessiter des opérations complexes comme le changement de source, facilitant l'utilisation pour les débutants (nécessite l'achat d'une carte SD ou M2WE)

### Ordinateur inférieur

    * MCU : STM32F407VGT6
    * Firmware :  KLIPPER/RRF/Marlin
    * Alimentation directe 12-24V DC, plus stable, évitant l'achat de modules de dérivation
    * Courant de sortie maximal pour la plaque chauffante :  15A
    * Courant de sortie maximal pour la résistance de chauffage :  6A
    * Courant de sortie maximal pour les ventilateurs :  1A
    * Modes de fonctionnement des moteurs pris en charge : SPI, UART, STEP/DIR
    * Moteurs pris en charge : supporte les moteurs externes TMC5160Pro, TMC5160, TMC2209, TMC2225, TMC2226, TMC2208, TMC2130, A4988, etc.
    * Interfaces d'extension : BLTouch (Servos, Probe), RGB x2, UART x1, 12864 x1
    * Ventilateurs : 6 ventilateurs numériques à deux lignes (support 24V, 12V, 5V) disponibles
    * Capteurs de température : 3 capteurs NTC 100K
    * Structures de machine prises en charge : Cartesian, Delta, Kossel, Ultimaker, CoreXY
