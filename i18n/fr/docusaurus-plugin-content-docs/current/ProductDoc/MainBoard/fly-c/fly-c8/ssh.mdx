# Connexion à la station supérieure via SSH


* Vous pouvez vous connecter à l'appareil via le port série ou le WIFI
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Veuillez vous assurer que la carte SD ou M2WE est gravée avec **le système correspondant à la station supérieure**
* **Si la station supérieure ne démarre pas correctement, veuillez retirer les périphériques tels que les pilotes, les limites, les ventilateurs, etc. !!!**
* **Enfin, veuillez vous connecter à la station supérieure via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, veuillez consulter le tutoriel sur la connexion à la station supérieure via le WIFI ou le câble réseau**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utilisation de l'adresse IP réseau pour se connecter à la station supérieure SSH

        * Connectez-vous à l'interface arrière de la **routeur** auquel la station supérieure est connectée
        * Assurez-vous que le système est **démarré normalement**
        * Assurez-vous que vous connaissez **le nom d'utilisateur et le mot de passe de la station supérieure**, le nom d'utilisateur par défaut pour le système FLY est **fly**, le mot de passe par défaut est **mellow**
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter à SSH via l'adresse IP
        * Ouvrez MobaXterm, effectuez les paramétrages suivants, entrez l'adresse IP que vous avez vue précédemment, ou vous pouvez la trouver dans l'interface arrière de la routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Remarque : ici, à l'étape 4, entrez le nom d'utilisateur de la station supérieure**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'affiche, cliquez sur `Accepter` pour continuer

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe, puis appuyez sur Entrée

        **Remarque : ici, le mot de passe est invisible lors de la saisie, appuyez simplement sur Entrée après la saisie**

        Le nom d'utilisateur par défaut pour le système FLY est : `fly` , le mot de passe par défaut est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />
