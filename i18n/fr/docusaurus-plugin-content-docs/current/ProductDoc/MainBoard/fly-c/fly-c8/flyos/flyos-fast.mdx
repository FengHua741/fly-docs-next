## Installation des antennes

    1. Installer les antennes pour le module `M2WE` ou `WiFi 2,4G`
    :::warning
    * Le logiciel supérieur FLY nécessite l'utilisation du module `M2WE` ou `module WiFi 2,4G`, sinon il ne sera pas possible d'utiliser le WiFi
    * Veuillez noter que le logiciel supérieur ne supporte pas les noms de réseau WiFi en chinois, et ne supporte pas le WiFi dual-band (2,4G et 5G fusionnés)
    :::
    * L'endroit pointé par la flèche rouge correspond à l'interface de l'antenne WiFi
    * L'interface de l'antenne WiFi est `ipex1`
    * Veuillez installer l'antenne sur M2WE, sinon, en raison d'un signal faible, il pourrait être impossible de se connecter au routeur
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - Une fois la carte système installée, vous pouvez passer à l'étape suivante : connexion SSH au logiciel supérieur
    :::

    2. Alimenter le logiciel supérieur.  
    **Ici, votre logiciel supérieur peut déjà être démarré !!!**  
    Veuillez exécuter l'étape suivante : <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c8/ssh">Connexion SSH au logiciel supérieur</Button>
