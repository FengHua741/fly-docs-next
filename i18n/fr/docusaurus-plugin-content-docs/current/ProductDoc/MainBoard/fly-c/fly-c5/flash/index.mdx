# Notice sur la gravure du firmware

    
## Comment savoir quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul firmware

    :::
    
    * Le C5 peut utiliser les méthodes suivantes pour se connecter à l'ordinateur hôte
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Firmware USB</Button>
        * Avantage : Communication fiable et stable
        * Inconvénient : Peut affecter la transmission des données si plusieurs périphériques USB sont utilisés
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware USB Bridge</Button>
        * Avantage : Peut se connecter à plusieurs appareils sans occuper de ports USB
        * Inconvénient : Peut provoquer des erreurs telles que le dépassement de temps d'attente lors de l'utilisation conjointe avec `KlipperScreen`.
