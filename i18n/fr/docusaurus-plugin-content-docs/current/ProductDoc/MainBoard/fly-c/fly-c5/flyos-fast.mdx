## Installation des antennes

    1. Installer l'antenne pour le module `M2WE` ou le module `WiFi 2,4G`
    :::warning
    * Le PC supérieur nécessite l'utilisation du module `M2WE` ou du module `WiFi 2,4G`, sinon il ne peut pas utiliser le WiFi
    * Veuillez noter que le PC supérieur ne prend pas en charge les noms de réseau WiFi en chinois et ne prend pas en charge le WiFi dual-band (2,4G et 5G fusionné)
    :::
    * L'endroit indiqué par la flèche rouge est l'interface de l'antenne WiFi
    * L'interface de l'antenne WiFi est `ipex1`
    * Veuillez installer l'antenne sur le M2WE, sinon une mauvaise qualité de signal peut entraîner une déconnexion du routeur
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - Une fois la carte système installée, vous pouvez passer à l'étape suivante : connexion au PC supérieur via SSH
    :::

    2. Alimenter le PC supérieur.  
    **Ici, votre PC supérieur est maintenant démarré !!!**  
    Veuillez passer à l'étape suivante : <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-c/fly-c5/ssh">Connexion SSH au PC supérieur</Button>
