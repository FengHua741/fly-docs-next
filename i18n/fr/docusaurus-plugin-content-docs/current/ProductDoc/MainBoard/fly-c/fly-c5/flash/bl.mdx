# Firmwares BL

    :::tip

    * Veuillez noter que le "bootloader" est préinstallé en usine, il n'est pas nécessaire de le réinstaller.

    :::

## Mode de démarrage en mode de programmation


    * Il existe deux méthodes pour entrer en mode de programmation
    * Première méthode : Sous tension complète du motherboard, maintenez enfoncé BOOT puis branchez un câble de transmission de données d'une extrémité sur la motherboard et de l'autre sur votre appareil, compatible avec WINDOWS
    * Seconde méthode : Après avoir branché un câble de transmission de données d'une extrémité sur la motherboard et de l'autre sur votre appareil, maintenez enfoncé BOOT puis entrez la commande, compatible avec le PC
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/flash/img/boot.webp').default} size="100%" align="center" />

                ```bash
                sudo wget http://cdn.mellow.klipper.cn/BL/c5.bin -O ~/c5.bin && sudo fly-flash -d c5 -u -f ~/c5.bin
                ```
            * Lorsque vous obtenez le message suivant, appuyez longuement sur le bouton BOOT du périphérique inférieur puis appuyez sur Entrée
