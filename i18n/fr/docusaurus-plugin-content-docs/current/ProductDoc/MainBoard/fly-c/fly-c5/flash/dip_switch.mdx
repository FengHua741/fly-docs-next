# Présentation du dip-switch

:::tip
FLY c8 se connecte à l'ordinateur hôte via le dip-switch. Si la position du dip-switch n'est pas correcte, il ne pourra pas être utilisé normalement.
:::

:::danger À savoir
Veuillez noter que le commutateur dip-switch doit être positionné correctement pour une connexion correcte avec l'ordinateur hôte.
:::
