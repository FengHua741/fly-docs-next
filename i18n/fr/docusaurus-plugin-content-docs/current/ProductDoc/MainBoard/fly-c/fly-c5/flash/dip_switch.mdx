# Présentation des commutateurs DIP

:::tip
FLY c8 se connecte à la station de base via les commutateurs DIP. Si la position des commutateurs n'est pas correcte, elle ne pourra pas fonctionner normalement.
:::

:::danger Avertissement
Veuillez noter que les commutateurs DIP doivent être placés dans les positions spécifiées pour une connexion correcte avec la station de base.
:::

## Connexion de la station de base intégrée

    * Placer les commutateurs DIP `1` et `2` vers la droite pour connecter le périphérique inférieur à la station de base via USB intégré.
    * Placer les commutateurs DIP `3` et `4` vers la gauche, sinon cela affectera la connexion entre le périphérique inférieur et la station de base.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip1.webp').default} size="100%" align="center" />
    * Après avoir ajusté les commutateurs DIP selon cette méthode, le port USB en bas sera occupé par le périphérique inférieur. Veuillez ne pas brancher d'autres appareils.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="100%" align="center" />


## Connexion de la station de base externe

    * Placer les commutateurs DIP `3` et `4` vers la droite pour connecter le périphérique inférieur à la station de base via USB intégré.
    * Placer les commutateurs DIP `1` et `2` vers la gauche, sinon cela affectera la connexion entre le périphérique inférieur et la station de base.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/dip2.webp').default} size="100%" align="center" />
