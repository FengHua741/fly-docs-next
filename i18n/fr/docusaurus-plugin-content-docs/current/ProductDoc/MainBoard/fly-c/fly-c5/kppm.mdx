# Fonction de reprise d'impression après coupure d'alimentation de Klipper

- La fonction de reprise d'impression après coupure d'alimentation de Klipper consiste à permettre à Klipper, après une coupure d'alimentation et une nouvelle mise sous tension, de restaurer automatiquement l'état d'impression de Klipper.
- C5 peut être utilisé sans KPPM
- Non compatible avec les modèles dont l'axe Z se déplace en cas de panne de courant.
