# Compilation du firmware USB


## Vérification des commutateurs

    :::danger À savoir
    - Lorsque vous l'utilisez normalement, veillez à ce que le `commutateur soit positionné vers le haut`
    - Lorsque vous effectuez les changements de commutateur, assurez-vous qu'il est positionné complètement vers le haut, sinon il pourrait ne pas être possible de se connecter au sous-système !!!
    :::
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/dip_switch.webp').default} size="60%" align="center" />


## Connexion SSH à la station de travail principale

    :::danger Avertissement
    - Avant de compiler le firmware, il est nécessaire d'utiliser `le wifi pour vous connecter via SSH à la station de travail principale`
    :::

    ### Programmation du firmware USB
        :::tip Astuce
        - Dans le système FLYOS-FAST, un firmware USB précompilé est disponible, il n'est donc pas nécessaire de compiler manuellement le firmware, exécutez simplement la commande ci-dessous pour le programmer
        - Vous pouvez également choisir de `compiler manuellement le firmware USB`, pour compiler le firmware vous-même
        :::

        * Connectez-vous via SSH à la station de travail principale et entrez la commande suivante
        ```bash
        fly-flash -d c5 -h -f /usr/lib/firmware/klipper-h723-128k-usb.bin
        ```
