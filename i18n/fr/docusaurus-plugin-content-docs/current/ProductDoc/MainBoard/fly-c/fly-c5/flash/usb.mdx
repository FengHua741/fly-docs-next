    ### Programmation du firmware USB
        :::tip Conseil
        - Dans le système FLYOS-FAST, un firmware USB précompilé est disponible, il n'est donc pas nécessaire de compiler manuellement le firmware, exécutez simplement la commande ci-dessous pour le flasher
        - Vous pouvez également choisir de `compiler manuellement le firmware USB`, compiler vous-même le firmware
        :::

        * Connectez-vous via SSH à la station de travail et entrez la commande suivante
        ```bash
        fly-flash -d c5 -h -f /usr/lib/firmware/klipper-h723-128k-usb.bin
        ```
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/flash/img/usb.webp').default} size="80%" align="center" />

    ## Compilation manuelle du firmware USB

        <MakeOperate/>
    
        <USB />
    
        * Appuyez sur `Q`, puis apparaît `Enregistrer la configuration`, appuyez alors sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    
        * Maintenant, la configuration est sauvegardée et vous êtes revenu à l'interface de ligne de commande
    
        * Entrez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/flyos-fast/img/firmware_build1.webp').default} size="60%" align="center" />
    
        * Enfin, si la sortie ressemble au contenu ci-dessous, la compilation a réussi
        <ImageView image={require('@site/docs/General/flyos-fast/img/firmware_build2.webp').default} size="60%" align="center" />
        ---
        
        ### Flashage du firmware
        
        :::tip
        Exécutez la commande suivante pour flasher le firmware
        :::
        
            ```bash
            fly-flash -d c5 -h -f ~/klipper/out/klipper.bin
            ```
