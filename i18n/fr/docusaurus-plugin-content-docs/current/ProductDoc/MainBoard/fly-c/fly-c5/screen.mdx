# Configuration et câblage de l'écran

## Méthode de configuration


- Entrez l'adresse IP du périphérique dans la barre d'URL du navigateur, par exemple : `http://192.168.6.179`
- Accédez à la page de configuration
    - Ouvrez l'adresse IP du périphérique dans le navigateur, par exemple : `http://192.168.1.2/`
    - Dans fluidd, décochez comme indiqué à gauche dans l'image ci-dessous -> `Filtrer les fichiers et dossiers cachés`.
    - Dans mainsail, cochez comme indiqué à droite dans l'image ci-dessous -> `Afficher les fichiers cachés`.

    <table>
      <tr>
        <td>
          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config1.webp').default} size="100%" align="center" />
        </td>
        <td>
          <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config2.webp').default} size="85%" align="center" />
        </td>
      </tr>
    </table>

    - Vous verrez alors le dossier `.flyos-config`. Entrez dans ce dossier pour trouver le fichier `sys-config.conf`.
    - Le fichier `sys-config.conf` est un lien symbolique vers le fichier de configuration `config.txt` du disque amovible `FlyOS-Conf`.

    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/config3.webp').default} size="60%" align="center" />

    - Ouvrez le fichier `sys-config.conf`, trouvez `screen=` et modifiez le type d'écran.
    - Modifiez le numéro de séquence ❶ `klipper_screen` après le `0` en `1` pour activer le service `klipper_screen`.
    - Modifiez l'élément de configuration ❷ après `screen` avec celui nécessaire, puis sauvegardez et fermez. Redémarrez ensuite le système.
    <ImageView image={require('@site/docs/General/flyos-fast/img/screen.webp').default} size="90%" align="center" />

    - Remarque : Le format des éléments de configuration dans le fichier `sys-config.conf` est `clé=valeur`, ne le modifiez pas en `clé: valeur`.
    - Après avoir modifié et enregistré, redémarrez le système pour que cela prenne effet.

## Branchement de l'écran C5

    :::danger À savoir
    - Éteignez le courant avant de connecter l'écran pour éviter d'endommager celui-ci.
    - Lors du branchement de l'écran, assurez-vous que la direction du fil fpc est correcte pour éviter d'endommager l'écran.
