# Configuration et câblage de l'écran

## Configuration et câblage de l'écran C5
Configuration et câblage de l'écran tft
:::tip
- Assurez-vous d'éteindre l'alimentation avant de connecter l'écran pour éviter tout dommage.
- Lors du câblage de l'écran, veillez à la direction de la ligne fpc pour éviter tout dommage.
:::

<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/screen.webp').default} size="80%" align="left" />

- Accédez à l'appareil via une adresse IP, par exemple : `http://192.168.1.1`
- Accédez à la page de configuration
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/screen1.webp').default} size="80%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/screen2.webp').default} size="80%" align="left" />
