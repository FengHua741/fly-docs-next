# Connexion au WiFi

## Installer l'antenne sur le C5

* L'endroit pointé par la flèche rouge correspond à l'interface de l'antenne WiFi
* L'interface de l'antenne WiFi est `ipex1`
* Installez l'antenne sur le C5, sinon, vous pourriez ne pas être en mesure de vous connecter en raison d'une mauvaise réception du signal
