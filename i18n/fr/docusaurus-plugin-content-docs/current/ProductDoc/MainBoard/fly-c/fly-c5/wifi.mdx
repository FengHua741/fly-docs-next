# Connexion au WiFi

## Installer l'antenne sur C5

* L'endroit pointé par la flèche rouge correspond à l'interface de l'antenne WiFi
* L'interface de l'antenne WiFi est `ipex1`
* Installez l'antenne sur C5, sinon une mauvaise réception du signal pourrait empêcher la connexion à Internet
