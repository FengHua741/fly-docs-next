# Connexion au WIFI

## Installer l'antenne sur le C5

* L'endroit pointé par la flèche rouge correspond à l'interface de l'antenne WIFI
* L'interface de l'antenne WIFI est `ipex1`
* Installez l'antenne sur le C5, sinon, vous pourriez ne pas pouvoir vous connecter en raison d'un signal faible.
