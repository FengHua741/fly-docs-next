# Connexion au WiFi

## Installation de l'antenne sur C5

* L'endroit pointé par la flèche rouge correspond à l'interface de l'antenne WiFi
* L'interface de l'antenne WiFi est `ipex1`
* Installer l'antenne sur C5, sinon il pourrait être impossible de se connecter à cause d'un signal faible
