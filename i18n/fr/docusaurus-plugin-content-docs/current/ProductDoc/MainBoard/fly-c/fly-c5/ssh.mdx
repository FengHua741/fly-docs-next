# Connexion à la station de travail SSH

* Vous pouvez utiliser le port série ou Wi-Fi pour vous connecter à l'appareil.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre outil de terminal SSH.
* Assurez-vous que la carte SD ou M2WE est gravée avec **le système correspondant à la station de travail**.
* **Si la station de travail ne démarre pas correctement, veuillez retirer les périphériques externes tels que le driver, les limites et les ventilateurs !**
* **Enfin, connectez-vous via le réseau pour compiler les firmwares !**
* **Pour compiler des firmwares, consultez le tutoriel sur la connexion à la station de travail via Wi-Fi ou Ethernet.**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utiliser une connexion réseau IP pour accéder au SSH de la station de travail

        * Connectez-vous à l'interface web de la **box du routeur**.
        * Assurez-vous que le système est **correctement démarré**.
        * Assurez-vous de connaître le nom d'utilisateur et le mot de passe de la station de travail, par défaut FLY utilise `fly` comme nom d'utilisateur et `mellow` comme mot de passe.
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter via IP au SSH.
        * Ouvrez MobaXterm et effectuez les paramétrages suivants, en entrant l'IP que vous avez vue précédemment, ou consultez-la dans l'interface web du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Remarque : Dans cette étape 4, entrez le nom d'utilisateur de la station de travail.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre de confirmation apparaîtra, cliquez sur `Accepter` pour continuer.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe demandé, puis appuyez sur Entrée.

        **Remarque : Le mot de passe que vous saisissez n'est pas visible, appuyez directement sur Entrée.**

        Par défaut, FLY utilise le nom d'utilisateur `fly` et le mot de passe `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * La connexion a réussi.
