# Connexion à la station de travail SSH

:::danger Attention
- Si votre station de travail est allumée pour la première fois après avoir flashé l'image, veuillez utiliser `connexion via port série à la station de travail SSH`.
- Lors du premier démarrage, il est interdit de connecter tout autre câble que l'alimentation et le Type-C !!!
- Lors de la compilation du firmware, veuillez utiliser `connexion WiFi à la station de travail SSH` pour faciliter la compilation et le flashage du firmware !
:::

* Avant de commencer la connexion, veuillez télécharger : <Button variant="contained" disableElevation component="a" rel="noopener" href="https://mobaxterm.mobatek.net/download-home-edition.html ">logiciel SSH</Button>. Vous pouvez choisir la version `Installer edition` à droite pour télécharger.

        ## Utiliser une connexion série pour accéder à la station de travail SSH

       :::tip Remarque
        - Veuillez vous assurer que la carte TF ou M2WE est correctement gravée avec le système et correctement installée sur FLY-C5.
        - Le système FLYOS-FAST a par défaut l'utilisateur `root` et le mot de passe `mellow`.
       :::

        * Connectez FLY-C5 à l'ordinateur via un câble de données Type-C.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/c5_usb.webp').default} size="60%" align="center" />

        * Si la connexion entre la carte mère et l'ordinateur fonctionne correctement, vous verrez le port CH340 dans le Gestionnaire de périphériques.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/pc-device-mgr.webp').default} size="50%" align="center" />

        * Ouvrez `MobaXterm` et effectuez les paramétrages suivants, en sélectionnant le port que vous avez vu dans le Gestionnaire de périphériques.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-2.webp').default} size="60%" align="center" />

        * Si tout se passe bien, vous verrez l'écran de démarrage.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial1.webp').default} size="60%" align="center" />  
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial2.webp').default} size="60%" align="center" />


        * Comme FLYOS-FAST démarre rapidement, vous risquez probablement de voir des codes incorrects, comme illustré ci-dessous. C'est un phénomène normal, ne vous inquiétez pas, **appuyez plusieurs fois sur Entrée**.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial3.webp').default} size="60%" align="center" />


        * Après avoir appuyé sur Entrée, vous verrez le contenu ci-dessous.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial4.webp').default} size="60%" align="center" />

        * Entrez le compte : `root`, puis appuyez sur Entrée, puis entrez le mot de passe : `mellow`, puis appuyez sur Entrée. Le mot de passe n'apparaît pas lors de l'entrée, c'est normal.

        :::warning Attention 
        - Le compte système FLYOS-FAST est `root`, le mot de passe est `mellow`.
        - Le système FLYOS-FAST n'a qu'un seul compte : `root` !!!
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial5.webp').default} size="60%" align="center" />

        * Si vous souhaitez vérifier l'adresse IP, tapez `ip a | grep inet` dans le SSH et trouvez l'IP correspondante au même segment que votre routeur pour y accéder.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_serial6.webp').default} size="60%" align="center" />

        ## Utiliser une connexion WiFi pour accéder à la station de travail SSH
       :::tip Remarque
        - Veuillez vous assurer que le système de la station de travail est correctement démarré.
        - Le système FLYOS-FAST a par défaut l'utilisateur `root` et le mot de passe `mellow`.
        - La condition préalable pour se connecter à la station de travail SSH via WiFi est que votre station de travail est déjà connectée au routeur et que vous connaissez l'adresse IP de la station de travail.
       :::

        * Connectez FLY-C5 à l'ordinateur via un câble de données Type-C.

        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c5/img/c5_usb.webp').default} size="60%" align="center" />

        * Ouvrez MobaXterm, configurez-le comme indiqué ci-dessous, et saisissez dans le champ `Remote host` l'adresse IP que vous avez obtenue via la connexion série, ou consultez l'adresse IP dans l'interface arrière du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Remarque : Dans cette étape, saisissez l'utilisateur `root` de la station de travail.**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_2.webp').default} size="60%" align="center" />

        * La première connexion affichera une fenêtre de confirmation, cliquez sur `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Saisissez le mot de passe demandé, puis appuyez sur Entrée.
        :::tip
        - Ici, le mot de passe entré ne sera pas affiché, appuyez directement sur Entrée après l'avoir saisi.
        :::
        
        Le système FLYOS-FAST a par défaut l'utilisateur `root` et le mot de passe `mellow`.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_3.webp').default} size="60%" align="center" />

        **Si une fenêtre similaire à celle-ci apparaît, elle vous demande si vous voulez enregistrer le mot de passe utilisateur, cliquez sur `Non`.**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Si vous voyez l'interface ci-dessous, la connexion est réussie !

        :::warning Attention 
        - Le compte système FLYOS-FAST est `root`, le mot de passe est `mellow`.
        - Le système FLYOS-FAST n'a qu'un seul compte : `root` !!!
        :::

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_5.webp').default} size="60%" align="center" />



    </TabItem>
