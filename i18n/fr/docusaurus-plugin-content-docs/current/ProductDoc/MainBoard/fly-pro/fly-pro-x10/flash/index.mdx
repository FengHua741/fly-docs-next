# Notice sur la gravure du firmware

    :::tip Attention

    * Le PRO X10 est pré-gravé avec Katapult
    * Une fois que le BL a été gravé, il n'est pas nécessaire de le graver à nouveau
    * Le PRO X10 dispose de deux types de BL, soit Katapult firmware, soit BL firmware gravé via une carte SD
    * Si nécessaire, vous pouvez choisir de re-graver le <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">BL firmware</Button>

    :::
    
    :::danger À savoir
    * La gravure du firmware de la carte mère doit se faire via Katapult pour graver Klipper
    * Si la carte mère est déjà gravée avec Katapult, il n'est pas nécessaire d'effectuer la dernière étape de gravure de Katapult firmware. Cela n'est qu'une précaution au cas où la gravure de Katapult firmware ne serait pas réussie
    * Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère. Si c'est le cas, cela signifie que la carte mère est entrée en mode Katapult firmware
    :::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de graver qu'un seul type de firmware

    :::
    
    * Le Pro X10 peut être connecté à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont utilisés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware pont USB</Button>
        * Avantage : il peut se connecter à plusieurs appareils sans occuper de ports USB
        * Inconvénient : l'utilisation conjointe avec `KlipperScreen` peut entraîner des erreurs telles qu'un délai de retour à zéro
