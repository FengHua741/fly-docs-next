# Références de branchement du panneau principal

    :::warning Remarques

    En raison de la résolution trop élevée de l'image, il est impossible de voir clairement en naviguant sur la page web. Veuillez placer le curseur de la souris sur le schéma de câblage, cliquer avec le bouton droit de la souris, choisir "Enregistrer l'image sous" et enregistrer l'image sur le bureau pour l'examiner !!!

    :::
