# BL firmware

    :::tip

    * Veuillez noter que le bootloader est déjà pré-écrit en usine, pas besoin de le réécrire.

    :::

## Entrée en mode de programmation

    * Il existe deux méthodes pour entrer en mode de programmation
    * Première méthode : Désactivez complètement l'alimentation du circuit imprimé, puis court-circuitez BOOT, puis connectez un câble de transmission de données d'un côté au circuit imprimé et de l'autre côté à votre appareil.
    * Deuxième méthode : Connectez un câble de transmission de données d'un côté au circuit imprimé et de l'autre côté à votre appareil, puis court-circuitez BOOT, appuyez sur le bouton RST, puis relâchez le bouton RST enfin déconnectez le court-circuit de BOOT.

            4. Programmation du firmware

            :::tip

            Vous n'avez besoin de programmer qu'un seul firmware.

            :::

            * Voici la commande de programmation du firmware katapult
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            * Voici la commande de programmation du bootloader
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/ProX10/ProX10_USB_katapult.bin
                ```

            5. Si vous voyez le contenu indiqué par la flèche ci-dessous ``File downloaded successfully``, alors la programmation a réussi.

                <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="100%" align="center" />
