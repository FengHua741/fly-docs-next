# Compilation et flashage du firmware

    :::tip

    * Le 407ZG utilise par défaut le flashage via la carte TF
    * Préparez une carte TF d'une capacité inférieure ou égale à `32G`
    * Les cartes mémoire de marque non officielle peuvent entraîner un échec du flashage, il est recommandé d'utiliser les cartes TF recommandées pour mettre à jour le firmware
    * La vitesse de transfert de la carte TF doit être de classe `10` ou supérieure, il est conseillé d'utiliser des cartes TF de marques comme SanDisk

    :::

    * Si la sortie finale ressemble au texte suivant, la compilation a réussi

        Creating hex file out/klipper.bin
