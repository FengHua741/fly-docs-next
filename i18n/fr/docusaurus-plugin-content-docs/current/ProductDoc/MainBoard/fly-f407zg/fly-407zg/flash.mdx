# Compilation et flashage du firmware

    :::tip

    * Le 407ZG utilise par défaut la carte TF pour le flashage du firmware
    * Préparez une carte TF d'une capacité maximale de `32 Go`
    * Les cartes mémoire de marque non recommandée peuvent entraîner un échec du flashage, il est donc préférable d'utiliser une carte TF des marques recommandées pour mettre à jour le firmware
    * La vitesse de transmission de la carte TF doit être de classe `10` ou supérieure, il est conseillé d’utiliser des cartes TF de marques comme SanDisk

    :::
