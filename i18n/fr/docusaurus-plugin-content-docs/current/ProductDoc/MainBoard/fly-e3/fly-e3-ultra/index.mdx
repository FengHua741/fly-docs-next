## Paramètres du produit

* Puce ARM Cortex-M4 32 bits série 550 MHz, puce STM32H723VGT6
* Firmware : Reprap/Klipper
* Intègre cinq pilotes TMC2209, les trois positions des moteurs XYZ prennent en charge le mode sans limite
* Affichage : écran tactile série, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
* 1 entrée haute tension, supporte les capteurs de détection de lit automatique : BLTouch, PL08N, etc., capteurs de proximité Klicky, Voron Tap, etc.
* En utilisant le firmware Repap, il est possible d'utiliser un module WiFi ESP32, avec une vitesse de 1,6 m/s
* Conception unique de la carte MOS du ventilateur PWM, facile à remplacer en cas de dommage, réduit la difficulté de réparation pour les clients, quatre connecteurs de ventilateur de 2 broches
* 3 interfaces ADC pour les capteurs de température
3 ports de limite peuvent être connectés aux interrupteurs de limite
* 4 interfaces IO qui peuvent être utilisées pour les capteurs de proximité, interrupteurs de limite, têtes laser, lampes RGB, etc.
* 3 interfaces de chauffage
* 1 interface de port série pour l'écran RRF
* Version Max prend en charge 1 thermocouple K et 1 capteur de température PT100
* Nouvelle fonctionnalité : commutateur à glissière pour choisir le mode 2/3/4 fils du capteur de température PT100
* Nouvelle conception de dissipateur thermique intégré, augmente la surface de refroidissement des pilotes
* Nouvelle interface CAN ajoutée
