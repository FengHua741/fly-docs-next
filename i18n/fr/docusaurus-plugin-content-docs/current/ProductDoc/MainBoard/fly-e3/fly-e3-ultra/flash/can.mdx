## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Appuyez sur `Q`, apparaît **Save configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration est maintenant enregistrée et vous êtes revenu à l'interface de commande
    * Entrez la commande ci-dessous pour commencer la compilation, cela prend un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient les lignes suivantes, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
