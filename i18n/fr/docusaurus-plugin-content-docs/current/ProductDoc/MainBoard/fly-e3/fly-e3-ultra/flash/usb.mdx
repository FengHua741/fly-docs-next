# Compilation du firmware USB

    :::tip

    * Un bootloader est déjà préinstallé sur la carte SD, pas besoin de le réinstaller

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur `Q`, apparaît **Save configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration doit maintenant être sauvegardée et vous êtes revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour démarrer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient ce qui suit, la compilation a réussi :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
