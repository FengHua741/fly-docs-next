# Firmwares BL

    :::tip

    * Veuillez noter que le "bootloader" est déjà préinstallé en usine, il n'est donc pas nécessaire de le réinstaller.

    :::

## Mode d'entrée en mode de programmation

    :::tip

    À droite se trouve le bouton "BOOT", à gauche se trouve le bouton "RESET".

    :::

    * Il existe deux méthodes pour entrer en mode de programmation
    * Première méthode : Lorsque la carte mère est complètement débranchée, appuyez sur le bouton BOOT puis branchez un câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur votre appareil.
    * Deuxième méthode : Après avoir branché un câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur votre appareil, appuyez sur le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST puis relâchez le bouton BOOT.
