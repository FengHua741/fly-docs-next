# BL firmware

    :::tip

    * Veuillez noter que le `bootloader` est préinstallé à la sortie d'usine et n'a pas besoin d'être réinstallé.

    :::

## Entrée en mode de programmation

    :::tip

    À droite se trouve le bouton `BOOT`, à gauche se trouve le bouton `RESET`.

    :::

    * Il existe deux méthodes pour entrer en mode de programmation.
    * Première méthode : Éteignez complètement la carte mère, appuyez sur le bouton BOOT, puis branchez un câble de données entre la carte mère et votre appareil.
    * Deuxième méthode : Branchez un câble de données entre la carte mère et votre appareil, appuyez sur le bouton BOOT, puis appuyez sur le bouton RST, relâchez le bouton RST, puis relâchez le bouton BOOT.
