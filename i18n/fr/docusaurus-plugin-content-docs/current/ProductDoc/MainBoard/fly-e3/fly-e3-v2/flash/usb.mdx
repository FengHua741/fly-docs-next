# Compilation du firmware USB

    :::tip

    * L'E3 V2 est livré avec un bootloader TF préinstallé, il n'est donc pas nécessaire de le réinstaller.

    :::

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur la touche `Q`, puis sur `Y` lorsque vous voyez **Save configuration**
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir sauvegardé la configuration et être revenu à l'interface de ligne de commande
    * Entrez la commande ci-dessous pour démarrer la compilation, qui prendra un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble à ceci, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
