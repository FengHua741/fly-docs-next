    :::tip

    * Le firmware par défaut pour E3 V2 utilise une carte TF pour la gravure
    * Préparez une carte TF dont la capacité n'excède pas `32G`
    * Les cartes mémoires de marque inconnue peuvent entraîner un échec de la gravure, veuillez utiliser les cartes mémoires recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte TF doit être de niveau `class10` ou supérieur, il est conseillé d'utiliser des cartes TF de marque SanDisk

    :::

# Instructions pour la gravure du firmware


## Comment déterminer quel type de firmware vous avez besoin

    :::tip

    * Notez que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * E3 V2 peut être connecté à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont présents, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware USB Bridge</Button>
        * Avantage : peut connecter plusieurs appareils sans occuper de ports USB
        * Inconvénient : peut causer des erreurs comme des timeouts de positionnement en utilisant `KlipperScreen` ensemble
