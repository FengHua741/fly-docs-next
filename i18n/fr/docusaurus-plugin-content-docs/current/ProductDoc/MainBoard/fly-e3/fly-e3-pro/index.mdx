## Paramètres du produit

    * Processeur 32-bit ARM Cortex-M4 série 168 MHz, puce STM32F407ZGT6
    * Firmware : Reprap/Klipper
    * Intègre cinq pilotes TMC2209
    * Affichage : écran tactile série, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1
    * 1 entrée haute tension, supporte les capteurs de planification automatique de lit : BLTouch, PL08N, etc., Klicky, Voron Tap, etc.
    * Avec le firmware Repap, peut utiliser le module WiFi ESP32, supporte 1,6 m/s
    * Conception unique de la carte MOS du ventilateur PWM, qui peut être remplacée directement en cas de dommage, réduit la difficulté de réparation pour le client, quatre connecteurs de ventilateurs 2 broches
    * Le PCB utilise une épaisseur de cuivre de 2OZ, capable de supporter un courant plus élevé
    * 2 interfaces ADC, prenant en charge jusqu'à deux thermistances
    * 4 interfaces IO, pouvant être utilisées pour l'équilibrage et les limites
    * 3 interfaces pour les résistances de chauffage, prenant en charge jusqu'à trois résistances de chauffage
