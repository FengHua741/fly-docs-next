# Compilation et écriture du firmware

    :::tip

    * Le firmware E3 Pro utilise par défaut une carte SD pour l'écriture du firmware
    * Préparez une carte SD d'une capacité ne dépassant pas `32 Go`
    * Les cartes SD de marque non recommandée peuvent provoquer un échec de l'écriture, il est donc préférable d'utiliser les cartes SD recommandées ci-dessous
    * La vitesse de transmission de la carte SD doit être de classe `10` ou supérieure, il est conseillé d'utiliser des cartes SD de marque SanDisk

    :::

        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir sauvegardé la configuration et être revenu à l'interface de ligne de commande
    * Entrez la commande suivante pour démarrer la compilation, cela prendra un certain temps
    ```bash
    make -j4
    ```

    * Une sortie similaire à celle-ci indique que la compilation a réussi

        ```bash
        Creating hex file out/klipper.bin
        ```
