# Compilation et écriture du firmware

    :::tip

    * E3 Pro utilise par défaut la carte SD pour l'écriture du firmware
    * Préparez une carte SD dont la capacité ne dépasse pas `32G`
    * Les cartes SD de marque non officielle peuvent entraîner un échec de l'écriture, veuillez utiliser les cartes SD recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte SD doit être de classe `10` ou supérieure, il est conseillé d'utiliser des cartes SD de marque SanDisk par exemple

    :::

    * Appuyez sur la touche `Q`, puis apparaît **Save configuration**, appuyez ensuite sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />

    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient ce qui suit, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
