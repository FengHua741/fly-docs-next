### Sauts de câblage du circuit imprimé principal

* **Mode STEP/DIR.** Dans ce mode, le réglage de la subdivision est effectué en utilisant une gaine de saut, et le réglage du courant est effectué en régulant un potentiomètre. Le plus souvent, A4988, 8825 nécessitent de consulter la table de configuration de subdivision fournie par le fabricant, puis de régler la subdivision à l'aide d'une gaine de saut.
* **Mode UART.** Les pilotes les plus couramment utilisés dans ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilotage peuvent communiquer de manière asynchrone en série via UART avec le contrôleur principal, et le réglage de la subdivision, du courant de fonctionnement et du mode silencieux peut être effectué en modifiant les fichiers de configuration.
* **Mode SPI.** Les pilotes les plus couramment utilisés dans ce mode sont : TMC5160, TMC2130, TMC2240, etc. Il est également possible de régler la subdivision, le courant de fonctionnement et le mode silencieux en modifiant les fichiers de configuration.
