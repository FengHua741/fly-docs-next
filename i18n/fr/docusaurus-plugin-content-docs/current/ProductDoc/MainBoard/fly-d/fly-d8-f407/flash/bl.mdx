## Entrer en mode de programmation

    :::tip

    À gauche se trouve le bouton `RST`, à droite se trouve le bouton `BOOT`

    :::

    * Il existe deux méthodes pour entrer en mode de programmation
    * Première méthode : Éteignez complètement la carte mère, appuyez sur le bouton BOOT puis branchez un câble de transfert de données d'un côté sur la carte mère et de l'autre côté sur votre appareil
    * Deuxième méthode : Après avoir branché un câble de transfert de données d'un côté sur la carte mère et de l'autre côté sur votre appareil, appuyez sur le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST et enfin relâchez le bouton BOOT
