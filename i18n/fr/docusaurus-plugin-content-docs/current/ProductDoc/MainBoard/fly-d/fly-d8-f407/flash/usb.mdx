## Vérifier si le mode de flashage est activé

    * Assurez-vous que l'ordinateur principal peut se connecter normalement à Internet et que le câble de connexion entre l'ordinateur principal et l'ordinateur secondaire possède une fonctionnalité de transmission de données.
    * Connectez-vous via SSH à l'ordinateur principal, puis entrez `lsusb` et appuyez sur Entrée. Si vous voyez les informations encerclées dans l'image ci-dessous, vous pouvez passer à l'étape suivante.
        * Si aucune information n'est retournée, il s'agit d'un problème avec le système de l'ordinateur principal, nous ne pouvons rien faire à ce sujet. Vous devez changer pour un système confirmé comme fonctionnant ou remplacer l'ordinateur principal.
        * Si vous obtenez un message indiquant qu'il n'y a pas de commande `lsusb`, exécutez cette commande pour l'installer :
            ```bash
            sudo apt-get install usbutils
            ```
            <ImageView image={require('@site/docs/General/get-id/img/6177.webp').default} size="80%" align="left" /> 
    * L'ID affiché dans le cadre rouge peut être utilisé comme référence mais il n'est pas fixe, il est variable. Le suffixe `OpenMoko, Inc. stm32f407xx` après `XX` est une indication.
        * Certains ordinateurs principaux peuvent ne pas afficher correctement ou ne pas afficher du tout en raison de problèmes liés au système.
