# Présentation

> Mellow Fly-D8 est une carte mère de haute performance pour imprimante 3D, développée par l'équipe de recherche et développement de la société Guangzhou Meilun Electronic Technology Co., Ltd. Elle peut prendre en charge jusqu'à 8 moteurs pas à pas.

## Caractéristiques du produit

* Les interfaces de fin de course et autres de la carte mère sont alimentées à 5V avec isolation de protection. Si le POWER n'est pas connecté, aucune puissance ne sera sortie.
* Utilise un processeur ARM Cortex-M4 de 32 bits, série 168 MHz, puce STM32F407.
* Supporte le bus CAN et la connexion RS232, ce qui rend la transmission de données plus stable, avec une latence réduite et une connexion plus stable.
* La carte mère dispose de boutons BOOT et RESET, permettant aux utilisateurs d'entrer en mode DFU via USB pour mettre à jour le firmware.
* L'interface d'alimentation a une protection contre la polarité inversée, évitant que les clients ne brûlent la carte lors de la réalisation de DIY en inversant les fils d'alimentation.
* Fournit deux ports USB 2.0, élargissant ainsi les possibilités de connexion, dont l'un des ports USB peut être commuté en port RS232 via un interrupteur, afin de s'adapter aux besoins de connexion de divers appareils.

# Spécifications du produit

* Puce principale : Processeur ARM Cortex-M4 de 32 bits, série 168 MHz, puce STM32F407ZGT6.
* Firmware : Marlin 2.0 / Klipper.
* Mode de pilotage pris en charge : TMC : UART / SPI.
* Huit pilotes pris en charge : A4988, LV8729, DRV8225, TMC2208 / 2209 / 5160, etc.
* Affichage : écran tactile sériel, mini12864 LCD, 12864 LCD, 2004 LCD, FLY 4.3/7.0 V1.
* Support des capteurs de plan de lit automatique : BLTouch, PL08N, etc., Klicky, Voron Tap, etc.
* Tension d'entrée : supporte 12V / 24V.
* Six interfaces de limiteur, permettant une limitation de plage et un contrôle de position précis.
* Deux interfaces RGB, offrant des effets visuels colorés.
* Trois interfaces de chauffage, fournissant un contrôle indépendant pour l'extrudeuse et le lit chaud.
* Quatre interfaces thermistances, pouvant surveiller simultanément quatre groupes de températures, assurant un contrôle de température précis.
* Interfaces USB : supporte deux interfaces USB 2.0, dont l'une peut être commutée en interface RS232 via un interrupteur.
* Fournit deux ports USB 2.0, élargissant ainsi les possibilités de connexion, dont l'un des ports USB peut être commuté en port RS232 via un interrupteur, afin de s'adapter aux besoins de connexion de divers appareils.
