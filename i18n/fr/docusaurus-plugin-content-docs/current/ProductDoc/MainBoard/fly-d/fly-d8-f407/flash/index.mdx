# Instructions pour la programmation de la mise à jour du firmware

:::danger À savoir
* La programmation du firmware de la carte mère doit être effectuée via Katapult pour programmer Klipper.
* Si vous êtes sûr que la carte mère a déjà été programmée avec Katapult, vous n'avez pas besoin d'effectuer la dernière étape de programmation du firmware Katapult, c'est juste une précaution au cas où la programmation du firmware Katapult ne serait pas réussie.
* Vous pouvez appuyer deux fois rapidement sur le bouton de réinitialisation et regarder si un LED clignote sur la carte mère, si c'est le cas, cela signifie que la carte mère est déjà dans le firmware Katapult.
:::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul des firmwares mentionnés ci-dessus.

    :::
    
    * La D8 peut être connectée à l'ordinateur hôte de la manière suivante :
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Firmware USB</Button>
        * Avantages : Communication fiable et stable.
        * Inconvénients : Peut affecter la transmission des données si plusieurs périphériques USB sont utilisés.
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware USB bridge</Button>
        * Avantages : Peut connecter plusieurs appareils sans utiliser de ports USB.
        * Inconvénients : Peut causer des erreurs telles que des retards de positionnement lors de l'utilisation conjointe avec `KlipperScreen`.
