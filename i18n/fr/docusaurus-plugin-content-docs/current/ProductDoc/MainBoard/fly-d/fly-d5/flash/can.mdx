# Flashage du firmware CAN via un pont USB

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Appuyez sur `Q`, une fenêtre **Save configuration** apparaît, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration devrait maintenant être enregistrée et vous êtes revenu à l'interface de ligne de commande
    * Entrez la commande ci-dessous pour démarrer la compilation, cela peut prendre un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble au texte suivant, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```


## Entrer en mode de flashage

    :::tip Astuce
        Les D5 sortis après le 1er septembre 2024 sont déjà équipés du firmware Katapult
    :::
    * Connectez la carte mère à l'ordinateur principal avec un câble Type-C, puis **appuyez deux fois sur le bouton reset**, une LED clignotera sur la carte mère.
    :::warning Avertissement
       * Si la LED ne clignote pas, veuillez réessayer la mise à jour du firmware Katapult : <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#entrer-en-mode-de-flashing-dfu">Mettre à jour le firmware Katapult</Button>
       * Si la LED ne clignote pas, veuillez exécuter à nouveau l'étape précédente après avoir mis à jour le firmware Katapult
    :::
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/img/d5_dfu.webp').default} size="100%" align="center" />
