# Flashage du firmware USB bridge CAN

## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Appuyez sur la touche `Q`, puis sur `Y` lorsque vous voyez **Save configuration**
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous êtes revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient les lignes suivantes, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Entrer en mode de flashage

    :::tip Astuce
        Les appareils Fly-D5 fabriqués après le 1er septembre 2024 sont déjà équipés du firmware Katapult
    :::
    * Connectez la carte mère au poste de travail avec un câble Type-C, puis appuyez deux fois sur le bouton reset. À ce moment, une LED clignotera sur la carte mère.
    :::warning Avertissement
       * Si aucune LED ne clignote, veuillez reprogrammer le firmware Katapult :<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#enter-dfu-flashing-mode">Programmer le firmware Katapult</Button>
       * Si aucune LED ne clignote, veuillez réexécuter cette étape après avoir programmé le firmware Katapult
    :::
