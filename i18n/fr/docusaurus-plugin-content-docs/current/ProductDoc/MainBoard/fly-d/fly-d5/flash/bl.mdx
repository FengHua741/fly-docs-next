# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::tip
    * Le firmware Katapult de D5 utilise le troisième mode de connexion : connexion USB
    * D5 est préinstallé avec le firmware Katapult dès sa sortie le 1er septembre 2024
    :::

## Entrer en mode de flashage DFU

    :::tip

    À gauche se trouve le bouton `BOOT`, à droite se trouve le bouton `RESET`

    :::

    * Il existe deux méthodes pour entrer en mode de flashage
    * Première méthode : Éteignez complètement la carte mère, appuyez sur le bouton BOOT puis branchez un câble de transfert de données d'un côté sur la carte mère et de l'autre côté sur votre appareil
    * Deuxième méthode : Après avoir branché un câble de transfert de données d'un côté sur la carte mère et de l'autre côté sur votre appareil, appuyez sur le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST puis relâchez le bouton BOOT
