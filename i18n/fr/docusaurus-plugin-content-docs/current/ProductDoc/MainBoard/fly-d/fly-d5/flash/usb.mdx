## Démarrer la compilation

    <SSH />

    <USB />

    * Appuyez sur `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration devrait maintenant être enregistrée et vous êtes sorti dans l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble à ceci, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```

## Entrer en mode de programmation

    :::tip Astuce
        Les D5 sortis après le 1er septembre 2024 sont déjà pré-programmés avec le firmware Katapult
    :::
    * Connectez la carte mère au poste de travail avec un câble de données Type-C, puis **appuyez deux fois sur le bouton reset**, à ce moment-là une LED clignotera sur la carte mère.
    :::warning Avertissement
       * Si la LED ne clignote pas, veuillez réenregistrer le firmware Katapult :<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d5/flash/bl#entrer-en-mode-de-programmation-dfu">Programmer le firmware Katapult</Button>
       * Si la LED ne clignote pas, veuillez réexécuter cette étape après avoir enregistré le firmware Katapult
    :::
