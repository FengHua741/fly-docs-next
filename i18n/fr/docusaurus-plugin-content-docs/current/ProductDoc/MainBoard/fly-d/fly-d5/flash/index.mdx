    Avant de compiler et de programmer le firmware, veillez à lire ce qui suit !!!

:::tip Remarques importantes
* Le D5 est pré-approvisionné avec le firmware Katapult à partir du 1er septembre 2024.
* Une fois que vous avez programmé BL une seule fois, vous n'avez pas besoin de le reprogrammer (le BL du D5 est Katapult).
* BL n'est qu'un programme de démarrage pour guider le firmware Katapult, il faut encore programmer le firmware Klipper.
:::



## Déterminez le firmware dont vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de programmer qu'un seul des firmwares.

    :::
    
    * Le D5 peut être connecté à l'ordinateur hôte via les deux méthodes suivantes :
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Firmware USB (le plus couramment utilisé)</Button>
        * Avantages : communication fiable, stable
        * Inconvénients : si plusieurs périphériques USB sont utilisés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware pont USB (souvent utilisé lors de l'utilisation d'une carte outil CAN)</Button>
        * Avantages : vous pouvez connecter plusieurs appareils sans occuper de ports USB
        * Inconvénients : l'utilisation conjointe avec `KlipperScreen` peut entraîner des erreurs telles que des retards de retour à zéro.
