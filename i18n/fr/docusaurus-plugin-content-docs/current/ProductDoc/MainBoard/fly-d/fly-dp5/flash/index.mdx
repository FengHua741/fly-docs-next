# Instructions pour la gravure du firmware

:::danger À savoir
* La gravure du firmware de la carte mère doit être effectuée via Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin de procéder à la gravure finale de Katapult. Celle-ci ne sert qu'à prévenir une éventuelle erreur de gravage du firmware Katapult.
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation pour voir si une LED clignote sur la carte mère. Si c'est le cas, cela signifie que la carte mère est déjà dans le firmware Katapult
:::

## Comment déterminer quel firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de graver qu'un seul firmware

    :::
    
    * DP5 peut utiliser les méthodes suivantes pour se connecter à l'ordinateur supérieur
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantages : communication fiable et stable
        * Inconvénients : Si plusieurs périphériques USB sont utilisés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware CAN</Button>
        * Avantages : il peut être connecté à plusieurs appareils sans occuper de ports USB
        * Inconvénients : peut causer des erreurs telles qu'une mise au point hors délai en raison de l'utilisation conjointe de `KlipperScreen`.
