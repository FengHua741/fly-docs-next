## Entrer en mode de flashage

    :::tip

    À gauche se trouve le bouton `BOOT`, à droite se trouve le bouton `RESET`.

    :::

    * Il existe deux méthodes pour entrer en mode de flashage.
    * Première méthode : Éteignez complètement la carte mère, appuyez sur le bouton BOOT, puis branchez un câble de transmission de données entre la carte mère et votre appareil.
    * Deuxième méthode : Branchez un câble de transmission de données entre la carte mère et votre appareil, appuyez sur le bouton BOOT, puis appuyez sur le bouton RST, relâchez le bouton RST en dernier, puis relâchez le bouton BOOT.

            :::tip

            * Voici la commande de flashage
                ```bash
                sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/FLY_Katapult/BL/D/FLY_DP5.bin
                ```
