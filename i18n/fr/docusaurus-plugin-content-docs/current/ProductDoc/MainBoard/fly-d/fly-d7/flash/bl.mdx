# Firmware Katapult

## Explication du firmware Katapult

    <CANBOOT />

    :::tip
        * Le firmware Katapult de D7 utilise la troisième méthode de connexion : la connexion USB
        * D7 est pré-écrit avec le firmware Katapult dès sa sortie le 1er septembre 2024
    :::

## Entrée en mode de flashage DFU

    :::tip

    À gauche se trouve le bouton `BOOT`, à droite se trouve le bouton `RESET`.

    :::

    * Il existe deux méthodes pour entrer en mode de flashage
    * Première méthode : Déconnectez complètement l'alimentation du circuit, appuyez sur le bouton BOOT puis branchez un câble de transfert de données d'un côté au circuit et de l'autre côté à votre appareil.
    * Deuxième méthode : Après avoir branché un câble de transfert de données d'un côté au circuit et de l'autre côté à votre appareil, appuyez sur le bouton BOOT puis appuyez sur le bouton RST, relâchez ensuite le bouton RST et enfin relâchez le bouton BOOT
