 ### Saute-circuits pour les pilotes du circuit imprimé principal

* **Mode STEP/DIR.** Dans ce mode, en utilisant un saute-circuit, on peut régler la subdivision en utilisant un saute-circuit et ajuster le courant en utilisant un potentiomètre. Il est le plus courant que les pilotes A4988 et 8825 nécessitent de consulter le tableau de configuration de subdivision fourni par le fabricant, puis de régler la subdivision à l'aide d'un saute-circuit.
* **Mode UART.** Les pilotes les plus courants utilisant ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilote peuvent communiquer de manière asynchrone en série avec le contrôleur principal via UART. On peut régler la subdivision, le courant de fonctionnement et le mode silencieux du pilote en modifiant le fichier de configuration.
* **Mode SPI.** Les pilotes les plus courants utilisant ce mode sont : TMC5160, TMC2130, TMC2240, etc. Ces puces de pilote peuvent également régler la subdivision, le courant de fonctionnement et le mode silencieux du pilote en modifiant le fichier de configuration.
