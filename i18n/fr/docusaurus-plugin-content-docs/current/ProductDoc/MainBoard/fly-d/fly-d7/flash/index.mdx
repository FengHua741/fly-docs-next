    Avant de compiler et graver le firmware, veuillez lire attentivement ce qui suit !!!

:::tip Remarques importantes
* Le D5 préinstallé avec le firmware Katapult a été fabriqué à partir du 1er septembre 2024.
* Une fois que le BL a été flashé une fois, il n'est plus nécessaire de le reflasher (le BL du D5 est Katapult).
* Le BL n'est qu'un programme de démarrage pour guider le firmware Katapult, vous devrez également flasher le firmware Klipper.
:::

## Déterminez le firmware dont vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul firmware

    :::
    
    * Le D7 peut être connecté à l'ordinateur hôte par les deux méthodes suivantes
    * <Button variant="contained" disableElevation href="./usb">Firmware USB (le plus couramment utilisé)</Button>
        * Avantages : communication fiable et stable
        * Inconvénients : si plusieurs périphériques USB sont utilisés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware pont USB (souvent utilisé lorsque vous utilisez une carte outil CAN)</Button>
        * Avantages : vous pouvez connecter plusieurs appareils sans utiliser de ports USB
        * Inconvénients : l'utilisation conjointe de `KlipperScreen` peut entraîner des erreurs telles que des retards de positionnement.
