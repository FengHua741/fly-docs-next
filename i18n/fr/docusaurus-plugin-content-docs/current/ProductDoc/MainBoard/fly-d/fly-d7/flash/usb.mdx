    * Appuyez sur la touche `Q`, apparaît **Enregistrer la configuration**, appuyez ensuite sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration devrait maintenant être enregistrée et vous êtes revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale est comme ci-dessous, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```

## Entrer en mode de programmation

    :::tip Astuce
        À partir du 1er septembre 2024, les D7 fabriqués après cette date sont déjà préprogrammés avec le firmware Katapult
    :::
    * Connectez la carte mère au PC avec un câble Type-C, puis **appuyez deux fois sur le bouton reset**, à ce moment-là une LED clignotera sur la carte mère.
    :::warning Avertissement
       * Si aucune LED ne clignote, rechargez le firmware Katapult :<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#entrer-en-mode-de-programmation">Recharger le firmware Katapult</Button>
       * Si aucune LED ne clignote, rechargez le firmware Katapult puis exécutez l'étape suivante
    :::
