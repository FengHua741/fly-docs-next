# Compilation du firmware USB


## Démarrer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur `Q`, apparaîtra **Save configuration**, appuyez ensuite sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Vous devriez maintenant avoir sauvegardé la configuration et être revenu à l'interface de ligne de commande
    * Entrez la commande ci-dessous pour commencer la compilation, cela prendra un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale est similaire à ce qui suit, la compilation a réussi :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Entrer en mode de flashage

    :::tip Astuce
        Les D7 sortis après le 1er septembre 2024 sont déjà équipés du firmware Katapult.
    :::
    * Connectez la carte mère à l'ordinateur via un câble Type-C, puis **appuyez deux fois sur le bouton reset**, une LED clignotera sur la carte mère.
    :::warning Avertissement
       * Si aucune LED ne clignote, veuillez reflasher le firmware Katapult : <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#entrer-en-mode-de-flashage">Flasher le firmware Katapult</Button>
       * Si aucune LED ne clignote, veuillez réessayer après avoir flashé le firmware Katapult.
    :::

    <ImageView image={require('@site//docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/img/d7_dfu.webp').default} size="50%" align="center" />
