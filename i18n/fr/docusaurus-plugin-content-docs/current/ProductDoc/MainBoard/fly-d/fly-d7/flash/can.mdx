    * Appuyez sur la touche `Q`, puis apparaît **Enregistrer la configuration**, appuyez ensuite sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Entrez la commande ci-dessous pour démarrer la compilation, elle peut prendre un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Une sortie similaire à celle-ci indique que la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Entrer en mode de mise à jour

    :::tip Astuce
        Les D7 sortis après le 1er septembre 2024 sont déjà préinstallés avec le firmware Katapult.
    :::
    * Connectez la carte mère à l'ordinateur principal via un câble USB Type-C, puis **appuyez deux fois sur le bouton reset**, à ce moment-là une LED clignotera sur la carte mère.
    :::warning Avertissement
       * Si la LED ne clignote pas, rechargez le firmware Katapult :<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#entrer-en-mode-de-mise-a-jour-dfu">Recharger le firmware Katapult</Button>
       * Si la LED ne clignote toujours pas, réessayez les étapes après avoir rechargé le firmware Katapult.
    :::
