## Démarrer la compilation

    <SSH />

    <CAN />

    * Appuyez sur la touche `Q`, apparaît **Enregistrer la configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * La configuration devrait maintenant être enregistrée et vous êtes revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prend un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale est comme ci-dessous, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Création du fichier hex out/klipper.bin
        ```

## Entrer en mode de flashage

    :::tip Astuce
        Les D7 sortis après le 1er septembre 2024 sont déjà préchargés avec le firmware Katapult
    :::
    * Connectez la carte mère au PC avec un câble USB-C, puis **appuyez deux fois sur le bouton reset**, à ce moment, une LED sur la carte mère clignotera.
    :::warning Avertissement
       * Si aucune LED ne clignote, rechargez le firmware Katapult :<Button variant="contained" disableElevation href="/docs/ProductDoc/MainBoard/fly-d/fly-d7/flash/bl#entrer-en-mode-de-flashe-dfu">Recharger le firmware Katapult</Button>
       * Si aucune LED ne clignote, répétez l'étape précédente après avoir rechargé le firmware Katapult
    :::
