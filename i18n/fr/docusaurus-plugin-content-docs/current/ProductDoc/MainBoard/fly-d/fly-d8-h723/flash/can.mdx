    * Appuyez sur la touche `Q`, apparaît **Save configuration**, puis appuyez à nouveau sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="40%" align="center" />
    * Vous devriez maintenant avoir enregistré la configuration et être revenu à l'interface de ligne de commande
    * Entrez la commande suivante pour commencer la compilation, cela prendra un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Une sortie similaire signifie que la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
