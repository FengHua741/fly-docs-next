## Accès au mode de programmation

    :::tip

    À gauche se trouve le bouton `RST`, à droite se trouve le bouton `BOOT`.

    :::

    * Il existe deux méthodes pour entrer en mode de programmation
    * Première méthode : Éteignez complètement la carte mère, puis appuyez sur le bouton BOOT et branchez le câble de données (un côté sur la carte mère et l'autre côté sur votre appareil).
    * Deuxième méthode : Branchez le câble de données (un côté sur la carte mère et l'autre côté sur votre appareil), puis appuyez sur le bouton BOOT et appuyez ensuite sur le bouton RST, relâchez le bouton RST en premier puis relâchez le bouton BOOT.
