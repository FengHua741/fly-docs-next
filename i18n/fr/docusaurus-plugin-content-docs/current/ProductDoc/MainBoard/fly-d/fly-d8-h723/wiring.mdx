### Sauts de connexion pour le contrôleur de la carte mère

* **Mode STEP/DIR.** Dans ce mode, en utilisant un capuchon de saut, on peut régler la subdivision en modifiant le potentiomètre pour ajuster le courant. La plupart du temps, les pilotes A4988 et 8825 nécessitent de consulter le tableau de configuration de subdivision fourni par le fabricant, puis de régler la subdivision à l'aide d'un capuchon de saut.
* **Mode UART.** Les pilotes les plus couramment utilisés dans ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilotage peuvent communiquer de manière asynchrone en série via UART avec le contrôleur principal. On peut configurer le pilote pour la subdivision, le courant de fonctionnement et le mode silencieux en modifiant le fichier de configuration.
* **Mode SPI.** Les pilotes les plus couramment utilisés dans ce mode sont : TMC5160, TMC2130, TMC2240, etc. Il est également possible de configurer le pilote pour la subdivision, le courant de fonctionnement et le mode silencieux en modifiant le fichier de configuration.
