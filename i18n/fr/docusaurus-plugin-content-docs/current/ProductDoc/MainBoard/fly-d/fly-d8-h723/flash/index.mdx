# Instructions pour la gravure du firmware

:::danger À savoir
* La gravure du firmware de la carte mère doit être effectuée via Katapult pour graver Klipper
* Si vous êtes sûr que la carte mère a déjà été gravée avec Katapult, vous n'avez pas besoin d'effectuer la dernière étape de gravure du firmware Katapult, il ne s'agit que d'une précaution au cas où la gravure du firmware Katapult n'aurait pas réussi
* Vous pouvez appuyer rapidement deux fois sur le bouton de réinitialisation et vérifier si une LED clignote sur la carte mère, si c'est le cas, cela signifie que la carte mère est déjà dans le firmware Katapult
:::

## Comment déterminer quel type de firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * Le D8 peut être connecté à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont utilisés, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware USB pour pont</Button>
        * Avantage : peut se connecter à plusieurs appareils sans occuper de ports USB
        * Inconvénient : peut causer des erreurs telles qu'un dépassement de temps d'attente lors de l'utilisation conjointe avec `KlipperScreen`
