# flash de la mise à jour du firmware USB vers CAN

## Début de la compilation

    <SSH />

    <MakeOperate/>

    <CAN />

    * Sélectionnez les broches GPIO à configurer au démarrage du microcontrôleur, entrez : gpio8
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/flash/img/can.webp').default} size="100%" align="center" />
    * Appuyez sur la touche `Q`, apparaît **Save configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration est maintenant sauvegardée et vous êtes revenu à l'interface de ligne de commande
    * Entrez la commande ci-dessous pour commencer la compilation, cela prendra un certain temps
    ```bash
    make -j4
    ```
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale contient les lignes suivantes, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating bin file out/klipper.bin
        ```
