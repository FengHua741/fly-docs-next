# Firmware Katapult

    :::tip

    * Veuillez noter que le firmware `katapult` est déjà préinstallé en usine, pas besoin de le réinstaller.

    :::

## Entrer en mode de programmation

    :::tip

    Le bouton marqué est le `BOOT button`

    :::

    * Pour entrer en mode de programmation, veuillez maintenir le bouton BOOT pendant que vous débranchez complètement la carte mère et que vous branchez un câble de transfert de données entre la carte mère et votre appareil.
