# Micro4

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-micro/fly-micro4/img/micro.webp').default} size="80%" align="center" />

## Caractéristiques du produit

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Prise en charge du pont CAN
    * Intègre 4x2209
    * Intègre la broche de fonction DIAG, qui permet l'utilisation d'un support sans fin sans besoin de modification grâce à un simple branchement/débranchement d'une languette de saut

## Paramètres du produit

    * MCU：Raspberry Pi rp2040，Dual core ARM Cortex-M0+@133MHz
    * Firmware:  Klipper
    * Alimentation : 12-24V DC, plus stable, sans module de dérivation
    * Modes de fonctionnement des moteurs : UART, STEP/DIR
    * Interfaces d'extension : BLTouch（Servos、Probe）、Switch de proximité X1
    * Ventilateurs : Ventilateur contrôlable à deux fils X2 , Ventilateur toujours actif X1
    * Capteurs de température : 2x NTC 100K
    * Interfaces de chauffage : 2x interfaces contrôlables supportant 10A
