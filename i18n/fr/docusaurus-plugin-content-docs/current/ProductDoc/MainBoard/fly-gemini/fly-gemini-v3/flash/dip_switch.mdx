# Présentation des commutateurs dip

:::tip
FLY Gemini se connecte à l'hôte via les commutateurs dip. Si la position des commutateurs n'est pas correcte, il ne peut pas être utilisé normalement.
:::

:::danger À savoir
Veuillez noter que les commutateurs dip doivent être placés à des positions spécifiques pour une connexion correcte à l'hôte.
:::

## Connexion à l'hôte embarqué

    * Les commutateurs dip `1`, `2` vers la droite, connectent le périphérique inférieur à l'hôte embarqué via USB intégré.
    * Les commutateurs dip `3`, `4` vers la gauche, sinon cela affectera la connexion du périphérique inférieur à l'hôte embarqué.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * Après avoir effectué ces étapes, la prise USB en dessous sera occupée par le périphérique inférieur, veuillez ne pas connecter d'autres appareils.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## Connexion à un hôte externe

    * Les commutateurs dip `3`, `4` vers la droite, connectent le périphérique inférieur à l'hôte externe via USB intégré.
    * Les commutateurs dip `1`, `2` vers la gauche, sinon cela affectera la connexion du périphérique inférieur à l'hôte externe.
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
