# Introduction aux commutateurs DIP

:::tip
FLY Gemini se connecte à l'ordinateur hôte via les commutateurs DIP. Une mauvaise position des commutateurs empêchera le bon fonctionnement.
:::

:::danger À savoir
Veuillez noter que les commutateurs DIP doivent être positionnés correctement pour se connecter correctement à l'ordinateur hôte.
:::

## Connexion à l'ordinateur hôte embarqué

    * Commutateurs DIP `1`,`2` vers la droite pour connecter le dispositif inférieur à l'ordinateur hôte via USB intégré
    * Commutateurs DIP `3`,`4` vers la gauche, sinon cela affectera la connexion entre le dispositif inférieur et l'ordinateur hôte
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip1.webp').default} size="50%" align="center" />
    * Après avoir positionné les commutateurs DIP comme indiqué, le port USB inférieur sera occupé par le dispositif inférieur. Veuillez ne pas connecter d'autres appareils
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-c/fly-c8/img/usb.webp').default} size="50%" align="center" />


## Connexion à l'ordinateur hôte externe

    * Commutateurs DIP `3`,`4` vers la droite pour connecter le dispositif inférieur à l'ordinateur hôte externe via USB intégré
    * Commutateurs DIP `1`,`2` vers la gauche, sinon cela affectera la connexion entre le dispositif inférieur et l'ordinateur hôte externe
        <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/dip3.webp').default} size="50%" align="center" />
