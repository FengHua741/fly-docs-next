# Gemini V3

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-gemini/fly-gemini-v3/img/geminiv3.webp').default} size="80%" align="center" />

## Caractéristiques du produit
    * Intègre un ordinateur principal, pouvant remplacer le Raspberry Pi pour fonctionner en tant qu'ordinateur principal, avec 1 Go de mémoire DDR3 embarquée.
    * Protection améliorée de l'alimentation 5V, réduisant la probabilité de dommage des puces.
    * Suppression de la configuration de la carte de stockage du microcontrôleur, facilitant l'écriture du firmware.
    * Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, plugins de compensation de résonance numpy et FLY-Tools, sans opérations complexes telles que le changement de source, ce qui facilite l'utilisation pour les débutants (nécessite l'achat d'une carte TF ou M2WE)


## Paramètres du produit

    * CPU : Puce H5 Allwinner à haute performance, 4 cœurs 64 bits Cortex-A53
    * GPU : GPU à haute performance 6 cœurs Mali 450, taux de remplissage de pixels supérieur à 2,7 gpixel/s
    * RAM : 1 Go DDR3 (partagé avec le GPU)
    * ROM : Supporte jusqu'à 128 Go de carte SD
    * Périphériques : Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1
    * Interfaces : Interface M.2 embarquée, permettant d'étendre le WIFI, eMMC, etc. (protocole privé, ne pas connecter d'appareils M.2 non spécifiques à FLY π)
    * Alimentation : Prend en charge l'alimentation continue de 12-24V DC, plus stable, évitant la nécessité d'un module de dérivation
    * Prévue une interface de ventilateur 5V contrôlable, utilisée pour le refroidissement de l'ordinateur principal
    * Interface CAN d'extension prévue
    * Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, plugins de compensation de résonance numpy et FLY-Tools, sans opérations complexes telles que le changement de source, ce qui facilite l'utilisation pour les débutants (nécessite l'achat d'une carte TF ou M2WE)
    * Pilotes : Pilotes haute tension *4
    * MCU : STM32F405
