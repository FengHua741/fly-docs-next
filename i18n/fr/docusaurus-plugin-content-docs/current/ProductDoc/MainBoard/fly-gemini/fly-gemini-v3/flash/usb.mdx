# Compilation du firmware USB

## Commencer la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur la touche `Q`, puis sur **Enregistrer la configuration**, et enfin sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être de retour à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la compilation se termine avec succès, vous verrez le message suivant :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

## Flashage du firmware

:::tip
Exécutez la commande ci-dessous pour flasher le firmware
:::

    ```bash
    sudo fly-flash -d gemini-v3 -h -f ~/klipper/out/klipper.bin
    ```
