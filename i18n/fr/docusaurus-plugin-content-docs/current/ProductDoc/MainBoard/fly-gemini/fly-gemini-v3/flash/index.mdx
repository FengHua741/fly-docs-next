# Notice sur la programmation de la mémoire interne

## Comment déterminer quel type de firmware vous avez besoin

    :::tip

    * Notez bien que vous n'avez besoin de flasher qu'un seul type de firmware

    :::

    * Gemini V3 peut être connecté à l'ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont présents, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware pont USB</Button>
        * Avantage : peut se connecter à plusieurs périphériques sans occuper de ports USB
        * Inconvénient : peut causer des erreurs telles qu'une erreur de temporisation lors du retour à zéro en utilisant `KlipperScreen`.
