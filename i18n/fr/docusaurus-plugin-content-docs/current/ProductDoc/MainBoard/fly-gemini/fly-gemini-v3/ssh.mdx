## Connexion au poste de travail SSH

* Vous pouvez vous connecter à l'appareil via le port série ou le WIFI
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre outil de terminal SSH
* Assurez-vous que la carte SD ou M2WE est gravée avec **le système correspondant au poste de travail**
* **Si le poste de travail ne démarre pas correctement, veuillez retirer les périphériques tels que les pilotes, les limites, les ventilateurs, etc. !!!**
* **Veuillez vous connecter au SSH via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, consultez le tutoriel sur la connexion au poste de travail via le WIFI ou le câble Ethernet**

        ## Utilisation de l'adresse IP réseau pour se connecter au poste de travail SSH

        * Connectez-vous au backend du routeur auquel le poste de travail est connecté
        * Assurez-vous que le système est bien démarré
        * Assurez-vous de connaître le nom d'utilisateur et le mot de passe du poste de travail, le nom d'utilisateur par défaut est **fly** et le mot de passe par défaut est **mellow** pour le système FLY
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter à SSH via l'adresse IP
        * Ouvrez MobaXterm, effectuez les paramétrages suivants, entrez l'adresse IP que vous avez vue précédemment, ou consultez-la dans le backend du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Note : Dans l'étape 4, entrez le nom d'utilisateur du poste de travail**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'affiche, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe du poste de travail et appuyez sur Entrée

        **Note : Ici, le mot de passe saisi ne sera pas visible, appuyez directement sur Entrée après avoir saisi**

        Le nom d'utilisateur par défaut est : `fly` , le mot de passe par défaut est : `mellow`
