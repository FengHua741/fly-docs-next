# Firmware du bootloader

## Entrée en mode de programmation

    :::tip
    
    Le bouton à droite est le `BOUTON BOOT`, et celui à gauche est le `BOUTON RAZ`

    :::

    * Il existe deux méthodes pour entrer en mode de programmation
    * Première méthode : Éteignez complètement la carte mère, appuyez sur le bouton BOOT puis branchez le câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur votre appareil
    * Deuxième méthode : Après avoir branché le câble de transmission de données d'un côté sur la carte mère et de l'autre côté sur votre appareil, appuyez sur le bouton BOOT, puis appuyez sur le bouton RAZ, relâchez ensuite le bouton RAZ puis le bouton BOOT
