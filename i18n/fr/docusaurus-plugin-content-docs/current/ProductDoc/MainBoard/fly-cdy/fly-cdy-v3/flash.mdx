# Compilation et écriture du firmware

    :::tip

    * CDV utilise par défaut une carte TF pour l'écriture du firmware
    * Préparez une carte TF d'une capacité ne dépassant pas `32G`
    * Les cartes mémoire de marque non officielle peuvent provoquer un échec de l'écriture, il est préférable d'utiliser les cartes TF recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte TF doit être de classe `10` ou supérieure, il est conseillé d'utiliser des cartes TF de marque SanDisk, etc.

    :::
