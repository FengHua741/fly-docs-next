# Compilation et flashage du firmware

    :::tip

    * CDV utilise par défaut la carte SD pour le flashage du firmware
    * Préparez une carte SD d'une capacité inférieure ou égale à `32G`
    * Les cartes SD de marque non officielle peuvent entraîner un échec du flashage, il est recommandé d'utiliser les cartes SD recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte SD doit être de classe `10` ou supérieure, il est conseillé d'utiliser des cartes SD de marque SanDisk, par exemple

    :::

    * Appuyez sur `Q` pour afficher **Save configuration**, puis appuyez sur `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Vous devriez maintenant avoir sauvegardé la configuration et être revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela prendra un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la sortie finale ressemble au texte suivant, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
