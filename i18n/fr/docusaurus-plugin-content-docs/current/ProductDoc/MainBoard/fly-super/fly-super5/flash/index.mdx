    :::tip

    * Le Super5 Pro utilise par défaut une carte TF pour la gravure du firmware
    * Préparez une carte TF d'une capacité inférieure ou égale à `32G`
    * Les cartes mémoire de marque inconnue peuvent provoquer des échecs de gravure, veuillez utiliser si possible les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte TF doit être de niveau `class10` ou supérieur, il est conseillé d'utiliser des cartes TF de marque SanDisk

    :::

# Instructions pour la gravure du firmware

    :::tip

    * Le Super5 Pro sort de l'usine avec un bootloader pré-gravé sur la carte TF, pas besoin de le graver à nouveau

    :::


## Comment savoir quel type de firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de graver qu'un seul type de firmware

    :::
    
* Le Super5 Pro peut se connecter à une machine supérieure de la manière suivante
* <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
    * Avantage : communication fiable, stable
    * Inconvénient : si plusieurs périphériques USB sont présents, cela peut affecter la transmission des données
* <Button variant="contained" disableElevation href="./can">Firmware USB Bridge</Button>
    * Avantage : peut connecter plusieurs appareils sans occuper de ports USB
    * Inconvénient : peut causer des erreurs telles que des retards de positionnement en raison de l'utilisation conjointe de `KlipperScreen`
