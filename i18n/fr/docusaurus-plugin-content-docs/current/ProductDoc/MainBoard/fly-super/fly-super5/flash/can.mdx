    * Appuyez sur la touche `Q`, puis sur **Enregistrer la configuration** et enfin sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être revenu à l'interface de ligne de commande
    * Tapez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si vous voyez le message ci-dessous, la compilation a réussi

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

            4. Flashage du firmware

            * Voici la commande pour flasher via DFU
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. Si le message ``File downloaded successfully`` s'affiche, le flashage est réussi

            :::warning

            Ignorez l'erreur qui apparaît après la ligne indiquée par la flèche rouge !!!

            :::

                <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super5/flash/img/boot.webp').default} size="100%" align="center" />

            :::tip

            Une fois le flashage terminé, retirez le jumper de la carte mère et appuyez sur `rst` pour redémarrer la carte mère
