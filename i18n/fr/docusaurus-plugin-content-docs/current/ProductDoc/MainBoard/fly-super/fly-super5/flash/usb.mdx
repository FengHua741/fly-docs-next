# Compilation du firmware USB

## Début de la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur la touche `Q`, puis sur **Save configuration**, puis sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être de retour à l'interface de commande
    * Entrez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la compilation est réussie, vous verrez le message suivant :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

            4. Flashing du firmware

            * Voici la commande pour flasher via DFU
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. Si vous voyez le message "File downloaded successfully" comme indiqué par la flèche sur l'image, le flashing a réussi

            :::warning

            Ignorez l'erreur après la ligne indiquée par la flèche rouge !!!
