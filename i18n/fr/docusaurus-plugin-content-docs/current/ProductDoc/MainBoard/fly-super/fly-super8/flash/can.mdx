## Début de la compilation

    <SSH />

    <CAN />

    * Appuyez sur la touche `Q`, **Save configuration** apparaît, puis appuyez sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * Maintenant, la configuration devrait être sauvegardée et vous devriez être revenu à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela prendra un peu de temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la compilation est réussie, le contenu suivant sera affiché à la fin

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

            4. Programmation du firmware

            * Voici la commande pour flasher via DFU
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. Si le contenu indiqué par la flèche apparaît, le firmware a été programmé avec succès

            :::warning

            Ignorez l'erreur après la ligne indiquée par la flèche rouge !!!
