### Sauterelles pour le pilote de la carte mère

* **Mode STEP/DIR.** Dans ce mode, le sous-diviseur est défini en utilisant un capot de saute-rue, et le courant est ajusté en régulant un potentiomètre. Les pilotes les plus courants qui nécessitent une recherche dans le tableau de configuration fourni par le fabricant pour définir le sous-diviseur sont A4988 et 8825.
* **Mode UART.** Les pilotes les plus courants utilisant ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilotage peuvent communiquer de manière asynchrone en série avec le contrôleur via UART, et le sous-diviseur, le courant de fonctionnement et le mode silencieux du pilote peuvent être configurés en modifiant le fichier de configuration.
* **Mode SPI.** Les pilotes les plus courants utilisant ce mode sont : TMC5160, TMC2130, TMC2240, etc. Ces puces de pilotage peuvent également configurer le sous-diviseur, le courant de fonctionnement et le mode silencieux du pilote en modifiant le fichier de configuration.
