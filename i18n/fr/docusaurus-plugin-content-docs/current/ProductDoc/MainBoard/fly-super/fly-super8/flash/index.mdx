    :::tip

    * Super8 utilise par défaut une carte SD pour la gravure du firmware
    * Préparez une carte SD dont la capacité n'excède pas `32G`
    * Les cartes SD non officielles peuvent entraîner un échec de la gravure, veuillez utiliser les cartes SD recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte SD doit être de classe `10` ou supérieure, il est conseillé d'utiliser des cartes SD de marque SanDisk, etc.

    :::

# Instructions pour la gravure du firmware


## Comment savoir quel type de firmware vous avez besoin

    :::tip

    * Notez que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
    * Super8 peut se connecter à un ordinateur hôte de la manière suivante
    * <Button variant="contained" disableElevation href="./usb">Firmware USB</Button>
        * Avantage : communication fiable et stable
        * Inconvénient : si plusieurs périphériques USB sont présents, cela peut affecter la transmission des données
    * <Button variant="contained" disableElevation href="./can">Firmware USB pont</Button>
        * Avantage : peut connecter plusieurs appareils sans occuper de ports USB
        * Inconvénient : peut causer des erreurs telles que des dépassements de temps d'attente lors de l'utilisation avec `KlipperScreen`.
