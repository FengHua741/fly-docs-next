
            4. Flasher le firmware

            * Voici la commande DFU pour flasher :
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. Si vous voyez le message ``File downloaded successfully`` comme indiqué par la flèche ci-dessous, le flashage est réussi
