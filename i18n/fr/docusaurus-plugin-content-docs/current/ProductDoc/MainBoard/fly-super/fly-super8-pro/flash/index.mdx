    :::tip

    * Le Super8 PRO utilise par défaut une carte TF pour la gravure du firmware
    * Préparez une carte TF d'une capacité inférieure ou égale à `32G`
    * Les cartes mémoire non officielles peuvent entraîner un échec de la gravure, veuillez utiliser si possible les cartes mémoire recommandées ci-dessous pour mettre à jour le firmware
    * La vitesse de transmission de la carte TF doit être de niveau `class10` ou supérieur, il est conseillé d'utiliser des cartes TF de marque SanDisk

    :::

# Instructions pour la gravure du firmware

    :::tip

    * Le bootloader de la carte TF du Super8 Pro est préinstallé en usine, vous n'avez pas besoin de le réinstaller

    :::

## Comment savoir quel type de firmware vous avez besoin

    :::tip

    * Veuillez noter que vous n'avez besoin de flasher qu'un seul type de firmware

    :::
    
* Le Super8 PRO peut être connecté à l'ordinateur hôte de la manière suivante
* <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">Firmware USB</Button>
    * Avantages : communication fiable et stable
    * Inconvénients : peut affecter la transmission des données si plusieurs périphériques USB sont présents
* <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">Firmware USB pont</Button>
    * Avantages : peut se connecter à plusieurs appareils sans occuper de ports USB
    * Inconvénients : peut causer des erreurs telles qu'un délai de retour à zéro lors de l'utilisation avec `KlipperScreen`
