# Compilation du firmware USB

## Début de la compilation

    <SSH />

    <MakeOperate/>

    <USB />

    * Appuyez sur la touche `Q`, lorsque **Save configuration** apparaît, appuyez ensuite sur la touche `Y`
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * La configuration devrait maintenant être sauvegardée et vous devriez être de retour à l'interface de ligne de commande
    * Entrez `make -j4` pour commencer la compilation, cela peut prendre un certain temps
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * Si la compilation est réussie, le message suivant sera affiché :

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```

            4. Flashage du firmware

            * Voici la commande pour flasher via dfu :
                ```bash
                cd ~/klipper
                ```
                ```bash
                make flash FLASH_DEVICE=0483:df11
                ```
            5. Si vous voyez le message **File downloaded successfully** comme indiqué par la flèche dans l'image ci-dessous, le flashage a réussi
