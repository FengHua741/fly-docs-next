### Sautez les explications pour le branchement de la carte mère

* **Mode STEP/DIR.** Dans ce mode, en utilisant un capot de saut, on définit la subdivision, et en ajustant le potentiomètre, on règle le courant. Les pilotes les plus courants comme A4988, 8825 nécessitent de consulter le tableau de configuration de subdivision fourni par le fabricant, puis de définir la subdivision avec un capot de saut.
* **Mode UART.** Les pilotes les plus courants utilisant ce mode sont : TMC2208, TMC2209, TMC2226, etc. Ces puces de pilotage peuvent communiquer de manière asynchrone en série avec le contrôleur via UART, et la subdivision, le courant de fonctionnement et le mode silencieux du pilote peuvent être configurés en modifiant le fichier de configuration.
* **Mode SPI.** Les pilotes les plus courants utilisant ce mode sont : TMC5160, TMC2130, TMC2240, etc. Ces puces de pilotage peuvent également configurer la subdivision, le courant de fonctionnement et le mode silencieux du pilote en modifiant le fichier de configuration.
