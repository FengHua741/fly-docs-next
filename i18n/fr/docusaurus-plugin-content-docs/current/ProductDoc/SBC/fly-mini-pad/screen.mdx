# Configuration de la connexion et de la configuration de l'écran

:::info

* FLY-TOOLS nécessite que l'ordinateur de bord soit connecté à Internet pour accéder à `IP+:9999` depuis l'ordinateur de bord.
* Par exemple `192.168.6.110:9999`
