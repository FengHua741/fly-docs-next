# Connexion au PC supérieur via SSH


* Vous pouvez utiliser le port série ou le WIFI pour vous connecter à l'appareil.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou tout autre outil de terminal SSH.
* Assurez-vous que la carte SD ou M2WE est gravée avec **le système correspondant au PC supérieur**.
* **Si le PC supérieur ne démarre pas correctement, retirez les périphériques tels que les pilotes, les limites, les ventilateurs, etc. !!!**
* **Enfin, connectez-vous via le réseau pour compiler les firmwares plus facilement !!!**
* **Pour compiler des firmwares, consultez le tutoriel sur la connexion au PC supérieur via le WIFI ou le câble Ethernet.**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utilisation de l'adresse IP réseau pour se connecter au PC supérieur via SSH

        * Connectez-vous à l'interface web de la **box Internet** du PC supérieur.
        * Assurez-vous que le système a démarré correctement.
        * Assurez-vous que vous connaissez le nom d'utilisateur et le mot de passe du PC supérieur. Le nom d'utilisateur par défaut pour le système FLY est **fly**, et le mot de passe par défaut est **mellow**.
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter via IP à SSH.
        * Ouvrez MobaXterm et effectuez les paramétrages suivants, en entrant l'adresse IP que vous avez vue précédemment, ou que vous pouvez voir dans l'interface web de la box Internet.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Notez : Dans cette étape 4, entrez le nom d'utilisateur du PC supérieur.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'affiche, cliquez sur `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe du PC supérieur et appuyez sur Entrée.

        **Notez : Ici, le mot de passe n'est pas visible pendant la saisie. Entrez-le simplement et appuyez sur Entrée.**

        Le nom d'utilisateur par défaut pour le système FLY est : `fly`, et le mot de passe par défaut est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * La connexion est réussie.
