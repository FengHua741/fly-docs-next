# Présentation

    Fly-MINI-Pad est une station supérieure de débutant pour Klipper

## Caractéristiques

    * CPU : puces Allwinner H3 à haute performance, quad-core 64-bit Cortex-A7
    * GPU : Mali-400 MP2
    * RAM : 512 Mo DDR3
    * ROM : supporte jusqu'à 128 Go de carte SD
    * WIFI : module intégré de 2,4 GHz
    * Périphériques : SPI x1, USB x2, Type-C x1
    * Interfaces : trois connecteurs XH2.54-2P prenant en charge respectivement, POWER-IN pour l'alimentation avec une entrée maximale de 24 V, CAN-BUS pour la connexion au bus CAN, LI-PO pour l'alimentation/charge de batterie LiPo de 3,7 V avec protection
    * Système préinstallé de klipper, moonraker, mainsail, fulidd, klipper-screen, plug-in de compensation de résonance numpy et FLY-Tools, sans opérations complexes comme le changement de source, plus facile pour les débutants (nécessite un achat de carte TF)
    * Écran capacitif/résistif, résolution de 320x480
