# Connexion au WiFi

## Installer l'antenne sur le MINI PAD

    * L'endroit indiqué par la flèche rouge est l'interface de l'antenne WiFi
    * La prise de l'antenne WiFi est `ipex1`
    * Installez l'antenne sur le MINI PAD, sinon, elle pourrait ne pas être en mesure de se connecter à cause d'un signal faible
