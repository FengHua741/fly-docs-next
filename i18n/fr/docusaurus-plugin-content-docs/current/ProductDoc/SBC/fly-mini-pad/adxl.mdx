# Utilisation de l'accéléromètre

    :::tip

    * Mini Pad peut connecter directement l'accéléromètre via l'interface embarquée ou utiliser le capteur d'accélération intégré dans la carte mère, etc.

    :::

## Interface embarquée pour connecter l'accéléromètre externe

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-mini-pad/img/adxl.webp').default} size="80%" align="left" />

    * Configuration de référence

    ```cfg
    [mcu host]
    serial: /tmp/klipper_host_mcu

    [adxl345]
    cs_pin: host:None
    spi_bus: spidev1.0

    [resonance_tester]
    accel_chip: adxl345
    #accel_chip_y: adxl345 bed
    probe_points:
        100, 100, 20  # un exemple
    ```
