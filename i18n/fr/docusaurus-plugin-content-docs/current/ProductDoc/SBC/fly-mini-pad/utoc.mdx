# Utilisation de l'UTOC embarqué

:::danger À savoir

Il n'est pas recommandé d'utiliser deux UTOIC, la gravure et la connexion seront très compliquées

:::
    * Si vous ne souhaitez pas utiliser l'UTOC embarqué, exécutez la commande ci-dessous, aucune indication ne sera affichée
        ```bash
        sudo sed -i 's/usbhost2/offusb2/g' /boot/FlyOS-Env.txt
        ```
    * Méthode de restauration, exécution sans indication
        ```bash
        sudo sed -i 's/offusb2/usbhost2/g' /boot/FlyOS-Env.txt
        ```
    * Le MINI Pad est déjà équipé d'un UTOC
