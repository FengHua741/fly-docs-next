# Connexion au WiFi
:::tip
- Les étapes et méthodes pour connecter le WiFi dans le système `FLYOS-FAST` et le système `FLYOS-Armbian` sont légèrement différentes, veuillez choisir la leçon correspondant au système que vous avez gravé !
:::

:::warning Attention
    * Le PC principal FLY nécessite d’utiliser le module `M2WE` ou le module `WiFi 2,4G` ou une carte réseau sans fil USB compatible Linux sans pilote, sinon le WiFi ne fonctionnera pas.
    * Veuillez noter que le PC principal ne prend pas en charge les noms de réseau WiFi en chinois et ne prend pas en charge les réseaux WiFi `2,4G et 5G combinés`.
    * Si vous utilisez un `mini pad` ou un `lite2/lite2.1`, ignorez cet avertissement.
:::
