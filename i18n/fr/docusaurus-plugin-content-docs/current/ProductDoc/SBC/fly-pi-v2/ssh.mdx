# Connexion à la station supérieure via SSH


* Il est possible d'utiliser le port série ou le WIFI pour se connecter à l'appareil
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre outil de terminal SSH
* Vérifiez que la carte SD ou M2WE a été gravée avec **le système correspondant à la station supérieure**
* **Si la station supérieure ne démarre pas correctement, retirez les périphériques tels que le pilote, les limites, les ventilateurs, etc. !!!**
* **Enfin, connectez-vous via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, consultez le tutoriel sur la connexion à la station supérieure via le WIFI ou le câble Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utilisation de l'adresse IP réseau pour se connecter à la station supérieure via SSH

        * Connectez-vous à l'interface arrière de la **routeur** auquel la station supérieure est connectée
        * Assurez-vous que le système a démarré correctement
        * Assurez-vous de connaître le nom d'utilisateur et le mot de passe de la station supérieure, par défaut pour le système FLY, l'utilisateur est **fly** et le mot de passe est **mellow**
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter à SSH via l'adresse IP
        * Ouvrez MobaXterm, effectuez les paramétrages suivants, entrez l'adresse IP que vous avez vue précédemment, vous pouvez également la trouver dans l'interface arrière du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Note : Ici, au quatrième étape, entrez le nom d'utilisateur de la station supérieure**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * La première connexion peut faire apparaître cette fenêtre, cliquez sur `Accept` pour continuer

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Une fois la demande de mot de passe affichée, entrez le mot de passe de la station supérieure et appuyez sur Entrée

        **Note : Ici, le mot de passe entré ne sera pas visible, appuyez directement sur Entrée après l'avoir saisi**

        L'utilisateur par défaut est : `fly` , le mot de passe par défaut est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * La connexion est réussie
