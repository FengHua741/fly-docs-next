# Connexion à la station de travail via SSH

* Vous pouvez utiliser le **port série** ou le **WiFi** pour vous connecter à l'appareil.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou tout autre outil de terminal SSH.
* Assurez-vous que la carte SD ou M2WE est gravée avec un système compatible avec la station de travail.
* **Si la station de travail ne démarre pas correctement, retirez les périphériques externes tels que le driver, les limites, les ventilateurs, etc. !**

:::attention Attention
- Si votre station de travail est en première utilisation, veuillez impérativement utiliser la **connexion série pour accéder à SSH de la station de travail**.
- Lors de la compilation du firmware, utilisez impérativement la **connexion WiFi pour accéder à SSH de la station de travail**, afin de faciliter la compilation et la gravure du firmware !
:::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        * Attendez quelques instants, cette image indique que le démarrage est terminé.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="60%" align="center" />

        * **Si après quelques minutes, l'écran reste vide, appuyez plusieurs fois sur la touche Entrée.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-5.webp').default} size="60%" align="center" />

        * **Après avoir appuyé sur Entrée, vous verrez le contenu suivant.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-6.webp').default} size="60%" align="center" />

        :::attention
        - Vous êtes connecté en tant qu'utilisateur root, saisissez : `su fly` pour passer à un utilisateur normal avant de poursuivre les opérations !!!

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-13.webp').default} size="60%" align="center" />

        * Si vous souhaitez consulter l'adresse IP, tapez dans SSH : `ip a | grep inet` puis appuyez sur Entrée. Recherchez l'IP dans le même sous-réseau que votre routeur pour y accéder.

    <TabItem value="net" label="Connexion WiFi FLYOS-Armbian" default>
        ## Connexion WiFi FLYOS-Armbian à SSH de la station de travail

        * Assurez-vous que le système de la station de travail est déjà démarré correctement.
        * Assurez-vous de connaître le nom d'utilisateur et le mot de passe de la station de travail. Le nom d'utilisateur par défaut de FLYOS-Armbian est `fly`, et le mot de passe est `mellow`.
        * La connexion WiFi à SSH de la station de travail repose sur une condition préalable : votre station de travail est déjà connectée au routeur et vous connaissez son adresse IP.

        * Ouvrez MobaXterm, configurez-le comme illustré ci-dessous, en entrant l'adresse IP de la station de travail obtenue lors de la connexion série dans le champ `Remote host`. Vous pouvez également obtenir l'adresse IP depuis le panneau de configuration du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Remarque : Entrez ici le nom d'utilisateur de la station de travail**
        :::astuce
        - Dans la quatrième étape, entrez le nom d'utilisateur de la station de travail. Pour une station de travail FLY utilisant le système FLYOS-Armbian, entrez `fly`. Pour les autres stations de travail, entrez le nom d'utilisateur approprié selon vos besoins.
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * Lors de la première connexion, une fenêtre s'affiche. Cliquez sur `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Saisissez le mot de passe de la station de travail, puis appuyez sur Entrée.
        :::astuce
        - Ici, le mot de passe n'est pas affiché. Appuyez directement sur Entrée après avoir saisi le mot de passe.
        :::

        Le nom d'utilisateur par défaut de FLYOS-Armbian est : `fly`, et le mot de passe est : `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        * Si une fenêtre similaire à celle ci-dessous apparaît, elle vous demande si vous souhaitez sauvegarder le mot de passe de l'utilisateur. Cliquez sur `Non`.

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Si vous voyez l'image suivante, la connexion est réussie !

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
