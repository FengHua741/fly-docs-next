# Connexion à la station de travail via SSH

* Il est possible d'utiliser **RS-232** ou **WiFi** pour se connecter à l'appareil.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou tout autre outil de terminal SSH.
* Assurez-vous que la carte SD ou M2WE a été gravée avec un système compatible avec la station de travail.
* **Si la station de travail ne démarre pas correctement, retirez les périphériques externes tels que les drivers, les limites, les ventilateurs, etc. !**

:::tip Note
- Si votre station de travail est en première utilisation, il est essentiel d'utiliser une **connexion RS-232 pour accéder au SSH de la station de travail**.
- Lors de la compilation du firmware, veuillez utiliser une **connexion WiFi pour accéder au SSH de la station de travail**, afin de faciliter la compilation et le flashage du firmware !
- Les opérations de connexion au SSH pour les systèmes `FLYOS-Armbian` et `FLYOS-FAST` diffèrent légèrement. Veuillez choisir le tutoriel correspondant en fonction du système gravé sur votre appareil !
::>

        * Ouvrez `MobaXterm` et effectuez les paramétrages suivants, en sélectionnant le port identifié précédemment dans le Gestionnaire de périphériques.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        * Si tout fonctionne correctement, vous verrez l'écran de démarrage.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-3.webp').default} size="60%" align="center" />

        * Attendez quelques instants, et si vous voyez cet écran, cela signifie que le démarrage s'est effectué.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-4.webp').default} size="60%" align="center" />

        * **Si après plusieurs minutes, l'écran ne montre aucun affichage, appuyez plusieurs fois sur la touche Entrée.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-5.webp').default} size="60%" align="center" />

        * **Après avoir appuyé sur Entrée, vous verrez le contenu suivant.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-6.webp').default} size="60%" align="center" />

        :::tip
        - À ce stade, vous êtes connecté en tant qu'utilisateur `root`. Veuillez entrer : `su fly` pour passer à un utilisateur normal avant de poursuivre les étapes suivantes !!!
        :::
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-13.webp').default} size="60%" align="center" />

        * Si vous souhaitez voir l'adresse IP, entrez `ip a | grep inet` dans le terminal SSH et cherchez l'adresse IP correspondante à votre routeur dans les résultats retournés.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-8.webp').default} size="70%" align="center" />

    <TabItem value="net" label="Connexion WiFi pour le système FLYOS-Armbian" default>
        ## Connexion WiFi au SSH de la station de travail FLYOS-Armbian

        * Assurez-vous que le système de la station de travail fonctionne correctement.
        * Assurez-vous d'avoir clairement identifié le nom d'utilisateur et le mot de passe de la station de travail. Le nom d'utilisateur par défaut pour FLYOS-Armbian est `fly`, et le mot de passe par défaut est `mellow`.
        * La connexion WiFi au SSH de la station de travail nécessite que celle-ci soit déjà connectée à un routeur et que vous connaissiez son adresse IP.

        * Ouvrez MobaXterm et configurez-le comme indiqué ci-dessous, en remplissant l'adresse IP de la station de travail que vous avez obtenue via la connexion RS-232 ou consultée depuis le panneau administratif du routeur dans le champ `Remote host`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Note : Dans cette étape 4, entrez le nom d'utilisateur de la station de travail.**
        :::warning
        - L'étape 4 consiste à entrer le nom d'utilisateur de la station de travail. Si vous utilisez un FLYOS-Armbian, entrez `fly`. Pour les autres stations de travail, entrez le nom d'utilisateur approprié selon vos besoins.
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * Lors de la première connexion, un message d'avertissement apparaîtra, cliquez sur `Accept`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Saisissez le mot de passe de la station de travail et appuyez sur Entrée.
        :::warning
        - Le mot de passe n'apparaîtra pas pendant la saisie, appuyez directement sur Entrée après l'avoir saisi.
        :::
        
        Le nom d'utilisateur par défaut pour FLYOS-Armbian est : `fly`, et le mot de passe par défaut est : `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        * Si un message d'alerte comme celui ci-dessous apparaît, cliquez sur `No` pour refuser la sauvegarde du mot de passe utilisateur.
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Si vous voyez l'écran suivant, la connexion a réussi !

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
