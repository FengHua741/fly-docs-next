# Connexion à l'ordinateur principal via SSH


* Vous pouvez utiliser le port série ou le Wi-Fi pour vous connecter à l'appareil.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre outil de terminal SSH.
* Veuillez vous assurer que la carte SD ou M2WE a été gravée avec **le système correspondant à l'ordinateur principal**.
* **Si l'ordinateur principal ne démarre pas normalement, veuillez retirer les périphériques externes tels que les pilotes, les limites, les ventilateurs, etc. !!!**
* **Enfin, veuillez vous connecter via le réseau à SSH pour compiler les firmwares !!!**
* **Pour compiler des firmwares, consultez le tutoriel sur la connexion à l'ordinateur principal via Wi-Fi ou Ethernet.**

        ## Connexion à l'ordinateur principal via IP réseau

        * Connectez-vous à l'interface de configuration du **routeur** auquel l'ordinateur principal est connecté.
        * Assurez-vous que le système a bien **démarré correctement**.
        * Assurez-vous de connaître **le nom d'utilisateur et le mot de passe de l'ordinateur principal**, par défaut le nom d'utilisateur est **fly** et le mot de passe est **mellow**.
        * Si la carte mère est démarrée et connectée au réseau, vous pouvez vous connecter via IP à SSH.
        * Ouvrez MobaXterm et configurez comme suit, en entrant l'IP que vous avez vue précédemment, ou consultez l'interface de configuration du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Remarque : Entrez ici le nom d'utilisateur de l'ordinateur principal.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * La première connexion peut afficher une fenêtre de confirmation, cliquez sur `Accept` pour continuer.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe et appuyez sur Entrée.

        **Remarque : Le mot de passe n'est pas affiché pendant la saisie, appuyez directement sur Entrée après avoir saisi le mot de passe.**

        Les valeurs par défaut du système FLY sont : Nom d'utilisateur : `fly`, Mot de passe : `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * La connexion est réussie.
