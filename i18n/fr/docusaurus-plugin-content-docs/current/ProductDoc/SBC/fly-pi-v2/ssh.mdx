# Connexion SSH à l'ordinateur principal

* Vous pouvez utiliser **RS232** ou **WiFi** pour vous connecter à l'appareil
* Veuillez préparer à l'avance **MobaXterm_Personal** ou d'autres outils de terminal SSH
* Veuillez vous assurer que la carte SD ou M2WE est gravée avec **un système compatible avec l'ordinateur principal**
* **Si l'ordinateur principal ne peut pas démarrer normalement, veuillez retirer les périphériques tels que le driver, les limites et les ventilateurs !!!!**

:::warning Attention
- Si votre ordinateur principal est un premier démarrage, il est essentiel d'utiliser **la connexion RS232 pour accéder à SSH de l'ordinateur principal**
- Lors de la compilation du firmware, il est impératif d'utiliser **la connexion WiFi pour accéder à SSH de l'ordinateur principal**, afin de faciliter la compilation et le flashage du firmware !
- Les opérations de connexion à SSH pour le système `FLYOS-Armbian` et le système `FLYOS-FAST` diffèrent, veuillez choisir le tutoriel correspondant en fonction du système gravé sur votre appareil !!!
:::

    <TabItem value="net" label="Connexion WiFi au système FLYOS-Armbian" default>
        ## Connexion WiFi au système FLYOS-Armbian via SSH

        * Assurez-vous que le système de l'ordinateur principal est déjà **démarré correctement**
        * Assurez-vous de connaître **nom d'utilisateur et mot de passe de l'ordinateur principal**. Le nom d'utilisateur par défaut de FLYOS-Armbian est `fly`, et le mot de passe par défaut est `mellow`
        * La condition préalable pour se connecter à SSH via WiFi est que votre **ordinateur principal est connecté au routeur, et vous savez l'IP de l'ordinateur principal**  

        * Ouvrez MobaXterm, configurez-le comme indiqué ci-dessous, dans la zone `Remote host`, saisissez l'IP de l'ordinateur principal que vous avez obtenu lors de la connexion RS232, ou consultez l'IP de l'ordinateur principal via le panneau de configuration du routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **Remarque : Dans cette étape 4, entrez le nom d'utilisateur de l'ordinateur principal**
        :::tip
        - Dans la quatrième étape, entrez le nom d'utilisateur de l'ordinateur principal. Pour un ordinateur principal FLY utilisant le système FLYOS-Armbian, entrez `fly`, pour d'autres ordinateurs principaux, entrez le nom d'utilisateur correct en fonction de vos besoins
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * Lors de la première connexion, une fenêtre de confirmation s'affichera, cliquez sur `Accept`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * Saisissez le mot de passe, appuyez sur Entrée après avoir saisi le mot de passe de l'ordinateur principal
        :::tip
        - Ici, le mot de passe n'est pas affiché, appuyez directement sur Entrée après avoir saisi
        :::
        
        Le nom d'utilisateur par défaut du système FLYOS-Armbian est : `fly`, et le mot de passe par défaut est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        **Si une fenêtre similaire à celle-ci apparaît, cette fenêtre demande si vous souhaitez enregistrer le mot de passe de l'utilisateur, cliquez sur `No`**

        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * Si vous voyez l'image suivante, cela signifie que la connexion est réussie !
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
