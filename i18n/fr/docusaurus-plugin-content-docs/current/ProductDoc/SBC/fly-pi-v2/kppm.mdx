# Fonction de reprise d'impression après coupure de courant Klipper

## Aperçu

- La fonction de reprise d'impression après coupure de courant Klipper permet à Klipper de reprendre automatiquement l'état d'impression après une coupure de courant, lorsqu'il est à nouveau alimenté.
- Cette fonction nécessite l'utilisation du système FAST pour être utilisée.
- Le PI-V2 nécessite l'utilisation du module de coupure de courant KPPM pour être utilisé.
- Non compatible avec les modèles où l'axe Z se déplace après une panne de courant.
