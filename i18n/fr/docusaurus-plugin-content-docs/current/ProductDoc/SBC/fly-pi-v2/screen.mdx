* FLY-TOOLS nécessite que la machine hôte soit connectée à Internet pour accéder à `IP+:9999` après avoir entré l'adresse IP de la machine hôte.
* Par exemple `192.168.6.110:9999`

# Configuration de l'écran TFT

    <TabItem value="tft" label="Configuration de l'écran TFT capacitif" default>
## Méthode un    
         * La configuration de l'écran capacitif nécessite d'utiliser la commande ci-dessous pour configurer en `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Assurez-vous que le commutateur sur le dos de l'écran est réglé sur `Cap`
    * Utilisez la commande ci-dessous pour vérifier si la modification a été effectuée avec succès
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Pour modifier une configuration d'écran résistif en écran capacitif, utilisez la commande ci-dessous

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-r/' -i /boot/FlyOS-Env.txt
```
                                * Assurez-vous que le commutateur sur le dos de l'écran est réglé sur `Resi`
        * Utilisez la commande ci-dessous pour vérifier si la modification a été effectuée avec succès
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
