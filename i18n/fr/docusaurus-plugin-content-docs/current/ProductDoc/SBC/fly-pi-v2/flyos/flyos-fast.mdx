## Introduction au système

* FlyOS-Fast est un système Linux allégé, spécialement conçu pour l'impression 3D, avec un temps de démarrage rapide (seulement 30 secondes). Il est compatible avec toutes les plateformes principales de Fly.
* Optimisation du noyau pour améliorer les performances en temps réel de Klipper.
* Optimisation du système de fichiers pour éviter tout dommage au système en cas d'interruption directe de courant.
* Pré-installation des services Klipper, RRF, Mainsail, Fluidd, Moonraker, KlipperScreen, etc.
* Prise en charge du basculement entre Klipper et RRF.
* Prise en charge du basculement en ligne entre Mainsail et Fluidd.
* Prise en charge de l'impression via USB.
* En combinaison avec le module KPPM, il est possible de reprendre une impression après coupure de courant.
* Prise en charge de la configuration avancée des langues par défaut et des noms d'imprimante pour Fluidd, etc.
* Pour plus de fonctionnalités de personnalisation OEM, comme le logo de démarrage, les fichiers de configuration pré-définis ou la désactivation de certaines pages dans Mainsail, contactez-nous.

## Préparation

1. Téléchargement de l'image du système
    - Lien de téléchargement de l'image : [Téléchargement de l'image Piv2](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.exe/version/50907014/overview "Cliquez ici pour accéder au lien") (compatible uniquement avec Windows)  
    - Lien de téléchargement de l'image : [Téléchargement de l'image Piv2](https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/flyos-fast/FlyOS-Fast-H5.7z/version/50907002/overview "Cliquez ici pour accéder au lien") (compatible avec Windows, MacOS, Linux)

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/flyos-fast_download.webp').default} size="90%" align="center" />


2. Téléchargement de l'outil de gravage. Le tutoriel utilisera `Rufus` à titre d'exemple. Vous pouvez également utiliser d'autres outils, comme `Etcher`, `Win32 Disk Imager`, etc.
    - Adresse de téléchargement de Rufus : [Cliquer pour télécharger en français](https://rufus.ie/zh/#download) ou [Cliquez pour télécharger en anglais](https://rufus.ie/en/#download) (compatible uniquement avec Windows).
    - Adresse de téléchargement de Etcher : [Cliquer pour télécharger](https://etcher.balena.io/#download-etcher) (compatible avec Windows, MacOS, Linux).
    - Adresse de téléchargement de Win32 Disk Imager : [Cliquer pour télécharger](https://sourceforge.net/projects/win32diskimager/) (compatible uniquement avec Windows).

3. Le tutoriel utilisera `FlyOS-Fast-H5.exe` à titre d'exemple.

4. Décompression du fichier image du système, obtention du fichier `.img`.
    - Double-cliquez ou ouvrez-le avec un clic droit sur `FlyOS-Fast-H5.exe`, attendez quelques secondes jusqu'à ce que la fenêtre suivante apparaisse (si le système affiche un avertissement de sécurité, choisissez « Plus d'informations » -> « Exécuter »).
    - Vous pouvez choisir le dossier de décompression, puis cliquez sur le bouton `Extract`, attendez la fin du processus de décompression.
    
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/ext1.webp').default} size="60%" align="center" />

    - Après la décompression, un fichier nommé `FlyOS-Fast-H5-vx.x.x.img` sera généré dans le dossier de décompression. `x.x.x` est le numéro de version du système.

## Gravure de l'image

    <Tabs>
        <TabItem value="TF" label="Gravure sur carte TF" default>

            1. - La carte TF utilisée pour la gravure doit être supérieure à 4Go, il est recommandé d'utiliser une carte TF SanDisk de 32Go avec classe 10 ou supérieure (il a été constaté que de nombreuses cartes bon marché ne peuvent pas démarrer après la gravure).
               - La carte TF utilisée pour la gravure doit être insérée dans un lecteur de carte SD, puis connectée à l'ordinateur via le lecteur de carte.
            2. Ouvrez `Rufus`, sélectionnez l'appareil. L'appareil sélectionné doit correspondre en capacité à celui inséré dans l'ordinateur.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash1.webp').default} size="40%" align="center" />
    
            3. Sélectionnez le fichier image, cliquez sur le bouton `Sélectionner` à droite du type de démarrage, puis sélectionnez le fichier `.img` décompressé.
            4. Après avoir sélectionné le fichier image, le type de démarrage changera pour afficher le nom du fichier image.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash2.webp').default} size="40%" align="center" />
            
            5. Cliquez sur le bouton `Démarrer`, si une fenêtre apparaît pour vous demander de nettoyer les données du disque, cliquez sur `OK`.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash3.webp').default} size="50%" align="center" />
            
            6. Une fois la gravure terminée, l'ordinateur émettra un signal sonore et la barre de progression affichera `Prêt`.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash4.webp').default} size="40%" align="center" />
            
            7. Ne retirez pas immédiatement la carte de gravure après la fin de celle-ci.
            8. Ouvrez l'explorateur de fichiers, vérifiez s'il existe un lecteur amovible ou une clé USB nommé `FlyOS-Conf`.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash5.webp').default} size="60%" align="center" />
            
            9. Ouvrez `FlyOS-Conf`, si les deux fichiers ci-dessous sont présents, cela signifie que la gravure a réussi.
            
            <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />
            
            10. Parmi eux, `config.txt` est le fichier de configuration du système, veuillez consulter le contenu suivant. Modifier le fichier de configuration.

## Connexion au WiFi

    1. Utilisez `vscode` ou un autre éditeur de texte professionnel pour ouvrir `config.txt` (**essayez d'éviter d'utiliser le Bloc-notes intégré de Windows pour éditer le fichier de configuration**) 
    <ImageView image={require('@site/docs/DebugDoc/flyos-fast/img/flash6.webp').default} size="70%" align="center" />

    2. Dans l'option `WIFI_SSID=""`, placez votre **nom de réseau WiFi** entre les `""`.  
       Dans l'option `WIFI_PASS="""`, placez votre **mot de passe WiFi** entre les `""`.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi-v2/img/osfast_wifi.webp').default} size="70%" align="center" />  
  
    3. À ce stade, votre station de travail peut être mise sous tension !!  
    Effectuez la prochaine étape : <Button variant="contained" disableElevation href="/docs/ProductDoc/SBC/fly-pi-v2/ssh">Connexion SSH à la station de travail</Button>
