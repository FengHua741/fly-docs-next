# Présentation

Nous sommes ravis de lancer le tout nouveau FLY-Pi V2. Cette carte a les mêmes dimensions d'installation que la Raspberry Pi et peut remplacer la Raspberry Pi comme ordinateur principal. Elle dispose de 1 Go de mémoire DDR3 embarquée, permettant de faire fonctionner un bureau LINUX complet (basé sur Armbian personnalisé et optimisé). Cette carte peut être utilisée avec les firmwares Klipper et RRF, et même fonctionner avec l'écran Klipper lors de l'utilisation de la carte d'outil SHT.
Le but de la conception de cet ordinateur principal est de rendre l'utilisation du firmware Klipper ou RRF plus facile pour les clients. Nous avons optimisé l'image système LINUX pour une utilisation plus adaptée à l'impression 3D, rendant l'installation et l'opération plus simples, même pour quelqu'un avec peu d'expérience. Cela réduit les coûts d'apprentissage par rapport aux autres cartes d'ordinateur principal, vous permettant de vous concentrer davantage sur le plaisir de l'impression 3D (l'achat d'une offre incluant une carte SD préinstallée avec le firmware Klipper permet de l'utiliser immédiatement sans aucune configuration).

## Caractéristiques

* Même format que la Raspberry Pi 3B et 4B
* CPU : Processeur Allwinner H5 à haute performance, 4 cœurs 64 bits Cortex-A53
* GPU : GPU haute performance 6 cœurs Mali 450, taux de remplissage de pixels supérieur à 2,7 gpixel/s
* RAM : 1 Go DDR3 (partagée avec le GPU) (512+512)
* ROM : Prise en charge jusqu'à 128 Go de carte SD
* Périphériques : Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN (module USB vers CAN embarqué, c'est-à-dire module UTOC)
* Interfaces : Interface M.2 embarquée, permettant d'étendre le WIFI, eMMC, etc. (protocole privé, ne pas connecter des périphériques M.2 non spécifiquement conçus pour FLY π)
* Interfaces 40 broches, compatibles Raspberry Pi
* Alimentation : Alimentation DC 12-24V, plus stable
* Prise en compte d'une interface 5V pour ventilateur ZH1.5
* Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, plugins de compensation de résonance numpy et FLY-Tools, sans opérations complexes comme le changement de source, facilitant ainsi l'apprentissage pour les débutants (nécessite l'achat d'une carte TF ou M2WE)
