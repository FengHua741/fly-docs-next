# Connexion au WiFi

## Installer l'antenne sur le LITE 2.1

    * L'endroit pointé par la flèche rouge est l'interface de l'antenne WiFi
    * La prise de l'antenne WiFi est `ipex1`
    * Installez l'antenne sur le LITE 2.1, sinon vous pourriez ne pas être en mesure de vous connecter à cause d'un signal faible

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="100%" align="left" />  

* Si vous constatez que le WiFi n'est pas correctement connecté, vérifiez d'abord si l'antenne est branchée, puis utilisez la méthode ci-dessous pour vous connecter au WiFi
