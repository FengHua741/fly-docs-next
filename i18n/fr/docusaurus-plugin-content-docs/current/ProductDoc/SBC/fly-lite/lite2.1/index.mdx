# Introduction

    Fly-Pi-lite2.1 est une station de travail de niveau débutant pour Klipper.
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/lite.webp').default} size="80%" align="center" />

## Caractéristiques

    * CPU : Puce H3 Allwinner à haute performance, quad-core 64-bit Cortex-A7
    * GPU : Mali-400 MP2
    * RAM : 512Mo DDR3
    * ROM : Prise en charge du maximum de 128Go de carte SD
    * WIFI : WiFi intégré 2,4G
    * Périphériques : Spi x1, URAT x2, USB x2, HDMI x1
    * Interfaces : FPC prévues, HDMI + USB FPC en un seul câble
    * Le système préinstalle klipper, moonraker, mainsail, fulidd, klipper-screen, plugins de compensation de résonance numpy et FLY-Tools, sans opérations complexes telles que le changement de source, plus facile pour les débutants (nécessite l'achat d'une carte TF)

## Fonctionnalités mises à jour

    * Mise à niveau de l'interface TFT V1 vers TFT V2
    * Ajout de broches de réinitialisation et d'arrêt
    * Ajout d'une DEL d'alimentation
    * Suppression du bouton BOOT, remplacement par un bouton de réinitialisation
    * Optimisation des câblages
