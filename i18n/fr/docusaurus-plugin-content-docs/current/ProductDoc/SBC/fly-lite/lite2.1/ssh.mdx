## Connexion au poste de travail SSH

* Il est possible de se connecter à l'appareil via le port série ou le WiFi.
* Préparez à l'avance **MobaXterm_Personal** ou un autre outil de terminal SSH.
* Assurez-vous que la carte SD ou M2WE a été gravée avec le système correspondant au poste de travail supérieur.
* **Si le poste de travail ne démarre pas correctement, retirez les périphériques tels que les pilotes, les limites, les ventilateurs, etc. !!!**
* **Enfin, connectez-vous via le réseau pour compiler les firmwares plus facilement !!!**
* **Pour compiler des firmwares, consultez le tutoriel sur la connexion au poste de travail via le WiFi ou le fil Ethernet.**

        ## Utilisation de l'adresse IP réseau pour se connecter au poste de travail SSH

        * Connectez-vous à l'interface web du routeur auquel le poste de travail est connecté.
        * Assurez-vous que le système a démarré correctement.
        * Assurez-vous de connaître le nom d'utilisateur et le mot de passe du poste de travail, le nom d'utilisateur par défaut pour le système FLY est **fly** et le mot de passe par défaut est **mellow**.
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter via l'adresse IP.
        * Ouvrez MobaXterm et configurez-le comme indiqué ci-dessous, en entrant l'adresse IP que vous avez vue précédemment, ou consultez-la dans l'interface web du routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Attention : Ici, au quatrième point, entrez le nom d'utilisateur du poste de travail.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * La première connexion peut afficher une fenêtre, cliquez sur `Accepter`.
        
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Lorsque vous êtes invité à entrer le mot de passe, tapez le mot de passe du poste de travail et appuyez sur Entrée.

        **Attention : Ici, le mot de passe que vous entrez ne s'affiche pas, appuyez simplement sur Entrée après avoir fini de taper.**

        Le nom d'utilisateur par défaut pour le système FLY est : `fly`, le mot de passe par défaut est : `mellow`.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />
