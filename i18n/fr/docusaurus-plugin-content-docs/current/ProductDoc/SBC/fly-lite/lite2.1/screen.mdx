# Configuration et câblage de l'écran

:::info

* lite2.1 nécessite d'utiliser SSH pour se connecter au système et d'exécuter la commande ci-dessous pour utiliser Fly Tools pour configurer l'écran
```bash
sudo sed -i 's/board=fly-lite2/board=fly-lite2.1/g' /boot/FlyOS-Env.txt
```
:::


* Fly Tools nécessite que l'ordinateur principal soit connecté à Internet pour accéder via `IP+:9999`
* Par exemple `192.168.6.110:9999`

## Méthode 1    
         * La configuration tactile capacitif nécessite d'exécuter la commande ci-dessous pour configurer en `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Vérifiez que le codeur sur l'écran est configuré sur `Cap`
    * Utilisez la commande suivante pour vérifier si la modification a été effectuée avec succès
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Pour modifier un écran tactile capacitif en écran tactile résistif, utilisez la commande suivante

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
```
