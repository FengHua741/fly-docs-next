# Connexion au WiFi

## Installation de l'antenne sur le LITE 2

    * L'endroit indiqué par une flèche rouge est l'interface de l'antenne WiFi
    * La prise de l'antenne WiFi est `ipex1`
    * Installez l'antenne sur le LITE 2, sinon vous pourriez ne pas être en mesure de vous connecter à cause d'un signal faible

    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2/img/wifi.webp').default} size="100%" align="left" />  

* Si vous constatez que la connexion au WiFi n'est pas normale, vérifiez d'abord si l'antenne est bien branchée, puis utilisez la méthode ci-dessous pour vous connecter au WiFi
