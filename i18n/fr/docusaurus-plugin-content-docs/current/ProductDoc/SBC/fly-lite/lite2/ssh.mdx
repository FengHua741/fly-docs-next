# Connexion au poste de travail SSH


* Il est possible de se connecter à l'appareil via le port série ou Wi-Fi
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre **outil de terminal SSH**
* Assurez-vous que la carte SD ou M2WE a été gravée avec **le système correspondant au poste de travail supérieur**
* **Si le poste de travail ne démarre pas correctement, retirez tous les périphériques externes tels que les pilotes, les limitateurs, les ventilateurs, etc. !!!**
* **Enfin, connectez-vous via le réseau pour faciliter la compilation du firmware !!!**
* **Pour compiler le firmware, consultez le tutoriel sur la connexion au poste de travail via Wi-Fi ou par câble Ethernet**

    <TabItem value="net" label="Connexion réseau" default>
        ## Utilisation de l'adresse IP réseau pour se connecter au poste de travail SSH

        * Connectez-vous au **panneau de configuration de la routeur** auquel est connecté le poste de travail
        * Assurez-vous que le système a bien démarré
        * Assurez-vous de connaître le nom d'utilisateur et le mot de passe du poste de travail. Le système FLY a par défaut le nom d'utilisateur **fly** et le mot de passe **mellow**
        * Si la carte mère est déjà allumée et connectée au réseau, vous pouvez vous connecter via l'adresse IP
        * Ouvrez MobaXterm, effectuez les paramétrages suivants, en saisissant l'adresse IP que vous avez vue précédemment, ou consultez-la dans le panneau de configuration de la routeur

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Attention : À l'étape 4, entrez le nom d'utilisateur du poste de travail**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * Lors de la première connexion, une fenêtre s'affiche. Cliquez sur `Accepter` pour continuer

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe du poste de travail et appuyez sur Entrée

        **Notez que le mot de passe n'est pas affiché lors de la saisie, appuyez simplement sur Entrée après l'avoir saisi**

        Le nom d'utilisateur par défaut du système FLY est : `fly`, le mot de passe par défaut est : `mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="40%" align="center" />

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="40%" align="center" />

        * La connexion est réussie
