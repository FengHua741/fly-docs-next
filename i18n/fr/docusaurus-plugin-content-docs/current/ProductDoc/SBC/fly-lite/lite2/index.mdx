# Introduction

    Fly-Pi-lite2 est un ordinateur supérieur de niveau d'entrée pour Klipper.

## Caractéristiques

    * CPU : puces Allwinner H3 à haute performance, 4 cœurs, 64 bits Cortex-A7
    * GPU : Mali-400 MP2
    * RAM : 512 Mo DDR3
    * ROM : prend en charge jusqu'à 128 Go de carte SD
    * WIFI : WiFi intégré 2,4G
    * Périphériques : Spi x1, URAT x2, USB x2, HDMI x1
    * Interfaces : FPC prévues, HDMI + USB FPC en une seule connexion
    * Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, des plugins de compensation de résonance numpy et FLY-Tools, sans opérations complexes telles que le changement de source, ce qui facilite l'utilisation pour les débutants (nécessite l'achat d'une carte TF)
