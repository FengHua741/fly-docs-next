## Préparation avant l'installation du système

    * Lecteur de cartes

## Installation du système

:::info
L'installation sur une carte TF et l'installation via une carte TF sur M2WE ne nécessitent qu'une seule méthode d'installation.
:::

:::info
* Si le système est installé sur M2WE et une carte mémoire, le système démarre par défaut à partir de la carte mémoire
* Il est recommandé d'utiliser une carte TF pour installer le système sur M2WE, car l'installation via USB peut échouer en raison de problèmes de compatibilité des pilotes. Il est conseillé d'utiliser directement [l'installation via carte mémoire sur M2WE](https://mellow-next.klipper.cn/docs/ProductDoc/SBC/fly-pi-v2/fly-os)
* Veuillez vérifier que l'image de système téléchargée correspond, sinon elle ne pourra pas démarrer
:::

## Installation de l'antenne

    1. Installer l'antenne pour le module `M2WE` ou le module `WiFi 2,4G`
    :::warning
    * L'ordinateur FLY nécessite l'utilisation du module `M2WE` ou du module `WiFi 2,4G`, sans quoi il ne pourra pas utiliser le WiFi
    * Veuillez noter que l'ordinateur FLY ne prend pas en charge les noms de réseau WiFi en chinois, ni les réseaux WiFi 2,4G et 5G combinés
    :::
    * L'endroit pointé par la flèche rouge est l'interface de l'antenne WiFi
    * L'interface de l'antenne WiFi est `ipex1`
    * Veuillez installer l'antenne sur PI V2, sinon cela peut provoquer une mauvaise connexion avec le routeur en raison d'une faible intensité du signal
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-pi/img/wifi.webp').default} size="80%" align="left" />
