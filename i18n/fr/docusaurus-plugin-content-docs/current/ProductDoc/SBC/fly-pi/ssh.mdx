## Connexion au poste de travail SSH

* Vous pouvez utiliser le port série ou le WIFI pour vous connecter à l'appareil.
* Veuillez préparer à l'avance **MobaXterm_Personal** ou un autre outil de terminal SSH.
* Assurez-vous que la carte SD ou M2WE a été gravée avec **le système correspondant au poste de travail supérieur**.
* **Si le poste de travail ne démarre pas correctement, veuillez retirer les périphériques tels que les pilotes, les limites, les ventilateurs, etc. !!!**
* **Veuillez enfin vous connecter via le réseau à SSH pour faciliter la compilation du firmware !!**
* **Pour compiler le firmware, consultez le tutoriel sur la connexion au poste de travail via le WIFI ou le câble Ethernet.**

        ## Utiliser l'adresse IP réseau pour se connecter au poste de travail SSH

        * Connectez-vous à l'interface web de la **box de votre routeur**.
        * Assurez-vous que le système a démarré correctement.
        * Assurez-vous que vous connaissez le nom d'utilisateur et le mot de passe du poste de travail, le nom d'utilisateur par défaut de FLY est **fly**, le mot de passe par défaut est **mellow**.
        * Si la carte mère a démarré et est connectée au réseau, vous pouvez vous connecter à SSH via l'adresse IP.
        * Ouvrez MobaXterm, effectuez les paramétrages suivants, entrez l'adresse IP que vous avez vue précédemment, vous pouvez également la trouver dans l'interface web de la box de votre routeur.

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="40%" align="center" />

        **Note : Ici, au quatrième pas, entrez le nom d'utilisateur du poste de travail.**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="40%" align="center" />

        * La première connexion affichera cette fenêtre, cliquez sur `Accept`.
        
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="40%" align="center" />

        * Saisissez le mot de passe et appuyez sur Entrée.
        
        **Note : Ici, le mot de passe saisi n'est pas affiché, appuyez directement sur Entrée après l'avoir saisi.**

        Le nom d'utilisateur par défaut de FLY est : `fly`, le mot de passe par défaut est : `mellow`.
