# Connexion au WiFi


## Installer l'antenne sur le PI

    * L'endroit pointé par la flèche rouge est l'interface de l'antenne WiFi
    * La prise de l'antenne WiFi est `ipex1`
    * Installez l'antenne sur le PI, sinon vous pourriez ne pas être en mesure de vous connecter à cause d'un signal faible
