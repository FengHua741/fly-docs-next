## Méthode 1    
         * La configuration de l'écran tactile capacitif nécessite d'utiliser la commande ci-dessous pour configurer en `fly-tft-v2-c`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v2-c/' -i /boot/FlyOS-Env.txt
```
         * Vérifiez que le codeur sur le dos de l'écran est réglé sur `Cap`
    * Utilisez la commande ci-dessous pour vérifier si la modification a été effectuée avec succès
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```
        <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tft-c.webp').default} size="80%" align="center" />
 * Pour modifier une configuration d'écran tactile capacitif en configuration d'écran tactile résistif, utilisez la commande suivante

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v2-r/}' /boot/FlyOS-Env.txt
``` 
</TabItem>
