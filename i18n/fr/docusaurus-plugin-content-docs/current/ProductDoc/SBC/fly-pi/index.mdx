# Introduction

Nous sommes ravis de lancer la nouvelle carte mère FLY-π. Cette carte peut remplacer le Raspberry Pi pour fonctionner en ordinateur principal et elle dispose de 1 Go de mémoire DDR3 embarquée, permettant d'exécuter un bureau LINUX complet (basé sur Armbian personnalisé et optimisé). La carte peut être utilisée avec les firmwares Klipper et RRF, et même peut exécuter l'écran Klipper lors de l'utilisation de l'outil de boîte à outils SHT.

Le but de la conception de cet ordinateur principal est de rendre l'utilisation du firmware Klipper ou RRF plus facile pour les clients. Nous avons optimisé l'image système LINUX pour une utilisation plus appropriée dans l'impression 3D, ce qui rend l'installation et l'opération plus simples, et même un débutant avec une base légère peut apprendre rapidement à l'utiliser. Par rapport aux autres cartes d'ordinateur principal, cela réduit les coûts d'apprentissage, vous permettant de vous concentrer davantage sur le plaisir que procure l'impression 3D (achetez un forfait avec une carte SD préinstallée avec le firmware Klipper, prêt à l'emploi dès la mise sous tension).

## Caractéristiques

* CPU : Puce Allwinner H5 haute performance, quad core 64-bit Cortex-A53
* GPU : GPU haute performance 6-core Mali 450, taux de remplissage de pixels supérieur à 2,7 gpixel/s
* RAM : 1 Go DDR3 (partagé avec le GPU) (512+512)
* ROM : Prise en charge jusqu'à 128 Go de carte SD
* Périphériques : Spi x2, UART x2, USB x4 (USB 2.0 x3, OTG x1), Micro HDMI x1, Eth x1 (100M), CAN x1 (à acheter séparément)
* Interfaces : Interface M.2 embarquée, pouvant être étendue avec Wi-Fi, eMMC, etc. (protocole privé, ne pas connecter des appareils M.2 non spécifiques à FLY π)
* Interface 40 broches, compatible Raspberry Pi
* Prise en charge de l'alimentation DC 12-24V, alimentation plus stable
* Une interface à courant continu de 5V est réservée pour le refroidissement du ventilateur
* Interface CAN d'extension réservée
* Le système est préinstallé avec klipper, moonraker, mainsail, fulidd, klipper-screen, Crowsmest, plug-in de compensation de résonance numpy et FLY-Tools, sans opérations complexes comme le changement de source, plus facile pour les débutants (nécessite l'achat d'une carte TF ou M2WE)
