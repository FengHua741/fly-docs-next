# ADXL345

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-adxl/img/345.webp').default} size="100%" align="left" />



## Caractéristiques du produit

    * Convient pour les endroits où se trouvent les moteurs de 36, etc.
    * Peut être connecté à des Raspberry Pi, FLY PI, Orange Pi, cartes mères, etc., via l'interface SPI

## Spécifications du produit

    * Interface sur la carte : connecteur 6P ZH1.5
    * Mode de communication : SPI
    * Tension de fonctionnement : 5V

## Remarques

    :::warning Remarques

    * La tension de fonctionnement est de 5V, veuillez ne pas vous tromper
    * Convient pour diverses firmwares, assurez-vous qu'il peut être utilisé avec une connexion SPI

    :::
