# Introduction

# Utilisation du tampon LLL

    Le tampon de consommable LLL est un dispositif conçu pour l'alimentation à distance d'un imprimante 3D, visant à simplifier les opérations de chargement et de déchargement des consommables. Ce tampon est équipé de deux boutons, l'un pour l'alimentation et l'autre pour le retrait, permettant aux utilisateurs d'effectuer des opérations d'alimentation et de retrait électriques.

### Fonction de tampon :

    Lorsque l'imprimante 3D alimente l'extrudeuse, si une traction est exercée sur le support de consommable distant, le tampon détecte cette action et envoie automatiquement une partie de la consommable vers le hot-end. Chaque fois qu'une traction se produit, le tampon répète cette opération d'alimentation. Cela permet de réduire efficacement les tractions exercées par l'extrudeuse sur la consommable, en maintenant une poussée constante de l'extrudeuse, améliorant ainsi la stabilité de la qualité d'impression.

### Fonction de détection de rupture :

    Le tampon est doté d'un détecteur de rupture qui envoie un signal à l'imprimante 3D pour suspendre l'impression lorsque la détection indique qu'il n'y a plus de consommable dans le tampon. Pour utiliser cette fonction, il est nécessaire de connecter les broches de signal et de modifier la configuration de l'imprimante 3D. En revanche, la fonction de tampon peut être utilisée directement sans modification de configuration.

    Grâce à ces conceptions de fonctionnalités, le tampon de consommable LLL simplifie non seulement les opérations de consommable, mais améliore également la stabilité du processus d'impression et la qualité d'impression.

* Veuillez noter que vous devez connecter la consommable avant d'utiliser les fonctions, sinon vous ne pouvez contrôler le moteur que par les boutons.

## Référence de configuration

    ```cfg
    [filament_switch_sensor Détection de rupture]
    pause_on_runout: true
    switch_pin: ^PG7
    runout_gcode:
            PAUSE
            M118 Manque de consommable
    insert_gcode:
            M118 Insertion de consommable
    ```
