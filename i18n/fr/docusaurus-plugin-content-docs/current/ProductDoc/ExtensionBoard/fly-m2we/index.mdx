## Précautions

:::danger Précautions

* M2WE est préinstallé avec FLYOS et peut être démarré directement à la sortie d'usine.
* M2WE ne peut être utilisé que sur les produits FLY, il n'est pas compatible avec d'autres appareils.
* Les bornes antennes de M2WE sont de type `IPEX1`.
* M2WE ne prend pas en charge la double fréquence 2.4G et 5G, les caractères spéciaux, les caractères chinois et les réseaux Wi-Fi par défaut masqués.

:::
