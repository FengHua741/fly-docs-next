## Configuration de référence

    ```cfg
    [fan]
    ##    Imprimante Ventilateur de refroidissement 
    pin: PF8
    max_power: 0.9
    cycle_time: 0.002
    hardware_pwm: false
    shutdown_speed: 0
    ```

## Solution pour le démarrage automatique en rotation

    :::info
    * Ceci n'est qu'une explication simplifiée
    * Ajoutez le signal de broche PWM lors de la compilation du firmware et inversez son état
    * Testé avec SUPER8PRO, broche utilisée PF8, lors de la compilation du firmware, choisissez `GPIO pins to set at micro-controller startup` et ajoutez `!PF8`
    * Veuillez noter que l'ajout de `!` est un point d'exclamation anglais
    * Le même principe s'applique aux autres cartes mères
    :::
