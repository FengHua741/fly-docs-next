# Configuration de RRF

## Configuration des fonctions de rupture de fil et de bouchon

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} size="70%" align="center" />

1. Ouvrez l'interface web de l'imprimante RRF, accédez à `Fichiers` -> `Système`, trouvez le fichier `config.g`, puis cliquez pour ouvrir et éditer le fichier.
2. Ajoutez le code suivant dans le fichier pour activer la fonction de détection de rupture de fil.

```gcode
M591 D0 P2 C"PA9" S1
```

3. Ajoutez le code suivant dans le fichier pour activer la fonction de détection de bouchon. (`L` est modifiable en fonction du disque utilisé, si des déclenchements erronés se produisent, augmentez cette valeur)

```gcode
M591 D0 P7 C"PA10" L3.7 R75:125 S1
```
