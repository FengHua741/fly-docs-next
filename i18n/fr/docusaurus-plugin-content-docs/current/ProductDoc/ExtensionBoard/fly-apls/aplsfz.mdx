# Comment ajuster la valeur de déclenchement du module
À l'aide d'un outil de terminal série, les commandes suivantes sont disponibles :
Envoyez "SET_PIN PIN=_probe_ready VALUE=1" dans le Klipper Console et observez la lampe indicatrice de déclenchement. Ajustez la valeur seuil de petite en grande jusqu'à ce que la lampe ne s'allume plus. Entrez la commande suivante :
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
Dans l'outil de terminal série, entrez la commande suivante pour ajuster la valeur de déclenchement :
```cfg
set valeur_déclenchement
```
* Commandes à utiliser dans l'outil de terminal série :

* Envoyez `v` via le terminal série pour afficher en temps réel les valeurs adc. a : avant filtrage ; b : après filtrage.
* Envoyez `uv` via le terminal série pour désactiver la sortie.
* Envoyez `rt` via le terminal série pour voir la valeur seuil actuelle.
