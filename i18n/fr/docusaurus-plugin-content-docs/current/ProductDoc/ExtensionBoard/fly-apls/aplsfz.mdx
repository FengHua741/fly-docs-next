# Comment ajuster la valeur de déclenchement du module
En utilisant l'outil série, les commandes suivantes sont disponibles :
Envoyer "SET_PIN PIN=_probe_ready VALUE=1" dans le Klipper Console et observer le voyant indicateur de déclenchement, ajustez la valeur seuil de petite en grande jusqu'à ce que l'indicateur ne s'allume plus. Entrez la commande ci-dessous :
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
Entrez la commande ci-dessous dans l'outil série pour ajuster la valeur de déclenchement :
```cfg
set valeur_déclenchement
```
Valeur seuil par défaut : 20000
* Commandes à utiliser avec l'outil série :

* Envoyer `v` via le port série pour afficher la sortie adc en temps réel. a : avant filtrage ; b : après filtrage.
* Envoyer `uv` via le port série pour désactiver la sortie.
* Envoyer `rt` via le port série pour consulter la valeur seuil actuelle.
