# Mise à jour et gravure du firmware

    :::danger À savoir

    * Le firmware est préinstallé en usine

    :::

## Téléchargement du firmware

    * Explications sur le firmware
    * `wbtn-vX.X.X-X-XXXXXX-full.bin` est un firmware complet qui doit être gravé à partir de l'adresse `0x0`, sans quoi toutes les configurations seront perdues
    * `wbtn-vX.X.X-X-XXXXXX-delta.bin` est un firmware incrémentiel qui doit être gravé à partir de l'adresse `0x10000`, il met à jour uniquement le firmware sans affecter les configurations existantes
    * `wbtn-vX.X.X-X-XXXXXX-www.bin` est un firmware pour l'interface utilisateur qui doit être gravé à partir de l'adresse `0x290000`, il met à jour uniquement l'interface utilisateur web sans affecter le firmware ou les configurations existantes
    * <Button variant="contained" disableElevation href="https://g-ofpa1390.coding.net/public-artifacts/3D-Printers/airclick/packages">Téléchargement du package de firmware</Button>

## Mise à jour du firmware
    * Il faut que l'AirClick soit déconnecté du courant, puis connecter le port de programmation ESP32 de l'AirClick à l'ordinateur en appuyant sur BOOT
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/host.webp').default} size="80%" align="Center" />

    1. Télécharger et décompresser l'outil de programmation USB <Button variant="contained" disableElevation href="https://cdn.mellow.klipper.cn/Utils/flash_download_tool_3.9.4.zip">Cliquez ici pour télécharger</Button>
    2. Ouvrir ``flash_download_tool_3.9.4.exe``, attention : après avoir ouvert l'outil de programmation, une fenêtre CMD s'ouvrira d'abord, puis l'interface de l'outil de programmation, ne fermez pas la fenêtre CMD
    3. Dans l'outil de programmation, choisir ``ESP32-C3`` dans ``ChipType:``
    4. Dans l'outil de programmation, choisir ``Develop`` dans ``WorkMode:``
    5. Dans l'outil de programmation, choisir ``USB`` dans ``LoadMode:``
    6. Cliquez ensuite sur ``OK``, comme indiqué dans l'image suivante
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash.webp').default} size="50%" align="Center" />
    7. Après avoir cliqué, l'image suivante apparaîtra
        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/flash-1.webp').default} size="50%" align="Center" />

    :::tip

    * La zone sélectionnée en vert dans l'image ci-dessus doit être identique à celle de l'image, ne modifiez pas

    :::

    8. Connectez AirClick à l'ordinateur via un câble USB-TypeC
    9. Cliquez sur le bouton bleu dans l'image ci-dessus, zone 1, pour sélectionner le firmware que vous souhaitez graver
    10. Cliquez sur le bouton bleu dans l'image ci-dessus, zone 2, pour entrer l'adresse de gravure
    11. Cliquez sur le bouton bleu dans l'image ci-dessus, zone 3, pour sélectionner le port série de l'AirClick (si ce n'est pas le cas, maintenez enfoncée la touche de programmation de l'AirClick pendant la connexion à l'ordinateur)
    12. Cliquez sur le bouton bleu dans l'image ci-dessus, zone 4, pour choisir le débit de 921600
    13. Cliquez sur le bouton bleu dans l'image ci-dessus, zone 5, pour commencer la gravure, attendez que la gravure soit terminée avant de débrancher le câble de l'AirClick
