# Se connecter à Klipper

    :::info Important
    * AirClick doit être connecté au même réseau que l'imprimante cible
    :::

1. Cliquez sur le paramètre de système sur la barre de gauche
    
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting.webp').default} size="80%" align="Center" />

2. Sélectionnez `Klipper` pour `Type d'imprimante`
3. Entrez l'adresse IP de la machine Klipper dans `Adresse IP de Moonraker`
4. Entrez le port du serveur supérieur de Klipper dans `Port de Moonraker`. La valeur par défaut est `7125`. Si le serveur supérieur utilise un autre port, entrez le port correspondant.
5. Ignorez `Apikey`, il n'est pas pris en charge temporairement.

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-4.webp').default} size="80%" align="Center" />

6. Après avoir configuré comme indiqué ci-dessus, cliquez sur `Soumettre` en bas à droite pour sauvegarder la configuration
7. Une fois correctement connecté, un message s'affichera en haut de la page web

     <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-airclick/img/setting-3.webp').default} size="80%" align="Center" />

8. Si la connexion échoue
    - Vérifiez si l'adresse IP et le port sont corrects
    - Vérifiez si l'imprimante et AirClick sont connectés au même réseau
    - Vérifiez si le serveur supérieur de Klipper fonctionne normalement
    - Vérifiez si le navigateur peut ouvrir la page web du serveur supérieur de Klipper
