# Configuration de RRF

## Configuration des fonctions de rupture et de bouchage de filament

<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-mdm/img/rrf.webp').default} size="70%" align="center" />

1. Ouvrez l'interface web de l'imprimante RRF, accédez à `Fichiers` -> `Système`, trouvez le fichier `config.g`, puis cliquez pour l'ouvrir et l'éditer.
2. Ajoutez le code suivant dans le fichier pour activer la fonction de détection de rupture de filament.

```gcode
M591 D0 P2 C"PA9" S1
```

3. Ajoutez le code suivant dans le fichier pour activer la fonction de détection de bouchage de filament. (Le paramètre `L` varie en fonction du disque codeur utilisé ; si une activation erronée se produit, augmentez cette valeur.)

```gcode
M591 D0 P7 C"PA10" L3.5 R75:125 S1
```
