### Description des fichiers de modèle

| Nom du fichier du modèle | Description |
| :---: | :--- |
| pcb_cover.stp | Cache PCB |
| switch_sensor_trigger_block.stp | Bloc de déclenchement de détection de filament |
| rear_cover.stp | Abattant arrière |
| upper_cover.stp | Abattant supérieur |
| main_body.stp | Objet principal du modèle |
| encoder_3_5.8mm.stp | Disque de codage encodeur, 3 bandes, distance de détection minimale 5.8mm <br/> `Ce modèle peut être utilisé pour une impression haute vitesse (> 20 mm³/s)` |
| encoder_5_3.5mm.stp | Disque de codage encodeur, 5 bandes, distance de détection minimale 3.5mm <br/> `Modèle recommandé` |
| encoder_10_1.73mm.stp | Disque de codage encodeur, 10 bandes, distance de détection minimale 1.73mm <br/> `Ce modèle peut être utilisé pour une impression haute précision à faible vitesse` <br/> Limité par la limitation de la fréquence d'échantillonnage du micrologiciel d'imprimante 3D pour la détection de consommable |

## Impression et assemblage

:::attention Remarques importantes sur l'impression du modèle

- Les coques ne sont pas recommandées pour être imprimées avec des matériaux comme le PETG ou les matériaux transparents, car cela affectera la sensibilité du capteur.

:::
