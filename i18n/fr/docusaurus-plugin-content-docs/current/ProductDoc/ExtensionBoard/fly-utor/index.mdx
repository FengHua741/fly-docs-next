# FLY-UTOR

    FLY-UTOR est une solution d'extension d'interface à haute performance et polyvalente, conçue pour fournir des interfaces USB supplémentaires et une capacité d'extension de puissance, tout en prenant en charge la communication UART vers RS232. Cette carte d'extension est compacte, facile à installer et s'adapte à diverses applications nécessitant des interfaces supplémentaires et des sorties de puissance.

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.webp').default} size="100%" align="left" />


## Caractéristiques du produit

    * Double extension d'interfaces USB : la carte d'extension propose deux ports USB supplémentaires, compatibles avec le standard USB 2.0, garantissant la compatibilité avec divers périphériques USB et un transfert de données rapide.
    * Prise en charge UART vers RS232 : convertisseur intégré UART vers RS232, permettant aux utilisateurs de connecter des appareils série au système et d'étendre les capacités de communication.
    * Double extension d'interfaces d'alimentation : deux interfaces d'alimentation permettent aux utilisateurs d'étendre simultanément deux sources externes d'alimentation, répondant ainsi aux besoins de puissance supplémentaire.
    * Conception stable d'alimentation : la carte d'extension est dotée d'un circuit d'alimentation stable, assurant un fonctionnement stable des appareils connectés.
    * Installation facile : conception standardisée adaptée à divers environnements, sans installation de pilotes supplémentaires requis.
    * Dimensions compactes : la taille raisonnable de la carte d'extension ne prend pas trop de place, facilitant son intégration dans divers projets.

## Spécifications du produit :

    * Interfaces disponibles : USB X2, UART vers RS232 X2, interfaces d'alimentation X2
    * Interface d'alimentation : 12V/24V
    * Compatibilité : ordinateurs de contrôle avec interface USB

## Avertissements :

    * Déconnectez l'alimentation et éteignez tous les appareils avant l'installation.
    * Insérez la carte d'extension USB multifonctionnelle dans l'interface USB de la carte mère ou d'autres interfaces d'extension USB.
    * Connectez les périphériques USB et/ou les sources d'alimentation nécessaires à la carte d'extension.
    * Si vous souhaitez utiliser la fonction UART vers RS232, connectez les appareils série appropriés.
    * Réconnectez l'alimentation et redémarrez les appareils.
    * Le système doit détecter automatiquement la carte d'extension et la configurer. Consultez les instructions de configuration si nécessaire.
