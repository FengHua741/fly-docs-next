# FLY-UTOR

    FLY-UTOR est une solution d'extension d'interface à haute performance et polyvalente conçue pour fournir des interfaces USB supplémentaires et une capacité d'extension de puissance, tout en prenant en charge la communication UART vers RS232. Cette carte d'extension est compacte, facile à installer et convient à divers scénarios d'utilisation nécessitant des interfaces et des sorties de puissance supplémentaires.

        <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-utor/img/utor.webp').default} size="100%" align="left" />


## Caractéristiques du produit

    * Extension de deux ports USB : La carte d'extension offre deux ports USB supplémentaires compatibles avec le standard USB 2.0, garantissant la compatibilité avec divers périphériques USB et un transfert de données à haut débit.
    * Support UART vers RS232 : Le convertisseur UART vers RS232 intégré facilite l'intégration de périphériques sériels dans le système, étendant les capacités de communication des appareils.
    * Extension de deux interfaces d'alimentation : Deux interfaces d'alimentation sont fournies, permettant aux utilisateurs d'étendre simultanément deux sources d'alimentation externes, répondant ainsi aux besoins de puissance supplémentaire.
    * Conception d'alimentation stable : La carte d'extension est dotée d'un circuit d'alimentation stable, assurant un fonctionnement stable des appareils connectés.
    * Installation facile : Conçue selon des dimensions standard, elle s'adapte à différents environnements sans nécessiter d'installation de pilotes supplémentaires.
    * Taille compacte : La taille de la carte d'extension est raisonnable, ne prenant pas trop de place, ce qui facilite son intégration dans divers projets.

## Spécifications du produit :

    * Interfaces disponibles : USB X2, UART vers RS232 X2, Interfaces d'alimentation X2
    * Interfaces d'alimentation : 12V/24V
    * Compatibilité : Ordinateurs supérieurs équipés d'interfaces USB

## Précautions :

    * Déconnectez l'alimentation et éteignez tous les appareils avant l'installation.
    * Branchez la carte d'extension multifonctionnelle USB dans une interface USB de la carte mère ou dans une autre interface d'extension USB.
    * Connectez les périphériques USB requis et/ou les alimentations à la carte d'extension.
    * Pour utiliser la fonction UART vers RS232, branchez les périphériques série correspondants.
    * Reconnectez l'alimentation et allumez les appareils.
    * Le système devrait détecter automatiquement la carte d'extension et la configurer. Si nécessaire, consultez les précautions de configuration après l'opération.
