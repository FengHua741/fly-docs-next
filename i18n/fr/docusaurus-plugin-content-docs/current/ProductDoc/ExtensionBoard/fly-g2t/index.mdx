# FLY-G2T

    La carte d'extension FLY-G2T est conçue pour étendre les fonctionnalités du Raspberry Pi et de l'interface supérieure FLY. Elle prend en charge la communication UART, intègre une interface FPC pour connecter le TFT-V2 à l'extérieur et supporte également le contrôle des ventilateurs de 5V. Cela en fait une solution idéale pour améliorer l'interactivité et la facilité d'utilisation du Raspberry Pi dans diverses applications, telles que l'automatisation industrielle, les imprimantes 3D, l'automatisation domestique, le contrôle robotique ou tout autre domaine nécessitant une interface homme-machine et une communication externe.

## Caractéristiques du produit

    * Compatibilité : Supporte plusieurs versions du Raspberry Pi, y compris le Raspberry Pi Zero/3B/3B+/4B, etc.
    * Conversion UART vers RS232 : Fournit une communication série stable, permettant au Raspberry Pi d'échanger des données avec d'autres périphériques matériels via l'interface RS232.
    * Écran tactile capacitif SPI : L'écran tactile capacitif TFT-V2 connecté via l'interface FPC fournit une opération tactile intuitive, adapté aux applications nécessitant une interface utilisateur graphique, telles que klipperscreen, etc.
    * Support des ventilateurs de 5V : Intègre une interface pour ventilateur de 5V, permettant de connecter un ventilateur pour le refroidissement, assurant la stabilité de l'interface supérieure lors d'une utilisation prolongée ou sous charge élevée.
    * Conception à enclenchement : La carte d'extension est conçue pour être facilement installée, en se branchant directement sur l'interface supérieure sans câblage supplémentaire ni configuration complexe.
    * Taille compacte : La taille de la carte d'extension est raisonnable, ne prenant pas trop de place, ce qui facilite son intégration dans divers projets.

## Spécifications du produit

    * Interfaces de communication : Conversion UART vers RS232
    * Interfaces d'affichage : Conversion SPI vers interface FPC
    * Alimentation du ventilateur : 5V
    * Compatibilité : Raspberry Pi Zero/3B/3B+/4B, FLY Pi V2, etc.

## Remarques

    :::warning Remarques

    * Lors de l'utilisation d'outils de connexion RS232 avec l'interface supérieure, assurez-vous qu'ils sont `en terre commune` !!!!!!

    :::

    * Assurez-vous que l'interface supérieure est éteinte avant d'installer la carte d'extension.
    * Évitez d'utiliser la carte d'extension dans des environnements humides ou poussiéreux.
    * Lors de l'installation de l'écran, veillez à ne pas le débrancher quand il est sous tension et à respecter l'orientation du câble d'écran pour éviter tout dommage.
    * Lors de l'installation du ventilateur, vérifiez les spécifications de courant et de tension du ventilateur pour éviter tout dommage.
