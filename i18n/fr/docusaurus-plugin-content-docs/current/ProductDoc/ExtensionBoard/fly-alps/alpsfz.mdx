# Comment ajuster la valeur de déclenchement du module
Avec l'outil série, les commandes suivantes sont disponibles :
Console Klipper en envoyant "SET_PIN PIN=_probe_ready VALUE=1", observer la lumière indicatrice de déclenchement, ajuster le seuil progressivement de plus petit en plus grand jusqu'à ce que la lumière s'éteigne. Entrez la commande ci-dessous :
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
Dans l'outil série, entrez la commande ci-dessous pour ajuster la valeur de déclenchement :
```cfg
set valeur_déclenchement
```
Valeur de déclenchement par défaut : 20000
* Commandes utilisées dans l'outil série :

* Envoyer `v` via le port série, sortie ADC en temps réel. a : avant filtrage ; b : après filtrage.
* Envoyer `uv` via le port série, désactiver la sortie.
* Envoyer `rt` via le port série, afficher la valeur seuil actuelle.
