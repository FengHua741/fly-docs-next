# Comment ajuster la valeur de déclenchement du module

## Ouvrir le module de calibrage
    Envoyez `SET_PIN PIN=_probe_ready VALUE=1` dans la console de `Klipper`, observez l'indicateur de déclenchement, ajustez la valeur seuil de petite en grande jusqu'à ce que l'indicateur ne s'allume pas. Entrez la commande suivante :
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```

## Ajuster la valeur de pression
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">Assistant outil de port série</Button>

    * Connectez le module à l'ordinateur avec un câble USB, ouvrez l'outil d'assistant port série, sélectionnez le port COM correspondant, la vitesse de communication 115200, puis cliquez sur "Connecter".
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    Entrez la commande suivante dans l'outil de port série pour ajuster la valeur de déclenchement :
    ```cfg
    set valeur_déclenchement
    ```
    Valeur de déclenchement par défaut : 20000
    Plus la valeur est petite, plus le module est sensible ; plus la valeur est grande, moins il est sensible.
    * Par exemple : `set 20000`, définir la valeur de déclenchement à 20000, plus la valeur est petite, plus le module est sensible ; plus elle est grande, moins il est sensible.
    * Commande utilisée dans l'outil de port série :

    * Envoi de la commande `v` via le port série, affiche en temps réel les valeurs abc. a : avant filtrage ; b : après filtrage.
    * Envoi de la commande `uv` via le port série, désactive la sortie.
    * Envoi de la commande `rt` via le port série, affiche la valeur seuil actuelle.
