# Comment ajuster la valeur de déclenchement du module
<Button variant="contained" disableElevation href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">Obtenir l'outil d'assistant de port série</Button>

En utilisant l'outil de port série, les commandes suivantes sont disponibles :
Envoyez "SET_PIN PIN=_probe_ready VALUE=1" dans le tableau de bord Klipper pour observer le voyant indicateur de déclenchement, ajustez la valeur seuil de petite en grande jusqu'à ce que le voyant ne s'allume plus. Entrez la commande suivante :
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
* Connectez le module à l'ordinateur via un câble USB, ouvrez l'outil d'assistant de port série, sélectionnez le port COM correspondant, définissez le débit à 115200, puis cliquez sur "Connecter".
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
<ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
Entrez la commande suivante dans l'outil de port série pour ajuster la valeur de déclenchement :
```cfg
set valeur_déclenchement
```
Valeur de déclenchement par défaut : 20000
Plus la valeur est petite, plus elle est sensible ; plus elle est grande, moins elle est sensible.
* Par exemple : `set 20000`, définir la valeur de déclenchement à 20000, plus la valeur est petite, plus elle est sensible ; plus la valeur est grande, moins elle est sensible.
* Commandes utilisées dans l'outil de port série :

* Envoi de la commande `v` via le port série : affiche les valeurs abc en temps réel. a : avant filtrage ; b : après filtrage.
* Envoi de la commande `uv` via le port série : désactive la sortie.
* Envoi de la commande `rt` via le port série : consultez la valeur seuil actuelle.
