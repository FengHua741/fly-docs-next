# FLY_halo

## 1.  Page d'accueil

    Cliquez sur le lit chaud ou l'extrudeuse, faites glisser depuis le haut de l'écran vers le bas dans le sens horaire jusqu'à la température souhaitée, puis cliquez à nouveau sur le lit chaud ou l'extrudeuse pour définir la température.

    Si vous souhaitez afficher les températures du MCU et de l'ordinateur principal, ajoutez cette configuration

    ```cfg
    [temperature_sensor mcu]
    sensor_type: temperature_mcu      # Associé au MCU (par défaut)

    [temperature_sensor host]        # Température de l'ordinateur principal
    sensor_type: temperature_host     # Associé à l'ordinateur principal
    ```


    :::warning Remarques

    * Notez que la température maximale est lue dans la configuration

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/heating.webp').default} size="50%" align="left" />


## 2. Extrusion

    * Appuyez longtemps sur l'icône pour régler la température de l'extrudeuse, la longueur d'extraction et la vitesse d'extraction

    :::warning Remarques

    * Halo ne supporte que le contrôle d'un seul extrudeur

    :::

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/Extrusion.webp').default} size="50%" align="left" />
