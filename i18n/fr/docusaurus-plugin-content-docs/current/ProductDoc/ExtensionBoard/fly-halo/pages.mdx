# Présentation des opérations de la page Halo

## Structure de la page

```bash
  Page d'impression de fichiers          Accueil               Page d'extrusion            Page des services système             Paramètres              Informations sur l'appareil
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
|             |    |             |    |             |    |             |    |             |    |             |
| Fichiers à imprimer |<-->|     Accueil    |<-->|   Extrusion  |<-->| Services systèmes |<-->|   Paramètres   |<-->|  Informations sur le périphérique   |
|             |    |             |    |             |    |             |    |             |    |             |
+-------------+    +-------------+    +-------------+    +-------------+    +-------------+    +-------------+
                         ∧
                         |
                         ∨
                   +-------------+
                   |             |
                   | Détection de blocage |
                   |             |
                   +-------------+
                     Page de détection de blocage
            (Cette page n'est disponible que si la détection de blocage est activée dans les paramètres)

```

## Page d'extrusion

    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-1.webp').default} size="50%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/jichu-2.webp').default} size="50%" align="left" />
    * Les zones opérationnelles de l'image 1 ci-dessus correspondent aux numéros en dessous
    1. Clic : Augmenter la température cible de l'extrudeur de 1 degré
    2. Appui long : Entrer dans la configuration de température d'extraction, ajuster la température en glissant la barre circulaire, cliquer à nouveau pour confirmer la température définie
    3. Clic : Diminuer la température cible de l'extrudeur de 1 degré
    4. Clic : Extrusion
    5. Glisser vers le haut ou vers le bas : Sélectionner la longueur d'extraction ou de retrait, unité mm
    6. Clic : Retrait
    7. Appui long : Entrer dans la configuration de vitesse d'extraction de l'image 2 ci-dessus, sélectionner la vitesse d'extraction en glissant vers le haut ou vers le bas, unité mm/s, cliquer sur une zone vide autour après la sélection pour quitter
