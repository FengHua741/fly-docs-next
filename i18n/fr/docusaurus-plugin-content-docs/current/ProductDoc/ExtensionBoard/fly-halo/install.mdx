## Guide d'installation

## Installation sur Voron Stealthburner

* L'installation de Halo sur Stealthburner nécessite l'utilisation des adaptateurs fournis
* Téléchargez d'abord les deux STL fournis et imprimez-les vous-même
    * [Stealthburner_main_body(Adapt_to_Halo).stl(Cliquez pour télécharger)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
    * [Halo_mounting_bracket.stl(Cliquez pour télécharger)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
    * [Stealthburner_CW2_Assembly-2 v9.step(Cliquez pour télécharger)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)

### Installation du panneau d'adaptation et des câbles de raccordement
    * La position de sortie peut être adaptée à vos préférences, elle n'a pas besoin d'être collée contre le dos du ventilateur
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-halo/img/install-13.webp').default} size="50%" align="left" />


    :::warning Remarques

    * Explication des couleurs de sortie : **noir est la masse GND**, **rouge est la tension positive VCC acceptée entre DC12 et DC24**, **blanc est GPIO47 utilisé pour la détection de blocage**, **jaune est GPIO48 utilisé pour l'arrêt après impression**

    :::
