# Utilisation du tutoriel

## Remarques importantes

    * Lors de la première utilisation, une connexion réseau est requise. Si votre réseau Wi-Fi ne peut pas accéder à Internet, halo ne pourra pas obtenir les paramètres initiaux au démarrage.
    * Toutes les modifications dans la page des paramètres ne prendront effet qu'après avoir cliqué sur **Enregistrer les paramètres** et redémarré l'appareil.
    * La fonction de détection de blocage est désactivée par défaut. Elle doit être activée dans les paramètres et redémarrée avant d'être accessible depuis l'écran d'accueil en faisant glisser vers le haut.
    * La fonction de fermeture automatique après impression est désactivée par défaut et peut être activée dans les paramètres.
    * Dans tous les fenêtres contextuelles nécessitant l'annulation ou la suspension de l'impression, le bouton doit être maintenu appuyé (pour éviter les touches accidentelles).
    * Dans la page des services système, le service Host est l'hôte principal.
    * En utilisant la version officielle du firmware, veuillez ne pas essayer de remplacer la puce principale ou la puce FLASH.

    :::tip

    * Attention : si votre fichier de configuration `moonraker.conf` dépasse 4 Mo, vous pourriez rencontrer des problèmes lors de son utilisation. Si des anomalies se produisent, réduisez la taille du fichier de configuration `moonraker.conf`.

    :::

## Mise à jour du firmware système

    * Les mises à jour de la version halo se font par mise à jour complète ou par mise à jour incrémentielle OTA.
        * Mise à jour complète : implique des mises à jour importantes au niveau du système, nécessitant l'utilisation d'un câble USB pour flasher le package de firmware complet.
        * Mise à jour incrémentielle : mise à jour normale via OTA. Vérifiez les mises à jour dans la page des informations sur l'appareil.

## Utilisation du balayage QR avec un téléphone mobile et configuration du Wi-Fi

    :::tip

    * Certains appareils mobiles peuvent ne pas avoir la fonction de connexion Wi-Fi via un balayage QR. Vous pouvez vous connecter au point d'accès nommé **FLY-halo-xxxxxxxx** dans les paramètres Wi-Fi de votre téléphone.

    :::

    :::tip

    * Si aucune page n'apparaît après la connexion au point d'accès, vous devrez ouvrir manuellement le lien dans un navigateur.

    :::

    :::tip

    * Veuillez noter que HALO ne peut utiliser que le Wi-Fi en 2,4 GHz.

    :::

    :::tip

    * L'écran de HALO ne peut pas rechercher des réseaux Wi-Fi cachés.

    :::
