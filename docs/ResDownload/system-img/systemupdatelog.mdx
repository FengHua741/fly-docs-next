            ## v2
                * 2022-1
                * 更换发行版本为Bullseye
                * 更新内核到5.10.85

                * Klipper使用Python3环境，解决中文文件名问题

                * 修复CatSerial没有连接设备出错

                * 修复Klipper缺少部分烧录环境问题
                * 修改配置模板
                * 新增mJpg支持摄像头
                * 新增V1,V1.1主板引脚配置。在FLY_Config切换

                * 支持Gemini v2

                * 修复udev规则冲突问题
                * 修复缺少libusb库导致klipper的flash报错问题
                * 修复FLY-Tools静态资源无法加载的问题
                * 修改默认主板型号为Gemini v2
                * 修改printer.cfg中默认serial为/dev/ttyACM0
