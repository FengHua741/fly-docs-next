# USB 固件编译

    :::tip

    * ERCF出厂预刷的CAN连接的Katapult固件
    * 如果使用USB固件烧录需要烧录USB的Katapult固件
    * 烧录一次后无需在烧录
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./bl">Katapult固件烧录</Button>

    :::

## 开始编译

    <SSH />

    <MakeOperate/>

    <USB />

    * 选择GPIO pins to set at micro-controller startup，输入：gpio17
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-ercf/ercfv2/flash/img/usb.webp').default} size="100%" align="center" />
    * 按 `Q` 键，出现 **Save configuration**，这时再按 `Y` 键
        <ImageView image={require('@site/docs/General/dfu-flash/img/configuration.webp').default} size="50%" align="center" />
    * 现在应该保存了配置并且退出到了命令行界面
    * 输入 `make -j4` 开始编译，时间有点长
        <ImageView image={require('@site/docs/General/dfu-flash/img/make.webp').default} size="50%" align="center" />
    * 最后输出下面的内容则编译成功

        ```bash
        Linking out/klipper.elf
        Creating hex file out/klipper.bin
        ```
