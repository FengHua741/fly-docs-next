# 固件烧录说明

    * ERCF V2可以使用RS232，CAN两种连接方式与上位机连接
    * RS232与CAN需要使用使用`XT30(2+2)-F`进行连接
    * 下方是`XT30(2+2)-F`数据线样式
        <ImageView image={require('@site/docs/General/get-id/img/xt30.webp').default} size="80%" align="left" />

:::danger 须知
* 主板固件烧录需要通过Katapult烧录Klipper
* 如果确定主板已经烧录了Katapult则无需进行最后面的烧录Katapult固件，那只是预防主板的Katapult固件没烧录成功
* 可以快速按两次重置后看主板上是否有一个LED闪烁起来，如果有则代表主板已经进去Katapult固件
:::

## 如何判断你需要的哪一种固件

    * ERCF V2可以使用以下方式连接上位机
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">CAN固件烧录</Button>
        * 好处：如果刷了`Katapult`固件可以无需触碰主板情况下更新固件，并且支持总线连接
        * 坏处：可能因使用`KlipperScreen`一起使用可能导致归位超时等报错。
        * 提示：如果需要使用CAN请确定上位机是否支持CAN，并且需要搭配UTOC或者刷了桥接固件的主板
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./rs232">RS232固件烧录</Button>
        * 好处：不需要特殊软件即可使用，通信可靠，稳定
        * 坏处：重新刷固件需要进入手动DFU才可以更新固件或者双击RST从可以进去`Katapult`来更新固件
        * 提示: 需要搭配G2T或者UTOR才可以使用
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB固件烧录</Button>
        * 好处：如果刷了Katapult固件可以无需触碰主板情况下更新固件，如果使用内存卡或者DFU烧录的还是需要按原来方式更新固件，通信可靠，稳定
        * 坏处：如果多个USB设备可能会影响数据传输
