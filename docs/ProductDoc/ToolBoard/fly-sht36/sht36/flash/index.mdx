# 固件烧录说明

    * SHT36/42可以使用CAN连接方式与上位机连接
    * CAN需要使用MX3.0金属端子压线
        <ImageView image={require('@site/docs/ProductDoc/ToolBoard/fly-sht36/sht36/img/utoc.webp').default} size="80%" align="left" />

## 如何判断你需要的哪一种固件

    * SHT36/42可以CAM方式连接上位机
    * <Button variant="contained" disableElevation href="./can">CAN固件烧录</Button>
        * 好处：如果刷了`Katapult`固件可以无需触碰主板情况下更新固件，并且支持总线连接
        * 坏处：可能因使用`KlipperScreen`一起使用可能导致归位超时等报错。
        * 提示：如果需要使用CAN请确定上位机是否支持CAN，并且需要搭配UTOC或者刷了桥接固件的主板
