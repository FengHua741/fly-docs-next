    * Halo安装到Stealthburner需要使用我们提供的适配件
    * 先下载我们提供的两个STL并自行打印
        * [Stealthburner_main_body(Adapt_to_Halo).stl(点击下载)](https://cdn.mellow.klipper.cn/Utils/Stealthburner_main_body(Adapt_to_Halo).stl)
        * [Halo_mounting_bracket.stl(点击下载)](https://cdn.mellow.klipper.cn/Utils/Halo_mounting_bracket.stl)
        * [Stealthburner_CW2_Assembly-2 v9.step(点击下载)](https://cdn.mellow.klipper.cn/STEP/Stealthburner_CW2_Assembly-2-v9.step)
