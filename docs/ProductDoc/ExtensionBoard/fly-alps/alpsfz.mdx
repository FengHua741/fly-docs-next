# 如何调整模块触发值

## 打开调平模块
    `Klipper`控制台发送`SET_PIN PIN=_probe_ready VALUE=1`，观察触发指示灯，调整阈值由小到大至指示灯不亮。输入下面命令：
    ```cfg
    SET_PIN PIN=_probe_ready VALUE=1
    ```

## 调整压力值
    <Button variant="contained" disableElevation component="a" rel="noopener" href="https://apps.microsoft.com/detail/9nblggh43hdm?launch=true&mode=full&hl=en&gl=hk&ocid=bingwebsearch">串口工具助手获取</Button>

    * 使用usb数据线连接模块跟电脑，打开串口工具助手，选择对应的端口号，波特率115200，点击连接。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-usb.webp').default} size="10%" align="left" />
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/fly-alps/img/apls-1.webp').default} size="100%" align="left" />
    串口工具输入下面命令调整触发值：
    ```cfg
    set 触发值
    ```
    默认触发值：20000
    数值越小越灵敏，数值越大越不灵敏。
    * 例如：`set 20000`，设置触发值为20000，数值越小越灵敏，数值越大越不灵敏。
    * 串口工具使用命令:

    * 串口发送`v`,实时输出abc值。a:滤波前；b:滤波后。
    * 串口发送`uv`，关闭输出。
    * 串口发送 `rt`,查看当前阈值。
