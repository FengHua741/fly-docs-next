# BDsensor-m

    BD传感器是一个可以与3D打印机配合使用的距离传感器，可以实时进行快速床调平和调整z轴。
    <ImageView image={require('@site/docs/ProductDoc/ExtensionBoard/bd-sensor/img/sensor.webp').default} size="50%" align="left" />

## 功能

    * 代替z轴限位
    * 扫描时无需升降z轴，提高打印速度
    * XY移动扫描,可进行密集网床探测



## 连接

    * 电源：5V
    * 固件：Klipper/Marlin
    * 连接：主板（BDsensor将使用主板的exp1接口）
    * 热床：支持：普通PEI，铝板热床等金属表面热床，不支持磁铁打孔热床与非金属表面热床


## 注意事项

    * KLipper需要下载需要下载klipper与最新的BD-SENSOR-m补丁，并且需要重新刷主板固件
    * 调试时候请将急停放着可以随时触发的地方，以免刮花PEI
    * 扫床速度建议200速！
    * Z轴归位速度不建议超过5
