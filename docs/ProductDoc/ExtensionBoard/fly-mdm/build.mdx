### 模型文件说明

| 模型文件名称 | 描述 |
| :---: | :--- |
| pcb_cover.stp | PCB盖子 |
| switch_sensor_trigger_block.stp | 断料检测触发块 |
| rear_cover.stp | 后盖 |
| upper_cover.stp | 上盖 |
| main_body.stp | 模型主体 |
| encoder_3_5.8mm.stp | 编码器码盘，3波段，最小检测距离5.8mm <br/> `高流量(大于 20 mm³/s)打印可使用这个` |
| encoder_5_3.5mm.stp | 编码器码盘，5波段，最小检测距离3.5mm <br/> `适中，建议使用` |
| encoder_10_1.73mm.stp | 编码器码盘，10波段，最小检测距离1.73mm <br/> `低速高精打印可使用这个` <br/> 受限于3D打印机固件耗材检测的采样率限制 |


## 打印组装

:::warning 模型打印注意事项

- 外壳不建议使用 `PETG` 和 `透明材料` 打印，会影响传感器感应。

:::
