# 产品简介

  - MDM断堵料传感器，体积小巧，使用简单，触发灵敏。
  - 可用于Klipper，Marlin，RRF等3D打印机
  -	断料检测：高速红外扫描，0.25 秒捕捉耗材断裂瞬间，立即暂停打印并报警，避免空打损耗；
  - 堵料预警：编码器光感信号识别，识别喷头 / 喉管堵塞前兆，减少废件率超 80%。
  - 广泛兼容：支持 PLA、ABS、PETG、TPU 等全类型线材（直径 1.75mm），无论是刚性材料还是柔性耗材，均能稳定识别；
  - 安装简单：提供孔距 图纸 接线图和配置文件的材料方便安装使用
