е®үиЈ… BDsensor

е°Ҷдј ж„ҹеҷЁз”өзјҶиҝһжҺҘеҲ°дё»жқҝжҲ–CANжҖ»зәҝе·Ҙе…·еӨҙжқҝгҖӮвҖӢ

  • иҜ·жіЁж„ҸSB2040ж— жі•дҪҝз”ЁBDsensor
  • иҜ·жіЁж„ҸSHT36йңҖиҰҒе°ҶBDsensorзҡ„CLK/SCL (Input)жҺҘеҲ°й«ҳеҺӢиҫ“е…ҘеҸЈпјҢе№¶дё”е°Ҷи·ізәҝжҺҘдёҠ
  • BDsensorзҡ„CKLе’ҢSDAзәҝеҸҜд»ҘиҝһжҺҘеҲ°з”өи·Ҝжқҝзҡ„д»»дҪ•GPIOеј•и„ҡгҖӮжӮЁд№ҹеҸҜд»Ҙе°ҶBDдј ж„ҹеҷЁз”өзјҶзӣҙжҺҘиҝһжҺҘеҲ°Bltouchз«ҜеҸЈпјҢдҫӢеҰӮпјҡ
BLtouch    |    BDsensor
5V       -->     5V
GND      -->     GND
S        -->     CLK/SCL    (Input)
GND      -->     GND
Zmin     -->     SDA    (Input/Output)

  • з”ұдәҺдё»жқҝиҝһжҺҘеҷЁдёӯзҡ„жҹҗдәӣеј•и„ҡеҸҜиғҪжІЎжңүзӣҙжҺҘиҝһжҺҘеҲ°MCUзҡ„gpiosпјҲдҫӢеҰӮпјҢе®ғд»¬дёҠеҸҜиғҪжңүж»Өжіўз”өе®№еҷЁжҲ–йҖҡиҝҮMOSFETпјҢдәҢжһҒз®ЎжҲ–е…үиҖҰеҗҲеҷЁйҡ”зҰ»пјҢдҪҶеҰӮжһңе®ғд»¬йҖҡиҝҮз”өйҳ»жҲ–з”өйҳ»дёҠжӢү/дёӢжӢүйҡ”зҰ»д№ҹеҸҜд»ҘпјүпјҢеӣ жӯӨе®ғд»¬дёҚиғҪдёҺBDsensorдёҖиө·дҪҝз”ЁгҖӮе№¶дё”еӣәд»¶е°ҶжҠҘе‘ҠиҝһжҺҘй”ҷиҜҜгҖӮдҫӢеҰӮ

  • йЈҺжүҮе’ҢеҠ зғӯеҷЁзҡ„иҝһжҺҘеҷЁйҖҡиҝҮ MOSFET йҡ”зҰ»пјҢ

  • жҹҗдәӣз”өи·Ҝжқҝдёӯз”ЁдәҺжё©еәҰзғӯж•Ҹз”өйҳ»е’Ңз«ҜжҢЎ/жҺўеӨҙзҡ„иҝһжҺҘеҷЁйҖҡеёёйҖҡиҝҮж»Өжіўз”өе®№еҷЁиҝһжҺҘеҲ° GNDпјҢ

  1. еҰӮдёӢеӣҫжүҖзӨәпјҢе°ҶBDдј ж„ҹеҷЁе®үиЈ…еңЁйқ иҝ‘зғӯз«Ҝзҡ„дҪҚзҪ®гҖӮ STL of mount, STL_mount_VzBot_Goliath short

зј–иҫ‘ printer.cfgвҖӢ

  • е°ҶиҜҘйғЁеҲҶеӨҚеҲ¶еҲ°еҲ°жӮЁзҡ„printer.cfgе№¶зј–иҫ‘[BDsensor] sda_pin scl_pinпјҢд№ҹдёҚиҰҒеҝҳи®°зҰҒз”Ёе…¶д»–жҺўй’ҲйғЁеҲҶпјҢеҰӮ BLtouchгҖӮжӮЁеҸҜд»ҘиҝһжҺҘдё»жқҝжҲ–е·Ҙе…·еӨҙCANжЁЎеқ—дёҠзҡ„BDдј ж„ҹеҷЁпјҢ

  • еңЁ[BDsensor]дёӯе°Ҷspeedдҝ®ж”№дёә 0.8гҖӮиҝҷд»…йҖӮз”ЁдәҺ z еҖҫж–ңе’ҢPROBE_ACCURACYе‘Ҫд»Ө.еҖји¶Ҡе°ҸпјҢжҺўжөӢж—¶зҡ„зІҫеәҰи¶Ҡй«ҳпјҢеӣ дёә MCU еңЁеҪ’дҪҚж—¶иҜ»еҸ–дё»еӣһи·Ҝдёӯзҡ„ BDдј ж„ҹеҷЁдёҚеғҸжӯЈеёёзҡ„жӯўеҠЁеҷЁйӮЈж ·е®һж—¶гҖӮ [BDsensor]

  • е°ҶBDдј ж„ҹеҷЁеңЁZиҪҙеӣһеҺҹзӮ№ж—¶дҪңдёәйҷҗдҪҚејҖе…ійңҖиҰҒе°Ҷ[stepper_z] дёӯзҡ„endstop_pinж”№дёәendstop_pin: probe:z_virtual_endstop

  • иҜ·зЎ®дҝқprinter.cfgдёӯжңү[safe_z_home]

  • е°Ҷ[bed_mesh]е’Ң[z_tilt]жҲ–иҖ…[quad_gantry_level]дёӯзҡ„[quad_gantry_level]еҖјж”№дёә1пјҲе»әи®®0.7-1.0mmпјүеңЁklipperзҡ„й»ҳи®ӨеҖјдёә5mmпјҢеҗҰеҲҷе®№жҳ“и¶…еҮәдј ж„ҹеҷЁиҢғеӣҙ

  • е–·еҳҙй«ҳеәҰеҸӘйҖӮеҗҲеңЁz_adjust:и®ҫзҪ®пјҢжӯЈж•°жҳҜиҙҙиҝ‘зғӯеәҠпјҢиҙҹж•°жҳҜиҝңзҰ»зғӯеәҠпјҢе…¶е®ғи°ғе–·еҳҙй«ҳеәҰзҡ„и®ҫзҪ®йғҪдјҡжңүbug

  • жү“ејҖеҝ«йҖҹжү«еәҠйңҖиҰҒе°Ҷno_stop_probe:trueеүҚйқўзҡ„#еҲ йҷӨ

  • дёӢйқўжҳҜдёҖдёӘй…ҚзҪ®зӨәдҫӢгҖӮ

    [BDsensor] 
    scl_pin:PC6  # иҲөжңәдҝЎеҸ·еҸЈ
    sda_pin:PC3  # йҷҗдҪҚдҝЎеҸ·еҸЈ
    delay: 20 # 20us per pulse, this value should be >=20 but must be below 50
    z_offset:0 # this `z_offset` must be set to 0. 
    z_adjust:0.0 # z axis adjustment, replace the z_offset function. within -0.3 to 0.3mm
    x_offset: -34
    y_offset: 0
    #no_stop_probe:true # enable this for fast probe, the toolhead will not stop at the probe point.
    position_endstop: 0.8 # the Z axis will stop at this position (mm) while homing z, recommend value is 0.4~1.0
    #speed:0.8 # this speed only works for the z tilt and PROBE_ACCURACY command.

    [stepper_z]
    endstop_pin: probe:z_virtual_endstop 
    #position_endstop: 0.5
    homing_speed: 5
    second_homing_speed: 0.8

    [bed_mesh]
    speed: 200
    horizontal_move_z:1
    algorithm: bicubic

    [quad_gantry_level]
    horizontal_move_z:1

  • жҺ§еҲ¶еҸ°еҸ‘йҖҒgcodeе‘Ҫд»ӨM102 S-6пјҢжү“еҚ°жңәжҜҸж¬Ўе°ҶZиҪҙзј“ж…ўеҗ‘дёҠз§»еҠЁ0.1mmпјҢзӣҙеҲ°иҫҫеҲ°4mmгҖӮиҜ·еӢҝеңЁе®үиЈ…дј ж„ҹеҷЁд№ӢеүҚиҝҗиЎҢM102 S-6пјҢд№ҹдёҚиҰҒеңЁж ЎеҮҶж—¶е…ій—ӯжү“еҚ°жңәз”өжәҗпјҢеҗҰеҲҷж—§зҡ„ж ЎеҮҶж•°жҚ®е°Ҷиў«еҲ йҷӨгҖӮеҰӮжһңеңЁиҝҷз§Қжғ…еҶөдёӢпјҢеҸӘйңҖеҶҚж¬Ўж ЎеҮҶ
