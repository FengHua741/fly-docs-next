# 如何调整模块触发值
使用串口工具，以下命令可用：
Klipper控制台发送“SET_PIN PIN=_probe_ready VALUE=1”，观察触发指示灯，调整阈值由小到大至指示灯不亮。输入下面命令：
```cfg
SET_PIN PIN=_probe_ready VALUE=1
```
串口工具输入下面命令调整触发值：
```cfg
set 触发值
```
* 串口工具使用命令:

* 串口发送`v`,实时输出adc值。a:滤波前；b:滤波后。
* 串口发送`uv`，关闭输出。
* 串口发送 `rt`,查看当前阈值。
