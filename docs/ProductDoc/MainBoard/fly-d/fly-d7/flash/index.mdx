    在编译烧录固件之前，请务必阅读此栏！！！

:::tip 注意事项
* D5在2024年9月1号出厂已经预刷好Katapult固件
* 只要烧录过一次BL，后续无需再烧录BL（D5的BL为Katapult固件）
* bl只是一个引导程序，用于引导Katapult固件，还需要刷入klipper固件
:::

## 判断你需要的固件

    :::tip

    * 请注意你只需要刷其中一种固件即可

    :::
    
    * D7可以使用以下两种方式连接上位机
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB固件（最常用）</Button>
        * 优点：通信可靠，稳定
        * 缺点：如果多个USB设备可能会影响数据传输
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB桥接固件（常在使用CAN工具板时使用）</Button>
        * 优点：可以接多个设备并且不占用USB口
        * 缺点：可能因使用`KlipperScreen`一起使用可能导致归位超时等报错。
