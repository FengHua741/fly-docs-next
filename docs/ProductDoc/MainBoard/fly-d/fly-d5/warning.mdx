# 主板注意事项

## 电源指示灯
    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-d/fly-d5/img/power.webp').default} size="80%" align="center" />
    * 只通过TYPE-C供电并不接其他主板电源时候至少亮两个灯`3.3V`与`5V`
    * 在接入主板电源后主板电源灯应该亮`3.3V`、`5V`、`VCC`

### 问题排查


    :::danger 须知

    * 在没有任何外接设备情况下（驱动、限位、12864等）
    * 只通过TYPE-C供电至少会常亮`3.3V`与`5V`两个LED

    :::

    * 如果没有灯亮起来，代表有短路，需要联系客服售后
    * 如果`3.3V`、`5V`中有一个灯闪烁，代表有短路，需要联系客服售后

## 驱动

    * 主板支持外置驱动     <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/ModuleDrive/servo-drive">接线与配置教程</Button>

## 限位口

    * 请注意接线顺序，三线限位需要注意5V、GND、S这三个如果接错很可能导致主板损坏
    * 如果两线直接接S与GND即可
    * PL08与其他接近开关需要在信号口外接二极管将电压降低到3.3V以下
