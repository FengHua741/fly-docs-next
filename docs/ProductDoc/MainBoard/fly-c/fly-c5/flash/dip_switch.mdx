# 拨码介绍

* FLY C5 通过拨码选择下位机与上位机的连接方式，如果拨码的位置不对会导致板载的下位机无法正常使用！！！



## 板载拨码说明

:::danger 须知
- 正常使用时，请将`拨码向上拨`
- 拨码时，请向`上拨到底`，否则可能无法连接下位机！！！
:::
