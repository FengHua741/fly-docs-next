## 驱动器安装

### 主板驱动跳线

* **STEP/DIR模式。** 在这个模式下，通过使用跳线帽来设置细分，通过调节电位器来调整电流。最常见的是A4988、8825驱动需要查询厂家提供的细分配置表，然后通过跳线帽来设置细分。
* **UART模式。** 最常见的使用这种模式的驱动有：TMC2208、TMC2209、TMC2226等。这类驱动芯片可以与主控进行UART异步串行通信，可以通过修改配置文件来设置驱动的细分、运行电流、静音模式等。
* **SPI模式。** 最常见的使用这种模式的驱动有：TMC5160、TMC2130、TMC2240样的也可以通过修改配置文件来设置驱动的细分、运行电流、静音模式等。

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/drive-dip.webp').default} size="40%" align="center" />

----

### 驱动的安装

* 驱动在安装前需要进行相应的检查，以免对驱动或主板造成损坏。
* **FLY驱动：** 如果不使用无限位归零功能，请将拨码开关拨到1的位置；相反如果要使用无限位归零功能，请将拨码开关拨到ON的位置。

    <ImageView image={require('@site/docs/ProductDoc/MainBoard/fly-super/fly-super8/img/wiring/fly2209.webp').default} size="40%" align="center" />

    :::danger

    * **注意！！！ 安装驱动模块时，一定要保证插入的方向正确，即EN引脚在左上角，否则会损坏驱动甚至主板！！！注意给驱动贴好散热片！！！**

    :::

## 步进电机接线

:::info
* 不管是哪个厂家生产的两相步进电机，它的最终出现方式对应的就是4根线。不管是什么颜色，什么黑，白，绿，黄好，对应的都是4根线。所以我们要把线进行分组，分成A,B两组来。
* 对于两相四线步进电机来说，我们不需要知道哪个是A组哪个是B组，主要是确定是一组即可，在配置里面修改电机方向即可
:::

* 在3D打印机中，最常用的是两相四线步进电机，其原理如图所示。鉴别步进电机线序有两种方法：
    * 将任意两根相线连接，手拧电机轴，若产生明显阻力，则两根相线互为一组。反之，则不为一组，需测试另外的相线。
    * 使用万用表，调至通断档位，测试步进电机的任意两根相线，若相通，则为一组；反之，则不为一组，需测试另外的相线。

## 限位开关接线

* 限位开关有常开（NO）和常闭（NC）两种。一般在3D打印机上，建议使用**常闭（NC）**，这样在限位开关线路出现问题时，系统会及时报错，可以避免一些不必要的撞车，避免损坏打印机。

### BL-Touch接线

* BL-touch一共有五根线，三根为第一组，负责传感器的供电和探针的收放，第二组为地线和信号线，输出限位信号。BL-touch接线时请仔细核对线序，错误的接线可能会永久性损坏传感器和主板！！！接线方法如下图所示。

### Voron Tap接线

* Tap是一种基于喷嘴的z探针，适用于V2和Trident打印机设计。整个工具头移动以触发光开关，精度比常规限位开关更好，可以使用几乎市面上所有的打印平台。

    :::warning

    Voron Tap不建议接 **24V** ，某些版本使用 **24V** 有一定概率会导致Tap传感器烧坏，这并非Fly产品的问题，是Voron Tap的设计缺陷，请悉知！！！

    :::

    :::danger

    注意：5V和GND不能接反，否则会损坏Tap传感器甚至主板！！！

    :::
