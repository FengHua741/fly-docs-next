# 固件烧录说明

    :::warning 注意事项

    * PRO X10预刷的是Katapult
    * 只要烧录过BL后续无需在烧录BL
    * PRO X10的BL有两种，分别为Katapult固件与TF卡烧录的BL固件
    * 如果有需要可以根据自己选择重新<Button variant="contained" disableElevation href="./bl">BL固件</Button>

    :::
    
    :::danger 须知
    * 主板固件烧录需要通过Katapult烧录Klipper
    * 如果确定主板已经烧录了Katapult则无需进行最后面的烧录Katapult固件，那只是预防主板的Katapult固件没烧录成功
    * 可以快速按两次重置后看主板上是否有一个LED闪烁起来，如果有则代表主板已经进去Katapult固件
    :::

## 如何判断你需要的哪一种固件

    :::tip

    * 请注意你只需要刷其中一种固件即可

    :::
    
    * Pro X10可以使用以下方式连接上位机
    * <Button variant="contained" disableElevation href="./usb">USB固件</Button>
        * 好处：通信可靠，稳定
        * 坏处：如果多个USB设备可能会影响数据传输
    * <Button variant="contained" disableElevation href="./can">USB桥接固件</Button>
        * 好处：可以接多个设备并且不占用USB口
        * 坏处：可能因使用`KlipperScreen`一起使用可能导致归位超时等报错。
