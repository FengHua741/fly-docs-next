# 主板注意事项

## 电源指示灯
    * 在使用typec供电并不接其他主板电源时候应该亮`SYS_5V`与SYS_3.3V
    * 在接入主板电源`HVIN`、`VIN`、`BOARO`、`POWER`后主板电源灯应该常亮

## 驱动

    * TP5160与TP2209只是命名不同，并且不需要跳线即可选择驱动模式
    * 驱动口只能安装TP驱动并不能安装其他设备

## 限位口
    * ProX10支持5V与12V电源选择，请注意如果使用接近开关需要安装二极管进行降压

## 上位机
    * ProX10支持使用外置上位机，拨码选择连接Type-c后即可使用

## 固件烧录

    * 主板支持使用TF卡进行烧录，但是出厂默认是`Katapult`
    * 如果需要使用TF卡进行烧录需要重新烧录BL
