    :::tip

    * Super8 PRO默认使用TF卡进行固件烧录
    * 请准备一张容量不超过`32G`TF卡
    * 杂牌内存卡可能导致烧录失败，请尽量使用下方推荐的内存卡进行更新固件
    * TF卡的传输速度最好为`class10`级或`class10`级以上，建议使用闪迪等品牌的TF卡

    :::

# 固件烧录说明

    :::tip

    * Super8 Pro出厂已经预刷TF卡bootloader，无需自己在刷一次

    :::

## 如何判断你需要的哪一种固件

    :::tip

    * 请注意你只需要刷其中一种固件即可

    :::
    
    * Super8 PRO可以使用以下方式连接上位机
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./usb">USB固件</Button>
        * 好处：通信可靠，稳定
        * 坏处：如果多个USB设备可能会影响数据传输
    * <Button variant="contained" disableElevation component="a" rel="noopener" href="./can">USB桥接固件</Button>
        * 好处：可以接多个设备并且不占用USB口
        * 坏处：可能因使用`KlipperScreen`一起使用可能导致归位超时等报错。
