## 产品简介

    * 屏幕尺寸：120.7mm*76.0mm
    * 屏幕显示区域：108mm*66.2mm
    * 分辨率：800*480
    * 屏幕类型：IPS
    * 通信方式：DSI/Mipi
    * 板载接口：15 pin DSI / 22pin Mipi
    * 工作稳定：0°-60°C
    * 触摸类型：Capacitive（电容式）
    * 逻辑电压：3.3V
    * 重量：135.5g


            * 如果使用0号接口需要按下方配置屏幕

            ```bash
            dtoverlay=vc4-kms-dsi-7inch,dsi0
            ```
