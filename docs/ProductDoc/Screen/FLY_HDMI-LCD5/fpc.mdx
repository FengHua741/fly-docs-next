# 屏幕参数修改教程
:::tip
浏览器打开你的IP:9999来访问FLY-Tools
:::

:::tip
FLY 上位机板载的HDMI接口是Micro HDMI，购买HDMI线时请注意！！！
:::

:::tip
FLY的上位机支持FPC一线通的有`FLY-PI-V2`,`FLY-C8`与`FLY-PRO-X10`或`FLY_LITE2/2.1`

:::
    * 使用HDMI需要将下方选项打开才可启用屏幕
    * 进入FLY Tools后
* 第一步需要进入fly_TOOLS修改参数
