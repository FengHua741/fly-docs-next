# 简介

* 该设备采用独特的 FPC（柔性印刷电路）端口设计，可直接连接到 Fly 设计的主机主板，无需使用 HDMI 和 USB 电缆。这种简化的 FPC 连接减少了电缆杂乱，并确保了干净高效的接口。尽管采用了这种先进的连接方法，但该设备还为喜欢传统连接选项的用户保留了标准 HDMI 端口。
除了灵活的连接性外，该设备还包括智能功能键，可轻松调节亮度、旋转屏幕和重启。高品质进口泡沫粘合剂可确保强大的屏幕粘合力和出色的耐用性。双面低阻散热技术增强了触摸灵敏度，为用户体验更流畅。黄光技术提供精确的色彩还原，标准亮度为 500，避免过度曝光。屏幕由高强度钢化玻璃保护，确保坚固的防损坏保护。
该设备与 Raspberry Pi 等主机系统完全兼容，还采用隐藏式 HDMI 端口设计，电缆整齐地隐藏在设备后面，外观干净专业。其高性能电路设计保证了无缝运行，使其成为性能稳定可靠的3D打印机的理想选择
