# 连接上位机SSH

* 可以使用**串口**或者**WiFi**连接到设备
* 请提前准备好**MobaXterm_Personal**或其他**SSH终端工具**
* 请确定SD卡或者M2WE已经烧录**与上位机相匹配的系统**
* **如果上位机不能正常启动请把驱动，限位，风扇，等外设拆除！！！**

:::warning 注意
- 如果您的上位机为第一次启动，请务必使用**串口连接到上位机SSH**
- 编译固件时，请务必使用**WiFi连接到上位机SSH**，以方便固件编译和烧录！
- `FLYOS-Armbian`系统与`FLYOS-FAST`系统 连接至SSH的操作有所区别，请根据您烧录的系统选择相应的教程！！！
:::

    <TabItem value="net" label="FLYOS-Armbian系统 WiFi连接" default>
        ## FLYOS-Armbian WiFi连接到上位机SSH

        * 请确保上位机系统已经**正常启动**
        * 请确保**明确上位机的用户名与密码**。FLY系统默认用户名`fly`, 默认密码`mellow`
        * 通过WiFi连接到上位机SSH的**前提**是：您的**上位机已经连接到路由器，并且知道上位机的IP**  

        * 打开MobaXterm，设置如下图所示，`Remote host`处则填写您在**串口连接时查询到的上位机IP**，也可以通过路由器后台查询上位机**IP**

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-1.webp').default} size="60%" align="center" />

        **注意：这里第4步输入的上位机的用户名**
        :::tip
        - 第四步输入上位机的用户名。fly上位机且使用**FLYOS-Armbian**系统的则填写`fly`，其他上位机请根据实际情况填写正确的用户名
        :::

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-9.webp').default} size="60%" align="center" />

        * 第一次连接回弹出这个窗口，点击`Accept`即可

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-10.webp').default} size="50%" align="center" />

        * 提示输入密码，输入上位机密码后回车
        :::tip
        - 这里输入的密码是看不到的，输完后直接回车即可
        :::
        
        **FLYOS-Armbian** 系统默认用户名为：`fly` ，密码为：`mellow`

        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-11.webp').default} size="60%" align="center" />

        **如果弹出下图所示界面，此弹窗提示是否保存用户密码，点击`No`即可**
        <ImageView image={require('@site/docs/General/flyos-fast/img/flyos-fast_ssh_4.webp').default} size="60%" align="center" />

        * 出现下图所示界面，则登录成功！
        <ImageView image={require('@site/docs/DebugDoc/BasicTutorial/img/mobaxterm-12.webp').default} size="60%" align="center" />


    </TabItem>
