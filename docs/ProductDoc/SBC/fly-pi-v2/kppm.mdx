# Klipper断电续打功能

## 概述

- Klipper断电续打功能是指在Klipper断电后，再次上电后可以自动恢复Klipper的打印状态。
- 需要搭配FAST系统才可以使用断电续打功能
- PI-V2需要搭配KPPM断电模块才可以使用
- 不适用于停电后Z轴会发生位移的机型。
