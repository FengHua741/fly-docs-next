    <TabItem value="tft" label="TFT电容屏配置" default>
## 方法一    
         * 电容配置需要使用下方命令配置成`fly-tft-v1`
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v1/' -i /boot/FlyOS-Env.txt
```
         * 请确认屏幕背面拨码拨到`Cap`
    * 使用下方命令查看是否修改成功
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```

 * 电容屏修改电阻屏配置使用下方命令

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-c/screen=fly-tft-v1/}' /boot/FlyOS-Env.txt
``` 
</TabItem>

        <TabItem value="hdmi" label="TFT电阻屏配置">
## 方法一        
           * 电阻屏配置需要使用下方命令配置成`fly-tft-v1` 
```cfg
sudo sed '11s/screen=none/screen=fly-tft-v1/' -i /boot/FlyOS-Env.txt
```
                                * 请确认屏幕背面拨码拨到`Resi`
        * 使用下方命令查看是否修改成功
```cfg
sed -n '9,11p' /boot/FlyOS-Env.txt | grep -v '^#'
```

 * 电阻屏修改电容屏配置使用下方命令

```cfg
sudo sed -i '11{/^#/!s/screen=fly-tft-v2-r/screen=fly-tft-v1/}' /boot/FlyOS-Env.txt
```
</TabItem>
