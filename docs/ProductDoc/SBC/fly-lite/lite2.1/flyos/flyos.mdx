# 系统烧录

## 上位机新手操作指南

    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/DebugDoc/baodian/host/">上位机新手操作指南</Button>

## 系统烧录前准备

    * 读卡器
    * TF卡一张，容量必须不小于16GB 或 不大于128G TF 卡，TF 卡的传输速度必须为 class10 级或 class10 级以上
    * 下载对应的 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/system-img/fly-lite2">系统镜像</Button>
    * 下载需要的 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">烧录工具</Button>

## 系统烧录
:::info
* MINI PAD 只能将系统烧录到TF卡,无法使用M2WE
* 系统烧录完成后要将TF卡安装到MINI PAD上
:::
    
    <TF />

## 安装系统卡

    * 在将系统烧录到TF卡后，需要将TF卡接到LITE2.1，否则系统无法启动
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/tf.webp').default} size="100%" align="left" />  

## 安装天线

    1. 安装好天线
    :::warning
    * 请注意，上位机不支持中文名字的WIFI，不支持5G的WiFi
    :::
    * 红色箭头所指的地方为WIFI天线接口
    * WIFI天线口是`ipex1`
    * 请将天线安装上，否则可能因为信号差导致无法连接至路由器
    <ImageView image={require('@site/docs/ProductDoc/SBC/fly-lite/lite2.1/img/wifi.webp').default} size="80%" align="left" /> 

    :::tip
    - 系统卡安装完成后即可看下一步，通过SSH连接上位机
    :::

    2. 给上位机供电。  
    **到这里，您的上位机已经可以启动了！！！**  
    请执行下一步：<Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ProductDoc/SBC/fly-lite/lite2.1/ssh">SSH连接上位机</Button>
