# 屏幕接线与配置

:::info

* FLY-TOOLS需要上位机联网后输入上位机的`IP+:9999`访问`使用浏览器打开`
* 比如你的ip地址是192.168.6.110,那么你需要在浏览器中输入`192.168.6.110:9999`

:::
