# 世界时区对照表

:::tip 必读

- 时区设置将时区ID填入`localtime`配置项中，例如：`Asia/Shanghai`。
- 重启系统后生效。
- 系统在没有联网前时间是错误的，联网后时间会自动同步。

:::
