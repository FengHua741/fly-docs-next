# 系统内置组件介绍

:::danger

- FlyOS-Fast系统内置的所有组件及服务均不支持用户自行修改，升级。
- 由您自行修改导致出现网页无法访问，系统无法启动等问题。
- 请重新烧录系统。

:::

## Klipper

:::warning

- 您可以尝试为现有Klipper添加第三方extras，但是不要修改现有的所有extras模块

:::

### 异常自动暂停

- 在打印过程中，Klipper出现温度异常，驱动异常等错误时，会自动暂停打印，不会直接停机。

### 实时性优化

- Klipper的实时性优化，通过修改内核与Klipper代码来实现，优化Klipper的实时性。
- 有效减少`Timer too close`错误。

### 断电续打，异常续打支持

- 支持断电续打。
- 支持报错停机续打。

### TCP套接字支持

- 支持同时启用Unix套接字和TCP套接字。
- 使用-p参数指定TCP端口来启用TCP套接字。

### U盘文件自动处理

- 支持自动处理U盘文件，避免打印中断。

## Moonraker

### TCP套接字支持

- 支持使用TCP套接字连接到klippy。
- `klippy_tcp_address`参数指定TCP地址。
- `klippy_tcp_port`参数指定TCP端口。

### 大文件上传优化

- 解决连续上传大文件出现错误的问题。

### 网络管理-适用于FLY5,7屏幕

- 支持无线网络管理，适用于FLY5,7屏幕。

### FLY屏幕支持

- 支持FLY屏幕。
- 支持模型首层图形生成。

## KlipperScreen

### 断电续打

- 支持断电续打。
- 检测到有续打时自动弹窗提醒。

## Fluidd

### 断电续打

- 支持断电续打。
- 检测到有续打时自动弹窗提醒。

## Mainsail

### 断电续打

- 支持断电续打。
- 检测到有续打时自动弹窗提醒。
