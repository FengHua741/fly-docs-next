# 上位机篇

## 新买的上位机我要如何才能跑起来

  1. 首先得确定你的上位机是什么型号是Lite 2还是PI V2等等上位机
  2. 有没有准备下面的东西
        * 烧录用的软件
        * 对应的系统镜像
        * 读卡器，TF卡，带数据传输功能的TYPE-C

  系统烧录前注意事项:
        * 请保证内存卡容量必须不小于`16GB`或不大于`128G`TF卡，并且传输速度必须为`class10`级或`class10`级以上，建议使用闪迪等品牌的TF卡
        * 请不要随意更新系统！！服务器在国外很容易更新出问题
        * 尽量使用我们推荐的软件来操作
        * 请尽量学一点LINUX知识我个人推荐 <Button variant="contained" disableElevation component="a" rel="noopener" href="https://item.jd.com/12443890.html">《鸟哥的Linux私房菜 基础学习篇》</Button> 等

  3. FLY系统镜像 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/category/系统镜像">系统镜像</Button>
  4. 需要的工具 <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">烧录工具</Button>

## 下载好各类工具与对应的系统镜像后

  * 解压系统，解压后文件后缀名为`.img`
  * 需要使用SDFormatter重新格式化内存卡
  * 使用`win32DiskImager`烧录到系统里面
  * 不建议使用`balenaetcher`进行系统烧录
  * 如果是FLY上位机需要打开`FlyOS-Env.txt`进行配置
  * 烧录好系统后请不要格式化！！！！

## 启动系统

  * 系统烧录好后需要将系统卡安装到上位机后在上电启动
  * 这个时候系统已经烧录配置完毕，需要使用带数据传输功能的TYPE-C连接电脑使用串口连接
  * 请注意其他上位机正常情况会显示日志加载，如果还没有显示需要按两下回车或者重新刷系统
  * 启动好后就可以连接WIFI


## 关于M2WE与内存卡使用说明

  * 如果机器安装了M2WE与内存卡，系统默认从内存卡启动
  * M2WE建议使用SD卡刷系统，使用通过USB烧录到M2W可能因为驱动兼容等原因烧录失败，建议直接使用内存卡烧录到M2WE
  * 请注意下载的系统镜像是否对应，如果不对应是无法启动的
  * 请注意Lite2与MINI PAD只能使用内存卡


## 日常使用注意事项

  * 请不要直接断电，需要正常关机否则可能会导致数据丢失
  * 因GitHud等服务器在国外，不建议自己更新或者安装各种插件
