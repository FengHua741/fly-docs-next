# 加热容差配置

    * 挤出机

    ```cfg title="printer.cfg"
    [verify_heater extruder]
    max_error: 120
    check_gain_time:120
    hysteresis: 50
    heating_gain: 2
    ```
       
    * 热床

    ```cfg title="printer.cfg"
    [verify_heater heater_bed]
    max_error: 120
    check_gain_time:120
    hysteresis: 50
    heating_gain: 2
    ```
