# 系统烧录TF卡

### 准备工作

    * 先安装好软件`SDFormatter` 与`Rufus`
    <Button variant="contained" disableElevation component="a" rel="noopener" href="/docs/ResDownload/auxiliary_software/">格式化工具</Button>
    * 将系统解压出来，解压后文件名的后缀有点为`.img`
    * 不建议使用`balenaetcher``Raspberry Pi`进行系统烧录
    * TF卡插入读卡器，将读卡器插到电脑USB接口
    * 如果TF卡有数据请备份刷完系统后系统卡不建议拆下来，除非需要烧录到M2WE后才可以拆下TF卡

### 格式化内存卡

    * 使用`SDFormatter.exe`软件格式化TF 卡
    * 选择你需要格式化的磁盘

    <ImageView image={require('@site/docs/General/fly-os/img/sdformatter.webp').default} size="80%" align="left" />

    * 或者使用WIN自带的磁盘管理删除分区,删除分区之后重新建立分区即可。
