


## 重要提示
    :::tip 必读

    - 配置文件中的`部分`设置只会在烧录后第一次启动系统时生效。
    - 因配置过程较为麻烦，强烈建议通过网络连接上机器后在通过浏览器进行配置
    - 本教程只提供可修改的选项

    :::

## 如何修改配置文件

### 烧录完成后修改配置文件

- 仅在`首次启动`时生效的配置项必须在烧录完成后，使用专业文本编辑器打开可移动磁盘`FlyOS-Conf`中的配置文件`config.txt`来修改。

### 系统已启动在线配置修改

- 如果是普通配置项在系统网页配置页面修改后重启系统即可生效。
    - 在浏览器打开设备`ip+:9999`地址，例如：`192.168.1.2:9999`
    - 支持主机名修改
    - 支持系统时区修改
    - 支持上位机40PIN扩展启用（请注意PWM启用后无法使用串口连接上位机，屏幕启用后可能无法使用上位机的加速度口）
    - 支持主板型号修改
    - 支持Klipper网页修改，支持Fuidd与Mainsail
    - 支持打开或者关闭KlipperScreen
    - 支持强制Klippy进程运行在指定的CPU核心，可指定Klipper在选择的核心下运行
