



## FLY上位机使用教程

:::tip
浏览器打开你的IP:9999来访问FLY-Tools
:::

:::tip
FLY 上位机板载的HDMI接口是Micro HDMI，购买HDMI线时请注意！！！
:::

:::tip
FLY的上位机支持FPC一线通的有`FLY-PI-V2`,`FLY-C8`与`FLY-PRO-X10`
:::

:::tip
使用FPC转HDMI模块可兼容树莓派等上位机
:::
