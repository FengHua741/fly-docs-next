


## CSI使用方法

    * 镜像烧写完成后，进入/boot/config.txt中找到`camera_auto_detect=1`并且修改成`camera_auto_detect=0`然后按着摄像头型号添加配置
    * 下方是参考配置

    ```bash
    dtoverlay=imx219,cam1
    ```
