

1. 安装烧录工具

* FLY上位机无需执行安装烧录工具
* 如果你需要安装请注意：上位机需要能上网才可以正常安装
* 如果已经安装过则无需安装

```bash
sudo apt update
sudo apt install dfu-util -y
```

2. 执行下面的命令查看是否连接成功。

```bash
lsusb
```

* 如果没有DFU设备请尝试重新进入**烧录模式**
* 因系统版本问题可能显示其他名字，如果有`0483:df11`则代表进入烧录模式

<ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.webp').default} size="100%" align="left" />


3. 拉取预先编译固件

* 这里面都是预先编译好的固件
* 请注意上位机需要能上网才可以安装
* 上位机需要执行下方命令下载固件包(如因上位机网络问题无法获取文件，请前往售后群联系管理员)

```bash
cd && git clone https://github.com/FengHua741/FLY_Katapult.git
```

* 如果已经下载过固件包的建议更新固件包后在进行剩下操作

```bash
cd ~/FLY_Katapult && git pull
```
