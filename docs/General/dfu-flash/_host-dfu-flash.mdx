

    1. 安装烧录工具

    * FLY上位机无需执行安装烧录工具
    * 如果你需要安装请注意：上位机需要能上网才可以正常安装
    * 如果已经安装过则无需安装

    ```bash
    sudo apt update
    sudo apt install dfu-util -y
    ```

    2. 执行下面的命令查看是否连接成功。

    ```bash
    lsusb
    ```

    * 如果没有DFU设备请尝试重新进入**烧录模式**
    * 因系统版本问题可能显示其他名字，如果有`0483:df11`则代表进入烧录模式


        <ImageView image={require('@site/docs/General/dfu-flash/img/lsusb.webp').default} size="80%" align="left" />    


    3. 烧录固件

    * 执行下方代码开始烧录
    
    ```bash
    sudo dfu-util -a 0 -d 0483:df11 --dfuse-address 0x08000000 -D ~/klipper/out/klipper.bin
    ```

    4. 出现下图箭头处所示内容``File downloaded successfully``则烧录成功

    <ImageView image={require('@site/docs/General/dfu-flash/img/make-flash.webp').default} size="80%" align="left" />    

    :::tip

    烧录完成后，需要给主板彻底断电一次。

    :::
