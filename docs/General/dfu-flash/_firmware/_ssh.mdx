

    * 需要使用安装好`klipper`服务的上位机  <Button variant="contained" disableElevation href="/docs/DebugDoc/BasicTutorial/">通过网络连接SSH</Button>
    * 并且将设备与上位机连接好
    * 注意：请使用普通的上位机，比如树莓派 FLY派等。如果是魔改上位机（例如WiFi棒子、红米等），这类上位机因问题太多，无法为您提供技术支持


    
    :::warning
    * 将`ROOT`用户切换到普通用户可以输入下方命令
    * 请不要使用`ROOT`用户
    * fly用户可直接输入：`su fly`，非fly用户请将`<您的上位机用户名>`,替换为您上位机的用户名！
    ```bash
     su <您的上位机用户名>
    ```
    :::
