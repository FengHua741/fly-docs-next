


    * 请注意当前工具板是处于固件烧录模式，只能烧后缀名为uf2的固件
    * 通过下方命令将固件烧录到工具板

        ```bash
        cd && cd ~/klipper/
        make flash FLASH_DEVICE=2e8a:0003
        ```
        :::tip
           如果是上位机写好系统后第一次烧录，可能会出现和下图红圈中一样的提示——出现部分红色提示，此为正常现象，并非报错。
        :::
